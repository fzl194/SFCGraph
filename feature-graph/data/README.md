## Files

- `data/feature_files.csv` — feature_id, vendor, file_path, page_type_candidate

## Usage
Query file list for a specific feature:
```python
import csv
with open('data/feature_files.csv', encoding='utf-8-sig') as f:
    for row in csv.DictReader(f):
        if row['feature_id'] == 'GWFD-110101' and row['vendor'] == 'UDG':
            print(row['file_path'])
```

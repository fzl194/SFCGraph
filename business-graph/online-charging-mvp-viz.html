在线计费 · 业务图谱实例化

按图索骥
从图谱到脚本

给定一条在线计费需求，先建立完整的业务图谱，再沿着图谱路径逐步产出配置脚本。

需求描述

某运营商5G用户在线计费策略：
• 视频业务：访问 video.huawei.com，按流量(VOLUME)计费，RG=100
• IM消息：IM类协议流量，按时长(DURATION)计费，RG=200
• 默认流量：未命中规则的流量，在线计费，RG=300
• 配额耗尽后阻断(BLOCK)

匹配方式：视频走L7 URL匹配，IM走L34协议匹配，默认走ANY匹配。

T-EXEC-002 · 配置三四层过滤条件
创建2个FILTER对象，为后续FLOWFILTER提供L34匹配基础
ADD FILTER:FILTERNAME="filter_any",L34PROTTYPE=STRING,L34PROTOCOL=ANY; ADD FILTER:FILTERNAME="filter_im",L34PROTTYPE=STRING,L34PROTOCOL=ANY;
产出：filter_any(ANY匹配，供视频和默认使用) + filter_im(IM协议匹配)
T-EXEC-003 · 配置七层过滤条件 gated
DP-03门控：仅视频业务需要L7FILTER，本案例仅创建1个
ADD L7FILTER:L7FILTERNAME="l7_video",URLTYPE=URL,URL="video.huawei.com/*";
产出：l7_video(匹配 video.huawei.com 的URL)
T-EXEC-004 · 配置流过滤器与绑定
创建3个FLOWFILTER，分别绑定FILTER和协议/L7条件
// 视频业务：L34 ANY + L7 URL ADD FLOWFILTER:FLOWFILTERNAME="ff_video"; ADD FLTBINDFLOWF:FLOWFILTERNAME="ff_video",FILTERNAME="filter_any"; ADD PROTBINDFLOWF:FLOWFILTERNAME="ff_video",PROTOCOLNAME="http",L7FILTERNAME="l7_video"; // IM消息：L34 + 协议绑定(无L7) ADD FLOWFILTER:FLOWFILTERNAME="ff_im"; ADD FLTBINDFLOWF:FLOWFILTERNAME="ff_im",FILTERNAME="filter_im"; ADD PROTBINDFLOWF:FLOWFILTERNAME="ff_im",PROTOCOLNAME="im"; // 默认流量：仅L34 ANY ADD FLOWFILTER:FLOWFILTERNAME="ff_any"; ADD FLTBINDFLOWF:FLOWFILTERNAME="ff_any",FILTERNAME="filter_any";
产出：ff_video(视频) + ff_im(IM) + ff_any(默认)，共3个FLOWFILTER + 3个FLTBINDFLOWF + 2个PROTBINDFLOWF
T-EXEC-005 · 配置计费动作链（3套三件套 + BLOCK）
每条业务创建 URR→URRGROUP→PCCPOLICYGRP 三件套；DP-02=BLOCK 额外创建 PCCACTIONPROP
// 视频业务：按流量在线计费 (DP-01=ONLINE, Feature=VOLUME) ADD URR:URRNAME="urr_video",URRID=1001,USAGERPTMODE=ONLINE,ONLMETERINGTYPE=VOLUME,RG=100; ADD URRGROUP:URRGROUPNAME="urrg_video",UPURRNAME1="urr_video",DOWNURRNAME1="urr_video"; ADD PCCPOLICYGRP:PCCPOLICYGRNM="PPG_video",URRGROUPNAME="urrg_video"; // IM消息：按时长在线计费 (DP-01=ONLINE, Feature=DURATION) ADD URR:URRNAME="urr_im",URRID=1002,USAGERPTMODE=ONLINE,ONLMETERINGTYPE=DURATION,RG=200; ADD URRGROUP:URRGROUPNAME="urrg_im",UPURRNAME1="urr_im",DOWNURRNAME1="urr_im"; ADD PCCPOLICYGRP:PCCPOLICYGRNM="PPG_im",URRGROUPNAME="urrg_im"; // 默认计费 + 配额耗尽BLOCK (DP-02=BLOCK) ADD URR:URRNAME="urr_default",URRID=1003,USAGERPTMODE=ONLINE,ONLMETERINGTYPE=VOLUME,RG=300; ADD URRGROUP:URRGROUPNAME="urrg_default",UPURRNAME1="urr_default",DOWNURRNAME1="urr_default"; ADD PCCACTIONPROP:PCCACTPROPNM="pccact_block",GATINGSTATUS=CLS; ADD PCCPOLICYGRP:PCCPOLICYGRNM="PPG_default",URRGROUPNAME="urrg_default",PCCACTPROPNAME="pccact_block";
产出：3个URR + 3个URRGROUP + 3个PCCPOLICYGRP + 1个PCCACTIONPROP(门控策略)
T-EXEC-008 · 配置Rule
创建3条RULE，每条绑定FLOWFILTER和PCCPOLICYGRP，按优先级裁决
ADD RULE:RULENAME="Rule_video",POLICYTYPE=PCC,FILTERINGMODE=FLOWFILTER,FLOWFILTERNAME="ff_video",PRIORITY=100,POLICYNAME="PPG_video"; ADD RULE:RULENAME="Rule_im",POLICYTYPE=PCC,FILTERINGMODE=FLOWFILTER,FLOWFILTERNAME="ff_im",PRIORITY=200,POLICYNAME="PPG_im"; ADD RULE:RULENAME="Rule_default",POLICYTYPE=PCC,FILTERINGMODE=FLOWFILTER,FLOWFILTERNAME="ff_any",PRIORITY=65000,POLICYNAME="PPG_default";
产出：Rule_video(P100) + Rule_im(P200) + Rule_default(P65000兜底)
T-EXEC-010 · 配置UserProfile与Rule绑定
创建UserProfile、设置默认URR组、绑定3条Rule、刷新生效
ADD USERPROFILE:USERPROFILENAME="up_online_charging"; SET URRGRPBINDING:USERPROFILENAME="up_online_charging",DFTURRGRPNAME="urrg_default",DFTSIGURRGNAME="urrg_default"; ADD RULEBINDING:USERPROFILENAME="up_online_charging",POLICYTYPE=PCC,RULENAME="Rule_video"; ADD RULEBINDING:USERPROFILENAME="up_online_charging",POLICYTYPE=PCC,RULENAME="Rule_im"; ADD RULEBINDING:USERPROFILENAME="up_online_charging",POLICYTYPE=PCC,RULENAME="Rule_default"; SET REFRESHSRV:REFRESHTYPE=ALL;
产出：up_online_charging(USERPROFILE) + 3个RULEBINDING + URRGRPBINDING + REFRESHSRV

## 3. ConfigObject 间关系边（§11.7）

### 3.1 contains 边（5条）

| 起点 | 关系 | 终点 | 说明 |
|------|------|------|------|
| MPLSSITE | `contains` | MPLSIF | MPLS 全局配置包含接口使能（SET MPLSSITE 后 ADD MPLSIF） |
| L3VPNINST | `contains` | VPNINSTAF | VPN 实例包含地址族（ipv4uni/ipv6uni，每实例可有多个地址族） |
| VPNINSTAF | `contains` | VPNTARGET | VPN 地址族包含 RT 属性（export_extcommunity + import_extcommunity 成对） |
| BGPVRF | `contains` | BGPVRFAF | BGP VPN 实例包含 BGP 地址族 |
| BGPPEER | `contains` | BGPPEERAF | BGP 对等体包含对等体地址族 |

### 3.2 refers_to 边（4条）

| 起点 | 关系 | 终点 | 说明 |
|------|------|------|------|
| MPLSIF | `refers_to` | INTERFACE | MPLS 接口引用物理/子接口（IFNAME 同名） |
| VPNINSTAF | `refers_to` | L3VPNINST | ★依赖：ADD VPNINSTAF 前 L3VPNINST 必须存在（手册原文） |
| BGPVRF | `refers_to` | L3VPNINST | ★依赖：BGP VPN 实例名须与 L3VPN 实例名一致（手册原文） |
| BGPPEER | `refers_to` | INTERFACE | BGP 对等体引用本地 Loopback 接口（LOCALIFADDR/LOCALIFNAME） |

### 3.3 depends_on 边（4条，配置顺序强约束）

| 起点 | 关系 | 终点 | 说明 |
|------|------|------|------|
| VPNINSTAF | `depends_on` | L3VPNINST | 地址族依赖 VPN 实例存在 |
| VPNTARGET | `depends_on` | VPNINSTAF（含 VRFRD） | ★RT 属性依赖地址族已配置 RD（手册原文：需确保 RD 已配置） |
| BGPVRF | `depends_on` | L3VPNINST + BGP | BGP VPN 实例依赖 L3VPN 实例+BGP 全局使能 |
| BGPPEERAF | `depends_on` | BGPPEER + BGPVRFAF | 对等体地址族依赖对等体+对应地址族存在（ADD BGPPEER 默认创建 ipv4uni 地址族） |

### 3.4 governs（控制平面归属，1条）

| 起点 | 关系 | 终点 | 说明 |
|------|------|------|------|
| BGPVRFAF（VRFNAME=_public_, AFTYPE=ipv4vpn/ipv6vpn） | `governs` | BGPPEERAF（同 _public_） | ★MP-EBGP 开关：BGPVPN 地址族是 MP-EBGP 控制平面的使能开关 |

## 8. 使用实例脚本（保留手册原文，UDG BGP/MPLS VPN 非SDN_手动 标准脚本）

### 8.1 UDG BGP/MPLS VPN 非SDN_手动 组网（来源：配置BGP_MPLS VPN（非SDN_手动）_70507877.md）

```
// 1. 全局激活BFD（MPLS VPN 与 BGP 配合，BFD 用于快速感知）。
SET BFD: BFDENABLE=TRUE;

// 2. 全局激活BGP。
SET BGP: BGPENABLE=TRUE, ASNUM="60001";

// 3. 配置MPLS全局配置（★LDPENABLE=DISABLE）。
SET MPLSSITE: MPLSLSRID="10.10.10.78", MPLSENABLE=ENABLE, LDPENABLE=DISABLE;

// 4. 配置外联口IP、BFD会话、BFD绑定静态路由（到BGP PEER IP的静态路由）。
//    N6外联口绑定VPN实例_public_。详见静态路由+BFD组网文档。

// 5. 使能接口的MPLS功能。
ADD MPLSIF:IFNAME="Ethernet66/0/4.100";
ADD MPLSIF:IFNAME="Ethernet66/0/5.200";

// 6. 配置VPN实例及VPN实例地址族。
ADD L3VPNINST: VRFNAME="VRF_PDN1";
ADD L3VPNINST: VRFNAME="VRF_PDN2";
// IPv4组网或IPv4v6组网：
ADD VPNINSTAF: VRFNAME="VRF_PDN1", AFTYPE=ipv4uni, VRFRD="100:100", VRFLABELMODE=perInstance;
ADD VPNINSTAF: VRFNAME="VRF_PDN2", AFTYPE=ipv4uni, VRFRD="200:200", VRFLABELMODE=perInstance;
// IPv6组网或IPv4v6组网（6VPE：用IPv4 eBGP传IPv6私网路由）：
ADD VPNINSTAF: VRFNAME="VRF_PDN1", AFTYPE=ipv6uni, VRFRD="100:100", VRFLABELMODE=perInstance;

// 7. 配置VPN Target（★export + import 成对）。
ADD VPNTARGET: VRFNAME="VRF_PDN1", AFTYPE=ipv4uni, VRFRTTYPE=export_extcommunity, VRFRTVALUE="1:1";
ADD VPNTARGET: VRFNAME="VRF_PDN1", AFTYPE=ipv4uni, VRFRTTYPE=import_extcommunity, VRFRTVALUE="1:1";

// 8. 配置Loopback接口（BGP本端IP，★不绑VPN，用公网BGP传私网路由）。
ADD INTERFACE: IFNAME="Loopback1";
ADD IFIPV4ADDRESS: IFNAME="Loopback1", IFIPADDR="10.10.10.78", SUBNETMASK="255.255.255.255";

// 9. 配置BGP。
//    a. 修改公网BGP VPN实例ROUTERID。
MOD BGPVRF: VRFNAME="_public_", ROUTERID="10.1.1.1";
//    b. 增加BGP VPN实例（私网）。
ADD BGPVRF: VRFNAME="VRF_PDN1", DEFAULTAFTYPE=noaf;
//    c. 增加BGP VPN实例地址族（★使能MP-EBGP，VRFNAME=_public_）。
ADD BGPVRFAF: VRFNAME="_public_", AFTYPE=ipv4vpn, APPLYLABELMODE=perNexthop;
ADD BGPVRFAF: VRFNAME="_public_", AFTYPE=ipv6vpn, APPLYLABELMODE=perNexthop;
//    d. 增加BGP VPN地址族（私网，负荷分担）。
ADD BGPVRFAF: VRFNAME="VRF_PDN1", AFTYPE=ipv4uni, MAXIMUMLOADBALANCE=2;
//    e. 增加BGP对等体（★VRFNAME=_public_，用IPv4 eBGP传IPv6路由）。
ADD BGPPEER: VRFNAME="_public_", ADDRESSTYPE=ipv4, PEERADDR="10.10.20.78", REMOTEAS="60002", LOCALIFADDR="10.10.10.78", LOCALIFNAME="Loopback1", EBGPMAXHOP=10;
//    f. 配置BGP对等体地址族。
ADD BGPPEERAF: VRFNAME="_public_", AFTYPE=ipv4vpn, REMOTEADDRESS="10.10.20.78";
ADD BGPPEERAF: VRFNAME="_public_", AFTYPE=ipv6vpn, REMOTEADDRESS="10.10.20.78";
//    g. 增加BGP引入wlr路由（★手机下行路由）。
ADD IMPORTROUTE: VRFNAME="VRF_PDN1", AFTYPE=ipv4uni, IMPORTPROTOCOL=wlr, MEDENABLE=FALSE;
ADD IMPORTROUTE: VRFNAME="VRF_PDN1", AFTYPE=ipv6uni, IMPORTPROTOCOL=wlr, MEDENABLE=FALSE;
```

### 8.2 UNC SDN MPLS VPN 关键差异（来源：配置VNF侧IP路由数据（SDN+IPv4）_44139027.md）

> **★UNC 侧关键差异**：UNC 没有独立的「配置BGP_MPLS VPN」初始配置文档（原 04 错误根源），MPLS VPN 命令零星出现在 SDN IPv4 路由配置文档中。UNC 侧特点：①Loopback 接口需 `ADD IPBINDVPN` 绑定 VPN（UDG 不绑）；②ROUTERID 在 `ADD BGPVRF` 直接配（UDG 用 MOD BGPVRF）；③SDN 场景大量用 `ADD AUTOSCALINGSERVICE/BFD/SRROUTE` 自动化模板。

```
// UNC SDN 场景 BGP/MPLS VPN 关键脚本片段：
ADD L3VPNINST: VRFNAME="VRF_Access";
ADD VPNINSTAF: VRFNAME="VRF_Access", AFTYPE=ipv4uni;
MOD VPNINSTAF: VRFNAME="VRF_Access", AFTYPE=ipv4uni, VRFRD="10.10.10.78:100";

SET BFD: BFDENABLE=TRUE;
SET AUTOCONFIG:SWITCHFLAG=FALSE;
// ... AUTOSCALINGSERVICE/BFD/SRROUTE 自动化模板 ...
SET AUTOCONFIG: SWITCHFLAG=TRUE;

SET BGP: BGPENABLE=TRUE, ASNUM="60001";

// UNC Loopback 接口绑定 VPN（★UDG 无此步）。
ADD INTERFACE: IFNAME="Loopback1";
ADD IPBINDVPN: IFNAME="Loopback1", VRFNAME="VRF_Access";
ADD IFIPV4ADDRESS: IFNAME="Loopback1", IFIPADDR="10.10.10.78", SUBNETMASK="255.255.255.255";

// UNC BGP（ROUTERID 在 ADD BGPVRF 直接配）。
ADD BGPVRF: VRFNAME="VRF_Access", DEFAULTAFTYPE=noaf, ROUTERID="10.10.10.78";
ADD BGPVRFAF: VRFNAME="VRF_Access", AFTYPE=ipv4uni, MAXIMUMLOADBALANCE=2;
ADD BGPPEER: VRFNAME="VRF_Access", ADDRESSTYPE=ipv4, PEERADDR="10.10.20.78", REMOTEAS="60002", LOCALIFADDR="10.10.10.78", LOCALIFNAME="Loopback1", EBGPMAXHOP=10;
ADD IMPORTROUTE: VRFNAME="VRF_Access", AFTYPE=ipv4uni, IMPORTPROTOCOL=wlr, MEDENABLE=FALSE;
```

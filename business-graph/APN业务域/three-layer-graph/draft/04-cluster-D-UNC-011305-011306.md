## 0. 命令清单总览（011305 + 011306 用到的全部命令）

### 0.1 按命令类型分布

| 类型 | 命令数 | 命令清单 |
|------|-------|---------|
| Radius 连接管理类（ADD/SET/MOD） | 10 | ADD RDSSVRGRP、ADD RDSSVR、MOD RDSSVR、MOD RDSSVRGRP、ADD APNRDSSVRGRP、ADD APNRDSCLIENTIP、ADD UPLIST4RDS、SET RDSRSPADDRCHK、LST RDSSVRGRP、LST APNAUTHATTR |
| Radius 计费/鉴权信元控制类（★原04丢失重灾区） | 3 | SET RDSACCTREQVSA（3GPP计费私有扩展）、SET RDSACCTREQATTR（基础信元可选属性）、SET RDSAUTHREQVSA（3GPP鉴权私有扩展） |
| APN 级 Radius 控制类 | 3 | SET APNAUTHATTR（★鉴权接入核心）、SET APNRDSACCTCTRL（APN计费控制）、SET APNRADIUSATTR（APN域名增/剥离） |
| Radius 故障放通/上报 APN 类 | 3 | SET FHBYPASS（故障旁路，高危）、ADD SPECIFICAPNVAL（指定上报 APN 映射）、TST AAA（AAA 连通性测试） |
| 4 组网场景差异命令（VPN/路由/隧道，非本特性拥有，仅登记引用） | 14 | ADD L3VPNINST、ADD VPNINSTAF、ADD VPNINST、ADD LOGICIP、ADD LOGICINF、ADD INTERFACE、ADD IPBINDVPN、ADD IFIPV4ADDRESS、ADD GRETUNNEL、ADD SRROUTE、ADD SRROUTE6、SET IFIPV6ENABLE、ADD IFIPV6ADDRESS、ADD ACLGROUP/ADDRULEADV4/IPSECPROPOSAL/IPSECINTFCFG（IPSec 场景，参考信息列出但 4 场景脚本未直接出现，归簇E） |

> **说明**：
> 1. SET APNAUTHATTR 跨 011305（鉴权接入核心）与 011306（DISCONNECT=ENABLE 开启 PoD）两个特性，是 011305 唯一的配置类命令，011306 步骤 6.h 引用其 DISCONNECT 参数。
> 2. SET RDSACCTREQVSA / SET RDSACCTREQATTR / SET RDSAUTHREQVSA 三条命令是 011306「3GPP扩展信元定制功能」的核心，原 04 完全丢失，本文件★补齐。
> 3. MOD RDSSVRGRP / MOD RDSSVR 原 04 仅登记 ADD，本文件★补齐 MOD（修改服务器组可选计费消息属性、修改服务器优先级）。
> 4. ADD SPECIFICAPNVAL（业务触发 APN 取值映射）原 04 丢失，本文件★补齐。
> 5. 4 组网场景（单平面+静态路由+BFD / 双平面+OSPF+BFD / 带内GRE VPN / 带外GRE VPN）共享 Radius 命令集，差异仅在 VPN/路由/隧道命令，在 §8 脚本维度区分。

## 1. UDG 侧 L2TP U 面 LAC 执行（10+ 参数）

### 1.1 SET APNL2TPATTR（CMD-UDG-065，★核心使能命令）

| 参数 | 取值 | 说明 |
|------|------|------|
| APN | 如 apn-l2tp | APN 名（需已 ADD APN） |
| L2TPSWITCH | ENABLE / DISABLE | APN 支持 L2TP 功能开关 |
| SUPPORTIPV6 | DISABLE（默认）/ ENABLE | L2TP 支持 IPv6（DISABLE 时仅 IPv4 用户接入） |
| **L2TPGROUPID** | 数字（如 1） | **方式A 专用**：本地 L2TP 组 ID |
| **RDSLNSMODE** | REDUNDANCY / TUNNEL_PREFER | **方式B 专用**：RADIUS 多 LNS 工作模式（REDUNDANCY 主备 / TUNNEL_PREFER Preference 区分） |
| ICRQ_CALLINGNO | MSISDN / IMSI / IMEI | ICRQ Calling-number AVP 用户标识 |
| ICCN_AUTH | ENABLE / DISABLE | ICCN 携带鉴权信元 |
| IPCP_NEGO | ENABLE / DISABLE | 主动发起 IPCP 协商 |
| ... | ... | ★共 10+ 参数（CR-APN-02：与 UNC APNL2TPCTRL 仅 2 参数不对称） |

### 1.2 ADD L2TPGROUP（CMD-UDG-067，方式A 本地配置核心）

| 参数 | 说明 |
|------|------|
| L2TPGROUPID | 组 ID |
| DOMAINNAME | 域名 |
| LOCALNAME | 本端名 |
| LOCALLNSMODE | 本地 LNS 模式 |
| FIRSTLNSIP / SECONDLNSIP | 主/备 LNS IP |
| HELLO 参数 | Hello 间隔/重试 |
| MAXSENDWINSIZE | 最大发送窗口 |

### 1.3 其他 U 面 L2TP 命令

| 命令 | command_id | 说明 |
|------|-----------|------|
| ADD L2TPLNSINFO | CMD-UDG-068 | L2TP LNS 信息（>6 LNS 或 IPv6 时） |
| ADD L2TPCLIENTIP | CMD-UDG-069 | 方式A：L2TP 组绑定源端 Giif |
| **ADD L2TPRDSCLIENT** | CMD-UDG-070 | **方式B**：AAA 下发 LNS（APN 绑定源端 Giif，互斥 L2TPGROUP，CR-APN-14） |
| SET GLOBALL2TP | CMD-UDG-071 | L2TP 缺省属性（LOCALNAME/HELLO/RETRY） |
| SET PPPCFG | CMD-UDG-072 | PPP 协商参数（MRU/HOSTNAME/TIMEOUT） |
| SET APNPPPACCESS | CMD-UDG-073 | APN PPP 鉴权 |
| **SET L2TPN4KEY** | CMD-UDG-074 | N4 接口 L2TP 加密密钥（★与 UNC L2TPKEY 须相同，CR-APN-03） |

## 3. 方式A 本地配置 vs 方式B AAA 下发（CR-APN-14，互斥）

| 方式 | LNS 参数来源 | U 面 L2TP 组 | SET APNL2TPATTR 关键参数 |
|------|------------|-------------|------------------------|
| **方式A 本地配置** | ADD L2TPGROUP 配置 | L2TPGROUP + L2TPCLIENTIP | L2TPGROUPID={组ID} |
| **方式B AAA 下发** | Radius Access-Accept | L2TPRDSCLIENT（互斥 L2TPGROUP） | RDSLNSMODE=REDUNDANCY/TUNNEL_PREFER |

> 两种方式互斥，不可同时配置。

## 4. 激活流程骨架（来源 GWFD-020412 §4.1）

```
步骤1：License（LKV3G5L2TP01，仅 UDG）+ ADD APN 前置
步骤2：（可选）ADD VPNINST: VPNINSTANCE="vpn_l2tp"
步骤3：SET GLOBALL2TP（缺省参数）
步骤4：SET APNL2TPATTR（★核心使能）
步骤5：SET SOFTPARAOFBIT（关闭快速流表 Byte671 bit7=1）
步骤6：ADD LOGICINF（Giif 接口）
=== 分支 ===
【方式A】步骤7a: ADD L2TPGROUP + 步骤8a: ADD L2TPCLIENTIP
【方式B】步骤7b: （无需 L2TPGROUP）+ 步骤8b: ADD L2TPRDSCLIENT
=== 共享可选 ===
步骤9: SET PPPCFG（可选）
步骤10: SET APNPPPACCESS（可选）
步骤11: SET L2TPN4KEY（可选，★U+C 密钥相同）
```

## 5. 配置实例（方式A 本地配置）

```mml
// UDG 侧
SET LICENSESWITCH:LICITEM="LKV3G5L2TP01",SWITCH=ENABLE;
ADD VPNINST:VPNINSTANCE="vpn_l2tp";
ADD LOGICINF:NAME="giif1/0/0",IPVERSION=ipv4,IPV4ADDRESS1="{giif_ip}",IPV4MASK1="{mask}",VPNINSTANCE="vpn_l2tp";
SET SOFTPARAOFBIT:DT2=BYTE,BYTENUM=671,BYTEPOSITION=7,BYTEVALUE=1;
ADD L2TPGROUP:GROUPID=1,DOMAINNAME="{domain}",LOCALNAME="{localname}",FIRSTLNSIP="{lns_ip}",...;
ADD L2TPCLIENTIP:L2TPGROUPID=1,INTERFACENAME="giif1/0/0";
SET APNL2TPATTR:APN="apn-l2tp",L2TPSWITCH=ENABLE,SUPPORTIPV6=DISABLE,L2TPGROUPID=1,...;
SET L2TPN4KEY:N4KEYVALUE="{key}",CFMN4KEYVALUE="{key}";   !-- 可选加密

// UNC 侧（C 面决策）
SET APNL2TPCTRL:APN="apn-l2tp",L2TPSWITCH=ENABLE;
SET L2TPKEY:KEYVALUE="{key}",CFMKEYVALUE="{key}";         !-- 与 UDG L2TPN4KEY 相同
```

## 6. 约束（BR/CR/TR）

- **License C-U 不对称**：仅 UDG 需 LKV3G5L2TP01，UNC WSFD-104410 无 License（BR-APN-L2TP-CU-ASYM / TR-APN-05）
- **APNL2TPATTR(U,10+) vs APNL2TPCTRL(C,2) 不对称**（CR-APN-02 / TR-APN-06）
- **L2TPN4KEY(U) 与 L2TPKEY(C) 密钥必须相同**（CR-APN-03）
- **L2TP 本地配置 vs AAA 下发互斥**（CR-APN-14）
- **L2TP × 地址自动检测互斥**（BR-APN-L2TP-ADDRAUTO-EXCL / CR-APN-13）
- **L2TP × 基于位置互斥**（BR-APN-LOC-L2TP-EXCL / CR-APN-13）
- **不支持 PPP 用户/DHCP 延迟分配/IPv6 PD/NAT**（CR-APN-14）
- **单栈限制**：SUPPORTIPV6=DISABLE 时仅 IPv4 用户接入，UDG 与 LNS 只进行 IPv4 单栈协商
- **企业网侧必须已部署 LNS 设备**（应用限制硬约束）

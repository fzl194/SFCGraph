# 04-地址分配规则与三级优先级

> 数据源：`04-command-graph.md` §1.4 + `feature/GWFD-010105-用户面地址分配.md` §3.3/4.3.4 + `feature/GWFD-020421-基于位置的地址分配.md`
> 作用：Phase 3 三级规则分析（必须用户确认）/ Phase 5 规则生成

---

## 1. 两级配置体系

UDG 地址分配规则分两级：

| 级别 | 命令 | 作用域 |
|------|------|--------|
| 全局 | `SET IPALLOCRULE`（CMD-UDG-019） | 所有 APN 默认规则 |
| APN 级 | `SET APNIPALLOCRULE`（CMD-UDG-020） | 指定 APN 覆盖全局 |

UNC 侧对应命令：`SET IPALLOCRULE`（CMD-UNC-016-1）/ `SET APNIPALLOCRULE`（CMD-UNC-016-2）/ `SET ADDRESSATTR`（CMD-UNC-016-3）/ `SET SOFTPARA`（CMD-UNC-016-4）。

---

## 2. 三级规则参数

| 参数 | 取值 | 说明 |
|------|------|------|
| FIRSTRULESW | ENABLE / DISABLE | 第一级开关（默认使能） |
| **FIRSTRULE** | 规则字符串 | 第一级规则 |
| SECONDRULESW | ENABLE / DISABLE | 第二级开关（默认不使能） |
| SECONDRULE | 规则字符串 | 第二级规则（多级回退场景） |
| THIRDRULESW | ENABLE / DISABLE | 第三级开关 |
| THIRDRULE | 规则字符串 | 第三级规则 |
| ALLOCATTR | LOCAL（仅 SET APNIPALLOCRULE） | 地址分配属性 |

---

## 3. ★规则字符串格式（严格三段式）

```
{APN-{0|1}}&{LOCATION-{0|1}}&{SMF-{0|1}}
```

- `APN-1`：基于 APN/DNN 匹配
- `LOCATION-1`：基于位置（LAC/TAC）匹配
- `SMF-1`：基于 SMF NodeID 匹配
- 三段通过 `&` 连接，1 表示启用该维度，0 表示不启用

### 3.1 单维度规则（DP-APN-ADDR-GRANULARITY 单选）

| 分配子方式 | FIRSTRULE | 全局开关 | 备注 |
|-----------|-----------|---------|------|
| 基于 APN/DNN | `APN-1&LOCATION-0&SMF-0` | 无 | 母特性 GWFD-010105 |
| 基于 SMF | `APN-0&LOCATION-0&SMF-1` | `SET IPALLOCBYSMFGLBSW:SWITCH=ENABLE` | 需 ADD CPNODEID |
| 基于位置 | `APN-0&LOCATION-1&SMF-0` | `SET IPALLOCBYLOCGLBSW:IPV4SWITCH=ENABLE` | 需 License LKV3G5LBAA01 |

### 3.2 双维度组合（DP-APN-ADDR-GRANULARITY 多选）

| 分配子方式 | FIRSTRULE | SECONDRULE（兜底） |
|-----------|-----------|-------------------|
| APN + SMF | `APN-1&LOCATION-0&SMF-1` | `APN-0&LOCATION-0&SMF-1`（回退到 SMF） |
| APN + LOCATION | `APN-1&LOCATION-1&SMF-0` | `APN-1&LOCATION-0&SMF-0`（回退到 APN） |
| SMF + LOCATION | `APN-0&LOCATION-1&SMF-1` | `APN-0&LOCATION-0&SMF-1`（回退到 SMF） |
| 三维度全选 | `APN-1&LOCATION-1&SMF-1` | `APN-1&LOCATION-1&SMF-0` / `APN-0&LOCATION-1&SMF-1` |

## 4. RADIUS 子方式特有（IGNOREV4/V6POOLID）

RADIUS 下发地址池名场景（DP-APN-ADDR-MODE=RADIUS下发），需配置：

```mml
SET APNADDRESSATTR:APN="{apn}",SUPPORTIPV4=ENABLE,IGNOREV4POOLID=DISABLE,IGNOREV6POOLID=DISABLE;
```

- `IGNOREV4POOLID=DISABLE`：支持 SMF 下发的地址池名（RADIUS 子方式必配）
- `IGNOREV4POOLID=ENABLE`：忽略 SMF 下发的地址池名（默认）

可选通配功能（`SET ADDRESSATTR`）：
- V4POOLWILDCARDSW / V6POOLWILDCARDSW：AAA 未下发通配符时是否开启通配

---

## 5. 三级规则匹配失败处理

**规则**：三级规则都匹配失败时，地址分配失败（GWFD-010105 §4.5）。

**兜底策略**：
- 启用 SECONDRULE/THIRDRULE 做维度回退（如 SMF+APN 主，SMF 兜底）
- 不同维度准备多个 POOLGRPMAP 映射

---

## 6. 配置实例（基于 SMF 单级）

```mml
// SMF NodeID
ADD CPNODEID:CPNAME="smfnode1",IPV4NODEID="10.0.0.1",IPV6NODEID="FC00:1111:...",FQDNNODEID="consumer.huawei.com";

// 池组映射（基于 SMF）
ADD POOLGRPMAP:MAPPINGNAME="mapping1",SMF="smfnode1",POOLGROUPNAME="poolgroup1";

// 三级规则（基于 SMF 单级）
SET IPALLOCRULE:FIRSTRULESW=ENABLE,FIRSTRULE=APN-0&LOCATION-0&SMF-1,SECONDRULESW=DISABLE,THIRDRULESW=DISABLE;

// 全局开关
SET IPALLOCBYSMFGLBSW:SWITCH=ENABLE;
```

> 来源：`feature/GWFD-010105-用户面地址分配.md` §5.2

## 7. 配置实例（基于 SMF+APN 两级回退）

```mml
// 地址池 2（SMF 单维度回退池）
ADD POOL:POOLNAME="testpool2",POOLTYPE=LOCAL,HASVPN=ENABLE,VPNINSTANCE="vpn1";
ADD SECTION:POOLNAME="testpool2",SECTIONNUM=1,IPVERSION=IPV4,V4STARTIP="10.10.1.21",V4ENDIP="10.10.1.30";

// 池组 2（带优先级算法）
ADD POOLGROUP:POOLGRPNAME="poolgroup2",IPV4ALLOCPRIALG=ENABLE;
ADD POOLBINDGROUP:POOLGROUPNAME="poolgroup2",POOLNAME="testpool2",PRIORITY=10;

// 两条映射规则
ADD POOLGRPMAP:MAPPINGNAME="mapping2",APN="apn-test1",SMF="smfnode1",POOLGROUPNAME="poolgroup1";
ADD POOLGRPMAP:MAPPINGNAME="mapping3",SMF="smfnode1",POOLGROUPNAME="poolgroup2";

// 三级规则（两级回退）
SET IPALLOCRULE:FIRSTRULESW=ENABLE,FIRSTRULE=APN-1&LOCATION-0&SMF-1,SECONDRULESW=ENABLE,SECONDRULE=APN-0&LOCATION-0&SMF-1,THIRDRULESW=DISABLE;
```

> 来源：`feature/GWFD-010105-用户面地址分配.md` §5.3

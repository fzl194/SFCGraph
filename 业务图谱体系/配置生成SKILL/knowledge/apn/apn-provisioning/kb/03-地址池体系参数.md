# 03-地址池体系参数

> 数据源：`04-command-graph.md` §1.3/1.4/1.17 + §5.1/5.2 + `feature/GWFD-010105-用户面地址分配.md` §4.3
> 作用：Phase 3 地址池参数收集 / Phase 5 地址池生成
> ★核心不对称：UDG POOL（POOLTYPE=LOCAL）vs UNC ADDRPOOL（POOLTYPE=UDM），CR-APN-01/04

## 1. UDG 侧地址池体系（POOL/SECTION/POOLGROUP/POOLBINDGROUP/POOLGRPMAP）

### 1.1 ADD POOL（CMD-UDG-014）

| 参数 | 类型 | 取值 | required_mode | 说明 |
|------|------|------|--------------|------|
| POOLNAME | string | — | required | 地址池名 |
| **POOLTYPE** | enum | **LOCAL / EXTERNAL** | required | ★UDG 用 LOCAL（本地池）；EXTERNAL 外部池；CR-APN-04 |
| IPVERSION | enum | IPV4 / IPV6 | required | 地址版本 |
| HASVPN | enum | ENABLE / DISABLE | optional | 是否绑定 VPN |
| VPNINSTANCE | string | — | conditional_required | VPN 名（CR-APN-07：与 APN VPN 一致；HASVPN=ENABLE 时必填） |
| REDUNDFUNC | enum | — | conditional_required | 冗余功能标识（静态路由冗余场景，与 REDUNDUSER 双使能） |

### 1.2 ADD SECTION（CMD-UDG-015）

| 参数 | 类型 | 取值 | required_mode | 说明 |
|------|------|------|--------------|------|
| POOLNAME | string | — | required | 地址池名 |
| SECTIONNUM | int | 1, 2, ... | required | 地址段号 |
| IPVERSION | enum | IPV4 / IPV6 | required | 地址版本（双栈需配两类池） |
| V4STARTIP | IPv4 | 如 10.10.1.1 | conditional_required | IPv4 起始（IPV4 时必填） |
| V4ENDIP | IPv4 | 如 10.10.1.10 | conditional_required | IPv4 终止 |
| V6PREFIXSTART | IPv6 | — | conditional_required | IPv6 起始 |
| V6PREFIXEND | IPv6 | — | optional | IPv6 终止 |
| **V6PREFIXLENGTH** | int | 64 / <64 | conditional_required | ★=64 普通 IPv6 单栈；<64 切换 PD 模式（CR-APN-05） |

### 1.3 ADD POOLGROUP（CMD-UDG-016）

| 参数 | 类型 | 取值 | required_mode | 说明 |
|------|------|------|--------------|------|
| POOLGRPNAME | string | — | required | 池组名 |
| IPV4ALLOCPRIALG | enum | ENABLE / DISABLE | optional | IPv4 优先级算法使能（双栈/多池必开） |
| IPV6ALLOCPRIALG | enum | ENABLE / DISABLE | optional | IPv6 优先级算法使能 |

### 1.4 ADD POOLBINDGROUP（CMD-UDG-017，★UDG 命名 GROUP）

| 参数 | 类型 | 取值 | required_mode | 说明 |
|------|------|------|--------------|------|
| POOLGROUPNAME | string | — | required | 池组名 |
| POOLNAME | string | — | required | 地址池名 |
| PRIORITY | int | 如 10 | optional | 优先级（同类池间比较；仅 IPv4/IPv4 或 IPv6/IPv6，跨类型不比） |

### 1.5 ADD POOLGRPMAP（CMD-UDG-018）

| 参数 | 类型 | 取值 | required_mode | 说明 |
|------|------|------|--------------|------|
| MAPPINGNAME | string | — | required | 映射名 |
| APN | string | — | optional（按粒度组合） | APN 名（基于 APN 时填） |
| SMF | string | — | optional | SMF NodeID（基于 SMF 时填） |
| LOCATION | string | — | optional | 位置区组名（基于位置时填） |
| POOLGROUPNAME | string | — | required | 池组名 |

> APN/SMF/LOCATION 三者按 DP-APN-ADDR-GRANULARITY 组合填，详见 §04 三级规则。

## 2. UNC 侧地址池体系（ADDRPOOL/SECTION/ADDRPOOLGRP/POOLBINDGRP/POOLBINDAPN/POOLGRPMAP）

> ★UNC 仅在 UDM 静态签约场景（CS-APN-01/04）配置；动态分配场景 UNC 侧不建池。

### 2.1 ADD ADDRPOOL（CMD-UNC-002）

| 参数 | 类型 | 取值 | required_mode | 说明 |
|------|------|------|--------------|------|
| ADDRPOOLNAME | string | — | required | ★UNC 地址池名（前缀 ADDR，与 UDG POOL 分离） |
| **POOLTYPE** | enum | **UDM** | required | ★UNC 仅 UDM（静态签约用），无 LOCAL；CR-APN-04 |
| IPVERSION | enum | IPV4 / IPV6 | required | 地址版本 |

### 2.2 ADD SECTION（CMD-UNC-006，与 UDG 同命令）

参数同 UDG ADD SECTION。

### 2.3 ADD ADDRPOOLGRP（CMD-UNC-005-1）+ POOLBINDGRP（CMD-UNC-007，★UNC 命名 GRP）

| 命令 | 关键参数 |
|------|---------|
| ADD ADDRPOOLGRP | ADDRPOOLGRPNAME |
| ADD POOLBINDGRP | ADDRPOOLGRPNAME, ADDRPOOLNAME（★UNC 命名 GRP 非 GROUP） |
| ADD POOLBINDAPN（UNC 独有） | APN, ADDRPOOLNAME |
| ADD POOLGRPMAP | MAPPINGNAME, APN, ADDRPOOLGRPNAME |

## 3. UDG 配置实例（基于 APN/DNN 单级，来源 GWFD-010105 §5.1）

```mml
// VPN 实例
ADD L3VPNINST:VRFNAME="vpn1";
ADD VPNINSTAF:VRFNAME="vpn1",AFTYPE=ipv4uni,VRFRD=100:1;
ADD VPNINST:VPNINSTANCE="vpn1";

// APN 使能地址分配
ADD APN:APN="apn-test1",HASVPN=ENABLE,VPNINSTANCE="vpn1";
SET APNADDRESSATTR:APN="apn-test1",SUPPORTIPV4=ENABLE,SUPPORTIPV6=ENABLE;

// 地址池
ADD POOL:POOLNAME="testpool1",POOLTYPE=LOCAL,HASVPN=ENABLE,VPNINSTANCE="vpn1";
ADD SECTION:POOLNAME="testpool1",SECTIONNUM=1,IPVERSION=IPV4,V4STARTIP="10.10.1.1",V4ENDIP="10.10.1.10";

// 池组绑定
ADD POOLGROUP:POOLGRPNAME="poolgroup1";
ADD POOLBINDGROUP:POOLGROUPNAME="poolgroup1",POOLNAME="testpool1";
ADD CONFLICTIP:POOLNAME="testpool1",IPADDRESS="10.1.1.1";

// 池组映射
ADD POOLGRPMAP:MAPPINGNAME="mapping1",APN="apn-test1",POOLGROUPNAME="poolgroup1";

// 地址分配规则
SET IPALLOCRULE:FIRSTRULESW=ENABLE,FIRSTRULE=APN-1&LOCATION-0&SMF-0,SECONDRULESW=DISABLE,THIRDRULESW=DISABLE;

// 下行路由
ADD OSPF:PROCID=100,VRFNAME="vpn1",SCHEMAROUID="10.10.10.1";
ADD OSPFAREA:PROCID=100,AREAID="0.0.0.0";
ADD OSPFNETWORK:PROCID=100,AREAID="0.0.0.0",IPADDRESS="10.13.21.0",WILDCARDMASK="0.0.0.255";
ADD OSPFIMPORTROUTE:PROCID=100,TOPOID=0,PROTOCOL=wlr;
```

## 4. SMF NodeID 与全局开关（基于 SMF 子方式）

### 4.1 ADD CPNODEID（CMD-UDG-021）

| 参数 | 取值 | 说明 |
|------|------|------|
| CPNAME | 如 smfnode1 | SMF 名称 |
| IPV4NODEID | IPv4（如 10.0.0.1） | IPv4 NodeID，需与对端 SMF 一致 |
| IPV6NODEID | IPv6 | IPv6 NodeID |
| FQDNNODEID | FQDN（如 consumer.huawei.com） | FQDN NodeID |

### 4.2 全局开关（两级控制）

```mml
// 全局开关
SET IPALLOCBYSMFGLBSW:SWITCH=ENABLE;
// 指定 SMF 开关（覆盖全局）
SET IPALLOCBYSMFSW:SMFID="{smf_id}",SWITCH=ENABLE;
```

### 4.3 基于位置全局开关

```mml
SET IPALLOCBYLOCGLBSW:IPV4SWITCH=ENABLE,IPV6SWITCH=DISABLE;
```

## 5. 约束（CR-APN-01/04/05/07，GWFD-010105 §4.5）

- **POOLTYPE 跨侧差异**：UDG=LOCAL，UNC=UDM（CR-APN-04），两侧同名参数取值集不同
- **V6PREFIXLENGTH<64 切换 PD**（CR-APN-05），需 License LKV3G5P6PD01
- **APN VPN 与 POOL VPN 必须一致**（CR-APN-07），双栈对应双 VPN 实例
- **VPN 三者一致**：POOL VPN = APN VPN = Gi/SGi/N6 外联口 VPN
- **地址池绑定数量**：每 APN/DNN 最多 16 个地址池
- **地址段绑定数量**：每本地池最多 64 个地址段
- **主机路由规格**：每 UDG 40 万
- **优先级仅同类池比较**：IPv4 vs IPv4，IPv6 vs IPv6，跨类型不比
- **冲突 IP 生效时机**：已分配地址配置冲突地址在下次分配时生效，不影响在线用户
- **强制回收风险**：必须先锁定地址池/段，否则强制下线在线用户

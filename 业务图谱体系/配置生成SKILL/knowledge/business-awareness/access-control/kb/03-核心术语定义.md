# 03 · 核心术语定义

> **定位**：访问限制场景术语速查。理解用户需求关键词与配置对象的映射。
> **数据来源**：`访问限制场景/three-layer-graph/01-business-graph.md`（SemanticObject SO-AC-01~07）、各特性 §1 概述。
> **关联文件**：`01-访问限制动作体系与双轨道.md`。

---

## 1. 动作类型术语（SO-AC-01）

| 术语 | 英文 | 语义 | 实现轨道 |
|------|------|------|---------|
| 阻塞 | DISCARD | 丢弃用户报文，禁止访问 | 轨道 A 隐式（规则不匹配丢弃）/ 轨道 B 显式 BLOCK |
| 头增强 | HEADEN | 在 HTTP/HTTPS/RTSP 报文头插入用户信息字段（MSISDN/IMSI/IMEI 等） | 轨道 A · HEADEN 子轨 |
| 重定向 | REDIRECT | 将用户业务流引导到指定服务器/页面 | 轨道 A · SMARTREDIRECT/WEBPROXY/PCC 子轨 + 轨道 B |
| 放行 | PERMIT | 显式允许访问（白名单机制） | **仅轨道 B**（URL 过滤，BR-AC-10） |

---

## 2. 动作轨道术语（SO-AC-07）

| 术语 | 定义 |
|------|------|
| 轨道 A（PCC 体系） | RULE.POLICYTYPE 隐式驱动，内含五子轨（PCC/ADC/HEADEN/SMARTREDIRECT/WEBPROXY） |
| 轨道 B（URL 过滤体系） | CFTEMPLATE/CONTCATEGBIND.ACTION 显式驱动，独立于 RULE 体系，唯一支持 PERMIT |
| 五子轨 | 轨道 A 内的 5 个 POLICYTYPE 分支：ADC / PCC / HEADEN / SMARTREDIRECT / WEBPROXY |

---

## 3. 过滤条件术语（SO-AC-02）

| 术语 | 配置对象 | 匹配维度 | 适用场景 |
|------|---------|---------|---------|
| 三四层过滤 | FILTER | IP / 端口 / 协议（五元组） | 所有轨道 A 特性共用 |
| 七层过滤 | L7FILTER | URL / Method / SNI | ADC / 头增强族 / Portal / 防欺诈 / URL 过滤 |
| 流过滤器 | FLOWFILTER | 组合过滤条件容器 | 所有特性匹配入口 |
| 扩展过滤器 | EXTENDEDFILTER | URL / UserAgent / ContentType / URLPostfix 多维 | 重定向族独有（SMARTREDIRECT 子轨） |
| 错误码范围 | ERRORCODE | HTTP 错误码（GT 400）/ DNS 错误码（EQUAL 3 NXDOMAIN） | SMARTREDIRECT 子轨触发条件 |
| URL 分类 | CONTCATEGBIND + CONTCATEGROUP | URL 分类 ID（CATEGORYID，来自 ICAP Server） | 轨道 B（URL 过滤）独有 |

## 4. 重定向目标术语（SO-AC-03）

| 术语 | 配置对象 | 协议层 | 适用方案 |
|------|---------|--------|---------|
| HTTP 智能重定向 | SMARTHTTPREDIR | L7 HTTP 响应改写（301/302/303） | CS-AC-03 |
| DNS 重写 | DNSOVERWRITING | DNS 解析层重写（构造新 DNS 响应） | CS-AC-04 |
| Portal 服务器集群 | IPFARM（Portal Server） | L7 HTTP 响应改写 + captive | CS-AC-05 Portal |
| WebProxy 服务器集群 | IPFARM（Proxy Server） | L3 IP NAT（透明代理） | CS-AC-05 WebProxy |
| RedirectInformation | (PCF 下发 URL，非 UDG 对象) | PCC 体系 | CS-AC-08 / CS-AC-09 |

## 5. 头增强字段术语（SO-AC-04）

| 术语 | DATATYPE 枚举 | 说明 |
|------|--------------|------|
| 用户标识 | IMSI1 / MSISDN1 / IMEI1 | 最常用，插入用户唯一标识 |
| 接入信息 | APN / MSIP / SGSNIP / GGSNIP / ULI / RAT | 接入网元与位置信息 |
| 业务信息 | CHGCHAR / Roaming / USERDEF / User-Profile-alias / SESSIONID / TIMESTAMP | 业务上下文 |
| 地址信息 | UPIPV4 / UPIPV6 | 用户 IP |
| 拼接字段 | MULTIPARA | 最多 3 个拼接字段 × 每字段 4 参数（HTTP 头增强 v02，20.10.0+） |
| 随机数 | RANDNUM | 防重放 |

**字段相关属性**：
- PREFIXNAME：字段前缀名（建议非 HTTP 标准头，如 X-msisdn / X-imsi，避免防欺诈失败）
- ENCRYALGORI：加密算法（NONE / MD5 / RC4 / AES128 / AES256 / RSA1024 / RSA2048 / SHA256）
- 编码方式：base64 / ASCII / INHERIT（HTTPS 额外支持十六进制）

## 6. 位置条件术语（SO-AC-05）

| 术语 | LOCATIONTYPE | 网络制式 | 标识组成 |
|------|-------------|---------|---------|
| CGI | CGI | 2G/3G | MCC + MNC + LAC + CI |
| ECGI | ECGI | 4G | MCC + MNC + ECI |
| NCGI | NCGI | 5G | MCC + MNC + NCI |
| PRA | — | 任意 | Presence Reporting Area（区域引导触发） |
| PLMN | — | 任意 | 漫游区域（PLMN_CH 触发） |
| TAI | — | 任意 | Tracking Area Identity（SAR 服务区域限制） |

**UNC 侧接入控制机制**：
- SAR（Service Area Restriction）：服务区域限制，TAI 允许/禁止列表
- 区域漫游限制：RAT / 核心网 / 区域
- ODB（Operator Determined Barring）：所有权闭锁，欠费禁用

## 7. 规则语义术语（SO-AC-06）

| 术语 | 语义 |
|------|------|
| RULE.POLICYTYPE | 决定动作类型（ADC/PCC/HEADEN/SMARTREDIRECT/WEBPROXY） |
| 规则类型 | 动态（PCF 实时下发）/ 预定义（UPF 本地，支持 L7）/ 本地（SMF 兜底） |
| 规则与策略关系 | 1:1（BR-AC-03），策略与动作 1:1 |
| 类型独立原则 | PCC/BWM/HEADEN 匹配独立进行，每种至多一条生效，不同类型可叠加（BR-AC-02） |

## 8. PCC 与计费属性术语（共享自计费场景）

| 术语 | 配置对象 | 访问限制场景用途 |
|------|---------|----------------|
| URR | OBJ-URR | ADC/Portal/WebProxy/URL 过滤的计费属性绑定（非 FUP 降速主用途） |
| URRGROUP | OBJ-URRGROUP | URR 组（UPURRNAME1/2/3） |
| PCCPOLICYGRP | OBJ-PCCPOLICYGRP | PCC 策略组，绑定 URRGROUP；ADC/Portal/WebProxy/URL 过滤共用（URL 过滤仅作匹配触发，动作不走 CFTEMPLATE） |

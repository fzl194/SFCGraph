## 访问限制场景核心机制

三种策略动作（均承载于PCC体系，这是本次修订补入PCC策略体系的根本原因）：

1. **PCC 阻塞**（DISCARD）→ PCCPOLICYGRP 包装 → 依赖 PCC基本功能(GWFD-020351/WSFD-109101)
2. **头增强**（HEADEN 插入 MSISDN+防欺诈）→ 可直接被 RULE 引用 → 依赖头增强族(110261/262/263)
3. **URL 重定向**（REDIRECT 指定目标 URL）→ PCCACTIONPROP 引用 REDIRECT → PCCPOLICYGRP 包装 → 依赖重定向族(110284/110283)

> 核心对象：PCCPOLICYGRP / RULE / FLOWFILTER / USERPROFILE / **PCCACTIONPROP(访问限制新增)**。前4个三场景共享，PCCACTIONPROP 为本场景独有。

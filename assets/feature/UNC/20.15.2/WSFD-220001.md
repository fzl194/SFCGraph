# NAS IMR信令上报

`WSFD-220001` · enhanced · required

## 所属目录

- [WSFD-220000](feature/UNC/20.15.2/WSFD-220000.md)

## 适用性

- 适用NF：AMF
- 代际支持矩阵：`SGSN(2.5&3G)=-; GGSN-C(2.5&3G)=-; MME(4G)=-; S/PGW-C(4G)=-; NB-IoT(4G)=-; 5G MME(5G NSA)=-; 5G S/PGW-C(5G NSA)=-; AMF(5G SA)=E; SMF(5G SA)=-; NRF(5G SA)=-`

## 适用NF（原文）

AMF

## 定义

NAS IMR信令上报是指AMF支持将NAS接口解密后的原始信令使用IMR（Initial Message Record）机制发送给单据服务器（如CloudUDN），单据服务器将原始信令通过xDR流北向接口发送给服务探针/SEQ系统/第三方服务器等。

## 客户价值

| 受益方 | 受益描述 |
| --- | --- |
| 运营商 | - 提升外置探针采集数据的完整性和准确性。<br>- 通过软探针，解决5G外置探针部署的问题。<br>- 降低网络运维复杂度。 |
| 用户 | 用户不感知该特性。 |

## 应用场景

为了应对Nas加密的场景下，外置探针对消息解密的准确性有影响的问题，开通本特性支持内置软探针功能，实现NAS信令通过IMR上报。

## 可获得性

**涉及网元**

| 涉及网元 | 支持版本 | 功能说明 |
| --- | --- | --- |
| AMF | UNC 20.6.2及后续版本 | - 支持数据采集，按照配置上报NAS信令，生成IMR单据，并向单据服务器上报。<br>- 将ASN编码的后的内容填入软采IMR的NGAPContainer，通过IMR接口上报出去。 |
| 单据服务器 | 无特殊要求 | - 支持采集AMF上报的原始信令，并进行海量存储。<br>- 预处理原始信令，输出北向数据至第三方服务器。 |

## 原理概述

**组网**

NAS IMR信令上报功能主要包括三层架构体系，统一拉通数据采集和数据分析平台，组网架构如 [图1](#ZH-CN_CONCEPT_0000001098759802__fig677091211430) 所示：

**图1** NAS IMR信令上报架构图

<br>

![](特性概述_98759802.assets/zh-cn_image_0000001145950905_2.png)

- 采集层：
  NAS信令指只有UE和核心网需要处理的信令流程，无线接入网络的基站无需处理的信令消息。
  UNC基于内置探针，实现NAS接口解密后的原始信令的采集。节省部署新探针所需要的硬件投资。
  软采通过IMR接口上报给单据服务器。
- 数据层：一般由全新云化架构的CloudUDN网元提供统一数据采集、汇聚、建模、存储、定位等基础功能，提供业务扩展能力，支持上层应用系统通过插件形式定制数据处理业务。
- 应用层：对数据层开放的数据进行分析和可视化呈现。
    - SEQ Analyst
    - Netprobe
    - 第三方IT系统

**系统实现**

当特性License加载，NAS消息上报开关打开时，AMF将接收/发送的NAS消息通过IMR事件上报接口发送给单据服务器（比如CloudUDN）。NAS IMR信令上报流程图如 [图2 NAS IMR信令上报流程图](#ZH-CN_CONCEPT_0000001098759802__fig1940601671915) 。

**图2** NAS IMR信令上报流程图

<br>

![](特性概述_98759802.assets/zh-cn_image_0000001099420494_2.png)

流程描述如下：

1. 用户通过U2020/OMPortal下发命令配置采集范围、IMR采集开关。
2. UNC判断NAS IMR信令上报功能开启，且UE上报的信令在采集范围内，启动信令上报功能。
3. UE和UNC间NAS信令交互。
4. UNC的AM微服务将收到/发送的NAS消息发送给vProbe微服务。
  UNC的内置软探针可使用TCP或SFTP的封装方式。
  > **说明**
  > - 配置变更后，新的信令采集配置参数在用户的下一个流程开始生效。
  > - UNC 20.10.0及后续版本UCF将不再转发IMR数据，vProbe服务用于承接IMR数据转发功能。
5. vProbe微服务通过IMR接口，把解密后的NAS信令封装到如下消息中，上报给单据服务器。
  AMF封装了NAS信令的消息包括：
    - PDU SESSION RESOURCE SETUP REQUEST
    - PDU SESSION RESOURCE RELEASE COMMAND
    - PDU SESSION RESOURCE MODIFY REQUEST
    - INITIAL CONTEXT SETUP REQUEST
    - HANDOVER REQUEST
    - INITIAL UE MESSAGE
    - DOWNLINK NAS TRANSPORT
    - UPLINK NAS TRANSPORT
    - NAS NON DELIVERY INDICATION
    - PDU SESSION RESOURCE SETUP RESPONSE
    - PDU SESSION RESOURCE MODIFY RESPONSE
    - INITIAL CONTEXT SETUP RESPONSE
    - HANDOVER REQUEST ACKNOWLEDGE
    - UE CONTEXT RELEASE REQUEST
    - UE CONTEXT RELEASE COMMAND
    - UE CONTEXT RELEASE COMPLETE
  单据服务器预处理，通过xDR接口输出北向数据至第三方服务器等。

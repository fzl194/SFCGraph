# 增加ICAP服务器（ADD ICAPSERVER）

> 命令静态知识见 [ADD ICAPSERVER](command/UDG/20.15.2/ADD-ICAPSERVER.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

本任务创建一条 ICAP Server 实体——UDG 与外部 ICAP Server（URL 分类查询服务）的互通参数。`ICAPSERVERNAME` 供后续 [ADD ICAPSVRBINDISG](task/UDG/20.15.2/0-00261.md) 引用绑定到 ICAP 服务器组。适用 NF：PGW-U、UPF。

### 配置维度 1：ICAP 服务器类型（参数 ICAPSERVERTYPE，必选）
| 取值 | 作用 | 配套必选/可选 |
|---|---|---|
| URL_FILTERING | URL 内容过滤（主流） | TCPRETRYTIME 可条件可选 |
| CONTENT_FILTERING | 内容过滤 | TCPRETRYTIME 一般不配 |

### 配置维度 2：IP 地址族（参数 ICAPSVRIPTYPE，必选）
| 取值 | 作用 | 配套条件必选 |
|---|---|---|
| IPV4 | IPv4 路径 | ICAPSERVERIPV4 |
| IPV6 | IPv6 路径 | ICAPSERVERIPV6 |

### 配置维度 3：ICAP 服务器地址（按 ICAPSVRIPTYPE 条件必选）
| 参数 | 含义 | 取值约束 |
|---|---|---|
| ICAPSERVERIPV4 | IPv4 地址 | 点分十进制；**不支持 0.0.0.0 与 255.255.255.255**（命令 notes 强约束） |
| ICAPSERVERIPV6 | IPv6 地址 | IPv6 字符串；**不支持全 F（ffff:ffff:...）**（命令 notes 强约束） |

### 配置维度 4：通信端口与 VPN（可选）
| 参数 | 默认值 | 含义 |
|---|---|---|
| ICAPSERVERPORT | 1344 | ICAP Server TCP 端口号，整数 1~65535 |
| VPNINSTANCE | absent | VPN 名（与 ADD LOGICINF 的 VPNINSTANCE 保持一致；ICAP Server 走专用 VPN） |

### 配置维度 5：TCP/OPTIONS 调优（可选）
| 参数 | 默认值 | 含义 | 配套 |
|---|---|---|---|
| TCPRETRYTIME | 5 秒 | TCP 尝试建链间隔，整数 1~300 秒 | ICAPSERVERTYPE=URL_FILTERING 时可选 |
| OPTIONINTERVAL | 10 秒 | OPTIONS 消息重发间隔，整数 1~600 秒 | 通用可选 |

### 配置维度 6：配置域（参数 CFGDOMAINNAME，可选）
| 取值 | 作用 |
|---|---|
| 字符串 1~31 字符 | 命令所属公共配置域 |

**主特性脚本样例**（GWFD-110471，IPv4 路径）

```
ADD ICAPSERVER: ICAPSERVERNAME="icapserver1", ICAPSERVERTYPE=URL_FILTERING, ICAPSVRIPTYPE=IPV4, ICAPSERVERIPV4="172.16.39.136", VPNINSTANCE="vpn-gcfif";
```

**步骤位置**：URL 过滤 / ICAP 互通数据准备阶段（本 atom 仅 1 特性命中 GWFD-110471，跨特性无分叉）。

## 决策点：IP 地址族选择（DP 0-00259）

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| IPv4（特性样例） | ICAPSVRIPTYPE=IPV4 + ICAPSERVERIPV4 | 当前主流：IPv4 ICAP Server 接入 |
| IPv6 | ICAPSVRIPTYPE=IPV6 + ICAPSERVERIPV6 | IPv6 单栈或双栈 ICAP Server 场景 |

## 约束

- **rule-0-00447**（critical）：ICAPSERVERIPV4 不支持 0.0.0.0 与 255.255.255.255；ICAPSERVERIPV6 不支持全 F — 被拒，命令失败
- **rule-0-00448**（critical）：命令最大记录数 **30**；每种类型的 ICAP 服务器最多 30 个；超限 ADD 失败
- **rule-0-00449**（warning）：VPNINSTANCE 需与 [ADD LOGICINF](task/UDG/20.15.2/0-00083.md) 的 VPNINSTANCE 保持一致 — 跨 atom 一致性编排，否则互通不通
- **rule-0-00450**（warning）：ICAPSRVGMNAME（外层上游，本属性无此参数）引用 ADD ICAPSVRGRP（0-00260）—— 本任务前置：VPN 实例先建（[ADD VPNINST](task/UDG/20.15.2/0-00041.md)）
- **rule-0-00451**（info）：ICAPSERVERPORT/TCPRETRYTIME/OPTIONINTERVAL 缺省值——主特性样例采用默认值即可
- **rule-0-00452**（info）：命令执行后立即生效

# 设置用户面接口模式(入不转板总开关)（SET DATAPLANEINFMODE）

> 命令静态知识见 [SET DATAPLANEINFMODE](command/UDG/20.15.2/SET-DATAPLANEINFMODE.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

本任务是 GWFD-020482（激活入不转板功能）的**全局总开关**第 2 步——设置数据面接口模式为 Instance，开启入不转板（上下行均不转板）。ROUTEENHANCEDSW 控制路由增强功能，通过 VNF 与 DC-GW 配合减少 DC-GW 对承载网发布的用户下行路由条数。适用 NF：SGW-U、PGW-U、UPF。

**参数取值规划（数据规划表样例）**

| 参数 | 含义 | 取值样例 | 获取方法 / 取值规律 |
|---|---|---|---|
| MODE | 模式 | `Instance` | **固定取值**（开启入不转板必须用样例值 Instance，默认 Group 为不入不转板） |
| ROUTEENHANCEDSW | 路由增强功能开关 | `ENABLE` | 本端规划，枚举 ENABLE/DISABLE（本特性必填） |

**脚本样例**

```
SET DATAPLANEINFMODE:MODE=Instance,ROUTEENHANCEDSW=ENABLE;
```

**路由增强生效条件**：ROUTEENHANCEDSW=ENABLE **且** ADD AUTOSCALINGBGPLF（★R1.5 atom 待补） 命令中的"发布的团体属性"（UDADVCOM）不为 0，二者同时满足才生效。UDADVCOM 属 BGP 自动化配置，不在本特性命令域。

**防误用（关键）**：开启入不转板必须用 **SET DATAPLANEINFMODE**（总开关，上下行均不转板）；不可误用 **SET DATAPLANEGIINFMODE（★R1.5 atom 待补）**（仅下行业务入不转板，上行仍入转板）。

**操作步骤定位**：本命令为入不转板流程全局开关，之后执行配置逻辑接口（ADD LOGICINF）等后续步骤，并通过 DSP IPSUIT（★R1.5 atom 待补） 查询每个 IP Suit 的逻辑接口绑定情况。

## 决策点：MODE 二路分叉 Instance / Group（DP 0-00124）

问：SET DATAPLANEINFMODE 的 MODE 取哪个分支？二路互斥（命令真相枚举），决定是否入不转板。

| 选项 | MODE | 含义 | 适用场景 |
|---|---|---|---|
| 入不转板（特性必选） | `Instance` | 上下行均不转板（总开关开启） | GWFD-020482 激活入不转板功能 |
| 入转板（默认） | `Group` | 走传统入转板路径 | 非入不转板场景 |

## 决策点：ROUTEENHANCEDSW 双路 ENABLE / DISABLE（DP 0-00125）

问：是否同时开启路由增强功能？决定 VNF+DC-GW 配合减少下行路由发布条数。

| 选项 | ROUTEENHANCEDSW | 联动 | 适用场景 |
|---|---|---|---|
| 开启增强（特性推荐） | `ENABLE` | 需 ADD AUTOSCALINGBGPLF（★R1.5 atom 待补） 的 UDADVCOM ≠ 0 | 大规模部署减少路由条目 |
| 关闭增强 | `DISABLE` | 无 | 默认场景 |

## 决策点：本命令与 SET DATAPLANEGIINFMODE（★R1.5 atom 待补） 的互斥（DP 0-00126）

问：开启入不转板应选 SET DATAPLANEINFMODE 还是 SET DATAPLANEGIINFMODE（★R1.5 atom 待补）？二命令功能相似但作用域不同（关键防误用）。

| 选项 | 命令 | 作用范围 | 误用后果 |
|---|---|---|---|
| 总开关（特性必选） | SET DATAPLANEINFMODE | 上下行均不转板 | —— |
| 仅下行（勿误用） | SET DATAPLANEGIINFMODE（★R1.5 atom 待补） | 仅下行不转板，上行仍入转板 | 上行业务入转板可能造成转发异常 |

## 约束

- 本命令属于**高危命令**，操作不当会导致配置失效。
- 修改本命令前需要**删除所有用户**（前置约束）。
- 配置前需检查当前 **POD Type 是否为 ISU/APU POD**，否则不能开启入不转板（前置检查 1）。
- 配置前需检查 SET CPTEIDUALLOC 命令的 SWITCH 是否为 DISABLE，否则不能开启（前置检查 2）。[SET CPTEIDUALLOC](command/UDG/20.15.2/SET-CPTEIDUALLOC.md)
- 当有**逻辑接口绑定切片**时，不能开启入不转板功能（切片约束）。
- ROUTEENHANCEDSW 变更（DISABLE↔ENABLE）前需将用户下线。
- 命令最大记录数 **1**；命令执行后**立即生效**。
- MODE 固定 Instance（特性 md 数据规划表强约束）。

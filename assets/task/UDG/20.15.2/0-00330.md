# 查询APN配置（LST APN）

> 命令静态知识见 [LST APN](command/UDG/20.15.2/LST-APN.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法（配置方法字典）

LST APN 是 APN 族**配置查询**命令，输出 APN / HASVPN / VPNINSTANCE 等配置字段。GWFD-010105 场景下重点核查 SUPPORTIPV4/SUPPORTIPV6/IGNOREV4POOLID/IGNOREV6POOLID 等地址分配属性。命令核心配置空间有 2 个维度：**查询范围**（APN 可选：指定单个/查全部）、**详细度**（VERBOSE 可选）。

### 配置维度 1：查询范围（参数 APN，可选）

| 取值 | 行为 |
|---|---|
| 指定 APN 名 | 仅查该 APN 配置 |
| 不配 | 查所有 APN 配置 |

### 配置维度 2：详细度（参数 VERBOSE，可选）

| 取值 | 行为 |
|---|---|
| ENABLE | 输出完整字段（含 IGNORE*POOLID 等地址分配属性） |
| 不配 | 输出关键字段 |

**典型脚本（GWFD-010105 调测：查所有 APN 配置）**：
```
LST APN:;
```

**典型脚本（指定 APN 名查询完整字段）**：
```
LST APN:APN="apn-test1",VERBOSE=ENABLE;
```

> 命令 notes：适用 NF：PGW-U、UPF；用于核查 APN 地址分配属性。

**步骤位置**：在所属 feature 编排中通常位于"调测/验证阶段"——前置条件：APN 已建；用于 APN 地址分配属性核查。

# 设置带宽管理生效范围（SET BANDWIDTHMNG）

> 命令静态知识见 [SET BANDWIDTHMNG](command/UDG/20.15.2/SET-BANDWIDTHMNG.md)

## 配置方法

全局 SET 类命令（整机一份），定位用户组级业务带宽/连接数配置的**生效汇总范围**。RANGE 是核心分叉：CPU（业务处理单元级，已有用户组级带宽配置时建议）vs SYSTEM（整机级汇总，新建局点建议）。整机生效需周期性统计汇总各业务处理单元状态，PERIOD 控制该周期；CIRMINTHRES/PIRMINTHRES 给出承诺/峰值带宽最低保护值。

**两特性范式对比**（GWFD-110311 业务感知带宽 / GWFD-110301 终端系统码率差异）：

| 范式 | 取值样例 | 脚本 | 适用 |
|---|---|---|---|
| 整机生效（基础） | RANGE=SYSTEM | `SET BANDWIDTHMNG:RANGE=SYSTEM;` | 110311 APN/切片/用户组/TOS/分级场景、110301 终端差异化 |
| 整机生效+保护值 | RANGE=SYSTEM, CIRMINTHRES=30000, PIRMINTHRES=500000 | `SET BANDWIDTHMNG:RANGE=SYSTEM,CIRMINTHRES=300000,PIRMINTHRES=500000;` | 110311 APN_DNN 场景 |
| 整机生效+切片规则 | RANGE=SYSTEM, BWMSUBSCRCTRL=PCC_RULE_IGNORE | `SET BANDWIDTHMNG:RANGE=SYSTEM,CIRMINTHRES=300000,PIRMINTHRES=500000,BWMSUBSCRCTRL=PCC_RULE_IGNORE;` | 110311 切片场景（用 BWM Rule 规则，忽略 PCC 规则） |

注：110311 数据规划表 CIRMINTHRES 样例 30000，但脚本取 300000——以脚本为准。

**BWMSUBSCRCTRL 取值语义**：用户级业务带宽规则来源。PCC_RULE_PRIOR（默认，PCC 规则优先）/ PCC_RULE_IGNORE（切片等用 BWM Rule 时，忽略 PCC 规则）。

**步骤位置**：在 ADD BWMCONTROLLER 之后、配置 BWM 用户组/BWMRULE 之前——先定生效范围，控制器和规则才能落到正确汇总层。

## 约束

- 最大记录数 1（全局 SET 类，整机一份） · error · 超限 SET 失败
- 本命令属高危命令，Range 变化后 bwmcontroller/bwmservice 等需同步修改 · error · 级联失效
- RANGE 由 CPU 切 SYSTEM 需整机无在线用户 · error · 否则切换期间统计混乱
- SYSTEM 范围需整机级汇总，执行前需重新容量评估 · warning · 无法评估联系华为技术支持
- 命令执行后立即生效 · info · 无需刷新

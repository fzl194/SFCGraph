# 设置APN的IMS属性（SET APNIMSATTR）

> 命令静态知识见 [SET APNIMSATTR](command/UDG/20.15.2/SET-APNIMSATTR.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。**本命令属高危命令**——配置错误将影响用户 VoLTE 业务。

## 配置方法

SET APNIMSATTR 用于将 APN 标记为 IMS 信令承载，使 VoLTE 语音业务在该 APN 上可用。**APN 必选**（引用已建 APN 实例）；**IMSSWITCH 可选**（典型 ENABLE）。**整机最多 10000 条 ApnImsAttr**。核心配置空间有 2 个维度：**APN 实例**（APN）、**IMS 使能**（IMSSWITCH）。1 个特性命中（GWFD-020251 VoLTE 基础语音业务）使用 `ims` + `ENABLE` 范式。

### 配置维度 1：APN 实例（参数 APN，必选）

| 取值 | 作用 |
|---|---|
| **字符串，1~63 字符** | APN 实例名；**引用** [0-00020](task/UDG/20.15.2/0-00020.md) ADD APN 已建实例；不可新建 |

### 配置维度 2：IMS 使能开关（参数 IMSSWITCH，可选）

| 取值 | 作用 |
|---|---|
| **DISABLE** | 关闭 IMS 功能（VoLTE 语音业务不可用） |
| **ENABLE** | 开启 IMS 功能（VoLTE 语音业务可用，020251 范式） |

**典型脚本（GWFD-020251 VoLTE 基础语音业务）**：
```
SET APNIMSATTR:APN="ims",IMSSWITCH=ENABLE;
```

**步骤位置**：在所属 feature 编排中通常位于"步骤 N：开 APN 的 IMS 属性"——前置 [0-00020](task/UDG/20.15.2/0-00020.md) ADD APN（建 APN 实例）+ [0-00155](task/UDG/20.15.2/0-00155.md) ADD APNIMSSIGFLTR（建 IMS 信令分类器），filter 匹配依赖本命令的 IMS 开关；后续按需启用 VoLTE 业务。

## 决策点：IMSSWITCH IMS 使能（DP 0-00156）

问：本 APN 是否启用 IMS 功能？

| 选项 | IMSSWITCH | 适用场景 |
|---|---|---|
| **关闭** | DISABLE | 非 VoLTE 业务；该 APN 不承载 IMS 信令 |
| **开启**（020251） | ENABLE | VoLTE 语音业务承载（020251 范式） |

> 1 个特性命中（020251）使用 ENABLE；本特性下固定 ENABLE，不允许关闭。

## 约束

- **APN 必选**（critical，rule-0-00156）：APN 是必选参数 — 后果：缺省命令拒收
- **高危命令**（critical，rule-0-00156）：本命令属高危命令，影响用户 VoLTE 业务 — 后果：误配导致用户 VoLTE 业务不可用
- **整机最多 10000 条**（warning，rule-0-00156）：整机 ApnImsAttr 最多 10000 条 — 后果：超限 SET 失败
- **执行后立即生效**（warning，rule-0-00156）：本命令执行后立即生效 — 后果：误操作即刻影响 VoLTE 业务
- **filter 匹配依赖前置**（warning，rule-0-00156）：IMS filter 匹配依赖 [0-00155](task/UDG/20.15.2/0-00155.md) ADD APNIMSSIGFLTR 已配 — 后果：未配 filter 时 IMS 信令无法识别
- **APN 前置依赖**（warning，rule-0-00156）：APN 须先经 [0-00020](task/UDG/20.15.2/0-00020.md) ADD APN 配置 — 后果：APN 不存在命令拒收
- **SGW-U/I-UPF 性能影响**（info，rule-0-00156）：SGW-U/I-UPF 上使能会触发 PF 的 SIP 信令解析、可能影响性能 — 后果：高负载下 IMS 信令处理延迟

# 计费三件套

> **跨域通用 backbone**（对齐 ConfigTask 计费 1-00001）：费率策略组三级链。计费/BWM/URL过滤共用（URL过滤用简化 URR 作计费属性）。被引用于 [2-00002](task/UDG/20.15.2/2-00002.md)。

## 配置方法

三级链：URR → URRGROUP（绑URR）→ PCCPOLICYGRP（绑URRGROUP）。

| 步骤 | 命令 | 关键参数 |
|---|---|---|
| 使用量上报规则 | ADD URR | URRNAME + URRID + USAGERPTMODE |
| URR 组 | ADD URRGROUP | URRGROUPNAME + UPURRNAME1/DOWNURRNAME1 |
| PCC 策略组 | ADD PCCPOLICYGRP | PCCPOLICYGRPNM + URRGROUPNAME |

**典型脚本（URL过滤简化：单一离线 URR 作计费属性）**：
```
ADD URR:URRNAME="urr_2",URRID=2,USAGERPTMODE=OFFLINE;
ADD URRGROUP:URRGROUPNAME="urrgroup1",UPURRNAME1="urr_2",DOWNURRNAME1="urr_2";
ADD PCCPOLICYGRP:PCCPOLICYGRPNM="ppg_test",URRGROUPNAME="urrgroup1";
```

**步骤位置**：承接 [1-00009](task/UDG/20.15.2/1-00009.md)（过滤链）；产出 PCCPOLICYGRPNM 供下游 RULE.POLICYNAME 引用。

## 场景差异

> 计费三件套被计费族 7 特性复用（2-00003~2-00009），差异集中在 URR 统计类型、计费模式 × URR 组装、PCCPOLICYGRP 信令参数三类。

### 1. METERINGTYPE 统计类型变种（URR 的 OFFMETERINGTYPE/ONLMETERINGTYPE）

| 取值 | 统计对象 | 对应 feature |
|---|---|---|
| VOLUME | 业务字节数（datavolumeFBCUplink/Downlink） | [2-00003](task/UDG/20.15.2/2-00003.md) 内容 / [2-00008](task/UDG/20.15.2/2-00008.md) 流量 |
| DURATION | 业务持续时间（CC-Time / 话单时间信元） | [2-00007](task/UDG/20.15.2/2-00007.md) 时长 |
| EVENT | 业务事件次数（numberOfEvents / serviceSpecificUnits，性能有影响） | [2-00009](task/UDG/20.15.2/2-00009.md) 事件 |
| FREE | 免费/防欺诈兜底（不计费，需配合 UNC 实现免费） | 各特性异常场景 |

> 注：DURATION/EVENT 取值来自各特性概述/实现原理；activation 模板复用，均演示 VOLUME。VOLUME 外的取值证据见各 feature。

### 2. 计费模式 × URR 组装（USAGERPTMODE）

| 模式 | URR 配置 | 对应 feature |
|---|---|---|
| 在线 | 单 online URR：`USAGERPTMODE=ONLINE`+`ONLMETERINGTYPE` | [2-00004](task/UDG/20.15.2/2-00004.md) |
| 离线 | 单 offline URR：`USAGERPTMODE=OFFLINE`+`OFFMETERINGTYPE`（8 种：VOLUME/TIME/EVENT/FREE/组合） | [2-00005](task/UDG/20.15.2/2-00005.md) |
| 融合 | **双 URR 同组**：`UP/DOWNURRNAME1`=offline URR + `UP/DOWNURRNAME2`=online URR | [2-00006](task/UDG/20.15.2/2-00006.md) |
| 融合（RGAPPLIED=DEFAULT） | **只能单配**（仅离线或仅在线二选一，critical 约束，否则计费冲突） | [2-00006](task/UDG/20.15.2/2-00006.md) DP-A |

### 3. 其他跨域差异

- **URL过滤**：简化单一离线 URR（仅作计费属性，USAGERPTMODE=OFFLINE）。
- **BWM**：URR 关联带宽控制（见 GWFD-110311）。
- **PCCPOLICYGRP 的 `SIGURRGRPNAME`**（信令URR组）：异常/防欺诈场景的信令费率组挂这里；[2-00003](task/UDG/20.15.2/2-00003.md) 内容 / [2-00006](task/UDG/20.15.2/2-00006.md) 融合用（异常 URRGROUP 通过 `SIGURRGRPNAME` 挂到 any 策略组）。

## 决策点

| 选项 | 影响（参数/命令/联动） |
|---|---|
| 计费方式 USAGERPTMODE=OFFLINE（联动特性级） | 离线计费；URL过滤通常用此 |
| 计费方式 USAGERPTMODE=ONLINE（联动特性级） | 在线计费 |

## 约束

- **URRID 全网唯一**（critical）：从 1000 起始 — 重复则 URR 冲突。
- **URRGROUP 引用一致**（critical）：URRGROUP.UPURRNAME1/DOWNURRNAME1 须与 URR.URRNAME 一致 — 不一致则组引用断链。
- **PCCPOLICYGRP 引用一致**（critical）：PCCPOLICYGRP.URRGROUPNAME 须与 URRGROUP 一致 — 不一致则策略组引用断链。

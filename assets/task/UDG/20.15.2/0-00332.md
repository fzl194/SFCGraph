# 查询SMF的NodeID（LST CPNODEID）

> 命令静态知识见 [LST CPNODEID](command/UDG/20.15.2/LST-CPNODEID.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法（配置方法字典）

LST CPNODEID 是 SMF NodeID 族**配置查询**命令，输出 CPNAME / IPV4NODEID / IPV6NODEID / FQDNNODEID 等配置字段。GWFD-010105 基于 SMF 分配地址场景下用于核查 SMF 实例。命令核心配置空间有 2 个维度：**查询范围**（CPNAME 可选：指定单个/查全部）、**详细度**（VERBOSE 可选）。

### 配置维度 1：查询范围（参数 CPNAME，可选）

| 取值 | 行为 |
|---|---|
| 指定 CPNAME | 仅查该 SMF 实例配置 |
| 不配 | 查所有 SMF 实例配置 |

### 配置维度 2：详细度（参数 VERBOSE，可选）

| 取值 | 行为 |
|---|---|
| ENABLE | 输出完整字段（含 IPv4/IPv6/FQDN 全 NodeID） |
| 不配 | 输出关键字段 |

**典型脚本（GWFD-010105 基于 SMF 调测：查所有 SMF 实例）**：
```
LST CPNODEID:;
```

**典型脚本（指定 CPNAME 查询完整字段）**：
```
LST CPNODEID:CPNAME="smfnode1",VERBOSE=ENABLE;
```

> 命令 notes：适用 NF：PGW-U、UPF；用于 SMF 实例核查。

**步骤位置**：在所属 feature 编排中通常位于"调测/验证阶段"——前置条件：CPNODEID 已由 [0-00303](task/UDG/20.15.2/0-00303.md) 建好；用于基于 SMF 地址分配链路核查。

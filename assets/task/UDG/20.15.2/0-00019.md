# 设置License开关（SET LICENSESWITCH）

> 命令静态知识见 [SET LICENSESWITCH](command/UDG/20.15.2/SET-LICENSESWITCH.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

本命令是**特性激活的通用前置开关**，跨 56+ 特性复用——每个特性有自己的 License 控制项码（LICITEM），配本命令打开该特性的 License 后才能配该特性的业务命令。**固定模式，两参数**：

**配置范式（每特性一行，激活=ENABLE，去激活=DISABLE）**

```
SET LICENSESWITCH:LICITEM="<特性License码>",SWITCH=ENABLE;
```

**各特性 LICITEM 取值（从证据数据规划表/脚本归纳）**

| 特性 | LICITEM | SWITCH | 说明 |
|---|---|---|---|
| 动态 PCC / 本地 PCC（GWFD-020351） | `LKV3G5PCCB01` | ENABLE | PCC 策略组前置 |
| TCP 重传识别（GWFD-020307） | `LKV3G5TCPR01` | ENABLE | TCP 重传计费前置 |
| 终端异常下行流量检测（GWFD-020305） | `LKV3G5ADTD01` | ENABLE | APN 下异常流量检测前置 |
| HTTP2.0 协议回落（GWFD-110202） | 特性对应码 | ENABLE | 见特性 md |
| Routing Behind MS（GWFD-110910） | 特性对应码 | ENABLE | 见特性 md |
| IPv6 承载上下文（GWFD-020401） | 特性对应码 | ENABLE | 见特性 md |
| IPv4v6 双栈接入（GWFD-020403） | 特性对应码 | ENABLE | 见特性 md |

**取值来源**：LICITEM 每特性固定取一个，由 feature task 决定（feature-level decision，不在本 atom 升 DP）。LICITEM 码格式如 `LKV3G5xxxxxx`，长度 0~255 字符串。

**前置条件**：License 文件必须已激活——未激活时无法设置任何 License 项；激活后只能设置已购买且支持配置开关的 License 项。

**执行顺序**：本命令是特性配置的**第 1 步**（打开 License），之后才配该特性的业务命令（如 PCC 特性先 `SET LICENSESWITCH` 开 PCC，再 [ADD PCCPOLICYGRP](command/UDG/20.15.2/ADD-PCCPOLICYGRP.md)）。

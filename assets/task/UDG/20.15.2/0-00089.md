# 配置错误码（ADD ERRORCODE）

> 命令静态知识见 [ADD ERRORCODE](command/UDG/20.15.2/ADD-ERRORCODE.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法

ADD ERRORCODE 是重定向触发的"错误码范围定义"命令：定义一组错误码范围，重定向动作命令通过 BINDERRCODENAME 引用该范围作为触发条件。本命令的核心配置空间有 3 个维度：**操作码 ERRORCODEOP**（决定范围语义）、**范围参数配对**（START/END 在不同 OP 下条件必选）、**错误码名 ERRORCODENAME**（全网规划命名）。

### 配置维度 1：操作码（参数 ERRORCODEOP，必选）

| 取值 | 作用 | START/END 配对 |
|---|---|---|
| **EQUAL** | 错误码等于指定值 | 仅 START 有效（END 不出现） |
| **LT** | 错误码小于等于指定值 | 仅 END 有效（START 不出现） |
| **GT** | 错误码大于等于指定值 | 仅 START 有效（END 不出现） |
| **RANGE** | 错误码在指定范围内（含边界值） | START + END 都必选 |

### 配置维度 2：范围参数（参数 ERRORCODESTART/ERRORCODEEND，按 OP 条件必选）

| 参数 | 取值范围 | 出现条件 |
|---|---|---|
| **ERRORCODESTART** | 整数 0~1000 | OP=EQUAL/GT/RANGE 时必选 |
| **ERRORCODEEND** | 整数 0~1000 | OP=LT/RANGE 时必选 |

### 配置维度 3：错误码名称（参数 ERRORCODENAME，必选，全网规划）

字符串 1~31（不区分大小写、不支持空格）；命名按业务语义约定（如 dnserrorcode/httperrorcode）；产出的 ERRORCODENAME 对象被 [0-00090 DNSOVERWRITING](task/UDG/20.15.2/0-00090.md) / [0-00091 SMARTHTTPREDIR](task/UDG/20.15.2/0-00091.md) 的 BINDERRCODENAME 引用。

> **典型范式（2 协议分叉）**：
> ```
> # DNS 纠错：NXDOMAIN（错误码=3）
> ADD ERRORCODE:ERRORCODENAME="dnserrorcode",ERRORCODEOP=EQUAL,ERRORCODESTART=3;
> # HTTP 智能重定向：4xx/5xx 错误码（>=400）
> ADD ERRORCODE:ERRORCODENAME="httperrorcode",ERRORCODEOP=GT,ERRORCODESTART=400;
> ```

**步骤位置**：在所属 feature 编排中通常为"步骤 2：配置错误码范围"——ADD EXTENDEDFILTER 之后、动作命令（DNSOVERWRITING/SMARTHTTPREDIR）之前。

## 决策点：操作码与协议分叉（DP 0-00089）

问：ADD ERRORCODE 的 ERRORCODEOP 取何值？

| 选项 | ERRORCODEOP | START/END | 适用协议 | 典型特性 |
|---|---|---|---|---|
| 等于指定值（EQUAL） | EQUAL | 仅 START | DNS NXDOMAIN（=3） | [2-00016](task/UDG/20.15.2/2-00016.md) DNS纠错 |
| 大于等于指定值（GT） | GT | 仅 START | HTTP 4xx/5xx（≥400） | [2-00017](task/UDG/20.15.2/2-00017.md) HTTP智能重定向 |
| 小于等于指定值（LT） | LT | 仅 END | 自定义场景 | 自定义 |
| 范围（RANGE） | RANGE | START+END | 自定义场景 | 自定义 |

## 约束

- **整机最大记录数为 100**（warning，rule-0-00377）：ErrorCodeName 数量上限 — 后果：超规格新增失败
- **单 ErrorCodeName 下最多 500 组错误码范围**（warning，rule-0-00377）：单名下范围组数上限 — 后果：超 500 新增失败
- **整机最多 2000 组错误码范围**（warning，rule-0-00377）：范围总组数上限 — 后果：超 2000 新增失败
- **立即生效**（info，隐含）：该命令执行后立即生效 — 后果：错误码范围变更实时影响重定向触发
- **ERRORCODENAME 全网规划**（warning，隐含）：跨网元统一命名，否则引用方无法识别 — 后果：跨网元引用失败
- **BINDERRCODENAME 引用须已配置**（critical，隐含）：重定向动作命令的 BINDERRCODENAME 须先经本命令配置 — 后果：引用不存在拒收

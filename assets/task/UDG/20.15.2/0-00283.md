# 设置RTSDNN参数（SET RTSDNNPARA）

> 命令静态知识见 [SET RTSDNNPARA](command/UDG/20.15.2/SET-RTSDNNPARA.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

本任务设置通用 DNN 漫游分流的全局参数。PGW-U/UPF 实例按**公网 UPF** 或**专网 UPF** 角色不同，参数配置完全不同——由 atom-级决策点 [DP 0-00152](task/UDG/20.15.2/0-00152.md) 编排。同一条 `SET RTSDNNPARA` 命令在特性 md 的任务示例中按角色执行 3 次（公网参数 / 专网参数 / 可选 NAT ALG）。

**公网 UPF 角色（样例取值，GWFD-020531）**

| 参数 | 含义 | 样例值 | 获取方法 |
|---|---|---|---|
| PUBLICNETSW | 公网通用 DNN 漫游分流开关 | ENABLE | 固定取值 |
| N4RPTSW | N4 上报通用 DNN 漫游分流锚点功能开关 | ENABLE | 与对端协商 |
| UEIPCONFLICTCHK | UE IP 地址冲突检测开关 | ENABLE | 本端规划 |
| UEIPREALLOC | UE IP 地址重分配开关 | ENABLE | 本端规划 |
| UEIPREALLOCNUM | UE IP 地址重分配次数 | 1000 | 本端规划（0～3000，仅 UEIPREALLOC=ENABLE 时必填） |
| UEIPCONFLICTPLY | UE IP 地址冲突处理策略 | BYPASS | 本端规划（仅 UEIPCONFLICTCHK=ENABLE 时必填；BYPASS=冲突时继续激活用户） |

```
SET RTSDNNPARA: PUBLICNETSW=ENABLE, UEIPCONFLICTCHK=ENABLE, UEIPREALLOC=ENABLE, UEIPREALLOCNUM=1000, UEIPCONFLICTPLY=BYPASS;
```

**专网 UPF 角色（样例取值）**

| 参数 | 含义 | 样例值 | 获取方法 |
|---|---|---|---|
| PRIVATENETSW | 专网通用 DNN 漫游分流开关 | ENABLE | 固定取值 |
| NATALGPROTV4 | IPv4 NAT ALG 协议 | FTP-1&RTSP-1 | 本端规划（可选） |
| NATALGPROTV6 | IPv6 NAT ALG 协议 | FTP-1 | 本端规划（可选） |

```
SET RTSDNNPARA: PRIVATENETSW=ENABLE;
SET RTSDNNPARA: NATALGPROTV4=FTP-1&RTSP-1, NATALGPROTV6=FTP-1;
```

**未绑参数说明（按 baseline §4.1 冷门不绑）**：`RPTCTRLINTERVAL`（MultiDNN 上报事件流控间隔，0～3600 秒）/`PKTPROCACTION`（报文处理动作）/`RESCHKINTERVAL`（MultiDNN 资源阈值核查间隔，120～1440 分钟）均为 optional，本特性场景不配（RESCHKINTERVAL 由整网阈值管控非本特性职责）。

## 决策点

[DP 0-00152 SET RTSDNNPARA UPF 角色分工](task/UDG/20.15.2/0-00152.md)：当前 UDG 实例是公网 UPF 还是专网 UPF？

- **公网 UPF**（`PUBLIC_UPF`，默认）：mandatory `PUBLICNETSW`/`UEIPCONFLICTCHK`/`UEIPREALLOC`，条件必填 `UEIPREALLOCNUM`(UEIPREALLOC=ENABLE 时)/`UEIPCONFLICTPLY`(UEIPCONFLICTCHK=ENABLE 时)，可选 `N4RPTSW`，**不配** `PRIVATENETSW`/`NATALG*`。
- **专网 UPF**（`PRIVATE_UPF`）：mandatory `PRIVATENETSW`，可选 `NATALGPROTV4`/`NATALGPROTV6`，**不配** `PUBLICNETSW`/`UEIPCONFLICTCHK`/`N4RPTSW`。

7 个参数绑定（`PUBLICNETSW`/`PRIVATENETSW`/`UEIPCONFLICTCHK`/`UEIPREALLOC`/`UEIPREALLOCNUM`/`UEIPCONFLICTPLY` 等）由该 DP 驱动。

## 约束

- 命令最大记录数 **1**；超出被拒。
- 执行后**立即生效**。
- 该命令存在系统初始记录（参数初始默认值，命令 notes）。
- 公网/专网 UPF 角色参数 mandatory 编排见 DP 0-00152。

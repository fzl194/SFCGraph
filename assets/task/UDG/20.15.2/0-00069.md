# 配置下行PHB映射（SET QOSPHB）

> 命令静态知识见 [SET QOSPHB](command/UDG/20.15.2/SET-QOSPHB.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

SET QOSPHB 在接口下行（egress）方向，把 UDG 内部的服务等级+报文颜色映射成外部 VLAN 报文的 802.1p 值（IP 网络→VLAN 的优先级映射）。PHBTYPE 固定 8021p，DSNAME 引用 DS 域，PHBVALUE/SERVICECLASS/COLOR 三参数按规划表逐条配置一组对应关系。

**配置范式（从证据数据规划表归纳）**

| DSNAME | PHBTYPE | PHBVALUE | SERVICECLASS | COLOR | 典型行 |
|---|---|---|---|---|---|
| default（GWFD-010201 示例） | 8021p | 3 | be | green | `SET QOSPHB:DSNAME="default",PHBTYPE=8021p,PHBVALUE=3,SERVICECLASS=be,COLOR=green;` |
| DS 域名（IPFD-012001 占位） | 8021p | PHB 值（0~7） | 服务分类 | 颜色分类 | `SET QOSPHB:DSNAME="DS域名",PHBTYPE=PHB类型,PHBVALUE=PHB值,SERVICECLASS=服务分类,COLOR=颜色分类;` |

**关键规则**
- **本 atom 仅出口方向**：与入口方向的 SET QOSBA（[0-00067](task/UDG/20.15.2/0-00067.md)）对偶；由 DP 0-00016 选 opt-egress 时使用，入口 BA 与出口 PHB 成对配置（rule 0-00283 入口+出口成对约束）。
- **PHBTYPE 固定 8021p**（本 atom 场景，802.1p 映射）。
- **DSNAME** 引用 [配置DS域](task/UDG/20.15.2/0-00066.md)（ADD QOSDIFFERSERV）。
- **PHBVALUE** 802.1p 取值 0~7，取值越大优先级越高；DSCP 场景为 0~63。
- 配置完后还需 [配置QoS接口信任](task/UDG/20.15.2/0-00068.md)（ADD QOSIFTRUST）把 DS 域绑到接口，简单流分类才生效。

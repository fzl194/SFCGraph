# 设置rule级HTTP2.0协议回落开关（MOD PCCPOLICYGRP）

> 命令静态知识见 [MOD PCCPOLICYGRP](command/UDG/20.15.2/MOD-PCCPOLICYGRP.md)。本页只讲配置生成实例化时这条命令**怎么配**。

> **单参数限定**：本 atom 严格限定 HTTP2DEGRADESW 一个参数；命令真相 18 参数中 16 个（PCCACTPROPNAME/EXTENDPROPNAME/FUPSESSIONEXC/Token*/Signal*/ADCMUTEFLAG/QOSPROPNAME/URRGROUPNAME/SIGURRGRPNAME/EVENT*/MMSCHARGETYPE/CFGDOMAINNAME）主特性场景不用，若后续计费/Token 特性要修改 PCC 策略组须新建独立 atom 承载。

## 配置方法

本命令是 **GWFD-110202（HTTP2.0 协议回落）rule 级开关**：基于 PCC 策略组粒度控制回落，**回落优先级最高**（rule > APN > 全局），一旦 rule 级显式配置，不论 APN/全局为何值均按 rule 级生效。立即生效。

**配置范式（单范式 + 决策开关）**

| 参数 | 取值样例 | 来源 | 说明 |
|---|---|---|---|
| PCCPOLICYGRPNM | `pcc-test` | 引用 [0-00003](task/UDG/20.15.2/0-00003.md).PCCPOLICYGRPNM | PCC 策略组名，ADD PCCPOLICYGRP 先建 |
| HTTP2DEGRADESW | `ENABLE` / `DISABLE` / `INHERIT` | decision_driven → DP 0-00143 | 3-way 编排，默认 INHERIT |

**任务示例**

```
MOD PCCPOLICYGRP: PCCPOLICYGRPNM="pcc-test", HTTP2DEGRADESW=ENABLE;
```

## 决策点

**DP 0-00143**（single_choice）rule 级 HTTP2.0 协议回落开关 3-way 编排（默认 INHERIT）

| 选项 | 取值 | 说明 |
|---|---|---|
| opt-rule-enable | `ENABLE` | rule 级强制开启回落（本特性示例采用，优先级最高） |
| opt-rule-disable | `DISABLE` | rule 级强制关闭回落 |
| opt-rule-inherit（默认） | `INHERIT` | 继承 APN 级 [0-00149](task/UDG/20.15.2/0-00149.md) HTTP2DEGRADESW |

与 APN 级 DP [[UDG@20.15.2@DecisionPoint@0-00142]] + 全局级 SRVCOMMONPARA 三层联动，防止 3 层均 INHERIT 时无生效源。

## 约束

- **rule-0-00354**（warning）立即生效；参数全 optional（无选配则配置空 PccPolicyGrp）；PCCPOLICYGRPNM 前置 ADD PCCPOLICYGRP(0-00003)——只有已添加成功的才能 MOD；HTTP2DEGRADESW=INHERIT 时继承 APN 级(0-00149)；rule 级优先级最高，生效时 APN/全局不生效

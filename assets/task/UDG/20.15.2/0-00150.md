# 设置rule级HTTP2.0协议回落开关（MOD PCCPOLICYGRP）

> 命令静态知识见 [MOD PCCPOLICYGRP](command/UDG/20.15.2/MOD-PCCPOLICYGRP.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

> **单参数限定**：本 atom 严格限定 HTTP2DEGRADESW 一个参数；命令真相 18 参数中 16 个（PCCACTPROPNAME/EXTENDPROPNAME/FUPSESSIONEXC/Token*/Signal*/ADCMUTEFLAG/QOSPROPNAME/URRGROUPNAME/SIGURRGRPNAME/EVENT*/MMSCHARGETYPE/CFGDOMAINNAME）主特性场景不用；若后续计费/Token 特性要修改 PCC 策略组须新建独立 atom 承载。

## 配置方法

MOD PCCPOLICYGRP 在 HTTP2.0 协议回落场景下作为 **rule 级开关**：基于 PCC 策略组粒度控制回落。**rule 级回落优先级最高**（rule > APN > 全局），一旦 rule 级显式配置，不论 APN/全局为何值均按 rule 级生效。立即生效。1 个特性命中（110202 HTTP2.0 协议回落）。

### 配置维度 1：PCC 策略组定位（参数 PCCPOLICYGRPNM，必选）

| 取值 | 来源 |
|---|---|
| **PCC 策略组名**（如 `pcc-test`） | 引用 [0-00003](task/UDG/20.15.2/0-00003.md).PCCPOLICYGRPNM（ADD PCCPOLICYGRP 先建） |

### 配置维度 2：回落开关（参数 HTTP2DEGRADESW，必选，DP 0-00150 决定）

| 取值 | 作用 | 适用场景 |
|---|---|---|
| **ENABLE** | rule 级强制开启回落（优先级最高） | 110202 范式采用 |
| **DISABLE** | rule 级强制关闭回落 | rule 不需要回落 |
| **INHERIT**（默认） | 继承 APN 级 [0-00149](task/UDG/20.15.2/0-00149.md) HTTP2DEGRADESW | 跟随 APN 策略 |

> **三层保护**：rule 级(本 atom) > APN 级(0-00149) > 全局级(0-00065)；rule 级显式配置后立即覆盖 APN/全局。

**任务示例（110202 范式）**：

```
MOD PCCPOLICYGRP: PCCPOLICYGRPNM="pcc-test", HTTP2DEGRADESW=ENABLE;
```

**步骤位置**：在所属 feature 编排中通常位于"步骤 N：rule 级回落开关"——前置 [0-00003](task/UDG/20.15.2/0-00003.md) ADD PCCPOLICYGRP；可选 APN 级联动 [0-00149](task/UDG/20.15.2/0-00149.md)；全局级 SRVCOMMONPARA(0-00065)。

## 决策点：rule 级回落开关（DP 0-00150）

| 选项 | HTTP2DEGRADESW | 适用场景 |
|---|---|---|
| **opt-rule-enable** | ENABLE | rule 级强制开启回落（110202 范式采用，优先级最高） |
| **opt-rule-disable** | DISABLE | rule 级强制关闭回落 |
| **opt-rule-inherit（默认）** | INHERIT | 继承 APN 级 SET APNHTTP2DGRD(0-00149) HTTP2DEGRADESW |

> 三层联动保护：与 APN 级 DP 0-00149 + 全局级 SRVCOMMONPARA 组合，防止 3 层均 INHERIT 时无生效源。

## 约束

- **立即生效**（warning，rule-0-00150）：执行后立即生效 — 后果：误配即刻影响该 PCC 策略组下用户的 HTTP2.0 回落
- **PCCPOLICYGRPNM 前置必备**（critical，rule-0-00150）：PCC 策略组须已由 ADD PCCPOLICYGRP(0-00003) 添加成功 — 后果：策略组未建命令拒收
- **参数全 optional**（info，rule-0-00150）：18 参数均为 optional，无选配时配置空 PccPolicyGrp — 后果：空配置时记录存在但无生效策略
- **HTTP2DEGRADESW=INHERIT 继承 APN 级**（warning，rule-0-00150）：INHERIT 时继承 APN 级 SET APNHTTP2DGRD(0-00149) HTTP2DEGRADESW — 后果：APN 未配则 rule 也无法生效
- **rule 级优先级最高**（critical，rule-0-00150）：rule 级显式 ENABLE/DISABLE 时覆盖 APN/全局 — 后果：误配 rule 级即覆盖下游所有层
- **三层联动保护**（critical，rule-0-00150）：rule(本 atom)/APN(0-00149)/全局(0-00065) 不可均 INHERIT — 后果：3 层均 INHERIT 时无生效源
- **16 个参数未绑**（info，rule-0-00150）：PCCACTPROPNAME/EXTENDPROPNAME/FUPSESSIONEXC/Token*/Signal*/ADCMUTEFLAG/QOSPROPNAME/URRGROUPNAME/SIGURRGRPNAME/EVENT*/MMSCHARGETYPE/CFGDOMAINNAME 在 110202 场景未用

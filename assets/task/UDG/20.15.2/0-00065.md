# 配置业务公共参数（SET SRVCOMMONPARA）

> 命令静态知识见 [SET SRVCOMMONPARA](command/UDG/20.15.2/SET-SRVCOMMONPARA.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法

SET SRVCOMMONPARA 是"业务公共参数全局开关/阈值"层——承载 40+ 项独立全局参数，各特性按需设其一或几项。本命令的核心配置空间有 3 个维度：**开关型参数**（ENABLE/DISABLE 二值，影响全局业务使能）、**配额型参数**（流量/时长配额及单位，控制默认配额）、**枚举型参数**（HTTPREDIRCODE 重定向码、RTSP 计费模式、协议层级等）。4 个特性命中覆盖 010201 TUNNELMARKINGID、110202 HTTP2DEGRADESW、110910 FDMAXNUM/USERFLOWADJSW、110281 HTTPREDIRCODE。

### 配置维度 1：开关型参数（ENABLE/DISABLE 二值）

| 参数 | 作用 | 典型场景 |
|---|---|---|
| **TUNNELMARKINGID** | TunnelMarking 开关（VLAN/IP 网络间优先级映射使能） | 010201 QoS 流量管理 |
| **HTTP2DEGRADESW** | HTTP2.0 协议回落开关（全局>APN>rule 优先级） | 110202 HTTP2.0 回落 |
| **USERFLOWADJSW** | 单用户最大流表数动态调整开关（与 FDMAXNUM 配对） | 110910 Routing Behind MS |
| **RTSPPAUSE** | RTSP 业务暂停时是否收费 | RTSP 计费控制 |
| **TUNNELMARKINGID** | TunnelMarking 开关 | QoS 优先级映射入口 |
| **XONLINEHOST** | X-Online-Host 生效标识（替代报文 URL HOST） | HTTP 头增强 |
| **UPDATERULESW** | 更新特征库规则开关 | SA 协议识别 |
| **HTTP2DEGRADESW** | HTTP2.0 协议回落 | HTTP2.0 协议 |
| **SAVOLUMESTATSW** | SA 流量统计开关 | SA 流量统计 |
| **SADISCOVERSW** | SA 发现协议开关 | SA 协议发现 |
| **RGAGESWITCH** | 计费节点老化开关 | 计费节点老化 |

### 配置维度 2：配额型参数（流量/时长配额 + 单位）

| 参数 | 取值范围 | 作用 |
|---|---|---|
| **DEFAULTVOLUME** | 整数 | 默认流量配额（单位参考 DFTQUOTAPARA） |
| **DEFAULTTIME** | 0~3600（秒） | 默认时长配额 |
| **DFTQUOTAPARA** | Byte / KByte | 默认流量配额单位（超过 4G 配额需配 KByte） |
| **FDMAXNUM** | 整数 | 用户 session 五元组节点最大数量（默认 600） |

### 配置维度 3：枚举型参数（多枚举值）

| 参数 | 枚举值 | 作用 | 典型场景 |
|---|---|---|---|
| **HTTPREDIRCODE** | RSP_CODE_301 / RSP_CODE_302 / RSP_CODE_303 | HTTP 重定向返回码（301永久/302临时/303 See Other） | 110281 用户 Portal |
| **RTSPCHGMETHOD** | ALL / PAUSE_ONLY / ... | RTSP 计费模式 | RTSP 计费 |
| **BWMPROTLEVEL** | APP / ... | 带宽规则匹配协议层级选择 | 带宽规则匹配 |
| **ADCPROTLEVEL** | APP / ... | ADC 规则匹配协议层级选择 | ADC 规则匹配 |
| **PERFPROTLEVEL** | APP / ... | 性能统计协议层级选择 | 性能统计 |
| **RTPPROTLEVEL** | APP / ... | 报表协议层级选择 | 报表 |
| **DNSOLENDOMAIN** | ... | DNS 超长域名截断处理 | DNS SA 处理 |
| **SAURLLEN** | ... | SA 支持 URL 最大长度（超长截断） | SA 处理 |

**典型脚本（010201 QoS TunnelMarking）**：
```
SET SRVCOMMONPARA:TUNNELMARKINGID=ENABLE;
```

**典型脚本（110202 HTTP2.0 回落）**：
```
SET SRVCOMMONPARA: HTTP2DEGRADESW=ENABLE;
SET SRVCOMMONPARA: HTTP2DEGRADESW=DISABLE;
```

**典型脚本（110910 Routing Behind MS）**：
```
SET SRVCOMMONPARA:FDMAXNUM=800,USERFLOWADJSW=ENABLE;
```

**典型脚本（110281 用户 Portal HTTP 重定向码）**：
```
SET SRVCOMMONPARA:HTTPREDIRCODE=RSP_CODE_302;
```

**步骤位置**：在所属 feature 编排中通常位于"步骤 1：全局参数使能"——入口/出口两方向各执行一次（010201 QoS）；或特性初始位置（110202/110281/110910）。

## 决策点：开关型参数取值（DP 0-00065）

问：SET SRVCOMMONPARA 的开关型参数取 ENABLE 还是 DISABLE？

| 选项 | 典型参数 | 适用场景 |
|---|---|---|
| 使能（ENABLE） | TUNNELMARKINGID / HTTP2DEGRADESW / USERFLOWADJSW | 010201 QoS 优先级映射使能、110202 HTTP2.0 回落启用、110910 动态调整启用 |
| 不使能（DISABLE） | TUNNELMARKINGID / HTTP2DEGRADESW | 系统初始 / 关闭回落 |

> HTTP2.0 回落优先级：rule > APN（[0-00231](task/UDG/20.15.2/0-00231.md) SET APNHTTP2DGRD）> 全局（本命令）。

## 决策点：HTTP 重定向码选择（DP 0-00066）

问：HTTPREDIRCODE 选哪个重定向返回码？

| 选项 | 含义 | 适用场景 |
|---|---|---|
| RSP_CODE_301 | 永久重定向 | URL 永久迁移 |
| RSP_CODE_302 | 临时重定向（推荐） | 用户 Portal 重定向（默认） |
| RSP_CODE_303 | See Other（GET 访问） | 表单提交后重定向（303 See Other） |

> UDG 构造的 HTTP 重定向回复报文返回码全局配置，**非裸数字 301/302/303**，必须用 RSP_CODE_xxx 枚举。

## 约束

- **高危命令**（critical，rule-0-00331）：改此配置前先确认无静态地址冗余用户，否则可能导致此部分用户业务不通；可能影响业务 — 后果：高危配置错误影响全局业务
- **最大记录数为 1**（critical，rule-0-00331）：全局唯一；多次 SET 后值覆盖 — 后果：违反唯一性命令拒收
- **执行后立即生效**（warning，隐含）：无 REFRESHSRV 缓冲 — 后果：配置错误立即影响业务
- **条件必选 IPV4EMBV6PFX**（warning，隐含）：IPV4EMBV6PFX 在 IPV4EMBV6SW=ENABLE 时为必选 — 后果：缺条件必选命令拒收
- **适用 NF 限定**（warning，隐含）：PGW-U、UPF — 后果：SGW-U 等不适用
- **FDMAXNUM 须 ≥ APN 级**（warning，隐含）：全局 FDMAXNUM 须**大于等于** [0-00231](task/UDG/20.15.2/0-00231.md) SET APNFLOWMAXNUM — 后果：全局小于 APN 时 APN 配置失败

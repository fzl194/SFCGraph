# 设置大流优化参数（SET FLOWLETPARA）

> 命令静态知识见 [SET FLOWLETPARA](command/UDG/20.15.2/SET-FLOWLETPARA.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

本任务是 5G-A 高通量会话特性的**激活核心开关**：SMF 把高通量会话标识下发到 UPF 后，只有 UPF 通过本命令开启大流优化功能开关（FLOWOPTSW=ENABLE），才会把该会话判定为高通量会话并开启微流调度。主场景只需显式设 FLOWOPTSW=ENABLE，其余调参为可选优化项（现网按需调测）。适用 NF：PGW-U、UPF、SGW-U。

**取值样例（数据规划表，主特性 GWFD-112002）**

| 参数 | 含义 | 样例值 | 获取方法 |
|---|---|---|---|
| FLOWOPTSW | 大流优化功能 | ENABLE | 固定取值（本特性激活开关） |
| THREADUPDINT | 线程负载更新间隔 | （现网调测） | 本端规划，整数 64～256 ms |
| FLOWCHKPPS | 大流检测速率 | （现网调测） | 本端规划，整数 50000～5000000 包/秒 |
| FLOWGUARDPKTNUM | 大流优化保护报文数 | （现网调测） | 本端规划，整数 0～65535 包 |
| FLOWGUARDTIME | 大流优化保护时间 | （现网调测） | 本端规划，整数 5～500 ms |

> 4 个调参为可选优化项，特性 md 未列具体值，取值待现网调测确定（默认全 optional）。前置先执行 `SET LICENSESWITCH: LICITEM="LKV3G55GHT01", SWITCH=ENABLE;` 开 License。

**主场景脚本样例**

```
SET FLOWLETPARA: FLOWOPTSW=ENABLE;
```

**未绑参数说明（按 baseline §4.1 冷门不绑）**：CPUOVERLOADTHR（线程 CPU 阈值，整数 50～90%）、CPUBIASDIFF（线程 CPU 偏差阈值，整数 5～50%），均 optional，现网按需调测。

## 约束

- 命令最大记录数 **1**（全局唯一参数集）。
- **修改参数会影响数据转发性能和大流优化效果**，请谨慎修改。
- 命令执行后**立即生效**。

# 配置地址池组与APN映射关系（ADD POOLGRPMAP）

> 命令静态知识见 [ADD POOLGRPMAP](command/UDG/20.15.2/ADD-POOLGRPMAP.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法（配置方法字典）

ADD POOLGRPMAP 是地址池族链路的"映射收口层"：把已建的 [ADD POOLGROUP](task/UDG/20.15.2/0-00042.md) POOLGROUPNAME 暴露为 APN/SMF/位置区组维度的可寻址规则，让外部控制面（SMF/CP）按映射键查到目标地址池组。命令核心配置空间有 4 个维度：**映射规则名**（MAPPINGNAME 必选 + 本端规划唯一）、**地址池组引用**（POOLGROUPNAME 必选 + 必须已建）、**映射键**（APN / SMF / LOCATIONGRPTYPE+LOCATIONGRPNAME 三选一或叠加）、**PLMN 限定**（MCC/MNC 可选，仅位置区组场景）。6 个特性命中覆盖 010104/010105/010107/020403/020406/020421，映射键在 APN/SMF/位置区组间分叉。

### 配置维度 1：映射规则名（参数 MAPPINGNAME，必选）

| 参数 | 范围 | 备注 |
|---|---|---|
| **MAPPINGNAME** | 字符串 1~79 | 本端规划的映射规则名；同一规则名下允许多条记录对应不同键组合（如 mapping1 同时配 APN / SMF / SMF+APN） |

### 配置维度 2：地址池组引用（参数 POOLGROUPNAME，必选）

| 参数 | 范围 | 备注 |
|---|---|---|
| **POOLGROUPNAME** | 字符串 1~79 | 引用已配的 [ADD POOLGROUP](task/UDG/20.15.2/0-00042.md) POOLGROUPNAME；前置依赖：POOLGROUP 必须先建 |

### 配置维度 3：映射键（参数 APN/SMF/LOCATIONGRPTYPE+LOCATIONGRPNAME，可选/必选）

| 键 | 必选性 | 取值 | 适用场景 |
|---|---|---|---|
| **APN**（APN 级映射） | 可选 | 字符串 1~63 | APN 级地址分配（010104/010107/020403/020406/020421 中 4 个特性） |
| **SMF**（SMF 级映射） | 可选 | 字符串 1~255 | SMF 级地址分配（010105/020406）；SMF 必须先 [ADD CPNODEID](task/UDG/20.15.2/0-00XXX.md) 建好 |
| **LOCATIONGRPTYPE**（位置区级映射） | 可选 | LAC / TAC | 基于位置区组的地址分配（020406/020421） |
| **LOCATIONGRPNAME** | 条件必选 | 字符串 | LOCATIONGRPTYPE=LAC/TAC 时必填；引用 [ADD TACGROUP](task/UDG/20.15.2/0-00XXX.md) 或 [ADD LACGROUP](task/UDG/20.15.2/0-00XXX.md) |

> 三键可叠加：同 MAPPINGNAME 可同时配 APN+SMF、APN+LOCATIONGRPNAME 等（010105 SMF+APN 范式）。

### 配置维度 4：PLMN 限定（参数 MCC/MNC，位置区组场景条件可选）

| 参数 | 取值 | 适用场景 |
|---|---|---|
| **MCC** | 整数 | 移动国家码；LOCATIONGRPTYPE=LAC/TAC 时条件可选 |
| **MNC** | 整数 | 移动网络号；LOCATIONGRPTYPE=LAC/TAC 时条件可选 |

**典型脚本（010104 APN 级映射）**：
```
ADD POOLGRPMAP: MAPPINGNAME="mapping1", APN="apn-test", POOLGROUPNAME="poolgroup1";
```

**典型脚本（010105 SMF 级 / SMF+APN 级）**：
```
ADD POOLGRPMAP: MAPPINGNAME="mapping1", SMF="smfnode1", POOLGROUPNAME="poolgroup1";
ADD POOLGRPMAP: MAPPINGNAME="mapping1", APN="apn-test1", SMF="smfnode1", POOLGROUPNAME="poolgroup1";
ADD POOLGRPMAP: MAPPINGNAME="mapping2", SMF="smfnode1", POOLGROUPNAME="poolgroup2";
```

**典型脚本（010107 APN 级，主备各一）**：
```
ADD POOLGRPMAP: MAPPINGNAME="mapping1", APN="test", POOLGROUPNAME="poolgroup1";
ADD POOLGRPMAP: MAPPINGNAME="mapping2", APN="test", POOLGROUPNAME="poolgroup2";
```

**典型脚本（020406 基于位置 LAC）**：
```
ADD POOLGRPMAP: MAPPINGNAME="mapping1", LOCATIONGRPTYPE=LAC, LOCATIONGRPNAME="lac1", POOLGROUPNAME="poolgroup1";
```

**典型脚本（020421 位置 + APN 叠加）**：
```
ADD POOLGRPMAP: MAPPINGNAME="mapping1", LOCATIONGRPTYPE=LAC, LOCATIONGRPNAME="lac1", APN="apn-test1", POOLGROUPNAME="poolgroup1";
```

> 命令 notes：适用 NF：PGW-U、UPF；执行后**只对新激活用户生效**；同 MAPPINGNAME 多条记录为不同键组合；MAPPINGNAME 本端规划唯一。

**步骤位置**：在所属 feature 编排中通常位于"步骤 4：映射收口"——前置依赖 [0-00042](task/UDG/20.15.2/0-00042.md)（POOLGROUPNAME）+ [0-00020](task/UDG/20.15.2/0-00020.md)（ADD APN，配 APN 键时）+ [ADD CPNODEID](task/UDG/20.15.2/0-00XXX.md)（配 SMF 键时）+ [ADD TACGROUP/LACGROUP](task/UDG/20.15.2/0-00XXX.md)（配位置键时）；完成 APN→地址池组的外部地址分配链路。

## 决策点：映射键选择（DP 0-00047）

问：ADD POOLGRPMAP 选哪个映射键（APN/SMF/位置）？

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| APN 级 | APN | 地址分配方式（010104）/ 静态地址冗余（010107）/ IPv4v6 双栈（020403）/ Prefix Delegation（020406）/ 基于位置（020421） |
| SMF 级 | SMF | 基于 SMF 分配（010105）/ Prefix Delegation（020406 子场景） |
| SMF+APN 叠加 | APN + SMF | 基于 SMF+APN 分配（010105 范式） |
| 位置区级 | LOCATIONGRPTYPE + LOCATIONGRPNAME | 基于位置/位置+APN 分配（020406/020421） |
| 位置+APN 叠加 | LOCATIONGRPTYPE + LOCATIONGRPNAME + APN | 基于位置+APN 分配（020421 范式） |

> 6 个特性命中以 APN 级（010104/010107/020403）和位置级（020406/020421）为主；010105 出现 SMF 级和 SMF+APN 叠加。

## 约束

- **POOLGROUPNAME 前置依赖**（critical，rule-0-00447）：POOLGROUPNAME 必须先由 [0-00042](task/UDG/20.15.2/0-00042.md)（ADD POOLGROUP）建好 — 后果：未建引用命令拒收
- **APN 前置依赖（配 APN 键时）**（critical，rule-0-00447）：APN 必须先由 [0-00020](task/UDG/20.15.2/0-00020.md)（ADD APN）建好 — 后果：未建 APN 命令拒收
- **SMF 前置依赖（配 SMF 键时）**（critical，rule-0-00447）：SMF 必须先 ADD CPNODEID 建好 — 后果：未建 SMF 命令拒收
- **位置区组前置依赖（配位置键时）**（critical，rule-0-00447）：LOCATIONGRPNAME 必须先 ADD TACGROUP/LACGROUP 建好 — 后果：未建位置组命令拒收
- **LOCATIONGRPNAME 必填条件**（critical，rule-0-00447）：LOCATIONGRPTYPE=LAC/TAC 时 LOCATIONGRPNAME 必填 — 后果：缺 LOCATIONGRPNAME 命令拒收
- **最大记录数 40000**（warning，rule-0-00447）：>98% 上报 ALM-135602215（≤95% 恢复，阈值可 `MOD CFGTHRESHOLD` 改） — 后果：超规格新增失败
- **只对新激活用户生效**（warning，rule-0-00447）：执行后已激活用户不受影响 — 后果：已激活用户不切换地址池组，需重激活
- **适用 NF 限定**（warning，隐含）：PGW-U、UPF — 后果：SGW-U 等不适用
- **MCC/MNC 仅位置区组场景条件可选**（warning，rule-0-00447）：非位置场景配 MCC/MNC 无意义 — 后果：非位置场景配 MCC/MNC 命令拒收

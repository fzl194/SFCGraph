# 设置网元形态（SET NETYPE）

> 命令静态知识见 [SET NETYPE](command/UDG/20.15.2/SET-NETYPE.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

本任务配置网元形态（NETYPEENUM 位图），用于 Proxy-UPF 漫游功能初始配置时合一形态对接北向网管。NETYPEENUM 按现网部署形态位图组合（SGWU/PGWU/UPF/GGSNU/ProxySGWU/ProxyUPF），跨场景取值由决策点 DP 0-00151 驱动，本特性场景为合一形态。适用 NF：SGW-U、PGW-U、UPF。

**取值样例（数据规划表，主特性 GWFD-112001 初始配置）**

| 参数 | 含义 | 样例值 | 获取方法 |
|---|---|---|---|
| NETYPEENUM | 网元形态 | GGSNU-0&PGWU-1&ProxySGWU-1&SGWU-0&UPF-0&ProxyUPF-1 | 决策驱动（本特性合一形态样例，实际取值见 DP 0-00151） |

**主场景脚本样例（合一形态）**

```
SET NETYPE: NETYPEENUM=GGSNU-0&PGWU-1&ProxySGWU-1&SGWU-0&UPF-0&ProxyUPF-1;
```

## 决策点

**[UDG@20.15.2@DecisionPoint@0-00151](decisionpoint/UDG/20.15.2/0-00151.md) — SET NETYPE 网元形态跨场景取值**（single_choice，required）

| 选项 | NETYPEENUM 取值 | 适用场景 |
|---|---|---|
| 合一形态（默认） | GGSNU-0&PGWU-1&ProxySGWU-1&SGWU-0&UPF-0&ProxyUPF-1 | 本特性 Proxy-UPF 漫游合一形态 |
| 独立 ProxySGWU | ProxySGWU-1 | 独立关口局承接国际漫游语音业务 |
| 独立 PGW-U | PGWU-1 | PGW-U 形态，无 Proxy 漫游 |
| 独立 UPF | UPF-1 | UPF 形态，无 Proxy 漫游 |

## 约束

- 命令最大记录数 **1**；NETYPEENUM 初始值由系统生成为 **NULL**。
- **ProxySGWU 仅限于设备作为独立关口局承接国际漫游语音业务的 ProxySGW 形态**，其他场景不能配置为 ProxySGWU。
- **ProxyUPF 同前限制**（仅独立关口局承接国际漫游语音业务）。
- 命令执行后**立即生效**。

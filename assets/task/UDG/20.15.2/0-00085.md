# 设置物联网能力上报（SET IOTCAPABILITY）

> 命令静态知识见 [SET IOTCAPABILITY](command/UDG/20.15.2/SET-IOTCAPABILITY.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

整机级 NB-IoT 能力开关，单参数 `NBIOT`，使能时取固定值 `ENABLE`。命令全局唯一（仅 1 条记录），系统初始值即 ENABLE，通常无需显式配置；激活 NB-IoT 特性时显式置位确认。

**取值**

| 参数 | 取值 | 语义 |
|---|---|---|
| NBIOT | ENABLE | 使能 NB-IoT 终端标准接入功能 |

命令真相标 optional（因系统初始已 ENABLE），本特性为表达"使能固定语义"故记 required。

**步骤**：配置 NB-IoT 相关功能前须先执行本命令使能接入能力。使能后 UDG 在与 UNC 偶联建立/更新时，于消息中携带 UP Function Features 信元表示支持 NB-IoT，UNC 随后向 UDG 分配 NB-IoT 用户。

> 完整 NB-IoT 功能需用户面 + 控制面共同配置，控制面配置参考 SGW-C、PGW-C 上 NB-IoT 相关章节（本特性仅用户面 UDG 侧开关）。

**脚本**

```
SET IOTCAPABILITY:NBIOT=ENABLE;
```

## 约束

- 全局唯一实例（最大记录数 1），非按 APN/会话维度；立即生效；系统初始记录 NBIOT=ENABLE。

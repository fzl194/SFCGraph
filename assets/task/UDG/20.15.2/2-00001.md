# 基于业务感知的带宽控制（GWFD-110311）

> 特性静态知识见 [GWFD-110311](feature/UDG/20.15.2/GWFD-110311.md)。本页讲配置生成实例化时这个特性**怎么配**——编排哪些业务阶段、9 种场景如何选。

## 配置概览

本特性通过 BWM 对业务流量做层次化速率控制。核心对象链：**BWMSERVICE**（业务识别 TOS/NONTOS）→ **BWMCONTROLLER**（CAR/Shaping + CIR/CBS/PIR/PBS + 三色）→ **BWMUSERGROUP**（用户集合）→ **BWMRULE**（绑定 Service+Controller，分用户级/用户组级/缺省）→ 绑定接入域（APN 或切片）。

9 种场景**共享同一套骨架**（前置 → 业务控制器 → 用户组规则绑定），差异由"控制层级 × 业务识别方式 × 触发维度"三维 DP 驱动：
- **多命令可复用模块**（PCC链/切片/时间段 + 通用骨架业务控制器/用户组规则）→ compound 承载
- **单命令步骤**（分级/三级）→ 直接复用 atom，参数增量记 DP 影响表 + 通用骨架场景差异（见 [构建SOP](task/特性步骤级构建SOP.md) §3.5）

4. **BWM 用户组、规则与绑定**（通用骨架，含场景差异）→ [1-00003](task/UDG/20.15.2/1-00003.md)
5. **单命令步骤**（直接引 atom，按 DP 选，可空）：
   - 分级带宽 → 在 [1-00002](task/UDG/20.15.2/1-00002.md) 后插入 [0-00108](task/UDG/20.15.2/0-00108.md)（BCSRRVLEVELPLY）
   - 三级整机 → 在 [1-00003](task/UDG/20.15.2/1-00003.md) 后追加 [0-00289](task/UDG/20.15.2/0-00289.md)（BWMRULEGLOBAL）

## 决策点

**DP（特性级）：带宽控制场景选择** —— 决定 compound 编排 + 参数增量 + 额外 atom。每 option 影响必全：

| 场景 | 控制层级 | 业务识别 | BWMRULETYPE | 走法 | 需BWMSERVICE | 接入域 | 关键联动 |
|---|---|---|---|---|---|---|---|
| APN/DNN 用户组级 | 用户组 | 不区分 | GROUP_DEFAULT | 纯骨架 | 否 | APN | USERLEVSRVTYPE=NONTOS、USERGLEVSRVTYPE=NONTOS；不配 BWMSERVICE；必配 SET BANDWIDTHMNG |
| PCC 预定义规则(FUP) | 用户 | CATEGORY_PROP | SUBSCRIBER_SPECIFIC | 骨架+[1-00004](task/UDG/20.15.2/1-00004.md) | 是 | APN | CATEPROPNAME==POLICYNAME；须 SET REFRESHSRV |
| 切片 | 用户组 | Protocol/TOS | GROUP_SPECIFIC | 骨架+[1-00005](task/UDG/20.15.2/1-00005.md) | 是 | 切片 | BWMSUBSCRCTRL=PCC_RULE_IGNORE |
| 时间段 | 用户组 | TOS | GROUP_SPECIFIC | 骨架+[1-00006](task/UDG/20.15.2/1-00006.md) | 是 | APN | TIMERANGENAME1/2/3 ↔ 控制器1/2/3 |
| 用户 TOS | 用户 | TOS | SUBSCRIBER_SPECIFIC | 纯骨架 | 是 | APN | USERLEVSRVTYPE=TOS |
| 用户组 TOS | 用户组 | TOS | GROUP_SPECIFIC | 纯骨架 | 是 | APN | USERGLEVSRVTYPE=TOS；SET BANDWIDTHMNG |
| 三级控制 | 用户/组/整机 | NONTOS+TOS | SUBSCRIBER+GROUP+GLOBAL | 骨架+[0-00289](task/UDG/20.15.2/0-00289.md) | 是 | APN | [1-00002](task/UDG/20.15.2/1-00002.md) 配三级控制器(用户<组<整机)；[1-00003](task/UDG/20.15.2/1-00003.md) 用户级+组级 BWMRULE（USERLEVSRVTYPE=NONTOS/USERGLEVSRVTYPE=TOS 双轨）；追加 BWMRULEGLOBAL（整机 BWMSERVICE 必须 TOS 且 TOS 值全局唯一） |
| 分级带宽 | 用户组 | TOS | GROUP_SPECIFIC | 骨架+[0-00108](task/UDG/20.15.2/0-00108.md) | 是 | APN | [1-00002](task/UDG/20.15.2/1-00002.md) 控制器加 SRVLEVELSPEC；[1-00003](task/UDG/20.15.2/1-00003.md) 规则加 SERVICELEVEL + 用户组 USERGLEVRULESW；CIRRATE 和≤100% |
| HTTP 用户限速 | 用户 | Protocol | SUBSCRIBER_SPECIFIC | 纯骨架 | 是 | APN | PROTOCOLNAME=http |

## 约束（特性级 rule）

- **License 前置**：`SET LICENSESWITCH` 开启 `LKV3G5TCSA01` 是所有场景第一步。
- **依赖特性**：[SA-Basic](feature/UDG/20.15.2/GWFD-110101.md)（GWFD-110101，业务识别）+ [PCC 基本功能](feature/UDG/20.15.2/GWFD-020351.md)（GWFD-020351，PCC 场景）。
- **CAR 参数推荐**：CIR=0.5×PIR；CBS≥CIR×1.5×1000/8；PBS≥PIR×1.5×1000/8。
- **SHAPING 限制**：仅用户级（SUBSCRIBER_SPECIFIC）支持；用户组级/整机级只能 CAR。
- **SET REFRESHSRV**：涉及 Rule/L7Filter 变更必须刷新（PCC 场景必刷）。

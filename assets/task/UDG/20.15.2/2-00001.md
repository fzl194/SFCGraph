# 基于业务感知的带宽控制（GWFD-110311）

> 特性静态知识见 [GWFD-110311](feature/UDG/20.15.2/GWFD-110311.md)。本页讲配置生成实例化时这个特性**怎么配**——编排哪些业务阶段、9 种场景如何选。

## 配置概览

本特性通过 BWM 对业务流量做层次化速率控制。核心对象链：

**BWMSERVICE**（业务识别 TOS/NONTOS）→ **BWMCONTROLLER**（CAR/Shaping + CIR/CBS/PIR/PBS + 三色）→ **BWMUSERGROUP**（用户集合）→ **BWMRULE**（绑定 Service+Controller，分用户级/用户组级/缺省）→ 绑定接入域（APN 或切片）。

9 种配置场景**共享同一套骨架**，差异由"控制层级 × 业务识别方式 × 触发维度"三维决策点驱动（见下），**不建 9 套独立流程**。

## 配置流程（compound 编排）

按业务阶段顺序，每阶段一个 compound；单命令步骤直接挂 atom：

1. **前置** → [1-00001](task/UDG/20.15.2/1-00001.md)（License + 接入域）
2. **BWM 业务与控制器**（通用骨架）→ [1-00002](task/UDG/20.15.2/1-00002.md)
3. **场景特化链**（按 DP 选，可空）：
   - PCC 预定义规则 → [1-00004](task/UDG/20.15.2/1-00004.md)
   - 切片 → [1-00005](task/UDG/20.15.2/1-00005.md)
   - 时间段 → [1-00006](task/UDG/20.15.2/1-00006.md)
   - 分级带宽 → [1-00007](task/UDG/20.15.2/1-00007.md)
   - 三级整机 → [1-00008](task/UDG/20.15.2/1-00008.md)
4. **用户组、规则与绑定**（通用骨架收尾）→ [1-00003](task/UDG/20.15.2/1-00003.md)

## 决策点

**DP（特性级）：带宽控制场景选择** —— 决定 compound 编排与参数分支。每个 option 的影响（编排时不能漏）：

| 场景(option) | 控制层级 | 业务识别 | BWMRULETYPE | 特化 compound | 需BWMSERVICE | 接入域 | 关键联动参数 |
|---|---|---|---|---|---|---|---|
| APN/DNN 用户组级 | 用户组 | 不区分 | GROUP_DEFAULT | 无 | 否 | APN | USERGLEVSRVTYPE=NONTOS；不配 BWMSERVICE |
| PCC 预定义规则(FUP) | 用户 | CATEGORY_PROP | SUBSCRIBER_SPECIFIC | [1-00004](task/UDG/20.15.2/1-00004.md) | 是 | APN | CATEPROPNAME==POLICYNAME；须 SET REFRESHSRV |
| 切片 | 用户组 | Protocol/TOS | GROUP_SPECIFIC | [1-00005](task/UDG/20.15.2/1-00005.md) | 是 | 切片 | BWMSUBSCRCTRL=PCC_RULE_IGNORE |
| 时间段 | 用户组 | TOS | GROUP_SPECIFIC | [1-00006](task/UDG/20.15.2/1-00006.md) | 是 | APN | TIMERANGENAME1/2/3 ↔ 控制器1/2/3 |
| 用户 TOS | 用户 | TOS | SUBSCRIBER_SPECIFIC | 无 | 是 | APN | USERLEVSRVTYPE=TOS |
| 用户组 TOS | 用户组 | TOS | GROUP_SPECIFIC | 无 | 是 | APN | USERGLEVSRVTYPE=TOS；SET BANDWIDTHMNG |
| 三级控制 | 用户/组/整机 | NONTOS+TOS | SUBSCRIBER+GROUP+GLOBAL | [1-00008](task/UDG/20.15.2/1-00008.md) | 是 | APN | 叠加 BWMRULEGLOBAL；三级带宽递增 |
| 分级带宽 | 用户组 | TOS | GROUP_SPECIFIC | [1-00007](task/UDG/20.15.2/1-00007.md) | 是 | APN | SRVLEVELSPEC；CIRRATE 和≤100% |
| HTTP 用户限速 | 用户 | Protocol | SUBSCRIBER_SPECIFIC | 无 | 是 | APN | PROTOCOLNAME=http |

**联动维度**：场景决定 `BWMRULETYPE`、`USERLEVSRVTYPE`/`USERGLEVSRVTYPE`、是否配 BWMSERVICE、接入域类型（APN vs 切片）、是否 SET REFRESHSRV、CTRLTYPE 可选范围（SHAPING 仅用户级）。

## 约束（特性级 rule）

- **License 前置**：`SET LICENSESWITCH` 开启 `LKV3G5TCSA01` 是所有场景第一步。
- **依赖特性**：[SA-Basic](feature/UDG/20.15.2/GWFD-110101.md)（GWFD-110101，业务识别）+ [PCC 基本功能](feature/UDG/20.15.2/GWFD-020351.md)（GWFD-020351，PCC 场景）。
- **CAR 参数推荐**：CIR=0.5×PIR；CBS≥CIR×1.5×1000/8；PBS≥PIR×1.5×1000/8。
- **SHAPING 限制**：仅用户级（SUBSCRIBER_SPECIFIC）支持；用户组级/整机级只能 CAR。
- **SET REFRESHSRV**：涉及 Rule/L7Filter 变更必须刷新（PCC 场景必刷）。

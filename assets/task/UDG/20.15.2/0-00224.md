# 创建TWAMP VPN实例（ADD TWAMPVPNINST）

> 命令静态知识见 [ADD TWAMPVPNINST](command/UDG/20.15.2/ADD-TWAMPVPNINST.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

本任务是 GWFD-110921（TWAMP）探测链路搭建的第 3 步前置，创建一个 TWAMP 探测专用的 VPN 实例，**Full 模式与 Light 模式共用**同一条命令、同一取值样例。后续 ADD TWAMPLOGICINF / ADD TWAMPCLIENT 都用该实例名。

**参数取值规划（数据规划表样例）**

| 参数 | 含义 | 取值样例 | 获取方法 |
|---|---|---|---|
| VRFNAME | VPN 实例名称 | `VPN_CN` | 全网规划，1~31 字符；须与外联口 / 逻辑接口 / 客户端的 VPN 实例一致 |

**脚本样例（Full / Light 共用）**

```
ADD TWAMPVPNINST: VRFNAME="VPN_CN";
```

**操作步骤定位**：该命令在 TWAMP 激活流程中紧接 License 开关（[SET LICENSESWITCH](command/UDG/20.15.2/SET-LICENSESWITCH.md)）之后、ADD TWAMPLOGICINF 之前——它是探测逻辑接口 VRFNAME 的引用源。

# 配置IPFarm全局参数（SET IPFARMGLOBAL）

> 命令静态知识见 [SET IPFARMGLOBAL](command/UDG/20.15.2/SET-IPFARMGLOBAL.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

整机级 IPFarm 心跳检测 + 负载均衡全局参数，6 个参数全部本端规划。`SERVERTYPE` 是类型主键（必选），其余参数由类型决定是否配置。命令按 SERVERTYPE 取值各自独立，全局最多 3 套（REDIRECT/LOADSHARE/PCSCF 各 1 套），同一类型重复配置覆盖前值。

**两套范式（按 SERVERTYPE 分叉）**

| SERVERTYPE | 场景 | 必配 | 可选/省略 |
|---|---|---|---|
| REDIRECT | 用户Portal(110281) / Web Proxy(110282) 重定向 | TIMETHRESHOLD, HEALTHSUCCLIMIT, HEALTHFAILLIMIT, **LBMETHOD** | TIMEOUT |
| PCSCF | P-CSCF 故障 IMS 恢复(020253) | TIMETHRESHOLD, HEALTHSUCCLIMIT, HEALTHFAILLIMIT | TIMEOUT, **LBMETHOD 不配** |

差异点：REDIRECT 场景需配 LBMETHOD（负载均衡模式），PCSCF 场景无需负载均衡。

**取值样例（来自各特性数据规划表）**

| 参数 | REDIRECT 取值 | PCSCF 取值 | 范围 |
|---|---|---|---|
| SERVERTYPE | REDIRECT | PCSCF | 枚举 |
| TIMETHRESHOLD（秒） | 10（激活）/ 6（修改） | 6 | 6~60 |
| HEALTHSUCCLIMIT | 2 / 3 | 4 | 1~10 |
| HEALTHFAILLIMIT | 4 / 2 | 4 | 1~10 |
| LBMETHOD | LEAST_LOAD / LEAST_RECENTLY_USED / ROUND_ROBIN | （不配） | 枚举 |
| TIMEOUT（秒） | （未显式配，按需） | （未显式配，按需） | 1~59 |

**脚本**

REDIRECT 场景（用户Portal/Web Proxy）：
```
SET IPFARMGLOBAL:SERVERTYPE=REDIRECT,TIMETHRESHOLD=10,HEALTHSUCCLIMIT=2,HEALTHFAILLIMIT=4,LBMETHOD=LEAST_LOAD;
```

PCSCF 场景（P-CSCF 故障 IMS 恢复，运营商无特殊要求建议保持默认）：
```
SET IPFARMGLOBAL:SERVERTYPE=PCSCF,HEALTHSUCCLIMIT=4,HEALTHFAILLIMIT=4,TIMETHRESHOLD=6;
```

## 决策点

**DP 0-00150：SERVERTYPE 跨特性取值**（single_choice，required）

| 选项 | 值 | 影响 |
|---|---|---|
| REDIRECT 重定向场景 | REDIRECT | 需配 LBMETHOD + TIMETHRESHOLD |
| PCSCF IMS 故障恢复场景 | PCSCF | 不配 LBMETHOD；必配 TIMETHRESHOLD（默认 6 秒） |

default=REDIRECT（主场景）。三个特性共享本 DP：110281/110282 选 REDIRECT，020253 选 PCSCF。

## 约束

- 全局最多 3 套（每 SERVERTYPE 取值各 1 套：REDIRECT/LOADSHARE/PCSCF）；立即生效；同类型重复配置覆盖前值。

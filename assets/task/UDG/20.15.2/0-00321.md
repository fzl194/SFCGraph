# 设置基于位置区分配地址的全局开关（SET IPALLOCBYLOCGLBSW）

> 命令静态知识见 [SET IPALLOCBYLOCGLBSW](command/UDG/20.15.2/SET-IPALLOCBYLOCGLBSW.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典",**不**逐特性罗列取值。该命令**仅 2 个可选参数 SWITCH/IPV6SWITCH**,配置空间极简;**适用 NF 限定 PGW-U/UPF**;整机唯一实例(max_records=1);系统初始记录默认两套开关均为 DISABLE。

## 配置方法(配置方法字典)

SET IPALLOCBYLOCGLBSW 是"基于位置区分配地址的全局开关"命令,适用于 PGW-U/UPF。作用:配置基于位置区分配地址的 IPv4/IPv6 总开关,作为位置区组级开关 IPALLOCBYLOCSW 的兜底(组级 SWITCH=INHERIT 时继承本全局值)。核心配置空间极简,**仅 1 个维度**:IPv4/IPv6 总开关(SWITCH/IPV6SWITCH)。2 个特性命中(GWFD-020406 IPv6 Prefix Delegation + GWFD-020421 基于位置的地址分配)共用同一范式(`SWITCH=ENABLE`)。

### 配置维度 1:IPv4/IPv6 总开关(参数 SWITCH/IPV6SWITCH,可选)

两套开关独立,各控制对应 IP 协议族的"基于位置区分配地址"全局使能:

| 取值 | 作用 | 典型场景 |
|---|---|---|
| **DISABLE**(默认) | 不使能基于位置区分配地址全局开关 | 默认出厂;非位置分配地址场景 |
| **ENABLE** | 使能基于位置区分配地址全局开关 | GWFD-020406 IPv6 Prefix Delegation + GWFD-020421 基于位置的地址分配(均配 `SWITCH=ENABLE`);位置分配地址功能启用前必配 |

> **两参数独立性**:SWITCH 与 IPV6SWITCH 独立设置,可同时 ENABLE/同时 DISABLE/任一组合。系统初始记录默认值两者均为 DISABLE。

> **注意事项**(命令 notes 投影):
> - 该命令执行后**立即生效**。
> - 该命令最大记录数为 **1**(整机全局唯一实例,SET 即覆盖,无 ADD/RMV 变体)。
> - 系统初始记录两套开关默认值均为 DISABLE。

**典型脚本(GWFD-020406 + GWFD-020421 共用范式)**:
```
SET IPALLOCBYLOCGLBSW: SWITCH=ENABLE;
```

**典型脚本(IPv4+IPv6 双栈全开)**:
```
SET IPALLOCBYLOCGLBSW: SWITCH=ENABLE, IPV6SWITCH=ENABLE;
```

**步骤位置**:在所属 feature 编排中位于"位置分配地址前置总开关"——前置 无;下游 触发位置区组级开关配置([0-00322](task/UDG/20.15.2/0-00322.md) SET IPALLOCBYLOCSW,组级 SWITCH=INHERIT 时继承本全局值)或白名单前置([0-00299](task/UDG/20.15.2/0-00299.md) ADD ADRLOCWHITELST)。

## 决策点

(无。本命令在所有特性下用法一致——仅 SWITCH/IPV6SWITCH 两开关 ENABLE/DISABLE 二选一,无特性分叉。)

## 约束

- **适用 NF 限定 PGW-U/UPF**(critical,rule-0-00321):本命令仅适用于 PGW-U/UPF,控制面 SMF 等网元不适用 — 后果:非 PGW-U/UPF 网元命令拒收
- **最大记录数 1**(critical,rule-0-00321):整机全局唯一实例 — 后果:重复 SET 覆盖上一条;无 ADD/RMV 变体
- **执行后立即生效**(warning,rule-0-00321):命令执行后立即生效 — 后果:误操作即刻生效
- **两参数独立配置**(warning,rule-0-00321):SWITCH 与 IPV6SWITCH 独立设置,无联动关系 — 后果:IPv4/IPv6 协议族使能各自独立,需按场景分别配
- **默认双 DISABLE**(info,rule-0-00321):系统初始记录两套开关默认值均为 DISABLE — 后果:出厂未配时位置分配地址全局关闭,启用需显式 SET ENABLE
- **组级 INHERIT 继承本全局值**(critical,rule-0-00321):位置区组级 IPALLOCBYLOCSW 的 SWITCH=INHERIT 时,按本全局开关配置进行地址分配 — 后果:全局 DISABLE + 组级 INHERIT 时位置分配地址关闭
- **命令无 ADD/RMV 变体**(info,rule-0-00321):本命令仅 SET/LST,无 ADD/RMV 变体(整机唯一,SET 即覆盖) — 后果:无显式删除命令,需 SET DISABLE 关闭

# 修改用户模板（MOD USERPROFILE）

> 命令静态知识见 [MOD USERPROFILE](command/UDG/20.15.2/MOD-USERPROFILE.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

MOD USERPROFILE 在 ADD USERPROFILE（[0-00011](task/UDG/20.15.2/0-00011.md)）创建用户模板后，调整模板级开关。承载多个独立属性，各特性按需设其一。

**关键参数的配置范式**

| 参数 | 取值 | 用途特性 | 典型行 |
|---|---|---|---|
| **ALIASMARKFLAG** | DISABLE/ENABLE | GWFD-020359 IM 无线资源管控：**DSCP 标记方式需 DISABLE**（DISABLE 时系统修改报文头 DSCP/ToS 值转发；ENABLE 时不修改原 DSCP/ToS）| `MOD USERPROFILE:USERPROFILENAME="up_test",ALIASMARKFLAG=DISABLE;` |
| DDOSCHECK | ENABLE | 防范攻击（用户模板级防 DDoS）| `MOD USERPROFILE:USERPROFILENAME="up_test",DDOSCHECK=ENABLE;` |
| CAPMODETHRES | 0-60 分钟 | Captive 模式定时器（重定向族用户Portal 等用，0=只重定向一次）| `MOD USERPROFILE:USERPROFILENAME="up_test",CAPMODETHRES=6;` |
| TETHERINGMAXNUM | 0-20/255 | Tethering 后台终端数上限（255=不开启）| `MOD USERPROFILE:USERPROFILENAME="up_test",TETHERINGMAXNUM=3;` |
| V4TCPMSSVALUE/V6TCPMSSVALUE | 496-1500 | 用户模板级 TCP MSS（优先级高于 APN 级）| `MOD USERPROFILE:USERPROFILENAME="up_test",V4TCPMSSVALUE=1200;` |

**脚本样例（GWFD-020359 DSCP 方式：禁用 Alias Marking）**
```
MOD USERPROFILE:USERPROFILENAME="up_test",ALIASMARKFLAG=DISABLE;
```

**配置链路**：ADD USERPROFILE（[0-00011](task/UDG/20.15.2/0-00011.md)）→ **MOD USERPROFILE**（调整模板级开关，如 020359 DSCP 方式在 ADD RULE(REMARK_FPI) 后禁用 Alias Marking）。

## 决策点

| 选项 | 影响 |
|---|---|
| ALIASMARKFLAG=DISABLE（020359 DSCP 方式）| 系统修改报文头 DSCP/ToS 按指定值转发（DSCP 标记生效）|
| ALIASMARKFLAG=ENABLE | 系统不修改原 DSCP/ToS（Alias Marking 生效）|

## 约束

- **立即生效**（info）：命令执行后立即生效（无需 SET REFRESHSRV）。
- **不支持实时位置上报参数**（info）：当前版本不支持 LOCRPTSWITCH 参数。
- **USERPROFILENAME 须已存在**（critical）：引用须为 ADD USERPROFILE 已创建的模板 — 不存在则修改失败。

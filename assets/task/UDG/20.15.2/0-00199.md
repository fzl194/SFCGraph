# 增加流行为(MQC)（ADD MQCBEHAVIOR）

> 命令静态知识见 [ADD MQCBEHAVIOR](command/UDG/20.15.2/ADD-MQCBEHAVIOR.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

MQCBEHAVIOR 是 CB 对的 **B 侧**——一个**空行为容器**，本命令只命名，流动作(permit/redirect/remark/deny/cascade/URPF/CAR)全部由后续子命令挂载。

**数据规划取值样例**（证据③，IPFD-012001 数据规划表）：

| 参数 | 取值样例 | 获取方法 |
|---|---|---|
| BEHAVIORNAME（行为名称） | `behavior1` | 本端规划 |

**典型范式**（先建空容器，再挂流动作）：
```
ADD MQCBEHAVIOR: BEHAVIORNAME="behavior1";
ADD QOSACTRDRNHP: BEHAVIORNAME="behavior1", IPVERSION=IPv4, NEXTHOP="192.168.2.3";  # 挂重定向动作 → 0-00200
```

> 流动作挂载子命令归口：重定向→[ADD QOSACTRDRNHP](0-00200.md)；permit/deny→SET QOSACTFILTER；重标记→ADD SQOSREMARK；级联→ADD QOSACTRDRPOLICY；URPF→ADD SQOSURPF；CAR→ADD SQOSCAR；重定向VPN组→ADD SQOSRDRVPNGROUP。

## 约束

- **rule-0-00357**（warning）最大记录数 **8192**；执行后**立即生效**；为空行为容器，流动作须由后续子命令挂载；流动作互斥（permit 可与其他共存，deny/重定向/重标记/级联/URPF/CAR 互斥）；B 侧须在 CB 节点绑定（0-00203）前先建。

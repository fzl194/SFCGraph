# 设置基于SMF分配地址的全局开关（SET IPALLOCBYSMFGLBSW）

> 命令静态知识见 [SET IPALLOCBYSMFGLBSW](command/UDG/20.15.2/SET-IPALLOCBYSMFGLBSW.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典",**不**逐特性罗列取值。该命令**仅 2 个可选参数 SWITCH/IPV6SWITCH**,配置空间极简;**适用 NF 限定 PGW-U/UPF**;整机唯一实例(max_records=1);系统初始记录默认两套开关均为 DISABLE。SMF 实例必须先经 [0-00303](task/UDG/20.15.2/0-00303.md) ADD CPNODEID 预建(决定后续 SET IPALLOCBYSMFSW 的 SMF 名可被引用)。

## 配置方法(配置方法字典)

SET IPALLOCBYSMFGLBSW 是"基于SMF分配地址的全局开关"命令,适用于 PGW-U/UPF。作用:配置基于SMF分配地址的 IPv4/IPv6 总开关,作为 SMF 级开关 IPALLOCBYSMFSW 的兜底(SMF 级 SWITCH=INHERIT 时继承本全局值)。核心配置空间极简,**仅 1 个维度**:IPv4/IPv6 总开关(SWITCH/IPV6SWITCH)。2 个特性命中(GWFD-010105 用户面地址分配 + GWFD-020406 IPv6 Prefix Delegation)共用同一范式(`SWITCH=ENABLE`)。

### 配置维度 1:IPv4/IPv6 总开关(参数 SWITCH/IPV6SWITCH,可选)

两套开关独立,各控制对应 IP 协议族的"基于SMF分配地址"全局使能:

| 取值 | 作用 | 典型场景 |
|---|---|---|
| **DISABLE**(默认) | 不使能基于SMF分配地址全局开关 | 默认出厂;非SMF分配地址场景 |
| **ENABLE** | 使能基于SMF分配地址全局开关 | GWFD-010105 用户面地址分配 + GWFD-020406 IPv6 Prefix Delegation(均配 `SWITCH=ENABLE`);SMF分配地址功能启用前必配 |

> **两参数独立性**:SWITCH 与 IPV6SWITCH 独立设置,可同时 ENABLE/同时 DISABLE/任一组合。系统初始记录默认值两者均为 DISABLE。

> **注意事项**(命令 notes 投影):
> - 该命令执行后**立即生效**。
> - 该命令最大记录数为 **1**(整机全局唯一实例,SET 即覆盖,无 ADD/RMV 变体)。
> - 系统初始记录两套开关默认值均为 DISABLE。

**典型脚本(GWFD-010105 + GWFD-020406 共用范式)**:
```
SET IPALLOCBYSMFGLBSW: SWITCH=ENABLE;
```

**典型脚本(IPv4+IPv6 双栈全开)**:
```
SET IPALLOCBYSMFGLBSW: SWITCH=ENABLE, IPV6SWITCH=ENABLE;
```

**步骤位置**:在所属 feature 编排中位于"SMF分配地址前置总开关"——前置 [0-00303](task/UDG/20.15.2/0-00303.md) ADD CPNODEID(SMF 实例标识预建,SMF 名作为下游 IPALLOCBYSMFSW 的 SMF 键);下游 触发 SMF 级开关配置([0-00324](task/UDG/20.15.2/0-00324.md) SET IPALLOCBYSMFSW,SMF 级 SWITCH=INHERIT 时继承本全局值)。

## 决策点

(无。本命令在所有特性下用法一致——仅 SWITCH/IPV6SWITCH 两开关 ENABLE/DISABLE 二选一,无特性分叉。)

## 约束

- **适用 NF 限定 PGW-U/UPF**(critical,rule-0-00323):本命令仅适用于 PGW-U/UPF,控制面 SMF 等网元不适用 — 后果:非 PGW-U/UPF 网元命令拒收
- **最大记录数 1**(critical,rule-0-00323):整机全局唯一实例 — 后果:重复 SET 覆盖上一条;无 ADD/RMV 变体
- **执行后立即生效**(warning,rule-0-00323):命令执行后立即生效 — 后果:误操作即刻生效
- **两参数独立配置**(warning,rule-0-00323):SWITCH 与 IPV6SWITCH 独立设置,无联动关系 — 后果:IPv4/IPv6 协议族使能各自独立,需按场景分别配
- **默认双 DISABLE**(info,rule-0-00323):系统初始记录两套开关默认值均为 DISABLE — 后果:出厂未配时SMF分配地址全局关闭,启用需显式 SET ENABLE
- **SMF 级 INHERIT 继承本全局值**(critical,rule-0-00323):SMF 级 IPALLOCBYSMFSW 的 SWITCH=INHERIT 时,按本全局开关配置进行地址分配 — 后果:全局 DISABLE + SMF 级 INHERIT 时SMF分配地址关闭
- **命令无 ADD/RMV 变体**(info,rule-0-00323):本命令仅 SET/LST,无 ADD/RMV 变体(整机唯一,SET 即覆盖) — 后果:无显式删除命令,需 SET DISABLE 关闭

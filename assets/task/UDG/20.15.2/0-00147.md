# 使能全局BGP并配置VRF实例与路由引入(路由交叉)（SET BGP）

> 命令静态知识见 [SET BGP](command/UDG/20.15.2/SET-BGP.md)。本页只讲配置生成实例化时这条命令**怎么配**。

> **聚合说明**：本 atom 合并 3 命令（SET BGP 全局 + ADD BGPVRF 实例 + ADD IMPORTROUTE 引入 wlr），按 SKILL §4.2 例外合并避免 BGP 族碎片化。ref 主指向 SET BGP；演进建议拆分为 0-00147/0-00148/0-00149 三 atom（CONVERGENCE-BACKLOG）。

## 配置方法

本 atom 是 **GWFD-020423 MPLS OptionB BGP 基础**：先全局使能 BGP 并配 AS 号，再为 VRF_networkA 增加 BGP VPN 实例，最后引入 wlr 本地路由。若 UDG 已使能 BGP 则全局 SET BGP 不重复配。高危命令，立即生效。

**配置范式（三段，全局→VRF→引入）**

| 阶段 | 参数 | 取值 | 来源 | 说明 |
|---|---|---|---|---|
| SET BGP 全局 | BGPENABLE | `TRUE` | 固定取值（必配） | 使能 BGP，新创建必须配 |
| SET BGP 全局 | ASNUM | `60001` | 全网规划（5G Core LLD AS号 VNF） | 自治域号，必配 |
| ADD BGPVRF 实例 | VRFNAME | `VRF_networkA` | 引用 [0-00071](task/UDG/20.15.2/0-00071.md).VRFNAME | VPN 实例名 |
| ADD BGPVRF 实例 | DEFAULTAFTYPE | `ipv4uni` | 本端规划（固定） | 默认地址族 |
| ADD IMPORTROUTE 引入 | IMPORTPROTOCOL | `wlr` | 本端规划（固定） | 引入本地路由协议 |

**任务示例**

```
SET BGP: BGPENABLE=TRUE, ASNUM="60001";
ADD BGPVRF: VRFNAME="VRF_networkA", DEFAULTAFTYPE=ipv4uni;
ADD IMPORTROUTE: VRFNAME="VRF_networkA", IMPORTPROTOCOL=wlr;
```

11 个 optional（GRACEFULRESTART/TIMEWAITFORRIB/ASPATHLIMIT/CONFED*/CHECKFIRSTAS/BGPRIDAUTOSEL/KEEPALLROUTES/MEMORYLIMIT/GRPEERRESET/ISSHUTDOWN）路由交叉场景均用系统默认，不绑。

## 约束

- **rule-0-00234**（error）高危命令；去使能 BGP（BGPENABLE=FALSE）会删除所有 BGP 历史配置（级联影响 ADD BGPVRF/ADD IMPORTROUTE）；GRACEFULRESTART、BGPRIDAUTOSEL 变更触发 BGP 邻居复位（运维窗口慎用）；新创建 BGP 必须至少配 ASNUM + BGPENABLE(TRUE)；存在系统初始记录（BGPENABLE=TRUE/GRACEFULRESTART=FALSE/TIMEWAITFORRIB=300s/BGPRIDAUTOSEL=FALSE）；立即生效

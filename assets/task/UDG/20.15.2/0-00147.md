# 使能全局BGP并配置VRF实例与路由引入(路由交叉)（SET BGP）

> 命令静态知识见 [SET BGP](command/UDG/20.15.2/SET-BGP.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

> **聚合说明**：本 atom 合并 3 命令（SET BGP 全局 + ADD BGPVRF 实例 + ADD IMPORTROUTE 引入 wlr），ref 主指向 SET BGP；演进建议拆分为 0-00147a/0-00147b/0-00147c 三 atom（CONVERGENCE-BACKLOG）。

## 配置方法

本 atom 是 **GWFD-020423 MPLS OptionB BGP 基础**：先全局使能 BGP 并配 AS 号，再为 VRF_networkA 增加 BGP VPN 实例，最后引入 wlr 本地路由。**若 UDG 已使能 BGP 则全局 SET BGP 不重复配**。高危命令，立即生效。1 个特性命中（020423 路由交叉）。

### 配置维度 1：全局使能（参数 BGPENABLE + ASNUM）

| 参数 | 取值 | 作用 |
|---|---|---|
| **BGPENABLE** | `TRUE`（新创建必须配） | 全局使能 BGP |
| **ASNUM** | 全网规划 AS 号（5G Core LLD AS 号 VNF） | 自治域号，必配 |

> 系统初始记录 BGPENABLE=TRUE；新创建 BGP 必须至少配 ASNUM + BGPENABLE(TRUE)。

### 配置维度 2：BGP VRF 实例（参数 VRFNAME + DEFAULTAFTYPE）

| 参数 | 取值 | 作用 |
|---|---|---|
| **VRFNAME** | `VRF_networkA`（引用 0-00071.VRFNAME） | VPN 实例名 |
| **DEFAULTAFTYPE** | `ipv4uni`（固定） | 默认地址族 |

> ADD BGPVRF 为指定 VRF 建立 BGP 路由上下文；当前路由交叉用 IPv4 单播。

### 配置维度 3：路由引入（参数 IMPORTPROTOCOL）

| 取值 | 作用 | 适用场景 |
|---|---|---|
| **wlr**（固定） | 引入本地 wlr 路由表 | 020423 路由交叉范式 |
| 其他 | 见命令真相表枚举（020423 未用） | - |

**任务示例（020423 范式三段：全局→VRF→引入）**：

```
SET BGP: BGPENABLE=TRUE, ASNUM="60001";
ADD BGPVRF: VRFNAME="VRF_networkA", DEFAULTAFTYPE=ipv4uni;
ADD IMPORTROUTE: VRFNAME="VRF_networkA", IMPORTPROTOCOL=wlr;
```

**步骤位置**：在所属 feature 编排中通常位于"步骤 N：BGP 基础三段配置"——前置 [0-00071](task/UDG/20.15.2/0-00071.md) ADD L3VPNINST 配 VRFNAME + [0-00145](task/UDG/20.15.2/0-00145.md) MOD VPNINSTAF 配 RD + [0-00146](task/UDG/20.15.2/0-00146.md) ADD VPNTARGET 配 RT。

## 约束

- **高危命令**（critical，rule-0-00147）：去使能 BGP（BGPENABLE=FALSE）会删除所有 BGP 历史配置（级联影响 ADD BGPVRF/ADD IMPORTROUTE）— 后果：去使能即清空 BGP 全量数据
- **GRACEFULRESTART/BGPRIDAUTOSEL 变更触发邻居复位**（warning，rule-0-00147）：变更 GRACEFULRESTART 或 BGPRIDAUTOSEL 参数触发 BGP 邻居复位 — 后果：运维窗口慎用，避免业务中断
- **新创建 BGP 必须至少配 ASNUM + BGPENABLE(TRUE)**（critical，rule-0-00147）：缺一拒收 — 后果：命令拒收
- **系统初始记录**（info，rule-0-00147）：BGPENABLE=TRUE/GRACEFULRESTART=FALSE/TIMEWAITFORRIB=300s/BGPRIDAUTOSEL=FALSE
- **立即生效**（warning，rule-0-00147）：执行后立即生效 — 后果：误配 AS 号即刻影响 BGP 邻居
- **11 个 optional 未绑**（info，rule-0-00147）：GRACEFULRESTART/TIMEWAITFORRIB/ASPATHLIMIT/CONFED*/CHECKFIRSTAS/BGPRIDAUTOSEL/KEEPALLROUTES/MEMORYLIMIT/GRPEERRESET/ISSHUTDOWN 在 020423 路由交叉场景均用系统默认

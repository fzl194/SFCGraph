# 创建OSPFv3进程(IPv6)（ADD OSPFV3）

> 命令静态知识见 [ADD OSPFV3](command/UDG/20.15.2/ADD-OSPFV3.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

ADD OSPFV3 是 IPv6 OSPFv3 动态路由组网的进程入口命令。4 个特性数据规划表（GWFD-010155/020401/020403/020406）仅用 6 个参数，其余 24 个进阶参数（BFD 间隔/SPF 计算/Stub 路由/Domain/SA 等）均未揭示，按通用配置筛不绑。PROCID/ROUTERID/VRFNAME/BFDALLINTFFLG/VPNINSCAPSIMFLG/VIRTUALSYSFLAG 六参即覆盖当前全部特性。

**两种范式（从证据数据规划表归纳）**

| 组网 | PROCID | ROUTERID | VRFNAME | BFDALLINTFFLG | VPNINSCAPSIMFLG | VIRTUALSYSFLAG | 脚本样例 |
|---|---|---|---|---|---|---|---|
| PGW-U 到 ePDG IPv6 动态路由（GWFD-010155） | 6（全网规划） | 10.8.20.1 | vpn-paif | TRUE | TRUE | TRUE | `ADD OSPFV3:PROCID=6,VRFNAME="vpn-paif",ROUTERID="10.8.20.1",BFDALLINTFFLG=TRUE,VPNINSCAPSIMFLG=TRUE,VIRTUALSYSFLAG=TRUE;` |
| IPv6 承载上下文（GWFD-020401） | 100（本端规划） | 10.10.10.1 | VPN_Internet | 不配 | 不配 | 不配 | `ADD OSPFV3:PROCID=100,ROUTERID="10.10.10.1",VRFNAME="VPN_Internet";` |
| IPv6 Prefix Delegation（GWFD-020406 系列） | 6（已配置数据中获取） | 10.8.25.1 | vpn1 | TRUE | TRUE | TRUE | `ADD OSPFV3:PROCID=6,VRFNAME="vpn1",ROUTERID="10.8.25.1",BFDALLINTFFLG=TRUE,VPNINSCAPSIMFLG=TRUE,VIRTUALSYSFLAG=TRUE;` |

**参数取值规则**
- **PROCID**：进程号，跨特性获取方法分叉（全网/本端/已配置数据中获取）但取值同构，保 global_planned；与 IPv4 ADD OSPF（0-00074）平行，由 DP 0-00017 ip-version 选择。
- **ROUTERID**：IPv4 点分十进制格式，**全网唯一**，必配否则进程无法生成 LSA。
- **VRFNAME**：reference 引用 [创建L3VPN实例](task/UDG/20.15.2/0-00071.md)，VPN 多实例时必填；可用 LST VPNINST 查询已配实例。
- **BFDALLINTFFLG=TRUE**：使能 BFD（GWFD-010155/020406 用，GWFD-020401 不配）。
- **VPNINSCAPSIMFLG=TRUE**：VPN 多实例场景**必配**去使能 VPN 路由环路检测，否则 OSPF 路由引入失败（GWFD-020406 md 明示双重强调）。
- **VIRTUALSYSFLAG=TRUE**：共网段虚拟系统使能（进程级），decision_driven→DP 0-00022 协调；须与 ADD OSPFV3INTERFACE 接口级 VIRTUALSYSFLAG 同时配。

## 决策点

- **DP 0-00022**（VIRTUALSYSFLAG 共网段虚拟系统使能）：共网段场景进程级 + 接口级须同时配 VIRTUALSYSFLAG=TRUE，由本 DP 协调取值。

## 约束

- **rule 0-00200**（constraint/critical）OSPFv3 进程实例数规格默认整机（命令最大记录数 8016）；一个接口只能属于某一个 OSPFv3 进程，进程实例（VPN/全局归属）不可更改，只能在第一次使能该进程时指定；ROUTERID 必配且全网唯一（IPv4 点分十进制格式）否则无法生成 LSA；VPN 多实例场景须配 VPNINSCAPSIMFLG=TRUE 去使能环路检测；共网段须配 VIRTUALSYSFLAG=TRUE（进程级 + 接口级须同时配）；接口上需先使能 IPv6 — 后果：路由引入失败 / Router ID 冲突 / 进程实例超规格 / 接口归属冲突 / LSA 无法生成

# 修改GRE隧道增强配置（MOD GRETUNNEL）

> 命令静态知识见 [MOD GRETUNNEL](command/UDG/20.15.2/MOD-GRETUNNEL.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

本命令承载 3 类**互斥可选增强**（IPFD-015002 GRE 特性按需各拆一条 MOD）。每条 MOD 只配一组增强参数，TNLNAME 引用 [ADD GRETUNNEL](task/UDG/20.15.2/0-00059.md) 已建隧道，TNLTYPE 固定 gre。源/目的重配参数（SRCTYPE/SRCIPADDR/SRCIFNAME/DSTIPADDR 等）在本 atom 不绑（按通用配置门槛）。

**配置范式（3 类增强，IPFD-015002 示例）**

| 增强项 | 必配参数 | 示例行 |
|---|---|---|
| 端到端校验 | TNLNAME, TNLTYPE=gre, CHECKSUMEN=TRUE | `MOD GRETUNNEL:TNLNAME="Tunnel1", TNLTYPE=gre, CHECKSUMEN=TRUE;` |
| 识别关键字 | TNLNAME, TNLTYPE=gre, GREKEYEN=TRUE, GREKEY="123" | `MOD GRETUNNEL:TNLNAME="Tunnel1", TNLTYPE=gre, GREKEYEN=TRUE, GREKEY="123";` |
| Keepalive | TNLNAME, TNLTYPE=gre, KEEPALVEN=TRUE, KEEPALVPERIOD=5, KEEPALVRETRYCNT=3 | `MOD GRETUNNEL:TNLNAME="Tunnel1", TNLTYPE=gre, KEEPALVEN=TRUE, KEEPALVPERIOD=5, KEEPALVRETRYCNT=3;` |

**参数取值约定**

| 参数 | 取值 | 来源 | 说明 |
|---|---|---|---|
| TNLNAME | 字符串 1~63 | 引用 [ADD GRETUNNEL](task/UDG/20.15.2/0-00059.md) 已建隧道 | 隧道名（核心键） |
| TNLTYPE | gre（固定） | 本端规划 | 固定值 |
| CHECKSUMEN | TRUE / FALSE | 本端规划 | 端到端校验开关 |
| GREKEYEN | TRUE / FALSE | 本端规划 | 识别关键字开关 |
| GREKEY | 密码 1~268 | 与对端协商 | **两端必须一致**（rule 0-00074） |
| KEEPALVEN | TRUE / FALSE | 本端规划 | Keepalive 开关 |
| KEEPALVPERIOD | 整数 1~32767 秒（示例=5） | 本端规划 | 报文发送周期 |
| KEEPALVRETRYCNT | 整数 1~255（示例=3） | 本端规划 | 不可达计数器 |

**未绑冷门参数**（命令真相有但特性脚本未用）：SRCTYPE/SRCTYPE6/SRCIPADDR/SRCIPV6ADDR/SRCIFNAME/DSTIPADDR/DSTIPV6ADDR/DSTVPNNAME（源/目的重配）/STATENABLE（报文统计）/REDUNDANCYEN（备份隧道）——按需时查命令 wiki。

## 约束

- **rule-0-00074**（warning）GREKEYEN=TRUE 时隧道两端 GREKEY 必须相同；或两端都不设置识别关键字 — 后果：关键字不匹配→报文丢弃/隧道不通

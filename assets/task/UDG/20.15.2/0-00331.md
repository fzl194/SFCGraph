# 查询VPN实例（LST VPNINST）

> 命令静态知识见 [LST VPNINST](command/UDG/20.15.2/LST-VPNINST.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法（配置方法字典）

LST VPNINST 是 VPN 族**配置查询**命令，输出 VPNINSTANCE / AFTYPE / VRFRD 等配置字段。GWFD-010105 场景下用于核查"VPN 三者一致"约束（地址池 / APN / Gi/SGi/N6 外联口）。命令核心配置空间有 2 个维度：**查询范围**（VPNINSTANCE 可选：指定单个/查全部）、**详细度**（VERBOSE 可选）。

### 配置维度 1：查询范围（参数 VPNINSTANCE，可选）

| 取值 | 行为 |
|---|---|
| 指定 VPNINSTANCE | 仅查该 VPN 配置 |
| 不配 | 查所有 VPN 配置 |

### 配置维度 2：详细度（参数 VERBOSE，可选）

| 取值 | 行为 |
|---|---|
| ENABLE | 输出完整字段（含 AFTYPE/VRFRD 等） |
| 不配 | 输出关键字段 |

**典型脚本（GWFD-010105 调测：查所有 VPN 实例配置）**：
```
LST VPNINST:;
```

**典型脚本（指定 VPNINSTANCE 查询完整字段）**：
```
LST VPNINST:VPNINSTANCE="vpn1",VERBOSE=ENABLE;
```

> 命令 notes：适用 NF：PGW-U、UPF；用于 VPN 配置核查。

**步骤位置**：在所属 feature 编排中通常位于"调测/验证阶段"——前置条件：VPN 已建；用于 VPN 三者一致约束核查。

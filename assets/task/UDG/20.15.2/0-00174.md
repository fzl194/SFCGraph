# 设置基于APN的下行数据缓存时长（SET APNDLBUFTIME）

> 命令静态知识见 [SET APNDLBUFTIME](command/UDG/20.15.2/SET-APNDLBUFTIME.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

本命令配置 **指定 APN 的下行数据缓存时长**（eDRX 模式下 APN 级配置，优先级高于全局 [0-00172](task/UDG/20.15.2/0-00172.md)；整机最多 10000 条 ApnDLBufTime）。NB-IoT/eMTC/5G 三特性共用本 atom，但**单次 SET 仅配一种用户类型时长**（互斥，由 DP 0-00136 路由）。

**配置范式（按用户类型三选一，APN 引用 + 单时长）**

| 用户类型 | 时长参数 | 取值样例 | 取值范围 | 适用特性 | APN 取值样例 |
|---|---|---|---|---|---|
| NB-IoT | NBIOTUSER | `10` | 3~200 秒 | GWFD-110601 | `apnnb` |
| Normal（普通） | NORMALUSER | `12` | 3~15 秒 | GWFD-110641 | `apnemtc` |
| RedCap-NR | REDCAPNRUSER | `10` | 3~200 秒 | GWFD-110661 | `apn-test` |

- **APN**：字符串 1~63，由 `-`/数字/字母/`.` 组成，不以 `.` 开头。引用 [0-00020](task/UDG/20.15.2/0-00020.md)（ADD APN）已建实例，可用 LST APN 查询。
- 三时长系统初始值均=6s。若 SMF 下发 DL Buffering Duration 信元则取下发值，否则取本参数。
- APN 级配置优先级 APN>全局（[0-00172](task/UDG/20.15.2/0-00172.md)）。

**配置步骤（按特性路由，三选一）**

1. 前置：先执行 [0-00020](task/UDG/20.15.2/0-00020.md)（ADD APN）创建 APN 实例。
2. 按目标用户类型选一时长参数，执行（NB-IoT / eMTC / 5G 各一）：
   ```
   SET APNDLBUFTIME: APN="apnnb", NBIOTUSER=10;
   SET APNDLBUFTIME: APN="apnemtc", NORMALUSER=12;
   SET APNDLBUFTIME: APN="apn-test", REDCAPNRUSER=10;
   ```

**示例脚本**（NB-IoT）：`SET APNDLBUFTIME: APN="apnnb", NBIOTUSER=10;`

## 决策点：APN级下行数据缓存时长 用户类型三选一（DP 0-00136）

问：SET APNDLBUFTIME 单次执行时，配置哪种用户类型的下行数据缓存时长？

| 选项 | 路由时长参数 | 必选性变化 | 适用特性 |
|---|---|---|---|
| NBIOT_USER（默认） | NBIOTUSER=required；NORMALUSER/REDCAPNRUSER=absent（不传） | 取值 3~200s | GWFD-110601 NB-IoT eDRX |
| NORMAL_USER | NORMALUSER=required；NBIOTUSER/REDCAPNRUSER=absent | 取值 3~15s | GWFD-110641 eMTC eDRX |
| REDCAP_NR_USER | REDCAPNRUSER=required；NBIOTUSER/NORMALUSER=absent | 取值 3~200s（与 SMF DL Buffering Duration 取最小） | GWFD-110661 5G eDRX |

> 依据：命令真相 3 时长均 optional，单次 SET 仅配其一，3 种用户类型互斥不可同时配。DP 把"特性路由→时长参数必选/不传"显化，防止误配多种时长。

## 约束

- **rule-0-00347**（warning）整机最多 10000 条 ApnDLBufTime，超限 SET 失败；命令立即生效；系统初始 NORMALUSER/NBIOTUSER/REDCAPNRUSER 均=6s；此命令生效范围限于 APN（与 [0-00172](task/UDG/20.15.2/0-00172.md) 优先级 APN>全局）；时长取值范围 NORMALUSER 3~15s、NBIOTUSER/REDCAPNRUSER 3~200s；DP 0-00136 编排 USER-TYPE 三选一，单次 SET 仅配一种 — 后果：超限 SET 失败 / 未先建 APN 则引用失败 / 时长超范围 SET 失败 / 三时长混传命令拒收

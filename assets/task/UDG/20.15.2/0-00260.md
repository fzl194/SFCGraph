# 增加ICAP服务器组（ADD ICAPSVRGRP）

> 命令静态知识见 [ADD ICAPSVRGRP](command/UDG/20.15.2/ADD-ICAPSVRGRP.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

本任务创建 ICAP 服务器组——作为 URL 过滤模板（ADD CFTEMPLATE 的 ICAPSRVGMNAME）与 ICAP 服务器绑定关系（ADD ICAPSVRBINDISG）引用的容器。`ICAPSVRGRPNAME` 供后续绑定关系引用。适用 NF：PGW-U、UPF。

ICAPSERVERTYPE 在本特性场景固定 URL_FILTERING（单一值，无跨特性分叉，未单独建 DP）。

**取值样例（数据规划表，主特性 GWFD-110471）**

| 参数 | 含义 | 样例值 | 获取方法 |
|---|---|---|---|
| ICAPSVRGRPNAME | ICAP服务器组名称 | icapsvrgrp1 | 全网规划（字符串 1～31，不支持空格、不区分大小写） |
| ICAPSERVERTYPE | ICAP服务器类型 | URL_FILTERING | 固定取值 |

**主场景脚本样例**

```
ADD ICAPSVRGRP:ICAPSVRGRPNAME="icapsvrgrp1", ICAPSERVERTYPE=URL_FILTERING;
```

**未绑参数说明（按 baseline §4.1 冷门不绑）**：CFGDOMAINNAME（配置域名称，optional，主场景不配）。

## 约束

- 命令最大记录数 **30**；每种类型下最大 **30 个**。
- ICAPSVRGRPNAME 字符串 1～31 字符，不支持空格、不区分大小写。
- 命令执行后**立即生效**。

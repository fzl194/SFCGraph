# 增加ICAP服务器绑定关系（ADD ICAPSVRBINDISG）

> 命令静态知识见 [ADD ICAPSVRBINDISG](command/UDG/20.15.2/ADD-ICAPSVRBINDISG.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

本任务把 ICAP 服务器绑定进 ICAP 服务器组——形成"服务器组 → 服务器"多对一容器关系，URL 过滤模板最终通过组名落到具体 ICAP Server。两个必填参数均为引用型（已配置数据中获取），自身不规划新值。适用 NF：PGW-U、UPF。

**取值样例（数据规划表，主特性 GWFD-110471）**

| 参数 | 含义 | 样例值 | 获取方法 |
|---|---|---|---|
| ICAPSVRGRPNAME | ICAP服务器组名称 | icapsvrgrp1 | 引用 [ADD ICAPSVRGRP](task/UDG/20.15.2/0-00260.md) 已配数据 |
| ICAPSERVERNAME | ICAP服务器名称 | icapserver1 | 引用 [ADD ICAPSERVER](task/UDG/20.15.2/0-00259.md) 已配数据 |

**主场景脚本样例**

```
ADD ICAPSVRBINDISG: ICAPSVRGRPNAME="icapsvrgrp1", ICAPSERVERNAME="icapserver1";
```

**未绑参数说明（按 baseline §4.1 冷门不绑）**：CFGDOMAINNAME（配置域名称，optional，主场景不配）。

## 约束

- 命令最大记录数 **300**；单个 ICAP Server Group 实例最多绑定 **10 个** ICAP Server。
- 前置依赖：必须先配置 [ADD ICAPSERVER](task/UDG/20.15.2/0-00259.md)（ICAPSERVERNAME）与 [ADD ICAPSVRGRP](task/UDG/20.15.2/0-00260.md)（ICAPSVRGRPNAME）。
- 命令执行后**立即生效**。

# 设置APN级HTTP2.0协议回落开关（SET APNHTTP2DGRD）

> 命令静态知识见 [SET APNHTTP2DGRD](command/UDG/20.15.2/SET-APNHTTP2DGRD.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法

SET APNHTTP2DGRD 用于设置 **APN 级** HTTP2.0 协议回落开关，是 **GWFD-110202（HTTP2.0 协议回落）APN 级开关**：当运营商网关不具备 HTTP2.0 处理能力时，按 APN 级别控制回落。**回落优先级 rule 级 > APN 级 > 全局**；APN 级 INHERIT 时继承全局 SRVCOMMONPARA(0-00065)。立即生效，仅 APN 生效。1 个特性命中（110202 HTTP2.0 协议回落）。

### 配置维度 1：APN 定位（参数 APN，必选）

| 取值 | 来源 |
|---|---|
| **APN 名**（如 `apn-test`） | 引用 [0-00020](task/UDG/20.15.2/0-00020.md).APN（ADD APN 先建） |

### 配置维度 2：回落开关（参数 HTTP2DEGRADESW，必选，DP 0-00149 决定）

| 取值 | 作用 | 适用场景 |
|---|---|---|
| **ENABLE** | APN 级强制开启回落 | 110202 范式（采用此值） |
| **DISABLE** | APN 级强制关闭回落 | APN 不需要回落 |
| **INHERIT**（默认） | 继承全局 SET SRVCOMMONPARA(0-00065) 的 HTTP2DEGRADESW | 跟随全局策略 |

> **三层保护**：rule 级(0-00150) > APN 级(本 atom) > 全局级(0-00065)；三层均 INHERIT 时无生效源，须至少一层显式配置 ENABLE/DISABLE。

**任务示例（110202 范式）**：

```
SET APNHTTP2DGRD: APN="apn-test", HTTP2DEGRADESW=ENABLE;
```

**步骤位置**：在所属 feature 编排中通常位于"步骤 N：APN 级回落开关"——前置 [0-00020](task/UDG/20.15.2/0-00020.md) ADD APN；可选全局继承 [0-00065](task/UDG/20.15.2/0-00065.md) SET SRVCOMMONPARA；rule 级联动 [0-00150](task/UDG/20.15.2/0-00150.md)。

## 决策点：APN 级回落开关（DP 0-00149）

| 选项 | HTTP2DEGRADESW | 适用场景 |
|---|---|---|
| **opt-apn-enable** | ENABLE | APN 级强制开启回落（110202 范式采用） |
| **opt-apn-disable** | DISABLE | APN 级强制关闭回落 |
| **opt-apn-inherit（默认）** | INHERIT | 继承全局 SRVCOMMONPARA(0-00065) HTTP2DEGRADESW |

> 三层联动保护：与 rule 级 DP 0-00150 + 全局级 SRVCOMMONPARA 组合，防止 3 层均 INHERIT 时无生效源。

## 约束

- **最大记录数 10000**（warning，rule-0-00149）：单命令最大 10000 条 — 后果：超规格新增失败
- **立即生效**（warning，rule-0-00149）：执行后立即生效 — 后果：误配即刻影响 APN 下用户的 HTTP2.0 回落
- **仅 APN 生效**（info，rule-0-00149）：配置只对指定 APN 生效，不影响其他 APN — 后果：每个 APN 须独立配置
- **HTTP2DEGRADESW=INHERIT 继承全局**（warning，rule-0-00149）：INHERIT 时继承 SET SRVCOMMONPARA(0-00065) 全局开关 — 后果：全局未配则 APN 也无法生效
- **三层联动保护**（critical，rule-0-00149）：rule(0-00150)/APN(本 atom)/全局(0-00065) 不可均 INHERIT — 后果：3 层均 INHERIT 时无生效源
- **APN 前置必备**（critical，rule-0-00149）：APN 须已由 ADD APN(0-00020) 创建 — 后果：APN 未建命令拒收
- **2/2 参数全绑定**（info，rule-0-00149）：APN + HTTP2DEGRADESW 全部必填，0 缺口

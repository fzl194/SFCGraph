# 设置APN级HTTP2.0协议回落开关（SET APNHTTP2DGRD）

> 命令静态知识见 [SET APNHTTP2DGRD](command/UDG/20.15.2/SET-APNHTTP2DGRD.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

本命令是 **GWFD-110202（HTTP2.0 协议回落）APN 级开关**：当运营商网关不具备 HTTP2.0 处理能力时，按 APN 级别控制回落。回落优先级 **rule 级 > APN 级 > 全局**；APN 级 INHERIT 时继承全局 [[UDG@20.15.2@Task@0-00148]]（SET SRVCOMMONPARA）。立即生效，仅 APN 生效。

**配置范式（单范式 + 决策开关）**

| 参数 | 取值样例 | 来源 | 说明 |
|---|---|---|---|
| APN | `apn-test` | 引用 [0-00020](task/UDG/20.15.2/0-00020.md).APN | APN 名称，ADD APN 先建 |
| HTTP2DEGRADESW | `ENABLE` / `DISABLE` / `INHERIT` | decision_driven → DP 0-00142 | 3-way 编排，默认 INHERIT |

**任务示例**

```
SET APNHTTP2DGRD: APN="apn-test", HTTP2DEGRADESW=ENABLE;
```

参数 2/2 全绑定，0 缺口。

## 决策点

**DP 0-00142**（single_choice）APN 级 HTTP2.0 协议回落开关 3-way 编排（默认 INHERIT）

| 选项 | 取值 | 说明 |
|---|---|---|
| opt-apn-enable | `ENABLE` | APN 级强制开启回落（本特性示例采用） |
| opt-apn-disable | `DISABLE` | APN 级强制关闭回落 |
| opt-apn-inherit（默认） | `INHERIT` | 继承全局 [[UDG@20.15.2@Task@0-00148]] HTTP2DEGRADESW |

与 rule 级 DP [[UDG@20.15.2@DecisionPoint@0-00143]] + 全局级 SRVCOMMONPARA 三层联动，防止 3 层均 INHERIT 时无生效源。

## 约束

- **rule-0-00353**（warning）单命令最大记录数 10000；立即生效；仅 APN 生效；HTTP2DEGRADESW=INHERIT 时继承 SET SRVCOMMONPARA 全局开关；前置 ADD APN(0-00020)；三层联动保护——rule(0-00150)/APN(本 atom)/全局(0-00148) 不可均 INHERIT

# 创建逻辑接口（ADD LOGICINF）

> 命令静态知识见 [ADD LOGICINF](command/UDG/20.15.2/ADD-LOGICINF.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。**适用 NF**：SGW-U、PGW-U、UPF。逻辑接口是"物理上不存在、需要通过配置建立的接口"，用于 SGW-U/PGW-U/UPF 实现数据交换；按用途划分为 25 种类型（Paif/N4if/Phif/Giif/S11-uif/Gcfif 等）。

## 配置方法（配置方法字典）

ADD LOGICINF 是"**创建逻辑接口**"命令。NAME/IPVERSION/IPV4ADDRESS1/IPV4MASK1（或 IPV6ADDRESS1/IPV6PREFIXLEN1）四参必选（IPv4 时）或五参必选（IPV6 时）；VPNINSTANCE 可选。核心配置空间有 5 个维度：**接口名**（NAME 格式 {类型}{ISU组号}/{实例类型}/{接口号}）、**IP 版本**（IPVERSION IPv4/IPv6/IPVER_ALL 门控）、**地址参数**（IPV4ADDRESS1+MASK1 / IPV6ADDRESS1+PREFIXLEN1 + 可选 IPV4ADDRESS2+MASK2 / IPV4MTU / IPV6MTU）、**VPN 绑定**（VPNINSTANCE 条件可选）、**接口类型**（Giif/Vxlanif/Swuif/Swmif 4 类不支持 IPv6）+ **切片属性**（SLICEATTRSW 非 N3/SA 口不能配）。8 个特性命中覆盖 6 种用途（Paif ePDG/IPFarm 心跳/IPFarm-IMS/L2TP Giif/S11-U 信令/ICAP Server）。

### 配置维度 1：接口名（参数 NAME，必选）

| 参数 | 取值范围 | 备注 |
|---|---|---|
| **NAME** | 字符串 1~63 字符 | 格式固定为 `{逻辑接口类型}{ISU组号}/{实例类型}/{逻辑接口号}`，如 `Paif1/1/0` / `phif1/0/0` / `giif1/0/1` / `s11-uif1/1/0` / `gcfif1/0/0` |

> **NAME 跨特性按用途约定**：Paif（ePDG 互通）/N4if（UPF 与 SMF/SGW-C/PGW-C 互通）/Phif（IPFarm 心跳）/Giif（L2TP 源端）/S11-uif（S11-U 信令面）/Gcfif（ICAP Server）/ N3if/N9cif/Scif/Tm3if/Tx-uif/N6mbif/N3mbif/M1if/Sgimbif/Swuif/Swmif/Vxlanif 等。
> **接口号范围**视 ISU 实例类型（0=组级；1~64=Instance 级）而定：`giif` 仅实例 0 用（0~1023），`s5-sif` 实例 1 用（0~31），`n4if` 仅实例 0（仅 0 号），`swuif/swmif` 实例 0 用（swuif=0；swmif=0~15）等。

### 配置维度 2：IP 版本（参数 IPVERSION，必选，3 选 1 enum）

IPVERSION 决定后续地址参数走 IPv4/IPv6 双栈：

| 取值 | 联动必选参数 | 适用场景 |
|---|---|---|
| **IPV4** | IPV4ADDRESS1 + IPV4MASK1 | 主流 IPv4 场景（GWFD-010155 Paif/IPFarm/S11-U/L2TP/ICAP） |
| **IPV6** | IPV6ADDRESS1 + IPV6PREFIXLEN1 | IPv6 场景（GWFD-010155 OSPFv3 Paif） |
| **IPVER_ALL** | 同时配 IPV4 + IPV6 | 双栈场景 |

### 配置维度 3：地址参数

**IPv4 路径**（IPVERSION=IPV4/IPVER_ALL 时）：

| 参数 | 必选性 | 取值范围 | 默认值 | 备注 |
|---|---|---|---|---|
| **IPV4ADDRESS1** | 必选 | IPv4 点分十进制 | — | 逻辑接口主 IP（按局点网络规划，A/B/C 类合法） |
| **IPV4MASK1** | 必选 | IPv4 点分十进制 | — | 主 IP 掩码（32 位） |
| **IPV4ADDRESS2** | 可选 | IPv4 点分十进制 | — | 逻辑接口子 IP（**仅 swmif 时使用**） |
| **IPV4MASK2** | 可选 | IPv4 点分十进制 | — | 子 IP 掩码（仅 swmif 时使用） |
| **IPV4MTU** | 可选 | 328~9600 | 1500 | NP 加速场景建议 <8500 |

**IPv6 路径**（IPVERSION=IPV6/IPVER_ALL 时）：

| 参数 | 必选性 | 取值范围 | 默认值 | 备注 |
|---|---|---|---|---|
| **IPV6ADDRESS1** | 必选 | IPv6 冒号分十六进制 | — | 主 IPv6 地址（不能全 0 全 1，不能环回 IP） |
| **IPV6PREFIXLEN1** | 必选 | 1~128 | — | 主 IPv6 前缀长度（**主流 128**） |
| **IPV6MTU** | 可选 | 1280~9600 | 1800 | NP 加速场景建议 <8500 |

> **Giif/Vxlanif/Swuif/Swmif 4 类接口不支持配 IPv6 地址**。

### 配置维度 4：VPN 绑定（参数 VPNINSTANCE，条件可选）

| 参数 | 必选性 | 取值范围 | 备注 |
|---|---|---|---|
| **VPNINSTANCE** | 可选（按用途） | 字符串 1~31 字符（区分大小写） | 绑定 VPN 时须确保该 VPN 已配（[0-00071](task/UDG/20.15.2/0-00071.md) ADD L3VPNINST 或 [0-00041](task/UDG/20.15.2/0-00041.md) ADD VPNINST）；**外联口 VPN 须与逻辑接口 VPN 相同否则报文查路由失败** |

> **VPNINSTANCE 跨用途分叉**：Paif（ePDG 互通）走 [0-00071](task/UDG/20.15.2/0-00071.md) ADD L3VPNINST（vpn-s2b/vpn-paif）；L2TP-Giif 走 [0-00041](task/UDG/20.15.2/0-00041.md) ADD VPNINST（vpn_l2tp，须与 APN 绑定 VPN 一致）；IPFarm-IMS 走 ims_vpn；IPFarm 心跳/S11-U 不配。

### 配置维度 5：切片属性（参数 SLICEATTRSW，可选）

| 参数 | 取值范围 | 备注 |
|---|---|---|
| **SLICEATTRSW** | DISABLE / ENABLE | **非 N3 和 SA 口不能配置** |

**典型脚本（Paif IPv4 ePDG 互通，vpn-s2b）**：
```
ADD LOGICINF: NAME="Paif1/1/0", IPVERSION=IPV4, IPV4ADDRESS1="10.10.10.10", IPV4MASK1="255.255.255.255", VPNINSTANCE="vpn-s2b";
```

**典型脚本（Paif IPv6 ePDG，vpn-paif）**：
```
ADD LOGICINF: NAME="Paif1/0/0", IPVERSION=IPV6, IPV6ADDRESS1="fc00::1", VPNINSTANCE="vpn-paif", IPV6PREFIXLEN1=128;
```

**典型脚本（N4if UPF 与 SMF 互通，vpn1）**：
```
ADD LOGICINF: NAME="n4if1/0/0", IPVERSION=IPV4, IPV4ADDRESS1="192.168.1.1", IPV4MASK1="255.255.255.255", VPNINSTANCE="vpn1";
```

**典型脚本（Phif IPFarm 心跳本端接口，预留地址）**：
```
ADD LOGICINF: NAME="phif1/0/0", IPVERSION=IPV4, IPV4ADDRESS1="10.10.10.10", IPV4MASK1="255.255.255.255";
```

**典型脚本（L2TP Giif 源端，vpn_l2tp）**：
```
ADD LOGICINF: NAME="giif1/0/1", IPVERSION=IPV4, IPV4ADDRESS1="10.8.20.2", IPV4MASK1="255.255.255.255", VPNINSTANCE="vpn_l2tp";
```

**典型脚本（S11-U 信令面数据传输）**：
```
ADD LOGICINF: NAME="s11-uif1/1/0", IPVERSION=IPV4, IPV4ADDRESS1="10.2.151.15", IPV4MASK1="255.255.255.255";
```

**典型脚本（ICAP Server 互通）**：
```
ADD LOGICINF: NAME="gcfif1/0/0", IPVERSION=IPV4, IPV4ADDRESS1="10.10.1.7", IPV4MASK1="255.255.255.255", VPNINSTANCE="vpn-gcfif";
```

> 命令 notes：**适用 NF**：SGW-U、PGW-U、UPF；**最大记录数 4096**（>98% 触发 ALM-135602215 告警，≤95% 恢复，阈值可 [MOD CFGTHRESHOLD](command/UDG/20.15.2/MOD-CFGTHRESHOLD.md) 改）；**执行后立即生效**；**绑定 VPNINSTANCE 须前置 ADD VPNINST/ADD L3VPNINST**；**配置接口如需跨安全域传输（如漫游/分流场景）须在安全域边界加防火墙**；**Giif/Vxlanif/Swuif/Swmif 4 类接口不支持配 IPv6 地址**。

**步骤位置**：在所属 feature 编排中通常位于"步骤 1：创建逻辑接口"——前置 [0-00071](task/UDG/20.15.2/0-00071.md) ADD L3VPNINST 或 [0-00041](task/UDG/20.15.2/0-00041.md) ADD VPNINST（绑定 VPN 时）；配套 [MOD LOGICINF](command/UDG/20.15.2/MOD-LOGICINF.md) / [RMV LOGICINF](command/UDG/20.15.2/RMV-LOGICINF.md)。

## 决策点：用途 + IP 版本 + VPN 绑定组合（DP 0-00083）

问：ADD LOGICINF 取何用途 + IP 版本 + VPN 绑定组合？

| 用途 | NAME 模式 | IPVERSION | VPNINSTANCE | 适用场景 |
|---|---|---|---|---|
| **Paif ePDG 互通 IPv4** | `Paif<x>/<y>/<z>` | IPV4 | ref [0-00071](task/UDG/20.15.2/0-00071.md) `vpn-s2b` | GWFD-010155 VPN+OSPF/缺省/静态+BFD |
| **Paif ePDG 互通 IPv6** | `Paif<x>/<y>/<z>` | IPV6 | ref [0-00071](task/UDG/20.15.2/0-00071.md) `vpn-paif` | GWFD-010155 OSPFv3 动态路由 |
| **N4if UPF 互通** | `n4if1/0/0`（固定） | IPV4 | ref [0-00071](task/UDG/20.15.2/0-00071.md) `vpn1` | UPF/SMF/SGW-C/PGW-C 控制面互通 |
| **Phif IPFarm 心跳** | `phif1/0/0`/`phif1/0/1` | IPV4 | 不配 / ims_vpn | GWFD-110281 Portal；GWFD-110282 Web Proxy；GWFD-020253 IMS 故障恢复 |
| **Giif L2TP 源端** | `giif1/0/0`/`giif1/0/1` | IPV4 | ref [0-00041](task/UDG/20.15.2/0-00041.md) `vpn_l2tp`（须与 APN 绑定 VPN 一致） | GWFD-020412 L2TP-VPN |
| **S11-U 信令面数据传输** | `s11-uif1/1/0` | IPV4 | 不配 | GWFD-110606 |
| **Gcfif ICAP Server** | `gcfif1/0/0`（固定） | IPV4 | ref [0-00071](task/UDG/20.15.2/0-00071.md) `vpn-gcfif` | GWFD-110471 ICAP 互通 |

> **关键约束**：**外联口 VPN 须与逻辑接口 VPN 相同否则报文查路由失败**；**IPFarm 心跳本端 IP 一般从终端地址池获取，配置地址池时需预留 IP**；**Giif/Vxlanif/Swuif/Swmif 不能配 IPv6**；**SLICEATTRSW 仅 N3/SA 口能配**。

## 约束

- **适用 NF 限定**（critical，rule-0-00483）：适用 NF：SGW-U、PGW-U、UPF — 后果：其他 NF 不适用
- **最大记录数 4096**（warning，rule-0-00483）：该命令最大记录数为 4096 — 后果：超规格新增失败
- **配置超阈值告警**（warning，rule-0-00483）：当配置记录数大于规格 98% 时上报 ALM-135602215 配置数量超阈值告警；≤95% 恢复；阈值可通过 MOD CFGTHRESHOLD 修改 — 后果：接近上限触发告警
- **执行后立即生效**（warning，rule-0-00483）：命令执行后立即生效 — 后果：误操作即刻生效
- **绑定 VPNINSTANCE 须前置 VPN**（critical，rule-0-00483）：若需输入 VPNINSTANCE（逻辑接口与 VPN 绑定），须提前配置 ADD VPNINST/ADD L3VPNINST — 后果：未建 VPN 即绑命令拒收
- **未配 VPNINSTANCE 默认下公网 VPN**（warning，rule-0-00483）：如果用户没有对 VPN 实例进行配置，系统将会下发缺省 VPN — 后果：未显式 VPNINSTANCE 走缺省 VPN（公网）
- **外联口 VPN 须与逻辑接口 VPN 一致**（critical，rule-0-00483）：外联口 VPN 须与逻辑接口 VPN 相同否则报文查路由失败 — 后果：VPN 不一致致转发失败
- **Giif/Vxlanif/Swuif/Swmif 不支持 IPv6**（critical，rule-0-00483）：逻辑接口类型为 giif、vxlanif、swuif 或 swmif 时不支持配置 IPv6 地址 — 后果：违反致命令拒收
- **IPV4ADDRESS2/MASK2 仅 swmif 可用**（warning，rule-0-00483）：IPV4ADDRESS2/IPV4MASK2 仅在逻辑接口为 swmif 时使用 — 后果：其他类型配次地址致命令拒收
- **SLICEATTRSW 仅 N3/SA 口能配**（warning，rule-0-00483）：非 N3 和 SA 口不能配置 SLICEATTRSW — 后果：违反致命令拒收
- **跨安全域传输须防火墙**（info，rule-0-00483）：配置接口如需跨安全域传输（如漫游/分流场景），应在安全域边界应用防火墙 — 后果：未加防火墙存在安全风险
- **IPV6ADDRESS 不能全 0 全 1/环回**（critical，rule-0-00483）：IPV6ADDRESS 不能是全 0 全 1，不能是环回 IP — 后果：违反致命令拒收

# 添加APN的IMS信令分类器（ADD APNIMSSIGFLTR）

> 命令静态知识见 [ADD APNIMSSIGFLTR](command/UDG/20.15.2/ADD-APNIMSSIGFLTR.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

ADD APNIMSSIGFLTR 配置 APN 下 IMS 信令专用上下文的 Filter，用于 VoLTE 语音业务识别 IMS 信令流。**关键配法**：IMS 信令通常 TCP(6)+UDP(17) 双协议，需**多次执行本命令配置两条分类器**（不同 PRIORITY、不同 PROTOCOL），属迭代配置非配法分叉。

**参数规划（IPv4 源 path，主特性样例）**

| 参数 | 含义 | 取值样例 | 说明 |
|---|---|---|---|
| APN | APN 名称 | `ims` | **引用** [0-00020](task/UDG/20.15.2/0-00020.md)（ADD APN）已配置实例；只能配做 IMS 业务的 APN |
| PRIORITY | 优先级 | `1`（TCP）/ `2`（UDP） | 1~64，**同一 APN 内唯一** |
| IPVERSION | IP 地址版本 | `IPV4` | 主特性用 IPv4 源 path |
| SRCIPV4SPECTYPE | 源 IPv4 指定方式 | `SPECIFY` | 配套 type 标记 |
| SRCIPV4ADDR | 源 IPv4 地址 | `10.1.1.1` | 点分十进制 |
| SRCIPV4MASKLEN | 源 IPv4 掩码长度 | `32` | 0~32 |
| SRCPORTSPECTYPE | 源端口指定方式 | `SPECIFY` | 配套 type 标记 |
| SRCPORTSTART | 源端口开始 | `1` | 1~65535 |
| SRCPORTEND | 源端口结束 | `2` | 1~65535 |
| PROSPECTYPE | 协议指定方式 | `SPECIFY` | 配套 type 标记 |
| PROTOCOL | 协议类型 | `6`（TCP）/ `17`（UDP） | TCP=6，UDP=17 |

**典型脚本（TCP + UDP 双分类器）**

```
ADD APNIMSSIGFLTR: APN="ims", PRIORITY=1, IPVERSION=IPV4, SRCIPV4SPECTYPE=SPECIFY, SRCIPV4ADDR="10.1.1.1", SRCIPV4MASKLEN=32, SRCPORTSPECTYPE=SPECIFY, SRCPORTSTART=1, SRCPORTEND=2, PROSPECTYPE=SPECIFY, PROTOCOL=6;
ADD APNIMSSIGFLTR: APN="ims", PRIORITY=2, IPVERSION=IPV4, SRCIPV4SPECTYPE=SPECIFY, SRCIPV4ADDR="10.1.1.1", SRCIPV4MASKLEN=32, SRCPORTSPECTYPE=SPECIFY, SRCPORTSTART=1, SRCPORTEND=2, PROSPECTYPE=SPECIFY, PROTOCOL=17;
```

**配置位置**（GWFD-020251 VoLTE 基础语音业务）
1. SET LICENSESWITCH 开 License。
2. **ADD APNIMSSIGFLTR**（本命令，建 IMS 信令 Filter）。
3. SET APNIMSATTR（配 IMS 使能开关 + IMS 信令空口增强，绑到 APN 实例）。

**冷门参数（主场景未用，15 个 optional/conditional）**：IPv6 源/目的 path（`SRCIPV6*` / `DSTIPV6*`）、目的 IPv4/端口（`DSTIPV4*` / `DSTPORT*`）、各种 `*SPECTYPE` 配套标记。主特性仅用 IPv4 源 path，`*SPECTYPE` 采用默认 ANY/SPECIFY 隐式表达。若后续 IPv6 IMS 信令识别特性启用，须新建独立 atom 覆盖 V6SRC*。

## 约束

- **rule-0-00355**（warning）单命令最大记录数 **64000**，超限 ADD 失败。
- **rule-0-00355**（warning）**仅对新激活用户生效**，已激活用户不受影响。
- **rule-0-00355**（warning）**高危命令**：须确认添加规则后合法 IMS 信令报文都能命中其中一条规则，配置不当将影响 IMS 语音信令转发、**造成语音不通**。
- **rule-0-00355**（warning）APN 参数**只能配做 IMS 业务的 APN**，非 IMS APN 配置无意义且干扰识别。
- **rule-0-00355**（warning）PRIORITY **同一 APN 内唯一**（1~64），多次配置须保证不冲突。
- **rule-0-00355**（warning）前置依赖：APN 必须先经 ADD APN [0-00020](task/UDG/20.15.2/0-00020.md) 配置，否则引用失败。

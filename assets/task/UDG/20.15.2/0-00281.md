# 配置SA指纹识别（ADD FINGERIDENT）

> 命令静态知识见 [ADD FINGERIDENT](command/UDG/20.15.2/ADD-FINGERIDENT.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

本任务开启 SA 指纹识别——在正常特征库识别之外对 HTTPS 业务报文做 TLS 指纹识别，提升未携带 SNI 场景的业务识别精度。指纹识别只增强不替代特征库。`CFGPROTMODE`（ALL/SPECIFIC）跨特性分叉由决策点 DP 0-00042 驱动；本特性主场景样例为 ALL（全协议）。适用 NF：PGW-U、UPF。

**取值样例（数据规划表，主特性 GWFD-110404）**

| 参数 | 含义 | 样例值 | 获取方法 |
|---|---|---|---|
| CFGPROTMODE | 协议配置方式 | ALL | 决策驱动（ALL=所有协议 / SPECIFIC=指定协议，见 DP 0-00042） |
| PROTOCOLNAME | 协议名称 | （主场景不配） | 本端规划，不支持空格不区分大小写；CFGPROTMODE=SPECIFIC 时必填 |
| FINGERIDENTCTRL | 指纹识别结果控制开关 | DISABLE（默认） | 本端规划，DISABLE=统计且参与控制；主场景用默认值 |
| CFGDOMAINNAME | 配置域名称 | （主场景不配） | 本端规划，字符串 1～31；CFGPROTMODE=SPECIFIC 时可选 |

**主场景脚本样例（全协议指纹识别）**

```
ADD FINGERIDENT:CFGPROTMODE=ALL;
```

> 前置先执行 [LOD SIGNATUREDB](command/UDG/20.15.2/LOD-SIGNATUREDB.md) 加载协议特征库文件（特征库前置依赖）。可通过 [LST SUPPFINGERIDEN](command/UDG/20.15.2/LST-SUPPFINGERIDEN.md) 查询系统支持指纹识别的协议。

## 决策点

**[[UDG@20.15.2@DecisionPoint@0-00042]] — SA 指纹识别协议范围**（feature-挂，驱动 CFGPROTMODE 取 ALL/SPECIFIC）

- 选 **ALL**：全协议指纹识别，PROTOCOLNAME 不配（本特性主场景）。
- 选 **SPECIFIC**：指定协议指纹识别，PROTOCOLNAME 必填、CFGDOMAINNAME 可选。

## 约束

- 命令最大记录数 **128**。
- 同时配置了指定协议和所有协议的 FINGERIDENTCTRL 参数时，**优先使用指定协议的配置**（SPECIFIC 优先）。
- 并不是所有特征库都能被指纹识别。
- 前置依赖：执行本命令前需先 [LOD SIGNATUREDB](command/UDG/20.15.2/LOD-SIGNATUREDB.md) 加载特征库文件。
- 命令执行后**立即生效**。

# 设置APN的L2TP属性（SET APNL2TPATTR）

> 命令静态知识见 [SET APNL2TPATTR](command/UDG/20.15.2/SET-APNL2TPATTR.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

SET APNL2TPATTR 使能指定 APN 的 L2TP 功能并下发该 APN 的 L2TP 属性，是 L2TP 激活的总开关。APN 引用上游 [ADD APN](task/UDG/20.15.2/0-00020.md)。两种激活方式差异在一个互斥参数：AAA 方式填 RDSLNSMODE（RADIUS 返回多 LNS 模式），本地方式填 L2TPGROUPID（引用 [ADD L2TPGROUP](task/UDG/20.15.2/0-00133.md) 的组号），由 [决策点](#决策点) 定。

**两种激活方式参数对比**（公共参 L2TPSWITCH/SUPPORTIPV6/ICRQ_CALLINGNO/ICCN_AUTH/IPCP_NEGO/DOMAINNAMEACT/DOMAINNAMEPOS 两脚本一致）

| 参数 | AAA 下发方式 | 本地配置方式 | 来源 |
|---|---|---|---|
| APN | `apn-l2tp` | `apn-l2tp` | reference（← 0-00020 ADD APN） |
| L2TPSWITCH | `ENABLE` | `ENABLE` | local_planned |
| SUPPORTIPV6 | `DISABLE` | `DISABLE` | local_planned（L2TP 内层 IPv6 开关，默认不使能） |
| RDSLNSMODE | `REDUNDANCY` | —（不填） | local_planned（AAA 方式） |
| L2TPGROUPID | —（不填） | `1` | reference（← 0-00133 ADD L2TPGROUP） |
| ICRQ_CALLINGNO | `MSISDN` | `MSISDN` | local_planned（可配 IMSI/IMEI） |
| ICCN_AUTH | `ENABLE` | `ENABLE` | local_planned |
| IPCP_NEGO | `ENABLE` | `ENABLE` | local_planned |
| DOMAINNAMEACT | `ADD_ENABLE_STRIP_DISABLE` | `ADD_ENABLE_STRIP_DISABLE` | local_planned（增/剥域名） |
| DOMAINNAMEPOS | `PREFIX` | `PREFIX` | local_planned（PREFIX/SUFFIX） |

**AAA 方式典型脚本**（不填 L2TPGROUPID，填 RDSLNSMODE）：

```
SET APNL2TPATTR:APN="apn-l2tp",L2TPSWITCH=ENABLE,SUPPORTIPV6=DISABLE,ICRQ_CALLINGNO=MSISDN,ICCN_AUTH=ENABLE,IPCP_NEGO=ENABLE,DOMAINNAMEACT=ADD_ENABLE_STRIP_DISABLE,DOMAINNAMEPOS=PREFIX;
```

**本地方式典型脚本**（填 L2TPGROUPID，不填 RDSLNSMODE）：

```
SET APNL2TPATTR:APN="apn-l2tp",L2TPSWITCH=ENABLE,SUPPORTIPV6=DISABLE,L2TPGROUPID=1,ICRQ_CALLINGNO=MSISDN,ICCN_AUTH=ENABLE,IPCP_NEGO=ENABLE,DOMAINNAMEACT=ADD_ENABLE_STRIP_DISABLE,DOMAINNAMEPOS=PREFIX;
```

## 决策点

**DP 0-00023 · L2TP 激活方式** — 决定填 RDSLNSMODE（AAA）还是 L2TPGROUPID（本地），两者**互斥**。

| 选项 | 含义 | 互斥参数影响 |
|---|---|---|
| AAA 下发 | RADIUS 返回 LNS 信息建隧道 | 填 RDSLNSMODE，不填 L2TPGROUPID |
| 本地配置 | 按本地 L2TP 组域名匹配建隧道 | 填 L2TPGROUPID，不填 RDSLNSMODE |

## 约束

- **rule-0-00366**（warning）系统最多支持配置 **10000 条** ApnL2tpAttr，**每 APN 一条**（APN 唯一索引）；执行后**立即生效**；存在系统初始记录（L2TPSWITCH=DISABLE / L2TPGROUPID=NULL / RDSLNSMODE=NULL / …）；适用 NF：PGW-U、UPF — 后果：超规格新增失败；未先 ADD APN 则无 APN 索引可绑
- RDSLNSMODE 与 L2TPGROUPID 互斥（由 DP 0-00023 决定填哪个）
- 前置：必须先 [配置APN](task/UDG/20.15.2/0-00020.md)；本地方式还须先 [创建 L2TP 组](task/UDG/20.15.2/0-00133.md)

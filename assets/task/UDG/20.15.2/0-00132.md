# 设置APN的L2TP属性（SET APNL2TPATTR）

> 命令静态知识见 [SET APNL2TPATTR](command/UDG/20.15.2/SET-APNL2TPATTR.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。**适用 NF：PGW-U、UPF**。

## 配置方法

SET APNL2TPATTR 为指定 APN 配置 L2TP 接入属性。**APN 必选**（1~63 字符）、**L2TPSWITCH 必选**（DISABLE/ENABLE）；其余参数 L2TPSWITCH=ENABLE 时才有效（L2TPGROUPID/RDSLNSMODE/ICRQ*/ICCN*/IPCP_NEGO/DOMAINNAME*/SUPPORTIPV6）。核心配置空间有 4 个维度：**L2TP 使能**（L2TPSWITCH）、**域指向**（L2TPGROUPID）、**L2TP 协商开关集合**（ICRQ_*/ICCN_*/IPCP_NEGO/SUPPORTIPV6）、**域名处理**（DOMAINNAMEACT/DOMAINNAMEPOS）。1 个特性命中（GWFD-020412 L2TP VPN 本地方式）配 L2TPSWITCH=ENABLE + L2TPGROUPID=1 + 域名处理 ADD_ENABLE_STRIP_DISABLE。

### 配置维度 1：L2TP 使能（参数 L2TPSWITCH，必选）

| 取值 | 作用 | 适用场景 |
|---|---|---|
| **DISABLE** | APN 不支持 L2TP 功能（默认） | 普通 APN 接入 |
| **ENABLE** | APN 支持 L2TP 功能（020412 范式） | L2TP VPN 用户接入 APN |

> 切换 L2TPSWITCH 后其余维度的可选参数才生效；ENABLE 时条件可选参数开始有效。

### 配置维度 2：域指向（参数 L2TPGROUPID，条件可选）

| 取值 | 作用 | 适用场景 |
|---|---|---|
| **不配 / SYSTEM_GLOBAL_GROUP** | 用户匹配 AAA 下发属性（AAA 方式） | 020412 AAA 方式范式 |
| **1~1500 整数** | 引用已建 L2TP 组号（[0-00133](task/UDG/20.15.2/0-00133.md) ADD L2TPGROUP） | 020412 本地方式范式（`L2TPGROUPID=1`） |

> L2TPSWITCH=ENABLE 时可选；取整数值时该 L2TP 组必须已建。

### 配置维度 3：L2TP 协商开关集合（参数 RDSLNSMODE / ICRQ_*/ICCN_*/IPCP_NEGO / SUPPORTIPV6，条件可选）

| 参数 | 取值 | 作用（020412 范式值） |
|---|---|---|
| **RDSLNSMODE** | FAILOVER/LOADSHARE | RADIUS 返回多 LNS 时工作模式 |
| **ICRQ_CALLINGNO** | MSISDN（默认）/DISABLE | ICRQ 携带 Calling-number AVP 值（020412 范式 MSISDN） |
| **ICCN_AUTH** | DISABLE/ENABLE | ICCN 携带鉴权信元开关（020412 范式 ENABLE） |
| **IPCP_NEGO** | DISABLE/ENABLE | 主动发起 IPCP 协商开关（020412 范式 ENABLE） |
| **ICCN_PGROUPID** | DISABLE/ENABLE | ICCN 携带 PrivateGroupId AVP 开关 |
| **ICCN_TXSPEED** | DISABLE/ENABLE | ICCN 携带 TxConnectSpeed 值 |
| **ICRQ_BEARER** | DISABLE/ENABLE | ICRQ 携带 Bearer Type AVP 开关 |
| **ICCN_LCPNEGO** | DISABLE/ENABLE | ICCN 携带 LCP CONFREQ 信元开关 |
| **SUPPORTIPV6** | DISABLE/ENABLE | L2TP 支持 IPv6 功能开关（020412 范式 DISABLE） |

> L2TPSWITCH=ENABLE 时全部条件可选；默认值见命令真相表。

### 配置维度 4：域名处理（参数 DOMAINNAMEACT+DOMAINNAMEPOS，条件可选）

| 参数 | 取值 | 作用 |
|---|---|---|
| **DOMAINNAMEACT** | ADD_ENABLE_STRIP_DISABLE（020412 范式）/ADD_ENABLE/STRIP/DISABLE | 入网用户名域名处理策略 |
| **DOMAINNAMEPOS** | PREFIX（020412 范式）/SUFFIX | 域名在用户名中的位置 |

> DOMAINNAMEPOS 仅在 DOMAINNAMEACT=ADD_ENABLE_STRIP_DISABLE 时为必选。

**典型脚本（020412 L2TP VPN 本地方式，L2TPGROUPID=1 + DOMAINNAMEACT=ADD_ENABLE_STRIP_DISABLE）**：
```
SET APNL2TPATTR:APN="apn-l2tp",L2TPSWITCH=ENABLE,SUPPORTIPV6=DISABLE,L2TPGROUPID=1,ICRQ_CALLINGNO=MSISDN,ICCN_AUTH=ENABLE,IPCP_NEGO=ENABLE,DOMAINNAMEACT=ADD_ENABLE_STRIP_DISABLE,DOMAINNAMEPOS=PREFIX;
```

**步骤位置**：在所属 feature 编排中通常位于"步骤 N：APN L2TP 属性"——前置 [0-00020](task/UDG/20.15.2/0-00020.md) ADD APN 建 APN + [0-00131](task/UDG/20.15.2/0-00131.md) SET GLOBALL2TP 设 L2TP 缺省；本地方式下后续 [0-00133](task/UDG/20.15.2/0-00133.md) ADD L2TPGROUP + [0-00135](task/UDG/20.15.2/0-00135.md) ADD L2TPCLIENTIP + [0-00139](task/UDG/20.15.2/0-00139.md) SET APNPPPACCESS + [0-00138](task/UDG/20.15.2/0-00138.md) SET PPPCFG；AAA 方式下后续 [0-00134](task/UDG/20.15.2/0-00134.md) ADD L2TPRDSCLIENT + [0-00139](task/UDG/20.15.2/0-00139.md) SET APNPPPACCESS。

## 决策点：APN L2TP 激活方式（DP 0-00132）

问：本 APN 下用户走 AAA 下发方式还是本地配置方式建 L2TP 隧道？

| 选项 | L2TPSWITCH | L2TPGROUPID | RDSLNSMODE | DOMAINNAMEACT/POS | 适用场景 |
|---|---|---|---|---|---|
| **本地配置方式**（020412 本地范式） | ENABLE | 1~1500（引用 ADD L2TPGROUP，本范式 `1`） | 一般不配或 FAILOVER | ADD_ENABLE_STRIP_DISABLE + PREFIX（020412 范式） | 域名匹配 L2TP 组接 LNS |
| **AAA 下发方式**（020412 AAA 范式） | ENABLE | 不配 / SYSTEM_GLOBAL_GROUP | 可配 FAILOVER/LOADSHARE | 一般不配 | 鉴权服务器返回 L2TP 属性建隧道 |

> 1 个特性命中（020412）；两种方式下 DOMAINNAMEACT 是关键分叉（PREFIX 域名剥离/PREFIX 域名添加）。

## 约束

- **APN 必须已建**（critical，rule-0-00132）：APN 必须已通过 `ADD APN` 创建 — 后果：APN 不存在命令拒收
- **L2TPSWITCH 必选**（critical，rule-0-00132）：L2TPSWITCH 是必选参数 — 后果：缺 L2TPSWITCH 命令拒收
- **L2TPGROUPID 仅本地方式需配**（warning，rule-0-00132）：L2TPGROUPID 仅在本地方式下需配且必须已通过 `ADD L2TPGROUP` 创建 — 后果：配未建组号命令拒收
- **DOMAINNAMEPOS 条件必选**（critical，rule-0-00132）：DOMAINNAMEPOS 仅在 DOMAINNAMEACT=ADD_ENABLE_STRIP_DISABLE 时为必选 — 后果：缺位置参数命令拒收
- **APN 名称约束**（info，rule-0-00132）：APN 1~63 字符，不支持空格与特殊字符 — 后果：超长或含特殊字符命令拒收
- **适用 NF**（info，rule-0-00132）：本命令仅适用 PGW-U、UPF — 后果：误在 SGW-U 上执行命令拒收
- **执行后立即生效**（warning，rule-0-00132）：本命令执行后立即生效 — 后果：误配即刻影响该 APN 下用户 L2TP 接入

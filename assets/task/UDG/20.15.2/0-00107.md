# 设置APN操作系统级带宽管理开关（SET APNOSLELBWMSW）

> 命令静态知识见 [SET APNOSLELBWMSW](command/UDG/20.15.2/SET-APNOSLELBWMSW.md)

## 配置方法

GWFD-110301 终端系统码率差异化的**末步开关**——开启后该 APN 下用户按终端操作系统(OS Type)差异化执行码率控制。APNNAME 引用 ADD APN 已配置的 APN 名；ISENABLE 直接显式 ENABLE/DISABLE。

**与全局开关形成"全局→APN"两级**：本 APN 级开关承继 SET GLBOSLELBWMSW 全局开关的 IsEnable 配置（全局开关 atom 待后续特性揭示）。

**配置范式**（110301）：

| 参数 | 取值样例 | 来源 | 说明 |
|---|---|---|---|
| APNNAME | apn-test | 已配置数据（ADD APN） | 引用 0-00020 配置的 APN 名 |
| ISENABLE | ENABLE | 本端规划 | 打开 APN 级开关 |

```
SET APNOSLELBWMSW:APNNAME="apn-test",ISENABLE=ENABLE;
```

**步骤位置**：终端系统码率差异化特性末步，在 ADD APNBINDBWMUSRG 之后。

## 约束

- 最大记录数 10000（APN 级） · warning · 超限 SET 失败
- 生效范围限于 APN（与全局 GLBOSLELBWMSW 两级开关） · info
- ISENABLE 设为继承则承继 SET GLBOSLELBWMSW 的 IsEnable 配置 · warning · 全局开关未配则静默失配
- 命令执行后立即生效 · info

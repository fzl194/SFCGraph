# 配置GTP路径管理IP地址（ADD ECHOIPLIST）

> 命令静态知识见 [ADD ECHOIPLIST](command/UDG/20.15.2/ADD-ECHOIPLIST.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

黑白名单成员按"每对端 IP 地址段一条 ADD ECHOIPLIST"逐条下发，最大 200 条。IPv4/IPv6 由 IPVERSION 二选一门控：IPv4 配 IPV4ADDR+IPV4MASK，IPv6 配 IPV6ADDR+IPV6MASK。本命令是 SET UPGTPPATH 的从属命令——前置 SET UPGTPPATH.ECHOLISTSWITCH=ENABLE，名单性质（黑/白）由 SET UPGTPPATH 决定，可 LST UPGTPPATH 查询。

**参数规划表（取值样例来自 GWFD-010102）**

| IPVERSION | 配对参数 | 样例值 | 取值范围 |
|---|---|---|---|
| IPV4 | IPV4ADDR | 192.168.1.10 / 192.168.10.10 / 192.168.100.10 | IPv4 地址 |
| IPV4 | IPV4MASK | 24 | 1~32 |
| IPV6 | IPV6ADDR | fc00::1 | IPv6 地址 |
| IPV6 | IPV6MASK | 64 | 1~128 |

**典型脚本（多条逐行）**
```
ADD ECHOIPLIST:IPVERSION=IPV4,IPV4ADDR="192.168.1.10",IPV4MASK=24;
ADD ECHOIPLIST:IPVERSION=IPV4,IPV4ADDR="192.168.10.10",IPV4MASK=24;
ADD ECHOIPLIST:IPVERSION=IPV4,IPV4ADDR="192.168.100.10",IPV4MASK=24;
ADD ECHOIPLIST:IPVERSION=IPV6,IPV6ADDR="fc00::1",IPV6MASK=64;
```

**配置要诀**
- 一个地址段一条记录，多地址段循环下发，累计不超 200 条。
- 名单类型（黑名单放通其余/白名单仅探测名单内）由前置 SET UPGTPPATH 决定，本命令只填成员。
- 命令立即生效，无需刷新。

## 约束

- **rule-0-00328**（warning）最大记录数 200（每对端 IP 地址段一条）；立即生效；前置 SET UPGTPPATH.ECHOLISTSWITCH=ENABLE 门控；IPVERSION=IPV4 配 IPV4ADDR+IPV4MASK，=IPV6 配 IPV6ADDR+IPV6MASK（二选一） — 违反后果：成员缺失致探测不到对端 / 超 200 条被拒

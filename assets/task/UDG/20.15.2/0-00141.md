# 配置外联口自动部署服务模板（ADD AUTOSCALINGSERVICE）

> 命令静态知识见 [ADD AUTOSCALINGSERVICE](command/UDG/20.15.2/ADD-AUTOSCALINGSERVICE.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法

ADD AUTOSCALINGSERVICE 用于配置外联口自动部署的服务模板，是路由交叉功能（GWFD-020423）网络 B 侧外联口自动化部署的基础。**一个服务模板按地址族独立配置**——IPv4v6 双栈需分别执行该命令两次（各配 IPv4 / IPv6 一次）。**SERVICENAME** 是下游 SRROUTE/BFD/SRBFD 三类模板的引用键。**AUTOCFGIFTYPE** 决定接口类型（SR-IOV bonding 走 ETHTRUNK；非 bonding 走 VNIC）。1 个特性命中（GWFD-020423 路由交叉）。

### 配置维度 1：地址族（参数 AFTYPE，必选）

| 取值 | 配套参数族 | 典型场景 |
|---|---|---|
| **IPv4** | BEGINADDR4/ENDADDR4/MASKLEN | 单栈 IPv4 020423 |
| **IPv6** | BEGINADDR6/ENDADDR6/PREFIXLEN | 单栈 IPv6 020423（双栈需再配一次） |

> 双栈场景需执行命令两次（各配一种 AFTYPE），**SERVICENAME 不变**。

### 配置维度 2：接口封装类型（参数 AUTOCFGIFTYPE，必选）

| 取值 | 联动参数 | 适用组网 |
|---|---|---|
| **ETHTRUNK** | ETHTRUNKTMPID（不填 VNICID） | SR-IOV bonding 组网（020423 范式） |
| **VNIC** | VNICID（不填 ETHTRUNKTMPID） | 非 SR-IOV bonding 组网 |

> ETHTRUNKTMPID 通过 `LST AUTOSCALINGETHTRUNK` 查询获取；VNICID 取值须在设备物理主接口号有效范围。

### 配置维度 3：地址池范围（参数 BEGINADDR*/ENDADDR*/MASKLEN/PREFIXLEN，AFTYPE 配套）

| 取值 | 作用 |
|---|---|
| **BEGINADDR4 ~ ENDADDR4**（IPv4） | 全网规划地址池（参见 5G Core LLD 业务IP对接设计起始/结束IP） |
| **MASKLEN**（IPv4） | 子网掩码长度（按 ISU VM 数量规划，IPv4 路由掩码 0~32） |
| **BEGINADDR6 ~ ENDADDR6**（IPv6） | IPv6 地址池起止（IPv6 路由前缀 0~128） |
| **PREFIXLEN**（IPv6） | IPv6 前缀长度 |

> 地址池范围（BEGINADDR*/ENDADDR*）不超过 **2048**。

**典型脚本（SR-IOV bonding + IPv4 单栈 020423 范式）**：

```
ADD AUTOSCALINGSERVICE: SERVICENAME="ServName_networkB", VPNNAME="VRF_networkB",
  AFTYPE=IPv4, IPALLOCTYPE4=USER_CONFIG, AUTOCFGIFTYPE=ETHTRUNK, ETHTRUNKTMPID=6,
  VLANID=20, BEGINADDR4="192.168.20.2", ENDADDR4="192.168.20.126", MASKLEN=25;
```

**步骤位置**：在所属 feature 编排中通常位于"步骤 1：建外联口服务模板"——前置 [0-00071](task/UDG/20.15.2/0-00071.md) ADD L3VPNINST 建 VRF_networkB；后续按 AUTOCFGIFTYPE 联动 ETHTRUNK/VNIC，下游 [0-00142](task/UDG/20.15.2/0-00142.md)/[0-00143](task/UDG/20.15.2/0-00143.md)/[0-00144](task/UDG/20.15.2/0-00144.md) 引用本任务的 SERVICENAME。

## 决策点：组网封装类型（DP 0-00141）

| 选项 | AUTOCFGIFTYPE | 联动参数 | 适用场景 |
|---|---|---|---|
| **SR-IOV bonding 组网**（020423 范式） | ETHTRUNK | ETHTRUNKTMPID（必填） | bonding 物理接口聚合场景 |
| **非 bonding 组网** | VNIC | VNICID（必填） | 普通 VNIC 场景 |

## 约束

- **最大记录数 4096**（warning，rule-0-00141）：系统最多 4096 条 AUTOSCALINGSERVICE — 后果：超规格新增失败
- **地址池范围上限 2048**（warning，rule-0-00141）：BEGINADDR*/ENDADDR* 范围不超过 2048 — 后果：超上限新增失败
- **VNICID 物理接口有效范围**（warning，rule-0-00141）：AUTOCFGIFTYPE=VNIC 时 VNICID 须在设备物理主接口号有效范围 — 后果：越界命令拒收
- **自动化配置开关先关闭**（critical，rule-0-00141）：执行前须先 `SET AUTOCONFIG 关闭`，执行后再开启 — 后果：未关开关命令拒收
- **立即生效**（warning，rule-0-00141）：执行后立即生效 — 后果：误配即刻影响自动化部署
- **VPNNAME 引用前置**（critical，rule-0-00141）：VPNNAME 须已通过 ADD L3VPNINST(0-00071) 建出 — 后果：VRF 未建命令拒收

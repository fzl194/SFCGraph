# 配置地址池IP地址段（ADD SECTION）

> 命令静态知识见 [ADD SECTION](command/UDG/20.15.2/ADD-SECTION.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

ADD SECTION 给上游 ADD POOL 建好的地址池（POOLNAME）添加地址段。**POOLNAME 引用上游 0-00043 的 POOLNAME**（`LST POOL` 可查）。一个地址池最多 64 段（SECTIONNUM 0~63），双栈池分别对 IPv4/IPv6 各执行一次本命令。

IPVERSION 决定本段配置哪一组起止参数（见决策点），起止地址全网规划、全网唯一地址空间。

**按 IPVERSION 的地址段范式**

| IPVERSION | 必配参数 | 典型特性 |
|---|---|---|
| IPV4 | V4STARTIP + V4ENDIP（IPv4 起止，起≤止） | 地址分配方式、双栈 IPv4 段、路由冗余 IPv4 段、入不转板 |
| IPV6 | V6PREFIXSTART + V6PREFIXEND + V6PREFIXLENGTH（前缀起止 + 长度 49~64） | 双栈 IPv6 段、路由冗余 IPv6 段、IPv6 Prefix Delegation |

**MASK 仅入不转板必配**：020482 场景 V4STARTIP/V4ENDIP 是完整子网起止（如 10.10.1.0~10.10.127.255），同时配 MASK（5~15）按 ISU POD 数量子段化；普通地址分配场景不配 MASK。

**典型行**

IPv4 段（010104 地址分配方式）：
```
ADD SECTION:POOLNAME="testpool",SECTIONNUM=1,IPVERSION=IPV4,V4STARTIP="10.10.1.1",V4ENDIP="10.10.1.10";
```
IPv6 Prefix Delegation 段（020406，V6PREFIXLENGTH=63 < 64 表示 PD 方式）：
```
ADD SECTION:POOLNAME="testpool",SECTIONNUM=1,IPVERSION=IPV6,V6PREFIXSTART="fc00:0000:0000:fcee:0000:0000:0000:0000",V6PREFIXEND="fc00:0000:0000:fcef:ffff:ffff:ffff:ffff",V6PREFIXLENGTH=63;
```
入不转板（020482，配 MASK=9 按 ISU POD 切子段）：
```
ADD SECTION:POOLNAME="testpool",SECTIONNUM=1,IPVERSION=IPV4,V4STARTIP="10.10.1.0",V4ENDIP="10.10.127.255",MASK=9;
```

**静态路由冗余特殊点**（010107）：主备 UDG 规划**相同的地址段**（如 pool1 IPv4 段与 pool3 IPv4 段都是 10.1.1.0~10.1.1.225），且地址段需包含 SMF 上配置的静态地址段；同一池内 IPv4 段和 IPv6 段 SECTIONNUM 递增（1/2/3/4）。

**IPv6 前缀长度语义**：V6PREFIXLENGTH=64 为标准 IPv6 段；<64（如 63）表示采用 IPv6 Prefix Delegation 方式分配，由外部网元委托前缀。

## 决策点：地址段 IP 地址类型（DP 0-00062）

问：本段 IPVERSION 取 IPV4 还是 IPV6？

| 选项 | 联动 |
|---|---|
| IPV4（默认） | requires V4STARTIP + V4ENDIP；excludes V6PREFIX* 参数 |
| IPV6 | requires V6PREFIXSTART + V6PREFIXEND + V6PREFIXLENGTH；excludes V4* 参数 |

依据：5 特性数据规划表 IPVERSION 取值分叉——IPV4（010104/010107 IPv4 段/020403 IPv4 段/020482）vs IPV6（010107 IPv6 段/020403 IPv6 段/020406）。双栈场景一池多次执行本命令分别配 IPv4/IPv6 段。

## 约束

- **rule-0-00142**（warning）最大记录数 40000（>98% 上报 ALM-135602215，阈值可 MOD CFGTHRESHOLD 改）；单地址池最多 64 段（SECTIONNUM 0~63）；起止不能倒置（V4STARTIP ≤ V4ENDIP / V6PREFIXSTART ≤ V6PREFIXEND）；本地 IPv4 段有容量上限；执行后立即生效 — 后果：超规格/超 64 段/起止倒置均新增失败

# 配置用户模板URR组绑定（SET URRGRPBINDING）

> 命令静态知识见 [SET URRGRPBINDING](command/UDG/20.15.2/SET-URRGRPBINDING.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

同一命令承载两种用途，按场景选参数子集。USERPROFILENAME 必填（reference），其余参数子集随用途分叉。

**用途一：缺省费率绑定（计费场景，2-00001 内容计费等）**——同时绑业务 + 信令 URR 组：
```
SET URRGRPBINDING: USERPROFILENAME="userprofile001", DFTURRGRPNAME="UrrGp_any",     DFTSIGURRGNAME="UrrGp_abnormal";
SET URRGRPBINDING: USERPROFILENAME="userprofile002", DFTURRGRPNAME="UrrGp_IMS",      DFTSIGURRGNAME="UrrGp_abnormal";
```

**用途二：缺省费率 + 继承全局缺省 URR 组（基于时长/流量/事件计费，2-00009/10/11）**——在用途一基础上加全局开关：
```
SET URRGRPBINDING: USERPROFILENAME="userprofile001", DFTURRGRPNAME="UrrGp_any", DFTSIGURRGNAME="UrrGp_abnormal",
                   GLBDFTURRGSW=ENABLE, GLBDFTSIGURRGSW=ENABLE;
```

**用途三：TCP 重传计费（GWFD-020307 激活 TCP 重传识别）**——固定标识 + TCP 重传 URR 组：
```
SET URRGRPBINDING: USERPROFILENAME="userprofile001", TCPCHGPROFLAG=CHARGING, TCPURRGRPNAME="cp_tcp";
```

**参数子集归属**：

| 参数 | 用途一 | 用途二 | 用途三 | 来源 |
|---|---|---|---|---|
| USERPROFILENAME | ✓ | ✓ | ✓ | 引 [0-00011](0-00011.md) |
| DFTURRGRPNAME | ✓ | ✓ | — | 引 [0-00002](0-00002.md) URRGROUP |
| DFTSIGURRGNAME | ✓ | ✓ | — | 引 [0-00002](0-00002.md) URRGROUP |
| GLBDFTURRGSW | — | ENABLE | — | 本端规划 |
| GLBDFTSIGURRGSW | — | ENABLE | — | 本端规划 |
| TCPCHGPROFLAG | — | — | CHARGING（fixed） | — |
| TCPURRGRPNAME | — | — | ✓ | 引 [0-00002](0-00002.md) URRGROUP |

## 约束

- **rule-0-00113**（warning）建议同时配 DFTURRGRPNAME + DFTSIGURRGNAME，否则使用该 UP 的用户部分流量无法计费。例外：仅在线计费且 UP 未绑任何 Rule 时，数据包按全局 URR 组计费，否则不计费。**立即生效**。

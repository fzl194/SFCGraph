# 切片绑定

> BWM 配置的场景特化阶段（仅切片场景）。把切片作为接入域，使切片下用户受 BWM 用户组规则管控。归属特性 [2-00001](task/UDG/20.15.2/2-00001.md)。

## 配置方法

切片三步链：切片标识 → 接口绑定 → 用户组绑定，并对齐全局开关使切片用 BWM Rule。

| 步骤 | 命令 | 关键参数 |
|---|---|---|
| 切片标识 | ADD SNSSAI | SST（切片类型）+ SD（区分符，可选） |
| 接口绑定 | ADD SNSSAIUPINTF | SST+SD + N3LOGICINF（N3 逻辑接口） |
| 用户组绑定 | ADD SNSSAIBWMUSRG | USERGROUPNAME + SST+SD |
| 全局开关（归属 [1-00003](task/UDG/20.15.2/1-00003.md) 的 0-00106） | SET BANDWIDTHMNG | 切片增量：BWMSUBSCRCTRL=PCC_RULE_IGNORE（关键） |

**典型脚本**：
```
ADD SNSSAI:SST=1,SD="123456";
ADD SNSSAIUPINTF:SST=1,SD="123456",N3LOGICINF="n3if1/1/1";
ADD SNSSAIBWMUSRG:USERGROUPNAME="ug_test",SST=1,SD="123456";
SET BANDWIDTHMNG:RANGE=SYSTEM,CIRMINTHRES=300000,PIRMINTHRES=500000,BWMSUBSCRCTRL=PCC_RULE_IGNORE;
```

**步骤位置**：替代 APN 作为接入域（与 [1-00001](task/UDG/20.15.2/1-00001.md) 的 APN 路径并列）；承接 [1-00002](task/UDG/20.15.2/1-00002.md) 控制器，配合 [1-00003](task/UDG/20.15.2/1-00003.md) 用户组规则。

## 决策点

| 选项 | 影响（参数/命令/联动） |
|---|---|
| 切片标识格式—双字段（SST+SD） | SD 传入，切片按 SST+SD 精确区分 |
| 切片标识格式—单字段（仅 SST） | SD 不传（缺省则全 F），切片仅按 SST 区分 |

## 约束

- **BWMSUBSCRCTRL=PCC_RULE_IGNORE**（critical）：切片场景必须 — 否则切片走 PCC Rule 而非 BWM Rule，控制不生效。
- **高危命令**（warning）：SNSSAIUPINTF / SNSSAIBWMUSRG 影响新 PFCP 会话 — 误配可能中断新建会话。
- **多用户组优先级**（info）：多用户组绑定切片时，仅最高优先级组生效 — 低优先级组不生效。

# 配置全局带宽管理规则（ADD BWMRULEGLOBAL）

> 命令静态知识见 [ADD BWMRULEGLOBAL](command/UDG/20.15.2/ADD-BWMRULEGLOBAL.md)

## 配置方法

把已建的 BWMSERVICE + BWMCONTROLLER（上下行）组装成一条**整机级**规则，对整机所有用户统一限速。无需指定用户组/接入类型/漫游类型；必须指定上下行或双向控制器之一 + 业务 + 优先级。与 ADD BWMRULE 同属 BwmRule 对象，共享 5000 记录配额。

| 参数 | 取值样例 | 说明 |
|---|---|---|
| BWMRULENAME | `br_global_test` | 必选，1~31 字符，不支持空格 |
| BWMSERVICENAME | `bs_group_test` | 必选，引用 BWMSERVICE；**仅支持 TOS 业务** |
| UPBWMCTRLNAME1 | `bc-global-up` | 可选，上行控制器，引用 BWMCONTROLLER |
| DNBWMCTRLNAME1 | `bc-global-down` | 可选，下行控制器，引用 BWMCONTROLLER |
| BWMRULEPRI | `1` | 必选，1~32768，全局规则间唯一，值越小优先级越高 |
| TIMERANGENAME1~4 | （可选） | 可选，引用 TIMERANGE；配 1 组以上控制器时必须指定时段且 ≥30 分钟 |
| UPBWMCTRLNAME2~4 / DNBWMCTRLNAME2~4 | （可选） | 多时段扩展，每时段一组上下行控制器 |
| SERVICELEVEL | `1`（默认） | 可选，1~30，同控制器不同业务流间优先级 |

**典型脚本**：

```
// 整机级规则：业务 bs_group_test，上下行分别绑整机控制器
ADD BWMRULEGLOBAL:BWMRULENAME="br_global_test",BWMSERVICENAME="bs_group_test",UPBWMCTRLNAME1="bc-global-up",DNBWMCTRLNAME1="bc-global-down",BWMRULEPRI=1;
```

**步骤位置**：三级带宽控制（用户/用户组/整机）的最后一步——在 BWM 三件套（BWMSERVICE/BWMCONTROLLER）就绪后执行；整机规则独立于用户组（不依赖 BWMUSERGROUP），是第三级"整机兜底"。

## 决策点

**DP 0-00289**：规则级别分叉——整机级（ADD BWMRULEGLOBAL）vs 用户级/用户组级（ADD BWMRULE）。两者同属 BwmRule 对象、共享配额，但适用范围和约束不同。

| 选项 | 命令 | 范围 | 业务类型 | 用户属性 |
|---|---|---|---|---|
| 整机级 | ADD BWMRULEGLOBAL | 整机所有用户 | **仅 TOS** | 不指定（接入/漫游类型均不传） |
| 用户级/用户组级 | ADD BWMRULE | 特定用户组下的用户 | TOS / NONTOS | 必传 USERGROUPNAME |

整机级只能配 TOS 业务，绑非 TOS 业务会失败；所有全局规则的 BWMSERVICENAME 的 TOS 值不能相同。

## 约束

- 最大记录数 8（全局规则自身配额） · warning · 超规格新增失败
- 与 ADD BWMRULE 共享 BwmRule 对象 5000 总配额 · warning · 跨命令联合计数
- 命令执行后立即生效 · info · 无需刷新，LST BWMRULEGLOBAL 查看
- 仅支持 TOS 业务，绑非 TOS 业务失败 · error · 业务类型限制
- 所有全局规则的 BWMSERVICENAME 的 TOS 值不能相同 · error · TOS 唯一性
- 必须指定上下行或双向控制器之一 + 业务 + 优先级 · error · 必备项
- 全局规则优先级（BWMRULEPRI）不能相同，值越小优先级越高 · warning · 优先级冲突
- 配 1 组以上控制器时必须指定 TIMERANGENAME 且时段 ≥30 分钟 · error · 多时段约束
- BwmRuleGlobal 不支持 Shaping 控制类型的 BwmController · error · 控制器类型限制
- 用户激活/更新时若所属用户组无用户级/组级规则，后续不再匹配带宽规则（含全局） · warning · 匹配中止
- SERVICELEVEL 用户级 ≤5，用户组级/全局 ≤30，且不超过 BwmController 的 SrvLevelSpec · error · 上限校验

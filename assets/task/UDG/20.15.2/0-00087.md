# 配置IPFarm（ADD IPFARM）

> 命令静态知识见 [ADD IPFARM](command/UDG/20.15.2/ADD-IPFARM.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

创建一个 IPFarm 实例（server 池容器），核心是 `IPFARMNAME` + `IPVERSION`（必选），再按 SERVERTYPE 类型选择性配心跳检测接口/虚拟 IP/VPN。多 server 负载均衡通过在 0-00088 ADD IPFARMSERVER 重复增加实现，本命令只建池。

**两套范式（按 SERVERTYPE 分叉，与 0-00086 共享 DP 0-00150）**

| SERVERTYPE | 必配 | 可选 | 典型场景 |
|---|---|---|---|
| REDIRECT（默认） | IPFARMNAME, IPVERSION, INTERFACENAME | HEALTHCHECKFLAG(=ENABLE), VIRTUALIP | 用户Portal(110281)/Web Proxy(110282) 重定向 |
| PCSCF | IPFARMNAME, IPVERSION, SERVERTYPE, VPNINSTANCE, VIRTUALIP | HEALTHCHECKFLAG(=ENABLE), INTERFACENAME | P-CSCF 故障 IMS 恢复(020253) |

差异点：PCSCF 场景须显式列 SERVERTYPE=PCSCF、绑 VPN(ims_vpn)、配虚拟 IP；REDIRECT 场景 SERVERTYPE 默认值即 REDIRECT 可省略。

**取值样例**

| 参数 | REDIRECT（farm_test） | PCSCF（farm_test / farm_test2） |
|---|---|---|
| IPFARMNAME | farm_test | farm_test / farm_test2 |
| IPVERSION | IPV4 | IPV4 |
| SERVERTYPE | （省略，默认 REDIRECT） | PCSCF |
| VPNINSTANCE | （不配） | ims_vpn（引用 0-00041 ADD VPNINST） |
| HEALTHCHECKFLAG | ENABLE | ENABLE |
| VIRTUALIP | （不配） | 10.1.1.1 / 10.2.2.2 |
| INTERFACENAME | phif1/0/0（引用 0-00083 ADD LOGICINF） | phif1/0/0（引用 0-00083 ADD LOGICINF） |

**引用关系**

- `INTERFACENAME` → [0-00083](task/UDG/20.15.2/0-00083.md):NAME（心跳检测逻辑接口，ADD LOGICINF 配置）
- `VPNINSTANCE` → [0-00041](task/UDG/20.15.2/0-00041.md):VPNINSTANCE（VPN 实例，ADD VPNINST 配置）

**脚本**

REDIRECT 场景（用户Portal/Web Proxy）：
```
ADD IPFARM:IPFARMNAME="farm_test",IPVERSION=IPV4,HEALTHCHECKFLAG=ENABLE,INTERFACENAME="phif1/0/0";
```

PCSCF 场景（P-CSCF 故障恢复，建两个 farm 做冗余）：
```
ADD IPFARM:IPFARMNAME="farm_test",IPVERSION=IPV4,SERVERTYPE=PCSCF,VPNINSTANCE="ims_vpn",VIRTUALIP="10.1.1.1",HEALTHCHECKFLAG=ENABLE,INTERFACENAME="phif1/0/0";
ADD IPFARM:IPFARMNAME="farm_test2",IPVERSION=IPV4,SERVERTYPE=PCSCF,VPNINSTANCE="ims_vpn",VIRTUALIP="10.2.2.2",HEALTHCHECKFLAG=ENABLE,INTERFACENAME="phif1/0/0";
```

## 决策点

**DP 0-00150：SERVERTYPE 跨特性取值**（与 0-00086 SET IPFARMGLOBAL 共享，详见 0-00086 决策点）。REDIRECT/PCSCF 分叉决定本命令配哪些可选参数。

## 约束

- 最大记录数 3600；>90% 规格上报 ALM-135602215（阈值可 MOD CFGTHRESHOLD 修改）；立即生效。
- IPFARMNAME 全局唯一；VIRTUALIP 与 IPFarm 一一对应（不允许同一 virtual IP 配到不同 IPFarm 上）。
- 不同 IPFarm 下的服务器须配不同心跳检测接口。
- 前置依赖：HEALTHCHECKFLAG=ENABLE 时须先 0-00083 ADD LOGICINF 创建心跳检测接口；VPNINSTANCE 引用时须先 0-00041 ADD VPNINST。

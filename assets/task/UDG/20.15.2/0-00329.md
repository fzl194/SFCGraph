# 查询地址池组（LST POOLGROUP）

> 命令静态知识见 [LST POOLGROUP](command/UDG/20.15.2/LST-POOLGROUP.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法（配置方法字典）

LST POOLGROUP 是地址池族**地址池组查询**命令，输出 POOLGRPNAME / IPV4ALLOCPRIALG / IPV6ALLOCPRIALG 等配置字段。命令核心配置空间有 2 个维度：**查询范围**（POOLGRPNAME 可选：指定单个/查全部）、**详细度**（VERBOSE 可选）。

### 配置维度 1：查询范围（参数 POOLGRPNAME，可选）

| 取值 | 行为 |
|---|---|
| 指定 POOLGRPNAME | 仅查该地址池组配置 |
| 不配 | 查所有地址池组配置 |

### 配置维度 2：详细度（参数 VERBOSE，可选）

| 取值 | 行为 |
|---|---|
| ENABLE | 输出完整字段 |
| 不配 | 输出关键字段 |

**典型脚本（GWFD-010105 调测：查所有地址池组配置）**：
```
LST POOLGROUP:;
```

**典型脚本（指定 POOLGRPNAME 查询）**：
```
LST POOLGROUP:POOLGRPNAME="poolgroup1";
```

> 命令 notes：适用 NF：PGW-U、UPF；查询输出用于配置核查。

**步骤位置**：在所属 feature 编排中通常位于"调测/验证阶段"——前置条件：POOLGROUP 已建；用于 POOLGROUP 配置核查。

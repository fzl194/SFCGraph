# 配置GRE隧道（ADD GRETUNNEL）

> 命令静态知识见 [ADD GRETUNNEL](command/UDG/20.15.2/ADD-GRETUNNEL.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

ADD GRETUNNEL 建 GRE 隧道。公共必填：TNLNAME（隧道名，引用 [ADD INTERFACE](task/UDG/20.15.2/0-00056.md) 的 Tunnel IFNAME）、TNLTYPE=gre、SRCTYPE=if_name、SRCIFNAME（本端 LoopBack IFNAME）、DSTIPADDR（对端 LoopBack IP）。开关类参数随场景分叉。

**3 种场景范式（从证据数据规划表归纳）**

| 场景 | REDUNDANCYEN | KEEPALVEN | DSTVPNNAME | 校验/Key/统计 | 典型行 |
|---|---|---|---|---|---|
| 冗余备份（010107） | TRUE | TRUE | 必填（L3VPNINST/VPNINST） | 不配 | `ADD GRETUNNEL:TNLNAME="Tunnel1",SRCTYPE=if_name,DSTIPADDR="10.10.0.1",TNLTYPE=gre,SRCIFNAME="LoopBack1",DSTVPNNAME="vpn2",REDUNDANCYEN=TRUE,KEEPALVEN=TRUE;` |
| 基本GRE（015002） | 不配(默认FALSE) | 不配(默认FALSE) | 不配 | 可选(由 MOD GRETUNNEL 后续使能) | `ADD GRETUNNEL:TNLNAME="Tunnel1",TNLTYPE=gre,SRCTYPE=if_name,SRCIFNAME="LoopBack1",DSTIPADDR="10.4.4.11";` |
| GRE over IPsec（015004） | 不配 | FALSE | 不配 | 全 FALSE | `ADD GRETUNNEL:TNLNAME="Tunnel1",SRCTYPE=if_name,DSTIPADDR="10.102.105.224",KEEPALVEN=FALSE,TNLTYPE=gre,SRCIFNAME="LoopBack1",GREKEYEN=FALSE,CHECKSUMEN=FALSE,STATENABLE=FALSE;` |

**TNLTYPE 固定 gre，SRCTYPE 固定 if_name**（源用接口名而非 IP）。DSTIPADDR=对端 LoopBack 的 IFIPADDR，须与 [ADD IFIPV4ADDRESS](task/UDG/20.15.2/0-00058.md) 协同。

**校验/关键字增强**（CHECKSUMEN/GREKEYEN）通常不在 ADD 时配，而由 MOD GRETUNNEL 后续使能；两端 Key 必须相同。

## 决策点：GRE隧道场景选择（DP 0-00074）

问：该 GRE 隧道用于哪种场景？（决定 REDUNDANCYEN/KEEPALVEN/DSTVPNNAME/校验类开关取值）

| 选项 | REDUNDANCYEN | KEEPALVEN | DSTVPNNAME | CHECKSUMEN/GREKEYEN | STATENABLE |
|---|---|---|---|---|---|
| 静态地址用户路由冗余备份（010107，默认） | TRUE | TRUE | 必填 | 不配 | 不配 |
| 基本GRE隧道（015002） | 不配(F) | 不配(F) | 不配 | 可选 TRUE（MOD 后续使能） | 不配 |
| GRE over IPsec（015004） | 不配(F) | FALSE | 不配 | FALSE | FALSE |

默认冗余备份（与现有 binding 默认值一致）。

## 约束

- **rule 0-00180**（capacity）命令最大记录数 4096 条 GRE 隧道 — 后果：超 4096 新增失败

# 配置地址池（ADD POOL）

> 命令静态知识见 [ADD POOL](command/UDG/20.15.2/ADD-POOL.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

ADD POOL 是地址池族链路起点：先建地址池（POOLNAME），再由 ADD SECTION 配地址段、ADD POOLBINDGROUP 绑到地址池组、ADD POOLGRPMAP 建 APN 映射。**POOLNAME 一旦建好，下游 4 条命令都引用它**，需全网唯一。

POOLTYPE 决定地址分配来源（见决策点），HASVPN+VPNINSTANCE 三者一致性约束（APN / 地址池 / SGi-N6 外联口同 VPN）。

**POOLNAME 命名约定**：双栈场景 IPv4/IPv6 各建一个池，建议 `testpool` / `testpoolv6` 分栈后缀；静态路由冗余主备各建 IPv4+IPv6 四个池（`pool1`~`pool4`）。

**按 POOLTYPE 的配置范式（数据规划表归纳）**

| 范式 | POOLTYPE | 特征参数 | 典型特性 |
|---|---|---|---|
| 外部地址分配 | EXTERNAL | 地址由 RADIUS/BRAS/DHCP 或静态分配，UDG 仅发布路由；可选 CHECKIPVALID=ENABLE 开静态地址白名单检测 | 地址分配方式、静态路由冗余、IPv6 Prefix Delegation |
| 本地地址池 | LOCAL | UDG 自维护地址段、为用户分配本地动态地址；需配套 ADD SECTION 配地址段 | IPv4v6 双栈、入不转板、基于 APN/位置分配 |

**典型行**

EXTERNAL 外部分配（010104 地址分配方式）：
```
ADD POOL:POOLNAME="testpool",POOLTYPE=EXTERNAL,HASVPN=ENABLE,VPNINSTANCE="vpn1";
```
EXTERNAL + 静态白名单（020406 Prefix Delegation，IPV6 段）：
```
ADD POOL:POOLNAME="testpool",POOLTYPE=EXTERNAL,IPVERSION=IPV6,CHECKIPVALID=ENABLE,HASVPN=ENABLE,VPNINSTANCE="vpn1";
```
EXTERNAL + 路由冗余（010107，IPv4/IPv6 主备各一组，REDUNDFUNC=ENABLE，主用 MASTERFLAG=ENABLE）：
```
ADD POOL: POOLNAME="pool1", POOLTYPE=EXTERNAL, IPVERSION=IPV4, HASVPN=ENABLE, VPNINSTANCE="vpn1", REDUNDFUNC=ENABLE;
```
LOCAL 本地双栈（020403，IPv4/IPv6 各一池）：
```
ADD POOL:POOLNAME="testpool",POOLTYPE=LOCAL,IPVERSION=IPV4,HASVPN=ENABLE,VPNINSTANCE="VRF_Internet";
ADD POOL:POOLNAME="testpoolv6",POOLTYPE=LOCAL,IPVERSION=IPV6,HASVPN=ENABLE,VPNINSTANCE="VRF_Internet";
```

**VPN 一致性**：HASVPN=ENABLE 时 VPNINSTANCE 必填，用 `LST VPNINST` 查询已配值；该 VPN 必须与同 APN、SGi-N6 外联口三者一致。

**未绑冷门可选**：IPLEASE / RELEASETIME / SINGLEIPFLAG / ALARMSWITCH / IMSSW 五参数任一特性数据规划表/脚本均未用到，按通用配置门槛不绑（RELEASETIME 在 LOCAL 池过大为高危项，见约束）。

## 决策点：地址池类型（DP 0-00061）

问：POOLTYPE 取 EXTERNAL 还是 LOCAL？

| 选项 | 联动 |
|---|---|
| EXTERNAL 外部地址分配（默认） | POOLTYPE=EXTERNAL；可选 CHECKIPVALID=ENABLE（静态白名单）/ REDUNDFUNC=ENABLE（路由冗余，需配 SET REDUNDUSER 全局开关） |
| LOCAL 本地地址池 | POOLTYPE=LOCAL；需配套 ADD SECTION 提供地址段（IPv4 V4* / IPv6 V6PREFIX*，由 ADD POOL.IPVERSION 决定） |

依据：5 特性数据规划表 POOLTYPE 取值分叉——EXTERNAL（010104/010107/020406）vs LOCAL（020403/020482）。

## 约束

- **rule-0-00141**（warning）最大记录数 40000，>98% 上报 ALM-135602215（≤95% 恢复，阈值可 MOD CFGTHRESHOLD 改）；POOLTYPE=LOCAL 且 RELEASETIME 过大为高危（地址延迟释放→池地址不足→激活失败）；APN 已绑特定 VPN 时本池必须绑同一 VPN（APN/地址池/SGi-N6 三者 VPN 一致）；执行后立即生效 — 后果：超规格新增失败 / LOCAL 池地址不足致激活失败 / VPN 不一致致地址分配或路由异常

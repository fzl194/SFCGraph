# BWM 用户组、规则与绑定

> BWM 配置的通用骨架收尾阶段（所有场景共享）。承接 [1-00002](task/UDG/20.15.2/1-00002.md) 的控制器/业务，产出可生效的带宽控制规则。归属特性 [2-00001](task/UDG/20.15.2/2-00001.md)。

## 配置方法

用户组（定义控制对象集合）→ 规则（绑 Service+Controller，类型由场景决定）→ 绑定接入域 → 生效范围与刷新。

| 步骤 | 命令 | 关键参数 |
|---|---|---|
| 用户组 | `ADD BWMUSERGROUP` | USERGROUPTYPE=`SPECIFIC_GROUP`（全特性统一）；USERGROUPNAME=`ug_test`（全网唯一）；USERGROUPPRI=`1`(组级)/`2`(用户级)（同级唯一）；USERLEVSRVTYPE=`NONTOS`/`TOS`（用户级路径，与 USERGLEVSRVTYPE 互斥）；USERGLEVSRVTYPE=`NONTOS`/`TOS`（用户组级路径） |
| 规则—SUBSCRIBER_SPECIFIC（用户级） | `ADD BWMRULE` | BWMRULETYPE=SUBSCRIBER_SPECIFIC（典型：用户 TOS / HTTP / PCC）；关联 BWMSERVICE=是；UPBWMCTRLNAME1/DNBWMCTRLNAME1 指控制器 |
| 规则—GROUP_SPECIFIC（用户组级） | `ADD BWMRULE` | BWMRULETYPE=GROUP_SPECIFIC（典型：切片 / 时间段 / 用户组 TOS / 分级）；关联 BWMSERVICE=是；UPBWMCTRLNAME1/DNBWMCTRLNAME1 指控制器 |
| 规则—GROUP_DEFAULT（用户组缺省） | `ADD BWMRULE` | BWMRULETYPE=GROUP_DEFAULT（典型：APN/DNN）；关联 BWMSERVICE=否；UPBWMCTRLNAME1/DNBWMCTRLNAME1 指控制器 |
| 绑定—APN 接入域 | `ADD APNBINDBWMUSRG` | USERGROUPNAME + APN |
| 绑定—切片接入域 | `ADD SNSSAIBWMUSRG` | USERGROUPNAME + SST+SD（见 [1-00005](task/UDG/20.15.2/1-00005.md)） |
| 生效范围（用户组级场景必配：APN/DNN、切片、分级） | `SET BANDWIDTHMNG` | RANGE=SYSTEM；切片须 BWMSUBSCRCTRL=PCC_RULE_IGNORE；CIRMINTHRES/PIRMINTHRES 阈值 |
| 刷新（涉及 Rule/L7Filter，PCC 场景） | `SET REFRESHSRV` | REFRESHTYPE=USERPROFILE |

**典型脚本（APN/DNN 用户组级）**：
```
ADD BWMUSERGROUP:USERGROUPTYPE=SPECIFIC_GROUP,USERGROUPNAME="ug_test",USERGROUPPRI=1,USERGLEVSRVTYPE=NONTOS;
ADD BWMRULE:USERGROUPTYPE=SPECIFIC_GROUP,USERGROUPNAME="ug_test",BWMRULETYPE=GROUP_DEFAULT,UPBWMCTRLNAME1="bc-up",DNBWMCTRLNAME1="bc-down";
ADD APNBINDBWMUSRG:USERGROUPNAME="ug_test",APN="apn-test";
```

**步骤位置**：承接 [1-00002](task/UDG/20.15.2/1-00002.md)（引用 BWMCNAME/BWMSERVICENAME）；BWM 主流程收尾。

## 场景差异（各场景对本步骤的参数增量）

- **分级带宽**：按 SERVICELEVEL 数量配**多条 BWMRULE**（每条绑对应 BWMSERVICE，如 3 业务→3 BWMSERVICE+3 BWMRULE，各带 SERVICELEVEL=1/2/3）；BWMUSERGROUP 加 `USERGLEVRULESW=ENABLE`。
- **三级控制**：BWMUSERGROUP 双业务类型 `USERLEVSRVTYPE=NONTOS,USERGLEVSRVTYPE=TOS`（用户级 NONTOS、组级 TOS）；配用户级（SUBSCRIBER_SPECIFIC）+ 用户组级（GROUP_SPECIFIC）两条 BWMRULE；整机级追加 [0-00289](task/UDG/20.15.2/0-00289.md)（BWMRULEGLOBAL，整机 BWMSERVICE 必须 TOS）。
- **切片**：`SET BANDWIDTHMNG` 须 `BWMSUBSCRCTRL=PCC_RULE_IGNORE`；绑定走切片（SNSSAIBWMUSRG）而非 APN。
- **APN/DNN 用户组级**：BWMRULETYPE=GROUP_DEFAULT，不关联 BWMSERVICE。

## 决策点

| 选项 | 影响（参数/命令/联动） |
|---|---|
| BWMRULETYPE=SUBSCRIBER_SPECIFIC（联动特性级场景 DP） | 用户级；关联 BWMSERVICE；典型 用户 TOS/HTTP/PCC |
| BWMRULETYPE=GROUP_SPECIFIC（联动特性级场景 DP） | 用户组级；关联 BWMSERVICE；典型 切片/时间段/用户组 TOS/分级 |
| BWMRULETYPE=GROUP_DEFAULT（联动特性级场景 DP） | 用户组缺省；不关联 BWMSERVICE；典型 APN/DNN |
| 业务类型 TOS/NONTOS（联动场景） | USERLEVSRVTYPE/USERGLEVSRVTYPE，与 [1-00002](task/UDG/20.15.2/1-00002.md) 的 BWMSERVICE 识别方式联动 |

> 三级类型 BWMRULETYPE 由 [2-00001](task/UDG/20.15.2/2-00001.md) 场景 DP 决定。

## 约束

- **APN/DNN 用户组级限定**（warning）：仅 `GROUP_DEFAULT` + `NONTOS`，不关联 BWMSERVICE — 违反则与缺省语义冲突。
- **切片 BWMSUBSCRCTRL**（critical）：`SET BANDWIDTHMNG` 必须 `BWMSUBSCRCTRL=PCC_RULE_IGNORE` — 否则切片走 PCC Rule 而非 BWM Rule，控制不生效。
- **新增用户组对在线用户不生效**（warning）：需重激活 — 否则已在线用户不受新组管控。
- **USERGROUPPRI 同级唯一**（warning）：同级 `USERGROUPPRI` 不能重复 — 重复则优先级歧义。

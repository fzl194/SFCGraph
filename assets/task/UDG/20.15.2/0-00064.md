# 配置静态地址用户路由冗余全局开关（SET REDUNDUSER）

> 命令静态知识见 [SET REDUNDUSER](command/UDG/20.15.2/SET-REDUNDUSER.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

SET REDUNDUSER 是激活"静态地址用户路由冗余"特性的**总开关**。SWITCH 固定 ENABLE（数据规划表标"固定取值"）。本 atom 属特性链路**最后一步**——所有冗余相关配置完成后才使能。

**配置范式**

| 参数 | 取值 | 来源 | 说明 |
|---|---|---|---|
| SWITCH | ENABLE | 固定取值 | 使能全局开关 |

```
SET REDUNDUSER: SWITCH=ENABLE;
```

**主备 UDG 均需配置**（备用 UDG 重复主用的全部步骤）。配置前必须先确认 APN 下**无正在使用或准备使用静态路由冗余的用户**，否则此部分用户业务不通。本开关须与地址池 `REDUNDFUNC=ENABLE`（ADD POOL 参数）**同时打开**才生效。系统初始记录 SWITCH=DISABLE。

## 约束

- **rule 0-00330**（cardinality，critical）最大记录数为 1（全局唯一）；**高危命令**，改前须确认无静态地址冗余用户，否则用户业务不通；执行后立即生效；此开关须与地址池 REDUNDFUNC=ENABLE（[0-00043](task/UDG/20.15.2/0-00043.md) ADD POOL 参数）同时打开才生效；主备 UDG 均需配置；存在系统初始记录 SWITCH=DISABLE — 后果：高危配置错误致用户业务不通 / REDUNDFUNC 未打开使开关不生效

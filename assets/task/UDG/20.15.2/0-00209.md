# 增加IPv6 ACL规则组与规则(ACL6)（ADD ACLGROUP6）

> 命令静态知识见 [ADD ACLGROUP6](command/UDG/20.15.2/ADD-ACLGROUP6.md)（建组）、[ADD ACLRULEADV6](command/UDG/20.15.2/ADD-ACLRULEADV6.md)（高级规则）、[ADD ACLRULEBAS6](command/UDG/20.15.2/ADD-ACLRULEBAS6.md)（基本规则）。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

本 atom 是**聚合 atom**：覆盖 IPv6 ACL 全链——`ADD ACLGROUP6`（建 IPv6 规则组）+ `ADD ACLRULEADV6`（高级 IPv6，五元组，3000-3999）/ `ADD ACLRULEBAS6`（基本 IPv6，仅源 IP，2000-2999）。**关键差异点**：IPv6 掩码参数 `ACLSRCWILD` 为**正掩码长度**（IPv4 为反掩码）。

> 聚合债：CONSOLIDATION-BACKLOG.md §B/E.2 已标记应拆为 `ACLGROUP6` / `ACLRULEADV6` / `ACLRULEBAS6` 三个独立 atom，本 pass 不拆。

**配置范式（按匹配类型分支，IPFD-012002 数据规划表归纳）**

| 步骤 | 命令 | ACLNAME（范围段） | 关键参数 | 示例行 |
|---|---|---|---|---|
| 1. 建 IPv6 规则组 | ADD ACLGROUP6 | 数字串（如 "3000"） | ACLNAME；ACLTYPE 数字串时可省 | `ADD ACLGROUP6:ACLNAME="3000";` |
| 2a. 基本规则（仅源 IP） | ADD ACLRULEBAS6 | 2000-2999 | ACLSOURCEIP + ACLSRCWILD（正掩码长度） | 见命令 wiki |
| 2b. 高级规则（五元组） | ADD ACLRULEADV6 | 3000-3999 | 五元组 + 协议；TCP/UDP 时端口段 | 见命令 wiki |

**参数取值约定**

| 参数 | 取值 | 来源 | 说明 |
|---|---|---|---|
| ACLNAME | IPv6 ACL 组标识 | 本端规划（建组）/ 引用（建规则） | **ACLNAME 为非数字时 ACLTYPE 默认高级**；为数字串时可省 ACLTYPE |
| ACLTYPE | 枚举（基本/高级） | 本端规划，可省 | 规则组类型 |
| ACLRULENAME | 字符串 1~32 | 本端规划 | 规则名称 |
| ACLACTION | Permit / Deny | 本端规划 | 规则行为 |
| ACLPROTOCOL | 整数 0~255 | 本端规划 | 协议号（高级规则） |
| ACLSOURCEIP | IPv6 | 全网规划 | 源 IP |
| ACLSRCWILD | **正掩码长度**（0~128，非反掩码） | 本端规划 | **IPv6 family 关键差异点**：与 IPv4 反掩码语义不同 |
| ACLSRCPORTOP / ACLDESTPORTOP | 枚举（Lt/Gt/Eq/Neq/Range） | 本端规划 | **TCP/UDP 时必填**（rule 0-00270 同款约束） |
| ACLSRCPORTBEGIN / ACLDESTPORTBEGIN | 整数 0~65535 | 本端规划 | **TCP/UDP 时必填**起始端口 |
| ACLDESTIP | IPv6 | 全网规划 | 高级规则目的 IP（条件必填） |

**未绑冷门参数**（ACLGROUP6 命令真相有但特性脚本未用）：ACLSTEP（默认 5）/ ACLMATCHORDER（默认 Config）/ ACLDESCRIPTION——按需时查命令 wiki。

## 决策点

- **[0-00032](task/UDG/20.15.2/0-00032.md)**（opt-acl-v6）IPv4 / IPv6 ACL family 互斥：已选 IPv6 时 excludes ACL4 链（0-00197/0-00198/0-00207/0-00208）；由本 family 顶层把控

## 约束

- **rule-0-00272**（warning）聚合 atom 内部依赖：ACLGROUP6 必须先发（规则组前置），后续 ACLRULEADV6（高级，五元组，3000-3999）或 ACLRULEBAS6（基本，仅源 IP，2000-2999）二选一——后果：规则组未建→规则创建失败；选错 type→范围段错位被拒
- **rule-0-00270 同款**：ACLPROTOCOL=TCP(6)/UDP(17) 时端口段参数（ACLSRCPORTOP/ACLDESTPORTOP + ACLSRCPORTBEGIN/ACLDESTPORTBEGIN）必填——后果：端口匹配参数缺失→规则无法精确匹配
- 最大记录数 65535；单组最多 16384 条规则——后果：超限→新增失败

# 创建TWAMP逻辑接口（ADD TWAMPLOGICINF）

> 命令静态知识见 [ADD TWAMPLOGICINF](command/UDG/20.15.2/ADD-TWAMPLOGICINF.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

本任务是 GWFD-110921（TWAMP）探测链路搭建的第 3 步，创建本端探测报文收发用的逻辑接口。**Full / Light 两模式共用**同一条命令，区别仅在接口名和 IP（Full 用 N3/S1-U 口、Light 用业务口），其余参数取值范式一致。

**参数取值规划（数据规划表样例）**

| 参数 | 含义 | Full 模式样例 | Light 模式样例 | 获取方法 / 取值规律 |
|---|---|---|---|---|
| NAME | 逻辑接口名 | `n3if1/1/1` | `n3if1/1/0` | 全网规划，1~63 字符；按 N3/S1-U 接口编号命名 |
| AFTYPE | 地址族类型 | `IPV4` | `IPV4` | 固定 IPV4 |
| IPV4ADDRESS | 逻辑接口地址 | `10.1.1.10` | `10.1.1.5` | 全网规划，本端 N3/S1-U 接口 IP（条件必填） |
| IPV4MASK | 逻辑接口掩码 | `255.255.255.255` | `255.255.255.255` | 固定 32 位主机掩码 |
| VRFNAME | VPN 实例名 | `VPN_CN` | `VPN_CN` | 引用 [创建TWAMP VPN实例](task/UDG/20.15.2/0-00224.md) |
| SHAREDTYPE | 是否共享 IP | `TRUE` | `TRUE` | 全网规划；与业务 N3/S1-U 共用 IP 时**必须 TRUE** |

**脚本样例**

```
ADD TWAMPLOGICINF: NAME="n3if1/1/1", AFTYPE=IPV4, IPV4ADDRESS="10.1.1.10", IPV4MASK="255.255.255.255", VRFNAME="VPN_CN", SHAREDTYPE=TRUE;
```

**操作步骤定位**：该命令紧接 ADD TWAMPVPNINST 之后执行，其 IPV4ADDRESS 将被下游 ADD TWAMPRESPONDER.LOCALIPV4（Full，0-00226）/ ADD TWAMPCLIENT.LOCALIPV4（Light，0-00227）**引用且必须一致**。IPV6 系列参数（IPV6ADDRESS/IPV6MASK）本特性场景未启用。

# 配置以太网子接口VLAN（ADD ETHSUBIF）

> 命令静态知识见 [ADD ETHSUBIF](command/UDG/20.15.2/ADD-ETHSUBIF.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

ADD ETHSUBIF 在**已建子接口**上关联 VLAN ID，实现二层网络接入三层 / VLAN 隔离。IFNAME 引用上游 [ADD INTERFACE](task/UDG/20.15.2/0-00056.md) 建出的子接口名（如 `Ethernet70/0/4.1`）。

**双平面组网典型配法**（GWFD-020402，两 trunk 两 VLAN 隔离）

| 子接口（IFNAME） | VLAN ID（VLANTYPEVID） | 平面 |
|---|---|---|
| `Ethernet70/0/4.1` | `100` | 平面A |
| `Ethernet70/0/5.1` | `200` | 平面B |
| `Ethernet71/0/4.1` | `100` | 平面A |
| `Ethernet71/0/5.1` | `200` | 平面B |

**参数取值样例**

| 参数 | 取值样例 | 来源 | 说明 |
|---|---|---|---|
| IFNAME | `Ethernet70/0/4.1` | reference（← 0-00056） | 子接口名（主接口+`.序号`），可用 LST INTERFACE 查询 |
| VLANTYPEVID | `100`、`200` | 本端规划 | 整数 1~4094，**同主接口下子接口必须不同 VLAN** |

**典型脚本**：

```
ADD ETHSUBIF:IFNAME="Ethernet70/0/4.1",VLANTYPEVID=100;
ADD ETHSUBIF:IFNAME="Ethernet70/0/5.1",VLANTYPEVID=200;
ADD ETHSUBIF:IFNAME="Ethernet71/0/4.1",VLANTYPEVID=100;
ADD ETHSUBIF:IFNAME="Ethernet71/0/5.1",VLANTYPEVID=200;
```

**VLAN 分配语义**：不同主接口下子接口可复用同一 VLAN（上表 100/200 各复用一次），但**同一主接口下**不同子接口必须不同 VLAN（Ethernet70/0/4.1=100 vs Ethernet70/0/5.1=200）。

**省略字段**：SUBIFTYPE（optional 默认 VlanType）——本命令即 VLAN 子接口场景走默认值，脚本均省略，不绑。

**部署时序**：① ADD INTERFACE 建子接口（0-00056）→ ② ADD ETHSUBIF 关联 VLAN（本 atom）→ ③ ADD IPBINDVPN 绑 VPN → ④ ADD IFIPV6ADDRESS 配 IPv6 地址。

## 约束

- **rule-0-00244**（warning）**只允许配在子接口上**，每个子接口**只可关联一个 VLAN**；**同主接口下不同子接口必须不同 VLAN**（不同主接口可复用）；**前置**必须先创建子接口（IFNAME 指向已建子接口）；命令最大记录数 262144；执行后立即生效 — 后果：超规格/非子接口/同子接口重复 VLAN/同主接口 VLAN 重复失败；IFNAME 引用悬空
  > 部署约束（md 注意事项，非命令 notes）：子接口配 VLAN 后**必须在交换机/对接网关同步 VLAN** 配置；**双平面组网两 trunk 的 VLAN 须不同**以避免环路 — 后果：VLAN 未同步致通信异常 / 同 VLAN 致环路

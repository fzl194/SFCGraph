# 配置以太网子接口VLAN（ADD ETHSUBIF）

> 命令静态知识见 [ADD ETHSUBIF](command/UDG/20.15.2/ADD-ETHSUBIF.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。用于实现 VLAN 间通信：接入用户终端的接口通过创建子接口关联 VLAN，实现二层接入三层、不同网段/不同 VLAN 间通信。

## 配置方法

ADD ETHSUBIF 用于在以太网上创建子接口并关联 VLAN。**IFNAME 必选**（子接口名 `<parent>.subid` 格式）；**VLANTYPEVID 必选**（关联 VLAN ID）。核心配置空间有 3 个维度：**子接口定位**（IFNAME）、**流封装类型**（SUBIFTYPE）、**VLAN ID**（VLANTYPEVID）。1 个特性命中（GWFD-020402 N6/Gi/SGi IPv6 组网）使用子接口关联 VLAN 100/200。

### 配置维度 1：子接口定位（参数 IFNAME，必选）

| 取值 | 作用 |
|---|---|
| **以太网子接口名**（`<parent>.<subid>` 格式） | 子接口名；如 `Ethernet70/0/4.1`、`Ethernet70/0/5.1`（020402 范式） |

> 子接口名由父接口名 + `.subid` 构成；父接口须先通过 `ADD INTERFACE` 创建；1~63 字符。

### 配置维度 2：流封装类型（参数 SUBIFTYPE，可选）

| 取值 | 作用 |
|---|---|
| **VlanType** | 按 VLAN 封装（典型） |
| **其他** | 见命令真相表枚举 |

> 默认 VlanType；其他封装类型见命令真相表。

### 配置维度 3：VLAN ID（参数 VLANTYPEVID，必选）

| 取值 | 作用 |
|---|---|
| **1~4094 整数** | 子接口关联的 VLAN ID |

> VLAN ID 须在本端网络规划内；不同 VLAN 的子接口可对应不同三层网络（IPv4/IPv6）。

**典型脚本（GWFD-020402 N6/Gi/SGi IPv6 组网，子接口关联 VLAN 100/200）**：
```
ADD ETHSUBIF:IFNAME="Ethernet70/0/4.1",VLANTYPEVID=100;
ADD ETHSUBIF:IFNAME="Ethernet70/0/5.1",VLANTYPEVID=200;
ADD ETHSUBIF:IFNAME="Ethernet71/0/4.1",VLANTYPEVID=100;
ADD ETHSUBIF:IFNAME="Ethernet71/0/5.1",VLANTYPEVID=200;
```

**步骤位置**：在所属 feature 编排中通常位于"步骤 N：建子接口关联 VLAN"——前置 [0-00056](task/UDG/20.15.2/0-00056.md) ADD INTERFACE 建父接口 + [0-00114](task/UDG/20.15.2/0-00114.md) MOD INTERFACE 启父接口 up；后续按需 [0-00058](task/UDG/20.15.2/0-00058.md) ADD IFIPV4ADDRESS / [0-00061](task/UDG/20.15.2/0-00061.md) ADD IFIPV6ADDRESS 配子接口 IP。

## 决策点：VLAN 子接口创建（DP 0-00115）

（本命令无独立决策分支，仅有 VLANTYPEVID 取值差异。）

| 选项 | IFNAME | VLANTYPEVID | 适用场景 |
|---|---|---|---|
| **VLAN 100 子接口**（020402） | `Ethernet70/0/4.1` / `Ethernet71/0/4.1` | 100 | IPv6 组网 VLAN 100 三层接入 |
| **VLAN 200 子接口**（020402） | `Ethernet70/0/5.1` / `Ethernet71/0/5.1` | 200 | IPv6 组网 VLAN 200 三层接入 |

> 1 个特性命中（020402）；IFNAME+VLANTYPEVID 必选；同一父接口下可建多个子接口对应不同 VLAN。

## 约束

- **父接口必须已建**（critical，rule-0-00115）：父接口（IFNAME 中 `.` 前的部分）必须是已通过 `ADD INTERFACE` 创建的以太网接口 — 后果：父接口不存在命令拒收
- **IFNAME+VLANTYPEVID 联合必选**（critical，rule-0-00115）：IFNAME 与 VLANTYPEVID 是必选参数 — 后果：缺一命令拒收
- **VLAN ID 范围**（warning，rule-0-00115）：VLANTYPEVID 取值范围 1~4094 — 后果：超范围命令拒收
- **子接口需对应父接口 up**（warning，rule-0-00115）：父接口须先通过 [0-00114](task/UDG/20.15.2/0-00114.md) MOD INTERFACE 启 up — 后果：父接口 down 时子接口不生效
- **执行后立即生效**（warning，rule-0-00115）：本命令执行后立即生效 — 后果：误配 VLAN ID 即刻影响该子接口的 VLAN 接入

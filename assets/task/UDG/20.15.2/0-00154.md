# 设置SA业务公共参数（SET SACOMMONPARA）

> 命令静态知识见 [SET SACOMMONPARA](command/UDG/20.15.2/SET-SACOMMONPARA.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

SET SACOMMONPARA 是 SA 业务公共参数的全局开关集合。本 atom 的主场景是 **HTTP3.0（QUIC）Host 识别 / 分析** 的前置开关——`QUICIDENFUNCEN`（识别增强）和 `QUICPSRFUNCEN`（解析增强）**跨特性二选一互斥**，由 DP 0-00141 编排特性路由。

**特性 → 开关映射（核心）**

| 特性 | 启用开关 | 关闭开关 | 典型脚本 |
|---|---|---|---|
| GWFD-110251 HTTP3.0 Host 识别 | `QUICIDENFUNCEN=ENABLE` | `QUICPSRFUNCEN=DISABLE` | `SET SACOMMONPARA: QUICIDENFUNCEN=ENABLE;` |
| GWFD-110252 HTTP3.0 Host 分析 | `QUICPSRFUNCEN=ENABLE` | `QUICIDENFUNCEN=DISABLE` | `SET SACOMMONPARA: QUICPSRFUNCEN=ENABLE;` |

两开关均数据规划表标"固定取值"=ENABLE，故 fixed；但取**哪个** ENABLE 由特性路由决定（见决策点）。

**配置位置**
1. 打开对应 QUIC 增强开关（本命令，全局前置）。
2. 若开关已 ENABLE，继续后续；若为 DISABLE，先 SET SACOMMONPARA 打开 License 配置开关。
3. 配内容计费 URR / URR Group 等后续步骤。

**冷门参数（主场景未用，按通用配置门槛不配）**：`NORSTPKTNUM`（默认 0=不控制，不宜过小）、`CDNENHANCEDSW`、`QUICSASW`、`USRRLTRFRSHSW`、`SPECMETHODSW`。

## 决策点

**DP 0-00141**：SA 业务公共参数 QUIC 增强功能特性路由（识别 / 解析）

问：SET SACOMMONPARA 配置时，选择哪种 QUIC 增强功能开关？

| 选项 | QUICIDENFUNCEN | QUICPSRFUNCEN | 适用特性 |
|---|---|---|---|
| QUIC 协议识别增强（默认） | ENABLE | DISABLE | GWFD-110251 HTTP3.0 Host 识别（Quic 密文增强 SA 协议识别） |
| QUIC 增强解析 | DISABLE | ENABLE | GWFD-110252 HTTP3.0 Host 分析（Quic 密文增强 SA 协议解析） |
| 两开关均关闭 | DISABLE | DISABLE | 本特性场景不采用 |

依据：两开关功能差异（识别 vs 解析）由特性路由分叉，**不应同开**。

## 约束

- **rule-0-00352**（warning）单命令最大记录数 **1**（SA 业务公共参数全局配置），超限 SET 失败。
- **rule-0-00352**（warning）**立即生效**。
- **rule-0-00352**（warning）`NORSTPKTNUM` 配置过小会导致报文解析提前返回失败、影响规则匹配；默认 0=不开启此控制。
- **rule-0-00352**（warning）QUICIDENFUNCEN / QUICPSRFUNCEN **只针对 Quic 密文业务增强 SA**，对非 Quic 业务无效。

# 创建OSPF进程(IPv4)（ADD OSPF）

> 命令静态知识见 [ADD OSPF](command/UDG/20.15.2/ADD-OSPF.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。**本命令是 OSPF IPv4 配置 backbone 根**——后续 [0-00075](task/UDG/20.15.2/0-00075.md) ADD OSPFAREA / [0-00076](task/UDG/20.15.2/0-00076.md) ADD OSPFNETWORK / [0-00077](task/UDG/20.15.2/0-00077.md) ADD OSPFIMPORTROUTE 都必须在已有 PROCID 进程的前提下执行。

## 配置方法（配置方法字典）

ADD OSPF 是"**创建 IPv4 OSPF 动态路由协议进程**"命令。PROCID 必选；VRFNAME 决定进程归属（首次创建时确定，后续不可更改）；SCHEMAROUID 决定 Router ID（全网唯一）。核心配置空间有 4 个维度：**进程号**（PROCID，全网规划唯一）、**VPN 归属**（VRFNAME，默认公网 `_public_`）、**Router ID**（SCHEMAROUID，全网规划 IPv4 点分十进制）、**协议进阶参数**（BFD 联动 / LSA/SPF 智能定时器 / Stub 路由器 / MPLS LDP 等 44 项可选，按需细化）。N 个特性命中（GWFD-010104 / GWFD-010105 / GWFD-010155 / GWFD-020406 / GWFD-020421 / IPFD-015004 等）覆盖 3 种 PROCID 取值（`本端规划/全网规划`）。

### 配置维度 1：进程号（参数 PROCID，必选）

| 参数 | 取值范围 | 数据来源 | 备注 |
|---|---|---|---|
| **PROCID** | 1~4294967295 | 本端规划 / 全网规划 | 新增 OSPF 进程号不能与现网重复 |

> **PROCID 跨特性分叉示例**：010155 用 6（vpn-s2b VPN+OSPF 组网）；010104/010105/020406/020421 用 100（IPv4 下行路由）；IPFD-015004 用 20214（OSPF over IPsec）。**PROCID 在 process_id_namespace 全局唯一**。

### 配置维度 2：VPN 归属（参数 VRFNAME，可选）

VRFNAME 决定 OSPF 进程属于哪个 VPN 实例：

| 取值 | 默认值 | 备注 |
|---|---|---|
| `_public_`（默认） | _public_ | 进程属于全局实例 |
| ref [0-00071](task/UDG/20.15.2/0-00071.md) ADD L3VPNINST 已建的 VPN 实例名 | — | 进程属于该实例 |

> **关键规则**：**进程实例不可更改**——只能在第一次使能该进程时指定 VPN/全局归属，后续 ADD/MOD 不可变更。**设备的一个接口只能属于某一个 OSPF 进程**。

### 配置维度 3：Router ID（参数 SCHEMAROUID，可选，全网规划 IPv4）

| 取值范围 | 备注 |
|---|---|
| IPv4 地址类型 | 全网唯一；可手工配置或系统从接口 IP 中选取（优先 Loopback 最大 → 其余接口最大） |

> **关键规则**：路由器的 ID 只能修改不能删除；**如有配置则不需接口 IP，否则需接口 IPv4 地址作为 Router ID**（DHCP 分配地址不能作为 Router ID）。**IP 地址均为自动分配方式时，需要配置 Router ID**。

### 配置维度 4：协议进阶参数（可选，按需细化）

| 子维度 | 关键参数 | 默认值 | 典型场景 |
|---|---|---|---|
| **BFD 联动** | BFDALLINTFFLG（全局使能 BFD）+ BFDRXCFGFLAG+BFDMINRXINTV + BFDTXCFGFLAG+BFDMINTXINTV + SCHEMABFDDETMUL（检测乘数 3~50）+ FRRBINDINGFLAG（FRR 联动） | FALSE / 200ms / 200ms / 3 / FALSE | OSPF BFD 加速收敛（010155 等 OSPF+BFD 联动场景） |
| **LSA 智能定时器** | LSAARRINTVFLAG/LSAARRINTV/LSAARRMAXINTV/LSAARRSTARINTV/LSAARRHLDINTV | FALSE / 1000ms / 500ms / 500ms | 自适应 LSA 重传抑制 |
| **SPF 智能定时器** | SPFINTVSEC/SPFINTVMILLISEC/SPFMAXINTERVAL/SPFSTARTINTV/SPFHOLDINTERVAL | 智能定时器 / 5000ms / 50ms / 200ms | 自适应 SPF 计算抑制 |
| **Stub 路由器** | STUBROUFLG（NoConfig/StubOnHand/StubOnBoot）+ STUBROUSTUPINTV + STUBROUINCSTUB | NoConfig / 500s / FALSE | 路由器倒换期/抑制被穿（生成 LSA 度量=65535） |
| **MPLS LDP** | MPLSLDPAUTOCFG（None/Enable）+ LDP 和 OSPF 联动（LDPSYNCENABLE） | None / FALSE | MPLS LDP 自动使能与同步（私网 OSPF 进程不支持） |
| **路由标签** | ROUTAGFLAG/ROUTAG | FALSE / — | VPN 引入路由 tag 值（私有 AS 防环） |
| **最大重传** | RXMTLIMITENABLE/RETRANSLIMIT | FALSE / 30 | OSPF 报文重传限制 |
| **接口抑制** | SILENTALLFLAG（抑制进程下所有接口收发 OSPF 报文） | FALSE | 静默接口 |
| **共网段** | VIRTUALSYSFLAG（OSPF 共网段虚拟系统使能，default=TRUE）+ FRRBINDINGFLAG | TRUE / FALSE | 共网段生效后 DRPRI 自动变 0 |
| **环路检测** | VPNINSCAPSIMFLG（MCE 多实例去使能 VPN 环路检测）+ RTTAGDISFLAG | FALSE / FALSE | VPN 多实例场景须去使能环路检测 |
| **Stub 链路最大开销** | STUBROUINCSTUB | FALSE | Stub 链路发布最大开销 |
| **安全同步** | SAFESYNCFLG | FALSE | 重启设备 LSDB 同步保护 |
| **Opaque LSA** | OPAQCAPFLG | FALSE | OSPFv2 Opaque-LSA 能力 |
| **协议安全** | 配套 [ADD OSPFAREAAUTH](command/UDG/20.15.2/ADD-OSPFAREAAUTH.md) / [ADD OSPFINTERFACEAUTH](command/UDG/20.15.2/ADD-OSPFINTERFACEAUTH.md) 配认证 | — | 建议配置协议认证增强协议安全性 |
| **自描述** | DESCRIPTIONTEXT（1~80 字符） | — | 进程描述信息 |

**典型脚本（010155 VPN+OSPF 组网，PROCID=6 / VPN-s2b / Router ID=10.8.20.1）**：
```
ADD OSPF: PROCID=6, VRFNAME="vpn-s2b", SCHEMAROUID="10.8.20.1";
```

**典型脚本（010104 IPv4 下行路由，PROCID=100 / vpn1 / Router ID=10.10.10.1）**：
```
ADD OSPF: PROCID=100, VRFNAME="vpn1", SCHEMAROUID="10.10.10.1";
```

**典型脚本（IPFD-015004 OSPF over IPsec，PROCID=20214 / vrf1 / Router ID=10.102.105.238）**：
```
ADD OSPF: PROCID=20214, VRFNAME="vrf1", SCHEMAROUID="10.102.105.238";
```

> 命令 notes：**最大记录数 8016**；**执行后立即生效**；设备的一个接口只能属于某一个 OSPF 进程；**进程实例不可更改 VRFNAME**（仅第一次使能时指定 VPN/全局归属）；**整机进程实例数规格默认 2000**（实际以服务规格文件为准）；**建议配置协议认证（ADD OSPFAREAAUTH/ADD OSPFINTERFACEAUTH）增强协议安全性**；运行 OSPF 协议必须存在 Router ID（未配 SCHEMAROUID 时由系统从接口 IP 选取，DHCP 分配地址不能作为 Router ID）。

**步骤位置**：在所属 feature 编排中通常位于"步骤 N：创建 OSPF 进程"——前置 [0-00071](task/UDG/20.15.2/0-00071.md) ADD L3VPNINST（VRFNAME 引用），下游 [0-00075](task/UDG/20.15.2/0-00075.md) ADD OSPFAREA / [0-00076](task/UDG/20.15.2/0-00076.md) ADD OSPFNETWORK / [0-00077](task/UDG/20.15.2/0-00077.md) ADD OSPFIMPORTROUTE。

## 决策点：OSPF 进程归属与 Router ID 策略（DP 0-00074）

问：ADD OSPF 的 VRFNAME / SCHEMAROUID 取何策略？

| 选项 | VRFNAME | SCHEMAROUID | 适用场景 |
|---|---|---|---|
| **公网全局实例** | `_public_`（默认） | 全网规划 IPv4 | UDG ePDG 公网路由对接（010155 vpn-s2b 例外） |
| **私网 VPN 实例** | ref [0-00071](task/UDG/20.15.2/0-00071.md) 已建的 VPN 名（如 vpn-s2b / vpn1 / vrf1） | 全网规划 IPv4 | 私网 OSPF（010104/010105/020406/020421/IPFD-015004 范式） |

> **关键规则**：VRFNAME 须在 ADD OSPF 首次创建时指定，后续不可修改；**VPN 多实例场景须配 `VPNINSCAPSIMFLG=TRUE`** 去使能 VPN 路由环路检测，否则路由引入失败。Router ID 选取优先级：手工配置 SCHEMAROUID > Loopback 最大 IP > 接口最大 IP。

## 约束

- **最大记录数 8016**（warning，rule-0-00474）：本命令最大记录数为 8016 — 后果：超规格新增失败
- **整机进程实例数默认 2000**（warning，rule-0-00474）：OSPF 进程实例数规格默认为整机 2000，实际以服务规格文件为准 — 后果：整机超过规格新增失败
- **执行后立即生效**（warning，rule-0-00474）：命令执行后立即生效 — 后果：误操作即刻生效
- **进程实例 VRFNAME 不可更改**（critical，rule-0-00474）：进程实例不可更改，只能在第一次使能该进程时指定 VPN 归属 — 后果：错填 VRFNAME 必须删除进程并重建
- **接口只能属一个 OSPF 进程**（critical，rule-0-00474）：设备的一个接口只能属于某一个 OSPF 进程 — 后果：接口归错进程需先 RMV 后重新绑定
- **Router ID 必须全网唯一**（critical，rule-0-00474）：SCHEMAROUID（Router ID）必须全网唯一 — 后果：冲突导致 OSPF 邻居无法建立
- **建议配置协议认证**（warning，rule-0-00474）：建议配置协议相关认证，增强协议安全性，否则可能存在协议安全风险 — 后果：未认证可能在协议层引入攻击风险
- **VPN 多实例须去环路检测**（critical，rule-0-00474）：VPN 多实例场景须配 VPNINSCAPSIMFLG=TRUE 去使能 VPN 路由环路检测，否则路由引入失败 — 后果：未去使能导致 OSPF 路由引入失败
- **OSPF 共网段须配套 AR**（warning，rule-0-00474）：当 OSPF 路由配置共网段时，需 MOD OSPFAREA VIRTUALSYSFLAG=TRUE — 后果：未配套 AR 共网段不生效
- **DHCP 分配地址不能作为 Router ID**（warning，rule-0-00474）：DHCP 分配的地址不能作为 Router ID — 后果：DHCP 场景需手工配置 SCHEMAROUID
- **BFD 联动需 OSPF+全局 BFD 都使能**（critical，rule-0-00474）：BFDALLINTFFLG=TRUE 需同时通过 SET BFD:BFDENABLE=TRUE 全局使能 BFD — 后果：BFD 全局未使能时进程下 BFD 不生效
- **私网 OSPF 不支持 LDP 联动**（info，隐含）：私网 OSPF 进程下不支持 LDP 和 OSPF 联动功能（LDPSYNCENABLE） — 后果：私网进程配 LDPSYNCENABLE 命令拒收

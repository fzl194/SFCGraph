# 前置准备（License 与接入域）

> BWM 配置的第一阶段（通用骨架，所有场景共享）。归属特性 [2-00001](task/UDG/20.15.2/2-00001.md)。

## 配置方法

| 步骤 | 命令 | 关键参数 |
|---|---|---|
| License 开关（所有场景第一步） | `SET LICENSESWITCH` | LICITEM=`LKV3G5TCSA01`（本特性 License 控制项），SWITCH=ENABLE |
| 接入域基础—APN（APN/DNN、PCC、TOS 族、HTTP、分级、三级场景） | `ADD APN` | APN=APN名（如 `apn-test`），HASVPN=DISABLE，HASVPNIPV6=DISABLE |
| 接入域基础—切片（切片场景） | 见 [1-00005](task/UDG/20.15.2/1-00005.md) | 切片标识 + 接口绑定（ADD SNSSAI/SNSSAIUPINTF），APN 可选 |

**典型脚本（APN 接入域）**：
```
SET LICENSESWITCH:LICITEM="LKV3G5TCSA01",SWITCH=ENABLE;
ADD APN:APN="apn-test",HASVPN=DISABLE,HASVPNIPV6=DISABLE;
```

**步骤位置**：BWM 配置流程最前；下游 [1-00002](task/UDG/20.15.2/1-00002.md)（业务与控制器）。

## 决策点

| 选项（接入域类型） | 影响（参数/命令/联动） |
|---|---|
| APN/DNN、PCC、TOS 族、HTTP、分级、三级 | 配 APN（[0-00020](task/UDG/20.15.2/0-00020.md)） |
| 切片 | 走切片准备（[1-00005](task/UDG/20.15.2/1-00005.md)） |

> 联动特性级场景 DP：接入域类型由 [2-00001](task/UDG/20.15.2/2-00001.md) 场景 DP 决定。

## 约束

- **License 必须先开**（critical）：后续 BWM 命令依赖 License 开启 — 否则 BWM 命令不生效。
- **依赖特性**（info）：[SA-Basic](feature/UDG/20.15.2/GWFD-110101.md)（GWFD-110101，业务识别）+ [PCC 基本功能](feature/UDG/20.15.2/GWFD-020351.md)（GWFD-020351，PCC 场景）。

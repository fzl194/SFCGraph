# 设置Gy接口一次重定向参数（SET GYONESHOT）

> 命令静态知识见 [SET GYONESHOT](command/UDG/20.15.2/SET-GYONESHOT.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

本命令是 **GWFD-020356（计费信息实时提醒）专属**，是 020356 三件套（扩展过滤器 → 重定向携带信息 → 一次重定向绑定）的**收尾激活步骤**。全局唯一（最大记录数 1），把前置配好的 EXTFLTNAME（URL 白名单）和 APPENDINFONAME（携带信息）绑定在一起，激活 Gy 接口一次重定向。匹配上白名单的报文将被重定向到 OCS 下发的 URL。立即生效。

**配置范式（单范式，双引用绑定）**

| 参数 | 取值 | 来源 | 说明 |
|---|---|---|---|
| EXTFLTNAME | 前置配好的扩展过滤器名（如 `gy-oneshot-url`） | 引用 [0-00022](task/UDG/20.15.2/0-00022.md).EXTFLTNAME | 一次重定向的 URL 白名单 |
| APPENDINFONAME | 前置配好的携带信息名（如 `testredirappendinfo`） | 引用 [0-00023](task/UDG/20.15.2/0-00023.md).APPENDINFONAME | 重定向携带的用户信息 |
| REDIRAGETIME | 整数 5~600（可选） | 本端规划 | 重定向完成时间（秒），主场景不配 |

**任务示例**

```
SET GYONESHOT:EXTFLTNAME="gy-oneshot-url",APPENDINFONAME="testredirappendinfo";
```

**前置链**：ADD EXTENDEDFILTER(0-00022) → ADD REDIRAPPENDINFO(0-00023) → SET GYONESHOT(0-00024)。

## 约束

- **rule-0-00334**（warning）最大记录数 1（全局唯一）；立即生效；EXTFLTNAME 引用前置 ADD EXTENDEDFILTER(0-00022)；APPENDINFONAME 引用前置 ADD REDIRAPPENDINFO(0-00023) — 后果：未配前置扩展过滤器或携带信息时一次重定向不生效

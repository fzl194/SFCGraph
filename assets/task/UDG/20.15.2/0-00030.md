# 绑定PCC策略组业务属性（ADD SRVPBINDPCCPG）

> 命令静态知识见 [ADD SRVPBINDPCCPG](command/UDG/20.15.2/ADD-SRVPBINDPCCPG.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法

ADD SRVPBINDPCCPG 是 PCC 业务属性链的"非默认组合装配层"：把已建好的 PCC 策略组 + 业务属性 + 可选 URR 组 + 可选 PCC 动作属性 + 可选扩展属性串成一条"非默认策略组合"。命令核心配置空间有 2 个维度：**必选引用**（PCC 策略组 + 业务属性，二者必备）+ **可选引用**（URR 组 / PCC 动作属性 / 扩展属性 / FUP 累计 4 个 optional，按需装配）。2 个必选 + 4 个可选全配即形成完整非默认组合；仅配 2 个必选即"空组合"骨架。

### 配置维度 1：必选引用（2 个必选 reference）

| 参数 | 引用对象 | 来源 atom | 何时配 |
|---|---|---|---|
| **PCCPOLICYGRPNM** | PCC 策略组名称 | 引用 [0-00003](task/UDG/20.15.2/0-00003.md)（ADD PCCPOLICYGRP）的 PCCPOLICYGRPNM | 总是必选 |
| **SRVPROPNAME** | 业务属性名称 | 引用 [0-00029](task/UDG/20.15.2/0-00029.md)（ADD SERVICEPROP）的 SRVPROPNAME | 总是必选 |

### 配置维度 2：可选引用（4 个 conditional reference）

| 参数 | 引用对象 | 来源 atom | 何时配 |
|---|---|---|---|
| **PCCACTPROPNAME** | PCC 动作属性名称 | 引用 [0-00028](task/UDG/20.15.2/0-00028.md)（ADD PCCACTIONPROP）的 PCCACTPROPNAME | 需要阻塞/转发门控动作时 |
| **URRGROUPNAME** | URR 组名称 | 引用 [0-00002](task/UDG/20.15.2/0-00002.md)（ADD URRGROUP）的 URRGROUPNAME | 需要计费时 |
| **EXTENDPROPNAME** | 扩展属性名称 | 命令参数齐全，无特性命中演示 | 扩展属性场景（冷门不配） |
| **FUPSESSIONEXC** | Session 级 FUP 流量累计标识（DISABLE/ENABLE） | 命令参数齐全，无特性命中演示 | Session 级 FUP 累计场景（冷门不配） |

**典型脚本（020352 IPv6 SA：PCC 策略组 + 业务属性 + 阻塞动作属性）**：
```
ADD SRVPBINDPCCPG:PCCPOLICYGRPNM="pg_test",SRVPROPNAME="sp_test",PCCACTPROPNAME="ap_test";
```

> 命令 notes：最大记录数 100000；一个 PCC 策略组最多 10 条非默认组合；多条组合时匹配优先级由各组合自身决定；适用 NF：PGW-U、UPF；执行后立即生效。

**步骤位置**：在所属 feature 编排中通常位于"步骤 N：PCC 策略组业务属性绑定"——前置 ADD PCCPOLICYGRP(0-00003) + ADD SERVICEPROP(0-00029) + （可选）ADD PCCACTIONPROP(0-00028) + （可选）ADD URRGROUP(0-00002)。

## 决策点：非默认组合的引用装配（DP 0-00030）

问：ADD SRVPBINDPCCPG 的非默认组合配哪些引用？

| 选项 | 装配内容 | 适用场景 |
|---|---|---|
| 仅必选（空组合骨架） | PCCPOLICYGRPNM + SRVPROPNAME | 命令级 PCC 触发（标准 PCCPOLICYGRP 骨架够用） |
| 必选 + 动作属性 | + PCCACTPROPNAME | 需要门控/转发动作（020352 IPv6 SA 阻塞业务） |
| 必选 + 计费 | + URRGROUPNAME | 需要按业务属性计费 |
| 全装配（必选 + 全部可选） | + EXTENDPROPNAME + FUPSESSIONEXC | 扩展属性 + Session 级 FUP 累计（命令参数齐全，无特性演示） |

## 约束

- **必选引用前置依赖**（critical，rule-0-00330）：PCCPOLICYGRPNM(0-00003) 与 SRVPROPNAME(0-00029) 必须先建成 — 后果：未建引用失败
- **可选引用前置依赖**（critical，rule-0-00330）：配 PCCACTPROPNAME 需先有 ADD PCCACTIONPROP(0-00028)；配 URRGROUPNAME 需先有 ADD URRGROUP(0-00002) — 后果：未建引用失败
- **每 PCC 策略组最多 10 条非默认组合**（warning，rule-0-00330）：单 PCC 策略组绑定组合数上限 10 — 后果：超 10 条新增失败
- **最大记录数 100000**（warning，rule-0-00330）：整板最大 100000 条 — 后果：超规格新增失败
- **适用 NF 限定**（warning，隐含）：PGW-U、UPF — 后果：SGW-U 等不适用

# 设置全局业务快速恢复配置（SET FASTRECOVERY）

> 命令静态知识见 [SET FASTRECOVERY](command/UDG/20.15.2/SET-FASTRECOVERY.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法

SET FASTRECOVERY 用于配置 SGW-C 故障重启场景下的全局业务快速恢复开关。**所有参数均为可选**（整机级全局配置，**最大记录数为 1**，单次 SET 配齐多开关）。核心配置空间有 5 个维度：**PGW 业务恢复**（RESTORPGWSWITCH）、**SGW 业务恢复**（RESTORSGWSWITCH）、**PDTN 触发 SGW-C 恢复**（PDTNSWITCH）、**防闪断定时器**（PRTFRQBRKTIME）、**保留承载超时**（HOLDONTIME）。1 个特性命中（GWFD-020254 VoLTE 业务快速恢复）三开关均 ENABLE 范式。

### 配置维度 1：故障重启业务恢复功能 PGW 开关（参数 RESTORPGWSWITCH，可选）

| 取值 | 作用 |
|---|---|
| **DISABLE** | 关闭 PGW 网络侧触发的业务恢复功能（默认） |
| **ENABLE** | 开启 PGW 网络侧触发的业务恢复功能（020254 范式，PGW-U 保留承载） |

### 配置维度 2：故障重启业务恢复功能 SGW 开关（参数 RESTORSGWSWITCH，可选）

| 取值 | 作用 |
|---|---|
| **DISABLE** | 关闭 SGW 网络侧触发的业务恢复功能（默认） |
| **ENABLE** | 开启 SGW 网络侧触发的业务恢复功能（020254 范式） |

### 配置维度 3：PDTN 功能开关（参数 PDTNSWITCH，RESTORPGWSWITCH=ENABLE 时可选）

| 取值 | 作用 |
|---|---|
| **DISABLE** | 关闭 PGW-U 触发 SGW-C 业务恢复功能（默认） |
| **ENABLE** | 开启 PGW-U 触发 SGW-C 业务恢复功能（保留承载收到下行数据时触发，020254 范式） |

### 配置维度 4：防闪断定时器时长（参数 PRTFRQBRKTIME，可选）

| 取值 | 作用 |
|---|---|
| **整数 0~300（秒）** | 防闪断定时器时长；0=不防闪断 |

### 配置维度 5：保留承载超时时长（参数 HOLDONTIME，可选）

| 取值 | 作用 |
|---|---|
| **整数 1~120（分钟）** | 保留承载的超时时长 |

**典型脚本（GWFD-020254 VoLTE 业务快速恢复，三开关全开）**：
```
SET FASTRECOVERY:RESTORPGWSWITCH=ENABLE,PDTNSWITCH=ENABLE,RESTORSGWSWITCH=ENABLE;
```

**步骤位置**：在所属 feature 编排中通常位于"步骤 N：开全局业务快速恢复"——全网规划统一确认容灾策略后一次性配齐三开关；整机级全局参数，跨网元一致。

## 决策点：快速恢复开关组合（DP 0-00171）

问：SGW-C 故障重启场景下需要开启哪些快速恢复能力？

| 选项 | RESTORPGWSWITCH | RESTORSGWSWITCH | PDTNSWITCH | 适用场景 |
|---|---|---|---|---|
| **三开关全开**（020254） | ENABLE | ENABLE | ENABLE | VoLTE 业务快速恢复（PGW + SGW + PDTN 联动） |

> 1 个特性命中（020254）三开关均 ENABLE；PDTNSWITCH 仅当 RESTORPGWSWITCH=ENABLE 时可选；整机最大记录数 1。

## 约束

- **PDTNSWITCH 前提条件**（critical，rule-0-00171）：PDTNSWITCH 在 RESTORPGWSWITCH=ENABLE 时为可选 — 后果：RESTORPGWSWITCH=DISABLE 时配 PDTNSWITCH 命令拒收
- **整机最大记录数 1**（warning，rule-0-00171）：本命令整机级全局配置，最大记录数 1 — 后果：超限 SET 失败
- **执行后立即生效**（warning，rule-0-00171）：本命令执行后立即生效 — 后果：误关开关即刻失去故障恢复能力
- **跨网元一致性**（warning，rule-0-00171）：三开关为网元故障容灾全局开关，跨网元需一致取值 — 后果：取值不一致时跨网元容灾协调异常
- **PGW-U 范围**（info，rule-0-00171）：RESTORPGWSWITCH 与 PDTNSWITCH 仅对 PGW-U 生效；RESTORSGWSWITCH 仅对 SGW-U 生效 — 后果：在不适用网元上配置无效

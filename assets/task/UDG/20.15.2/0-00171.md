# 设置全局业务快速恢复配置（SET FASTRECOVERY）

> 命令静态知识见 [SET FASTRECOVERY](command/UDG/20.15.2/SET-FASTRECOVERY.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

本命令配置 **SGW-C 故障重启场景下的全局业务快速恢复开关**（GWFD-020254 VoLTE 业务快速恢复专属，SGW-U/PGW-U 共用）。整机级全局参数，最大记录数为 1，单次 SET 配齐三开关。

**配置范式（三个开关同条命令一次配齐）**

| 参数 | 取值样例 | 获取方法 | 说明 |
|---|---|---|---|
| RESTORPGWSWITCH | `ENABLE` | 全网规划 | PGW-U 开关：SGW-C 故障重启时，PGW 是否具备业务恢复功能（PGW 支持保留承载）。仅对 PGW-U 生效 |
| RESTORSGWSWITCH | `ENABLE` | 全网规划 | SGW 开关：打开网络侧触发的业务恢复功能 |
| PDTNSWITCH | `ENABLE` | 全网规划 | PGW-U 开关：SGW-C 故障重启场景下，开启 PGW-U 触发 SGW-C 业务恢复功能（保留承载收到下行数据时触发） |

> 三开关均为**网元故障容灾全局开关**（非本端运维开关），故 variable_source=global_planned（按全网规划统一取值，跨网元一致）。

**配置步骤**

1. 全网规划确认三开关策略（容灾方案统一开 ENABLE）。
2. 一次执行三开关同条命令：
   ```
   SET FASTRECOVERY:RESTORPGWSWITCH=ENABLE,PDTNSWITCH=ENABLE,RESTORSGWSWITCH=ENABLE;
   ```

**示例脚本**：`SET FASTRECOVERY:RESTORPGWSWITCH=ENABLE,PDTNSWITCH=ENABLE,RESTORSGWSWITCH=ENABLE;`

**未绑冷门参数**（命令真相 optional 且 GWFD-020254 数据规划表未用到）：PRTFRQBRKTIME（0~300s 防闪断定时器）、HOLDONTIME（1~120min 保留承载超时）——按通用配置门槛不绑，需要时查命令 wiki。

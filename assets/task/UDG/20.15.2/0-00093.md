# 设置OTT业务规则匹配开关（SET EXTOTTMATCHSW）

> 命令静态知识见 [SET EXTOTTMATCHSW](command/UDG/20.15.2/SET-EXTOTTMATCHSW.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

SET EXTOTTMATCHSW 是 GWFD-110321 专属的全局开关命令，使能后 FLOWFILTER(FLOWFILTERTYPE=EXTEROTTDB) 才能匹配 OTT 业务规则库。系统初始值均为 DISABLE，本特性使能固定为 ENABLE。IPV4MATCH/IPV6MATCH 均本端规划（全局开关由本端运维决定）。命令真相 optional（系统初始 DISABLE），atom 录 required+value=ENABLE 表达本特性使能语义。

**主场景范式（从证据数据规划表归纳）**

| 字段 | 取值样例 | 数据来源 | 说明 |
|---|---|---|---|
| IPV4MATCH | ENABLE | 本端规划 | IPv4 流量 OTT 匹配开关，系统初始 DISABLE |
| IPV6MATCH | ENABLE | 本端规划 | IPv6 流量 OTT 匹配开关，系统初始 DISABLE |

**脚本样例**
```
SET EXTOTTMATCHSW:IPV4MATCH=ENABLE,IPV6MATCH=ENABLE;
```

**配置链路**（GWFD-110321 步骤 3→4→5）：先 ADD FLOWFILTER(FLOWFILTERTYPE=EXTEROTTDB) → **SET EXTOTTMATCHSW** 使能全局开关 → ADD URR 配置在线计费费率标识。

## 约束

- **rule 0-00378**（capacity_limit/error）全局唯一实例，最大记录数 1；立即生效；属**高危命令**，操作不当导致性能下降明显，须谨慎使用并联系华为技术支持；系统初始 IPV4MATCH=DISABLE/IPV6MATCH=DISABLE；仅当 SET EXTOTTMATCHSW 使能后 FLOWFILTER(FLOWFILTERTYPE=EXTEROTTDB) 才能匹配 OTT 库，应先 ADD FLOWFILTER 再 SET EXTOTTMATCHSW — 后果：重复执行覆盖前值；未先建 FLOWFILTER 时 SET 开关无效

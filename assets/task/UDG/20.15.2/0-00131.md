# 设置L2TP缺省配置（SET GLOBALL2TP）

> 命令静态知识见 [SET GLOBALL2TP](command/UDG/20.15.2/SET-GLOBALL2TP.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。**适用 NF：PGW-U、UPF**。

## 配置方法

SET GLOBALL2TP 设置 L2TP 隧道的**全局唯一**缺省属性——仅 1 个实例（命令最大记录数 1）。它同时为 **AAA 下发方式**与**本地配置方式**两种 L2TP 激活方式提供缺省：AAA 方式（[0-00132](task/UDG/20.15.2/0-00132.md) SET APNL2TPATTR + [0-00134](task/UDG/20.15.2/0-00134.md) ADD L2TPRDSCLIENT）取本命令的 LAC 名称/HELLO/Magic-Number/窗口/初始隧道/会话上限；本地方式（[0-00133](task/UDG/20.15.2/0-00133.md) ADD L2TPGROUP + [0-00135](task/UDG/20.15.2/0-00135.md) ADD L2TPCLIENTIP）只共享本端名称/HELLO/重发次数，其余下放 ADD L2TPGROUP 覆盖。核心配置空间有 2 个维度：**缺省隧道身份属性**（LOCALNAME/HELLO*/RETRYTIMES）、**缺省隧道参数（仅 AAA 方式生效）**（PPPMAGICNUMBER/MAXSENDWINSIZE/INITTUNNELNUM/MAXSESSIONNUM/INVTUNLEXISTDUR/LNSDETECT*）。1 个特性命中（GWFD-020412 L2TP VPN）按 AAA 方式配全参、本地方式配本端名+HELLO+重发。

### 配置维度 1：缺省隧道身份属性（参数 LOCALNAME / HELLOINTERVALSW / HELLOINTERVAL / RETRYTIMES，均可选）

| 参数 | 取值 | 作用 | 配套必选 |
|---|---|---|---|
| **LOCALNAME** | 字符串 1~30（区分大小写） | 隧道本端 LAC 缺省名称（020412 AAA 范式 `huawei`） | — |
| **HELLOINTERVALSW** | DISABLE/ENABLE | HELLO 保活报文开关 | — |
| **HELLOINTERVAL** | 60~10000 秒 | HELLO 报文间隔（020412 AAA 范式 `60`） | HELLOINTERVALSW=ENABLE 时必选 |
| **RETRYTIMES** | 1~16 | L2TP 报文重发次数（020412 AAA 范式 `3`） | — |

> LOCALNAME 也被本地方式 ADD L2TPGROUP 引用（GROUP 内未配置时取全局缺省）；HELLOINTERVAL 仅在 HELLOINTERVALSW=ENABLE 时必选。

### 配置维度 2：缺省隧道参数（AAA 方式生效，参数 PPPMAGICNUMBER / MAXSENDWINSIZE / INITTUNNELNUM / MAXSESSIONNUM / INVTUNLEXISTDUR / LNSDETECTSW / LNSDETECTINR，可选）

| 参数 | 取值 | 作用 | 配套必选 |
|---|---|---|---|
| **PPPMAGICNUMBER** | DISABLE/ENABLE | PPP LCP 协商是否支持 Magic-Number（020412 AAA 范式 DISABLE） | — |
| **MAXSENDWINSIZE** | 整数 | LAC 侧 L2TP 发送窗口上限（020412 AAA 范式 `64`） | — |
| **INITTUNNELNUM** | 整数 | LAC 与每 LNS 初始建立隧道数（020412 AAA 范式 `1`） | — |
| **MAXSESSIONNUM** | 整数 | 每条隧道承载的会话数上限（020412 AAA 范式 `32767`） | — |
| **INVTUNLEXISTDUR** | 整数（小时） | 无效隧道存活时长；超时释放 | — |
| **LNSDETECTSW** | DISABLE/ENABLE | 对 LNS 服务器探测消息开关 | — |
| **LNSDETECTINR** | 整数（分钟） | 对 LNS 探测消息时间间隔 | LNSDETECTSW=ENABLE 时必选 |

> 本地配置方式下 020412 范式不配 PPPMAGICNUMBER/MAXSENDWINSIZE/INITTUNNELNUM/MAXSESSIONNUM，这几个参数下放 ADD L2TPGROUP（每组覆盖）。

**AAA 方式典型脚本（020412，全参）**：
```
SET GLOBALL2TP:LOCALNAME="huawei",HELLOINTERVALSW=ENABLE,HELLOINTERVAL=60,RETRYTIMES=3,PPPMAGICNUMBER=DISABLE,MAXSENDWINSIZE=64,INITTUNNELNUM=1,MAXSESSIONNUM=32767;
```

**本地方式典型脚本（020412，仅共享本端名 + HELLO + 重发）**：
```
SET GLOBALL2TP:LOCALNAME="huawei",HELLOINTERVALSW=ENABLE,HELLOINTERVAL=60,RETRYTIMES=3;
```

**步骤位置**：在所属 feature 编排中通常位于"步骤 1：先设置 L2TP 缺省属性"——两种激活方式都先于 APN L2TP 属性（[0-00132](task/UDG/20.15.2/0-00132.md)）/ L2TP 组（[0-00133](task/UDG/20.15.2/0-00133.md)）执行；后续 AAA 方式配 [0-00134](task/UDG/20.15.2/0-00134.md) ADD L2TPRDSCLIENT + [0-00139](task/UDG/20.15.2/0-00139.md) SET APNPPPACCESS；本地方式配 [0-00135](task/UDG/20.15.2/0-00135.md) ADD L2TPCLIENTIP + [0-00138](task/UDG/20.15.2/0-00138.md) SET PPPCFG。

## 决策点：L2TP 缺省参数覆盖范围（DP 0-00131）

问：本缺省配置是 AAA 方式全配，还是本地方式仅配本端名+HELLO+重发？

| 选项 | 维度 1 参数 | 维度 2 参数（PPPMAGICNUMBER/MAXSENDWINSIZE/INITTUNNELNUM/MAXSESSIONNUM） | 适用场景 |
|---|---|---|---|
| **AAA 方式全参**（020412 AAA 范式） | 全配 | 全配（值 64/1/32767 等） | 鉴权服务器返回 L2TP 属性建隧道场景；不建 L2TP 组 |
| **本地方式精简**（020412 本地范式） | 仅 LOCALNAME/HELLO/RETRYTIMES | 不配（走 ADD L2TPGROUP 覆盖） | 本地配置 L2TP 组场景 |

> 1 个特性命中（020412）；AAA 方式可用 `SET APNL2TPATTR L2TPGROUPID` 不填走系统缺省；HELLOINTERVAL 仅在 HELLOINTERVALSW=ENABLE 时必选。

## 约束

- **全局唯一实例**（critical，rule-0-00131）：命令最大记录数 1；执行后立即生效（AAA 下发或本地建隧道时立即读新值） — 后果：超规格（多实例）新增失败；非立即生效会导致隧道参数未刷新
- **HELLOINTERVAL 条件必选**（critical，rule-0-00131）：HELLOINTERVAL 仅在 HELLOINTERVALSW=ENABLE 时为必选 — 后果：开关启用后缺间隔命令拒收
- **HELLOINTERVAL 范围**（warning，rule-0-00131）：HELLOINTERVAL 取值 60~10000 秒；RETRYTIMES 1~16；INITTUNNELNUM/MAXSESSIONNUM 受规格约束 — 后果：超范围命令拒收
- **适用 NF**（info，rule-0-00131）：本命令仅适用 PGW-U、UPF；SGW-U 不支持 — 后果：误在 SGW-U 上执行命令拒收
- **存在系统初始记录**（info，rule-0-00131）：系统初始记录 LOCALNAME=空 / HELLOINTERVALSW=DISABLE / 其他参数取默认值；不执行则走初始 — 后果：未配则按初始值生效
- **执行后立即生效**（warning，rule-0-00131）：本命令执行后立即生效 — 后果：误配即刻影响后续 L2TP 隧道建立

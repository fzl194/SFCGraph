# 增加内容分类组绑定关系（ADD CONTCATEGBIND）

> 命令静态知识见 [ADD CONTCATEGBIND](command/UDG/20.15.2/ADD-CONTCATEGBIND.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

本任务把内容过滤策略（CFPROFILE）与内容分类组（CONTCATEGROUP）绑定，并指定匹配命中后的动作（ACTION）。两个引用键均为前置依赖。适用 NF：PGW-U、UPF。

`CFPROFILENAME` 与 `CONTCATEGNAME` 均为"已配置数据中获取"——分别引用 [ADD CFPROFILE](task/UDG/20.15.2/0-00252.md) 与 [ADD CONTCATEGROUP](task/UDG/20.15.2/0-00256.md) 已建的实例名，二者必须先建。

**ACTION 三路取值（数据规划表，主特性 GWFD-110471）**

| ACTION 分支 | 含义 | REDIRECTNAME | 样例值 |
|---|---|---|---|
| PASS（缺省默认，可不显式传） | 报文转发 | 不配 | ACTION=PASS |
| BLOCK（主特性样例） | 报文丢弃 | 不配 | ACTION=BLOCK |
| REDIRECT | 报文重定向 | 必选（指向 ADD REDIRECT（★R1.5 atom 待补） 已建实例，atom 待建） | ACTION=REDIRECT, REDIRECTNAME="rd1" |

**主场景脚本样例**（BLOCK 路径）

```
ADD CONTCATEGBIND:CFPROFILENAME="cf1", CONTCATEGNAME="ccgp1", ACTION=BLOCK;
```

**未绑参数说明（按 baseline §4.1 冷门不绑）**：TIMERANGENAME（TimeRange 名称，optional，对应 ADD TIMERANGE，主场景不配）；CFGDOMAINNAME（配置域，optional，主场景不配）。

## 决策点

**DP [UDG@20.15.2@DecisionPoint@0-00120] —— ACTION 三路分叉（PASS/BLOCK/REDIRECT）**
决定 ACTION 取值与 REDIRECTNAME 配套必选/必不传。默认 PASS 路径，BLOCK 为主特性样例，REDIRECT 需先建 ADD REDIRECT（★R1.5 atom 待补） atom（atom 待建，隐性前置）。本 atom 仅 1 特性使用，无跨特性分叉。

## 约束

- 命令最大记录数 **10000**；每个 CFPROFILE 绑定的 CONTCATEGROUP 数量 **≤20 个**。
- 前置依赖：CFPROFILENAME 引用 ADD CFPROFILE（0-00252）、CONTCATEGNAME 引用 ADD CONTCATEGROUP（0-00256），两者必须先建。
- ACTION optional 默认 PASS（命令真相），不强制必选。
- 命令执行后**立即生效**。

# 配置ADC参数（ADD ADCPARA）

> 命令静态知识见 [ADD ADCPARA](command/UDG/20.15.2/ADD-ADCPARA.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法

ADD ADCPARA 是 ADC（应用检测控制）增强机制的"应用级参数配置层"：把"待检测流过滤器 + 是否上报流信息 + 迟滞时间 + 关键流检测 + 重定向"用一条记录描述。仅当 PCRF/PCF 经预定义规则下发 APPLICATION_START/STOP Event-Trigger 时才需要配置。命令核心配置空间有 3 个维度：**流信息上报维度**（FLOWINFORPT 开关 + ADCHYSTIMER 迟滞，核心高频）、**关键流检测维度**（KEYFLOWDETECTSW + KEYFLOWTIME + KEYFLOWSPEED 阈值三参，冷门）、**重定向维度**（REDIRECTNAME 引用重定向对象，冷门）。当前 020357 演示只配核心维度 1。

### 配置维度 1：流信息上报维度（参数 FLOWINFORPT / ADCHYSTIMER）

| 参数 | 取值范围 | 必选 | 作用 |
|---|---|---|---|
| **FLOWINFORPT** | DISABLE/ENABLE（缺省 DISABLE） | optional | UDG 是否把"应用↔流"对应关系随 APPLICATION_START/STOP 上报给 PCRF/PCF（核心开关） |
| **ADCHYSTIMER** | 整数 0~3600 秒（缺省 0） | optional | 应用级上报迟滞时间；0=关闭延迟上报；Start/Stop 过频时调大以抑制抖动 |

### 配置维度 2：关键流检测维度（参数 KEYFLOWDETECTSW / KEYFLOWTIME / KEYFLOWSPEED）

| 参数 | 取值范围 | 必选 | 作用 |
|---|---|---|---|
| **KEYFLOWDETECTSW** | DISABLE/ENABLE（缺省 DISABLE） | optional | 关键业务流识别总开关 |
| **KEYFLOWTIME** | 整数（条件可选，KEYFLOWDETECTSW=ENABLE 时） | conditional | 关键流时长阈值 |
| **KEYFLOWSPEED** | 整数（二进制千比特每秒，条件可选） | conditional | 关键流速率阈值 |

### 配置维度 3：重定向维度（参数 REDIRECTNAME）

| 参数 | 取值范围 | 必选 | 作用 |
|---|---|---|---|
| **REDIRECTNAME** | 字符串 1~31 | optional | 重定向名称（引用前置 ADD REDIRAPPENDINFO/ADD PCCACTIONPROP 的重定向对象） |

**典型脚本（020357 增强的 ADC 基本功能，核心维度）**：
```
ADD ADCPARA:FLOWFILTERNAME="flowfilter_ADC01",FLOWINFORPT=ENABLE,ADCHYSTIMER=0;
```

> 命令 notes：最大记录数 8000；适用 NF：PGW-U、UPF；执行后立即生效。

**步骤位置**：在所属 feature 编排中通常位于"步骤 N：ADC 应用级参数"——前置 ADD FLOWFILTER(0-00007) + License + PCCPOLICYGRP；产出供 PCRF/PCF 下发 APPLICATION_START/STOP Event-Trigger 触发。

## 决策点：ADC 参数应用范围（DP 0-00031）

问：ADD ADCPARA 的配置范围（仅核心 / 加关键流检测 / 加重定向）？

| 选项 | 配置范围 | 适用场景 |
|---|---|---|
| 仅核心维度（默认） | FLOWINFORPT + ADCHYSTIMER | 020357 增强的 ADC 基本功能（演示） |
| 核心 + 关键流检测 | + KEYFLOWDETECTSW=ENABLE + KEYFLOWTIME + KEYFLOWSPEED | 关键业务流识别场景（命令参数齐全，无特性演示） |
| 核心 + 重定向 | + REDIRECTNAME | 应用触发重定向场景（命令参数齐全，无特性演示） |
| 全维度 | 上述全部 | 复合场景（冷门） |

## 约束

- **FLOWFILTERNAME 必须已配置**（critical，rule-0-00331）：FLOWFILTERNAME 必须先 ADD FLOWFILTER(0-00007) 配置 — 后果：未配 ADD 失败
- **关键流三参配套约束**（warning，rule-0-00331）：KEYFLOWDETECTSW=ENABLE 时 KEYFLOWTIME / KEYFLOWSPEED 可选配；KEYFLOWDETECTSW=DISABLE 时不允许配 KEYFLOWTIME/KEYFLOWSPEED — 后果：误配拒收
- **最大记录数 8000**（warning，rule-0-00331）：单板最大 8000 条 — 后果：超规格 ADD 失败
- **ADCHYSTIMER 范围**（warning，rule-0-00331）：必须 0~3600 整数；超出范围拒收 — 后果：参数越界失败
- **适用 NF 限定**（warning，隐含）：PGW-U、UPF — 后果：SGW-U 等不适用

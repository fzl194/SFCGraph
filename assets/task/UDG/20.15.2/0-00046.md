# 配置APN地址分配属性（SET APNADDRESSATTR）

> 命令静态知识见 [SET APNADDRESSATTR](command/UDG/20.15.2/SET-APNADDRESSATTR.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法（配置方法字典）

SET APNADDRESSATTR 是 APN 级"地址分配属性开关"：针对指定 APN 配地址分配相关的全维度开关，控制该 APN 下用户的地址类型、防欺诈、路由方式、是否接受 SMF 下发地址池名等。命令核心配置空间有 7 个维度：**APN 引用**（APN 必选 + 必须已建）、**双栈支持**（SUPPORTIPV4/SUPPORTIPV6）、**路由方式**（HOSTROUTEIP 主机地址路由 / FRAMEDROUTE 手机后路由）、**防欺诈**（ANTISPOOFINGDL/UL）、**地址池名忽略**（IGNOREV4/6POOLID）、**双栈返回地址类型**（IPTYPEFORDUALIP/CPCTRL/SINGLEADDRCAUSE）、**IPv6 RA**（IPV6RAMTUSW/IPV6MTU/IPV6RALIFETIME）。5 个特性命中覆盖主机地址接入（010104）、双栈地址（010105/020406/020421）、Routing Behind MS（110910）。

### 配置维度 1：APN 引用（参数 APN，必选）

| 参数 | 范围 | 备注 |
|---|---|---|
| **APN** | 字符串 1~63 | 引用已配的 [ADD APN](task/UDG/20.15.2/0-00020.md) APN；前置依赖：APN 必须先建；**每个 APN 仅一条本配置**（重复配置失败） |

### 配置维度 2：双栈支持（参数 SUPPORTIPV4/SUPPORTIPV6，可选）

| 参数 | 取值 | 作用 |
|---|---|---|
| **SUPPORTIPV4** | ENABLE / DISABLE | 该 APN 是否支持 IPv4 地址 |
| **SUPPORTIPV6** | ENABLE / DISABLE | 该 APN 是否支持 IPv6 地址 |

> 多数地址分配子场景配 `SUPPORTIPV4=ENABLE + SUPPORTIPV6=ENABLE`（010105/020406/020421 范式）。

### 配置维度 3：路由方式（参数 HOSTROUTEIP/FRAMEDROUTE，可选）

| 参数 | 取值 | 作用 |
|---|---|---|
| **HOSTROUTEIP** | ENABLE / DISABLE | APN 是否支持 CP 使用**主机地址**接入（主机地址路由） |
| **FRAMEDROUTE** | ENABLE / DISABLE | APN 是否允许**手机后路由**用户接入（Routing Behind MS） |

> 二者互斥：HOSTROUTEIP 用于主机地址场景（010104 范式），FRAMEDROUTE 用于 MS 后路由场景（110910 范式）。

### 配置维度 4：防欺诈（参数 ANTISPOOFINGDL/ANTISPOOFINGUL，可选）

| 参数 | 取值 | 作用 |
|---|---|---|
| **ANTISPOOFINGDL** | ENABLE / DISABLE | APN 下行数据是否支持防欺诈 |
| **ANTISPOOFINGUL** | ENABLE / DISABLE | APN 上行数据是否支持防欺诈 |

### 配置维度 5：地址池名忽略（参数 IGNOREV4POOLID/IGNOREV6POOLID，可选）

| 参数 | 取值 | 作用 |
|---|---|---|
| **IGNOREV4POOLID** | DISABLE（默认） | 不忽略 SMF 下发 IPv4 地址池名，从 SMF 下发池分配 |
| **IGNOREV6POOLID** | DISABLE（默认） | 不忽略 SMF 下发 IPv6 地址池名，从 SMF 下发池分配 |
| ENABLE | 忽略 SMF 下发地址池名 | 从本地配置池分配 |

> 010105 RADIUS 场景显式配 `IGNOREV4POOLID=DISABLE + IGNOREV6POOLID=DISABLE`（接受 SMF 下发）。

### 配置维度 6：双栈返回地址类型（参数 IPTYPEFORDUALIP/CPCTRL/SINGLEADDRCAUSE，可选）

| 参数 | 取值 | 作用 |
|---|---|---|
| **IPTYPEFORDUALIP** | IPV4V6 / IPV4 / IPV6 | 双栈用户请求时分配双栈/IPv4 单栈/IPv6 单栈 |
| **CPCTRL** | ENABLE / DISABLE | 是否开启 CP 控制双栈地址分配（IPTYPEFORDUALIP=IPV4V6 时条件可选） |
| **SINGLEADDRCAUSE** | 整数 | 单栈分配时激活响应消息中携带的原因值 |

### 配置维度 7：IPv6 RA（参数 IPV6RAMTUSW/IPV6MTU/IPV6RALIFETIME，可选）

| 参数 | 取值 | 作用 |
|---|---|---|
| **IPV6RAMTUSW** | ENABLE / DISABLE | IPv6 RA 消息是否携带 MTU 选项 |
| **IPV6MTU** | 整数 | IPv6 MTU 值（IPV6RAMTUSW=ENABLE 时条件可选） |
| **IPV6RALIFETIME** | 整数 3600~ | IPv6 RA 消息路由生命周期（秒） |

**典型脚本（010104 主机地址路由接入）**：
```
SET APNADDRESSATTR: APN="apn-test", HOSTROUTEIP=ENABLE;
```

**典型脚本（010105 双栈支持 + 接受 SMF 下发地址池名）**：
```
SET APNADDRESSATTR:APN="apn-test1",SUPPORTIPV4=ENABLE,SUPPORTIPV6=ENABLE,IGNOREV4POOLID=DISABLE,IGNOREV6POOLID=DISABLE;
```

**典型脚本（020406/020421 双栈支持）**：
```
SET APNADDRESSATTR:APN="apn-test1",SUPPORTIPV4=ENABLE,SUPPORTIPV6=ENABLE;
```

**典型脚本（110910 Routing Behind MS）**：
```
SET APNADDRESSATTR:APN="apn1",FRAMEDROUTE=ENABLE;
```

> 命令 notes：适用 NF：PGW-U、UPF；本命令属高危命令（可能影响用户激活）；执行后立即生效；每个 APN 仅一条本配置。

**步骤位置**：在所属 feature 编排中通常位于"步骤 N：APN 地址属性配置"——前置依赖 [0-00020](task/UDG/20.15.2/0-00020.md)（ADD APN）；按需在地址池族链路（[0-00043](task/UDG/20.15.2/0-00043.md)~[0-00047](task/UDG/20.15.2/0-00047.md)）之前或之后插入。

## 决策点：路由方式与场景组合（DP 0-00046）

问：SET APNADDRESSATTR 选哪个路由方式 + 双栈/地址池名参数组合？

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| 主机地址路由 | HOSTROUTEIP=ENABLE | 地址分配方式（010104 范式） |
| Routing Behind MS | FRAMEDROUTE=ENABLE | Routing Behind MS（110910 范式） |
| 双栈地址 | SUPPORTIPV4=ENABLE + SUPPORTIPV6=ENABLE | SMF 分配（010105）/ Prefix Delegation（020406）/ 基于位置（020421） |
| 双栈+接受 SMF 下发 | SUPPORTIPV4=ENABLE + SUPPORTIPV6=ENABLE + IGNOREV4POOLID=DISABLE + IGNOREV6POOLID=DISABLE | 基于 RADIUS 下发地址池名（010105 子场景） |

> 5 个特性命中：010104 选主机地址；110910 选 MS 后路由；010105/020406/020421 选双栈（其中 010105 补 IGNORE*=DISABLE）。

## 约束

- **APN 前置依赖**（critical，rule-0-00446）：APN 必须先由 [0-00020](task/UDG/20.15.2/0-00020.md)（ADD APN）建好 — 后果：未建 APN 命令拒收
- **APN 唯一**（critical，rule-0-00446）：每个 APN 仅一条地址属性配置 — 后果：同 APN 重复配置命令拒收
- **HOSTROUTEIP 与 FRAMEDROUTE 互斥**（warning，rule-0-00446）：二者通常不同时配 ENBALE（路由方式二选一） — 后果：同时 ENABLE 行为未定义
- **高危命令**（warning，rule-0-00446）：配置可能影响用户激活 — 后果：配置错误可能致激活失败
- **最大记录数 10000**（warning，rule-0-00446）：超规格新增失败 — 后果：超 10000 拒收
- **适用 NF 限定**（warning，隐含）：PGW-U、UPF — 后果：SGW-U 等不适用
- **CPCTRL 仅 IPTYPEFORDUALIP=IPV4V6 时可配**（warning，rule-0-00446）：CPCTRL 是条件可选 — 后果：IPTYPEFORDUALIP 非 IPV4V6 配 CPCTRL 命令拒收
- **IPV6MTU 仅 IPV6RAMTUSW=ENABLE 时可配**（warning，rule-0-00446）：IPV6MTU 条件可选 — 后果：未开 RA MTU 开关配 IPV6MTU 命令拒收

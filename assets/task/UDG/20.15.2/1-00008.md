# CF 内容过滤业务配置

> URL/内容过滤的**核心业务配置**（多命令可复用模块，内容过滤域通用）。建立策略-模板-分类体系。被引用于 [2-00002](task/UDG/20.15.2/2-00002.md)。

## 配置方法

策略链六步：策略 → 模板(绑ICAP组+缺省动作) → APN绑模板 → 策略绑模板 → 分类组 → 策略绑分类组(动作)。

| 步骤 | 命令 | 关键参数 |
|---|---|---|
| 过滤策略 | ADD CFPROFILE | CFPROFILENAME |
| 过滤模板 | ADD CFTEMPLATE | CFTEMPLATENAME + ICAPSRVGMNAME(绑ICAP组) + ACTION(缺省动作 BLOCK/PERMIT/REDIRECT) |
| APN 绑模板 | SET APNCFTEMPLATE | APNNAME + CFTEMPLATENAME |
| 策略绑模板 | ADD CFPROFBINDCFT | CFTEMPLATENAME + CFPROFILENAME |
| 分类组 | ADD CONTCATEGROUP | CONTCATEGNAME + CATEGORYTYPE=SPECIFIC + CATEGORYID(ICAP返回的分类ID) |
| 策略绑分类组 | ADD CONTCATEGBIND | CFPROFILENAME + CONTCATEGNAME + ACTION(分类绑定动作) |

**典型脚本（分类匹配模式）**：
```
ADD CFPROFILE:CFPROFILENAME="cf1";
ADD CFTEMPLATE:CFTEMPLATENAME="icaptemplate",ICAPSRVGMNAME="icapsvrgrp1",ACTION=BLOCK;
SET APNCFTEMPLATE:APNNAME="apn-test",CFTEMPLATENAME="icaptemplate";
ADD CFPROFBINDCFT:CFTEMPLATENAME="icaptemplate",CFPROFILENAME="cf1";
ADD CONTCATEGROUP:CONTCATEGNAME="ccgp1",CATEGORYTYPE=SPECIFIC,CATEGORYID=1;
ADD CONTCATEGBIND:CFPROFILENAME="cf1",CONTCATEGNAME="ccgp1",ACTION=BLOCK;
```

**步骤位置**：承接 [1-00007](task/UDG/20.15.2/1-00007.md)（CFTEMPLATE 引用 ICAPSRVGMNAME）+ License/APN 粒度开关；下游供 PCC 触发链引用 CFPROFILE。

## 决策点

| 选项 | 影响（参数/命令/联动） |
|---|---|
| 动作匹配模式—分类匹配（主流，特性级联动） | 配 CONTCATEGROUP + CONTCATEGBIND，ICAP 返回 CATEGORYID → UDG 本地按分类匹配动作 |
| 动作匹配模式—直接动作（特性级联动） | 省略 CONTCATEGROUP/CONTCATEGBIND，ICAP 直接返回动作，UDG 仅按 CFTEMPLATE.ACTION 兜底 |
| 缺省动作 vs 分类绑定动作优先级 | `CONTCATEGBIND.ACTION`（分类绑定）**优先于** `CFTEMPLATE.ACTION`（缺省）；CFTEMPLATE.ACTION 仅在分类未命中或 ICAP 无应答时兜底 |

## 场景差异（各场景对本步骤的参数增量）

- **直接动作模式**：省略 CONTCATEGROUP/CONTCATEGBIND；CFTEMPLATE.ACTION 兜底（如 PERMIT）。
- **全局粒度**：SET APNCFTEMPLATE 换 ADD GLBCFTEMPLATE（全局模板，全网 APN 生效）。
- **HTTPS SNI**：CFTEMPLATE/CONTCATEGROUP 配置不变（ICAP 返回的 CATEGORYID 仍本地匹配），但需前置 HTTP2.0 Host 识别 License。

## 约束

- **CONTCATEGBIND.ACTION 优先于 CFTEMPLATE.ACTION**（critical）：分类匹配模式核心 — 顺序错则动作决策反转。
- **ICAPSRVGMNAME 一致**（critical）：CFTEMPLATE.ICAPSRVGMNAME 须与 [1-00007](task/UDG/20.15.2/1-00007.md) 的 ICAPSVRGRPNAME 一致 — 不一致则 ICAP 调用断链。
- **APNNAME 一致**（warning）：SET APNCFTEMPLATE.APNNAME 须与 ADD APN 一致 — 不一致则模板绑定错误 APN。
- **规格上限**（warning）：整机 ≤500 内容过滤套餐 — 超规格则配置失败。

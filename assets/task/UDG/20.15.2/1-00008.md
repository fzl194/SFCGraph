# 三级整机控制

> BWM 配置的场景特化阶段（仅三级控制场景）。整机级规则对全网用户统一限速，与用户级/组级规则叠加。归属特性 [2-00001](task/UDG/20.15.2/2-00001.md)。

## 配置方法

三级控制器带宽递增（用户级 < 用户组级 < 整机级），用户级/组级用 BWMRULE，整机级用 BWMRULEGLOBAL。

| 级别 | 规则命令 | CIR 示例(kbps) | 说明 |
|---|---|---|---|
| 用户级 | ADD BWMRULE（SUBSCRIBER_SPECIFIC）| 30 | 每用户 |
| 用户组级 | ADD BWMRULE（GROUP_SPECIFIC）| 300 | 每用户组 |
| 整机级 | ADD BWMRULEGLOBAL | 3000 | 全网 |

**典型脚本（整机级部分；用户级/组级见 [1-00003](task/UDG/20.15.2/1-00003.md)）**：
```
ADD BWMRULEGLOBAL:BWMRULENAME="br_global_test",BWMSERVICENAME="bs_group_test",UPBWMCTRLNAME1="bc-global-up",DNBWMCTRLNAME1="bc-global-down",BWMRULEPRI=1;
```

**步骤位置**：在 [1-00003](task/UDG/20.15.2/1-00003.md)（用户级/组级规则）之后叠加整机级规则。三级控制器在 [1-00002](task/UDG/20.15.2/1-00002.md) 按级配（`bc-subs`/`bc-group`/`bc-global`）。

## 决策点

**DP：三级控制器带宽梯度** —— 用户级 < 用户组级 < 整机级（如 30/300/3000），保证层次化生效。
**DP（联动）：业务类型组合** —— 用户级可 NONTOS、用户组/整机级 TOS（`USERLEVSRVTYPE` 与 `USERGLEVSRVTYPE` 可不同）。

## 约束

- 整机级（BWMRULEGLOBAL）仅支持 TOS 业务。
- 三级带宽须递增，否则层次化失效。
- 与用户级/组级 BWMRULE 共享 BwmRule 5000 配额。

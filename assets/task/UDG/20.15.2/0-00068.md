# 配置QoS接口信任（ADD QOSIFTRUST）

> 命令静态知识见 [ADD QOSIFTRUST](command/UDG/20.15.2/ADD-QOSIFTRUST.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法

ADD QOSIFTRUST 是 QoS 5 件套的"接口绑定层"——把物理接口绑定到指定 DS 域，使接口支持简单流分类并对报文优先级进行修改。入口/出口两方向都要绑定一次。本命令的核心配置空间有 3 个维度：**DSNAME 绑定**（引 [0-00066](task/UDG/20.15.2/0-00066.md) DS 域）、**接口类型**（决定 FLAG8021P 是否可用）、**802.1p 标志**（仅特定接口类型支持）。

### 配置维度 1：DSNAME 绑定（参数 DSNAME，必选）

| 取值 | 作用 | 典型场景 |
|---|---|---|
| **default** | 系统预置 DS 域（两方向主场景） | 010201 QoS 流量管理 |
| **5p3d** | 系统预置 DS 域（多 DS 域 VLAN→IP 映射分别绑定） | IPFD-012001 QoS 简单流分类 |
| **自定义** | 自建 DS 域（避开 5p3d） | 特殊业务场景 |

### 配置维度 2：接口类型（参数 IFNAME，必选）

| 接口类型 | FLAG8021P 支持 | 说明 |
|---|---|---|
| **以太主接口** | ✅ 支持 | 默认支持 802.1p |
| **以太 Trunk 主接口** | ✅ 支持 | 默认支持 802.1p |
| **GE 子接口** | ✅ 支持 | 默认支持 802.1p |
| **其他接口类型** | ❌ 不支持 | 置 TRUE 命令拒收 |

### 配置维度 3：802.1p 标志（参数 FLAG8021P，可选）

| 取值 | 作用 | 联动 |
|---|---|---|
| **FALSE**（默认） | 不支持 802.1p 映射 | 其他接口类型 |
| **TRUE** | 支持 802.1p 映射 | 仅以太子/GE 子/Eth-Trunk 主接口 |

**典型脚本（010201 QoS 主场景）**：
```
ADD QOSIFTRUST:DSNAME="default",IFNAME="Ethernet66/0/5";
```

**步骤位置**：在所属 feature 编排中位于"步骤 4：接口绑定 DS 域"——前置 [0-00066](task/UDG/20.15.2/0-00066.md) ADD QOSDIFFERSERV（DSNAME 来源）+ [0-00056](task/UDG/20.15.2/0-00056.md) ADD INTERFACE（IFNAME 来源）+ [0-00067](task/UDG/20.15.2/0-00067.md) SET QOSBA 入口映射 / [0-00069](task/UDG/20.15.2/0-00069.md) SET QOSPHB 出口映射。

## 决策点：方向绑定选择（DP 0-00068）

问：ADD QOSIFTRUST 绑入口方向还是出口方向？

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| 入口绑定（INGRESS） | 与 SET QOSBA 配对 | 入接口绑 DS 域启用入口 BA 映射 |
| 出口绑定（EGRESS） | 与 SET QOSPHB 配对 | 出接口绑 DS 域启用出口 PHB 映射 |

> 同一接口入口/出口方向各自需要一条 ADD QOSIFTRUST 绑定。

## 约束

- **最大记录数 65536**（critical，rule-0-00284）：系统最大记录数 65536 — 后果：超限 ADD 失败
- **FLAG8021P 接口类型约束**（critical，rule-0-00284）：FLAG8021P 仅以太子接口/GE 子接口/Eth-Trunk 主接口支持 — 后果：其他接口类型置 FLAG8021P=TRUE 命令拒收
- **入口/出口方向各绑定一次**（critical，rule-0-00284）：同一接口的 ingress（SET QOSBA 入口 BA）与 egress（SET QOSPHB 出口 PHB）各自需要本命令绑定一次 — 后果：方向未绑→对应映射不生效
- **DSNAME 前置依赖**（critical，隐含）：DSNAME 须先建（[0-00066](task/UDG/20.15.2/0-00066.md)） — 后果：未建域命令拒收
- **IFNAME 前置依赖**（critical，隐含）：IFNAME 须先建（[0-00056](task/UDG/20.15.2/0-00056.md)） — 后果：未建接口命令拒收

# 绑定用户组到APN（ADD APNBINDBWMUSRG）

> 命令静态知识见 [ADD APNBINDBWMUSRG](command/UDG/20.15.2/ADD-APNBINDBWMUSRG.md)

## 配置方法

仅 2 个参数，纯引用型绑定：把已建好的 BWMUSERGROUP 绑到指定 APN 下，使该 APN 用户纳入此用户组的带宽管控。

| 参数 | 取值样例 | 来源 |
|---|---|---|
| USERGROUPNAME | `ug_test` / `usergroup_test` | 引用 ADD BWMUSERGROUP（0-00033）已建用户组名 |
| APN | `apn-test` / `huawei.com` | 引用 ADD APN（0-00020）已建 APN 名 |

**典型脚本**：

```
ADD APNBINDBWMUSRG:USERGROUPNAME="ug_test",APN="apn-test";
```

**前置依赖**：APN 必须先通过 ADD APN 配置；USERGROUPNAME 必须先通过 ADD BWMUSERGROUP 配置。可 LST APN / LST BWMUSERGROUP 查询确认。

**步骤位置**：BWM 流程末步——用户组、业务、控制器、规则都建好后，最后把 APN 用户纳入用户组生效。在 110301 终端差异化场景下，本命令之后还要 SET APNOSLELBWMSW 打开 APN 操作系统级带宽管理开关。

## 约束

- 最大记录数 31984 · warning · 超规格新增失败；90% 阈值告警 ALM-135602215
- 一个用户组下 APN+UserProfile+切片数量总和 ≤ 16 · warning · per-user-group 配额超限失败
- APN/USERGROUPNAME 必须前置存在 · warning · 引用失败

# 设置内容过滤缓存参数（SET CFCACHEPARA）

> 命令静态知识见 [SET CFCACHEPARA](command/UDG/20.15.2/SET-CFCACHEPARA.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

本任务开启 URL 过滤本地 cache，降低 UDG 对 ICAP Server 的查询频次。两参数均为 optional，命令规格全局唯一实例。适用 NF：PGW-U、UPF。

**取值样例（数据规划表，主特性 GWFD-110471）**

| 参数 | 含义 | 样例值 | 说明 |
|---|---|---|---|
| CACHESW | 缓存开关 | ENABLE | 开启本地 cache（DISABLE=关闭）。命令真相无默认值，本特性显式 ENABLE。 |
| CACHEIDLETIME | 本地缓存过期时间（秒） | 7200 | 全网规划。整数 0～4294967294；<60s 系统实际按 60s 处理。 |

**主场景脚本样例**

```
SET CFCACHEPARA:CACHEIDLETIME=7200, CACHESW=ENABLE;
```

**CACHEIDLETIME 取值权衡**：配置过小→缓存策略更新过于频繁影响性能；配置过大→策略更新延迟（命中的分类可能过时）。需结合 ICAP Server 端缓存过期时间一起评估实际更新周期（两者共同决定策略生效时延）。

## 约束

- 命令最大记录数 **1**（全局唯一实例）。
- 命令执行后**立即生效**。
- 开启 cache 会导致 URL 过滤策略不能及时更新，更新周期取决于本命令与 ICAP Server 缓存过期时间。
- CACHEIDLETIME 配置值 <60 秒时，系统实际按 **60s** 处理（截断规则）。

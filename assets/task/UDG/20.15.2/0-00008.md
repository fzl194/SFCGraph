# 绑定三四层过滤条件到流过滤器（ADD FLTBINDFLOWF）

> 命令静态知识见 [ADD FLTBINDFLOWF](command/UDG/20.15.2/ADD-FLTBINDFLOWF.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法

ADD FLTBINDFLOWF 是过滤链 backbone 的"三四层绑定原子"：把上游 ADD FILTER（IPv4）/ADD FILTERIPV6（IPv6）产出的三四层过滤器，绑到 ADD FLOWFILTER 产出的流过滤器上。两参数都来自"已配置数据"（reference），不本端重新规划——只做引用对接。本命令的核心配置空间有 2 个维度：**绑定协议族**（FILTERNAME 引用 IPv4 还是 IPv6 FILTER）、**一对多绑定关系**（一个 FLOWFILTER 可同时绑多个 FILTER）。

### 配置维度 1：绑定协议族（参数 FILTERNAME，引用上游已配置数据）

| 取值 | 引用的源 atom | 典型过滤器名 |
|---|---|---|
| **IPv4 FILTER**（默认） | [ADD FILTER](task/UDG/20.15.2/0-00004.md) | `L34_dns` / `L34_any` / `filter_red` / `filter_test` |
| **IPv6 FILTERIPV6** | [ADD FILTERIPV6](task/UDG/20.15.2/0-00005.md) | `ipv6_ims` / `L34_any_ipv6` |

> 同一 FILTERNAME 参数名可绑定两种过滤器（IPv4/IPv6 名字空间独立），实际取值由上游 FILTER/FILTERIPV6 配置决定。一个 FlowFilter 可同时绑 IPv4 和 IPv6 过滤器（多次执行本命令）。

### 配置维度 2：一对多绑定关系（参数 FLOWFILTERNAME，引用上游已配置数据）

| 关系 | 说明 |
|---|---|
| **一对一**（单条 FILTER → 单条 FLOWFILTER） | 在线/离线/PCC 等单条规则场景 |
| **一对多**（多条 FILTER → 同一条 FLOWFILTER） | 同一 FlowFilter 既绑定 IPv4 也绑定 IPv6 FILTER（双协议族兜底） |

**典型范式**（一个 FLOWFILTER 同时绑 IPv4 + IPv6）：
```
ADD FLTBINDFLOWF: FLOWFILTERNAME="ff_L7", FILTERNAME="L34_dns";          # IPv4 dns
ADD FLTBINDFLOWF: FLOWFILTERNAME="ff_IMS", FILTERNAME="ipv6_ims";        # IPv6 IMS
ADD FLTBINDFLOWF: FLOWFILTERNAME="ff_L34any", FILTERNAME="L34_any";      # IPv4 兜底
ADD FLTBINDFLOWF: FLOWFILTERNAME="ff_L34any_IPv6", FILTERNAME="L34_any_ipv6";  # IPv6 兜底
```

**对接约定**：
- `FLOWFILTERNAME` → 引 [ADD FLOWFILTER](task/UDG/20.15.2/0-00007.md) 产出的流过滤器名
- `FILTERNAME` → 引 [ADD FILTER](task/UDG/20.15.2/0-00004.md)（IPv4）或 [ADD FILTERIPV6](task/UDG/20.15.2/0-00005.md)（IPv6）

**步骤位置**：在所属 feature 编排中通常位于"步骤 5：流过滤器与过滤条件绑定"——ADD FILTER/L7FILTER + ADD FLOWFILTER 之后，ADD PROTBINDFLOWF 可选；一个 FlowFilter 可多次执行本命令绑不同 FILTER。

## 决策点：绑定的三四层过滤器协议族（DP 0-00058）

问：FLTBINDFLOWF 的 FILTERNAME 引用何种协议族的三四层过滤器？

| 选项 | 联动 FILTERNAME 来源 | 典型过滤器名 |
|---|---|---|
| 绑定 IPv4 过滤器（IPV4，默认） | 引 ADD FILTER(0-00004) | `L34_dns` / `L34_any` / `filter_red` |
| 绑定 IPv6 过滤器（IPV6） | 引 ADD FILTERIPV6(0-00005) | `ipv6_ims` / `L34_any_ipv6` |

> 选项不互斥：一个 FlowFilter 可同时绑 IPv4 和 IPv6 过滤器（多次执行本命令）。选项决定"这条绑定引哪个源"。

## 约束

- **ADD FLTBINDFLOWF 刷新生效与记录数上限**（critical，rule-0-00124）：执行后需 **SET REFRESHSRV 刷新**后对新数据流生效（非立即，非固定延时）；FLTBINDFLOWF 最大记录数 **300000**，超规格 90% 上报 ALM-135602215；单个 FlowFilter 可配置的 Filter 数量有上限（规格约束，见命令原始 md） — 后果：新数据流不生效 / 容量超限告警
- **FLOWFILTERNAME 与 FILTERNAME 必须已配置**（critical，隐含）：本命令仅做引用对接——FLOWFILTERNAME 必须已 ADD FLOWFILTER 配置；FILTERNAME 必须已 ADD FILTER 或 ADD FILTERIPV6 配置 — 后果：未配置则绑定失败
- **绑定数量级**（warning，隐含）：单个 FlowFilter 可绑 Filter 数量有规格上限 — 后果：超规格绑定失败

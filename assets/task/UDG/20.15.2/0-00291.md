# 增加头增强（ADD HEADEN）

> 命令静态知识见 [ADD HEADEN](command/UDG/20.15.2/ADD-HEADEN.md)

## 配置方法

新建一个头增强对象（HEADERENNAME），通过 DATATYPE 指定要插入的用户信息字段类型，PREFIXNAME 设定 HTTP 头扩展字段前缀。**多字段插入需多次执行本命令**（同一 HEADERENNAME 下最多 40 种不同 DATATYPE）。加密/防欺诈/灰名单/编码按需配。

| 参数 | 取值样例 | 说明 |
|---|---|---|
| HEADERENNAME | `header_test` | 头增强名称；1~31 字符无空格；**不能配为 HTTP 标准头域名**（如 host、accept），否则防欺诈失效 |
| DATATYPE | `APN` / `IMSI1` / `MSIP1` / `MSISDN1` / `IMEI1` / `SGSNIP1` / `GGSNIP1` / `RAT1` / `ULI1` / `CHGCHAR1` / `ROAMING1` / `USERDEF1` / `TIMESTAMP1` / `UPIPV4` / `UPIPV6` / `RANDNUM` / `MULTIPARA1` 等 | 数据类型（要插入的用户信息）；同一 HEADERENNAME 多字段需多次执行 |
| PREFIXNAME | `pre-msisdn` / `X-imsi` | HTTP 头扩展字段前缀（非标准头域名）；1~31 字符；同名头增强内不可重复 |
| ENCRYALGORI | `NONE` / `MD5` / `RC4` / `AES128` / `AES256` / `RSA1024` / `RSA2048` / `SHA256` / `AES128_GCM` / `AES256_GCM` 等 | 加密算法；默认 AES128_GCM；MD5/RC4/RSA1024/AES128(CBC)/AES256(CBC)/NONE 有安全风险不建议 |
| PSWDKEY / PSWDKEYCONFIRM | `XXXXXX` | 加密密钥（ENCRYALGORI≠NONE/RSA*时必配）；两参数须一致 |
| CIPHERTXTENCODE | `INHERIT` / `NONE` / `BASE64` / `ASCII` | 密文编码方式；INHERIT 由 SET BASE64 + 软参 BIT475 控（base64/ASCII）|
| **ANTIFRAUD** | `ENABLE` / `DISABLE`（默认） | **防欺诈标识（GWFD-110401 强耦合开关）**；ENABLE=使能头防欺诈（插入前先检测/纠正已有错误字段）|
| GRAYLIST | `ENABLE` / `DISABLE`（默认） | 灰名单；ENABLE=**只防欺诈不插入头增强字段**（配合 ANTIFRAUD=ENABLE）|
| SALTSW | `ENABLE`（默认）/ `DISABLE` | 是否加盐（仅 MD5/SHA256 可配）；建议 ENABLE |
| MULTIPARA 拼接参数 | PARADATATYPE1/2/3 + SEPARATOR | DATATYPE=MULTIPARA1/2/3 时配；每字段最多 4 参数（仅 MSISDN/SGSNIP/MSIP），分隔符默认空格 |

**典型脚本（基于 URL 触发的标准部署：4 字段不加密，activation 原样）**：
```
ADD HEADEN:HEADERENNAME="header_test",DATATYPE=APN,PREFIXNAME="pre-apn",ENCRYALGORI=NONE;
ADD HEADEN:HEADERENNAME="header_test",DATATYPE=IMSI1,PREFIXNAME="pre-imsi",ENCRYALGORI=NONE;
ADD HEADEN:HEADERENNAME="header_test",DATATYPE=MSIP1,PREFIXNAME="pre-msip",ENCRYALGORI=NONE;
ADD HEADEN:HEADERENNAME="header_test",DATATYPE=MSISDN1,PREFIXNAME="pre-msisdn",ENCRYALGORI=NONE;
```

**变体脚本（防欺诈 + AES128 加密，命令 wiki 使用实例）**：
```
ADD HEADEN:HEADERENNAME="headen1",DATATYPE=MSISDN1,PREFIXNAME="TEST",ENCRYALGORI=AES128,PSWDKEY="******",PSWDKEYCONFIRM="******",ANTIFRAUD=DISABLE;
```

**步骤位置**：承接 [1-00009](task/UDG/20.15.2/1-00009.md)（过滤链，产出 FLOWFILTERNAME）+ 协议识别（WELLKNOWNPORT/SIGNATUREDB/PARSERDB）；本命令产出 HEADERENNAME，供下游 [0-00010](task/UDG/20.15.2/0-00010.md) RULE 经 `POLICYTYPE=HEADEN, POLICYNAME=<HEADERENNAME>` 直接引用（**不经 PCCPOLICYGRP**）。

## 决策点

**DP（阶段级）**：加密算法 × 编码方式（联动特性级 feature DP）。

| 选项 | 取值 | 联动 |
|---|---|---|
| 不加密 | ENCRYALGORI=NONE | 无 PSWDKEY；CIPHERTXTENCODE 无意义 |
| 对称加密（推荐安全） | ENCRYALGORI=AES128_GCM/AES256_GCM/AES128_CTR/AES256_CTR | 配 PSWDKEY；SET AES128/SET AES256 配 IV 值（HEADENRANDOMIV=ENABLE 支持随机 IV）|
| 对称加密（CBC，有风险） | ENCRYALGORI=AES128/AES256 | 不建议；同上 |
| 哈希 | ENCRYALGORI=MD5/SHA256 | 配 PSWDKEY+SALTSW；MD5 不建议；RANDNUM 可作盐 |
| 非对称 | ENCRYALGORI=RSA2048/RSA3072/RSA1024 | 配 RSA*NAME 公钥（ADD RSA2048/RSA3072 命令生成）；RSA1024 不建议 |
| 编码继承（默认） | CIPHERTXTENCODE=INHERIT | SET BASE64 控 base64；软参 BIT475 控 ASCII |
| 编码显式 base64 | CIPHERTXTENCODE=BASE64 | 命令内直接开 base64；可配等号/斜杠/加号替换字符 |
| 编码显式 ASCII | CIPHERTXTENCODE=ASCII | 命令内直接开 ASCII |

## 约束

- **PREFIXNAME 不可为 HTTP 标准头域名**（critical）：如 host、accept 等会导致防欺诈失效 — 后果：防欺诈无法识别/纠正已有字段。
- **多字段多次执行**（info）：单条 ADD HEADEN 只能配一个 DATATYPE；同 HEADERENNAME 下最多 40 种不同 DATATYPE — 后果：超规格新增失败。
- **整机规格**（warning）：≤100 个不同名称的头增强实例；≤500 个前缀字段；≤32 个前缀开防欺诈；单报文 ≤16 个前缀开防欺诈 — 后果：超规格新增失败。
- **加密密钥强度**（warning）：建议 ≥8 字符且含大小写/数字/特殊字符 ≥2 类 — 后果：弱密钥有安全风险。
- **DATATYPE=MULTIPARA 限制**（info）：每拼接字段最多 4 参数（仅 MSISDN/SGSNIP/MSIP）；PARADATATYPE1/2/3 不能同时 NONE 且任意两值不可相同。
- **BILLINGTYPE/SESSIONID 前置**（info）：插 BILLINGTYPE 需 SMF `SET PFCPPVTEXT:FEATURE=CHARGING_TYPE,SWITCH=ENABLE`；插 SESSIONID 需 `FEATURE=AUTH_CLASS,SWITCH=ENABLE`。
- **加密仅作用于字段取值**（info）：字段名称（前缀）不加密（抓包：`msisdn:` 明文，仅取值是加密后字符串），属预期设计。

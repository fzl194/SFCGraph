# 时间段控制

> BWM 配置的场景特化阶段（仅时间段场景）。一条规则绑多个时段对应多个控制器。归属特性 [2-00001](task/UDG/20.15.2/2-00001.md)。

## 配置方法

时间段两层结构：容器（TIMERANGE）+ 周期时段（PERITIMERANGE）。BWMRULE 最多绑 3 个时段（`TIMERANGENAME1/2/3`），分别对应 3 组上/下行控制器。

| 步骤 | 命令 | 关键参数 |
|---|---|---|
| 建时段容器 | `ADD TIMERANGE` | TIMERANGENAME |
| 填周期时段 | `ADD PERITIMERANGE` | TIMERANGENAME + PERITMRANGESEQ + PERITIMETYPE(DAYSOFWEEK/STARTTOEND) + DAYOFWEEK + STARTTIME/ENDTIME(HH&MM，如 `8&30`) |
| 规则绑时段（协同 [1-00003](task/UDG/20.15.2/1-00003.md)） | `ADD BWMRULE` | TIMERANGENAME1/2/3 ↔ UPBWMCTRLNAME1/2/3 + DNBWMCTRLNAME1/2/3（一一对应，第 3 个常作"其他时段"兜底） |

**典型脚本（三时段三控制器）**：
```
ADD TIMERANGE:TIMERANGENAME="timerange1";
ADD TIMERANGE:TIMERANGENAME="timerange2";
ADD PERITIMERANGE:TIMERANGENAME="timerange1",PERITMRANGESEQ=1,PERITIMETYPE=DAYSOFWEEK,DAYOFWEEK=DAILY,STARTTIME=8&30,ENDTIME=10&00;
ADD PERITIMERANGE:TIMERANGENAME="timerange2",PERITMRANGESEQ=1,PERITIMETYPE=DAYSOFWEEK,DAYOFWEEK=DAILY,STARTTIME=22&00,ENDTIME=23&00;
ADD BWMRULE:...,UPBWMCTRLNAME1="bc2",DNBWMCTRLNAME1="bc2",TIMERANGENAME1="timerange1",
  UPBWMCTRLNAME2="bc3",DNBWMCTRLNAME2="bc3",TIMERANGENAME2="timerange2",
  UPBWMCTRLNAME3="bc1",DNBWMCTRLNAME3="bc1";
```

（BWMRULE 完整见 [1-00003](task/UDG/20.15.2/1-00003.md)；时段控制器来自 [1-00002](task/UDG/20.15.2/1-00002.md)）

**步骤位置**：在 [1-00002](task/UDG/20.15.2/1-00002.md)（控制器，需按时段配多个）之后、[1-00003](task/UDG/20.15.2/1-00003.md)（规则绑时段）之中。

## 决策点

| 选项 | 影响（参数/命令/联动） |
|---|---|
| 周期类型 PERITIMETYPE=DAYSOFWEEK | 周内同天，DAYOFWEEK=DAILY/周一… |
| 周期类型 PERITIMETYPE=STARTTOEND | 跨天时段 |
| 时段-控制器映射 | `TIMERANGENAME1/2/3` ↔ `UPBWMCTRLNAME1/2/3` + `DNBWMCTRLNAME1/2/3`（一一对应，第 3 个常作"其他时段"兜底） |

## 约束

- **时段数量上限**（warning）：一条 BWMRULE 最多 3 个时段（`TIMERANGENAME1/2/3`） — 超过则无法绑定。
- **时间格式**（warning）：STARTTIME/ENDTIME 格式 `HH&MM`（如 `8&30`） — 格式错则解析失败。
- **控制器-时段匹配**（warning）：控制器数量须与时段数量匹配 — 不匹配则部分时段无控制器。

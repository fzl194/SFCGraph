# 配置APN下异常下行流量检测开关（ADD ABNTRAFFICDT）

> 命令静态知识见 [ADD ABNTRAFFICDT](command/UDG/20.15.2/ADD-ABNTRAFFICDT.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

本命令是 **GWFD-020305（激活终端异常下行流量检测）专属**。在指定 APN 下使能终端异常下行流量检测，含常规异常流检测和基于流行为的检测两档开关。前置：APN 已配（[ADD APN](command/UDG/20.15.2/ADD-APN.md)，见 [0-00020](task/UDG/20.15.2/0-00020.md)）+ License `LKV3G5ADTD01`（见 [0-00019](task/UDG/20.15.2/0-00019.md)）。

**配置范式（单范式，两开关全开）**

| 参数 | 取值 | 来源 | 说明 |
|---|---|---|---|
| APN | 已配 APN 名（如 `apn1.com`） | 引用 [0-00020](task/UDG/20.15.2/0-00020.md) | 引用已配 ADD APN |
| SWITCH | `ENABLE` | 本端规划 | 总检测开关 |
| TRAFFICBEHAVIORSWITCH | `ENABLE` | 本端规划 | 基于流行为的检测开关（命令默认 DISABLE，020305 数据规划显式 ENABLE 故按 required） |

**任务示例**

```
ADD ABNTRAFFICDT:APN="apn1.com",SWITCH=ENABLE,TRAFFICBEHAVIORSWITCH=ENABLE;
```

## 约束

- **rule-0-00332**（warning）最大记录数 10000；只对新激活用户生效（对在线用户不生效）；Normal 用户不支持异常流检测；APN 引用前置 ADD APN(0-00020)；前置 License LKV3G5ADTD01 — 后果：超规格被拒 / 对在线用户无效 / Normal 用户配异常流检测无效

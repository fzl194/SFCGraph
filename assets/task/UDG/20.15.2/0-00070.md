# 配置QoS CAR功能（SET QOSCAR）

> 命令静态知识见 [SET QOSCAR](command/UDG/20.15.2/SET-QOSCAR.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法

SET QOSCAR 是 QoS 5 件套的"CAR 全局开关层"——按"用户漫游类型 × 上/下行 RAT"组合限速：当使能 CAR 功能后，对于 E-UTRAN 接入的非 GBR 承载，系统会根据用户协商后的 APN-AMBR 对用户的报文发送速率做控制；对于 E-UTRAN 接入的 GBR 承载和其他接入方式下的承载，系统会根据用户协商后的上下行最大带宽和保证带宽对用户的报文发送速率做控制。当用户的报文发送速率超过最大带宽或 APN-AMBR 允许的值后，超过部分的数据包会被系统丢弃。本命令的核心配置空间有 3 个维度：**用户漫游类型**（USERTYPE 二选一）、**上行 RAT 类型**（ULRAT 位域）、**下行 RAT 类型**（DLRAT 位域）。

### 配置维度 1：用户漫游类型（参数 USERTYPE，必选）

| 取值 | 作用 | 典型场景 |
|---|---|---|
| **HOME** | 本地用户 | 010201 QoS 流量管理（4G 本地用户限速 eutran-1） |
| **ROAMING** | 漫游用户 | 020381 会话类 QoS 保证（避免异常大话单） |

### 配置维度 2：上行 RAT 类型（参数 ULRAT，可选）

| 取值 | 含义 |
|---|---|
| **eutran-1** | 4G LTE 接入（位域） |
| **geran-1** | 2G/GSM 接入（位域） |
| **unknown** | 未知 RAT |
| 其他位域（utran-1、ngran-1 等） | 3G/5G 接入 |

### 配置维度 3：下行 RAT 类型（参数 DLRAT，可选）

| 取值 | 含义 |
|---|---|
| **eutran-1** | 4G LTE 下行接入 |
| **geran-1** | 2G/GSM 下行接入 |
| 其他位域 | 各种接入方式 |

> ULRAT 和 DLRAT 是位域类型，可同时配置多个 RAT 类型；010201 演示 eutran-1/eutran-1（上下行对称）；020381 演示 geran-1/不配。

**典型脚本（010201 4G 本地用户限速）**：
```
SET QOSCAR: USERTYPE=HOME, ULRAT=eutran-1, DLRAT=eutran-1;
```

**典型脚本（020381 漫游用户限速）**：
```
SET QOSCAR:USERTYPE=ROAMING,ULRAT=geran-1;
```

**步骤位置**：在所属 feature 编排中位于"步骤 N：CAR 全局开关"——通常与 [0-00049](task/UDG/20.15.2/0-00049.md) SET APNQOSATTR 协同完成 QoS CAR 功能链；适用 NF 限定：SGW-U、PGW-U、UPF。

## 决策点：用户漫游类型选择（DP 0-00070）

问：SET QOSCAR 的 USERTYPE 取何值？

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| HOME（本地用户） | ULRAT/DLRAT 默认 eutran-1 | 010201 QoS 流量管理（4G 本地用户限速） |
| ROAMING（漫游用户） | ULRAT/DLRAT 按漫游接入 RAT 配 | 020381 会话类 QoS 保证（避免异常大话单） |

## 约束

- **最大记录数 3**（critical，rule-0-00285）：系统最大记录数 3 — 后果：超限 ADD 失败
- **仅对新激活用户生效**（critical，rule-0-00285）：本命令仅对新激活用户生效，对已在线用户不生效 — 后果：期望限速对存量用户不生效
- **CAR/Shaping 互斥**（critical，rule-0-00285）：CAR 与 shaping 同开时系统优先按 shaping 控制（不可同时启用 SET QOSSHAPE（★R1.5 atom 待补）） — 后果：本 atom 配置无效
- **ULRAT/DLRAT 在初始记录不生效**（warning，rule-0-00285）：首配本命令为 ADD 而非 SET；ULRAT/DLRAT 在系统初始记录中不生效，后续新增记录才作用于 RAT 匹配 — 后果：首配记录 RAT 维度不生效
- **开启会导致性能下降**（warning，rule-0-00285）：容量评估需考虑 PERF 影响 — 后果：高负载场景性能下降
- **适用 NF 限定**（warning，隐含）：SGW-U、PGW-U、UPF — 后果：SGW-C 等控制面不适用

# 查询用户会话上下文（DSP SESSIONINFO）

> 命令静态知识见 [DSP SESSIONINFO](command/UDG/20.15.2/DSP-SESSIONINFO.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法（配置方法字典）

DSP SESSIONINFO 是用户面**会话上下文查询**命令，输出含 IPv4 PDP address / IPv4 Address type / APN name / IMSI 等关键字段，是 GWFD-010105 地址分配调测的核心命令（步骤 2/3 验证）。命令核心配置空间有 2 个维度：**查询类型**（QUERYTYPE 决定后续必选参数集）、**关键标识**（按 QUERYTYPE 类型条件必选）。

### 配置维度 1：查询类型（参数 QUERYTYPE，必选）

| 取值 | 必选参数 | 输出关键字段 |
|---|---|---|
| **IMSI** | IMSI | IMSI / PDP Type / IPv4 PDP address / APN name / IPv4 Address type |
| **IPV4** | IPV4ADDR | IPv4 地址所属会话信息 |
| **IPV6** | IPV6ADDR | IPv6 地址所属会话信息 |
| **MSISDN** | MSISDN | MSISDN 所属会话信息 |

### 配置维度 2：关键标识（按 QUERYTYPE 条件必选）

| 参数 | QUERYTYPE | 范围 |
|---|---|---|
| **IMSI** | IMSI | 字符串（010105 调测范式：`460000123456789`） |
| **IPV4ADDR** | IPV4 | IPv4 地址 |
| **IPV6ADDR** | IPV6 | IPv6 地址 |
| **MSISDN** | MSISDN | 用户 MSISDN |

**典型脚本（010105 调测步骤 2：按 IMSI 查询会话上下文）**：
```
DSP SESSIONINFO:QUERYTYPE=IMSI,IMSI="460000123456789";
```

**典型脚本（010105 调测步骤 3：再次查询 IPv4 Address type）**：
```
DSP SESSIONINFO:QUERYTYPE=IMSI,IMSI="460000123456789";
```

**预期输出关键字段**：
- `IPv4 PDP address` = `10.10.10.1`（应在规划地址池内）
- `IPv4 Address type` = `UPF ALLOC IP ADDRESS`（用户面本地分配）/ `EXTERNAL ALLOC IP ADDRESS`（外部地址分配）
- `APN name` = `apn-test1`

> 命令 notes：适用 NF：PGW-U、UPF；QUERYTYPE 与对应查询参数强绑定；IPv4 Address type 是判断"本地/外部分配"的关键字段。

**步骤位置**：在所属 feature 编排中通常位于"调测/验证阶段"——前置条件：用户已发起激活；用于 GWFD-010105 地址分配调测的核心命令。

## 决策点：查询维度选择（DP 0-00327）

问：根据已知信息选择查询维度？

| 选项 | QUERYTYPE | 联动参数 | 适用场景 |
|---|---|---|---|
| 按 IMSI 查 | IMSI | IMSI | 已知用户 IMSI，010105 调测主路径 |
| 按 IPv4 查 | IPV4 | IPV4ADDR | 已知用户 IPv4，反查会话 |
| 按 IPv6 查 | IPV6 | IPV6ADDR | 已知用户 IPv6，反查会话（双栈场景） |
| 按 MSISDN 查 | MSISDN | MSISDN | 已知用户 MSISDN，反查会话 |

## 约束

- **QUERYTYPE 与查询参数绑定**（critical，rule-0-00486）：QUERYTYPE 选定后必须配对应参数（IMSI/IPV4/IPV6/MSISDN） — 后果：缺参数命令拒收
- **IMSI 前置条件**（critical，rule-0-00486）：用户必须已发起激活且未去活，否则查不到 — 后果：空结果 (Number of results = 0)
- **适用 NF 限定**（warning，隐含）：PGW-U、UPF — 后果：SGW-U 等不适用
- **调测依赖 QUERYTYPE=IMSI + IMSI**（warning，rule-0-00486）：010105 调测主路径，无其他选项替代 — 后果：按其他维度查不能定位"IPv4 Address type=UPF/EXTERNAL"关键字段

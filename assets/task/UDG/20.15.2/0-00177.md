# 设置APN Non-IP配置（SET APNNONIPFUNC）

> 命令静态知识见 [SET APNNONIPFUNC](command/UDG/20.15.2/SET-APNNONIPFUNC.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

本命令把 Non-IP 数据传输从**全局场景收敛到非全局场景**：一旦配置，只有 NONIPSWITCH=ENABLE 的 APN 才具备 Non-IP 功能。UDG 作 PGW-U 时必须配本命令。控制面需配相同 APN 并使能 Non-IP 开关（不在 UDG 命令域）。

**配置范式（GWFD-110607 Non-IP 数据传输）**

| APN（引用） | NONIPSWITCH | USERPORT | 示例行 |
|---|---|---|---|
| apnnonip（已配 ADD APN） | ENABLE | 1010 | `SET APNNONIPFUNC:APN="apnnonip",NONIPSWITCH=ENABLE,USERPORT=1010;` |

**参数取值约定**

| 参数 | 取值 | 来源 | 说明 |
|---|---|---|---|
| APN | 字符串 1~64 | 引用 [ADD APN](task/UDG/20.15.2/0-00020.md) 已配置数据 | 必须先经 ADD APN 配置 |
| NONIPSWITCH | ENABLE / DISABLE（系统初始 DISABLE） | 本端规划 | ENABLE=该 APN 开启 Non-IP 功能。配本命令即转入非全局场景 |
| USERPORT | 整数 1~65535（系统初始 5683） | 与对端协商 | Non-IP 用户面端口号，需与控制面/对端一致 |

**全局/非全局场景语义**

- **配置前（全局场景）**：打开 License → 全局默认开启 Non-IP；不打开 License → 全局关闭。
- **配置后（非全局场景）**：必须先打开 License（[0-00019](task/UDG/20.15.2/0-00019.md) SET LICENSESWITCH，LKV3G5NOIP01），再用本命令按 APN 粒度开启。

## 约束

- **rule-0-00070**（warning）必须先 SET LICENSESWITCH(LKV3G5NOIP01) 打开 License 本命令才生效；本命令一旦配置即由全局场景转入非全局场景（仅 NONIPSWITCH=ENABLE 的 APN 生效）；APN 必须先经 ADD APN 配置 — 后果：License 未开/未配 APN→命令不生效

# 设置APN Radius服务器组（ADD APNRDSSVRGRP）

> 命令静态知识见 [ADD APNRDSSVRGRP](command/UNC/20.15.2/ADD-APNRDSSVRGRP.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

该命令将一个已存在的 RADIUS 服务器组（由 `ADD RDSSVRGRP` 建立）绑定到指定 APN 实例（由 `ADD APN` 建立），用于对接 AAA Server 完成用户鉴权/计费。配置空间主要由**是否抄送中间话单**这一维度决定。

### 配置维度1：抄送中间话单（参数 COPYINTERIMUPD）

| 取值 | 作用 | 典型场景 |
|---|---|---|
| DISABLE（默认） | 不抄送中间话单 | 常规 RADIUS 鉴权计费 |
| ENABLE | 抄送 Interim-Update Request 中间话单 | 激活支持 RADIUS 抄送功能；AAA 负荷分担场景下的多份话单 |

### 配置维度2：APN 与服务器组的绑定关系（参数 APN / RDSSVRGRPNAME）

| 参数 | 作用 | 约束 |
|---|---|---|
| APN | 指定 APN 实例名称 | 必须是 `ADD APN` 已配置的 APN |
| RDSSVRGRPNAME | RADIUS 服务器组名称 | 必须是 `ADD RDSSVRGRP` 已配置的服务器组 |

> 一个 APN 只能绑定一个 RADIUS 服务器组。

**步骤位置**：在所属 feature 编排中，于 `ADD APN`（建 APN 实例）+ `ADD RDSSVRGRP`（建 RADIUS 服务器组）之后执行。

## 决策点

**是否启用抄送**（UNC pre-build 内嵌，不编号——按 `assets/CLAUDE.md` §5.7）：

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| DISABLE | COPYINTERIMUPD=DISABLE | WSFD-011306 Radius功能（基础鉴权计费） |
| ENABLE | COPYINTERIMUPD=ENABLE | WSFD-011307 支持Radius抄送功能；WSFD-011308 AAA负荷分担 |

## 约束

- **立即生效**（info）：命令执行后立即生效 — 后果：无延迟
- **一个 APN 只能绑一个组**（critical）：一个 APN 只能绑定一个 RADIUS 服务器组 — 后果：重复绑定失败
- **最多 10000 条记录**（warning）：本命令最大记录数为 10000 — 后果：超规格新增失败

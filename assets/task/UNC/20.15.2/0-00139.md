# 删除Diameter链路（RMV DIAMCONNECTION）

> 命令静态知识见 [RMV DIAMCONNECTION](command/UNC/20.15.2/RMV-DIAMCONNECTION.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法

按"链路组 + 本端接口 + 对端地址端口"组合定位删除；`LOCALPORT` 三态语义不同，需显式规划；调测场景常成对 `RMV DIAMLOCINFO` 重建 Gx 设备标识。

### 参数维度表

| 维度 | 参数 | 数据来源 | 取值 | 作用 | 适用场景 |
|------|------|----------|------|------|----------|
| 链路组名 | DIAMCONNGRP | 本端规划 | 字符串 1～31，不支持空格、不区分大小写 | 定位链路所属组；指定其它参数时必选 | 精确删指定组下链路 |
| 本端接口 | LOCINTERFACE | 本端规划 | 字符串 1～31，不支持空格、不区分大小写 | 收窄到本端某接口 | 进一步精确定位 |
| 对端地址类型 | PEERADDRTYPE | 对端协商 | IPv4 / IPv6 / SCTP | 决定用哪种对端标识匹配 | 删除特定地址类型链路 |
| 对端 IPv4 | PEERIPV4ADDR | 对端协商 | IPv4 地址（不含 0.0.0.0、255.255.255.255） | PEERADDRTYPE=IPv4 时必选 | 删 IPv4 链路 |
| 对端 IPv6 | PEERIPV6ADDR | 对端协商 | IPv6 地址（不含全 F） | PEERADDRTYPE=IPv6 时必选 | 删 IPv6 链路 |
| 对端端口 | PEERPORT | 对端协商 | 1～65534 | IPv4/IPv6 时可选收窄 | 精确删某端口链路 |
| 对端 SCTP 端点名 | PEERSCTPENDPT | 对端协商 | 字符串 1～63，不支持空格、不区分大小写 | PEERADDRTYPE=SCTP 时必选 | 删 SCTP 链路 |
| 本端端口 | LOCALPORT | 本端规划 | 0 / 13200～13263（Gy）/ 16400～16463（Gx）/ 19765～19784（S6b） | 三态语义不同，见 DP 0-03018 | 按端口范围过滤 |

> 说明：高危命令。全部参数都不指定会**删除所有记录**；未指定 `DIAMCONNGRP` 时禁止执行（除非软参 BYTE976=169）。

### 典型脚本

删指定链路组下一条 Gx 链路：

```
RMV DIAMCONNECTION:DIAMCONNGRP="Grp1",LOCINTERFACE="gxif1/0/0";
```

按对端地址端口精确删：

```
RMV DIAMCONNECTION: DIAMCONNGRP=gx-grp, PEERIPV4ADDR=10.1.1.2, PEERPORT=3868;
```

### 步骤位置

Diameter 链路删除/重建流程：
1. `LST DIAMCONNECTION` — 核对目标链路是否承载在用会话
2. `RMV DIAMCONNECTION` — 删除链路（本案）
3. 调测场景成对：`RMV DIAMLOCINFO` 删除原 Gx 设备标识
4. 如需重建：`ADD DIAMLOCINFO` + `ADD DIAMCONNECTION`

## 决策点

### 决策点：删除范围（精确单条 vs 全删）（DP 0-03017）

| 选项 | 联动参数 | 适用场景 |
|------|----------|----------|
| 精确删单条 | `DIAMCONNGRP` + 对端地址端口等组合锁定 | 调测/运维常规场景 |
| 全删 | 所有参数都不指定 | 极端回收，生产环境禁止裸执行 |

> 注意：全删参数等同留空 = 删除所有记录，属高危；未指定 `DIAMCONNGRP` 时禁止执行，若需执行需将软参 BYTE976 设为 169。删前必须确认链路未承载在用会话，否则可能导致用户被去活等业务影响。

### 决策点：LOCALPORT 三态语义（DP 0-03018）

| 选项 | 联动参数 | 适用场景 |
|------|----------|----------|
| 省略 LOCALPORT | 删除由其它参数锁定的所有 CONNECTION 记录（含系统指定端口和用户指定端口） | 仅按其它参数过滤 |
| LOCALPORT=0 | 只删系统指定端口的 CONNECTION 记录 | 清理系统默认端口链路 |
| LOCALPORT=具体值 | 删该具体端口（13200～13263 / 16400～16463 / 19765～19784）的记录 | 精确删某应用端口链路 |

> 注意：端口范围按应用区分：Gx 应用 16400～16463、Gy 应用 13200～13263、S6b 应用 19765～19784；配置/删除时按应用选择正确端口段。

## 约束

- **全删禁止**（critical，rule-0-03041）：全删禁止：所有参数都不指定会删除全部记录，生产环境禁止裸执行；若需在未指定 DIAMCONNGRP 时执行，需将软参 BYTE976 设为 169
- **链路组名前置**（critical，rule-0-03042）：链路组名前置：未指定 DIAMCONNGRP 时禁止执行（除非软参 BYTE976=169）；指定其它参数则必须同时指定 DIAMCONNGRP
- **高危业务影响**（warning，rule-0-03043）：高危业务影响：删除 Diameter 链路可能导致链路中断、用户被去活等业务影响，删前必须确认链路未承载在用会话
- **立即生效**（warning，rule-0-03044）：立即生效：执行后立即生效，不可回滚；多实例并发删除需评估对在用业务的影响
- **配套删除**（info，rule-0-03045）：配套删除：调测场景常成对使用本命令 + RMV DIAMLOCINFO 删除原配置，再 ADD DIAMLOCINFO + ADD DIAMCONNECTION 重建

# 增加中转UPF与RADIUS客户端绑定（ADD UPFRDSCLIENTIP）

> 命令静态知识见 [ADD UPFRDSCLIENTIP](command/UNC/20.15.2/ADD-UPFRDSCLIENTIP.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

该命令配置 UPF 中转 RADIUS 的客户端绑定（客户端类型 + IP + VPN），SMF 将该 Client IP 下发至 UPF，UPF 据此转发 RADIUS 消息。配置空间核心维度为 **客户端类型**（CLIENTYPE）和 **IP 版本**（IPVERSION），绑定到指定 UP 列表。

### 配置维度1：客户端类型（参数 CLIENTYPE）

| 取值 | 作用 | 典型场景 |
|---|---|---|
| AUTHENTICATION | 鉴权客户端（UPF 中转鉴权 RADIUS） | WSFD-108007 终端二次鉴权（CLIENTYPE=AUTHENTICATION） |
| ACCOUNTING | 计费客户端（UPF 中转计费 RADIUS） | 中转计费 |
| ACCT_AND_AUTH | 计费与鉴权共用客户端 | 单客户端同时承载鉴权+计费 |

### 配置维度2：IP 版本（参数 IPVERSION）

| 取值 | 作用 | 配套必选参数 | 典型场景 |
|---|---|---|---|
| IPV4 | 客户端 IPv4 | CLIENTIPV4 | WSFD-108007：CLIENTIPV4="192.168.10.1" |
| IPV6 | 客户端 IPv6 | CLIENTIPV6 | IPv6 中转场景 |

### 配置维度3：UP 列表 × VPN 实例 × 网络实例来源（参数 UPLISTNAME / VPNINSTANCE / NETINSTNAMESRC）

| 取值 | 作用 | 典型场景 |
|---|---|---|
| UPLISTNAME（已有 `ADD UPLIST4RDS`） | 该客户端绑定归属的 UP 列表 | WSFD-108007：UPLISTNAME="uplist1" |
| VPNINSTANCE（可选） | UPF 与 DN-AAA 对接的 VPN 实例 | WSFD-108007：VPNINSTANCE="vpnInstance1" |
| NETINSTNAMESRC=VPN（默认）| 网络实例名 = VPN 实例名（空时为 _public_） | 默认 |
| NETINSTNAMESRC=Specified + SPECNETINSTNAME | 显式指定网络实例名（用于 PFCP 会话下行检测规则） | 精细控制 PFCP 下行规则 |

**步骤位置**：在 WSFD-108007 终端二次鉴权编排中，于 `ADD UPLIST4RDS`（建 UP 列表）之后执行，常与 `ADD UPFRDSSVR`（UPF 中转 RADIUS 服务器）配套。

## 决策点

**客户端类型选择**（UNC pre-build 内嵌，不编号——按 `assets/CLAUDE.md` §5.7）：

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| AUTHENTICATION | + IPVERSION + CLIENTIPV4/V6 + UPLISTNAME | 鉴权中转（WSFD-108007） |
| ACCOUNTING | 同上 | 计费中转 |
| ACCT_AND_AUTH | 同上 | 鉴权计费共客户端 |

> 网络实例来源：默认 NETINSTNAMESRC=VPN（与 VPNINSTANCE 一致），需精细控制 PFCP 会话时改为 Specified + SPECNETINSTNAME。

## 约束

- **立即生效**（info）：命令执行后立即生效 — 后果：无回滚窗口
- **最大记录数6000**（warning）：最多可输入 6000 条记录（超 98% 上报告警，≤95% 恢复） — 后果：超规格新增失败
- **UPLISTNAME 必须先建**（critical）：UPLISTNAME 必须由 `ADD UPLIST4RDS` 已配置 — 后果：列表不存在则新增失败
- **CLIENTIPV4/V6 条件必选**（critical）：IPVERSION=IPV4 必填 CLIENTIPV4；IPV6 必填 CLIENTIPV6 — 后果：缺则新增失败
- **VPNINSTANCE 必须先建**（critical）：VPNINSTANCE 若填必须由 `ADD VPNINST` 已配置 — 后果：VPN 不存在新增失败
- **SPECNETINSTNAME 条件必选**（critical）：NETINSTNAMESRC=Specified 时必填 SPECNETINSTNAME — 后果：缺则新增失败
- **SPECNETINSTNAME 字符约束**（warning）：字符串 1~100，仅字母/数字/`.`/`_`/`-`，大小写不敏感 — 后果：违规命名新增失败

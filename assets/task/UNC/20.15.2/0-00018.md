# ADD CHGDCHAR 原子任务

> 命令静态知识见 [ADD CHGDCHAR](command/UNC/20.15.2/ADD-CHGDCHAR.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

ADD CHGDCHAR 为 SGSN 离线计费登记一条"缺省计费属性"。当外网漫游用户的 HLR 没有为其指定计费属性时，SGSN 按用户的 MCC（移动国家码）+ MNC（移动网号）匹配本表，用登记的缺省计费属性（DCC）对其计费。配置时按整网规划给出用户类型 ST、MCC、MNC，并指定缺省属性。

**参数取值样例（漫游用户缺省计费属性范式）**

| 参数 | 取值样例 | 默认 | 说明 |
|---|---|---|---|
| ST | `ROAMING` | — | 用户类型（必填）；ROAMING=使用归属 PLMN GGSN 的外网用户 |
| MCC | `460` | — | 移动国家码，3 位十进制（必填） |
| MNC | `00` / `001` | — | 移动网号，2 或 3 位十进制（必填） |
| DCC | `NORMAL` | NORMAL | 缺省计费属性；NORMAL=普通计费 |

**典型行（登记一条漫游用户普通计费缺省属性）**

```
ADD CHGDCHAR: ST=ROAMING, MCC=460, MNC=00, DCC=NORMAL;
```

**配置位置**：在"配置离线计费参数（SGSN）"流程中作为第 10 步可选项，用于为特定 PLMN 配置缺省计费属性。

## 决策点

### 是否需要配置缺省计费属性

| 选项 | 适用 |
|---|---|
| 配置 | 现网存在外网漫游用户且其 HLR 未指定计费属性，需 SGSN 按归属 PLMN 兜底计费 |
| 不配 | 仅服务本网用户，或外网用户 HLR 已指定计费属性 |

## 约束

- **立即生效** · severity=LOW。
- **同 ST 记录最多 10 条，本表总数不超过 20** · severity=MEDIUM · 超过上限新增失败。
- **MNC 唯一性**：相同 ST 且相同 MCC 下，有效长度 2 和 3 的 MNC 前两位不允许相同 · severity=HIGH · 违反导致匹配二义性，新增失败。
- **计费属性选择优先级**：S-CDR 填写的计费属性按多级优先级选择，缺省属性是兜底层，LST CHGGNCCCFG 等更细粒度策略优先 · severity=MEDIUM。

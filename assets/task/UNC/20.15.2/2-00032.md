# 别名APN（WSFD-106203）

> 特性静态知识见 [WSFD-106203](feature/UNC/20.15.2/WSFD-106203.md)。本页讲配置生成实例化时这个特性**怎么配**——编排哪些步骤、两代际（SGSN/MME vs GGSN/PGW-C/SMF）如何选。

## 配置概览

核心对象链：**真实 APN**（`ADD APN` → [0-00004](task/UNC/20.15.2/0-00004.md)）→ **别名映射**（`ADD APNALIAS` [0-00220](task/UNC/20.15.2/0-00220.md) 或 `ADD ALIASAPN`，依 NF 而异）→ 可选 **APN 上报属性**（`SET APNREPORTATTR` [0-00152](task/UNC/20.15.2/0-00152.md) PCF/CHF=SERVICE）。

两代际**共享接入域骨架**（[UNC 1-00024](task/UNC/20.15.2/1-00024.md)），差异在命令族（`ADD APNALIAS` vs `ADD ALIASAPN`）+ 映射方向（SGSN/MME = 协商APN→别名APN；GGSN/PGW-C/SMF = 别名APN→真实APN）+ License 项（LKV2ALIASAPN02 vs LKV2AAPN01）。由特性级 DP（NF 代际选择）驱动。

**License 信息**：需 [SET LICENSESWITCH](task/UNC/20.15.2/0-00180.md) 前置 — SGSN/MME 配 `LKV2ALIASAPN02`，GGSN/PGW-C/SMF 配 `LKV2AAPN01`。

## 配置流程（步骤编排，每步显示含命令）

1. **接入域基础设施**（L3VPN + VPN 实例 + 真实 APN 实例）：`ADD L3VPNINST` + `ADD VPNINSTAF` + `ADD VPNINST` + `ADD APN` → [UNC 1-00024](task/UNC/20.15.2/1-00024.md)（含 ADD APN 步骤）
2. **License 前置**（按 NF 选对应控制项）：`SET LICENSESWITCH: LICITEM=LKV2ALIASAPN02/LKV2AAPN01, SWITCH=ENABLE` → [0-00180](task/UNC/20.15.2/0-00180.md)
3. **APN 上报属性**（真实 APN 配 PCF/CHF=SERVICE，确保下游按真实 APN 上报）：`SET APNREPORTATTR` → [0-00152](task/UNC/20.15.2/0-00152.md)
4. **别名映射**（按 NF 代际选命令）：
   - **GGSN/PGW-C/SMF**：`ADD APNALIAS`（`SUBRANGE` + `ALIASAPN` + `CONVERTAPN`，可附 `SST`+`SD` 切片维度）→ [0-00220](task/UNC/20.15.2/0-00220.md)
   - **SGSN/MME**：`ADD ALIASAPN`（`SUBRANGE=IMSI_PREFIX` + `IMSIPRE` + `OLDAPN` + `NEWAPN`，按 IMSI 最长匹配）
     - 关键参数：SUBRANGE=IMSI_PREFIX, IMSIPRE=IMSI 前缀, OLDAPN=用户激活请求 APN, NEWAPN=别名 APN
     - **★ R1.5 待补 atom**：`ADD ALIASAPN` 命令真实存在 UNC OM（`业务服务管理/Pre 5G接入业务管理/控制面管理/`），UNC pre-build 阶段未建 atom；配置生成时**待 atom 补齐后**才产出。
5. **可选·锁定 + 批量去活 + 删除**（运维操作）：`LCK APNALIAS` + `DEA SMCTX`（`APNTYPE=ALIAS_APN`）+ `SET DEACTIVERATE` + `RMV APNALIAS`
   - 关键参数：`LCK APNALIAS: APN` / `DEA SMCTX: APNTYPE=ALIAS_APN` / `RMV APNALIAS: APN`
   - **★ R1.5 待补 atom**：`LCK APNALIAS`（Pre-5G 别名APN 锁定）/ `RMV APNALIAS`（Pre-5G 别名APN 删除）/`SET DEACTIVERATE`（去激活速率）命令真实存在 UNC OM 文档（`业务服务管理/Pre 5G接入业务管理/控制面管理/`），atom 未建。
   - **`LCK APNALIAS` 命令文档路径**：`业务服务管理/Pre 5G接入业务管理/控制面管理/.../锁定APN别名配置(LCK APNALIAS)_28567625.md`
   - **`RMV APNALIAS` 命令文档路径**：`.../删除APN别名配置(RMV APNALIAS)_28567655.md`

> **★ 代际差异驱动 DP（按 NF 选）**：
> - **GGSN/PGW-C/SMF 代际**：别名 APN 是用户激活请求 APN，`CONVERTAPN` 是真实 APN；用户激活时 UNC 把请求 APN 映射到真实 APN，使用真实 APN 上的业务。
> - **SGSN/MME 代际**：别名 APN 是 UNC 协商后映射的输出 APN（用于 DNS 解析），`OLDAPN` 是用户请求协商后的 APN，`NEWAPN` 是别名 APN；S-CDR 与后续 PDP 消息携带别名 APN。
> - **切片维度（GGSN/PGW-C/SMF 专属）**：`SUBRANGE=SPECIFIC_NS` 时附 `SST`+`SD`；不同切片下相同 ALIASAPN 可映射到不同 CONVERTAPN（5G 特性）。

## 决策点（特性级：NF 代际 × 映射模式）

| NF × 映射模式 | 走法（复用 compound）| 关键差异 |
|---|---|---|
| **GGSN/PGW-C/SMF × ALL_USER（最常用）**| 骨架 [1-00024](task/UNC/20.15.2/1-00024.md) + License LKV2AAPN01 + SET APNREPORTATTR + ADD APNALIAS(SUBRANGE=ALL_USER) | 别名APN→真实APN，单一映射 |
| **GGSN/PGW-C/SMF × SPECIFIC_NS（5G 切片维度）**| 骨架 1-00024 + License LKV2AAPN01 + SET APNREPORTATTR + ADD APNALIAS(SUBRANGE=SPECIFIC_NS, SST, SD) | 5G 用户，不同切片下可映射到不同真实 APN |
| **SGSN/MME × ALL_USER（按 IMSI 前缀）**| 骨架 1-00024 + License LKV2ALIASAPN02 + ADD ALIASAPN(SUBRANGE=IMSI_PREFIX, IMSIPRE, OLDAPN, NEWAPN) | IMSI 最长匹配；2/3G SGSN 仅 UNC 20.3.0+；4G MME UNC 20.0.0+ |
| **删除别名 APN（运维）**| LCK APNALIAS → DEA SMCTX(APNTYPE=ALIAS_APN) → RMV APNALIAS | 存在用户/会话时直接 RMV 被拒；须先锁定、批量去活、再删除 |

> **走法列**：两代际**共享骨架 [1-00024](task/UNC/20.15.2/1-00024.md)**（必先建真实 APN）；GGSN/PGW-C/SMF 用 [0-00220](task/UNC/20.15.2/0-00220.md) ADD APNALIAS，SGSN/MME 用 Pre-5G 命令族 ADD ALIASAPN（独立命名空间，atom 待补）。

## 约束（特性级 rule）

- **真实 APN 与别名 APN 不能重名**（critical）：`ADD APN` 的 APN 与 `ADD APNALIAS` 的 ALIASAPN（`ADD ALIASAPN` 的 NEWAPN）不可重复 — 后果：命名冲突配置失败。
- **CONVERTAPN/OLDAPN 必须已 `ADD APN`**（critical）：`ADD APNALIAS` 的 CONVERTAPN 必须先建真实 APN；`ADD ALIASAPN` 的 OLDAPN 是协商 APN — 后果：引用未建资源配置失败。
- **别名 APN 多对一但不可反向**（warning）：同一真实 APN 下可挂多个别名 APN，但同一别名 APN 不可映射到多个真实 APN（5G 切片下例外：`SPECIFIC_NS` 不同 SST/SD 下别名 APN 可映射到不同真实 APN）。
- **系统上限 1000 条 / APN 下最多 500 条**（warning）：系统最大 1000 个 APN 别名；单个真实 APN 最多 500 个别名。
- **删除别名 APN 须先清空用户**（critical）：存在用户/会话时 `RMV APNALIAS` 直接被 UNC 拒绝 — 后果：未按锁定+批量去活顺序时删除失败；先 `LCK APNALIAS` + `DEA SMCTX(APNTYPE=ALIAS_APN)` 再删除。
- **S-CDR 携带别名 APN**（warning，SGSN/MME 代际专属）：S-CDR 与后续 PDP 激活消息携带别名 APN（系统默认，不可通过参数修改）— 后果：CDR 上报侧需按别名 APN 做计费关联。
- **请求信息纠正特性前置**（info，SGSN/MME 代际）：[WSFD-106004](feature/UNC/20.15.2/WSFD-106004.md) 请求信息纠正功能先于别名 APN 执行（用户激活消息携带错误 APN 时）。
- **License 控制项必配**（critical）：未 `SET LICENSESWITCH` 启用 LKV2ALIASAPN02 / LKV2AAPN01 时，本特性功能不可用 — 后果：别名 APN 映射不生效。

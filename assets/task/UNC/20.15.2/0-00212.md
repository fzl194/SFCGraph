# 修改用户模板组绑定（MOD UPBINDUPG）

> 命令静态知识见 [MOD UPBINDUPG](command/UNC/20.15.2/MOD-UPBINDUPG.md)。本页讲实例化怎么配。

## 配置方法

| 参数 | 取值样例 | 来源 |
|---|---|---|
| USERPROFGNAME | upg_test1 | 已配置数据中获取（[ADD USRPROFGROUP](task/UNC/20.15.2/0-00096.md)）|
| UPBINDTYPE | SPECIFIC | 固定取值 |
| USERPROFILENAME | up_test1 | 已配置数据中获取（[ADD USERPROFILE](task/UNC/20.15.2/0-00094.md)）|
| LOCGROUPNAME | locgrp_test | 已配置数据中获取（[ADD USRLOCATIONGRP](task/UNC/20.15.2/0-00211.md)）|

**典型脚本**：
```
MOD UPBINDUPG:USERPROFGNAME="upg_test1",UPBINDTYPE=SPECIFIC,USERPROFILENAME="up_test1",LOCGROUPNAME="locgrp_test";
```

## 决策点

| 选项 | 影响 |
|---|---|
| 用户模板已绑用户模板组 | 用 MOD UPBINDUPG 追加位置组 |
| 未绑 | 用 [ADD UPBINDUPG](task/UNC/20.15.2/0-00089.md) 一次绑定模板+位置组 |

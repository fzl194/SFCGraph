# OFCTemplate 绑定 UserProfile

> 阶段定位：离线计费参数绑到 UserProfile 层（4 层次中优先级最高）。被引用于：[2-00001 离线计费](task/UNC/20.15.2/2-00001.md)。
> 域通用 compound（在线/融合计费模板绑 UserProfile 同此结构，后续特性可复用）。

## 配置方法

命令序列（建 UserProfile → 绑模板 → 组 → 组绑 APN）：

1. [ADD USERPROFILE](task/UNC/20.15.2/0-00094.md) — 用户模板（USERPROFILENAME 全网规划，UPTYPE=PCC）；已配则跳过
2. [SET USRPROFCHARGE](task/UNC/20.15.2/0-00207.md) — UserProfile 绑 OFCTemplate（PGWOFCTEMPLATE/SGWOFCTEMPLATE 引用模板名）
3. [ADD USRPROFGROUP](task/UNC/20.15.2/0-00096.md) — 用户模板组容器
4. [ADD UPBINDUPG](task/UNC/20.15.2/0-00089.md) — UserProfile 绑到 USRPROFGROUP
5. [ADD APNUSRPROFG](task/UNC/20.15.2/0-00008.md) — USRPROFGROUP 绑到 APN 实例

**典型脚本**
```
ADD USERPROFILE: USERPROFILENAME="up-test", UPTYPE=PCC;
SET USRPROFCHARGE: USRPROFNAME="up-test", PGWOFCTEMPLATE="offlinecharge-test", SGWOFCTEMPLATE="offlinecharge-test";
ADD USRPROFGROUP: USERPROFGNAME="upg-test";
ADD UPBINDUPG: USERPROFGNAME="upg-test", USERPROFILENAME="up-test";
ADD APNUSRPROFG: APN="apn-test", USERPROFGNAME="upg-test";
```

## 约束
- 前置：[1-00001 模板参数](task/UNC/20.15.2/1-00001.md)（OFCTEMPLATENAME 须已存在）
- USERPROFILENAME/USERPROFGNAME 全网规划（跨网元一致）

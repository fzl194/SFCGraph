# OFCTemplate 绑定 UserProfile

> 离线计费参数绑到 UserProfile 层（4 层次优先级最高）。被引用于 [2-00001 离线计费](task/UNC/20.15.2/2-00001.md)。域通用（在线/融合计费模板绑 UserProfile 同此结构）。

## 配置方法

| 步骤 | 命令 | 关键参数 |
|---|---|---|
| 建用户模板（已配可跳过） | `ADD USERPROFILE` | USERPROFILENAME（全网规划）；UPTYPE=PCC |
| 模板绑 UserProfile | `SET USRPROFCHARGE` | USRPROFNAME；PGWOFCTEMPLATE/SGWOFCTEMPLATE=模板名 |
| 建用户模板组 | `ADD USRPROFGROUP` | USERPROFGNAME（全网规划） |
| 模板绑到组 | `ADD UPBINDUPG` | USERPROFGNAME + USERPROFILENAME |
| 组绑到 APN | `ADD APNUSRPROFG` | APN + USERPROFGNAME |

**典型脚本**：
```
ADD USERPROFILE: USERPROFILENAME="up-test", UPTYPE=PCC;
SET USRPROFCHARGE: USRPROFNAME="up-test", PGWOFCTEMPLATE="offlinecharge-test", SGWOFCTEMPLATE="offlinecharge-test";
ADD USRPROFGROUP: USERPROFGNAME="upg-test";
ADD UPBINDUPG: USERPROFGNAME="upg-test", USERPROFILENAME="up-test";
ADD APNUSRPROFG: APN="apn-test", USERPROFGNAME="upg-test";
```

**步骤位置**：紧跟 [1-00001](task/UNC/20.15.2/1-00001.md)（模板参数）之后；仅当特性级 DP 选 UserProfile 层时编排。

## 场景差异

- **热计费（WSFD-011202）**：无参数增量，仅模板名引用热计费模板（阈值更小）。activation 未单独演示 UserProfile 层，结构复用离线计费。被引用于 [2-00002 热计费](task/UNC/20.15.2/2-00002.md)。

## 约束

- **前置模板**（critical）：OFCTEMPLATENAME 须已在 [1-00001](task/UNC/20.15.2/1-00001.md) 建立 — 否则 SET USRPROFCHARGE 绑定失败。
- **名称跨网元一致**（warning）：USERPROFILENAME/USERPROFGNAME 全网规划。

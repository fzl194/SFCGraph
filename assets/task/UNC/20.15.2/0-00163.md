# SET CHGPLMNCHAR 原子任务
> 命令静态知识见 [SET CHGPLMNCHAR](command/UNC/20.15.2/SET-CHGPLMNCHAR.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法
SET CHGPLMNCHAR 用于 SGSN 按 PLMN 类型（`PLMN=HPLMN` 本网 / `VPLMN` 拜访）全局开关 8 类话单：`MP`(M-CDR)、`SP`(S-CDR)、`SMOP`(S-SMO-CDR)、`SMTP`(S-SMT-CDR)、`LCSMOP`(LCS-MO-CDR)、`LCSMTP`(LCS-MT-CDR)、`LCSNIP`(LCS-NI-CDR)。本命令是**全局总闸**：一旦本命令禁止生成某种话单，无论 ADD CHGPLMNCFG/ADD CHGIMSICFG/SET CHGCHAR 如何配置，该话单都不生成。

| PLMN 类型 | 关键参数 | 典型脚本 |
|---|---|---|
| 漫游 VPLMN 用户抑制 LCS-NI 话单 | `PLMN=VPLMN` + `LCSNIP=NO`（其余话单默认 YES） | `SET CHGPLMNCHAR:PLMN=VPLMN,LCSNIP=NO;` |
| 漫游 VPLMN 用户开启全部话单 | 7 类话单全 `YES` | `SET CHGPLMNCHAR:PLMN=VPLMN,MP=YES,SP=YES,SMOP=YES,SMTP=YES,LCSMOP=YES,LCSMTP=YES,LCSNIP=YES;` |
| 全局 S-CDR 生成（供细分策略生效） | `PLMN=VPLMN` + `SP=YES` | `SET CHGPLMNCHAR:PLMN=VPLMN,SP=YES;` |

**话单抑制优先级（仅对 S-CDR 举例）**：SET CHGPLMNCHAR 全局策略 > ADD CHGPLMNCFG（PLMN+APN NI）> ADD CHGIMSICFG（IMSI 前缀+APN NI+拜访类型+计费属性）。SET CHGCDR 的 `PLMNCTRL` 控制话单生成策略基于哪种 PLMN 来源（IMSI/LOCINFO）。

## 决策点

### PLMN 类型选择
问：配置本网用户还是漫游用户的话单属性？
- `PLMN=HPLMN`：本网用户计费属性。
- `PLMN=VPLMN`：拜访 PLMN（漫游）用户计费属性——漫游控制场景必选（证据 WSFD-011001）。

### 各类话单生成开关
问：该 PLMN 下 8 类话单分别生成还是抑制？
- 默认全局策略：按运营商整网规划，漫游场景通常 7 类 `YES` + 个别 `NO`（如 `LCSNIP=NO` 抑制定位话单）。
- 抑制后无法被下层细粒度命令（ADD CHGPLMNCFG/ADD CHGIMSICFG/SET CHGCHAR）覆盖——本命令是总闸。

## 约束
- 本命令禁止生成某种话单后，无论 ADD CHGPLMNCFG/ADD CHGIMSICFG/SET CHGCHAR 如何配置都不生成（总闸优先）。`severity=critical`
- 全局 S-CDR 策略生效前提是本命令开启 `SP=YES`，否则三种 S-CDR 细分策略（PLMN+APN NI / IMSI 前缀等）都不生效。`severity=high`
- 系统初次运行执行系统初始设置值。`severity=medium`
- 该命令执行后立即生效。`severity=low`

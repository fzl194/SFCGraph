# ADD ROAMCHGMODE 原子任务
> 命令静态知识见 [ADD ROAMCHGMODE](command/UNC/20.15.2/ADD-ROAMCHGMODE.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法
本命令按"终端+接入类型（TMACCTYPE）× 漫游属性（ROAMINGTYPE）"维度指定计费接口选择方式，是 UNC 三级计费接口选择中**优先级最高**的一级（基于漫游属性 > 基于 APN/DNN [ADD APNCHGMODE](task/UNC/20.15.2/0-00005.md) > 全局 [SET CHGMODE](task/UNC/20.15.2/0-00162.md)）。控制类型 CTRLTYPE 区分 APN 级别等维度；实际计费接口由 FORCED（强制指定）、FORSGWONLY（SGW 计费模式）、FORVSMFONLY（V-SMF 计费模式）等参数给出，并可由 BY5GSIWKI（按 5GS 互操作指示）、BY5GCNRI（按 5GC 无限制接入标识）两个开关进一步细化选择条件。

| 参数 | 配置方法 | 取值样例 |
|---|---|---|
| TMACCTYPE | 指定终端和接入类型，本端规划 | `UE5G_RAT4G`（5G 终端 4G 接入） |
| CTRLTYPE | 控制类型，全网规划 | `APN_LEVEL`（APN 级别） |
| APN | APN 名称，本端规划，1~63 字符（CTRLTYPE=APN_LEVEL 时必配） | `apn-test` |
| ROAMINGTYPE | 漫游属性，本端规划 | `ROAMING`（漫游用户） |
| FORCED | 指定的计费接口，全网规划 | `NchfMode` / `GaGyMode` |
| FORSGWONLY | 作为 SGW 计费模式，全网规划（SGW 独立部署时指定 SGW 计费接口） | `GaGyMode` |
| FORVSMFONLY | 作为 V-SMF 计费模式，全网规划（5G 国漫漫游地接入场景） | `GaGyMode` |
| BY5GSIWKI | 按 5GS 互操作指示选择计费接口，全网规划（参考 S5/S8 的 5GS Interworking Indication） | `False` |
| BY5GCNRI | 按 5GC 无限制接入标识选择计费接口，本端规划 | `False` |

典型脚本（5G 终端 4G 接入漫游用户，强制 Nchf）：
```
ADD ROAMCHGMODE:ROAMINGTYPE="ROAMING", TMACCTYPE=UE5G_RAT4G, BY5GSIWKI=False, FORCED=NchfMode;
```
典型脚本（SGW 独立部署指定 SGW 计费接口 + V-SMF 计费接口）：
```
ADD ROAMCHGMODE:ROAMINGTYPE="ROAMING", TMACCTYPE=UE5G_RAT4G, FORCED=GaGyMode, FORSGWONLY=GaGyMode, FORVSMFONLY=GaGyMode;
```

## 决策点

### 计费接口选择的三级优先级
- 问什么：本命令（漫游属性维度）在整体计费接口选择中的位置。
| 优先级 | 来源命令 | 维度 |
|---|---|---|
| 最高 | 本命令 ADD ROAMCHGMODE | 终端+接入类型 × 漫游属性 |
| 中 | [ADD APNCHGMODE](task/UNC/20.15.2/0-00005.md) | APN/DNN |
| 最低 | [SET CHGMODE](task/UNC/20.15.2/0-00162.md) | 全局 |

### 计费接口指定的多个场景
- 问什么：FORCED / FORSGWONLY / FORVSMFONLY 分别覆盖哪些组网场景。
| 参数 | 场景 | 是否必配 |
|---|---|---|
| FORCED | 通用强制指定计费接口 | 常配 |
| FORSGWONLY | SGW 独立部署时 SGW 发起计费会话的接口 | 仅 SGW 独立部署时配 |
| FORVSMFONLY | 5G 终端国漫场景下漫游地接入 5G 核心网的 V-SMF 计费接口 | 仅 V-SMF 场景配 |

### 附加选择条件开关
- 问什么：是否启用 BY5GSIWKI / BY5GCNRI 作为计费接口的二次细化选择条件。
| 开关 | 判据 | 备注 |
|---|---|---|
| BY5GSIWKI | S5/S8 的 5GS Interworking Indication | True 时按互操作指示选择 |
| BY5GCNRI | 5GC 无限制接入标识 | True 时按 CNRI 选择 |

## 约束
- 生效时机：命令执行后**只对新激活用户生效**（已激活用户不重配）。`severity=high`
- 记录上限：最多可输入 **10000** 条记录。`severity=medium`
- 优先级覆盖：选择计费接口时，本命令优先级**高于** [SET CHGMODE](task/UNC/20.15.2/0-00162.md)（全局）。`severity=high`
- 互操作不变性：4G/5G 互操作流程的计费接口不变；2G/3G/4G 互操作流程的计费接口不变（即互操作流程不重新选择接口）。`severity=medium`
- 2G 接入 Nchf 前置：当需要用本命令配置 2G 接入使用 Nchf 接口时，须**提前和对端 CHF 确认是否支持**。`severity=high`
- 5G 接入 GaGy License：当 5G 终端 5G 接入时配置为 GaGy 模式，须**开启特定 License 项**，提前联系华为技术支持确认。`severity=high`
- 风险提示：配置不当可能选错计费策略接口，相应计费策略接口未配置时会导致计费相关流程失败（如用户无法正常激活）。`severity=high`
- 前置校验：CTRLTYPE=APN_LEVEL 时 APN 须已配置；SGW/V-SMF 维度参数须对应实际组网角色。`severity=high`

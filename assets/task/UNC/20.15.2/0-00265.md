# 修改APN配置（MOD APN）

> 命令静态知识见 [MOD APN](command/UNC/20.15.2/MOD-APN.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

该命令修改已存在的 APN 实例（由 `ADD APN` 建立）的属性。APN 属性庞大，配置空间按**功能族**组织——一次 MOD 通常修改某功能族的若干参数。

### 配置维度1：VPN 绑定（参数 HASVPN / VPNINSTANCE / HASVPNIPV6 / VPNINSTANCEIPV6）

| 取值 | 作用 | 配套参数 |
|---|---|---|
| HASVPN=ENABLE | 绑定 IPv4 VPN | VPNINSTANCE（必填） |
| HASVPNIPV6=ENABLE | 绑定 IPv6 VPN | VPNINSTANCEIPV6（必填） |
| DISABLE | 解绑 VPN | — |

### 配置维度2：虚拟 APN（参数 VIRTUALAPN）

| 取值 | 作用 | 约束 |
|---|---|---|
| ENABLE | 使能虚拟 APN | 必须先通过 `ADD VIRTUALAPNRULE（★R1.5 atom 待补）` 配置映射规则；不能同时是 Parking APN |
| DISABLE | 关闭虚拟 APN | — |

### 配置维度3：ULCL / Parking APN / LADN（参数 ULCLFUNC / PARKINGAPN / LADN）

| 参数 | 取值 | 作用 |
|---|---|---|
| ULCLFUNC | ENABLE/INHERIT/DISABLE | ULCL 功能开关（仅 5G；与 Parking APN 互斥） |
| PARKINGAPN | ENABLE/DISABLE | Parking APN（仅 5G；与虚拟 APN/ULCL 互斥） |
| LADN | Support/NotSupport | 局域数据网络支持 |

### 配置维度4：业务恢复与计费策略（参数 NTSRSWITCH / RESTORPGWSWITCH / PDTNSWITCH / CHARGEPROFILE）

| 参数 | 作用 |
|---|---|
| NTSRSWITCH | 网络侧触发业务恢复功能（ENABLE/INHERIT/DISABLE） |
| RESTORPGWSWITCH | 故障重启业务恢复 PGW（ENABLE/INHERIT/DISABLE） |
| PDTNSWITCH | PDTN 功能（条件：RESTORPGWSWITCH=ENABLE） |
| CHARGEPROFILE | 计费策略（NULL/HLR/PREPAID_WITH_CREDIT_CONTROL/POSTPAID_WITH_CREDIT_CONTROL/POSTPAID/PREPAID） |

### 配置维度5：紧急呼叫/二次激活/去活（参数 EMERGENCYSWITCH / USRINTSECACTSW / REACWITHDEL）

| 参数 | 取值 | 作用 |
|---|---|---|
| EMERGENCYSWITCH | ENABLE/DISABLE | 支持紧急呼叫 |
| USRINTSECACTSW | ENABLE/DISABLE | 用户发起的二次激活 |
| REACWITHDEL | DISABLE/ENABLE/SPECIFIC_SCENE | 去活消息携带 reactivation-request（SPECIFIC_SCENE 时配 SCENELIST） |

**步骤位置**：在所属 feature 编排中，于 `ADD APN`（建实例）之后按需修改。

## 决策点

**功能开关选择**（UNC pre-build 内嵌，不编号——按 `assets/CLAUDE.md` §5.7）：

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| 绑定 IPv6 VPN（启用 IPv6 业务） | HASVPNIPV6=ENABLE, VPNINSTANCEIPV6 | WSFD-104004 IPv6 前缀代理 |
| 开启 ULCL | ULCLFUNC=ENABLE | ULCL 分流场景（与 Parking APN 互斥） |
| 开启虚拟 APN | VIRTUALAPN=ENABLE（需先 ADD VIRTUALAPNRULE（★R1.5 atom 待补）） | 虚拟 APN 场景 |

## 约束

- **立即生效**（info）：命令执行后立即生效 — 后果：无延迟
- **部分参数已弃用**（warning）：APPLAYERVOLUME/QCHATSWITCH/PSEUDOACTSWITCH 参数当前版本不支持 — 后果：配置无效
- **虚拟 APN 开启需先建映射规则**（critical）：VIRTUALAPN=ENABLE 前必须通过 `ADD VIRTUALAPNRULE（★R1.5 atom 待补）` 配置映射规则 — 后果：用户激活失败
- **软参 APN_BIT36 影响虚拟 APN 缺省规则校验**（warning）：APN_BIT36=0 时 SMF 校验缺省映射规则；无缺省规则返回 51795 异常码 — 后果：开启失败
- **虚拟 APN 与 Parking APN 互斥**（critical）：虚拟 APN 不能同时是 Parking APN — 后果：配置失败
- **ULCL 与 Parking APN 互斥**（critical）：开启 ULCL 的 APN 不能是 Parking APN — 后果：配置失败

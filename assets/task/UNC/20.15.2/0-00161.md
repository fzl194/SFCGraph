# SET CHGGA 原子任务
> 命令静态知识见 [SET CHGGA](command/UNC/20.15.2/SET-CHGGA.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法
SET CHGGA 用于 SGSN 计费 Ga 接口全局参数的整网规划配置，分四类范式：话单协议版本（GCR/UCR/CHGVER98/99/4/5/6/7/9 + QOSVER）、CDR 重发策略（ITVRES/RSNUM）、硬盘空间保护（HDDETH/HDDFSL/REDICTFRMMAXOCCRATE/CDROVERWRITE）、网络层参数（GASENDUDPCHECK/GAREVUDPCHECK/IPPLCY）。话单协议版本必与对接 CG 的协议版本对齐，参数错误可致计费中心无法正确计费。

| 配置目标 | 关键参数 | 典型脚本 |
|---|---|---|
| 话单可靠性保证（硬盘话单覆盖） | `HDDFSL=90` + `CDROVERWRITE=YES` | `SET CHGGA:HDDFSL=90,CDROVERWRITE=YES;` |
| 离线话单版本与 CG 对齐 | `UCR` + 对应 `CHGVER99`（与 ADD CHGCG 的 CG 协议版本一致） | `SET CHGGA:UCR=R99,CHGVER99=GPP32015V360;` |
| 热计费 UMTS 话单版本对齐 | `UCR=R99` + `CHGVER99=GPP32015V3.a.0` | `SET CHGGA:UCR=R99,CHGVER99=GPP32015V3a0;` |

## 决策点

### 话单协议版本选择
问：GPRS/UMTS 用户走哪个 CDR 协议版本？
- `GCR=R98` + `UCR=R99`：SGSN 同时支持 2G/3G，分别为 2G/3G 用户指定话单格式（证据 WSFD-011005）。
- `CHGVER98/99/4/5/6/7/9`：选定版本对应遵循的标准（如 `CMCCV130`/`GPP32015V360`/`GPP32215V440` 等）——必须与对接 CG 协议版本一致（证据 WSFD-011202）。

### IP 地址选择策略
问：IPv4/IPv6 改造阶段如何选 Ga 接口 IP？
- `IPPLCY=IPV4`：纯 IPv4 网络。
- 改造过程取值见命令联机帮助（IPv4 向 IPv6 改造场景）。

## 约束
- 计费参数错误修改可能导致计费中心无法正确计费，需谨慎修改。`severity=critical`
- 部分参数修改需复位 CDP/SPP/GTP 才生效：`ITVRES`/`HDDETH`/`HDDFSL`/`REDICTFRMMAXOCCRATE` 需复位 CDP；`GCR`/`UCR`/`CHGVER98` 等（协议版本类）需复位 SPP 和 GTP——详见命令联机帮助。`severity=high`
- 硬盘话单覆盖功能依赖 `CDROVERWRITE=YES` 且硬盘使用达 `HDDFSL` 门限；SFTP 上传场景下删除最旧文件不保证（系统只有最后修改时间）。`severity=medium`
- 各话单协议版本与 CG 协议版本必须一致（与 [ADD CHGCG](task/UNC/20.15.2/0-00017.md) 联动）。`severity=high`
- 执行后部分立即生效、部分复位生效（见上）。`severity=low`

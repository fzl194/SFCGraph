# 配置融合计费异常处理

> 融合计费故障处理——SMF 超时未收到 CHF 响应 + PDU/RG 级异常返回码 + CHF 响应信元异常的处理动作。被引用于 [2-00003 融合计费](task/UNC/20.15.2/2-00003.md)。特性专属（离线/热计费无 Nchf 交互，无故障处理机制）。

## 配置方法

| 步骤 | 命令 | 关键参数 |
|---|---|---|
| 超时故障处理 | `SET FAILHANDLING` | CCTMPLTNAME；FHACTION=CONTINUE/RETRY_AND_TERM/TERMINATE；FAILOVERSUP；AUTOFAILBACK；TXTIMER；HOLDINGTIME |
| PDU 级异常返回码动作 | `ADD PDUSCACT` | CCTMPLTNAME；CODETYPE=VALUE；STATUSCODE（如 502/404）；SCACT=FAILOVER/CONTINUE/TERMINATE |
| RG 级异常返回码动作 | `ADD RGRCACT` | CCTMPLTNAME；RCCODE（如 QUOTAMNOTAPPL）；RCACT=BLCK_IMMED_RPT；BLKTIMER |
| 忽略 CHF 响应信元 | `SET CNVRGDCHGPARA` | IGNORERSPAVP（位域：INVOCATIONNUM/FUI/SESSIONFAILOVER/FAILUREHANDLING）；BADRSPACT=TERMINATE/CONTINUE |

**典型脚本**：
```
SET FAILHANDLING: CCTMPLTNAME="cct-test", FHACTION=CONTINUE, FAILOVERSUP=ENABLE, AUTOFAILBACK=ENABLE, TXTIMER=5, HOLDINGTIME=10;
ADD PDUSCACT: CCTMPLTNAME="cct-test", CODETYPE=VALUE, STATUSCODE=502, SCACT=FAILOVER;
ADD RGRCACT: CCTMPLTNAME="cct-test", RCCODE=QUOTAMNOTAPPL, RCACT=BLCK_IMMED_RPT, BLKTIMER=30;
SET CNVRGDCHGPARA: IGNORERSPAVP=FUI, BADRSPACT=TERMINATE;
```

**步骤位置**：CCT 模板（[1-00005](task/UNC/20.15.2/1-00005.md)）+ CHF 选择（[1-00007](task/UNC/20.15.2/1-00007.md)，主备组）之后；故障处理依赖主备 CHF 组配置。

## 决策点

| 选项 | 影响（参数/联动） |
|---|---|
| FHACTION=CONTINUE | 超时后继续业务（不进行配额管理）—— 消息缓存前提 |
| FHACTION=RETRY_AND_TERM | 重试后终止会话 |
| FHACTION=TERMINATE | 直接终止会话 |
| SCACT=FAILOVER | PDU 异常向备 CHF 重发 —— 消息缓存前提 |
| SCACT=CONTINUE | 继续业务 —— 消息缓存前提 |
| SCACT=TERMINATE | 终止会话 |
| RCACT=BLCK_IMMED_RPT | RG 异常阻塞业务并立即上报 |
| BADRSPACT=TERMINATE | CHF 响应信元异常 → PDU 会话去活 |
| BADRSPACT=CONTINUE | 信元异常 → 继续业务，不再配额管理 |

> 联动特性级 DP：FHACTION=CONTINUE + SCACT=FAILOVER/CONTINUE 是开启消息缓存（[1-00011](task/UNC/20.15.2/1-00011.md)）的硬前提。FAILOVERSUP=ENABLE 时主 CHF 故障向备 CHF 发消息（依赖 [1-00007](task/UNC/20.15.2/1-00007.md) 配主备组）。

## 约束

- **前置 CCT 模板**（critical）：CCTMPLTNAME 须已在 [1-00005](task/UNC/20.15.2/1-00005.md) 建立。
- **前置主备 CHF 组**（warning）：FAILOVERSUP=ENABLE 须先在 [1-00007](task/UNC/20.15.2/1-00007.md) 配主备 CHF 组 — 否则无备 CHF 可切换。
- **缓存前提**（critical）：开启消息缓存（[1-00011](task/UNC/20.15.2/1-00011.md)）必须 FHACTION=CONTINUE 且 SCACT=FAILOVER 或 CONTINUE — 否则缓存不生效。
- **IGNORERSPAVP 位域**（info）：可单独或组合配置 INVOCATIONNUM/FUI/SESSIONFAILOVER/FAILUREHANDLING；FUI 表示忽略 FinalUnitIndication 按 ADD RGRCACT 动作执行。

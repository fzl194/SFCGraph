# SET CONTAINERTRIGGER 原子任务
> 命令静态知识见 [SET CONTAINERTRIGGER](command/UNC/20.15.2/SET-CONTAINERTRIGGER.md)。本页只讲配置生成实例化时这条命令**怎么配**。

SGW-C/PGW-C/SMF 侧**离线计费模板粒度**的容器产生开关：每个离线计费模板（OFCTEMPLATE）一条记录，勾选哪些"计费条件变更"会触发容器产生。容器概念：一张话单内含多个容器，分别记录不同计费条件下的流量，避免话单量暴增。

## 配置方法
以离线计费模板名为主键，按运营颗粒度勾选触发条件；多数场景按需开启，默认行为可全 DISABLE。

| 参数 | 配置方法 | 取值样例 |
|---|---|---|
| OFCTEMPLATENAME | 离线计费模板名（主键，前置已建） | `"offlinecharge-test"` |
| CONTTRIGRATCHNG | Container RAT 更新 | `DISABLE` / `ENABLE` |
| CONTTRIGQOSCHNG | QoS 更新 | `DISABLE` / `ENABLE` |
| PCCINIT | PCRF 发起的 QoS 更新（conditional，CONTTRIGQOSCHNG=ENABLE 时配） | `ENABLE` |
| CONTTRIGSNCHNG | Serving Node 更新 | `DISABLE` / `ENABLE` |
| CONTTRIGCGISAI | CGI/SAI 更新 | `DISABLE` / `ENABLE` |
| CONTTRIGECGI | ECGI 更新 | `DISABLE` / `ENABLE` |
| CONTTRIGTAI | TAI 更新 | `DISABLE` / `ENABLE` |
| CONTTRIGULI | ULI 更新 | `DISABLE` / `ENABLE` |
| CONTTRIGPLMNID | Container Serving Node PLMN 标识更新 | `DISABLE` / `ENABLE` |
| CONTTRIGRAI | RAI 更新 | `DISABLE` / `ENABLE` |
| CONTTRIGTARRIF | 费率切换 | `DISABLE` / `ENABLE` |
| CONTTRIGENB | eNodeB 更新 | `DISABLE` / `ENABLE` |
| USERPLANECHNG | 用户面更新 | `DISABLE` / `ENABLE` |
| SPLMNRATECHNG | 服务 PLMN 控制速率改变 | `DISABLE` / `ENABLE` |
| APNRATECHNG | APN 控制速率改变 | `DISABLE` / `ENABLE` |
| APNRATECHNGCLOSECAUSE | APN 速率控制改变流量容器关闭原因值（conditional，APNRATECHNG=ENABLE 时配，对端协商） | 枚举 |

典型脚本（基础离线计费，仅模板名，全部 DISABLE，覆盖系统初始值）：
```
SET CONTAINERTRIGGER: OFCTEMPLATENAME="offlinecharge-test", CONTTRIGRATCHNG=DISABLE, CONTTRIGQOSCHNG=DISABLE;
```

## 决策点

### 容器触发条件勾选集合
- 问什么：该离线计费模板下，哪些计费条件变化需要单独记一个容器。
- 取值由运营计费颗粒度决定：细颗粒（需要按位置/费率/QoS 分别统计流量）→ 多开；粗颗粒（仅周期/流量阈值）→ 全 DISABLE。
- 位置类（CGI/SAI/ECGI/TAI/ULI/RAI）按 2G/3G/4G/5G 制式选配；费率切换（CONTTRIGTARRIF）配合费率切换特性。

### APN/服务 PLMN 速率控制场景（终端接入速率控制特性）
- 问什么：是否启用基于 APN 或服务 PLMN 的终端接入速率控制触发的容器。
- 仅 WSFD-216104/216105（eMTC）/WSFD-215204/215205（NB-IoT）等终端接入速率控制特性场景需要配 APNRATECHNG/SPLMNRATECHNG=ENABLE，并配套 APNRATECHNGCLOSECAUSE。

## 约束
- 前置：必须先配置离线计费模板（[ADD OFCTEMPLATE](task/UNC/20.15.2/0-00049.md)），才能配本命令。`severity=high`
- 唯一性：最大记录数 100。`severity=medium`
- 不支持项：当前版本不支持 CONTTRIGENB 参数。`severity=medium`
- 生效时机：立即生效。`severity=low`
- 初始值：系统存在初始记录，参数默认值见命令静态表。`severity=low`
- conditional：PCCINIT 在 CONTTRIGQOSCHNG=ENABLE 时配；APNRATECHNGCLOSECAUSE 在 APNRATECHNG=ENABLE 时配。`severity=medium`

# TST CHGCDR 原子任务
> 命令静态知识见 [TST CHGCDR](command/UNC/20.15.2/TST-CHGCDR.md)。本页只讲配置生成实例化时这条命令**怎么配**。

本命令适用网元为 **SGSN**（Pre-5G 接入业务），不用于 5GC/UPF 侧。用途是新建局时模拟一条 SGSN 话单走完 SPP/UPP→CDP→CG 全链路，验证计费网络与对接数据配置的正确性。属调测验证类原子，非常规配置项。

## 配置方法

| 参数 | 配置方法 | 取值样例 |
|---|---|---|
| CDRTYPE | 话单类型（整网规划，required） | `MCDR` |
| SYSTYPE | 系统类型（整网规划，默认 `UMTS`） | `UMTS` / `GPRS` |
| IMSI | IMSI（整网规划） | 5～15 位十进制数字 |
| MSISDN | MSISDN（整网规划） | 5～15 位十进制数字 |
| CC | 计费属性（整网规划，默认 `NORMAL`） | `NORMAL` |
| LAC | 位置区域码（整网规划） | `0x0000`～`0xFFFF` |
| RAC | 寻呼范围路由区域码（整网规划） | 1～2 位十六进制 |
| CI | 小区标识（整网规划） | 1～4 位十六进制 |
| RUNAME | RU 名称（整网规划） | 1～63 位字符串 |
| PID | 进程号（整网规划） | `0`～`20` |

典型脚本（仅必填 CDRTYPE 即可触发，其余按调测场景补）：
```
TST CHGCDR:CDRTYPE=MCDR,SYSTYPE=UMTS,IMSI=460001234567890,MSISDN=8613800138000,CC=NORMAL;
```

## 决策点

### 调测范围（话单参数组合）
- 问什么：模拟话单的 IMSI/MSISDN/CC/LAC/RAC/CI 取什么值。
- 取值由整网规划决定（调测目标用户/区域），无固定值。
- 用途定位：调测命令，配置生成场景一般不落常规配置，仅在新建局或接口验证时触发。

### 触发进程（PID）
- 问什么：指定由哪个 CDP 进程生成/处理话单。
- 取值 `0`～`20`，可选；不填则由系统按进程状态选择。
- 前置：所选 PID 对应的 CDP 进程必须状态正常。

## 约束
- 生效时机：执行后立即生效。`severity=low`
- 触发前置（4 个条件须全部满足）：`severity=high`
  1. 至少存在一个状态正常的 CDP 进程；
  2. SPU 硬盘没有满；
  3. 与 CG 通讯正常；
  4. SET CHGCHAR 中配置的生成话单类型为 `YES(Generate)`。
- NF 限制：仅适用 SGSN（Pre-5G 接入业务），不用于 5GC/UPF。`severity=medium`
- 用途限制：调测验证类命令，非常规配置项，仅用于计费接口对接验证。`severity=medium`
- 必填：CDRTYPE 为必填。`severity=low`

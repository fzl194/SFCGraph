# 增加目标NF实例（ADD TNFINS）

> 命令静态知识见 [ADD TNFINS](command/UNC/20.15.2/ADD-TNFINS.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法

按"主备 CHF 各注册一个实例"配置，固定类型 CHF。

| 维度 | 参数 | 取值 | 默认 | 作用 | 适用场景 |
|---|---|---|---|---|---|
| 索引 | TNFINSINDEX | 0~2047 整数，本端规划 | 无 | 唯一标识一个 NF 实例，供 TNFINSIP/TNFGRP 引用 | 主备 CHF 各取一个唯一索引 |
| 类型 | TNFTYPE | CHF / UPF / PCSCF / PCF | 无 | 指定目标 NF 类型 | 融合计费场景固定为 CHF |
| 实例名 | TNFINSNAME | 字符串，0~63 字符，与 PCF 下发 FQDN 一致 | 无 | 与 PCF/NRF 对端实例标识对齐；与 PNFPROFILE.NFINSTANCEID 相同时触发 IP 包含约束 | 从对端获取，需与 PCF 下发 FQDN 名称一致 |
| 服务名 | SRVNAME | NrfNfm / NnrfDisc / NudmSdm 等 | NrfNfm（默认） | 指定目标 NF 提供的服务名称 | 常规注册填 NrfNfm（NRF 的 Nnrf_NFManagement 服务） |
| 服务实例标识 | SRVINSID | 字符串，0~50 字符 | 空（按实例级匹配） | 服务级精细化发现时填写；为空时按 NF 实例级匹配 | 仅当需要服务级精细化发现时填写 |
| 协议模式 | SCHEMA | http / https | 无 | 本端与对端通信协议，建议对齐否则无法建立 HTTP 链接 | 全网规划，建议对端一致 |
| 域名 | FQDN | 字符串，0~127 字符 | 无 | CHF 域名，区分大小写 | 从对端获取，如 chf1.hw.com |
| NF 描述名称 | NFDESCNAME | 字符串，0~255 字符 | 无 | NF 实例描述信息 | 本端规划，可选 |

典型脚本（主备 CHF，带 FQDN）：
```
ADD TNFINS: TNFINSINDEX=1, TNFTYPE=CHF, TNFINSNAME="target_chf_0", SCHEMA=http, FQDN="chf1.hw.com";
ADD TNFINS: TNFINSINDEX=2, TNFTYPE=CHF, TNFINSNAME="target_chf_1", SCHEMA=http, FQDN="chf2.hw.com";
```

步骤位置：全局 NF 实例配置 → NRF 基础数据 → 目标 NF 实例管理。

## 决策点

### 决策点：TNFINSNAME 与 ADD PNFPROFILE 的 IP 包含关系（DP 0-01210）

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| TNFINSNAME = PNFPROFILE.NFINSTANCEID（相同 CHF） | 关联的 TNFINSIP 下所有 IP 必须被 PNFPROFILE 中该 CHF 的 IP 包含 | 融合计费 CHF 注册，本端与对端实例名一致时触发 |
| TNFINSNAME ≠ PNFPROFILE.NFINSTANCEID（不同 NF） | 无 IP 包含约束 | 非融合计费场景，或注册的为其他类型 NF |

### 决策点：SRVINSID 与 SRVNAME 的取舍（DP 0-01211）

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| 不填 SRVINSID（空） | 按 NF 实例级匹配，SRVNAME 填 NrfNfm | 常规融合计费 CHF 注册 |
| 填写 SRVINSID | 服务实例标识精确匹配，SRVNAME 填对应服务名 | 需要服务级精细化发现时（如同一 NF 实例提供多服务） |

## 约束

- **该命令执行后立即生效**（info，rule-0-01461）：该命令执行后立即生效
- **与 ADD PNFPROFILE 的 IP 包含关系**（warning，rule-0-01462）：与 ADD PNFPROFILE 的 IP 包含关系：当 TNFINSNAME 与 PNFPROFILE 的 NFINSTANCEID 相同时，本实例关联的 TNFINSIP 全部 IP 必须被 PNFPROFILE 中该 CHF 的 IP 包含，否则配置失败
- **TNFINSINDEX 唯一性**（warning，rule-0-01463）：TNFINSINDEX 唯一性：同一索引标识一个实例，跨 TNFINS/TNFINSIP/TNFGRP 引用需一致
- **TNFINSNAME 长度**（info，rule-0-01464）：TNFINSNAME 长度：0~63 字符
- **记录上限**（info，rule-0-01465）：记录上限：最多 2048 条

# SET N40APIVER 原子任务
> 命令静态知识见 [SET N40APIVER](command/UNC/20.15.2/SET-N40APIVER.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法
系统部署后已存在初始记录（`APIVER=F30, FEATURE=NODEFUNC-1`）。`APIVER` 全网统一为 F30（遵循 3GPP TS 32.291 F30 定义的 Nchf_ConvergedCharging），`FEATURE` 为位域叠加型——支持多值用 `&` 拼接，使能时后缀 `-1`、不使能后缀 `-0`，不修改的功能不携带。配置需与 CHF 协议版本和接口能力**完全一致**，否则 CHF 无法处理 SMF 发送的消息。

| 启用能力组合 | FEATURE 取值 | 典型脚本 |
|---|---|---|
| 仅基础（NodeFunction，初始默认） | `NODEFUNC-1` | `SET N40APIVER:APIVER=F30,FEATURE=NODEFUNC-1;` |
| NB-IoT + LTE-M 计费增强 | `NBIOTCHG-1&LTEMCHG-1` | `SET N40APIVER:APIVER=F30,FEATURE=NBIOTCHG-1&LTEMCHG-1;` |
| UTRAN 3GPP 计费增强 | `UTRANCHG3GPP-1` | `SET N40APIVER:APIVER=F30,FEATURE=UTRANCHG3GPP-1;` |
| Trigger 2/3G 计费增强 | `TRIGGER23G-1` | `SET N40APIVER:APIVER=F30,FEATURE=TRIGGER23G-1;` |
| GERAN 计费增强 | `GERANCHG-1` | `SET N40APIVER:APIVER=F30,FEATURE=GERANCHG-1;` |
| WLAN 计费增强 | `WLANCHG-1` | `SET N40APIVER:APIVER=F30,FEATURE=WLANCHG-1;` |
| 多能力叠加 | `NODEFUNC-1&GERANCHG-1&WLANCHG-1` | `SET N40APIVER:APIVER=F30,FEATURE=NODEFUNC-1&GERANCHG-1&WLANCHG-1;` |

**FEATURE 位域叠加规则**：`-1` 使能、`-0` 不使能、不携带=不修改；多值用 `&` 拼接。

## 决策点

### API 接口版本选择
问：APIVER 取哪个值？
- 证据中全网规划统一为 `F30`（3GPP TS 32.291 F30）。
- 该值为全网规划，需与 CHF 一致；当前证据仅出现 F30，跨版本升级时由全网规划决定。

### 增强功能集组合（FEATURE）
问：本局点启用哪些 N40 增强计费能力？
| 选项 | 联动 | 适用场景 |
|---|---|---|
| `NODEFUNC-1` | 仅基础 NodeFunction | 系统初始默认，无特殊接入类型 |
| `NBIOTCHG-1` | 联动 [ADD N40MSGTEMP](task/UNC/20.15.2/0-00046.md) `NBEXTENDATTR=ENABLE` | NB-IoT 接入融合计费 |
| `LTEMCHG-1` | 常与 `NBIOTCHG-1` 同启 | LTE-M 接入融合计费 |
| `UTRANCHG3G-1` / `GERANCHG-1` / `WLANCHG-1` / `TRIGGER23G-1` | 各自独立 | 对应 3G/GERAN/WLAN/2-3G Trigger 计费场景 |
- 决策依据：本局点实际接入的 RAT 类型集合（NB-IoT / LTE-M / UTRAN / GERAN / WLAN 等），与 CHF 能力对齐。

## 约束
- 执行后立即生效，且会修改 N40 接口功能和信元携带能力。`severity=critical` 后果：CHF 不支持时无法处理 SMF 消息。
- `APIVER`/`FEATURE` 需与 CHF 协议版本和接口能力**完全一致**（全网规划）。`severity=critical` 后果：N40 接口对接失败。
- 系统部署后已存在初始记录（`F30 + NODEFUNC-1`）；不输入 `FEATURE` 时保持当前值不变。`severity=low`
- `FEATURE` 为位域叠加型，使能带 `-1`、不使能带 `-0`、不修改不携带。`severity=medium` 后果：拼接错误导致能力误开/误关。
- NB-IoT 计费增强开启后，需同步在 [ADD N40MSGTEMP](task/UNC/20.15.2/0-00046.md) 配 `NBEXTENDATTR=ENABLE` 才能生效。`severity=medium`

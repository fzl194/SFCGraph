# ADD PCRFGRPBNDAPN 原子任务
> 命令静态知识见 [ADD PCRFGRPBNDAPN](command/UNC/20.15.2/ADD-PCRFGRPBNDAPN.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法
本命令把 PCRF 分组（来自 [ADD PCRFGROUP](task/UNC/20.15.2/0-00059.md)）与 APN（来自 [ADD APN](task/UNC/20.15.2/0-00004.md)）关联起来，使通过该 APN 接入的用户按规则选择 PCRF 组。APN 下 PCRF 组关联有两条独立轨道：DEFAULT（APN 缺省 PCRF 组）与 IMSI_MSISDN_SEG（按号段细分 PCRF 组），可只配其一或都配。

| 参数 | 配置方法 | 取值样例 |
|---|---|---|
| APN | APN 名称，从已配置的 `ADD APN` 数据中获取 | `apn-test1` |
| DEFAULTFLAG | 缺省标记，本端规划 | `DEFAULT` / `IMSI_MSISDN_SEG` |
| PCRFGRPNAME | PCRF 组名称，从已配置的 `ADD PCRFGROUP` 数据中获取 | `pcrf_group_2` |
| IMSIMSISDNSEG | IMSI/MSISDN 号段名称，本端规划（仅 DEFAULTFLAG=IMSI_MSISDN_SEG 时必配，从 `ADD IMSIMSISDNSEG` 获取） | `msisdnseg1` |
| PRIORITY | 优先级，本端规划，1~65535，号段维度内唯一（conditional） | `1` |
| DESCRIPTION | 描述，本端规划，1~63 字符（可选） | `apn1 default pcrf group` |

典型脚本（APN 缺省 PCRF 组）：
```
ADD PCRFGRPBNDAPN:APN="apn1",DEFAULTFLAG=DEFAULT,PCRFGRPNAME="pcrf.group.1";
ADD PCRFGRPBNDAPN:APN="ims",DEFAULTFLAG=DEFAULT,PCRFGRPNAME="aaa";
```
典型脚本（按号段绑定 PCRF 组）：
```
ADD PCRFGRPBNDAPN:APN="apn-test1",DEFAULTFLAG=IMSI_MSISDN_SEG,IMSIMSISDNSEG="msisdnseg1",PCRFGRPNAME="pcrf_group_2",PRIORITY=1;
```

## 决策点

### APN 与 PCRF 组的关联方式
- 问什么：该 APN 是配置一个缺省 PCRF 组（覆盖所有用户），还是按 IMSI/MSISDN 号段细分到不同 PCRF 组。
| DEFAULTFLAG | 行为 | 必配参数 |
|---|---|---|
| DEFAULT | 设置 APN 的缺省 PCRF Group（APN 下唯一不带号段） | PCRFGRPNAME |
| IMSI_MSISDN_SEG | 按 IMSI/MSISDN 号段绑定 PCRF 组（号段命中时用该组） | PCRFGRPNAME + IMSIMSISDNSEG + PRIORITY |

### 是否在号段维度配置优先级（号段策略并存时）
- 问什么：当同一 APN 下按多个号段绑定不同 PCRF 组时，号段命中重叠时的选择顺序。
| 是否多号段 | PRIORITY 配置 |
|---|---|
| 单号段 | 可不配或配任意值 |
| 多号段并存 | 每条记录配 PRIORITY，号段维度内唯一，值小者优先（命中冲突时） |

## 约束
- 生效时机：命令执行后**立即生效**。`severity=medium`
- 记录上限：最大记录数 **3000**。`severity=medium`
- APN 缺省唯一性：APN 下只允许配置**一个不带号段**的 PCRF 组名称（即 DEFAULT 记录唯一）。`severity=high`
- 前置校验：添加记录前须确认 APN 已配置（[ADD APN](task/UNC/20.15.2/0-00004.md)）；PCRFGRPNAME 须已由 [ADD PCRFGROUP](task/UNC/20.15.2/0-00059.md) 配置；号段模式须已由 [ADD IMSIMSISDNSEG](task/UNC/20.15.2/0-00039.md) 配置号段。`severity=high`
- PRIORITY 唯一性：优先级在号段维度内唯一。`severity=high`
- 回退策略：若某 APN PCC 开关为 INHERIT，且通过该 APN 接入用户的特定号段未绑定 PCRF 组，则使用全局缺省 PCRF 组（由 [0-00169](task/UNC/20.15.2/0-00169.md) SET DFTGLBPCRFGRP 配置）。`severity=medium`

# ADD APN 原子任务
> 命令静态知识见 [ADD APN](command/UNC/20.15.2/ADD-APN.md)。本页只讲配置生成实例化时这条命令**怎么配**。

ADD APN 是多特性共用的"APN 实例入口"，44 个特性在此命令上叠加不同开关组合。配置时先确定 APN 名称，再按场景叠加可选开关。

## 配置方法

### 基础（仅创建 APN 实例）
| 参数 | 配置方法 | 取值样例 |
|---|---|---|
| APN | APN 名称（命名约束：`-`/数字/字母/`.`，不以 `.` 开头/结尾） | `apn1`、`Huawei.com`、`ims` |

仅建实例：
```
ADD APN: APN="apn1";
```

### 按场景叠加的可选开关（同一 APN 可组合）
| 场景 | 参数 | 取值 | 说明 |
|---|---|---|---|
| 绑定 VPN（Radius/企业专线） | HASVPN / VPNINSTANCE | ENABLE / `vpn_pdn` | VPNINSTANCE 必须先由 `ADD VPNINST` 创建 |
| 绑定 IPv6 VPN | HASVPNIPV6 / VPNINSTANCEIPV6 | ENABLE / `vpn_v6` | 同上，独立于 v4 |
| 开启 ULCL 分流 | ULCLFUNC | ENABLE | 使能 SMF 上的 ULCL 功能 |
| 开启虚拟 APN | VIRTUALAPN | ENABLE | 开启前必须先 `ADD VIRTUALAPNRULE` 配映射规则 |
| 开启漫游地动态签约分流 | TRAFFICDIST | ENABLE | 与 ULCLFUNC 联用，AMF 据此判断专网 DNN |
| 主锚点 Always 分流 | ALWAYSPSAULCLSW | ENABLE | 跨域业务漫游专用 DNN |
| 支持 UE 发起二次激活 | USRINTSECACTSW | ENABLE | |
| 按 SGSN/SGW 映射 PLMN | SERVINGNODEPLMN | ENABLE | 与 `ADD SRVNODEPLMN` 联用 |
| 局域数据网络（LADN） | LADN | Support | |
| 紧急呼叫 | EMERGENCYSWITCH | ENABLE | |
| 智能业务 UPF 选择 | INTELLSERSELUPF | ENABLE | |

典型组合脚本：
```
// ULCL 分流（预定义规则）
ADD APN: APN="Huawei.com", ULCLFUNC=ENABLE;

// 漫游地动态签约分流（专网 DNN）
ADD APN: APN="dnn-ar", ULCLFUNC=ENABLE, TRAFFICDIST=ENABLE;

// 跨域业务 Always 分流
ADD APN: APN="Huawei.com", ALWAYSPSAULCLSW=ENABLE;

// 虚拟 APN
ADD APN: APN="example.com", VIRTUALAPN=ENABLE;

// VPN 绑定（Radius/AAA）
ADD APN: APN="apn1", HASVPN=ENABLE, VPNINSTANCE="vpn_pdn", HASVPNIPV6=DISABLE, RESTORPGWSWITCH=INHERIT, REACWITHDEL=DISABLE;

// 二次激活
ADD APN: APN="huawei.com", USRINTSECACTSW=ENABLE;
```

## 决策点（若有）

### APN 名称规划来源
- 问什么：APN 名称是本端规划还是全网统一规划。
- 大多数激活场景数据规划表取值样例为 `apn`/`apn1`/`apn-test`，获取方法在本端规划与全网规划间分叉（PCC 动态/计费等场景倾向全网规划，QoS/本地控制倾向本端规划）——按网络整体规划确定，本命令不强制。

### VPN 绑定方式
- 问什么：APN 是否需要绑定 VPN 实例（Radius/AAA/企业专线场景）。
- 不绑 VPN → HASVPN/HASVPNIPV6 保持 DISABLE。
- 绑 v4 VPN → HASVPN=ENABLE + VPNINSTANCE=已存在实例名。
- 双栈 → 同时 HASVPNIPV6=ENABLE + VPNINSTANCEIPV6。
- VPNINSTANCE 必须先由 `ADD VPNINST` 创建，可 `LST VPNINST` 查询。

### 分流功能开关组合
- 问什么：启用哪种分流能力。
- 预定义规则分流 → 仅 ULCLFUNC=ENABLE。
- 漫游地动态签约分流 → ULCLFUNC=ENABLE + TRAFFICDIST=ENABLE。
- 跨域 Always 分流 → ALWAYSPSAULCLSW=ENABLE。

### 虚拟 APN
- 问什么：APN 是否作为虚拟 APN（用户请求 APN 映射到真实 APN）。
- 是 → VIRTUALAPN=ENABLE，且必须先 `ADD VIRTUALAPNRULE`。
- 否 → 保持 DISABLE（默认）。

## 约束（若有）
- 虚拟 APN 前置：VIRTUALAPN=ENABLE 前必须通过 `ADD VIRTUALAPNRULE` 配置规划的虚拟 APN 映射规则，否则该 APN 下用户激活失败。`severity=high`
- 别名不冲突：ADD APN 的 APN 名称不能与 `ADD APNALIAS` 的别名 APN 重复。`severity=high`
- VPN 前置：VPNINSTANCE 必须先由 `ADD VPNINST` 创建存在。`severity=high`
- 当前版本不支持：APPLAYERVOLUME、QCHATSWITCH、PSEUDOACTSWITCH 参数。`severity=medium`
- 规格上限：最多可输入 20000 条记录。`severity=medium`
- 生效时机：执行后立即生效。`severity=low`

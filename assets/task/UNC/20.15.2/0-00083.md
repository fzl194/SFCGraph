# ADD TARIFFGROUP 原子任务
> 命令静态知识见 [ADD TARIFFGROUP](command/UNC/20.15.2/ADD-TARIFFGROUP.md)。本页只讲配置生成实例化时这条命令**怎么配**。

本命令是计费管理费率切换链路的核心建模任务：在费率切换组下按"计费属性 CC + 费率类型 + 时间段"逐条定义费率切换点。配合上游的 `ADD FESTIVAL`（节假日）/`ADD WEEKDAY`（工作日表）和下游的 `SET GLBTARIFFGROUP`（全局费率切换组）使用。

## 配置方法
按"费率切换组名 + 计费CC + 费率类型 + 起止时间"四元组配置一条费率切换点。一个组下不同费率类型（工作日/周末/节假日）需多次执行。

| 参数 | 配置方法 | 取值样例 |
|---|---|---|
| TARIFFGRPNAME | 费率切换组名（本端规划，1~31） | `"tar_nb"` |
| GLOBALFLG | 全局配置标识（本端规划，枚举；计费CC维度用 CHARGE_CHARACT） | `CHARGE_CHARACT` |
| CCVALUE | Charge Characteristic 值（本端规划，十六进制，1~6） | `"0x0800"` |
| TARIFFTYPE | 费率类型（全网规划，枚举 WORKDAY/WEEKEND/FESTIVAL） | `WORKDAY` |
| STARTTIME | 费率切换起始时间（全网规划，HH:MM） | `06&00` |
| ENDTIME | 费率切换终止时间（全网规划，HH:MM） | `17&00` |

典型脚本（组 tar_nb 的三类时间段）：
```
ADD TARIFFGROUP: TARIFFGRPNAME="tar_nb", GLOBALFLG=CHARGE_CHARACT, CCVALUE="0x0800", TARIFFTYPE=WORKDAY,  STARTTIME=06&00, ENDTIME=17&00;
ADD TARIFFGROUP: TARIFFGRPNAME="tar_nb", GLOBALFLG=CHARGE_CHARACT, CCVALUE="0x0800", TARIFFTYPE=WEEKEND,  STARTTIME=06&00, ENDTIME=17&00;
ADD TARIFFGROUP: TARIFFGRPNAME="tar_nb", GLOBALFLG=CHARGE_CHARACT, CCVALUE="0x0800", TARIFFTYPE=FESTIVAL, STARTTIME=06&00, ENDTIME=17&00;
```

## 决策点

### 计费属性 CC 取值
- 问什么：本费率切换点作用于哪个计费属性维度。
- 取值（本端规划）：十六进制 CC 值（如 `0x0800`）。
- 禁止：不能取 `0x0200`（统一费率计费不涉及费率切换，配了也无效）。`severity=high`
- 与 GLOBALFLG 联动：GLOBALFLG=CHARGE_CHARACT 表示按计费 CC 维度配置；另有 GLOBAL 全局维度（本批证据未展开）。

### 费率类型枚举
- 问什么：本切换点对应哪类日期的费率时段。
- WORKDAY（工作日）/ WEEKEND（周末）/ FESTIVAL（节假日）三选一，跨场景枚举。
- 三类通常都要配（覆盖完整周历），各执行一次本命令。

### 是否需要强制切换模式
- 问什么：除按本命令配置的时间点切换外，是否叠加每日/每月强制切换。
- 不叠加 → 下游 `SET N40QUOTACTRL` 的 TCMODE=DEFAULT（按本命令生效）。
- 每日 00:00 强切 → TCMODE=DAILY。
- 月初 00:00 强切 → TCMODE=MONTHLY。
- 属下游命令决策，此处仅声明联动。

## 约束
- 规格上限：最大记录数 1000；每组最多 144 个费率切换时间段；整系统最多 1000 个（费率切换组 × CC）组合；整系统最多 2000 个费率切换点。`severity=medium`
- 取值约束（CC 掩码一致性）：要求 `CCVALUE AND CCMASK == CCVALUE`（CCMASK 必须覆盖 CCVALUE 的所有有效位）。`severity=high`
- 取值范围：CCVALUE/CCMASK 十六进制 1~6；CCPRIORITY 1~65535；STARTTIME/ENDTIME 格式 HH:MM（脚本中用 `HH&MM`）。`severity=low`
- 禁止：CCVALUE 不能取 `0x0200`（统一费率不切换）。`severity=high`
- 生效时机：命令执行后立即生效。`severity=low`
- 前置：节假日切换需先 `ADD FESTIVAL` 建节假日表、`ADD WEEKDAY` 建工作日表（否则 FESTIVAL/WEEKEND 类型无日历依据）。`severity=medium`

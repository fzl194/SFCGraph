# SET GLBCDRFLDTEMP 原子任务

> 命令静态知识见 [SET GLBCDRFLDTEMP](command/UNC/20.15.2/SET-GLBCDRFLDTEMP.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

SET GLBCDRFLDTEMP 为三类话单（G-CDR / PGW-CDR / SGW-CDR）各绑定一个**全局**话单字段模板。模板本身由 [ADD CDRFIELDTEMP](task/UNC/20.15.2/0-00010.md) 先建，本命令只做"全局绑定"。全局唯一一条记录（最大记录数=1）。输入空格表示解除某类话单的全局绑定。

**参数取值样例（离线计费特性配置范式）**

| 参数 | 取值样例 | 默认 | 说明 |
|---|---|---|---|
| GCDRTEMPLATE | `cdrfield-test` | 无 | G-CDR 话单字段模板名，1~63 字符 |
| PGWCDRTEMPLATE | `cdrfield-test` | 无 | PGW-CDR 话单字段模板名 |
| SGWCDRTEMPLATE | `cdrfield-test` | 无 | SGW-CDR 话单字段模板名 |

**典型行（仅绑 PGW-CDR 全局模板）**

```
SET GLBCDRFLDTEMP: PGWCDRTEMPLATE="cdrfield-test";
```

**典型行（解除某类绑定，输入空格）**

```
SET GLBCDRFLDTEMP: PGWCDRTEMPLATE=" ";
```

## 决策点

### 话单字段模板的生效优先级（全局 vs APN 级）

全局绑定的话单字段优先级较低。决定是否需要本命令，取决于是否还有 APN 级绑定。

| 选项 | 配置位置 | 生效逻辑 |
|---|---|---|
| APN 级绑定优先 | `ADD APN`/APN 话单字段配置 | APN 下绑了该类模板就用 APN 的 |
| 全局兜底 | `SET GLBCDRFLDTEMP` | APN 下没绑该类模板，才用全局的 |
| 都没绑 | — | 使用系统默认 |

### 是否为三类话单都建全局模板

| 选项 | 配置 | 适用 |
|---|---|---|
| 仅 PGW-CDR（PGW-C 场景） | 只填 PGWCDRTEMPLATE | UNC 作为 PGW-C |
| 仅 SGW-CDR（SGW-C 场景） | 只填 SGWCDRTEMPLATE | UNC 作为 SGW-C |
| 仅 G-CDR（GGSN 场景） | 只填 GCDRTEMPLATE | UNC 作为 GGSN |
| 按需组合 | 多个参数 | SP 合设场景 |

## 约束

- **立即生效** · severity=LOW。
- **最大记录数=1**（全局唯一） · severity=HIGH · 重复执行为改写。
- **前置依赖**：执行本命令前必须先 [ADD CDRFIELDTEMP](task/UNC/20.15.2/0-00010.md) 建话单字段模板 · severity=HIGH。
- **解除绑定用空格**：输入空格（`" "`）表示解除该类话单的全局绑定 · severity=MEDIUM。
- **优先级低于 APN 级**：APN 下绑了该类模板则 APN 优先，全局仅在 APN 未绑时兜底 · severity=MEDIUM。

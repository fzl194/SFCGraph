# 设置全局话单字段模板（SET GLBCDRFLDTEMP）（SET GLBCDRFLDTEMP）

> 命令静态知识见 [SET GLBCDRFLDTEMP](command/UNC/20.15.2/SET-GLBCDRFLDTEMP.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法

SET GLBCDRFLDTEMP 为 G-CDR / PGW-CDR / SGW-CDR 三类话单类型分别绑定一个**全局**话单字段模板。模板本身由 [ADD CDRFIELDTEMP](task/UNC/20.15.2/0-00010.md) 先建，本命令只做"全局绑定"。全局唯一一条记录（最大记录数=1），三个参数全部可选，输入空格表示解除该类话单的全局绑定。

话单字段模板的生效优先级：**APN 级绑定 > 全局绑定（本命令）> 系统默认**。即 APN 下绑了该类模板就用 APN 的，APN 未绑才用全局，全局也未绑则用系统默认配置。本命令是 APN 未覆盖时的全局兜底。本命令的核心配置空间有 2 个维度：**话单类型选择**（按 UNC 网元角色 GGSN/PGW-C/SGW-C 决定配哪些类）、**模板名取值**（指向 ADD CDRFIELDTEMP 已建的模板名，或空格解除绑定）。

### 配置维度 1：话单类型选择（参数 GCDRTEMPLATE/PGWCDRTEMPLATE/SGWCDRTEMPLATE）

| 参数 | 控制 CDR 类型 | 适用 NF 角色 | 取值 | 默认 |
|---|---|---|---|---|
| `GCDRTEMPLATE` | G-CDR（GGSN 话单） | GGSN | 模板名 1~63 字符 / 空格(解除) | 无 |
| `PGWCDRTEMPLATE` | PGW-CDR（PGW-C 话单） | PGW-C | 模板名 1~63 字符 / 空格(解除) | 无 |
| `SGWCDRTEMPLATE` | SGW-CDR（SGW-C 话单） | SGW-C | 模板名 1~63 字符 / 空格(解除) | 无 |

> 三类话单对应不同网元角色：UNC 作 GGSN 配 GCDRTEMPLATE、作 PGW-C 配 PGWCDRTEMPLATE、作 SGW-C 配 SGWCDRTEMPLATE；SP 合设场景按需组合多类。

### 配置维度 2：模板名取值（参数值指向 ADD CDRFIELDTEMP）

| 取值 | 含义 | 前置 |
|---|---|---|
| 模板名字符串（如 `cdrfield-test`） | 绑定该类话单的全局模板 | 必须由 [ADD CDRFIELDTEMP](task/UNC/20.15.2/0-00010.md) 先建 |
| 空格（`" "`） | 解除该类话单的全局绑定 | — |

> 模板名数据来源为全网规划，引用的模板必须已存在；解除绑定用空格而非空字符串。

典型脚本（GGSN/PGW-C/SGW-C 三类话单各绑一个全局模板）：
```
SET GLBCDRFLDTEMP: GCDRTEMPLATE="gcdr", PGWCDRTEMPLATE="pgwcdr", SGWCDRTEMPLATE="sgwcdr";
```
典型脚本（仅 PGW-C 场景绑全局 PGW-CDR 模板）：
```
SET GLBCDRFLDTEMP: PGWCDRTEMPLATE="cdrfield-test";
```
典型脚本（解除 PGW-CDR 全局绑定，输入空格）：
```
SET GLBCDRFLDTEMP: PGWCDRTEMPLATE=" ";
```

**步骤位置**：在所属 feature 编排中位于"配置话单字段模板"步骤——[ADD CDRFIELDTEMP](task/UNC/20.15.2/0-00010.md)（建模板）之后，作为全局兜底绑定（证据：[配置话单字段（GGSN_SGW-C_PGW-C）](evidence/UNC/20.15.2/配置话单字段（GGSN_SGW-C_PGW-C）_95923364.md)）；APN 级绑定（若需）在 APN 配置任务中覆盖本命令。

## 决策点

### 决策点：是否为三类话单建全局模板（DP 0-03085）

问：是否需要全局绑定？为哪些话单类型（G-CDR/PGW-CDR/SGW-CDR）配全局模板？取决于 UNC 网元角色与 APN 级是否已覆盖。

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| 仅 PGW-CDR | PGWCDRTEMPLATE=模板名 | UNC 作为 PGW-C |
| 仅 SGW-CDR | SGWCDRTEMPLATE=模板名 | UNC 作为 SGW-C |
| 仅 G-CDR | GCDRTEMPLATE=模板名 | UNC 作为 GGSN |
| 按需组合 | 多个参数 | SP 合设场景（如 PGW-C+SGW-C 合设） |
| 不配（用系统默认） | 三参数均不输 | APN 级已全覆盖，或暂不需要自定义话单字段 |

> 取值由 UNC 网元角色决定。全局绑定是 APN 未覆盖时的兜底；若 APN 级已为每类话单绑了模板，可不配本命令。

### 决策点：模板名取值来源（DP 0-03086）

问：每类话单绑哪个模板名？模板名必须由 ADD CDRFIELDTEMP 先建，或输入空格解除绑定。

| 选项 | 取值 | 前置/说明 |
|---|---|---|
| 绑定已建模板 | 模板名字符串（1~63） | 必须由 [ADD CDRFIELDTEMP](task/UNC/20.15.2/0-00010.md) 先建，引用未建模板无效 |
| 解除全局绑定 | 空格（`" "`） | 解除该类话单的全局绑定，回退到系统默认 |

> 模板名数据来源为全网规划；解除绑定必须用空格，不能用空字符串。

## 约束

· rule-0-03211 · severity=high · 前置依赖：执行本命令前必须先 [ADD CDRFIELDTEMP](task/UNC/20.15.2/0-00010.md) 建话单字段模板，GCDRTEMPLATE/PGWCDRTEMPLATE/SGWCDRTEMPLATE 指向的模板名必须已存在，否则绑定无效

· rule-0-03212 · severity=high · 全局唯一一条记录（最大记录数=1），重复执行为改写而非新增；执行后立即生效，且对新激活用户立即生效

· rule-0-03213 · severity=warning · 解除绑定用空格：输入空格（`" "`）表示解除该类话单的全局绑定，不能用空字符串

· rule-0-03214 · severity=warning · 优先级低于 APN 级：全局绑定的话单字段优先级较低，APN 下绑了该类模板则 APN 优先，全局仅在 APN 未绑时兜底；全局也未绑则用系统默认配置

· rule-0-03215 · severity=info · 适用 NF：SGW-C、PGW-C、SMF（离线计费话单字段场景）

# 设置APN RADIUS计费控制（SET APNRDSACCTCTRL）

> 命令静态知识见 [SET APNRDSACCTCTRL](command/UNC/20.15.2/SET-APNRDSACCTCTRL.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

该命令以 APN 粒度控制 RADIUS 客户端计费信令处理流程。配置空间核心维度为 **阈值上报**（TIMETHRESHOLD/VOLUMETHRESHOLD）、**业务触发**（SRVTRIGGER）、**响应等待**（SUPPORTACCTRSP/SUPPORTACCTUPD）和 **变化触发**（RATTRIGGER/SGSNTRIGGER/QOSTRIGGER/ULITRIGGER 等，SUPPORTACCTUPD=ENABLE 时条件可选）。

### 配置维度1：阈值上报（参数 TIMETHRESHOLD / VOLUMETHRESHOLD）

| 取值 | 作用 | 典型场景 |
|---|---|---|
| TIMETHRESHOLD（整数，分钟） | 用户使用流量到达该时长阈值时上报 RADIUS 计费 | WSFD-011306：TIMETHRESHOLD=30（30 分钟） |
| VOLUMETHRESHOLD（整数，千字节） | 用户流量到达该阈值时上报 RADIUS 计费 | WSFD-011306：VOLUMETHRESHOLD=20（20 KB） |

### 配置维度2：业务触发（参数 SRVTRIGGER / BUFFERTIMEVALUE）

| 取值 | 作用 | 联动参数 | 典型场景 |
|---|---|---|---|
| SRVTRIGGER=ENABLE | 在用户访问特殊业务报文（3/4 层规则）时触发计费请求消息（非用户上线触发） | BUFFERTIMEVALUE（条件可选，业务触发延时秒） | WSFD-011306：SRVTRIGGER=ENABLE + BUFFERTIMEVALUE=2 |
| SRVTRIGGER=DISABLE（默认） | 用户上线时触发计费请求 | 无 | 默认 |

### 配置维度3：响应等待与超时（参数 SUPPORTACCTRSP / DEACTIVE / SUPPORTACCTSTOP）

| 组合 | 作用 | 典型场景 |
|---|---|---|
| SUPPORTACCTRSP=ENABLE | 等待计费开始响应消息后才回应激活成功应答 | 严格计费场景（WSFD-011306：SUPPORTACCTRSP=ENABLE） |
| SUPPORTACCTRSP=DISABLE + DEACTIVE | 未收到响应时是否去活用户（条件可选） | DEACTIVE=CONTINUE 不去活 |
| DEACTIVE=CONTINUE + SUPPORTACCTSTOP | 未收到 Start 响应时是否发计费停止消息（条件可选） | 精细控制 |

### 配置维度4：变化触发计费更新（参数 SUPPORTACCTUPD + 一族变化触发开关）

| SUPPORTACCTUPD | 联动条件可选（变化触发） | 作用 |
|---|---|---|
| ENABLE | RATTRIGGER / QOSTRIGGER / SGSNTRIGGER / ULITRIGGER / IPRELEASETRIGER / SUPPORTUPDRSP / CRBNCHANGE | 任一变化时触发计费更新（WSFD-011306：SUPPORTACCTUPD=ENABLE + RATTRIGGER=ENABLE） |
| DISABLE | 上述均无效 | 不发计费更新 |

### 配置维度5：其他开关（节选）

| 参数 | 作用 | 默认 |
|---|---|---|
| ACCTALLBEAR | 全部承载/PDP 均开 RADIUS 计费 | DISABLE |
| UELOCIPTRIGGER | WLAN 地址变化触发计费更新 | DISABLE |
| CACHEACCTSTOPSW | 缓存 Account Stop（RADIUS 无响应时） | DISABLE |
| CFCATETRIGGER | URL 过滤规则变化触发计费更新 | DISABLE |

**步骤位置**：在 RADIUS 功能（WSFD-011306）编排中，于 `ADD APN`、`ADD APNRDSSVRGRP`、`SET APNRADIUSATTR` 之后执行，用于调优 RADIUS 计费信令流程。

## 决策点

**计费触发模式选择**（UNC pre-build 内嵌，不编号——按 `assets/CLAUDE.md` §5.7）：

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| 业务触发 | SRVTRIGGER=ENABLE + BUFFERTIMEVALUE | 业务报文触发 RADIUS（WSFD-011306 配置业务触发） |
| 阈值+等待+更新全套 | TIMETHRESHOLD + VOLUMETHRESHOLD + SUPPORTACCTRSP=ENABLE + SUPPORTACCTUPD=ENABLE + RATTRIGGER | 严格计费场景（WSFD-011306 综合示例） |
| 简单（默认） | 仅 APN，其他默认 | 基础 RADIUS 计费 |

## 约束

- **APN 必须先建**（critical）：APN 必须由 `ADD APN` 已配置 — 后果：APN 不存在则配置失败
- **条件可选参数前提**（warning）：BUFFERTIMEVALUE 需 SRVTRIGGER=ENABLE；DEACTIVE 需 SUPPORTACCTRSP=DISABLE；SUPPORTACCTSTOP 需 DEACTIVE=CONTINUE；RATTRIGGER/QOSTRIGGER/SGSNTRIGGER/ULITRIGGER/IPRELEASETRIGER/SUPPORTUPDRSP/CRBNCHANGE 需 SUPPORTACCTUPD=ENABLE — 后果：前提不满足时配置无效
- **APN 命名约束**（warning）：字符串 1~63 — 后果：违规命名配置失败
- **TIMETHRESHOLD/VOLUMETHRESHOLD 范围**（warning）：取值范围见命令 md — 后果：超范围配置失败
- **BUFFERTIMEVALUE 范围**（warning）：业务报文延时秒，范围见命令 md — 后果：超范围配置失败

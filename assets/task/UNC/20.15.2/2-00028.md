# Radius 抄送功能（WSFD-011307）

> 特性静态知识见 [WSFD-011307](feature/UNC/20.15.2/WSFD-011307.md)。本页讲配置生成实例化时这个特性**怎么配**——精简版 RADIUS 链路（RDSSVRGRP+RDSSVR+APNRDSSVRGRP）+ `PRIFLAG=CARBON_COPY` 标识抄送。

## 配置概览

核心对象链：**AAA 服务器组**（RDSSVRGRP）→ **AAA 服务器**（RDSSVR，PRIFLAG=PRIMARY/SECONDARY/CARBON_COPY 区分主备抄送）→ **APN 绑 AAA 组**（APNRDSSVRGRP）→ **APN 计费控制**（APNRDSACCTCTRL，可选）→ **APN 域名剥离**（APNRADIUSATTR，可选）。

本特性是 [WSFD-011306](feature/UNC/20.15.2/WSFD-011306.md) Radius 功能的**精简扩展**——仅启用计费/鉴权主链路 + 通过 `PRIFLAG=CARBON_COPY` 标识抄送服务器。**前提**：[WSFD-011306](feature/UNC/20.15.2/WSFD-011306.md) 已激活主链路。

## 配置流程（步骤编排，每步显示含命令）

1. **前置依赖**（必先开）：[WSFD-011306](feature/UNC/20.15.2/WSFD-011306.md) Radius 功能（建 RDSSVRGRP+鉴权/计费 RDSSVR+APNRDSSVRGRP 主链路）
2. **接入域基础设施**（建 APN 实例 + VPN）：`ADD L3VPNINST` + `ADD VPNINSTAF` + `ADD VPNINST` + `ADD APN` + `SET APNADDRESSATTR` → [UNC 1-00024](task/UNC/20.15.2/1-00024.md)
3. **（按需）建独立抄送 AAA 服务器组**：`ADD RDSSVRGRP: RDSSVRGRPNAME="isprg"` → [0-00255](task/UNC/20.15.2/0-00255.md)
4. **建 VPN 实例**（抄送服务器独立 VPN 场景）：`ADD VPNINST: VPNINSTANCE="vpntest"` → [0-00098](task/UNC/20.15.2/0-00098.md)
5. **（按需）建 UP 列表**（UPF 中转 + 抄送组合场景）：`ADD UPLIST4RDS` → [0-00260](task/UNC/20.15.2/0-00260.md)
6. **配置 AAA 服务器（含抄送服务器）**：
   - 主 AAA 计费服务器：`ADD RDSSVR: RDSSVRGRPNAME="isprg", SERVERTYPE=ACCOUNTING, PRIFLAG=PRIMARY, SERVERIPV4="192.168.8.181", PORT=1819, VPNINSTANCE="vpntest", CIPHERKEY="*****"` → [0-00254](task/UNC/20.15.2/0-00254.md)
   - 备 AAA 鉴权服务器：`ADD RDSSVR: RDSSVRGRPNAME="isprg", SERVERTYPE=AUTHENTICATION, PRIFLAG=SECONDARY, SERVERIPV4="192.168.8.182", PORT=1819, VPNINSTANCE="vpntest", CIPHERKEY="*****"` → [0-00254](task/UNC/20.15.2/0-00254.md)
   - **抄送 AAA 服务器**：`ADD RDSSVR: RDSSVRGRPNAME="isprg", SERVERTYPE=AUTHENTICATION, PRIFLAG=CARBON_COPY, SERVERIPV4="192.168.8.183", PORT=1819, VPNINSTANCE="vpntest", CIPHERKEY="*****"` → [0-00254](task/UNC/20.15.2/0-00254.md)
7. **APN 绑 AAA 组**：`ADD APN: APN="apn-test"` + `ADD APNRDSSVRGRP: APN="apn-test", RDSSVRGRPNAME="isprg"` → [0-00004](task/UNC/20.15.2/0-00004.md) + [0-00222](task/UNC/20.15.2/0-00222.md)
8. **（可选）RADIUS 私有扩展**：3GPP VSA + APN 计费控制 + APN 域名剥离 → [UNC 1-00026](task/UNC/20.15.2/1-00026.md) 步骤 1-3

> **★ 抄送服务器标识**：`PRIFLAG=CARBON_COPY` 是本特性**核心差异**——同 RDSSVR 命令，加此参数即标识为抄送服务器。每 RADIUS Server Group 最多配置 64 个抄送服务器。
>
> **★ 与 [WSFD-011306](feature/UNC/20.15.2/WSFD-011306.md) 复用**：主链路（RDSSVRGRP+RDSSVR 鉴权+计费+APNRDSSVRGRP）**共用** [WSFD-011306](feature/UNC/20.15.2/WSFD-011306.md) 步骤 2；本特性**仅**新增抄送服务器（1+条 `PRIFLAG=CARBON_COPY` RDSSVR）。

## 决策点（特性级：场景影响表）

| 抄送服务器数 × 主备模式 × 是否独立 AAA 组 | 走法（核心步骤） | 联动 / 关键参数 |
|---|---|---|
| **1 个抄送服务器** × 主备 + 抄送 × 复用主 AAA 组 | 步骤 1+6（加抄送 RDSSVR）+ 7 | PRIFLAG=CARBON_COPY |
| **多抄送服务器**（增值服务多接收方）| 步骤 1+6 重复 `ADD RDSSVR PRIFLAG=CARBON_COPY` + 7 | 每组最多 64 抄送 |
| **独立抄送 AAA 组**（独立 VPN）| 步骤 1+3+4+6+7 | 独立 RDSSVRGRPNAME |
| **UPF 中转 + 抄送**（5G UPF 下沉）| 步骤 1+5+6+7 | 步骤 5 `ADD UPLIST4RDS` |
| **3GPP VSA + 抄送** | 步骤 1+6+7+8 | 步骤 8 `SET RDSACCTREQVSA` |
| **APN 级计费控制 + 抄送** | 步骤 1+6+7+8 | 步骤 8 `SET APNRDSACCTCTRL` |
| **APN 域名剥离 + 抄送** | 步骤 1+6+7+8 | 步骤 8 `SET APNRADIUSATTR` |

> **走法列**：所有场景**共享步骤 1 前置依赖**；差异在步骤 3-8 子命令的**重复/独立配置**。

## 约束（特性级 rule）

- **必先开 WSFD-011306**（critical）：本特性使用前要激活 Radius 功能 — 后果：未激活前置特性致抄送链路无意义。
- **每 RADIUS Server Group 最多 64 抄送服务器**（warning）：超出配置失败。
- **抄送服务器可与主备服务器共享 RDSSVR 命令**（info）：同 `ADD RDSSVR` 命令，仅 `PRIFLAG` 参数区分。
- **PRIFLAG 取值**（info）：3 取值 PRIMARY（主）/ SECONDARY（备）/ CARBON_COPY（抄送）；本特性专用于 CARBON_COPY 场景。
- **用户在线时删除抄送服务器配置**（warning）：可能造成抄送消息失败 — 约束归本特性对系统的影响段。
- **本特性无需 License**（info）：适用 NF 为 GGSN/PGW-C/SMF。
- **首次发布版本**（info）：UNC **20.3.0**（首次发布）。
- **替代 AAA Server 转发**（info）：本特性可替换现网 AAA Server 转发消息的方式，降低对 AAA Server 的性能要求。
- **同步 AAA 信息到增值服务器**（info）：提升增值服务能力（IP 地址/APN/MSISDN）。

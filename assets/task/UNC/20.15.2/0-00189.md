# SET NBIOTRATVALUE 原子任务
> 命令静态知识见 [SET NBIOTRATVALUE](command/UNC/20.15.2/SET-NBIOTRATVALUE.md)。本页只讲配置生成实例化时这条命令**怎么配**。

该命令按周边网元类型分 9 个 RAT 值参数，控制 NB-IoT 终端接入时各接口消息中 RAT-Type 信元的填写。系统部署后已存在初始记录（默认 EUTRAN_NB_IOT 等）。被 3 个特性引用，核心场景为 PCRF 免升级（将 EUTRAN_NB_IOT 替换为 PCRF 可识别的 EUTRAN）。

## 配置方法
按对接网元是否支持 EUTRAN-NB-IoT 类型，逐个网元参数配置对应 RAT 值；未输入参数保持系统当前值不变。下表列实际激活 md 中显式配置的网元参数。

| 参数 | 描述 | 配置方法 | 取值样例 | 典型场景 |
|---|---|---|---|---|
| PCRF | 和 PCRF 交互使用的 RAT 值 | 本端规划 | `EUTRAN` / `EUTRAN_NB_IOT` | PCRF 免升级（WSFD-215302）：PCRF 不支持 EUTRAN-NB-IoT 时配 EUTRAN |
| PCF | 和 PCF 交互使用的 RAT 值 | 本端规划 | `EUTRAN` / `EUTRAN_NB_IOT` | PCC 基本功能（WSFD-109101 SMF 场景）：PCF 不支持时配 EUTRAN |
| OCS | 和 OCS 交互使用的 RAT 值 | 对端协商 | 枚举 | 按对端能力 |
| CG | 和 CG 交互使用的 RAT 值 | 对端协商 | 枚举 | 按对端能力 |
| AAAACCT | 和 AAA 计费服务器交互使用的 RAT 值 | 对端协商 | 枚举 | 按对端能力 |
| AAAAUTH | 和 AAA 鉴权服务器交互使用的 RAT 值 | 对端协商 | 枚举 | 按对端能力 |
| PGW | SGW 发送给 PGW 使用的 RAT 值 | 对端协商 | 枚举 | 按对端能力 |
| CHF | 和 CHF 交互使用的 RAT 值 | 对端协商 | 枚举 | 按对端能力 |
| UPF | 和 UPF 交互使用的 RAT 值 | 对端协商 | 枚举 | 按对端能力 |

典型脚本（WSFD-215302 PCRF 免升级，PCRF 不支持 EUTRAN-NB-IoT）：
```
SET NBIOTRATVALUE: PCRF=EUTRAN;
```
典型脚本（WSFD-109101 PGW-C 对接，PCRF/PCF 支持 EUTRAN-NB-IoT）：
```
SET NBIOTRATVALUE: PCRF=EUTRAN_NB_IOT, PCF=EUTRAN_NB_IOT;
```
典型脚本（WSFD-109101 SMF 对接，PCF 支持 EUTRAN-NB-IoT）：
```
SET NBIOTRATVALUE: PCF=EUTRAN_NB_IOT;
```

## 决策点

### 各网元 RAT 值（PCRF/PCF 等）—— 对端能力二选一
- 问什么：每个对接网元的 RAT 信元填 EUTRAN_NB_IOT 还是 EUTRAN。
- 取值由对端网元是否支持 EUTRAN-NB-IoT 类型决定：支持→EUTRAN_NB_IOT（系统初始值，透传）；不支持→EUTRAN（替换，需配合 License）。跨场景枚举，是本命令核心决策。
- 三特性（WSFD-011601/WSFD-215302/WSFD-109101）均围绕 PCRF/PCF 这一对参数决策。

### PCRF=EUTRAN 的连锁（PCRF 免升级 License）
- 问什么：PCRF 设为 EUTRAN 时还需要做什么。
- 跨命令前置：PCRF=EUTRAN 时还需开启 PCRF 免升级 License（对应 WSFD-215302 激活流程，SET LICENSESWITCH 开对应 LICITEM）。

## 约束
- 唯一性：全局一份（最大记录数为 1，系统部署后已存在初始记录）。`severity=high`
- 生效时机：执行后立即生效。`severity=high`
- 前置（License 联动）：PCRF=EUTRAN 时需开启 PCRF 免升级 License（WSFD-215302）。`severity=high`
- 对端协商一致性：OCS/CG/AAA/PGW/CHF/UPF 等 6 个参数取值依赖对端能力，需逐网元确认。`severity=high`
- 默认值：系统初始 PCRF/PCF=EUTRAN_NB_IOT。`severity=low`
- 适用 NF：SGW-C、PGW-C、SMF。`severity=low`

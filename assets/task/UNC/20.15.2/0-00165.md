# SET CNVRGDCHGPARA 原子任务
> 命令静态知识见 [SET CNVRGDCHGPARA](command/UNC/20.15.2/SET-CNVRGDCHGPARA.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法
SET CNVRGDCHGPARA 用于 PGW-C/SMF 配置融合计费（N40 接口）全局参数，共 25 个参数。系统部署后存在初始记录，仅按需修改偏离默认值的参数。按功能分为六组：CHF 响应异常处理、用量上报与配额管理、Trigger 控制与填写、ChargingDataRef 生成、位置信息与时间戳、消息缓存与放通。

| 功能组 | 关键参数 | 典型脚本 |
|---|---|---|
| CHF 响应异常信元处理 | `IGNORERSPAVP=FUI` + `BADRSPACT=TERMINATE` | `SET CNVRGDCHGPARA:IGNORERSPAVP=FUI,BADRSPACT=TERMINATE;` |
| ChargingDataRef 由 SMF 生成（消息缓存回放前提） | `CHGDATAREFGEN=SMF` | `SET CNVRGDCHGPARA:CHGDATAREFGEN=SMF;` |
| 多 UPF 配额管理 | `MULTIUPFQUOTAM=QUOTAINDEP` | `SET CNVRGDCHGPARA:MULTIUPFQUOTAM=QUOTAINDEP;` |
| 用量上报方式 | `RPTBASEDONGU=DISABLE`（按实际用量）/ `ENABLE`（按 GrantedUnit） | `SET CNVRGDCHGPARA:RPTBASEDONGU=DISABLE;` |
| RG VT 事件合并上报 | `MERGERGVTSW=ENABLE`（合并）/ `DISABLE`（单独） | `SET CNVRGDCHGPARA:MERGERGVTSW=ENABLE;` |

**位域类参数**（支持单独或组合配置）：`IGNORERSPAVP`（忽略 CHF 响应信元：`INVOCATIONNUM`/`FUI`/...）、`IGNORETRIGGER`（忽略 CHF 下发 Trigger 列表：`MAXNUMCCC`/会话级时间/会话级流量/RG 级时间/RG 级流量/配额保持时长）、`IGNOREERRTRIGS`。

## 决策点

### CHF 响应异常处理动作
问：CHF 响应携带异常信元致 SMF 无法计费时，去活还是放通？
- `BADRSPACT=TERMINATE`：去活 PDU 会话（默认推荐，计费严谨）。
- `BADRSPACT=CONTINUE`：用户业务继续，不再进行配额管理（放通不计费，需配合 `CONTINUEALARM=ENABLE` 开启放通告警）。

### 用量上报方式
问：SMF 按实际用量还是按 CHF 下发 GrantedUnit 上报？
- `RPTBASEDONGU=DISABLE`：按实际使用量上报（默认）。
- `RPTBASEDONGU=ENABLE`：基于 CHF 下发 GrantedUnit 做配额管理，用量上报不超过配额。

### Trigger 控制来源
问：CHF 下发的 Trigger 与 SMF 本地配置（ADD PDUTRIGGER/ADD RGTRIGGER）哪个生效？
- 默认采用 CHF 响应携带的 Trigger。
- `IGNORETRIGGER` 配置位域后：忽略 CHF 指定 Trigger，采用本地配置生效；若 CHF 响应未携带被忽略的 Trigger，该 Trigger 仍用本地值。

### 异常去活 triggerType 填写（SPTABNTRIGGER）
问：用户异常去活时 Nchf Release 消息 triggerType 填 FINAL 还是 AbnormalRelease？
- 由 [ADD N40DIAGTRIGGER](task/UNC/20.15.2/0-00045.md) 的 `RELEASETRIGGER` 优先控制（配置了去活原因时）。
- 未配置时：`SPTABNTRIGGER=ENABLE` 支持 AbnormalRelease；`DISABLE` 改填 FINAL。

### ChargingDataRef 生成与格式
问：ChargingDataRef 由谁生成、什么格式、如何使用？
- `CHGDATAREFGEN=SMF`/`CHF`：生成方（消息缓存回放功能要求 `SMF`）。
- `CHGDATAREFFMT`：格式（conditional）。
- `CHGDATAREFUSE`：使用方式（conditional）。

## 约束
- `OFLRGTIMECALC` 和 `CHGDATAREFFMT` 功能仅针对新用户生效（已激活用户不追溯）。`severity=high`
- 消息缓存回放功能需三参数联动：`CHGDATAREFGEN=SMF`（本命令）+ [SET N40MSGSTG](task/UNC/20.15.2/0-00184.md) 的 `STGSWITCH=ENABLE` + [ADD PDUSCACT](task/UNC/20.15.2/0-00061.md) 的 `SCACT=FAILOVER/CONTINUE`。`severity=high`
- `IGNORERSPAVP`/`IGNORETRIGGER` 为位域类型，支持单独或组合配置（不互斥）。`severity=medium`
- `BADRSPACT=CONTINUE` 放通场景建议配合 `CONTINUEALARM=ENABLE` 开启告警。`severity=medium`
- 系统部署完成后存在初始记录（各参数默认值见命令联机帮助表），仅按需修改偏离默认值参数。`severity=low`
- 该命令执行后立即生效。`severity=low`

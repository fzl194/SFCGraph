# ADD PCFSELPLCY 原子任务
> 命令静态知识见 [ADD PCFSELPLCY](command/UNC/20.15.2/ADD-PCFSELPLCY.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法
本命令为 AMF 增加 PCF 选择策略，按**用户范围（SUBRANGE）**圈定用户群，再用一组开关（SUPI/GPSI/网络切片/目标 PLMN/重选/Serving Scope）决定匹配维度。系统部署后已有初始记录，本命令在其上增加差异化策略。

| 用户范围 | 必备参数 | 典型脚本 |
|---|---|---|
| HOME_USER（归属用户，按 SUPI） | `SUBRANGE=HOME_USER` + `TGTPLMNSW=NO` + `SUPISW=YES` + `GPSISW=NO` + `NSSW=NO` + `RETRYSW=YES` | `ADD PCFSELPLCY:SUBRANGE=HOME_USER,TGTPLMNSW=NO,SUPISW=YES,GPSISW=NO,NSSW=NO,RETRYSW=YES,DESC="for home user";` |
| USER_GROUP（指定用户群，白名单+Serving Scope） | `SUBRANGE=USER_GROUP` + `USRGRPID`（已配 NGUSRGRP） + `CTRLMODE=WHITELIST` + `SSSW=YES` + `SERVINGSCOPE` | `ADD PCFSELPLCY:SUBRANGE=USER_GROUP,USRGRPID=20,CTRLMODE=WHITELIST,SSSW=YES,SERVINGSCOPE="YES",DESC="for roaming user";` |

**SUBRANGE 取值语义**：`ALL_USER`（所有用户）/ `HOME_USER`（归属用户）/ `USER_GROUP`（指定用户群，需配 USRGRPID）/ `IMSIPRE`（IMSI 前缀，需配 IMSIPRE）等。

## 决策点

### 用户范围 SUBRANGE
问：本策略针对哪类用户？
- `ALL_USER`：所有用户（系统初始记录即此）。
- `HOME_USER`：归属用户，通常按 SUPI 选择 PCF（SUPISW=YES）。
- `USER_GROUP`：指定用户群，须先用 ADD NGUSRGRP 创建 USRGRPID，并配 CTRLMODE（白名单等）。
- `IMSIPRE`：按 IMSI 前缀细分，须配 IMSIPRE（5~15 位数字）。

### 用户标识维度（多开关组合）
问：用哪些标识匹配 PCF？
- `SUPISW`（SUPI，默认 YES）/ `GPSISW`（GPSI，默认 NO）/ `NSSW`（网络切片，默认 NO）/ `TGTPLMNSW`（目标 PLMN，默认 YES）——按规划开关，可叠加。
- `SSSW=YES` 时须配 `SERVINGSCOPE`（服务区，可为多个用 `:` 分隔，如 "pudong:xuhui"）+ 可选 `NBEGIN/NEND`（区域标识起止位置 1~12），用于为指定用户群选特定区域服务的 PCF。

### CTRLMODE（仅 USER_GROUP）
问：用户群匹配用白名单还是其他？
- `WHITELIST`：白名单（证据中漫游地分流场景采用）。

### 是否重选 PCF
问：初选 PCF 收到 5xx 时是否重选？
- `RETRYSW=YES`（默认）：重选 PCF 再次重试 AM 策略/UE 策略偶联创建。
- `RETRYSW=NO`：不重试。

## 约束
- SUBRANGE=USER_GROUP 时 USRGRPID 必填（conditional），且 USRGRPID 须已通过 ADD NGUSRGRP 创建。`severity=high`
- SUBRANGE=USER_GROUP 时 CTRLMODE 必填（conditional）。`severity=high`
- SSSW=YES 时 SERVINGSCOPE 必填（conditional）。`severity=high`
- USRGRPID 取值 0~4294967294。`severity=medium`
- IMSIPRE 长度 5~15，仅十进制数字。`severity=medium`
- SERVINGSCOPE 长度 0~64，多个用 `:` 分隔。`severity=medium`
- NBEGIN/NEND 取值 1~12。`severity=medium`
- DESC 长度 0~32，自定义便于运维。`severity=low`
- 执行后立即生效。`severity=low`

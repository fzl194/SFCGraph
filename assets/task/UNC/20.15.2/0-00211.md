# 配置用户位置组（ADD USRLOCATIONGRP）

> 命令静态知识见 [ADD USRLOCATIONGRP](command/UNC/20.15.2/ADD-USRLOCATIONGRP.md)。本页讲实例化怎么配。

## 配置方法

| 参数 | 取值样例 | 来源 |
|---|---|---|
| LOCGROUPNAME | locgrp_test | 本端规划（不重复）|
| LOCATIONNAME | loc_test1 / loc_test2 / loc_test3 | 已配置数据中获取（[ADD USRLOCATION](task/UNC/20.15.2/0-00210.md) 定义）|

**典型脚本**（逐位置一条，绑到同一组）：
```
ADD USRLOCATIONGRP:LOCGROUPNAME="locgrp_test",LOCATIONNAME="loc_test1";
ADD USRLOCATIONGRP:LOCGROUPNAME="locgrp_test",LOCATIONNAME="loc_test2";
ADD USRLOCATIONGRP:LOCGROUPNAME="locgrp_test",LOCATIONNAME="loc_test3";
```

## 约束

- **位置名须先建**（critical）：LOCATIONNAME 引用 [ADD USRLOCATION](task/UNC/20.15.2/0-00210.md) — 否则绑定失败。
- **逐位置一条**（info）：多位置场景重复执行 ADD USRLOCATIONGRP。

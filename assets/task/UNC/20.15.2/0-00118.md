# MOD CHGIMSICFG 原子任务
> 命令静态知识见 [MOD CHGIMSICFG](command/UNC/20.15.2/MOD-CHGIMSICFG.md)。本页只讲调测/运维实例化时这条命令**怎么用**。

适用网元：SGSN。本命令修改已存在的 IMSI 前缀计费配置条目，按 IMSI 维度落实 S-CDR 生成策略（按周期/流量/计费条件变更/位置更新），是 SGSN 离线计费参考信息（WSFD-011201）中 IMSI 维度计费配置的修改入口。

## 配置方法
按"用户范围 + IMSI 前缀 + APN"定位条目，修改 S-CDR 生成策略各开关与阈值。

| 参数 | 配置方法 | 取值样例 |
|---|---|---|
| SUBRANGE | 用户范围（必选） | `ALL_USER` |
| IMSIPRE | IMSI 前缀（条件必选，5~15 位十进制） | `46000` |
| APNNI | APN（必选，≤62 字符） | `cmnet` |
| VISITTYPE | 拜访类型 | `ROAMING`（使用归属地 GGSN 的漫游用户） |
| CC | 计费属性 | `HOT_BILLING`（实时计费） |
| SP | 生成 S-CDR | `NO`/`YES` |
| SPP | 周期生成 S-CDR | `FORBID`（不生成） |
| SPL | S-CDR 生成周期（0~1440 min） | `60` |
| SVP | 流量生成 S-CDR | `NO`/`YES` |
| SVL | S-CDR 流量阈值（0~1000000 KB） | `1024` |
| SCCP | 计费条件变更生成 S-CDR | `NO`/`YES` |
| SCCL | 最大计费条件变更次数（0~10） | `5` |
| SLCP | 位置更新生成 S-CDR | `NO`/`YES` |

## 决策点

### S-CDR 生成策略组合（周期 / 流量 / 计费条件变更 / 位置更新）
S-CDR 生成有四个并列触发维度（SPP 周期、SVP 流量、SCCP 计费条件变更、SLCP 位置更新），每个维度开关 + 阈值。具体启用哪些触发维度由话单规划决定，存在跨场景取值分叉（如某些场景只开周期，某些场景只开流量）。获取方法稳定（整网规划），但组合取值随场景分叉。

### PLMN ID 控制策略联动（LOCINFO 分支）
notes 指出：当 SET CHGCDR 的 PLMNCTRL 取值为"LOCINFO 中的 PLMN ID"时，应选择 ADD/MOD CHGPLMNCFG（而非 CHGIMSICFG）配置。即 PLMN 控制策略的取值决定走 IMSI 维度还是 PLMN 维度的计费配置命令。该选择由 [SET CHGCDR](task/UNC/20.15.2/0-00159.md) 的 PLMNCTRL 参数驱动。

## 约束
- **仅对之后激活的用户有效**：配置立即生效，但只影响新激活用户。
- **默认值为无效值**：各字段默认值都为无效值，需显式规划；不规划的字段不会生效。
- **条目存在性前置**：MOD 修改前需先 ADD CHGIMSICFG 建立条目；可用 LST CHGIMSICFG 校验。
- **IMSIPRE 格式**：5~15 位十进制字符串。
- **SPL 取值范围**：0~1440 min；SVL：0~1000000 KB；SCCL：0~10。
- **PLMNCTRL 联动**：PLMNCTRL=LOCINFO 时走 CHGPLMNCFG 而非本命令。

# 配置 PCC 开关与模板

> PCC 功能使能的开关层——License 前置 + 全局 PCC 开关（SET PCCFUNC）+ PCC 模板（ADD PCCTEMPLATE，承载故障处理动作）+ APN 级 PCC 开关与模板绑定（SET APNPCCFUNC）。被引用于 [2-00005 PCC基本功能](task/UNC/20.15.2/2-00005.md) · [2-00007 会话级FUP](task/UNC/20.15.2/2-00007.md) · [2-00011 业务级FUP](task/UNC/20.15.2/2-00011.md)（FUP 仅复用 SET PCCFUNC atom 增 MKPARSEFORMAT，见场景差异）。域通用（动态 PCC / 本地 PCC 共享本骨架，差异在开关取值，见场景差异）。

## 配置方法

| 步骤 | 命令 | 关键参数 |
|---|---|---|
| 激活 License | `SET LICENSESWITCH` | LICITEM=LKV2PCCBF01/LKV3W9SPCC11；SWITCH=ENABLE（前置） |
| 配全局 PCC 开关 | `SET PCCFUNC` | HOMEPCCSWITCH/ROAMPCCSWITCH/VISITPCCSWITCH=ENABLE/DISABLE（ENABLE=动态 PCC；DISABLE=本地 PCC）；COMPOLICY=LOCALAAA 等（对端协商） |
| 配 PCC 模板 | `ADD PCCTEMPLATE` | PCCTEMPNAME；SELPEERFAILACT（选 PCRF 失败动作）；INITIALFAILACT（Initial 流程故障动作=FORBIDDEN 等）；UPDATEFAILACT（Update 流程故障动作=INHERIT 等）；NTFRSRURI/N7FAILOVERSW/PCFLBPARA（5G/主备场景，见场景差异） |
| 配 APN 级 PCC 开关 + 绑模板 | `SET APNPCCFUNC` | APN；HOMEPCCSWITCH/ROAMPCCSWITCH/VISITPCCSWITCH=ENABLE/DISABLE/INHERIT；PCCTEMPLATE（PCC 模板名）；PCFSELECTMODE（5G 按 DNN/SERVINGSCOPE 选 PCF，见场景差异） |

**典型脚本（动态 PCC，全局+APN 级使能，绑 PCC 模板）**：
```
SET LICENSESWITCH: LICITEM="LKV2PCCBF01", SWITCH=ENABLE;
SET LICENSESWITCH: LICITEM="LKV3W9SPCC11", SWITCH=ENABLE;
SET PCCFUNC: HOMEPCCSWITCH=ENABLE, ROAMPCCSWITCH=ENABLE, VISITPCCSWITCH=ENABLE, COMPOLICY=LOCALAAA;
ADD PCCTEMPLATE: PCCTEMPNAME="pcctemplate", SELPEERFAILACT=DEFAULT, INITIALFAILACT=FORBIDDEN, UPDATEFAILACT=INHERIT;
SET APNPCCFUNC: APN="huawei.com", HOMEPCCSWITCH=ENABLE, ROAMPCCSWITCH=ENABLE, VISITPCCSWITCH=ENABLE, PCCTEMPLATE="pcctemplate";
```

**步骤位置**：在 [1-00012](task/UNC/20.15.2/1-00012.md)（PCRF 对接链）+ [1-00013](task/UNC/20.15.2/1-00013.md)（PCRF 选择）之后；之后是本地规则与业务策略组合（[2-00005](task/UNC/20.15.2/2-00005.md) 步骤 5-7，复用费率标识链 atom 子集）。

## 场景差异

- **动态 PCC（网络部署 PCRF/PCF）**：`SET PCCFUNC`/`SET APNPCCFUNC` 的 HOMEPCCSWITCH/ROAMPCCSWITCH/VISITPCCSWITCH=ENABLE；UNC 从 PCRF/PCF 获取策略。须先完成 [1-00012](task/UNC/20.15.2/1-00012.md)（PCRF 对接）+ [1-00013](task/UNC/20.15.2/1-00013.md)（PCRF 选择）。被引用于 [2-00005 PCC基本功能](task/UNC/20.15.2/2-00005.md)。
- **本地 PCC（网络未部署 PCRF）**：`SET PCCFUNC` 的 HOMEPCCSWITCH/ROAMPCCSWITCH/VISITPCCSWITCH=DISABLE（关闭全局动态 PCC = 开启全局本地 PCC）；UNC 本地配静态规则（[ADD RULE](task/UNC/20.15.2/0-00071.md)），按全局优先级匹配。无需配 PCRF 对接链（[1-00012](task/UNC/20.15.2/1-00012.md) 跳过）。ADD PCCTEMPLATE 的 INITIALFAILACT=FORBIDDEN（PCRF 故障即拒绝，本地 PCC 不查 PCRF）。
- **5G 代际（Npcf 接口）**：ADD PCCTEMPLATE 增加 5G 容灾参数——NTFRSRURI=ENABLE（基于 Notify 消息 ResourceURI 触发 PCF 重选）、N7FAILOVERSW=ENABLE（N7 Failover）、PCFLBPARA=GROUPID/PRIORITY（PCF 负荷分担）。SET APNPCCFUNC 增 PCFSELECTMODE=DNN/SERVINGSCOPE/NFLOC/SUPI（5G PCF 选择维度）。COMPOLICY=LOCALAAA 仍配。
- **APN 级优先级高于全局**（info）：`SET APNPCCFUNC` 的 PCC 开关优先级高于 `SET PCCFUNC`；APN 级 INHERIT=继承全局开关。
- **APN 级 PCF 选择方式（5G）**：SET APNPCCFUNC 的 PCFSELECTMODE 复选 DNN/SERVINGSCOPE/NFLOC/SUPI；选 SERVINGSCOPE 需配 [ADD PCFSSCOPE](task/UNC/20.15.2/0-00055.md)+[ADD USRTAIRANGE](task/UNC/20.15.2/0-00097.md)+[ADD PCFSSCOPEBIND](task/UNC/20.15.2/0-00056.md)；选 NFLOC 需 [MOD NFPROFILE](task/UNC/20.15.2/0-00120.md) 加 LOCALITY。
- **FUP 累计流量增量（仅 SET PCCFUNC atom 复用，Gx 路径）**：[2-00007 会话级FUP](task/UNC/20.15.2/2-00007.md) 与 [2-00011 业务级FUP](task/UNC/20.15.2/2-00011.md) 均复用本 compound 的 `SET PCCFUNC` atom，但**参数焦点不同**——FUP 仅增 `MKPARSEFORMAT=UNSIGNED32`（Monitoring-Key 解析方式，须与 PCRF 一致），**不重配 HOMEPCCSWITCH**（已在 PCC 基本功能配过）。本 compound 典型脚本演示的是 HOMEPCCSWITCH/COMPOLICY 主开关；FUP 场景配置生成时对已激活 PCC 基本功能的局点走 `SET PCCFUNC:MKPARSEFORMAT=UNSIGNED32;` 单参数增量（MOD 语义）。两个 FUP 特性的差异在 URR 标识链：会话级（2-00007）不配 URR 链（仅 3 参数增量），业务级（2-00011）复用费率标识链 [1-00009](task/UNC/20.15.2/1-00009.md) 的 MONITORINGKEY 变种 + PCCPOLICYGRP 带 FUPSESSIONEXC。FUP 另两增量（MOD PCRF 使能 UMCH / MOD PCCPOLICYGRP 设 FUPSESSIONEXC）是单命令修改，直接引 atom 不走本 compound。

## 决策点

| 选项 | 影响（参数/联动） |
|---|---|
| 全局 HOMEPCCSWITCH=ENABLE | 动态 PCC；须配 PCRF 对接 [1-00012](task/UNC/20.15.2/1-00012.md) + PCRF 选择 [1-00013](task/UNC/20.15.2/1-00013.md) |
| 全局 HOMEPCCSWITCH=DISABLE | 本地 PCC；UNC 本地配静态规则；不配 PCRF 对接 |
| APN 级 HOMEPCCSWITCH=ENABLE/DISABLE/INHERIT | 覆盖全局；ENABLE=该 APN 动态 PCC；DISABLE=该 APN 本地 PCC；INHERIT=继承全局 |
| INITIALFAILACT=FORBIDDEN | PCRF 故障时拒绝用户（本地 PCC 典型） |
| UPDATEFAILACT=INHERIT | Update 流程继承 Initial 故障动作 |
| NTFRSRURI=ENABLE（5G） | 支持 PCF 通过 Notify 消息 ResourceURI 触发 PCF 重选 |

> 联动特性级 DP：动态 PCC / 本地 PCC / 5G 的选择由 [2-00005](task/UNC/20.15.2/2-00005.md) 的 PCC 模式 DP 与代际 DP 驱动。

## 约束

- **License 前置**（critical）：SET LICENSESWITCH 激活 LKV2PCCBF01 + LKV3W9SPCC11 须先于其他步骤 — 否则 PCC 功能不生效。
- **动态 PCC 须先配 PCRF 对接**（critical）：动态 PCC 场景下，SET PCCFUNC 使能前须完成 [1-00012](task/UNC/20.15.2/1-00012.md) PCRF 对接链 — 否则用户激活选不到 PCRF。
- **PCCTEMPLATE 名跨命令一致**（warning）：ADD PCCTEMPLATE 的 PCCTEMPNAME 须与 SET APNPCCFUNC 的 PCCTEMPLATE 一致。
- **APN 级覆盖全局**（info）：SET APNPCCFUNC 优先级高于 SET PCCFUNC；APN 级 INHERIT 才继承全局。

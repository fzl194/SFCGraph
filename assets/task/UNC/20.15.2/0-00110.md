# 查询PDP上下文数（DSP SMPDPNUM）

> 命令静态知识见 [DSP SMPDPNUM](command/UNC/20.15.2/DSP-SMPDPNUM.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法

按"查询范围 + 查询分类 + 过滤条件"组合统计。QRY_CLASS 省略表示查汇总；指定分类（如 APN/RAT/FWA 等）时按维度细分。

适用 NF：SGW-C、PGW-C、SMF、GGSN。本命令是容量/在线用户数统计入口——验证当前在线 PDP 上下文数与规划一致，常用于 FWA 接入、离线计费、Gy 在线计费场景的容量核对。

### 查询范围（QRY_SCOPE）

| 维度 | 参数 | 取值 | 默认 | 作用 | 适用场景 |
|------|------|------|------|------|----------|
| 查询范围 | QRY_SCOPE | `SUMMARY`（汇总信息） | 默认 SUMMARY | 查询汇总信息，以汇总方式呈现 | 最常用，容量总览 |
| | | `ALL_POD_INFO`（所有POD信息） | | 查询所有 POD 信息，以 pod 粒度呈现 | 按 POD 明细统计 |
| | | `SPECIFIED_POD_INFO`（指定POD信息） | | 查询指定 POD 信息，需配合 POD_ID | 指定 POD 核对 |

### 查询分类（QRY_CLASS）

| 维度 | 参数 | 取值 | 默认 | 作用 | 适用场景 |
|------|------|------|------|------|----------|
| 查询分类 | QRY_CLASS | `SUMMARY`（汇总信息） | 默认 SUMMARY | 查询所有类型的汇总会话数信息 | 总容量核对 |
| | | `FWA`（FWA） | | 查询当前在线的 FWA 用户的 PDP 上下文数 | FWA 接入调测 |
| | | `APN`（APN） | | 查询使用该 APN 激活的当前在线 PDP 上下文数，需配合 APN | APN 维度统计 |
| | | `RAT`（无线接入类型） | | 查询不同无线接入类型的 PDP 上下文数目，需配合 RAT | 无线接入类型统计 |
| | | `UPF`（UPF） | | 查询使用该 UPF 激活的当前在线 PDP 上下文数，需配合 UPF_NAME | UPF 维度统计 |
| | | `SNSSAI`（SNSSAI） | | 查询使用该切片的 QFI 上下文数，需配合 SST/SD | 切片维度统计 |
| | | `ALIASAPN`（别名APN） | | 查询使用该别名 APN 激活的当前在线 PDP 上下文数，需配合 ALIASAPN | 别名 APN 统计 |
| | | `CHG_OFFLINE`（离线计费） | | 查询离线计费的 PDP 上下文数 | 离线计费场景 |
| | | `CHG_ONLINE`（在线计费） | | 查询在线计费的 PDP 上下文数 | 在线计费场景 |
| | | `CHG_CONVERGED`（融合计费） | | 查询融合计费的会话上下文数 | 融合计费场景 |
| | | `CHARGING`（计费方式） | | 指定 APN 下按照计费方式查看当前在线上下文数，需配合 APN | 按计费方式细分 |
| | | `IP_ALLOCATION`（IP分配方式） | | 指定 APN 下按照 IP 地址分配方式查看当前在线上下文数，需配合 APN | 按 IP 分配方式细分 |
| | | `NSA`（NSA） | | 非独立组网 | NSA 场景统计 |
| | | `IMS`（IMS） | | 查询当前在线的 IMS 会话的 PDP 上下文数 | IMS 会话统计 |
| | | `L2TP`（L2TP） | | 查询当前在线的 L2TP 用户的 PDP 上下文数 | L2TP 场景统计 |

### 过滤参数（条件必选，依赖 QRY_CLASS）

| 维度 | 参数 | 取值 | 默认 | 作用 | 适用场景 |
|------|------|------|------|------|----------|
| APN | APN | 字符串（1~63） | 无 | 指定 APN 名称 | QRY_CLASS=APN/CHARGING/IP_ALLOCATION 时必选 |
| 无线接入类型 | RAT | `UTRAN`/`GERAN`/`EUTRAN`/`NGRAN`/`EUTRAN_NB_IOT`/`WLAN`/`LTE_M`/`REDCAP` | 无 | 指定无线接入类型 | QRY_CLASS=RAT 时必选 |
| UPF 名称 | UPF_NAME | 字符串（1~255） | 无 | 指定 UPF 名称 | QRY_CLASS=UPF 时必选 |
| 切片/服务类型 | SST | 整数 0~255 | 无 | 指定切片/服务类型 | QRY_CLASS=SNSSAI 时必选 |
| 切片区分码 | SD | 字符串（1~8，十六进制） | 无 | 指定切片区分码 | QRY_CLASS=SNSSAI 时可选 |
| POD 名称 | POD_ID | 字符串（1~255） | 无 | 指定 POD 名称 | QRY_SCOPE=SPECIFIED_POD_INFO 时必选 |
| 别名 APN | ALIASAPN | 字符串（1~63） | 无 | 指定 APN 别名 | QRY_CLASS=ALIASAPN 时必选 |

### 典型脚本

**FWA 场景核对当前在线 FWA 用户 PDP 上下文数：**
```
DSP SMPDPNUM: QRY_SCOPE=SUMMARY, QRY_CLASS=FWA;
```

**汇总查询（最常用）：**
```
DSP SMPDPNUM: QRY_SCOPE=SUMMARY, QRY_CLASS=SUMMARY;
```

**按 APN 统计：**
```
DSP SMPDPNUM: QRY_SCOPE=SUMMARY, QRY_CLASS=APN, APN=apn-test;
```

**按切片统计：**
```
DSP SMPDPNUM: QRY_SCOPE=SUMMARY, QRY_CLASS=SNSSAI, SST=1, SD=010203;
```

### 调测语义

- **FWA 接入调测**：核对当前在线 FWA 用户的 PDP 上下文数与实际规划一致。
- **离线计费（CG/抄送 CG 组/CG 绑定）与 Gy Diameter 在线计费调测**：作为容量与在线用户数旁路核对命令，配合业务验证使用。

### 步骤位置

本命令为查询类命令，无需配置步骤。直接在网元上执行查询即可。

## 约束

- **QRY_CLASS 参数不输入时表示查询汇总信息（与**（info，rule-0-01581）：QRY_CLASS 参数不输入时表示查询汇总信息（与 SUMMARY 等效为汇总呈现）

- **"指定查询条件"指的是指定查询范围（ALL_POD_INFO**（info，rule-0-01582）："指定查询条件"指的是指定查询范围（ALL_POD_INFO、SPECIFIED_POD_INFO）。例如"指定查询条件的 PGW-C 上激活的 5G 承载上下文数"实际指的是该查询范围内的统计

- **APN/UPF_NAME/SST/SD/RAT/ALIASAPN**（info，rule-0-01583）：APN/UPF_NAME/SST/SD/RAT/ALIASAPN 等过滤参数仅在对应 QRY_CLASS 下生效，按数据规划取值

- **适用 NF 为 SGW-C**（info，rule-0-01584）：适用 NF 为 SGW-C、PGW-C、SMF、GGSN，不同网元形态支持的上下文类型不同（GGSN-C 支持 GTPv1 PDP 上下文，S/PGW-C 支持 GTPv2 承载上下文）

- **查询结果中的"承载上下文总数"等于 PGW 上激活数 +**（info，rule-0-01585）：查询结果中的"承载上下文总数"等于 PGW 上激活数 + SGW 上激活数 + S/PGW 上激活数之和；"QFI 上下文总数"等于各 SMF 上激活数之和

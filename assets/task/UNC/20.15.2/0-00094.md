# 新增用户模板（ADD USERPROFILE）

> 命令静态知识见 [ADD USERPROFILE](command/UNC/20.15.2/ADD-USERPROFILE.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法

User Profile 是策略/计费绑定的载体。配置后由后续命令往这个模板上挂 RULE（[ADD RULEBINDING](task/UNC/20.15.2/0-00073.md)）/ URR 组（[SET URRGRPBINDING](task/UNC/20.15.2/0-00206.md)）/ 计费属性（[SET USRPROFCHARGE](task/UNC/20.15.2/0-00207.md)）/ DNAI（[ADD USRPROBINDDNAI](task/UNC/20.15.2/0-00095.md)）。

### 按 UPTYPE 维度

| 维度 | 参数 | 取值 | 默认 | 作用 | 适用场景 |
|---|---|---|---|---|---|
| 业务类别 | `USERPROFILENAME` | 字符串 1~63 | 无 | 全网规划名称，PCF/SMF/UPF 间必须一致 | 所有场景 |
| 业务类别 | `UPTYPE` | `PCC` / `ULCL` | 无（必选） | 决定业务类别 | PCC：计费/QoS/ADC/Radius；ULCL：分流策略 |
| 下发范围 | `PROFILERANGE` | `ALL` / `CENTRAL` / `LOCAL` | `ALL` | 决定 UserProfile 下发到哪些 UPF | LOCAL 时需配 ADD USRPROBINDDNAI |
| 规格 | `UPSPECTYPE` | `DEFAULT` / `SPECIFICATION_LIMITED` | `DEFAULT` | DEFAULT 规格 5000，SPECIFICATION_LIMITED 规格 100000 | 高容量场景 |
| 免费计费 | `FREESERONLCHG` / `FREESEROFFCHG` / `FREESERCVGCHG` | `ENABLE` / `DISABLE` | `ENABLE` | 控制免费业务是否产出在线/离线/融合计费信息 | 话单/配额上报观察免费业务计费信息时使能 |

### 典型脚本

```
ADD USERPROFILE:USERPROFILENAME="up_test",UPTYPE=PCC,PROFILERANGE=ALL;
ADD USERPROFILE:USERPROFILENAME="testuserprofilename",UPTYPE=ULCL;
ADD USERPROFILE:USERPROFILENAME="up_test",UPTYPE=PCC,FREESERONLCHG=ENABLE;
```

### 步骤位置

UserProfile 三段式组装的第 1 段（建模板）。完整链路：ADD USERPROFILE（本命令）→ [ADD USRPROFGROUP](task/UNC/20.15.2/0-00096.md) → [ADD UPBINDUPG](task/UNC/20.15.2/0-00089.md) → [ADD APNUSRPROFG](task/UNC/20.15.2/0-00008.md)。

## 决策点

### 决策点：UPTYPE 业务类别选择（DP 0-01226）

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| `PCC` | 计费/QoS/ADC/Radius 等通用策略承载 | 绝大多数场景（WSFD-109101/109102/109107/211005/211009 等） |
| `ULCL` | ULCL 分流策略承载 | WSFD-108002/223001/223003/223101 分流场景 |

### 决策点：PROFILERANGE 下发范围选择（DP 0-01227）

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| `ALL`（默认） | 主+辅锚点 UPF 都下发 | 默认场景，SMF 同时下发中心和所有边缘 UPF |
| `CENTRAL` | 仅主锚点 UPF | 核心网策略，不涉及边缘计算 |
| `LOCAL` | 需额外配置 [ADD USRPROBINDDNAI](task/UNC/20.15.2/0-00095.md) 指定边缘 UPF | 边缘计算/MEC 辅锚点场景 |

> 说明：`PROFILERANGE=CENTRAL` 时该模板不能绑定任何边缘 UPF 的 DNAI；`LOCAL` 时才需要 [ADD USRPROBINDDNAI](task/UNC/20.15.2/0-00095.md) 绑定 DNAI。

## 约束

- **USERPROFILENAME 全网唯一**（critical，rule-0-01501）：`USERPROFILENAME` 全网唯一、跨 PCF/SMF/UPF 名称一致；不一致时 PCF 下发策略失效

- **UPTYPE 必选**（warning，rule-0-01502）：`UPTYPE` 必选，固定取值（PCC/ULCL）；选错导致业务策略承载错误

- **PROFILERANGE=CENTRAL 时不能绑定任何边缘**（critical，rule-0-01503）：`PROFILERANGE=CENTRAL` 时不能绑定任何边缘 UPF 的 DNAI（绑定后 SMF 不下发到边缘，策略失效）；`PROFILERANGE=ALL` 时绑 DNAI 无实际意义

- **记录数上限**（info，rule-0-01504）：记录数上限：`UPSPECTYPE=DEFAULT` 时 5000；`SPECIFICATION_LIMITED` 时 100000；超 98% 触发 ALM-135602215 告警

- **执行后立即生效**（info，rule-0-01505）：执行后立即生效

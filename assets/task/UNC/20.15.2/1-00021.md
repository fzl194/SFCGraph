## 约束

- **本骨架不引入独立命令**（critical，`§3.7`）：本骨架 `cmd: 无`，PCC/CHF 配置归各被依赖特性 wiki。
- **PCC 与 CHF 链路必须先建立**（critical）：会话建立前 N7（→ PCF）+ N40（→ CHF）链路必须与外部网元建立 TCP/SBI 偶联 — 链路建立归 SBI 本端实体（已建 atom 0-00074/37/38 + HTTPCONF 0-00179）。
- **PCRF/CHF 故障保活**（critical）：PCRF/CHF 全故障保活依赖各计费域特性配置（[2-00001]/[2-00003] 含故障处理），否则链路异常 → 会话被强制释放。
- **PCC 规则与计费属性协同**（warning）：5G 融合计费场景必须 PCC 规则（QER/URR）与 CHF 计费属性（Quota Trigger）协同，否则 URR 与 CHF 状态不一致 → 计费异常。
- **背靠背会话的 PCF/CHF ID 全网唯一**（warning）：PCF/CHF 实例的 NF ID（FQDN/IP）必须全网唯一，避免 PCC/CHF 路由错乱。

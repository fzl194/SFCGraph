# ADD CHGAPN 原子任务

> 命令静态知识见 [ADD CHGAPN](command/UNC/20.15.2/ADD-CHGAPN.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

ADD CHGAPN 仅适用于 SGSN，在要求忽略 HLR 签约的 APN 计费属性时（由 `SET CHGCDR` 设 IGNOREFLG 触发），为指定 APN 配置本端计费属性，用于 S-CDR 中计费属性的选择。

**参数取值样例**

| 参数 | 取值样例 | 获取方法 | 说明 |
|---|---|---|---|
| APNNI | APN 网络标识（1~62 字符） | 整网规划 | 唯一键，各记录不可重复 |
| CC | 计费属性（默认 `NORMAL` 普通计费） | 整网规划 | 可选，缺省按普通计费 |

**配置链路（本命令位置）**

```
SET CHGCDR (设 IGNOREFLG)   # 开启忽略 HLR 签约 APN 计费属性
ADD CHGAPN   ← 本命令       # 配置本端 APN 计费属性（可选步骤，按需）
```

## 约束

- **立即生效但仅对之后激活的用户有效** · severity=MEDIUM。
- **最大 1000 条记录** · severity=LOW。
- **APNNI 唯一性** · severity=HIGH · 各记录的 APNNI 字段不能重复。
- **生效前置** · severity=MEDIUM · 本命令生效前提是 `SET CHGCDR` 已设 IGNOREFLG 忽略 HLR 签约 APN 计费属性，否则本端 APN 计费属性不参与选择。
- **S-CDR 计费属性选择优先级** · severity=MEDIUM · 证据 notes 指出 S-CDR 计费属性按优先等级选择（如按 IMSI 匹配等，详见 `LST CHGGNCCCFG`），本命令仅为其中一条来源。

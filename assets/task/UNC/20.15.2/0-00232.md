# 增加IKE对等体（ADD IKEPEER）

> 命令静态知识见 [ADD IKEPEER](command/UNC/20.15.2/ADD-IKEPEER.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

该命令增加一个 IPv4 IKE 对等体，对端协商参数由本命令承载。配置空间主要由三个维度决定：**IKE 版本**（协商协议版本）、**认证方式**（与对端相互验证身份的方式）、**交换模式**（IKEv1 协商阶段 1 的模式）。

### 配置维度1：IKE 版本（参数 VERSION1 / VERSION2）

| 取值 | 作用 | 配套必选参数 | 典型场景 |
|---|---|---|---|
| VERSION1=TRUE, VERSION2=FALSE | 仅 IKEv1 | — | 国密 IPsec 支持 IKEv1 场景；与旧对端兼容 |
| VERSION1=FALSE, VERSION2=TRUE | 仅 IKEv2（推荐） | 可配 IKEMSGSYNC/IPSECMSGSYNC（仅 v2 生效） | 推荐场景；IKEv2 更安全 |
| VERSION1=TRUE, VERSION2=TRUE | 双版本支持 | — | 过渡或对端版本未知 |

### 配置维度2：认证方式（通过 PROPOSAL 引用 IKE 提议决定 + 证书参数）

| 取值组合 | 作用 | 配套必选参数 | 典型场景 |
|---|---|---|---|
| 预共享密钥 | 用预共享密钥认证对端 | PRESHAREDKEY；PROPOSAL 引用的 IKEPROPOSAL 中 AUTHMETHOD=Pre_share | 常规 IPSec 隧道（IPv4） |
| 数字信封（国密） | 用国密数字信封认证对端 | PROPOSAL 引用的 IKEPROPOSAL 中 AUTHMETHOD=Digital_envelope；CERTLOCALFILE + ENCCERTLOCFILE（国密签名/加密证书文件名） | 激活国密 IPsec 支持 IKEv1 功能 |
| 证书（非国密） | 用证书认证对端 | CERTLOCALFILE；可选 CERTSCENARIO/ENCCERTSCENARIO 绑定场景 | 证书认证场景 |

### 配置维度3：交换模式（参数 EXCHANGEMODE，仅 IKEv1 有意义）

| 取值 | 作用 | 典型场景 |
|---|---|---|
| Main（主模式，默认） | IKEv1 主模式协商，身份保护 | 推荐；国密 IKEv1 仅支持主模式 |
| Aggressive（野蛮模式） | IKEv1 野蛮模式协商 | 对端 IP 动态获取场景 |

### 配置维度4：本地 ID 类型（参数 LOCALIDTYPE）

| 取值 | 作用 | 配套参数 |
|---|---|---|
| Ip（IP，默认） | 用 IP 地址作本地 ID | — |
| Fqdn（FQDN） | 用全限定域名作本地 ID | 必配 REMOTEID |
| User_fqdn（User-FQDN） | 用用户型 FQDN 作本地 ID | 必配 REMOTEID |
| Dn（DN） | 用证书 DN 作本地 ID | — |
| Ip_disable（IP-disable） | 关闭本地 IP 校验 | 仅支持 IKEv1 |

**步骤位置**：在所属 feature（IPFD-016000 IPSec功能）编排中，于 `ADD IKEPROPOSAL`（建立 IKE 安全提议，PROPOSAL 字段引用其 PROPOSALNUMBER）之后、`ADD IPSECPOLICY`（建立 IPsec 策略并引用本对等体）之前执行。

## 决策点

**IKE 版本与认证方式选择**（UNC pre-build 内嵌，不编号——按 `assets/CLAUDE.md` §5.7）：

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| IKEv1 + 数字信封（国密） | VERSION1=TRUE, VERSION2=FALSE；CERTLOCALFILE + ENCCERTLOCFILE；PROPOSAL 引用 AUTHMETHOD=Digital_envelope 的 IKEPROPOSAL | 激活国密 IPsec 支持 IKEv1（普通 IPv4 IPsec 隧道）；不支持 NAT 穿越、主备隧道 |
| IKEv1 + 预共享密钥 | VERSION1=TRUE；PRESHAREDKEY | 与旧对端兼容 |
| IKEv2 + 预共享密钥 | VERSION2=TRUE；PRESHAREDKEY | 推荐场景 |

## 约束

- **立即生效**（info）：命令执行后立即生效 — 后果：无延迟
- **IKEv1 不安全**（warning）：IKEv1 是不安全的协议，建议使用 IKEv2 — 后果：安全性弱
- **国密 IKEv1 限制**（critical）：IKEv1 国密 IPSEC 仅支持主模式；不支持 NAT 穿越；不支持主备隧道；仅支持数字信封认证 — 后果：配置不符限制会协商失败
- **IP-disable 仅 IKEv1**（warning）：LOCALIDTYPE=Ip_disable 仅支持 IKEv1 — 后果：IKEv2 下配置无效
- **证书字段与场景字段互斥**（warning）：证书字段（CERTLOCALFILE 等）和场景字段（CERTSCENARIO 等）不能同时配置 — 后果：配置失败
- **策略模板模式不带对端地址**（warning）：用于策略模板模式时不能配置对端地址（LOWREMOTEADDR） — 后果：配置失败
- **FQDN/User-FQDN 必配 REMOTEID**（warning）：LOCALIDTYPE=Fqdn 或 User_fqdn 时，REMOTEID 必须配置 — 后果：协商失败
- **最多 4000 条记录**（warning）：本命令最多可输入 4000 条记录 — 后果：超规格新增失败
- **IKEMSGSYNC/IPSECMSGSYNC 仅 v2 生效**（info）：参数 IKEMSGSYNC、IPSECMSGSYNC 只有在 VERSION2=TRUE 时才生效 — 后果：v1 下配置无效

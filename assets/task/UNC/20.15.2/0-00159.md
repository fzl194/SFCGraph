# SET CHGCDR 原子任务
> 命令静态知识见 [SET CHGCDR](command/UNC/20.15.2/SET-CHGCDR.md)。本页只讲配置生成实例化时这条命令**怎么配**。

SGSN 侧全局 CDR 配置命令，影响 S-CDR/M-CDR/S-SMO/S-SMT/LCS 各类话单的字段产生、缺省计费属性、PLMN ID 来源、强制话单生成等。计费 CDR 参数由系统缺省增加（初次运行有初始设置值）。话单字段优先级：CC 级（[SET CHGCHAR](task/UNC/20.15.2/0-00160.md)）> 全局级（本命令）> HLR 签约。

## 配置方法
按运营场景选择性配置子集；参数众多，常用三类：话单内容定制、PLMN 控制策略、强制生成话单。

| 参数 | 配置方法 | 取值样例 |
|---|---|---|
| ML / SMOL / SMTL / SL | 各类 CDR 选项（如填 IMEI） | `SERVED_IMEI` |
| LCSMTL / LCSMOL / LCSNIL | LCS 各类 CDR 选项（填 MSISDN/SGSN_ADDR） | `SERVED_MSISDN` / `SGSN_ADDR` |
| NODEID | SGSN 节点标识（≤19 字符串） | `sgsn01` |
| HCC / VCC / RCC | 本地/拜访/漫游用户缺省计费属性 | `NORMAL`（普通计费）等 |
| VP | 缺省非本地用户类型 | `ROAMING`（漫游用户）等 |
| LRSNP | 配置本地话单序列号 | `PRESENT`（有）/ `ABSENT`（无） |
| IGNOREFLG | 忽略签约计费属性 | `HOME_SUBSCRIBER` 等 |
| CRC | 话单关闭原因值 | `NORMAL`（正常释放）等 |
| PLMNCTRL | PLMN ID 控制策略（话单 PLMN 来源） | `LOCINFO`（位置信息 PLMN） / `IMSI` |
| EXTRANGE | PLMNCG 范围扩展 | `YES`（允许非 PLMN CG） |
| FRCGEN | 强制生成话单方式 | `SCDRONLY`（只生成 S-CDR） |
| FRCTIME | 强制生成话单时间（FRCGEN 配套） | `00&00&00`~`23&59&59` |
| CRQOS | 保持请求 QoS | `YES`（保持原始值） |
| CDRPLMNSELECT | 话单填写 PLMN ID 选择策略 | `LOCINFO` |
| CGPLMNSELECT | 话单发送 PLMN ID 选择策略 | `LOCINFO` |
| NONSUPPUEPLMN | MOCN 中 Non-supporting UE 的 PLMN 获取策略 | `SRV_PLMN` / `COMMON_PLMN` |

典型脚本（MOCN 网络共享场景，Non-supporting UE 用 Serving PLMN）：
```
SET CHGCDR: NONSUPPUEPLMN=SRV_PLMN;
```
话单可靠性（生成本地话单序列号）：
```
SET CHGCDR: LRSNP=PRESENT;
```

## 决策点

### PLMN ID 控制策略（PLMNCTRL）
- 问什么：话单/CG 发送时 PLMN ID 从哪里取，决定 S-CDR 话单生成策略的匹配维度。
- `LOCINFO`（位置信息中的 PLMN）→ 话单策略基于 PLMN + APN NI 共同配置。
- `IMSI`（IMSI 中的 PLMN）→ 话单策略基于 IMSI 前缀 + APN NI + 拜访类型 + 计费属性共同配置，可精细抑制特定号段漫游用户。
- 与 `SET CHGPLMNCHAR`（全局配置策略）配合：仅当全局策略设为生成 S-CDR 时，上述策略才生效。

### MOCN Non-supporting UE PLMN 获取（NONSUPPUEPLMN）
- 问什么：网络共享（MOCN）下，不能解析共享信息的 Non-supporting UE，话单/Gn 用哪个 PLMN。
- `SRV_PLMN`（Serving PLMN）→ 配合 `ADD HNOSRVPLMN` 指定 Serving PLMN，替换 Common PLMN 填入话单，解决 Billing Center 无法识别话单问题。
- `COMMON_PLMN`（公共 PLMN）→ 直接用广播的 Common PLMN。
- 视 Billing Center 是否归属本网决定。

### 强制生成话单（FRCGEN / FRCTIME）
- 问什么：是否在指定时间点强制生成所有用户话单（如日终结算）。
- `SCDRONLY`（只生成 S-CDR）→ 配 FRCTIME 指定时间。
- 不强制 → 不配。

## 约束
- 前置/初始：系统初次运行即有初始设置值；CDR 参数由系统缺省增加。`severity=low`
- 生效时机：执行后立即生效。`severity=low`
- PLMN 策略生效前置：PLMNCTRL 相关话单生成策略仅在 `SET CHGPLMNCHAR`（[0-00163](task/UNC/20.15.2/0-00163.md)，待建）全局策略设为生成 S-CDR 时生效。`severity=medium`
- 计费属性优先级：S-CDR 计费属性选择优先级为 IMSI 匹配的 SPEC 配置 > 本命令（全局缺省）> HLR 签约；CC 级 `SET CHGCHAR` 配置优先级高于本命令。`severity=medium`
- 话单序列号固定值：话单可靠性场景 LRSNP 固定选 `PRESENT`。`severity=low`

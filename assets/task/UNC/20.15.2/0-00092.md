# ADD UPFGLOCGBNDGRP 原子任务
> 命令静态知识见 [ADD UPFGLOCGBNDGRP](command/UNC/20.15.2/ADD-UPFGLOCGBNDGRP.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法
绑定关系组是一个容器对象，建组只用 `UPFGLOCGBNDGNAME`（本端规划命名），实际绑定关系由 [ADD UPFGBINDLOCG](task/UNC/20.15.2/0-00091.md) 逐条登记到这个组里。

| 场景 | 必备参数 | 典型脚本 |
|---|---|---|
| 建立 Gx 本端主机选择容器（多 PGW-U 用户 IP 重叠场景） | `UPFGLOCGBNDGNAME` | `ADD UPFGLOCGBNDGRP: UPFGLOCGBNDGNAME="test";` |

## 约束
- `UPFGLOCGBNDGNAME` 规划多个时该参数不能重复。`severity=critical` 后果：组名重复导致后续绑定登记错组。
- 字符串 1~63，不支持空格，不区分大小写。
- 最大记录数 3000。`severity=medium`
- 执行后立即生效。`severity=low`

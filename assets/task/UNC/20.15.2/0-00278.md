# 设置IKE全局配置（SET IKEGLOBALCONFIG）

> 命令静态知识见 [SET IKEGLOBALCONFIG](command/UNC/20.15.2/SET-IKEGLOBALCONFIG.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

该命令设置 IKE 全局配置，作为 IPsec 策略的缺省值来源（如 `ADD IPSECPOLICY` 的 SALIFETIMESEC=0 时取本命令 TIMESADURTN）。配置空间按**功能族**组织。

### 配置维度1：SA 生命周期（参数 TRAFFICSADURTN / TIMESADURTN）

| 参数 | 取值 | 作用 |
|---|---|---|
| TIMESADURTN | 480~604800（秒，初始 3600） | 全局 SA 按时计长缺省值（IPSECPOLICY 的 SALIFETIMESEC=0 取此值） |
| TRAFFSADISFLG | TRUE/FALSE | 流量方式触发 SA 过期标记 |
| TRAFFICSADURTN | 8000~200000000（千字节，初始 1843200） | 流量方式触发 SA 过期的流量数（TRAFFSADISFLG=FALSE 时可选） |

### 配置维度2：抗重放（参数 ANTIREPLFLG / WINDOWSIZE）

| 参数 | 取值 | 作用 |
|---|---|---|
| ANTIREPLFLG | True/False | 抗重放开关（初始 True） |
| WINDOWSIZE | Size_32~Size_1024（初始 Size_1024） | 抗重放窗口大小（ANTIREPLFLG=True 时可选） |

### 配置维度3：DPD（参数 DPDTYPE / DPDINTERVAL / DPDRETRYINTRVL）

| DPDTYPE | 作用 | 配套参数 |
|---|---|---|
| None（初始） | 不做 DPD | — |
| Periodic | 周期性 DPD 检测 | DPDINTERVAL（必填，10~3600）+ DPDRETRYINTRVL（可选，2~60） |
| Ondemand | 按需 DPD 检测 | DPDINTERVAL + DPDRETRYINTRVL |

### 配置维度4：分片与 NAT 保活（参数 DFBITCLEAR / FRAGBEFOREENCR / NATKLI）

| 参数 | 取值 | 作用 |
|---|---|---|
| DFBITCLEAR | TRUE/FALSE（初始 FALSE） | 清除分片标记位 |
| FRAGBEFOREENCR | TRUE/FALSE（初始 FALSE） | 加密前分片 |
| NATKLI | 5~300（秒，初始 20） | NAT 保活报文发送间隔（NAT 穿越时有效；用户级模板模式不生效） |

### 配置维度5：CPU 阈值（参数 CPUREPORTTHRES / CPUCLEARTHRES / FLOWCTRLSTHRES / FLOWCTRLRTHRES）

| 参数 | 取值 | 作用 |
|---|---|---|
| CPUREPORTTHRES | 50~100（%，初始 80） | IPSEC 转发 CPU 过载告警上报阈值 |
| CPUCLEARTHRES | 50~100（%，初始 70） | IPSEC 转发 CPU 过载告警恢复阈值（需 ≤ CPUREPORTTHRES，建议差至少 10） |
| FLOWCTRLSTHRES / FLOWCTRLRTHRES | 50~100 | 开启/恢复流控的 CPU 阈值 |

**步骤位置**：在所属 feature（IPFD-016000 IPSec功能）编排中，是 IPSec 基础配置——通常在建立具体策略（`ADD IPSECPOLICY`）之前执行一次，作为全局缺省值。

## 决策点

（无。本命令为全局参数设置，无特性级分支。）

## 约束

- **立即生效**（info）：命令执行后立即生效 — 后果：无延迟
- **CPU 阈值关系**（warning）：需 CPUREPORTTHRES ≥ CPUCLEARTHRES，且建议上报阈值大至少 10 个百分点 — 后果：告警可能持续上报
- **NATKLI 用户级模板模式不生效**（warning）：在用户级模板模式场景下 NATKLI 参数不生效 — 后果：该场景 NAT 保活参数无效
- **安全日志阈值非 0 可能日志风暴**（warning）：如果 DOSTHRESHOLD 不为 0，链路协商失败时可能导致日志大量打印，影响问题定位 — 后果：日志风暴
- **系统初始记录存在**（info）：系统部署完成后已存在初始记录（TIMESADURTN=3600 等） — 后果：无需重复配置

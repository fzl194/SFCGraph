# 配置用户位置与模板绑定

> 位置策略控制链：建位置（代际分叉）→位置组→绑用户模板组。被引用于 [2-00013 基于初始接入位置的策略控制](task/UNC/20.15.2/2-00013.md)。访问限制域专属。

## 配置方法

| 步骤 | 命令 | 关键参数 |
|---|---|---|
| 建用户位置（代际分叉） | `ADD USRLOCATION` | LOCATIONNAME；LOCATIONTYPE=CGI/ECGI/NCGI；MCC/MNC；位置码（LAC+CI / ECI / NCI）|
| 建位置组（逐位置一条） | `ADD USRLOCATIONGRP` | LOCGROUPNAME + LOCATIONNAME |
| 绑位置组到用户模板组 | `MOD UPBINDUPG`（已绑）/ `ADD UPBINDUPG`（未绑）| USERPROFGNAME + UPBINDTYPE=SPECIFIC + USERPROFILENAME + LOCGROUPNAME |

**典型脚本**：
```
ADD USRLOCATION:LOCATIONNAME="loc_test1",LOCATIONTYPE=CGI,MCC="333",MNC="111",LAC="0xFFFD",CI="0xFFFD";
ADD USRLOCATION:LOCATIONNAME="loc_test2",LOCATIONTYPE=ECGI,MCC="333",MNC="111",ECI="0xffffffd";
ADD USRLOCATIONGRP:LOCGROUPNAME="locgrp_test",LOCATIONNAME="loc_test1";
ADD USRLOCATIONGRP:LOCGROUPNAME="locgrp_test",LOCATIONNAME="loc_test2";
MOD UPBINDUPG:USERPROFGNAME="upg_test1",UPBINDTYPE=SPECIFIC,USERPROFILENAME="up_test1",LOCGROUPNAME="locgrp_test";
```

**步骤位置**：License 前置之后；下游绑 APN（[0-00008](task/UNC/20.15.2/0-00008.md)）。

## 决策点

| 选项 | 影响 |
|---|---|
| LOCATIONTYPE=CGI/ECGI/NCGI | ADD USRLOCATION 位置码不同（2/3G LAC+CI / 4G ECI / 5G NCI）|
| 用户模板已绑/未绑模板组 | MOD UPBINDUPG（追加位置组）vs ADD UPBINDUPG（一次绑模板+位置组）|

## 约束

- **位置名不重复**（warning）：LOCATIONNAME 跨记录唯一。
- **位置名须先建**（critical）：ADD USRLOCATIONGRP 引用 ADD USRLOCATION 的 LOCATIONNAME。
- **多位置逐条**（info）：ADD USRLOCATIONGRP 每位置一条。

# ADD SBIAPLE 原子任务
> 命令静态知识见 [ADD SBIAPLE](command/UNC/20.15.2/ADD-SBIAPLE.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法
SBIAPLE 是本端 NF 在服务化接口上的"接入点身份证"：声明本端是谁（NFTYPE）、用哪组 HTTP 本端实体（HTTPLEGRPIDX）收发。当本端 NF 要和目的 NF 建 SBI 连接时必须配置。每条命令对应一个 (INDEX, NFTYPE) 实体。

| 业务场景 | NFTYPE | HTTPLEGRPIDX | 对端 NF 参数 | 典型脚本 |
|---|---|---|---|---|
| SBI 接口加密（AMF 本端） | NFTypeAMF | 引用已配 HTTPLEGRP（如 1） | 不配 | `ADD SBIAPLE:INDEX=1,HTTPLEGRPIDX=1,NFTYPE=NFTypeAMF,DESCRIPTION="AMF";` |
| SBI 接口加密（NRF 本端） | NFTypeNRF | 引用已配 HTTPLEGRP（如 2） | 不配 | `ADD SBIAPLE:INDEX=2,HTTPLEGRPIDX=2,NFTYPE=NFTypeNRF,DESCRIPTION="NRF";` |
| QoS 能力开放（BSF 本端） | NFTypeBSF | 引用已配 HTTPLEGRP（如 3） | 不配 | `ADD SBIAPLE:INDEX=1,HTTPLEGRPIDX=3,NFTYPE=NFTypeBSF,DESCRIPTION="BSF";` |
| 需对接 CHF（融合计费场景） | 本端 NF 类型 | 引用已配 HTTPLEGRP | PEERNFTYPE/CHFPEERNFTYPE/PEERSERVICE | 见决策点 |

**前置**：必须先用 ADD HTTPLE（或其未建编号）和 ADD HTTPLEGRP 建 HTTP 本端实体组，且该组内必须含一个 Server 端 + 一个 Client 端 HTTP 本端实体。HTTPLEGRPIDX 引用该组 INDEX。

## 决策点

### NFTYPE 取值
问：本端 NF 是什么类型？
- 按本端网元角色定：AMF→NFTypeAMF，NRF→NFTypeNRF，BSF→NFTypeBSF，SMF→NFTypeSMF 等。
- 证据中 SBI 加密场景配 AMF 和 NRF 两条；QoS 能力开放场景配 BSF 一条。一个网元上可配多个 SBIAPLE（每种 NF 角色一条）。

### 是否配置对端 NF 信息
问：本端 NF 是否要在此声明目的 NF（CHF 等）？
- **不配对端 NF 信息**（多数场景）：只能配置一个服务化接口实体，纯本端接入点声明。证据中 SBI 加密、QoS 能力开放均如此。
- **配对端 NF 信息**（融合计费/CHF 对接）：需填 PEERNFTYPE（仅支持 Nchf）、CHFPEERNFTYPE（仅支持 Nnrf/Nocs）、PEERSERVICE（仅支持 Nchf_ConvCharg 服务）、CHFPEERNFINSTID。用于本端 NF 直接定位 CHF。

### INDEX 与 HTTPLEGRPIDX 取值
问：两个 INDEX 取已配数据还是本端规划？
- **INDEX（本端实体索引）**：本端规划，1~128，本端唯一即可。
- **HTTPLEGRPIDX（HTTP 本端实体组索引）**：引用已配 HTTPLEGRP 的 INDEX（1~64）。证据中多与 INDEX 同值（如 AMF 均 1、NRF 均 2），但语义不同——INDEX 是 SBIAPLE 自己的编号，HTTPLEGRPIDX 指向 HTTP 实体组。

## 约束
- INDEX 取值 1~128，本端唯一。`severity=medium`
- HTTPLEGRPIDX 取值 1~64，必须指向已配 HTTPLEGRP，且该组内必须存在一个 Server 端 + 一个 Client 端 HTTP 本端实体。`severity=critical`
- NFTYPE 不能为 INVALID。`severity=high`
- 不配对端 NF 信息时，只能配一个服务化接口实体。`severity=medium`
- 配对端 NF 时：PEERNFTYPE 仅支持 Nchf；CHFPEERNFTYPE 仅支持 Nnrf/Nocs；PEERSERVICE 仅支持 Nchf_ConvCharg。`severity=high`
- DESCRIPTION 长度 0~255。`severity=low`
- CHFPEERNFINSTID 长度 0~255，配 CHF 对端时必填。`severity=medium`
- 执行后立即生效。`severity=low`

# OFCTemplate 绑定计费属性 CC

> 离线计费参数绑到 Charge Characteristic 层（优先级次于 APN）。被引用于 [2-00001 离线计费](task/UNC/20.15.2/2-00001.md)。域通用（在线/融合计费模板绑 CC 同此结构）。

## 配置方法

| 步骤 | 命令 | 关键参数 |
|---|---|---|
| 建计费属性（已配可跳过） | `ADD CHARGECHAR` | CCNAME（本端规划） |
| CC 绑模板 | `ADD GLBOFCTEMPLATE` | GLOBALFLAG=CHARGE_CHARACT；CCVALUE；PGWTEMPLATE/SGWTEMPLATE=模板名 |

**典型脚本**：
```
ADD CHARGECHAR: CCNAME="testchgcha";
ADD GLBOFCTEMPLATE: GLOBALFLAG=CHARGE_CHARACT, CCVALUE=0x0003, PGWTEMPLATE="offlinecharge-test", SGWTEMPLATE="offlinecharge-test";
```

**步骤位置**：紧跟 [1-00001](task/UNC/20.15.2/1-00001.md) 之后；仅当特性级 DP 选 CC 层时编排。

## 约束

- **前置模板**（critical）：OFCTEMPLATENAME 须已在 [1-00001](task/UNC/20.15.2/1-00001.md) 建立。
- **GLOBALFLAG 区分**（warning）：CC 层用 CHARGE_CHARACT；全局层用 GLOBAL（见特性级 DP 走法）。

# 增加AMF全局标识（ADD GUAMI）

> 命令静态知识见 [ADD GUAMI](command/UNC/20.15.2/ADD-GUAMI.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法

增加 GUAMI 在"初始配置 > 配置 AMF"阶段完成。PLMN / Region / Set 三段标识同一 AMF Set（对应一个 AMF Pool），Pointer 在 Set 内区分不同 AMF 实例。同一 AMF 实例可配多个 GUAMI（多 PLMN 场景）。

| 维度 | 参数 | 取值 | 默认 | 作用 | 适用场景 |
|---|---|---|---|---|---|
| GUAMI 索引 | INDEX | 本端规划，整数 0~255 | 无 | 在 UNC 系统内唯一标识某个 GUAMI | 所有记录必填 |
| PLMN 索引 | PLMNIDX | 全网规划，整数 0~127，引用 `ADD NGSRVPLMN` 已配置 PLMN | 无 | 指定组成 GUAMI 的 PLMN 信息 | 所有记录必填 |
| AMF 区域标识 | AMFREGIONID | 全网规划，2 位十六进制（无 `0x` 前缀，0-9/A-F/a-f） | 无 | AMF 所在区域标识 | 所有记录必填 |
| AMF 集合标识 | AMFSETID | 全网规划，3 位十六进制，首字符只能 0-3 | 无 | AMF 所在集合（Pool）标识 | 所有记录必填 |
| AMF 指示符 | AMFPOINTER | 全网规划，2 位十六进制，首字符只能 0-3 | 无 | 组成 GUAMI 的 AMF 指示符信息 | 所有记录必填 |
| 备用 AMF 名称 | BACKUPAMFNAME | 全网规划，0~150 字符，字母/数字/`-`/`.`，首尾只能是数字或字母 | 无 | GUAMI 粒度指定备用 AMF（输入单空格删除已有配置） | 可选 |

典型脚本（实际值由网络规划给定）：
```
ADD GUAMI:INDEX=0,PLMNIDX=0,AMFREGIONID="2A",AMFSETID="1F4",AMFPOINTER="00";
```

步骤位置：建 PLMN（`ADD NGSRVPLMN`）→ 本任务 →（可选）建备用 AMF 信息（`ADD BACKUPGUAMI` / 对端 AMF 的 `ADD AMFINFO`）

## 决策点

### 决策点：Pool 内多 AMF 的 GUAMI 划分（DP 0-02049）

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| 同 Set 内多 AMF 实例 | 各实例 GUAMI 三段标识（PLMNIDX + AMFREGIONID + AMFSETID）相同，AMFPOINTER 不同 | AMF Pool 内多实例部署 |
| 多 PLMN 部署（同一 AMF 实例） | 同一 AMF 实例配多条 GUAMI，PLMNIDX 不同时 AMFPOINTER 可同可不同 | 一网元承载多 PLMN |
| 融合网元 MMEGI 约束 | MMEGI + MMEC ≠ AMF Region ID + AMF Set ID + AMF Pointer（且 MMEC ≠ AMF Set ID 0~1 位 + AMF Pointer） | MME 与 AMF 融合部署 |

### 决策点：BACKUPAMFNAME 是否指定（DP 0-02050）

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| 启用 GUAMI 粒度备份 | 填 BACKUPAMFNAME（须为 Pool 内合法 AMF，由对端 `ADD AMFINFO` 配置） | 1:1 备份规划，AMF 故障/升级时业务迁移 |
| 不做粒度备份 / 走 Pool 级冗余 | BACKUPAMFNAME 留空 | Pool 级冗余兜底 |

## 约束

- **同一性约束**（critical，rule-0-02121）：同一性约束：同一 AMF 实例多条 GUAMI 间，AMFREGIONID 和 AMFSETID 必须相同；当 PLMNIDX 相同时 AMFPOINTER 必须不同，PLMNIDX 不同时 AMFPOINTER 可同可不同
- **唯一性**（warning，rule-0-02122）：唯一性：同一个 AMF Set 内，各 AMF 配置的 GUAMI 不能相同，违反则 Pool 内实例标识冲突、用户路由错乱
- **互斥约束**（warning，rule-0-02123）：互斥约束：本命令配置的 GUAMI 与 `ADD BACKUPGUAMI` 配置的 GUAMI 的 PLMN 相同时，AMFREGIONID 和 AMFSETID 必须相同，但 AMFPOINTER 必须不同
- **规格上限**（warning，rule-0-02124）：规格上限：一个 AMF 最多可输入 256 条记录
- **生效时机**（info，rule-0-02125）：生效时机：命令执行后立即生效

# 增加高级IPv4 ACL规则（ADD ACLRULEADV4IPSEC）

> 命令静态知识见 [ADD ACLRULEADV4IPSEC](command/UNC/20.15.2/ADD-ACLRULEADV4IPSEC.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

该命令向一个已存在的 IPv4 ACL 规则组挂载一条高级 ACL 规则，用于基于五元组匹配报文（决定哪些流量进入 IPSec 保护）。配置空间主要由两个维度决定：**规则行为**（Permit/Deny）和**协议类型**（决定后续端口/ICMP 参数的可选范围）。

### 配置维度1：规则行为（参数 ACLACTION）

| 取值 | 作用 | 典型场景 |
|---|---|---|
| Permit | 允许匹配报文（IPSec 保护流匹配用此） | IPSec 隧道保护流 |
| Deny | 禁止匹配报文 | 报文过滤场景 |

### 配置维度2：协议类型（参数 ACLPROTOCOL）

| 取值 | 作用 | 配套端口/ICMP 参数 | 典型场景 |
|---|---|---|---|
| 0（IP，所有 IP 报文） | 匹配所有 IP 报文 | 无 | 任意流（典型 IPSec 用此，仅按源/目的 IP 匹配） |
| 1（ICMP） | 匹配 ICMP 报文 | ACLICMPNAME 或 (ACLICMPTYPE + ACLICMPCODE) | ICMP 报文过滤 |
| 6（TCP） | 匹配 TCP 报文 | ACLSRCPORTOP/BEGIN/END、ACLDESTPORTOP/BEGIN/END、ACLSYNFLAG | TCP 报文过滤 |
| 17（UDP） | 匹配 UDP 报文 | ACLSRCPORTOP/BEGIN/END、ACLDESTPORTOP/BEGIN/END | UDP 报文过滤 |
| 其他（0~255） | 匹配指定协议号 | 无 | 特定协议 |

### 配置维度3：端口范围操作符（参数 ACLSRCPORTOP / ACLDESTPORTOP，仅 TCP/UDP 生效）

| 取值 | 配套端口参数 | 作用 |
|---|---|---|
| Lt | 仅结束端口 | 小于某端口 |
| Eq | 仅起始端口 | 等于某端口 |
| Gt | 仅起始端口 | 大于某端口 |
| Range | 起始 + 结束端口 | 端口范围 |
| Unconfiged | 无 | 不校验端口 |

**步骤位置**：在所属 feature（IPFD-016000 IPSec功能）编排中，于 `ADD ACLGROUPIPSEC`（建规则组）之后、`ADD IPSECPOLICY`（策略引用本规则组的 ACLNUMBER）之前执行。

（无。本命令用法由保护流规划决定，无特性级分支。）

## 约束

- **立即生效**（info）：命令执行后立即生效 — 后果：无延迟
- **必须先建规则组**（critical）：增加 ACL 规则前必须先执行 `ADD ACLGROUPIPSEC` 指定所在规则组 — 后果：未建组则规则无归属
- **TCP/UDP 才生效端口参数**（warning）：ACLPROTOCOL 为 TCP(6)/UDP(17) 时，端口号相关参数才生效；Lt 仅结束端口可输，Gt/Eq/Neq 仅起始端口可输，Range 起始和结束都必须输 — 后果：错配端口参数无效
- **ACLSYNFLAG 仅 TCP 生效**（warning）：ACLSYNFLAG 只有在 ACLPROTOCOL=TCP(6) 时才生效 — 后果：非 TCP 下配置无效
- **ICMP 参数仅 ICMP 生效**（warning）：ACLICMPNAME/ACLICMPTYPE/ACLICMPCODE 只有在 ACLPROTOCOL=ICMP(1) 时才生效；自定义需 ACLICMPNAME=Custom 再配 ICMPTYPE/ICMPCODE — 后果：非 ICMP 下配置无效
- **源目的地址不可重复或包含**（critical）：ACL 规则中不可带有重复或具有包含关系的源目的地址 — 后果：配置冲突
- **IKEv1 不支持 narrow down**（warning）：IKEv1 场景下 ACL 规则不支持 narrow down（一端范围被包含于另一端时仍能协商 SA） — 后果：IKEv1 下两端 ACL 范围需一致
- **ACLPRECEDENCE/ACLTOS/ACLDSCP 互斥**（warning）：ACLPRECEDENCE 不允许和 ACLDSCP 一起配置；ACLTOS 不允许和 ACLDSCP 一起配置 — 后果：配置失败
- **最多 65535 条记录**（warning）：本命令最多可输入 65535 条记录 — 后果：超规格新增失败

# 设置APN RADIUS域名属性（SET APNRADIUSATTR）

> 命令静态知识见 [SET APNRADIUSATTR](command/UNC/20.15.2/SET-APNRADIUSATTR.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

该命令设置 APN 与 AAA 交互时 RADIUS 消息用户名的域名处理策略。配置空间核心维度为 **域名动作**（DOMAINNAMEACT）和 **域名位置**（DOMAINNAMEPOS，仅增加域名时条件必选）。

### 配置维度1：域名动作（参数 DOMAINNAMEACT）

| 取值 | 作用 | 典型场景 |
|---|---|---|
| ADD_DISABLE_STRIP_DISABLE（默认） | 不支持增加或剥离域名功能 | 默认（无域名处理） |
| ADD_ENABLE_STRIP_DISABLE | 仅支持增加域名功能（需配套 DOMAINNAMEPOS） | WSFD-011306：APN="apn-test"，DOMAINNAMEPOS=SUFFIX 后缀域名 |
| ADD_DISABLE_STRIP_ENABLE | 仅支持剥离域名功能 | 用户名带域名需剥离场景 |

### 配置维度2：域名位置（参数 DOMAINNAMEPOS，DOMAINNAMEACT=ADD_ENABLE_STRIP_DISABLE 时条件必选）

| 取值 | 作用 | 典型场景 |
|---|---|---|
| NULL | 无 | 不增加 |
| PREFIX | 前缀域名（域名加在用户名前） | 前缀场景 |
| SUFFIX | 后缀域名（域名加在用户名后） | WSFD-011306：SUFFIX |

**步骤位置**：在 RADIUS 功能（WSFD-011306）编排中，于 `ADD APN`（建 APN）、`ADD APNRDSSVRGRP`（APN 绑服务器组）、`ADD APNRDSCLIENTIP`（APN 绑 Gi 接口）之后/同时执行，用于调整 RADIUS 用户名格式。

## 决策点

**域名处理策略**（UNC pre-build 内嵌，不编号——按 `assets/CLAUDE.md` §5.7）：

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| 不处理 | DOMAINNAMEACT=ADD_DISABLE_STRIP_DISABLE | 默认（用户名原样发 AAA） |
| 增加后缀域名 | DOMAINNAMEACT=ADD_ENABLE_STRIP_DISABLE + DOMAINNAMEPOS=SUFFIX | WSFD-011306 RADIUS 功能 |
| 增加前缀域名 | 同上 + PREFIX | 用户名加前缀 |
| 剥离域名 | DOMAINNAMEACT=ADD_DISABLE_STRIP_ENABLE | 用户名带域名需剥离 |

## 约束

- **立即生效**（info）：命令执行后立即生效 — 后果：无回滚窗口
- **APN 必须先建**（critical）：APN 必须由 `ADD APN` 已配置 — 后果：APN 不存在则配置失败
- **DOMAINNAMEPOS 条件必选**（critical）：DOMAINNAMEACT=ADD_ENABLE_STRIP_DISABLE 时必填 DOMAINNAMEPOS — 后果：缺则配置失败
- **APN 命名约束**（warning）：字符串 1~63，不支持空格及 `_`/`#`/`$`/`&` 等，不能以 `.` 开头或连续 `.`，不区分大小写 — 后果：违规命名配置失败

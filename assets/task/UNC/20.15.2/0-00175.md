# SET GLBCHARGECHAR 原子任务

> 命令静态知识见 [SET GLBCHARGECHAR](command/UNC/20.15.2/SET-GLBCHARGECHAR.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

SET GLBCHARGECHAR 配置**全局**计费属性（CC，Charging Characteristics）。CC 决定一个用户走在线计费还是离线计费。本命令按三类用户（本地 HOME / 漫游 ROAM / 拜访 VISIT）各给一个十六进制 CC 值，再加三个标志位（HOMESGSN/ROAMSGSN/VISITSGSN）决定该类用户是否使用对端网元（UDM/HSS/MME/SGSN-C）下发的签约 CC，还是用本地全局 CC。

全局唯一一条记录（最大记录数=1）。配置粒度有**三层**：全局（本命令）→ APN/DNN（SET APNCHARGECTRL）→ UserProfile（SET USRPROFCHARGE），逐层细化。

**参数取值样例（计费属性配置范式）**

| 参数 | WSFD-011201(4G离线) | WSFD-011206(5G融合) | WSFD-109001(Gy在线) | 说明 |
|---|---|---|---|---|
| HOME | `0x0100` | `0x0100` | `0x0100` | 本地用户 CC，十六进制 1~6 字符 |
| ROAM | `0x0100` | `0x0100` | `0x0100` | 漫游用户 CC |
| VISIT | `0x0100` | `0x0100` | `0x0100` | 拜访用户 CC |
| HOMESGSN | `ENABLE` | `DISABLE` | `ENABLE` | 本地用户是否用签约 CC |
| ROAMSGSN | `ENABLE` | `DISABLE` | `ENABLE` | 漫游用户是否用签约 CC |
| VISITSGSN | `ENABLE` | `DISABLE` | `ENABLE` | 拜访用户是否用签约 CC |

**典型行（4G 离线/Gy 在线，使用签约 CC）**

```
SET GLBCHARGECHAR: HOME="0x0100", ROAM="0x0100", VISIT="0x0100",
  HOMESGSN=ENABLE, ROAMSGSN=ENABLE, VISITSGSN=ENABLE;
```

**典型行（5G 融合计费，使用本地全局 CC）**

```
SET GLBCHARGECHAR: HOME="0x0100", ROAM="0x0100", VISIT="0x0100",
  HOMESGSN=DISABLE, ROAMSGSN=DISABLE, VISITSGSN=DISABLE;
```

## 决策点

### xSGSN 标志位：使用签约 CC 还是本地全局 CC

HOMESGSN/ROAMSGSN/VISITSGSN 是本命令的核心决策。当对端网元（5G 的 UDM、4G 的 MME/SGSN-C）下发签约 CC 时，这三个标志决定每类用户用哪个 CC。

| 选项 | xSGSN 取值 | 含义 | 典型场景 |
|---|---|---|---|
| 用签约 CC | `ENABLE` | 使用 UDM/HSS/MME 下发的签约 CC | 4G 离线（WSFD-011201）、Gy 在线（WSFD-109001） |
| 用本地全局 CC | `DISABLE` | 使用本命令 HOME/ROAM/VISIT 配的 CC | 5G 融合计费（WSFD-011206），SMF 直接动态获取 |

> 注意：从 UDM/对端获取签约 CC 时，SMF 直接动态获取计费属性，与本地配置 CC 无关。

### CC 配置粒度选择

| 选项 | 命令 | 适用 |
|---|---|---|
| 全局（本命令） | `SET GLBCHARGECHAR` | 全局默认 CC |
| APN/DNN 级 | [SET APNCHARGECTRL](task/UNC/20.15.2/0-00149.md) | 按 APN/DNN 区分 CC |
| UserProfile 级 | [0-00207](task/UNC/20.15.2/0-00207.md)（SET USRPROFCHARGE） | 按用户模板区分 CC |

### CC 值（HOME/ROAM/VISIT）来源

| 选项 | 来源 | 说明 |
|---|---|---|
| 签约下发 | UDM/HSS | xSGSN=ENABLE 时用此 |
| 本地配置 | 本命令 HOME/ROAM/VISIT | xSGSN=DISABLE 时用此，十六进制如 `0x0100` |

## 约束

- **只对新激活用户生效** · severity=MEDIUM · 已在线用户不受影响。
- **最大记录数=1**（全局唯一） · severity=HIGH · 重复执行为改写。
- **xSGSN 与配置粒度联动**：当对端携带签约 CC 且本地基于**全局**配置 CC 时，由本命令 HOMESGSN/ROAMSGSN/VISITSGSN 控制；若本地基于 **UserProfile/APN** 配置 CC，则由 `ADD CHARGECHAR` 的对应参数控制 · severity=HIGH。
- **CC 值为十六进制**：HOME/ROAM/VISIT 仅支持 0x/X、a-f/A-F、0~9，长度 1~6 · severity=MEDIUM。
- **4G 签约 CC 携带方**：SP 合设时由 MME 携带（从 HSS 获取）；SP 分离时 SGW-C 由 MME 携带、PGW-C 由 SGSN-C 携带 · severity=MEDIUM。
- **5G 签约 CC 携带方**：由 UDM 下发 · severity=MEDIUM。

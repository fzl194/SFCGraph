# 设置APN上报属性（SET APNREPORTATTR）

> 命令静态知识见 [SET APNREPORTATTR](command/UNC/20.15.2/SET-APNREPORTATTR.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法

按"对接网元维度 × APN 类型映射"建模。APN 类型取值常见 `REQUESTED`（请求的，用终端上报的 APN）、`SERVICE`（真实的，用真实 APN）。每次 ADD APN 自动建一条记录（CONGESTIONRPT=DISABLE, INTELLIGENTSEL=DISABLE, LOCATIONREPORT=DISABLE, MAPTRANSDATA=DISABLE，其余 REQUESTED/SERVICE 初值见证据）。配置仅当 APN 参数为真实 APN 时生效。

| 维度 | 参数 | 取值 | 默认 | 作用 | 适用场景 |
|---|---|---|---|---|---|
| PCF/CHF/PCRF/OCS/CG APN 类型 | PCF, CHF, PCRF, OCS, CG | REQUESTED/SERVICE（本端规划/与对端协商） | ADD APN 自动建记录初值 | 上报给各网元的 APN 名 | 别名 APN 场景需真实 APN 的 PCF/CHF=SERVICE |
| AAA 类 APN 类型 | AAAACCT, AAAAUTH, DIAMETERAAA | REQUESTED/SERVICE（本端规划） | — | 上报给 AAA 鉴权/计费的 APN 名 | AAA 交互 |
| 话统/用户面 APN 类型 | PERFORMANCE, UPF | REQUESTED/SERVICE（本端规划） | — | 话统/用户面 APN 名 | 性能统计/用户面交互 |
| 小区拥塞上报 | CONGESTIONRPT | ENABLE/DISABLE（本端规划） | DISABLE | 基于 APN 的小区拥塞上报开关 | WSFD-211101 基于小区负荷上报 |
| 周边上报（多数已弃用） | INTELLIGENTSEL, LOCATIONREPORT, MAPTRANSDATA | ENABLE/DISABLE（本端规划） | DISABLE | 智能网关选择/实时位置/VLR ID 获取 | 证据标注已弃用 |

典型脚本（激活别名 APN：真实 APN 的 PCF/CHF 配为 SERVICE）：
```
SET APNREPORTATTR: APN="apn5", PCF=SERVICE, CHF=SERVICE;
```
（基于小区负荷上报：开 CONGESTIONRPT）：
```
SET APNREPORTATTR: APN="apn1", CONGESTIONRPT=ENABLE;
```
（VoLTE 快速恢复：PCRF=REQUESTED 映射真实 APN）：
```
SET APNREPORTATTR: APN="ims.real", PCRF=REQUESTED;
```

步骤位置：建 APN（`ADD APN`）→ 本任务 → （可选）开 License（`SET LICENSESWITCH`，小区负荷上报前置）

## 决策点

### 决策点：各交互网元 APN 类型映射（DP 0-03043）

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| REQUESTED | PCF/CHF/PCRF/OCS/CG/AAA/话统/UPF=REQUESTED | 用终端激活请求里的 APN（别名 APN 时用别名） |
| SERVICE | 对应网元=SERVICE | 用真实/业务 APN，别名 APN 场景需真实 APN 的 PCF 和 CHF 配为 SERVICE |
| 各网元独立 | 各网元参数独立取值 | 不同网元可不同策略 |

### 决策点：周边上报开关组（DP 0-03044）

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| 开小区拥塞上报 | CONGESTIONRPT=ENABLE | WSFD-211101 基于小区负荷上报（License 前置 SET LICENSESWITCH） |
| 关闭/默认 | CONGESTIONRPT=DISABLE | 不上报小区拥塞 |
| 已弃用开关 | INTELLIGENTSEL/LOCATIONREPORT/MAPTRANSDATA | 证据标注已弃用，按需规划 |

## 约束

- **生效时机**（warning，rule-0-03106）：生效时机：只对新激活用户生效，不立即作用于在线用户
- **一致性（前置）**（warning，rule-0-03107）：一致性（前置）：APN 参数须为真实 APN，配置仅当 APN 为真实 APN 时生效；APN 须先通过 `ADD APN` 建好
- **规格上限**（info，rule-0-03108）：规格上限：每个 APN 支持一条配置，不允许多条同 APN 记录
- **唯一性**（info，rule-0-03109）：唯一性：ADD APN 自动建一条初始记录，随后用本命令改写；不输入的可选参数保持当前值不变（可 LST APNREPORTATTR 查询）
- **一致性**（warning，rule-0-03110）：一致性：REQUESTED 遇别名 APN 时对该网元按别名 APN 处理，计费/统计口径需评估

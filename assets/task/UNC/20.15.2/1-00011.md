# 配置计费消息缓存

> 主备 CHF 均故障时 SMF 缓存计费消息、CHF 恢复后回放。被引用于 [2-00003 融合计费](task/UNC/20.15.2/2-00003.md)。特性专属（融合计费独有可靠性机制）。

## 配置方法

| 步骤 | 命令 | 关键参数 |
|---|---|---|
| 缓存前提（故障处理放通） | `SET FAILHANDLING` | FHACTION=CONTINUE（固定）；TXTIMER；UPDCONTTERM=ENABLE；CNVCHGRECOVER=ENABLE；RECOVERINTERVAL |
| 缓存前提（PDU 异常放通） | `MOD PDUSCACT` | SCACT=FAILOVER 或 CONTINUE（固定） |
| 缓存开关与回放参数 | `SET N40MSGSTG` | STGSWITCH=ENABLE（固定）；REPLAYINTERVAL；REPLAYRATE |
| 缓存期间 Trigger | `SET STGTRIGGER` | VOLUMELIMIT=IMMEDIATE；PDUVOLUMELIMIT |
| ChargingDataRef 生成方式 | `SET CNVRGDCHGPARA` | CHGDATAREFGEN=SMF（固定） |
| 全局默认 CHF 组 | `SET GLBDFTCHFGROUP` | PRIMARYCHFGRP；SECONDARYCHFGRP（须已在 [1-00007](task/UNC/20.15.2/1-00007.md) 配） |
| 缓存文件超期 | `SET CDRSTORAGECTRL` | CDREXPIREDAY；CDREXPIREWEEK |
| 缓存告警 | `SET STGALARMCTRL` | ALMTHD；MONITORTIME |
| N4 计费消息池扩展 | `SET N4CHGMSGCTRL` | MSGCACHEEPDSW=ENABLE；CACHEFULLACT=CONTINUE |

**典型脚本**：
```
SET FAILHANDLING: CCTMPLTNAME="cct-test", FHACTION=CONTINUE, TXTIMER=5, UPDCONTTERM=ENABLE, CNVCHGRECOVER=ENABLE, RECOVERINTERVAL=20;
MOD PDUSCACT: CCTMPLTNAME="cct-test", CODETYPE=VALUE, STATUSCODE=502, SCACT=FAILOVER;
SET N40MSGSTG: STGSWITCH=ENABLE, REPLAYINTERVAL=30, REPLAYRATE=500;
SET STGTRIGGER: VOLUMELIMIT=IMMEDIATE, PDUVOLUMELIMIT=500;
SET CNVRGDCHGPARA: CHGDATAREFGEN=SMF;
SET GLBDFTCHFGROUP: PRIMARYCHFGRP="ChfGroup1", SECONDARYCHFGRP="ChfGroup2";
SET CDRSTORAGECTRL: CDREXPIREDAY=2, CDREXPIREWEEK=4;
SET STGALARMCTRL: ALMTHD=5120, MONITORTIME=60;
SET N4CHGMSGCTRL: MSGCACHEEPDSW=ENABLE, CACHEFULLACT=CONTINUE;
```

**步骤位置**：异常处理（[1-00010](task/UNC/20.15.2/1-00010.md)）+ CHF 选择（[1-00007](task/UNC/20.15.2/1-00007.md)，全局默认组）之后；可选增强模块（可靠性场景）。

## 约束

- **缓存前提（critical）**：必须同时满足——`SET FAILHANDLING` FHACTION=CONTINUE + `MOD PDUSCACT` SCACT=FAILOVER/CONTINUE + `SET N40MSGSTG` STGSWITCH=ENABLE + `SET CNVRGDCHGPARA` CHGDATAREFGEN=SMF — 任一不满足缓存不生效。
- **全局默认 CHF 组（critical）**：`SET GLBDFTCHFGROUP` 必须配置 —— 缓存消息回放须有默认 CHF 组（确保配置变更导致 CHF 删除时仍有可回放目标）。须先在 [1-00007](task/UNC/20.15.2/1-00007.md) 配 CHF 组。
- **告警监控时长（warning）**：`SET STGALARMCTRL` MONITORTIME 建议为 `SET N40MSGSTG` REPLAYINTERVAL 的 2 倍以上；=0 时受软参 DWORD507 控制。
- **超期文件不回放（info）**：超过 CDREXPIREDAY/CDREXPIREWEEK 的缓存文件不再回放。

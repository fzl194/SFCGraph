# ADD DIAMCONNECTION 原子任务

> 命令静态知识见 [ADD DIAMCONNECTION](command/UNC/20.15.2/ADD-DIAMCONNECTION.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

ADD DIAMCONNECTION 在已建好的 [DIAMCONNGRP](command/UNC/20.15.2/ADD-DIAMCONNGRP.md) 链路组内增加一条 Diameter 链路，绑定本端逻辑接口（LOCINTERFACE）与对端地址。对端地址通过 `PEERADDRTYPE` 二选一：**TCP 模式**走 IPv4/IPv6 地址（可选 PEERPORT），**SCTP 模式**走已用 ADD SCTPENDPOINT 建好的对端 SCTP 端点名（PEERSCTPENDPT）。LOCALPORT 是否必配取决于 [SET CONCENPOINT](command/UNC/20.15.2/SET-CONCENPOINT.md) 的集中点模式。

**参数取值样例**

| 参数 | 取值样例 | 获取方法 | 说明 |
|---|---|---|---|
| DIAMCONNGRP | `dconn_gx` / `dconn_pcrf1` / `dia1` | 已配置数据中获取 | 必须先用 ADD DIAMCONNGRP 建好，可 LST DIAMCONNGRP 查 |
| LOCINTERFACE | `gxif1/0/0` / `gyif1/0/0` / `s6bif1/0/0` | 已配置数据中获取 | 已用 ADD LOGICINF 建好，可 LST LOGICINF 查 |
| LOCALPORT | `16400`（Gx SCTP）/ `13200`（Gy SCTP）/ `16450`（Gx LOCALPORT）/ `19766`（S6b）/ `0` | 本端规划 | LOCALPORT 模式必配非 0；否则可省或 0 |
| PEERADDRTYPE | `IPv4` / `IPv6` / `SCTP` | 与对端协商 | 不选则与对端全部地址建链 |
| PEERIPV4ADDR | `10.11.21.59` | 与对端协商 | PEERADDRTYPE=IPv4 时配 |
| PEERPORT | `3868` | 与对端协商 | TCP 模式可选 |
| PEERSCTPENDPT | `pcrfendb` / `ocsend` / `endpoint0` | 已配置数据中获取 | PEERADDRTYPE=SCTP 时配，已 ADD SCTPENDPOINT |
| REVERSEIP | `ENABLE` / `DISABLE` | 与对端协商 | SCTP 模式下对端端口为奇数时可 ENABLE 颠倒主从 IP |

**特性场景取值对比**

| 特性场景 | DIAMCONNGRP | LOCINTERFACE | LOCALPORT | PEERADDRTYPE | 对端地址/端点 | 说明 |
|---|---|---|---|---|---|---|
| Gx 直连 PCRF IPv4（WSFD-109101） | dconn_pcrf1 | gxif1/0/0 | 16450（LOCALPORT 模式）/0 | IPv4 | PEERIPV4ADDR=10.11.21.59 | LOCALPORT 仅当 GXCONCENMODE=LOCALPORT 时配非 0 |

| Gx over DRA（WSFD-011132） | dconn_gx1 | gxif1/0/0 | 16401 | 不选/SCTP | （DRA 地址由 ADD DIAMPEERADDR 配） | 不选地址类型=与对端全部地址建链 |

## 决策点

### PEERADDRTYPE 对端地址类型选择（TCP vs SCTP）

承载协议决定后续参数集与对端准备物，必须与对端协商一致：

| PEERADDRTYPE | 承载 | 后续参数 | 对端前置准备 |
|---|---|---|---|
| IPv4 / IPv6 | TCP | PEERIPV4ADDR/PEERIPV6ADDR（+可选 PEERPORT） | 对端主机 IP 可达 |
| SCTP | SCTP | PEERSCTPENDPT | 已用 [ADD SCTPENDPOINT](command/UNC/20.15.2/ADD-SCTPENDPOINT.md) 建好对端端点 |
| 不选 | — | — | 本端接口与对端主机**全部地址**建立链接 |

### LOCALPORT 是否必配（集中点模式联动）

| SET CONCENPOINT 模式 | LOCALPORT 配置 |
|---|---|
| GXCONCENMODE/GYCONCENMODE/S6BCONCENMODE = LOCALPORT | **必须配非 0 有效值**（如 16450/13201/19766） |
| 其他模式 | 不配或配 0（系统自动分配） |

### 多 PGW-U 用户 IP 相同场景的链路组数量

同一 PGW-C 对接多个 PGW-U 且不同 PGW-U 下用户 IP 重叠、PCRF 要求 UE IP + GxLocalHost 唯一确定用户时（CU Full Mesh 组网），每对接一个 PCRF 须规划 **2 个** Diameter 链路组（dconn_pcrf1~4），本命令相应建多条链路。否则只需 1 个链路组。

## 约束

- **立即生效** · severity=LOW · 命令执行后立即建链。
- **最大 3000 条记录** · severity=LOW。
- **同 ConnGrp+LocInterface 允许多条记录，但每条地址必须不同** · severity=HIGH · 地址重复则配置失败。
- **LOCALHOSTNAME 已配 Local Info 前置依赖** · severity=HIGH · 必须先 ADD DIAMLOCINFO。
- **PEERHOSTNAME 已配且 APPLICATION 类型一致前置依赖** · severity=HIGH · 必须先建对应 APPLICATION（GX/GY/S6B）的对端服务器（PCRF/OCS/AAA/DRA）。
- **PEERADDRTYPE 与 SCTPENDPOINT/IP 地址联动** · severity=HIGH · SCTP 模式须先 ADD SCTPENDPOINT；TCP 模式须对端 IP 可达。
- **配 DiamConnection 后再增加 Addr 不会自动建链** · severity=MEDIUM · 必须显式配 DiamConnection 才会建链，仅加地址不建链。
- **不选地址类型=与对端全部地址建链** · severity=MEDIUM · 仅在已用 ADD DIAMPEERADDR 配好对端多地址时使用。

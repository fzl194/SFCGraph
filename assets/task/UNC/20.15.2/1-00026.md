# UNC Radius 私有扩展

> **UNC 域鉴权族扩展骨架**（SOP §3.5 域通用）。被引用于 2 个 WSFD 鉴权族特性（WSFD-011306 + WSFD-011307）。
>
> 设计参考：本骨架的 `SET RDSACCTREQVSA`/`SET APNRDSACCTCTRL`/`SET APNRADIUSATTR` 3 条**全部是 APN/AAA 服务器组级别的可选扩展**——主骨架（[UNC 1-00025](task/UNC/20.15.2/1-00025.md)）搭好鉴权+计费主链路后，本骨架按需开启高级功能。

## 配置方法

本骨架在 [UNC 1-00025](task/UNC/20.15.2/1-00025.md) 主链路之后启用 Radius 私有扩展——3GPP VSA 私有属性、APN 级计费控制、域名增加/剥离。

| 步骤 | 命令 | 关键参数 |
|---|---|---|
| 1. RADIUS 计费 3GPP 扩展（WSFD-011306 必选）| `SET RDSACCTREQVSA` → [0-00280](task/UNC/20.15.2/0-00280.md) | RDSSVRGRPNAME, THREEGPP=ENABLE/DISABLE, THREEGPP2=ENABLE/DISABLE |
| 2. APN RADIUS 计费控制（WSFD-011306/011307 选）| `SET APNRDSACCTCTRL` → [0-00274](task/UNC/20.15.2/0-00274.md) | APN, TIMETHRESHOLD/VOLUMETHRESHOLD, SUPPORTACCTRSP, SUPPORTACCTUPD, RATTRIGGER, SRVTRIGGER, BUFFERTIMEVALUE, DEACTIVE（计费服务器无响应处理）|
| 3. APN 域名增加/剥离（WSFD-011306/011307 选）| `SET APNRADIUSATTR` → [0-00273](task/UNC/20.15.2/0-00273.md) | APN, DOMAINNAMEACT=ADD_ENABLE_STRIP_DISABLE/..., DOMAINNAMEPOS=PREFIX/SUFFIX |

**典型脚本（WSFD-011306 主链路扩展）**：
```
SET RDSACCTREQVSA: RDSSVRGRPNAME="isprg", THREEGPP=ENABLE, THREEGPP2=DISABLE;
SET APNRDSACCTCTRL: APN="apn-test", TIMETHRESHOLD=30, VOLUMETHRESHOLD=20, SUPPORTACCTRSP=ENABLE, SUPPORTACCTUPD=ENABLE, RATTRIGGER=ENABLE, SRVTRIGGER=ENABLE, BUFFERTIMEVALUE=2;
SET APNRADIUSATTR: APN="apn-test", DOMAINNAMEACT=ADD_ENABLE_STRIP_DISABLE, DOMAINNAMEPOS=SUFFIX;
```

**步骤位置**：在 [UNC 1-00025](task/UNC/20.15.2/1-00025.md) 主链路 `ADD APNRDSSVRGRP`（步骤 5）之后；`SET RDSACCTREQVSA` 须有 RDSSVRGRPNAME 引用（步骤 1 之后），`SET APNRDSACCTCTRL`+`SET APNRADIUSATTR` 须有 APN 引用（[UNC 1-00024](task/UNC/20.15.2/1-00024.md) 步骤 4 之后）。

## 约束

- **RDSACCTREQVSA 必配**（critical，WSFD-011306 场景）：3GPP 私有扩展属性是 AAA 服务器协议互通前提 — 未配可能致 AAA 不识别 UNC 上报字段；激活文档（32909765）步骤 6a 标注必选。
- **APNRDSACCTCTRL 优先级 > APNAUTHATTR AAANORSPCTRL**（warning）：当 APNRDSACCTCTRL DEACTIVE=CONTINUE 而 AAANORSPCTRL=DEACTIVE 时，APNRDSACCTCTRL 优先级更高（见 [WSFD-011306 实现原理](evidence/UNC/20.15.2/WSFD-011306/WSFD-011306-实现原理_31922672.md) 表 2）；约束归 [WSFD-011306](feature/UNC/20.15.2/WSFD-011306.md) RADIUS 鉴权响应处理。
- **BUFFERTIMEVALUE 范围**（info）：0~1440 秒（业务触发 RADIUS 消息发送时的业务报文延时时间）。
- **域名 PREFIX/SUFFIX 分隔符**（info）：由 `SET DOMAINSEPARATOR` 命令配置；本骨架不直接涉及该命令。
- **不引入独立命令外的命令**：本骨架不重复主骨架 [UNC 1-00025](task/UNC/20.15.2/1-00025.md) 的 RDSSVR/RDSSVRGRP/APNRDSSVRGRP 等命令；不重复 [UNC 1-00024](task/UNC/20.15.2/1-00024.md) 的 ADD APN/SET APNADDRESSATTR。
- **RDSRSPADDRCHK 不在本骨架**（warning）：`SET RDSRSPADDRCHK`（响应消息源 IP/端口检查）属 WSFD-011306 可选全局开关，不在私有扩展骨架内；约束归 [WSFD-011306](feature/UNC/20.15.2/WSFD-011306.md)。
- **FHBYPASS 不在本骨架**（critical）：`SET FHBYPASS`（一键放通，**仅紧急使用**）优先级最高，覆盖 APNAUTHATTR/ APNRDSACCTCTRL；不在私有扩展骨架内，约束归 [WSFD-011306](feature/UNC/20.15.2/WSFD-011306.md)。

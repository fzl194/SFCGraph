# 增加IPv6 IKE对等体（ADD IKEPEER6）

> 命令静态知识见 [ADD IKEPEER6](command/UNC/20.15.2/ADD-IKEPEER6.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

该命令增加一个 IPv6 IKE 对等体，对端协商参数由本命令承载（与 `ADD IKEPEER` 为 IPv4/IPv6 对版）。配置空间主要由三个维度决定：**IKE 版本**、**认证方式**、**交换模式**。

### 配置维度1：IKE 版本（参数 VERSION2）

| 取值 | 作用 | 典型场景 |
|---|---|---|
| VERSION2=TRUE（默认） | 使用 IKEv2（推荐） | 推荐场景 |
| VERSION2=FALSE | 使用 IKEv1 | 国密 IPsec 支持 IKEv1 |

> 参数 IKEMSGSYNC、IPSECMSGSYNC 只有在 VERSION2=TRUE 时才生效。

### 配置维度2：认证方式（通过 PROPOSAL 引用 IKE 提议 + 证书参数）

| 取值组合 | 作用 | 配套必选参数 | 典型场景 |
|---|---|---|---|
| 预共享密钥 | 用预共享密钥认证 | PRESHAREDKEY；PROPOSAL 引用 AUTHMETHOD=Pre_share 的 IKEPROPOSAL | 常规 IPv6 IPSec 隧道 |
| 数字信封（国密） | 用国密数字信封认证 | PROPOSAL 引用 AUTHMETHOD=Digital_envelope；CERTLOCALFILE + ENCCERTLOCFILE | 激活国密 IPsec 支持 IKEv1（IPv6） |

### 配置维度3：本地 ID 类型（参数 LOCALIDTYPE）

| 取值 | 作用 | 配套参数 |
|---|---|---|
| IPv6（默认） | 用 IPv6 地址作本地 ID | — |
| Fqdn_ipv6 | 用 IPv6 FQDN 作本地 ID | 必配 REMOTEID |
| User_fqdn_ipv6 | 用 IPv6 User-FQDN 作本地 ID | 必配 REMOTEID |
| Dn_ipv6 | 用 IPv6 DN 作本地 ID | — |
| Ip_disable | 关闭本地 IP 校验 | 仅支持 IKEv1 |

### 配置维度4：交换模式（参数 EXCHANGEMODE）

| 取值 | 作用 | 典型场景 |
|---|---|---|
| Main（主模式，默认） | IKEv1 主模式 | 国密 IKEv1 仅支持主模式 |
| Aggressive（野蛮模式） | IKEv1 野蛮模式 | 对端地址动态获取 |

**步骤位置**：在所属 feature（IPFD-016000 IPSec功能）编排中，于 `ADD IKEPROPOSAL` 之后、`ADD IPSECPOLICY6` 之前执行。

## 决策点

**IKE 版本与认证方式选择**（UNC pre-build 内嵌，不编号——按 `assets/CLAUDE.md` §5.7）：

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| IKEv1 + 数字信封（国密） | VERSION2=FALSE；CERTLOCALFILE + ENCCERTLOCFILE | 激活国密 IPsec 支持 IKEv1（普通 IPv6 IPSec 隧道） |
| IKEv2 + 预共享密钥 | VERSION2=TRUE；PRESHAREDKEY | 推荐场景 |

## 约束

- **立即生效**（info）：命令执行后立即生效 — 后果：无延迟
- **IKEv1 不安全**（warning）：IKEv1 是不安全的协议，建议使用 IKEv2 — 后果：安全性弱
- **国密 IKEv1 限制**（critical）：IKEv1 国密 IPSEC 仅支持主模式；不支持 NAT 穿越；不支持主备隧道；仅支持数字信封认证 — 后果：配置不符限制会协商失败
- **IP-disable 仅 IKEv1**（warning）：LOCALIDTYPE=Ip_disable 仅支持 IKEv1 — 后果：IKEv2 下配置无效
- **证书字段与场景字段互斥**（warning）：证书字段（CERTLOCALFILE 等）和场景字段（CERTSCENARIO 等）不能同时配置 — 后果：配置失败
- **FQDN/User-FQDN 必配 REMOTEID**（warning）：LOCALIDTYPE=Fqdn_ipv6 或 User_fqdn_ipv6 时，REMOTEID 必须配置 — 后果：协商失败
- **IKEMSGSYNC/IPSECMSGSYNC 仅 v2 生效**（info）：参数 IKEMSGSYNC、IPSECMSGSYNC 只有在 VERSION2=TRUE 时才生效 — 后果：v1 下配置无效
- **最多 4000 条记录**（warning）：本命令最多可输入 4000 条记录 — 后果：超规格新增失败

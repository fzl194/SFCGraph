# 绑定Diameter本端主机到组（ADD LOCALHOSTBIND）

> 命令静态知识见 [ADD LOCALHOSTBIND](command/UNC/20.15.2/ADD-LOCALHOSTBIND.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法

ADD LOCALHOSTBIND 把已通过 [ADD DIAMLOCINFO](command/UNC/20.15.2/ADD-DIAMLOCINFO.md) 创建的 Diameter 本端主机，逐一绑定到已通过 [ADD LOCALHOSTGRP](task/UNC/20.15.2/0-00042.md) 创建的本端主机组中。它服务于 PCC 对接（PCRF/PCF）场景，建立 PGW-C/SMF 的 Diameter 连接归属关系；多 PGW-C 实例时按实例分组绑定。

核心配置空间有 2 个维度：**组名**（必选，引用已建组）、**主机名**（必选，引用已建主机）。

### 配置维度 1：组名（参数 LOCGRPNAME，必选）

| 参数 | 取值 | 数据来源 | 约束 |
|---|---|---|---|
| `LOCGRPNAME` | `pgwc_host_group_01` / `pgwc_host_group_02` | 本端规划 | 1~63 字符，不支持空格，不区分大小写；必须已由 ADD LOCALHOSTGRP 创建 |

> 组不存在则绑定失败；本参数为 local_planned（本端规划），引用 ADD LOCALHOSTGRP 的命名。

### 配置维度 2：主机名（参数 LOCHOSTNAME，必选）

| 参数 | 取值 | 数据来源 | 约束 |
|---|---|---|---|
| `LOCHOSTNAME` | `pgwc_1` / `pgwc_2` | 本端规划 | 1~127 字符，不支持空格；必须已由 ADD DIAMLOCINFO 创建 |

> 主机名不存在则绑定失败；本参数为 local_planned（本端规划），按 PGW-C/SMF 实例规划从已配置数据中取。

**典型脚本（两实例分组绑定）**

```
ADD LOCALHOSTBIND: LOCGRPNAME="pgwc_host_group_01", LOCHOSTNAME="pgwc_1";
ADD LOCALHOSTBIND: LOCGRPNAME="pgwc_host_group_02", LOCHOSTNAME="pgwc_2";
```

**步骤位置**：在"PCC 基本功能（5G）"对接流程中，位于 ADD LOCALHOSTGRP（建组）+ ADD DIAMLOCINFO（建主机）之后。典型顺序：ADD LOCALHOSTGRP（建组）→ ADD DIAMLOCINFO（建主机）→ ADD LOCALHOSTBIND（绑定）。

## 决策点

### 决策点：主机组与主机名对应关系（DP 0-02061）

问：每个 PGW-C/SMF 实例的本端主机归属到哪个本端主机组（决定 LOCGRPNAME 与 LOCHOSTNAME 配对）？

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| 一实例一组、单主机绑定 | 一对一：`LOCGRPNAME=pgwc_host_group_01, LOCHOSTNAME=pgwc_1` | 证据范例范式，每实例独占一组 |
| 一组多主机（负载分担） | 一对多：同 `LOCGRPNAME` 下绑多个 `LOCHOSTNAME`（最多 64 个） | 单组内多主机负载分担 |

> 对应关系由 Diameter 路由与 PCRF/PCF 对接拓扑规划决定（global_planned / 对端协商）；证据范例为"一实例一组、单主机绑定"。

### 决策点：绑定时机与前置就绪（DP 0-02062）

问：ADD LOCALHOSTBIND 执行时机与前置命令的关系？

| 选项 | 前置条件 |
|---|---|
| 先建组+建主机再绑定 | ADD LOCALHOSTGRP（组存在）+ ADD DIAMLOCINFO（主机存在）均已执行成功 |
| 组或主机未建直接绑定 | 不允许——LOCGRPNAME 与 LOCHOSTNAME 均必须先由各自命令创建，引用失败 |

> 绑定命令为引用型，组与主机必须先存在；本决策为顺序约束，无取值分叉。

## 约束

- rule-0-02151 · severity=high · 前置依赖：LOCGRPNAME 必须已通过 ADD LOCALHOSTGRP 创建存在，LOCHOSTNAME 必须已通过 ADD DIAMLOCINFO 创建存在，否则引用失败
- rule-0-02152 · severity=warning · 规格上限（组内）：单个 Diameter 本端主机分组内最多 64 个本端主机，超过上限新增失败
- rule-0-02153 · severity=warning · 规格上限（表）：本命令最大记录数为 1000，超过上限新增失败
- rule-0-02154 · severity=high · 命名约束：LOCGRPNAME 1~63 字符不支持空格不区分大小写，LOCHOSTNAME 1~127 字符不支持空格，违规被拒
- rule-0-02155 · severity=info · 立即生效：命令执行后立即生效，无需重启

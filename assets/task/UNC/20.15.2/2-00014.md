# 会话管理（WSFD-010501）

> **能力型底座特性**——本特性无需 License、无需 MML 配置、无激活步骤。特性静态知识见 [WSFD-010501](feature/UNC/20.15.2/WSFD-010501.md)。
>
> **本页定位**：作为"底座骨架"显式记录**本特性是其他特性的被依赖锚点**——会话级 NAS 信令与 N4 PFCP 链路建立归各被依赖特性，本骨架 wiki 解决"会话承载"的可追溯性问题，**不**重复描述被依赖特性的配置。

## 配置概览

本特性作为 **AMF/SMF/MME/SGW-C/PGW-C/SGSN 控制面网元** 的底座能力，覆盖三种接入制式：

- **5G PDU 会话**（UNC 20.0.0+，NAS 信令 + N4 PFCP，AMF/SMF 主控）
- **4G EPS 承载**（UNC 早于 20.0.0，S1/S5/S8 + Sxb）
- **2/3G PDP 上下文**（UNC 早于 20.0.0，Gn/Gp）

**核心动作**：UNC 控制面**主动触发**会话的建立/修改/释放。本特性**不引入新命令、不引入新对象**——"配置生成"在本特性上**无内容可生成**。

**被依赖骨架 compound（指向 3 维度）**：

| 会话建立依赖维度 | 骨架 compound | 实际配置发生于 |
|---|---|---|
| 地址分配（DNN/PDU UE IP）| [UNC 1-00020](task/UNC/20.15.2/1-00020.md) | [WSFD-010502](feature/UNC/20.15.2/WSFD-010502.md|
| PCC/CHF 策略（PCF N7 / CHF N40）| [UNC 1-00021](task/UNC/20.15.2/1-00021.md) | 计费域已建 [2-00001](task/UNC/20.15.2/2-00001.md)~[2-00011](task/UNC/20.15.2/2-00011.md) |
| 控制面-用户面接口（N4/Sxb/Gn）| [UNC 1-00022](task/UNC/20.15.2/1-00022.md) \| UNC N4 PFCP 偶联特性（**★ R1.5 待建**）+ SBI 基础（已建 atom 支撑）|

> 其他相关被依赖特性（APN/DNN 实例 / 用户签约数据 / 接入控制等）：见 [UNC 1-00020](task/UNC/20.15.2/1-00020.md) "被依赖特性索引"段。

## 配置流程（骨架——3 个被依赖维度的空骨架步骤）

> **每步不显示具体命令**——本骨架的所有"步骤"都**直接跳转到被依赖特性**生成配置。

1. **会话建立依赖-地址分配**：无本骨架配置 → [UNC 1-00020](task/UNC/20.15.2/1-00020.md)
2. **会话建立依赖-PCC/CHF 策略**：无本骨架配置 → [UNC 1-00021](task/UNC/20.15.2/1-00021.md)
3. **会话建立依赖-N4/Sxb/Gn 接口**：无本骨架配置 → [UNC 1-00022](task/UNC/20.15.2/1-00022.md)
4. **会话建立触发**（运行时事件，非配置）：UE→AMF→SMF NAS → SMF N4 PFCP → UPF；无需 UNC 配置
5. **会话修改/释放**（运行时事件）：UE/PCF/AMF/SMF 发起；无需 UNC 配置

## 约束

- **无需 License**（info）：本特性无需获得 License 许可，无需配置即可使用。
- **C-U 面接口偶联必须就绪**（critical）：所有会话信令交互的前提是 N4（5G）/Sxb（4G CU 分离）/Gn（2-3G）链路已建立 — 链路建立归 [UNC 1-00022](task/UNC/20.15.2/1-00022.md) 被依赖特性。
- **用户面网元（UPF）已部署**（critical）：AMF/SMF 等控制面与 UPF 偶联完成。
- **用户签约数据就绪**（critical）：UDM/HSS/HLR 已配置用户签约（DNN/APN、QoS、SSC mode、静态地址等）— 归 [WSFD-010400](feature/UNC/20.15.2/WSFD-010400.md)。
- **接入网就绪**（info）：(R)AN/eNodeB/5G-RAN 已部署并提供空口承载。
- **PDU 会话数量控制**（info）：可通过 `SET SMPDUCTRL`（5G）/计费域特性 控制单个 DNN 最大 PDU 会话数。
- **承载数限制（4G/2-3G）**（warning）：每个用户可以激活的最大承载数为 **11**（无线侧限制）。
- **PDN 连接-承载关系（4G）**（warning）：同一 PDN 连接必须且只能建立一个缺省承载；缺省承载删除则 PDN 连接释放。
- **APN-PDN 一一对应（4G）**（warning）：一个 PDN 由一个 APN 代表；不同 APN 连接不同 PDN。
- **首次发布版本**（warning）：5G N4 接口首次发布版本为 UNC **20.0.0**；4G S1/S5/S8 控制在 UNC 早于 20.0.0；2/3G Gn/Gp 控制在 UNC 早于 20.0.0—— 配置 5G PDU 会话需确认网元版本 ≥ 20.0.0。

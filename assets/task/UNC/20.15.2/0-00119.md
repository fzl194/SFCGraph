# MOD CHGPLMNCFG 原子任务
> 命令静态知识见 [MOD CHGPLMNCFG](command/UNC/20.15.2/MOD-CHGPLMNCFG.md)。本页只讲调测/运维实例化时这条命令**怎么用**。

适用网元：SGSN。本命令修改已存在的 PLMN 计费配置条目，按 PLMN（MCC+MNC）维度落实 S-CDR 生成策略（按周期/流量/计费条件变更/位置更新），是 SGSN 离线计费参考信息（WSFD-011201）中 PLMN 维度计费配置的修改入口。

## 配置方法
按"用户范围 + MCC + MNC + APN"定位条目，修改 S-CDR 生成策略各开关与阈值。

| 参数 | 配置方法 | 取值样例 |
|---|---|---|
| SUBRANGE | 用户范围（必选） | `ALL_USER` |
| MCC | 移动国家码（条件必选，3 位十进制） | `460` |
| MNC | 移动网号（条件必选，2 或 3 位十进制） | `00`、`06` |
| APNNI | APN（必选，1~62 字符） | `cmnet` |
| SP | 生成 S-CDR | `NO`/`YES` |
| SPP | 周期生成 S-CDR | `FORBID`（不生成） |
| SPL | S-CDR 生成周期（0~1440 min） | `60` |
| SVP | 流量生成 S-CDR | `NO`/`YES` |
| SVL | S-CDR 流量阈值（0~1000000 KB） | `1024` |
| SCCP | 计费条件变更生成 S-CDR | `NO`/`YES` |
| SCCL | 最大计费条件变更次数（0~10） | `5` |
| SLCP | 位置更新生成 S-CDR | `NO`/`YES` |

## 决策点

### S-CDR 生成策略组合（周期 / 流量 / 计费条件变更 / 位置更新）
与 [MOD CHGIMSICFG](task/UNC/20.15.2/0-00118.md) 同构：S-CDR 生成有四个并列触发维度（SPP 周期、SVP 流量、SCCP 计费条件变更、SLCP 位置更新），每个维度开关 + 阈值。具体启用哪些触发维度由话单规划决定，存在跨场景取值分叉。

### IMSI 维度 vs PLMN 维度选择
notes 指出：本命令与 ADD CHGIMSICFG 各字段默认值都为无效值，按 PLMN ID 控制策略（SET CHGCDR 的 PLMNCTRL）取值决定走 PLMN 维度（CHGPLMNCFG）还是 IMSI 维度（CHGIMSICFG）。当 PLMNCTRL=LOCINFO 时走 PLMN 维度。该选择由 [SET CHGCDR](task/UNC/20.15.2/0-00159.md) 驱动。

## 约束
- **仅对新激活用户生效**：配置立即生效，但只影响新激活用户的话单生成策略。
- **默认值为无效值**：各字段默认值都为无效值，需显式规划；不规划的字段不会生效。本命令与 ADD CHGIMSICFG 默认值均为无效值，若需使用需显式赋值。
- **条目存在性前置**：MOD 修改前需先 ADD CHGPLMNCFG 建立条目；可用 LST CHGPLMNCFG 校验。
- **MCC 格式**：3 位十进制；**MNC 格式**：2 或 3 位十进制。
- **SPL 取值范围**：0~1440 min；SVL：0~1000000 KB；SCCL：0~10。
- **PLMNCTRL 联动**：PLMNCTRL=LOCINFO 时走本命令，否则走 CHGIMSICFG。

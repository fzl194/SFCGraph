# MOD APNCHGMODE 修改APN计费接口选择方式（MOD APNCHGMODE）

> 命令静态知识见 [MOD APNCHGMODE](command/UNC/20.15.2/MOD-APNCHGMODE.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法

按配置维度组织如下"字典"：

| 维度 | 参数 | 取值 | 默认 | 作用 | 适用场景 |
|---|---|---|---|---|---|
| APN 名称 | `APN` | 1~63 字符（`-`/数字/大小写字母，首尾仅字母数字） | 必选 | 指定 APN | 所有场景必选 |
| 终端/接入类型 | `TMACCTYPE` | 枚举值（如 `UE5G_RAT4G`） | 必选 | 指定终端和接入类型 | 所有场景必选，决定 PDN 激活场景下的计费接口 |
| 强制计费接口 | `FORCED` | GaGyMode / NchfMode | — | 强制指定计费接口轨道，对所有 Serving Node 生效 | 融合计费主路径 |
| SGW 计费模式 | `FORSGWONLY` | GaGyMode / NchfMode | — | SGW 单独指定计费模式 | 按网元角色细分 |
| V-SMF 计费模式 | `FORVSMFONLY` | GaGyMode / NchfMode | — | V-SMF 单独指定计费模式 | 按网元角色细分 |
| I-SMF 计费开关 | `ISMFCHGSW` | ENABLE / DISABLE | DISABLE | I-SMF 变更场景下是否对该 APN 计费 | 仅漫游 I-SMF 组网考虑 |
| 5GS 互操作指示 | `BY5GSIWKI` | True / False | — | 发生 5GS 互操作时按指示切换 | 按 5GS 互操作指示选择接口 |
| 5GC 无限制接入标识 | `BY5GCNRI` | True / False | — | 按 5GC 无限制接入标识选择 | 按 5GC NRI 选择接口 |

### 典型脚本

```mml
MOD APNCHGMODE:APN="cmnet",TMACCTYPE="UE5G_RAT4G",FORCED="NchfMode";
MOD APNCHGMODE:APN="cmnet",TMACCTYPE="UE5G_RAT4G",FORCED="GaGyMode";
MOD APNCHGMODE:APN="cmnet",TMACCTYPE="UE5G_RAT4G",FORSGWONLY="GaGyMode";
MOD APNCHGMODE:APN="cmnet",TMACCTYPE="UE5G_RAT4G",FORVSMFONLY="NchfMode";
```

### 步骤位置

在全局计费模式（SET CHGMODE）配置完成后，按 APN 粒度调整具体计费接口轨道；融合计费场景（WSFD-011206）典型动作为将指定用户的 `FORCED` 改为 `NchfMode` 切换到 N40 接口。修改后只对新激活用户生效，已激活用户需重激活或等待自然重连。

## 决策点

### 决策点：计费接口选择维度（DP 0-01264）

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| 强制指定 | `FORCED`=`GaGyMode` 或 `NchfMode` | 对所有 Serving Node 生效；融合计费主路径（切 N40） |
| 按 5GS 互操作指示 | `BY5GSIWKI`=True/False + 对应模式 | 发生 5GS 互操作时按指示切换 |
| 按 5GC 无限制接入标识 | `BY5GCNRI`=True/False + 对应模式 | 按 5GC NRI 选择接口 |
| SGW 计费模式单独指定 | `FORSGWONLY`=`GaGyMode`/`NchfMode` | 按网元角色细分 |
| V-SMF 计费模式单独指定 | `FORVSMFONLY`=`GaGyMode`/`NchfMode` | 按网元角色细分 |
| I-SMF 计费开关 | `ISMFCHGSW`=ENABLE/DISABLE | I-SMF 变更场景下是否计费；仅漫游 I-SMF 组网考虑 |

### 决策点：终端与接入类型（DP 0-01265）

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| 5G 终端 4G 接入 | `TMACCTYPE=UE5G_RAT4G` | 5G 终端通过 4G 接入场景下的 PDN 激活 |
| 其他枚举值 | 按实际终端/接入组合 | 决定 PDN 激活场景下的计费接口 |

## 约束

· rule-0-01596 · severity=critical · `APN` 必选，1~63 字符（`-`/数字/大小写字母，首尾仅字母数字）。
· rule-0-01597 · severity=critical · `TMACCTYPE` 必选（指定终端和接入类型枚举）。
· rule-0-01598 · severity=high · 计费接口选择方式互斥：`FORCED`、`BY5GSIWKI`+对应模式、`BY5GCNRI`+对应模式为不同决策路径，按规划选其一，避免冲突。选错计费策略接口，未配置时计费流程失败，用户无法正常激活。
· rule-0-01599 · severity=medium · 只对新激活用户生效（已激活用户不重选接口）。变更需用户重激活或等待自然重连。
· rule-0-01600 · severity=medium · 适用 NF：SGW-C、PGW-C、SMF、GGSN。

# ADD CHGIMSICFG 原子任务

> 命令静态知识见 [ADD CHGIMSICFG](command/UNC/20.15.2/ADD-CHGIMSICFG.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

ADD CHGIMSICFG 是 UNC 提供的三种 S-CDR 话单生成策略之一（另两种为 ADD CHGPLMNCFG 基于 PLMN+APN NI、SET CHGCHAR 全局基于计费属性）。本命令用"IMSI 前缀 + APN NI + 拜访类型 + 计费属性"四元组定位一组用户，定制其 S-CDR 生成行为（是否生成、周期、流量阈值、条件变更/位置更新触发）。典型用途是抑制特定号段漫游用户对特定 APN NI 的话单。

激活前提：SET CHGCDR 的 PLMNCTRL=IMSI（取 IMSI 中的 PLMN ID）时，本策略才生效；若 PLMNCTRL=LOCINFO 则走 ADD CHGPLMNCFG。

**参数取值样例（抑制特定 IMSI 前缀用户话单范式）**

| 参数 | 取值样例 | 默认 | 说明 |
|---|---|---|---|
| SUBRANGE | `IMSI_PREFIX` / `ALL_USER` | — | 用户范围（必填）；IMSI_PREFIX=按前缀定位 |
| IMSIPRE | `12303` | — | IMSI 前缀，5~15 位；SUBRANGE=IMSI_PREFIX 时必填 |
| VISITTYPE | `ROAMING-1` | ROAMING&VISITING | 拜访类型 |
| APNNI | `huawei.com` | — | APN NI，1~62 字符（必填） |
| CC | `NORMAL_BILLING-1` | HOT_BILLING&FLAT_RATE&PREPAID | 计费属性 |
| SP | `NO` | DFT | 生成 S-CDR；NO=不生成（抑制），DFT=走缺省策略 |
| SPP | `FORBID` / `DFT` | DFT | 周期生成 S-CDR |
| SPL | `30` | 0 | S-CDR 生成周期(min)，0~1440 |
| SVP | `NO` / `DFT` | DFT | 流量生成 S-CDR |
| SVL | `1000` | 0 | 生成 S-CDR 流量阈值(KB)，0~1000000 |
| SCCP | `NO` / `DFT` | DFT | 计费条件变更生成 S-CDR |
| SCCL | `3` | 0 | 最大计费条件变更次数，0~10 |
| SLCP | `NO` / `DFT` | DFT | 位置更新生成 S-CDR |

**典型行（抑制 12303 前缀漫游用户对 huawei.com 的话单）**

```
ADD CHGIMSICFG: SUBRANGE=IMSI_PREFIX, IMSIPRE="12303", VISITTYPE=ROAMING-1,
  APNNI="huawei.com", CC=NORMAL_BILLING-1, SP=NO;
```

**前置（典型流程）**：
```
SET CHGPLMNCHAR: PLMN=VPLMN, SP=YES;   # 允许 VPLMN 生成话单
SET CHGCDR: PLMNCTRL=IMSI;             # 切到 IMSI 来源，本策略才生效
ADD CHGIMSICFG: ...;                    # 再细化抑制
```

## 决策点

### 话单生成策略选哪一层

UNC 三种 S-CDR 策略互斥于不同定位维度，由 SET CHGCDR 的 PLMNCTRL 决定走哪条：

| 策略层 | 定位维度 | 命令 | 生效条件 |
|---|---|---|---|
| IMSI 细化 | IMSI 前缀 + APN NI + 拜访类型 + 计费属性 | ADD CHGIMSICFG | PLMNCTRL=IMSI |
| PLMN 细化 | PLMN(MCC+MNC) + APN NI | ADD CHGPLMNCFG | PLMNCTRL=LOCINFO |
| 全局 | 计费属性 | SET CHGCHAR | 始终 |

### 各触发开关填 DFT 还是显式 NO/YES

SP/SPP/SVP/SCCP/SLCP 五个开关默认 DFT（走缺省策略），显式填 NO/FORBID 表示抑制该类话单。抑制场景需把这些开关显式置 NO。

| 选项 | 取值 | 含义 |
|---|---|---|
| 走缺省策略 | DFT | 不在本层覆盖，沿用上层（SET CHGCHAR）配置 |
| 抑制生成 | NO / FORBID | 该维度不生成 S-CDR |

## 约束

- **立即生效，但只对配置生效后激活的用户有效** · severity=MEDIUM · 已激活用户不立即受影响。
- **本表最大 2048 条记录** · severity=MEDIUM · 超过上限新增失败。
- **生效前置**：仅当 SET CHGCDR 的 PLMNCTRL=IMSI 时本策略生效；PLMNCTRL=LOCINFO 时走 ADD CHGPLMNCFG · severity=HIGH · 配错前置会导致策略不生效。
- **字段默认值均为"无效值"**：与 ADD CHGPLMNCFG 一致，未显式填的字段按 DFT 走缺省策略而非覆盖 · severity=MEDIUM。
- **上层闸门**：若 SET CHGPLMNCHAR 禁止生成某种话单，则无论本命令如何配置都不生成 · severity=HIGH。

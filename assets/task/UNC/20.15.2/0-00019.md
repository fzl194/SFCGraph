# ADD CHGHOLI 原子任务

> 命令静态知识见 [ADD CHGHOLI](command/UNC/20.15.2/ADD-CHGHOLI.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

ADD CHGHOLI 登记一条"计费节假日"。节假日被识别为特殊费率时段，与 ADD CHGTARI（费率时段配置）配合，使 S-CDR 话单在节假日采用不同的费率。配置时按整网规划给出年/月/日三个日期字段，并指定该节假日适用的计费属性 CC。

**参数取值样例（节假日费率切换范式）**

| 参数 | 取值样例 | 默认 | 说明 |
|---|---|---|---|
| YEAR | `2010` | — | 年，1998~2099（必填） |
| MONTH | `10` | — | 月，1~12（必填） |
| DAY | `1` | — | 日，1~31（必填） |
| CC | `NORMAL` | — | 计费属性（必填）；NORMAL=普通计费 |

**典型行（登记一个普通计费属性的国庆节假日）**

```
ADD CHGHOLI: YEAR=2010, MONTH=10, DAY=1, CC=NORMAL;
```

**配置位置**：在"配置离线计费费率切换（SGSN）"流程中作为第 2 步可选项（可选地增加节假日配置），其后常跟 SET CHGWKDY 设置星期配置。

## 决策点

### 是否需要配置节假日费率

| 选项 | 适用 |
|---|---|
| 配置节假日 + 费率时段 | 需对节假日（如国庆、春节）采用特殊费率，与 ADD CHGTARI 联动 |
| 仅配星期/日常费率时段 | 无节假日差异化需求，只靠 SET CHGWKDY + ADD CHGTARI |

## 约束

- **立即生效** · severity=LOW。
- **同一计费属性节假日总数不超过 50，本表总数不超过 200** · severity=MEDIUM · 超过上限新增失败。
- **唯一性**：年 + 月 + 日 + 计费属性 四者唯一确定一条记录 · severity=HIGH · 重复登记新增失败。
- **联动依赖**：节假日需与 ADD CHGTARI（费率时段配置）配合才能实现费率切换，单独配节假日不产生费率效果 · severity=MEDIUM。

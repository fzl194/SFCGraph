# RMV CHARGECHAR 原子任务
> 命令静态知识见 [RMV CHARGECHAR](command/UNC/20.15.2/RMV-CHARGECHAR.md)。本页只讲调测/运维实例化时这条命令**怎么用**。

适用 NF：SGW-C、PGW-C、SMF。本命令删除已存在的计费属性条目，是热计费功能（WSFD-011202）中计费属性管理的删除入口，与 ADD/MOD/LST CHARGECHAR 构成完整 CRUD。

## 配置方法
按计费属性名称定位条目删除。

| 参数 | 配置方法 | 取值样例 |
|---|---|---|
| CCNAME | 计费属性名称（必选，本地规划，1~31 字符） | `cc1` |

典型用法：计费属性停用或重建时，先用 LST CHARGECHAR 校验存在性，再用本命令按名称删除；如需重建则删后用 ADD CHARGECHAR 重新创建。

## 决策点

本命令仅一个必选参数 CCNAME，获取方法稳定（本地规划）、取值单值，无双值/跨场景分叉，无独立决策点。是否删除由业务计费属性停用决策决定。

## 约束
- **立即生效**：命令执行后立即生效。
- **条目存在性前置**：删除前条目需存在（先 ADD CHARGECHAR 建立），否则删除失败；可用 LST CHARGECHAR 校验。
- **引用影响**：删除前应确认无 RULE/策略引用该 CCNAME，避免残留引用导致配置不一致。
- **CCNAME 命名**：1~31 字符。

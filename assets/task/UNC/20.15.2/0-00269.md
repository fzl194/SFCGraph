# 删除GRE隧道（RMV GRETUNNEL）

> 命令静态知识见 [RMV GRETUNNEL](command/UNC/20.15.2/RMV-GRETUNNEL.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

该命令删除一个已存在的 GRE 隧道（由 `ADD GRETUNNEL` 建立），用于网络重新规划、隧道不再使用的场景。配置空间仅需定位参数。

### 配置维度1：隧道定位（参数 TNLNAME + TNLTYPE）

| 参数 | 作用 |
|---|---|
| TNLNAME | 指定要删除的 GRE 隧道接口名称（必须是 `ADD GRETUNNEL` 已建的） |
| TNLTYPE | 指定隧道类型（gre / gre6，需与建隧道时一致） |

**步骤位置**：在所属 feature（IPFD-015002 GRE）去激活编排中执行。

## 决策点

（无。本命令为按需删除，无特性级分支。）

## 约束

- **定位参数必填**（critical）：TNLNAME + TNLTYPE 是必选参数 — 后果：缺一无法定位
- **隧道必须已存在**（critical）：TNLNAME 必须是 `ADD GRETUNNEL` 已建的隧道 — 后果：删除失败

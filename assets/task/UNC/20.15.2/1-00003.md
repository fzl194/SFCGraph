# OFCTemplate 绑定 APN

> 阶段定位：离线计费参数绑到 APN 层（优先级次于 UserProfile）。被引用于：[2-00001 离线计费](task/UNC/20.15.2/2-00001.md)。
> 域通用 compound（在线/融合计费模板绑 APN 同此结构）。

## 配置方法

命令序列（建 APN → 绑模板）：

1. [ADD APN](task/UNC/20.15.2/0-00004.md) — APN 实例（APN 全网规划）；已配则跳过
2. [SET APNCHARGECTRL](task/UNC/20.15.2/0-00149.md) — APN 绑 OFCTemplate（PGWOFCTEMPLATE/SGWOFCTEMPLATE 引用模板名）

**典型脚本**
```
ADD APN: APN="apn-test";
SET APNCHARGECTRL: APN="apn-test", PGWOFCTEMPLATE="offlinecharge-test", SGWOFCTEMPLATE="offlinecharge-test";
```

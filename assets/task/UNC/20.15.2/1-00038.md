# UNC 位置区域 DNS 族

> **UNC 域对等网元选择骨架**（SOP §3.5 特性专属）。被引用于 **WSFD-010202**（基于位置区域的对等网元选择）。
>
> **触发场景**：减少对等网元（SGSN/MME/MSC）DNS 解析配置记录——基于 LAI/RAI/TAI 或区域名称（ZONE）定制 DNS 域名。适用 2G/3G SGSN + 4G MME + MSC（SRVCC 流程）。

## 配置方法

| 步骤 | 命令 | 关键参数 |
|---|---|---|
| 1. （可选）Hostfile A 解析记录 | `ADD IPV4DNSH` | HOSTNAME（格式：`ZONE.MNCxxx.MCCxxx.GPRS` / `TOPON.MME-S10.MMECxx.MMEGIxxx.MME.EPC.MNCxxx.MCCxxx.3GPPNETWORK.ORG`），IPV4ADDR |
| 2. （可选）NAPTR 解析记录 | `ADD DNSN` | HOSTNAME, PRIORITY, WEIGHT, REPLACEMENT |
| 3. 位置区域 DNS 域名策略（核心）| `ADD AREADNS` | DNTYPE=SGSN_RAI/SGSN_LAI/MME_TAI/MSC_LAI/MSC_RAI（网元类型 × 区域类型）；LAC+RACRANGE / TAC+TACRANGE（范围）；ZONESW=YES+ZONENAME=ZONE（区域名称模式）|
| 4. （按需）MSC 选择策略（SRVCC 流程）| `SET MSCSELPLCY` | MSCSELPLCTYPE=RAI_FQDN/LAI_FQDN |
| 5. （按需）SGSN 域名策略后缀匹配 | `ADD SGSNDNS` | SGSNDNSPOLICY=本端域名后缀 |

**典型脚本（SGSN_RAI × 区域名称 ZONE）**：
```
ADD AREADNS: DNTYPE=SGSN_RAI, LAC="0x0001", RAC="0x01", RACRANGE="0x05", ZONESW=YES, ZONENAME="ZONE";
```

**步骤位置**：所有对等网元选择特性的核心——前置仅在与其他 APN 域特性同场景时按需先建 [UNC 1-00024](task/UNC/20.15.2/1-00024.md)，对等网元选择不依赖 APN 接入域。

## 场景差异

- **[WSFD-010202](task/UNC/20.15.2/2-00037.md) 基于位置区域的对等网元选择**：步骤 1-2 可选（按运营商 DNS 配置情况）+ 步骤 3 核心 + 步骤 4 MSC 场景必选 + 步骤 5 SGSN 后缀按需

## 决策点

| NF 类型 × 区域类型 | 走法 |
|---|---|
| SGSN × RAI × 起始 RAC（默认）| 步骤 3 `ADD AREADNS` DNTYPE=SGSN_RAI |
| SGSN × RAI × 区域名称 ZONE | 步骤 3 `ADD AREADNS` +ZONESW=YES |
| SGSN × LAI × 起始 LAC | 步骤 3 `ADD AREADNS` DNTYPE=SGSN_LAI |
| MME × TAI × 起始 TAC | 步骤 3 `ADD AREADNS` DNTYPE=MME_TAI |
| MSC × LAI × 区域名称 ZONE（SRVCC 典型）| 步骤 3 + 步骤 4 |
| MSC × RAI × 区域名称 ZONE | 步骤 3 DNTYPE=MSC_RAI +ZONESW + 步骤 4 |

## 约束

- **★ 5 个命令 atom 均待补**：本骨架 5 条命令 UNC pre-build 阶段**未建 atom**（_numbering.json 未收录），**命令真实存在 UNC OM 文档**—— 命令文档路径：

  - `ADD DNSN`：`.../DNS NAPTR管理/增加DNS NAPTR记录(ADD DNSN)_72225569.md`
  - `ADD AREADNS`：`.../GnGp-SGSN_S10_S16_S3接口管理/位置域名管理/增加位置区域DNS域名(ADD AREADNS)_72345559.md`
  - `SET MSCSELPLCY`：`.../Sv接口管理/MSC选择策略/设置MSC选择策略(SET MSCSELPLCY)_26305786.md`

- **AREADNS 必先建**（critical）：网元发起对等网元选择前必须先 `ADD AREADNS` 配置位置区域 DNS 域名策略 — 后果：未配置致对等网元查询失败
- **区域范围必填**（critical）：区域类型（LAI/RAI/TAI）必填**起始码 + 范围**（LAC+RACRANGE / TAC+TACRANGE） — 后果：范围未填致全网范围 DNS 查询无法收敛
- **ZONESW 与 ZONENAME 联动**（critical）：启用 ZONESW=YES 时**强制必填** ZONENAME — 后果：未填致命令失败

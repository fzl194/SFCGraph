# 配置缺省主用PCRF（SET MASTERPCRF）

> 命令静态知识见 [SET MASTERPCRF](command/UNC/20.15.2/SET-MASTERPCRF.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法

SET MASTERPCRF 是 PCRF 组内缺省主用 PCRF 主机名的设置命令，**仅当 PCRF 组工作模式为主备模式时才生效**。本命令为可选配置，前置依赖：PCRF 组与 PCRF-组绑定关系已通过 [ADD PCRFGROUP](task/UNC/20.15.2/0-00059.md)、[ADD PCRFBINDGRP](task/UNC/20.15.2/0-00058.md) 建好。本命令有 2 个配置维度：**PCRF 组名（定位目标组）**、**主用 PCRF 主机名（组内选择主用）**。

### 配置维度 1：PCRF 组名（参数 PCRFGRPNAME，必选）

| 维度 | 参数 | 取值 | 默认 | 作用 | 适用场景 |
|---|---|---|---|---|---|
| PCRF 组名 | `PCRFGRPNAME` | 字符串 1~128 字符，不支持空格不区分大小写 | 无 | 指定目标 PCRF 组 | 取自已配置数据（必须先建 PCRFGROUP） |

> 数据来源：本端规划（运营商规划）。必须为系统内已存在的 PCRF 组名，可通过 `LST PCRFGROUP` 查询。

### 配置维度 2：主用 PCRF 主机名（参数 MASTERPCRF，必选）

| 维度 | 参数 | 取值 | 默认 | 作用 | 适用场景 |
|---|---|---|---|---|---|
| 主用 PCRF | `MASTERPCRF` | 字符串 1~127 字符，不支持空格不区分大小写 | 无 | 该组缺省主用 PCRF 主机名 | 取自本端规划，必须为该组已绑定的 PCRF 成员之一 |

> 数据来源：本端规划。`MASTERPCRF` 必须是该组已通过 [ADD PCRFBINDGRP](task/UNC/20.15.2/0-00058.md) 绑定的 PCRF 成员，否则配置失败。

**典型脚本**

```
# 修改某 PCRF 组的缺省主用 PCRF
SET MASTERPCRF: PCRFGRPNAME="pcrf_group_2", MASTERPCRF="pcrf_3";
```

**步骤位置**：Gx Failover 功能流程的可选步骤。位于 [Gx-Failover功能](evidence/UNC/20.15.2/Gx-Failover功能_31422950.md) 流程"配置 PCRF 信息"步骤的第 c 子步（可选），在 [ADD PCRFGROUP](task/UNC/20.15.2/0-00059.md)（配置 PCRF 组，含主备模式 LOADBALANCEMODE=MASTER_SLAV）和 [ADD PCRFBINDGRP](task/UNC/20.15.2/0-00058.md)（绑定 PCRF 到组）之后执行；仅主备模式且需修改缺省主用 PCRF 时才配置。

## 决策点

### 决策点：PCRF 组工作模式是否为主备模式（DP 0-03103）

问：是否需要执行本命令？

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| 主备模式 | 需配置缺省主用 PCRF，本命令生效 | PCRFGROUP 的 `LOADBALANCEMODE=MASTER_SLAV` |
| 负载分担模式 | 无主备概念，本命令无意义，不执行 | PCRFGROUP 的 `LOADBALANCEMODE` 为非主备值 |

> 工作模式由 [ADD PCRFGROUP](task/UNC/20.15.2/0-00059.md) 的 `LOADBALANCEMODE` 参数决定，本命令不自带该开关。

### 决策点：主用 PCRF 选择（DP 0-03104）

问：该 PCRF 组内选哪台作主用？

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| 组内已绑定成员 | `MASTERPCRF` 取该组已绑定的 PCRF 主机名之一 | 必须是 [ADD PCRFBINDGRP](task/UNC/20.15.2/0-00058.md) 已绑定的成员，否则配置失败 |
| 未绑定主机 | 配置失败 | MASTERPCRF 不在该组绑定成员中，命令拒绝 |

> 取值为本端规划主机名（如 `pcrf_3`），须先完成 PCRFBINDGRP 绑定。

## 约束

- **前置（组内绑定）**（critical，rule-0-03256）：前置（组内绑定）：`MASTERPCRF` 必须已通过 [ADD PCRFBINDGRP](task/UNC/20.15.2/0-00058.md) 绑定到该 `PCRFGRPNAME` 组，否则配置失败或主用指向不存在的主机 — 后果：配置失败或主用指向不存在的主机
- **工作模式前提**（warning，rule-0-03257）：工作模式前提：本命令仅在 PCRF 组工作模式为主备模式（`LOADBALANCEMODE=MASTER_SLAV`）时生效；负载分担模式无主备概念，本命令无意义 — 后果：负载分担模式下配置不生效
- **规格限制**（warning，rule-0-03258）：规格限制：最大记录数 100，超规格新增失败 — 后果：超规格新增失败
- **生效与查询**（info，rule-0-03259）：生效与查询：执行后立即生效；本命令无独立查询，设定结果需通过 `LST PCRFGROUP` 查看 — 后果：立即生效，需借 LST PCRFGROUP 验证
- **参数格式**（info，rule-0-03260）：参数格式：`PCRFGRPNAME`/`MASTERPCRF` 均不支持空格、不区分大小写 — 后果：含空格或大小写不一致需按规则归一化

# DSP CPPDPCHGINFO 原子任务

> 命令静态知识见 [DSP CPPDPCHGINFO](command/UNC/20.15.2/DSP-CPPDPCHGINFO.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。


## 配置方法

本命令为**只读查询**，`SUBSCRIBERID` 决定用哪种用户标识定位，对应标识字段联动：

| 用户标识类型 | `SUBSCRIBERID` | 联动参数 | 取值 | 典型脚本 |
|---|---|---|---|---|
| 按 IMSI | `IMSI` | IMSI | 1~15 位字符串 | `DSP CPPDPCHGINFO:SUBSCRIBERID=IMSI,IMSI="460000123456789";` |
| 按 MSISDN | `MSISDN` | MSISDN | 1~15 位字符串 | `DSP CPPDPCHGINFO:SUBSCRIBERID=MSISDN,MSISDN="13900000000";` |
| 按 IPv4 | `IPv4` | IPV4 | IPv4 地址 | `DSP CPPDPCHGINFO:SUBSCRIBERID=IPv4,IPV4="192.168.1.1";` |
| 按 IPv6 | `IPv6` | IPV6 | X:X:X:X:X:X:X:X | `DSP CPPDPCHGINFO:SUBSCRIBERID=IPv6,IPV6="::1";` |

**可选精度提升**：带 `NSAPI`（1~63）可精确定位到单条 PDP 承载；不带则查该用户所有 PDP 上下文。

**典型用途（故障恢复定位）**：GGSN/PGW-C 计费接口故障恢复后，用本命令查询被放通用户的归属信息，决定后续处理（转离线或继续放通）。相比 `SET FHBYPASS` 的 `HOLDINGTIME` 全局控制，本命令可精准定位单个放通用户。

**配额用量语义（重要）**：查询到的用户配额用量是 SMF/PGW-C **已收集但未上报**的部分——既非用户上线以来总流量，也非上次上报以来总流量，而是**上次上报计费中心后、从 UPF/PGW-U 获取且尚未上报**的流量。

**典型脚本**（按 IMSI 查所有承载）：
```
DSP CPPDPCHGINFO:SUBSCRIBERID=IMSI,IMSI="231011223344551";
```

**典型脚本**（按 IMSI + NSAPI 精确定位单承载）：
```
DSP CPPDPCHGINFO:SUBSCRIBERID=IMSI,IMSI="231011223344551",NSAPI=5;
```

## 决策点

### 决策点：用户标识类型选择（DP 0-01248）

问：用哪种标识定位目标用户？

| 选项 | `SUBSCRIBERID` | 联动字段 | 适用场景 |
|---|---|---|---|
| IMSI | `IMSI` | IMSI（1~15 位） | 按用户身份定位（拨测、精确归属核查） |
| MSISDN | `MSISDN` | MSISDN（1~15 位） | 按号码定位（终端侧定位） |
| IPv4 | `IPv4` | IPV4 | 按 IPv4 地址定位（按会话/IP 定位） |
| IPv6 | `IPv6` | IPV6（冒号分十六进制） | 按 IPv6 地址定位 |

> `SUBSCRIBERID` 为 required 枚举，取值决定联动哪个标识字段，atom 绑定为 decision_driven（标识维度分叉）。

### 决策点：是否带 NSAPI 精确到承载（DP 0-01249）

问：是否用 `NSAPI`（1~63）进一步定位到具体 PDP 承载？

| 选项 | `NSAPI` | 适用场景 |
|---|---|---|
| 不带 NSAPI（optional） | 不填 | 查该用户所有 PDP 上下文（全局视图） |
| 带 NSAPI | 填值（1~63） | 精确定位单条 PDP 承载的计费上下文 |

## 约束

- **查询到的配额用量是**已收集未上报**部分**（info，rule-0-01556）：查询到的配额用量是**已收集未上报**部分，不是总流量，解读时需注意口径；后果：误读为总流量导致计费核账错误
- **SUBSCRIBERID 为 required**（warning，rule-0-01557）：SUBSCRIBERID 为 required，必须指定标识类型并配对应字段（conditional 互斥）
- **IMSI/MSISDN 为 1~15 位字符串**（info，rule-0-01558）：IMSI/MSISDN 为 1~15 位字符串；NSAPI 为 1~63 整数，optional
- **本命令为只读诊断**（info，rule-0-01559）：本命令为只读诊断，适用于 PGW-C、SMF 网元

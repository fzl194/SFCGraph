# 修改融合计费模板（CCT）配额与终结参数（MOD CCT）

> 命令静态知识见 [MOD CCT](command/UNC/20.15.2/MOD-CCT.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法

按配置维度组织，每维度表格（维度/参数/取值/默认/作用/适用场景）。先 `ADD CCT` 建模板（或复用系统默认 `global` 模板），再用 `MOD CCT` 按业务维度填参。模板分两层：①全局通用模板（名称 `global`，配 `QHT` 等基础参）；②业务级 CCT（如时长计费 `cct-test` 配 `QHT/TQT/VT`，流量计费配 `VQT`，可叠加 PDU/RG/QF 各级流量/时长阈值与漫游阈值）。

| 维度 | 参数 | 取值 | 默认 | 作用 | 适用场景 |
|---|---|---|---|---|---|
| 全局通用模板（基础配额空闲时间） | `CCTMPLTNAME=global` + `QHT` | `QHT` 0 或 5~3600 秒 | 无 | 配额空闲时间门限，超时触发配额更新 | 通用基础配置 |
| 业务时长计费模板 | `CCTMPLTNAME` + `QHT/TQT/VT` | `QHT` 5~3600s；`TQT` 0~50%；`VT` 0 或 5~1440min | 无 | 时长配额触发百分比（QHT 触发+TQT 百分比+超后时长阈值） | 时长计费场景 |
| 业务流量计费模板 | `CCTMPLTNAME` + `VQT` | `VQT` 0~50% | 无 | 流量配额触发百分比 | 流量计费场景 |
| 业务事件计费模板 | `CCTMPLTNAME` + `UQT` | `UQT` 0~50% | 无 | 事件配额触发百分比（事件阈值触发百分比） | 事件计费场景 |
| 漫游用户独立阈值 | `CCTMPLTNAME` + `ROAMVOLUMELIMIT`/`ROAMFUATERMACT` | `ROAMVOLUMELIMIT` 0~2147483647 MB；`ROAMFUATERMACT` 枚举 | 无 | 漫游用户独立配额限制与终结方式 | 漫游场景必填 |
| PDU 会话级阈值 | `PDUVOLUMELIMIT`/`PDUTIMELIMIT` | 0~2147483647 MB / 0~1440min | 无 | PDU 会话级流量/时长阈值 | 按需 |
| RG 业务级阈值 | `RGVOLUMELIMIT`/`RGTIMELIMIT` | 0~2147483647 MB / 0~1440min | 无 | Rating Group 级流量/时长阈值 | 按需 |
| QF 级阈值 | `QFVOLUMELIMIT`/`QFTIMELIMIT` | 0~2147483647 MB / 0~1440min | 无 | Quota Holding Key 级流量/时长阈值 | 按需 |
| QHT 上报触发 | `QHTRPTTRIGGER`/`QHTEXPIREDRSU`/`NOQUOTATRIGGER` | `QHT`/`WITH_RSU`/`ENABLE` | 无 | QHT 超时触发时 Nchf 消息容器 Trigger 内容与 RSU 携带规则 | 按需 |

典型脚本：

```
MOD CCT:CCTMPLTNAME="global",QHT=5;
MOD CCT:CCTMPLTNAME="cct-test",QHT=40,TQT=30,VT=40;
MOD CCT:CCTMPLTNAME="cct-test",VQT=30;
MOD CCT:CCTMPLTNAME="cct-event",UQT=20;
MOD CCT:CCTMPLTNAME="cct-roam",ROAMVOLUMELIMIT=1024,ROAMFUATERMACT="TERM_SERVICE";
```

## 决策点

### 决策点：阈值触发百分比维度（流量/时长/事件多值）（DP 0-01266）

问：模板按哪种计量维度触发配额更新？

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| 流量 | `VQT`（0~50%） | 流量计费场景 |
| 时长 | `TQT`（0~50%） | 时长计费场景 |
| 事件 | `UQT`（0~50%） | 事件计费场景 |
| 三者同模板并存 | `VQT`+`TQT`+`UQT` 同时配置 | 融合计费普遍同时配流量+时长 |

### 决策点：配额到期/耗尽终结方式（DP 0-01267）

问：最终配额动作指示终结方式（`FUATERMINATE`）怎么处理？

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| `TERM_SERVICE`（阻塞业务） | `FUATERMACT="TERM_SERVICE"` | 配额耗尽阻塞用户业务 |
| 漫游用户独立设置 | `ROAMFUATERMACT`（conditional） | 漫游场景必填 |

## 约束

- **CCTMPLTNAME 必选**（critical，rule-0-01601）：`CCTMPLTNAME` 必选，1~63 字符，不区分大小写；模板不存在则修改失败。

- **阈值取值范围**（warning，rule-0-01602）：阈值取值范围：`QHT` 0 或 5~3600 秒；`VQT/TQT/UQT` 0~50%；`VT` 0 或 5~1440 分；`PDUVOLUMELIMIT/RGVOLUMELIMIT/QFVOLUMELIMIT/ROAMVOLUMELIMIT` 0~2147483647 MB（1MB=1048576 字节）；`PDUTIMELIMIT/RGTIMELIMIT/QFTIMELIMIT` 0~1440 分；超规格参数被拒。

- **漫游用户相关参数（ROAMFUATERMACT/ROAMVOLU**（warning，rule-0-01603）：漫游用户相关参数（`ROAMFUATERMACT`/`ROAMVOLUMELIMIT`）为 conditional，漫游场景必填。

- **适用 NF**（info，rule-0-01604）：适用 NF：PGW-C、SMF；非适用 NF 执行无效。

- **配套前置**（info，rule-0-01605）：配套前置：DCC 在线计费模板/离线计费模板需先建好（参见激活 Gy 在线计费、激活离线计费、激活融合计费），本命令在其上修改融合相关参数。

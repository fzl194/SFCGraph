# 配置话单字段模板（ADD CDRFIELDTEMP）

> 命令静态知识见 [ADD CDRFIELDTEMP](command/UNC/20.15.2/ADD-CDRFIELDTEMP.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法

话单字段模板是 50+ 个 CDR 字段的使能开关表。绝大多数字段默认 `ENABLE`，少数（IMSIUNAUTHFLAG/PDNCONNECTIONID/PGWADDRESSUSED/PGWPLMNID/RECORDEXTENTION/STARTTIME/STOPTIME 等）默认 `DISABLE`。实际配置时**只显式改与默认不同的字段**，其余走默认即可。模板建好后通过 `SET GLBCDRFLDTEMP`（全局模板）或 `SET OFCCDRPARA` 关联到话单生成路径。

### 配置维度 1：字段使能开关（ENABLE / DISABLE）

按字段语义分组，便于按需开关：

| 分组 | 字段示例 | 默认值 | 说明 |
|---|---|---|---|
| 用户/接入标识 | MSISDN、SERVEDIMEISV、APNNETWORKID、APNSELECTMODE、PDPTYPE、SGSNSGWPLMNID、SERVEDPDPADDR、DYNADDRFLAG | ENABLE | 基础用户/会话信息 |
| 位置/接入类型 | ULI、RATTYPE、MSTIMEZONE、THREEGPP2USRLOC、UWANULI | ENABLE | 双端支持才生效（见约束） |
| PGW/节点信息 | NODEID、PGWADDRESSUSED、PGWADDRESSIPV6、PGWPLMNID、SGWADDRESSIPV6 | PGWADDRESSUSED/PGWPLMNID 默认 DISABLE | PGW 特定字段 |
| 扩展字段开关 | RECORDEXTENTION | DISABLE | 条件必选；开启时需配 RECORDEXTTYPE |
| 运营商扩展类型 | RECORDEXTTYPE | CCB（当 RECORDEXTENTION=ENABLE 时） | 决定扩展字段内容来源 |
| 序列/时间 | RECSEQNUMBER、LOCALRECSEQNUM、STARTTIME、STOPTIME | STARTTIME/STOPTIME 默认 DISABLE | 会话起止时间 |
| 流量/业务 | LOTV、LOSD、IMSSIGCONTEXT、PSFCI、DIAGNOSTICS、NETINITPDPCONT | ENABLE | 业务用量明细 |
| CIoT/NB-IoT 专用 | CPEPSOINDICATOR、UNIPDUCPONLY、PDPTYPEEXT、SGIPTPTUNMETH、APNRATECTRL、SRVPLMNRATECTRL、LOWPRIIND、MOEXCPTDATA | DISABLE | NB-IoT/信令面优化 |
| 5G/NSA | SNSSAI、SSCMODE、SECRATUSAGE | DISABLE | 5G 特有字段 |

> **实际配置原则**：只改与默认不同的字段。例如"用户属性反射"场景只需 RATTYPE/ULI/MSTIMEZONE=ENABLE，其余全部走默认。

### 配置维度 2：RECORDEXTTYPE 扩展类型（条件必选，当 RECORDEXTENTION=ENABLE）

| 取值 | 扩展字段内容 | 适用场景 |
|---|---|---|
| CCB | 内容计费（第一个 URL + 使用时长） | 内容计费业务话单 |
| HP | 高精度时间 | 时间敏感计费 |
| DT | Direct Tunnel 模式上下行流量及 RNC/NodeB 地址更新列表 | 3G direct tunnel 场景 |
| GLB | 携带业务报文数和 AAA 计费信息 | 全局通用扩展 |
| TAG | 携带 Transparent-Data 数据 | 自定义扩展 |
| CMHK | 携带 Gy 接口相关信息 | 香港运营商定制 |
| COMM | 携带通用字段（可含 SNSSAI/SSCMODE） | 5G 通用扩展 |

> RECORDEXTTYPE=GLB/DT 时，PGW 扩展字段是否携带 IP 层报文包数**不受 PacketNum 控制**（命令 wiki notes）。

**步骤位置**：在所属 feature 编排中通常位于"配置话单参数"步骤——OFCTEMPLATE 之后、绑定之前。

## 决策点

### 字段使能组合策略（DP 0-01003）

问：按什么意图选择需要显式 ENABLE/DISABLE 的字段组合？

| 选项 | 典型字段组合 | 适用场景 |
|---|---|---|
| 用户属性反射 | RATTYPE=ENABLE, ULI=ENABLE, MSTIMEZONE=ENABLE | 接入层位置信息需在话单携带 |
| NB-IoT 信令面优化 | CPEPSOINDICATOR=ENABLE, UNIPDUCPONLY=ENABLE | 信令面数据传输场景 |
| Non-IP 数据传输 | PDPTYPEEXT=ENABLE, SGIPTPTUNMETH=ENABLE | NB-IoT Non-IP PDN 类型 |
| 离线计费字段精简 | APNNETWORKID=DISABLE, MSTIMEZONE=DISABLE, RECORDEXTENTION=DISABLE | 减少话单冗余字段 |
| NSA 双栈支持 | SECRATUSAGE=ENABLE | 携带 RAN-SecondaryRAT-Usage-Report |

### 双端支持字段是否开启（DP 0-01004）

问：RATTYPE/ULI/MSTIMEZONE 等"双端支持"字段是否在话单中使能？

| 选项 | 取值 | 前提 |
|---|---|---|
| 双端均支持且话单需要 | ENABLE | 已与对端（SGSN/S-GW 与 PGW-C/SMF）确认支持该字段 |
| 对端不支持或话单不需要 | DISABLE | 避免无效配置（开了对端不支持也不携带） |

### RECORDEXTTYPE 扩展类型选择（DP 0-01005）

问：当需要开启话单扩展字段时，选用哪种扩展类型？

| 选项 | 前提 | 扩展内容 |
|---|---|---|
| CCB | RECORDEXTENTION=ENABLE | 内容计费首个 URL + 使用时长 |
| HP | RECORDEXTENTION=ENABLE | 高精度时间戳 |
| DT | RECORDEXTENTION=ENABLE | Direct Tunnel 模式流量+地址列表 |
| GLB | RECORDEXTENTION=ENABLE | 业务报文数 + AAA 计费信息 |
| COMM | RECORDEXTENTION=ENABLE | 通用字段（含 SNSSAI/SSCMODE） |

## 约束

- **立即生效**（info，rule-0-01006）：ADD CDRFIELDTEMP 执行后立即生效，无需重启
- **最大 100 条记录**（warning，rule-0-01007）：超限新增失败 — 规划前查现有模板数
- **当前版本不支持字段**（high，rule-0-01008）：EXTCHARGEID/THREEGPP2USRLOC/ULIWLAN 当前版本不支持；配置无效 — 避免配置不支持的参数
- **双端支持才生效**（high，rule-0-01009）：RATTYPE/ULI/MSTIMEZONE 等需 SGSN/S-GW 与 PGW-C/SMF 双端都支持，开了对端不支持也不携带 — 易被误判为配置失效
- **PACKETNUM=ENABLE 需先配费率**（warning，rule-0-01010）：仅当配置了费率时 PGW 扩展字段才携带 IP 层报文包数 — 未配费率时 PACKETNUM 配置无效
- **RECORDEXTTYPE=GLB/DT 时 PacketNum 不受控**（warning，rule-0-01011）：此时 PGW 扩展字段是否携带 IP 层报文包数不受 PacketNum 控制 — 话单包数统计失效

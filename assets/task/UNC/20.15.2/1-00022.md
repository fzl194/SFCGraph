## 约束

- **本骨架不引入独立命令**（critical，`§3.7`）：本骨架 `cmd: 无`，控制面-用户面接口链路建立归各被依赖特性 wiki。
- **接口链路先于会话建立**（critical）：5G N4 PFCP Association / 4G Sxb PFCP Association / 2-3G Gn/Gp TCP 链路必须在会话建立之前完成建立与活跃状态 — 链路建立归被依赖特性。
- **UPF Node ID 全网唯一**（critical）：UPF Node ID（IP/FQDN）必须全网唯一，避免 PFCP 路由错乱。
- **PFCP Association 恢复**（warning）：SMF/UPF 重启后需 PFCP Association Setup Recovery 重连，否则 Association 失活 → 会话建立失败。
- **GTP-U 路径可达**（critical）：N4 PFCP 控制面链路 + GTP-U 用户面隧道都需要配置，5G 场景下 PFCP 偶联成功是会话建立的前提。

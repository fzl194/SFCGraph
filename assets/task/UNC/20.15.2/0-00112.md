# MOD AMUEPLCYCTRL 原子任务
> 命令静态知识见 [MOD AMUEPLCYCTRL](command/UNC/20.15.2/MOD-AMUEPLCYCTRL.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法
按"用户范围 × 策略维度"修改控制参数：`SUBRANGE` 圈定作用域（ALL_USER 或 IMSI/MSISDN 前缀、切片群组），再按业务维度打开对应开关并填关键字。两大典型场景：①PCC 基本功能里把网络侧 UE 策略偶联打开（`ISUEASSOC=YES`），或回退本地配置（`ISAMASSOC/ISUEASSOC=NO`）；②MEC/本地特色业务（动态 UE Logo、就近接入）按区域+用户群使能就近接入开关。

| 业务维度 | 必备参数 | 典型脚本 |
|---|---|---|
| 应用网络侧 AM/UE 策略（PCC 基本功能） | `SUBRANGE=ALL_USER` + `ISUEASSOC=YES`（+可选 `ISAMASSOC=YES`） | `MOD AMUEPLCYCTRL:SUBRANGE="ALL_USER",ISUEASSOC="YES";` |
| 回退本地配置的 AM/UE 策略 | `SUBRANGE=ALL_USER` + `ISAMASSOC=NO` + `ISUEASSOC=NO` | `MOD AMUEPLCYCTRL:SUBRANGE="ALL_USER",ISAMASSOC="NO",ISUEASSOC="NO";` |
| 就近接入（MEC 本地特色） | `SUBRANGE` + `NEARBYACCSW=YES` + `NEARBYKEYWD` | `MOD AMUEPLCYCTRL:SUBRANGE="ALL_USER",NEARBYACCSW="YES",NEARBYKEYWD="mec_kw";` |
| 基于 DNN 创建 AM/UE 策略偶联 | `DNNAMPLCYSW=YES`+`DNNAMPLCYKEYWD`（+ UE 侧 `DNNUEPLCYSW`+`DNNUEPLCYKEYWD`） | `MOD AMUEPLCYCTRL:SUBRANGE="ALL_USER",DNNAMPLCYSW="YES",DNNAMPLCYKEYWD="dnn1";` |

## 决策点

### 策略来源：网络侧 vs 本地配置
问：AM/UE 策略遵从 PCF 下发还是 AMF 本地配置？
- 网络侧（PCF）→ `ISAMASSOC=YES` / `ISUEASSOC=YES`（建立策略偶联，向 PCF 请求策略）。
- 本地配置 → `ISAMASSOC=NO` / `ISUEASSOC=NO`（不建立偶联，AMF 用本地或签约策略）。

### 用户范围选定方式（`SUBRANGE` 多值）
问：用哪种方式圈定受控用户群？
- `ALL_USER`：所有用户。
- IMSI 前缀（`IMSIPRE`，5~15 位数字）：按号段。
- MSISDN 前缀（`MSISDNPRE`，1~15 位数字）：按号码段。
- 切片群组（`NSGRPID`，0~16；0=无效即不按切片区分）：按网络切片群组。

### 策略失败/终止处理策略
问：AM/UE 策略建立或更新失败、终止时怎么处理？
- AM 失败/终止：`AMFAILPLCY`/`AMTERPLCY` 取 `LOCAL_OR_SUB_AMPLCY`（向 PCF 请求失败后回退本地/签约 AM 策略）。
- UE 失败/终止：`UEFAILPLCY`/`UETERPLCY` 取 `NOPROCESS`（忽略失败/终止，不异常处理，用户按现有策略继续）。

### 就近接入是否发现 H-PCF
问：就近接入场景下是否向归属 PCF 上报？（仅 `NEARBYACCSW=YES` 时触发）
- `NEARBYHPCFSW=YES`：发现 H-PCF。
- `NEARBYHPCFSW=NO`：不发现。

## 约束
- `SUBRANGE` 为必选；当取值不是 `ALL_USER` 时，对应的 `IMSIPRE`/`MSISDNPRE`/`NSGRPID` 必须填写（conditional）。`severity=critical` 后果：作用域未定义，策略无法生效。
- `NEARBYACCSW=YES` 时 `NEARBYKEYWD` 必填（0~63 字符，字母/数字/`-`/`.`，首尾仅字母数字）。`severity=high`
- `DNNAMPLCYSW=YES` 时 `DNNAMPLCYKEYWD` 必填；`DNNUEPLCYSW=YES` 时 `DNNUEPLCYKEYWD` 必填（均 1~63 字符）。`severity=high`
- `IMSIPRE` 5~15 位纯数字；`MSISDNPRE` 1~15 位纯数字；`NSGRPID` 0~16（0 表无效）。`severity=medium`
- `DESC` 描述信息 0~32 字符。`severity=low`
- 适用 NF：AMF。`severity=medium` 后果：在非 AMF 网元执行无效。
- 执行后立即生效。`severity=low`

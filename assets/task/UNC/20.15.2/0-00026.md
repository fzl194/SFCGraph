# ADD DIAMPEERADDR 原子任务

> 命令静态知识见 [ADD DIAMPEERADDR](command/UNC/20.15.2/ADD-DIAMPEERADDR.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法

ADD DIAMPEERADDR 指定对端主机的地址类型与地址值，HOSTNAME 必须已用对应服务器命令（[ADD PCRF](command/UNC/20.15.2/ADD-PCRF.md) / [ADD OCS](command/UNC/20.15.2/ADD-OCS.md) / [ADD DRA](command/UNC/20.15.2/ADD-DRA.md) / ADD DIAMETERAAA）建好。地址二选一：**IPv4/IPv6 地址**（TCP，可选 PORT，缺省 3868）或 **SCTP 端点名**（SCTP，端点须已用 [ADD SCTPENDPOINT](command/UNC/20.15.2/ADD-SCTPENDPOINT.md) 建好）。同一主机可重复本命令加多个地址/端点构成多链路。

**参数取值样例**

| 参数 | 取值样例 | 获取方法 | 说明 |
|---|---|---|---|
| HOSTNAME | `pcrf_1` / `ocs` / `host1.example.com` / `3gppaaa` | 已配置数据中获取 | 必须已建对应类型对端主机 |
| ADDRTYPE | `IPv4` / `IPv6` / `SCTP` | 与对端协商 | 决定后续参数集 |
| IPV4ADDRESS | `10.11.21.59` / `192.168.40.10` | 与对端协商或全网规划 | ADDRTYPE=IPv4 时配，禁 0.0.0.0 / 255.255.255.255 |
| IPV6ADDRESS | IPv6 地址 | 与对端协商 | ADDRTYPE=IPv6 时配，禁全 F |
| PORT | `3868`（缺省）/ `3396` / `3397` | 全网规划 | TCP 模式可选 |
| SCTPENDPOINT | `pcrfendb` / `ocsend` / `endpoint0` | 已配置数据中获取 | ADDRTYPE=SCTP 时配，已 ADD SCTPENDPOINT |

**特性场景取值对比**

| 特性场景 | HOSTNAME | ADDRTYPE | 地址/端点 | PORT | 说明 |
|---|---|---|---|---|---|
| Gx 直连 PCRF IPv4（WSFD-109101） | pcrf_1/pcrf_2 | IPv4 | 10.11.21.59/60/61 | — | 多 IP 重复本命令建多链路 |
| Gx over SCTP（WSFD-104508） | pcrf | SCTP | pcrfendb/pcrfendc | — | 本端 1 端点对对端 2 端点 |
| Gy over SCTP（WSFD-104508） | ocs | SCTP | ocsend | — | — |
| Gy 直连 OCS IPv4（WSFD-109001） | ocs1/ocs_host | IPv4 | 10.11.21.59/192.168.40.10 | 3868 | PORT 缺省 3868 |
| Gx over DRA IPv4（WSFD-011132） | host1.example.com | IPv4 | 10.10.0.1/10.10.0.2 | — | 同一 DRA 不允许同时配 IP 和 SCTP 端点 |
| Gx over DRA SCTP（WSFD-011132） | host1.example.com | SCTP | endpoint0/endpoint1 | — | 与 IPv4 互斥 |
| Gy over DRA（WSFD-011133） | host1.example.com/ocs1 | IPv4/SCTP | 10.10.10.1/endpoint0 | 3396/3397 | OCS 直连 + DRA 非直连并存 |
| S6b over DRA（WSFD-011134） | host1.example.com | IPv4/SCTP | 10.10.10.1/endpoint0 | — | — |
| S6b 直连 3GPP AAA（WSFD-010102） | 3gppaaa | SCTP | dc1_AAA01-S6B-SCTPEP01 | — | TCP 时改 IPv4 |

## 决策点

### ADDRTYPE 对端地址类型选择（TCP IPv4/IPv6 vs SCTP）

承载协议决定后续参数集与前置依赖：

| ADDRTYPE | 承载 | 后续参数 | 前置依赖 |
|---|---|---|---|
| IPv4 | TCP | IPV4ADDRESS（+可选 PORT，缺省 3868） | 对端主机已建（ADD PCRF/OCS/DRA/AAA） |
| IPv6 | TCP | IPV6ADDRESS（+可选 PORT） | 对端主机已建 |
| SCTP | SCTP | SCTPENDPOINT | 已 ADD SCTPENDPOINT 建好对端端点 |

### 同一 DRA 不允许 IP 与 SCTP 端点并存

over-DRA 场景对同一 DRA 主机：

| 选择 | ADDRTYPE 配法 |
|---|---|
| 走 IP 地址 | ADDRTYPE=IPv4 + IPV4ADDRESS（不可再配 SCTP） |
| 走 SCTP 端点 | ADDRTYPE=SCTP + SCTPENDPOINT（不可再配 IPv4） |

> PCRF/OCS 直连场景无此互斥限制；仅 DRA 强制二选一。

### 直连 OCS vs 非直连 OCS（Gy over DRA 场景）

| OCS 类型 | 配置 | 路由方式 |
|---|---|---|
| 直连 OCS | ADD OCS（Realm）+ ADD DIAMPEERADDR（IP） | 直连链路 |
| 非直连 OCS | ADD OCS（Realm），**不配** ADD DIAMPEERADDR | 基于 Realm 查 Diameter 路由经 DRA 转发 |

## 约束

- **立即生效** · severity=LOW。
- **最大 510 条记录** · severity=LOW。
- **HOSTNAME 前置依赖** · severity=HIGH · 对端主机不存在则不能配对端地址，配置失败。
- **PCRF/OCS/DRA 单主机最多 20 个 IP 或 20 个 SCTP 端点** · severity=HIGH · 超规格配置失败。
- **3GPP AAA 单主机最多 2 个 IP 或 2 个 SCTP 端点** · severity=HIGH · 超规格配置失败。
- **同一 DRA 不允许同时配 IP 和 SCTP 端点** · severity=HIGH · 二选一，否则行为异常。
- **ADD PCRF/ADD OCS 的 HOSTNAME/REALMNAME 须与本命令一致** · severity=HIGH · 主机名/域名不一致对端拒绝。
- **App 与 IP/SCTP 都配置才生效** · severity=MEDIUM · 仅建主机未配地址不建链。
- **DRA 与 PCRF/OCS/AAA 主机名不能重复** · severity=HIGH · over-DRA 场景特别约束。

# 配置PCRF返回消息属性（SET PCCPCRFMSGATTR）

> 命令静态知识见 [SET PCCPCRFMSGATTR](command/UNC/20.15.2/SET-PCCPCRFMSGATTR.md)。本页只讲配置生成实例化时这条命令**怎么配**——按配置维度组织成"字典"，**不**逐特性罗列取值。

## 配置方法

SET PCCPCRFMSGATTR 控制激活和更新流程中 UNC 是否根据 DRA 或 PCRF 消息触发的 PCRF 重选。三个参数分别对应三种触发消息（CCA-I/CCA-U/RAR 的 Origin-Host AVP）。命令存在系统初始记录，三参数初始值均为 DISABLE（缺省不支持重选）。需启用时置 ENABLE。本命令有 1 个配置维度：**PCRF 重选触发消息开关**。

### 配置维度 1：PCRF 重选触发消息开关（参数 ORGHOSTCCAI/ORGHOSTCCAU/ORGHOSTRAR，可选）

| 维度 | 参数 | 取值 | 默认 | 作用 | 适用场景 |
|---|---|---|---|---|---|
| CCA-I 触发 | `ORGHOSTCCAI` | `ENABLE`/`DISABLE` | DISABLE | 激活流程中基于 CCA-I Origin-Host AVP 触发 PCRF 重选 | 激活流程（Initial CCA） |
| CCA-U 触发 | `ORGHOSTCCAU` | `ENABLE`/`DISABLE` | DISABLE | 更新流程中基于 CCA-U Origin-Host AVP 触发 PCRF 重选 | 更新流程（Update CCA） |
| RAR 触发 | `ORGHOSTRAR` | `ENABLE`/`DISABLE` | DISABLE | 更新流程中基于 RAR Origin-Host AVP 触发 PCRF 重选 | PCRF 主动推送（Re-Auth-Request） |

> 数据来源：本端规划。三参数缺省均为 DISABLE（不支持重选）。是否启用取决于网络是否需要 UNC 根据 DRA/PCRF 的 Origin-Host 重定向指示动态重选 PCRF；属本端规划决策，需结合组网（是否经 DRA、PCRF 是否支持重选指示）决定。证据中 WSFD-109101 将此命令列为"可选配置"。

**典型脚本**

```
# 激活+更新流程均支持 PCRF 重选（DRA/PCRF 触发）
SET PCCPCRFMSGATTR: ORGHOSTCCAI=ENABLE,ORGHOSTCCAU=ENABLE,ORGHOSTRAR=ENABLE;

# 仅激活流程支持 PCRF 重选
SET PCCPCRFMSGATTR: ORGHOSTCCAI=ENABLE;
```

**步骤位置**：PCC 基本功能（WSFD-109101）整机粒度可选配置步骤，用于在激活/更新过程中根据 PCRF 返回的重定向指示重选 PCRF；缺省 DISABLE 时按常规 PCRF 选择流程处理。

## 决策点

### 决策点：是否启用 PCRF 重选（DP 0-03129）

问：是否需要 UNC 根据 DRA/PCRF 的 Origin-Host 重定向指示动态重选 PCRF？

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| 启用（按需组合） | 对应参数置 `ENABLE` | 网络经 DRA 或 PCRF 支持重选指示，需 UNC 动态响应重定向 |
| 缺省不支持（不配置） | 三参数沿用初始 DISABLE | 常规 PCRF 选择流程，无重选需求 |

> 属本端规划决策；需结合组网（是否经 DRA、PCRF 是否支持重选指示）决定。证据中标记为"可选配置"，缺省不支持时按常规 PCRF 选择流程处理。

### 决策点：触发消息类型选择（DP 0-03130）

问：启用 PCRF 重选时，在哪些流程阶段响应 Origin-Host 重定向指示？

| 选项 | 联动参数 | 适用场景 |
|---|---|---|
| 仅激活流程 | `ORGHOSTCCAI=ENABLE` | 仅在激活流程响应 CCA-I 重定向 |
| 仅更新流程（CCA-U） | `ORGHOSTCCAU=ENABLE` | 仅在更新流程响应 CCA-U 重定向 |
| 仅 PCRF 主动推送 | `ORGHOSTRAR=ENABLE` | 仅响应 RAR 重定向 |
| 组合使能 | 多参数 `ENABLE` | 激活+更新+推送多阶段均响应重定向 |

> 三种触发消息对应不同流程阶段（CCA-I 激活/CCA-U 更新/RAR 推送），可按需单独或组合使能。

## 约束

- **系统初始值 DISABLE**（critical，rule-0-03321）：系统初始值 DISABLE：三参数（ORGHOSTCCAI/ORGHOSTCCAU/ORGHOSTRAR）初始值均为 DISABLE，未配置时缺省不支持 PCRF 重选，按常规 PCRF 选择流程处理 — 后果：缺省不支持重选，需显式 ENABLE 才启用
- **整机粒度控制**（warning，rule-0-03322）：整机粒度控制：本命令为整机粒度控制 PCRF 重选，非 APN/用户粒度；影响所有激活/更新流程 — 后果：整机生效，无法按 APN/用户差异化配置
- **组网依赖**（warning，rule-0-03323）：组网依赖：启用前须确认组网是否经 DRA、PCRF 是否支持重选指示；不匹配的组网下启用可能导致重选异常 — 后果：组网不匹配导致 PCRF 重选失败
- **最大 1 条记录**（info，rule-0-03324）：最大 1 条记录：全局唯一配置，最大记录数为 1 — 后果：超出记录数限制导致命令失败
- **立即生效**（info，rule-0-03325）：立即生效：命令执行后立即生效，非仅对新激活用户 — 后果：配置即时生效，影响所有后续激活/更新流程

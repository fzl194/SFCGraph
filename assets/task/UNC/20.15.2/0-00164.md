# SET CHGWKDY 原子任务
> 命令静态知识见 [SET CHGWKDY](command/UNC/20.15.2/SET-CHGWKDY.md)。本页只讲配置生成实例化时这条命令**怎么配**。

## 配置方法
SET CHGWKDY 用于 SGSN 设置某计费属性用户（`CC`）周一到周日（`WKDAY`）每天的费率类型（`TT`：`WORK` 工作日 / `WEEKEND` 休息日）。该配置与 [ADD CHGTARI](task/UNC/20.15.2/0-00022.md)（费率时段配置）结合，对 S-CDR 话单进行不同费率切换。同一计费属性周记录固定 7 条，全表固定 28 条（4 计费属性 × 7 天）。

| 计费属性 | 星期 | 费率类型 | 典型脚本 |
|---|---|---|---|
| 实时计费（HOTBILLING） | 周五 | 休息日 | `SET CHGWKDY:WKDAY=FRI,CC=HOTBILLING,TT=WEEKEND;` |
| 普通计费（NORMAL） | 周一~周五 | 工作日 | `SET CHGWKDY:WKDAY=MON,CC=NORMAL,TT=WORK;`（逐天下发） |
| 包月制（FLATRATE） | 周六/周日 | 休息日 | `SET CHGWKDY:WKDAY=SAT,CC=FLATRATE,TT=WEEKEND;` |

**计费属性（CC）取值**：`NORMAL`(普通计费) / `PREPAID`(预付费) / `FLATRATE`(包月制) / `HOTBILLING`(实时计费)。
**费率类型（TT）取值**：`WORK`(工作日) / `WEEKEND`(休息日)。

## 决策点

### 每日费率类型
问：某计费属性用户在某天按工作日还是休息日费率？
- `TT=WORK`：工作日费率。
- `TT=WEEKEND`：休息日费率（费率切换场景，如周五提前进入周末费率）。
- 取值由整网规划，需与费率时段配置（ADD CHGTARI）的时段定义配套。

## 约束
- "星期序号 + 计费属性"唯一确定一条记录（同一组合不能重复配置）。`severity=critical`
- 同一计费属性的周记录数固定为 7，本表记录总数固定为 28（4 计费属性 × 7 天）。`severity=high`
- 必须与 [ADD CHGTARI](task/UNC/20.15.2/0-00022.md) 联动才能实现灵活费率时段控制；可选与 [ADD CHGHOLI](task/UNC/20.15.2/0-00019.md)（节假日配置）联动覆盖节假日费率。`severity=medium`
- 系统初次运行执行系统初始设置值。`severity=low`
- 该命令执行后立即生效。`severity=low`

# ADD AMFROAMNSMAP（增加漫游切片映射配置）

## 功能

**适用NF：AMF**

该命令用于增加漫游切片映射配置。

## 注意事项

- 该命令执行后立即生效。

- 最多可输入8192条记录。
- 当SET NSMAPPARA中本地映射开关参数设置为“ON(打开)”，AMF判断不能为UE提供需要的切片映射服务时，仍会发起到NSSF的网络切片映射服务。
- 增加漫游切片映射配置时，对已注册用户不会立即生效。如果需要立即生效，请先执行STR NGUSRBATCHDEREG命令对漫游用户启动批量去注册任务，并指定去注册类型为需要重新注册，触发用户重新注册，获取最新切片映射关系。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组

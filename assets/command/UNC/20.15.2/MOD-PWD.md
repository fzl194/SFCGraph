# MOD PWD（修改在线用户自身密码）

## 功能

该命令用于修改在线用户自身密码。5G场景中，在线用户表示用户internaluser，即MML连接管道时所用的用户。

## 注意事项

- 该命令执行后立即生效。
- 该命令修改在线用户密码后，会断开与网元的连接，需要重新登录网元。
- 修改密码时，确认密码必须和第一次输入的新密码一样。
- 该命令只能用于修改本地用户的自身密码。

## 权限

G_1，管理员级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| OLDPASSWORD | 旧密码 | 可选必选说明：必选参数<br>参数含义：旧密码。<br>数据来源：本端规划<br>取值范围：密码类型，输入长度范围为8～64。<br>默认值：无 |
| NEWPASSWORD | 新密码 | 可选必选说明：必选参数<br>参数含义：新密码。<br>数据来源：本端规划<br>取值范围：密码类型，输入长度范围为8～64。<br>默认值：无<br>配置原则：<br>- 区分大小写。<br>- 密码必须包含数字、大写字母、小写字母、特殊字符，不包括逗号、分号、等号、双引号、单引号、问号和空格。<br>- 密码不能包含账号或账号的反向字符串。<br>- 密码不能与之前10次历史密码相同（与当前密码相同也视为与历史密码相同，即当前密码包含在10次历史密码之内）。<br>- 新密码和老密码需要至少在两个字符位上有不同。 |
| CFMPASSWORD | 确认密码 | 可选必选说明：必选参数<br>参数含义：确认密码。<br>数据来源：本端规划<br>取值范围：密码类型，输入长度范围为8～64。必须与新密码一致。<br>默认值：无 |

## 操作的配置对象

- [[configobject/UNC/20.15.2/PWD]] · 在线用户自身密码（PWD）

## 使用实例

已登录用户修改自己的登录密码：

```
MOD PWD: OLDPASSWORD="*****", NEWPASSWORD="*****", CFMPASSWORD="*****";
```

# STR CNTRLTRCTASK（启动跨层统一联动跟踪任务）

## 功能

![](启动跨层统一联动跟踪任务（STR CNTRLTRCTASK）_28855492.assets/notice_3.0-zh-cn_2.png)

本功能消耗转发性能，请在系统空闲时使用。

该命令用于启动跨层统一联动跟踪任务。

## 注意事项

- 该命令执行后立即生效。

- 该命令仅在虚机场景下支持，并且仅支持FusionSphere用户态EVS和用户态OVS组网。
- 当已存在跨层统一联动跟踪任务时，该命令会执行失败。
- IP四元组不能全为空，否则该命令会执行失败。
- 外联口接口名称1/2/3/4取值之间不能重复，否则该命令会执行失败。
- 在MAE-CN VNF-LCM V100R024C10SPC120及之后版本与NFV_FusionSphere 24.1.0及之后版本下，参数“有效时间”支持取值1~60，否则参数取值仅支持1~10，若填写为1~10以外的值时，该命令会执行失败。
- 跨层联动抓包、外联口抓包功能和染色功能不可以同时开启，否则可能导致功能无法正常使用。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组


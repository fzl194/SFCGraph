# SET FWDPARA（设置转发参数）

## 功能

![](设置转发参数（SET FWDPARA）_29627123.assets/notice_3.0-zh-cn_2.png)

可能会影响报文转发，请谨慎使用并联系华为技术支持协助操作。

该命令用于设置CSLB的转发参数。

## 注意事项

- 该命令执行后立即生效。
- 系统初次上电运行时，会执行系统初始设置值。
- 通常情况下不需要执行本命令，维持系统初始设置值即可。如果需要修改转发参数，请联系华为技术支持。
- 当参数FWDSWITCH设置为打开(ON)时，对系统性能有影响，且影响大小与分片报文流量大小有关，一般情况估计下降10%~30% 。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RSTIME | 报文重组超时时长（秒） | 可选必选说明：可选参数<br>参数含义：该参数表示报文重组超时时长，指收到第一片分片报文到重组成功所允许的最长时间。<br>数据来源：本端规划<br>取值范围：2~60<br>默认值：无<br>系统初始设置值：6<br>说明：设置太长的报文重组超时时长，会造成资源的持续占用，降低设备的性能，同时可能会遭受网络攻击。推荐使用系统初始设置值。<br>配置原则：无 |
| IPV6RSTIME | IPv6报文重组超时时长（秒） | 可选必选说明：可选参数<br>参数含义：该参数表示报文重组超时时长，指收到第一片分片报文到重组成功所允许的最长时间。<br>数据来源：本端规划<br>取值范围：2~60<br>默认值：无<br>系统初始设置值：6<br>说明：设置太长的报文重组超时时长，会造成资源的持续占用，降低设备的性能，同时可能会遭受网络攻击。推荐使用系统初始设置值。<br>配置原则：无 |
| FWDSWITCH | 分片报文转板开关 | 可选必选说明：可选参数<br>参数含义：该参数表示分片报文转板开关状态。<br>数据来源：本端规划<br>取值范围：<br>- “打开(ON)：分片报文大概率会进行一次转板 ”<br>- “关闭(OFF)：如果分片报文重组超时，会直接丢弃 ”<br>默认值：无<br>系统初始设置值：打开(ON)<br>配置原则：无 |
| BATCHRCVNUM | 批量收包数 | 可选必选说明：可选参数<br>参数含义：该参数表示批量收包数。<br>数据来源：本端规划<br>取值范围：1~128<br>默认值：无<br>系统初始设置值：128<br>配置原则：无 |
| BATCHRCVTIMES | 批量收包次数 | 可选必选说明：可选参数<br>参数含义：该参数表示批量收包次数。<br>数据来源：本端规划<br>取值范围：1~16<br>默认值：无<br>系统初始设置值：4<br>配置原则：无 |

## 操作的配置对象

- [[UNC@20.15.2@ConfigObject@FWDPARA]] · 转发参数（FWDPARA）

## 使用实例

设置报文重组超时时长为6秒，分片报文转板开关为关闭:

SET FWDPARA: RSTIME=6, FWDSWITCH=OFF;

# SET MSSWORKFILTER（设置Work过滤规则信息）

## 功能

![](设置Work过滤规则信息（SET MSSWORKFILTER）_50120582.assets/notice_3.0-zh-cn_2.png)

本命令属于高危命令，操作不当会导致性能下降，请谨慎使用并联系华为技术支持协助操作。

该命令用于设置Work的过滤规则信息。

Work是转发调度的基本单元，可以通过设置特定的规则，将符合规则的Work运行信息记录下来并可以查询，进行故障诊断。

设置Work过滤规则之后，系统性能会下降0.2%左右。

## 注意事项

- 该命令执行后立即生效。
- 本命令用于使能work过滤规则开关，使能后会降低性能且在用户指定的时间之后会自动去使能，关闭后会恢复性能。默认时间是60秒。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组

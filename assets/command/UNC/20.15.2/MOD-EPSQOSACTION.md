# MOD EPSQOSACTION（修改EPS QoS控制动作配置）

## 功能

**适用NF：PGW-C**

该命令用于修改4G用户QoS上下行保证带宽门限、最高带宽门限以及带宽超过门限值时对QoS用户的处理动作。

## 注意事项

- 命令执行后只对新接入用户生效。

- 上行保证带宽不能大于上行最大带宽。
- 下行保证带宽不能大于下行最大带宽。
- 带宽取值为4294967295时，表示无效值。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组

## 使用实例

修改“QOSPROFILENAME”为“test”，“QCI”为“2”的“下行保证带宽”门限值为“3333”bit/s：

```
MOD EPSQOSACTION:QOSPROFILENAME="test",QCI=2,GBRDL=3333,GBRDLACTION=DEGRADE;
```

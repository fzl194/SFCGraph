# MOD PRER8SUBQOS（修改Pre-R8签约QoS配置）

## 功能

**适用NF：GGSN**

该命令用于修改用户的签约QoS属性信息。

## 注意事项

- 命令执行后只对新接入用户生效。

- 确保同类型同方向上的最大带宽值不小于保证带宽值。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组

## 使用实例

修改SUBQOSINDEX为1的QoS签约信息，设置ARP为HIGH，STRMMBRUL为7，INTERMBRDL为12，BACKMBRUL为30222：

```
MOD PRER8SUBQOS: SUBQOSINDEX=1, ARP=HIGH, STRMMBRUL=7, INTERMBRDL=12, BACKMBRUL=30222;
```

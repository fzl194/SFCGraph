# MOD CDRAUDIT（修改话单稽核）

## 功能

**适用NF：NCG**

该命令用于修改某通道下或所有通道下第二份最终话单的稽核规则。

## 注意事项

- 该命令执行后立即生效。
- 执行该命令后，可能会影响稽核文件的格式，比如：文件名、稽核参数等，修改后的格式将在第二天生成的正式文件中体现，第二天生成的正式文件的格式以最后一次修改的配置为准。 不能通过该命令修改“通道名称”。

## 权限

manage-ug；system-ug
G_1，管理员级别命令组；G_2，操作员级别命令组

## 使用实例

修改标识为“Audit”话单稽核任务中稽核文件保存天数。示例如下：

```
MOD CDRAUDIT: CDRAUDITID="Audit", AGID="PS_GROUP_1", AUDITSTORDAYS=10;
```

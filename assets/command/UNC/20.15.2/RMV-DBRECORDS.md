# RMV DBRECORDS（删除数据库中的表记录）

## 功能

![](删除数据库中的表记录（RMV DBRECORDS）_70132935.assets/notice_3.0-zh-cn_2.png)

本操作删除数据后，数据无法恢复。请谨慎操作。

该命令用于使用DDF插件侧的网元在紧急情况下进行数据表记录的删除操作。

## 注意事项

- 该命令执行后立即生效。

- 使用该命令时，请勿进行数据的插入或更新操作，并联系华为技术开发人员。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |

| CELLTYPE | 进程类型 | 可选必选说明：该参数在"DBGTYPE"配置为"CELLTYPE"时为条件必选参数。<br>参数含义：该参数用于表示发给同一类型的进程。通过DSP MSPROCTYPE来查询。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~4294967295。<br>默认值：无<br>配置原则：无 |

| DATABASENAME | 数据库名称 | 可选必选说明：可选参数<br>参数含义：该参数用于表示数据库名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~255。<br>默认值：无<br>配置原则：<br>查询sdb类型数据库的库名举例如下：<br>OPR DBGDATA: DBGTYPE=CELLTYPE, CELLTYPE=1005, DEBUGNAME="DDF LST DATABASES WHERE DATABASETYPE IS sdb";<br>查询ddb类型数据库的库名举例如下：<br>OPR DBGDATA: DBGTYPE=CELLTYPE, CELLTYPE=1005, DEBUGNAME="DDF LST DATABASES WHERE DATABASETYPE IS ddb"; |
| TABLENAME | 表名 | 可选必选说明：可选参数<br>参数含义：该参数用于表示表名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~255。<br>默认值：无<br>配置原则：<br>查询sdb类型数据库的表名举例如下，其中数据库名为参数DATABASENAM的取值：<br>OPR DBGDATA: DBGTYPE=CELLTYPE, CELLTYPE=1005, DEBUGNAME="DDF LST TABLES WHERE DATABASETYPE IS sdb AND DATABASENAME IS SmcSvc";<br>查询ddb类型数据库的表名举例如下，其中数据库名为参数DATABASENAM的取值：<br>OPR DBGDATA: DBGTYPE=CELLTYPE, CELLTYPE=1005, DEBUGNAME="DDF LST TABLES WHERE DATABASETYPE IS ddb AND DATABASENAME IS SmcSvc"; |

| PRIMARYKEY | 主键 | 可选必选说明：可选参数<br>参数含义：该参数用于表示主键。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~5120。<br>默认值：无<br>配置原则：<br>当使用指定主键删除表数据记录时，可使用此参数，字段构造举例为：含有一个字段的主键 ：k1:Imsi\|\|string\|\|1；含有两个字段的主键 ：k1:Imsi\|\|string\|\|1+Tmsi\|\|string\|\|2；k1 是索引名，Imsi是字段名，string是字段类型，1是字段的值；Tmsi是字段名，string是字段类型，2是字段的值。构造的主键字段的数据类型只支持string、int、int8、int16、int32、int64、uint、uint8、uint16、uint32、uint64、float32和float64。 |
| PARA1 | 参数1 | 可选必选说明：可选参数<br>参数含义：扩展参数1。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~1024。<br>默认值：无<br>配置原则：<br>当使用指定字段删除表数据记录时，可使用此参数，字段构造举例为：Imsi\|\|string\|\|1；Imsi是字段名，string是字段类型，1是字段的值。字段的数据类型只支持string、int、int8、int16、int32、int64、uint、uint8、uint16、uint32、uint64、float32和float64。 |
| PARA2 | 参数2 | 可选必选说明：可选参数<br>参数含义：扩展参数2。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~1024。<br>默认值：无<br>配置原则：<br>数据删除速度，最大值为2000条/秒，如果不配置默认速度为100条/秒。 |
| PARA3 | 参数3 | 可选必选说明：可选参数<br>参数含义：扩展参数3。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~1024。<br>默认值：无<br>配置原则：<br>该值为表的记录数。<br>查询sdb类型数据库的表记录数操作举例如下：<br>OPR DBGDATA: DBGTYPE=CELLTYPE, CELLTYPE=1005, DEBUGNAME="DDF DSP NUM WHERE DATABASETYPE IS sdb AND DATABASENAME IS testsdb";<br>查询ddb类型数据库的表记录数操作举例如下：<br>OPR DBGDATA: DBGTYPE=CELLTYPE, CELLTYPE=1005, DEBUGNAME="DDF LST TABLES WHERE DATABASETYPE IS ddb AND DATABASENAME IS testddb"; |
| PARA4 | 参数4 | 可选必选说明：可选参数<br>参数含义：扩展参数。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~1024。<br>默认值：无<br>配置原则：无 |

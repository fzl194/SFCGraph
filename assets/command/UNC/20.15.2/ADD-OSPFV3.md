# ADD OSPFV3（创建OSPFv3进程配置）

## 功能

该命令用于增加OSPFv3特性的进程。

## 注意事项

- 该命令执行后立即生效。
- 该命令最大记录数为8016。
- 设备的一个接口只能属于某一个OSPFv3进程。
- 如果指定了VPN实例，OSPFv3进程属于此实例，否则属于全局实例。进程实例不可更改，只能在第一次使能该进程时指定。
- 只有在OSPFv3进程下配置了Router ID，OSPFv3进程才能正常运行，否则只能看到该进程，但无法生成LSA。
- OSPFv3进程实例数规格默认为整机2000，实际以服务规格文件为准。
- 建议配置协议相关认证，增强协议安全性，否则可能存在协议安全风险。认证可以通过ADD OSPFV3AUTH、ADD OSPFV3AREAAUTH或ADD OSPFV3INTERFACEAUTH命令进行配置。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组

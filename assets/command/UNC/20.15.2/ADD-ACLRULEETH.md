# ADD ACLRULEETH（增加以太ACL规则）

## 功能

该命令用于增加以太ACL规则。以太ACL规则支持按照源目的MAC，协议号，VLAN ID等信息进行匹配。主要应用场景为端口流量策略。

## 注意事项

- 该命令执行后立即生效。
- 该命令最大记录数为65535。
- 增加ACL规则前必须先执行ADD ACLGROUP指定所在规则组，单个ACL规则组下最多支持16384条ACL规则。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组

## 使用实例

用户使用流分类，需要过滤源MAC地址为00E0-FC12-3456的报文，可以配置以太ACL规则：

```
ADD ACLRULEETH:ACLNAME="4000",ACLRULENAME="rule_5",ACLACTION=Deny,ACLSRCMAC="00E0-FC12-3456";
```

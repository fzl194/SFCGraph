# SET FLOWCTRLPARA（设置流控系统参数）

## 功能

![](设置流控系统参数(SET FLOWCTRLPARA)_26305982.assets/notice_3.0-zh-cn_2.png)

此命令需要华为技术支持人员指导下才能执行，请慎重使用。

**适用网元：SGSN、MME**

该命令用于设置某个进程上每秒数据流量的接入限制数。

## 注意事项

- 系统初次上电运行时，会执行系统初始设置值。
- 此命令执行后立即生效。
- 此命令需要华为技术支持人员指导下才能执行，请慎重使用。

## 权限

manage-ug；system-ug
G_1，管理员级别命令组；G_2，操作员级别命令组

## 使用实例

设置流控系统参数：进程类型为GBP，窗口最小值为30，窗口最大值为5000，用户面窗口最小值为100，用户面窗口最大值为16000，寻呼窗口最大值为800：

SET FLOWCTRLPARA: PROCTYPE=GBP, MINWAL=30, MAXWAL=5000, MINWAL_GBDATA=100, MAXWAL_GBDATA=16000, MAXWAL_GBPAGING=800;

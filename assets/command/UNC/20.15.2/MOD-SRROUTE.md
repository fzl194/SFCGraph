# MOD SRROUTE（修改IPv4静态路由）

## 功能

该命令用于修改IPv4静态路由。

静态路由是一种需要管理员手工配置的特殊路由。

当网络结构比较简单时，只需配置静态路由就可以使网络正常工作。当设备不能使用动态路由协议或者不能建立到达目的网络时，也可以使用静态路由。合理的静态路由可以改进网络性能，并为重要业务保证带宽。

## 注意事项

- 该命令执行后立即生效。
- 修改该静态路由时，要保证该路由添加过。
- 不能在VPN实例__mpp_vpn_inner__和__mpp_vpn_inner_server__下修改默认静态路由。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组

## 使用实例

修改一条目的地址为10.0.0.1，掩码为32，下一跳是10.0.0.3，出接口是NULL0口的公网IPv4静态路由，tag为10：

```
MOD SRROUTE:AFTYPE=ipv4unicast,PREFIX="10.0.0.1",MASKLENGTH=32,VRFNAME="_public_",DESTVRFNAME="_public_",IFNAME="NULL0",NEXTHOP="10.0.0.3",TAG=10;
```

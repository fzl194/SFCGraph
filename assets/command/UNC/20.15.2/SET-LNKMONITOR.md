# SET LNKMONITOR（设置链路频繁闪断监控功能）

## 功能

**适用网元：SGSN、MME**

本命令用于设置当系统中大量链路频繁闪断时是否上报告警提示，以及大量链路频繁闪断的告警上报条件。

当前仅支持S1链路的频繁闪断监控，其原理说明如下：当系统中的S1链路发生瞬间闪断，闪断时间小于防闪断定时器时长时，系统不会有告警提示。但当某条链路频繁闪断时，会上报此链路频繁闪断的异常事件，并记录文件。当短时间内系统中有大量链路发生频繁闪断时，对系统业务会有明显影响，需要告警提示。

> **说明**
> - 防闪断定时器时长默认40s，可通过[**SET S1APPARA**](../../../S1接口管理/S1AP协议参数/设置S1AP协议参数(SET S1APPARA)_72225935.md)命令的“防闪断定时器(s)”参数进行配置。
> - 频繁闪断指某条链路在T时间内闪断5次及以上，T的取值与“防闪断定时器(s)”取值有关：当“防闪断定时器(s)”小于等于1分钟时，T取5分钟；当“防闪断定时器(s)”超过1分钟时，T取5倍“防闪断定时器(s)”时长。

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| LNKTYPE | 链路类型 | 可选必选说明：必选参数<br>参数含义：本参数用于指定链路类型。<br>数据来源：本端规划<br>取值范围：<br>“S1AP(S1AP 链路)”<br>系统初始设置值：<br>“S1AP(S1AP 链路)”<br>配置原则：当前仅支持S1AP链路。 |
| FLASHSWITCH | 瞬间闪断监控开关 | 可选必选说明：必选参数<br>参数含义：本参数用于决定是否开启瞬间闪断监控功能。<br>数据来源：本端规划<br>取值范围：<br>- “OFF(关闭)”<br>- “ON(开启)”<br>系统初始设置值：<br>“ON(开启)”<br>说明：当存在<br>**ALM-14025 批量链路频繁闪断**<br>告警时，若将该参数由<br>“ON(开启)”<br>设置为<br>“OFF(关闭)”<br>，告警恢复，且不再保存故障链路信息。 |
| PERIOD | 统计时长（s） | 可选必选说明：可选参数<br>参数含义：本参数用于指定<br>**ALM-14025 批量链路频繁闪断**<br>告警的统计周期。<br>前提条件：该参数在<br>“瞬间闪断监控开关”<br>参数配置为<br>“ON(开)”<br>后生效。<br>数据来源：本端规划<br>取值范围：60～300<br>系统初始设置值：60<br>配置原则：建议使用初始设置值。<br>说明：- 该参数和“统计门限”配合使用，组合后的含义是每隔“统计时长(s)”，系统对当前所有发生频繁闪断的链路数量进行统计，当其大于等于“统计门限”时，则上报**ALM-14025 批量链路频繁闪断**告警。<br>- 该参数配置后需要等待下一周期才会生效，等待时间与该参数之前的配置有关。 |
| THRESHOLD | 统计门限 | 可选必选说明：可选参数<br>参数含义：本参数用于指定<br>**ALM-14025 批量链路频繁闪断**<br>告警的统计门限。<br>前提条件：该参数在<br>“瞬间闪断监控开关”<br>参数配置为<br>“ON(开)”<br>后生效。<br>数据来源：本端规划<br>取值范围：1～1000<br>系统初始设置值：50<br>配置原则：建议使用初始设置值。<br>说明：- 该参数和“统计时长(s)”配合使用，组合后的含义是每隔“统计时长(s)”，系统对当前所有发生频繁闪断的链路数量进行统计，当其大于等于“统计门限”时，则上报**ALM-14025 批量链路频繁闪断**告警。<br>- 该参数配置后需要等待下一周期才会生效，等待时间与之前的“统计时长(s)”的配置有关。 |

## 操作的配置对象

## 使用实例

设置S1AP链路频繁闪断监控信息， “链路类型” 为 “S1AP(S1AP 链路)” ， “瞬间闪断监控开关” 为 “ON(开启)” ， “统计时长(s)” 为 “60” ， “统计门限” 为 “50” 。

SET LNKMONITOR: LNKTYPE=S1AP, FLASHSWITCH=ON, PERIOD=60, THRESHOLD=50;

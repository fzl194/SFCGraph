# RMV PNFWILDCARD（删除对端NF的通配策略）

## 功能

**适用NF：AMF、SMF、NRF、NSSF、SMSF、NCG**

该命令用于删除对端NF的通配策略。

## 注意事项

该命令执行后立即生效。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组

## 操作的配置对象

- [[UNC@20.15.2@ConfigObject@PNFWILDCARD]] · 对端NF的通配策略（PNFWILDCARD）

## 使用实例

运营商A需要删除NFType为Nf5G_EIR的对端NF的通配策略。

```
RMV PNFWILDCARD: NFTYPE=Nf5G_EIR;
```

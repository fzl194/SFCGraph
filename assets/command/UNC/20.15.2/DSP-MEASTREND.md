# DSP MEASTREND（查询话统指标统计趋势）

## 功能

该命令用于查询网元的话统指标统计趋势。

## 注意事项

- 当系统处于存储故障状态时，只支持查询KPI类型的指标统计趋势，查询结果不支持“最近一周同时间段(按时间倒序)”数据的对比。
- 网元升级后未结束升级观察期期间，支持查询高低版本指标统计趋势；结束升级观察期后，普通域仅支持查询高版本指标统计趋势。
- 该命令返回结果条数存在如下限制：最多返回1500条。

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| MEID | 网元ID | 可选必选说明：可选参数。<br>参数含义：标识网元ID，可以通过<br>[**LST ME**](../../系统管理/版本信息/查询网元配置信息（LST ME）_47084797.md)<br>命令查询获取。<br>取值范围：0~65535。<br>默认值：无。<br>配置原则：无。 |
| PRD | 测量周期 | 可选必选说明：必选参数。<br>参数含义：标识指标查询的测量周期。<br>取值范围：<br>- FIVE(5分钟)<br>- FIFTEEN(15分钟)<br>- THIRTY(30分钟)<br>- SIXTY(1小时)<br>- ONE_DAY(一天)<br>默认值：FIVE(5分钟)。<br>配置原则：无。 |
| QRY_NUM | 查询周期个数 | 可选必选说明：可选参数。<br>参数含义：标识指标查询的测量周期个数<br>取值范围：整形，5-30<br>默认值：无。<br>配置原则：若不输入，则查询最近5个周期。 |
| QRY_TYPE | 查询类型 | 可选必选说明：必选参数。<br>参数含义：话统指标时间的查询类型。取值范围：<br>- STD_QUERY(标准查询) ，整周期数据完整后才支持查询。<br>- REAL_QUERY(实时查询)，不依赖整周期数据是否完整，数据入库后即可查询。<br>默认值：STD_QUERY(标准查询)。<br>配置原则：<br>当配置<br>“查询类型”<br>为<br>“REAL_QUERY(实时查询)”<br>时，最新一个周期的数据可能为因为没有统计结果显示为nil。 |
| TYPE | 指标分类 | 可选必选说明：必选参数。<br>参数含义：指标分类。<br>取值范围：<br>- KPI(KPI指标)<br>- ALL(不限分类)<br>- EXTEND(EXTEND指标)<br>默认值：KPI(KPI指标)。<br>配置原则：无。 |
| MUID | 测量单元ID | 可选必选说明：该参数在<br>“指标分类”<br>配置为<br>“ALL(不限分类)”<br>时为条件可选参数。<br>参数含义：测量单元ID，可以通过<br>[**LST MEASUNIT**](查询测量指标模型(LST MEASUNIT)_47814449.md)<br>命令查询获取。<br>取值范围：整数类型，取值范围为0~4294967294。<br>默认值：无。<br>配置原则：无。 |
| MUNAME | 测量单元名称 | 可选必选说明：该参数在<br>“指标分类”<br>配置为<br>“ALL(不限分类)”<br>时为条件可选参数。<br>参数含义：测量单元名称，支持中英文模糊搜索。<br>取值范围：字符串类型，长度不超过256个字符。<br>默认值：无。<br>配置原则：若不输入，则表示该参数不作为查询的限制条件。 |
| METRICID | 测量指标ID | 可选必选说明：该参数在<br>“指标分类”<br>配置为<br>“ALL(不限分类)”<br>时为条件可选参数。<br>参数含义：测量指标ID，可以通过<br>[**LST MEASUNIT**](查询测量指标模型(LST MEASUNIT)_47814449.md)<br>命令查询获取。<br>取值范围：整数类型，取值范围为0~4294967294。<br>默认值：无。<br>配置原则：无。 |
| METRICNAME | 测量指标名称 | 可选必选说明：该参数在<br>“指标分类”<br>配置为<br>“ALL(不限分类)”<br>时为条件可选参数。<br>参数含义：测量指标名称，支持中英文模糊搜索。<br>取值范围：字符串类型，长度不超过256个字符。<br>默认值：无。<br>配置原则：若不输入，则表示该参数不作为查询的限制条件。 |
| MOIID | 对象实例ID | 可选必选说明：该参数在<br>“指标分类”<br>配置为<br>“ALL(不限分类)”<br>时为条件可选参数。<br>参数含义：对象实例ID，可以通过<br>[**DSP MEASOBJ**](查询话统测量对象实例(DSP MEASOBJ)_01094728.md)<br>命令查询获取。<br>取值范围：字符串类型，长度不超过256个字符。<br>默认值：无。<br>配置原则：无。 |

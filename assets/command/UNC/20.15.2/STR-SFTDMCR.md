# STR SFTDMCR（启动软调操作）

## 功能

**适用网元：MME**

该命令为专家维护命令，用于向主机发送软件调试命令。

## 注意事项

- 该命令执行可能造成业务受损，需要慎重使用，必须在华为技术支持人员的指导下操作。
- 该命令执行须确保软调名称和参数输入准确，否则不会出现错误提示。

## 权限

manage-ug；system-ug
G_1，管理员级别命令组；G_2，操作员级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SOFTDEBUGNAME | 软调命令名字 | 可选必选说明：必选参数<br>参数含义：该参数用于指定软件调试名称。<br>取值范围：1~31位字符串<br>默认值：无<br>配置原则：<br>- 如果输入软件调试名称为GetRunTime，则会将进程的编译时间和运行时间记录到日志(目录为：/opt/MCR_VNFC/*/*/vrpv8/logfile/)中。<br>- 如果输入其他软件调试名称，则不会记录日志，不会对系统产生影响。 |
| RUNAME | RU名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定软件调试Ru的名称。<br>数据来源：该参数可以通过<br>[**DSP RU**](../../../../../../平台服务管理/单体服务公共功能管理/系统管理/资源管理/RU管理/显示资源单元信息（DSP RU）_59103857.md)<br>命令查询获取。<br>取值范围：1~63位字符串<br>默认值：无<br>配置原则：<br>- 如果输入RU名称，则对相应RU执行该软调命令。<br>- 如果不输入该参数，则默认对所有RU执行该软调命令。 |
| PROCESSTYPE | 进程类型 | 可选必选说明：可选参数<br>参数含义：该参数用于指定软件调试进程类型。<br>数据来源：该参数可以通过<br>[**DSP PROCESSMCR**](../../../../../../平台服务管理/操作维护/VNFC公共功能管理/操作维护/系统调测/进程管理/查询MCR进程信息(DSP PROCESSMCR)_77522651.md)<br>命令查询获取。<br>取值范围：<br>- “SAP（SAP）”：表示在SAP进程上执行软调函数。<br>- “SRP（SRP）”：表示在SRP进程上执行软调函数。<br>- “OMP（OMP）”：表示在OMP进程上执行软调函数。<br>默认值：无<br>配置原则：<br>- 如果输入进程类型，则只对输入类型的进程执行该软调命令。<br>- 如果不输入该参数，则默认对所有类型的进程执行该软调命令。 |
| PROCESSNO | 同类进程序列号 | 可选必选说明：可选参数<br>参数含义：该参数用于指定软件调试进程号，使用同类进程在RU中的进程号。<br>数据来源：该参数可以通过<br>[**DSP PROCESSMCR**](../../../../../../平台服务管理/操作维护/VNFC公共功能管理/操作维护/系统调测/进程管理/查询MCR进程信息(DSP PROCESSMCR)_77522651.md)<br>命令查询获取，选取Same Process Type Ins字段显示的进程号。<br>取值范围：0~1023<br>默认值：无<br>配置原则：<br>- 如果输入进程号，则只对该进程执行该软调命令。<br>- 如果不输入该参数，则默认对所有符合条件的进程执行该软调命令。 |
| PARAM1 | 参数1 | 可选必选说明：可选参数<br>参数含义：该参数用于指定软件调试函数的第一个参数。<br>取值范围：1~63位字符串<br>默认值：无 |
| PARAM2 | 参数2 | 可选必选说明：可选参数<br>参数含义：该参数用于指定软件调试函数的第二个参数。<br>取值范围：1~63位字符串<br>默认值：无 |
| PARAM3 | 参数3 | 可选必选说明：可选参数<br>参数含义：该参数用于指定软件调试函数的第三个参数。<br>取值范围：1~63位字符串<br>默认值：无 |
| PARAM4 | 参数4 | 可选必选说明：可选参数<br>参数含义：该参数用于指定软件调试函数的第四个参数。<br>取值范围：1~63位字符串<br>默认值：无 |

## 使用实例

目前可执行的软调函数：

- 查看进程编译时间以及运行时间：
  当需要查看进程的编译和运行时间时可以使用该软调命令。执行后查询结果将打印到使用MCR_VNFC插件的网元的诊断日志(目录为：/opt/MCR_VNFC/*/*/vrpv8/logfile/)中。
  该软调函数入参说明：

  - “软件调试名称”固定为GetRunTime。
    - “Ru名称”为RU名称，可以通过[**DSP RU**](../../../../../../平台服务管理/单体服务公共功能管理/系统管理/资源管理/RU管理/显示资源单元信息（DSP RU）_59103857.md)命令查询获取。
    - “进程类型”为进程类型，可以通过[**DSP PROCESSMCR**](../../../../../../平台服务管理/操作维护/VNFC公共功能管理/操作维护/系统调测/进程管理/查询MCR进程信息(DSP PROCESSMCR)_77522651.md)命令查询获取。
    - “进程号”为进程号，可以通过[**DSP PROCESSMCR**](../../../../../../平台服务管理/操作维护/VNFC公共功能管理/操作维护/系统调测/进程管理/查询MCR进程信息(DSP PROCESSMCR)_77522651.md)命令查询获取。
    - “参数1”、“参数2”、“参数3”、“参数4”为空。
  以查看资源单元为MCR_SP_RU_0065的0号SAP进程的运行时间举例： STR SFTDMCR: SOFTDEBUGNAME="GetRunTime", RUNAME="MCR_SP_RU_0065", PROCESSTYPE=SAP, PROCESSNO=0;

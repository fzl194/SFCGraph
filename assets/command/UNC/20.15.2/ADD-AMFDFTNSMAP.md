# ADD AMFDFTNSMAP（增加漫游切片默认映射配置）

## 功能

**适用NF：AMF**

该命令用于增加漫游切片默认映射配置。在漫游场景下，不同运营商对切片的定义可能不同，在处理漫游切片业务时，需要把归属地切片映射为拜访地切片，以便完成漫游业务。

## 注意事项

- 该命令执行后只对新激活用户生效。

- 最多可输入1024条记录。
- ADD AMFROAMNSMAP配置的切片映射规则优先级高于本命令配置的缺省映射规则。
- 基于HMCC和HMNC配置的漫游切片默认映射规则优先级高于基于VMCC和VMNC配置的漫游切片默认映射规则。
- 增加漫游切片映射配置时，对已注册用户不会立即生效。如果需要立即生效，请先执行STR NGUSRBATCHDEREG命令对漫游用户启动批量去注册任务，并指定去注册类型为需要重新注册，触发用户重新注册，获取最新切片映射关系。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组

## 使用实例

如果需要增加“控制模式”为“拜访地PLMN”、“拜访地移动国家码”为“123”、“拜访地移动网号”为“45”、“拜访地默认切片服务类型”为“8”、“拜访地默认切片细分标识”为“ABC123”的漫游切片默认映射配置，执行如下命令：

```
ADD AMFDFTNSMAP: CTRLMODE=VPLMN, VMCC="123", VMNC="45", VSST=8, VSD="ABC123";
```

# MOD SAIGEO（修改服务区标识与地理坐标对照关系）

## 功能

**适用网元：SGSN**

此命令用于修改服务区标识SAI与地理位置经纬度坐标的对应关系。

## 注意事项

此命令执行后立即生效。

## 权限

manage-ug；system-ug
G_1，管理员级别命令组；G_2，操作员级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SAI | 服务区标识 | 可选必选说明：必选参数<br>参数含义：该参数用于指定服务区标识。服务区由一个或多个小区构成。服务区标识由位置区标识LAI(Location Area Identification)和服务区码SAC(ServIce Area Code)组成。位置区标识由移动国家码MCC(Mobile Country Code)、移动网络码MNC(Mobile Network Code)和位置区码LAC(Location Area Code)组成。移动国家码MCC由3位数字组成，移动网络码MNC由2～3位数字组成，位置区码是在GSM PLMN中标识一个位置区的4位十六进制数。服务区码是在一个位置区中标识一个服务区的4位十六进制数。<br>数据来源：整网规划<br>取值范围：13～14位字符串<br>默认值：无 |
| LAT | 纬度类型 | 可选必选说明：可选参数<br>参数含义：该参数用于指定位置坐标的纬度类型。<br>数据来源：整网规划<br>取值范围：<br>- “NLA(北纬)”<br>- “SLA(南纬)”<br>默认值：无<br>说明：SOUTH_LATITUDE，南纬；NORTH_LATITUDE，北纬。 |
| LAD | 纬度（度） | 可选必选说明：可选参数<br>参数含义：该参数用于指定位置坐标纬度的度值。<br>数据来源：整网规划<br>取值范围：0度～90度<br>默认值：无 |
| LAM | 纬度（分） | 可选必选说明：可选参数<br>参数含义：该参数用于指定位置坐标的纬度的分的值。<br>数据来源：整网规划<br>取值范围：0分～59分<br>默认值：无<br>说明：当LAD取值为90度时，LAM只能为0分。 |
| LAS | 纬度（秒） | 可选必选说明：可选参数<br>参数含义：该参数用于指定位置坐标的纬度的秒的值。<br>数据来源：整网规划<br>取值范围：0秒～59秒<br>默认值：无<br>说明：当LAD取值为90度时，LAS只能为0秒。 |
| LOT | 经度类型 | 可选必选说明：可选参数<br>参数含义：该参数用于指定位置坐标的经度类型。<br>数据来源：整网规划<br>取值范围：<br>- “ELO(东经)”<br>- “WLO(西经)”<br>默认值：无<br>说明：EAST_LONGITUDE，东经；WEST_LONGITUDE，西经。 |
| LOD | 经度（度） | 可选必选说明：可选参数<br>参数含义：该参数用于指定位置坐标的经度的度值。<br>数据来源：整网规划<br>取值范围：0度～180度<br>默认值：无 |
| LOM | 经度（分） | 可选必选说明：可选参数<br>参数含义：该参数用于指定位置坐标的经度的分的值。<br>数据来源：整网规划<br>取值范围：0分～59分<br>默认值：无<br>说明：当LOD取值为180度时，LOM只能为0分。 |
| LOS | 经度（秒） | 可选必选说明：可选参数<br>参数含义：该参数用于指定位置坐标的经度的秒的值。<br>数据来源：整网规划<br>取值范围：0秒～59秒<br>默认值：无<br>说明：当LOD取值为180度时，LOS只能为0秒。 |
| RAD | 服务区半径（米） | 可选必选说明：可选参数<br>参数含义：该参数用于指定服务区半径的值，是指该服务区的覆盖半径。<br>数据来源：整网规划<br>取值范围：0米～1800000米<br>默认值：无<br>说明：在LCS MT/NI/MO流程中当只能获得被定位用户的服务区时，服务区半径就是该用户坐标偏离该服务区中心坐标的误差。 |

## 操作的配置对象

- [[configobject/UNC/20.15.2/SAIGEO]] · 服务区标识与地理坐标对照关系（SAIGEO）

## 使用实例

将SAI为1230907551234 对应的地理位置坐标由北纬5度15分25秒西经35度45分55秒服务区半径为35000米修改为南纬2度12分22秒东经32度42分52秒服务区半径为45000米：

MOD SAIGEO: SAI="1230907551234", LAT=SLA, LAD=2, LAM=12, LAS=22, LOT=ELO, LOD=32, LOM=42, LOS=52, RAD=45000;

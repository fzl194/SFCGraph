# SET UGTP（设置GTP-C路径扫描参数）

## 功能

**适用网元：SGSN、MME**

该命令用于配置GTP信息表，GTP信息表包含了SPP和UPP进程一些运行控制参数。

## 注意事项

无。

## 权限

manage-ug；system-ug
G_1，管理员级别命令组；G_2，操作员级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| ICI | 空闲计数器扫描间隔（秒） | 可选必选说明：可选参数<br>参数含义：该参数用于指定GTP路径管理，空闲计数器扫描时间间隔。扫描的目的是为了老化空闲路径。时间到达以后，空闲计数器次数加1。<br>数据来源：整网规划<br>取值范围：60s~1800s<br>系统初始设置值：“180”。<br>配置原则：<br>- 建议值为180s。 |
| SSI | SM信令路径空闲计数器 | 可选必选说明：可选参数<br>参数含义：该参数用于指定GTP路径管理，SM信令路径的空闲计数器上限。上限到达以后，清除该路径的SM属性，当该路径的所有属性都被清除，该路径将被老化。<br>数据来源：整网规划<br>取值范围：10~200<br>系统初始设置值：“60”<br>配置原则：建议值为60，该参数一般不要设置。 |
| MSI | MM信令路径空闲计数器 | 可选必选说明：可选参数<br>参数含义：该参数用于指定GTP路径管理，MM信令路径的空闲计数器上限。上限到达以后，清除该路径的MM属性，当该路径的所有属性都被清除，该路径将被老化。<br>数据来源：整网规划<br>取值范围：10~200<br>系统初始设置值：“60”<br>配置原则：建议值为60，该参数一般不要设置。 |
| GMSI | GTPMAP路径空闲计数器 | 可选必选说明：可选参数<br>参数含义：该参数用于指定GTP路径管理，GTPMAP信令路径的空闲计数器上限。上限到达以后，清除该路径的GTPMAP属性，当该路径的所有属性都被清除，该路径将被老化。<br>数据来源：整网规划<br>取值范围：10~200<br>系统初始设置值：<br>“60”<br>配置原则:该参数一般不要设置。 |
| PATHVERCHKINT | 路径版本探测扫描周期（分钟） | 可选必选说明：可选参数<br>参数含义：对路径版本进行探测的周期，该探测用于将低版本路径升级到较高版本。<br>数据来源：整网规划<br>取值范围：1min~2400min<br>系统初始设置值：“60”<br>配置原则：系统缺省值为60min，该参数一般不要设置。 |

## 操作的配置对象

- [[configobject/UNC/20.15.2/UGTP]] · GTP-C路径扫描参数（UGTP）

## 使用实例

设置GTP配置信息表的各个参数：

SET UGTP: ICI=180, SSI=60, MSI=60, PATHVERCHKINT=60;

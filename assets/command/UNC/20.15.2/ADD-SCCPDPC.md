# ADD SCCPDPC（增加SCCP目的信令点）

## 功能

**适用网元：SGSN、MME、SMSF**

此命令用于增加SCCP目的信令点。

与本局信令点有SCCP消息交互的信令点都需要在本表中增加数据。

## 注意事项

- 此命令执行后立即生效。
- 此命令的最大记录数为1280。
- 目的信令点编码在同一网络中唯一。
- 目的信令点编码、本局信令点索引在表中唯一确定一条记录。
- SCCP目的信令点表中的本局信令点索引在本局信令点表中必须存在。
- 本局信令点不能够添加到SCCP目的信令点表中。
- SCCP目的信令点对应的信令点编码和网络指示语记录必须在M3UA目的实体表中存在。
- 备用/负荷分担信令点索引和目的信令点索引不能相同，且应该对应相同的本局信令点。
- 执行此命令之前，应该先执行下面的命令：
    - 配置信令点属性表:[**SET SIGATTR**](../../信令网属性管理/设置信令网属性(SET SIGATTR)_72226021.md)。
    - 配置本局信令点表：[**ADD SCCPOPC**](../SCCP本局信令点/增加SCCP本局信令点(ADD SCCPOPC)_72226009.md)。

## 权限

manage-ug；system-ug
G_1，管理员级别命令组；G_2，操作员级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| DPX | 目的信令点索引 | 可选必选说明：必选参数<br>参数含义：该参数用于指定SCCP目的信令点索引。<br>数据来源：本端规划<br>取值范围：0~1279<br>默认值：无 |
| OPX | 本局信令点索引 | 可选必选说明：必选参数<br>参数含义：该参数用于指定SCCP本局信令点索引。<br>数据来源：本端规划<br>取值范围：1~64<br>默认值：无<br>说明：该本局信令点索引对应的记录在SCCPOPC表中必须有效，可以通过命令<br>[**LST SCCPOPC**](../SCCP本局信令点/查询SCCP本局信令点(LST SCCPOPC)_26146332.md)<br>查看。 |
| DPC | 目的信令点编码 | 可选必选说明：必选参数<br>参数含义：该参数用于指定目的信令点编码，可以采用24位信令点编码或14位信令编码。<br>数据来源：与对端网元协商<br>取值范围：长度不超过8的字符串<br>默认值：无<br>说明：- 取值为0x1~0x3FFF（对应14位编码），或0x1~0xFFFFFF（对应24位编码）。<br>- 配置的信令点编码结构需要和信令属性表中对应的信令网络的网络结构保持一致。<br>- 不输入0x也取输入为十六进制值，即输入123等价于0x123。 |
| SS7T | 协议类型 | 可选必选说明：可选参数<br>参数含义：该参数用于指定信令网协议类型。表示No.7信令网所遵循的协议类型，包括ITUT和ANSI两种类型。数据来源：整网规划<br>取值范围：<br>- “ITUT_SS7(ITUT_SS7)”<br>- “ANSI_SS7(ANSI_SS7)”<br>默认值：<br>“ITUT_SS7(ITUT_SS7)”<br>配置原则：此参数配置后并不生效（该参数当前无效），当前通过<br>[**SET SIGATTR**](../../信令网属性管理/设置信令网属性(SET SIGATTR)_72226021.md)<br>命令中的SS7T配置。 |
| LDP | 负荷分担类型 | 可选必选说明：可选参数<br>参数含义：该参数用于指定负荷分担类型，指示此目的信令点是否使用负荷分担或者主备用功能。<br>数据来源：整网规划<br>取值范围：<br>- “NOUSE(不使用)”<br>- “BACKUP(主备方式)”<br>- “LOADSHARE(动态负荷分担方式)”<br>- “MULTIDPCLOADSHARE(多点负荷分担方式)”<br>默认值：<br>“NOUSE(不使用)” |
| SDPX | 负荷分担目的信令点索引 | 可选必选说明：条件必选参数<br>参数含义：该参数用于指定负荷分担信令点对应的索引值。<br>前提条件：该参数在<br>“负荷分担类型”<br>参数设置为<br>“LOADSHARE(动态负荷分担方式)”<br>时，才需要配置。数据来源：整网规划<br>取值范围：0~1279<br>默认值：无 |
| BDPX | 备用目的信令点索引 | 可选必选说明：条件必选参数<br>参数含义：该参数用于指定备用目的信令点索引。<br>前提条件：该参数在<br>“负荷分担类型”<br>参数设置为<br>“BACKUP(主备方式)”<br>时，才需要配置。<br>数据来源：整网规划<br>取值范围：0~1279<br>默认值：无 |
| DPN | 目的信令点名 | 可选必选说明：可选参数<br>参数含义：该参数用于指定目的信令点名。<br>数据来源：与对端网元协商<br>取值范围：长度不超过32的字符串<br>默认值：noname |

## 操作的配置对象

- [[UNC@20.15.2@ConfigObject@SCCPDPC]] · SCCP目的信令点（SCCPDPC）

## 使用实例

增加SCCP目的信令点，其目的信令点索引、本局信令点索引为1，目的信令点编码为0x12，支持负荷分担，负荷分担目的信令点索引为2，目的信令点名为rnc：

ADD SCCPDPC: DPX=1, OPX=1, DPC="0x12", LDP=LOADSHARE, SDPX=2, DPN="rnc";

# LST OPTLOG（查询操作日志）

## 功能

本命令用于查询操作日志。

该命令获取日志与 OM Portal 界面 “ 安全>日志审计>操作日志 ” 相同。

## 注意事项

最大限度为1000条。如果超过最大限度，则返回满足条件的最新的1000条。

## 参数

| **参数标识** | **参数名称** | **参数说明** |
| --- | --- | --- |
| OPERATIONUSER | 操作用户 | 可选必选说明：可选参数。<br>参数含义：用于命令按照操作用户名称来查询操作日志；若不输入，则表示查询前1000条操作日志。<br>取值范围：操作用户名称。可从菜单<br>“安全>用户管理”<br>中获取。<br>默认值：无。<br>配置原则：无。 |
| MENAME | 网元名称 | 可选必选说明：可选参数。<br>参数含义：用于命令按照网元名称来查询操作日志，<br>[**LST ME**](../../系统管理/版本信息/查询网元配置信息（LST ME）_47084797.md)<br>命令可获取；若不输入，则表示查询前1000条操作日志。<br>取值范围：字符串类型，输入长度范围为0~64。<br>默认值：无。<br>配置原则：无。 |
| OPERATIONNAME | 操作名称 | 可选必选说明：可选参数。<br>参数含义：用于命令按照操作名称来查询操作日志；若不输入，则表示查询前1000条操作日志。<br>取值范围：字符串类型，输入长度范围为0~100。<br>默认值：无。<br>配置原则：无。 |
| SDT | 开始时间 | 可选必选说明：可选参数。<br>参数含义：用于命令按照操作日志开始时间来查询操作日志；若不输入，则表示查询前1000条操作日志。<br>取值范围：1990/01/01 00:00:00-2037/12/31 23:59:59。<br>默认值：无。<br>配置原则：无。 |
| EDT | 结束时间 | 可选必选说明：可选参数。<br>参数含义：用于命令按照操作日志结束时间来查询操作日志；若不输入，则表示查询前1000条操作日志。<br>取值范围：1990/01/01 00:00:00-2037/12/31 23:59:59。<br>默认值：无。<br>配置原则：无。 |
| LEVEL | 级别 | 可选必选说明：可选参数。<br>参数含义：用于命令按照操作级别来查询操作日志；若不输入，则表示查询前1000条操作日志。<br>取值范围：<br>- Minor(次要)<br>- Important(重要)<br>- Urgent(紧急)<br>- Auto(自动查询)<br>- Manual(手动查询)<br>默认值：无。<br>配置原则：无。 |
| RESULT | 操作结果 | 可选必选说明：可选参数。<br>参数含义：用于命令按照操作结果状态来查询操作日志；若不输入，则表示查询前1000条操作日志。<br>取值范围：<br>- Successful(成功)<br>- Failed(失败)<br>- PartiallySuccessful(部分成功)<br>- Executing(执行中)<br>默认值：无。<br>配置原则：无。 |

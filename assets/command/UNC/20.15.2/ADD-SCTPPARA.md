# ADD SCTPPARA（增加SCTP协议参数）

## 功能

**适用网元：SGSN、MME、AMF** 、 **SMSF**

该命令用于增加基于IP的宽带信令SCTP(Stream Control Transmission Protocol)偶联的协议参数。偶联就是两个SCTP端点通过SCTP协议规定的4步握手机制建立起来的进行数据传递的逻辑联系或者通道。

## 注意事项

- 该命令执行后立即生效。
- 该表最大记录数为1024。
- 需要根据现网网络状况、接口类型，与对端网元协商合适的SCTP协议参数。
- 该命令针对不同接口有两组推荐值。
  对于S6a/S6d/SGs/Gr/Iu接口以及采用双主端口组网方式的S1-MME接口，推荐值与系统默认参数一致：

  - “RTO最小值（ms）”：“500”
    - “RTO最大值（ms）”：“1500”
    - “RTO初始值（ms）”：“500”
    - “心跳间隔（ms）”：“3000”
    - “偶联最大重发次数”：“5”
    - “路径最大重发次数”：“4”
  对于采用主备端口组网方式的S1-MME接口，推荐采用如下取值：

  - “RTO最小值（ms）”：“1000”
    - “RTO最大值（ms）”：“3000”
    - “RTO初始值（ms）”：“1000”
    - “心跳间隔（ms）”：“5000”
    - “偶联最大重发次数”：“10”
    - “路径最大重发次数”：“5”
- Diameter链路不支持ADLER32校验和算法。

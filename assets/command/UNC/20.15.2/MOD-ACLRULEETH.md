# MOD ACLRULEETH（修改以太ACL规则）

## 功能

该命令用于修改以太ACL规则，可以修改要匹配的源目的MAC地址，VLAN等。

## 注意事项

- 该命令执行后立即生效。
- 修改ACL规则前必须先执行ADD ACLRULEETH创建相应的规则。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组

## 使用实例

用户使用流分类，需要将过滤源MAC地址修改为00E0-FC12-3456的报文，可以修改以太ACL规则：

```
MOD ACLRULEETH:ACLNAME="4000",ACLRULENAME="rule_5",ACLACTION=Deny,ACLSRCMAC="00E0-FC12-3456";
```

# LST CFGSNSSAI（查询支持的S-NSSAI）

## 功能

**适用NF：NSSF**

该命令用于查询NSSF支持的S-NSSAI列表。

## 注意事项

查询条件都不填时，即查询所有。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

| --- | --- | --- |

## 使用实例

- 若运营商希望查询所有的数据，执行下列命令。
  ```
  LST CFGSNSSAI:;
  %%LST CFGSNSSAI:;%%
  RETCODE = 0  Operation succeeded

  The result is as follows
  ------------------------
  索引  移动国家码  移动网号  切片服务类型  切片细分标识  优先级  激活状态  描述  切片细分标识区间的起始值  切片细分标识区间的结束值  

  1      460            05            1               NULL       1024     TRUE     NULL         111111          222222        
  2      460            03            1              111111      1024     TRUE     NULL         NULL            NULL          
  (结果个数 = 2)
  ```
- 若运营商希望查询移动国家码为"460"、移动网号为"03"，支持的切片或者切片区间，执行下列命令。
  ```
  LST CFGSNSSAI: MCC="460", MNC="03";
  %%LST CFGSNSSAI: MCC="460", MNC="03";%%
  RETCODE = 0  操作成功

  结果如下
  --------

      移动网号  =  03

  切片细分标识  =  111111
      优先级为  =  1024
      激活状态  =  TRUE
  	    描述  =  NULL
  切片细分标识区间的起始值 = NULL
  切片细分标识区间的结束值 = NULL
  (结果个数 = 1)
  ```

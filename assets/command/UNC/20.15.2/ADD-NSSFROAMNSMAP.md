# ADD NSSFROAMNSMAP（增加漫游切片映射配置）

## 功能

**适用NF：NSSF**

该命令用于增加漫游切片映射配置。在漫游场景下，不同运营商对切片的定义可能不同，在处理漫游切片业务时，需要把归属地切片映射为拜访地切片，以便完成漫游业务。

## 注意事项

- 该命令执行后立即生效。

- 最多可输入65535条记录。由于当前协议标准没有通知AMF变更的机制，当NSSF上配置的切片映射规则发生变更时，需要NSSF管理的所有AMF重新从NSSF获取变更后的切片映射关系。
- HSD与HSD区间（HSDSTART与HSDEND）配置二选一，不能同时使用。
- HSD配置时，HSDSTART与HSDEND必须同时为空，如果HSD也为空，HSD需要填入FFFFFF补齐。
- HSD区间配置，HSDSTART与HSDEND必须同时有值。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组

## 使用实例

- 假如运营商希望增加一条归属地移动国家码为"245"、归属地移动网号为"38"、归属地切片服务类型为1、归属地切片细分标识为"123FFF"、拜访地移动国家码为"123"、拜访地移动网号为"45"、拜访地切片服务类型为1、拜访地切片细分标识为"FFFFFF"的记录，执行下列命令：
  ```
  ADD NSSFROAMNSMAP: HMCC="245", HMNC="38", HSST=1, HSD="123FFF", VMCC="123", VMNC="45", VSST=1, VSD="FFFFFF";
  ```
- 假如运营商希望增加一条归属地移动国家码为"245"、归属地移动网号为"38"、归属地切片服务类型为1、归属地切片细分标识区间的起始值为"000000"、归属地切片细分标识区间的结束值为"123FFF"、拜访地移动国家码为"123"、拜访地移动网号为"45"、拜访地切片服务类型为1、拜访地切片细分标识为"FFFFFF"的记录，执行下列命令：
  ```
  ADD NSSFROAMNSMAP: HMCC="245", HMNC="38", HSST=1, HSDSTART="000000", HSDEND="123FFF", VMCC="123", VMNC="45", VSST=1, VSD="FFFFFF";
  ```

# MOD CDRSTAT（修改话单统计）

## 功能

**适用NF：NCG**

该命令用于修改话单统计任务配置。通常在任务配置错误需要变更的时候才执行此命令。

## 注意事项

- 该命令中“接入网元分组标识”、“通道名称”、“数据网络名称”、“话单文件开始时间”、“话单文件结束时间”、“是否开启匿名化”的参数有变化时会重启任务。
- 如果任务重启会删除话单统计任务生成的结果文件，在执行修改操作之前，需确认具体的修改参数。

## 权限

manage-ug；system-ug
G_1，管理员级别命令组；G_2，操作员级别命令组

## 使用实例

修改话单统计任务的任务时间段，配置举例如下：

```
MOD CDRSTAT: CDRSTATID="cdrstat1", STARTTIME=02&00, ENDTIME=00&50;
```

# SET GTPUPARA（设置GTP-U协议参数）

## 功能

**适用网元：SGSN、MME**

此命令用于配置GTP-U用户面协议参数。GTP-U参数与Iu接口用户面和Gn/Gp/S11/S4接口相关。

## 注意事项

- 系统初次运行时，会执行系统初始设置值。
- 该命令执行后立即生效。

## 权限

manage-ug；system-ugDSP GTPUPATHNUM
G_1，管理员级别命令组；G_2，操作员级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| GTPUUPECHO | GTPU上游接口发送Echo信令 | 可选必选说明：可选参数<br>参数含义：该参数定义用户面上游接口（Gn接口、Gp接口、S11接口和S4接口）是否发送ECHO信令，ECHO信令用于路径管理。参考3GPP 29060。<br>数据来源：整网规划<br>取值范围：<br>- “OFF(关闭)”<br>- “ON(启用)”<br>系统初始设置值：<br>“ON(启用)”<br>。 |
| GTPUDNECHO | GTPU下游接口发送Echo信令 | 可选必选说明：可选参数<br>参数含义：该参数定义用户面下游接口（Iu接口）是否发送ECHO信令。参考3GPP 29060。<br>数据来源：整网规划<br>取值范围：<br>- “OFF(关闭)”<br>- “ON(启用)”<br>系统初始设置值：<br>“ON(启用)”<br>。 |
| GTPUEI | GTPU路径Echo探测发送间隔 | 可选必选说明：可选参数<br>参数含义：该参数定义用户面GTP-U路径发送Echo Request的间隔时间。参考3GPP 29060。<br>说明：- 一对节点最多支持20480条GTPU路径发送Echo探测。<br>- 特例：处于故障状态下的GTP-U路径的Echo探测间隔取120秒、2*“GTPU路径的Echo探测的T3 Response时长”*“GTPU路径的Echo探测的消息发送次数”取值中大的值，不受本参数控制。<br>数据来源：整网规划<br>取值范围：120~3600 秒<br>系统初始设置值：<br>“239”<br>。 |
| GTPUT3 | GTPU路径的Echo探测的T3 Response时长 | 可选必选说明：可选参数<br>参数含义：该参数定义用户面GTP-U路径的Echo探测的T3 Response时长。T3 Response时长表示等待一条路径探测请求消息的响应消息的最大时长。参考3GPP 29060。<br>数据来源：整网规划<br>取值范围：1s~20s<br>系统初始设置值：<br>“7”<br>。 |
| GTPUN3 | GTPU路径的Echo探测的消息发送次数 | 可选必选说明：可选参数<br>参数含义：该参数定义用户面GTP-U路径的Echo探测消息的重发次数。如果<br>[**ADD GTPUPATHDP**](../GTP-U路径管理自定义策略/增加GTP-U路径管理自定义策略(ADD GTPUPATHDP)_72225521.md)<br>未配置GTP-U路径管理自定义策略，则当重发次数达到<br>“GTPUN3”<br>时还没有收到响应消息，路径状态会变成故障；否则按照<br>[**ADD GTPUPATHDP**](../GTP-U路径管理自定义策略/增加GTP-U路径管理自定义策略(ADD GTPUPATHDP)_72225521.md)<br>配置的策略进行GTP-U路径的故障判定。参考3GPP 29060。<br>数据来源：整网规划<br>取值范围：1次~6次<br>系统初始设置值：<br>“2”<br>。<br>说明：特例：处于故障状态下的GTP-U路径的Echo探测重发次数为1，不受本参数控制。 |
| PRVEXT | 发送私有信息 | 可选必选说明：可选参数<br>参数含义：该参数指示是否发送私有信息，私有信息包含了运行商或设备商定义的信息。<br>数据来源：整网规划<br>取值范围：<br>- “OFF(关闭)”<br>- “ON(打开)”<br>系统初始设置值：<br>“OFF(关闭)”<br>。<br>说明：此参数选择<br>“ON(打开)”<br>时，需要设置参数<br>“PE（私有信息）”<br>、<br>“PEID（企业号）”<br>。 |
| PEID | 企业号 | 可选必选说明：可选参数<br>参数含义：运营商或设备商的企业编号，是私有信息的一部分。<br>数据来源：整网规划<br>取值范围：0~65535<br>系统初始设置值：<br>“2011”<br>。 |
| PE | 私有信息 | 可选必选说明：可选参数<br>参数含义：私有信息包含了运营商或设备商定义的信息。<br>前提条件：该参数在<br>“PRVEXT”<br>设置为<br>“ON”<br>时有效。<br>取值范围：长度不超过255的字符串<br>系统初始设置值：<br>“usn”<br>。 |
| GTPEXT | 检查GTP扩展头类型个数上限 | 可选必选说明：可选参数<br>参数含义：该参数定义检查用户面GTP-U扩展头类型个数上限。系统检测的用户面GTP扩展头类型个数的最大数值，若大于该值则判为异常GTP-U报文，异常GTP-U报文将被丢弃。<br>数据来源：整网规划<br>取值范围：1~255<br>系统初始设置值：<br>“10”<br>。 |
| GTPEXLEN | 检查GTP扩展头类型长度上限 | 可选必选说明：可选参数<br>参数含义：该参数定义检查用户面GTP-U扩展头类型长度上限。当网元接收到协议未定义的扩展头，且其长度大于该阈值，判定为异常GTP-U报文，异常GTP-U报文将被丢弃。<br>数据来源：整网规划<br>取值范围：1~255字节<br>系统初始设置值：<br>“100”<br>。 |
| GTPSUPEXTLIST | 检查GTP扩展头列表个数上限 | 可选必选说明：可选参数<br>参数含义：该参数定义检查用户面GTP-U支持扩展头列表通知长度上限。当网元接收到来自RNC或GSN的支持扩展头列表通知消息，且列表中扩展头列表长度大于该阈值，就认为该扩展头列表为巨型扩展头列表，判定为异常GTP-U报文，异常GTP-U报文将被丢弃。<br>数据来源：整网规划<br>取值范围：1~255字节<br>系统初始设置值：<br>“100”<br>。 |
| POTHD | 路径数过载门限（%） | 可选必选说明：可选参数<br>参数含义：该参数定义了GTP-U路径占用比（当前路径数/系统支持路径数）的过载门限，包含了系统中所有的路径。如果当前路径占用比超过此门限值，并且此前不存在用户面资源过载告警(告警ID:11605)，将触发该告警。<br>数据来源：整网规划<br>取值范围：0~100<br>系统初始设置值：<br>“85”<br>。<br>配置原则：<br>“路径过载门限（POTHD）”<br>的值应该大于<br>“路径过载恢复门限（PNTHD）”<br>的值。 |
| PNTHD | 路径数过载恢复（%） | 可选必选说明：可选参数<br>参数含义：该参数定义了GTP-U路径占用比（当前路径数/系统支持路径数）的过载恢复门限，包含了系统中所有的路径。如果当前路径占用比小于此门限值，并且此前已经存在用户面资源过载告警(告警ID:11605)，则恢复该告警。<br>数据来源：整网规划<br>取值范围：0~100<br>系统初始设置值：<br>“80”<br>。 |
| PCTHD | 路径数拥塞门限（%） | 可选必选说明：可选参数<br>参数含义：该参数定义了GTP-U路径占用比（当前路径数/系统支持路径数）拥塞门限，包含了系统中所有的路径。如果当前路径占用比超过此门限值，并且此前不存在用户面资源拥塞告警(告警ID:11606)，将触发该告警。<br>数据来源：整网规划<br>取值范围：0~100<br>系统初始设置值：<br>“95”<br>。<br>配置原则：<br>“路径拥塞门限（PCTHD）”<br>的值应该大于<br>“路径拥塞恢复门限（PRTHD）”<br>的值。 |
| PRTHD | 路径数拥塞恢复（%） | 可选必选说明：可选参数<br>参数含义：该参数定义了GTP-U路径占用比（当前路径数/系统支持路径数）拥塞恢复门限，包含了系统中所有的路径。如果当前路径占用比小于此门限值，并且此前已经存在用户面资源拥塞告警(告警ID:11606)，则恢复该告警。<br>数据来源：整网规划<br>取值范围：0~100<br>系统初始设置值：<br>“90”<br>。 |
| UDPCHKSW | GTPU信令面报文校验功能开关 | 可选必选说明：可选参数<br>参数含义：该参数用于设置是否支持对GTP-U信令面报文的UDP校验功能。开关打开时，支持UDP校验功能，此时发送GTP-U信令面报文需要填写UDP校验和，接收GTP-U信令面报文需要检查UDP校验和，并且校验失败的报文将被丢弃。开关关闭时，不支持UDP校验功能，此时发送GTP-U信令面报文不需要填写UDP校验和，接收GTP-U信令面报文需要检查UDP校验和，并且校验失败的报文将会继续接收，不会被丢弃。<br>数据来源：整网规划<br>取值范围：<br>- “OFF(关闭)”<br>- “ON(打开)”<br>系统初始设置值：<br>“ON(打开)”<br>。 |

## 使用实例

设置GTP-U协议参数：

SET GTPUPARA: GTPUUPECHO=ON, GTPUDNECHO=ON, GTPUEI=239, GTPUT3=7, GTPUN3=2, GTPEXT=10, GTPEXLEN=200, GTPSUPEXTLIST=100, POTHD=85, PNTHD=80, PCTHD=95, PRTHD=90, UDPCHKSW=ON;

# RMV CACHEUPDATEPLCY（删除缓存更新策略）

## 功能

**适用NF：AMF、SMF、NRF、NSSF、SMSF、NCG**

该命令用于删除缓存更新策略。

## 注意事项

该命令执行后立即生效。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组

## 操作的配置对象

- [[UNC@20.15.2@ConfigObject@CACHEUPDATEPLCY]] · 缓存更新策略（CACHEUPDATEPLCY）

## 使用实例

运营商A需要删除缓存更新策略，对端NF类型为OCS。

```
RMV CACHEUPDATEPLCY: PEERTYPE=NfCUSTOM_OCS;
```

# SET PFRES（设置带宽资源管理参数）

## 功能

**适用网元：SGSN、MME**

该命令用于设置支持用户面GTP-U转发的资源管理需要的参数。

## 注意事项

- 系统初次运行时，会执行系统初始设置值。
- 该命令执行后立即生效。

## 权限

manage-ug；system-ug
G_1，管理员级别命令组；G_2，操作员级别命令组

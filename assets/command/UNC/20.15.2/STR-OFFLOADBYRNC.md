# STR OFFLOADBYRNC（启动RNC迁移任务）

## 功能

**适用网元：SGSN**

此命令用于启动RNC迁移任务：

1. 启动以RNC为单位的迁移任务，如指定目的SGSN则将本SGSN上指定RNC下的用户按设定的迁入百分比迁移到各个目的SGSN中。
2. 未指定目的SGSN则由RAN侧选择目的SGSN。

## 注意事项

- 此命令执行后立即生效。
- 启动迁移前必须确保已通过[**SET OFFLOADINF**](设置迁移配置表（SET OFFLOADINF）_72345695.md)命令设置过迁移参数信息，且保证迁移参数中的“NULL NRI值”已配置，否则将无法启动迁移任务。
- 迁移启动前需确保已在指定RNC上将本SGSN的状态设置为“Offload”，以避免指定RNC将被迁移用户和漫游至该指定RNC节点的新用户接入到本SGSN。
- 此命令将导致指定RNC下的用户被迁移到其它SGSN必须慎重使用。
- 所指定的RNC必须支持Iu-flex功能。
- 如果系统当前已启动DCN迁移或POOL迁移任务且未结束，则不能再启动新的迁移任务。请执行[**DSP DCNOFFLOAD**](../DCN迁移控制/查询DCN迁移状态(DSP DCNOFFLOAD)_26305932.md)和[**DSP OFFLOAD**](显示迁移进度(DSP OFFLOAD)_72345691.md)命令查询系统迁移任务状态。

## 使用实例

将本SGSN中RNC移动国家代码为123，移动网号为00，RNC标识为1104下的用户迁移到其它SGSN，不指定目的SGSN：

STR OFFLOADBYRNC: RNCMCC_1="123", RNCMNC_1="00", RNCID_1=1104;

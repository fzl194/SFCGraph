# MOD NCGALMSRV（修改告警服务器相关配置信息）

## 功能

**适用NF：NCG**

该命令用于修改告警服务器相关配置信息。

## 注意事项

- 该命令执行后立即生效。

- 修改IPTYPE或IP主/备用地址后，所有活动告警会被重新发送给SMSF。
- 如果一个告警ID同时在派单告警列表、不派单告警列表中配置，则不派单告警列表配置生效。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组

## 使用实例

修改告警服务器配置标识为AlarmServerCfg的配置中主用告警服务器IP地址为192.168.100.3：

```
MOD NCGALMSRV:ALMSRVNAME="AlarmServerCfg",IPTYPE=IPV4,PIPV4="192.168.100.3";
```

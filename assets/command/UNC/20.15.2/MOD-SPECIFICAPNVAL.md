# MOD SPECIFICAPNVAL（修改用户APN与消息交互使用APN的映射关系）

## 功能

**适用NF：SMF、PGW-C、GGSN**

该命令用于修改用户使用的APN与信令流程中使用的指定APN的映射关系。用户使用的APN包括：别名APN、虚拟APN或真实APN，信令流程包括：PCRF交互、PCF交互、CG话单、同AAA鉴权交互、计费交互以及同OCS和CHF交互。

## 注意事项

- 该命令执行后只对新激活用户生效。

- 若一条数据中的所有的映射APN都为NULL时，删除该条记录。
- 输入“AAAAUTHAPN”时，不检查是否与用户APN取值相同。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组

## 使用实例

- 运营商需要修改一个用户APN和上报APN之间的映射关系，用户APN名称为“huawei.com”，上报APN名称为“reportname”，上报对象选择PCRF，配置命令如下：
  ```
  MOD SPECIFICAPNVAL: SUBSCRIBERAPN="huawei.com", PCRFAPN="reportname";
  ```
- 运营商需要修改映射关系，使上报特殊APN服务仅对连接在名为“huawei.com”的APN下的漫游用户生效，配置命令如下：
  ```
  MOD SPECIFICAPNVAL: SUBSCRIBERAPN="huawei.com", ONLYROAMSWITCH=ENABLE, PCRFAPN="reportname";
  ```

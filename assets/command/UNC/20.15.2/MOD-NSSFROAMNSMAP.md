# MOD NSSFROAMNSMAP（修改漫游切片映射配置）

## 功能

**适用NF：NSSF**

该命令用于修改漫游切片映射配置。

## 注意事项

- 该命令执行后立即生效。

- 由于当前协议标准没有通知AMF变更的机制，当NSSF上配置的切片映射规则发生变更时，需要NSSF管理的所有AMF重新从NSSF获取变更后的切片映射关系。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组

## 使用实例

- 修改根据归属地切片细分标识添加的漫游切片映射配置中的拜访地切片服务类型为2，执行如下命令：
  ```
  MOD NSSFROAMNSMAP: HMCC="245", HMNC="38", HSST=1, HSD="123FFF", VMCC="123", VMNC="45", VSST=2;
  ```
- 修改根据归属地切片细分标识区间添加的漫游切片映射配置中的拜访地切片服务类型为2，执行如下命令：
  ```
  MOD NSSFROAMNSMAP: HMCC="245", HMNC="38", HSST=1, HSDSTART="000000", HSDEND="123FFF", VMCC="123", VMNC="45", VSST=2;
  ```

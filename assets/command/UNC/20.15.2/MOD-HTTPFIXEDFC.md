# MOD HTTPFIXEDFC（修改HTTP流控组固定速率流控信息）

## 功能

![](修改HTTP流控组固定速率流控信息（MOD HTTPFIXEDFC）_83653662.assets/notice_3.0-zh-cn_2.png)

该命令中是否携带Retry-After消息头参数可能会导致对端网元不发请求消息，需慎用。

该命令用于修改HTTP固定速率流控门限值信息。

## 注意事项

- 该命令执行后立即生效。

- 流控阈值不支持修改为零，如果涉及流控模式的切换，支持在原有模式基础上新增流控模式，不支持删除原有模式基础上新增流控模式。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组

## 操作的配置对象

## 使用实例

修改HTTP单链路粒度固定速率流控门限值，将发送消息流控门限从1000改为100：

```
MOD HTTPFIXEDFC: GROUP=1, SENDTHD=100;
```

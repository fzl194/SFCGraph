# RMV AREATZ（删除区域时区参数）

## 功能

**适用网元：SGSN、MME**

此命令用于删除一条区域时区记录。即删除特定区域与时区标识的映射关系。

## 注意事项

此命令执行后，只有在UE触发位置移动后删除的操作才会对该UE生效。

## 权限

manage-ug；system-ug
G_1，管理员级别命令组；G_2，操作员级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| AREA | 区域范围 | 可选必选说明：必选参数<br>参数含义：该参数用于指定该区域的标识类型。<br>取值范围：<br>- “LA(位置区)”：表示该区域标识类型为位置区。<br>- “RA(路由区)”：表示该区域标识类型为路由区。<br>- “TA(跟踪区)”：表示该区域标识类型为跟踪区。<br>默认值：无 |
| MCC | 移动国家代码 | 可选必选说明：必选参数<br>参数含义：该参数用于指定移动国家代码。<br>取值范围：3位的十进制数字<br>默认值：无 |
| MNC | 移动网号 | 可选必选说明：必选参数<br>参数含义：该参数用于指定移动网号。<br>取值范围：2～3位的十进制数字<br>默认值：无 |
| LAC | 位置区域码 | 可选必选说明：条件必选参数<br>参数含义：该参数用于指定位置区域码。<br>取值范围：0x0000～0xFFFF<br>默认值：无<br>说明：该参数在<br>“AREA(区域范围)”<br>设置为<br>“LA(位置区)”<br>或<br>“RA(路由区)”<br>时生效。 |
| LACRANGE | 位置区域码范围 | 可选必选说明：可选参数<br>参数含义：该参数用于指定位置区域码范围的结束值。<br>取值范围：0x0000～0xFFFF<br>默认值：无<br>说明：该参数在<br>“AREA(区域范围)”<br>设置为<br>“LA(位置区)”<br>时生效。 |
| RAC | 路由区域码 | 可选必选说明：条件必选参数<br>参数含义：该参数用于指定路由区域码。<br>取值范围：0x00～0xFF<br>默认值：无<br>说明：该参数在<br>“AREA(区域范围)”<br>设置为<br>“RA(路由区)”<br>时生效。 |
| RACRANGE | 路由区域码范围 | 可选必选说明：可选参数<br>参数含义：该参数用于指定路由区域码范围的结束值。<br>取值范围：0x00～0xFF<br>默认值：无<br>说明：该参数在<br>“AREA(区域范围)”<br>设置为<br>“RA(路由区)”<br>时生效。 |
| TAC | 跟踪区域码 | 可选必选说明：条件必选参数<br>参数含义：该参数用于指定该跟踪区域码。<br>取值范围：0x0000～0xFFFF<br>默认值：无<br>说明：该参数在<br>“AREA(区域范围)”<br>设置为<br>“TA(跟踪区)”<br>时生效。 |
| TACRANGE | 跟踪区域码范围 | 可选必选说明：可选参数<br>参数含义：该参数用于指定跟踪区域码范围的结束值。<br>取值范围：0x0000～0xFFFF<br>默认值：无<br>说明：前提条件：该参数在<br>“AREA(区域范围)”<br>设置为<br>“TA(跟踪区)”<br>时生效。 |

## 操作的配置对象

- [[configobject/UNC/20.15.2/AREATZ]] · 区域时区参数（AREATZ）

## 使用实例

删除一条位置区域时区记录，将 “区域范围” 为 “LA(位置区)” ， “移动国家代码” 为 “123” ， “移动网号” 为 “03” ， “位置区域码” 为 “112” ， “位置区域码范围” 为 “350” 的位置区与时区的映射关系删除：

RMV AREATZ: AREA=LA, MCC="123", MNC="03", LAC="112", LACRANGE="350";

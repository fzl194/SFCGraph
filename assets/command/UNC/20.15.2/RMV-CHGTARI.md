# RMV CHGTARI（删除费率时段配置）

## 功能

**适用网元：SGSN**

该命令用来在费率时段配置表中删除一条费率时段配置。

## 注意事项

- 该命令执行后立即生效。
- “费率类型”、“起始时间”、“结束时间”和“计费属性”确定某一条费率时段记录。

## 权限

manage-ug；system-ug
G_1，管理员级别命令组；G_2，操作员级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| TT | 费率类型 | 可选必选说明：必选参数<br>参数含义：该参数用于表示费率时段的费率类型。<br>取值范围：<br>- “WORK(工作日)”：表示该费率时段适用于工作日。<br>- “WEEKEND(周休日)”：表示该费率时段适用于周休日。<br>- “FESTIVAL(节假日)”：表示该费率时段适用于节假日。<br>默认值：无 |
| ST | 起始时间 | 可选必选说明：必选参数<br>参数含义：该参数用于费率时段的起始时间。<br>取值范围：00&00&00~23&59&59<br>默认值：无 |
| ET | 结束时间 | 可选必选说明：必选参数<br>参数含义：该参数用于费率时段的结束时间。<br>取值范围：00&00&00~23&59&59<br>默认值：无 |
| CC | 计费属性 | 可选必选说明：必选参数<br>参数含义：该参数用于表示费率时段是对普通计费属性用户、预付费计费属性用户、包月制计费属性用户还是对实时计费属性用户有效。<br>取值范围：<br>- “NORMAL(普通计费)”：表示普通计费属性，按照此种方式计费的用户按照常规的方式支付费用。<br>- “PREPAID(预付费)”：表示预付费计费属性，按照此种方式计费的用户在获取某种服务之前需要预支付一定的费用。<br>- “FLATRATE(包月制)”：表示包月制计费属性，按照此种方式计费的用户在一个月内的收费是固定的。<br>- “HOTBILLING(实时计费)”：表示实时计费属性，按照此种方式计费的用户将在短时间或流量达到某个值时及时生成话单，保证运营商对此类用户及时收费。<br>默认值：无 |

## 操作的配置对象

- [[configobject/UNC/20.15.2/CHGTARI]] · 费率时段配置（CHGTARI）

## 使用实例

删除一条包月制计费属性的费率时段记录：费率类型为周休日、起始时间为11点4分、结束时间为23点4分，配置格式为：

RMV CHGTARI: TT=WEEKEND, ST=11&04&00, ET=23&04&00, CC=FLATRATE;

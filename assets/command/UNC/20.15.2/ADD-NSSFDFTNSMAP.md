# ADD NSSFDFTNSMAP（增加漫游切片默认映射配置）

## 功能

**适用NF：NSSF**

该命令用于增加漫游切片默认映射配置。在漫游场景下，不同运营商对切片的定义可能不同，在处理漫游切片业务时，需要把归属地切片映射为拜访地切片，以便完成漫游业务。

## 注意事项

- 该命令执行后立即生效。

- 由于当前协议标准没有通知AMF变更的机制，当NSSF上配置的切片映射规则发生变更时，需要NSSF管理的所有AMF重新从NSSF获取变更后的切片映射关系。
- 最多可输入1024条记录。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组

| HMCC | 归属地移动国家码 | 可选必选说明：该参数在"CTRLMODE"配置为"HPLMN"时为条件必选参数。<br>参数含义：该参数用于指定归属地移动国家码。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度是3。<br>默认值：无<br>配置原则：无 |

## 使用实例

如果需要增加“拜访地移动国家码”为“123”、“拜访地移动网号”为“45”、“拜访地默认切片服务类型”为“8”、“拜访地默认切片细分标识”为“ABC123”的漫游切片默认映射配置，执行如下命令：

```
ADD NSSFDFTNSMAP: CTRLMODE=VPLMN, VMCC="123", VMNC="45", VSST=8, VSD="ABC123";
```

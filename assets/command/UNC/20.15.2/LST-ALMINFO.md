# LST ALMINFO（查询告警配置信息）

## 功能

本命令用于查询系统内告警配置信息。

## 注意事项

默认查询64条，最大支持1000条告警。

## 参数

| **参数标识** | **参数名称** | **参数说明** |
| --- | --- | --- |
| MEID | 网元ID | 可选必选说明：可选参数。<br>参数含义：用于指示系统需要按照哪个网元来查询告警配置信息。<br>取值范围：0~65535<br>默认值：无。<br>配置原则：若不输入，则表示查询所有网元的告警配置信息，网元ID可以通过<br>[**LST ME**](../../系统管理/版本信息/查询网元配置信息（LST ME）_47084797.md)<br>命令查询。 |
| SAID | 起始告警ID | 可选必选说明：可选参数。<br>参数含义：用于指示系统查询时，告警ID的取值下限。<br>取值范围：0~999999999<br>默认值：无。<br>配置原则：若不输入，则表示取值下限<br>**大于等于**<br>**0**<br>且<br>**小于等于**<br>结束告警ID。 |
| MASKED | 是否屏蔽 | 可选必选说明：必选参数。<br>参数含义：用于指示告警是否屏蔽的标识。<br>取值范围：<br>- “ALL(全部)”：表示查询屏蔽和未屏蔽告警。<br>- “UNMASKED(未屏蔽)”<br>：表示只查询未屏蔽告警。<br>- “MASKED(已屏蔽)”：标识只查询屏蔽告警。<br>默认值：<br>“ALL(全部)”<br>。<br>配置原则：无。 |
| EAID | 结束告警ID | 可选必选说明：可选参数。<br>参数含义：用于指示系统查询时，告警ID的取值上限。<br>取值范围：0~999999999<br>默认值：无。<br>配置原则：若不输入，则表示取值上限<br>**小于等于999999999**<br>且<br>**大于等于**<br>起始告警ID。 |
| MASK_TYPE | 屏蔽类型 | 可选必选说明：可选参数。<br>参数含义：用于指示系统需要以哪种屏蔽类型来查询告警配置信息。<br>取值范围：<br>- “BYID(按告警ID)”：表示按告警ID查询告警配置信息。<br>- “BYPARAMS(按告警参数)”：表示按告警参数查询告警配置信息。<br>默认值：<br>“BYID(按告警ID)”<br>。<br>配置原则：若不输入，则表示按告警ID查询告警配置信息。 |
| CNT | 返回记录数 | 可选必选说明：可选参数。<br>参数含义：用于指示系统返回记录条数的最大值。<br>取值范围：1~1000<br>默认值：无。<br>配置原则：若不输入，则表示返回记录的最大数目是64。 |

## 使用实例

1. 按告警ID查询告警配置信息：
  ```
  %%LST ALMINFO: MEID=0, SAID=5521, EAID=5521, MASKED=ALL, CNT=1;%%
  RETCODE = 0  操作成功

  告警配置信息
  ------------
    网元ID  =  0
  网元名称  =  omteam15-v6-2
    告警ID  =  5521
  告警名称  =  服务中心和服务通信故障
  是否屏蔽  =  已屏蔽
  (结果个数 = 1)

  统计信息
  --------
  符合查询条件的总记录数  =  1
          实际返回记录数  =  1
  (结果个数 = 1)

  ---    END
  ```
2. 按告警参数查询已屏蔽的告警配置信息：
  ```
  %%LST ALMINFO: MASKED=MASKED, MASK_TYPE=BYPARAMS;%% 
  RETCODE = 0  操作成功  

  告警配置信息 
  ------------   
  网元ID    =  12 
  网元名称  =  superom07-v4   
  告警ID    =  36161 
  告警名称  =  GO服务进程故障告警 
  屏蔽规则  =  进程号=0,进程名=Test 
  (结果个数 = 1)  

  统计信息 
  -------- 
  符合查询条件的总记录数  =  1         
          实际返回记录数  =  1 
  (结果个数 = 1)  

  ---    END
  ```

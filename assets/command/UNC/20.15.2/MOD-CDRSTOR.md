# MOD CDRSTOR（修改话单存储）

## 功能

**适用NF：NCG**

该命令用于修改原始话单、第一份最终话单或第二份最终话单的存储规则。

## 注意事项

- 该命令执行后立即生效。
- 该命令除最终话单文件名的最大序列号外其他参数均可动态生效。
- 可能重复帧的存储规则不允许修改。
- 当前版本不支持Bi口SingleIP功能，请勿开启。

## 权限

manage-ug；system-ug
G_1，管理员级别命令组；G_2，操作员级别命令组

## 使用实例

修改第二份最终话单default通道下的话单存储规则：

- 只按大小生成第二份最终话单，话单大小为1024K。
- 定制第二份最终话单文件名出现局点名“ABC”，文件序列号为9位。

```
MOD CDRSTOR: CDRSTORID="FinalCDR_2ndCopy", ISSINGLEBI=No, FILETYPE=SecondFinal,AGID="PS_GROUP_1", CHLNAME="ALL", CLOSEFILECOND=BySize, SFCDRSIZE=1024, NAMINGRULE="b%09LABC.dat";
```

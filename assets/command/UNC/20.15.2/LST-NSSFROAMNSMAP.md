# LST NSSFROAMNSMAP（查询漫游切片映射配置）

## 功能

**适用NF：NSSF**

该命令用于查询漫游切片映射配置。

## 注意事项

无

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| HMCC | 归属地移动国家码 | 可选必选说明：可选参数<br>参数含义：该参数用于描述归属地移动国家码。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度是3。3位十进制数。<br>默认值：无<br>配置原则：无 |
| HMNC | 归属地移动网号 | 可选必选说明：可选参数<br>参数含义：该参数用于描述归属地移动网号。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是2~3。2位或者3位十进制数。<br>默认值：无<br>配置原则：无 |
| HSST | 归属地切片服务类型 | 可选必选说明：可选参数<br>参数含义：该参数用于描述归属地切片服务类型。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是0~255。<br>默认值：无<br>配置原则：无 |
| HSD | 归属地切片细分标识 | 可选必选说明：可选参数<br>参数含义：该参数用于表示归属地切片细分标识。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是1~6。该参数输入长度为6或1。长度为6时，只能由字母（A-F或者a-f）、数字（0-9）组成，且大小写不敏感。长度为1时，只能为“*”，表示通配符，任意SD值都可以匹配。<br>默认值：无<br>配置原则：<br>HSDSTART与HSDEND不配置时，如果HSD为空，需要填入FFFFFF补齐。<br>HSD与HSD区间（HSDSTART与HSDEND）配置二选一，不能同时使用。 |
| HSDSTART | 归属地切片细分标识区间的起始值 | 可选必选说明：可选参数<br>参数含义：该参数用于表示归属地切片细分标识的起始值。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是4~6。<br>默认值：无<br>配置原则：<br>HSD与HSD区间（HSDSTART与HSDEND）配置二选一，不能同时使用。<br>HSD配置时，HSDSTART与HSDEND必须同时为空。<br>HSD区间配置，HSDSTART与HSDEND必须同时有值。 |
| HSDEND | 归属地切片细分标识区间的结束值 | 可选必选说明：可选参数<br>参数含义：该参数用于表示归属地切片细分标识的结束值。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是4~6。<br>默认值：无<br>配置原则：<br>HSD与HSD区间（HSDSTART与HSDEND）配置二选一，不能同时使用。<br>HSD配置时，HSDSTART与HSDEND必须同时为空。<br>HSD区间配置，HSDSTART与HSDEND必须同时有值。 |
| VMCC | 拜访地移动国家码 | 可选必选说明：可选参数<br>参数含义：该参数用于描述拜访地移动国家码。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度是3。3位十进制数。<br>默认值：无<br>配置原则：无 |
| VMNC | 拜访地移动网号 | 可选必选说明：可选参数<br>参数含义：该参数用于描述拜访地移动网号。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是2~3。2位或者3位十进制数。<br>默认值：无<br>配置原则：无 |
| VSST | 拜访地切片服务类型 | 可选必选说明：可选参数<br>参数含义：该参数用于描述拜访地切片服务类型。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是0~255。<br>默认值：无<br>配置原则：无 |
| VSD | 拜访地切片细分标识 | 可选必选说明：可选参数<br>参数含义：该参数用于表示拜访地切片细分标识。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度是6。该参数只能由字母（A-F或者a-f）、数字（0-9）组成，且大小写不敏感，转大写处理。<br>默认值：无<br>配置原则：无 |

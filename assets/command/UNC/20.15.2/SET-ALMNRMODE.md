# SET ALMNRMODE（设置网管北向告警上报模式）

## 功能

本命令用于设置网管北向告警上报模式。

本命令的使用场景为：日常的操作维护中，操作员可根据实际需要使用此命令设置网管北向告警上报模式，使告警上报北向网管系统时使用统一的网元ID或者指定某个源网元ID的告警使用目标网元ID上报到北向网管系统。

## 注意事项

- 此命令允许修改本站点所有网元告警的北向上报模式，影响设备与北向网管系统的对接。系统初始预置“告警北向上报模式”为“MN(多网元独立模式)”，需要根据具体局点的要求进行设置。
- 单网元合并模式下，删除目标网元或源网元时，针对该网元的设置都将失效。
- 单网元合并模式下，删除所有的源网元和目标网元时，“告警北向上报模式”自动恢复到“MN(多网元独立模式)”。

## 参数

| **参数标识** | **参数名称** | **参数说明** |
| --- | --- | --- |
| ALMNRMODE | 告警北向上报模式 | 可选必选说明：必选参数。<br>参数含义：用于设置本站点网元的告警北向上报模式，上报模式包含单网元合并模式和多网元独立模式两种模式。<br>取值范围：<br>- “SN(单网元合并模式)”：表示本站点所有网元或源网元的告警从网管上报到北向网管系统时，将被合并到目标网元上报，但网管客户端和本地操作维护系统上仍按原始网元显示告警。<br>- “MN(多网元独立模式)”：表示本站点各网元将按本网元类型和网元实例独立上报告警到北向网管系统。<br>默认值：无。<br>配置原则：无。 |
| SMEID | 源网元ID | 可选必选说明：条件必选参数。<br>参数含义：仅当本命令中的“告警北向上报模式”参数为“单网元合并模式”时该参数有效，用于指定系统需要将哪个网元的告警合并。网元ID可以通过<br>[**LST ME**](../../系统管理/版本信息/查询网元配置信息（LST ME）_47084797.md)<br>命令获取。<br>取值范围：0～65535<br>默认值：无。<br>配置原则：无。 |
| TMEID | 目标网元ID | 可选必选说明：条件必选参数。<br>参数含义：仅当本命令中的“告警北向上报模式”参数为“单网元合并模式”时该参数有效，用于指定系统需要将告警合并到哪个网元。网元ID可以通过<br>[**LST ME**](../../系统管理/版本信息/查询网元配置信息（LST ME）_47084797.md)<br>命令获取。<br>取值范围：0～65535<br>默认值：无。<br>配置原则：无。 |

## 操作的配置对象

## 使用实例

1. 设置网管告警北向上报模式，设置“告警北向上报模式”为“单网元合并模式”，“源网元ID”为“0”，“目标网元ID”为“75”：
  ```
  %%SET ALMNRMODE: ALMNRMODE=SN, SMEID=0, TMEID=75;%%
  RETCODE = 0  操作成功
  ```
2. 设置网管告警北向上报模式，设置“告警北向上报模式”为“多网元独立模式”：
  ```
  %%SET ALMNRMODE: ALMNRMODE=MN;%%
  RETCODE = 0  操作成功
  ```
3. 存在多个网元设置了“单网元合并模式”，设置其中一个网元为“多网元独立模式”：
  ```
  %%SET ALMNRMODE: ALMNRMODE=SN, SMEID=0, TMEID=0;%%
  RETCODE = 0  操作成功
  ```

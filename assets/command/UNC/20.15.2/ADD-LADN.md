# ADD LADN（增加本地数据网络）

## 功能

**适用NF：AMF**

该命令用于增加本地数据网络（LADN）的生效区域。本地数据网络生效区域由一个或者若干个跟踪区组成。

## 注意事项

- 该命令执行后立即生效。

- DNN、起始TAI、终止TAI的组合确定唯一一条LADN记录。
- DNN可以重复，一个DNN可对应多个TAI区间。
- 相同DNN最多只能配置16个TAI。
- 同一个DNN的TAI区间不允许有重叠区域，即为LADN新增TAI时，不能和相同DNN的已有TAI区间存在交集。

- 最多可输入100条记录。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| DNN | 数据网络名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定本地数据网络。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是0~63。可输入的字符有字母、十进制数字、“-”和“.”，并且开头和结尾只能是数字或者字母。不能出现连续两个“.”。字母大小写不敏感。<br>默认值：无<br>配置原则：无 |
| BGNTAI | 跟踪区标识起始值 | 可选必选说明：必选参数<br>参数含义：该参数用于指定本地数据网络（LADN）的生效区域范围。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是11~12。TAI由MCC、MNC和TAC组成。MCC为3位数字，MNC为2个或者3位数字，填写时请遵循实际长度。TAC编码为16进制数，按照字符串格式输入，字符串长度为6，只能由数字（0-9），字母（A-F、a-f）组成。字母大小写不敏感。<br>默认值：无<br>配置原则：无 |
| ENDTAI | 跟踪区标识结束值 | 可选必选说明：可选参数<br>参数含义：该参数用于指定本地数据网络的生效区域范围。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是11~12。TAI由MCC、MNC和TAC组成。MCC为3位数字，MNC为2个或者3位数字，填写时请遵循实际长度。TAC编码为16进制数，按照字符串格式输入，字符串长度为6，只能由数字（0-9），字母（A-F、a-f）组成。字母大小写不敏感。<br>默认值：无<br>配置原则：<br>当为LADN配置一个TAI时，本参数可以不输入，也可以与Begin TAI输入相同的值。 |
| DESC | 描述信息 | 可选必选说明：可选参数<br>参数含义：该参数表示对本地数据网络的描述信息，在运维中起到助记的作用。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~32。<br>默认值：无<br>配置原则：<br>输入单空格将删除该参数已有配置项。 |

## 使用实例

为值为"huawei.com"的DNN增加一个数据网络区域定义，覆盖范围从TAI 12300123888到12300123889：

```
ADD LADN: DNN="huawei.com", BGNTAI="12300123888", ENDTAI="12300123889", DESC="for Pudong Jinqiao";
```

# LST AREAGPMEM（查询区域群成员）

## 功能

**适用网元：SGSN、MME**

此命令用于查询区域群成员。

## 注意事项

- 此命令执行后立即生效。
- 查询支持范围匹配，即如果用户输出范围内的某个值，则将该值对应的范围记录显示出来。

## 权限

manage-ug；system-ug；monitor-ug
G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| AREAID | 区域群标识 | 可选必选说明：可选参数<br>参数含义：待查询的区域群标识。<br>取值范围：1~50<br>默认值：无 |
| IDTYPE | 标识类型 | 可选必选说明：可选参数<br>参数含义：待查询的标识类型。<br>取值范围：<br>- “LA(位置区)”：表示该区域标识类型为位置区。<br>- “RA(路由区)”：表示该区域标识类型为路由区<br>- “TA(跟踪区)”：表示该区域标识类型为跟踪区。<br>默认值：无<br>说明：同一个<br>“AREAID”<br>下的<br>“IDTYPE”<br>必须相同。 |
| MCC | 移动国家码 | 可选必选说明：可选参数<br>参数含义：待查询的移动国家码。<br>取值范围：3位的十进制数字<br>默认值：无 |
| MNC | 移动网号 | 可选必选说明：可选参数<br>参数含义：待查询的移动网号。<br>取值范围：2~3位的十进制数字<br>默认值： 无 |
| LAC | 位置区域码 | 可选必选说明：条件可选参数<br>参数含义：待查询的位置区域码。<br>前提条件：该参数在<br>“IDTYPE(标识类型)”<br>设置为<br>“LA(位置区)”<br>、<br>“RA(路由区)”<br>时生效。<br>取值范围：0x0000~0xFFFF<br>默认值：无 |
| LACRANGE | 位置区域码范围 | 可选必选说明：条件可选参数<br>参数含义：待查询的位置区域码范围。<br>前提条件：该参数在<br>“IDTYPE(标识类型)”<br>设置为<br>“LA(位置区)”<br>时生效。<br>取值范围：0x0000~0xFFFF<br>默认值：无<br>说明：- 该参数的取值需要大于或等于“LAC”。<br>- 该参数与“LAC”参数构成一个LAC区段，方便运维人员配置连续的位置区域。<br>- 如果不输入，表示配置单个LAC。 |
| RAC | 路由区域码 | 可选必选说明：条件可选参数<br>参数含义：待查询的路由区域码。<br>前提条件：该参数在<br>“IDTYPE(标识类型)”<br>设置为<br>“RA(路由区)”<br>时生效。<br>取值范围：0x00~0xFF<br>默认值：无 |
| RACRANGE | 路由区域码范围 | 可选必选说明：条件可选参数<br>参数含义：待查询的路由区域码范围。<br>前提条件：该参数在<br>“IDTYPE(标识类型)”<br>设置为<br>“RA(路由区)”<br>时生效。<br>取值范围：0x00~0xFF<br>默认值：无<br>说明：- 该参数的取值需要大于或等于“RAC”。<br>- 该参数与“RAC”参数构成一个RAC区段，方便运维人员配置连续的路由区域。<br>- 如果不输入，表示配置单个RAC。 |
| TAC | 跟踪区域码 | 可选必选说明：条件可选参数<br>参数含义：待查询的跟踪区域码。<br>前提条件：该参数在<br>“IDTYPE(标识类型)”<br>设置为<br>“TA(跟踪区)”<br>时生效。<br>取值范围：0x0000~0xFFFF<br>默认值：无 |
| TACRANGE | 跟踪区域码范围 | 可选必选说明：条件可选参数<br>参数含义：待查询的跟踪区域码范围。<br>前提条件：该参数在<br>“IDTYPE(标识类型)”<br>设置为<br>“TA(跟踪区)”<br>时生效。<br>取值范围：0x0000~0xFFFF<br>默认值：无<br>说明：- 该参数的取值需要大于或等于“TAC”。<br>- 该参数与“TAC”参数构成一个TAC区段，方便客户配置连续的跟踪区域。<br>- 如果不输入，表示配置单个TAC。 |

## 使用实例

1. 不输入查询条件，查询表中全部区域群成员的信息：
  LST AREAGPMEM:;
  ```
  %%LST AREAGPMEM:;%%
  RETCODE = 0  操作成功。

  输出结果如下
  --------------
   区域群标识  标识类型  移动国家码  移动网号  位置区域码  位置区域码范围  路由区域码  路由区域码范围  跟踪区域码  跟踪区域码范围

   1           位置区    123         03        0x12        0x12            NULL        NULL            NULL        NULL          
   2           路由区    123         03        0x34        0x34            0x56        0x56            NULL        NULL          
  (结果个数 = 2)

  ---    END
  ```
2. 查询一条区域群标识为1的区域群成员记录：
  LST AREAGPMEM: AREAID=1;
  ```
  %%LST AREAGPMEM: AREAID=1;%%
  RETCODE = 0  操作成功。

  输出结果如下
  --------------
      区域群标识  =  1
        标识类型  =  位置区
      移动国家码  =  123
        移动网号  =  03
      位置区域码  =  0x12
  位置区域码范围  =  0x12
      路由区域码  =  NULL
  路由区域码范围  =  NULL
      跟踪区域码  =  NULL
  跟踪区域码范围  =  NULL
  (结果个数 = 1)

  ---    END
  ```

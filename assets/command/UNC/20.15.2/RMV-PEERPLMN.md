# RMV PEERPLMN（删除对等PLMN配置）

## 功能

![](删除对等PLMN配置(RMV PEERPLMN)_72345697.assets/notice_3.0-zh-cn_2.png)

删除对等PLMN配置，UE将只能选择本网服务PLMN网络。

**适用网元：SGSN、MME**

此命令用于删除一条对等PLMN配置记录。

## 注意事项

- 此命令执行后立即生效。
- 删除对等PLMN配置，UE将只能选择本网服务PLMN网络。

## 权限

manage-ug；system-ug
G_1，管理员级别命令组；G_2，操作员级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |

| LAC | 位置区域码 | 可选必选说明：条件必选参数<br>参数含义：该参数用于显示指定位置区域码的信息。<br>前提条件：该参数在当<br>“AREA（区域范围）”<br>为<br>“LA（指定位置区）”<br>或<br>“RA（指定路由区）”<br>时生效。<br>取值范围：0x0000~0xFFFF<br>默认值：无 |
| RAC | 路由区域码 | 可选必选说明：条件必选参数<br>参数含义：该参数用于显示指定路由区域码的信息。<br>前提条件：该参数在当<br>“AREA（区域范围）”<br>为<br>“RA（指定路由区）”<br>时生效。<br>取值范围：0x00~0xFF<br>默认值：无 |

| IMSI | IMSI | 可选必选说明：条件必选参数<br>参数含义：该参数用于指定IMSI，对该IMSI所在号段进行删除。<br>前提条件：只有<br>“用户范围”<br>为<br>“IMSI_RANGE（指定IMSI范围）”<br>时，该参数才有效。<br>取值范围：1~15位十进制数字字符串<br>默认值：无 |
| SUBID | 用户群标识 | 可选必选说明：条件必选参数<br>参数含义：该参数用于指定用户群标识 。<br>前提条件：只有<br>“用户范围”<br>为<br>“SUBGP（用户群）”<br>时，该参数才有效。<br>取值范围：1~100<br>默认值：无<br>说明：该参数必须已经通过<br>[**ADD SUBGP**](../../移动性管理/区域漫游限制管理/用户群管理/增加用户群(ADD SUBGP)_72225241.md)<br>添加成功，可执行<br>[**LST SUBGP**](../../移动性管理/区域漫游限制管理/用户群管理/查询用户群(LST SUBGP)_26145562.md)<br>进行查看。 |

## 使用实例

删除一条对等PLMN配置记录， “区域范围” 为 “指定位置区” ， “移动国家代码” 为 “111” ， “移动网号” 为 “11” ， “位置区域码” 为 “0x0111” ， “用户范围” 为 “指定IMSI前缀” ， “IMSI前缀” 为 “12345” 。

RMV PEERPLMN: AREA=LA, MCC="111", MNC="11", LAC="0x0111", SUBRANGE=IMSI_PREFIX, IMSIPRE="12345";

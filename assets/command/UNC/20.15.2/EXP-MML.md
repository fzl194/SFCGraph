# EXP MML（导出MML文件）

## 功能

本命令可以对系统中的所有配置数据进行备份，并以特定的数据格式将系统配置数据导出，提升系统可靠性。

- 若要导出某一网元的配置数据，请输入“网元ID”。
- 若要导出全部网元的配置数据，请不要输入任何参数。

## 注意事项

- 配置导出文件系统默认保留时长为7天，每天凌晨2点会定时删除超过7天的文件，执行EXP MML导出命令时也会对超过7天的文件进行删除。
- 配置导出文件系统最大存储空间500M，最大文件保留个数500个，执行EXP MML导出命令时，若存储空间或文件保留个数超过最大值，系统按照文件产生时间删除最老的文件，直到本次导出文件可以正常存储为止。
- 执行命令后请及时下载，如果文件链接失效，请重新执行本命令获取新MML导出文件的链接。
- 当通过网管执行该命令时，需要复制导出链接，然后在浏览器打开。系统默认跳转至OM Portal的登录界面，请完成登录后，再次打开下载链接，完成下载操作。
- 当通过OM Portal执行该命令时，需要复制导出链接，然后在浏览器打开，完成下载操作。
- IE浏览器在IPV6环境下，OM Portal呈现的导出链接中IP格式为x-x-x-x-x-x-x-x.ipv6-literal.net。
- 当前OM portal登录的IP格式若与导出链接中IP格式不同，直接通过导出链接下载文件时系统会要求重新登录，若不想重新登录，可只复制链接中端口号后面的URL，替换当前登录页面链接端口号后面的URL。
- admin用户可以通过导出链接下载文件。非admin用户需要在OM Portal界面，通过单击“ 安全 > 用户管理 > 角色 ”菜单，设置目录权限中“导出目录”为“只读”时才能下载文件。
- 导出文件除可以用EXP MML命令返回的url进行下载外，也可以在OM Portal界面，通过单击“系统 > 文件传输 > 导出目录 > EXP MML”，进入配置导出文件系统进行下载。
- 导出文件中若出现两个USE ME切换相同网元ID的情况，为当前导出机制产生的正常现象，不影响正常的导入导出功能。比如：/*USE ME:MEID=0;*/在导出文件中出现两次。
- 跨版本/跨环境之间配置导入导出，要对导出的命令参数进行修正，再导入到新环境。
- 导出文件中，命令前后存在注释符时，表示该命令不支持配置导入，注释符为/* */。
- 导出文件的名称需要记录在数据库中，因此数据库的函数名称不能作为导出文件的名称，常见的数据库函数名称如下(以下函数名称不区分大小写)：and、exec、insert、select、drop、grant、alter、delete、desc、union、create、update、count、chr、mid、master、truncate、char、dec、are、or。
- 导出的系统配置数据中包含用户密钥时，无法直接导入成功。如果需要将该配置数据成功导入，则需要在执行本命令之前，修改MML导入导出密钥。

## 参数

| **参数标识** | **参数名称** | **参数说明** |
| --- | --- | --- |
| MEID | 网元ID | 可选必选说明：可选参数<br>参数含义：标识网元对象，用于指示系统需要将哪一个网元的配置数据文件导出。<br>取值范围：0～65535。<br>默认值：无。<br>配置原则：该参数不填写默认导出所有网元的配置命令，网元ID可以通过<br>**LST ME**<br>命令查询。 |
| FNM | 导出文件名称 | 可选必选说明：可选参数<br>参数含义：用于指定导出文件的名称。<br>取值范围：长度不超过128的英文字符串。<br>默认值：无。<br>配置原则：导出文件名称不能为点（.）或点点（..）；不能包含如下特殊字符：, ; = \ / : * ? " < > \| ' ` ! $ # & ( ) ~ @ % ^ +++ 和空格以及"--- END"；导出文件名称必须以字母或者数字开头。 |
| PWDENCRPT | 密码字段是否匿名化 | 可选必选说明：可选参数<br>参数含义：用于指定是否需要对业务所需的密文密码进行匿名化导出。<br>取值范围：<br>- TRUE(是)<br>- FALSE(否)<br>默认值：TRUE(是)<br>配置原则：导出的命令中包含密码字段且该字段需要再次导入成功才选择FALSE(否)，进行去匿名化导出，否则都选择TRUE(是)，进行匿名化导出。 |
| ISULDFTP | 是否上传到SFTP服务器 | 可选必选说明：可选参数<br>参数含义：用于指定是否上传到SFTP服务器。<br>取值范围：<br>- TRUE(是)<br>- FALSE(否)<br>默认值：无。<br>配置原则：无。 |
| BKSERVERIPTYPE | SFTP服务器IP地址类型 | 可选必选说明：该参数在<br>**“是否上传到SFTP服务器”**<br>配置为<br>“TRUE(是)”<br>时为条件必选参数。<br>参数含义：用于标识SFTP服务器的IP地址类型。<br>取值范围：<br>- IPV4(IPV4)<br>- IPV6(IPV6)<br>默认值：无。<br>配置原则：无。 |
| IPV4 | SFTP服务器IP地址 | 可选必选说明：该参数在<br>**“是否上传到SFTP服务器”**<br>配置为<br>“TRUE(是)”<br>，并且<br>**“SFTP服务器IP地址类型”**<br>配置为<br>“IPV4(IPV4)”<br>时为条件必选参数。<br>参数含义：用于标识SFTP服务器的IP地址。<br>取值范围：IPv4地址。<br>默认值：无。<br>配置原则：无。 |
| IPV6 | SFTP服务器IP地址 | 可选必选说明：该参数在<br>**“是否上传到SFTP服务器”**<br>配置为<br>“TRUE(是)”<br>，并且<br>**“SFTP服务器IP地址类型”**<br>配置为<br>“IPV6(IPV6)”<br>时为条件必选参数。<br>参数含义：用于标识SFTP服务器的IP地址。<br>取值范围：IPv6地址。<br>默认值：无。<br>配置原则：无。 |
| FTPUSER | SFTP用户名 | 可选必选说明：该参数在<br>**“是否上传到SFTP服务器”**<br>配置为<br>“TRUE(是)”<br>时为条件必选参数。<br>参数含义：用于标识登录SFTP服务器的用户名。<br>取值范围：长度不超过128的字符串。<br>默认值：无。<br>配置原则：无。 |
| FTPPSSWD | SFTP密码 | 可选必选说明：该参数在<br>**“是否上传到SFTP服务器”**<br>配置为<br>“TRUE(是)”<br>时为条件必选参数。<br>参数含义：用于标识登录SFTP服务器的密码。<br>取值范围：长度不超过128的密码。<br>默认值：无。<br>配置原则：密码字符支持范围为大写字母（A~Z），小写字母（a~z），数字（0~9）和特殊字符（包括!"#$%&'()*+,-./\:;<=>?@[]^{_\|}~），填写范围之外的字符可能会导致导出文件上传到SFTP服务器失败。 |
| DSTF | 目标目录 | 可选必选说明：该参数在<br>**“是否上传到SFTP服务器”**<br>配置为<br>“TRUE(是)”<br>时为条件必选参数。<br>参数含义：用于标识SFTP传输的目标路径。<br>取值范围：长度不超过256的字符串。<br>默认值：无。<br>配置原则：无。 |
| FTPPORT | SFTP端口号 | 可选必选说明：该参数在<br>**“是否上传到SFTP服务器”**<br>配置为<br>“TRUE(是)”<br>时为条件可选参数。<br>参数含义：用于标识SFTP服务器的端口号。<br>取值范围：1～65535。<br>默认值：22。<br>配置原则：需要和真实的SFTP服务器端口一致，不填写则使用默认端口22。 |

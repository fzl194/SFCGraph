# ADD M3LE（增加M3UA本地实体）

## 功能

**适用网元：SGSN、MME、SMSF**

该命令用于增加M3UA本地实体。

## 注意事项

- 该命令执行后立即生效。
- 在配置本地实体前，信令网属性表必须先配置成功（使用[**SET SIGATTR**](../../信令网属性管理/设置信令网属性(SET SIGATTR)_72226021.md)设置）。
- 相同网络指示语（NI）下，M3UA本地实体编码（OPC）和路由上下文（RC）的组合不能重复。
- 相同网络指示语（NI）下，M3UA本地实体编码（OPC）与目的实体编码（DPC）相同时，该M3UA实体工作于代理模式。
- M3UA本地实体个数不能超过64。
- 如果本地实体配置了路由上下文，则不能再配置网络指示语、实体编码相同而路由上下文不指定（即路由上下文为无效值）的本地实体。 反之，如果一个本地实体没有配置路由上下文，则不能再配置一个网络指示语、实体编码相同而且路由上下文有效的本地实体。

## 权限

manage-ug；system-ug
G_1，管理员级别命令组；G_2，操作员级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| LEX | 本地实体索引 | 可选必选说明：必选参数<br>参数含义：该参数用于指定唯一标识一个本地实体的索引。<br>数据来源：本端规划<br>取值范围：0~63<br>默认值：无<br>配置原则：无 |
| NI | 网络指示语 | 可选必选说明：必选参数<br>参数含义：该参数用于标识本信令点编码所在的网络，No.7信令网规定了4种网络。<br>数据来源：整网规划<br>取值范围：<br>- “INT（国际网）”<br>- “INTB（国际备用网）”<br>- “NAT（国内网）”<br>- “NATB（国内备用网）”<br>默认值：无<br>配置原则：无<br>说明：网络指示语在信令网属性表中必须设置为有效值。 |
| OPC | 本地实体编码 | 可选必选说明：必选参数<br>参数含义：该参数用于指定该本地实体所对应的本局信令点编码。本地实体编码（OPC）不允许配置为0。<br>数据来源：整网规划<br>取值范围：0x1~0x3FFF（对应14位编码）或0x1~0xFFFFFF（对应24位编码）<br>默认值：无<br>配置原则：<br>- 信令点编码结构与选定的网络指示语在信令网属性表中的设置一致。<br>- 本地实体编码为16进制数，可以输入0x为前缀。即输入“0xB23”和“B23”均表示实体编码为“0xB23”。<br>说明：不输入0x也取输入为十六进制值。 |
| LET | 本地实体类型 | 可选必选说明：必选参数<br>参数含义：该参数用于指定M3UA链路中的应用模式。<br>数据来源：整网规划<br>取值范围：<br>- “ASP（应用服务器进程）”<br>- “SGP（信令网关进程）”<br>- “IPSP（IP服务进程）”<br>默认值：无<br>配置原则：<br>- 在非对等组网模式中，当SGSN作为信令网关时，该参数设为“SGP（信令网关进程）”。<br>- 在非对等组网模式中，当SGSN与信令网关相连时，该参数设为“ASP（应用服务器进程）”。<br>- 在对等组网模式中，当SGSN通过M3UA链路（不通过SG转发）直接与其它设备对接时，该参数设为“IPSP（IP服务进程）”。<br>说明：- 对等组网：两信令点（SP）直接连接，不通过信令转接点（SG）。<br>- 非对等组网：两信令点（SP）通过信令转接点（SG）相连。 |
| RC | 路由上下文 | 可选必选说明：可选参数<br>参数含义：该参数用于当本地实体的网络指示语和实体编码相同时，用不同的路由上下文来区分不同的本地实体。本地实体的路由上下文不能重复。当不同的本地实体的网络指示语和实体编码都相同，导致无法区分不同的本地实体时，才需要配置路由上下文，用不同的路由上下文来区分不同的本地实体。<br>数据来源：整网规划<br>取值范围：0~4294967295<br>默认值：无<br>配置原则：无<br>说明：- 取值为“4294967295”表示在接口不发送Route context。<br>- 本地实体的路由上下文不能重复，即在本局唯一即可。<br>- 不输入时为无效值即0xFFFFFFFF，显示为NULL。 |
| NA | 网络外貌 | 可选必选说明：可选参数<br>参数含义：该参数用于识别SS7信令网上下文，有利于逻辑上把SGP和ASP间公共SCTP偶联上的信令业务分开。网络外貌隐含地定义了SS7信令点编码、网络表示语和MTP3协议类型、不同的版本。<br>数据来源：整网规划<br>取值范围：0~4294967295<br>默认值：无<br>配置原则：无<br>说明：- 取值为“4294967295”表示在接口不发送Network Appearance。<br>- 本地实体的一个网络外貌值只能配置给一个网络指示语，但可以给一个网络指示语配置多个网络外貌值。<br>- 不输入时为无效值即0xFFFFFFFF，显示为NULL。 |
| LEN | 本地实体名 | 可选必选说明：可选参数<br>参数含义：本地实体名称，标识本地实体。<br>数据来源：本端规划<br>取值范围：1~32位字符串<br>默认值：noname<br>配置原则：无 |

## 操作的配置对象

## 使用实例

增加M3UA本地实体，LEX为1，NI为“国内网”，OPC为“0x222222”，LET为“ASP”，LEN为“TEST”：

```
ADD M3LE: LEX=1, NI=NAT, OPC="222222", LET=ASP, LEN="TEST";
```

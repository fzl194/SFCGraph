# LST TZ（查询时区和夏令时信息）

## 功能

本命令用于查询时区和夏令时信息。

## 注意事项

当 “标准时类型” 配置为 “UTC(UTC)” 时，查询结果中 “时区” 显示为UTC，否则显示为GMT。

## 参数

无。

## 操作的配置对象

- [时区和夏令时信息（TZ）](configobject/UNC/20.15.2/TZ.md)

## 使用实例

星期输入方式时区信息查询结果如下：

```
%%LST TZ:;%% 
RETCODE = 0  操作成功  
操作结果如下
------------               
                时区  =  UTC+08:00     
      是否执行夏令时  =  是         
        日期输入方式  =  星期方式          
            开始月份  =  1月 
    开始的第几个星期  =  第一个  
            开始星期  =  星期日       
            开始时间  =  18:54           
            结束月份  =  2月
    结束的第几个星期  =  第二个   
            结束星期  =  星期日       
            结束时间  =  18:55
  时间偏移量（分钟）  =  10    
 (结果个数 = 1)  
---    END 
```

日期输入方式时区信息查询结果如下：

```
%%LST TZ:;%% 
RETCODE = 0  操作成功  
操作结果如下 
------------               
                 时区  =  GMT+08:00     
       是否执行夏令时  =  是         
         日期输入方式  =  日期方式          
             开始月份  =  1月             
               开始日  =  2           
             开始时间  =  18:49           
             结束月份  =  2月             
               结束日  =  1           
             结束时间  =  18:50 
   时间偏移量（分钟）  =  10 
  (结果个数 = 1)  
---    END 
```

# ADD CDRAUDIT（增加话单稽核）

## 功能

**适用NF：NCG**

该命令用于配置第二份最终话单的稽核任务。

每天生成的稽核文件对前一天的第二份最终话单文件信息进行统计，如果调整了系统时间使某一天产生了两个稽核文件（其中一个文件的后缀为_old），这时需要将两个文件同时取走才能得到那天的完整统计信息。

CG支持定制稽核文件名，通过配置可以在话单稽核文件名中体现各类信息。例如：

- 加入时间信息
  当计费中心（本文档中又称BS（Billing System））要求在稽核文件名中体现稽核话单的生成时间时，可以在稽核文件名中加入时间信息。
- 加入局点信息
  当采用大本地网配置时，通常要将各个归属地的稽核文件派发到不同的计费中心。这时可以在稽核文件名中加入局点信息，方便管理。
- 加入接入点信息
  当多对CGRU上的稽核话单送往同一计费中心的同一文件夹下时，需要在稽核文件名上加入AP模块信息。用于区分不同对RU上的稽核文件，避免稽核文件被覆盖。
- 增加序列号
  序列号能清晰的体现每个话单稽核文件产生的先后顺序。

稽核文件名定制配置项有：

- %Y%m%d%H%M%S或%y%m%d%H%M%S或%Y%m%d%I%M%S或%y%m%d%I%M%S
- %c或%C
- %0xL

> **说明**
> - 稽核文件名的默认配置为“%Y%m%d.log”。
> - 用户自行配置文件名时，必须配置[表1](增加话单稽核（ADD CDRAUDIT）_51174239.md#ZH-CN_CONCEPT_0251174239__table_0365FEC2)中列出的“增加时间信息”。

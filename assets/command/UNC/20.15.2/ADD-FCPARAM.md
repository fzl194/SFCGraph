# ADD FCPARAM（增加流控参数）

## 功能

该命令用于增加指定微服务实例的流控参数，本命令可以配置该微服务实例在流控过程中的最大令牌数、最小令牌数、权重生成方法、权重参数等。

## 注意事项

- 该命令执行后下次流控时生效。

- 使用本命令会修改流控参数配置，配置错误可能导致流控效果达不到预期，需要联系华为技术支持来调整参数。

- 最多可输入1000条记录。

## 权限

G_1，管理员级别命令组

## 使用实例

- 增加微服务类型为UesmCtrlSvc、微服务实例ID为999、流控套餐ID为0的流控参数，最小令牌数为30，最大令牌数为5000。
  ```
  ADD FCPARAM: SERVICETYPE="UesmCtrlSvc", SERVICEID=999, CASEID=0, PARAMCAT=GENERAL, MINTOKEN=30, MAXTOKEN=5000;
  ```
- 增加微服务类型为UesmCtrlSvc、微服务实例ID为999、流控套餐ID为0的流控参数，使用统计学习模式，需要流控的消息分别使用了4个流控优先级1~4，调整这4个流控优先级的权重预期系数为4:3:2:1。
  ```
  ADD FCPARAM: SERVICETYPE="UesmCtrlSvc", SERVICEID=999, CASEID=0, PARAMCAT=WEIGHTS, WEIGHTMETHOD=FCLEARN, EXPECTRATIO1=4, EXPECTRATIO2=3, EXPECTRATIO3=2, EXPECTRATIO4=1;
  ```
- 增加微服务类型为UesmExecSvc、微服务实例为999、流控套餐为0的流控参数，节点起控参考容器CPU功能打开，容器CPU超过阈值30后，节点才可以起控；节点流控起控阈值为80，连续5s超过起控阈值后起控，停控阈值为75，连续60s的满足停控算法后停控。
  ```
  ADD FCPARAM: SERVICETYPE="UesmExecSvc", SERVICEID=999, CASEID=0, PARAMCAT=THRESHOLD, NODEUPPERTH=80, NODEUPTHDELTA=5, NODELOWERTH=75, NODELOWTHDELTA=60, CNTTHOFNODE=30, ENABLECNTTH=ENABLE;
  ```
- 增加微服务类型为UesmExecSvc、微服务实例为999、流控套餐为0的流控参数，节点起控后，目标CPU值为78，容器起控后，目标CPU值为78。
  ```
  ADD FCPARAM: SERVICETYPE="UesmExecSvc", SERVICEID=999, CASEID=0, PARAMCAT=TARGETCPU, TARGETCPU=78, CNTTARGETCPU=78;
  ```
- 增加微服务类型为UesmExecSvc、微服务实例为999、流控套餐为0的流控参数，节点CPU轻度过载阈值为75，中度过载阈值为80，重度过载阈值为85，容器CPU轻度过载阈值为75，中度过载阈值为80，重度过载阈值为85。
  ```
  ADD FCPARAM: SERVICETYPE="UesmExecSvc", SERVICEID=999, CASEID=0, PARAMCAT=OVERLOADLEVEL, MILDOVERLOAD=75, MODEROVERLOAD=80, SEVEREOVERLOAD=85, CNTMILDOVERLOAD=75, CNTMODEOVERLOAD=80, CNTSEROVERLAOD=85;
  ```
- 增加微服务类型为UesmExecSvc、微服务实例为999、流控套餐为0的流控参数，节点CPU停控算法为连续停控，容器CPU停控算法为平均停控。连续停控（CONTINUE）：CPU值在连续Node/Container Lower Threshold Delta参数时长范围内（秒）低于Node/Container Lower Threshold时停控；平均停控（AVERAGE）：CPU值在连续Node/Container Lower Threshold Delta参数时长（秒）均值低于Node/Container Lower Threshold时停控。
  ```

  ```

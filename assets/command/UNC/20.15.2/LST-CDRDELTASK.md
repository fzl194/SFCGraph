# LST CDRDELTASK（查询话单删除任务）

## 功能

**适用NF：NCG**

该命令用于查询当前系统已配置的话单删除任务信息。

## 注意事项

该命令执行后立即生效。

## 权限

manage-ug；system-ug；monitor-ug
G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| CDRDELTASKID | 删除任务标识 | 可选必选说明：可选参数<br>参数含义：话单删除任务标识，全局唯一。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～32。<br>默认值：无<br>配置原则：无 |
| RUID | RU的ID | 可选必选说明：可选参数<br>参数含义：RU的ID。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为64～4294967294。<br>默认值：无<br>配置原则：该值需要执行<br>[**LST SERVICERUSTATE**](../../../../../平台服务管理/单体服务编排功能管理/系统管理/资源管理/RU信息/查询RU的信息(LST SERVICERUSTATE)_29626965.md)<br>命令，查询出存在的RU ID进行填写。 |

| ISCOMPRESSION | 是否压缩话单 | 可选必选说明：条件可选参数<br>前提条件：该参数在“CDRDELTYPE”配置为“DELETEBYTIME”时为条件可选参数。<br>参数含义：是否压缩话单，其为开关型参数。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- NO：否。<br>- YES：是。<br>默认值：无<br>配置原则：无 |
| CDRDELDIR | 话单删除目录 | 可选必选说明：条件可选参数<br>前提条件：该参数在“CDRDELTYPE”配置为“DELETEBYFILENAME”时为条件可选参数。<br>可选必选说明：条件可选参数<br>前提条件：该参数在“ISCOMPRESSION”配置为“NO”时为条件可选参数。<br>参数含义：当选择“是否压缩话单”为“NO(否)”时，此参数有效。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～128。<br>默认值：无<br>配置原则：无 |
| COMPRESSSRC | 话单压缩源目录 | 可选必选说明：条件可选参数<br>前提条件：该参数在“ISCOMPRESSION”配置为“YES”时为条件可选参数。<br>参数含义：当选择“是否压缩话单”为“YES(是)”时，此参数有效。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～128。<br>默认值：无<br>配置原则：无 |
| COMPRESSDES | 话单压缩目的目录 | 可选必选说明：条件可选参数<br>前提条件：该参数在“ISCOMPRESSION”配置为“YES”时为条件可选参数。<br>参数含义：当选择“是否压缩话单”为“YES(是)”时，此参数有效。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～128。<br>默认值：无<br>配置原则：无 |
| COMPRESSDAYS | 话单压缩天数 | 可选必选说明：条件可选参数<br>前提条件：该参数在“ISCOMPRESSION”配置为“YES”时为条件可选参数。<br>参数含义：当选择“是否压缩话单”为“YES(是)”时，此参数有效。 自定义目录下话单文件的压缩天数，NCG自动将“话单压缩源目录”下超过此期限的话单文件进行压缩，并删除源文件。压缩后的文件日期和源文件保持一致，保存在“话单压缩目的目录”下。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～180。<br>默认值：无<br>配置原则：无 |
| STOREDAYS | 话单保存天数 | 可选必选说明：条件可选参数<br>前提条件：该参数在“CDRDELTYPE”配置为“DELETEBYTIME”时为条件可选参数。<br>参数含义：自定义目录下话单文件的保存天数，当配置为非压缩删除任务时，NCG将自动删除“话单删除目录”下超过此期限的话单文件；当配置为压缩删除任务时，NCG将自动删除“话单压缩目的目录”下超过此期限的压缩文件。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～180。<br>默认值：无<br>配置原则：无 |
| CDRDELPREFIX | 待删除话单文件名前缀 | 可选必选说明：条件可选参数<br>前提条件：该参数在“CDRDELTYPE”配置为“DELETEBYFILENAME”时为条件可选参数。<br>参数含义：该参数用于指定待删除话单的前缀。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～128。<br>默认值：无<br>配置原则：无 |
| CDRDELSUFFIX | 待删除话单文件名后缀 | 可选必选说明：条件可选参数<br>前提条件：该参数在“CDRDELTYPE”配置为“DELETEBYFILENAME”时为条件可选参数。<br>参数含义：该参数用于指定待删除话单的后缀。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～128。<br>默认值：无<br>配置原则：无 |
| CDRDELINTERVAL | 话单删除时间间隔 | 可选必选说明：条件可选参数<br>前提条件：该参数在“CDRDELTYPE”配置为“DELETEBYFILENAME”时为条件可选参数。<br>参数含义：该参数用于指定执行完一次删除任务后到下一次删除任务开始的时间间隔，单位为秒。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～3600，单位是秒。<br>默认值：无<br>配置原则：无 |
| TIMETYPE | 话单删除时间类型 | 可选必选说明：条件可选参数<br>前提条件：该参数在“CDRDELTYPE”配置为“DELETEBYFILENAME”时为可选参数。<br>参数含义：该参数用于指定删除重命名话单文件时的时间类型。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- SECOND：话单保存的时间单位为秒。<br>- DAY：话单保存的时间单位为天。<br>默认值：无<br>配置原则：无 |
| RENMSTOREDAYS | 重命名话单保存天数 | 可选必选说明：条件可选参数<br>前提条件：该参数在“TIMETYPE”配置为“DAY”时为可选参数。<br>参数含义：自定义目录下重命名话单文件的保存天数，NCG将自动删除“话单删除目录”下超过此期限的重命名话单文件。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～180。<br>默认值：无<br>配置原则：<br>- 参数配置为“0”时，表示不删除。<br>- 参数填写不为“0”时，删除自定义目录及其一级子目录下超过保存天数的重命名话单文件。 |
| CDRDELSUBLEV | 话单删除子目录层级 | 可选必选说明：可选参数<br>参数含义：该参数用于指定话单删除子目录层级。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～32。<br>默认值：无<br>配置原则：无 |

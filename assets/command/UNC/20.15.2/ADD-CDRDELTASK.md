# ADD CDRDELTASK（增加话单删除任务）

## 功能

![](增加话单删除任务（ADD CDRDELTASK）_51174264.assets/notice_3.0-zh-cn_2.png)

如果目录配置为系统文件目录将会造成误删系统文件，请确认配置的目录为话单目录。

**适用NF：NCG**

该命令用于配置话单删除任务，删除NCG上自定义目录下的过期话单文件，防止保存的话单数量过多导致磁盘空间不足。

## 注意事项

- 该命令最大记录数为2048。
- 请确保配置的目录为话单文件所在目录，否则可能由于误删除文件导致NCG系统功能异常。
- NCG会对“话单删除目录”、“话单压缩源目录”、“话单压缩目的目录”参数的输入值进行规整，保证输入目录的最后一个字符为“/”。例如，当输入的“话单删除目录”为“./frontsave/destpath”时，规整后为“./frontsave/destpath/”。
- 此命令可以动态生效。
- 当“话单删除类型”配置为“按文件名删除”时不支持压缩删除。
- 话单删除任务支持自定义目录下话单文件的过期删除，自定义目录需要满足以下条件：
    - 目录必须以“./”开头，但不能仅为“./”。
    - 目录不能为“./frontsave”和“./backsave”。
    - 目录不能为保存状态文件的目录及其子目录，即不能以“./StatusFile”、“./StatusFileB”和“./status”开头。
    - 目录不能为备份、分发任务对应的硬链接目录及其子目录，即不能以“./frontsave/hdcdrs”和“./backsave/hdcdrs”开头。
    - 目录不能为备份、分发任务对应的自定义源目录及其父、子目录，可使用[查询话单备份（LST CDRBACKUP）](../话单备份/查询话单备份（LST CDRBACKUP）_51174247.md)、[查询话单分发（LST CDRDISTR）](../话单分发/查询话单分发（LST CDRDISTR）_51174257.md)命令查询。
    - 如果本地备份、本地分发任务的目的路径配置了过期删除，则目录不能为本地备份、本地分发任务的目的路径及其父、子目录，可使用[查询话单备份（LST CDRBACKUP）](../话单备份/查询话单备份（LST CDRBACKUP）_51174247.md)、[查询话单分发（LST CDRDISTR）](../话单分发/查询话单分发（LST CDRDISTR）_51174257.md)命令查询。
    - 如果目录以“./frontsave”或“./backsave”开头，则其下一级子目录不能与AP名相同。
  请确保配置的目录为话单文件所在目录，否则可能由于误删除文件导致NCG系统功能异常。
- 话单删除任务有两种删除方式：
    - 非压缩删除：当参数“是否压缩话单”取值为“NO(否)”时，根据“话单保存天数”直接删除“话单删除目录”及其一级子目录下的过期话单文件。
    - 压缩删除：当参数“是否压缩话单”取值为“YES(是)”时，先将“话单压缩源目录”及其一级子目录下超过“话单压缩天数”的话单文件进行压缩（压缩后的.gz文件保存在“话单压缩目的目录”下），并删除源文件，再根据“话单保存天数”删除“话单压缩目的目录”及其一级子目录下的过期压缩文件。
  请根据需求选择不同的删除方式配置话单删除任务。

## 使用实例

- 增加非压缩删除任务，过期删除目录“./third”及其一级子目录下超过40天的话单文件：
  ```
  ADD CDRDELTASK: CDRDELTASKID="cdr_deltask1", CDRDELTYPE=DELETEBYTIME, ISCOMPRESSION=NO, CDRDELDIR="./third", STOREDAYS=40;
  ```
- 增加压缩删除任务，先将压缩源目录“./frontsave/scdr”及其一级子目录下的超过70天的话单文件进行压缩（压缩后的文件保存在压缩目的目录“./scdr”及其相应的一级子目录中），并删除源文件，再将压缩目的目录“./scdr”下超过90天的压缩文件过期删除：
  ```
  ADD CDRDELTASK: CDRDELTASKID="cdr_deltask2", RUID=65, CDRDELTYPE=DELETEBYTIME, ISCOMPRESSION=YES, COMPRESSSRC="./frontsave/scdr", COMPRESSDES="./scdr", COMPRESSDAYS=70, STOREDAYS=90;
  ```
- 增加非压缩删除任务，过期删除目录“./third”及其一级子目录下超过30天的重命名话单文件：
  ```
  ADD CDRDELTASK: CDRDELTASKID="renameTask", CDRDELTYPE=DELETEBYFILENAME, CDRDELDIR="./third", TIMETYPE=DAY, RENMSTOREDAYS=30;
  ```

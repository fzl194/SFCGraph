# SWAP DR（容灾实例主备倒换）

## 功能

![](容灾实例主备倒换（SWAP DR）_23235166.assets/notice_3.0-zh-cn_2.png)

本命令属于高危命令，操作不当可能会导致业务的短时中断，请谨慎使用。

该命令用于容灾实例主备倒换。

## 注意事项

- 该命令执行后立即生效。

- 该命令只用于在UEG-L/UEN网元采用主备（冷备）容灾模式下执行。
- 该命令只用于在UEG-M/UEG网元采用主备（热备）容灾模式下执行。
- 在容灾实例间通信通道正常时，可在运行主执行该命令进行倒换。
- 在容灾实例间通信通道异常时，可在运行备执行该命令进行倒换。
- 若执行该命令后，返回结果上报错误码67680（超过最大倒换限制次数），可选择“强制倒换”进行倒换。
- 若错误码是67679（运行备关键服务异常），则需要查看容灾网元是否有ALM-100500容灾实例内关键服务异常，如果有则先根据告警信息进行处理；如果没有则请等待15分钟后再重新执行该命令。
- 命令返回成功后可使用[**DSP DRGROUPSTATUS**](显示容灾组的运行状态信息（DSP DRGROUPSTATUS）_74554621.md)命令查询倒换结果。
- 软参“DWORD_521 容灾联动复位或备升主开关”的优先级（级别最高）高于本命令，当该软参设置为1时，即便满足容灾实例主备倒换的条件也不会触发倒换。当该软参设置为0时，则再以本命令回显结果为准。
- 执行该命令会导致整系统复位，低概率返回3574返回码，属正常现象。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| DRGROUPID | 容灾组标识 | 可选必选说明：必选参数<br>参数含义：该参数用于指定容灾组标识。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是1~8。<br>默认值：无<br>配置原则：<br>可使用<br>[**LST DRGROUPINFO**](查询容灾组信息（LST DRGROUPINFO）_74835153.md)<br>返回的DRGROUPID作为参数输入。 |
| FORCE | 是否强制倒换 | 可选必选说明：可选参数<br>参数含义：该参数用于配置是否强制倒换。<br>数据来源：本端规划<br>取值范围：<br>- YES（强制倒换）<br>- NO（不强制倒换）<br>默认值：NO<br>配置原则：<br>在容灾实例间通信通道正常或异常时，运行备关键服务正常但超过最大倒换次数限制，需配置为强制倒换。 |

## 操作的配置对象

## 使用实例

执行容灾实例主备倒换：

```
%%SWAP DR: DRGROUPID=1, FORCE=NO;%%
RETCODE = 0  操作成功

---    END
```

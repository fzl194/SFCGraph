# DSP MSSTBMDETINFO（通过表号查询表的详细信息）

## 功能

该命令用于查询指定表的表项详细信息。当业务表下表项后业务未生效时，可以用该命令定界表项丢失位置。

## 注意事项

该命令执行后立即生效。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RUNAME | RU名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定RU名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，区分大小写。<br>默认值：无<br>配置原则：使用DSP RU查看RU名称。 |
| TABLEID | 表号 | 可选必选说明：必选参数<br>参数含义：该参数用于表示表号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～4294967295。<br>默认值：无 |

## 操作的配置对象

- [[UNC@20.15.2@ConfigObject@MSSTBMDETINFO]] · 通过表号查询表的详细信息（MSSTBMDETINFO）

## 使用实例

查询表号查询表的详细信息：

```
DSP MSSTBMDETINFO:TABLEID = 1,RUNAME = "VNODE_VNRS_VNFC_IPU_0064";
```

# MOD CELLGEO（修改CELLID与地理坐标对应关系）

## 功能

**适用网元：SGSN**

此命令用于修改小区标识CID与地理位置经纬度坐标的对应关系。可选参数没有输入时该参数保持原有值不变。

## 注意事项

此命令执行后立即生效。

## 权限

manage-ug；system-ug
G_1，管理员级别命令组；G_2，操作员级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| CID | 小区标识 | 可选必选说明：必选参数<br>参数含义：该参数用于指定小区的标识，CID = MCC + MNC + LAC + RAC + CI。<br>数据来源：整网规划<br>取值范围：15~16位十六进制数<br>默认值：无 |
| LAT | 纬度类型 | 可选必选说明：可选参数<br>参数含义：该参数用于指定位置坐标的纬度类型。<br>数据来源：整网规划<br>取值范围：<br>- “NLA(北纬)”<br>- “SLA(南纬)”<br>默认值：无<br>说明：SOUTH_LATITUDE，南纬；NOUTH_LATITUDE，北纬。 |
| LAD | 纬度（度） | 可选必选说明：可选参数<br>参数含义：该参数用于指定位置坐标的纬度的度值。<br>数据来源：整网规划<br>取值范围：0度～90度<br>默认值：无 |
| LAM | 纬度（分） | 可选必选说明：可选参数<br>参数含义：该参数用于指定位置坐标的纬度的分的值。<br>数据来源：整网规划<br>取值范围：0分～59分<br>默认值：无<br>说明：当LAD取值为90度时，LAM只能为0分。 |
| LAS | 纬度（秒） | 可选必选说明：可选参数<br>参数含义：该参数用于指定位置坐标的纬度的秒的值。<br>数据来源：整网规划<br>取值范围：0秒～59秒<br>默认值：无<br>说明：当LAD取值为90度时，LAS只能为0秒。 |
| LOT | 经度类型 | 可选必选说明：可选参数<br>参数含义：该参数用于指定位置坐标的经度类型。<br>数据来源：整网规划<br>取值范围：<br>- “ELO(东经)”<br>- “WLO(西经)”<br>默认值：无<br>说明：EAST_LONGITUDE，东经；WEST_LONGITUDE，西经。 |
| LOD | 经度（度） | 可选必选说明：可选参数<br>参数含义：该参数用于指定位置坐标的经度的度值。<br>数据来源：整网规划<br>取值范围：0度～180度<br>默认值：无 |
| LOM | 经度（分） | 可选必选说明：可选参数<br>参数含义：该参数用于指定位置坐标的经度的分的值。<br>数据来源：整网规划<br>取值范围：0分～59分<br>默认值：无<br>说明：当LOD取值为180度时，LOM只能为0分。 |
| LOS | 经度（秒） | 可选必选说明：可选参数<br>参数含义：该参数用于指定位置坐标的经度的秒的值。<br>数据来源：整网规划<br>取值范围：0秒～59秒<br>默认值：无<br>说明：当LOD取值为180度时，LOS只能为0秒。 |
| RAD | 小区半径（m） | 可选必选说明：可选参数<br>参数含义：该参数用于指定小区半径的值，就是该小区覆盖的半径。<br>数据来源：整网规划<br>取值范围：0m～1800000m<br>默认值：无<br>说明：在LCS MT/NI流程中当只能获得被定位用户的小区标识时，小区半径就是该用户坐标偏离该小区中心坐标的误差。 |

## 操作的配置对象

- [[UNC@20.15.2@ConfigObject@CELLGEO]] · CELLID与地理坐标对应关系（CELLGEO）

## 使用实例

将CID为0123456789012345对应的地理位置坐标由北纬5度15分25秒西经35度45分55秒小区半径为5500米修改为南纬2度12分22秒东经32度42分52秒小区半径为4500米：

MOD CELLGEO: CID="0123456789012345", LAT=SLA, LAD=2, LAM=12, LAS=22, LOT=ELO, LOD=32, LOM=42, LOS=52, RAD=4500;

# MOD MULDNNPRILIST（修改本地专网DNN就近接入优先级）

## 功能

**适用NF：SMF、PGW-C**

该命令用于修改指定智能分流专网DNN接入省/市的优先级。

## 注意事项

- 该命令执行后只对新激活用户生效。

- 智能分流就近接入省/市后缀名称的优先级顺序由高到低为，就近接入省市第一级后缀，就近接入省市第二级后缀，就近接入省市第三级后缀。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组

## 使用实例

修改本地专网DNN就近接入优先级，智能分流专网DNN为“special.dnn”，就近接入省市第一级后缀为“abcd”，就近接入省市第二级后缀为“defg”，就近接入省市第三级后缀为“ghij”。

```
MOD MULDNNPRILIST: DEDDNN="special.dnn",NEARBYSUFF1ST="adcd",NEARBYSUFF2ND="defg",NEARBYSUFF3RD="ghij";
```

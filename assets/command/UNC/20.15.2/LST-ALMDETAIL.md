# LST ALMDETAIL（查询告警详细信息）

## 功能

**适用网元：SGSN、MME、AMF**

该命令用于查询批量告警的详细故障信息。

## 注意事项

- 当“详细信息”选择为“YES(是)”时，系统内可能存在许多原始告警，导致该命令返回时间过长。可通过在命令中限定“起始时间”和“结束时间”来减少查询记录数。
- 由于批量告警是在原始告警产生后按照一定周期合并的结果，所以原始告警的实际故障时间早于批量告警的产生时间。
- 如果批量告警进行了刷新，原始告警和批量告警的时间可能相差很多，因此当根据某条批量告警产生时间作为查询条件时，只能作为命令的“结束时间”。

## 权限

manage-ug；system-ug；monitor-ug
G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| ALMTYPE | 告警类型 | 可选必选说明：可选参数<br>参数含义：该参数用于指定告警类型。<br>数据来源：本端规划<br>取值范围：<br>- “S1apLinkDown(S1ap链路故障)”<br>- “SCTPPacketErrorRateTooHigh(SCTP链路错包率过高)”<br>- “SCTPFlowControl(S1接口SCTP层流控)”<br>- “FrequentIntermittenceofLinks(批量链路频繁闪断)”<br>- “RanSctpFlowControl(RAN侧SCTP层链路拥塞流控)”<br>默认值：无 |
| DETAIL | 详细信息 | 可选必选说明：分支可选参数<br>参数含义：该参数用于指定是否输出<br>“告警类型”<br>对应的详细信息。<br>前提条件：该参数在<br>“告警类型”<br>参数配置后生效。<br>数据来源:本段规划<br>取值范围：<br>- “NO(否)”<br>- “YES(是)”<br>默认值：<br>“NO(否)” |
| SGT | 起始时间 | 可选必选说明：分支可选参数<br>参数含义：该参数用于指定系统查询告警产生时间的起始时间。<br>前提条件：该参数在<br>“详细信息”<br>参数配置为<br>“YES(是)”<br>后生效。<br>数据来源：本端规划<br>取值范围：1970-01-01～2099-12-31<br>默认值：无 |
| EGT | 结束时间 | 可选必选说明：分支可选参数<br>参数含义：该参数用于指定系统查询告警产生时间的结束时间。<br>前提条件：该参数在<br>“详细信息”<br>参数配置为<br>“YES(是)”<br>后生效。<br>数据来源：本端规划<br>取值范围：1970-01-01～2099-12-31<br>默认值：无 |
| MAXRECORD | 最大记录数 | 可选必选说明：分支可选参数<br>参数含义：该参数用于指定系统查询告警的最大记录数。<br>前提条件：该参数在<br>“详细信息”<br>参数配置为<br>“YES(是)”<br>后生效。<br>数据来源：本端规划<br>取值范围：1～1000<br>默认值：1000 |

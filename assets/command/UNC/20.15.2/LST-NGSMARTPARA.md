# LST NGSMARTPARA（查询5G信令抑制参数）

## 功能

**适用NF：AMF**

该命令用于查询AMF针对5G用户异常信令抑制的相关参数。

## 注意事项

无

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## 参数

无

## 操作的配置对象

- [[UNC@20.15.2@ConfigObject@NGSMARTPARA]] · 5G信令抑制参数（NGSMARTPARA）

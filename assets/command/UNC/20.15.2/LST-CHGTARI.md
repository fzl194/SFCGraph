# LST CHGTARI（查询费率时段配置）

## 功能

**适用网元：SGSN**

该命令用于查询计费费率时段配置表中费率时段相关配置。费率时段配置表与 [**SET CHGWKDY**](../计费星期配置/设置星期配置(SET CHGWKDY)_26305210.md) 、 [**ADD CHGHOLI**](../计费节假日配置/增加节假日配置(ADD CHGHOLI)_26145382.md) 相结合，实现灵活的费率配置。

## 注意事项

如果有输入参数，则显示与输入参数均匹配的费率时段记录；如果没有输入参数，则显示所有费率时段记录。

## 权限

manage-ug；system-ug；monitor-ug
G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| TT | 费率类型 | 可选必选说明：可选参数<br>参数含义：该参数用于指定费率时段的费率类型。<br>取值范围：<br>- “WORK(工作日)”：表示该费率时段适用于工作日。<br>- “WEEKEND(周休日)”：表示该费率时段适用于周休日。<br>- “FESTIVAL(节假日)”：表示该费率时段适用于节假日。<br>默认值：无 |
| CC | 计费属性 | 可选必选说明：可选参数<br>参数含义：该参数用于指定费率时段是对普通计费属性用户、预付费计费属性用户、包月制计费属性用户还是对实时计费属性用户有效。<br>取值范围：<br>- “NORMAL(普通计费)”：表示普通计费属性，按照此种方式计费的用户按照常规的方式支付费用<br>- “PREPAID(预付费)”：表示预付费计费属性，按照此种方式计费的用户在获取某种服务之前需要预支付一定的费用。<br>- “FLATRATE(包月制)”：表示包月制计费属性，按照此种方式计费的用户在一个月内的收费是固定的。<br>- “HOTBILLING(实时计费)”：表示实时计费属性，按照此种方式计费的用户将在短时间或流量达到某个值时及时生成话单，保证运营商对此类用户及时收费。<br>默认值：无 |

## 操作的配置对象

- [[configobject/UNC/20.15.2/CHGTARI]] · 费率时段配置（CHGTARI）

## 使用实例

1. 查询费率类型为工作日的普通计费属性的费率时段配置信息，配置格式为：
  LST CHGTARI: TT=WORK, CC=NORMAL;
  ```
  %%LST CHGTARI: TT=WORK, CC=NORMAL;%%
  RETCODE = 0  操作成功。

  输出结果如下
  ------------------
   费率类型  起始时间     结束时间     计费属性    费率变更允许的用户类型

   工作日    07:00:00     08:00:00     普通计费    本地用户&拜访用户
   工作日    08:00:00     08:53:00     普通计费    漫游用户
   工作日    09:05:00     11:05:00     普通计费    拜访用户
   工作日    11:05:00     11:16:00     普通计费    本地用户&漫游用户&拜访用户
  (结果个数 = 4)
  ---    END
  ```
2. 查询所有的费率时段配置信息，配置格式为：
  LST CHGTARI:;
  ```
  %%LST CHGTARI:;%%
  RETCODE = 0  操作成功。

  输出结果如下
  ------------------
   费率类型   起始时间    结束时间     计费属性     费率变更允许的用户类型

   工作日    07:00:00    08:00:00     普通计费     本地用户&拜访用户         
   工作日    08:00:00    08:53:00     普通计费     漫游用户                     
   工作日    09:05:00    11:05:00     普通计费     拜访用户                    
   工作日    11:05:00    11:16:00     普通计费     本地用户&漫游用户&拜访用户
   周休日    09:19:00    19:19:00     包月制       本地用户                     
   节假日    00:00:00    00:12:00     普通计费      漫游用户&拜访用户          
   节假日    09:19:00    19:19:00     实时计费      本地用户&漫游用户           
  (结果个数 = 7)

  ---    END
  ```

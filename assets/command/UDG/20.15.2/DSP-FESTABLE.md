# DSP FESTABLE（显示FES表项信息）

## 功能

该命令用于显示FES表项信息。

## 注意事项

无。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| TABLENAME | 转发数据表名称 | 可选必选说明：可选参数<br>参数含义：转发数据表名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～15。<br>默认值：无 |
| TABLEID | 转发数据表ID | 可选必选说明：可选参数<br>参数含义：转发数据表ID。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～4294967295。<br>默认值：无 |
| FIRSTRECNUM | 获取前几条记录 | 可选必选说明：可选参数<br>参数含义：获取前几条记录。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～4294967295。<br>默认值：无 |
| FIELD | 字段名称 | 可选必选说明：可选参数<br>参数含义：字段名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～15。<br>默认值：无 |
| OPERFLAG | 命令操作标记 | 可选必选说明：可选参数<br>参数含义：命令操作标记。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～4294967295。<br>默认值：无 |
| MBORLC | 操作维护单元还是接口处理单元 | 可选必选说明：可选参数<br>参数含义：操作维护单元还是接口处理单元。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- mbfes：操作维护单元。<br>- lcfes：接口处理单元。<br>默认值：无 |
| RUNAME | 资源单元名称 | 可选必选说明：可选参数<br>参数含义：资源单元名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。<br>默认值：无<br>配置原则：区分大小写，不支持空格，下发本MML命令前可使用DSP RU查看资源单元信息。 |

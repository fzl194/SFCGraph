# DSP DBGDATAEX（显示调试信息）

## 功能

该命令用于查询进程运行中的调试信息。当管理员需要查询流控功能当前的算法参数、服务订阅信息、流量等数据时，可以使用本命令查询。

> **说明**
> 无

## 权限

G_1，管理员级别命令组

## 使用实例

查询SDRC进程（Cell类型为104）上注册的流控参数列表。

```
DSP DBGDATAEX: CELLTYPE=104, PARAM1="DspFccRegSvc";
RETCODE = 0  操作成功

结果如下
------------------------
输出0        输出1  输出2  输出3  输出4  输出5  输出6  输出7  输出8  输出9  

Service      TID   GID    -      -     -      -      -     -      -       
UesmCtrlSvc  999   0      -      -     -      -      -     -      -       
(结果个数 = 2)
```

# LOD SIGNATUREDB（加载协议特征库文件）

## 功能

**适用NF：PGW-U、UPF**

该命令用于加载特定的完整知识库文件。如果已经加载知识库文件，则该命令用于升级新知识库文件。升级知识库文件不中断和影响业务。

## 注意事项

- 该命令执行后立即生效。
- 新增知识库可以通网管/MAE的“网元文件传输”功能或者OM Portal的“文件传输”功能进行库文件上传。
- 待更新的知识库文件相比较当前使用版本的知识库文件，不允许删除协议或者修改协议名称。如果发生了上述的情况，需确保在加载新知识库文件之前已经将该协议相关配置删除或修改。
- 系统未加载协议特征知识库文件，将无法进行七层协议解析业务。加载旧版本的知识库文件可能会影响系统对七层协议业务的识别能力。
- 知识库文件的加载大约需要5分钟，5分钟后可使用DSP SIGNATUREDB命令来查看当前系统加载上的知识库版本。如果重复执行加载，建议加载之后5分钟以上再执行加载命令，否则可能加载失败。
- 在环境初始部署时，会默认以LATEST模式加载知识库。
- 按照指定最新的特征库版本加载协议特征知识库的时候，库文件有两种来源，默认加载csp远端最新的库文件，如果期望加载csp远端和本地缓存的最新库文件，可以根据软参BYTE687进行配置。
- 加载不同版本的知识库可能会导致产生协议变更并产生告警，请使用DSP RULECHECK查看相关信息并确认变更。
- 该命令会导致用户匹配范围发生变化，可能导致性能下降。执行命令前请评估对性能的影响，如果无法评估请联系华为技术支持。
- 执行该命令5分钟后，通过DSP SIGNATUREDB查看如果知识库已加载完成，且无“ALM-81011 SIG知识库升级失败”告警产生，则认为知识库加载成功。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| LOADMODE | 特征库加载模式 | 可选必选说明：必选参数<br>参数含义：该参数用于指定特征库的加载模式。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- SPECIFIC：指定特定的特征库版本加载。<br>- LATEST：指定最新的特征库版本加载。<br>- REFRESH：指定最新的特征库版本更新，但不改变配置保存方式。<br>默认值：无<br>配置原则：<br>- 如果运营商需要加载知识库时配置该文件，该参数用于指定加载知识库时加载为指定版本或最新版本。<br>- 配置为SPECIFIC时加载特定的特征库版本；配置为LATEST时加载为最新的特征库版本；配置为REFRESH时加载为最新的特征库版本，但不改变配置保存方式。 |
| VERSION | 特征库版本号 | 可选必选说明：条件必选参数<br>前提条件：该参数在“LOADMODE”配置为“SPECIFIC”时为必选参数。<br>参数含义：该参数用于指定加载的知识库的版本号。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～16。长度范围是1～16，版本号形式如01.002.ABCD.01，A、B、C、D值为0~9的数字。<br>默认值：无<br>配置原则：如果运营商需要加载知识库时配置该命令，该参数用于在加载特定知识库版本时指定知识库版本号。 |

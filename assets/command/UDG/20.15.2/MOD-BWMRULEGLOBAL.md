# MOD BWMRULEGLOBAL（修改全局带宽管理规则）

## 功能

**适用NF：PGW-U、UPF**

该命令用于修改一条全局业务带宽管理规则。当运营商希望修改全局规则的用户业务类型和控制器参数时，则执行该命令。

## 注意事项

- 该命令执行后立即生效。
- 需要为每一个业务控制规则指定优先级，且不能相同。进行业务规则匹配时，优先匹配优先级高的业务控制规则，优先级的值越小，优先级越高。
- 全局用户组下无默认的业务带宽管理规则。
- 全局的业务带宽管理规则只支持配置TOS类业务，如果将非TOS业务绑定到全局规则上，会提示失败，并且所有的全局带宽管理规则下的带宽管理业务的TOS值不能相同。
- 全局的业务带宽管理规则只能绑定到全局用户组下。
- 该命令设定后的数据，需要通过LST BWMRULEGLOBAL命令进行查看。
- BwmRuleGlobal不支持配置Shaping控制类型的BwmController。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组

## 使用实例

假如运营商需要修改全局的带宽管理规则，该规则名称为“testbwmruleglobal”：

```
MOD BWMRULEGLOBAL:BWMRULENAME="testbwmruleglobal",BWMSERVICENAME="bs3",UPBWMCTRLNAME1="testupbwmctrl",BWMRULEPRI=16,SERVICELEVEL=2;
```

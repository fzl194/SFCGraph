# LST NODE（查询节点信息）

## 功能

该命令用于查询网元节点的信息。

## 注意事项

无。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| NODENAME | 节点名称 | 可选必选说明：可选参数<br>参数含义：该参数表示所要查询的网元节点名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，区分大小写。<br>默认值：无 |
| NODETYPE | 节点类型 | 可选必选说明：可选参数<br>参数含义：该参数表示所要查询的网元节点类型。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，区分大小写。<br>默认值：无 |
| USERLABEL | 用户标签 | 可选必选说明：可选参数<br>参数含义：用户可通过该参数增加备注信息。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～255。<br>默认值：无 |
| SWVERSION | 软件版本 | 可选必选说明：可选参数<br>参数含义：该参数表示软件版本名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，区分大小写。<br>默认值：无 |
| SWHOTPATCHVERSION | 软件热补丁版本 | 可选必选说明：可选参数<br>参数含义：该参数表示软件的热补丁号。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，区分大小写。<br>默认值：无 |
| NERMVERSION | 网元资源模型版本 | 可选必选说明：可选参数<br>参数含义：不同的网元软件，如果其模型定义（MOC定义、属性定义、配置规则等）相同，则模型版本相同。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，区分大小写。<br>默认值：无 |
| INTERFACEID | 产品接口号 | 可选必选说明：可选参数<br>参数含义：该属性表征和M2000的南向接口，一个编号对应于一套完整的南向接口，值来自VNFC的vercfg.xml文件里的Version字段。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～127。不支持空格，区分大小写。<br>默认值：无 |
| CONTAINERAPPLICATIONID | 组件名称 | 可选必选说明：可选参数<br>参数含义：建立Function、Node与CenuxNode内的Application对象的对应关系。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～65535。不支持空格。<br>默认值：无 |
| NODEID | 网元节点号 | 可选必选说明：可选参数<br>参数含义：对应网元节点号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～65536。不支持空格。<br>默认值：无 |
| SERVICEINSTANCE | 服务实例 | 可选必选说明：必选参数<br>参数含义：该参数表示大颗粒服务实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，通过LST VNFC命令获取。<br>默认值：无<br>配置原则：只能填写通过LST VNFC命令查询到的管理代理标识，但不能填写0，0表示VNFP。 |

# MOD MPACIPV6RULE（修改IPv6 MPAC策略规则）

## 功能

该命令用于修改MPAC策略规则。

## 注意事项

该命令执行后立即生效。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| POLICYNAME | 策略名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定创建规则的策略名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。大小写敏感，英文字母开头，不支持空格。<br>默认值：无<br>配置原则：<br>- 必须指定策略名字。<br>- 数据库中已经存在的策略名。 |
| RULENAME | 规则名字 | 可选必选说明：必选参数<br>参数含义：该参数用于指定规则的名字。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。大小写敏感，英文字母开头，不支持空格。<br>默认值：无<br>配置原则：字符串形式，必须指定，不能为空。 |
| RULEID | 规则ID | 可选必选说明：可选参数<br>参数含义：该参数用于指定规则的编号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～4294967294。ID在策略中唯一。<br>默认值：无 |
| ACTION | 规则动作 | 可选必选说明：必选参数<br>参数含义：该参数用于指定匹配本条规则后要执行的动作类型。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- permit：允许报文上送。<br>- deny：拒绝报文上送。<br>默认值：无<br>配置原则：必须给出动作类型。 |
| PROTTYPE | 协议方式 | 可选必选说明：必选参数<br>参数含义：该参数用于指定匹配规则的协议方式是协议号还是协议名。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- Number：IP协议号。<br>- Name：协议名称。<br>默认值：无 |
| PROTNAME | 协议名称 | 可选必选说明：条件必选参数<br>前提条件：该参数在“PROTTYPE”配置为“Name”时为必选参数。<br>参数含义：该参数用于指定规则匹配的协议名称。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- None：未指定协议名称。<br>- BGP：BGP协议报文。<br>- OSPF：OSPF协议报文。<br>- TCP：TCP协议报文。<br>- UDP：UDP协议报文。<br>- LDP：LDP协议报文。<br>- LSP-PING：LSP-PING协议报文。<br>- NTP：NTP协议报文。<br>- SNMP：SNMP协议报文。<br>- SSH：SSH协议报文。<br>- ANY：所有协议报文。<br>默认值：无 |
| PROTNUM | IPv6协议号 | 可选必选说明：条件必选参数<br>前提条件：该参数在“PROTTYPE”配置为“Number”时为必选参数。<br>参数含义：该参数用于指定规则匹配的协议号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～255。<br>默认值：无 |
| SRCIP | 源地址 | 可选必选说明：可选参数<br>参数含义：该参数用于指定源IPv6地址。<br>数据来源：本端规划<br>取值范围：IPv6地址类型。<br>默认值：无<br>配置原则：只有协议名称不是ANY时支持此参数配置。 |
| SRCIPPREFIX | 源地址前缀长度 | 可选必选说明：可选参数<br>参数含义：该参数用于指定源IPv6地址前缀长度。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～128。<br>默认值：无<br>配置原则：<br>- 配置了源地址，必须设置此参数值。<br>- 不配置此参数，查询出默认值为0。<br>- 只有协议名称不是ANY时支持此参数配置。 |
| DSTIP | 目的地址 | 可选必选说明：可选参数<br>参数含义：该参数用于指定目的IPv6地址。<br>数据来源：本端规划<br>取值范围：IPv6地址类型。<br>默认值：无<br>配置原则：只有协议名称不是ANY时支持此参数配置。 |
| DSTIPPREFIX | 目的地址前缀长度 | 可选必选说明：可选参数<br>参数含义：该参数用于指定目的IPv6地址前缀长度。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～128。<br>默认值：无<br>配置原则：<br>- 配置了目的地址，必须设置此参数值。<br>- 不配置此参数，查询出默认值为0。<br>- 只有协议名称不是ANY时支持此参数配置。 |
| SRCPORT | 源端口号 | 可选必选说明：可选参数<br>参数含义：该参数用于指定源端口号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～65535。0为无效端口号。<br>默认值：无<br>配置原则：<br>- 只有TCP和UDP协议支持此参数配置。<br>- 不配置此参数，查询出默认值为0。 |
| DSTPORT | 目的端口号 | 可选必选说明：可选参数<br>参数含义：该参数用于指定目的端口号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～65535。0为无效端口号。<br>默认值：无<br>配置原则：<br>- 只有TCP和UDP协议支持此参数配置。<br>- 不配置此参数，查询出默认值为0。 |

## 操作的配置对象

- [[UDG@20.15.2@ConfigObject@MPACIPV6RULE]] · IPv6 MPAC策略规则（MPACIPV6RULE）

## 使用实例

修改IPv6策略规则：

```
MOD MPACIPV6RULE:POLICYNAME="policyV6",RULENAME="name1",ACTION=deny,PROTTYPE=Number,PROTNUM=17,SRCIP="2001:db8::10",SRCIPPREFIX=32;
```

# DSP RMMSGDETAIL（查询路由管理消息明细）

## 功能

该命令用来查询路由管理消息明细。

## 注意事项

无。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| ADDRESSFAMILY | 地址族 | 可选必选说明：必选参数<br>参数含义：该参数用于表示地址族。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- ipv4unicast：IPv4单播。<br>- ipv6unicast：IPv6单播。<br>默认值：无 |
| MSGTRACETYPE | 消息类型 | 可选必选说明：必选参数<br>参数含义：该参数用于表示消息类型。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- send：发送。<br>- receive：接收。<br>默认值：无 |

## 操作的配置对象

- [[UDG@20.15.2@ConfigObject@RMMSGDETAIL]] · 路由管理消息明细（RMMSGDETAIL）

## 使用实例

查询路由管理消息明细：

```
DSP RMMSGDETAIL:ADDRESSFAMILY=ipv4unicast,MSGTRACETYPE=send;
```

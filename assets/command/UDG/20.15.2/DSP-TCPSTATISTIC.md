# DSP TCPSTATISTIC（查询TCP报文统计）

## 功能

该命令用来查看TCP连接报文统计信息，用于检查网络的连接状态。

## 注意事项

无。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IPVERSION | IP版本 | 可选必选说明：可选参数<br>参数含义：该参数表示当前显示的统计信息的IP版本。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- IPv4：IPv4协议族。<br>- IPv6：IPv6协议族。<br>默认值：IPv4 |

## 操作的配置对象

- [[UDG@20.15.2@ConfigObject@TCPSTATISTIC]] · TCP报文统计（TCPSTATISTIC）

## 使用实例

显示当前系统IPv4协议的TCP报文相关统计信息：

```
DSP TCPSTATISTIC:;
```

# DSP MEASMOC（查询话统测量对象类模型）

## 功能

该命令用于查询网元的测量对象类模型信息。

> **说明**
> 无。

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| MEID | 网元ID | 可选必选说明：必选参数。<br>参数含义：用于指定系统按照网元ID来查询测量对象类模型。<br>取值范围：整数类型，取值范围为0~65535。<br>默认值：无。<br>配置原则：本参数需要与<br>[**LST ME**](../../系统管理/版本信息/查询网元配置信息（LST ME）_47084797.md)<br>中查询到的MEID值保持一致。 |
| MOCID | 测量对象类ID | 可选必选说明：可选参数。<br>参数含义：用于指定系统按照测量对象类ID来查询测量对象类模型。<br>取值范围：整数类型，取值范围为0~4294967294。<br>默认值：无。<br>配置原则：无。 |
| MUID | 测量单元ID | 可选必选说明：可选参数。<br>参数含义：用于指定系统按照测量指标组ID来查询测量对象类模型。<br>取值范围：整数类型，取值范围为0~4294967294。<br>默认值：无。<br>配置原则：无。 |

## 操作的配置对象

- [话统测量对象类模型（MEASMOC）](configobject/UDG/20.15.2/MEASMOC.md)

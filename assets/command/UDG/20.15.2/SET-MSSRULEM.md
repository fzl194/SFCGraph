# SET MSSRULEM（设置匹配规则）

## 功能

该命令用于设置匹配规则。

用户只有设置规则后，才能绑定该规则对消息或者报文进行匹配，匹配规则是从消息、报文的偏移类型对应的位置加上偏移大小开始，比较“规则内容&规则掩码”和“报文内容&规则掩码”是否一致。

## 注意事项

该命令执行后立即生效。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组;

## 使用实例

设置微服务类型为104的微服务实例csdb-pod-0172-16-0-247__103__0的匹配规则：

```
%%SET MSSRULEM: CELLTYPE="104", CELLINSTANCE="csdb-pod-0172-16-0-247__103__0", ENABLE=TRUE, RULEID=1, OFFSETTYPE=ufp-pkt-ctrl, OFFSET=0, PATTERN="0000", MASK="ffff";%%
RETCODE = 0  操作成功

---    END
```

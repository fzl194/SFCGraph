# DSP SERVICEINST（显示注册的服务实例信息）

## 功能

该命令用于查询在CMF上注册的服务实例信息。

若需要查询指定网元在CMF上注册的服务实例信息，请输入“网元ID”参数。若不输入“网元ID”参数，表示查询所有网元在CMF上注册的服务实例信息。

若需要查询在CMF上注册的所有服务实例信息，请将输入参数中的“查询类型”选择为“所有”。

若需要通过服务名称查询在CMF上注册的服务实例信息，请将输入参数中的“查询类型”选择为“通过服务名称”，并输入“服务名称”参数。

若需要通过服务实例ID查询在CMF上注册的服务实例信息，请将输入参数中的“查询类型”选择为“通过服务实例ID”，并输入“服务实例ID”参数。

若需要通过服务实例标签查询在CMF上注册的服务实例信息，请将输入参数中的“查询类型”选择为“通过服务实例标签”，并输入“服务实例标签键”和“服务实例标签值”参数。其中，键为必选参数，值为可选参数。

若需要根据节点ID、Pod ID或进程ID查询在CMF上注册的服务实例信息，请将输入参数中的“查询类型”选择为“通过服务实例标签”，然后在“服务实例标签键”参数中输入“Node ID”、“Pod ID”或“Cell ID”，并输入“服务实例标签值”，注意此时“服务实例标签值”不能为空。

> **说明**
> 当输入参数中的“查询类型”选择为“通过服务实例标签”时，“服务实例标签键”和“服务实例标签值”只支持通过服务实例单个标签的键和值查询，不支持同时通过服务实例多个标签的键和值查询。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| TYPE | 查询类型 | 可选必选说明：必选参数<br>参数含义：该参数用于指定查询在CMF上注册的服务实例信息的方式。<br>数据来源：本端规划<br>取值范围：<br>- “ALL（所有）”：查询所有在CMF上注册的服务实例信息<br>- “BYNAME（通过服务名称）”：通过服务名称查询在CMF上注册的服务实例信息<br>- “BYID（通过服务实例ID）”：通过服务实例ID查询在CMF上注册的服务实例信息<br>- “BYLABEL（通过服务实例标签）”：通过服务实例标签查询在CMF上注册的服务实例信息<br>默认值：无<br>配置原则：无 |
| SERVICENAME | 服务名称 | 可选必选说明：该参数在"TYPE"配置为"BYNAME"时为条件必选参数。<br>参数含义：该参数用于表示服务名称。服务名称可通过<br>[**DSP SERVICENAME**](显示注册的服务名称（DSP SERVICENAME）_52606173.md)<br>命令查询获取。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~31。不区分大小写。<br>默认值：无<br>配置原则：<br>当按服务名称查询实例信息时，该参数支持模糊匹配。 |
| INSTANCEID | 服务实例ID | 可选必选说明：该参数在"TYPE"配置为"BYID"时为条件必选参数。<br>参数含义：该参数用于表示服务实例ID。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~64。<br>默认值：无<br>配置原则：无 |
| LABELKEY | 服务实例标签键 | 可选必选说明：该参数在"TYPE"配置为"BYLABEL"时为条件必选参数。<br>参数含义：该参数用于表示服务实例标签的键。服务实例标签是KV键值对。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~128。服务实例标签键不允许输入以下字符：,（逗号）、;（分号）、=（等号）、"（双引号）、'（单引号）。<br>默认值：无<br>配置原则：无 |
| LABELVALUE | 服务实例标签值 | 可选必选说明：该参数在"TYPE"配置为"BYLABEL"时为条件可选参数。<br>参数含义：该参数用于表示服务实例标签的值。服务实例标签是KV键值对。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~128。服务实例标签值不允许输入以下字符：,（逗号）、;（分号）、=（等号）、"（双引号）、'（单引号）。<br>默认值：无<br>配置原则：无 |
| MEID | 网元ID | 可选必选说明：该参数在"TYPE"配置为"BYID"、"BYNAME"、"BYLABEL"、"ALL"时为条件可选参数。<br>参数含义：该参数用于指定网元来查询在CMF上注册的服务实例信息，可通过<br>[**LST ME**](../../系统管理/版本信息/查询网元配置信息（LST ME）_47084797.md)<br>命令查询获取。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~40。<br>默认值：无<br>配置原则：<br>该参数若不输入，则表示查询所有网元在CMF上注册的服务实例信息。 |

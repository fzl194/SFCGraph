# SET ALMFILTER（设置告警闪断振荡配置）

## 功能

本命令用于设置系统内告警的闪断振荡配置信息。

> **说明**
> 告警的闪断振荡配置信息只对未屏蔽告警有效。对于被屏蔽的告警，闪断振荡配置信息实际中不起作用。

> **说明**
> 告警的闪断振荡配置信息只对未屏蔽告警有效。

## 参数

| **参数标识** | **参数名称** | **参数说明** |
| --- | --- | --- |
| MEID | 网元ID | 可选必选说明：必选参数。<br>参数含义：用于指示系统需要设置哪个网元的告警配置信息。<br>取值范围：0~65535<br>默认值：无。<br>配置原则：网元ID可以通过<br>[**LST ME**](../../系统管理/版本信息/查询网元配置信息（LST ME）_47084797.md)<br>命令查询。 |
| ALARMID | 告警ID | 可选必选说明：必选参数。<br>参数含义：用于指示系统需要设置哪个告警ID的配置信息。<br>取值范围：0~999999999<br>默认值：无。<br>配置原则：无。 |
| SUPPRESSTYPE | 抑制类型 | 可选必选说明：必选参数。<br>参数含义：告警被系统抑制上报的方式。<br>取值范围：<br>- “NO(未抑制)”：表示系统不抑制告警上报。<br>- “INTERMITTENT(闪断抑制)”：告警闪断指一条告警的产生时间与恢复时间的间隔小于某一特定值（“闪断周期（秒）”），则不在告警页面显示该告警，系统通过“闪断周期”抑制告警闪断。<br>- “TOGGLING(振荡抑制)”：告警振荡指在单位时间内（“振荡产生周期（秒）”），同一对象的同一告警（即告警产生对象和告警ID都相同的告警）交替产生/恢复的次数大于某一特定值（“振荡阈值（秒）”），则该告警进入振荡抑制，新上报的告警不再显示在告警页面。进入振荡抑制的告警，在指定时间内（“振荡恢复周期（秒）”），不再继续交替产生/恢复，则退出振荡抑制，新上报的告警可显示在告警页面。<br>- “ALL(闪断振荡抑制)”：系统根据闪断周期、振荡产生周期、振荡恢复周期、振荡阈值等参数对上报的告警进行检查和抑制，检测顺序为先检测振荡告警，不满足则检测闪断告警。<br>默认值：无。<br>配置原则：无。 |
| TP | 闪断周期（秒） | 可选必选说明：在抑制类型配置为“INTERMITTENT(闪断抑制)”或者“ALL(闪断振荡抑制)”时，该参数为必选参数。<br>参数含义：用于指定检测告警产生闪断的时间值。闪断周期必须小于振荡产生周期。<br>取值范围：1～1800，如果是VNFC类型服务的告警，则取值范围是1～600。<br>默认值：无。<br>配置原则：无。 |
| TSP | 振荡产生周期（秒） | 可选必选说明：在抑制类型配置为“TOGGLING(振荡抑制)”或者“ALL(闪断振荡抑制)”时，该参数为必选参数。<br>参数含义：用于指定检测告警是否产生振荡的时间段。振荡产生周期必须大于闪断周期。<br>取值范围：1～3600<br>默认值：无。<br>配置原则：无。 |
| TEP | 振荡恢复周期（秒） | 可选必选说明：在抑制类型配置为“TOGGLING(振荡抑制)”或者“ALL(闪断振荡抑制)”时，该参数为必选参数。<br>参数含义：用于指定检测一个告警的振荡是否结束的时间段。<br>取值范围：1～3600<br>默认值：无。<br>配置原则：无。 |
| TT | 振荡阈值（次） | 可选必选说明：在抑制类型配置为“TOGGLING(振荡抑制)”或者“ALL(闪断振荡抑制)”时，该参数为必选参数。<br>参数含义：用于指定在振荡产生周期内，判断告警是否产生振荡的故障告警和清除告警交替产生的次数。<br>取值范围：1～3600<br>默认值：无。<br>配置原则：无。 |

## 使用实例

设置告警闪断振荡配置：

```
%%SET ALMFILTER: MEID=0, ALARMID=5521, SUPPRESSTYPE=ALL, TP=11, TSP=22, TEP=33, TT=4;%%
RETCODE = 0  操作成功

---    END
```

# MOD URRGROUP（修改URR组）

## 功能

**适用NF：PGW-U、UPF**

该命令用于修改使用量上报规则组，通过该命令可以指定上下行发起使用的URR名称，即指定上下行报文如何计费。

## 注意事项

- 该命令执行后立即生效。
- 如果引用了该使用量上报规则组的上下文激活时特定URR记录存在，则不允许修改。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组

## 使用实例

假如运营商需要修改使用量上报规则组，修改上行发起在线的URR信息为uponurr,下行发起在线的URR信息为downonurr：

```
MOD URRGROUP: URRGROUPNAME="urrgroup2", UPURRNAME1="uponurr", DOWNURRNAME1="downonurr";
```

# DSP FEICOM_CSTDATA（显示存储池存储的表项信息）

## 功能

该命令用于显示存储池中存储的表项信息。

## 注意事项

该命令仅适用于非NP卡场景和NP卡非加速模式场景。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| CSTID | 存储池ID | 可选必选说明：必选参数<br>参数含义：存储池ID。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～4294967294。<br>默认值：无 |
| RUNAME | RU名称 | 可选必选说明：可选参数<br>参数含义：指定资源单元名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。<br>默认值：无<br>配置原则：<br>- 区分大小写，不支持空格。<br>- 使用DSP RU查看RU名称。 |
| KEY | 表项关键值 | 可选必选说明：可选参数<br>参数含义：表项的关键值，用于唯一标识一条表项。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～255。<br>默认值：无 |

## 操作的配置对象

- [存储池存储的表项信息（FEICOM_CSTDATA）](configobject/UDG/20.15.2/FEICOM_CSTDATA.md)

DSP FEICOM_CSTDATA: CSTID=0, RUNAME="VNODE_VNRS_VNFC_IPU_0064";

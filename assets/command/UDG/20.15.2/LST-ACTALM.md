# LST ACTALM（查询活动告警）

## 功能

本命令用于查询系统内活动告警，可以通过OM Portal的 “ 监控分析 > 告警管理 > 活动告警 ” 进行查看，查询结果为最新产生的告警。

> **说明**
> 该MML除能查到OM Portal的 “ 监控分析 > 告警管理 > 活动告警 ” 的故障告警外，还支持查询 “事件告警” 和 “未确定告警” 。故障告警对系统有不同程度影响，而 “事件告警” 对系统无影响， “未确定告警” 则对系统影响程度无法确定，因此不会在OM Portal界面呈现。

> **说明**
> 默认查询64条，最大支持1000条告警。

## 参数

| **参数标识** | **参数名称** | **参数说明** |
| --- | --- | --- |
| MEID | 网元ID | 可选必选说明：可选参数。<br>参数含义：用于指示系统需要按照哪个网元来查询活动告警；若不输入，则表示所有网元；网元ID可以通过<br>[**LST ME**](../../系统管理/版本信息/查询网元配置信息（LST ME）_47084797.md)<br>命令查询。<br>取值范围：0~65535<br>默认值：无。<br>配置原则：无。 |
| ALARMNAME | 告警名称 | 可选必选说明：可选参数。<br>参数含义：用于指示系统查询告警时需要精确查找哪个名称的告警；若不输入，则表示所有告警。<br>取值范围：最大64个字节。<br>默认值：无。<br>配置原则：无。 |
| ALARMID | 告警ID | 可选必选说明：可选参数。<br>参数含义：用于指示系统查询告警时需要精确查找哪个告警ID；若不输入，则表示所有告警ID。告警ID：用于确定一类告警，代表一种特定的故障，是告警项的唯一索引。<br>取值范围：0~999999999<br>默认值：无。<br>配置原则：无。 |
| SAID | 起始告警ID | 可选必选说明：可选参数。<br>参数含义：用于指示系统查询时，告警ID的取值下限；若不输入，则表示取值下限<br>**大于等于**<br>**0**<br>且<br>**小于等于**<br>结束告警ID。<br>取值范围：0~999999999<br>默认值：无。<br>配置原则：无。 |
| EAID | 结束告警ID | 可选必选说明：可选参数。<br>参数含义：用于指示系统查询时，告警ID的取值上限；若不输入，则表示取值上限<br>**小于等于999999999**<br>且<br>**大于等于**<br>起始告警ID。<br>取值范围：0~999999999<br>默认值：无。<br>配置原则：无。 |
| ALVL | 告警级别 | 可选必选说明：可选参数。<br>参数含义：用于指示系统需要按照哪个告警级别来查询活动告警；若不输入，则表示所有告警级别。告警级别：是故障告警对业务的影响程度，按严重程度递减分为6级，详见取值范围。若输入是清空、全部选择、全部置灰，则表示不作限制。<br>取值范围：<br>- “紧急告警”：此类级别的故障影响到系统提供的服务，需要立即采取相应动作。如某设备或资源完全不可用，需进行恢复，即使该故障在非工作时间内发生，也需立即采取措施。<br>- “重要告警”：此类级别的故障影响到服务质量，需要采取紧急动作。如某设备或资源服务质量下降，需对其进行还原，恢复全部能力，需在工作时间内立即采取措施。<br>- “次要告警”：此类级别的故障还未影响到服务质量，但为了避免更严重的故障，需要在适当时候进行处理或进一步观察。<br>- “提示告警”：此类级别的故障指示可能有潜在的错误影响到提供的服务，相应的措施根据不同的错误进行处理。<br>- “事件告警”：此类级别的故障对服务质量没有影响，用于提示用户系统运行过程中产生的事件。此级别为预留告警级别，当前无告警使用。<br>- “未确定告警”：此类级别的故障可能存在错误影响到提供的服务，影响程度无法确定，需要根据相应措施确认影响后进行处理。此级别为预留告警级别，当前无告警使用。<br>默认值：全部灰化。<br>配置原则：无。 |
| STARTTIME | 起始时间 | 可选必选说明：可选参数。<br>参数含义：用于指示系统查询活动告警时告警产生时间的取值下限；若不输入，则表示不作限制。<br>取值范围：1990/01/01 00:00:00~2037/12/31 23:59:59<br>默认值：无。<br>配置原则：无。 |
| ENDTIME | 结束时间 | 可选必选说明：可选参数。<br>参数含义：用于指示系统查询活动告警时告警产生时间的取值上限；若不输入，则表示不作限制。<br>取值范围：1990/01/01 00:00:00~2037/12/31 23:59:59<br>默认值：无。<br>配置原则：无。 |
| SS | 起始流水号 | 可选必选说明：可选参数。<br>参数含义：用于指示系统查询活动告警时流水号的取值下限；若不输入，则表示没有下限。流水号：是在一个网元内对故障告警进行的编号，恢复告警的流水号与其对应的故障告警流水号相同，由告警服务自动生成。<br>取值范围：0~2147483647<br>默认值：无。<br>配置原则：无。 |
| ES | 结束流水号 | 可选必选说明：可选参数。<br>参数含义：用于指示系统查询活动告警时流水号的取值上限；若不输入，则表示没有上限。<br>取值范围：0~2147483647<br>默认值：无。<br>配置原则：无。 |
| CNT | 返回记录数 | 可选必选说明：可选参数。<br>参数含义：用于指示系统返回记录条数的最大值；若不输入，则表示返回记录的最大数目是64。<br>取值范围：1~1000<br>默认值：无。<br>配置原则：无。 |

# ADD GRETUNNEL（增加GRE隧道）

## 功能

该命令用于创建GRE隧道。

通用路由封装GRE（Generic Routing Encapsulation）是对某些网络层协议的报文进行封装，使这些被封装的报文能够在另一网络层协议（如IP）中传输。GRE可以作为VPN的第三层隧道协议，为VPN数据提供透明传输通道。

## 注意事项

- 该命令执行后立即生效。
- 该命令最大记录数为4096。

## 权限

G_1，管理员级别命令组；G_2，操作员级别命令组

| CHECKSUMEN | 使能端到端校验功能 | 可选必选说明：可选参数<br>参数含义：该参数用于使能隧道端到端校验功能。使能后该功能，对端将对接收到的报文计算校验和，并与报文中的校验和比较，不一致则丢弃报文。<br>数据来源：本端规划<br>取值范围：布尔类型，输入格式为“TRUE”或者“FALSE”。<br>默认值：FALSE<br>配置原则：如果本端配置了校验和而对端没有配置，则本端将不对接收到的报文进行校验，但对本端发送的报文计算校验和；如果本端没有配置校验和而对端配置，则本端对从对端发来的报文进行校验和检查，但对本端发送的报文不计算校验和。 |

## 使用实例

建立GRE隧道，此接口封装的报文实际出口为Ethernet64/0/3，目的端地址为10.3.3.11，配置识别关键字为200：

```
ADD GRETUNNEL:TNLNAME="Tunnel1",TNLTYPE=gre, SRCTYPE=if_name,SRCIFNAME="Ethernet64/0/3",DSTIPADDR="10.3.3.11", GREKEYEN=TRUE, GREKEY="200";
```

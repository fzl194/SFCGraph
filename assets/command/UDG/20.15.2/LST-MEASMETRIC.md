# LST MEASMETRIC（查询话统测量指标参数）

## 功能

该命令用于查询话统测量指标参数。

> **说明**
> 无。

## 参数

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| MEID | 网元ID | 可选必选说明：可选参数。<br>参数含义：标识网元ID，可以通过<br>[**LST ME**](../../系统管理/版本信息/查询网元配置信息（LST ME）_47084797.md)<br>命令查询获取。<br>取值范围：0~65535。<br>默认值：无。<br>配置原则：若不输入，则表示该参数不作为查询的限制条件。 |
| METRICID | 测量指标ID | 可选必选说明：可选参数。<br>参数含义：测量指标ID，可以通过<br>[**LST MEASUNIT**](查询测量指标模型(LST MEASUNIT)_47814449.md)<br>命令查询获取。<br>取值范围：整数类型，取值范围为0~4294967294。<br>默认值：无。<br>配置原则：若不输入，则表示该参数不作为查询的限制条件。 |
| METRIC_TYPE | 指标分类 | 可选必选说明：可选参数。<br>参数含义：指标的分类选项。<br>取值范围：位域项<br>- NORTH(北向指标)<br>- BASIC(基础指标)<br>- EXTEND(扩展指标)<br>- KPI(KPI指标)<br>默认值：无。<br>配置原则：<br>- 不输入或“全部灰化”，表示该参数不作为查询条件。<br>- “全部清空”，表示查询不属于任何指标分类指标。<br>- “全部选中”，表示查询满足所有取值范围的指标。<br>- 勾选取值范围中的多个，表示查询满足所勾选多个指标分类的指标。<br>- 指标分类去勾选或者非全部灰化，表示查询不属于该指标分类的指标。 |

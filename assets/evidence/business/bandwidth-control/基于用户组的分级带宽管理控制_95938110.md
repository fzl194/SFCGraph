# 基于用户组的分级带宽管理控制

## [操作场景](#ZH-CN_OPI_0000001295938110)

当总带宽一定的情况下实现差异化业务的分级带宽管理控制：

多个业务优先级之间共享带宽，每个优先级的业务提供基本的带宽，高优先级的业务在保障低优先级业务基础带宽的前提下可以优先获得带宽，高优先级的业务需求带宽较低时可以把剩余的带宽共享给低优先级的业务使用。

这样可以为低优先级业务提供基本保障，同时又保障高优先级业务的带宽优先诉求，实现业务差异化控制管理策略。

> **说明**
> 适用于PGW-U、UPF。

## [操作步骤](#ZH-CN_OPI_0000001295938110)

1. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09587387.md)
2. 配置BWM业务。
    a. 基于TOS定义BWM业务。
      [**ADD BWMSERVICE**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理业务/增加带宽管理业务（ADD BWMSERVICE）_82837473.md)
    b. 配置BWM控制器，设置CAR参数。
      [**ADD BWMCONTROLLER**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理控制器/增加带宽管理控制器（ADD BWMCONTROLLER）_82837460.md)
    c. 配置BWM控制器生效范围等管理参数。
      [**SET BANDWIDTHMNG**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理参数/设置带宽管理参数（SET BANDWIDTHMNG）_82837465.md)
3. 配置BWM业务级别策略。
  [**ADD BCSRVLEVELPLY**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理控制器业务级别策略/增加带宽管理控制器业务级别策略（ADD BCSRVLEVELPLY）_93097052.md)
4. 配置带宽管理用户组。
    a. 定义带宽管理用户组，配置优先级和业务类型。
      [**ADD BWMUSERGROUP**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理用户组/增加带宽管理用户组（ADD BWMUSERGROUP）_82837468.md)
5. 配置带宽管理规则。
    a. 配置用户组级带宽控制规则。
      [**ADD BWMRULE**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理规则/增加带宽管理规则（ADD BWMRULE）_82837478.md)
    b. 配置APN/DNN实例。
      [**ADD APN**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/DN管理/APN管理/APN/添加APN配置（ADD APN）_82837014.md)
    c. 将 APN 下的用户包含到该用户组。
      [**ADD APNBINDBWMUSRG**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理用户组APN绑定/增加带宽管理用户组APN绑定（ADD APNBINDBWMUSRG）_82837487.md)

## [任务示例](#ZH-CN_OPI_0000001295938110)

任务描述

本实例中，操作员在 UDG 上进行数据配置实现以下要求：

对APN用户组下的用户区分业务Service1/Service2/Service3实施分级带宽管理控制策略，保障各业务基本带宽的同时，优先满足Service1的上下行带宽，对APN用户的指定IP TOS等级的上下行流量进行流量监管。

*表1 针对不同Service业务实施分级带宽管理控制策略*

| 带宽控制项目 | Service1 | Service2 | Service3 |
| --- | --- | --- | --- |
| 业务等级（ServiceLevel） | 1 | 2 | 3 |
| 上下行保障带宽（CIR） | CIR<br>1 | CIR<br>2 | CIR<br>3 |
| 上下行保障突发大小<br>（CBS） | CBS<br>1 | CBS<br>2 | CBS<br>3 |
| 上下行最大带宽<br>（PIR） | PIR<br>1 | PIR<br>2 | PIR<br>3 |
| 上下行最大突发大小<br>（PBS） | PBS<br>1 | PBS<br>2 | PBS<br>3 |

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH:LICITEM="LKV3G5TCSA01",SWITCH=ENABLE;
```

//配置BWM业务。

```
ADD BWMSERVICE:BWMSERVICENAME="bs_test1",BWMSERVICETYPE=TOS,TOSCFGTYPE=CLASS,TOSCLASSTYPE=BE;
ADD BWMSERVICE:BWMSERVICENAME="bs_test2",BWMSERVICETYPE=TOS,TOSCFGTYPE=CLASS,TOSCLASSTYPE=AF1;
ADD BWMSERVICE:BWMSERVICENAME="bs_test3",BWMSERVICETYPE=TOS,TOSCFGTYPE=CLASS,TOSCLASSTYPE=AF2;
```

```
ADD BWMCONTROLLER:BWMCNAME="bc_test",CTRLTYPE=CAR,CIR=300,CBS=60000,PIR=500,PBS=100000,GREEN=PASS,YELLOW=PASS,RED=DISCARD,SRVLEVELSPEC=3;
```

```
SET BANDWIDTHMNG:RANGE=SYSTEM;
```

//配置BWM业务级别策略。

```
ADD BCSRVLEVELPLY: BWMCNAME="bc_test", SERVICELEVEL=1, CIRRATE=50, PIRRATE=70;
ADD BCSRVLEVELPLY: BWMCNAME="bc_test", SERVICELEVEL=2, CIRRATE=30, PIRRATE=50;
ADD BCSRVLEVELPLY: BWMCNAME="bc_test", SERVICELEVEL=3, CIRRATE=20, PIRRATE=100;
```

//配置带宽管理用户组。

```
ADD BWMUSERGROUP:USERGROUPTYPE=SPECIFIC_GROUP,USERGROUPNAME="ug_test",USERGROUPPRI=1,USERGLEVSRVTYPE=TOS,USERGLEVRULESW=ENABLE;
```

//配置带宽管理规则。

```
ADD BWMRULE:USERGROUPTYPE=SPECIFIC_GROUP,USERGROUPNAME="ug_test",BWMRULETYPE=GROUP_SPECIFIC,BWMRULENAME="br_test1",BWMSERVICENAME="bs_test1",UPBWMCTRLNAME1="bc_test",DNBWMCTRLNAME1="bc_test",BWMRULEPRI=10,SERVICELEVEL=1;
ADD BWMRULE:USERGROUPTYPE=SPECIFIC_GROUP,USERGROUPNAME="ug_test",BWMRULETYPE=GROUP_SPECIFIC,BWMRULENAME="br_test2",BWMSERVICENAME="bs_test2",UPBWMCTRLNAME1="bc_test",DNBWMCTRLNAME1="bc_test",BWMRULEPRI=15,SERVICELEVEL=2;
ADD BWMRULE:USERGROUPTYPE=SPECIFIC_GROUP,USERGROUPNAME="ug_test",BWMRULETYPE=GROUP_SPECIFIC,BWMRULENAME="br_test3",BWMSERVICENAME="bs_test3",UPBWMCTRLNAME1="bc_test",DNBWMCTRLNAME1="bc_test",BWMRULEPRI=18,SERVICELEVEL=3;
```

//配置APN绑定用户组。

```
ADD APN
:APN="apn-test", HASVPN=DISABLE, HASVPNIPV6=DISABLE;
ADD APNBINDBWMUSRG:USERGROUPNAME="ug_test",APN="apn-test";
```

## [操作步骤](#ZH-CN_OPI_0249155915)

1. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09587387.md)
2. 配置APN/DNN实例。
  [**ADD APN**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/DN管理/APN管理/APN/添加APN配置（ADD APN）_82837014.md)
3. 配置BWM业务。
    a. 配置两个BWM控制器，设置CAR参数并配置带宽控制算法。
      [**ADD BWMCONTROLLER**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理控制器/增加带宽管理控制器（ADD BWMCONTROLLER）_82837460.md)
      > **说明**
      > - 进行带宽控制时，如果要优先保证用户组内标识为绿色的用户流量，则需要配置[**ADD BWMCONTROLLER**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理控制器/增加带宽管理控制器（ADD BWMCONTROLLER）_82837460.md)“COLORISAWARE”为“ENABLE”。
      > - 该带宽管理功能只支持“CAR”的带宽管理控制类型。
    b. 配置BWM控制器中的用户组级业务带宽配置数据基于整机生效。
      [**SET BANDWIDTHMNG**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理参数/设置带宽管理参数（SET BANDWIDTHMNG）_82837465.md)
4. 配置带宽管理用户组。
    a. 定义带宽管理用户组，配置优先级和业务类型。
      [**ADD BWMUSERGROUP**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理用户组/增加带宽管理用户组（ADD BWMUSERGROUP）_82837468.md)
      > **说明**
      > 该带宽管理功能只支持 “NONTOS” 的用户组级业务类型。
    b. 配置一条用户组级带宽控制规则。
      [**ADD BWMRULE**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理规则/增加带宽管理规则（ADD BWMRULE）_82837478.md)
    c. 将 APN 下的用户包含到该用户组。
      [**ADD APNBINDBWMUSRG**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理用户组APN绑定/增加带宽管理用户组APN绑定（ADD APNBINDBWMUSRG）_82837487.md)

## [任务示例](#ZH-CN_OPI_0249155915)

任务描述

本实例中，操作员在 UDG 上进行数据配置实现以下要求：

UDG 对某个APN/DNN下的一个用户组进行流量监管。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH:LICITEM="LKV3G5TCSA01",SWITCH=ENABLE;
```

//配置APN。

```
ADD APN
: APN="apn-test", HASVPN=DISABLE, HASVPNIPV6=DISABLE;
```

//配置BWM业务。

```
ADD BWMCONTROLLER:BWMCNAME="bc-up",CTRLTYPE=CAR,CIR=2000000,CBS=320000000,PIR=4000000,PBS=
800000000,GREEN=PASS,YELLOW=PASS,RED=DISCARD,PRIORITYISAWARE=DISABLE;
```

```
ADD BWMCONTROLLER:BWMCNAME="bc-down",CTRLTYPE=CAR,CIR=2000000,CBS=320000000,PIR=4000000,PBS=
800000000,GREEN=PASS,YELLOW=PASS,RED=DISCARD,PRIORITYISAWARE=DISABLE;
```

```
SET BANDWIDTHMNG:RANGE=SYSTEM,CIRMINTHRES=300000,PIRMINTHRES=500000;
```

//配置带宽管理用户组。

```
ADD BWMUSERGROUP:USERGROUPTYPE=SPECIFIC_GROUP,USERGROUPNAME="ug_test",USERGROUPPRI=1,USERLEVSRVTYPE=NONTOS,USERGLEVSRVTYPE=NONTOS;
```

```
ADD BWMRULE:USERGROUPTYPE=SPECIFIC_GROUP,USERGROUPNAME="ug_test",BWMRULETYPE=GROUP_DEFAULT,UPBWMCTRLNAME1="bc-up",DNBWMCTRLNAME1="bc-down";
```

```
ADD APNBINDBWMUSRG:USERGROUPNAME="ug_test",APN="apn-test";
```

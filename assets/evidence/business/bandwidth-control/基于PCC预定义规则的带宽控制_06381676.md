## [操作场景](#ZH-CN_OPI_0206381676)

PCF/PCRF作为用户计费与控制策略管理中心，能够基于用户签约、套餐使用量等多种场景动态安装或者删除预定义规则。基于PCC预定义规则的带宽控制可应用于以下两种场景：

- PCF/PCRF可感知用户套餐的使用量，当用户业务访问量超出限额时通过安装预定义规则通知UDG执行带宽控制策略，以降低用户的访问速率，达到公平使用网络的原则，使访问相同业务的用户得到同样的网络体验。
- PCF/PCRF可感知SMF的PRA（Presence Reporting Area）位置上报，当用户进入指定区域时通过安装预定义规则通知UDG执行带宽控制策略，达到对指定区域的业务做限速的目的。

> **说明**
> 适用于PGW-U、UPF。

## [操作步骤](#ZH-CN_OPI_0206381676)

1. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09587387.md)
2. 配置BWM业务。
    a. 设置分类属性。
      [**ADD CATEGORYPROP**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/分类属性/增加分类属性（ADD CATEGORYPROP）_82837504.md)
    b. 设置基于分类属性的BWM业务。
      [**ADD BWMSERVICE**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理业务/增加带宽管理业务（ADD BWMSERVICE）_82837473.md)
    c. 配置BWM控制器，设置CAR参数。
      [**ADD BWMCONTROLLER**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理控制器/增加带宽管理控制器（ADD BWMCONTROLLER）_82837460.md)
    d. 定义带宽管理用户组，配置优先级和业务类型。
      [**ADD BWMUSERGROUP**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理用户组/增加带宽管理用户组（ADD BWMUSERGROUP）_82837468.md)
    e. 配置用户级带宽控制规则。
      [**ADD BWMRULE**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理规则/增加带宽管理规则（ADD BWMRULE）_82837478.md)
    f. 将 APN 下的用户包含到该用户组。
      [**ADD APNBINDBWMUSRG**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理用户组APN绑定/增加带宽管理用户组APN绑定（ADD APNBINDBWMUSRG）_82837487.md)
3. 配置业务感知规则。
    a. 配置七层过滤规则。
      [**ADD L7FILTER**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/七层规则管理/七层过滤器/增加七层过滤器（ADD L7FILTER）_82837397.md)
    b. 配置FlowFilter。
      [**ADD FLOWFILTER**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/流过滤器/增加流过滤器（ADD FLOWFILTER）_82837361.md)
    c. 将指定L7Filter绑定到FlowFilter。
      [**ADD PROTBINDFLOWF**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/流过滤器的协议绑定/增加流过滤器协议绑定关系（ADD PROTBINDFLOWF）_82837370.md)
    d. 将新配置的信息置为生效。
      [**SET REFRESHSRV**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/业务刷新/业务刷新（SET REFRESHSRV）_82837355.md)
    e. 配置业务使用的rule规则。
      [**ADD RULE**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务匹配公共配置/业务规则管理/规则/增加规则（ADD RULE）_82837266.md)
      > **说明**
      > - PCF/PCRF和UNC上仅配置预定义规则的名称， UDG 上配置该预定义规则的名称及具体的规则内容，如QoS管理、计费控制策略等。且PCF/PCRF、UNC 和 UDG 上的预定义规则名称必须规划一致。
      > - 带宽控制策略通过[**ADD RULE**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务匹配公共配置/业务规则管理/规则/增加规则（ADD RULE）_82837266.md)中的“POLICYNAME”和[**ADD BWMSERVICE**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理业务/增加带宽管理业务（ADD BWMSERVICE）_82837473.md)中的“CATEPROPNAME”关联获得，因此两者的值必须规划一致。

## [任务示例](#ZH-CN_OPI_0206381676)

任务描述

用户购买了某P2P业务的套餐， UDG 对该用户的业务进行带宽控制，具体控制策略：套餐耗尽前，不限速。套餐耗尽后，限制上行访问速率在700kbps之内，下行访问速率为600kbps之内。PCF/PCRF感知用户套餐的使用情况，在套餐耗尽后能通过下发预定义规则实现带宽控制，预定义规则名称为rule_p2pbwm。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH:LICITEM="LKV3G5TCSA01",SWITCH=ENABLE;
```

//配置BWM业务。

```
ADD CATEGORYPROP:CATEPROPNAME="cp_700_600";
```

```
ADD BWMSERVICE:BWMSERVICENAME="my-bwm-service1", BWMSERVICETYPE=NONTOS, NONTOSSRVTYPE=CATEGORY_PROP, CATEPROPNAME="cp_700_600";
```

```
ADD BWMCONTROLLER:BWMCNAME="bc-up-p2p", CTRLTYPE=CAR,CIR=100, PIR=700, GREEN=PASS, YELLOW=PASS, RED=DISCARD;
```

```
ADD BWMCONTROLLER:BWMCNAME="bc-dn-p2p", CTRLTYPE=CAR,CIR=100, PIR=600, GREEN=PASS, YELLOW=PASS, RED=DISCARD;
```

```
ADD BWMUSERGROUP:USERGROUPTYPE=SPECIFIC_GROUP,USERGROUPNAME="ug-bwm",USERGROUPPRI=1,USERLEVSRVTYPE=NONTOS;
```

```
ADD BWMRULE: USERGROUPTYPE=SPECIFIC_GROUP, USERGROUPNAME="ug-bwm", BWMRULETYPE=SUBSCRIBER_SPECIFIC, BWMRULENAME="br-p2p", BWMSERVICENAME="my-bwm-service1", UPBWMCTRLNAME1="bc-up-p2p", DNBWMCTRLNAME1="bc-dn-p2p", BWMRULEPRI=1;
```

```
ADD APNBINDBWMUSRG:USERGROUPNAME="ug-bwm", APN="testapn";
```

//配置业务感知规则。

```
ADD L7FILTER:L7FILTERNAME="l7-p2p",SUBL7FLTNAME="sl7-p2p",URL="www.example.com*/*";
```

```
ADD FLOWFILTER: FLOWFILTERNAME="flowfilter_p2p"; 
```

```
ADD PROTBINDFLOWF:FLOWFILTERNAME="flowfilter_p2p",PROTOCOLNAME="http",L7FILTERNAME="l7-p2p";
```

```
SET REFRESHSRV:REFRESHTYPE=USERPROFILE;
```

```
ADD RULE: RULENAME="rule_p2pbwm", POLICYTYPE=BWM, FILTERINGMODE=FLOWFILTER, FLOWFILTERNAME="flowfilter_p2p", PRIORITY=1, POLICYNAME="cp_700_600";
```

# 头增强功能之间的差异

*表1 HTTP/HTTPS/RTSP头增强功能之间的差异*

| 差异点 | **HTTP头增强** | **HTTPS头增强** | **RTSP头增强** |
| --- | --- | --- | --- |
| 触发条件 | - 基于特定IP地址触发HTTP头增强<br>- 基于特定URL触发HTTP头增强<br>- 基于HTTP协议触发HTTP头增强 | - 基于特定IP地址触发HTTPS头增强<br>- 基于特定SNI(Server Name Identification)触发HTTPS头增强<br>- 基于HTTPS协议触发HTTPS头增强 | - 基于特定IP地址触发RTSP头增强<br>- 基于特定URL触发RTSP头增强<br>- 基于RTSP协议触发RTSP头增强 |
| 支持协议 | 只支持HTTP1.x协议，不支持HTTP2.0协议。 | 只支持TLS1.0/1.1/1.2协议。 | 只支持RTSP协议，不支持RTSP over HTTP协议。 |
| 交互特性 | 对HTTP报文进行解析时需要开通如下特性：<br>[GWFD-110101 SA-Basic](../../特性指南/UDG特性指南/业务感知功能/GWFD-110101 SA-Basic_24082506.md)<br>[GWFD-110103 SA-Web Browsing](../../特性指南/UDG特性指南/业务感知功能/GWFD-110103 SA-Web Browsing_24082511.md)<br>[GWFD-110105 SA-Mobile](../../特性指南/UDG特性指南/业务感知功能/GWFD-110105 SA-Mobile_24082513.md) | 对HTTPS报文进行解析时需要开通如下特性：<br>[GWFD-110101 SA-Basic](../../特性指南/UDG特性指南/业务感知功能/GWFD-110101 SA-Basic_24082506.md)<br>[GWFD-110103 SA-Web Browsing](../../特性指南/UDG特性指南/业务感知功能/GWFD-110103 SA-Web Browsing_24082511.md)<br>[GWFD-110105 SA-Mobile](../../特性指南/UDG特性指南/业务感知功能/GWFD-110105 SA-Mobile_24082513.md)<br>[GWFD-110201 HTTP2.0 Host识别](../../特性指南/UDG特性指南/业务感知功能/GWFD-110201 HTTP2.0 Host识别_70518421.md) | 对RTSP报文进行解析时需要开通如下特性：<br>[GWFD-110101 SA-Basic](../../特性指南/UDG特性指南/业务感知功能/GWFD-110101 SA-Basic_24082506.md)<br>[GWFD-110107 SA-Streaming](../../特性指南/UDG特性指南/业务感知功能/GWFD-110107 SA-Streaming_24082515.md) |
| 业务 | Web浏览 | Web浏览 | 视频 |
| 业务服务器 | Web Server | Web Server | Streaming Server |
| 加密算法 | 支持MD5、RC4、AES-128、AES-256、RSA-1024、RSA-2048、SHA-256加密算法。MD5和SHA256算法支持盐值加密。 | 支持MD5、RC4、AES-128、AES-256、SHA-256加密算法。MD5和SHA256算法支持盐值加密。 | 支持MD5、RC4、AES-128、AES-256、RSA-1024、RSA-2048、SHA-256加密算法。MD5和SHA256算法支持盐值加密。 |
| 编码能力 | - base64<br>- ASCII | - base64<br>- ASCII | - base64<br>- ASCII |
| 头防欺诈 | 支持 | 支持 | 不支持 |

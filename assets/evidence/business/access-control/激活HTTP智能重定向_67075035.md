## [操作场景](#ZH-CN_OPI_0167075035)

当需要根据返回的错误码进行URL纠错，并将用户访问的HTTP页面重定向到第三方服务器时，需要开启HTTP智能重定向功能。

> **说明**
> 适用于PGW-U、UPF。

前提条件

- 请仔细阅读 [GWFD-110284 HTTP智能重定向](../GWFD-110284 HTTP智能重定向_67075034.md) 。
- UDG 上已经加载了支持业务解析 SA -Basic/ SA -Web Browsing的License。

## [任务示例](#ZH-CN_OPI_0167075035)

任务描述

本例采用SMF下发预定义规则的方式进行说明。

对于用户发起的URL为www.huawei.com/*，并且user agent为Mozilla/* HTTP请求，当HTTP服务器返回的错误码值大于400时， UDG 进行HTTP智能重定向，重定向之后URL为www.example.com，并在重定向动作中携带request-url-name和imsi-name。

脚本

// 打开本特性的License配置开关。

```
SET LICENSESWITCH:LICITEM="LKV3G5SHPR01",SWITCH=ENABLE;
```

//配置HTTP智能重定向使用的三四层过滤条件。

```
ADD FILTER:FILTERNAME="f-any",L34PROTTYPE=STRING,L34PROTOCOL=ANY;
```

```
ADD FLOWFILTER:FLOWFILTERNAME="flow-filter1";
```

```
ADD FLTBINDFLOWF:FLOWFILTERNAME="flow-filter1",FILTERNAME="f_any";
```

```
SET REFRESHSRV:REFRESHTYPE=USERPROFILE;
```

//配置HTTP智能重定向使用的扩展过滤器。

```
ADD EXTENDEDFILTER:EXTFLTNAME="filter_http-redirect1",URL="www.huawei.com/*";
```

```
ADD EXTENDEDFILTER:EXTFLTNAME="filter_http-redirect2",USERAGENT="Mozilla/*";
```

//配置HTTP智能重定向错误码。

```
ADD ERRORCODE: ERRORCODENAME="httperrorcode", ERRORCODEOP=GT, ERRORCODESTART=400;
```

//配置HTTP智能重定向动作策略。


# 套餐3：访问限制场景

- [操作场景](#ZH-CN_TOPIC_0294838086__1.3.1.1)
- [数据](#ZH-CN_TOPIC_0294838086__1.3.3.1)
- [脚本](#ZH-CN_TOPIC_0294838086__1.3.4.1)

#### [操作场景](#ZH-CN_TOPIC_0294838086)

运营商需要布置对用户访问进行管控的业务套餐：

- 业务1——阻塞业务：http://www.example.com非法，用户禁止访问。
- 业务2——头增强业务：当用户访问http://www.vmall.com网站时，为了保证用户交易安全，需要对用户数据插入用户的MSISDN号，并进行防欺诈处理。
- 业务3——重定向业务：用户不使用业务1、2且用户的接入端口号在1000-20000之间时，重定向至10.100.111.222。
- 业务4——重定向业务2：话费余额耗尽场景下，用户访问任何网站都会重定向到充值页面http://www.huawei.com。

#### [数据](#ZH-CN_TOPIC_0294838086)

将该操作场景业务感知信息提炼出来如下：

*表1 访问限制业务套餐信息表*

| 业务编号 | 过滤条件 | 过滤条件 | 策略类型 | 策略 | 业务优先级 |
| --- | --- | --- | --- | --- | --- |
| 业务编号 | 三四层 | 七层 | 策略类型 | 策略 | 业务优先级 |
| 1 | 任意 | http://www.example.com | PCC | 阻塞 | 中 |
| 2 | 任意 | http://www.vmall.com | HEADEN | 插入MSISDN号+防欺诈处理 | 中 |
| 3 | 1000-20000之间的手机端口号 | 任意 | IPREDIR | 重定向至10.100.111.222 | 低 |
| 4 | 任意 | 任意 | PCC | 重定向到http://www.huawei.com | 高 |

将 [表1](#ZH-CN_TOPIC_0294838086__table168581921182911) 内容转化为配置信息，分析如下：

- **三四层过滤条件：**
  业务1、2、4对三四层过滤条件均为“任意”，可以共用三四层过滤器 **filterA** ；业务3的三四层过滤条件为 **1000-20000** 之间的手机端口号，可配置为 **filterC** 。
- **七层过滤条件：**
  业务1、2均涉及到七层过滤网址，过滤器为 **l7filterA、l7filterB** 。业务3、4均不涉及七层过滤条件。
- **流过滤器：**
  流过滤器是三四层过滤条件与七层过滤条件的组合，因此所有业务的流过滤器不相同，可分别配置为 **flowfilterA-flowfilterD** 。
- **策略类型、优先级：**
  业务1、4属于PCC类型策略，由于业务4下发后需要优先于其他业务生效，优先级较高。业务3属于IP重定向，需要在业务1、2匹配失败后才生效，因此优先级较低。业务2是头增强策略，且与其他业务无冲突。综上所述，四个业务的优先级可分别配置为 **500、550、700、400** 。
- **流动作：**
  业务1的流动作是阻塞，可以通过配置PCC动作属性完成。业务2的流动作是插入MSISDN号和防欺诈，可以通过配置头增强完成。业务3的流动作是IP重定向，需要在配置Rule时实现。业务4的流动作是URL重定向，可以通过PCC动作属性完成。

#### [脚本](#ZH-CN_TOPIC_0294838086)

基于以上分析，下面对业务套餐进行配置：

1. 配置三四层过滤条件。
  ```
  ADD FILTER:FILTERNAME="filterA",L34PROTTYPE=STRING,L34PROTOCOL=ANY;
  ADD FILTER:FILTERNAME="filterC",L34PROTTYPE=STRING,L34PROTOCOL=ANY,MSSTARTPORT=1000,MSENDPORT=20000;
  ```
2. 配置七层过滤条件。
  ```
  ADD L7FILTER:L7FILTERNAME="l7filterA",SUBL7FLTNAME="sl7A",URLTYPE=URL,URL="www.huawei.com";
  ADD L7FILTER:L7FILTERNAME="l7filterB",SUBL7FLTNAME="sl7B",URLTYPE=URL,URL="www.vmall.com";
  ```
3. 配置流过滤器（组），并将三四层、七层过滤条件与流过滤器绑定。
  //业务1
  ```
  ADD FLOWFILTER:FLOWFILTERNAME="flowfilterA";
  ADD FLTBINDFLOWF:FLOWFILTERNAME="flowfilterA",FILTERNAME="filterA";
  ADD PROTBINDFLOWF:FLOWFILTERNAME="flowfilterA",PROTOCOLNAME="HTTP",L7FILTERNAME="l7filterA";
  ```
  //业务2
  ```
  ADD FLOWFILTER:FLOWFILTERNAME="flowfilterB";
  ADD FLTBINDFLOWF:FLOWFILTERNAME="flowfilterB",FILTERNAME="filterA";
  ADD PROTBINDFLOWF:FLOWFILTERNAME="flowfilterB",PROTOCOLNAME="HTTP",L7FILTERNAME="l7filterB";
  ```
  //业务3
  ```
  ADD FLOWFILTER:FLOWFILTERNAME="flowfilterC";
  ADD FLTBINDFLOWF:FLOWFILTERNAME="flowfilterC",FILTERNAME="filterC";
  ```
  //业务4
  ```
  ADD FLOWFILTER:FLOWFILTERNAME="flowfilterD";
  ADD FLTBINDFLOWF:FLOWFILTERNAME="flowfilterD",FILTERNAME="filterA";
  ```
4. 配置流动作。
  //业务1
  ```
  ADD PCCACTIONPROP:PCCACTPROPNAME="pccactA",UPINITUPGATE=DISCARD,UPINITDNGATE=DISCARD,DNINITUPGATE=DISCARD,DNINITDNGATE=DISCARD;
  ADD PCCPOLICYGRP: PCCPOLICYGRPNM="pccgrpA",PCCACTPROPNAME="pccactA";
  ```
  //业务2
  ```
  ADD HEADEN:HEADERENNAME="headenB",DATATYPE=MSISDN1,ANTIFRAUD=ENABLE; 
  ```
  //业务4
  ```
  ADD REDIRECT: REDIRECTNAME="redirectD", URL="www.huawei.com";
  ADD PCCACTIONPROP:PCCACTPROPNAME="pccactD",UPINITREDIRNM="redirectD";
  ADD PCCPOLICYGRP: PCCPOLICYGRPNM="pccgrpD",PCCACTPROPNAME="pccactD";
  ```
5. 配置Rule，并绑定过滤条件与流动作。
  ```
  ADD RULE:RULENAME="ruleA",POLICYTYPE=PCC,FILTERINGMODE=FLOWFILTER,FLOWFILTERNAME="flowfilterA",POLICYNAME="pccgrpA",PRIORITY=500;
  ADD RULE:RULENAME="ruleB",POLICYTYPE=HEADEN,FILTERINGMODE=FLOWFILTER,FLOWFILTERNAME="flowfilterB",POLICYNAME="headenB",PRIORITY=550;
  ADD RULE:RULENAME="ruleC",POLICYTYPE=IPREDIR,FILTERINGMODE=FLOWFILTER,FLOWFILTERNAME="flowfilterC",REDIRIPVERFLAG=IPV4,IPREDIRECTIP=10.100.111.222;
  ADD RULE:RULENAME="ruleD",POLICYTYPE=PCC,FILTERINGMODE=FLOWFILTER,FLOWFILTERNAME="flowfilterD",POLICYNAME="pccgrpD",PRIORITY=400;
  ```
6. 配置User Profile，并绑定规则。
  ```
  ADD USERPROFILE:USERPROFILENAME="up_policy";
  ADD RULEBINDING: USERPROFILENAME="up_policy",RULENAME="ruleA",POLICYTYPE=PCC;
  ADD RULEBINDING: USERPROFILENAME="up_policy",RULENAME="ruleB",POLICYTYPE=HEADEN;
  ADD RULEBINDING: USERPROFILENAME="up_policy",RULENAME="ruleC",POLICYTYPE=IPREDIR;
  ADD RULEBINDING: USERPROFILENAME="up_policy",RULENAME="ruleD",POLICYTYPE=PCC;
  SET REFRESHSRV:REFRESHTYPE=USERPROFILE;
  ```

## [操作场景](#ZH-CN_OPI_0000001955040304)

该任务用于指导操作人员完成5G融合计费所需的相关参数。

## [必备事项](#ZH-CN_OPI_0000001955040304)

前提条件

- 具备管理员权限的操作人员已经登录华为操作维护系统（登录MAE客户端或登录OM Portal）。

- UNC软件安装已完成。

## [操作步骤](#ZH-CN_OPI_0000001955040304)

1. 配置默认Failover模板。
  ```
  ADD CCPFOT: FOTNM="FR1", FOENABLE=TRUE, RESPPLCY=TIME-0&SUMVOLUME-1, UVDQ=102400, DVDQ=102400; 
  ```
2. 配置基于CC值的处理模板
  ```
  ADD CCPCCACT:CCTYPE=VALUE, CHGCHAR="0x0400", CHGCHARMASK="0xFFFF", FOTNM="FR1", FWDOCSENABLE=TRUE;
  ADD CCPCCACT:CCTYPE=VALUE, CHGCHAR="0x0800", CHGCHARMASK="0xFFFF", FOTNM="FR1", FWDOCSENABLE= FALSE;
  ```
3. 配置基于异常返回码的处理模板
  ```
  ADD CCPRCACT:RCTYPE=VALUE, CODETYPE=408, FOTNM="FR1", PEERUNAVAILABLE=FALSE;
  ADD CCPRCACT:RCTYPE=VALUE, CODETYPE=429, FOTNM="FR1", PEERUNAVAILABLE=FALSE;
  ADD CCPRCACT:RCTYPE=VALUE, CODETYPE=500, FOTNM="FR1", PEERUNAVAILABLE=FALSE;
  ADD CCPRCACT:RCTYPE=VALUE, CODETYPE=503, FOTNM="FR1", PEERUNAVAILABLE=FALSE;
  ADD CCPRCACT: RCTYPE=DEFAULT, FOTNM="FR1";
  ```
4. 配置AGF功能参数。此处配置create消息生成话单、NCG等待OCS计费响应超时后不重发响应超时的计费请求，根据GPSI选择OCS。
  ```
  SET AGFFUNCPARA: OCSRETRYCNT=0, OCSDISCPLCY=SUPI-0&GPSI-1, CDFIPKGPLCY=TRUE;
  ```
5. CHF(OCS)超时时长配置，默认值为3秒。
  ```
  SET CCPTIMER: TID= CCTIMEOUT, VALUE=3;
  ```
6. 主备CHF(OCS)故障后NCG再次尝试转发CHF(OCS)时长配置，默认值为5分钟。
  ```
  SET CCPTIMER: TID= RECOVERINTERVAL, VALUE=300;
  ```

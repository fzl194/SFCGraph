## [操作场景](#ZH-CN_OPI_0000001990839473)

本指导描述在UNC上配置融合计费模板的相关操作过程。

融合计费模板CCT（Converged Charging Template）中包含本地配置的融合计费参数，如计费的时间/流量门限等。UNC上默认配置一个名为global的模板，即全局通用参数模板CCT-global，用于配置整机通用的融合计费参数。除了CCT-global外，还可以配置其他普通CCT模板，用于配置以下更细粒度的融合计费参数。

- User Profile粒度：根据User Profile配置CCT，将CCT绑定到特定User Profile下。
- DNN粒度：根据DNN配置CCT，将CCT绑定到特定DNN下。
- 计费属性粒度：根据用户的计费属性Charging Characteristics配置CCT，将CCT绑定到特定CC下。

上述几种粒度范围的优先级从高到低依次是： User Profile下的配置 > DNN下的配置 > 计费属性Charging Characteristics下的配置 > 整机通用参数。如果高级别的参数没有配置，则依次向下继承低一级别的参数取值。

一旦确定了整个系统的融合计费参数配置粒度，则原则上所有融合计费参数的配置都必须统一使用该粒度。

## [操作步骤](#ZH-CN_OPI_0000001990839473)

- 配置全局通用CCT模板。
    1. 修改系统默认配置的全局通用参数CCT模板，该模板名称为 “global” 。
      **MOD CCT**
- 配置CC粒度的CCT。
    1. 配置CCT模板。
      **ADD CCT**
    2. 配置计费属性。
      请参见 [配置计费属性](../配置计费属性_55040284.md#ZH-CN_OPI_0000001955040284) 。
    3. 配置CCT模板绑定CC。
      **ADD SELECTCCTBYCC**
- 配置DNN粒度的CCT。
    1. 配置CCT模板。
      **ADD CCT**
    2. 配置CCT模板绑定DNN。
      **SET APNCHARGECTRL**
- 配置User Profile粒度的CCT。
    1. 配置CCT模板。
      **ADD CCT**
    2. 配置User Profile，将CCT绑定到User Profile上。
      **SET USRPROFCHARGE**

## [任务示例](#ZH-CN_OPI_0000001990839473)

任务描述

任务一：修改全局通用CCT模板。

任务二：创建融合计费模板cct_test，漫游用户的PDU流量阈值配置为20MB。模板绑定与计费属性0x0800绑定。

任务三：创建融合计费模板cct_test，漫游用户的PDU流量阈值配置为20MB。模板绑定与APN apn-test绑定。

任务四：创建融合计费模板cct_test，漫游用户的PDU流量阈值配置为20MB。模板绑定与User Profile up_test绑定。

脚本

//任务一：修改全局通用CCT模板。

```
MOD CCT:CCTMPLTNAME="global", QHT=0, VQT=20, TQT=20, UQT=20, VT=30, MAXNUMOFCCC=3, PDUVOLUMELIMIT=500, PDUTIMELIMIT=30, RGVOLUMELIMIT=500, RGTIMELIMIT=30, QFVOLUMELIMIT=500, QFTIMELIMIT=30, FUATERMINATE=TERM_SERVICE, TIMEFORMAT=LocalTime, MAXSVCCONTAINER=100, SECRUTHRESHOLD=100, MAXUSEDSRVNUM=60, QHTRPTTRIGGER=QHT, ROAMFUATERMACT=TERM_SERVICE, ROAMVOLUMELIMIT=0, BLKFREESRV=PASS, NOQUOTATRIGGER=ENABLE;
```

//任务二：创建融合计费模板cct_test，漫游用户的PDU流量阈值配置为20MB。模板绑定与计费属性0x0800绑定。

```
ADD CCT: CCTMPLTNAME="cct_test", QHT=0, VQT=20, TQT=20, UQT=20, VT=30, MAXNUMOFCCC=3, PDUVOLUMELIMIT=500, PDUTIMELIMIT=30, RGVOLUMELIMIT=500, RGTIMELIMIT=30, QFVOLUMELIMIT=500, QFTIMELIMIT=30, FUATERMINATE=TERM_SERVICE, TIMEFORMAT=LocalTime, MAXSVCCONTAINER=100, SECRUTHRESHOLD=100, MAXUSEDSRVNUM=60, QHTRPTTRIGGER=QHT, ROAMFUATERMACT=TERM_SERVICE, ROAMVOLUMELIMIT=20, BLKFREESRV=PASS, NOQUOTATRIGGER=ENABLE;
ADD SELECTCCTBYCC: CCTYPE=VALUE, CCVALUE="0x0800", CCTMPLTNAME="cct_test";
```

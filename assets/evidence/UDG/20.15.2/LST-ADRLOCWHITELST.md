#### [命令功能](#ZH-CN_CONCEPT_0206054825)

**适用NF：PGW-U、UPF**

该命令用于查询位置区域分配地址用户白名单。

#### [注意事项](#ZH-CN_CONCEPT_0206054825)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0206054825)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0206054825)

无。

#### [使用实例](#ZH-CN_CONCEPT_0206054825)

查询位置区域分配地址用户白名单：

```
LST ADRLOCWHITELST:;
```

```

RETCODE = 0  操作成功。

位置区域地址分配白名单
----------------------
MSISDN  =  123456
(结果个数 = 1)
---    END
```

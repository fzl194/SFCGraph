## [命令功能](#ZH-CN_MMLREF_0209587909)

此命令用于查看话务高峰的相关信息。

> **说明**
> 该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0209587909)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209587909)

无

## [使用实例](#ZH-CN_MMLREF_0209587909)

查询当年发生的话务峰值事件信息：

```
%%DSP LICENSEPEAK:;%%
RETCODE = 0  操作成功

已使用的峰值License记录
-----------------------
     使用日期  =  NULL
过载License项  =  NULL
(结果个数 = 1)

剩余次数
--------
 剩余使用次数  =  10
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209587909)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 使用日期 | 该参数用于指定本次话务高峰的起始日期。 |
| 过载License项 | 该参数用于指定本次话务高峰内的过载License项。 |
| 剩余使用次数 | 该参数用于指定本年度剩余可用的高峰次数。 |

#### [命令功能](#ZH-CN_CONCEPT_0000201764675045)

**适用NF：PGW-U、UPF**

该命令用于查看用户面下行模式配置。

#### [注意事项](#ZH-CN_CONCEPT_0000201764675045)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000201764675045)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000201764675045)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000201764675045)

查看当前配置的用户面下行模式配置，可以使用该命令：

```
LST DATAPLANEGIINFMODE:;
```

```

RETCODE = 0  操作成功。

用户面下行模式配置
------------------
模式  =  RU
(结果个数 = 1)
---    END
```

#### [命令功能](#ZH-CN_CONCEPT_0000207394871989)

**适用NF：UPF、PGW-U**

该命令用于查询媒体中继白名单URL列表。

#### [注意事项](#ZH-CN_CONCEPT_0000207394871989)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000207394871989)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000207394871989)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RELAYURLLISTNM | 媒体中继URL列表名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定媒体中继URL列表名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。不区分大小写。<br>默认值：无<br>配置原则：无 |
| RELAYURLNAME | 媒体中继URL名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定媒体中继URL名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~31。不区分大小写。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000207394871989)

查询媒体中继URL白名单列表：

```
LST RELAYWURLLIST: RELAYURLLISTNM="list01";
```

```

RETCODE = 0  操作成功

结果如下
------------------------

	媒体中继URL名称	=  url01
		      URL   =  www.huawei.com
		     优先级  =  1
	      配置域名称	=  NULL
(结果个数 = 1)

---    END
```

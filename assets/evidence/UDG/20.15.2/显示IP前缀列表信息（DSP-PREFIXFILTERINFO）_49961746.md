#### [命令功能](#ZH-CN_CONCEPT_0000001549961746)

该命令用来显示前缀过滤器的基本信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001549961746)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549961746)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549961746)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| NAME | IP前缀列表名字 | 可选必选说明：可选参数<br>参数含义：该参数用来指定IP前缀列表名字。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～169。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549961746)

显示前缀过滤器基本信息：

```
DSP PREFIXFILTERINFO:;
```

```

RETCODE = 0  操作成功

结果如下
-------------------------
IP前缀列表名字  =  a
        允许数  =  0
        拒绝数  =  0
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001549961746)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 允许数 | 用来表示通过匹配的路由数。 |
| 拒绝数 | 用来表示没有通过匹配的路由数。 |

其余输出项请参见ADD PREFIXFILTERNODE的参数说明。

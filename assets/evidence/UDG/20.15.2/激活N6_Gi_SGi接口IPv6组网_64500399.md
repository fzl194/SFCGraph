## [操作步骤](#ZH-CN_OPI_0164500399)

1. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09587387.md)
2. 查询接口名。
  [**LST INTERFACE**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/接口管理/接口配置/查询接口（LST INTERFACE）_49801850.md) :;
3. 配置VPN实例。
  [**ADD L3VPNINST**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/VPN管理/L3VPN管理/L3VPN实例/增加L3VPN实例（ADD L3VPNINST）_49802446.md) ;
  [**ADD VPNINSTAF**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/VPN管理/L3VPN管理/L3VPN实例地址族/增加L3VPN实例地址族（ADD VPNINSTAF）_49802178.md) ;
4. 配置VNRS_VNFC外联口物理IP。
    a. 配置主接口的管理状态为up。
      [**MOD INTERFACE**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/接口管理/接口配置/修改接口（MOD INTERFACE）_50281674.md) ;
    b. 配置以太网子接口。
      [**ADD INTERFACE**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/接口管理/接口配置/增加接口（ADD INTERFACE）_49960870.md) ;
    c. 增加接口绑定VPN。
      [**ADD IPBINDVPN**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/接口管理/IP绑定VPN/增加接口绑定VPN（ADD IPBINDVPN）_50120734.md) ;
    d. 在以太网子接口上关联VLAN ID。
      [**ADD ETHSUBIF**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/VLAN管理/VLAN子接口/增加子接口配置（ADD ETHSUBIF）_49801486.md) ;
      > **说明**
      > - 子接口配置了VLAN，则需要在交换机、对接网关上调整VLAN的配置。
      > - 双平面组网场景下，要求双平面的两个trunk允许通过的Vlan不同，避免环路风险。
    e. 使能IPv6接口。
      [**SET IFIPV6ENABLE**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/接口管理/IPv6使能/修改接口IPv6使能（SET IFIPV6ENABLE）_00601329.md) ;
    f. 配置外联口的物理IP。
      [**ADD IFIPV6ADDRESS**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/接口管理/IPv6地址/增加接口IPv6地址（ADD IFIPV6ADDRESS）_49961222.md) ;
5. 添加IPv6路径MTU配置。
  [**ADD IPV6PATHMTU**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/IP协议栈/IPv6管理/IPv6 Path MTU/添加IPv6 Path MTU配置（ADD IPV6PATHMTU）_50280838.md) ;
6. **可选：**配置静态路由。
  [**ADD SRROUTE6**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/路由管理/静态路由管理/IPv6静态路由/增加IPv6静态路由（ADD SRROUTE6）_50280922.md) ;
  > **说明**
  > 如果需要为静态路由绑定静态BFD会话，通过 [**ADD SRROUTE6**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/路由管理/静态路由管理/IPv4静态路由/增加IPv4静态路由（ADD SRROUTE）_49961498.md) 的“静态BFD会话名”来指定静态路由所联动的静态BFD的会话名。

## [任务示例](#ZH-CN_OPI_0164500399)

任务描述

UDG 使用IPv6地址与PDN/DN互通。

- 开启UDG的IPv6功能。
- 为UDG的接口配置IPv6地址。
- 为UDG配置到PDN/DN的IPv6静态路由。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH:LICITEM="LKV3G5V6NF01",SWITCH=ENABLE;
```

//配置VPN实例。

```
ADD L3VPNINST:VRFNAME="VPN_CN";
ADD VPNINSTAF:VRFNAME="VPN_CN",AFTYPE=ipv6uni;
```

//配置主接口的管理状态为up。

```
MOD INTERFACE:IFNAME="Ethernet70/0/4",IFADMINSTATUS=up; 
MOD INTERFACE:IFNAME="Ethernet70/0/5",IFADMINSTATUS=up; 
MOD INTERFACE:IFNAME="Ethernet71/0/4",IFADMINSTATUS=up; 
MOD INTERFACE:IFNAME="Ethernet71/0/5",IFADMINSTATUS=up;
```

//配置以太网子接口。

```
ADD INTERFACE:IFNAME="Ethernet70/0/4.1"; 
ADD INTERFACE:IFNAME="Ethernet70/0/5.1"; 
ADD INTERFACE:IFNAME="Ethernet71/0/4.1"; 
ADD INTERFACE:IFNAME="Ethernet71/0/5.1";
```

//增加接口绑定VPN。

```
ADD IPBINDVPN:IFNAME="Ethernet70/0/4.1",VRFNAME="VPN_CN"; 
ADD IPBINDVPN:IFNAME="Ethernet70/0/5.1",VRFNAME="VPN_CN"; 
ADD IPBINDVPN:IFNAME="Ethernet71/0/4.1",VRFNAME="VPN_CN"; 
ADD IPBINDVPN:IFNAME="Ethernet71/0/5.1",VRFNAME="VPN_CN";
```

//在以太网子接口上关联VLAN ID。

```
ADD ETHSUBIF:IFNAME="Ethernet70/0/4.1",VLANTYPEVID=100; 
ADD ETHSUBIF:IFNAME="Ethernet70/0/5.1",VLANTYPEVID=200; 
ADD ETHSUBIF:IFNAME="Ethernet71/0/4.1",VLANTYPEVID=100; 
ADD ETHSUBIF:IFNAME="Ethernet71/0/5.1",VLANTYPEVID=200;
```

//使能IPv6接口。

```
SET IFIPV6ENABLE:IFNAME="Ethernet70/0/4.1", ENABLEFLAG=TRUE; 
SET IFIPV6ENABLE:IFNAME="Ethernet70/0/5.1", ENABLEFLAG=TRUE; 
SET IFIPV6ENABLE:IFNAME="Ethernet71/0/4.1", ENABLEFLAG=TRUE; 
SET IFIPV6ENABLE:IFNAME="Ethernet71/0/5.1", ENABLEFLAG=TRUE;
```

//配置VNRS_VNFC外联口物理IP。

```
ADD IFIPV6ADDRESS:IFNAME="Ethernet70/0/4.1",IPV6ADDRESS="fc00:1000::1",PREFIXLEN=64; 
ADD IFIPV6ADDRESS:IFNAME="Ethernet70/0/5.1",IPV6ADDRESS="fc00:2000::1",PREFIXLEN=64; 
ADD IFIPV6ADDRESS:IFNAME="Ethernet71/0/4.1",IPV6ADDRESS="fc00:1000::2",PREFIXLEN=64; 
ADD IFIPV6ADDRESS:IFNAME="Ethernet71/0/5.1",IPV6ADDRESS="fc00:2000::2",PREFIXLEN=64;
```

//配置PMTU。

```
ADD IPV6PATHMTU:VRFNAME="VPN_CN",IPV6ADDRESS="fc00:1000::1",PATHMTU=1300;
ADD IPV6PATHMTU:VRFNAME="VPN_CN",IPV6ADDRESS="fc00:2000::1",PATHMTU=1300;
ADD IPV6PATHMTU:VRFNAME="VPN_CN",IPV6ADDRESS="fc00:1000::2",PATHMTU=1300;
ADD IPV6PATHMTU:VRFNAME="VPN_CN",IPV6ADDRESS="fc00:2000::2",PATHMTU=1300;
```

//配置静态路由数据

```
ADD SRROUTE6:VRFNAME="VPN_CN",AFTYPE=ipv6unicast,PREFIX="fc00:1000::1",MASKLENGTH=64,DESTVRFNAME="VPN_CN",NEXTHOP="fc00:1000::3",IFNAME="Ethernet70/0/4.1"; 
ADD SRROUTE6:VRFNAME="VPN_CN",AFTYPE=ipv6unicast,PREFIX="fc00:2000::1",MASKLENGTH=64,DESTVRFNAME="VPN_CN",NEXTHOP="fc00:2000::3",IFNAME="Ethernet70/0/5.1"; 
ADD SRROUTE6:VRFNAME="VPN_CN",AFTYPE=ipv6unicast,PREFIX="fc00:1000::2",MASKLENGTH=64,DESTVRFNAME="VPN_CN",NEXTHOP="fc00:1000::3",IFNAME="Ethernet71/0/4.1"; 
ADD SRROUTE6:VRFNAME="VPN_CN",AFTYPE=ipv6unicast,PREFIX="fc00:2000::2",MASKLENGTH=64,DESTVRFNAME="VPN_CN",NEXTHOP="fc00:2000::3",IFNAME="Ethernet71/0/5.1"; 
```

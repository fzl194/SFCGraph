#### [命令功能](#ZH-CN_CONCEPT_0000001111153474)

该命令用于显示进程内的文件句柄使用信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001111153474)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001111153474)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001111153474)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RESOURCENAME | 资源名称 | 可选必选说明：可选参数<br>参数含义：用于说明节点名称或容器名称。通过<br>[**DSP RES**](../../../系统管理/资源管理/资源实例管理/显示资源信息（DSP RES）_59036939.md)<br>命令可以查询资源信息。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，区分大小写。<br>默认值：无 |
| PROCID | 进程ID | 可选必选说明：可选参数<br>参数含义：用于说明进程的编号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～4294967295。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001111153474)

显示进程内的文件句柄使用情况：

```
DSP RESPROCFILEHDLSTAT:RESOURCENAME="OMU1";
```


#### [输出结果说明](#ZH-CN_CONCEPT_0000001111153474)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 进程ID | 用于说明进程的编号。 |
| 资源名称 | 用于说明节点名称或容器名称。通过<br>[**DSP RES**](../../../系统管理/资源管理/资源实例管理/显示资源信息（DSP RES）_59036939.md)<br>命令可以查询资源信息。 |
| 文件句柄数目 | 用于说明进程内文件句柄的数目。 |
| 文件句柄最大数目 | 用于说明进程内文件句柄的最大数目。 |

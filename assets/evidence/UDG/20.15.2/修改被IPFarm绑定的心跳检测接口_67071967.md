  [**MOD IPFARM**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/DN管理/IP Farm 管理/IP Farm参数/修改IPFarm（MOD IPFARM）_82837051.md)

## [任务示例](#ZH-CN_OPI_0167071967)

任务描述

farm_test的心跳检测接口改为phif1/0/1。

脚本

//配置IPFarm的心跳检测接口。

```
ADD LOGICINF:NAME="phif1/0/1",IPVERSION=IPV4,IPV4ADDRESS1="10.10.16.16",IPV4MASK1="255.255.255.255";
```

//修改心跳检测接口。

```
RMV RULE:RULENAME="rule_test1";
```

```
MOD IPFARM:IPFARMNAME="farm_test",HEALTHCHECKFLAG=ENABLE,INTERFACENAME="phif1/0/1";
```

```
ADD RULE:RULENAME="rule_test1",POLICYTYPE=WEBPROXY,FILTERINGMODE=FLOWFILTER,FLOWFILTERNAME="flow-l34",PRIORITY=10,POLICYNAME="farm_test";
```

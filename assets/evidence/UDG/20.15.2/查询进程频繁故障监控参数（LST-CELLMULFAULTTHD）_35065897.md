## [命令功能](#ZH-CN_MMLREF_0000001835065897)

该命令用于查询进程频繁故障监控参数值。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000001835065897)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001835065897)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCESSNAME | 进程名称 | 可选必选说明：可选参数<br>参数含义：该参数用于表示微服务的进程类型名称。进程类型名称可以通过<br>[**DSP MSPROCTYPE**](显示微服务进程类型（DSP MSPROCTYPE）_09587905.md)<br>命令查询。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~31。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001835065897)

查询服务进程频繁告警参数配置。

```
%%LST CELLMULFAULTTHD:;%%
RETCODE = 20111  无数据。

没有查到相应结果
---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001835065897)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 进程名称 | 该参数用于表示微服务的进程类型名称。进程类型名称可以通过<br>[**DSP MSPROCTYPE**](显示微服务进程类型（DSP MSPROCTYPE）_09587905.md)<br>命令查询。 |
| 故障告警监控时间 | 该参数表示进程频繁故障监控周期。 |
| 故障告警故障次数 | 该参数表示故障监控周期内发生故障次数。 |
| 恢复告警监控时间 | 该参数表示进程频繁故障恢复监控周期。 |
| 恢复告警监控次数 | 该参数表示恢复监控周期内发生故障次数。 |

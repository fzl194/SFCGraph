#### [命令功能](#ZH-CN_CONCEPT_0000001550280774)

该命令用于查询基于IPv6信息的ACL匹配下一跳的配置结果。

#### [注意事项](#ZH-CN_CONCEPT_0000001550280774)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550280774)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550280774)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| POLICYNAME | 路由策略名字 | 可选必选说明：可选参数<br>参数含义：该参数用于指定配置操作的路由策略名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～200。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001550280774)

查询IPv6下Acl过滤器匹配下一跳的配置信息：

```
LST MATCHNEXTHOPACL6:;
```

```

RETCODE = 0  操作成功

结果如下
-------------------------
  路由策略名字  =  a
路由策略节点号  =  19
ACL名字或ACL号  =  a
(结果个数 = 1)
---    END
```

#### [命令功能](#ZH-CN_TOPIC_0000001883902420)

该命令用于查询当前环境中桶的信息。

#### [注意事项](#ZH-CN_TOPIC_0000001883902420)

无。

#### [参数说明](#ZH-CN_TOPIC_0000001883902420)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| BUCKETNAME | 桶名称 | 可选必选说明：可选参数。<br>参数含义：桶名称。<br>取值范围：字符串类型，输入长度范围为0~100。<br>默认值：无。<br>配置原则：无。 |

#### [使用实例](#ZH-CN_TOPIC_0000001883902420)

1、查询model桶的基本信息。

```
%%DSP DFSBUCKET: BUCKETNAME="model";
%% RETCODE = 0  操作成功  

存储桶信息 
----------       
             应用ID  =  0      
             桶名称  =  model      
       总大小（MB）  =  20480   
   已使用空间（MB）  =  93.693     
     剩余空间（MB）  =  20386.307 
单个文件大小限额(MB) =  100     
           备份数量  =  2 
(结果个数 = 1)  

---    END 
```

2、查询所有桶的基本信息

```
%%DSP DFSBUCKET:;%% 
RETCODE = 0  操作成功
  
存储桶信息 
---------- 
应用ID  桶名称   总大小（MB）  已使用空间（MB）  剩余空间（MB）    单个文件大小限额（MB）  备份数量    
0       dataset  20480         0                 20480             100                      2         
0       model    20480         93.693            20386.307         100                      2          
0       sample   20480         2.0031            20477.9969        100                      2          
(结果个数 = 3)  

---    END 
```

#### [输出结果说明](#ZH-CN_TOPIC_0000001883902420)

命令执行正常，会返回桶的基本信息，输出结果项信息项目如 [表1 输出项说明](#ZH-CN_TOPIC_0000001883902420__table819976173217) 。

命令执行异常，会返回对应的错误码。常见错误码如 [表2 错误码列表](#ZH-CN_TOPIC_0000001883902420__table18477416112710) 所示。

*表1 输出项说明*

| 输出项名称 | 输出项解释 |
| --- | --- |
| 应用ID | 用于返回当前应用ID。 |
| 桶名称 | 用于返回桶名称。 |
| 总大小 | 用于返回数据桶总大小，单位MB。 |
| 已使用空间 | 用于返回数据桶已使用空间大小，单位MB。 |
| 剩余空间 | 用于返回数据桶剩余空间大小，单位MB。 |
| 单个文件大小限额 | 用于返回数据桶支持最大上传单个文件大小，单位MB。 |
| 备份数量 | 用于返回数据桶对象副本数。 |

*表2 错误码列表*

| 错误码 | 错误码解释 | 原因分析 | 处理建议 |
| --- | --- | --- | --- |
| 180001 | 下发的MML参数不正确 | 非法的命令参数 | 请联系华为技术支持。 |
| 180002 | MML处理失败 | 内部错误 | 请联系华为技术支持。 |
| 180003 | 数据桶不存在 | 输入参数的数据桶名称不存在 | 请联系华为技术支持。 |

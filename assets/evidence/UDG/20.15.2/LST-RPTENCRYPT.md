#### [命令功能](#ZH-CN_CONCEPT_0000201106561550)

**适用NF：PGW-U、UPF**

此命令用于查询业务报表加密算法配置。

#### [注意事项](#ZH-CN_CONCEPT_0000201106561550)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000201106561550)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000201106561550)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000201106561550)

假如运营商需要查询业务报表加密算法配置：

```
LST RPTENCRYPT:;
```

```

RETCODE = 0 操作成功

业务报表加密算法配置信息
---------------------------
加密模式 = DEFAULT
加密算法 = SHA256
     
(结果个数 = 1)
--- END
```

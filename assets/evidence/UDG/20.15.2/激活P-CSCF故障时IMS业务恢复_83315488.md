## [操作场景](#ZH-CN_OPI_0183315488)

在 UDG 上激活P-CSCF故障时IMS业务恢复功能，降低P-CSCF服务器故障时对用户业务的影响。

> **说明**
> 适用于PGW-U、UPF。

## [必备事项](#ZH-CN_OPI_0183315488)

前提条件

- 请仔细阅读[GWFD-020253 P-CSCF故障时IMS业务恢复](../GWFD-020253 P-CSCF故障时IMS业务恢复_83315487.md)。
- 完成[配置VoLTE基础语音业务](../GWFD-020251 VoLTE基础语音业务/配置VoLTE基础语音业务_65928706.md)。
- 完成 **加载license** （如果有需求，请联系华为技术支持工程师处理）。

## [操作步骤](#ZH-CN_OPI_0183315488)

1. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09587387.md) ;
2. 配置IP-Farm参数。
    a. **可选：**配置整机的IP Farm心跳检测功能的相关参数。
      [**SET IPFARMGLOBAL**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/DN管理/IP Farm 管理/IP Farm全局参数/设置IPFarm全局参数（SET IPFARMGLOBAL）_82837062.md)
      > **说明**
      > 如果运营商没有特殊要求，建议保持默认值。
    b. 配置VPN。
      [**ADD VPNINST**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/DN管理/VPN管理/VPN/增加VPN实例（ADD VPNINST）_82837045.md)
      > **说明**
      > 如果运营商规划该IP Farm在指定VPN网络中，则需要对该IP Farm绑定相应的VPN。
    c. 配置逻辑接口。
      [**ADD LOGICINF**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/DN管理/逻辑接口管理/逻辑接口配置/接口/增加逻辑接口（ADD LOGICINF）_82835378.md)
    d. 创建IP Farm实例，配置IP Farm对应的虚拟IP地址，使能IP Farm心跳检测功能。
      [**ADD IPFARM**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/DN管理/IP Farm 管理/IP Farm参数/增加IPFarm（ADD IPFARM）_82837050.md)
    e. 在IP Farm中加入P-CSCF服务器地址。
      [**ADD IPFARMSERVER**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/DN管理/IP Farm 管理/IP Farm服务器/增加IPFarmServer（ADD IPFARMSERVER）_82837056.md)
      > **说明**
      > IP Farm配置完成后，可以执行命令 [**LST IPFARMSERVER**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/DN管理/IP Farm 管理/IP Farm服务器/查询IPFarmServer（LST IPFARMSERVER）_82837059.md) 查询IP Farm中server的状态。为确保业务正常进行，需保证至少有一个server的状态为up。

## [任务示例](#ZH-CN_OPI_0183315488)

任务描述

使能APN为“ims”的IMS业务恢复功能。

脚本

1. 打开本特性的License配置开关。
  ```
  SET LICENSESWITCH:LICITEM ="LKV3G5IMSR01",SWITCH=ENABLE;
  ```
2. 配置IP-Farm参数。
  //配置整机的IP Farm心跳检测功能的相关参数。
  ```
  SET IPFARMGLOBAL:SERVERTYPE=PCSCF,HEALTHSUCCLIMIT=4,HEALTHFAILLIMIT=4,TIMETHRESHOLD=6;
  ```
  //配置VPN。
  ```
  ADD VPNINST:VPNINSTANCE="ims_vpn";
  ```
  //配置逻辑接口。
  ```
  ADD LOGICINF:NAME="phif1/0/0",IPVERSION=IPV4,IPV4ADDRESS1="192.168.6.6",VPNINSTANCE="ims_vpn",IPV4MASK1="255.255.255.255";
  ```
  //创建IP Farm实例，配置IP Farm对应的虚拟IP地址,使能IP Farm心跳检测功能。
  ```
  ADD IPFARM:IPFARMNAME="farm_test",IPVERSION=IPV4,SERVERTYPE=PCSCF,VPNINSTANCE="ims_vpn",VIRTUALIP="10.1.1.1",HEALTHCHECKFLAG=ENABLE,INTERFACENAME="phif1/0/0";
  ```
  //在IP Farm中加入P-CSCF服务器地址。
  ```
  ADD IPFARMSERVER:IPFARMNAME="farm_test",IPVERSION=IPV4,SERVERIPV4="192.168.253.251";
  ```
  ```
  ADD IPFARMSERVER:IPFARMNAME="farm_test",IPVERSION=IPV4,SERVERIPV4="192.168.253.252";
  ```
  ```
  ADD IPFARMSERVER:IPFARMNAME="farm_test",IPVERSION=IPV4,SERVERIPV4="192.168.253.253";
  ```
  ```
  ADD IPFARMSERVER:IPFARMNAME="farm_test",IPVERSION=IPV4,SERVERIPV4="192.168.253.254";
  ```
  //创建IP Farm实例，配置IP Farm对应的虚拟IP地址,使能IP Farm心跳检测功能。
  ```
  ADD IPFARM:IPFARMNAME="farm_test2",IPVERSION=IPV4,SERVERTYPE=PCSCF,VPNINSTANCE="ims_vpn",VIRTUALIP="10.2.2.2",HEALTHCHECKFLAG=ENABLE,INTERFACENAME="phif1/0/0";
  ```
  //在IP Farm中加入P-CSCF服务器地址。
  ```
  ADD IPFARMSERVER:IPFARMNAME="farm_test2",IPVERSION=IPV4,SERVERIPV4="192.168.254.251";
  ```
  ```
  ADD IPFARMSERVER:IPFARMNAME="farm_test2",IPVERSION=IPV4,SERVERIPV4="192.168.254.252";
  ```
  ```
  ADD IPFARMSERVER:IPFARMNAME="farm_test2",IPVERSION=IPV4,SERVERIPV4="192.168.254.253";
  ```
  ```
  ADD IPFARMSERVER:IPFARMNAME="farm_test2",IPVERSION=IPV4,SERVERIPV4="192.168.254.254";
  ```

## [操作场景](#ZH-CN_OPI_0000001940507709)

合一ProxyUPF/ProxySGW-U/PGW-U支持镜像SIP信令功能，该功能支持重定向VPN和SIP报文 IPV6大包按照固定MTU分片。重定向VPN是指配置路由到GRE隧道，将SIP报文重定向到GRE隧道并发送到指定服务器。重定向VPN需要单独规划。

## [必备事项](#ZH-CN_OPI_0000001940507709)

前提条件

- 网元运行正常，且处于运行状态。
- 已规划好重定向VPN的底层组网业务配置。

数据

| 类别 | 参数名称 | 取值样例 | 获取方法 | 说明 |
| --- | --- | --- | --- | --- |
| **ADD VPNINST** | VPN实例名(VPNINSTANCE) | SIP_OM | 全网规划 | - |
| **SET SIPSIGMIRRORPARA** | 镜像功能开关（SWITCH） | ENABLE | 固定值 | - |
| **SET SIPSIGMIRRORPARA** | 内层IPv6报文分片MTU（IPV6FRAGMTU） | 1800 | 本端规划 | 控制镜像SIP信令功能发送IPV6报文分片MTU。<br>- 该参数取值为0，表示不开启报文分片。<br>- 该参数取值小于IPV6报文MTU最小值1280时，按最小值1280字节控制分片MTU，大于9600时，按9600字节控制分片MTU。 |
| **SET SIPSIGMIRRORPARA** | 重定向VPN（RDRVPN） | SIP_OM | 本端规划 | - |
| **SET SOFTPARA** | 软参记录数据类型（DT） | BYTE | 本端规划 | - |
| **SET SOFTPARA** | DWord索引（DWORDNUM） | 1116 | 本端规划 | - |
| **SET SOFTPARA** | Dword值（DWORDVALUE） | 72 | 本端规划 | - |

## [操作步骤](#ZH-CN_OPI_0000001940507709)

1. 安装镜像SIP信令功能插件，完成插件安装后再进行配置操作。请联系华为技术支持获取《UDG镜像SIP信令操作指导》。
2. 设置镜像SIP信令功能开启，并设置IPv6报文分片值。
  SET SIPSIGMIRRORPARA
3. 开启插件惯性软参。
  SET SOFTPARA

## [任务示例](#ZH-CN_OPI_0000001940507709)

任务描述

开启镜像SIP信令功能，并设置IPv6报文分片值。

脚本

1. 新增OM网络的重定向VPN，VPN规划参考接口配置。
  ```
  ADD VPNINST:VPNINSTANCE="SIP_OM";
  ```
2. 配置镜像SIP信令的功能参数。设置IPv6报文分片值。
  ```
  SET SIPSIGMIRRORPARA: SWITCH=ENABLE, IPV6FRAGMTU=1800, RDRVPN="SIP_OM";
  ```
3. 配置镜像SIP信令惯性运行时间为6小时（72*5分钟），确保插件进行升级操作时，镜像功能不受影响。
  ```
  SET SOFTPARA: DT=BYTE, BYTENUM=1116, BYTEVALUE=72;
  ```

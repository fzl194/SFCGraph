## [操作场景](#ZH-CN_OPI_0184034134)

> **说明**
> 适用于PGW-U、UPF。

通过部署基于业务累计流量的策略控制功能，实现对特定业务的控制，如用户业务访问量超出限额时降低其访问速率或提高费率，达到网络公平使用的原则，使访问相同业务的用户得到同样的网络体验。

## [任务示例](#ZH-CN_OPI_0184034134)

任务描述

本例采用3GPP标准的Create URR机制实现业务累计上报：对满足过滤条件的业务实现在线流量计费和累计流量上报功能。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH:LICITEM="LKV3G5FPBS01",SWITCH=ENABLE;
```

//配置三四层过滤条件。

```
ADD FILTER:FILTERNAME="filter_red",L34PROTTYPE=STRING,L34PROTOCOL=TCP,SVRSTARTPORT=80,SVRENDPORT=80;
```

```
ADD FLOWFILTER:FLOWFILTERNAME="fg_red";
```

```
ADD FLTBINDFLOWF:FLOWFILTERNAME="fg_red",FILTERNAME="filter_red";
```

```
SET REFRESHSRV:REFRESHTYPE=USERPROFILE;
```

//配置使用量上报规则信息。

```
ADD URR:URRNAME="urr_online", URRID=1000, USAGERPTMODE=ONLINE, DFTQUOTAVOLUME=4294967295, ONLMETERINGTYPE=VOLUME;
```

```
ADD URR:URRNAME="urr_mk", URRID=1002, USAGERPTMODE=MONITORINGKEY;
```

```
ADD URRGROUP:URRGROUPNAME="cp_red",UPURRNAME1="urr_online",DOWNURRNAME1="urr_online",UPURRNAME2="urr_mk", DOWNURRNAME2="urr_mk";
```

//配置PCC策略组。当需要计费时，请配置参数URRGROUPNAME。


# 恢复NSH头增强全局策略（RTR GLBNSHHDNPLY）

- [注意事项](#ZH-CN_CONCEPT_0000203719641390__1.3.2.1)

#### [命令功能](#ZH-CN_CONCEPT_0000203719641390)

**适用NF：PGW-U、UPF**

该命令用于恢复NSH头增强全局策略为系统初始设置值。

#### [注意事项](#ZH-CN_CONCEPT_0000203719641390)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000203719641390)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000203719641390)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000203719641390)

恢复NSH头增强全局策略为系统初始设置值：

```
RTR GLBNSHHDNPLY:;
```

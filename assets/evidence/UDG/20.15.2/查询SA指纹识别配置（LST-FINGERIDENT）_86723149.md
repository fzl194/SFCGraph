#### [命令功能](#ZH-CN_CONCEPT_0000202386723149)

**适用NF：PGW-U、UPF**

该命令用来查询整机配置的SA指纹识别功能。

#### [注意事项](#ZH-CN_CONCEPT_0000202386723149)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000202386723149)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000202386723149)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000202386723149)

查询所有SA指纹识别功能的配置信息：

```
LST FINGERIDENT:;
```

```

RETCODE = 0  操作成功

指纹识别配置信息
----------------
协议名称      指纹识别结果控制开关  指纹识别开关    配置域名称  

facebook      只做统计              使能（开启）    NULL        
youtube       不使能                使能（开启）    NULL        
all-protocol  不使能                不使能（关闭）  NULL        
(结果个数 = 3)

---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000202386723149)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 指纹识别开关 | 用于显示指纹识别开关。 |

其余输出项请参见ADD FINGERIDENT的参数说明。

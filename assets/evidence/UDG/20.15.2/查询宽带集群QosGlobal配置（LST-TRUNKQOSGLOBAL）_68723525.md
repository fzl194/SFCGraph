#### [命令功能](#ZH-CN_CONCEPT_0000202968723525)

**适用NF：SGW-U、PGW-U**

该命令用于查询整机宽带集群Qos功能的使能情况。

#### [注意事项](#ZH-CN_CONCEPT_0000202968723525)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000202968723525)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000202968723525)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000202968723525)

查询整机宽带集群Qos功能的使能情况：

```
LST TRUNKQOSGLOBAL:;
```

```

RETCODE = 0  操作成功

宽带集群全局QoS配置
-------------------
  宽带集群QoS功能开关  =  使能
   宽带集群QoSCar功能  =  使能
宽带集群QoSRemark功能  =  使能
(结果个数 = 1)

---    END
```

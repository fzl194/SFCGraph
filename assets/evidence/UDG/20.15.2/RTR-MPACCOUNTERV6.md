# 清除IPv6 MPAC策略匹配统计计数（RTR MPACCOUNTERV6）

#### [命令功能](#ZH-CN_CONCEPT_0000001549961078)

该命令用于清除当前策略匹配统计计数。

#### [注意事项](#ZH-CN_CONCEPT_0000001549961078)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549961078)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549961078)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| POLICYNAME | 策略名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定策略名字。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。大小写敏感，英文字母开头，不支持空格。<br>默认值：无<br>配置原则：不指定策略名字，默认清除IPv6所有策略匹配统计计数。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549961078)

清除IPv6 MPAC策略匹配统计计数：

```
RTR MPACCOUNTERV6:POLICYNAME="policyV6";
```

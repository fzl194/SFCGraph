#### [命令功能](#ZH-CN_CONCEPT_0182837576)

**适用NF：PGW-U、UPF**

该命令用于指定RSA2048加密使用的公钥文件及相关参数配置。

#### [注意事项](#ZH-CN_CONCEPT_0182837576)

- 该命令执行后立即生效。
- 该命令最大记录数为20。
- 当前命令中 RSAPSWDTYPE 中的 CERTIFICATE_FILE 项功能不支持。

#### [操作用户权限](#ZH-CN_CONCEPT_0182837576)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [使用实例](#ZH-CN_CONCEPT_0182837576)

假如运营商希望配置RSA2048公钥中N值为ABCDEF，E值为101010，开启头增强前缀，国家码为123，网络码为456：

```
ADD RSA2048: RSA2048NAME="test", RSAPSWDTYPE=PUBLIC_KEY, RSAPSWDKEYN="ABCDEF", RSAPSWDKEYE="101010", HEADPREFIXSW=ENABLE, COUNTRYCODE="123", NETWORKCODE="456";
```

#### [命令功能](#ZH-CN_CONCEPT_0000001549961686)

该命令用于查询隧道选择器。

#### [注意事项](#ZH-CN_CONCEPT_0000001549961686)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549961686)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549961686)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001549961686)

查询所有配置的隧道选择器信息：

```
LST TUNNELSELECTOR:;
```

```

RETCODE = 0  操作成功

结果如下
-------------------------
隧道选择器名字  =  a
(结果个数 = 1)
---    END
```

#### [命令功能](#ZH-CN_CONCEPT_0000201254023310)

**适用NF：PGW-U、UPF**

该命令用于显示与指定VXLAN Group绑定的VTEP的信息，包括VTEP的名称。

#### [注意事项](#ZH-CN_CONCEPT_0000201254023310)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000201254023310)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000201254023310)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| VXLANGRPNAME | VXLAN隧道组名称 | 可选必选说明：必选参数<br>参数含义：该参数用于表示VXLAN链路组名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，不区分大小写。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000201254023310)

查询与VXLAN Group“test”绑定的VTEP信息：

```
DSP VTEPSTAT: VXLANGRPNAME="test";
```

```

RETCODE = 0  操作成功。

VTEP状态
-----------
VXLAN隧道端点名称   =  test
地址信息  =  10.0.0.1
服务器状态 =  正常
(结果个数 = 1)

---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000201254023310)

| 输出项名称 | 输出项解释 |
| --- | --- |
| VXLAN隧道端点名称 | 用于表示VXLAN隧道端点名称。 |
| 地址信息 | 用于表示VTEP的地址信息。 |
| 服务器状态 | 用于表示Vtep服务器状态。 |

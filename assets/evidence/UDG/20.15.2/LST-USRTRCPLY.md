#### [命令功能](#ZH-CN_CONCEPT_0000201281234600)

**适用NF：UPF**

该命令用于显示用户跟踪策略。

#### [注意事项](#ZH-CN_CONCEPT_0000201281234600)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000201281234600)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000201281234600)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000201281234600)

查询用户跟踪策略：

```
LST USRTRCPLY:;
```

```

RETCODE = 0 操作成功。

用户跟踪策略
-----------------
最大保留天数  =  7
匿名化开关  =  使能
(结果个数 = 1)
--- END
```

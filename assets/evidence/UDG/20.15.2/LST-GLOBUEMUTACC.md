#### [命令功能](#ZH-CN_CONCEPT_0182837774)

**适用NF：UPF**

该命令用于查询整机终端互访控制功能。

#### [注意事项](#ZH-CN_CONCEPT_0182837774)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0182837774)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0182837774)

无。

#### [使用实例](#ZH-CN_CONCEPT_0182837774)

查看整机粒度APN内终端互访控制功能状态及APN间终端互访控制功能状态：

```
LST GLOBUEMUTACC:;
```

```

RETCODE = 0  操作成功。

全局用户互访禁止开关
--------------------
不同APN间的控制开关  =  使能 
 同APN内的控制开关  =  使能
(结果个数 = 1)

---    END
```

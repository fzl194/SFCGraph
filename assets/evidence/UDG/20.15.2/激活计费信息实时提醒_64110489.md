- [操作场景](#ZH-CN_TOPIC_0164110489__1.3.1.1)
- [必备事项](#ZH-CN_TOPIC_0164110489__1.3.2.1)

#### [操作场景](#ZH-CN_TOPIC_0164110489)

当运营商部署在线计费用户上网欢迎页面提醒、上网过程流量提醒或者欠费充值提醒业务时，需要激活计费信息实时提醒特性。

> **说明**
> 适用于SGW-U、PGW-U、UPF。

#### [必备事项](#ZH-CN_TOPIC_0164110489)

**前提条件**

- 请仔细阅读[GWFD-110314 计费信息实时提醒](GWFD-020356 计费信息实时提醒特性概述_61160472.md)。
- 完成**加载license**（如果有需求，请联系华为技术支持工程师处理）。
- 完成激活在线计费。

#### [操作步骤](#ZH-CN_TOPIC_0164110489)

1. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09587387.md)

2. 配置一次重定向的参数。
    a. 配置一次重定向URL白名单列表，匹配上白名单的URL才会触发重定向。
      [**ADD EXTENDEDFILTER**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/扩展过滤器/增加扩展过滤器（ADD EXTENDEDFILTER）_82837379.md)
    b. 配置一次重定向携带的用户原始请求URL、IMSI、IMEI或MS-IP信息。
      [**ADD REDIRAPPENDINFO**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/重定向控制/重定向公共参数管理/重定向携带信息/增加重定向携带信息（ADD REDIRAPPENDINFO）_82837561.md)
    c. 配置一次重定向的URL白名单列表。
      [**SET GYONESHOT**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/重定向控制/Gy接口一次重定向控制/Gy接口一次重定向/设置Gy一次重定向参数（SET GYONESHOT）_82837567.md)

3. 配置一次重定向中重定向页面的计入的URR。
    a. 配置URR。
      [**ADD URR**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/计费控制/使用率上报规则/增加URR（ADD URR）_82837629.md)
    b. 配置URR组。
      [**ADD URRGROUP**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/计费控制/使用率上报规则组/增加URR组（ADD URRGROUP）_82837634.md)
    c. 配置PCC策略组，绑定上面配置的URR组。
      [**ADD PCCPOLICYGRP**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/PCC控制策略/PCC策略组/增加PCC策略组（ADD PCCPOLICYGRP）_82837606.md) ;
      > **说明**
      > 多次执行 [步骤4.1](#ZH-CN_TOPIC_0164110489__li155121876118) ~ [步骤4.3](#ZH-CN_TOPIC_0164110489__li1951320751113) ，可以分别配置欢迎页面和流量提醒页面的PCC策略组，也可以使用同一PCC策略组，具体根据运营商的需求规划。
    d. 配置三四层过滤条件。

#### [任务示例](#ZH-CN_TOPIC_0164110489)

**任务描述**

本例中，当用户访问www.huawei.com URL时触发欢迎页面提醒和流量页面提醒，用户账户余额耗尽，访问www.huawei.com页面触发欠费充值提醒。

为欢迎提醒和流量提醒的重定向页面规划相同URR ID。

**脚本**

//打开本特性的License配置开关。

```
SET LICENSESWITCH:LICITEM="LKV3G5RCIN01",SWITCH=ENABLE;
```

//配置一次重定向的参数。

```
ADD EXTENDEDFILTER:EXTFLTNAME="gy-oneshot-url",URL="www.huawei.com";
ADD REDIRAPPENDINFO:APPENDINFONAME="testredirappendinfo",REQURLFLAG=ENABLE,MSISDNFLAG=DISABLE,IMSIFLAG=ENABLE,IMSIENCRYALGORI=AES256,IMEIFLAG=ENABLE,MSIPFLAG=ENABLE;
SET GYONESHOT:EXTFLTNAME="gy-oneshot-url",APPENDINFONAME="testredirappendinfo";
```

//配置一次重定向中重定向页面的URR ID。

//配置计费PCC策略组。

```
ADD URR:URRNAME="urr_red",URRID=1000,USAGERPTMODE=OFFLINE;
ADD URRGROUP:URRGROUPNAME="cp_red",UPURRNAME1="urr_red",DOWNURRNAME1="urr_red";
ADD PCCPOLICYGRP:PCCPOLICYGRPNM="cg_red",URRGROUPNAME="cp_red";
```

//配置三四层和七层信息。

```
ADD FILTER:FILTERNAME="filter_red",L34PROTTYPE=STRING,L34PROTOCOL=TCP,SVRSTARTPORT=80,SVRENDPORT=80;
ADD L7FILTER:L7FILTERNAME="l7-red",SUBL7FLTNAME="sl7-red",URL="10.13.4.3/*";
ADD FLOWFILTER:FLOWFILTERNAME="fg_red";
ADD FLTBINDFLOWF:FLOWFILTERNAME="fg_red",FILTERNAME="filter_red";
ADD PROTBINDFLOWF:FLOWFILTERNAME="fg_red",PROTOCOLNAME="http",L7FILTERNAME="l7-red";
```

//配置重定向页面的内容计费规则。

```
ADD RULE:RULENAME="rule_red",POLICYTYPE=PCC,FILTERINGMODE=FLOWFILTER, FLOWFILTERNAME="fg_red",POLICYNAME="cg_red";
SET REFRESHSRV:REFRESHTYPE=USERPROFILE;
```

//配置计费信息实时提醒使用的业务策略组合。

```
ADD USERPROFILE:USERPROFILENAME="up_test";
ADD RULEBINDING:USERPROFILENAME="up_test",RULENAME="rule_red";
```

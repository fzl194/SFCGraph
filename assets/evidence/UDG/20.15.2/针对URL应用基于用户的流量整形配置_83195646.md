## [操作场景](#ZH-CN_OPI_0183195646)

用户访问特定URL的业务时，需要公平分享带宽资源，对流量进行整形。

> **说明**
> 适用于PGW-U、UPF。

## [必备事项](#ZH-CN_OPI_0183195646)

前提条件

- UDG与网络实体之间的网络环境已经构建完成。
- 本特性License项依赖于其他特性，需开启依赖特性后才能开启本特性。详细仔细阅读[GWFD-110311 基于业务感知的带宽控制](../../GWFD-110311 基于业务感知的带宽控制_77219468.md)。
- 已加载特征库和解析库。

数据

使用三四层过滤条件（源IP、源Port、目的IP、目的Port、四层协议类型）触发流量整形时，需要规划 [表1](#ZH-CN_OPI_0183195646__table182854220145) 中的数据，不使用三四层过滤条件触发流量整形时，不需要规划 [表1](#ZH-CN_OPI_0183195646__table182854220145) 中的数据。

## [操作步骤](#ZH-CN_OPI_0183195646)

1. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09587387.md) :LICITEM="LKV3G5SBTS01",SWITCH=ENABLE;
2. 配置流过滤器。
  [**ADD FLOWFILTER**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/流过滤器/增加流过滤器（ADD FLOWFILTER）_82837361.md)
3. 使用源IP/Port、目的IP/Port、四层协议类型作为过滤条件时，配置三四层过滤规则。
    a. 配置三四层过滤条件。
      [**ADD FILTER**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/过滤器/增加过滤器（ADD FILTER）_82837348.md)
    b. 配置三四层过滤条件与流过滤器的绑定关系。
      [**ADD FLTBINDFLOWF**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/流过滤器的过滤器绑定/增加流过滤器的过滤器绑定关系（ADD FLTBINDFLOWF）_82837366.md)
    c. 将新配置的Filter置为生效。
      [**SET REFRESHSRV**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/业务刷新/业务刷新（SET REFRESHSRV）_82837355.md)
4. 使用URL作为过滤条件时，配置七层过滤条件。
    a. 配置七层过滤器。
      [**ADD L7FILTER**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/七层规则管理/七层过滤器/增加七层过滤器（ADD L7FILTER）_82837397.md)
    b. 绑定七层过滤信息。
      [**ADD PROTBINDFLOWF**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/流过滤器的协议绑定/增加流过滤器协议绑定关系（ADD PROTBINDFLOWF）_82837370.md)
    c. 将新配置的L7Filter置为生效。
      [**SET REFRESHSRV**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/业务刷新/业务刷新（SET REFRESHSRV）_82837355.md)
      > **说明**
      > 系统自动在60s后将配置的L7Filter置为生效。如果需要将新配置的L7Filter立即生效，可执行本步骤。
5. 配置规则。
  **[ADD RULE](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务匹配公共配置/业务规则管理/规则/增加规则（ADD RULE）_82837266.md)**
  若需配置多个规则，则需要多次执行 [步骤 5](#ZH-CN_OPI_0183195646__step3630133623814) 。
6. 配置BWM使用的UserProfile（规则与用户模板的绑定关系） 。
    a. 创建UserProfile。
      [**ADD USERPROFILE**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务匹配公共配置/业务规则管理/用户模板/增加用户模板（ADD USERPROFILE）_82837279.md)

      [**ADD RULEBINDING**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务匹配公共配置/业务规则管理/规则绑定/增加用户模板和规则的绑定关系（ADD RULEBINDING）_82837272.md)
7. 基于分类属性（例如URL）时，配置分类属性。
  [**ADD CATEGORYPROP**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/分类属性/增加分类属性（ADD CATEGORYPROP）_82837504.md)

    a. 增加带宽管理用户组。
      [**ADD BWMUSERGROUP**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理用户组/增加带宽管理用户组（ADD BWMUSERGROUP）_82837468.md)
    b. 配置带宽管理的业务范围。
      [**ADD BWMSERVICE**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理业务/增加带宽管理业务（ADD BWMSERVICE）_82837473.md)
    c. 绑定带宽管理用户组BwmUserGroup到指定APN下。
      [**ADD APNBINDBWMUSRG**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理用户组APN绑定/增加带宽管理用户组APN绑定（ADD APNBINDBWMUSRG）_82837487.md)
    d. 配置BWM控制器。

10. 配置带宽管理规则。
  [**ADD BWMRULE**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理规则/增加带宽管理规则（ADD BWMRULE）_82837478.md)


//配置三四层过滤规则。仅采用七层过滤条件时请跳过该脚本。

```
ADD FILTER: FILTERNAME="filter_test1", L34PROTTYPE=STRING, L34PROTOCOL=TCP, MSIPMODE=IP, MSIP="0.0.0.0", MSIPMASKTYPE=IPTYPE, MSIPMASK="255.255.255.255", SVRIPMODE=IP, SVRIP="10.10.10.20", SVRIPMASKTYPE=IPTYPE, SVRIPMASK="0.0.0.0", MSSTARTPORT=0, MSENDPORT=65535, SVRSTARTPORT=80, SVRENDPORT=80;
ADD FILTER: FILTERNAME="filter_test2", L34PROTTYPE=STRING, L34PROTOCOL=TCP, MSIPMODE=IP, MSIP="0.0.0.0", MSIPMASKTYPE=IPTYPE, MSIPMASK="255.255.255.255", SVRIPMODE=IP, SVRIP="10.10.10.20", SVRIPMASKTYPE=IPTYPE, SVRIPMASK="0.0.0.0", MSSTARTPORT=0, MSENDPORT=65535, SVRSTARTPORT=8080, SVRENDPORT=8080;
ADD FLOWFILTER:FLOWFILTERNAME="flowfilter_test";
ADD FLTBINDFLOWF:FLOWFILTERNAME="flowfilter_test",FILTERNAME="filter_test1";
ADD FLTBINDFLOWF:FLOWFILTERNAME="flowfilter_test",FILTERNAME="filter_test2";
SET REFRESHSRV:REFRESHTYPE=ALL;
```

//配置七层过滤条件（URL）。如已配置四层过滤规则，则不需要重复通过ADD FLOWFILTER命令配置流过滤器。

```
ADD L7FILTER:L7FILTERNAME="l7filter_test",SUBL7FLTNAME="sl7_test1",URL="www.huawei.com/*";
ADD L7FILTER:L7FILTERNAME="l7filter_test",SUBL7FLTNAME="sl7_test2",URL="www.example.com";
ADD FLOWFILTER:FLOWFILTERNAME="flowfilter_test";
ADD PROTBINDFLOWF:FLOWFILTERNAME="flowfilter_test",PROTOCOLNAME="http",L7FILTERNAME="l7filter_test";
SET REFRESHSRV:REFRESHTYPE=ALL;
```

//配置业务分类属性。

```
ADD CATEGORYPROP:CATEPROPNAME="cateprop_test";
```

```
ADD RULE:RULENAME="rule_test2",POLICYTYPE=BWM,
FILTERINGMODE=FLOWFILTER,
FLOWFILTERNAME="flowfilter_test",PRIORITY=20,POLICYNAME="cateprop_test";
```

//配置BWM使用的UserProfile。

```
ADD USERPROFILE:USERPROFILENAME="up_test";
ADD RULEBINDING:USERPROFILENAME="up_test",RULENAME="rule_test2";
```

//配置带宽管理的业务范围。

```
ADD BWMSERVICE:BWMSERVICENAME="bwmservice_test",BWMSERVICETYPE=NONTOS,NONTOSSRVTYPE=CATEGORY_PROP,CATEPROPNAME="cateprop_test";
```

```
ADD BWMUSERGROUP:USERGROUPTYPE=SPECIFIC_GROUP,USERGROUPNAME="usergroup_test",USERGROUPPRI=2,USERLEVSRVTYPE=NONTOS;
```

```
ADD APNBINDBWMUSRG:USERGROUPNAME="usergroup_test",APN="huawei.com";
```

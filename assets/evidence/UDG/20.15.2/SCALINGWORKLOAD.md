## [命令功能](#ZH-CN_MMLREF_0209587919)

此命令用于显示当前系统中每个资源对象的负载情况，作为自动扩缩容的参考依据。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0209587919)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209587919)

无

## [使用实例](#ZH-CN_MMLREF_0209587919)

DSP SCALINGWORKLOAD:;

```
%%DSP SCALINGWORKLOAD:;%%
RETCODE = 0  操作成功
结果如下
--------
资源对象  KPI类型   负载
PBU_P-L   workload  25    
OMU_L     workload  29    
SDU       workload  7 
(结果个数 = 3)
---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209587919)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 资源对象 | 该参数用于表示资源对象，虚机场景下是虚机类型，裸机场景下是common-pod类型。 |
| KPI类型 | 该参数用于表示KPI类型，当前定义为负载(workload)。 |
| 负载 | 该参数用于表示负载，这里指kpi的使用率占比，单位是百分比。 |

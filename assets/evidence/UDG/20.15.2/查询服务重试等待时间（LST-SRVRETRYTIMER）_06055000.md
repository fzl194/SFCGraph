#### [命令功能](#ZH-CN_CONCEPT_0206055000)

**适用NF：PGW-U、UPF**

此命令用于显示服务上报出现上报失败/上报响应超时/响应失败的异常场景后重新发起上报需要等待的时间。

#### [注意事项](#ZH-CN_CONCEPT_0206055000)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0206055000)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0206055000)

无。

#### [使用实例](#ZH-CN_CONCEPT_0206055000)

查询服务重新上报需要等待的时间：

```
LST SRVRETRYTIMER:;
```

```

RETCODE = 0 操作成功。
服务重试定时器信息
------------------
      ADC重新上报等待时间（秒） = 40
      QOS重新上报等待时间（秒） = 50
Tethering重新上报等待时间（秒） = 60
(结果个数 = 1)
--- END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0206055000)

参见SET SRVRETRYTIMER的参数说明。

# 查询PBUF轨迹开关状态和持续时间（LST MSSPBUFTRACELT）

## [命令功能](#ZH-CN_MMLREF_0000001761621989)

该命令用于查询PBUF轨迹开关状态和持续时间。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000001761621989)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001761621989)

无

## [使用实例](#ZH-CN_MMLREF_0000001761621989)

查询当前PBUF轨迹开关状态和持续时间:

```
%%LST MSSPBUFTRACELT:;%%
RETCODE = 0  操作成功

结果如下
--------
            PBUF轨迹开关  =  开
轨迹开关持续时间（分钟）  =  1440
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001761621989)

| 输出项名称 | 输出项解释 |
| --- | --- |
| PBUF轨迹开关 | 该参数用于表示轨迹开关标记。 |
| 轨迹开关持续时间（分钟） | 该参数用于表示本次规则设置的持续时间，单位:分钟。 |

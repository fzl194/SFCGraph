## [命令功能](#ZH-CN_MMLREF_0000001180751066)

该命令用于查询IPsec策略。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000001180751066)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001180751066)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| POLICYNAME | 策略名称 | 可选必选说明：可选参数<br>参数含义：策略名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~15。不区分大小写。<br>默认值：无<br>配置原则：无 |
| SEQUENCENUMBER | 序列号 | 可选必选说明：可选参数<br>参数含义：序列号。<br>数据来源：对端协商<br>取值范围：整数类型，取值范围是1~10000。<br>默认值：无<br>配置原则：无 |

## [输出结果说明](#ZH-CN_MMLREF_0000001180751066)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 策略名称 | 策略名。 |
| 序列号 | 序列号。 |
| 策略模式 | 该参数用于指定策略模式。 |
| 模板模式 | 模板模式-绑定一个策略模板到ISAKMP（使用策略模板建立IPsec安全联盟）。 |
| ACL编号 | 通过策略指定需要保护的报文，指定ACL的编号，只能是高级ACL。 |
| ACL名称 | 通过策略指定需要保护的报文，指定ACL的名字（名字是由1-32个字符组成的字符串，需要用a-z或A-Z开头）。 |
| 去使能SA按流量计长 | 流量方式触发SA过期。 |
| SA按流量计长 (mbyte) | 基于流量的SA生命周期。 |
| SA按时计长 (s) | SA的生命周期。 |
| PFS | 在IKE协商的第二阶段使用前向安全保护（PFS）。 |
| 选择DSCP入方向值 | 输入入方向的DSCP值。 |
| 输入DSCP入方向值 | 输入入方向的DSCP值。 |
| 选择DSCP出方向值 | 输入出方向的DSCP值。 |
| 输入DSCP出方向值 | 输入出方向的DSCP值。 |
| 抗重放 | 抗重放。 |
| 抗重放窗口大小 | 抗重放窗口的大小，超过这个流量的报文会被丢弃。 |
| 清除分片标记 | 清除分片标记。 |
| 加密前分片报文 | 加密前分片报文。 |
| 入方向限速 (kbyte/s) | 入方向流量限速。 |
| 出方向限速 (kbyte/s) | 出方向流量限速。 |
| 日志使能 | 日志使能。 |
| 策略组标识 | 策略组标识。 |
| 工作模式 | 工作模式，如轮循模式、主备模式。 |
| 自动切回开关 | 若开启自动切回开关，主恢复后，流量重新切回主隧道；不开启自动切回开关，主恢复后，流量仍在备隧道。 |
| 扩展序列号 | 该参数用于表示扩展序列号是否使能。 |
| 数据流可信使能 | 数据流可信使能。 |

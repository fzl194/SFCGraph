## [命令功能](#ZH-CN_MMLREF_0209587931)

该命令用于显示自愈功能配置信息。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0209587931)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209587931)

无

## [使用实例](#ZH-CN_MMLREF_0209587931)

显示自愈功能配置信息。

```
%%LST PODHEALCTRL:;%%
RETCODE = 0  操作成功

结果如下
--------
自愈开关  =  不使能
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209587931)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 自愈开关 | 该参数用于表示自愈开关。 |

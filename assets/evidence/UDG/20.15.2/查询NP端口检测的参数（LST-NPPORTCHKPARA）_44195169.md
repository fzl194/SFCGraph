#### [命令功能](#ZH-CN_TOPIC_0000002144195169)

该命令用来查询NP端口检测的参数。

#### [注意事项](#ZH-CN_TOPIC_0000002144195169)

- 该命令执行后立即生效。
- 该命令仅适用于NP卡加速模式场景。

#### [操作用户权限](#ZH-CN_TOPIC_0000002144195169)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_TOPIC_0000002144195169)

无

#### [使用实例](#ZH-CN_TOPIC_0000002144195169)

查询NP端口检测的参数：

```
LST NPPORTCHKPARA:;
RETCODE = 0  操作成功

结果如下
--------
内联口带宽利用率告警上报阈值（%）  =  95
内联口带宽利用率告警恢复阈值（%）  =  90
外联口带宽利用率告警上报阈值（%）  =  90
外联口带宽利用率告警恢复阈值（%）  =  80
             NP间端口故障自愈开关  =  使能
       NP间端口故障时长阈值（秒）  =  3
                 TB出端口检测开关  =  使能
   TB出端口无效的时长阈值（毫秒）  =  2000
       检测TB出端口刷新的周期倍数  =  15
(结果个数 = 1)

---    END
```

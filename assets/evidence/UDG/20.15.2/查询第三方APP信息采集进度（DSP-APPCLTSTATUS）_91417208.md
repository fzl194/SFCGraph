#### [命令功能](#ZH-CN_CONCEPT_0000206691417208)

该命令用于查询第三方APP信息采集进度。

#### [注意事项](#ZH-CN_CONCEPT_0000206691417208)

- 该命令执行后立即生效。
- 该命令当前仅支持第三方APP。

#### [操作用户权限](#ZH-CN_CONCEPT_0000206691417208)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000206691417208)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000206691417208)

查询第三方应用信息采集的进度：

```
DSP APPCLTSTATUS:;
```

```

RETCODE = 0  操作成功

执行百分比
----------
  执行百分比  =  100
执行阶段信息  =  NULL
执行详细信息  =  No found the collect information task.
(结果个数 = 1)

---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000206691417208)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 执行百分比 | 用于返回执行采集进度百分比。 |
| 执行阶段信息 | 用于返回执行采集阶段。 |
| 执行详细信息 | 用户返回执行采集详细信息。 |

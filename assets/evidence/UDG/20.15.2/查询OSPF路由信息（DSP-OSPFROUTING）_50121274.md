#### [命令功能](#ZH-CN_CONCEPT_0000001550121274)

该命令用于显示OSPF路由信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001550121274)

此命令最多支持显示1000条路由信息，如果路由信息比较多，建议指定参数进行查询。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550121274)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550121274)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCESSID | 进程号 | 可选必选说明：可选参数<br>参数含义：OSPF进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |
| DESTIPADDR | 目的IP地址 | 可选必选说明：可选参数<br>参数含义：目的IP地址。<br>数据来源：全网规划<br>取值范围：IPv4地址类型。点分十进制格式。<br>默认值：无 |
| NEXTHOPIPADDR | 下一跳IP地址 | 可选必选说明：可选参数<br>参数含义：下一跳IP地址。<br>数据来源：全网规划<br>取值范围：IPv4地址类型。点分十进制格式。<br>默认值：无 |
| ADVROUTERID | 宣告路由器标识 | 可选必选说明：可选参数<br>参数含义：宣告路由器标识。<br>数据来源：全网规划<br>取值范围：IPv4地址类型。点分十进制格式。<br>默认值：无 |
| IFNAME | 接口名 | 可选必选说明：可选参数<br>参数含义：接口名称。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围为1～63。<br>默认值：无<br>配置原则：请使用LST INTERFACE命令查看可用接口。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001550121274)

显示设备OSPF进程号为1的所有路由信息：

```
DSP OSPFROUTING: PROCESSID=1;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001550121274)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 进程号 | 该参数指定用户创建的OSPF进程号。 |
| 目的IP地址 | 该参数指定目的路由IP地址。 |
| 掩码长度 | 该参数指定目的路由掩码长度。 |
| 路径Cost值 | 该参数指定目的路由路径Cost值。 |
| 下一跳IP地址 | 该参数指定目的路由下一跳IP地址。 |
| 宣告路由标识 | 该参数指定目的路由宣告路由器的标识。 |
| 区域标识 | 该参数指定用户创建的OSPF区域号。 |
| 路由类型 | 该参数指定目的路由的类型。 |
| 接口名 | 该参数指定目的路由的出接口名称。 |
| 标签 | 该参数指定目的路由标签值。 |
| 优先级 | 该参数指定目的路由的优先级。 |
| 下一跳数目 | 该参数指定目的路由的下一跳数目。 |
| 下一跳标志 | 该参数指定目的路由的下一跳标志。 |
| 备份下一跳 | 该参数指定目的路由的备份下一跳IP地址。 |
| 备份接口 | 该参数指定目的路由的备份出接口名称。 |
| 备份类型 | 该参数指定目的路由备份路径类型。 |

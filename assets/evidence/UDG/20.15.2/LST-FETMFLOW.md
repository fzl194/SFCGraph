#### [命令功能](#ZH-CN_TOPIC_0000001259131112)

该命令用来查询FETM流控配置：包括流控开关，流控阈值。

#### [注意事项](#ZH-CN_TOPIC_0000001259131112)

- 该命令执行后立即生效。
- 该命令仅适用于NP卡加速CSLB卸载模式场景。

#### [操作用户权限](#ZH-CN_TOPIC_0000001259131112)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_TOPIC_0000001259131112)

无

#### [使用实例](#ZH-CN_TOPIC_0000001259131112)

查询FETM流控配置：

```
LST FETMFLOW:;
RETCODE = 0  操作成功

结果如下
--------
流控开关  =  启用
流控阈值  =  70
(结果个数 = 1)

---    END
```

#### [输出结果说明](#ZH-CN_TOPIC_0000001259131112)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 流控开关 | FETM流控开关。 |
| 流控阈值 | FETM流控阈值，1个单位代表1000次下表操作。 |

#### [命令功能](#ZH-CN_CONCEPT_0235373538)

**适用NF：PGW-U、UPF**

该命令用于查询L2TP组和源端Gi接口之间的绑定关系。用户查看所有L2TP组和源端接口绑定关系场景，可以不指定参数，查询结果将按照L2TP组顺序显示全部显示；用户查看指定L2TP组绑定的源端接口场景，可以指定L2TP组号；用户查看与指定源端接口绑定的L2TP组场景，可以指定源端接口名称。

#### [注意事项](#ZH-CN_CONCEPT_0235373538)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0235373538)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0235373538)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| LISTTPYE | 查询方式 | 可选必选说明：可选参数<br>参数含义：指定查询类型。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- GROUP：表示根据L2TP组号查询绑定的源端接口。<br>- INTERFACE：表示根据源端Gi接口查询L2TP组号。<br>默认值：无<br>配置原则：当该参数不配置时，会查询所有的L2TP组和源端接口之间的绑定关系。 |
| L2TPGROUPID | L2TP组号 | 可选必选说明：条件必选参数<br>前提条件：该参数在“LISTTPYE”配置为“GROUP”时为必选参数。<br>参数含义：指定L2TP组号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～1500。<br>默认值：无<br>配置原则：无 |
| INTERFACENAME | 接口名称 | 可选必选说明：条件必选参数<br>前提条件：该参数在“LISTTPYE”配置为“INTERFACE”时为必选参数。<br>参数含义：指定源端Gi接口名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0235373538)

- 查询所有L2TP组和源端接口之间的绑定关系记录：
  ```
  LST L2TPCLIENTIP:;
  ```
  ```

  RETCODE = 0  操作成功。

  L2TP绑定的接口信息
  ------------------
  L2TP组号  =  1
  接口名称  =  giif1/0/0
  (结果个数 = 1)
  ---    END
  ```
- 查询L2TP组1绑定的源端接口：
  ```
  LST L2TPCLIENTIP: LISTTPYE=GROUP,L2TPGROUPID=1;
  ```
  ```

  RETCODE = 0  操作成功。

  L2TP绑定的接口信息
  ------------------
  L2TP组号  =  1
  接口名称  =  giif1/0/0
  (结果个数 = 1)
  ---    END
  ```

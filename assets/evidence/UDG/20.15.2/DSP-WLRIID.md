#### [命令功能](#ZH-CN_CONCEPT_0000001600840505)

该命令用于显示无线路由IID信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600840505)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600840505)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600840505)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| AFTYPE | 地址族 | 可选必选说明：必选参数<br>参数含义：该参数用于表示地址族。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- ipv4unicast：IPv4单播。<br>- ipv6unicast：IPv6单播。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600840505)

显示无线路由IID信息：

```
DSP WLRIID:AFTYPE=ipv4unicast;
```

| 标识 | 用于表示无线路由标识。 |
| 组ID | 用于表示组ID。 |
| 下一跳VPN名字 | 用于表示下一跳所属VPN名字。 |

| TB high | 用于表示TB high。 |

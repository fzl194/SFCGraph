#### [命令功能](#ZH-CN_CONCEPT_0182837073)

**适用NF：SGW-U、PGW-U、UPF**

此命令用于显示当前被收集运行信息的所有用户，输出当前被收集运行信息的用户的IMSI/MSISDN。

#### [注意事项](#ZH-CN_CONCEPT_0182837073)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0182837073)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0182837073)

无。

#### [使用实例](#ZH-CN_CONCEPT_0182837073)

- 系统中只有一条记录时，显示当前系统中进行信息记录的用户：
  ```
  LST USERRUNINFO:;
  ```
  ```

  RETCODE = 0  操作成功。

  结果如下
  --------
  用户类型  =  IMSI
      IMSI  =  460011223344551
    MSISDN  =  NULL
  (结果个数 = 1)
  ---    END
  ```
- 系统中有多条记录时，显示当前系统中进行信息记录的用户：
  ```
  LST USERRUNINFO:;
  ```
  ```

  RETCODE = 0  操作成功。

  结果如下
  --------
  用户类型    IMSI               MSISDN    

  IMSI        460011223344551    NULL      
  MSISDN      NULL               1223344551
  (结果个数 = 2)
  ---    END
  ```

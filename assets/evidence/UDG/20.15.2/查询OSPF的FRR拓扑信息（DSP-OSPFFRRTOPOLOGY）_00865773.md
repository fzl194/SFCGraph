#### [命令功能](#ZH-CN_CONCEPT_0000001600865773)

该命令用于查询OSPF的FRR拓扑信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600865773)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600865773)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600865773)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | OSPF进程号 | 可选必选说明：可选参数<br>参数含义：OSPF进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |
| AREAID | 区域号 | 可选必选说明：可选参数<br>参数含义：OSPF区域号。<br>数据来源：全网规划<br>取值范围：IPv4地址类型。点分十进制格式。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600865773)

查询OSPF进程1的FRR拓扑信息：

```
DSP OSPFFRRTOPOLOGY:PROCID=1;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600865773)

| 输出项名称 | 输出项解释 |
| --- | --- |
| OSPF进程号 | 该参数用于表示OSPF进程号。 |
| 区域号 | 该参数用于表示OSPF区域号。 |
| 路由器标识 | 该参数用于表示OSPF路由器标识。 |
| SPF类型 | 该参数用于表示SPF类型-Router节点、Network节点。 |
| SPF节点标识 | 该参数用于表示SPF节点标识，当SPF类型为Router时，取得是生成LSA的Router ID；当SPF类型为Network时，取得是DR的接口IP地址。 |
| 节点类型 | 该参数用于表示该节点的路由能力（比如ASR、ASBR、NSSA等）。 |
| 开销值 | 该参数用于表示链路的开销值。 |
| 下一跳 | 该参数用于表示主路径下一跳。 |
| 出接口 | 该参数用于表示主路径出接口。 |
| 备份下一跳 | 该参数用于表示备份路径下一跳。 |
| 备份出接口 | 该参数用于表示备份路径出接口。 |
| LFA开销值 | 该参数用于表示LFA链路的开销值。 |
| 备份类型 | 该参数用于表示LFA FRR流量保护类型：链路保护、节点链路双保护。 |

# 激活静态地址用户路由冗余

- [必备事项](#ZH-CN_OPI_0319612403__1.3.2)
- [操作流程](#ZH-CN_OPI_0319612403__1.3.3)

## [操作场景](#ZH-CN_OPI_0319612403)

网络同时部署主用备用两台 UDG 。当主用 UDG 故障或者主用 UDG 通信链路故障的时候通过备份tunnel，将静态地址用户的下行数据备份到备用 UDG 上。

> **说明**
> 适用于PGW-U、UPF。

## [必备事项](#ZH-CN_OPI_0319612403)

前提条件

- 请仔细阅读[GWFD-010107 静态地址用户路由冗余](../GWFD-010107 静态地址用户路由冗余_19612408.md)。
- 已通过初始配置完成主备LoopBack接口的路由互通配置。
- 静态地址用户路由冗余特性开启和关闭前都要确保APN下无使用静态路由冗余功能的用户。
- 本特性与UNC配合使用生效，请参考UNC中的**WSFD-107021 静态地址用户路由冗余**。

## [操作步骤](#ZH-CN_OPI_0319612403)

1. 配置主用 UDG 静态地址用户路由冗余功能。
    a. 如果用户为静态IP地址用户或RADIUS分配地址用户，执行以下配置。
          1. 配置EXTERNAL类型地址池。
            [**ADD POOL**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/会话地址管理/地址池配置/添加地址池（ADD POOL）_82837132.md)
            > **说明**
            > - 主备UPF是针对地址池发布的路由优先级而定的，地址池发布的路由优先级高，所在的UPF就是主UPF。
            > - 当使能上行转发功能，且需要具备UE互访能力时，需在主UPF上将MASTERFLAG配置为ENABLE。
          2. 配置静态地址池的地址段。
                  - 配置IPV4地址段。
                    [**ADD SECTION**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/会话地址管理/地址段配置/添加地址池IP地址段（ADD SECTION）_82837114.md)
                  - 配置IPV6地址段。
                    [**ADD SECTION**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/会话地址管理/地址段配置/添加地址池IP地址段（ADD SECTION）_82837114.md)
            ![](激活静态地址用户路由冗余_19612403.assets/notice_3.0-zh-cn.png)

            - 静态地址池下需要配置地址段，一个静态地址池最多可以配置64个地址段。
                  - 配置地址池后，必须配置地址段。如果不配置地址段，将导致用户激活失败。
                  - 静态地址池在配置时，建议尽量使用完整的网段地址。
          3. 配置地址池组。
            [**ADD POOLGROUP**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/会话地址管理/地址池组/添加地址池组（ADD POOLGROUP）_82837138.md)
          4. 将地址池绑定地址池组。
            [**ADD POOLBINDGROUP**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/会话地址管理/地址池绑定地址池组/绑定地址池到地址池组（ADD POOLBINDGROUP）_82837143.md)
          5. 配置APN名称。
            [**ADD APN**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/DN管理/APN管理/APN/添加APN配置（ADD APN）_82837014.md)
          6. 配置地址池组与APN的映射关系。
            [**ADD POOLGRPMAP**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/会话地址管理/地址池组映射关系/添加地址池组映射关系（ADD POOLGRPMAP）_82837148.md)
    b. 配置冗余备份重定向IP，将业务流重定向到GRE Tunnel。
      [**ADD REDUNDRDTIP**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/会话地址管理/静态地址用户路由冗余/冗余备份重定向IP/增加冗余备份重定向IP地址（ADD REDUNDRDTIP）_75097445.md)
    c. 配置LoopBack接口。
      [**ADD INTERFACE**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/接口管理/接口配置/增加接口（ADD INTERFACE）_49960870.md)
    d. 将LoopBack接口绑定VPN实例。Loopback接口和Tunnel接口绑定的VPN实例不同，完成物理和业务上的划分。
      [**ADD IPBINDVPN**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/接口管理/IP绑定VPN/增加接口绑定VPN（ADD IPBINDVPN）_50120734.md)
    e. 配置GRE Tunnel接口。
      [**ADD INTERFACE**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/接口管理/接口配置/增加接口（ADD INTERFACE）_49960870.md)
      > **说明**
      > 针对IPv4生效，MTU配置原则：封装GRE隧道出外联口不会再次分片。
    f. 配置备份GRE Tunnel。
      [**ADD GRETUNNEL**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/VPN管理/GRE管理/GRE隧道/增加GRE隧道（ADD GRETUNNEL）_00841729.md)
    g. 将Tunnel接口绑定VPN实例。
      [**ADD IPBINDVPN**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/接口管理/IP绑定VPN/增加接口绑定VPN（ADD IPBINDVPN）_50120734.md)
    h. 配置逻辑接口的IP地址。
          - 配置IPV4地址。

      [**ADD SRROUTE**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/路由管理/静态路由管理/IPv4静态路由/增加IPv4静态路由（ADD SRROUTE）_49961498.md)

## [任务示例](#ZH-CN_OPI_0319612403)

任务描述

配置主用 UDG 和备用 UDG 的静态地址用户路由冗余功能， UDG 上规划IPv4和IPv6两个静态地址池。

脚本

1. 配置主用 UDG 的静态地址用户路由冗余功能，静态地址段分为IPv4和IPv6两类。
    a. 配置地址池信息。
      ```
      ADD POOL: POOLNAME="pool1", POOLTYPE=EXTERNAL, IPVERSION=IPV4, HASVPN=ENABLE, VPNINSTANCE="vpn1", REDUNDFUNC=ENABLE;
      ```
      ```
      ADD POOL: POOLNAME="pool2", POOLTYPE=EXTERNAL, IPVERSION=IPV6, HASVPN=ENABLE, VPNINSTANCE="vpn1", REDUNDFUNC=ENABLE;
      ```
      ```
      ADD SECTION: POOLNAME="pool1", SECTIONNUM=1, IPVERSION=IPV4, V4STARTIP="10.1.1.0", V4ENDIP="10.1.1.225";
      ```
      ```
      ADD SECTION: POOLNAME="pool2", SECTIONNUM=2, IPVERSION=IPV6, V6PREFIXSTART="fc00:1::1", V6PREFIXEND="fc00:1::1111", V6PREFIXLENGTH=64;
      ```
      ```
      ADD POOLGROUP: POOLGRPNAME="poolgroup1", IPV4ALLOCPRIALG=ENABLE;
      ```
      ```
      ADD POOLBINDGROUP:POOLGROUPNAME="poolgroup1", POOLNAME="pool1";
      ```
      ```
      ADD POOLBINDGROUP:POOLGROUPNAME="poolgroup1", POOLNAME="pool2";
      ```
      ```
      ADD APN: APN="test", HASVPN=ENABLE, VPNINSTANCE="vpn1";
      ```
      ```
      ADD POOLGRPMAP: MAPPINGNAME="mapping1", APN="test", POOLGROUPNAME="poolgroup1";
      ```
    b. 配置冗余备份重定向IP。
      ```
      ADD REDUNDRDTIP: IPVERSION=IPv4, IPV4ADDRESS="192.168.0.1";
      ```
      ```
      ADD REDUNDRDTIP: IPVERSION=IPv6, IPV6ADDRESS="fc00:3::1";
      ```
    c. 配置loopBack接口。
      ```
      ADD INTERFACE: IFNAME="LoopBack1", IFADMINSTATUS=up;
      ```
      ```
      ADD IPBINDVPN: IFNAME="LoopBack1", VRFNAME="vpn2";
      ```
      ```
      ADD IFIPV4ADDRESS: IFNAME="LoopBack1", IFIPADDR="10.11.1.1", SUBNETMASK="255.255.255.255";
      ```
    d. 配置GRE Tunnel。
      ```
      ADD INTERFACE: IFNAME="Tunnel1", IFMTU=1500;
      ```
      ```
      ADD GRETUNNEL: TNLNAME="Tunnel1", SRCTYPE=if_name, DSTIPADDR="10.10.0.1", TNLTYPE=gre, SRCIFNAME="LoopBack1", DSTVPNNAME="vpn2", REDUNDANCYEN=TRUE, KEEPALVEN=TRUE;
      ```
      ```
      ADD IPBINDVPN: IFNAME="Tunnel1", VRFNAME="vpn1";
      ```
      ```
      ADD IFIPV4ADDRESS: IFNAME="Tunnel1", IFIPADDR="10.11.1.3", SUBNETMASK="255.255.255.0";
      ```
      ```
      SET IFIPV6ENABLE: IFNAME="Tunnel1", ENABLEFLAG=TRUE, MTU=1500;
      ```
      ```
      ADD IFIPV6ADDRESS: IFNAME="Tunnel1", IPV6ADDRESS="fc00:2::1", PREFIXLEN=64;
      ```
    e. 配置IPv4静态路由。
      ```
      ADD SRROUTE: AFTYPE=ipv4unicast, PREFIX="192.168.0.1", MASKLENGTH=32, VRFNAME="vpn1", IFNAME="Tunnel1";
      ```
    f. 配置IPv6静态路由。
      ```
      ADD SRROUTE6: AFTYPE=ipv6unicast, PREFIX="fc00:3::1", MASKLENGTH=128, VRFNAME="vpn1", IFNAME="Tunnel1"; 
      ```
    g. （可选）PGW-U/UPF与PDN/DN之间配置GRE隧道，并且由PDN/DN侧服务器端配置高低优先级路由。此场景下应在主 UDG 配置下行缺省低优先级静态地址用户网段路由入冗余GRE隧道。
      ```
      ADD SRROUTE: AFTYPE=ipv4unicast, PREFIX="10.1.1.0", MASKLENGTH=24, VRFNAME="vpn1", IFNAME="Tunnel1", BFDENABLE=FALSE, DHCPENABLE=FALSE;
      ```
      ```
      ADD SRROUTE6: AFTYPE=ipv6unicast, PREFIX="fc00:1::1", MASKLENGTH=64, VRFNAME="vpn1", IFNAME="Tunnel1", BFDENABLE=FALSE, DHCPENABLE=FALSE;
      ```
    h.
      主备LoopBack接口的路由需要打通，当使用OSPF路由时，可以使用ADD OSPFINTERFACE命令打通路由。

      ```
      ADD OSPFINTERFACE: PROCID=6, AREAID="0.0.0.5", IFNAME="LoopBack1";
      ```
    i. 对于N6口路由，要检查路由发布优先级，主UPF优先级要高于备UPF优先级。BGP方式，通过MOD IMPORTROUTE命令配置，OSPF方式，通过MOD OSPFIMPORTROUTE命令配置。
      ```
      MOD OSPFIMPORTROUTE: PROCID=600, TOPOID=0, PROTOCOL=wlr, IMPTCOSTCFG=TRUE, COST=100;
      ```
    j. 使能静态地址用户路由冗余功能。
      ```
      SET REDUNDUSER: SWITCH=ENABLE;
      ```

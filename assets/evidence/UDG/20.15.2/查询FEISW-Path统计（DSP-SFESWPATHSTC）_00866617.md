#### [命令功能](#ZH-CN_CONCEPT_0000001600866617)

该命令用来查询FEISW Path统计。

#### [注意事项](#ZH-CN_CONCEPT_0000001600866617)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600866617)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600866617)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RUNAME | 资源单元名称 | 可选必选说明：必选参数<br>参数含义：指定系统中有效的资源单元。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。<br>默认值：无<br>配置原则：<br>- 区分大小写，不支持空格。<br>- 使用DSP RU查看RU名称。 |
| PATHID | Path ID | 可选必选说明：必选参数<br>参数含义：指定Path的编号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～4294967295。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600866617)

查询FEISW Path统计：

```
DSP SFESWPATHSTC:RUNAME="VNRS_IPCTRL_EX_RU_0201",PATHID=90232;
```

```

RETCODE = 0  操作成功

结果如下
------------------------
         Path ID  =  90232
        Path数量  =  63
上次接收Path时间  =  2017-07-02 17:08:01
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600866617)

| 输出项名称 | 输出项解释 |
| --- | --- |
| Path ID | 指定Path的编号。 |
| Path数量 | 指定Path的数量。 |
| 上次接收Path时间 | FEISW上次接收Path时间。 |

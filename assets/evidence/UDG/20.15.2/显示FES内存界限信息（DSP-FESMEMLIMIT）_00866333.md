#### [命令功能](#ZH-CN_CONCEPT_0000001600866333)

该命令用于显示FES内存界限信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600866333)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600866333)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600866333)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001600866333)

显示FES内存界限信息：

```
DSP FESMEMLIMIT:;
```

```

RETCODE = 0  操作成功。

结果如下
------------------------
               RU号  =  66
       内存过载状态  =  正常状态
       过载状态标记  =  正常状态
(结果个数 = 1)
--- END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600866333)

| 输出项名称 | 输出项解释 |
| --- | --- |
| RU号 | 用来表示RU号。 |
| 内存过载状态 | 该参数用于表示内存过载状态：<br>- 正常状态。<br>- 超限状态。 |
| 过载状态标记 | 该参数用于表示过载状态标记：<br>- 正常状态。<br>- 矛盾状态。 |

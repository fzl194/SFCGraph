#### [命令功能](#ZH-CN_CONCEPT_0000203928361127)

**适用NF：PGW-U、UPF**

该命令用于查询智能板的大流判断速率阈值。

#### [注意事项](#ZH-CN_CONCEPT_0000203928361127)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000203928361127)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000203928361127)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000203928361127)

查询智能板的大流判断速率阈值：

```
%%LST SSUBIGFLOWCTRL:;
```

```
%%
RETCODE = 0  操作成功

查询智能板的大流判断速率阈值
----------------------------
VVIP业务大流速率（千比特/秒）  =  800
 CCO业务大流速率（千比特/秒）  =  800
(结果个数 = 1)

---    END
```

# 删除高级团体属性过滤器节点（RMV ADVANCEDCOMMUNITYNODE）

- [注意事项](#ZH-CN_CONCEPT_0000001550281786__1.3.2.1)

#### [命令功能](#ZH-CN_CONCEPT_0000001550281786)

该命令用来删除高级团体属性过滤器节点。

#### [注意事项](#ZH-CN_CONCEPT_0000001550281786)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550281786)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550281786)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| CMNTYNAMEORNUM | 团体属性过滤器名字或号 | 可选必选说明：必选参数<br>参数含义：该参数用于指定团体属性过滤器名字或团体属性过滤器号。<br>数据来源：本端规划<br>取值范围：字符串类型，团体属性过滤器名称其长度范围是1～51。<br>默认值：无 |
| NODESEQUENCE | 团体属性过滤器节点ID | 可选必选说明：必选参数<br>参数含义：团体属性过滤器节点ID。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |
| REGULAR | 正则表达式 | 可选必选说明：可选参数<br>参数含义：正则表达式。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～1024。<br>默认值：无<br>配置原则：支持空格。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001550281786)

删除高级团体属性过滤器节点：

```
RMV ADVANCEDCOMMUNITYNODE:NODESEQUENCE=1,CMNTYNAMEORNUM="ab";
```

#### [命令功能](#ZH-CN_CONCEPT_0000202386851945)

**适用NF：PGW-U、UPF**

该命令用来开启SA指纹识别功能。

#### [注意事项](#ZH-CN_CONCEPT_0000202386851945)

- 该命令执行后立即生效。
- 该命令最大记录数为128。
- SA指纹识别功能开启后，如果用户访问开启该功能的业务，系统会在正常的特征库识别之外，对业务做更进一步的指纹识别，以增强识别精度。
- 可以通过FingerIdentCtrl参数控制指纹识别结果只用于统计，不用于业务控制。如果同时配置了指定协议和所有协议的FingerIdentCtrl参数，则优先使用指定协议的配置。
- 并不是所有特征库可以识别的协议都支持SA指纹识别功能。如果该命令配置的协议不支持指纹识别，则该配置无效，业务仍然只进行特征库识别。支持指纹识别的协议，可以通过命令LST SUPPFINGERIDEN查看。
- 该命令可以指定最多128个协议的SA指纹识别功能，以及一个所有协议的场景。
- 开启指纹识别功能后，对SA性能会有部分影响，详情请联系华为工程师。
- 该命令执行后配置立即生效，SA指纹识别功能2分钟后生效且对新流生效。
- 该命令误配后会影响系统性能。执行命令前请评估对性能的影响，如果无法评估请联系华为技术支持。

#### [操作用户权限](#ZH-CN_CONCEPT_0000202386851945)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000202386851945)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| CFGPROTMODE | 协议配置方式 | 可选必选说明：必选参数<br>参数含义：该参数用于设置SA指纹识别功能协议的配置方式。<br>数据来源：本端规划<br>取值范围：<br>- ALL：所有协议。<br>- SPECIFIC：指定协议。<br>默认值：无<br>配置原则：无 |
| PROTOCOLNAME | 协议名称 | 可选必选说明：条件必选参数<br>前提条件：该参数在“CFGPROTMODE”配置为“SPECIFIC”时为必选参数。<br>参数含义：该参数用于设置协议名称。<br>数据来源：本端规划<br>取值范围：不支持空格，不区分大小写。<br>默认值：无<br>配置原则：无 |
| FINGERIDENTCTRL | 指纹识别结果控制开关 | 可选必选说明：可选参数<br>参数含义：该参数用于设置指纹识别结果控制开关。<br>数据来源：本端规划<br>取值范围：<br>- DISABLE：SA指纹识别结果既用于统计，又参与业务控制。<br>- STATIC_ONLY：SA指纹识别结果只用于统计，不参与业务控制。<br>默认值：DISABLE<br>配置原则：无 |
| CFGDOMAINNAME | 配置域名称 | 可选必选说明：条件可选参数<br>前提条件：该参数在“CFGPROTMODE”配置为“SPECIFIC”时为可选参数。<br>参数含义：该参数表示命令所属公共配置域的名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000202386851945)

- 开启所有协议SA指纹识别功能，配置如下：
  ```
  ADD FINGERIDENT: CFGPROTMODE=ALL;
  ```
- 开启Facebook协议SA指纹识别功能，配置如下：
  ```
  ADD FINGERIDENT: CFGPROTMODE=SPECIFIC, PROTOCOLNAME="facebook";
  ```

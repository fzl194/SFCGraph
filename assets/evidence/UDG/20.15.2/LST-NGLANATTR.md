#### [命令功能](#ZH-CN_CONCEPT_0000206783372126)

**适用NF：UPF**

该命令用于查询指定5G LAN组的属性配置。

#### [注意事项](#ZH-CN_CONCEPT_0000206783372126)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000206783372126)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000206783372126)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| VNINSTANCE | 5G LAN组名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定5G LAN会话实例名称。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围为18～37。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000206783372126)

查询指定5G LAN组总的MAC地址超限个数：

```
LST NGLANATTR: VNINSTANCE="a0000001-460-003-50";
```

```

%%LST NGLANATTR: VNINSTANCE="a0000001-460-003-50";%%
RETCODE = 0 操作成功

结果如下
--------
     5G LAN组名称  =  a0000001-460-003-50
支持的总MAC地址数  =  2000
(结果个数 = 1)

---    END
```

#### [命令功能](#ZH-CN_CONCEPT_0182837786)

**适用NF：PGW-U、UPF**

该命令用于查询系统中当前使用的防欺诈业务规则数据库的版本信息。

#### [注意事项](#ZH-CN_CONCEPT_0182837786)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0182837786)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0182837786)

无。

#### [使用实例](#ZH-CN_CONCEPT_0182837786)

查询防欺诈业务规则数据库版本信息：

```
DSP AFDB:;
```

```

RETCODE = 0  操作成功

防欺诈业务规则数据库版本信息
----------------------------
防欺诈业务规则数据库版本号  =  0001.01.0086.0001.001
                引擎版本号  =  01
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0182837786)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 防欺诈业务规则数据库版本号 | 用于指定防欺诈业务规则数据库的版本号。 |
| 引擎版本号 | 用于指定加载的防欺诈业务规则数据库的引擎版本号。 |

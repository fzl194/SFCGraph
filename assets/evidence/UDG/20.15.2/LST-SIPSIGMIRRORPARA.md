#### [命令功能](#ZH-CN_CONCEPT_0000205630165424)

**适用NF：SGW-U、UPF**

该命令用于查询SIP镜像功能是否使能。

#### [注意事项](#ZH-CN_CONCEPT_0000205630165424)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000205630165424)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000205630165424)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000205630165424)

该命令用于查询SIP镜像功能是否使能：

```
LST SIPSIGMIRRORPARA:;
```

```

RETCODE = 0  操作成功

结果如下
--------
       镜像功能开关  =  不使能
内层IPv6报文分片MTU  =  0
          重定向VPN  =  NULL
(结果个数 = 1)

---    END
```

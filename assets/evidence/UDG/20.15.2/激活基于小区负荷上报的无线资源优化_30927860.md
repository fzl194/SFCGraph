## [操作场景](#ZH-CN_OPI_0000001130927860)

运营商规划基于小区负荷状态对某些用户或某些业务进行策略控制以解除小区拥塞状况，需要激活基于小区负荷上报的无线资源优化功能。

> **说明**
> 适用于PGW-U。

## [任务示例](#ZH-CN_OPI_0000001130927860)

任务描述

在 UDG 上使能APN“apn-test”的基于小区负荷上报的无线资源优化功能。

脚本

1. 打开本特性的License配置开关。
  ```
  SET LICENSESWITCH:LICITEM="
  LKV3G5WOCR01
  ",SWITCH=ENABLE;
  ```
2. 开启小区负荷上报功能。
  ```
  SET APNREPORTATTR:APN="apn-test",CONGESTIONRPT=ENABLE;
  ```

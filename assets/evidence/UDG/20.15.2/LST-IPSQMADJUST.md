#### [命令功能](#ZH-CN_CONCEPT_0000201708991781)

**适用NF：SGW-U、UPF**

该命令用于查询业务处理节点（POD）的最低保护带宽及突发尺寸。

#### [注意事项](#ZH-CN_CONCEPT_0000201708991781)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000201708991781)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000201708991781)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000201708991781)

业务处理节点（POD）的最低保护带宽及突发尺寸：

```
LST IPSQMADJUST:;
```

```

RETCODE = 0  操作成功

IPSQM调整参数
-------------
                阈值（兆比特/秒）  =  30
低带宽域的最低保护值（千比特/秒）  =  300
       低带宽域的突发尺寸（字节）  =  2000
高带宽域的最低保护值（千比特/秒）  =  1000
       高带宽域的突发尺寸（字节）  =  10000
(结果个数 = 1)

---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000201708991781)

#### [命令功能](#ZH-CN_CONCEPT_0000207394632057)

**适用NF：UPF、PGW-U**

该命令用于查询媒体中继全局回源TCP控制。

#### [注意事项](#ZH-CN_CONCEPT_0000207394632057)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000207394632057)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000207394632057)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000207394632057)

查询媒体中继全局回源TCP控制：

```
LST GLBRLYOTCPCTRL:;
```

```

```

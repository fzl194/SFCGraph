#### [命令功能](#ZH-CN_CONCEPT_0000001600841773)

该命令用于查询OSPF等价路由优先级配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001600841773)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600841773)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600841773)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | 进程号 | 可选必选说明：可选参数<br>参数含义：OSPF进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |
| NBRIPADDRESS | 下一跳IP地址 | 可选必选说明：可选参数<br>参数含义：下一跳IP地址。<br>数据来源：本端规划<br>取值范围：IPv4地址类型。点分十进制格式。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600841773)

查询OSPF等价路由优先级配置：

```
LST OSPFNEXTHOP:;
```

```

RETCODE = 0  操作成功。

结果如下
-----------
      进程号  =  1
    拓扑标识  =  0
下一跳IP地址  =  10.4.4.4
  下一跳权重  =  10
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600841773)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 进程号 | 该参数指定用户创建的OSPF进程号。 |
| 拓扑标识 | 该参数指定用户创建的OSPF拓扑标识。 |
| 下一跳IP地址 | 该参数指定等价路由的下一跳IP地址。 |
| 下一跳权重 | 该参数指定等价路由的下一跳权重。该值越小，路由优先级越高。 |

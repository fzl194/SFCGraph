#### [命令功能](#ZH-CN_CONCEPT_0000001600600693)

该命令用于查看ARP系统老化时间、全局ARP严格学习使能、ARP日志告警上报时间间隔和免费ARP发送类型。

#### [注意事项](#ZH-CN_CONCEPT_0000001600600693)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600600693)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600600693)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001600600693)

查询ARP系统配置信息：

```
LST ARPSYS:;
```

```

RETCODE = 0  操作成功。

结果如下
--------
           表项老化时间（s）  =  1200
         全局ARP严格学习使能  =  FALSE
ARP日志告警上报时间间隔（s）  =  80
             免费ARP发送类型  =  请求
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600600693)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 表项老化时间（s） | 该参数表示表项老化时间。 |
| 全局ARP严格学习使能 | 该参数用于指定全局ARP严格学习是否使能。 |
| ARP日志告警上报时间间隔（s） | 该参数用于指定ARP写日志和发送告警的时间间隔。 |
| 免费ARP发送类型 | 该参数用于指定发送免费ARP报文的类型。 |

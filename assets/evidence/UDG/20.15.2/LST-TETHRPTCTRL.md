#### [命令功能](#ZH-CN_CONCEPT_0000201415208993)

**适用NF：PGW-U、UPF**

该命令用于查询tethering事件上报的相关参数。

#### [注意事项](#ZH-CN_CONCEPT_0000201415208993)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000201415208993)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000201415208993)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000201415208993)

查询tethering事件上报的相关参数：

```
LST TETHRPTCTRL:;
```

```

RETCODE = 0 操作成功

设置tethering事件上报的相关参数
----------------------------------------------------
          Tethering Stop消息上报开关 = 不使能
Tethering Stop消息上报迟滞时间（秒） = 0
               Tethering上报事件模式 = TETHERING行为模式
(结果个数 = 1)

--- END
```

#### [命令功能](#ZH-CN_CONCEPT_0182837784)

**适用NF：PGW-U、UPF**

该命令用于加载或升级防欺诈业务规则数据库，系统在做防欺诈业务检测时需要使用防欺诈业务规则数据库作为参考，可以通过无损升级数据库的方式增强欺诈行为的检测能力。

#### [注意事项](#ZH-CN_CONCEPT_0182837784)

- 该命令执行后立即生效。
- 新增防欺诈业务规则数据库可以通过网管/MAE的“网元文件传输”功能或者OM Portal的“文件传输”功能进行库文件上传。
- 系统如果未加载防欺诈业务规则数据库文件，只能依赖内置的条件做欺诈行为检测，将无法确保检测结果的准确性和全面性。加载旧版本的数据库文件可能会影响系统对最新欺诈行为的检测能力识别能力。
- 数据库加载或更新需要一段时间，如果在执行完毕之前重新执行本命令，系统会提示错误。
- 加载或者卸载命令两次执行间隔不能小于10s。
- 如果加载防欺诈业务规则数据文件不存在时，会上报81151告警。
- 使用本命令时，对性能会产生一定影响。

#### [操作用户权限](#ZH-CN_CONCEPT_0182837784)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0182837784)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| VERSION | 防欺诈业务规则数据库版本号 | 可选必选说明：必选参数<br>参数含义：该参数用于指定防欺诈业务规则数据库的版本号。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～21。版本号形式如AAAA.BB.CCCC.DDDD.EEE，ABCDE都是数字。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0182837784)

加载防欺诈业务规则数据库：

```
LOD AFDB: VERSION="0001.01.0086.0001.001";
```

#### [命令功能](#ZH-CN_CONCEPT_0000001600866425)

该命令用于查询调度部署详细信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600866425)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600866425)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600866425)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RUNAME | RU名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定RU名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，区分大小写。<br>默认值：无<br>配置原则：使用DSP RU查看RU名称。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600866425)

查询调度模块统计信息：

```
DSP MSSSCHSUMTAIL:RUNAME = "VNODE_VNRS_VNFC_IPU_0064";
```

```

1             2               1                 2                     0                     NULL                  3                           OFF
2             3               1                 2                     0                     NULL                  4                           OFF
(结果个数 = 2)
---    END
```

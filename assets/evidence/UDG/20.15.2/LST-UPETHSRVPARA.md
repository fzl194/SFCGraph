#### [命令功能](#ZH-CN_CONCEPT_0000206875556851)

**适用NF：UPF**

该命令用于查询用户面以太业务相关参数配置。

#### [注意事项](#ZH-CN_CONCEPT_0000206875556851)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000206875556851)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000206875556851)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000206875556851)

查询用户面以太业务相关参数配置：

```
LST UPETHSRVPARA:;
```

```

RETCODE = 0 操作成功。

用户面以太业务参数配置
-----------------
用户面以太业务开关  =  ENABLE          
用户面以太业务MAC地址  =  10-11-11-11-11-11
(结果个数 = 1)
--- END
```

#### [命令功能](#ZH-CN_CONCEPT_0000001600440365)

该命令用于查询定时器全局信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600440365)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600440365)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600440365)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RUNAME | RU名称 | 可选必选说明：必选参数<br>参数含义：该参数用于表示该参数用于指定RU名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，区分大小写。<br>默认值：无<br>配置原则：使用DSP RU查看RU名称。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600440365)

查询定时器全局信息：

```
DSP MSSTIMERINFOHEAD:RUNAME = "VNODE_VNRS_VNFC_IPU_0064";
```

```

RETCODE = 0  操作成功。

结果如下
-------------------------
              定时器总数  =  8192
      当前使用定时器数量  =  6
              定时器峰值  =  6
          启动定时器次数  =  6
          停止定时器次数  =  0
同一时刻触发的定时器峰值  =  12
(结果个数 = 1)
---    END
```

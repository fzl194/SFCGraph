#### [命令功能](#ZH-CN_CONCEPT_0000001600841313)

该命令用于查询DHCPv6客户端的DUID生成方式。

#### [注意事项](#ZH-CN_CONCEPT_0000001600841313)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600841313)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600841313)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001600841313)

查询当前DHCPv6客户端的DUID生成方式：

```
LST DHCP6CLIENTDUID:;
```

```

RETCODE = 0  操作成功

结果如下
------------------------
DUID生成方式  =  链路地址加虚拟局域网标识号模式
(结果个数 = 1)
---    END
```

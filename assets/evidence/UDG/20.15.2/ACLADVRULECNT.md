#### [命令功能](#ZH-CN_CONCEPT_0000001550281558)

该命令用于显示高级ACL规则的匹配计数信息。可应用于路由策略等使用场景的匹配计数信息查询。

#### [注意事项](#ZH-CN_CONCEPT_0000001550281558)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550281558)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550281558)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| ACLNAME | ACL规则组标识 | 可选必选说明：必选参数<br>参数含义：该参数用于指定规则属于哪个规则组。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～32。不支持空格，区分大小写。以英文字母a～z或A～Z开始，可以是英文字母、数字、连字符“-”、下划线“_”或中文字符的组合。<br>默认值：无 |
| ACLRULENAME | 规则名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定规则名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～32。不支持空格，区分大小写。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001550281558)

显示当前ACL规则组3000下规则名称为"rule_5"的匹配计数：

```
DSP ACLADVRULECNT:ACLNAME="3000",ACLRULENAME="rule_5";
```

```

RETCODE = 0  操作成功。

结果如下
--------
ACL规则组标识  =  3000
     规则名称  =  rule_5
     匹配计数  =  0
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001550281558)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 匹配计数 | 用于显示规则的匹配计数。 |

其余输出项请参见LST ACLGROUP的参数说明。

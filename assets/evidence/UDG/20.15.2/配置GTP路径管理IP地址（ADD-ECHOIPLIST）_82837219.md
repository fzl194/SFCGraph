#### [命令功能](#ZH-CN_CONCEPT_0182837219)

**适用NF：SGW-U、PGW-U、UPF**

该命令用来增加对端网元所属的GTP数据面IP地址段，即配置GTP路径管理黑白名单中的地址段。

#### [注意事项](#ZH-CN_CONCEPT_0182837219)

- 该命令执行后立即生效。
- 该命令最大记录数为200。
- ADD ECHOIPLIST是配合SET UPGTPPATH使用的。
- 使用LST UPGTPPATH查看EchoList类型为黑名单还是白名单。
- 如果对端网元的IP地址落在ADD ECHOIPLIST配置的地址段内：
    - 配置GTP路径管理名单属性为白名单时，系统向对端网元发送GTP信令路径Echo探测或是GTP数据路径Echo探测。
    - 配置GTP路径管理名单属性为黑名单时，系统不会向对端网元发送GTP信令路径Echo探测或是GTP数据路径Echo探测。
- 如果对端网元的IP地址没有落在ADD ECHOIPLIST配置的地址段内：
    - 配置GTP路径管理名单属性为黑名单时，系统向对端网元发送GTP信令路径Echo探测或是GTP数据路径Echo探测。
    - 配置GTP路径管理名单属性为白名单时，系统不会向对端网元发送GTP信令路径Echo探测或是GTP数据路径Echo探测。

#### [操作用户权限](#ZH-CN_CONCEPT_0182837219)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [使用实例](#ZH-CN_CONCEPT_0182837219)

增加IP地址为10.36.0.2，mask为27的IP地址段为GTP路径管理的黑白名单控制的网段：

```
ADD ECHOIPLIST: IPVERSION=IPV4, IPV4ADDR="10.36.0.2", IPV4MASK=27;
```

#### [命令功能](#ZH-CN_CONCEPT_0000001550281734)

该命令用于查询静态路由迭代隧道功能开关。

#### [注意事项](#ZH-CN_CONCEPT_0000001550281734)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550281734)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550281734)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001550281734)

查询静态路由迭代隧道功能开关：

```
LST SRROUTERELAYTUNNEL:;
```

```

RETCODE = 0  操作成功.
结果如下
------------------------
    迭代隧道使能开关 =  TRUE
          前缀列表名 =  NULL
         隧道策略名  =  tp
(结果个数 = 1)
---    END
```

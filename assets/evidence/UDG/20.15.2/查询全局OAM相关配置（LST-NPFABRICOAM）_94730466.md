## [命令功能](#ZH-CN_MMLREF_0294730466)

该命令用于查询全局OAM相关配置：包括OAM使能，OAM报文检测周期，OAM报文超时检测倍数。

> **说明**
> - 该命令执行后立即生效。
> - 该命令仅适用于NP卡加速模式场景。

#### [操作用户权限](#ZH-CN_MMLREF_0294730466)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0294730466)

无

## [使用实例](#ZH-CN_MMLREF_0294730466)

查询全局OAM相关配置

```
LST NPFABRICOAM:;
RETCODE = 0  操作成功

结果如下
--------
            OAM使能  =  使能
OAM报文发送周期(ms)  =  100
OAM报文超时检测倍数  =  30
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0294730466)

| 输出项名称 | 输出项解释 |
| --- | --- |
| OAM使能 | 该参数用于表示OAM使能开关。 |
| OAM报文发送周期(ms) | 该参数用于表示OAM报文发送周期，单位为毫秒。 |
| OAM报文超时检测倍数 | 该参数用于表示OAM报文超时检测倍数。 |

#### [命令功能](#ZH-CN_CONCEPT_0000207114541477)

**适用NF：UPF、PGW-U**

该命令用于查询媒体中继功能配置。

#### [注意事项](#ZH-CN_CONCEPT_0000207114541477)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000207114541477)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000207114541477)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000207114541477)

查询媒体中继功能配置：

```
LST RELAYFUNC:;
```

```

RETCODE = 0  操作成功

媒体中继功能配置
-----------------------------
媒体中继功能开关  =  禁用
(结果个数 = 1)

---    END
```

#### [命令功能](#ZH-CN_CONCEPT_0000001600866037)

该命令用于显示所有FIB路由表项的统计信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600866037)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600866037)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600866037)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RUNAME | RU名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定资源单元名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。区分大小写。<br>默认值：无<br>配置原则：<br>- 区分大小写，不支持空格。<br>- 使用DSP RU查看RU名称。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600866037)

显示指定资源单元的总路由数量：

```
DSP SFEFIBSTATALL:RUNAME="VNODE_VNRS_VNFC_IPU_0064";
```

```

RETCODE = 0  操作成功

结果如下
-------------------------
      路由总数   =  8
   公网路由数量  =  7
总FIB表规格容量  =  1048576
(结果个数 = 1)
---    END
```

| 公网路由数量 | 公网路由数量。 |

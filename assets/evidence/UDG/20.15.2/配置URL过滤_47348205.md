## [操作场景](#ZH-CN_OPI_0000001547348205)

当运营商需要为用户提供URL过滤套餐，以满足特殊用户群的业务需求，则需要部署URL过滤基本功能。

> **说明**
> 适用于PGW-U、UPF。

## [必备事项](#ZH-CN_OPI_0000001547348205)

前提条件

- 请仔细阅读[GWFD-110471 URL过滤基本功能](../../GWFD-110471 URL过滤基本功能_05929912.md)。

- 完成[配置到ICAP Server的互通数据](配置到ICAP Server的互通数据_75400117.md)。
- 完成激活PCC 基本功能。
- 在ICAP Server上已导入URL分类数据信息。

## [操作步骤](#ZH-CN_OPI_0000001547348205)

1. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09587387.md)
2. 配置URL过滤用户使用的APN。
    a. 添加APN。
      [**ADD APN**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/DN管理/APN管理/APN/添加APN配置（ADD APN）_82837014.md)
    b. 基于APN粒度开启URL过滤功能。
      **[SET APNCFFUNC](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/内容过滤/APN内容过滤开关配置/设置APN内容过滤开关（SET APNCFFUNC）_54188227.md)**
3. 配置URL过滤。
    a. 配置过滤策略。
      [**ADD CFPROFILE**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/内容过滤/内容过滤策略配置/增加内容过滤策略（ADD CFPROFILE）_43236705.md)
    b. 配置过滤策略模板。
      [**ADD CFTEMPLATE**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/内容过滤/内容过滤模板配置/增加内容过滤模板（ADD CFTEMPLATE）_73992447.md)
    c. 配置APN的URL过滤模板。
      **[SET APNCFTEMPLATE](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/内容过滤/APN 内容过滤模板绑定配置/设置APN内容过滤模板（SET APNCFTEMPLATE）_54628141.md)**
    d. 配置URL过滤策略和模板绑定。
      [**ADD CFPROFBINDCFT**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/内容过滤/内容过滤策略绑定配置/增加内容过滤策略绑定关系（ADD CFPROFBINDCFT）_43236707.md)
    e. 配置分类组。
      [**ADD CONTCATEGROUP**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/内容过滤/内容分类组配置/增加内容分类组（ADD CONTCATEGROUP）_39478824.md)
    f. 配置过滤策略与分类组绑定。
      [**ADD CONTCATEGBIND**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/内容过滤/内容分类组绑定配置/增加内容分类组绑定关系（ADD CONTCATEGBIND）_39478826.md)
4. 开启本地cache功能。
  [**SET CFCACHEPARA**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/内容过滤/内容过滤缓存参数配置/设置内容过滤缓存参数（SET CFCACHEPARA）_54389381.md)
5. 配置URL过滤条件。
    a. 配置Filter。
      [**ADD FILTER**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/过滤器/增加过滤器（ADD FILTER）_82837348.md)
    b. 将Filter配置为生效。
      **[SET REFRESHSRV](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/业务刷新/业务刷新（SET REFRESHSRV）_82837355.md)**
6. 配置流过滤器，绑定过滤条件。
    a. 配置流过滤器。
      [**ADD FLOWFILTER**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/流过滤器/增加流过滤器（ADD FLOWFILTER）_82837361.md)
    b. 配置过滤条件与流过滤器的绑定关系。
      [**ADD FLTBINDFLOWF**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/流过滤器的过滤器绑定/增加流过滤器的过滤器绑定关系（ADD FLTBINDFLOWF）_82837366.md)
7. 配置计费属性。
    a. 配置URR。
      [**ADD URR**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/计费控制/使用率上报规则/增加URR（ADD URR）_82837629.md)
    b. 配置上行和下行业务对应的URR。
      [**ADD URRGROUP**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/计费控制/使用率上报规则组/增加URR组（ADD URRGROUP）_82837634.md)
8. 配置PCC策略。
  [**ADD PCCPOLICYGRP**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/PCC控制策略/PCC策略组/增加PCC策略组（ADD PCCPOLICYGRP）_82837606.md)
9. 配置业务规则。
  [**ADD RULE**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务匹配公共配置/业务规则管理/规则/增加规则（ADD RULE）_82837266.md)
  > **说明**
  > 如果运营商希望基于流过滤器组设置过滤条件时，配置 “流过滤器或流过滤器组选择(FILTERINGMODE)” 为 “FLOWFILTERGRP” ，并且使用 [**ADD FLOWFILTERGRP**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/流过滤器组/增加流过滤器组（ADD FLOWFILTERGRP）_82837384.md) 命令配置流过滤器组。
10. 配置业务策略组合。
    a. 创建UserProfile。
      [**ADD USERPROFILE**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务匹配公共配置/业务规则管理/用户模板/增加用户模板（ADD USERPROFILE）_82837279.md)
    b. 将已配置的Rule绑定到UserProfile。
      [**ADD RULEBINDING**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务匹配公共配置/业务规则管理/规则绑定/增加用户模板和规则的绑定关系（ADD RULEBINDING）_82837272.md)

## [任务示例](#ZH-CN_OPI_0000001547348205)

任务描述

配置对指定APN下用户开启URL过滤功能。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH:LICITEM="LKV3G5UFBF01",SWITCH=ENABLE;
```

//配置URL过滤用户使用的APN。

```
ADD APN: APN="apn-test";
```

```
SET APNCFFUNC: APNNAME="apn-test", CFSWITCHVALUE=ENABLE;
```

//配置URL过滤。

```
ADD CFPROFILE: CFPROFILENAME="cf1";
```

```
ADD CFTEMPLATE: CFTEMPLATENAME="icaptemplate",ICAPSRVGMNAME="icapsvrgrp1", ACTION=BLOCK;
```

```
SET APNCFTEMPLATE: APNNAME="apn-test", CFTEMPLATENAME="icaptemplate";
```

```
ADD CFPROFBINDCFT: CFTEMPLATENAME="icaptemplate", CFPROFILENAME="cf1";
```

```
ADD CONTCATEGROUP: CONTCATEGNAME="ccgp1", CATEGORYTYPE=SPECIFIC, CATEGORYID=1;
```

```
ADD CONTCATEGBIND: CFPROFILENAME="cf1", CONTCATEGNAME="ccgp1", ACTION=BLOCK;
```

//开启本地cache功能。

```
SET CFCACHEPARA: CACHEIDLETIME=7200, CACHESW=ENABLE;
```

//配置URL过滤条件。

```
ADD FILTER:FILTERNAME="filter_test",L34PROTTYPE=STRING,L34PROTOCOL=ANY;
```

```
SET REFRESHSRV:REFRESHTYPE=USERPROFILE;
```

//配置流过滤器，绑定过滤条件。

```
ADD FLOWFILTER:FLOWFILTERNAME="flowfilter_test1";
```

```
ADD FLTBINDFLOWF:FLOWFILTERNAME="flowfilter_test1",FILTERNAME="filter_test";
```

//配置计费属性。

```
ADD URR: URRNAME="urr_2", URRID=2, USAGERPTMODE=OFFLINE;
```

```
ADD URRGROUP: URRGROUPNAME="urrgroup1", UPURRNAME1="urr_2",DOWNURRNAME1="urr_2";
```

//配置PCC策略。

```
ADD PCCPOLICYGRP:PCCPOLICYGRPNM="ppg_test",URRGROUPNAME="urrgroup1";
```

//配置业务规则。

```
ADD RULE:RULENAME="rule_test",POLICYTYPE=PCC,FILTERINGMODE=FLOWFILTER,FLOWFILTERNAME="flowfilter_test1",PRIORITY=1,POLICYNAME="ppg_test";
```

//配置业务策略组合。

```
ADD USERPROFILE:USERPROFILENAME="up_test";
```

```
ADD RULEBINDING:USERPROFILENAME="up_test",RULENAME="rule_test";
```

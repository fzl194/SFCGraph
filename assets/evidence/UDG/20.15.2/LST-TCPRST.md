#### [命令功能](#ZH-CN_CONCEPT_0182837627)

**适用NF：PGW-U、UPF**

该命令用来显示欠费用户下行TCP RST报文的处理动作。

#### [注意事项](#ZH-CN_CONCEPT_0182837627)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0182837627)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0182837627)

无。

#### [使用实例](#ZH-CN_CONCEPT_0182837627)

显示欠费用户下行RST报文的处理动作：

```
LST TCPRST:;
```

```

RETCODE = 0  操作成功。

下行RST报文处理动作
-------------------
欠费用户下行RST报文处理动作  =  允许通过
  欠费用户纯RST报文处理动作  =  丢弃
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0182837627)

参见SET TCPRST的参数说明。

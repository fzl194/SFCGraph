## [命令功能](#ZH-CN_MMLREF_0000001228971837)

该命令用于显示HTTP历史链路状态。

> **说明**
> 每个进程可以查询最多50条历史记录。

#### [操作用户权限](#ZH-CN_MMLREF_0000001228971837)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001228971837)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PODNAME | POD名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定HTTP链路所在的POD名称。该POD名称可通过<br>[**DSP HTTPPROCESS**](../HTTP进程状态管理/显示HTTP进程信息（DSP HTTPPROCESS）_29053327.md)<br>命令查询获取。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~255。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001228971837)

查询HTTP历史链路信息，可以执行下面的命令。

## [输出结果说明](#ZH-CN_MMLREF_0000001228971837)

| 输出项名称 | 输出项解释 |
| --- | --- |
| POD名称 | 该参数用于指定HTTP链路所在的POD名称。该POD名称可通过<br>[**DSP HTTPPROCESS**](../HTTP进程状态管理/显示HTTP进程信息（DSP HTTPPROCESS）_29053327.md)<br>命令查询获取。 |
| 进程标识 | 该参数用于显示进程标识。 |
| 源IPv4地址 | 该参数用于显示HTTP历史链路的源IPv4地址。 |
| 目的IPv4地址 | 该参数用于显示HTTP历史链路的目的IPv4地址。 |
| 源IPv6地址 | 该参数用于显示HTTP历史链路的源IPv6地址。 |
| 目的IPv6地址 | 该参数用于显示HTTP历史链路的目的IPv6地址。 |
| 接口类型 | 该参数用于显示HTTP历史链路的接口类型。 |
| 状态变更 | 该参数用于显示HTTP历史链路的链路变更状态。 |
| 时间戳 | 该参数用于显示HTTP历史链路的状态变化的时间戳。 |

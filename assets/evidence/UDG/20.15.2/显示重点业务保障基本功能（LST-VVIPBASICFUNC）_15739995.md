#### [命令功能](#ZH-CN_CONCEPT_0000204915739995)

**适用NF：PGW-U、UPF**

该命令用于查询重点业务保障基本功能的开关及上报周期等参数。

#### [注意事项](#ZH-CN_CONCEPT_0000204915739995)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000204915739995)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000204915739995)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000204915739995)

查询当前重点业务保障基本功能的设置：

```
%%LST VVIPBASICFUNC:;
```

```
%%
RETCODE = 0  操作成功

重点业务保障基本功能配置
------------------------
重点业务保障基本功能  =  使能
  质差上报周期（秒）  =  5
非质差上报周期（秒）  =  300
    质差报表上报范围  =  全部上报
  采集单据周期（秒）  =  5
        上报用户类型  =  上报5G用户
(结果个数 = 1)

---    END
```

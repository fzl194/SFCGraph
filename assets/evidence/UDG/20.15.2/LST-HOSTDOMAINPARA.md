#### [命令功能](#ZH-CN_CONCEPT_0000202125540725)

**适用NF：PGW-U、UPF**

该命令用来查看主机域相关参数配置。

#### [注意事项](#ZH-CN_CONCEPT_0000202125540725)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000202125540725)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000202125540725)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000202125540725)

查看主机域相关参数配置：

```
%%LST HOSTDOMAINPARA:;
```

```
%%
RETCODE = 0  操作成功

Host主机域配置信息
------------------
多Host学习开关  =  不使能（关闭）
(结果个数 = 1)

---    END
```

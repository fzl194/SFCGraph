#### [命令功能](#ZH-CN_CONCEPT_0000203343357967)

**适用NF：PGW-U、UPF**

该命令用于查询已开启内容过滤功能的协议列表。

#### [注意事项](#ZH-CN_CONCEPT_0000203343357967)

HTTP与WAP协议无需配置，缺省做内容过滤。

#### [操作用户权限](#ZH-CN_CONCEPT_0000203343357967)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000203343357967)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000203343357967)

查询已开启内容过滤功能的协议列表：

```
LST CFPROTOCOLLST:;
```

```

RETCODE = 0  操作成功

内容过滤协议类型列表
--------------------
内容过滤协议名称  =  HTTPS协议
      配置域名称  =  NULL
(结果个数 = 1)

---    END
```

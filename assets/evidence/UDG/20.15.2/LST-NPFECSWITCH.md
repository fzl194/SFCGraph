#### [命令功能](#ZH-CN_TOPIC_0250068406)

该命令用来查询NP卡端口FEC功能配置。

#### [注意事项](#ZH-CN_TOPIC_0250068406)

- 该命令仅适用于NP卡加速模式场景。

#### [操作用户权限](#ZH-CN_TOPIC_0250068406)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_TOPIC_0250068406)

无

#### [使用实例](#ZH-CN_TOPIC_0250068406)

- 查询所有的NP卡端口的FEC功能配置：
  ```
  LST NPFECSWITCH:;
  ```
  ```
  RETCODE = 0  操作成功

  结果如下
  --------
  接口名称     FEC 状态     

  100GE66/0/8  FEC_MOD_RS   
  100GE66/0/9  NP_FEC_NONE  
  100GE66/0/6  FEC_MOD_RS   
  (结果个数 = 3)

  ---    END
  ```

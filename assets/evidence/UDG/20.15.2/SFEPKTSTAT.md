#### [命令功能](#ZH-CN_CONCEPT_0000001550280870)

设置报文统计记录开关。

#### [注意事项](#ZH-CN_CONCEPT_0000001550280870)

该命令仅适用于非NP卡场景和NP卡非加速模式场景。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550280870)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550280870)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| ISENABLE | 报文统计记录开关使能 | 可选必选说明：必选参数<br>参数含义：报文统计记录开关使能。<br>数据来源：本端规划<br>取值范围：布尔类型，输入格式为“TRUE”或者“FALSE”。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001550280870)

软转发报文统计记录使能：

```
SET SFEPKTSTAT: ISENABLE = TRUE;
```

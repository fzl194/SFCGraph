# 查询OSPFv3上次Down的邻居（DSP OSPFV3LASTDOWNNBR）

#### [命令功能](#ZH-CN_CONCEPT_0000001549961850)

该命令用于显示OSPFv3 断开的邻居信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001549961850)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549961850)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549961850)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | OSPFv3进程号 | 可选必选说明：可选参数<br>参数含义：OSPFv3进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549961850)

显示设备OSPFv3进程号为1的所有断开的路由信息：

```
DSP OSPFV3LASTDOWNNBR: PROCID=1;
```

| 输出项名称 | 输出项解释 |

| 接口名称 | 该参数指定本端建立邻居的接口名称。 |
| 实例号 | 该参数指定接口实例号。 |
| 邻居路由器标识 | 该参数指定邻居路由器标识。 |
| 邻居Down的直接原因 | 该参数指定邻居Down的直接原因。 |
| 邻居Down的次要原因 | 该参数指定邻居Down的次要原因。 |
| 邻居Down的时间 | 该参数指定邻居Down的时间点。 |
| 序号 | 该参数指定记录序号。 |

#### [命令功能](#ZH-CN_CONCEPT_0000206297080147)

**适用NF：UPF**

该命令用于查询Diameter字典加载路径。

#### [注意事项](#ZH-CN_CONCEPT_0000206297080147)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000206297080147)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000206297080147)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000206297080147)

当需要查询当前配置的Diameter字典加载路径时：

```
LST UPDIAMDICTPATH:;
```

```

RETCODE = 0  操作成功
Diameter字典加载路径
--------------------
        应用  =  SWM
    字典序号  =  1
字典加载路径  =  EPC标准字典路径
(结果个数 = 1)
---    END
```

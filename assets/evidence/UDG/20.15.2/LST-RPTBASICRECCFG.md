#### [命令功能](#ZH-CN_CONCEPT_0000201961159772)

**适用NF：PGW-U、UPF**

该命令用于查询基础单据上报功能开关。

#### [注意事项](#ZH-CN_CONCEPT_0000201961159772)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000201961159772)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000201961159772)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000201961159772)

加入运营商需要查询基础单据上报功能开关：

```
LST RPTBASICRECCFG:;
```

```

RETCODE = 0 操作成功。

基础单据上报功能开关配置信息
------------------------
用户信息开关 = 不使能（关闭）
用户统计开关 = 不使能（关闭）
 APP统计开关 = 不使能（关闭）
    资源开关 = 不使能（关闭）

--- END
```

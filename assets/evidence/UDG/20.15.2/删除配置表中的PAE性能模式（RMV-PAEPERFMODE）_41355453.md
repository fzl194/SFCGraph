# 删除配置表中的PAE性能模式（RMV PAEPERFMODE）

- [注意事项](#ZH-CN_MMLREF_0000001941355453__1.3.2)

## [命令功能](#ZH-CN_MMLREF_0000001941355453)

该命令用于删除配置表中的PAE性能模式。

> **说明**
> 该命令执行后需要重新启动系统才能生效。

#### [操作用户权限](#ZH-CN_MMLREF_0000001941355453)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001941355453)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PODTYPE | Pod类型 | 可选必选说明：必选参数<br>参数含义：该参数用于指定POD类型，可以通过使用命令<br>[**DSP POD**](../../../操作维护/Pod管理/POD查询（DSP POD）_69830277.md)<br>获取。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~255。<br>默认值：无<br>配置原则：<br>PODTYPE数值来源为DSP POD命令的查询结果。 |

## [使用实例](#ZH-CN_MMLREF_0000001941355453)

删除配置表中的PAE性能模式

```
+++    UNC/*MEID:0 MENAME:unc*/        2024-05-20 18:54:35
O&M    #75
%%RMV PAEPERFMODE: PODTYPE="sfpod";%%
RETCODE = 0  操作成功

---    END
```

## [命令功能](#ZH-CN_MMLREF_0000001761502717)

该命令用于查询是否放通就绪状态检测。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000001761502717)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001761502717)

无

## [使用实例](#ZH-CN_MMLREF_0000001761502717)

查询是否放通就绪状态检测。

```
%%LST KUBEPROBE:;%%
RETCODE = 0  操作成功

结果如下
--------
                  是否检测就绪状态 = 开启
        特定网元下是否检测就绪状态 = 开启
 PSP平台大颗粒服务是否检测就绪状态 = 开启
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001761502717)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 是否检测就绪状态 | 该参数用于表示是否放通就绪状态检测。 |
| 特定网元下是否检测就绪状态 | 该参数用于表示特定网元下是否放通就绪状态检测。 |
| PSP平台大颗粒服务是否检测就绪状态 | 该参数用于表示PSP平台大颗粒服务是否放通就绪状态检测。 |

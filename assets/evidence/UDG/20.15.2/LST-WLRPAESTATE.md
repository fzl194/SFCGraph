#### [命令功能](#ZH-CN_CONCEPT_0000001550281814)

该命令用于查询PAE使能状态。

#### [注意事项](#ZH-CN_CONCEPT_0000001550281814)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550281814)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550281814)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001550281814)

查询PAE使能状态：

```
LST WLRPAESTATE:;
```

```

RETCODE = 0  操作成功

结果如下
-------------------------
PAE使能状态 =  TRUE
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001550281814)

# 删除OSPFv3默认路由宣告配置（RMV OSPFV3DEFAULTROUTE）

- [命令功能](#ZH-CN_CONCEPT_0000001549802410__1.3.1.1)
- [注意事项](#ZH-CN_CONCEPT_0000001549802410__1.3.2.1)
- [操作用户权限](#ZH-CN_CONCEPT_0000001549802410__1.3.3.1)

- [使用实例](#ZH-CN_CONCEPT_0000001549802410__1.3.5.1)

#### [命令功能](#ZH-CN_CONCEPT_0000001549802410)

该命令用于删除OSPFv3默认路由宣告。

#### [注意事项](#ZH-CN_CONCEPT_0000001549802410)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549802410)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549802410)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | OSPFv3进程号 | 可选必选说明：必选参数<br>参数含义：OSPFv3进程号。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |
| TOPOID | 拓扑标识 | 可选必选说明：必选参数<br>参数含义：拓扑标识。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围为0。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549802410)

删除OSPFv3进程1下的默认路由：

```
RMV OSPFV3DEFAULTROUTE:PROCID=1,TOPOID=0;
```

## [操作步骤](#ZH-CN_OPI_0269258754)

3. 配置S11-U接口。
  **[ADD LOGICINF](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/DN管理/逻辑接口管理/逻辑接口配置/接口/增加逻辑接口（ADD LOGICINF）_82835378.md)**

## [任务示例](#ZH-CN_OPI_0269258754)

任务描述

开启基于信令面的数据传输功能。在本例中，使用的S11-U接口地址类型为IPv4，地址为10.2.151.15，掩码为255.255.255.255。

脚本

//打开本特性的License开关。

```
SET LICENSESWITCH:LICITEM="LKV3G5DOVN01", SWITCH=ENABLE;
```

//配置APN。

```
ADD APN: APN="apncp";
```

//配置信令面数据传输使用的S11-U接口。

```
ADD LOGICINF: NAME="s11-uif1/1/0", IPVERSION=IPV4, IPV4ADDRESS1="10.2.151.15", IPV4MASK1="255.255.255.255";
```

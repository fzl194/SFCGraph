## [命令功能](#ZH-CN_MMLREF_0000002148110861)

该命令用于显示Fabric亚健康通信质量信息。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000002148110861)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000002148110861)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| CELLINSTANCE | 微服务实例号 | 可选必选说明：可选参数<br>参数含义：该参数用于指定微服务实例号。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~127。<br>默认值：无<br>配置原则：<br>使用<br>[**DSP PAENODE**](../../服务通信管理/策略查询/显示PAE节点信息（DSP PAENODE）_92520008.md)<br>查看工作角色为数据转发对应的微服务实例号。 |

## [使用实例](#ZH-CN_MMLREF_0000002148110861)

查询Fabric亚健康通信质量。

```
%%DSP FABSUBDETAIL: CELLINSTANCE="sfpod-0__103__0";%%
RETCODE = 0  操作成功

结果如下
```

## [输出结果说明](#ZH-CN_MMLREF_0000002148110861)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 源Cell名称 | 该参数用于表示Fabric平面亚健康源Pod的Cell名称。 |
| 源节点名称 | 该参数用于表示Fabric平面亚健康源Pod部署的节点名称。 |
| 源主机名称 | 该参数用于表示Fabric平面亚健康源Pod部署的主机名称。 |
| 目的Cell名称 | 该参数用于表示Fabric平面亚健康目的Pod的Cell名称。 |
| 目的节点名称 | 该参数用于表示Fabric平面亚健康目的Pod部署的节点名称。 |
| 目的主机名称 | 该参数用于表示Fabric平面亚健康目的Pod部署的主机名称。 |
| 平面组ID | 该参数用于表示Fabric平面亚健康Pod的平面组ID。 |
| 平面ID | 该参数用于表示Fabric平面亚健康Pod的平面ID。 |
| 亚健康发生时间 | 该参数用于表示Fabric平面亚健康发生时间。 |

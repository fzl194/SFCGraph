#### [命令功能](#ZH-CN_CONCEPT_0000206961692892)

**适用NF：CloudEPSN**

该命令用于查询具有API权限和管理员权限的用户名和密码、配置模板信息、最后一次修改这些信息的时间。

#### [注意事项](#ZH-CN_CONCEPT_0000206961692892)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000206961692892)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000206961692892)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000206961692892)

查询具有API权限和管理员权限的用户名和密码、配置模板名称、最后一次修改这些信息的时间：

```
%%LST DNSINFO:;
```

```
%%
RETCODE = 0  操作成功

结果如下
--------
配置模板  =  config_provA
  用户名  =  scale
    密码  =  *****
修改时间  =  2024-02-20 13:41:43
(结果个数 = 1)

---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000206961692892)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 配置模板 | 表示DNS的配置模板名称。 |
| 用户名 | 表示DNS运维界面上具有API权限和管理员权限的用户名。 |
| 密码 | 表示DNS运维界面上具有API权限和管理员权限的用户密码。 |
| 修改时间 | 表示最后一次同步配置信息的时间。 |

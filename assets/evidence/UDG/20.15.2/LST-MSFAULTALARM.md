## [命令功能](#ZH-CN_MMLREF_0209587938)

该命令用于查询告警抑制的配置。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0209587938)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209587938)

无

## [使用实例](#ZH-CN_MMLREF_0209587938)

查询告警抑制使能开关的配置：

```
%%LST MSFAULTALARM:;%%
RETCODE = 0  操作成功

结果如下
------------------------
告警抑制使能开关  =  开启
告警抑制时间(秒)  =  300
(结果个数 = 1)
```

## [输出结果说明](#ZH-CN_MMLREF_0209587938)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 告警抑制使能开关 | 该参数用于表示告警抑制使能开关。 |
| 告警抑制时间(秒) | 该参数用于表示告警抑制时间。 |

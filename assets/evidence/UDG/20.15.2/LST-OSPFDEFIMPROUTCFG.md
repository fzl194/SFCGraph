# 查询OSPF引入路由默认配置（LST OSPFDEFIMPROUTCFG）

#### [命令功能](#ZH-CN_CONCEPT_0000001549801726)

该命令用于查询OSPF引入路由默认配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001549801726)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549801726)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549801726)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | 进程号 | 可选必选说明：可选参数<br>参数含义：OSPF进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549801726)

查询OSPF引入路由默认配置：

```
LST OSPFDEFIMPROUTCFG:;
```

```

RETCODE = 0  操作成功。

结果如下
------------------------
                            进程号  =  1
                          拓扑标识  =  0
                  等价路由最大数量  =  64
                        默认开销值  =  1
引入路由的开销值为路由自带的cost值  =  FALSE
                    外部路由的标记  =  1
                      外部路由类型  =  Type2
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001549801726)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 进程号 | 该参数指定用户创建的OSPF进程号。 |
| 拓扑标识 | 该参数指定用户创建的OSPF拓扑标识。 |
| 等价路由最大数量 | 该参数指定本OSPF进程引入的等价路由最大数量。 |
| 默认开销值 | 该参数指定本OSPF进程引入路由的默认开销值。 |
| 引入路由的开销值为路由自带的cost值 | 如果指定该参数说明引入路由的开销值为路由自带的cost值。如果没有指定开销值，则使用默认开销值。 |
| 外部路由的标记 | 该参数指定本OSPF进程引入路由的标记。 |
| 外部路由类型 | 该参数指定本OSPF进程引入路由的类型。 |

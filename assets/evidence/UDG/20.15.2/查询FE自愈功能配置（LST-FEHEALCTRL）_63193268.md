## [命令功能](#ZH-CN_MMLREF_0000001263193268)

该命令用于查询FE自愈功能配置：包括Mesh网口故障自愈功能开关和FE服务丢失心跳自愈开关。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000001263193268)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001263193268)

无

## [使用实例](#ZH-CN_MMLREF_0000001263193268)

查询FE自愈功能配置

```
LST FEHEALCTRL:;
RETCODE = 0  操作成功

结果如下
--------
Mesh网口故障自愈功能开关  =  使能
  FE服务丢失心跳自愈开关  =  使能
(结果个数 = 1)
```

## [输出结果说明](#ZH-CN_MMLREF_0000001263193268)

| 输出项名称 | 输出项解释 |
| --- | --- |
| Mesh网口故障自愈功能开关 | Mesh网口故障自愈功能开关。 |
| FE服务丢失心跳自愈开关 | FE服务丢失心跳自愈开关。 |

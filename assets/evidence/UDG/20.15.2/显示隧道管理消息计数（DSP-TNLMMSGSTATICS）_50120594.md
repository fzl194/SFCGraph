#### [命令功能](#ZH-CN_CONCEPT_0000001550120594)

该命令用于显示本设备上与隧道管理交互的消息计数。

#### [注意事项](#ZH-CN_CONCEPT_0000001550120594)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550120594)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550120594)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001550120594)

显示本设备上与隧道管理交互的消息计数：

```
DSP TNLMMSGSTATICS:;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001550120594)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 组件ID | 用于表示隧道管理交互的组件PID。 |
| 收到消息的总数 | 用于表示隧道管理收到的消息总数。 |
| 收到丢弃消息数 | 用于表示隧道管理收到的丢弃消息数。 |
| 发送消息总数 | 用于表示隧道管理交互的发送消息总数。 |
| 失败消息数 | 用于表示隧道管理向源组件发送失败消息数。 |
| 重传消息数 | 用于表示隧道管理向源组件重传消息数。 |
| 收到序列号失序的消息数 | 用于表示隧道管理序列号失序的消息数。 |
| 流控消息数 | 用于表示隧道管理流控消息数。 |

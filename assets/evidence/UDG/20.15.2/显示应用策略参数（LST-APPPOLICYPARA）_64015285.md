#### [命令功能](#ZH-CN_CONCEPT_0264015285)

**适用NF：PGW-U、UPF**

该命令用于显示app策略参数。

#### [注意事项](#ZH-CN_CONCEPT_0264015285)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0264015285)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0264015285)

无。

#### [使用实例](#ZH-CN_CONCEPT_0264015285)

查询app策略参数：

```
LST APPPOLICYPARA:;
```

```

RETCODE = 0 操作成功。

应用策略开关
------------------------
黑白名单策略开关 = ENABLE
应用带宽策略开关 = ENABLE
   DNS重定向开关 = DISABLE
(结果个数 = 1)
--- END
```

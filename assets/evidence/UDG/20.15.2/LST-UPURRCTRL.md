#### [命令功能](#ZH-CN_CONCEPT_0000201645582088)

**适用NF：PGW-U、UPF**

查询URR控制参数。

#### [注意事项](#ZH-CN_CONCEPT_0000201645582088)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000201645582088)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000201645582088)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000201645582088)

查询URR控制参数：

```
LST UPURRCTRL:;
```

```

RETCODE = 0  操作成功

URR控制参数
----------------------
                    QHT上报时UPF是否删除URR开关  =  是
                      上报用量为空的URR消息类型  =  上报请求消息
              被聚合的URR用量为空时是否强制上报  =  是
                URR暂停测量期间是否触发费率切换  =  是
是否支持PDR关联未通过PFCP消息创建的预定义URR开关 =  否
(结果数 = 1)
```

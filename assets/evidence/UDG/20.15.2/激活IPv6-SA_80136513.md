## [操作场景](#ZH-CN_OPI_0180136513)

为了保证网络资源或安全， UDG 可以通过对IPv6报文进行七层解析对特定业务进行阻塞，即过滤掉符合配置特征的IPv6报文。

> **说明**
> 适用于PGW-U、UPF。

请仔细阅读 [GWFD-020352 IPv6 SA](../GWFD-020352 IPv6 SA_80136510.md) 。

## [操作步骤](#ZH-CN_OPI_0180136513)

1. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09587387.md) ;
2. 配置阻塞业务三四层过滤条件。
    a. 配置阻塞业务三四层过滤条件。
      [**ADD FILTERIPV6**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/过滤器/增加IPv6过滤器（ADD FILTERIPV6）_82837350.md)
    b. 配置流过滤器。
      [**ADD FLOWFILTER**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/流过滤器/增加流过滤器（ADD FLOWFILTER）_82837361.md)
    c. 绑定三四层过滤信息。
      [**ADD FLTBINDFLOWF**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/流过滤器的过滤器绑定/增加流过滤器的过滤器绑定关系（ADD FLTBINDFLOWF）_82837366.md)
    d. 将新配置的Filter置为生效。
      [**SET REFRESHSRV**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/业务刷新/业务刷新（SET REFRESHSRV）_82837355.md)
3. 配置触发阻塞的七层过滤条件。
    a. 配置七层过滤器。
      [**ADD L7FILTER**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/七层规则管理/七层过滤器/增加七层过滤器（ADD L7FILTER）_82837397.md)
    b. 绑定七层过滤信息。
      [**ADD PROTBINDFLOWF**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/流过滤器的协议绑定/增加流过滤器协议绑定关系（ADD PROTBINDFLOWF）_82837370.md)
    c. 将新配置的L7Filter置为生效。
      [**SET REFRESHSRV**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/业务刷新/业务刷新（SET REFRESHSRV）_82837355.md)
      > **说明**
      > 系统自动在60s后将配置的L7Filter置为生效。如果需要将新配置的L7Filter立即生效，可执行本步骤。
4. 配置阻塞业务的动作策略。
  [**ADD PCCACTIONPROP**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/PCC控制策略/PCC动作属性/增加PCC动作属性（ADD PCCACTIONPROP）_82837601.md)
5. 配置阻塞业务使用的PCC策略组。
    a. 配置PCC策略组。
      [**ADD PCCPOLICYGRP**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/PCC控制策略/PCC策略组/增加PCC策略组（ADD PCCPOLICYGRP）_82837606.md)
    b. 配置业务属性。
      [**ADD SERVICEPROP**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/业务控制公共配置/业务属性/增加业务属性（ADD SERVICEPROP）_82837590.md)
    c. 绑定阻塞业务的动作策略。
      [**ADD SRVPBINDPCCPG**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/PCC控制策略/PCC策略组业务属性绑定/增加PCC组业务属性绑定（ADD SRVPBINDPCCPG）_82837611.md)
6. 配置阻塞业务规则。
  [**ADD RULE**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务匹配公共配置/业务规则管理/规则/增加规则（ADD RULE）_82837266.md)
  > **说明**
  > 如果运营商希望基于流过滤器组设置过滤条件时，配置 “流过滤器或流过滤器组选择(FILTERINGMODE)” 为 “FLOWFILTERGRP” ，并且使用 [**ADD FLOWFILTERGRP**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/流过滤器组/增加流过滤器组（ADD FLOWFILTERGRP）_82837384.md) 命令配置流过滤器组。
7. 配置阻塞业务策略组合。
    a. 创建UserProfile。
      [**ADD USERPROFILE**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务匹配公共配置/业务规则管理/用户模板/增加用户模板（ADD USERPROFILE）_82837279.md)
    b. 将已配置的Rule绑定到UserProfile。
      [**ADD RULEBINDING**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务匹配公共配置/业务规则管理/规则绑定/增加用户模板和规则的绑定关系（ADD RULEBINDING）_82837272.md)

## [任务示例](#ZH-CN_OPI_0180136513)

任务描述

本实例中，操作员在 UDG 上进行数据配置实现以下要求：

- UDG将IPv6用户发起的到“www.example.com”服务器的上行报文全部丢弃。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH:LICITEM ="LKV3G5SAV601",SWITCH=ENABLE;
```

//配置阻塞业务三四层过滤条件。

```
ADD FILTERIPV6:FILTERNAME="filter_test",L34PROTTYPEV6=STRING,L34PROTOCOL=TCP,SVRIPMODE=IP,SVRIPV6="FC00::6",SVRIPV6MASKTYPE=LENGTHTYPE,SVRIPV6MASKLEN=64;
```

```
ADD FLOWFILTER:FLOWFILTERNAME="flow-filter1";
```

```
ADD FLTBINDFLOWF:FLOWFILTERNAME="flow-filter1",FILTERNAME="filter_test";
```

```
SET REFRESHSRV:REFRESHTYPE=USERPROFILE;
```

//配置七层协议识别数据。

```
LOD SIGNATUREDB:LOADMODE=LATEST;
```

//配置触发阻塞的七层过滤条件。

```
ADD L7FILTER:L7FILTERNAME="l7_test",SUBL7FLTNAME="sl7-test",URL="www.example.com*/*";
```

```
ADD PROTBINDFLOWF:FLOWFILTERNAME="flow-filter1",PROTOCOLNAME="http",L7FILTERNAME="l7_test";
SET REFRESHSRV:REFRESHTYPE=USERPROFILE;
```

//配置阻塞业务的动作策略组。 **当需要计费时，ADD PCCPOLICYGRP命令请配置参数URRGROUPNAME** 。

```
ADD PCCACTIONPROP:PCCACTPROPNAME="ap_test",UPINITUPGATE=DISCARD,UPINITDNGATE=DISCARD;
```

```
ADD PCCPOLICYGRP:PCCPOLICYGRPNM="pg_test",URRGROUPNAME="urrgp_test";
```

```
ADD SERVICEPROP:SRVPROPNAME="sp_test",SERVICEID=1,PRIORITY=1;
```

```
ADD SRVPBINDPCCPG:PCCPOLICYGRPNM="pg_test",SRVPROPNAME="sp_test",PCCACTPROPNAME="ap_test";
```

//配置阻塞业务规则。

```
ADD RULE:RULENAME="rule_test",POLICYTYPE=PCC,FILTERINGMODE=FLOWFILTER,FLOWFILTERNAME="flow-filter1",PRIORITY=1,POLICYNAME="pg_test";
```

//配置阻塞业务策略组合。

```
ADD USERPROFILE:USERPROFILENAME="up_test";
```

```
ADD RULEBINDING:USERPROFILENAME="up_test",RULENAME="rule_test";
```

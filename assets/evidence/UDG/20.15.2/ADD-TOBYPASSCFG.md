#### [命令功能](#ZH-CN_CONCEPT_0000202314369777)

**适用NF：UPF**

该命令用于设置旁路Pod配置。

#### [注意事项](#ZH-CN_CONCEPT_0000202314369777)

- 该命令执行后对新数据流生效。
- 该命令最大记录数为64。

#### [操作用户权限](#ZH-CN_CONCEPT_0000202314369777)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000202314369777)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PODNAME | Pod名称 | 可选必选说明：必选参数<br>参数含义：Pod名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，区分大小写。<br>默认值：无<br>配置原则：无 |
| MODE | 旁路模式 | 可选必选说明：必选参数<br>参数含义：设置该Pod的Bypass模式。<br>数据来源：本端规划<br>取值范围：无。<br>- mode1：所有的流量都不上送to pod。<br>- mode2：所有的流量都上送to pod，但是不上送内核。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000202314369777)

增加名称为“to-pod-0”的pod旁路模式为mode1模式：

```
ADD TOBYPASSCFG: PODNAME="to-pod-0", MODE=mode1;
```

#### [参考信息](#ZH-CN_CONCEPT_0000202314369777)

可使用如下命令获取c16规格的Pod名称：DSP POD: TYPE=byType, PODTYPE="to-pod";。

可使用如下命令获取c56规格的Pod名称：DSP POD: TYPE=byType, PODTYPE="toc56-pod";。

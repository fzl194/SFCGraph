#### [命令功能](#ZH-CN_CONCEPT_0000001600841245)

该命令用于显示LDP的邻居信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600841245)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600841245)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600841245)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| VRFNAME | VPN实例名称 | 可选必选说明：可选参数<br>参数含义：该参数用于表示VPN实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。<br>默认值：_public_ |
| IFNAME | 接口名称 | 可选必选说明：可选参数<br>参数含义：该参数用于表示LDP邻居的接口名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。<br>默认值：无<br>配置原则：请使用LST INTERFACE命令查看可用接口。 |
| PEERID | 对等体的LSR ID | 可选必选说明：可选参数<br>参数含义：该参数用于表示LDP对等体的LSR ID。<br>数据来源：对端协商<br>取值范围：IPv4地址类型。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600841245)

显示LDP的邻居信息：

```
DSP LDPNEMGADJ:;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600841245)

| 输出项名称 | 输出项解释 |
| --- | --- |
| VPN实例名称 | 用于表示VPN实例名称。 |
| 对等体的LSR ID | 用于表示LDP对等体的LSR ID。 |
| 实体编号 | 用于表示LDP邻居实体整数形式的编号。 |
| 状态 | 该参数用于表示LDP邻居状态的标识，包括：<br>- 空闲状态。<br>- LDP邻居处于稳态。 |
| 类型 | 该参数用于表示LDP邻居类型的标识，包括：<br>- 本地邻居。<br>- 远端邻居。 |
| 源地址 | 用于表示LDP邻居的源地址。 |
| 接口索引值 | 用于表示LDP邻居的接口索引值。 |
| 接口名称 | 用于表示LDP邻居的接口名称。 |
| 传输地址 | 用于表示LDP建立LDP会话使用的传输地址。 |
| 对等体Hello保持时间（s） | 用于表示LDP对等体的Hello保持时间（s）。 |
| 本端Hello保持时间（s） | 用于表示LDP本端Hello保持时间（s）。 |
| 备份版本号 | 用于表示LDP邻居备份版本号。 |
| 平滑版本号 | 用于表示LDP邻居平滑版本号。 |
| 调度状态 | 该参数用于表示LDP对等体发布的调度状态标识，包括：<br>- 完成发布或未发布。<br>- 等待发布对等体。<br>- 正在发布对等体。<br>- 还有对等体要发布。 |

# 从N2TAC组内删除一个TAC（RMV N2TACID）

- [命令功能](#ZH-CN_CONCEPT_0197358682__1.3.1.1)
- [注意事项](#ZH-CN_CONCEPT_0197358682__1.3.2.1)
- [操作用户权限](#ZH-CN_CONCEPT_0197358682__1.3.3.1)
- [参数说明](#ZH-CN_CONCEPT_0197358682__1.3.4.1)
- [使用实例](#ZH-CN_CONCEPT_0197358682__1.3.5.1)

#### [命令功能](#ZH-CN_CONCEPT_0197358682)

**适用NF：PGW-U、UPF**

该命令用来删除N2TAC组内绑定的N2TAC号段。

#### [注意事项](#ZH-CN_CONCEPT_0197358682)

该命令执行后只对新激活用户生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0197358682)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0197358682)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| TACGROUPNAME | 指定N2TAC组名 | 可选必选说明：必选参数<br>参数含义：该参数用于指定N2TAC组名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～32。不支持空格，不区分大小写。<br>默认值：无<br>配置原则：该参数使用ADD TACGROUP命令配置生成。 |
| TACSECNUM | N2Tac 段编号 | 可选必选说明：可选参数<br>参数含义：该参数用于指定TAC段编号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～15999。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0197358682)

假设运营商需要在一个本地已经配置的N2TAC组删除一个N2TAC号段：

```
RMV N2TACID:TACGROUPNAME="beijing",TACSECNUM=2;
```

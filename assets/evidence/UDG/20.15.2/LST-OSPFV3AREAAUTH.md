#### [命令功能](#ZH-CN_CONCEPT_0000001549802374)

该命令用于查询OSPFv3区域认证配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001549802374)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549802374)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549802374)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | OSPFv3进程号 | 可选必选说明：可选参数<br>参数含义：OSPFv3进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |
| AREAID | OSPFv3区域号 | 可选必选说明：可选参数<br>参数含义：OSPFv3区域号。<br>数据来源：本端规划<br>取值范围：IPv4地址类型。点分十进制格式。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549802374)

查询OSPFv3区域认证配置：

```
LST OSPFV3AREAAUTH:;
```

```

RETCODE = 0  操作成功。

结果如下
------------------------
    OSPFv3进程号  =  1
    OSPFv3区域号  =  0.0.0.0
        认证模式  =  HMAC-SHA256
    认证密码类型  =  Cipher
    密文验证密码  =  *****
密文验证字标识符  =  1000
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001549802374)

| 输出项名称 | 输出项解释 |
| --- | --- |
| OSPFv3进程号 | 该参数指定用户创建的OSPFv3进程号。 |
| OSPFv3区域号 | 该参数指定用户创建的OSPFv3区域号。 |
| 认证模式 | 该参数指定OSPFv3认证模式，OSPFv3支持HMAC-SHA256密文验证模式。 |
| 认证密码类型 | 该参数指定OSPFv3认证模式认证密码类型，OSPFv3支持cipher类型。 |
| 密文验证密码 | 该参数指定密文验证密码。 |
| 密文验证字标识符 | 该参数指定密文验证字标识符，标识符需要跟对端保持一致。 |

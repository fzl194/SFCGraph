## [命令功能](#ZH-CN_MMLREF_0000001184132100)

本命令用于查询HTTP连接信息上报的控制参数信息。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000001184132100)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001184132100)

无

## [使用实例](#ZH-CN_MMLREF_0000001184132100)

如果想查询HTTP连接信息上报的控制参数，可以用如下命令：

```
%%LST HTTPINFORPT:;%%
RETCODE = 0  操作成功

结果如下
--------
          HTTP连接信息上报列表  =  HTTP动态头压缩字典上报
HTTP动态头域压缩字典上报定时器  =  60
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001184132100)

| 输出项名称 | 输出项解释 |
| --- | --- |
| HTTP连接信息上报列表 | 本参数用于指定需要上报的HTTP连接信息。 |
| HTTP动态头域压缩字典上报定时器 (s) | 本参数用于配置HTTP动态头域压缩字典信息的周期上报定时器。 |

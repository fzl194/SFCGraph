#### [命令功能](#ZH-CN_CONCEPT_0182837104)

**适用NF：SGW-U、PGW-U、UPF**

该命令用于查询白名单中的UE信息。

#### [注意事项](#ZH-CN_CONCEPT_0182837104)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0182837104)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0182837104)

无。

#### [使用实例](#ZH-CN_CONCEPT_0182837104)

查询UE下行灌包功能的白名单：

```
LST UEINJECTLIST:;
```

```

RETCODE = 0  操作成功。

用户ID信息  =  12345678901234
(结果个数 = 1)

```

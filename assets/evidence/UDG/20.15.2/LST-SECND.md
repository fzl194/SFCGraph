#### [命令功能](#ZH-CN_CONCEPT_0000001600441197)

该命令用来查询ND快回使能配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001600441197)

- 该命令执行后立即生效。
- 该命令仅适用于非NP卡场景和NP卡非加速模式场景。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600441197)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600441197)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001600441197)

查询ND快回使能配置：

```
LST SECND:;
```

```

RETCODE = 0  操作成功

结果如下
-------------------------
使能标记    是
(结果个数 = 1)
---    END
```

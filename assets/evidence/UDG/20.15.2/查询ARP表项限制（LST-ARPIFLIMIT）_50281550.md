#### [命令功能](#ZH-CN_CONCEPT_0000001550281550)

该命令用于查询接口的表项限制配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001550281550)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550281550)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550281550)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IFNAME | 接口名称 | 可选必选说明：可选参数<br>参数含义：该参数用于需要设置表项限制的接口名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。以太网接口名称由接口类型和接口编号组成。<br>默认值：无<br>配置原则：请使用LST INTERFACE命令查看可用接口。 |
| VLANID | VLAN ID值 | 可选必选说明：可选参数<br>参数含义：接口所属的VLAN ID值。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～4094。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001550281550)

查询Ethernet64/0/5的表项限制：

```
LST ARPIFLIMIT:IFNAME="Ethernet64/0/5";
```

```

RETCODE = 0  操作成功

结果如下
-------------------------

     VLAN ID值  =  0
      限制数量  =  20
学习到表项数量  =  0
(结果个数 = 1)
---    END
```

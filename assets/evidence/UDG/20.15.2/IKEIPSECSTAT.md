## [命令功能](#ZH-CN_MMLREF_0000001197874058)

该命令用于显示IKE IPsec统计信息。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000001197874058)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001197874058)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PODNAME | Pod名称 | 可选必选说明：可选参数<br>参数含义：该参数表示Pod名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~63。<br>默认值：无<br>配置原则：无 |

## [输出结果说明](#ZH-CN_MMLREF_0000001197874058)

| 输出项名称 | 输出项解释 |
| --- | --- |
| Pod名称 | 该参数表示Pod名称。 |
| SA首次建立的AH或ESP数目 | 该参数表示SA首次建立的IP认证头协议AH(Authentication Header)或IP封装安全载荷协议ESP(Encapsulating Security Payload)数目。 |
| SA首次建立的AH和ESP数目 | 该参数表示SA首次建立的IP AH和IP ESP数目。 |
| 重新申请密钥或重认证后SA更新的AH或ESP数目 | 该参数表示重新申请密钥或重认证后SA更新的IP AH或IP ESP数目。 |
| 重新申请密钥或重认证后SA更新的AH和ESP数目 | 该参数表示重新申请密钥或重认证后SA更新的IP AH和IP ESP数目。 |
| 第一次软过期数 | 该参数表示第一次软过期数(重新申请密钥的70%已经配置)。 |
| 第二次软过期数 | 该参数表示第二次软过期数(重新申请密钥的80%已经配置)。 |
| 第三次软过期数 | 该参数表示第三次软过期数(重新申请密钥的90%已经配置)。 |
| 硬过期数 | 该参数表示硬过期数(重新申请密钥的100%已经配置)。 |
| 删掉的SA数 | 该参数表示删掉的SA数目(包括由于清除SA API、重新申请密钥和重认证而导致的新老SA)。 |
| 删掉的新SA数 | 该参数表示删掉的新SA数目。 |
| 一秒内删掉的旧SA数 | 该参数表示一秒钟内删掉的旧SA数目。这是一秒钟内的提醒事件，若老的SA被删掉了而数据包可能丢弃，则这个事件就并不是期待的行为了。 |
| 一秒后删掉的旧SA数 | 该参数表示一秒钟后删掉的旧SA数目。这是一秒钟后的提醒事件，并且是期待的行为。 |
| 保留的入IPSec隧道描述字模块数目 | 该参数表示保留的入IPSec隧道描述字模块TDB(Tunnel Descriptor Block)数目。 |
| 到达的入DHCP包数目 | 该参数表示到达的入动态主机配置协议DHCP包数目。 |
| 到达的出DHCP包数目 | 该参数表示到达的出动态主机配置协议DHCP包数目。 |

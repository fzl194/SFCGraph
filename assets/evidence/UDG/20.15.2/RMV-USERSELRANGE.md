#### [命令功能](#ZH-CN_CONCEPT_0286133376)

**适用NF：UPF**

该命令用来删除用户选择范围的属性配置，应用于该策略选定用户的性能统计。

#### [注意事项](#ZH-CN_CONCEPT_0286133376)

该命令执行后只对新激活用户生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0286133376)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0286133376)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| NAME | 用户选择范围名称 | 可选必选说明：必选参数<br>参数含义：用户选择范围集合配置名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63，单位是字节。只能由“_”、“-”、数字、大小写字母和“.”组成，不能以“.”开头且不能出现连续两个“.”。不支持空格及“#”、“$”、“&”、“%”、“^”、“（”、“）”、“，”、“/”、“;”、“:”、“””、“`”等特殊字符，不区分大小写。<br>默认值：无<br>配置原则：name必须是通过ADD USERSELRANGE命令添加。 |
| RANGETYPE | 用户标识类型 | 可选必选说明：可选参数<br>参数含义：该参数用于设置策略类型。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- APN：范围类型为APN。<br>- SNSSAI：范围类型为切片。<br>默认值：无<br>配置原则：无 |
| APN | APN名称 | 可选必选说明：条件必选参数<br>前提条件：该参数在“RANGETYPE”配置为“APN”时为必选参数。<br>参数含义：APN名称信息。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63，单位是字节。只能由“-”、数字、大小写字母和“.”组成，不能以“.”开头且不能出现连续两个“.”。不支持空格及“_”、“#”、“$”、“&”、“%”、“^”、“（”、“）”、“，”、“/”、“;”、“:”、“””、“`”等特殊字符，不区分大小写。<br>默认值：无<br>配置原则：该参数使用ADD APN命令配置生成。 |
| SNSSAI | 网络切片选择标识 | 可选必选说明：条件必选参数<br>前提条件：该参数在“RANGETYPE”配置为“SNSSAI”时为必选参数。<br>参数含义：网络切片选择标识名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～6。<br>默认值：无<br>配置原则：字符串类型，输入长度范围为1～6。对于同一用户选择列表名称，最大可配置10个切片。 |

#### [使用实例](#ZH-CN_CONCEPT_0286133376)

当运营商需要删除一个用户策略配置：

```
RMV USERSELRANGE: NAME="test";
```

## [操作场景](#ZH-CN_OPI_0000001151849611)

运营商通过综合业务保障级别、业务签约及用户级别等信息规划业务流优先级，需要激活基于业务流标识的无线资源优化功能。

> **说明**
> 适用于PGW-U、UPF。

## [必备事项](#ZH-CN_OPI_0000001151849611)

前提条件

- 请仔细阅读 [GWFD-110331 基于业务流标识的无线资源优化](../GWFD-110331 基于业务流标识的无线资源优化_51769519.md) 。
- 完成 **加载License** （如果有需求，请联系华为技术支持工程师处理）。

## [操作步骤](#ZH-CN_OPI_0000001151849611)

1. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09587387.md)
2. 配置FPI使能开关，选择资源优化方法。
  **[SET FPIFUNC](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/无线资源优化/FPI/设置FPI差异化控制功能（SET FPIFUNC）_25006876.md)**
3. 配置三四层过滤规则。
    a. 配置业务使用的三四层过滤规则为TCP项。
      [**ADD FILTER**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/过滤器/增加过滤器（ADD FILTER）_82837348.md)
    b. 配置流过滤器。
      [**ADD FLOWFILTER**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/流过滤器/增加流过滤器（ADD FLOWFILTER）_82837361.md)
    c. 流过滤器绑定过滤器。
      [**ADD FLTBINDFLOWF**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/流过滤器的过滤器绑定/增加流过滤器的过滤器绑定关系（ADD FLTBINDFLOWF）_82837366.md)
    d. 将新配置的Filter置为生效。
      [**SET REFRESHSRV**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/业务刷新/业务刷新（SET REFRESHSRV）_82837355.md)
4. 配置PCC动作属性。
  [**ADD PCCACTIONPROP**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/PCC控制策略/PCC动作属性/增加PCC动作属性（ADD PCCACTIONPROP）_82837601.md)
5. 配置PCC策略组。
  [**ADD PCCPOLICYGRP**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/PCC控制策略/PCC策略组/增加PCC策略组（ADD PCCPOLICYGRP）_82837606.md)
6. 配置业务规则。
  [**ADD RULE**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务匹配公共配置/业务规则管理/规则/增加规则（ADD RULE）_82837266.md)
  > **说明**
  > 如果运营商希望基于流过滤器组设置过滤条件时，配置 “流过滤器或流过滤器组选择(FILTERINGMODE)” 为 “FLOWFILTERGRP” ，并且使用 [**ADD FLOWFILTERGRP**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/流过滤器组/增加流过滤器组（ADD FLOWFILTERGRP）_82837384.md) 命令配置流过滤器组。

## [任务示例](#ZH-CN_OPI_0000001151849611)

任务描述

在 UDG 上使能基于业务流标识的无线资源优化功能，FPI信息以DSCP方式向RAN侧传递。

脚本

1. 打开本特性的License配置开关。
  ```
  SET LICENSESWITCH:LICITEM="LKV3G5WOFR01",SWITCH=ENABLE;
  ```
2. 配置FPI使能开关，选择资源优化方法。
  ```
  SET FPIFUNC:FPISWITCH=ENABLE,FPITRANSMETHOD=DSCP;
  ```
3. 配置三四层过滤规则。
  ```
  ADD FILTER:FILTERNAME="filter_test",L34PROTTYPE=STRING,L34PROTOCOL=TCP;
  ```
  ```
  ADD FLOWFILTER:FLOWFILTERNAME="ff_test";
  ```
  ```
  ADD FLTBINDFLOWF:FLOWFILTERNAME="ff_test",FILTERNAME="filter_test";
  ```
  ```
  SET REFRESHSRV:REFRESHTYPE=ALL;
  ```
4. 配置业务动作属性。
  ```
  ADD PCCACTIONPROP:PCCACTPROPNAME="pap_test",UPINITDNGATE=PASS;
  ```
5. 配置PCC策略组。当需要计费时，请配置参数URRGROUPNAME。
  ```
  ADD PCCPOLICYGRP:PCCPOLICYGRPNM="ppg_test",PCCACTPROPNAME="pap_test",
  URRGROUPNAME
  ="urrgp_test";
  ```
6. 配置业务规则。
  ```
  ADD RULE:RULENAME="rule_test1",POLICYTYPE=PCC,
  FILTERINGMODE=FLOWFILTER,
   FLOWFILTERNAME="ff_test",PRIORITY=1,POLICYNAME="ppg_test";
  ```
  ```
  ADD RULE:RULENAME="rule_test2",POLICYTYPE=REMARK_FPI,
  FILTERINGMODE=FLOWFILTER,
   FLOWFILTERNAME="ff_test",PRIORITY=2,REMARKFPISEL=FPI,FPIVALUE=63;
  ```

#### [命令功能](#ZH-CN_CONCEPT_0182837790)

**适用NF：PGW-U、UPF**

该命令用于查询当前防欺诈功能配置的可信代理服务器信息。

#### [注意事项](#ZH-CN_CONCEPT_0182837790)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0182837790)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0182837790)

无。

#### [使用实例](#ZH-CN_CONCEPT_0182837790)

查询所有可信代理服务器IP地址：

```
LST AFPROXYSERVER:;
```

```

RETCODE = 0  操作成功。

防欺诈代理服务器信息
--------------------
  IP地址版本类型  =  IPV4
代理服务器IP地址  =  192.168.0.2
(结果个数 = 1)
---    END
```

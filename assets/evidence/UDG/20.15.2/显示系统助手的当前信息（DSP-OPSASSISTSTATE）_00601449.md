#### [命令功能](#ZH-CN_CONCEPT_0000001600601449)

该命令用于显示系统助手的当前信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600601449)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600601449)

G_1，管理员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600601449)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001600601449)

显示系统助手的当前信息：

```
DSP OPSASSISTSTATE:;
```

```

 RETCODE = 0  操作成功。

结果如下
------------------------
      助手ID = 2
    助手名称 = _get_all_env.py
    助手状态 = 准备好状态
触发条件类型 = 多条件组合
    助手类型 = 脚本
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600601449)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 助手ID | 助手ID。 |
| 助手名称 | 助手名称。 |
| 助手状态 | 助手状态。 初始状态: 助手初始状态，等待助手注册，注册后进入准备状态。 准备好状态: 助手准备好状态，助手已完成注册，等待订阅事件被触发。 等待状态: 助手等待状态，助手正在排队待处理，被调度到则进入运行状态。 运行状态: 助手运行状态，助手脚本正在运行。 关闭状态: 助手关闭状态，当前状态下订阅事件不能被触发，助手可以再次被启动。 悬挂状态: 助手悬挂状态，不响应订阅事件触发，悬挂状态可以被取消。 |
| 触发条件类型 | 触发条件类型。 无: 无订阅。 时间: 时间订阅。 事件: 事件订阅。 启动: 启动订阅。 告警: 告警订阅。 用户日志: 用户日志订阅。 SNMP通知上报: SNMP通知订阅。 自动订阅: 自动触发订阅。 命令订阅: 命令触发订阅。 相对定时器: 相对定时器订阅。 绝对定时器: 绝对定时器订阅。 倒数计时定时器: 倒数计时定时器订阅。 多条件组合: 多条件组合订阅。 订阅事件: 触发事件订阅。 去订阅事件: 去除事件订阅。 |
| 助手类型 | 助手类型。 命令: 命令助手。 脚本: 脚本助手。 |

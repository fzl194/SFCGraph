## [命令功能](#ZH-CN_MMLREF_0000001175539859)

该命令用于查询网元的话统测量结果。

> **说明**
> 网元升级后未结束升级观察期期间，支持查询高低版本话统结果；结束升级观察期后，普通域仅支持查询高版本话统结果。
>
> 当不填写测量单元ID或测量单元名称时，该命令执行时长可能会很长，建议填写这些参数，减少执行时长，当参数过少时可能会执行失败。

## [参数说明](#ZH-CN_MMLREF_0000001175539859)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| MEID | 网元ID | 可选必选说明：必选参数。<br>参数含义：标识网元ID，可以通过<br>[**LST ME**](../../系统管理/版本信息/查询网元配置信息（LST ME）_47084797.md)<br>命令查询获取。<br>取值范围：整数类型，取值范围为0~65535。<br>默认值：无。<br>配置原则：无。 |
| MOCID | 测量对象类ID | 可选必选说明：可选参数。<br>参数含义：测量对象类ID，可以通过<br>[**LST MEASUNIT**](查询测量指标模型(LST MEASUNIT)_47814449.md)<br>命令查询获取。<br>取值范围：整数类型，取值范围为0~4294967294。<br>默认值：无。<br>配置原则：无。 |
| MUID | 测量单元ID | 可选必选说明：可选参数。<br>参数含义：测量单元ID，可以通过<br>[**LST MEASUNIT**](查询测量指标模型(LST MEASUNIT)_47814449.md)<br>命令查询获取。<br>取值范围：整数类型，取值范围为0~4294967294。<br>默认值：无。<br>配置原则：若不输入，则表示该参数不作为查询的限制条件。 |
| MUNAME | 测量单元名称 | 可选必选说明：可选参数。<br>参数含义：测量单元名称，支持中英文模糊搜索。<br>取值范围：字符串类型，长度不超过256个字符。<br>默认值：无。<br>配置原则：若不输入，则表示该参数不作为查询的限制条件。 |
| METRICID | 测量指标ID | 可选必选说明：可选参数。<br>参数含义：测量指标ID，可以通过<br>[**LST MEASUNIT**](查询测量指标模型(LST MEASUNIT)_47814449.md)<br>命令查询获取。<br>取值范围：字符串类型，长度不超过256个字符。<br>默认值：无。<br>配置原则：支持多个测量指标ID，以&字符分割，不能以&结尾。例如 1&2&3。 |
| MOIID | 对象实例ID | 可选必选说明：可选参数。<br>参数含义：对象实例ID，可以通过<br>[**DSP MEASOBJ**](查询话统测量对象实例(DSP MEASOBJ)_01094728.md)<br>命令查询获取。<br>取值范围：字符串类型，长度不超过256个字符。<br>默认值：无。<br>配置原则：无。 |
| PRD | 测量周期 | 可选必选说明：可选参数。<br>参数含义：标识查询的周期。<br>取值范围：<br>- ALL(全部)<br>- FIVE(5分钟)<br>- FIFTEEN(15分钟)<br>- THIRTY(30分钟)<br>- SIXTY(1小时)<br>- ONE_DAY(一天)<br>默认值：无。<br>配置原则：无。 |
| SDT | 查询起始时间 | 可选必选说明：可选参数。<br>参数含义：查询起始时间。<br>取值范围：年月日时分秒，规则为YYYY/MM/DD HH:MM:SS。<br>默认值：客户端当前时间。<br>配置原则：若不输入，则把查询结束时间7天之前的时间当做查询开始时间。 |
| EDT | 查询结束时间 | 可选必选说明：可选参数。<br>参数含义：查询结束时间。<br>取值范围：年月日时分秒，规则为YYYY/MM/DD HH:MM:SS。<br>默认值：客户端当前时间。<br>配置原则：若不输入，则把当前时间当做查询结束时间。 |

## [输出结果说明](#ZH-CN_MMLREF_0000001175539859)

命令执行正常，会返回命令执行成功的提示信息，输出项如 [表1 输出项说明](#ZH-CN_MMLREF_0000001175539859__table7632103012435) 所示。

命令执行异常，请联系华为技术支持处理。输出错误码说明如 [表2](#ZH-CN_MMLREF_0000001175539859__table16741512233) 所示。

*表1 输出项说明*

| **输出项名称** | **输出项解释** |
| --- | --- |
| 开始时间 | 测量结果的开始时间。 |
| 周期(秒) | 测量结果的测量周期。 |
| 测量对象类名称 | 测量对象类名称。 |
| 对象实例ID | 测量对象实例ID。 |
| 对象实例名称 | 测量对象实例名称，如未定义对象枚举，则不呈现此列或呈现为NULL。 |
| 测量单元名称 | 测量单元名称。 |
| 测量指标名称 | 测量指标名称。 |
| 测量指标ID | 测量指标ID。 |
| 指标值 | 测量结果的测量指标值。 |
| 指标单位名称 | 指标单位名称。 |
| 提示信息 | 帮助理解输出结果的提示信息。 |
| 存储时间 | 指标值的存入数据库的时间。服务存储故障状态下，存储时间为 - 。此时间用于辅助分析，历史数据的存储时间不随时区变化而变化。 |
| 可信度 | 该话统结果是否可信，取值范围：<br>- 可信<br>- 不可信 |
| 不可信原因 | 该话统结果不可信的原因。输出格式为“数字（文字）”：数字代表网元内部不可信原因标识，文字代表北向上报的不可信原因，其中文字为可选项，取值范围：<br>- 新增任务：新增任务，客户端首个测量周期数据不全。<br>- 新增对象：新增对象，客户端首个测量周期数据不全。<br>- 服务启动：服务启动，客户端首个测量周期数据不全。<br>- 时间跳变：客户端时间跳变，测量周期数据不全。<br>- 客户端计算溢出：指标在客户端的计算结果超过指标类型定义的最大值。<br>- 实例不全 ：上报指标的业务实例个数少于注册实例个数。<br>- 复合周期不全：在服务端进行复合周期计算的原始测量数据不全。<br>- 服务端计算溢出：指标在服务端的计算结果超过指标类型定义的最大值。<br>- 指标不全：上报指标个数与模型不一致或新增指标测量。<br>仅数字无文字：表示测量对象所属域为“大颗粒”的不可信，不可信原因未收编，具体请联系华为技术支持 。 |

# 调测计费信息实时提醒

- [操作场景](#ZH-CN_TOPIC_0164110506__1.3.1.1)
- [必备事项](#ZH-CN_TOPIC_0164110506__1.3.2.1)
- [操作步骤](#ZH-CN_TOPIC_0164110506__1.3.3.1)

#### [操作场景](#ZH-CN_TOPIC_0164110506)

当运营商部署了计费信息实时提醒业务时，需要调测该功能，确保该功能可正常使用。

> **说明**
> 适用于SGW-U、PGW-U、UPF。

#### [必备事项](#ZH-CN_TOPIC_0164110506)

**前提条件**

- 请仔细阅读[GWFD-020356 计费信息实时提醒](../GWFD-020356 计费信息实时提醒_67644325.md)。

- 完成[激活计费信息实时提醒](激活计费信息实时提醒_64110489.md)。
- 已经安装了第三方抓包工具。

**数据**

| 类别 | 参数名称 | 取值样例 | 获取方法 | 说明 |
| --- | --- | --- | --- | --- |
| 用户信息查询 | IMSI | 460043004307442 | 测试终端自带 | - |
| 测试终端使用的APN | APN名称（APN） | apn-test | 已配置数据中获取 | 取自已配置的APN实例名。 |
| 欢迎页面提醒、流量提醒、欠费充值提醒使用的URL | URL | www.huawei.com | 已配置数据中获取 | 取自激活计费信息实时提醒中配置的页面实时提醒URL白名单。 |

**工具**

- 测试终端
- 第三方抓包工具

#### [操作步骤](#ZH-CN_TOPIC_0164110506)

1. 执行 [**LST LICENSESWITCH**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09587387.md) 命令查询 UDG 上计费信息实时提醒对应的License配置开关是否打开。
  ```
  LST LICENSESWITCH:LICITEM="LKV3G5RCIN01";
  ```
    - 若回显中“Switch”为“ENABLE”，请执行[步骤2](#ZH-CN_TOPIC_0164110506__step1)。
    - 若回显中“Switch”为“DISABLE”，请开启License后继续调测本功能。
2. 打开抓包工具，准备抓取测试终端的出入报文。
3. 测试终端使用APN“apn-test”接入网络。
    - 如果测试终端成功接入网络，请执行[步骤5](#ZH-CN_TOPIC_0164110506__step4)。
    - 如果测试终端无法接入网络，请调测UDG的接入功能。
4. **调测欢迎页面提醒功能。**
    a. 在测试终端上打开web浏览器，使用URL方式访问www.huawei.com网站。检查用户终端上是否显示欢迎提醒页面。
      > **说明**
      > 进行本调测任务前请不要上网访问www.huawei.com，确保调测时是首次访问该网页。
          - 如果是，说明欢迎提醒功能正常，请继续执行[步骤5](#ZH-CN_TOPIC_0164110506__step4)。
          - 如果否，请继续执行[步骤4.2](#ZH-CN_TOPIC_0164110506__substepb)。
    b. 检查镜像接口的抓包信息，查看用户激活时Gy接口的报文，检查CCA-I消息中是否下发欢迎页面的一次重定向策略。
      > **说明**
      > OCS在“Service-Information AVP > PS-Information AVP”中下发一次重定向策略，其中“Redirect-Server > Redirect-Address-Type AVP”指示重定向的类型：IP地址格式、URL格式；“Redirect-Server > Redirect-Server-Address AVP”指示重定向服务器名；“Deactivate-By-Redirect AVP”指示重定向开关：使能、不使能。
          - 如果是，请确保本次访问www.huawei.com是本提醒周期内首次访问，如果还不能解决问题，请继续执行[步骤4.3](#ZH-CN_TOPIC_0164110506__substepe)。
          - 如果否，请联系OCS侧技术支持处理，确保OCS能下发一次重定向策略。
    c. 查看用户消息跟踪报文，检查用户访问www.huawei.com网页时，UDG是否正常返回重定向返回码。
          - 如果是，说明UDG构造重定向报文功能正常，请继续执行[步骤4.4](#ZH-CN_TOPIC_0164110506__substepc)。
          - 如果否，请执行[步骤7](#ZH-CN_TOPIC_0164110506__step9)。
    d. 执行[**LST GYONESHOT**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/重定向控制/Gy接口一次重定向控制/Gy接口一次重定向/查询Gy一次重定向参数（LST GYONESHOT）_86528909.md)命令，查询欢迎页面提醒功能绑定的URL白名单列表是否与规划值一致。
      ```
      LST GYONESHOT:;
             扩展过滤器名字  =  gy-oneshot-url
       重定向完成时间（秒）  =  15
          重定向携带信息名称 = test
      (结果个数 = 1) 
      ---    END
      ```
          - 如果是，请继续执行[步骤4.5](#ZH-CN_TOPIC_0164110506__substepd)。
          - 如果否，请根据激活计费信息实时提醒重新配置，之后返回[步骤4](#ZH-CN_TOPIC_0164110506__cn_20_110314_12_01_mmccppss_step3)重新调测。
    e. 执行[**LST EXTENDEDFILTER**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/扩展过滤器/查询扩展过滤器（LST EXTENDEDFILTER）_82837382.md)命令，查询欢迎页面提醒的URL白名单列表绑定的URL是否与规划值一致。
      ```
      LST EXTENDEDFILTER:EXTFLTNAME="gy-oneshot-url";
      ...... 
      ------------------------- 
          扩展过滤器名字  =  gy-oneshot-url 
      扩展过滤器记录类型  =  URL 
                     URL  =  www.huawei.com 
            Content Type  =  NULL 
              客户端类型  =  NULL 
                 Url后缀  =  NULL 
          后缀名匹配标识  =  不使能 
            分组匹配标识  =  不使能 
      (结果个数 = 1) 
      ---    END
      ```
    - 如果是，说明问题属于异常情况，请执行[步骤7](#ZH-CN_TOPIC_0164110506__step9)联系华为技术支持处理。
    - 如果否，请根据激活计费信息实时提醒重新配置，之后返回[步骤4](#ZH-CN_TOPIC_0164110506__cn_20_110314_12_01_mmccppss_step3)重新调测。
5. **调测流量提醒功能。**
    a. 用户继续浏览网页，同时可以访问其他数据业务。一段时间后，估约用户访问流量达到设置的提醒阈值后，查看用户终端是否收到流量提醒页面。
          - 如果是，说明流量提醒功能正常，请继续执行[步骤6](#ZH-CN_TOPIC_0164110506__step5)。
          - 如果否，请继续执行[步骤5.2](#ZH-CN_TOPIC_0164110506__substep4b)。
    b. 查看Gy接口消息抓包，参考欢迎页面提醒功能调测步骤的[步骤4.2](#ZH-CN_TOPIC_0164110506__substepb)～[步骤4.5](#ZH-CN_TOPIC_0164110506__substepd)的操作，查看CCA-U消息中是否下发流量提醒的一次重定向策略。
    - 如果是，说明问题属于异常情况，请执行[步骤7](#ZH-CN_TOPIC_0164110506__step9)联系华为技术支持处理。
    - 如果否，请等待一段时间当访问流量达到设置阈值后再检查。
6. **调测FUI** **重定向功能。**
    a. 继续访问业务，直到用户帐户金额耗尽。检查用户终端是否收到充值提醒。
          - 如果是，说明欠费充值重定向功能正常，本调测任务结束。
          - 如果否，说明欠费充值重定向功能异常，请继续执行[步骤6.2](#ZH-CN_TOPIC_0164110506__substep5b)。
    b. 查看Gy接口消息抓包，参考欢迎页面提醒功能调测步骤的[步骤4.2](#ZH-CN_TOPIC_0164110506__substepb)～[步骤4.5](#ZH-CN_TOPIC_0164110506__substepd)的操作，查看CCA-U消息中是否下发欠费充值提醒的一次重定向策略。
    - 如果是，说明问题属于异常情况，请执行[步骤7](#ZH-CN_TOPIC_0164110506__step9)联系华为技术支持处理。
    - 如果否，请联系OCS侧技术支持处理。
7. 收集信息并寻求技术支持。
    a. 在镜像接口或服务器上开启抓包工具，执行[步骤5](#ZH-CN_TOPIC_0164110506__step4)并保存报文。
    b. 执行**EXP MML**命令将当前配置数据导出为MML脚本文件并保存。
    c. 收集归纳所有信息并联系华为技术支持解决。

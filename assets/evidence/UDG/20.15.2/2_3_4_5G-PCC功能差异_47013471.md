# 2/3/4/5G PCC功能差异

*表1 2/3/4G PCC与5G PCC差异点比较*

| 差异点 | 2/3/4G | 5G |
| --- | --- | --- |
| 关键网元 | AF, SPR,<br>PCRF, PGW(PCEF) | AF,UDR,<br>PCF, AMF, PGW-C/SMF,<br>UPF |
| 接口 | Gx | N7，N15，N4 |
| 协议 | 基于Diameter的Gx应用协议 | PFCP协议（N4）<br>HTTPS（N7，N15） |
| 策略 | 会话管理策略 | - 会话管理策略（SM策略）<br>- 接入和移动性策略（AM策略）<br>- 选网和路由策略（UE策略） |
| 漫游支持 | 策略在归属网络生成 | - 策略在归属网络生成<br>- 策略在访问网络生成 |
| 主要流程 | - IP-CAN会话流程<br>- 专有承载流程 | - PCF发现和选择（AMF,PGW-C/SMF,AF请求）<br>- AM策略关联流程<br>- SM策略关联流程<br>- PFD管理流程 |
| 规则来源 | - PCRF下发<br>- AAA Server下发<br>- 本地配置 | - PCF下发<br>- 本地配置 |
| 规则分类 | - 动态规则<br>- 预定义规则<br>- 预定义规则组（即本地userprofile） | - 动态规则<br>- 预定义规则（可以匹配预定义规则或userprofile，即将4G的预定义规则和预定义规则组统一） |

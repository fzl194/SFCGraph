# 查询OSPFv3引入路由配置（LST OSPFV3IMPORTROUTE）

#### [命令功能](#ZH-CN_CONCEPT_0000001549961414)

该命令用于查询OSPFv3引入路由配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001549961414)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549961414)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549961414)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | OSPFv3进程号 | 可选必选说明：可选参数<br>参数含义：OSPFv3进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |
| PROTOCOL | 协议号 | 可选必选说明：可选参数<br>参数含义：协议号。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- direct：直连路由。<br>- static：静态路由。<br>- bgp：BGP协议。<br>- ospfv3：OSPFv3协议。<br>- wlr：无线路由。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549961414)

查询OSPFv3引入路由配置：

```
LST OSPFV3IMPORTROUTE:;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001549961414)

| 输出项名称 | 输出项解释 |
| --- | --- |
| OSPFv3进程号 | 该参数指定用户创建的OSPFv3进程号。 |
| 拓扑标识 | 该参数指定拓扑标识。 |
| 协议号 | 该参数指定OSPFv3引入外部路由的协议类型。 |
| 协议进程号 | 该参数指定引入路由协议的进程号，当协议号配置为OSPFv3时生效。 |
| 引入路由开销值配置 | 该参数指定允许配置引入的外部路由的开销值。 |
| 引入路由标签配置 | 该参数指定允许配置引入的外部路由的标签。 |
| 引入路由类型配置 | 该参数指定允许配置引入的外部路由的类型。 |
| 路径Cost值 | 该参数指定自治系统外部路径的开销，当引入路由开销值配置为true时生效。 |
| 标签 | 该参数指定外部LSA中的标记，当引入路由标签配置为true时生效。 |
| 引入路由类型 | 该参数指定外部LSA的度量类型，当引入路由Type使能标志配置为true时生效。 |
| 路由策略名称 | 该参数指定引入时依据的路由策略名称。 |
| 允许BGP配置 | 该参数指定允许使能引入IBGP路由，当协议号配置为BGP时生效。 |
| 指定引入路由时继承原路由的开销值 | 该参数指定引入路由时继承原路由的开销值。 |

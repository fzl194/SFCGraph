#### [命令功能](#ZH-CN_TOPIC_0000001111910784)

用于查询OM网络故障探测参数。

#### [注意事项](#ZH-CN_TOPIC_0000001111910784)

无。

#### [参数说明](#ZH-CN_TOPIC_0000001111910784)

无。

#### [使用实例](#ZH-CN_TOPIC_0000001111910784)

查询OM网络探测参数：

```
%%LST OMNWCONF:;%% 
RETCODE = 0  操作成功  
网络探测配置如下 
---------------- 
IP类型      IP地址类型  探测周期（秒）  倒换阈值  告警阈值  网络探测开关  网络亚健康告警阈值（%）  网络亚健康检测周期（分钟）  
PHYSICALIP  IPV4        15              NULL      15        ON            10                       5
PHYSICALIP  IPV6        10              NULL      15        ON            10                       5
VIRTUALIP   IPV6        2               3         NULL      ON            NULL                     NULL
(结果个数 = 3)  
---    END
```

#### [输出结果说明](#ZH-CN_TOPIC_0000001111910784)

命令执行正常，会返回命令执行成功的提示信息。输出结果说明如 [表1](#ZH-CN_TOPIC_0000001111910784__table73043432286) 所示。

该命令执行异常，请联系华为技术支持处理。错误码说明如 [表2](#ZH-CN_TOPIC_0000001111910784__table130424322820) 所示。

*表1 输出项解释*

| **输出项名称** | **输出项解释** |
| --- | --- |
| IP类型 | IP类型，取值为：<br>- PHYSICALIP<br>- VIRTUALIP |
| IP地址类型 | IP地址类型，取值为：<br>- IPV4<br>- IPV6 |
| 探测周期（秒） | 网络探测的间隔时间，单位为秒(s)。 |
| 倒换阈值 | 网络探测失败次数达到阈值时，主备倒换。 |
| 告警阈值 | 网络探测失败或成功次数达到告警阈值时上报或清除告警。 |
| 网络探测开关 | 网络探测是否开启或关闭。 |
| 网络亚健康告警阈值（%） | 用于指定在<br>“网络亚健康检测周期（分钟）”<br>内，丢包率达到该参数取值时，上报<br>[ALM-136001 OM网络资源故障](../../../../../../../网络运维/故障处理/Framework告警/ALM-136001 OM网络资源故障_54986794.md)<br>告警。 |
| 网络亚健康检测周期（分钟） | 用于指定执行网络亚健康检测时，统计丢包率的周期。 |

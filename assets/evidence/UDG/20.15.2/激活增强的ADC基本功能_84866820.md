## [操作场景](#ZH-CN_OPI_0184866820)

PCRF/PCF需要感知到应用数据流（L7），并为应用数据流提供QoS保障时，需要在 UDG 上激活增强的ADC基本功能。

UDG 检测应用并通过 PGW-C/SMF 上报给PCRF/PCF。

## [必备事项](#ZH-CN_OPI_0184866820)

前提条件

- 请仔细阅读[GWFD-020357 增强的ADC基本功能](../GWFD-020357 增强的ADC基本功能_84866817.md)。
- 完成加载License（如果有需求，请联系华为技术支持工程师）。
- 完成[激活PCC基本功能](../GWFD-020351 PCC基本功能/激活PCC基本功能_74096528.md)。

## [操作步骤](#ZH-CN_OPI_0184866820)

1. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09587387.md) :LICITEM="LKV3G5ADCF01",SWITCH=ENABLE;
2. 配置ADC业务使用的三四层过滤条件。
    a. 配置ADC业务使用的三四层过滤条件。
      [**ADD FILTER**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/过滤器/增加过滤器（ADD FILTER）_82837348.md)
    b. 配置流分类器。
      [**ADD FLOWFILTER**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/流过滤器/增加流过滤器（ADD FLOWFILTER）_82837361.md)
      > **说明**
      > - PCRF/PCF通过动态规则下发appid时，动态规则中携带的TDF-Application-Identifier AVP映射到本地配置的FlowFilter。
      > - PCRF/PCF通过预定义规则下发APPLICATION_START和APPLICATION_STOP Event-Trigger时，FlowFilter与ADC功能无关。
    c. 绑定三四层过滤信息。
      [**ADD FLTBINDFLOWF**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/流过滤器的过滤器绑定/增加流过滤器的过滤器绑定关系（ADD FLTBINDFLOWF）_82837366.md)
    d. 将新配置的Filter置为生效。
      [**SET REFRESHSRV**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/业务刷新/业务刷新（SET REFRESHSRV）_82837355.md)
3. 配置ADC业务使用的七层过滤条件。
    a. 配置七层过滤器。
      [**ADD L7FILTER**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/七层规则管理/七层过滤器/增加七层过滤器（ADD L7FILTER）_82837397.md)
    b. 绑定七层过滤信息。
      [**ADD PROTBINDFLOWF**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/流过滤器的协议绑定/增加流过滤器协议绑定关系（ADD PROTBINDFLOWF）_82837370.md)
    c. 将新配置的L7Filter置为生效。
      [**SET REFRESHSRV**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/业务刷新/业务刷新（SET REFRESHSRV）_82837355.md)
      > **说明**
      > 系统自动在60s后将配置的L7Filter置为生效。如果需要将新配置的L7Filter立即生效，可执行本步骤。
4. PCRF/PCF通过预定义规则下发APPLICATION_START和APPLICATION_STOP Event-Trigger时，配置ADC参数。
  [**ADD ADCPARA**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/ADC/ADC参数/增加ADC参数（ADD ADCPARA）_01905960.md)
  通过本命令开启流信息上报开关，则 UDG 可向PCRF/PCF上报流信息，便于PCRF/PCF获取到应用与流的对应关系，从而可以下发更精确的策略对业务进行控制。
  > **说明**
  > PCRF/PCF通过预定义规则下发APPLICATION_START和APPLICATION_STOP Event-Trigger时，PGW-U/UPF可规划单独的ADC类型的rule，也可复用PCC类型的rule。
  >
  > - 通过单独的策略类型为ADC的rule实现ADC功能，请执行[步骤 5](#ZH-CN_OPI_0184866820__step8431111985416)。
  > - 通过复用策略类型为PCC的rule实现ADC功能，请执行[步骤 6](#ZH-CN_OPI_0184866820__step1593210319318)～[步骤 8](#ZH-CN_OPI_0184866820__step3630133623814)。
5. 配置单独的策略类型为ADC的rule。
  [**ADD RULE**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务匹配公共配置/业务规则管理/规则/增加规则（ADD RULE）_82837266.md)
6. 配置ADC业务使用的计费属性。
  [步骤 6](#ZH-CN_OPI_0184866820__step1593210319318) ～ [步骤 9](#ZH-CN_OPI_0184866820__step1730816462404) 步骤的详细规划等请参见计费配置，区别在于：配置增强的ADC基本功能时，必须将 “ADCMUTEFLAG” 参数设置为 “DISABLE” 。
    a. 配置URR。
      [**ADD URR**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/计费控制/使用率上报规则/增加URR（ADD URR）_82837629.md)
    b. 配置上下行业务对应的URR。
      [**ADD URRGROUP**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/计费控制/使用率上报规则组/增加URR组（ADD URRGROUP）_82837634.md)
7. 配置ADC业务使用的PCC动作策略组。
  [**ADD PCCPOLICYGRP**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/PCC控制策略/PCC策略组/增加PCC策略组（ADD PCCPOLICYGRP）_82837606.md)
8. 配置策略类型为PCC的rule。
  [**ADD RULE**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务匹配公共配置/业务规则管理/规则/增加规则（ADD RULE）_82837266.md)
9. 配置ADC业务使用的业务策略组合（规则与用户模板的绑定关系） 。
    a. 创建UserProfile。
      [**ADD USERPROFILE**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务匹配公共配置/业务规则管理/用户模板/增加用户模板（ADD USERPROFILE）_82837279.md)
    b. 将已配置的rule绑定到UserProfile。
      [**ADD RULEBINDING**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务匹配公共配置/业务规则管理/规则绑定/增加用户模板和规则的绑定关系（ADD RULEBINDING）_82837272.md)

## [任务实例一：PCRF/PCF下发预定义规则](#ZH-CN_OPI_0184866820)

任务描述

使能增强的ADC基本功能，PCRF/PCF下发携带APPLICATION_START和APPLICATION_STOP Event-Trigger的预定义规则给 UDG ， UDG 本地配置需要识别的URL、具体的规则内容及上报的应用，并开启流信息上报开关，实现对HTTP业务的QoS保障。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH:LICITEM="LKV3G5ADCF01",SWITCH=ENABLE;
```

//配置ADC业务使用的三四层过滤条件。

```
ADD FILTER: FILTERNAME="filter_test1", L34PROTTYPE=STRING, L34PROTOCOL=ANY;
ADD FLOWFILTER:FLOWFILTERNAME="flowfilter_ADC01";
ADD FLTBINDFLOWF:FLOWFILTERNAME="flowfilter_ADC01",FILTERNAME="filter_test1";
SET REFRESHSRV:REFRESHTYPE=ALL;
```

//配置ADC业务的七层过滤条件（URL）。

```
ADD L7FILTER:L7FILTERNAME="l7filter_test",SUBL7FLTNAME="sl7_test_1",URL="www.huawei.com/*";
ADD L7FILTER:L7FILTERNAME="l7filter_test",SUBL7FLTNAME="sl7_test_2",URL="www.example.com";
ADD PROTBINDFLOWF:FLOWFILTERNAME="flowfilter_ADC01",PROTOCOLNAME="http",L7FILTERNAME="l7filter_test";
SET REFRESHSRV:REFRESHTYPE=ALL;
```

//配置ADC参数。

```
ADD ADCPARA: FLOWFILTERNAME="flowfilter_ADC01",FLOWINFORPT=ENABLE,ADCHYSTIMER=0;
```

//（通过策略类型为ADC的rule实现）配置ADC业务使用的规则。

```
ADD RULE:RULENAME="rule_test2",POLICYTYPE=ADC,FILTERINGMODE=FLOWFILTER,FLOWFILTERNAME="flowfilter_ADC01",PRIORITY=15,ADCMUTEFLAG=
DISABLE
;
```

//（通过策略类型为PCC的rule实现）配置ADC业务使用的计费属性。

```
ADD URR:URRNAME="urr01",URRID=1000,USAGERPTMODE=OFFLINE;
ADD URRGROUP:URRGROUPNAME="cp_test", UPURRNAME1="urr01", DOWNURRNAME1="urr01";
```

//（通过策略类型为PCC的rule实现）配置ADC业务使用的PCC动作策略组。

```
ADD PCCPOLICYGRP:PCCPOLICYGRPNM="pg_ADC01",ADCMUTEFLAG=
DISABLE
,URRGROUPNAME="cp_test";
```

//（通过策略类型为PCC的rule实现）配置ADC业务使用的规则。

```
ADD RULE:RULENAME="rule_test2",POLICYTYPE=PCC,FILTERINGMODE=FLOWFILTER,FLOWFILTERNAME="flowfilter_ADC01",PRIORITY=20,POLICYNAME="pg_ADC01";
```

//配置ADC业务使用的业务策略组合（规则与用户模板的绑定关系）。

```
ADD USERPROFILE:USERPROFILENAME="up_test";
ADD RULEBINDING:USERPROFILENAME="up_test",RULENAME="rule_test2";
```

流信息上报开关以SMF下发的为准。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH:LICITEM="LKV3G5ADCF01",SWITCH=ENABLE;
```

//配置ADC业务使用的三四层过滤条件。

```
ADD FILTER: FILTERNAME="filter_test1", L34PROTTYPE=STRING, L34PROTOCOL=ANY;
ADD FLOWFILTER:FLOWFILTERNAME="flowfilter_ADC01";
ADD FLTBINDFLOWF:FLOWFILTERNAME="flowfilter_ADC01",FILTERNAME="filter_test1";
SET REFRESHSRV:REFRESHTYPE=ALL;
```

//配置ADC业务的七层过滤条件（URL）。

```
ADD L7FILTER:L7FILTERNAME="l7filter_test",SUBL7FLTNAME="sl7_test_1",URL="www.huawei.com/*";
ADD L7FILTER:L7FILTERNAME="l7filter_test",SUBL7FLTNAME="sl7_test_2",URL="www.example.com";
```

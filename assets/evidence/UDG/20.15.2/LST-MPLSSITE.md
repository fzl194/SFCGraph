#### [命令功能](#ZH-CN_CONCEPT_0000001549961326)

该命令用于查询MPLS全局配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001549961326)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549961326)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549961326)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001549961326)

查询MPLS全局配置：

```
LST MPLSSITE:;
```

```

RETCODE = 0  操作成功。

结果如下
--------
MPLS LSR ID  =  192.168.1.1
   MPLS能力  =  DISABLE
    LDP能力  =  DISABLE
 空标签类型  =  隐式空标签
(结果个数 = 1)
---    END
```

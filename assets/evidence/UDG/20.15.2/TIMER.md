#### [命令功能](#ZH-CN_CONCEPT_0259103751)

该命令用于显示定时器信息。

#### [注意事项](#ZH-CN_CONCEPT_0259103751)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0259103751)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0259103751)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RUNAME | RU名称 | 可选必选说明：必选参数<br>参数含义：用于表示RU名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，区分大小写。<br>默认值：无 |
| SERVICEINSTANCE | 服务实例 | 可选必选说明：必选参数<br>参数含义：该参数表示大颗粒服务实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，通过LST VNFC命令获取。<br>默认值：无<br>配置原则：只能填写通过LST VNFC命令查询到的管理代理标识。 |

#### [使用实例](#ZH-CN_CONCEPT_0259103751)

显示VNODE_CSLB_VNFC_OMU_0002的定时器信息：

```
DSP TIMER:RUNAME="VNODE_CSLB_VNFC_OMU_0002"
,SERVICEINSTANCE="vnfc"
;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0259103751)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 进程ID | 用于表示进程ID。 |
| 进程名称 | 用于表示进程名称。 |
| 定时器超时时长 | 用于表示定时器的超时时长。单位不定，由定时器选项这一参数指定。若是定时器选项选择了时长单位为秒，则此参数的单位为秒，否则为毫秒。 |
| 定时器参数 | 用于表示定时器的参数，由各使用者自定义。 |
| 定时器类型 | 用于表示定时器的类型，由各使用者自定义。 |
| 定时器选项 | - 用于表示定时器的选项。此数据以掩码的形式选择定时器的优先级/平衡算法。<br>- 规则如下：把数据转为2进制数，从低位算起：- 若是第2位为1，则表示选择10%的时长偏移。<br>- 若是第3位为1，则表示为高优先级。<br>- 若是第4位为1，则表示为定时器时长为秒。<br>- 若是第5位为1，则表示时长偏移为负值，允许实际超时时长小于配置的定时器超时时长。 |
| 组件ID | 用于表示组件ID。 |
| 组件名称 | 用于表示组件名称。 |

## [操作场景](#ZH-CN_OPI_0000001256038531)

当前加密协议的业务识别主要基于SNI和Domain进行，对于未携带SNI的场景无法准确识别业务。在 UDG 开启HTTPS业务行为识别功能后， UDG 会对HTTPS业务报文做TLS指纹识别，提升识别精度。

> **说明**
> 适用于PGW-U、UPF。

## [操作步骤](#ZH-CN_OPI_0000001256038531)

1. 打开本特性的License特性配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09587387.md)
2. 加载特定的完整的特征库文件。指纹库内置在特征库中。如果已经加载特征库，则升级新特征库文件。
  [**LOD SIGNATUREDB**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/协议识别/特征库管理/识别特征库/加载协议特征库文件（LOD SIGNATUREDB）_82837716.md)
3. 开启HTTPS业务行为识别功能。
  [**ADD FINGERIDENT**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/协议识别/SA指纹识别/SA指纹识别/增加SA指纹识别配置（ADD FINGERIDENT）_86851945.md)
  > **说明**
  > 可通过 [**LST SUPPFINGERIDEN**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/协议识别/SA指纹识别/SA指纹识别/查询SA指纹识别配置（LST FINGERIDENT）_86723149.md) 命令查询系统内哪些协议支持指纹识别功能。

## [任务示例](#ZH-CN_OPI_0000001256038531)

任务描述

在 UDG 开启HTTPS业务行为识别功能，并加载特定的知识库版本，指定所有SA协议进行指纹识别功能。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH:LICITEM="LKV3G5HSBI01",SWITCH=ENABLE;
```

//加载知识库的版本为01.0002.1061.01。

```
LOD SIGNATUREDB:LOADMODE=SPECIFIC,VERSION="01.0002.1061.01";
```

//开启所有协议SA指纹识别功能。

```
ADD FINGERIDENT:CFGPROTMODE=ALL;
```

#### [命令功能](#ZH-CN_CONCEPT_0000001549802062)

该命令用于查询OSPF的平滑重启配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001549802062)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549802062)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549802062)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | 进程号 | 可选必选说明：可选参数<br>参数含义：OSPF进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |
| HELPERROLEFLAG | 使能Helper模式 | 可选必选说明：可选参数<br>参数含义：使能Helper模式。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- FALSE：不使能。<br>- TRUE：使能。<br>默认值：无 |
| HELPERFLAG | 使能GR | 可选必选说明：可选参数<br>参数含义：使能GR。<br>数据来源：本端规划<br>取值范围：布尔类型，输入格式为“TRUE”或者“FALSE”。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549802062)

查询OSPF的平滑重启配置：

```
LST OSPFGR:;
```


#### [输出结果说明](#ZH-CN_CONCEPT_0000001549802062)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 进程号 | 该参数指定用户创建的OSPF进程号。 |
| 只支持计划GR | 该参数指定在路由器上是否支持计划内的平滑重启。默认支持计划内和计划外的。 |
| 使能Helper模式 | 该参数指定是否使能Helper模式。 |
| 不检查Type-5和Type-7类的LSA | 该参数指定是否不检查Type-5和Type-7类的LSA。 |
| 不支持Helper模式 | 该参数指定是否不支持Helper模式。 |
| 使能GR | 该参数指定是否使能平滑重启。 |
| Helper过滤值 | 该参数指定Helper过滤值，当Helper过滤类型配置为acl_name，ip_prefix或acl_num时生效。 |
| Helper过滤类型 | 该参数指定Helper过滤类型，当使能Helper模式为true时生效。 |

#### [命令功能](#ZH-CN_CONCEPT_0000201556092939)

**适用NF：PGW-U、UPF**

该命令用于根据功能开关FUNCTION，删除指定功能的配置信息。

#### [注意事项](#ZH-CN_CONCEPT_0000201556092939)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000201556092939)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000201556092939)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| FUNCTION | 功能开关 | 可选必选说明：必选参数<br>参数含义：该参数用于配置功能开关。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- HIGH_RATE_USRSTATS：高带宽用户统计功能。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000201556092939)

删除高带宽用户统计功能：

```
RMV USERSTATISTICS: FUNCTION=HIGH_RATE_USRSTATS;
```

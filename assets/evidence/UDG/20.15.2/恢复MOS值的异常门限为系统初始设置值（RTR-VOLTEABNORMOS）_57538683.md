# 恢复MOS值的异常门限为系统初始设置值（RTR VOLTEABNORMOS）

- [注意事项](#ZH-CN_CONCEPT_0000201457538683__1.3.2.1)
- [操作用户权限](#ZH-CN_CONCEPT_0000201457538683__1.3.3.1)
- [参数说明](#ZH-CN_CONCEPT_0000201457538683__1.3.4.1)

#### [命令功能](#ZH-CN_CONCEPT_0000201457538683)

**适用NF：PGW-U**

该命令用于恢复异常MOS值的门限值为系统初始设置值。

#### [注意事项](#ZH-CN_CONCEPT_0000201457538683)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000201457538683)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000201457538683)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000201457538683)

恢复异常MOS值的门限值为系统初始设置值：

```
RTR VOLTEABNORMOS:;
```

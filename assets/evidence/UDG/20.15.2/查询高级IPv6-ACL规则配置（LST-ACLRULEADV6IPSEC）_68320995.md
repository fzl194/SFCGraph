## [命令功能](#ZH-CN_MMLREF_0000001168320995)

该命令用于查询当前系统中高级ACL的配置信息。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000001168320995)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001168320995)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| ACLNAME | 高级ACL规则组名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定规则属于哪个规则组。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~32。不支持空格，区分大小写。以英文字母a～z或A～Z开始，可以是英文字母、数字、连字符“-”、下划线“_”或中文字符的组合。整数形式，取值范围是3000～3999。<br>默认值：无<br>配置原则：无 |
| ACLRULENAME | 规则名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定规则名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~32。不支持空格，区分大小写。<br>默认值：无<br>配置原则：无 |

## [输出结果说明](#ZH-CN_MMLREF_0000001168320995)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 高级ACL规则组名称 | 该参数用于指定规则属于哪个规则组。 |
| 规则名称 | 该参数用于指定规则名称。 |
| 规则ID | 该参数用于指定规则的ID。如果不输入该参数，则规则ID按照规则组步长增长。 |
| 规则行为 | 该参数用于指定对符合本规则的报文采取的动作。 |
| 协议类型值 | 该参数用于指定规则协议类型。 |
| 源IPv6地址 | 该参数用于指定规则要匹配的源IPv6地址。 |
| 源IPv6地址正掩码 | 该参数用于指定规则要匹配的源IPv6地址正掩码。 |
| 目的IPv6地址 | 该参数用于指定规则要匹配的目的IPv6地址。 |
| 目的IPv6地址正掩码 | 该参数用于指定规则要匹配的目的IPv6地址正掩码。 |
| 报文分片类型 | 该参数用于指定规则所的报文分片类型，共有分片报文中的首片、分片报文中的非首片、非分片报文三种报文类型。如果不输入该参数，则表示不校验报文的分片信息。 |
| 服务优先级 | 该参数用于指定规则的数据包过滤服务优先级。如果不输入该参数，则表示不校验报文的服务优先级。 |
| 源端口范围类型 | 该参数用于指定规则源端口范围类型。如果不输入该参数，则表示不校验报文的源端口。 |
| 源端口的开始端口号 | 该参数用于指定规则源端口号的开始端口。 |
| 源端口的结束端口号 | 该参数用于指定规则源端口号的结束端口。 |
| 目的端口范围类型 | 该参数用于指定规则目的端口范围类型。如果不输入该参数，则表示不校验报文的目的端口。 |
| 目的端口的开始端口号 | 该参数用于指定规则目的端口号的开始端口。 |
| 目的端口的结束端口号 | 该参数用于指定规则目的端口号的结束端口。 |
| 网络控制管理协议类型 | 该参数用于指定规则的网络控制管理协议报文的类型。 |
| 网络控制管理协议消息码 | 该参数用于指定规则的网络控制管理协议消息码。 |
| DSCP值 | 该参数用于指定规则的差分服务码。如果不输入该参数，则表示不校验报文的差分服务码。 |
| 报文优先级 | 该参数用于指定规则的报文优先级。如果不输入该参数，则表示不校验报文优先级。 |
| VPN实例名称 | 该参数用于指定规则匹配要生效的VPN实例。 |
| ICMP名称 | 该参数用于指定规则的网络控制管理协议名称。如果不输入该参数，则表示不校验报文的ICMP名称。 |
| 规则描述 | 该参数用于指定规则描述。 |
| 协议选项类型 | 协议选项类型。 |
| 协议名称 | 协议名称。 |

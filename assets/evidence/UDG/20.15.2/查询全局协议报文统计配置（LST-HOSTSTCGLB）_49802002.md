#### [命令功能](#ZH-CN_CONCEPT_0000001549802002)

该命令用于查询全局协议报文统计配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001549802002)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549802002)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549802002)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001549802002)

查询全局协议报文统计配置：

```
LST HOSTSTCGLB:;
```

```

RETCODE = 0  操作成功。

结果如下
--------
协议报文统计功能状态  =  使能
(结果个数 = 1)
---    END
```

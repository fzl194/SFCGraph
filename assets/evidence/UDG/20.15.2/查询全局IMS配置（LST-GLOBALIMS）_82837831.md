#### [命令功能](#ZH-CN_CONCEPT_0182837831)

**适用NF：PGW-U、UPF**

该命令用来查询全局IMS互通配置信息。

#### [注意事项](#ZH-CN_CONCEPT_0182837831)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0182837831)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0182837831)

无。

#### [使用实例](#ZH-CN_CONCEPT_0182837831)

显示全局IMS配置：

```
LST GLOBALIMS:;
```

```

RETCODE = 0  操作成功。

全局IMS配置信息
---------------
              IMS功能开关  =  使能
(结果个数 = 1)
---    END
```

## [命令功能](#ZH-CN_MMLREF_0000001840700732)

通过 **DSP BKMPOLICY** 命令可获取备份服务的自动备份策略。

> **说明**
> 网络备份开关为关时sftp信息为无效，不返回相关参数。

## [参数说明](#ZH-CN_MMLREF_0000001840700732)

无。

## [使用实例](#ZH-CN_MMLREF_0000001840700732)

1. 查询自动备份策略：
  ```
  %%DSP BKMPOLICY:;%%
  RETCODE = 0  操作成功

  操作结果如下
  ------------
                  自动备份开关  =  开
            自动备份周期（天）  =  1
                  自动备份时间  =  03:00
        本地备份包存留数（个）  =  7
            是否支持不安全算法  =  关
          网络备份能力检查开关  =  开
              是否进行网络备份  =  开
  SFTP服务器备份包存留数（个）  =  -1
              SFTP服务器IP地址  =  10.0.0.0
              SFTP服务器端口号  =  22
                    SFTP用户名  =  user
                      SFTP密码  =  *****
                      目标目录  =  /home
  (结果个数 = 1)

  ---    END
  ```

## [输出结果说明](#ZH-CN_MMLREF_0000001840700732)

命令执行正常，会返回命令执行成功的提示信息，输出项如 [表1](#ZH-CN_MMLREF_0000001840700732__table7632103012435) 所示。

命令执行异常，请联系华为技术支持处理。输出错误码说明如 [表2](#ZH-CN_MMLREF_0000001840700732__table16741512233) 所示。

*表1 输出项说明*

| **输出项名称** | **输出项解释** |
| --- | --- |
| 自动备份开关 | 是否开启自动备份功能。 |
| 自动备份周期（天） | 系统自动备份的间隔时间。 |
| 自动备份时间 | 系统开始自动备份的时间点。 |
| 本地备份包存留数（个） | 系统本地存留的备份包数量，默认一天备份一份。 |
| 是否支持不安全算法 | 是否支持不安全算法。 |
| 网络备份能力检查开关 | 是否支持网络备份能力检查。 |
| 是否进行网络备份 | 可选择是否进行SFTP存储。 |
| SFTP服务器备份包存留数（个） | 可以上传到SFTP服务端备份包的总个数。 |
| SFTP服务器IP地址 | 保存备份数据包的远端SFTP服务器的IP地址。 |
| SFTP服务器端口号 | 保存备份数据包的远端SFTP服务器的端口号。 |
| SFTP用户名 | 登录远端SFTP服务器的用户名。 |
| SFTP密码 | 登录远端SFTP服务器的密码。 |
| 目标目录 | 备份数据包保存在远端SFTP服务器的具体路径。 |

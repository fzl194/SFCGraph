## [命令功能](#ZH-CN_MMLREF_0000002241105089)

该命令用于查询DCS直通存储慢盘检测参数。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000002241105089)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000002241105089)

无

## [使用实例](#ZH-CN_MMLREF_0000002241105089)

查询DCS直通存储慢盘检测参数

```
%%LST DCSSDPARA:;%%
RETCODE = 0  操作成功

结果如下
------------------------
   慢盘检测开关  =  开启
       检测次数  =  30
    检测周期(s)  =  1
 IO服务时间(ms)  =  30
    IO使用率(%)  =  98
慢盘故障阈值(%)  =  70
慢盘恢复阈值(%)  =  50
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000002241105089)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 慢盘检测开关 | 该参数用于开启或关闭DCS直通存储慢盘检测功能。 |
| 检测次数 | 该参数表示一轮直通慢盘检测的总次数。 |
| 检测周期(s) | 该参数用于设置直通存储慢盘检测周期，表示连续两次检测间的时间间隔。 |
| IO服务时间(ms) | 该参数表示系统进行IO操作的响应时间，IO操作表示在磁盘上进行读写操作。 |
| IO使用率(%) | 该参数表示系统IO操作所占用的CPU负载。 |
| 慢盘故障阈值(%) | 该参数表示一轮慢盘检测中，慢盘状态次数在总检测次数中占比的故障阈值，大于等于该阈值判定为直通存储慢盘。 |
| 慢盘恢复阈值(%) | 该参数表示一轮慢盘检测中，慢盘状态次数在总检测次数中占比的恢复阈值，小于等于该阈值则表示直通存储慢盘恢复。 |

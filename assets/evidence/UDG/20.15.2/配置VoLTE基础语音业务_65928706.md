## [操作场景](#ZH-CN_OPI_0165928706)

当部署 GPRS / UMTS / EPC 网络接入IMS域，需要配置VoLTE基础语音业务。

> **说明**
> 适用于SGW-U、PGW-U。

## [操作步骤](#ZH-CN_OPI_0165928706)

- 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09587387.md)
- 基于APN配置IMS业务。
  [**ADD APNIMSSIGFLTR**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/VOLTE管理/APN的IMS信令分类器/添加APN的IMS分类器（ADD APNIMSSIGFLTR）_82837825.md)
- 配置IMS使能开关和IMS信令空口增强开关，绑定到指定APN实例下。
  [**SET APNIMSATTR**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/VOLTE管理/APN的IMS属性/设置ApnImsAttr配置（SET APNIMSATTR）_82837822.md)

## [任务示例](#ZH-CN_OPI_0165928706)

任务描述

配置IMS业务。

脚本

// 打开本特性的License配置开关。

```
SET LICENSESWITCH:LICITEM ="LKV3G5VLEF01",SWITCH=ENABLE;
```

//基于APN配置IMS业务。

```
ADD APNIMSSIGFLTR: APN="ims", PRIORITY=1, IPVERSION=IPV4, SRCIPV4SPECTYPE=SPECIFY, SRCIPV4ADDR="10.1.1.1", SRCIPV4MASKLEN=32, SRCPORTSPECTYPE=SPECIFY, SRCPORTSTART=1, SRCPORTEND=2, PROSPECTYPE=SPECIFY, PROTOCOL=6;
```

```
ADD APNIMSSIGFLTR: APN="ims", PRIORITY=2, IPVERSION=IPV4, SRCIPV4SPECTYPE=SPECIFY, SRCIPV4ADDR="10.1.1.1", SRCIPV4MASKLEN=32, SRCPORTSPECTYPE=SPECIFY, SRCPORTSTART=1, SRCPORTEND=2, PROSPECTYPE=SPECIFY, PROTOCOL=17;
```

//配置IMS使能开关和IMS信令空口增强开关，绑定到指定APN实例下。

```
SET APNIMSATTR:APN="ims",IMSSWITCH=ENABLE;
```

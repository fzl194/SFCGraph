# 激活支持OTT业务规则库功能

- [操作场景](#ZH-CN_OPI_0305455409__1.3.1)

## [操作场景](#ZH-CN_OPI_0305455409)

在 UDG 上激活支持百万级OTT业务规则库功能。配置加载外部定义的业务规则库，并将其绑定在指定的flow-filter下，配置OTT业务对应的计费策略和动作策略，从而实现对不同的业务流执行不同的计费策略和动作策略。

> **说明**
> 适用于PGW-U、UPF。

## [必备事项](#ZH-CN_OPI_0305455409)

前提条件

- 请仔细阅读[GWFD-110321 支持内置百万级业务规则库](../GWFD-110321 支持内置百万级业务规则库_05455410.md)。
- ESA VMPod正常运行，且部署该特性时不影响ESA VMPod上已承载的业务。若网络中未部署ESA VMPod，请参考[安装ESA服务](../../../../../网络部署/软件安装/基于VNF LCM安装UDG（非SDN）/安装可选服务/安装ESA服务_08952123.md)完成ESA VMPod的部署。
- OTT业务规则库已上传至UDG。

## [任务示例](#ZH-CN_OPI_0305455409)

任务描述

设置加载外部业务规则库功能，对匹配OTT业务规则库成功的报文做在线计费策略。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH:LICITEM="LKV3G5MSFM01",SWITCH=ENABLE;
```

//加载OTT业务规则库。

```
LOD EXTERNALDB: DBTYPE=OTT,VERSION="0020.01.0000.0000.001";
```

```
SET REFRESHSRV:REFRESHTYPE=USERPROFILE;
```

//使能外部OTT业务规则库。

```
ADD FLOWFILTER:FLOWFILTERNAME="ff1",FLOWFILTERTYPE=EXTEROTTDB;
```

//打开OTT业务规则库匹配开关。

```
SET EXTOTTMATCHSW:IPV4MATCH=ENABLE,IPV6MATCH=ENABLE;
```

//配置在线计费的费率标识。

```
ADD URR:URRNAME="urr_red",URRID=1000,USAGERPTMODE=ONLINE;
ADD URRGROUP:URRGROUPNAME="cp_red",UPURRNAME1="urr_red",DOWNURRNAME1="urr_red";
ADD PCCPOLICYGRP:PCCPOLICYGRPNM="cg_red",URRGROUPNAME="cp_red";
```

//配置计费规则。

```
ADD RULE:RULENAME="rule_red",POLICYTYPE=PCC,FILTERINGMODE=FLOWFILTER, FLOWFILTERNAME="ff1",POLICYNAME="cg_red";
SET REFRESHSRV:REFRESHTYPE=USERPROFILE;
```

//配置计费规则绑定用户模板。

```
ADD USERPROFILE:USERPROFILENAME="up_test";
ADD RULEBINDING:USERPROFILENAME="up_test",RULENAME="rule_red";
```

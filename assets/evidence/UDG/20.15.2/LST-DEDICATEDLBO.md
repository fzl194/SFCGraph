#### [命令功能](#ZH-CN_CONCEPT_0000204243025982)

**适用NF：PGW-U、UPF**

该命令用来查看专网UPF动态分流功能开关状态。

#### [注意事项](#ZH-CN_CONCEPT_0000204243025982)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000204243025982)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000204243025982)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000204243025982)

查看专网UPF动态分流功能开关状态，可以使用该命令：

```
LST DEDICATEDLBO:;
```

```

RETCODE = 0  操作成功。

开关状态
------------------
专网UPF动态分流功能开关  =  使能
(结果个数 = 1)
---    END
```

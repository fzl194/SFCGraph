#### [命令功能](#ZH-CN_CONCEPT_0129627085)

该命令用于查询隧道信息，辅助定位隧道转发失败的问题。

#### [注意事项](#ZH-CN_CONCEPT_0129627085)

该命令批量下发可能导致执行超时。

#### [操作用户权限](#ZH-CN_CONCEPT_0129627085)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0129627085)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| TNGRPID | 隧道组ID | 可选必选说明：可选参数<br>参数含义：该参数用于表示业务VNFC定义的隧道组的标识。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0~4294967295。<br>默认值：无 |
| TNID | 隧道ID | 可选必选说明：可选参数<br>参数含义：该参数用于表示业务VNFC定义的隧道的标识。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0~4294967295。<br>默认值：无 |
| SRVVNFCID | 业务VNFCID | 可选必选说明：可选参数<br>参数含义：该参数用于指定业务VNFC的唯一标识，通过在业务VNFC下执行<br>**[LST NODE](../../../../单体服务公共功能管理/系统管理/基础参数/查询节点信息/查询节点信息（LST NODE）_27372977.md)**<br>获得，所得NODE ID（节点ID）即为业务VNFC ID。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0~4294967295<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0129627085)

查询所有隧道信息。

DSP LBTN:;


#### [输出结果说明](#ZH-CN_CONCEPT_0129627085)

| 输出项名称 | 输出项解释 |
| --- | --- |
| IP类型 | 参数含义：该参数用于显示隧道的IP地址类型。<br>取值范围：<br>- “IPV4”<br>- “IPV6” |
| 隧道组ID | 参数含义：该参数用于显示业务VNFC定义的隧道组的标识。<br>取值范围：整数类型，取值范围为0~4294967295。 |
| 隧道ID | 参数含义：该参数用于显示业务VNFC定义的隧道的标识。<br>取值范围：整数类型，取值范围为0~4294967295。 |
| 业务VNFCID | 参数含义：该参数用于显示业务VNFC的唯一标识。<br>取值范围：整数类型，取值范围为0~4294967295。 |
| 隧道组名称 | 参数含义：该参数用于显示隧道所属隧道组的名称。<br>取值范围：字符串类型，输入长度范围为0~15。 |
| VPN名称 | 参数含义：该参数用于显示业务VNFC配置的VPN名称。<br>取值范围：字符串类型，输入长度范围为0~31。 |
| 源IPV4 | 参数含义：该参数用于显示隧道源IPV4地址。<br>取值范围：0.0.0.0~255.255.255.255。 |
| 源端口号 | 参数含义：该参数用于显示隧道源端口号。<br>取值范围：整数类型，取值范围为0~65535。 |
| 目的IPV4 | 参数含义：该参数用于显示隧道目的IPV4地址。<br>取值范围：0.0.0.0~255.255.255.255。 |
| 目的端口号 | 参数含义：该参数用于显示隧道目的端口号。<br>取值范围：整数类型，取值范围为0~65535。 |
| 隧道状态 | 参数含义：该参数用于显示隧道状态。<br>取值范围：<br>- “初始态”：隧道探测报文已发送，尚未收到探测结果的状态。<br>- “正常”：隧道探测成功的状态。<br>- “故障”：隧道探测失败的状态。 |
| 源IPV6 | 参数含义：该参数用于显示隧道源IPV6地址。<br>取值范围：0:0:0:0:0:0:0:0~FFFF:FFFF:FFFF:FFFF:FFFF:FFFF:FFFF。 |
| 目的IPV6 | 参数含义：该参数用于显示隧道目的IPV6地址。<br>取值范围：0:0:0:0:0:0:0:0~FFFF:FFFF:FFFF:FFFF:FFFF:FFFF:FFFF。 |

#### [命令功能](#ZH-CN_CONCEPT_0000001600866237)

该命令用于查询IPv6扩展头选项安全配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001600866237)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600866237)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600866237)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001600866237)

查询IPv6扩展头选项安全配置：

```
LST IPV6EXTHSECALL:;
```

```

RETCODE = 0  操作成功

结果如下
-------------------------
IPv6过滤配置  =  使能
(返回结果 = 1)
---    END
```

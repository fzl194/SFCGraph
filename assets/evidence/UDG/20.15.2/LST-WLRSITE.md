#### [命令功能](#ZH-CN_CONCEPT_0000001549961286)

该命令用于查询无线路由全局属性配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001549961286)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549961286)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549961286)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001549961286)

查询无线路由全局属性配置：

```
LST WLRSITE:;
```

```

RETCODE = 0  操作成功

结果如下
-------------------------
     GR定时器（s）  =  1
PAE延时定时器（s）  =  30
(结果个数 = 1)
---    END
```

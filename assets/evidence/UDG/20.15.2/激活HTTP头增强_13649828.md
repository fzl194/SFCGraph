## [操作场景](#ZH-CN_OPI_0000001113649828)

当需要将用户信息（如APN、IMEI等）中的一个或多个信息添加到HTTP报文头并传递给Web Server时，需要完成HTTP报文头的内容增强处理。

> **说明**
> 适用于PGW-U、UPF。

## [必备事项](#ZH-CN_OPI_0000001113649828)

前提条件

- 请仔细阅读[HTTP头增强原理描述](HTTP头增强原理描述_59374533.md)。
- 操作员已经获取到运营商对使用HTTP头增强特性的插入字段、字段前缀名称以及需要HTTP头增强信息的Web Server的IP地址、URL等信息。
- 完成加载license（如果有需求，请联系华为技术支持工程师处理）。

## [任务示例](#ZH-CN_OPI_0000001113649828)

任务描述

运营商的Web Server需要通过获取用户的APN、IMSI、MSIP、MSISDN给某些用户提供特殊定制业务，因此，运营商需要 UDG 将用户发起的上行HTTP报文进行HTTP头增强，采用基于特定IP地址、基于特定URL或HTTP协议的触发头增强方式，将用户的APN、IMSI、MSIP、MSISDN字段插入用户报文，将其携带给Web Server。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV3G5HTHE01",SWITCH=ENABLE;
```

//配置流过滤器。

```

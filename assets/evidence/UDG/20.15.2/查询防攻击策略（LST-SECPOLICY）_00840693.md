#### [命令功能](#ZH-CN_CONCEPT_0000001600840693)

该命令用于查询安全策略。

#### [注意事项](#ZH-CN_CONCEPT_0000001600840693)

- 该命令执行后立即生效。
- 该命令仅适用于非NP卡场景和NP卡非加速模式场景。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600840693)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600840693)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| POLICYID | 安全策略编号 | 可选必选说明：可选参数<br>参数含义：策略编号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～30。<br>默认值：无 |
| DESCRIPTION | 策略描述 | 可选必选说明：可选参数<br>参数含义：防攻击策略描述。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～64。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600840693)

- 查询安全策略：
  ```
  LST SECPOLICY:;
  ```
  ```
  RETCODE = 0  操作成功

  结果如下
  -------------------------
  安全策略编号    策略描述

  1               NULL
  2               NULL
  (结果个数 = 2)
  ---    END
  ```
- 查询指定条件的安全策略：
  ```
  LST SECPOLICY:POLICYID=1;
  ```
  ```
  RETCODE = 0  操作成功

  结果如下
  -------------------------
  安全策略编号  =  1
      策略描述  =  NULL
  (结果个数 = 1)
  ---    END
  ```

| 策略描述 | 该参数为策略描述信息。 |

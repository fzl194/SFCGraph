#### [命令功能](#ZH-CN_CONCEPT_0000001600866733)

该命令用于显示LDP邻居管理的接口信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600866733)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600866733)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600866733)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| VRFNAME | VPN实例名称 | 可选必选说明：可选参数<br>参数含义：该参数用于表示VPN实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。<br>默认值：_public_ |
| IFNAME | 接口名称 | 可选必选说明：可选参数<br>参数含义：该参数用于表示LDP邻居的接口名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。<br>默认值：无<br>配置原则：请使用LST INTERFACE命令查看可用接口。 |

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600866733)

| 输出项名称 | 输出项解释 |
| --- | --- |
| VPN实例名称 | 用于表示VPN实例名称。 |
| 接口索引值 | 用于表示LDP邻居的接口索引值。 |
| 接口名称 | 用于表示LDP邻居的接口名称。 |
| 使用传输地址的次数 | 用于表示传输地址设为接口的次数。 |
| 传输地址 | 用于表示LDP建立LDP会话使用的传输地址。 |
| 配置序列号 | 用于表示LDP接口上的配置序列号。 |
| 发送Hello消息的时长（ms） | 用于表示LDP接口上发送Hello消息的时间（ms）。 |
| Hello消息发送的间隔时间（s） | 用于表示LDP接口上发送Hello消息的间隔时间（s）。 |
| Hello消息保持的时间（s） | 用于表示LDP接口上Hello消息保持的时间（s）。 |
| 协商的保持Hello消息的时间（s） | 用于表示LDP协商的保持Hello消息的时间（s）。 |
| 发送保活消息时间间隔（s） | 用于表示LDP接口上配置的发送保活消息时间间隔（s）。 |
| 保活时间（s） | 用于表示LDP接口上配置的保活时间（s）。 |
| 加入多播的返回码 | 用于表示LDP接口加入多播的返回码。 |
| 加入多播的状态标识 | 该参数用于表示LDP接口加入多播的状态标识，包括：<br>- 空闲状态。<br>- 正在加入多播。<br>- 正在尝试重新加入多播。<br>- 加入多播成功。<br>- 正在退出多播。 |
| 加入或退出多播的标志 | 用于表示LDP接口加入多播或退出多播的标志，0表示加入多播。 |
| 等待加入组播 | 用于表示LDP接口存在于等待加入多播或退出多播的队列的标志。 |
| 订阅的状态 | 该参数用于表示LDP接口订阅的状态，包括：<br>- 空闲状态。<br>- 正在订阅。<br>- 订阅成功。<br>- 去订阅。 |
| LDP接口的标志 | 用于表示LDP接口的标志。 |
| 备份的版本号 | 用于表示LDP接口备份的版本号。 |
| 接收Hello消息的数量 | 用于表示LDP接口上接收到的Hello消息的数量。 |
| 发送Hello消息的数量 | 用于表示LDP接口上发送的Hello消息的数量。 |
| 地址数量 | 用于表示LDP接口上的地址的数量。 |

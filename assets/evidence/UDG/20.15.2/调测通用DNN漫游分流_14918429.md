## [操作场景](#ZH-CN_OPI_0000001614918429)

本操作指导介绍在运行网络中调测通用DNN漫游分流功能的操作过程。

## [对系统的影响](#ZH-CN_OPI_0000001614918429)

无。

## [必备事项](#ZH-CN_OPI_0000001614918429)

前提条件

- 操作人员已了解本功能的原理。

- 完成[激活通用DNN漫游分流](激活通用DNN漫游分流_07687889.md)。

数据

该操作无需准备数据。

工具

- 测试终端
- OM Portal跟踪

## [操作步骤](#ZH-CN_OPI_0000001614918429)

1. 执行 [**LST LICENSESWITCH**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/查询License配置项开关（LST LICENSESWITCH）_09587900.md) 命令，查询本特性对应的License项是否开启。
  ```
  LST LICENSESWITCH:LICITEM="LKV3G5STCA01";  
  RETCODE = 0  操作成功

  操作结果如下
  ------------
           License项  =  LKV3G5STCA01
         License名称  =  移动VPN智能分流园区接入
                开关  =  打开
   (结果个数 = 1)  
   ---    END
  ```
    - 如果License的“开关”为“打开”，请执行[步骤 2](#ZH-CN_OPI_0000001614918429__step1949305824820)。
    - 如果License的“开关”为“关闭”，则执行[**SET LICENSESWITCH**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09587387.md)命令打开本特性对应的License配置开关。
2. 在公网UPF对应的 UDG 上创建用户跟踪任务。
3. 在专网UPF对应的 UDG 上创建用户跟踪任务。
4. 测试终端 在漫游地发起5G用户激活流程。
    - 如果测试终端成功接入网络，请执行[步骤 5](#ZH-CN_OPI_0000001614918429__step427244210501)。
    - 如果测试终端无法接入网络，调测UDG的接入功能。
5. 用户激活成功后，在公网UPF上执行 [**DSP SESSIONINFO**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/会话信息管理/会话信息查询/显示用户上下文（DSP SESSIONINFO）_86526407.md) 查询用户会话的MultiDnn Session Type属性。
    - 如果查询结果为两个会话，其中一个会话MultiDnn Session Type属性为RtsDnn Common UPF，另外一个会话MultiDnn Session Type属性为RtsDnn Special I-UPF，请执行[步骤 6](#ZH-CN_OPI_0000001614918429__step176001122185617)。
    - 否则，请参考[激活通用DNN漫游分流](激活通用DNN漫游分流_07687889.md)重新配置后再执行本操作。
6. 用户激活成功后，在专网UPF上执行 [**DSP SESSIONINFO**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/会话信息管理/会话信息查询/显示用户上下文（DSP SESSIONINFO）_86526407.md) 查询用户会话的MultiDnn Session Type属性。
    - 如果查询结果为一个会话，其MultiDnn Session Type属性为RtsDnn SpecialDnn UPF，请执行[步骤 7](#ZH-CN_OPI_0000001614918429__step15938311165811)。
    - 否则，请参考[激活通用DNN漫游分流](激活通用DNN漫游分流_07687889.md)重新配置后再执行本操作。
7. 测试终端 访问Internet，观察公网UPF的用户跟踪中的业务流走向。
    - 如果业务流通过N6口直接发送到本地Internet，则执行[步骤 8](#ZH-CN_OPI_0000001614918429__step147991587593)。
    - 否则，请执行[步骤 9](#ZH-CN_OPI_0000001614918429__step1578773210337)。
8. 测试终端 访问归属地专网业务，观察业务流是否正常放通。
    - 如果业务流正常放通，则业务调测完成。
    - 否则，请执行[步骤 9](#ZH-CN_OPI_0000001614918429__step1578773210337)。
9. 收集信息并寻求技术支持。
    a. [**EXP MML**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/配置管理/配置导出管理/导出MML文件（EXP MML）_47200033.md) ，将当前配置数据导出为MML脚本文件并保存。
    b. 收集并保存上述所有查询信息。
    c. 收集归纳所有信息并联系华为技术支持解决。

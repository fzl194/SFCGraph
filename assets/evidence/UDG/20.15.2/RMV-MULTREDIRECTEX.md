# 删除扩展的多级重定向密码（RMV MULTREDIRECTEX）

- [命令功能](#ZH-CN_CONCEPT_0000204327370105__1.3.1.1)
- [注意事项](#ZH-CN_CONCEPT_0000204327370105__1.3.2.1)
- [操作用户权限](#ZH-CN_CONCEPT_0000204327370105__1.3.3.1)
- [参数说明](#ZH-CN_CONCEPT_0000204327370105__1.3.4.1)

#### [命令功能](#ZH-CN_CONCEPT_0000204327370105)

**适用NF：PGW-U、UPF**

该命令用于删除多级重定向字段加密的密码。

#### [注意事项](#ZH-CN_CONCEPT_0000204327370105)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000204327370105)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000204327370105)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000204327370105)

假如运营商想要删除加密密码，配置如下：

```
RMV MULTREDIRECTEX:;
```

#### [命令功能](#ZH-CN_CONCEPT_0275097447)

**适用NF：PGW-U、UPF**

该命令用来查询全局的冗余备份隧道虚拟重定向IP。

#### [注意事项](#ZH-CN_CONCEPT_0275097447)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0275097447)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0275097447)

无。

#### [使用实例](#ZH-CN_CONCEPT_0275097447)

查询冗余备份隧道虚拟重定向地址：

```
LST REDUNDRDTIP:;
```

```

RETCODE = 0 操作成功。

冗余备份重定向IP
----------------
IP版本 = IPv4
IPV4地址 = 192.168.0.1
IPV6地址 = ::
(结果个数 = 1)
--- END
```

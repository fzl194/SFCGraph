#### [命令功能](#ZH-CN_CONCEPT_0182837251)

**适用NF：SGW-U、PGW-U、UPF**

查询UPF工作状态配置。

#### [注意事项](#ZH-CN_CONCEPT_0182837251)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0182837251)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0182837251)

无。

#### [使用实例](#ZH-CN_CONCEPT_0182837251)

查询UPF工作状态配置：

```
LST UPSTATUS:;
```

```

RETCODE = 0  操作成功。

UP 状态信息:
----------
               UP状态  =  UP
       强制停工开关  =  DISABLE
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0182837251)

## [命令功能](#ZH-CN_MMLREF_0000001840859456)

该命令用于设置备份服务的自动备份策略。

> **说明**
> - 由于公钥校验需要下载公钥， 这里不提供公钥校验开关的参数。
> - 执行命令时未传值的可选参数，在自动备份策略中对应参数值保持不变。

## [参数说明](#ZH-CN_MMLREF_0000001840859456)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SCHEDULEFLAG | 自动备份开关 | 可选必选说明：必选参数。<br>参数含义：是否开启自动备份功能。<br>取值范围：<br>- ON(开)：开启自动备份。<br>- OFF(关)：不自动备份。<br>默认值：无。<br>配置原则：无。 |
| PERIOD | 自动备份周期（天） | 可选必选说明：该参数在<br>“SCHEDULEFLAG(自动备份开关)”<br>为<br>“ON(开)”<br>时，为条件可选参数。<br>参数含义：系统自动备份的间隔时间。<br>取值范围：整数类型，取值范围为1~7。<br>默认值：无。<br>配置原则：无。 |
| STARTTIME | 自动备份时间 | 可选必选说明：该参数在<br>“SCHEDULEFLAG(自动备份开关)”<br>为<br>“ON(开)”<br>时，为条件可选参数。<br>参数含义：系统开始自动备份的时间点。<br>取值范围：24小时制时间，格式HH:MM。<br>默认值：无。<br>配置原则：无。 |
| VALIDPERIOD | 本地备份包存留数（个） | 可选必选说明：该参数在<br>“SCHEDULEFLAG(自动备份开关)”<br>为<br>“ON(开)”<br>时，为条件可选参数。<br>参数含义：系统本地存留的备份包数量。<br>说明：默认在每次备份任务完成后会清除存留数之外的备份包。若此时存在异常无法清除，则在23:10清除存留数量之外的数据包。<br>取值范围：整数类型，取值范围为1~7。<br>默认值：无。<br>配置原则：无。 |
| REMOTEFLAG | 是否进行网络备份 | 可选必选说明：该参数在<br>“SCHEDULEFLAG(自动备份开关)”<br>为<br>“ON(开)”<br>时，为条件可选参数。<br>参数含义：可选择是否进行SFTP存储。<br>取值范围：<br>- ON(开)：进行网络备份，并展开SFTP服务器参数配置界面。<br>- OFF(关)：不进行网络备份。<br>- DEFAULT(默认)：使用原有的网络备份配置信息。<br>默认值：无。<br>配置原则：无。 |
| COMPATIBLE | 是否支持不安全算法 | 可选必选说明：该参数在<br>“SCHEDULEFLAG(自动备份开关)”<br>为<br>“ON(开)”<br>时，为条件可选参数。<br>参数含义：是否支持不安全算法。<br>- 支持的安全算法为：curve25519-sha256，curve25519-sha256@libssh.org，diffie-hellman-group-exchange-sha256。<br>- 支持的不安全算法为：diffie-hellman-group-exchange-sha1，diffie-hellman-group1-sha1，diffie-hellman-group14-sha1，diffie-hellman-group-exchange-sha256，ecdh-sha2-nistp256，ecdh-sha2-nistp384，ecdh-sha2-nistp521。<br>取值范围：<br>- ON(开)：支持不安全算法。<br>- OFF(关)：不支持不安全算法。<br>默认值：无。<br>配置原则：根据SFTP服务器支持的算法选择打开或关闭开关。<br>说明：打开<br>“是否支持不安全算法”<br>开关，可能会存在安全风险。非必要情况下建议关闭此开关。 |
| CHECKNETWORKALARM | 网络备份能力检查开关 | 可选必选说明：该参数在<br>“SCHEDULEFLAG(自动备份开关)”<br>为<br>“ON(开)”<br>时，为条件可选参数。<br>参数含义：是否支持网络备份能力检查。<br>取值范围：<br>- ON(开)：支持网络备份能力检查。<br>- OFF(关)：不支持网络备份能力检查。<br>默认值：无。<br>配置原则：无。 |
| REMOTEVALIDPERIOD | SFTP服务器备份包存留数（个） | 可选必选说明：该参数在<br>“REMOTEFLAG(是否进行网络备份)”<br>为<br>“ON(开)”<br>时，为条件可选参数。<br>参数含义：SFTP服务器上存留的备份包数量。<br>取值范围：整数类型，取值范围为-1和3~30，-1代表不控制SFTP服务器备份包存留数。<br>默认值：无。<br>配置原则：无。 |
| SFTPIPTYPE | SFTP服务器IP地址类型 | 可选必选说明：该参数在<br>“REMOTEFLAG(是否进行网络备份)”<br>为<br>“ON(开)”<br>时，为条件必选参数。<br>参数含义：SFTP服务器的IP地址的类型。<br>取值范围：<br>- IPV4(IPV4)：IPV4类型的SFTP服务器IP地址。<br>- IPV6(IPV6)：IPV6类型的SFTP服务器IP地址。<br>默认值：无。<br>配置原则：无。 |
| IPV4 | SFTP服务器IP地址 | 可选必选说明：该参数在<br>“SFTPIPTYPE(SFTP服务器IP地址类型)”<br>为<br>“IPV4(IPV4)”<br>时，为条件必选参数。<br>参数含义：远端SFTP服务器的IPv4地址。<br>取值范围：点分十进制格式，即X.X.X.X，其中X为0~255的数，如10.119.147.178。<br>默认值：无。<br>配置原则：无。 |
| IPV6 | SFTP服务器IP地址 | 可选必选说明：该参数在<br>“SFTPIPTYPE(SFTP服务器IP地址类型)”<br>为<br>“IPV6(IPV6)”<br>时，为条件必选参数。<br>参数含义：远端SFTP服务器的IPv6地址。<br>取值范围：冒号分割的16进制格式，即X:X:X:X:X:X:X:X或者为压缩的格式X::X，如果其中X出现连续为0的情况允许出现压缩格式，即将连续为0的两段或多段可以压缩成连续两个冒号，连续冒号只允许出现一次，其中X为0~FFFF的十六进制数，如fc00:0:0:0:0:0:0:7344或者fc00::7344。<br>默认值：无。<br>配置原则：无。 |
| SFTPPORT | SFTP服务器端口号 | 可选必选说明：该参数在<br>“REMOTEFLAG(是否进行网络备份)”<br>为<br>“ON(开)”<br>时，为条件必选参数。<br>参数含义：远端SFTP服务器的端口号。<br>取值范围：整数类型，取值范围为0~65535。<br>默认值：无。<br>配置原则：无。 |
| USERNAME | SFTP用户名 | 可选必选说明：该参数在<br>“REMOTEFLAG(是否进行网络备份)”<br>为<br>“ON(开)”<br>时，为条件必选参数。<br>参数含义：登录远端SFTP服务器的用户名。<br>取值范围：长度不超过128的字符串。<br>默认值：无。<br>配置原则：无。 |
| USERPSWD | SFTP密码 | 可选必选说明：该参数在<br>“REMOTEFLAG(是否进行网络备份)”<br>为<br>“ON(开)”<br>时，为条件必选参数。<br>参数含义：登录远端SFTP服务器的密码。<br>取值范围：长度不超过128的字符串。<br>默认值：无。<br>配置原则：无。 |
| REMOTEPATH | 目标目录 | 可选必选说明：该参数在<br>“REMOTEFLAG(是否进行网络备份)”<br>为<br>“ON(开)”<br>时，为条件必选参数。<br>参数含义：备份数据包保存在远端SFTP服务器的具体路径，填写的SFTP用户名需要有访问该路径的权限，否则网络备份设置失败。<br>取值范围：长度不超过256的字符串。<br>默认值：无。<br>配置原则：无。 |

## [使用实例](#ZH-CN_MMLREF_0000001840859456)

1. 设置自动备份策略：
  ```
  %%SET BKMPOLICY: SCHEDULEFLAG=ON, PERIOD=1, STARTTIME=03&00, VALIDPERIOD=7, REMOTEFLAG=ON, COMPATIBLE=ON, CHECKNETWORKALARM=ON, REMOTEVALIDPERIOD=30, SFTPIPTYPE=IPV4, IPV4="10.0.0.0", SFTPPORT=22, USERNAME="user", USERPSWD="*****", REMOTEPATH="/home";%%
  RETCODE = 0  操作成功

  ---    END
  ```

#### [命令功能](#ZH-CN_CONCEPT_0182837449)

**适用NF：PGW-U、UPF**

该命令用来查询Tethering用户终端数量检测全局开关。

#### [注意事项](#ZH-CN_CONCEPT_0182837449)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0182837449)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0182837449)

无。

#### [使用实例](#ZH-CN_CONCEPT_0182837449)

查询Tethering用户终端数量检测全局开关：

```
LST TETHERDETGLBSW:;
```

```

RETCODE = 0  操作成功。

Tethering 用户终端数量检测全局开关
-------------------------------------------------
开关表述  =  开启
(结果个数 = 1)
---    END
```

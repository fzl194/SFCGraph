#### [命令功能](#ZH-CN_CONCEPT_0000001549801426)

该命令用于查询无线路由与对端平滑开始、结束时间以及平滑状态机类型和状态。

#### [注意事项](#ZH-CN_CONCEPT_0000001549801426)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549801426)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549801426)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| AFTYPE | 地址族 | 可选必选说明：必选参数<br>参数含义：该参数用于表示地址族。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- ipv4unicast：IPv4单播。<br>- ipv6unicast：IPv6单播。<br>默认值：无 |
| PEERIP | 对端地址 | 可选必选说明：可选参数<br>参数含义：该参数用于表示对端地址。<br>数据来源：对端协商<br>取值范围：字符串类型，输入长度范围为1～64。IP地址。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549801426)

显示无线路由平滑状态机的状态：

```
DSP WLRSMOOTHSTAT:AFTYPE=ipv4unicast;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001549801426)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 地址族 | 用于表示地址族。 |
| 对端地址 | 用于表示对端地址。 |
| 状态机类型 | 该参数用于表示平滑状态机类型：<br>- ROUTE：路由状态机。<br>- PAEG：PAEG状态机。<br>- FLOWRULE：引流表状态机。 |
| 状态机状态 | 该参数用于表示平滑状态机状态：<br>- INIT：初始状态，表示状态机未启动。<br>- REG：注册状态，表示对端发起注册。<br>- BATCH：批备状态，表示对端发起平滑数据。<br>- NORMAL：完成状态，表示本轮平滑结束。 |
| 上次开始时间 | 用于表示上次平滑开始时间。 |
| 上次结束时间 | 用于表示上次平滑结束时间。 |

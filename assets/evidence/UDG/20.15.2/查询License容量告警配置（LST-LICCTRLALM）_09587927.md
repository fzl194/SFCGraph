## [命令功能](#ZH-CN_MMLREF_0209587927)

该命令用于查询系统中License容量告警配置信息。

> **说明**
> 该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0209587927)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209587927)

无

## [使用实例](#ZH-CN_MMLREF_0209587927)

查询系统中当前的容量告警配置信息：

```
%%LST LICCTRLALM:;%%
RETCODE = 0  操作成功

操作结果如下
------------
    容量告警百分比(%)  =  80
容量告警恢复百分比(%)  =  70
          持续时间(h)  =  3
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209587927)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 容量告警百分比(%) | 该参数用于指定容量告警百分比。当系统中已使用的资源控制项数目在“持续时间”内的平均值与License允许的数目的比值大于该参数设定的值时，系统将产生扩容告警。 |
| 容量告警恢复百分比(%) | 该参数用于指定容量告警恢复百分比。系统已经产生该资源控制项的扩容告警后，当已使用的资源控制项数目在“持续时间”内的平均值与License允许的数目的比值小于该参数设定的值时，系统将恢复该扩容告警。 |
| 持续时间(h) | 该参数用于指定系统中计算已使用的资源控制项数目平均值的“持续时间”，当该参数设置的时间内的平均值与License允许的数目的比值大于系统设定的“容量告警百分比”时，系统将产生扩容告警。 |

#### [命令功能](#ZH-CN_CONCEPT_0182837424)

**适用NF：SGW-U、PGW-U、UPF**

该命令用于修改指定时间段下，特定序号的绝对时间段信息，如果运营商需要修改系统的业务生效的绝对时间段，则用此命令进行设置。

#### [注意事项](#ZH-CN_CONCEPT_0182837424)

- 该命令执行后立即生效。
- 绝对时间段的开始时间不能大于等于结束时间。
- 绝对时间段的结束时间不能小于等于系统当前时间。
- 如果时间段被BwmRule绑定，则修改后的绝对时间段配置不能与BwmRule绑定的其它时间段下的绝对时间段配置重叠。
- 如果时间段被BwmRule绑定，则修改后的绝对时间段时间间隔不能小于30分钟。
- 对于PCC用户，如果在时间段内出现夏令时切换或系统时间修改，以TimeRange开始时间为准，按照结束时间与开始时间之差（不考虑夏令时）标识的时间长度计算结束时间。

#### [操作用户权限](#ZH-CN_CONCEPT_0182837424)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0182837424)

| 参数标识 | 参数名称 | 参数说明 |

| STARTDATE | 起始日期，格式MM/DD/YYYY | 可选必选说明：可选参数<br>参数含义：该参数用于设置绝对时间段的开始日期。<br>数据来源：本端规划<br>取值范围：日期类型，输入格式是MM/DD/YYYY。<br>默认值：无<br>配置原则：无 |
| ENDTIME | 结束时间，格式HH:MM | 可选必选说明：可选参数<br>参数含义：该参数用于设置绝对时间段的结束时间。只输入结束时间不输入开始时间时，表明系统的业务立即生效。如果不设置该值，表明系统的业务从开始时间起一直生效。<br>数据来源：本端规划<br>取值范围：时间类型，输入格式是HH:MM。<br>默认值：无<br>配置原则：无 |
| ENDDATE | 结束日期，格式MM/DD/YYYY | 可选必选说明：可选参数<br>参数含义：该参数用于设置绝对时间段的结束日期。<br>数据来源：本端规划<br>取值范围：日期类型，输入格式是MM/DD/YYYY。<br>默认值：无<br>配置原则：无 |
| CFGDOMAINNAME | 配置域名称 | 可选必选说明：可选参数<br>参数含义：该参数表示命令所属公共配置域的名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。<br>默认值：无<br>配置原则：输入单空格将删除该参数已有配置项。 |

#### [使用实例](#ZH-CN_CONCEPT_0182837424)

假设运营商需要修改系统的业务在2015年9月20日的10点到2015年9月20日的11点生效，则按如下命令进行配置：

```
MOD ABSTIMERANGE:TIMERANGENAME="t1",ABSTIMERANGESEQ=1,STARTTIME=10&00,STARTDATE=2015&09&20,ENDTIME=11&00,ENDDATE=2015&09&20;
```

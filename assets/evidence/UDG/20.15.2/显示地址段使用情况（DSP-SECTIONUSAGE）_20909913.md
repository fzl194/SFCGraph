#### [命令功能](#ZH-CN_CONCEPT_0000201320909913)

**适用NF：PGW-U、UPF**

此命令用于显示地址段使用情况的信息。

#### [注意事项](#ZH-CN_CONCEPT_0000201320909913)

- 查询结果相对地址实际释放的时间有1-5秒的延迟。
- 该命令只支持本地地址池下的地址段使用情况查询。

#### [操作用户权限](#ZH-CN_CONCEPT_0000201320909913)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000201320909913)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| POOLNAME | 地址池名称 | 可选必选说明：必选参数<br>参数含义：地址池名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～79。<br>默认值：无<br>配置原则：该参数使用ADD POOL命令配置生成。 |
| SECTIONNUM | 地址段号 | 可选必选说明：必选参数<br>参数含义：该参数用于指定配置Section的编号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～63。<br>默认值：无<br>配置原则：该参数使用ADD SECTION命令配置生成。 |

#### [使用实例](#ZH-CN_CONCEPT_0000201320909913)

查询地址池名称为pool1的地址池，地址段号为0的使用率信息：

```
DSP SECTIONUSAGE: POOLNAME="pool1", SECTIONNUM=0;
```

```

-------------------------------
                  地址池名称  =  pool1
                    地址段号  =  0
                等待释放IP数  =  0

           地址段使用率（%）  =  0

```

#### [输出结果说明](#ZH-CN_CONCEPT_0000201320909913)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 地址池名称 | 地址池名称。 |
| 地址段号 | 用于指定配置Section的编号。 |
| 等待释放IP数 | 本地地址池等待释放IPv4地址/IPv6前缀个数，如果IP地址租约功能使能，则也是租约IPv4地址/IPv6前缀个数。 |
| 地址总数 | 地址段中IPv4地址/IPv6前缀的总数。 |
| 地址使用数 | 地址段中IPv4地址/IPv6前缀的使用数。 |
| 地址无效数 | 本地地址段中无效的IPv4地址/IPv6前缀数。 |
| 锁定地址数 | 地址段中地址被锁定的IP数。 |
| 地址使用数(锁定地址) | 分配给UE但后被锁定的地址数。 |
| 地址使用数(冲突地址) | 分配给UE但是后来被置为冲突的地址数。 |
| 地址段使用率（%） | 地址段的IPv4地址/IPv6前缀的占用率。 |

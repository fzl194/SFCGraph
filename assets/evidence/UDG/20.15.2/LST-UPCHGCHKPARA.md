#### [命令功能](#ZH-CN_CONCEPT_0000202968962491)

**适用NF：SGW-U、PGW-U、UPF**

用于查询用户面计费检查相关参数。

#### [注意事项](#ZH-CN_CONCEPT_0000202968962491)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000202968962491)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000202968962491)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000202968962491)

使用LST MULTIDNNPARA命令查询用户面计费检查相关参数：

```
LST UPCHGCHKPARA:;
```

```

RETCODE = 0 操作成功

计费检查参数
----------------------
用户资源告警开关 = 否
流量核查告警开关 = 否
流量核查告警上报阈值 = 500
流量核查告警恢复阈值 = 300
(结果数 = 1)
```

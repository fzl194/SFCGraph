## [命令功能](#ZH-CN_MMLREF_0000001599126968)

本命令用于查询系统日志。

该命令获取的日志与 OM Portal 界面 “ 安全>日志审计>系统日志 ” 相同。

> **说明**
> 查询系统日志最多显示1000条。

## [参数说明](#ZH-CN_MMLREF_0000001599126968)

| **参数标识** | **参数名称** | **参数说明** |
| --- | --- | --- |
| STARTTIME | 开始时间 | 可选必选说明：可选参数。<br>参数含义：用于命令按照系统日志开始时间来查询系统日志；若不输入，则表示查询前1000条系统日志。<br>取值范围：1990/01/01 00:00:00-2037/12/31 23:59:59。<br>默认值：无。<br>配置原则：无。 |
| ENDTIME | 结束时间 | 可选必选说明：可选参数。<br>参数含义：用于命令按照系统日志结束时间来查询系统日志；若不输入，则表示查询前1000条系统日志。<br>取值范围：1990/01/01 00:00:00-2037/12/31 23:59:59。<br>默认值：无。<br>配置原则：无。 |
| MENAME | 网元名称 | 可选必选说明：可选参数。<br>参数含义：用于命令按照网元名称来查询系统日志，可通过<br>[LST ME](../../系统管理/版本信息/查询网元配置信息（LST ME）_47084797.md)<br>命令获取；若不输入，则表示查询前1000条系统日志。<br>取值范围：字符串类型，输入长度范围为0~64。<br>默认值：无。<br>配置原则：无。 |
| EVENTNAME | 事件名称 | 可选必选说明：可选参数。<br>参数含义：用于命令按照事件名称来查询系统日志；若不输入，则表示查询前1000条系统日志。<br>取值范围：字符串类型，输入长度范围为0~64。<br>默认值：无。<br>配置原则：无。 |
| LEVEL | 日志级别 | 可选必选说明：可选参数。<br>参数含义：用于命令按照日志级别来查询系统日志；若不输入，则表示查询前1000条系统日志。<br>取值范围：<br>- Emergency(紧急)<br>- Alert(警报)<br>- Critical(严重)<br>- Error(错误)<br>- Warning(警告)<br>- Notice(通知)<br>- Information(信息)<br>- Debug(调试)<br>默认值：无。<br>配置原则：无。 |
| RESULT | 事件结果 | 可选必选说明：可选参数。<br>参数含义：用于命令按照事件结果状态来查询系统日志；若不输入，则表示查询前1000条系统日志。<br>取值范围：<br>- Successful(成功)<br>- Failed(失败)<br>- PartiallySuccessful(部分成功)<br>- Unknown(未知)<br>默认值：无。<br>配置原则：无。 |

## [使用实例](#ZH-CN_MMLREF_0000001599126968)

查询系统日志：

LST SYSTEMLOG:;

```
%%LST SYSTEMLOG:;%% 
RETCODE = 0  操作成功

结果信息 
-------- 
事件名称             日志级别    网元名称     开始时间                 结束时间                 服务名称   事件结果  详细信息                                                                                      
OM网络检测           严重        csp          2023-07-15 19:36:11.851  2023-07-15 19:36:11.851  OMLB       失败      OM网络资源故障，可能导致的原因是网络丢包，IP地址=10.0.0.0，网关IP地址=10.0.0.0
dmsetup              错误        csp          2023-07-21 07:30:36.697  2023-07-21 07:30:36.697  OS         失败      Device does not exist.

(结果个数 = 2)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001599126968)

该命令执行正常，会返回命令执行成功的提示信息和相关参数，各参数含义请参见如 [表1](#ZH-CN_MMLREF_0000001599126968__table47787675) 所示参数说明。

该命令执行异常，会返回对应的错误码。常见的错误码如 [表2](#ZH-CN_MMLREF_0000001599126968__table0350152692312) 所示。

*表1 输出项解释*

| **输出项名称** | **输出项解释** |
| --- | --- |
| 事件名称 | 用于表示产生系统日志的事件名称。 |
| 日志级别 | 用于表示产生系统日志的级别。 |
| 开始时间 | 用于表示产生系统日志的开始时间。 |
| 结束时间 | 用于表示产生系统日志的结束时间。 |
| 网元名称 | 用于表示产生系统日志的网元名称。 |
| 服务名称 | 用于表示产生系统日志的服务名称。 |
| 事件结果 | 用于表示产生系统日志的结果：成功、失败、部分成功或者未知。 |
| 详细信息 | 用于表示系统日志的详细信息，方便维护人员定位分析该事件对于系统的影响。 |

*表2 错误码列表*

| 错误码 | 错误码解释 | 原因分析 | 处理建议 |
| --- | --- | --- | --- |
| 101100 | 请求参数不合法 | 请求参数不合法。 | 请检查输入参数并重新输入。 |
| 101108 | 系统内部错误 | 系统内部错误。 | 请联系华为技术支持。 |

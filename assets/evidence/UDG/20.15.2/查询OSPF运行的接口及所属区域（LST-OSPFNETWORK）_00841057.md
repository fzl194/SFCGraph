#### [命令功能](#ZH-CN_CONCEPT_0000001600841057)

该命令用于查询在OSPF进程下的区域网段的配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001600841057)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600841057)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600841057)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | 进程号 | 可选必选说明：可选参数<br>参数含义：OSPF进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |
| AREAID | 区域号 | 可选必选说明：可选参数<br>参数含义：OSPF区域号。<br>数据来源：对端协商<br>取值范围：IPv4地址类型。点分十进制格式。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600841057)

查询在OSPF进程下的区域查询网段的配置：

```
LST OSPFNETWORK:;
```

```

RETCODE = 0  操作成功。

结果如下
------------------------
进程号  =  1
区域号  =  0.0.0.0
IP地址  =  192.168.7.0
反掩码  =  0.0.0.255
  描述  =  NULL
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600841057)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 进程号 | 该参数指定用户创建的OSPF进程号。 |
| 区域号 | 该参数指定用户创建的OSPF区域号。 |
| IP地址 | 该参数指定接口所在的网段地址。 |
| 反掩码 | 该参数指定IP地址的反码，相当于将IP地址的掩码反转（0变1，1变0）。其中，“1”表示忽略IP地址中对应的位，“0”表示必须保留此位。 |
| 描述 | 该参数指定OSPF网段的描述信息。 |

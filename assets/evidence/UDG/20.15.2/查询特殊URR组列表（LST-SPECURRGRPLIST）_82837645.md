#### [命令功能](#ZH-CN_CONCEPT_0182837645)

**适用NF：PGW-U、UPF**

该命令用于查询当前系统的特殊使用量上报规则组列表配置信息。

#### [注意事项](#ZH-CN_CONCEPT_0182837645)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0182837645)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0182837645)

无。

#### [使用实例](#ZH-CN_CONCEPT_0182837645)

查询特殊URR组列表配置信息：

```
LST SPECURRGRPLIST:;
```

```

RETCODE = 0 操作成功。

特殊URR组列表信息：
----------------------------------
使用量上报规则组名称 = aaa
         配置域名称  =  NULL
(结果个数 = 1)
--- END
```

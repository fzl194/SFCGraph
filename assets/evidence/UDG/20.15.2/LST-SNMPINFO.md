#### [命令功能](#ZH-CN_TOPIC_0186563519)

本命令用于查询SNMP服务建立的长连接信息，本命令不需要输入任何参数。

#### [注意事项](#ZH-CN_TOPIC_0186563519)

无。

#### [参数说明](#ZH-CN_TOPIC_0186563519)

无。

#### [使用实例](#ZH-CN_TOPIC_0186563519)

1. 查询网元长连接信息（网管网元建立长连接）。
  ```
  %%LST SNMPINFO:;%%
  RETCODE = 0  操作成功  
  操作结果如下 
  ------------ 
  订阅者IP       订阅ID    订阅时间                  订阅者外出端口    
  10.128.0.8     1         2019-08-15 17:59:50       8001
  10.128.0.9     2         2019-08-15 17:59:55       8002
      
  (结果个数 = 2)  
  ---    END
  ```
2. 查询网元长连接信息（网管网元没有建立长连接）。
  ```
  %%LST SNMPINFO:;%% 
  RETCODE = 0  操作成功
  没有查到相应结果  
  ---    END
  ```

#### [输出结果说明](#ZH-CN_TOPIC_0186563519)

命令执行正常，会返回命令执行成功的提示信息。输出结果说明如 [表1](#ZH-CN_TOPIC_0186563519__table782715528281) 所示。

命令执行异常，请联系华为技术支持处理。

*表1 输出项解释*

| 输出项名称 | 输出项解释 |
| --- | --- |
| 订阅者IP | 订阅者和SNMP建立连接的IP地址信息。 |
| 订阅者ID | 订阅者编号，类序列号。 |
| 订阅时间 | 订阅者向SNMP发起订阅并成功订阅的时间。 |
| 订阅者外出端口 | SNMP和某一订阅者通信的内部端口，取值范围为8001-8010。 |

#### [命令功能](#ZH-CN_CONCEPT_0182837720)

**适用NF：PGW-U、UPF**

该命令用于查询系统当前使用的协议解析数据库的版本信息。

#### [注意事项](#ZH-CN_CONCEPT_0182837720)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0182837720)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0182837720)

无。

#### [使用实例](#ZH-CN_CONCEPT_0182837720)

查询协议解析库：

```
DSP PARSERDB:;
```

```

RETCODE = 0  操作成功。

协议解析数据库信息
------------------------------------
解析库加载模式  =  最新解析数据库信息
　　　　版本号  =  009.013.010.003
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0182837720)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 版本号 | 用于指定加载的解析库的版本号。 |

其余输出项请参见LOD PARSERDB的参数说明。

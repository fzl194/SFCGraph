#### [命令功能](#ZH-CN_CONCEPT_0000001550280554)

该命令用于查询接口的借用IPv4地址。

不指定IFNAME参数时，查询所有接口的IPv4借用地址；当指定IFNAME参数时，可以查询指定接口的IPv4借用地址。

#### [注意事项](#ZH-CN_CONCEPT_0000001550280554)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550280554)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550280554)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IFNAME | 接口名 | 可选必选说明：可选参数<br>参数含义：该参数用于指定接口名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。接口名称由接口类型+接口编号组成。<br>默认值：无<br>配置原则：请使用LST INTERFACE命令查看可用接口。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001550280554)

查询tunnel口借用的IPv4地址：

```
LST IFIPV4UNNUM:IFNAME="Tunnel1";
```

```

RETCODE = 0  操作成功

结果如下
--------
      接口名  =  Tunnel1
被借用接口名  =  LoopBack1
(结果个数 = 1)
---    END
```

# 删除OSPF默认宣告路由配置（RMV OSPFDEFAULTROUTE）

- [命令功能](#ZH-CN_CONCEPT_0000001550281658__1.3.1.1)
- [注意事项](#ZH-CN_CONCEPT_0000001550281658__1.3.2.1)

- [使用实例](#ZH-CN_CONCEPT_0000001550281658__1.3.5.1)

#### [命令功能](#ZH-CN_CONCEPT_0000001550281658)

该命令用于取消缺省路由通告到普通OSPF区域。

#### [注意事项](#ZH-CN_CONCEPT_0000001550281658)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550281658)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550281658)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | 进程号 | 可选必选说明：必选参数<br>参数含义：OSPF进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |
| TOPOID | 拓扑标识 | 可选必选说明：必选参数<br>参数含义：拓扑标识。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围为0。<br>默认值：无<br>配置原则：目前只支持默认拓扑0。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001550281658)

取消缺省路由通告到普通OSPF进程1区域：

```
RMV OSPFDEFAULTROUTE:PROCID=1,TOPOID=0;
```

# 初始配置

合一ProxyUPF/ProxySGW-U/PGW-U配置操作流程与UDG产品的初始配置流程基本相同（仅“ **配置网元和网管对接** ”章节有区别），如 [图1](#ZH-CN_TOPIC_0000001980688502__fig327733361417) 所示，详细配置步骤请参见“UDG产品文档”>“网络部署”>“初始配置”手册。参考手册，配置N9c/N9a/N16c/N16a/SGi等接口的数据，为V-SMF/V-UPF与H-SMF/H-UPF之间的数据路径建立逻辑连接。

在“ **配置网元和网管对接** ”章节基础上，需要增加网元形态配置，用于对接北向。如果已对接网管，无需重新配置。

//配置网元形态为合一ProxyUPF/ProxySGW-U/PGW-U。

```
SET NETYPE: NETYPEENUM=GGSNU-0&PGWU-1&ProxySGWU-1&SGWU-0&UPF-0&ProxyUPF-1;
```

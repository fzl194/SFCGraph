#### [命令功能](#ZH-CN_CONCEPT_0000001600601349)

该命令用于显示DHCPv4客户端报文统计信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600601349)

该命令在Ethernet接口上配置执行。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600601349)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600601349)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IFNAME | 接口名称 | 可选必选说明：可选参数<br>参数含义：该参数用于显示接口名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。<br>默认值：无<br>配置原则：请使用LST INTERFACE命令查看可用接口。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600601349)

显示DHCPv4客户端报文统计信息：

```
DSP DHCPCLIENTSTC: IFNAME="Ethernet64/0/4";
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600601349)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 接口名称 | 用于显示接口名称。 |

| 发送请求报文个数 | 用于显示发送的requeset报文个数。 |

| 发送续租状态下请求报文个数 | 用于显示发送的renew requeset报文个数。 |
| 发送重绑定状态下请求报文个数 | 用于显示发送的rebind requeset报文个数。 |
| 发送地址冲突报文个数 | 用于显示发送的decline报文个数。 |
| 发送地址释放报文个数 | 用于显示发送的release报文个数。 |
| 接收ACK报文个数 | 用于显示接收的ACK报文个数。 |
| 接收NAK报文个数 | 用于显示接收的NAK报文个数。 |
| 收到报文个数 | 用于显示接收的报文总数。 |
| 发出报文个数 | 用于显示发送的报文总数。 |

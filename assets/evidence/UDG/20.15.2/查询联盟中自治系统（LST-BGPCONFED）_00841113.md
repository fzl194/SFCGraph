#### [命令功能](#ZH-CN_CONCEPT_0000001600841113)

该命令用于查询联盟中子自治系统号。

#### [注意事项](#ZH-CN_CONCEPT_0000001600841113)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600841113)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600841113)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| CONFEDASNUMBER | 联盟对等体的自治域号 | 可选必选说明：可选参数<br>参数含义：该参数用于指定与本地AS连接的其他EBGP对等体所属的子自治系统号。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围为1～11。取值范围是number<1-4294967295>或者number<1-65535>.number<0-65535>。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600841113)

查询联盟中子自治系统号：

```
LST BGPCONFED:;
```

```

RETCODE = 0  操作成功。

结果如下
-------------------------
联盟对等体的自治域号  =  65002
(结果个数 = 1)
---    END
```

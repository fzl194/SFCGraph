## [操作场景](#ZH-CN_OPI_0000001088299168)

UDG 通过对用户三四层数据报文进行分析和检测，识别出TCP重传报文，并根据配置对重传报文流量实施灵活的计费策略。

> **说明**
> 适用于SGW-U、PGW-U、UPF。

## [必备事项](#ZH-CN_OPI_0000001088299168)

前提条件

请仔细阅读 [GWFD-020307 TCP重传识别](../GWFD-020307 TCP重传识别_88170678.md) 。

数据

详细配置请参考 [部署UPF](../GWFD-020301 内容计费基本功能/部署UPF_28493406.md) ，此处仅描述差异配置。

## [操作步骤](#ZH-CN_OPI_0000001088299168)

1. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09587387.md)
2. 请参考 [部署UPF](../GWFD-020301 内容计费基本功能/部署UPF_28493406.md) 配置，配置的差异在于UserProfile绑定TCP重传使用量上报规则组，为用户模板添加TCP重传使用量上报规则，并打开TCP重传计费。
  **[SET URRGRPBINDING](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务匹配公共配置/业务规则管理/用户模板/设置用户模板的URR组绑定关系（SET URRGRPBINDING）_82837281.md)**

## [任务示例](#ZH-CN_OPI_0000001088299168)

任务描述

UDG 将TCP重传业务报文记入URR ID为1100的URR中，进行离线计费。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH:LICITEM="LKV3G5TCPR01",SWITCH=ENABLE;
```

//参考 [部署UPF](../GWFD-020301 内容计费基本功能/部署UPF_28493406.md) 配置，配置的差异在于UserProfile绑定TCP重传使用量上报规则组。

```
SET URRGRPBINDING:USERPROFILENAME="userprofile001", TCPCHGPROFLAG=CHARGING, TCPURRGRPNAME="cp_tcp";
```

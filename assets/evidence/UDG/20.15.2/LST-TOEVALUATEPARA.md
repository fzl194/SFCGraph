#### [命令功能](#ZH-CN_CONCEPT_0000201308865124)

**适用NF：PGW-U、UPF**

该命令用于查询TCP优化评估配置。

#### [注意事项](#ZH-CN_CONCEPT_0000201308865124)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000201308865124)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000201308865124)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000201308865124)

查询设置TCP优化评估配置：

```
LST TOEVALUATEPARA:;
```

```

RETCODE = 0 操作成功。

结果如下
------------------------
TCP优化评估开关 = Enable
TCP优化评估抽样比率 = 50
(结果个数 = 1)
--- END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000201308865124)

参见SET TOEVALUATEPARA的参数说明。

#### [命令功能](#ZH-CN_CONCEPT_0000204449785469)

**适用NF：SGW-U、PGW-U、UPF**

该命令用来设置系统是否开启专网UPF动态分流功能。

#### [注意事项](#ZH-CN_CONCEPT_0000204449785469)

- 该命令执行后只对新激活用户生效。
- 该命令最大记录数为1。
- 该命令存在系统初始记录，参数的初始设置值如下表：

| 参数标识 | ULCLSWITCH |
| --- | --- |
| 初始值 | DISABLE |

#### [操作用户权限](#ZH-CN_CONCEPT_0000204449785469)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000204449785469)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| ULCLSWITCH | 专网UPF动态分流功能开关 | 可选必选说明：必选参数<br>参数含义：该参数用于指定系统是否开启专网UPF动态分流功能。<br>数据来源：本端规划<br>取值范围：<br>- DISABLE：不使能。<br>- ENABLE：使能。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000204449785469)

设置系统开启专网UPF动态分流功能，可以使用该命令设置：

```
SET DEDICATEDLBO: ULCLSWITCH= ENABLE;
```

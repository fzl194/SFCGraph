#### [命令功能](#ZH-CN_CONCEPT_0000206297080145)

**适用NF：UPF**

该命令用于查询Diameter TW定时器。

#### [注意事项](#ZH-CN_CONCEPT_0000206297080145)

该命令相关的功能当前版本暂不支持。

#### [操作用户权限](#ZH-CN_CONCEPT_0000206297080145)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000206297080145)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000206297080145)

查询Diameter TW定时器，则可按如下配置：

```
LST UPTWTIMER:;
```

```

RETCODE = 0  操作成功。
TW定时器配置
------------
      TW定时器时长（秒）  =  3
 DWR消息响应超时次数上限  =  3
超时次数达到上限复位连接  =  复位TCP
(结果个数 = 1)
---    END
```

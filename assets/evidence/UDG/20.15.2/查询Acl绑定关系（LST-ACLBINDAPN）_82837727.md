#### [命令功能](#ZH-CN_CONCEPT_0182837727)

**适用NF：SGW-U、PGW-U、UPF**

该命令用于查询指定APN下所有方向绑定的ACL信息。

#### [注意事项](#ZH-CN_CONCEPT_0182837727)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0182837727)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0182837727)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| APN | APN | 可选必选说明：必选参数<br>参数含义：该参数用于指定APN实例名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。只能由“-”、数字、大小写字母和“.”组成，不能以“.”开头且不能出现连续两个“.”。不支持空格及“_”、“#”、“$”、“&”、“%”、“^”、“（”、“）”、“，”、“/”、“;”、“:”、“””、“`”特殊字符，不区分大小写。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0182837727)

假如运营商需要查询APN“testapn”下已绑定的ACL信息：

```
LST ACLBINDAPN:APN="testapn";
```

```

RETCODE = 0  操作成功。

ACL绑定信息
-----------
       APN  =  testapn
      方向  =  上行入
   ACL名称  =  testacl1
时间段名称  =  NULL
(结果个数 = 1)
---    END
```

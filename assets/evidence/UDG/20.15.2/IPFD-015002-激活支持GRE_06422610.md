## [操作场景](#ZH-CN_OPI_0206422610)

本操作指导介绍在运行网络中创建GRE隧道的操作过程。

通用路由封装GRE（Generic Routing Encapsulation）是对某些网络层协议的报文进行封装，使这些被封装的报文能够在另一网络层协议（如IP）中传输。GRE可以作为VPN的第三层隧道协议，为VPN数据提供透明传输通道。

## [对系统的影响](#ZH-CN_OPI_0206422610)

该操作对系统正常运行无影响。

## [必备事项](#ZH-CN_OPI_0206422610)

前提条件

- 隧道源端口和目的端口已路由可达。
- 待创建的GRE隧道不存在。
- 如果运营商规划采用VPN组网方式，则操作员在执行本操作前应已创建了相应的VPN实例。

## [操作步骤](#ZH-CN_OPI_0206422610)

1. 配置LoopBack接口。
  在 “MML命令行-UDG” 窗口上执行：
  [**ADD INTERFACE**](../../../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/接口管理/接口配置/增加接口（ADD INTERFACE）_49960870.md) :IFNAME="接口名";
  [**ADD IFIPV4ADDRESS**](../../../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/接口管理/IPv4地址/增加接口IPv4地址（ADD IFIPV4ADDRESS）_00865509.md) : IFNAME="接口名", IFIPADDR="接口IPv4地址", SUBNETMASK="接口IPv4地址掩码", ADDRTYPE=接口IPv4地址类型;
2. 配置Tunnel接口。
  在 “MML命令行-UDG” 窗口上执行：
  [**ADD GRETUNNEL**](../../../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/VPN管理/GRE管理/GRE隧道/增加GRE隧道（ADD GRETUNNEL）_00841729.md) :TNLNAME="隧道名称", TNLTYPE=隧道类型, SRCTYPE= IPv4 源类型, SRCIPADDR=源IP v4 地址, DSTIPADDR="目的IP v4 地址";
  [**ADD IFIPV4ADDRESS**](../../../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/接口管理/IPv4地址/增加接口IPv4地址（ADD IFIPV4ADDRESS）_00865509.md) :IFNAME="接口名", IFIPADDR="接口IPv4地址", SUBNETMASK="接口IPv4地址掩码", ADDRTYPE=接口IPv4地址类型;
  > **说明**
  > 为了不影响GRE隧道收发报文性能，建议使用 [**MOD INTERFACE**](../../../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/接口管理/接口配置/修改接口（MOD INTERFACE）_50281674.md) 命令配置GRE Tunnel的MTU值，该MTU值小于等于（出接口的MTU值-GRE头的长度），GRE头封装格式为：GRE header + 外层IP header。
3. 配置隧道间静态路由
  在 “MML命令行-UDG” 窗口上执行：
  **[**ADD SRROUTE**](../../../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/路由管理/静态路由管理/IPv4静态路由/增加IPv4静态路由（ADD SRROUTE）_49961498.md)** : AFTYPE=地址族, PREFIX="路由前缀", MASKLENGTH=路由掩码长度, IFNAME="路由接口名字";
4. （可选）当配置GRE隧道时，使能隧道端到端校验功能。
  在 “MML命令行-UDG” 窗口上执行：
  [**ADD GRETUNNEL**](../../../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/VPN管理/GRE管理/GRE隧道/增加GRE隧道（ADD GRETUNNEL）_00841729.md) :TNLNAME="隧道名称", TNLTYPE=隧道类型, CHECKSUMEN=使能端到端校验功能;
5. （可选）当配置GRE隧道时，使能隧道识别关键字功能。
  在 “MML命令行-UDG” 窗口上执行：
  [**ADD GRETUNNEL**](../../../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/VPN管理/GRE管理/GRE隧道/增加GRE隧道（ADD GRETUNNEL）_00841729.md) :TNLNAME="隧道名称", TNLTYPE=隧道类型, GREKEYEN=使能识别关键字功能, GREKEY="识别关键字";
  > **说明**
  > 如果在隧道两端设置识别关键字，则必须指定相同的识别关键字；或隧道两端都不设置识别关键字。
6. （可选）当配置GRE隧道时，使能隧道Keepalive功能。
  在 “MML命令行-UDG” 窗口上执行：
  [**ADD GRETUNNEL**](../../../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/VPN管理/GRE管理/GRE隧道/增加GRE隧道（ADD GRETUNNEL）_00841729.md) :TNLNAME="隧道名称", TNLTYPE=隧道类型, KEEPALVEN=使能Keepalive功能, KEEPALVPERIOD=Keepalive报文发送周期, KEEPALVRETRYCNT=不可达计数器参数;
  > **说明**
  > - 若不填写参数KEEPALVPERIOD，则Keepalive报文发送周期默认为5秒。
  > - 若不填写参数KEEPALVRETRYCNT，则不可达计数器参数默认为3次。

## [任务示例](#ZH-CN_OPI_0206422610)

任务描述

**本端GRE隧道配置：**

创建LoopBack1接口，配置接口源IP地址为10.3.3.11/32，目的IP地址为10.4.4.11。

创建Tunnel1接口，Tunnel接口为“GRE”隧道模式。配置接口源IP地址为10.10.1.201/24，目的地址为10.10.1.202。

**对端GRE隧道配置：**

创建LoopBack1接口，配置接口源IP地址为10.4.4.11/32，目的IP地址为10.3.3.11。

创建Tunnel1接口，Tunnel接口为“GRE”隧道模式。配置接口源IP地址为10.10.1.202/24，目的地址为10.10.1.201。

> **说明**
> 隧道的源接口IP地址和目的IP地址分别对应GRE报文头中的源IP地址和目的IP地址。

**图1** 激活GRE特性组网

<br>

![](激活支持GRE_06422610.assets/zh-cn_image_0206422614.png)

脚本

**本端GRE隧道配置脚本：**

//创建LoopBack接口，并配置IP地址10.3.3.11/32。

```
ADD INTERFACE:IFNAME="LoopBack1";
```

```
ADD IFIPV4ADDRESS:IFNAME="LoopBack1", IFIPADDR="10.3.3.11", SUBNETMASK="255.255.255.255", ADDRTYPE=main;
```

//创建Tunnel接口，并配置IP地址10.10.1.201/24。

```
ADD GRETUNNEL:TNLNAME="Tunnel1", TNLTYPE=gre, SRCTYPE=if_name, SRCIFNAME="LoopBack1", DSTIPADDR="10.4.4.11";
```

```
ADD IFIPV4ADDRESS:IFNAME="Tunnel1", IFIPADDR="10.10.1.201", SUBNETMASK="255.255.255.0", ADDRTYPE=main;
```

//增加隧道间静态路由

```
ADD SRROUTE:AFTYPE=ipv4unicast,PREFIX="10.10.1.202",MASKLENGTH=24,IFNAME="Tunnel1";
```

//使能隧道端到端校验功能。

```
MOD GRETUNNEL:TNLNAME="Tunnel1", TNLTYPE=gre, CHECKSUMEN=TRUE;
```

//使能隧道识别关键字功能。

```
MOD GRETUNNEL:TNLNAME="Tunnel1", TNLTYPE=gre, GREKEYEN=TRUE, GREKEY="123";
```

//使能隧道Keepalive功能。

```
MOD GRETUNNEL:TNLNAME="Tunnel1", TNLTYPE=gre, KEEPALVEN=TRUE, KEEPALVPERIOD=5, KEEPALVRETRYCNT=3;
```

**对端GRE隧道配置脚本：**

//创建LoopBack接口，并配置IP地址10.4.4.11/32。

```
ADD INTERFACE:IFNAME="LoopBack1";
```

```
ADD IFIPV4ADDRESS:IFNAME="LoopBack1", IFIPADDR="10.4.4.11", SUBNETMASK="255.255.255.255", ADDRTYPE=main;
```

//创建Tunnel接口，并配置IP地址10.10.1.202/24。

```

```

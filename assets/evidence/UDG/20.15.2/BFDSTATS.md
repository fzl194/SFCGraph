#### [命令功能](#ZH-CN_CONCEPT_0000001600440309)

该命令用于显示BFD会话的统计信息。可以按照本地描述符或者目的地址过滤。

#### [注意事项](#ZH-CN_CONCEPT_0000001600440309)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600440309)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600440309)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| LOCALDISCR | 本地标识符 | 可选必选说明：可选参数<br>参数含义：该参数表示会话本地描述符。用于唯一标识本端会话。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～49152。<br>默认值：无 |
| DESTADDR | 目的IP地址 | 可选必选说明：可选参数<br>参数含义：该参数表示会话检测链路的目的地址。<br>数据来源：对端协商<br>取值范围：字符串类型，输入长度范围为1～128。IPv4或者IPv6地址。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600440309)

显示本地描述符为222会话的统计信息：

```
DSP BFDSTATS:LOCALDISCR=222;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600440309)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 本地标识符 | 表示会话本地描述符。用于唯一标识本端会话。 |
| 收到的报文数目 | 表示收到的报文数目。收到的报文数目统计数据是用来定位会话不UP的原因，因此只有Down的时候有统计计数，UP的时候不做统计。 |
| 发送的报文数目 | 表示发送的报文数目。发送的报文数目统计数据是用来定位会话不UP的原因，因此只有Down的时候有统计计数，UP的时候不做统计。 |
| 收到的坏包数目 | 表示收到的错误报文的数目。 |
| 发送的坏包数目 | 表示发送的错误报文的数目。 |
| 会话Down次数 | 表示BFD会话Down的次数。 |
| 会话创建时间 | 表示BFD会话的创建时间。 |
| 上一次Down的时间 | 表示BFD会话上一次Down的时间。 |
| 上一次Up的时间 | 表示BFD会话上一次Up的时间。 |
| 会话创建的时长 | 表示BFD会话创建的时长。 |
| 距离上次会话Down的时长 | 表示BFD会话距离上次会话Down的时长。 |
| 上一次Up持续时长 | 表示BFD会话上一次Up持续时长。 |
| 会话名称 | 表示BFD会话名称。 |

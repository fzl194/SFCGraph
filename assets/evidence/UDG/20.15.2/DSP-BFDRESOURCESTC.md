#### [命令功能](#ZH-CN_CONCEPT_0000001600441293)

该命令用于查询RU上BFD会话资源的统计信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600441293)

BFD将不会联动接口状态。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600441293)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600441293)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001600441293)

查询RU上BFD会话资源统计信息：

```
DSP BFDRESOURCESTC:;
```

```

RETCODE = 0  操作成功

结果如下
----------------
             RU名称  =  VNODE_VNRS_VNFC_IPU_0064
    BFD会话总资源数  =  300
  剩余BFD会话资源数  =  300
(结果个数 = 1)
--- END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600441293)

| 输出项名称 | 输出项解释 |
| --- | --- |
| RU名称 | 表示资源单元名称。 |
| BFD会话总资源数 | 可以配置的BFD会话总个数。 |
| 剩余BFD会话资源数 | 当前剩余可以配置的BFD会话个数。 |

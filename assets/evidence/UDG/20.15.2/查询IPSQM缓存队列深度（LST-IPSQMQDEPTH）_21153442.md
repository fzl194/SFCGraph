#### [命令功能](#ZH-CN_CONCEPT_0000201521153442)

**适用NF：SGW-U、UPF**

该命令用于查询整形队列深度。

#### [注意事项](#ZH-CN_CONCEPT_0000201521153442)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000201521153442)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000201521153442)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000201521153442)

查询整形队列深度：

```
LST IPSQMQDEPTH:;
```

```

RETCODE = 0  操作成功

-----------------

(结果个数 = 1)

---    END
```

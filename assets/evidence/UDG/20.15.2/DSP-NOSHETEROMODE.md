#### [命令功能](#ZH-CN_CONCEPT_0000001574671533)

该命令用来查询NOS异构模式。

在日常的维护时，可使用本命令查询OMU异构开关状态。

该命令仅在 Full-stack 虚机场景下支持。

#### [注意事项](#ZH-CN_CONCEPT_0000001574671533)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001574671533)

G_1，管理员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001574671533)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SERVICEINSTANCE | 服务实例 | 可选必选说明：必选参数<br>参数含义：该参数表示大颗粒服务实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，通过<br>**LST VNFC**<br>命令获取。<br>默认值：无<br>配置原则：只能填写通过<br>**LST VNFC**<br>命令查询到的管理代理标识。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001574671533)

查询NOS异构模式：

```
DSP NOSHETEROMODE:SERVICEINSTANCE="vnfc";
```

```
RETCODE = 0  操作成功

结果如下:
-------------------------
OMU异构开关  =  打开
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001574671533)

| 输出项名称 | 输出项解释 |
| --- | --- |
| OMU异构开关 | 表示OMU的异构开关状态，包括打开和关闭两种状态。 |

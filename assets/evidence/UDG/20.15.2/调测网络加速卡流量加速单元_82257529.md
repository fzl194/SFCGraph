# 调测网络加速卡流量加速单元

- [操作场景](#ZH-CN_OPI_0000001482257529__1.3.1)
- [必备事项](#ZH-CN_OPI_0000001482257529__1.3.2)
- [操作步骤](#ZH-CN_OPI_0000001482257529__1.3.3)

## [操作场景](#ZH-CN_OPI_0000001482257529)

激活网络加速开流量加速单元后，请参考本页面进行调测，确保网络加速开流量加速单元功能正常。

## [必备事项](#ZH-CN_OPI_0000001482257529)

前提条件

已完成 [激活网络加速卡流量加速单元](激活网络加速卡流量加速单元_82336245.md) 。

数据

无。

## [操作步骤](#ZH-CN_OPI_0000001482257529)

1. 执行 **LST LICENSESWITCH** 命令，查询网络加速卡流量加速单元对应的License配置开关是否打开。
  ```
  LST LICENSESWITCH: LICITEM="LKV3G5NPTO01";
  ```
    - “SWITCH”值为“ENABLE”时，调测结束。
    - “SWITCH”值为“DISABLE”时，请执行[步骤 2](#ZH-CN_OPI_0000001482257529__step8545181719331)。
2. 执行 [**SET LICENSESWITCH**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09587387.md) 命令打开本特性对应的License配置开关。
  ```
  SET LICENSESWITCH: LICITEM="LKV3G5NPTO01", SWITCH=ENABLE;
  ```
  重新执行 [步骤 1](#ZH-CN_OPI_0000001482257529__step811720172748) 进行检查。
    - “SWITCH”值为“ENABLE”时，调测结束。
    - “SWITCH”值为“DISABLE”时，请执行[步骤 3](#ZH-CN_OPI_0000001482257529__step09731640103412)。
3. 收集信息并寻求技术支持。
    a. 在 “MML命令行-UDG” 窗口上执行 [**EXP MML**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/配置管理/配置导出管理/导出MML文件（EXP MML）_47200033.md) 命令将当前配置数据导出为MML脚本文件并保存。
    b. 收集并保存上述所有查询信息。
    c. 收集归纳所有信息并联系华为技术支持解决。

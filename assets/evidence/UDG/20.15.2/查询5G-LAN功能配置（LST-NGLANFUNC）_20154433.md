#### [命令功能](#ZH-CN_CONCEPT_0000202020154433)

**适用NF：UPF**

该命令用于查询5G LAN功能开关状态。

#### [注意事项](#ZH-CN_CONCEPT_0000202020154433)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000202020154433)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000202020154433)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000202020154433)

查询5G LAN功能开关：

```
LST NGLANFUNC:;
```

```

返回值 = 0  操作成功

5G LAN 功能配置
-----------------------------
5G LAN 功能开关  =  开启
N19数据转发开关  =  开启
(结果数= 1)

---    结束
```

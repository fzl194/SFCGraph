## [命令功能](#ZH-CN_MMLREF_0000002211530321)

该命令用于显示指定点播视频信息。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000002211530321)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000002211530321)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| OTTNAME | OTT名称 | 可选必选说明：必选参数<br>参数含义：该参数用于表示OTT名称，OTT名称可通过DSP DCSSTATS命令查询。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~256。<br>默认值：无<br>配置原则：无 |
| URL | URL名称 | 可选必选说明：必选参数<br>参数含义：该参数用于表示视频文件的URL名称。URL名称可以通过DSP DCSSTATS命令查询。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~1024。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000002211530321)

显示OTT名称为“douyin”，URL为“aaa.douyincdn.com/fantasy/stream-404357031821050550.mp4”的点播视频信息。

```
%%DSP DCSVODVIDEO: OTTNAME="douyin", URL="aaa.douyincdn.com/fantasy/stream-404357031821050550.mp4";%%
RETCODE = 0  操作成功

结果如下
```

## [输出结果说明](#ZH-CN_MMLREF_0000002211530321)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 实例ID | 该参数用于表示SERVER分配的实例ID。 |
| 文件状态 | 该参数用于表示视频文件状态。<br>取值说明：<br>- FILE_STATUS_EXIST（文件存在）<br>- FILE_STATUS_NOT_EXIST（文件不存在）<br>- FILE_STATUS_NOT_RETURN（文件未返回） |
| 视频创建时间 | 该参数用于表示视频创建时间。 |
| 视频大小（KB） | 该参数用于表示视频大小。 |
| 最后访问时间 | 该参数用于表示指定点播视频的最后访问时间。 |
| 最后核查时间 | 该参数用于表示指定点播视频的最后核查时间。 |
| 最后修改时间 | 该参数用于表示指定点播视频的最后修改时间。 |
| 热度 | 该参数用于表示热度。 |

#### [命令功能](#ZH-CN_CONCEPT_0259104140)

该命令用于显示系统中的活动告警，状态为活动的告警即为活动告警。

#### [注意事项](#ZH-CN_CONCEPT_0259104140)

- 主备OMU均重启后，重启前显示的活动告警将被自动清除。
- 最多支持显示1000条活动告警详细信息，若实际活动记录条数超出此规格，追加输出统计信息，包含输出记录条数和记录总条数。

#### [操作用户权限](#ZH-CN_CONCEPT_0259104140)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0259104140)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SEQUENCE | 告警序列号 | 可选必选说明：可选参数<br>参数含义：告警的序列号，设备为每一个告警分配的实例编号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |
| LEVEL | 告警级别 | 可选必选说明：可选参数<br>参数含义：告警级别。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- Critical：告警级别为紧急。<br>- Major：告警级别为重要。<br>- Minor：告警级别为次要。<br>- Warning：告警级别为提示。<br>默认值：无 |
| STARTTIME | 起始时间 | 可选必选说明：可选参数<br>参数含义：查询起始时间。<br>数据来源：本端规划<br>取值范围：日期时间类型，输入格式是YYYY&MM&DD&HH&NN&SS。<br>默认值：无 |
| ENDTIME | 终止时间 | 可选必选说明：可选参数<br>参数含义：查询终止时间。<br>数据来源：本端规划<br>取值范围：日期时间类型，输入格式是YYYY&MM&DD&HH&NN&SS。<br>默认值：无 |
| STARTALARMID | 起始告警ID | 可选必选说明：可选参数<br>参数含义：起始告警ID。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |
| ENDALARMID | 终止告警ID | 可选必选说明：可选参数<br>参数含义：终止告警ID。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |
| SERVICEINSTANCE | 服务实例 | 可选必选说明：必选参数<br>参数含义：该参数表示大颗粒服务实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，通过LST VNFC命令获取。<br>默认值：无<br>配置原则：只能填写通过LST VNFC命令查询到的管理代理标识。 |

#### [使用实例](#ZH-CN_CONCEPT_0259104140)

- 显示所有活动告警的信息：
  ```
  DSP ACTALM:
  SERVICEINSTANCE="vnfc"
  ;
  ```

  告警的所属VR名  =  Admin-VS
    根源告警标识  =  独立告警

#### [输出结果说明](#ZH-CN_CONCEPT_0259104140)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 告警序列号 | 告警的序列号，设备为每一个告警分配的实例编号。 |
| 告警名称 | 告警名称。 |
| 告警ID | 告警ID。 |
| 告警级别 | 告警级别。 |
| 告警的发生时间 | 告警的发生时间（支持显示夏令时）。 |
| 告警的所属VR名 | 告警所属虚拟路由器的名称。 |
| 根源告警标识 | 根源告警类型标识。有根源告警、非根源告警和独立告警三种标识，告警相关性使能开关状态为开，若某个告警产生会引起其它告警产生，则该告警为根源告警，其它告警为非根源告警，否则该告警为独立告警。 |
| 根源告警序列号 | 告警的根源告警序列号，只有非根源告警该序列号有效。 |
| 告警状态 | 有告警活动、变更、清除三种状态，活动告警为活动状态。 |
| 告警类型 | 告警所属类型：<br>- 通信告警。<br>- 业务质量告警。<br>- 处理出错告警。<br>- 设备告警。<br>- 环境告警。<br>- 电源告警。<br>- 信令告警。<br>- 中继告警。<br>- 软件系统告警。<br>- 运行系统告警。<br>- 机柜系统。<br>- 完整性违背。<br>- 操作违背。<br>- 物理违背。<br>- 安全违背或机制违背。<br>- 时间域违背。 |
| 告警描述信息 | 告警描述信息。 |
| 输出记录条数 | 输出记录条数，活动告警大于1000条时查询结果会被截断，在命令执行的最后打印此参数。 |
| 记录总条数 | 记录总条数，活动告警大于1000条时查询结果会被截断，在命令执行的最后打印此参数。 |

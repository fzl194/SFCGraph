#### [命令功能](#ZH-CN_TOPIC_0000001292313044)

![](跟踪路由（OMTRACERT）_92313044.assets/notice_3.0-zh-cn.png)

跟踪路由是基于应用中业务流程抓取网络路径信息报文来辅助故障定位定界的手段。跟踪路由获得的信息可能涉及敏感数据，使用该功能时需遵循所使用国家的法律和运营商的用户隐私保护政策，并采取足够的措施，以确保用户的涉敏感数据受到充分保护。

该命令用来查看OM网络平面与目标主机之间的网络路径。

#### [注意事项](#ZH-CN_TOPIC_0000001292313044)

- 执行命令期间，请不要执行复位OMLB主节点或者主备倒换OMLB操作，否则会终止跟踪路由任务。
- 命令返回报文中包含“？”属于正常现象，因为某一跳的IP不通或者内部存在拦截规则，导致这一跳的IP显示成“？”。

#### [参数说明](#ZH-CN_TOPIC_0000001292313044)

| **参数标识** | **参数名称** | **参数说明** |
| --- | --- | --- |
| IPTYPE | IP版本 | 可选必选说明：必选参数。<br>参数含义：源IP地址和目的IP地址的类型。<br>取值范围：<br>- IPV4(IPv4类型)<br>- IPV6(IPv6类型)<br>默认值：IPV4(IPv4类型)。<br>配置原则：无。 |
| SRCIPV4 | 源IPv4地址 | 可选必选说明：条件可选参数，当IP版本为IPV4时为可选参数。<br>参数含义：发送数据包时使用的源IP地址，参数为空时，将使用OM Portal浮动IP地址。<br>取值范围：合法的IPv4地址，且必须是OM Portal浮动IP地址或OMLB主节点物理IP地址。<br>默认值：无。<br>配置原则：可以通过执行<br>[**LST OMIP**](../代理管理/查询OM IP (LST OMIP)_76163349.md)<br>查询获得。 |
| DSTIPV4 | 目的IPv4地址 | 可选必选说明：条件必选参数，当IP版本为IPV4时为必选参数。<br>参数含义：发送数据包时使用的目的IP地址。<br>取值范围：合法的IPv4地址。<br>默认值：无。<br>配置原则：无。 |
| SRCIPV6 | 源IPv6地址 | 可选必选说明：条件可选参数，当IP版本为IPV6时为可选参数。<br>参数含义：发送数据包时使用的源IP地址，参数为空时，将使用OM Portal浮动IP。<br>取值范围：合法的IPv6地址，且必须是OM Portal浮动IP或OMLB主节点物理IP。<br>默认值：无。<br>配置原则：可以通过执行<br>[**LST OMIP**](../代理管理/查询OM IP (LST OMIP)_76163349.md)<br>查询获得。 |
| DSTIPV6 | 目的IPv6地址 | 可选必选说明：条件必选参数，当IP版本为IPV6时为必选参数。<br>参数含义：发送数据包时使用的目的IP地址。<br>取值范围：合法的IPv6地址。<br>默认值：无。<br>配置原则：无。 |
| PKTCOUNT | 发包数 | 可选必选说明：可选参数。<br>参数含义：设置每一跳的发包数量。<br>取值范围：1~50<br>默认值：3<br>配置原则：无。 |
| MAXHOP | 最大跳 | 可选必选说明：可选参数。<br>参数含义：设置检测数据包的最大存活数值TTL的大小。<br>取值范围：1~128<br>默认值：32<br>配置原则：无。 |

#### [使用实例](#ZH-CN_TOPIC_0000001292313044)

执行跟踪路由命令，查看OM Portal浮动IP与目的IP之间的网络路径：

```
%%OMTRACERT: IPTYPE=IPV4, DSTIPV4="192.168.1.2";%% 
RETCODE = 0  操作成功

操作结果如下
------------
记录索引  丢包率  发包数量  最小响应时间(毫秒)  最大响应时间(毫秒)  平均响应时间(毫秒)  IP版本  对端IP地址 

1         0.0%    3         3.4                 13.0                8.2                 IPv4    192.168.1.254 
2         0.0%    3         3.5                 3.8                 3.6                 IPv4    192.168.1.2   
(结果个数 = 2)

---    END
```

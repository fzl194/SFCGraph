#### [命令功能](#ZH-CN_CONCEPT_0182837317)

**适用NF：SGW-U、PGW-U、UPF**

该命令用于设置软件参数表比特位。

#### [注意事项](#ZH-CN_CONCEPT_0182837317)

- 该命令执行后立即生效。
- 该命令最大记录数为1。

#### [操作用户权限](#ZH-CN_CONCEPT_0182837317)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [使用实例](#ZH-CN_CONCEPT_0182837317)

设置单字节类型1号软参Bit1位的值为1：

```
SET SOFTPARAOFBIT: DT2=BYTE, BYTENUM=1, BYTEPOSITION=1, BYTEVALUE=1;
```

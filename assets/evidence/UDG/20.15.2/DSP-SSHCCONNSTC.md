#### [命令功能](#ZH-CN_CONCEPT_0000001600841345)

该命令用于显示SSH客户端连接统计信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600841345)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600841345)

G_1，管理员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600841345)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001600841345)

显示SSH客户端连接统计信息：

```
DSP SSHCCONNSTC:;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600841345)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 总连接数 | 表示SSH客户端总连接数。 |
| SFTP会话数 | 表示SFTP会话数。 |
| SNetconf会话数 | 表示SNetconf会话数。 |
| STelnet会话数 | 表示STelnet会话数。 |
| MML会话数 | 表示MML会话数。 |
| MML关闭连接数 | 表示被MML关闭的连接数。 |
| CLI关闭连接数 | 表示被CLI关闭的连接数。 |
| SNetconf关闭连接数 | 表示被SNetconf关闭的连接数。 |
| SSHS关闭连接数 | 表示被SSH服务器关闭的连接数。 |
| Socket关闭连接数 | 表示被Socket关闭的连接数。 |
| CTRL+T关闭连接数 | 表示被CTRL+T关闭的连接数。 |
| CTRL+]关闭连接数 | 表示被CTRL+]关闭的连接数。 |
| 鉴权关闭连接数 | 表示鉴权拒绝连接数。 |
| ACL拒绝连接数 | 表示ACL拒绝连接数。 |
| Socket创建失败数 | 表示创建失败的Socket数。 |
| Channel失败连接数 | 表示失败的通道数。 |
| Channel关闭连接数 | 表示被SSH服务器关闭的通道数。 |
| 断连消息数 | 表示SSH服务器断开的连接数。 |
| 在线连接数 | 表示在线客户端连接数。 |
| 重传次数 | 表示客户端重传次数。 |

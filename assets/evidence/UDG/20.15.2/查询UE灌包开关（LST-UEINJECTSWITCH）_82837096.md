#### [命令功能](#ZH-CN_CONCEPT_0182837096)

**适用NF：SGW-U、PGW-U、UPF**

该命令用于查询UE下行灌包是否使能。

#### [注意事项](#ZH-CN_CONCEPT_0182837096)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0182837096)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0182837096)

无。

#### [使用实例](#ZH-CN_CONCEPT_0182837096)

查询UE下行灌包功能是否使能：

```
LST UEINJECTSWITCH:;
```

```

RETCODE = 0  操作成功。

UE下行灌包开关信息
------------------
开关标识  =  使能
(结果个数 = 1)
---    END
```

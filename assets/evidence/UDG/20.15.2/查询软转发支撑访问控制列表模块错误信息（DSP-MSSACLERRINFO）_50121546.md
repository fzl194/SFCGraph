#### [命令功能](#ZH-CN_CONCEPT_0000001550121546)

该命令用于查询访问控制列表错误信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001550121546)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550121546)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550121546)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RUNAME | RU名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定RU名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，区分大小写。<br>默认值：无<br>配置原则：使用DSP RU查看RU名称。 |
| TABLEID | 表ID | 可选必选说明：必选参数<br>参数含义：该参数用于表示访问控制列表ID。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～4294967295。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001550121546)

查询软转发支撑访问控制列表错误信息：

```
DSP MSSACLERRINFO:TABLEID=7,RUNAME="VNODE_VNRS_VNFC_IPU_0064";
```

```

RETCODE = 0  操作成功。

结果如下
--------
      操作  =  add-rule
规则优先级  =  10
    错误码  =  0x12C3346B
      时间  =  2016-10-19 10:26:28.756
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001550121546)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 操作 | 用于表示访问控制列表错误标识。 |
| 规则优先级 | 用于表示访问控制列表优先级。 |
| 错误码 | 用于表示访问控制列表模块错误码。 |
| 时间 | 用于表示访问控制列表模块错误时间。 |

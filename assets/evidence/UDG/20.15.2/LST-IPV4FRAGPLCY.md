#### [命令功能](#ZH-CN_CONCEPT_0186526767)

**适用NF：SGW-U、PGW-U、UPF**

该命令用于查询逻辑接口在IPv4组网场景下的分片处理策略。

#### [注意事项](#ZH-CN_CONCEPT_0186526767)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0186526767)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0186526767)

无。

#### [使用实例](#ZH-CN_CONCEPT_0186526767)

查询逻辑接口在IPv4组网场景下的分片处理策略：

```
LST IPV4FRAGPLCY:;
```

```

RETCODE = 0  操作成功。

IPV4分片策略
------------
内层为IPv4的分片策略  =  外层分片
内层为IPv6的分片策略  =  外层分片
             分片长度选项  = 内层总包长
(结果个数 = 1)
---    END
```

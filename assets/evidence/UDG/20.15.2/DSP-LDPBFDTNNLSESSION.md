#### [命令功能](#ZH-CN_CONCEPT_0000001550121702)

该命令用于显示BFD For LDP Tunnel的BFD会话信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001550121702)

查询会话类型为LDP隧道会话的BFD会话，只有在主动端才能查询到BFD会话信息，在被动端需要通过DSP BFDSESSION查询BFD会话信息。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550121702)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550121702)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| FECADDRESS | FEC地址 | 可选必选说明：可选参数<br>参数含义：该参数用于表示BFD For LDP Tunnel的FEC地址。<br>数据来源：对端协商<br>取值范围：IPv4地址类型。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001550121702)

显示BFD For LDP Tunnel的BFD会话信息：

```
DSP LDPBFDTNNLSESSION:;
```


#### [输出结果说明](#ZH-CN_CONCEPT_0000001550121702)

| 输出项名称 | 输出项解释 |
| --- | --- |
| FEC地址 | 用于表示BFD For LDP Tunnel的FEC地址。 |
| LSP索引 | 用于表示BFD For LDP Tunnel的LDP LSP的索引值，也即XC索引。 |
| BFD会话的本地标识符 | 用于表示BFD For LDP Tunnel的BFD会话的标识符。 |
| 会话状态 | 该参数用于表示BFD For LDP Tunnel的BFD会话的状态。<br>- 管理Down状态：BFD会话处于建立管理Down的状态。<br>- Down状态：BFD会话处于Down的状态。<br>- 初始化状态：BFD会话处于初始化的状态。<br>- 建立状态：BFD会话处于Up的状态。 |
| 最小发送间隔（ms） | 用于表示本地BFD报文实际发送时间间隔。本地BFD报文实际发送时间间隔＝MAX { 本地配置的发送时间间隔，对端配置的接收时间间隔 }。对端配置的接收时间间隔为固定的200ms。 |
| 最小接收间隔（ms） | 用于表示本地BFD报文实际接收时间间隔。本地实际接收时间间隔＝MAX { 对端配置的发送时间间隔，本地配置的接收时间间隔 }。对端配置的发送时间间隔为固定的200ms。 |
| 检测倍数 | 用于表示对端检测倍数，为固定值3。 |
| 会话建立时长（s） | 用于表示BFD For LDP Tunnel的BFD会话建立时长。 |

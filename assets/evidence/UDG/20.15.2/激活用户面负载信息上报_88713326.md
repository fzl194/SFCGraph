## [操作场景](#ZH-CN_OPI_0000001088713326)

当SGW-C/PGW-C/SMF需要基于网元负载信息选择SGW-U/PGW-U/UPF时，可激活本特性。

> **说明**
> 适用于SGW-U、PGW-U、UPF。

## [操作步骤](#ZH-CN_OPI_0000001088713326)

1. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09587387.md)
2. 开启用户面负载信息上报的功能开关，并设置上报阈值。
  [**SET PFCPLOADRPT**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/路径管理/PFCP路径管理/PFCP负荷上报管理/PFCP负荷上报/设置负荷上报开关（SET PFCPLOADRPT）_93753137.md)

## [任务示例](#ZH-CN_OPI_0000001088713326)

任务描述

SGW-U/PGW-U/UPF开启用户面负载信息上报的功能，上报阈值设置为50%，变化范围阈值配置为5%，当负载超过50%时上报。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH:LICITEM="LKV3G5UPLR01",SWITCH=ENABLE;
```

//打开本特性的开关，上报阈值设置为50%，变化范围阈值配置为5%。

```
SET PFCPLOADRPT:LOADRPTSW=ENABLE,RPTTHD=50,STOPTHD=45,RANGETHD=5;
```

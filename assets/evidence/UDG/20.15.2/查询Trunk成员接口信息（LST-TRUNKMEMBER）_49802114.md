#### [命令功能](#ZH-CN_CONCEPT_0000001549802114)

该命令用于显示Trunk成员接口信息。

若不指定IFNAME参数时，则显示所有Trunk成员信息；若指定IFNAME参数时，则可以显示特定Trunk成员信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001549802114)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549802114)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549802114)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| TRUNKIFNAME | Trunk接口名 | 可选必选说明：可选参数<br>参数含义：该参数用于指定需要配置Trunk接口名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。接口名称由接口类型+接口编号组成。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549802114)

查询Eth-Trunk1的成员接口信息：

```
LST TRUNKMEMBER:TRUNKIFNAME="Eth-Trunk1";
```

```

RETCODE = 0  操作成功

结果如下
-------------------------
Trunk接口名       成员接口名
Eth-Trunk1        Ethernet64/0/3
Eth-Trunk1        Ethernet64/0/4
(结果个数 = 2)
---   END
```

## [命令功能](#ZH-CN_MMLREF_0000001538848818)

该命令用于查询内部升级配置表。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000001538848818)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001538848818)

无

## [使用实例](#ZH-CN_MMLREF_0000001538848818)

查询内部升级配置表，执行如下命令：

```
%%LST UPDATECTRLCFG:;%%
RETCODE = 0  操作成功

结果如下
------------------------
升级状态 = 0
低版本号 = 0
高版本号 = 100
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001538848818)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 升级状态 | 升级状态。 |
| 低版本号 | 低版本号。 |
| 高版本号 | 高版本号。 |

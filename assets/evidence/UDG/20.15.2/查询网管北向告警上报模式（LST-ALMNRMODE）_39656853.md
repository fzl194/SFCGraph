## [命令功能](#ZH-CN_MMLREF_0000001139656853)

本命令用于查询网管向北向网管系统上报告警的模式。

> **说明**
> 无。

## [参数说明](#ZH-CN_MMLREF_0000001139656853)

无。

## [使用实例](#ZH-CN_MMLREF_0000001139656853)

1. 查询当前系统中设置的北向上报模式（单网元合并模式）：
  ```
  %%LST ALMNRMODE:;%%
  RETCODE = 0  操作成功

  操作结果
  --------
  告警北向上报模式  =  单网元合并模式
          源网元ID  =  0
        源网元名称  =  sourceMeName
        目标网元ID  =  75
      目标网元名称  =  targetMeName
  (结果个数 = 1)

  ---    END
  ```
2. 查询当前系统中设置的北向上报模式（多网元独立模式）：
  ```
  %%LST ALMNRMODE:;%%
  RETCODE = 0  操作成功

  操作结果
  --------
  告警北向上报模式  =  多网元独立模式
  (结果个数 = 1)
  ---    END
  ```

## [输出结果说明](#ZH-CN_MMLREF_0000001139656853)

命令执行正常，会返回命令执行成功的提示信息，输出结果说明如 [表1](#ZH-CN_MMLREF_0000001139656853__table47787675) 所示。

命令执行异常，请联系华为技术支持处理。输出结果说明如 [表2](#ZH-CN_MMLREF_0000001139656853__table698716490471) 所示。

*表1 输出项解释*

| **输出项名称** | **输出项解释** |
| --- | --- |
| 告警北向上报模式 | 用于表示本站点网元的告警北向上报模式，取值范围：<br>- “单网元合并模式”：表示本站点所有网元或源网元的告警从网管上报到北向网管系统时，将被合并到目标网元上报，但网管和网元OM Portal上仍按原始网元显示告警。<br>- “多网元独立模式”：表示本站点各网元将按本网元类型和网元实例独立上报告警到北向网管系统。 |
| 源网元ID | 用于表示系统需要将告警合并的网元的ID。 |
| 源网元名称 | 用于表示系统需要将告警合并的网元的名称。 |
| 目标网元ID | 用于表示系统需要将告警合并到的目标网元的ID。 |
| 目标网元名称 | 用于表示系统需要将告警合并到的目标网元的名称。 |

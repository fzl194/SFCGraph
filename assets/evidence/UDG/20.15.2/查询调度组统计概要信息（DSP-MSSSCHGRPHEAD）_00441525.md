#### [命令功能](#ZH-CN_CONCEPT_0000001600441525)

该命令用于查询调度组统计概要信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600441525)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600441525)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600441525)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RUNAME | RU名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定RU名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，区分大小写。<br>默认值：无<br>配置原则：使用DSP RU查看RU名称。 |
| INGROUPID | 调度组ID | 可选必选说明：必选参数<br>参数含义：该参数用于表示调度组ID。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～4294967295。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600441525)

查询调度组概要信息：

```
DSP MSSSCHGRPHEAD:INGROUPID = 0,RUNAME = "VNODE_VNRS_VNFC_IPU_0064";
```

```

RETCODE = 0  操作成功。

结果如下
--------
        调度组状态  =  used
        调度组类型  =  io type
          队列个数  =  1
调度组绑定线程个数  =  2
  调度组绑定线程号  =  1 2
  端口调度任务个数  =  2
(结果个数 = 1)
---    END
```

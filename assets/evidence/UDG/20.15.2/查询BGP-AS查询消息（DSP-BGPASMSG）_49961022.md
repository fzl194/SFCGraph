#### [命令功能](#ZH-CN_CONCEPT_0000001549961022)

该命令用来查询BGP AS查询消息。

#### [注意事项](#ZH-CN_CONCEPT_0000001549961022)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549961022)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549961022)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001549961022)

查询BGP AS查询消息：

```
DSP BGPASMSG:;
```

```

RETCODE = 0  操作成功。

结果如下
-------------------------
VPN名称        =  vrf1
 返回码        =  1
原始AS号       =  200
查询时间       =  2016-12-12T03:46:40
查询模式       =  2
目的地址       =  192.168.3.1
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001549961022)

| 输出项名称 | 输出项解释 |
| --- | --- |
| VPN名称 | 用来指定用户配置的VPN实例的名称。 |
| 返回码 | 用于表示返回码。 |
| 原始AS号 | 用于表示原始AS号。 |
| 查询时间 | 用于表示查询时间。 |
| 查询模式 | 用于指定查询模式（严格和宽松）。 |
| 目的地址 | 用于表示目的地址。 |

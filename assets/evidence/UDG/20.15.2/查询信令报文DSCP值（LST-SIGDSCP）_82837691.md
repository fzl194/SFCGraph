#### [命令功能](#ZH-CN_CONCEPT_0182837691)

**适用NF：SGW-U、PGW-U、UPF**

该命令用于查询信令报文的DSCP值。

#### [注意事项](#ZH-CN_CONCEPT_0182837691)

TM信令和GTP信令在未配置时不显示。

#### [操作用户权限](#ZH-CN_CONCEPT_0182837691)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0182837691)

无。

#### [使用实例](#ZH-CN_CONCEPT_0182837691)

查询所有协议的DSCP值：

```
LST SIGDSCP:;
```

```

RETCODE = 0  操作成功。

信令报文DSCP值
--------------
信令协议      DSCP值

PFCP信令      46          
L2tp信令      46          
PPP信令       46          
(结果个数 = 3)

---    END
```

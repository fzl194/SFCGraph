## [命令功能](#ZH-CN_MMLREF_0000001930144141)

该命令用于查询分布式文件系统集群信息。

> **说明**
> 无

## [参数说明](#ZH-CN_MMLREF_0000001930144141)

无。

## [使用实例](#ZH-CN_MMLREF_0000001930144141)

查询hofs集群的基本信息。

```
%%DSP DFSINFO:;%% 
RETCODE = 0  操作成功
  
分布式文件系统信息 
------------------        
          应用ID  =  0       
         OSD数量  =  0      
        对象数量  =  9599
      读成功次数  =  0    
      写成功次数  =  0      
        失败次数  =  0    
      对象总大小  =  4872
    PV总大小(MB)  =  120371.5625  
PV已使用大小(MB)  =  38366.0664      
     PV使用率(%)  =  36.9902     
    Inode总数(B)  =  7864320 
Inode使用数量(B)  =  122422   
  Inode使用率(%)  =  1.5567 
(结果个数 = 1) 
 
---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001930144141)

命令执行正常，会返回分布式文件系统的基本信息，输出结果项信息项目如 [表1](#ZH-CN_MMLREF_0000001930144141__table69822584408) 。

命令执行异常，会返回对应的错误码。常见错误码如 [表2](#ZH-CN_MMLREF_0000001930144141__table398214418524) 所示。

*表1 输出项说明*

| 输出项名称 | 输出项解释 |
| --- | --- |
| 应用ID | 用于返回当前应用ID。 |
| OSD数量 | 用于返回OSD数量。 |
| 对象数量 | 用于返回存储的文件个数。 |
| 读成功次数 | 用于返回操作数据桶读成功次数。 |
| 写成功次数 | 用于返回操作数据桶写成功次数。 |
| 失败次数 | 用于返回操作数据桶失败次数。 |
| 对象总大小 | 用于返回存储的文件总大小。 |
| PV总大小 | 用于返回实际存储空间大小，单位M。 |
| PV已使用大小 | 用于返回实际存储空间已使用大小，单位M。 |
| PV使用率 | 用于返回实际存储空间使用率，单位%。 |
| Inode总数 | 用于返回文件系统对象总大小<br>，单位B。 |
| Inode使用数量 | 用于返回文件系统对象使用数量，单位B。 |
| Inode使用率 | 用于返回文件系统对象使用率，单位%。 |

*表2 错误码列表*

| 错误码 | 错误码解释 | 原因分析 | 处理建议 |
| --- | --- | --- | --- |
| 180001 | 下发的MML参数不正确 | 非法的命令参数 | 请联系华为技术支持。 |
| 180002 | MML处理失败 | 内部错误 | 请联系华为技术支持。 |

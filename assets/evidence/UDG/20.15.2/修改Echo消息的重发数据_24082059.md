## [操作场景](#ZH-CN_OPI_0124082059)

在组网改变、网络调整中修改调整系统参数中Echo消息的重发时间的间隔和最大尝试发送次数。

> **说明**
> 适用于SGW-U、PGW-U、UPF。

## [对系统的影响](#ZH-CN_OPI_0124082059)

该配置除了影响Echo消息外，还影响 UDG 发送的去活请求消息和更新请求消息。

## [操作步骤](#ZH-CN_OPI_0124082059)

1. 设置 UDG 非N4/Sxb接口的GTP请求消息的重发时间间隔和最大尝试发送次数。
  [**SET UPGTPPATH**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/路径管理/GTP路径管理/GTP协议参数管理/GTP路径全局参数/设置GTP路径相关属性（SET UPGTPPATH）_82837227.md)
2. 设置 UDG N4/Sxb接口的GTP请求消息的重发时间间隔和最大尝试发送次数。
  [**SET UPN4UPATH**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/路径管理/GTP路径管理/GTP协议参数管理/N4U路径全局参数/设置N4U路径相关属性（SET UPN4UPATH）_18761567.md)

## [任务示例](#ZH-CN_OPI_0124082059)

任务描述

将Echo消息的重发时间的间隔设置为“10秒”和最大尝试发送次数设置为“3次”。

脚本

//设置非N4/Sxb接口的GTP请求消息的重发时间间隔和最大尝试发送次数。

```
SET UPGTPPATH:T3RESPONSE=10,N3REQUEST=3;
```

//设置N4/Sxb接口的GTP请求消息的重发时间间隔和最大尝试发送次数。

```
SET UPN4UPATH:N4T3RESPONSE=10,N4N3REQUEST=3;
```

## [命令功能](#ZH-CN_MMLREF_0232217475)

该命令用于查询Base平面的亚健康检测参数。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0232217475)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0232217475)

无

## [使用实例](#ZH-CN_MMLREF_0232217475)

查询Base平面的亚健康检测参数：

```
%%LST COMBASEHEALTH:;%%
RETCODE = 0  操作成功

结果如下
--------
      检测周期(秒)  =  1
      统计周期(秒)  =  30
亚健康告警阈值(‰)  =  50
          使能开关  =  使能
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0232217475)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 检测周期(秒) | 该参数用于标识Base平面亚健康检测的检测周期，即每过一个检测周期发一次探测报文。 |
| 统计周期(秒) | 该参数用于标识Base平面亚健康检测的统计周期，即每过一个统计周期统计一次该周期内各项亚健康指标。 |
| 亚健康告警阈值(‰) | 该参数用于标识Base平面亚健康检测的阈值。 |
| 使能开关 | 该参数用于标识Base平面亚健康检测的使能开关。 |

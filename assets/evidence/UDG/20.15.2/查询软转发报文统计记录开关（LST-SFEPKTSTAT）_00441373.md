#### [命令功能](#ZH-CN_CONCEPT_0000001600441373)

该命令用于查询软转发报文统计记录开关配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001600441373)

- 该命令执行后立即生效。
- 该命令仅适用于非NP卡场景和NP卡非加速模式场景。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600441373)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600441373)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001600441373)

查询软转发报文统计记录开关配置：

```
LST SFEPKTSTAT:;
```

```

RETCODE = 0  操作成功

结果如下
------------------------
IsEnable = FALSE
(结果个数 = 1)
---    END
```

#### [命令功能](#ZH-CN_CONCEPT_0182837860)

**适用NF：SGW-U、PGW-U、UPF**

该命令用来显示ALM-81100 GTPU大量路径断告警的上报阈值和恢复阈值。

#### [注意事项](#ZH-CN_CONCEPT_0182837860)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0182837860)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0182837860)

无。

#### [使用实例](#ZH-CN_CONCEPT_0182837860)

查询大量路径断告警的上报阈值和恢复阈值：

```
LST PATHCTRLALMTHD:;
```

```

RETCODE = 0  操作成功。

大量路径断告警告警阈值
----------------------
大量路径断告警上报的阈值  =  100
大量路径断告警恢复的阈值  =  50
(结果个数 = 1)
---    END
```

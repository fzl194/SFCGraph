## [命令功能](#ZH-CN_MMLREF_0000001559336676)

查询证书开关状态。

> **说明**
> - [**SET NEWCERTSWITCH**](设置证书开关状态（SET NEWCERTSWITCH）_10015761.md)命令触发开关切换，请1~3分钟后使用[**LST NEWCERTSWITCH**](查询证书开关状态（LST NEWCERTSWITCH）_59336676.md)命令查询开关切换任务结果。

## [参数说明](#ZH-CN_MMLREF_0000001559336676)

无。

## [使用实例](#ZH-CN_MMLREF_0000001559336676)

查询证书开关状态。

```
%%LST NEWCERTSWITCH:;%% 
RETCODE = 0  操作成功  

操作结果如下 
------------ 
    证书开关  =  开 
    加载状态  =  成功 
(结果个数 = 1)  

---    END
```

证书开关状态切换失败时，查询证书开关状态。

```
%%LST NEWCERTSWITCH:;%% 
RETCODE = 0  操作成功  

操作结果如下 
------------ 
    证书开关  =  开 
    加载状态  =  失败 
(结果个数 = 1)  

失败服务信息如下
------------ 
网元ID   失败服务名称   节点信息
0        Privilege      172.16.1.1
(结果个数 = 1) 
---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001559336676)

命令执行正常，会返回命令执行成功的提示信息，输出结果说明如 [表1](#ZH-CN_MMLREF_0000001559336676__table47787675) 所示。

命令执行异常，请联系华为技术支持处理。输出结果说明如 [表2](#ZH-CN_MMLREF_0000001559336676__table5257824152) 、 [表3](#ZH-CN_MMLREF_0000001559336676__table698716490471) 所示。

*表1 操作结果输出项解释*

| **输出项名称** | **输出项解释** |
| --- | --- |
| 证书开关 | 证书开关状态，取值包括：<br>- 开<br>- 关 |
| 加载状态 | 切换证书开关状态，取值包括：<br>- 成功。表示证书开关状态切换成功。<br>- 失败。表示证书开关状态切换失败。可能原因：与部分服务建链失败或部分服务状态异常、客户端返回加载失败或客户端没有在超时时间内返回加载成功。<br>- 执行中。表示证书开关状态正在切换中。 |

*表2 失败服务信息输出项解释*

| **输出项名称** | **输出项解释** |
| --- | --- |
| 网元ID | 网元ID。 |
| 失败服务名称 | 失败服务的具体名称。 |
| 节点信息 | 失败服务实例所在的节点信息。 |

*表3 错误码列表*

| 错误码 | 错误码解释 | 原因分析 | 处理建议 |

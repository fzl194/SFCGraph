- [注意事项](#ZH-CN_CONCEPT_0000201106561536__1.3.2.1)

#### [命令功能](#ZH-CN_CONCEPT_0000201106561536)

**适用NF：UPF**

该命令用于删除指定的NF实例ID。

#### [注意事项](#ZH-CN_CONCEPT_0000201106561536)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000201106561536)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000201106561536)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| UUID | UUID | 可选必选说明：必选参数<br>参数含义：NF实例标识ID。<br>数据来源：全网规划<br>取值范围：字符串类型，NF实例ID的长度为36个字节，以连字号分为五段，格式为：xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx (8-4-4-4-12)，其中每个 x 是 0-9 或 a-f 范围内的一个十六进制的数字。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000201106561536)

删除UPF配置的NF实例ID：

```
RMV UPINSTID: UUID="4947a69a-f61b-4bc1-b9da-47c9c5d14b64";
```

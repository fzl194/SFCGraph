# 调测ACL

- [操作场景](#ZH-CN_OPI_0206381761__1.3.1)
- [必备事项](#ZH-CN_OPI_0206381761__1.3.2)
- [操作步骤](#ZH-CN_OPI_0206381761__1.3.3)

## [操作场景](#ZH-CN_OPI_0206381761)

ACL功能调测用于校验本局与对端设备的ACL功能能否正常工作以及数据配置是否正确，以保证ACL过滤功能的正常运行。

## [必备事项](#ZH-CN_OPI_0206381761)

前提条件

- 操作人员已经登录网络管理系统NMS（Network Management System）。
- 硬件安装已经完成。

数据

无

## [操作步骤](#ZH-CN_OPI_0206381761)

1. 查询ACL规则组信息。
  在 “MML命令行-UDG” 窗口上执行：
  [**LST ACLGROUP**](../../../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/ACL管理/ACL规则组/查询ACL规则组配置（LST ACLGROUP）_00841277.md) ：ACLNAME="ACL规则组标识";
    - 预期结果：存在ACL规则组，规则组参数与所配置的一致。
      ```
      LST ACLGROUP: ACLNAME="1000";
      RETCODE = 0  操作成功

      结果如下
      -------------------------

       ACL规则组标识  =  1000

          规则组步长  =  5
          规则组类型  =  接口型ACL

      (结果个数 = 1)
      ---END
      ```
    - 异常结果：查询不到规则组1000配置信息。

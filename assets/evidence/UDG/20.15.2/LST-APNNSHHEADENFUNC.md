#### [命令功能](#ZH-CN_CONCEPT_0000203719881180)

**适用NF：PGW-U、UPF**

该命令用于查询NSH头增强功能开关。

#### [注意事项](#ZH-CN_CONCEPT_0000203719881180)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000203719881180)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000203719881180)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| APN | APN | 可选必选说明：可选参数<br>参数含义：该参数用于指定APN名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。只能由“-”、数字、大小写字母和“.”组成，不能以“.”开头且不能出现连续两个“.”。不支持空格及“_”、“#”、“$”、“&”、“%”、“^”、“（”、“）”、“，”、“/”、“;”、“:”、“””、“`”等特殊字符，不区分大小写。<br>默认值：无<br>配置原则：该参数使用ADD APN命令配置生成。 |

#### [使用实例](#ZH-CN_CONCEPT_0000203719881180)

查询NSH头增强功能开关：

```
LST APNNSHHEADENFUNC:;
```

```

RETCODE = 0  操作成功

NSH头增强功能开关
-----------------
     APN  =  apn
功能开关  =  不使能（关闭）
(结果个数 = 1)

---    END
```

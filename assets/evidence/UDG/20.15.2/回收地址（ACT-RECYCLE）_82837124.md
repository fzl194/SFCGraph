#### [命令功能](#ZH-CN_CONCEPT_0182837124)

**适用NF：SGW-U、PGW-U、UPF**

![](回收地址（ACT RECYCLE）_82837124.assets/notice_3.0-zh-cn.png)

本命令属于高危命令，使用该命令后会将强制回收正在使用的地址，对应的正常在线用户将被强制下线。

ACT RECYCLE命令用于回收本地地址池、组播地址池或远端地址池中的一个或一段地址。修改系统配置时，如果需要对某个地址池/段进行动态修改或删除，但地址池内有地址已经分配出去，这时候就需要利用ACT RECYCLE命令来强制回收分配出去的地址。

#### [注意事项](#ZH-CN_CONCEPT_0182837124)

- 该命令执行后立即生效。
- 强制回收已分配的地址或地址段前，首先要锁定地址所在的地址池或地址段。如果想要修改和删除的地址池中地址已经分配出去，需要强制回收后才能够对地址池/段进行动态修改或删除。
- 回收地址范围不可以跨地址段。这里回收的不一定是整个段，可以是地址段内的某个子段。当起始地址和终止地址相同时，表示只回收一个地址。
- 如果对配置了Releasetime的地址池执行本命令后，ISU板在releasetime时间发生主备倒换，当地址又变为占用态时，需要再次执行ACT RECYCLE命令进行地址回收。
- 静态地址回收不需要锁定地址池。
- 执行ACT RECYCLE后，地址池回收需要时间，不要立刻使用命令SET DeactiveRate修改去活用户会话速率。如果立即修改且速率修改得过大，会导致N4接口消息速率过大，冲击SMF。

#### [操作用户权限](#ZH-CN_CONCEPT_0182837124)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0182837124)

| 参数标识 | 参数名称 | 参数说明 |

| POOLNAME | 地址池名称 | 可选必选说明：必选参数<br>参数含义：地址池的名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～79，单位是字节。由“_”、“-”、数字、大小写字母和“.”组成，不能以“.”开头且不能出现连续两个“.”，不区分大小写。<br>默认值：无<br>配置原则：无 |
| IPVERSION | IP地址类型 | 可选必选说明：必选参数<br>参数含义：用于指定IP地址类型。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- IPV4：IPv4地址类型。<br>- IPV6：IPv6地址类型。<br>默认值：无<br>配置原则：无 |

| ENDIP | 终止IP地址 | 可选必选说明：条件必选参数<br>前提条件：该参数在“IPVERSION”配置为“IPV4”时为必选参数。<br>参数含义：地址段的终止IPv4地址。<br>数据来源：全网规划<br>取值范围：IPv4地址类型。点分十进制格式。<br>默认值：无<br>配置原则：终止IP地址必须大于等于起始IP地址。当起始地址和终止地址相同时，表示只回收一个地址。 |
| PREFIXSTART | IPv6前缀 | 可选必选说明：条件必选参数<br>前提条件：该参数在“IPVERSION”配置为“IPV6”时为必选参数。<br>参数含义：起始IPv6前缀。<br>数据来源：全网规划<br>取值范围：IPv6地址类型。冒号分十六进制，格式为X:X:X:X:X:X:X:X。除了组播地址、链路本地地址、环回地址或未指定的地址为非法地址外，其他都为合法地址。<br>默认值：无<br>配置原则：无 |
| PREFIXEND | IPv6后缀 | 可选必选说明：条件必选参数<br>前提条件：该参数在“IPVERSION”配置为“IPV6”时为必选参数。<br>参数含义：终止IPv6前缀。<br>数据来源：全网规划<br>取值范围：IPv6地址类型。冒号分十六进制，格式为X:X:X:X:X:X:X:X。除了组播地址、链路本地地址、环回地址或未指定的地址为非法地址外，其他都为合法地址。<br>默认值：无<br>配置原则：一个IPv6地址前缀只能配置一个前缀地址段，前缀地址段的起始地址必须小于或等于终止地址，前缀地址段内的地址总数不能超过32768，且同一VPN下，各个前缀地址段里的地址不能相互重叠。 |

#### [使用实例](#ZH-CN_CONCEPT_0182837124)

强制回收本地IPV4地址池lap中的地址10.1.1.0~10.1.1.20，POOLNAME为lap，IPVERSION为IPV4，STARTIP为10.1.1.0，ENDIP为10.1.1.20：

```
ACT RECYCLE:POOLNAME="lap",IPVERSION=IPV4,STARTIP="10.1.1.0",ENDIP="10.1.1.20";
```

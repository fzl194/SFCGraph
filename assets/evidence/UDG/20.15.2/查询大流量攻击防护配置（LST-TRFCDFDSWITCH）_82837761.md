#### [命令功能](#ZH-CN_CONCEPT_0182837761)

**适用NF：UPF**

该命令用来查询整机粒度是否开启大流量攻击检测功能。

#### [注意事项](#ZH-CN_CONCEPT_0182837761)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0182837761)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0182837761)

无。

#### [使用实例](#ZH-CN_CONCEPT_0182837761)

查询整机粒度大流量攻击检测功能开关状态：

```
LST TRFCDFDSWITCH:;
```

```

RETCODE = 0 操作成功。

大流量攻击防护配置信息
----------------------
上行大流量攻击检测开关  =  使能
下行大流量攻击检测开关  =  使能
(结果个数 = 1)
```

## [操作场景](#ZH-CN_OPI_0000001383167737)

> **说明**
> 适用于PGW-U。

在线计费中， UDG 支持按用户使用的业务进行在线计费， UDG 与UNC消息交互时需要用到URR ID，上报计费信息。

## [操作步骤](#ZH-CN_OPI_0000001383167737)

配置业务级在线计费的费率标识。

1. 配置内容计费使用的URR。
    a. 配置URR。
      **ADD URR**
    b. 配置上行和下行发起的业务对应的URR。
      **ADD URRGROUP**
2. 配置内容计费使用的PCC策略组。
  **ADD PCCPOLICYGRP**
3. 配置过滤条件。
    a. 配置三四层过滤条件。
      **ADD FILTER**
    b. 配置流过滤器。
      **ADD FLOWFILTER**
    c. 绑定三四层过滤条件。
      **ADD FLTBINDFLOWF**
4. 配置内容计费使用的规则。
  **ADD RULE**
5. 配置内容计费规则绑定用户模板。
    a. 创建UserProfile实例。
      **ADD USERPROFILE**
    b. 将已配置的内容计费规则绑定到UserProfile。
      **ADD RULEBINDING**
6. **可选** ：设置全局默认配额开关。
  **SET UPDEFAULTQUOTA**
  > **说明**
  > 新业务触发申请配额期间是否允许使用默认配额由 **SET UPDEFAULTQUOTA** 命令控制。
  >
  > - 使能情况下使用**ADD URR**配置的默认配额正常处理报文。
  > - 不使能情况下根据**SET UPGLBCHGPARA**命令决定缓存报文或者丢弃报文。
7. 可选：设置全局Credit Pooling功能开关。
  **SET UPGLBCHGPARA**
  > **说明**
  > - 是否使能Credit Pooling功能由**SET UPDEFAULTQUOTA**命令的参数**CREDITPOOLSW**控制。
  > - 该功能与DefaultQuota功能不能同时使用。
8. **可选** ：设置URR上报失败后的动作处理参数。
  ****SET URRFAILACTION****
  > **说明**
  > - 该命令在在线计费需要申请配额的场景下生效。
  > - 当业务部署调整导致PGW-U上的URR ID在GGSN/PGW-C上不存在时(比如，先添加PGW-U上**ADD URR**，后添加GGSN/PGW-C上**ADD URR**)，会导致GGSN/PGW-C无法处理PGW-U的计费用量上报。为减小该场景下对用户业务的影响，可通过该命令设置PGW-U向GGSN/PGW-C进行计费用量上报失败时的处理动作。
  > - 当计费URR上报失败，业务被阻塞时，如果想要放通业务，可以执行**SET URRFAILACTION**命令，配置“RETRYFAILACT”参数为“CONTINUE”，则URR对应的业务将不计费。在PGW-U向GGSN/PGW-C进行用量上报恢复后，将恢复对该URR的计费。

## [任务示例](#ZH-CN_OPI_0000001383167737)

任务描述

用户进行在线计费，其访问的业务的URR ID为1100。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH:LICITEM="LKV3G5OLCH01",SWITCH=ENABLE;
```

//配置在线计费的费率标识

```
ADD URR:URRNAME="onlineURR",URRID=1100,USAGERPTMODE=ONLINE,ONLMETERINGTYPE=EVENT_VOLUME_TIME;
ADD URRGROUP:URRGROUPNAME="urrgroup001",UPURRNAME1="onlineURR",DOWNURRNAME1="onlineURR";
ADD PCCPOLICYGRP:PCCPOLICYGRPNM="cg_red",URRGROUPNAME="urrgroup001";
```

//配置三四层信息

```
ADD FILTER:FILTERNAME="filter_red",L34PROTTYPE=STRING,L34PROTOCOL=TCP,SVRSTARTPORT=80,SVRENDPORT=80;
ADD FLOWFILTER:FLOWFILTERNAME="fg_red";
ADD FLTBINDFLOWF:FLOWFILTERNAME="fg_red",FILTERNAME="filter_red";
```

//配置内容计费规则

```
ADD RULE:RULENAME="Rule002",POLICYTYPE=PCC,FILTERINGMODE=FLOWFILTER, FLOWFILTERNAME="fg_red",POLICYNAME="cg_red",PRIORITY=120;
SET REFRESHSRV:REFRESHTYPE=USERPROFILE;
```

//配置内容计费规则绑定用户模板

```
ADD USERPROFILE:USERPROFILENAME="up_test";
ADD RULEBINDING:USERPROFILENAME="up_test",RULENAME="Rule002";
```

#### [命令功能](#ZH-CN_TOPIC_0000002171168450)

该命令用于显示NP微码补丁加载状态信息。

#### [注意事项](#ZH-CN_TOPIC_0000002171168450)

- 该命令仅适用于NP卡加速模式场景。

#### [操作用户权限](#ZH-CN_TOPIC_0000002171168450)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_TOPIC_0000002171168450)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RUID | RU编号 | 可选必选说明：必选参数。<br>参数含义：RU编号。<br>数据来源：本端规划。<br>取值范围：整数类型，取值范围为0～4294967294。<br>配置原则：使用<br>[DSP RU](../../../../单体服务公共功能管理/系统管理/资源管理/RU管理/显示资源单元信息（DSP RU）_59103857.md)<br>命令查询RU编号。<br>默认值：无。 |

#### [使用实例](#ZH-CN_TOPIC_0000002171168450)

显示RUID为66的NP卡微码补丁加载状态信息：

```
DSP NPMICCODEPATCHSTATUS:RUID=66;
```

```
RETCODE = 0  操作成功

结果如下
--------
补丁状态  =  未加载        
(结果个数 = 1)

---    END
```

#### [输出结果说明](#ZH-CN_TOPIC_0000002171168450)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 补丁状态 | NP微码补丁加载状态。<br>补丁状态枚举类型：<br>- 未加载：未加载NP微码补丁。<br>- 加载中：正在加载NP微码补丁。<br>- 加载成功：加载NP微码补丁成功。<br>- 回退中：正在回退NP微码补丁。<br>- 回退成功：回退NP微码补丁成功。 |

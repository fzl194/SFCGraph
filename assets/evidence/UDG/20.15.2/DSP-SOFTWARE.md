#### [命令功能](#ZH-CN_CONCEPT_0259036808)

该命令用于显示软件仓软件的信息。

#### [注意事项](#ZH-CN_CONCEPT_0259036808)

该指令只在VNFP侧提供。

#### [操作用户权限](#ZH-CN_CONCEPT_0259036808)

G_1，管理员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0259036808)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| VNFCTYPE | VNFC类型 | 可选必选说明：可选参数<br>参数含义：该参数用于指定VNFC类型。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～39。<br>默认值：无<br>配置原则：当不输入时显示所有的VNFC的信息。 |

#### [使用实例](#ZH-CN_CONCEPT_0259036808)

显示软件仓软件信息，请执行以下命令：

```
DSP SOFTWARE:VNFCTYPE="VNRS_VNFC";
```

#### [输出结果说明](#ZH-CN_CONCEPT_0259036808)

| 输出项名称 | 输出项解释 |
| --- | --- |
| VNFC类型 | 表示VNFC类型。 |
| 包类型 | 表示包的类型，包括版本包、风险排查包和App软件包。 |
| 版本应用类型 | 表示版本的应用类型。 |
| 版本类型 | 表示版本类型，包括基础版本、冷补丁、热补丁。 |
| 版本号 | 表示软件版本号。 |
| 基础版本号 | 表示软件基础版本号。 |
| CPU架构类型 | 表示软件支持的CPU架构类型。<br>- **X86**：表示软件支持X86架构。<br>- **ARM**：表示软件支持ARM架构。<br>- **Any**：仅当“版本类型”为“冷补丁”或“热补丁”时，有此输出值，表示补丁软件同时支持X86架构和ARM架构。 |

#### [命令功能](#ZH-CN_CONCEPT_0182837755)

**适用NF：SGW-U、PGW-U、UPF**

该命令用于查询DDoS攻击防护使用的流控阈值。DDoS攻击防护是通过对用户上行TCP SYN报文进行流控来实施的。

#### [注意事项](#ZH-CN_CONCEPT_0182837755)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0182837755)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0182837755)

无。

#### [使用实例](#ZH-CN_CONCEPT_0182837755)

使用LST DDOS命令查询当前配置的DDoS攻击防护使用的流控阈值：

```
LST DDOS:;
```

```

RETCODE = 0  操作成功。

结果如下
--------
DDos防护阈值  =  5000
(结果个数 = 1)
---    END
```

#### [命令功能](#ZH-CN_CONCEPT_0000001600866753)

该命令用于显示OSPFv3引入路由聚合配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001600866753)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600866753)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600866753)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | OSPFv3进程号 | 可选必选说明：可选参数<br>参数含义：OSPFv3进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |
| TOPOID | 拓扑标识 | 可选必选说明：可选参数<br>参数含义：拓扑标识。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围为0。<br>默认值：无 |
| IPADDRESS | IPv6地址 | 可选必选说明：可选参数<br>参数含义：IPv6地址。<br>数据来源：本端规划<br>取值范围：IPv6地址类型。<br>默认值：无 |
| IPV6MASKLEN | IPv6前缀长度 | 可选必选说明：可选参数<br>参数含义：IPv6前缀长度。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～128。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600866753)

显示OSPFv3引入路由聚合配置：

```
LST OSPFV3ASBRSUMMARY:;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600866753)

| 输出项名称 | 输出项解释 |
| --- | --- |
| OSPFv3进程号 | 该参数指定用户创建的OSPFv3进程号。 |
| 拓扑标识 | 该参数指定用户创建的OSPFv3拓扑标识。 |
| IPv6地址 | 该参数指定聚合路由的IPv6地址。 |
| IPv6前缀长度 | 该参数指定聚合路由的IPv6前缀长度。 |
| 使能聚合路由的开销值 | 该参数指定是否使能聚合路由的开销值。 |
| 聚合路由的开销 | 该参数指定聚合路由的开销，仅当使能聚合路由的开销值为true生效。 |
| 使能延迟发布聚合路由 | 该参数指定是否延迟发布聚合路由。 |
| 延迟发布聚合路由的时间（s） | 该参数用于配置发布缺省路由的延时定时器，配置延迟发布聚合路由的时间，仅当使能延迟发布聚合路由为true生效。 |
| 不发布路由 | 该参数用来指定发布或者不发布聚合路由，默认值是发布聚合路由。 |
| 使能聚合路由的标记 | 该参数指定是否使能聚合路由的标签。 |
| 路由标签 | 该参数指定聚合路由的标签，仅当配置使能聚合路由的标记为true生效，默认值为1。 |

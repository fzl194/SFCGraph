#### [命令功能](#ZH-CN_CONCEPT_0000001600441233)

该命令用于查询OSPFv3进程配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001600441233)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600441233)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600441233)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | OSPFv3进程号 | 可选必选说明：可选参数<br>参数含义：OSPFv3进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600441233)

查询OSPFv3进程配置：

```
LST OSPFV3:PROCID=1;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600441233)

| 输出项名称 | 输出项解释 |
| --- | --- |
| OSPFv3进程号 | 该参数指定用户创建的OSPFv3进程号。 |
| 路由器标识 | 该参数指定OSPF进程的路由器标识。 |
| VPN名称 | 该参数指定虚拟专用网(VPN)实例名称。 |
| 抑制接口收发OSPFv3报文 | 该参数指定是否抑制OSPFv3进程下所有接口收发OSPFv3报文标志。 |
| 使能BFD | 该参数指定是否为OSPFv3进程使能动态创建BFD会话特性。 |
| 是否配置BFD最小接收报文间隔 | 该参数指定是否配置了BFD最小接收报文间隔。 |
| 接收BFD包的最小时间间隔（ms） | 该参数指定期望从对端接收BFD报文的最小接收间隔，单位是毫秒，当是否配置BFD最小接收报文间隔为true时生效。 |
| 是否配置BFD最小发送报文间隔 | 该参数指定是否配置了BFD最小发送报文间隔。 |
| 发送BFD包的最小时间间隔（ms） | 该参数指定向对端发送BFD报文的最小发送间隔，单位是毫秒，当是否配置BFD最小发送报文间隔为true时生效。 |
| BFD检测乘数间隔 | 该参数指定会话检测倍数，如果BFD会话在配置的检测周期内没有收到对端发来的BFD报文，则认为链路发生了故障，BFD会话的状态将会置为Down。 |
| VPN引入路由的tag值 | 该参数指定绑定VPN的OSPFv3进程引入外部路由的tag值，当配置VPN引入路由的tag值为true时生效。 |
| 计算链路开销参考值（Mbps） | 该参数指定计算链路开销时所依据的参考值， 单位是兆比特每秒。OSPFv3接口默认开销值的计算公式为：接口开销=带宽参考值/接口带宽，取计算结果的整数部分作为接口开销值（当结果小于1时取1）。 |
| 使能最大重传次数 | 该参数指定是否使能了最大重传上限数目。 |
| 最大重传限制 | 该参数指定限制OSPFv3报文重传次数，并在达到指定的次数后断开邻居，避免在邻居收不到报文的情况下，一直重传造成死循环。当使能最大重传数为true时生效。 |
| 配置Domain ID为空 | 该参数指定删除OSPFv3进程下所有Domain ID配置。 |
| 去使能VPN路由环路检测 | 该参数指定禁止路由环路检测，直接进行路由计算。 |
| 配置VPN引入路由的tag值 | 该参数指定VPN场景下OSPFv3进程引入的外部路由的tag值。 |
| 去使能标签环路检测 | 该参数指定禁止使用VPN的路由标记检测环路。 |
| OSPFv3 Stub路由器 | 该参数指定配置本进程为Stub路由器，配置本参数通过增大该设备所生成的LSA中的链路的度量值（65535），告知其它OSPFv3设备不要使用这个Stub设备来转发数据。但由于度量值不是无穷大，因此仍然可以拥有一个到Stub设备的路由。Stub设备生成的Router LSA中，所有链路的度量值都配置为比较大。 |
| On-Startup时间间隔（s） | 该参数指定路由器在发生主备倒换过程中保持为Stub路由器的时间间隔，当OSPFv3 Stub路由器配置为StubOnBoot时生效。 |
| 描述信息 | 该参数指定OSPFv3进程描述信息。 |
| OSPFv3共网段虚拟系统使能标志 | 该参数指定使能OSPFv3共网段虚拟系统功能。 |
| IPsec SA名称 | 该参数指定IPsec SA的名称。 |
| BFD会话与接口链路状态绑定标志 | 该参数指定将BFD会话状态与接口的链路状态进行绑定。 |

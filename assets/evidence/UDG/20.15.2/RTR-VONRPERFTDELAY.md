# 恢复理想到达报文的最小时延偏差、固定传输时延（RTR VONRPERFTDELAY）

#### [命令功能](#ZH-CN_CONCEPT_0000203295174169)

**适用NF：UPF**

该命令用于恢复理想报文的最小时延偏差、固定传输时延为系统初始设置值。

#### [注意事项](#ZH-CN_CONCEPT_0000203295174169)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000203295174169)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000203295174169)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000203295174169)

恢复理想报文的最小时延偏差、固定传输时延为系统初始设置值：

```
RTR VONRPERFTDELAY:;
```

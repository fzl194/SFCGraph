#### [命令功能](#ZH-CN_CONCEPT_0000001600440897)

该命令用于显示Diameter消费者统计信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600440897)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600440897)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600440897)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001600440897)

显示Diameter消费者统计信息：

```
DSP CONSSTATIC:;
```


#### [输出结果说明](#ZH-CN_CONCEPT_0000001600440897)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 用户PID | 该参数用于表示消费者PID。 |
| 发送用户信息 | 该参数用于表示CSLB发送到消费者网络报文消息统计。 |
| 接收用户信息 | 该参数用于表示CSLB接收消费者网络报文消息统计。 |
| 丢弃用户信息 | 该参数用于显示CSLB丢弃消费者网络报文消息统计。 |
| 最大延迟时间 | 该参数用于显示最大延迟时间。 |
| 消息重传计数 | 该参数用于显示发给消息者的报文消息重传次数。 |

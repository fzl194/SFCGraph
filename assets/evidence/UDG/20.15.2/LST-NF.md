#### [命令功能](#ZH-CN_CONCEPT_0186526378)

**适用NF：SGW-U、PGW-U、UPF**

该命令用于查询NF的状态。

#### [注意事项](#ZH-CN_CONCEPT_0186526378)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0186526378)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0186526378)

无。

#### [使用实例](#ZH-CN_CONCEPT_0186526378)

查询当前所有的NF的信息：

```
LST NF:;
```

```

RETCODE = 0  操作成功

NF信息
--------------
NF锁定状态开关  =  使能
(结果个数 = 1)

---    结束
```

#### [输出结果说明](#ZH-CN_CONCEPT_0186526378)

| 输出项名称 | 输出项解释 |
| --- | --- |
| NF锁定状态开关 | 使能时，NF为锁定状态，该参数为不使能时，NF为解锁状态。 |

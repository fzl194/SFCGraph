## [操作步骤](#ZH-CN_OPI_0000001144919837)

1. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09587387.md)
2. 配置BWM业务。
    a. 用NONTOS定义BWM业务。
      [**ADD BWMSERVICE**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理业务/增加带宽管理业务（ADD BWMSERVICE）_82837473.md)
    b. 配置两个BWM控制器，设置CAR参数并配置带宽控制算法。
      [**ADD BWMCONTROLLER**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理控制器/增加带宽管理控制器（ADD BWMCONTROLLER）_82837460.md)
      > **说明**
      > 进行用户级/用户组级/整机级带宽控制时，如果要优先保证用户组内标识为绿色的用户流量，则需要配置 [**ADD BWMCONTROLLER**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理控制器/增加带宽管理控制器（ADD BWMCONTROLLER）_82837460.md) “COLORISAWARE” 为 “ENABLE” 。
3. 配置带宽管理用户组。
    a. 定义带宽管理用户组，配置优先级和业务类型。
      [**ADD BWMUSERGROUP**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理用户组/增加带宽管理用户组（ADD BWMUSERGROUP）_82837468.md)
    b. 配置用户组级带宽控制规则。
      [**ADD BWMRULE**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理规则/增加带宽管理规则（ADD BWMRULE）_82837478.md)
4. 配置网络切片和带宽管理用户组的绑定关系。
    a. 添加一个新的网络切片选择标识。
      **[ADD SNSSAI](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/网络切片管理/网络切片选择标识/增加NF支持的网络切片选择标识（ADD SNSSAI）_95089577.md)**
    b. 增加网络切片和逻辑接口绑定关系。
      **[ADD SNSSAIUPINTF](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/网络切片管理/网络切片与逻辑接口绑定关系配置/增加网络切片和逻辑接口绑定关系（ADD SNSSAIUPINTF）_51061265.md)**
    c. 绑定网络切片到带宽管理用户组。
      **[ADD SNSSAIBWMUSRG](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理用户切片绑定/增加带宽管理用户组切片绑定（ADD SNSSAIBWMUSRG）_16930642.md)**
5. 配置BWM控制器中的用户组级业务带宽和连接数的生效范围（CPU、整机），以及配置基于切片的带宽控制规则使用BWM Rule。
  [**SET BANDWIDTHMNG**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理参数/设置带宽管理参数（SET BANDWIDTHMNG）_82837465.md)

## [任务示例](#ZH-CN_OPI_0000001144919837)

任务描述

本实例中，操作员在 UDG 上进行数据配置实现以下要求：

UDG 对某个切片下的一个用户组进行流量监管。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH:LICITEM="LKV3G5TCSA01",SWITCH=ENABLE;
```

//配置BWM业务。

```
ADD BWMSERVICE:BWMSERVICENAME="bs_test",BWMSERVICETYPE=NONTOS,NONTOSSRVTYPE=PROTOCOL,PROTOCOLNAME="bt";
```

```
ADD BWMCONTROLLER:BWMCNAME="bc-up",CTRLTYPE=CAR,CIR=300,CBS=10000,PIR=500,PBS=12000,GREEN=PASS,YELLOW=PASS,RED=DISCARD,PRIORITYISAWARE=DISABLE;
```

```
ADD BWMCONTROLLER:BWMCNAME="bc-down",CTRLTYPE=CAR,CIR=300,CBS=10000,PIR=500,PBS=12000,GREEN=PASS,YELLOW=PASS,RED=DISCARD,PRIORITYISAWARE=DISABLE;
```

//配置带宽管理用户组。

```
ADD BWMUSERGROUP:USERGROUPTYPE=SPECIFIC_GROUP,USERGROUPNAME="ug_test",USERGROUPPRI=1,USERGLEVSRVTYPE=NONTOS;
```

```
ADD BWMRULE:USERGROUPTYPE=SPECIFIC_GROUP,USERGROUPNAME="ug_test",BWMRULETYPE=GROUP_SPECIFIC,BWMRULENAME="br_test",BWMSERVICENAME="bs_test",UPBWMCTRLNAME1="bc-up",DNBWMCTRLNAME1="bc-down",BWMRULEPRI=1;
```

//配置网络切片和带宽管理用户组的绑定关系。

```
ADD SNSSAI: SST=1,SD="123456";
```

```
ADD SNSSAIUPINTF: SST=1,SD="123456",N3LOGICINF="n3if1/1/1";
```

```
ADD SNSSAIBWMUSRG:USERGROUPNAME="ug_test",SST=1,SD="123456";
```

配置BWM控制器中的用户组级业务带宽和连接数的生效范围（CPU、整机），以及配置基于切片的带宽控制规则使用BWM Rule。

```
SET BANDWIDTHMNG:RANGE=SYSTEM,CIRMINTHRES=300000,PIRMINTHRES=500000,BWMSUBSCRCTRL=PCC_RULE_IGNORE;
```

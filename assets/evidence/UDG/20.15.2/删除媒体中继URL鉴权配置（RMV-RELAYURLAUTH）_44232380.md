# 删除媒体中继URL鉴权配置（RMV RELAYURLAUTH）

- [注意事项](#ZH-CN_CONCEPT_0000207244232380__1.3.2.1)

#### [命令功能](#ZH-CN_CONCEPT_0000207244232380)

**适用NF：UPF、PGW-U**

该命令用于删除媒体中继URL鉴权配置。

#### [注意事项](#ZH-CN_CONCEPT_0000207244232380)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000207244232380)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000207244232380)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RELAYURLAUTHNM | 媒体中继URL鉴权名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定媒体中继URL鉴权名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。不支持空格，不区分大小写。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000207244232380)

删除媒体中继URL鉴权配置：

```
RMV RELAYURLAUTH: RELAYURLAUTHNM="urlauth";
```

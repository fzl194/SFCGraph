## [命令功能](#ZH-CN_MMLREF_0000001448164178)

该命令用于查询内联口的网关转发地址信息。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000001448164178)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001448164178)

无

## [使用实例](#ZH-CN_MMLREF_0000001448164178)

显示内联口的网关地址信息：

```
%%LST PAEPORTGATEWAY:;%%
RETCODE = 0  操作成功

结果如下
--------
网段索引  平面ID  IP地址类型  网关IPv4地址  IPv4地址掩码  网关IPv6地址  IPv6地址前缀  描述  

1         0       IPv4        192.168.2.3   255.255.0.0   ::            0             NULL  
2         0       IPv4        172.16.52.1   255.255.0.0   ::            0             NULL  
3         0       IPv4        172.28.78.8   255.255.0.0   ::            0             NULL  
(结果个数 = 3)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001448164178)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 网段索引 | 该参数用于指定网段索引。 |
| 平面ID | 该参数用于指定平面ID。可以通过<br>[**DSP PAEPORTGROUPINFO**](../端口/显示PAE端口组信息（DSP PAEPORTGROUPINFO）_44950521.md)<br>指令查询。 |
| IP地址类型 | 该参数用于指定IP地址的类型。 |
| 网关IPv4地址 | 该参数用于指定网关IPv4地址。 |
| IPv4地址掩码 | 该参数用于指定网关IPv4地址的网络掩码。 |
| 网关IPv6地址 | 该参数用于指定网关IPv6地址。 |
| IPv6地址前缀 | 该参数用于指定网关IPv6地址前缀。 |
| 描述 | 该参数用于描述逻辑IP地址的其他信息。 |

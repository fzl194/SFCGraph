#### [命令功能](#ZH-CN_CONCEPT_0000001600601029)

该命令用于查询组件侧转发过载告警参数。

#### [注意事项](#ZH-CN_CONCEPT_0000001600601029)

- 该命令执行后立即生效。
- 该命令仅适用于非NP卡场景和NP卡非加速模式场景。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600601029)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600601029)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| OVERLOADTYPE | 过载类型 | 可选必选说明：必选参数<br>参数含义：该参数用于表示转发过载告警的类型。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- queue：队列告警。<br>默认值：无 |
| RUNAME | 资源单元名称 | 可选必选说明：可选参数<br>参数含义：指定资源单元名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600601029)

查询组件侧转发过载告警参数：

```
DSP SFELOADALARM:OVERLOADTYPE=queue;
```

```

RETCODE = 0 操作成功。

结果如下
-------------------------
 触发阈值（%） = 92
 恢复阈值（%） = 60
检测间隔（ms） = 1000
      检测次数 = 10
      使能标记 = TRUE
  资源单元名称 = VNODE_VNRS_VNFC_IPU_0066
(结果个数 = 1)
--- 结束
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600601029)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 触发阈值（%） | 用于表示转发过载告警的触发阈值。 |
| 恢复阈值（%） | 用于表示转发过载告警的恢复阈值。 |
| 检测间隔（ms） | 用于表示转发过载告警的检测间隔，单位为ms。 |
| 检测次数 | 用于表示转发过载告警的检测次数。 |
| 使能标记 | 用于表示转发过载告警是否使能。 |
| 资源单元名称 | 指定资源单元名称。 |

#### [命令功能](#ZH-CN_CONCEPT_0000201106213368)

**适用NF：PGW-U、UPF**

本命令用于查询当配额耗尽时，UPF是否阻塞最后一个超出配额范围的数据报文。

#### [注意事项](#ZH-CN_CONCEPT_0000201106213368)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000201106213368)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000201106213368)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000201106213368)

查询UPF是否阻塞最后一个超出配额范围的数据报文：

```
LST DROPFINALPKT:;
```

```

RETCODE = 0  操作成功

配额耗尽时末包动作
-----------------
开关  =  ENABLE
(结果个数 = 1)

---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000201106213368)

参见SET DROPFINALPKT的参数说明。

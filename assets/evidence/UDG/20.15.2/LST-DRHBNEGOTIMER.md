## [命令功能](#ZH-CN_MMLREF_0000001323235162)

该命令用于查询容灾定时器。

> **说明**
> - 该命令只用于在UEG-L/UEN网元采用主备（冷备）容灾模式下执行。
> - 该命令只用于在UEG-M/UEG网元采用主备（热备）容灾模式下执行。

#### [操作用户权限](#ZH-CN_MMLREF_0000001323235162)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001323235162)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| DRGROUPID | 容灾组标识 | 可选必选说明：可选参数<br>参数含义：该参数用来标识一个容灾组，该标识必须存在于容灾组信息中，容灾组信息可以使用<br>[**LST DRGROUPINFO**](查询容灾组信息（LST DRGROUPINFO）_74835153.md)<br>命令查询。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是1~8。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001323235162)

- 查询所有的容灾定时器:
  ```
  %%LST DRHBNEGOTIMER:;%%
  RETCODE = 0  操作成功

  结果如下
  --------
                   容灾组标识 = 1
    心跳消息重传时间(100毫秒) = 10
    协商消息重传时间(100毫秒) = 20
                     重传次数 = 3
  OM通道心跳探测时长(100毫秒) = 10
           OM通道心跳重传次数 = 3
  (结果个数 = 1)

  ---    END
  ```
- 查询指定容灾组的容灾定时器:
  ```
  %%LST DRHBNEGOTIMER: DRGROUPID=1;%%
  RETCODE = 0  操作成功

  结果如下
  --------
                   容灾组标识 = 1
    心跳消息重传时间(100毫秒) = 10
    协商消息重传时间(100毫秒) = 20
                     重传次数 = 3
  OM通道心跳探测时长(100毫秒) = 10
           OM通道心跳重传次数 = 3
  (结果个数 = 1)

  ---    END
  ```

## [输出结果说明](#ZH-CN_MMLREF_0000001323235162)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 容灾组标识 | 该参数用来标识一个容灾组，该标识必须存在于容灾组信息中，容灾组信息可以使用<br>[**LST DRGROUPINFO**](查询容灾组信息（LST DRGROUPINFO）_74835153.md)<br>命令查询。 |
| 心跳消息传递间隔(100毫秒) | 该参数用于配置容灾组内网元间发送心跳消息的间隔时间。 |
| 协商消息传递间隔(100毫秒) | 该参数用于配置容灾组内网元间发送协商消息的间隔时间。 |
| 重传次数 | 该参数用于配置容灾组内网元间发送心跳和协商消息的重传次数。 |
| OM通道心跳重传次数 | 该参数用于配置容灾组内OM通道发送心跳的重传次数。 |
| OM通道心跳探测时长(100毫秒) | 该参数用于配置容灾组OM通道发送心跳消息的间隔时间。 |

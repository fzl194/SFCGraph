#### [命令功能](#ZH-CN_CONCEPT_0000001549802622)

该命令用于显示FES平滑状态信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001549802622)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549802622)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549802622)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IDENTITYFLAG | 身份标志位 | 可选必选说明：可选参数<br>参数含义：该参数用于表示身份标志位。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- RUNAME：表示输入资源单元名称。<br>- COMPID：表示输入组件ID。<br>默认值：无 |
| COMPONENTID | 组件ID | 可选必选说明：条件必选参数<br>前提条件：该参数在“IDENTITYFLAG”配置为“COMPID”时为必选参数。<br>参数含义：该参数用于表示组件ID。<br>数据来源：本端规划<br>取值范围：十六进制整数类型，取值范围为0x0～0xFFFFFFFF。<br>默认值：无 |
| RUNAME | 资源单元名称 | 可选必选说明：条件必选参数<br>前提条件：该参数在“IDENTITYFLAG”配置为“RUNAME”时为必选参数。<br>参数含义：该参数用于表示资源单元名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。<br>默认值：无<br>配置原则：区分大小写，不支持空格，下发本MML命令前可使用DSP RU查看资源单元信息。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549802622)

显示FES平滑状态信息：

```
DSP FESSMOOTHSTAT:;
```

```

RETCODE = 0 操作成功

结果如下
------------------------
                  订阅的Class ID  =  0
生产者APP组件与FES之间的平滑状态  =  平滑结束
生产者CFG组件与FES之间的平滑状态  =  平滑结束
                     对端组件PID  =  0xffffff
       FES与消费者之间的平滑状态  =  平滑结束
(结果个数 = 1)
----- END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001549802622)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 订阅的Class ID | 用于表示订阅的Class ID。 |
| 生产者APP组件与FES之间的平滑状态 | 该参数用于表示生产者APP与FES之间的平滑状态，包括：<br>- 平滑结束。<br>- 平滑不结束。 |
| 生产者CFG组件与FES之间的平滑状态 | 该参数用于表示生产者CFG与FES之间的平滑状态，包括：<br>- 平滑结束。<br>- 平滑不结束。 |
| 对端组件PID | 用于表示对端组件PID。 |
| FES与消费者之间的平滑状态 | 该参数用于表示FES与消费者之间的平滑状态，包括：<br>- 平滑结束。<br>- 平滑不结束。 |

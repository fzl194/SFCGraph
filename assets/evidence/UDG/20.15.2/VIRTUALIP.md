## [命令功能](#ZH-CN_MMLREF_0000001147819609)

用于查询登录OM Portal的浮动IP类型、浮动IP、网关、子网掩码和前缀长度信息。

> **说明**
> 无。

## [参数说明](#ZH-CN_MMLREF_0000001147819609)

无。

## [使用实例](#ZH-CN_MMLREF_0000001147819609)

查询浮动IP信息：

```
%%LST VIRTUALIP:;%%
RETCODE = 0  操作成功

操作结果如下
------------
浮动IP类型  =  IPv4
  IPv4地址  =  192.168.40.47
  IPv4网关  =  192.168.40.1
  子网掩码  =  255.255.255.0
      端口  =  31943
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001147819609)

命令执行正常，会返回命令执行成功的提示信息。输出结果说明如 [表1](#ZH-CN_MMLREF_0000001147819609__table47787675) 所示。

*表1 输出项解释*

| **输出项名称** | **输出项解释** |
| --- | --- |
| 浮动IP类型 | 登录OM Portal的浮动IP类型，取值为IPv4或IPv6。 |
| IPv4地址 | 登录OM Portal的浮动IPv4地址。 |
| IPv6地址 | 登录OM Portal的浮动IPv6地址。 |
| 端口 | 登录OM Portal的端口。 |
| IPv4网关 | 登录OM Portal的浮动IPv4地址的网关。 |
| IPv6网关 | 登录OM Portal的浮动IPv6地址的网关。 |
| 子网掩码 | 登录OM Portal的子网掩码，<br>浮动IP类型是IPV6的时候，不显示子网掩码的取值。 |
| 前缀长度 | 登录OM Portal的前缀长度，<br>浮动IP类型是IPV4的时候，不显示前缀长度<br>的取值。 |

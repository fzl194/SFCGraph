#### [命令功能](#ZH-CN_CONCEPT_0000207394871985)

**适用NF：UPF、PGW-U**

该命令用于查询证书更新拆链配置参数。

#### [注意事项](#ZH-CN_CONCEPT_0000207394871985)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000207394871985)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000207394871985)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000207394871985)

假如需要查询证书更新拆链配置参数，则命令如下：

```
LST SRVCERTUPDCFG:;
```

```

RETCODE = 0  操作成功
结果如下
------------------------
          证书更新时重建链开关  =  打开
单进程每批次重建链路间隔（秒）  =  4
  单进程每批次重建链路数（条）  =  10
    证书延迟重建链的时间(分钟)  =  5
(结果个数 = 1)
---    END
```

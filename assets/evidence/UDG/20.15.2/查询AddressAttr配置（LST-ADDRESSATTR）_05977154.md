#### [命令功能](#ZH-CN_CONCEPT_0000201105977154)

**适用NF：PGW-U、UPF**

该命令用来查询全局地址分配属性。

#### [注意事项](#ZH-CN_CONCEPT_0000201105977154)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000201105977154)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000201105977154)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000201105977154)

查询全局地址分配属性：

```
LST ADDRESSATTR:;
```

```

返回值 = 0  操作成功

全局地址分配属性
-----------------------------------
IPv4地址池通配功能开关  =  ENABLE
IPv6地址池通配功能开关  =  DISABLE
         地址租期（秒） =  30
  IPv4主机路由发布开关  =  ENABLE
  IPv6主机路由发布开关  =  ENABLE
```

## [操作步骤](#ZH-CN_OPI_0202455576)

1. 打开本特性的License配置开关。
  **[SET LICENSESWITCH](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09587387.md)**
2. 配置APN下使能终端异常下行流量检测功能。
  [**ADD ABNTRAFFICDT**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务防欺诈/APN下终端异常下行流量检测开关/新增异常流量检测开关（ADD ABNTRAFFICDT）_82837035.md)

## [任务示例](#ZH-CN_OPI_0202455576)

任务描述

在 UDG 上开启终端异常下行流量检测功能。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH:LICITEM ="LKV3G5ADTD01",SWITCH=ENABLE;
```

//配置开启终端异常下行流量检测功能。

```
ADD ABNTRAFFICDT:APN="apn1.com",SWITCH=ENABLE,TRAFFICBEHAVIORSWITCH=ENABLE;
```

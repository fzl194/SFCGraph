#### [命令功能](#ZH-CN_CONCEPT_0000001600440757)

该命令用于显示LDP LSP。

#### [注意事项](#ZH-CN_CONCEPT_0000001600440757)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600440757)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600440757)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| VRFNAME | VPN实例名称 | 可选必选说明：可选参数<br>参数含义：该参数用于表示VPN实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。<br>默认值：无 |
| LSPADDR | 目的地址 | 可选必选说明：可选参数<br>参数含义：该参数用于表示目的地址。<br>数据来源：本端规划<br>取值范围：IPv4地址类型。<br>默认值：无<br>配置原则：当“PREFIXLENGTH”参数被输入的时候，该参数也必须输入。 |
| PREFIXLENGTH | 前缀长度 | 可选必选说明：可选参数<br>参数含义：该参数用于表示目的地址的前缀长度。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～32。<br>默认值：无<br>配置原则：当“LSPADDR”参数被输入的时候，该参数也必须输入。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600440757)

显示LDP LSP：

```
DSP LDPLSP:;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600440757)

| 输出项名称 | 输出项解释 |
| --- | --- |
| VPN实例名称 | 用于表示VPN实例名称。 |
| 目的地址 | 用于表示目的地址。 |
| 前缀长度 | 用于表示目的地址的前缀长度。 |
| LSP索引 | 用于表示LSP的标识号。 |
| LSP下一跳 | 用于表示流量到该节点后的出口方向。 |
| 入标签 | 用于表示流量入口的标签值。 |
| 出标签 | 用于表示流量出口的标签值。 |
| 出接口 | 用于表示流量到该节点后从哪个接口流出。 |
| LSP类型 | 用于表示LSP类型。包含入节点、中间节点、出节点、入口和中间节点、bud节点。 |
| LSP MTU值 | 用于表示LSP MTU值。MPLS在Ingress根据此MTU来决定MPLS转发报文的大小，从而避免在Ingress发送的报文较大，导致Transit转发失败。 |
| FRR LSP | 用于表示该LSP是否是FRR LSP。 |
| RLFA LSP | 用于表示该LSP是否是RLFA LSP。 |
| LSP建立的时长（s） | 用于表示LSP建立的时长，单位是秒。 |

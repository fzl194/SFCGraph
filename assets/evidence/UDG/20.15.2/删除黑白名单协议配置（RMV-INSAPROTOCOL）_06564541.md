# 删除黑白名单协议配置（RMV INSAPROTOCOL）

- [注意事项](#ZH-CN_CONCEPT_0000206906564541__1.3.2.1)

#### [命令功能](#ZH-CN_CONCEPT_0000206906564541)

**适用NF：PGW-U、UPF**

删除黑白名单协议配置。

#### [注意事项](#ZH-CN_CONCEPT_0000206906564541)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000206906564541)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000206906564541)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROTOCOLNAME | 协议名称 | 可选必选说明：必选参数<br>参数含义：协议组包含的协议的名字。<br>数据来源：本端规划<br>取值范围：1、字符串类型，输入长度范围为1～31; 2、不支持空格，不区分大小写。<br>默认值：无<br>配置原则：数据源为系统支持识别的子协议，可以通过工程命令smctrldsp protocol-list sub-protocol查询。 |

#### [使用实例](#ZH-CN_CONCEPT_0000206906564541)

删除http协议的黑白名单协议配置：

```
RMV INSAPROTOCOL:PROTOCOLNAME="http";
```

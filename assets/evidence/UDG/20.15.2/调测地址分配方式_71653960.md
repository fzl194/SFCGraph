# 调测地址分配方式

- [操作场景](#ZH-CN_OPI_0171653960__1.3.1)

## [操作场景](#ZH-CN_OPI_0171653960)

用户入网时采用外部地址分配方式激活。

> **说明**
> 适用于PGW-U、UPF。

## [必备事项](#ZH-CN_OPI_0171653960)

前提条件

- 请仔细阅读[GWFD-010104 地址分配方式](../GWFD-010104 地址分配方式_71653946.md)。
- 完成[激活地址分配方式](激活地址分配方式_71653959.md)。

数据

| 类别 | 参数名称 | 取值样例 | 获取方法 | 说明 |
| --- | --- | --- | --- | --- |
| 用户信息查询 | 用户IMSI号（IMSI） | 460000123456789 | 测试终端自带 | - |
| 测试终端使用的APN | APN名称（APN） | apn-test | 已配置数据中获取 | 取自终端接入运营商自身IP网络中配置的APN实例名。 |

工具

- 测试终端
- OM Portal

## [操作步骤](#ZH-CN_OPI_0171653960)

1. 测试终端使用“apn-test”APN发起接入网络请求。
2. 执行 [**DSP SESSIONINFO**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/会话信息管理/会话信息查询/显示用户上下文（DSP SESSIONINFO）_86526407.md) 命令，通过 测试终端 的IMSI查看 测试终端 是否成功激活。
  ```
  DSP SESSIONINFO:QUERYTYPE=IMSI,IMSI="460000123456789";
  ```
  ```
  ......

  -------------------------------
                                   IMSI  =  460000123456789
                                       ......              
                               PDP Type  =  IPv4 
                      IPv4 Address type  =  EXTERNAL ALLOC IP ADDRESS
                       IPv4 PDP address  =  10.10.10.1         
                                       ......    
                               APN name  =  apn-test
                                       ......       
  (Number of results = 1)
  ---    END
  ```
    - 如果测试终端激活成功，“IPv4 Address type”的值为“EXTERNAL ALLOC IP ADDRESS”，且使用的“IPv4 PDP address”规划值一致，测试终端接入成功，调测结束。
    - 如果测试终端激活成功，但使用的“IPv4 Address type”或“IPv4 PDP address”与规划值不一致，或测试终端激活失败，执行[步骤 3](#ZH-CN_OPI_0171653960__step7)。
3. 收集信息并寻求技术支持。
    a. 在OM Portal上创建用户跟踪任务 并保存报文。
    b. 执行 [**EXP MML**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/配置管理/配置导出管理/导出MML文件（EXP MML）_47200033.md) 命令将当前配置数据导出为MML脚本文件并保存。
    c. 收集并保存上述所有查询信息。
    d. 收集归纳所有信息并联系华为技术支持解决。

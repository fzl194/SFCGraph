# 调测静态地址用户路由冗余

- [操作场景](#ZH-CN_OPI_0319612411__1.3.1)
- [必备事项](#ZH-CN_OPI_0319612411__1.3.2)
- [操作步骤](#ZH-CN_OPI_0319612411__1.3.3)

## [操作场景](#ZH-CN_OPI_0319612411)

当运营商需要部署静态地址用户路由冗余功能时，需对 UDG 的静态地址用户路由冗余功能进行调测，确保本功能可以正常使用。

当主用 UDG 故障时，通过主备两台 UDG 间建立的备份GRE Tunnel，备用 UDG 及时承担起主用 UDG 的网络功能。本章节需要手工构造主用 UDG 故障场景，因此建议在空载场景下进行特性调测， 避免影响正常的在线用户业务。

> **说明**
> 适用于PGW-U、UPF。

## [必备事项](#ZH-CN_OPI_0319612411)

前提条件

- 请仔细阅读[GWFD-010107 静态地址用户路由冗余](../GWFD-010107 静态地址用户路由冗余_19612408.md)。
- 完成主备UDG组网。
- 完成[激活静态地址用户路由冗余](激活静态地址用户路由冗余_19612403.md)。

数据

| 类别 | 参数名称 | 取值样例 | 获取方法 | 说明 |
| --- | --- | --- | --- | --- |
| 用户跟踪任务 | 用户IMSI号（IMSI） | 460000123456789 | 测试终端自带 | 终端地址是静态地址 |
| 测试终端使用的APN | APN名称（APN） | apn1 | 已配置数据中获取 | - |

工具

- 测试终端
- OM Portal
- U2020/MAE

## [操作步骤](#ZH-CN_OPI_0319612411)

1. 对于主备 UDG ， 在OM Portal上创建 UDG 用户跟踪任务 。
2. 测试终端 使用“apn1”APN接入网络，查看主用 UDG 的OM Portal用户跟踪任务结果。
    - 如果测试终端成功接入网络，请执行[步骤 3](#ZH-CN_OPI_0319612411__step4)。
    - 如果测试终端无法接入网络，请调测UDG的接入功能。
3. 测试终端 访问 **www.huawei.com** 网页。
    - 如果测试终端可以正常访问网页，请执行[4](#ZH-CN_OPI_0319612411__cmd567556178175813)。
    - 如果测试终端无法正常访问网页，请调测UDG的Web浏览功能。
4. 手工构造主用 UDG 故障，例如：复位主用 UDG 。 测试终端 重新接入网络，查看备用 UDG 的OM Portal用户跟踪任务结果。
    - 如果 测试终端 成功接入网络，请执行 [5](#ZH-CN_OPI_0319612411__cmd1386518493175813) 。
    - 如果测试终端无法接入网络，请调测UDG的接入功能。
5. 测试终端 访问 **www.huawei.com** 网页。
    - 如果测试终端可以正常访问网页，请执行[6](#ZH-CN_OPI_0319612411__cmd1593606220175813)。
    - 如果测试终端无法正常访问网页，请调测UDG的Web浏览功能。
6. 主用 UDG 故障恢复后，在U2020/MAE上创建如下性能指标：
    - 主 UDG ：
      [1914307630 用户平面静态备份用户发送千字节数](../../../../../OM参考/性能指标/UDG性能指标/用户面指标/转发测量/用户平面转发测量/1914307630 用户平面静态备份用户发送千字节数_73546562.md)
      [1914307631 用户平面静态备份用户发送包数](../../../../../OM参考/性能指标/UDG性能指标/用户面指标/转发测量/用户平面转发测量/1914307631 用户平面静态备份用户发送包数_73546563.md)
      [1914307632 用户平面静态备份用户发送峰值千字节速率](../../../../../OM参考/性能指标/UDG性能指标/用户面指标/转发测量/用户平面转发测量/1914307632 用户平面静态备份用户发送峰值千字节速率_73546564.md)
      [1914307633 用户平面静态备份用户发送峰值包速率](../../../../../OM参考/性能指标/UDG性能指标/用户面指标/转发测量/用户平面转发测量/1914307633 用户平面静态备份用户发送峰值包速率_73546565.md)
    - 备 UDG ：
      [1914307626 用户平面静态备份用户接收千字节数](../../../../../OM参考/性能指标/UDG性能指标/用户面指标/转发测量/用户平面转发测量/1914307626 用户平面静态备份用户接收千字节数_73546558.md)
      [1914307627 用户平面静态备份用户接收包数](../../../../../OM参考/性能指标/UDG性能指标/用户面指标/转发测量/用户平面转发测量/1914307627 用户平面静态备份用户接收包数_73546559.md)
      [1914307628 用户平面静态备份用户接收峰值千字节速率](../../../../../OM参考/性能指标/UDG性能指标/用户面指标/转发测量/用户平面转发测量/1914307628 用户平面静态备份用户接收峰值千字节速率_73546560.md)
      [1914307629 用户平面静态备份用户接收峰值包速率](../../../../../OM参考/性能指标/UDG性能指标/用户面指标/转发测量/用户平面转发测量/1914307629 用户平面静态备份用户接收峰值包速率_73546561.md)
      [1914307822 用户平面当前在线静态备份会话数](../../../../../OM参考/性能指标/UDG性能指标/用户面指标/会话测量/用户平面会话测量/1914307822 用户平面当前在线静态备份会话数_70439308.md)
    - 如果主备UDG上成功收集到静态备份功能相关的性能指标，调测结束。
    - 如果主备UDG上不能成功收集到静态备份功能相关的性能指标，请执行[步骤 7](#ZH-CN_OPI_0319612411__step11032013621)。
7. 执行 [**DSP TUNNELINFO**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/VPN管理/VPN隧道管理/隧道信息/显示隧道信息（DSP TUNNELINFO）_00841297.md) 查看GRE隧道状态是否正常。
    - 隧道状态为UP，请执行 [步骤 12](#ZH-CN_OPI_0319612411__step42841220155514) 。
    - 隧道状态为DOWN，请执行 [步骤 8](#ZH-CN_OPI_0319612411__step6a) 。
8. 执行 [**LST REDUNDUSER**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/会话地址管理/静态地址用户路由冗余/路由冗余开关/查询静态地址用户路由冗余功能（LST REDUNDUSER）_71074368.md) 命令，查看静态地址用户路由冗余功能是否使能。
    - 如果使能，请执行 [步骤 9](#ZH-CN_OPI_0319612411__step7) 。
    - 如果不使能，请参考 [激活静态地址用户路由冗余](激活静态地址用户路由冗余_19612403.md) 重新配置。
9. 执行 [**LST POOL**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/会话地址管理/地址池配置/显示地址池（LST POOL）_82837135.md) 命令，查看 UDG 上配置的地址池数据是否与规划数据一致。
  ```
  RETCODE = 0  操作成功
  地址池信息
  --------------------
   地址池名称      地址池类型      IP地址类型      ······      绑定VPN      VPN实例名     冗余备份功能开关

   pool1           EXTERNAL         IPV4            ······      使能         vpn1         使能     
   pool2           EXTERNAL         IPV6            ······      使能         vpn1         使能
  (结果个数 = 2)
  ---    END
  ```
    - 如果是，请执行 [步骤 10](#ZH-CN_OPI_0319612411__step8) 。
    - 如果不是，请参考 [激活静态地址用户路由冗余](激活静态地址用户路由冗余_19612403.md) 重新配置。
10. 执行 [**LST GRETUNNEL**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/VPN管理/GRE管理/GRE隧道/查询GRE隧道（LST GRETUNNEL）_49802638.md) 命令，查看 UDG 上配置Tunnel的隧道数据是否与规划数据一致。
  ```
  LST GRETUNNEL:;
  ```
  ```
  RETCODE = 0  操作成功  
  结果如下 
  -------------------------
                隧道名称  =  Tunnel1
                隧道类型  =  GRE隧道
              源IPv4地址  =  10.11.1.1
            目的IPv4地址  =  10.10.0.1
       使能Keepalive功能  =  使能
             目的VPN名称  =  vpn2
        使能备份隧道功能  =  TRUE 
  (结果个数 = 1) 
  ---    END
  ```
    - 如果与规划数据一致，请执行 [步骤 11](#ZH-CN_OPI_0319612411__step9) 。
    - 如果不一致，请参考 [激活静态地址用户路由冗余](激活静态地址用户路由冗余_19612403.md) 重新配置。
11. 执行 [**LST REDUNDRDTIP**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/会话地址管理/静态地址用户路由冗余/冗余备份重定向IP/查询冗余备份重定向IP（LST REDUNDRDTIP）_75097447.md) 命令，查看 UDG 上配置的冗余备份重定向IP地址是否与规划数据一致。
  ```
  LST REDUNDRDTIP:;
  ```
  ```
  RETCODE = 0  操作成功
  冗余备份重定向IP
  --------------------
  IP版本      IPV4地址      IPV6地址

  IPv4        192.168.0.1   ::
  IPv6        0.0.0.0       fc00:3::1
  (结果个数  = 2)
  ---    END
  ```
    - 如果IP地址一致，请执行 [步骤 12](#ZH-CN_OPI_0319612411__step42841220155514) 。
    - 如果不一致，请参考 [激活静态地址用户路由冗余](激活静态地址用户路由冗余_19612403.md) 重新配置。
12. 收集信息并寻求技术支持。
    a. 在OM Portal上创建用户跟踪任务，执行 [步骤 2](#ZH-CN_OPI_0319612411__step3) 和 [步骤 4](#ZH-CN_OPI_0319612411__step5) 并保存主备 UDG 对应的报文。
    b. 执行 [**EXP MML**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/配置管理/配置导出管理/导出MML文件（EXP MML）_47200033.md) 命令将当前配置数据导出为MML脚本文件并保存。
    c. 收集并保存上述所有查询信息。
    d. 收集归纳所有信息并联系华为技术支持解决。

## [命令功能](#ZH-CN_MMLREF_0000002211535957)

该命令用于查询DCS点播参数。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000002211535957)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000002211535957)

无

## [使用实例](#ZH-CN_MMLREF_0000002211535957)

查询DCS点播参数。

```
%%LST DCSVODPARAS:;%%
RETCODE = 0  操作成功

操作结果如下：
------------------------
          热度更新频次  =  100
          系统最大热度  =  20000
     热度衰减速率（%）  =  50
   L0加入热度阈值（%）  =  95
   L2最大节点容量（G）  =  14336
 磁盘淘汰起始阈值（G）  =  500
 磁盘老化对象大小（G）  =  300
元数据淘汰起始阈值（%） =  95
    元数据老化对象个数  =  2000
 对象强制老化周期（h）  =  168
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000002211535957)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 热度更新频次 | 该参数用于表示元数据被访问的次数达到设置值，更新一次热度信息。 |
| 系统最大热度 | 该参数用于配置视频的热度上限。 |
| 热度衰减速率（%） | 该参数用于设置视频热度每天衰减的比例。 |
| L0加入热度阈值（%） | 该参数用于表示资源热度在L0资源中的排位达到阈值时，将该资源加入L0区域，L0表示极热片头存储区。 |
| L2最大节点容量（G） | 该参数用于配置L2层的磁盘容量，L2用于存储温热差异区视频。 |
| 磁盘淘汰起始阈值（G） | 该参数用于表示磁盘剩余存储空间达到阈值时，开始老化磁盘中存储的视频。 |
| 磁盘老化对象大小（G） | 该参数用于配置磁盘老化视频的大小。 |
| 元数据淘汰起始阈值（%） | 该参数用于表示元数据数量占比达到阈值时，开始老化元数据。 |
| 元数据老化对象个数 | 该参数用于配置元数据老化的个数。 |
| 对象强制老化周期（h） | 该参数用于配置强制老化视频的周期。 |

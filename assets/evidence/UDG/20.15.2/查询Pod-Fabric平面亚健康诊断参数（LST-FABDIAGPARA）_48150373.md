## [命令功能](#ZH-CN_MMLREF_0000002148150373)

该命令用来显示Pod Fabric平面亚健康诊断参数。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000002148150373)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000002148150373)

无

## [使用实例](#ZH-CN_MMLREF_0000002148150373)

显示Pod Fabric平面亚健康诊断参数

```
%%LST FABDIAGPARA:;%%
RETCODE = 0  操作成功

结果如下
--------
POD亚健康阈值 = 50
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000002148150373)

| 输出项名称 | 输出项解释 |
| --- | --- |
| POD亚健康阈值(%) | 该参数用于指定Pod级fabric平面亚健康诊断阈值，单位：%。 |

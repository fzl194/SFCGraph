#### [命令功能](#ZH-CN_CONCEPT_0000203719172436)

**适用NF：PGW-U、UPF**

该命令用于查询主动触发用户位置实时通知功能。

#### [注意事项](#ZH-CN_CONCEPT_0000203719172436)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000203719172436)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000203719172436)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000203719172436)

查询主动触发用户位置实时通知功能：

```
%%LST USRREALLOCNTY:;
```

```
%%
RETCODE = 0  操作成功

主动触发用户位置实时通知功能
----------------------------
主动触发用户位置实时通知功能开关  =  不使能
                      用户抽样比  =  10
(结果个数 = 1)

---    END
```

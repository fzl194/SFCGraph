#### [命令功能](#ZH-CN_CONCEPT_0000001549802210)

该命令用于查询LDP实例配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001549802210)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549802210)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549802210)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| VRFNAME | VPN实例名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定VPN实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549802210)

查询LDP实例配置：

```
LST LDPINSTANCE:;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001549802210)

| 输出项名称 | 输出项解释 |
| --- | --- |
| VPN实例名称 | 该参数用于表示VPN实例名称。 |
| 实例的LSR ID | 该参数用于表示LDP实例的LSR ID。当下发0.0.0.0时表示LSR ID被删除。 |
| IGP联动延迟时间（s） | 该参数用于表示IGP联动延迟时间，单位是秒。 |
| 优雅删除标志 | 该参数用于表示优雅删除标志。通过配置Graceful Delete功能，可以提高LDP-IGP联动的切换速度，提高整网可靠性。 |
| 优雅删除时间（s） | 该参数用于表示优雅删除时间，单位是秒。 |
| 禁止向remote-peer发送mapping | 该参数用于表示是否禁止向remote-peer发送mapping。 |
| LDP全局认证模式 | 该参数用于表示LDP全局认证模式。 |
| LDP全局认证类型 | 该参数用于表示LDP全局认证类型。当前支持MD5和KEYCHAIN认证类型。 |
| LDP全局KeyChain名字 | 该参数用于表示LDP全局KeyChain名字。配置LDP KeyChain认证，可以提高LDP会话连接的安全性。需要在会话两端的LSR上进行配置。 |
| LDP全局MD5密码 | 该参数用于表示LDP全局MD5密码。在对安全性要求较高的网络中，可以通过配置LDP MD5认证来提高网络的安全性。注意，MD5属于不安全的加密算法，建议使用Keychain认证。 |
| 水平分割 | 该参数用于表示水平分割。通过配置LDP水平分割策略，可以限制不必要的LSP的建立，从而节省内存。 |
| 发送所有loopback地址 | 该参数用于表示是否发送所有loopback地址。 |
| 标签控制模式 | 该参数用于表示标签控制模式。在LSP的建立过程中，LSR分配标签时采用的处理方式。 |

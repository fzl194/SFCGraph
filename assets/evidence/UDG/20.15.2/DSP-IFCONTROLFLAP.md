#### [命令功能](#ZH-CN_CONCEPT_0000001550281650)

该命令用于显示接口震荡抑制动态信息。

若不指定IFNAME参数时，则查询所有接口的震荡抑制动态信息；若指定IFNAME参数时，则可以查询指定接口的震荡抑制动态信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001550281650)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550281650)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550281650)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IFNAME | 接口名 | 可选必选说明：可选参数<br>参数含义：该参数用于指定震荡抑制的接口名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。接口名称由接口类型+接口编号组成。<br>默认值：无<br>配置原则：请使用LST INTERFACE命令查看可用接口。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001550281650)

显示接口震荡抑制动态信息：

```
DSP IFCONTROLFLAP:;
```

```

RETCODE = 0  操作成功

结果如下
-------------------------
接口名            当前震荡次数    当前惩罚值    当前抑制状态
Ethernet64/0/3    0               0.000         未抑制
Ethernet64/0/4    1000            0.100         抑制
(结果个数 = 2)
---   END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001550281650)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 接口名 | 用于指定震荡抑制的接口名。 |
| 当前震荡次数 | 用于查询接口当前震荡次数。 |
| 当前惩罚值 | 用于查询接口当前惩罚值。 |
| 当前抑制状态 | 用于查询接口当前处于抑制还是未抑制状态。 |

#### [命令功能](#ZH-CN_CONCEPT_0000204780175090)

**适用NF：PGW-U、UPF**

该命令用于查询5G LAN计费控制相关参数。

#### [注意事项](#ZH-CN_CONCEPT_0000204780175090)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000204780175090)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000204780175090)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000204780175090)

查询5G LAN计费控制相关参数：

```
LST NGLANCHGCTRL:;
```

```

RETCODE = 0  操作成功

5GLAN计费控制相关参数信息
----------------------
                    5GLAN 本地离线计费策略开关  =  是
(结果数 = 1)
```

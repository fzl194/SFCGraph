#### [命令功能](#ZH-CN_CONCEPT_0186527101)

**适用NF：SGW-U、UPF**

该命令用来查询报文DDN触发控制策略。

#### [注意事项](#ZH-CN_CONCEPT_0186527101)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0186527101)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0186527101)

无。

#### [使用实例](#ZH-CN_CONCEPT_0186527101)

查询报文DDN触发控制策略：

```
LST DDNCTRL:;
```

```

RETCODE = 0 操作成功。

DDN控制策略
------------------
  报文类型 = Tcp 信令
  使能开关 = 使能
  报文策略 = 丢弃报文 
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0186527101)

参见SET DDNCTRL的参数说明。

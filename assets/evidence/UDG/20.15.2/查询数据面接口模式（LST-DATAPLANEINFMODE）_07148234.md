#### [命令功能](#ZH-CN_CONCEPT_0000203207148234)

**适用NF：SGW-U、PGW-U、UPF**

该命令用于查看用户面IP模式配置。

#### [注意事项](#ZH-CN_CONCEPT_0000203207148234)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000203207148234)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000203207148234)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000203207148234)

查看当前配置的用户面IP模式配置，可以使用该命令：

```
LST DATAPLANEINFMODE:;
```

```

RETCODE = 0  操作成功。

用户面IP模式配置
------------------
            模式  =  Instance
路由增强功能开关  =  DISABLE
(结果个数 = 1)
---    END
```

> **说明**
> 无。

## [参数说明](#ZH-CN_MMLREF_0000001469362704)

无。

## [使用实例](#ZH-CN_MMLREF_0000001469362704)

查询NCC策略时，执行以下命令：

```
%%LST NCCPOLICY:;%% 
RETCODE = 0  操作成功  

操作结果如下 
------------ 
策略标识  =  内存优化开关策略 
策略内容  =  关闭 
(结果个数 = 1)  

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001469362704)

| **输出项名称** | **输出项解释** |
| --- | --- |
| 策略标识 | 用于返回查询到的策略标识 |
| 策略内容 | 用于返回查询到的策略内容 |

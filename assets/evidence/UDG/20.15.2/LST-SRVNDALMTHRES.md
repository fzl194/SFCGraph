#### [命令功能](#ZH-CN_CONCEPT_0182837875)

**适用NF：SGW-U、PGW-U、UPF**

该命令用于查询业务节点告警阈值和恢复告警阈值。当业务节点资源使用率超过配置的告警门限时产生告警。当使用率低于配置的恢复门限时恢复告警。

#### [注意事项](#ZH-CN_CONCEPT_0182837875)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0182837875)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0182837875)

无。

#### [使用实例](#ZH-CN_CONCEPT_0182837875)

假如运营商想要查询业务节点的告警阈值和恢复告警阈值，使用此命令：

```
LST SRVNDALMTHRES:;
```

```

RETCODE = 0  操作成功。

业务节点资源不足告警阈值信息
----------------------------
    业务节点告警阈值  =  85
业务节点恢复告警阈值  =  75
(结果个数 = 1)
---    END
```

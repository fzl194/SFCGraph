## [操作场景](#ZH-CN_OPI_0000001576107785)

当基于整形的业务流量控制想要得到公平丢包率时，请进行本节操作。

## [必备事项](#ZH-CN_OPI_0000001576107785)

前提条件

- UDG与网络实体之间的网络环境已经构建完成。
- 本特性License项依赖于其他特性，需开启依赖特性后才能开启本特性。详细仔细阅读[GWFD-020354 基于业务的Shaping](../GWFD-020354 基于业务的Shaping_83823060.md)。
- 请仔细阅读[GWFD-110313 基于智能Shaping的组级带宽控制](../GWFD-110313 基于智能Shaping的组级带宽控制_25866666.md)。
- 已加载特征库和解析库。

## [任务示例](#ZH-CN_OPI_0000001576107785)

任务描述

打开带宽管理控制器的用户级公平调动功能开关。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH:LICITEM="LKV3G5FSHP01",SWITCH=ENABLE;
```

//配置手动支持用户级公平调度功能。（手动智能二选一。）

```
ADD BWMCONTROLLER:BWMCNAME="bc1",CTRLTYPE=SHAPING,RATE=400,QUEDEPTH=25,SRVLEVELSPEC=10,USERFAIREN=ENABLE,WORKMODE=MANUAL;
```

//配置智能支持用户级公平调度功能。（手动智能二选一。）

```
ADD BWMCONTROLLER:BWMCNAME="bc1",CTRLTYPE=SHAPING,RATE=400,QUEDEPTH=40,SRVLEVELSPEC=3,USERFAIREN=ENABLE,WORKMODE=AUTO,MAXPKTLOSTRATE=5000,PKTLOSTRATEDTL=50,ASSUREMODE=EXPFIRST;
```

//配置带宽管理控制器业务等级策略。

```
ADD BCSRVLEVELPLY:BWMCNAME="bc1",SERVICELEVEL=2,SHAPRATE=100,MAXPKTLOSTRATE=10000, MAXINCRDELAY=100;
```

#### [命令功能](#ZH-CN_CONCEPT_0000203432260087)

**适用NF：PGW-U、UPF**

该命令用来显示所有已经配置的ICAP Server或者指定名字的ICAP Server的工作状态信息。

#### [注意事项](#ZH-CN_CONCEPT_0000203432260087)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000203432260087)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000203432260087)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| ICAPSERVERNAME | ICAP服务器名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定ICAP Server的名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。不支持空格。不区分大小写。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000203432260087)

查询所有已经配置的ICAP Server的工作状态信息：

```
DSP ICAPSVRSTATUS:;
```

```

RETCODE = 0  操作成功

ICAP服务器状态
--------------
        Pod 名称  =  icapc-pod-0
        源IP地址  =  192.168.0.1
        源端口号  =  16735
ICAP服务器IP地址  =  192.168.22.64
  ICAP服务器端口  =  1344
  ICAP服务器状态  =  NULL
(结果个数 = 1)

---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000203432260087)

| 输出项名称 | 输出项解释 |
| --- | --- |
| Pod 名称 | 用于指定IcapServer所在资源单元的名称。 |
| 源IP地址 | 用于指定ICAP客户端的IP地址。 |
| 源端口号 | 用于指定ICAP Server的目的端口。 |
| ICAP服务器状态 | ICAP Server的通信状态。 |

其余输出项请参见ADD ICAPSERVER的参数说明。

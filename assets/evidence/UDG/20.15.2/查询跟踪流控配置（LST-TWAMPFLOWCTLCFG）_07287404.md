## [命令功能](#ZH-CN_MMLREF_0000002107287404)

用于查询TWAMP功能的跟踪流控配置。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000002107287404)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000002107287404)

无

## [使用实例](#ZH-CN_MMLREF_0000002107287404)

查询TWAMP功能的跟踪流控配置参数。

```
%%LST TWAMPFLOWCTLCFG:;%%
RETCODE = 0  操作成功

结果如下
--------
    跟踪流控开关  =  打开
    流控恢复阈值  =  60
    流控触发阈值  =  70
允许跟踪创建阈值  =  60
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000002107287404)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 跟踪流控开关 | 该参数用于控制流控功能开关。 |
| 流控恢复阈值 | 该参数用于设置流控恢复阈值。 |
| 流控触发阈值 | 该参数用于设置流控触发阈值。 |
| 允许跟踪创建阈值 | 该参数用于设置允许跟踪创建阈值。 |

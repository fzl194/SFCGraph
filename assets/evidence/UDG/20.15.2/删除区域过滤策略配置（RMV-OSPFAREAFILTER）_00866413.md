# 删除区域过滤策略配置（RMV OSPFAREAFILTER）

#### [命令功能](#ZH-CN_CONCEPT_0000001600866413)

该命令用于取消对区域内的Type 3 LSA进行过滤配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001600866413)

- 该命令执行后立即生效。
- 此命令仅在ABR上配置。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600866413)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [使用实例](#ZH-CN_CONCEPT_0000001600866413)

在进程1的区域0.0.0.1下取消出方向的类型为ip-prefix的过滤策略：

```
RMV OSPFAREAFILTER:PROCID=1,AREAID="0.0.0.1",TOPONAME=base,FILTERTYPE=export,TYPE=ip_prefix,IPPREFIX="ip-prefix1";
```

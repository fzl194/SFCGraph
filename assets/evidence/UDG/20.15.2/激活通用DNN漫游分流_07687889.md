## [操作步骤](#ZH-CN_OPI_0000001607687889)

1. 在公网UPF上打开特性License开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09587387.md)
2. 在公网UPF上配置通用DNN漫游分流开关。
  [**SET RTSDNNPARA**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/RTSDNN业务控制/RTSDNN参数/设置RTSDNN参数（SET RTSDNNPARA）_11829348.md)
3. 在专网UPF上配置通用DNN漫游分流开关。
  [**SET RTSDNNPARA**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/RTSDNN业务控制/RTSDNN参数/设置RTSDNN参数（SET RTSDNNPARA）_11829348.md)
4. （可选）在专网UPF上配置针对指定协议的地址替换。
  [**SET RTSDNNPARA**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/RTSDNN业务控制/RTSDNN参数/设置RTSDNN参数（SET RTSDNNPARA）_11829348.md)

## [任务示例](#ZH-CN_OPI_0000001607687889)

任务描述

在公网UPF、专网UPF上开启通用DNN漫游分流功能，打开需要支持的license开关。

脚本

- 公网UPF
  //开启公网UPF的通用DNN漫游分流功能相关的License。
  ```
  SET LICENSESWITCH:LICITEM="LKV3G5STCA01",SWITCH=ENABLE;
  ```
  //开启公网UPF的通用DNN漫游分流功能开关和通过N4接口上报通用DNN漫游分流能力的开关，同时开启UE IP和园区服务IP冲突检测功能，设置UE IP冲突的策略为继续激活会话。
  ```
  SET RTSDNNPARA: PUBLICNETSW=ENABLE, UEIPCONFLICTCHK=ENABLE, UEIPREALLOC=ENABLE, UEIPREALLOCNUM=1000, UEIPCONFLICTPLY=BYPASS;
  ```
- 专网UPF
  //开启专网UPF的通用DNN漫游分流功能开关。
  ```
  SET RTSDNNPARA: PRIVATENETSW=ENABLE;
  ```
  //（可选）专网UPF开启针对FTP和RTSP协议的IPv4地址替换，针对FTP协议的IPv6地址替换。
  ```
  SET RTSDNNPARA: NATALGPROTV4=FTP-1&RTSP-1, NATALGPROTV6=FTP-1;
  ```

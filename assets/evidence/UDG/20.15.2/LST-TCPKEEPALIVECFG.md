## [命令功能](#ZH-CN_MMLREF_0000001127102476)

该命令用于查询TWAMP的Full模式TCP保活参数配置。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000001127102476)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001127102476)

无

## [使用实例](#ZH-CN_MMLREF_0000001127102476)

查询TWAMP的TCP保活参数的实例：

```
%%LST TCPKEEPALIVECFG: ;%%
RETCODE = 0  操作成功
 
结果如下
--------
保活时间  =  90
保活间隔  =  30
保活重试  =  9
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001127102476)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 保活时间 | 保活时间是在空闲条件下两次保活传输之间的持续时间。 |
| 保活间隔 | 保活间隔是两次连续保活重传之间的持续时间，如果没有收到对上一次保活传输的确认。 |
| 保活重试 | 保活重试是在声明远程端不可用之前要执行的重传次数。 |

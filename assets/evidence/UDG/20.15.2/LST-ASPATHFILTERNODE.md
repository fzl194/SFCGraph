#### [命令功能](#ZH-CN_CONCEPT_0000001600840729)

该命令用来查询一个AS路径过滤器表项。

#### [注意事项](#ZH-CN_CONCEPT_0000001600840729)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600840729)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600840729)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| NAME | AS路径过滤器名字或号 | 可选必选说明：可选参数<br>参数含义：该参数用于指定AS路径过滤器名字或号。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～51。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600840729)

查询AS路径过滤器：

```
LST ASPATHFILTERNODE:;
```

```

RETCODE = 0  操作成功

结果如下
-------------------------
AS路径过滤器名字或号  =  a
  AS路径过滤器节点号  =  10
            匹配模式  =  允许
          正则表达式  =  [1-9]
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600840729)

| 输出项名称 | 输出项解释 |
| --- | --- |
| AS路径过滤器名字或号 | 用于指定AS路径过滤器名字或号。 |

其余输出项请参见ADD ASPATHFILTERNODE的参数说明。

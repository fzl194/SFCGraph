# 删除全局计费属性（RMV GLBURRGROUP）

- [注意事项](#ZH-CN_CONCEPT_0186528078__1.3.2.1)

#### [命令功能](#ZH-CN_CONCEPT_0186528078)

**适用NF：PGW-U、UPF**

RMV GLBURRGROUP命令用于将全局计费属性恢复成默认记录。

#### [注意事项](#ZH-CN_CONCEPT_0186528078)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0186528078)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0186528078)

无。

#### [使用实例](#ZH-CN_CONCEPT_0186528078)

假如用户要删除全局计费属性：

```
RMV GLBURRGROUP:;
```

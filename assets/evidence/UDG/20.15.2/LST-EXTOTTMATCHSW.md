#### [命令功能](#ZH-CN_CONCEPT_0000201293531881)

**适用NF：PGW-U、UPF**

该命令用来查询OTT业务规则匹配功能是否使能。

#### [注意事项](#ZH-CN_CONCEPT_0000201293531881)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000201293531881)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000201293531881)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000201293531881)

查询外置OTT业务规则匹配功能是否使能信息：

```
LST EXTOTTMATCHSW:;
```

```

RETCODE = 0 操作成功.

外置OTT业务规则匹配功能开关信息
--------------------------------------------
外置OTT业务规则匹配功能开关IPv4 = ENABLE
外置OTT业务规则匹配功能开关IPv6 = ENABLE

(结果个数 = 1)
--- END
```

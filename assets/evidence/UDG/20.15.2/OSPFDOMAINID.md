#### [命令功能](#ZH-CN_CONCEPT_0000001600866173)

该命令用于查询OSPF域标识配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001600866173)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600866173)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600866173)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | 进程号 | 可选必选说明：可选参数<br>参数含义：OSPF进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |
| DOMAINID | 域标识 | 可选必选说明：可选参数<br>参数含义：域标识。<br>数据来源：本端规划<br>取值范围：IPv4地址类型。点分十进制格式。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600866173)

查询OSPF域标识配置：

```
LST OSPFDOMAINID:;
```

```

RETCODE = 0  操作成功。

结果如下
---------------
          进程号  =  1
          域标识  =  10.0.0.123
域标识符类型的值  =  0x64
  域标识符的类型  =  Type8005
    次级域标识符  =  FALSE
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600866173)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 进程号 | 该参数指定用户创建的OSPF进程号。 |
| 域标识 | 该参数指定OSPF域标识，用来标识和区分不同的域。 |
| 域标识符类型的值 | 该参数指定OSPF域标识符类型的值，域标识符用来标识和区分不同的域。 |
| 域标识符的类型 | 该参数指定OSPF域标识符的类型，默认值为Type0005。 |
| 次级域标识符 | 该参数指定OSPF次级域标识符。 |

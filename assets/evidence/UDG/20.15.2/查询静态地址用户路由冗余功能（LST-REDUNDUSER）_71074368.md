#### [命令功能](#ZH-CN_CONCEPT_0271074368)

**适用NF：PGW-U、UPF**

该命令用来显示当前设备静态地址用户路由冗余功能使能状态。

#### [注意事项](#ZH-CN_CONCEPT_0271074368)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0271074368)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0271074368)

无。

#### [使用实例](#ZH-CN_CONCEPT_0271074368)

显示静态地址用户路由冗余功能是否使能：

```
LST REDUNDUSER:;
```

```

RETCODE = 0 操作成功。
结果如下 
---------------------- 
开关 = 不使能 (结果个数 = 1) 
--- END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0271074368)

参见SET REDUNDUSER的参数说明。

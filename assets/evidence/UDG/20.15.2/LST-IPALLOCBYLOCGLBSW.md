#### [命令功能](#ZH-CN_CONCEPT_0186527067)

**适用NF：PGW-U、UPF**

该命令用于显示基于位置区分配地址的全局开关。

#### [注意事项](#ZH-CN_CONCEPT_0186527067)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0186527067)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0186527067)

无。

#### [使用实例](#ZH-CN_CONCEPT_0186527067)

查询基于位置区分配地址的全局开关：

```
LST IPALLOCBYLOCGLBSW:;
```

```

RETCODE = 0 操作成功。

基于位置区分配地址的全局开关
-----------------------------------------------
IPv4开关 = 使能
IPv6开关 = 使能
(结果个数 = 1)
--- END
```

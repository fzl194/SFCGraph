#### [命令功能](#ZH-CN_CONCEPT_0000201133110040)

**适用NF：SGW-U、PGW-U、UPF**

此命令用于显示当前纯转发用户配置，输出纯转发配置的IMSI、MSISDN。

#### [注意事项](#ZH-CN_CONCEPT_0000201133110040)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000201133110040)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000201133110040)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000201133110040)

显示当前的纯转发配置：

```
LST PUREFWDUSER:;
```

```

RETCODE = 0 操作成功。

结果如下
--------
用户类型 = IMSI
IMSI = 460011223344551
MSISDN = NULL
(结果个数 = 1)
--- END
```

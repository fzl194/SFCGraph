## [命令功能](#ZH-CN_MMLREF_0000002072648877)

该命令用于查询配置对象告警阈值。系统通过检测配置对象当前记录数与配置对象最大记录数的比值是否大于配置对象告警阈值决定是否上报“ALM-135602215 配置数量超阈值”。

> **说明**
> 无。

## [参数说明](#ZH-CN_MMLREF_0000002072648877)

| **参数标识** | **参数名称** | **参数说明** |
| --- | --- | --- |
| OBJECTNAME | 配置对象名称 | 可选必选说明：可选参数。<br>参数含义：用于指定配置对象名称。<br>取值范围：长度不超过16的字符串。<br>默认值：无。<br>配置原则：当不输入该参数时，表示查询所有配置对象的对象告警阈值。 |

## [使用实例](#ZH-CN_MMLREF_0000002072648877)

查询指定配置对象的配置对象告警阈值，执行以下命令：

```
%%LST CFGTHRESHOLD: OBJECTNAME="TcmcTypeInner";%%
RETCODE = 0  操作成功

操作结果如下

         是否存在默认告警阈值  =  否
(结果个数 = 1)
```

## [输出结果说明](#ZH-CN_MMLREF_0000002072648877)

该命令执行正常，会返回命令执行成功的提示信息。

| **输出项名称** | **输出项解释** |
| --- | --- |
| 配置对象名称 | 配置对象的名称。 |
| 配置对象告警阈值（%） | 当<br>“配置对象当前记录数”<br>与<br>“配置对象最大记录数”<br>的比值大于该输出项取值时，系统上报“ALM-135602215 配置数量超阈值”。 |
| 配置对象告警恢复阈值（%） | 当<br>“配置对象当前记录数”<br>与<br>“配置对象最大记录数”<br>的比值小于或等于该输出项取值时，“ALM-135602215 配置数量超阈值”告警恢复。 |
| 配置对象当前记录数 | 系统当前记录的配置对象的数量。 |
| 配置对象最大记录数 | 系统允许记录的配置对象的最大数量。 |
| 配置对象当前记录百分比（%） | 配置对象当前记录数与配置对象最大记录数的比值。 |
| 是否存在默认告警阈值 | 在增加配置对象告警阈值之前，系统中是否存在配置对象告警阈值。 |

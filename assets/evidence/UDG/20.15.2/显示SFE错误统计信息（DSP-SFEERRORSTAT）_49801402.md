#### [命令功能](#ZH-CN_CONCEPT_0000001549801402)

该命令用于显示SFE错误统计信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001549801402)

该命令仅适用于非NP卡场景和NP卡非加速模式场景。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549801402)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549801402)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RUNAME | RU名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定资源单元名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。区分大小写。<br>默认值：无<br>配置原则：<br>- 区分大小写，不支持空格。<br>- 使用DSP RU查看RU名称。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549801402)

显示指定资源单元的SFE的错误计数信息：

```
DSP SFEERRORSTAT:RUNAME="VNODE_VNRS_VNFC_IPU_0064";
```

```

RETCODE = 0  操作成功

结果如下
-------------------------
RU名称    =  VNODE_VNRS_VNFC_IPU_0064
错误类型  =  VFP_CAUSE_ERR_LPM_TBL_FIB_ADD
报文计数  =  1392
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001549801402)

| 输出项名称 | 输出项解释 |
| --- | --- |
| RU名称 | 用于指定资源单元名称。 |
| 错误类型 | 用于表示错误类型。 |
| 报文计数 | 用于表示错误报文计数。 |

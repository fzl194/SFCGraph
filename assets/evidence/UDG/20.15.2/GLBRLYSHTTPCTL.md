#### [命令功能](#ZH-CN_CONCEPT_0000207394871983)

**适用NF：UPF、PGW-U**

该命令用于设置媒体中继全局业务HTTP控制。

#### [注意事项](#ZH-CN_CONCEPT_0000207394871983)

- 该命令执行后立即生效。
- 该命令最大记录数为1。
- 该命令存在系统初始记录，参数的初始设置值如下表：

| 参数标识 | HTTPCONLDLETIME |
| --- | --- |
| 初始值 | 60 |

#### [操作用户权限](#ZH-CN_CONCEPT_0000207394871983)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000207394871983)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| HTTPCONLDLETIME | HTTP连接空闲时间（秒） | 可选必选说明：可选参数<br>参数含义：该参数用于指定Http连接空闲时间。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是为5～300，单位是秒。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000207394871983)

设置媒体中继全局业务HTTP控制：

```
SET GLBRLYSHTTPCTL: HTTPCONLDLETIME=60;
```

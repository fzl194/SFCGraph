#### [命令功能](#ZH-CN_CONCEPT_0000001600440813)

该命令用于查询隧道选择器节点。

#### [注意事项](#ZH-CN_CONCEPT_0000001600440813)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600440813)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600440813)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| TNLSELECTORNAME | 隧道选择器名字 | 可选必选说明：必选参数<br>参数含义：该参数用于指定隧道选择器名字。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～40。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600440813)

查询隧道选择器节点信息：

```
LST TUNNELSELECTORNODE:TNLSELECTORNAME="a";
```

```

RETCODE = 0  操作成功
结果如下
-------------------------
隧道选择器节点  =  6
隧道选择器名字  =  a
      匹配模式  =  允许
(结果个数 = 1)
---    END
```

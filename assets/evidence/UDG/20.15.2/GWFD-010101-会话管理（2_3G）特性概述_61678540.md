#### [适用NF](#ZH-CN_CONCEPT_0261678540)

PGW-U

#### [定义](#ZH-CN_CONCEPT_0261678540)

会话管理是 UDG 的基本功能， 是用户与外部PDN（ Packet Data Network）建立连接，进行数据业务的基础。华为 UDG 支持3GPP标准定义的GERAN/UTRAN无线网络接入，实现GPRS/UMTS网络用户的接入。

#### [客户价值](#ZH-CN_CONCEPT_0261678540)

| 受益方 | 受益描述 |
| --- | --- |
| 运营商 | 会话管理功能使运营商得以向授权的用户提供从移动终端到数据网络间的数据传输服务，从而丰富其网络业务。 |
| 用户 | 会话管理使UE可以通过GERAN/UTRAN网络接入外部PDN，享用Internet或企业网提供的数据业务。 |

#### [应用场景](#ZH-CN_CONCEPT_0261678540)

当用户需要通过GERAN/UTRAN网络接入外部PDN时，需要支持本功能。

#### [可获得性](#ZH-CN_CONCEPT_0261678540)

**涉及NF**

| 涉及NF | 支持版本 | 功能说明 |
| --- | --- | --- |
| MS | 无特殊要求 | 3GPP终端，支持UMTS功能。 |
| RAN | 无特殊要求 | 为MS提供无线资源及承载。 |
| HSS/HLR | 无特殊要求 | 保存用户签约信息，为MS提供QoS等业务相关数据。 |
| Gn/Gp-SGSN | UNC 20.0.0及后续版本 | 传统UMTS网络中的Pre-Rel-8 SGSN，为MS提供附着、去附着等移动性管理功能。<br>和PGW-C、PGW-U配合，为MS提供PDP上下文激活、去激活、更新等会话管理功能。 |
| PGW-C | UNC 20.0.0及后续版本 | 接收MS发送的数据，选路到相应的外部网络；或接收外部网络的数据，根据其目的地址选择EPC网络中的传输通道，传给相应的Gn/Gp-SGSN。 |
| PGW-U | UDG 20.3.2及后续版本 | - 为MS接入外部PDN提供数据转发功能。<br>- 通过Gn接口和RNC互通。和PGW-C配合，为MS提供PDP上下文激活修改等会话管理功能。 |

**会话管理流程**

2/3G网络会话管理通过 [表1](#ZH-CN_CONCEPT_0261678540__table4913185915411) 中的流程管理用户的缺省和专有承载，以下介绍会话管理相关流程的主要应用场景。

*表1 会话管理流程介绍*

| 流程 | 描述 |
| --- | --- |
| PDP上下文激活 | MS附着到GPRS/UMTS网络后，可以发起PDP上下文激活请求流程，该流程用于在MS和PDN之间建立一条虚拟的数据通道。PDP上下文激活包含MS发起一次PDP上下文激活流程和二次PDP上下文激活流程。激活过程中，网络侧根据用户请求及签约数据完成IP地址分配、QoS参数协商、以及MS接入请求ISP或企业网的权限检查等。PDP上下文激活成功后，用户即可通过GPRS/UMTS网络与PDN进行数据通信。 |
| PDP上下文去激活 | 该流程用于释放PDP上下文，可以由MS、<br>UDG<br>发起。去激活过程中，MS的IP地址被释放，<br>UDG<br>中的PDP上下文被删除。PDP上下文去激活成功后，用户即不能与PDN进行数据通信。 |
| PDP上下文更新 | 该流程用于修改已激活PDP上下文的参数，如QoS和TFT，可以由MS、RNC、PCRF、<br>UDG<br>发起。 |

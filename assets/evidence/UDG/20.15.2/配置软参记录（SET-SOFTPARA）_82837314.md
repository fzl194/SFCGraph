#### [命令功能](#ZH-CN_CONCEPT_0182837314)

**适用NF：SGW-U、PGW-U、UPF**

该命令用于设置软件参数。

#### [注意事项](#ZH-CN_CONCEPT_0182837314)

- 该命令执行后立即生效。
- 该命令最大记录数为5250。
- 该命令存在系统初始记录，参数的初始设置值如下表：

#### [使用实例](#ZH-CN_CONCEPT_0182837314)

设置软参，参数类型是字节类型，参数索引为181，参数取值为3：

```
SET SOFTPARA: DT=BYTE, BYTENUM=181, BYTEVALUE=3;
```

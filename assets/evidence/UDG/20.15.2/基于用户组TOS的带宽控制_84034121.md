## [操作场景](#ZH-CN_OPI_0184034121)

对于一个用户组基于TOS作为区分业务的条件，即 UDG 可以不进行业务的识别和解析，只是根据报文的DSCP就可以确定是否对其进行流量监管。

> **说明**
> 适用于PGW-U、UPF。

## [操作步骤](#ZH-CN_OPI_0184034121)

1. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09587387.md)
2. 配置BWM业务。
    a. 用TOS来定义BWM业务。
      [**ADD BWMSERVICE**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理业务/增加带宽管理业务（ADD BWMSERVICE）_82837473.md)
    b. 配置两个BWM控制器，设置CAR参数。
      [**ADD BWMCONTROLLER**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理控制器/增加带宽管理控制器（ADD BWMCONTROLLER）_82837460.md)
    c. 配置BWM控制器中的用户组级业务带宽和连接数等配置数据是基于业务处理单元生效还是整机生效。
      [**SET BANDWIDTHMNG**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理参数/设置带宽管理参数（SET BANDWIDTHMNG）_82837465.md)
3. 配置带宽管理用户组
    a. 定义带宽管理用户组，配置优先级和业务类型。
      [**ADD BWMUSERGROUP**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理用户组/增加带宽管理用户组（ADD BWMUSERGROUP）_82837468.md)
    b. 配置用户组级带宽控制规则。
      [**ADD BWMRULE**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理规则/增加带宽管理规则（ADD BWMRULE）_82837478.md)
    c. 将 APN 下的用户包含到该用户组。
      [**ADD APNBINDBWMUSRG**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/带宽控制/带宽管理用户组APN绑定/增加带宽管理用户组APN绑定（ADD APNBINDBWMUSRG）_82837487.md)

## [任务示例](#ZH-CN_OPI_0184034121)

任务描述

本实例中，操作员在 UDG 上进行数据配置实现以下要求：

UDG 对一个用户组下的所有用户的指定IP TOS等级的上行、下行用户面数据流量进行流量监管。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH:LICITEM="LKV3G5TCSA01",SWITCH=ENABLE;
```

//配置BWM业务。

```
ADD BWMSERVICE:BWMSERVICENAME="bs_test",BWMSERVICETYPE=TOS,TOSCFGTYPE=CLASS,TOSCLASSTYPE=BE;
```

```
ADD BWMCONTROLLER:BWMCNAME="bc-up",CTRLTYPE=CAR,CIR=300,CBS=10000,PIR=500,PBS=12000,GREEN=PASS,YELLOW=PASS,RED=DISCARD;
```

```
ADD BWMCONTROLLER:BWMCNAME="bc-down",CTRLTYPE=CAR,CIR=300,CBS=10000,PIR=500,PBS=12000,GREEN=PASS,YELLOW=PASS,RED=DISCARD;
```

```
SET BANDWIDTHMNG:RANGE=SYSTEM;
```

//配置带宽管理用户组。

```
ADD BWMUSERGROUP:USERGROUPTYPE=SPECIFIC_GROUP,USERGROUPNAME="ug_test",USERGROUPPRI=1,USERGLEVSRVTYPE=TOS;
```

```
ADD BWMRULE:USERGROUPTYPE=SPECIFIC_GROUP,USERGROUPNAME="ug_test",BWMRULETYPE=GROUP_SPECIFIC,BWMRULENAME="br_test",BWMSERVICENAME="bs_test",UPBWMCTRLNAME1="bc-up",DNBWMCTRLNAME1="bc-down",BWMRULEPRI=1;
```

```
ADD APNBINDBWMUSRG:USERGROUPNAME="ug_test",APN="apn-test";
```

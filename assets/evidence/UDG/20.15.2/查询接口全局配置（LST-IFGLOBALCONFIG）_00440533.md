#### [命令功能](#ZH-CN_CONCEPT_0000001600440533)

该命令用于显示接口全局属性。

#### [注意事项](#ZH-CN_CONCEPT_0000001600440533)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600440533)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600440533)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001600440533)

查询全局的流量统计间隔和接口首次Down告警上报的使能状态：

```
LST IFGLOBALCONFIG:;
```

```

RETCODE = 0  操作成功

结果如下
-------------------------
   全局流量统计间隔（s） = 5
使能接口首次Down告警上报 = FALSE
(结果个数 = 1)
---   END
```

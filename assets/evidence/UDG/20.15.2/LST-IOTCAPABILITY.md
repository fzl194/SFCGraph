#### [命令功能](#ZH-CN_CONCEPT_0270824409)

**适用NF：UPF**

该命令用来查看UPF的物联网接入能力上报功能选择指示。

#### [注意事项](#ZH-CN_CONCEPT_0270824409)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0270824409)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0270824409)

无。

#### [使用实例](#ZH-CN_CONCEPT_0270824409)

查询UPF的物联网接入能力上报功能开关配置：

```
LST IOTCAPABILITY:;
```

```

RETCODE = 0 操作成功。

物联网能力
------------------------------------
NB-IoT接入能力上报 = ENABLE
(结果个数 = 1)
--- END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0270824409)

参见SET IOTCAPABILITY的参数说明。

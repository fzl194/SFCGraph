#### [命令功能](#ZH-CN_CONCEPT_0000201531139797)

**适用NF：UPF**

该命令用于查询TCP MSS值。

#### [注意事项](#ZH-CN_CONCEPT_0000201531139797)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000201531139797)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000201531139797)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000201531139797)

查询TCP MSS值：

```
LST TOMSSCFG:;
```

```

RETCODE = 0 操作成功。

TCP MSS配置
-----------
IPv4 TCP报文长度（字节） = 496
IPv6 TCP报文长度（字节） = 1500
(结果个数 = 1)

--- END
```

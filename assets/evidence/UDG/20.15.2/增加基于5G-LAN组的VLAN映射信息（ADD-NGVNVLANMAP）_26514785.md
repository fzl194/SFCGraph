#### [命令功能](#ZH-CN_CONCEPT_0000206026514785)

**适用NF：UPF**

该命令用于配置基于5GLAN组的VLAN映射信息。

#### [注意事项](#ZH-CN_CONCEPT_0000206026514785)

- 该命令执行后立即生效。
- 该命令最大记录数为4096。
- 5G LAN组会话实例PDN连接类型必须为以太连接类型。

#### [操作用户权限](#ZH-CN_CONCEPT_0000206026514785)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000206026514785)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| VNINSTANCE | 5G LAN组名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定5G LAN会话实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为18～37。VNINSTANCE以连字号分为4段，形式为GroupServiceID-MCC-MNC-LocalGroupID。其中，GroupServiceID长度为8，只能输入数字或者范围为A到F或a-f的字母；MCC长度为3，只能输入数字；MNC长度为2~3，只能输入数字；LocalGroupID长度为2~20的偶数，只能输入数字或者范围为A到F或a-f的字母。例如，A0000001-460-003-01，A0000001-460-003-A000000001。不区分大小写。<br>默认值：无<br>配置原则：无 |
| ORIGINALVLANID | 原始VLAN ID | 可选必选说明：必选参数<br>参数含义：该参数用于指定5G LAN接收的上行报文中的原始VLAN ID。<br>数据来源：对端协商<br>取值范围：整数类型，取值范围为1～4094。<br>默认值：无<br>配置原则：无 |
| MAPPEDVLANID | 映射的VLAN ID | 可选必选说明：必选参数<br>参数含义：该参数用于指定5G LAN接收的上行报文中的原始VLAN ID映射后的VLAN ID。<br>数据来源：对端协商<br>取值范围：整数类型，取值范围为1～4094。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000206026514785)

增加VLAN映射信息：

```
ADD NGVNVLANMAP: VNINSTANCE="a0000001-460-003-01", ORIGINALVLANID=10, MAPPEDVLANID=11;
```

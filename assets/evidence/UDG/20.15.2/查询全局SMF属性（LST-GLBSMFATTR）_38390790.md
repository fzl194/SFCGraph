#### [命令功能](#ZH-CN_CONCEPT_0000204338390790)

**适用NF：SGW-U、PGW-U、UPF**

该命令用于查询UPF对接SMF是否为同一供应商提供的设备。

#### [注意事项](#ZH-CN_CONCEPT_0000204338390790)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000204338390790)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000204338390790)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000204338390790)

UPF对接SMF是否为同一供应商提供的设备：

```
LST GLBSMFATTR:;
```

```

RETCODE = 0  操作成功。

SMF属性
-------------
SMF是否为同一供应商  =  否
UPF去活会话  =  使能
N4计费上报消息冲突检查时长 = 0
(结果个数 = 1)

---    END
```

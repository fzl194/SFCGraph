#### [命令功能](#ZH-CN_CONCEPT_0000201287728428)

**适用NF：UPF**

该参数用于查询当前配置的IP防欺诈功能开关状态。

#### [注意事项](#ZH-CN_CONCEPT_0000201287728428)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000201287728428)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000201287728428)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000201287728428)

使用LST ANTISPOOFINGTHD命令查询当前配置的IP防欺诈功能开关状态：

```
LST IPSPOOFFUNC:;
```

```

RETCODE = 0 操作成功

IPSpoof攻击防护
-------------
上行IP防攻击开关 = 已使能
下行IP防攻击开关 = 已使能
(结果个数 = 1)

--- 结束
```

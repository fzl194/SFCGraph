#### [命令功能](#ZH-CN_CONCEPT_0259036254)

该命令用来查询不同资源类型的磁盘分区过载阈值。

当资源的磁盘分区使用率超过阈值时，会触发磁盘分区使用率超限告警。

#### [注意事项](#ZH-CN_CONCEPT_0259036254)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0259036254)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0259036254)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RESOURCETYPE | 资源类型 | 可选必选说明：可选参数<br>参数含义：该参数用于指定资源类型。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，区分大小写。<br>默认值：无<br>配置原则：请使用<br>[**LST RESINSTANCETYPE**](../../../../单体服务公共功能管理/系统管理/资源管理/RU管理/查询资源实例类型（LST RESINSTANCETYPE）_59103378.md)<br>命令查询存在的资源类型。 |

| 输出项名称 | 输出项解释 |
| --- | --- |
| 资源类型 | 该参数用于表示资源的类型。 |
| 磁盘分区名称 | 该参数用于表示磁盘分区的名称，系统中支持的磁盘分区以网元部署的实际情况为准。 |
| 分区使用率过载次要级别告警恢复阈值（%） | 该参数用于表示分区使用率过载次要级别告警恢复阈值。在触发分区使用率过载次要级别告警后，只有当分区使用率低于此阈值一段时间后，次要级别告警才会消除。 |
| 分区使用率过载次要级别告警上报阈值（%） | 该参数用于表示分区使用率过载次要级别告警上报阈值。当分区使用率长期高于此阈值时，将会触发分区使用率过载次要告警。 |
| 分区使用率过载重要级别告警恢复阈值（%） | 该参数用于表示分区使用率过载重要级别告警恢复阈值。在触发分区使用率过载重要级别告警后，只有当分区使用率低于此阈值一段时间后，重要级别告警才会消除。 |
| 分区使用率过载重要级别告警上报阈值（%） | 该参数用于表示分区使用率过载重要级别告警上报阈值。当分区使用率长期高于此阈值时，将会触发分区使用率过载重要告警。 |
| 分区使用率过载紧急级别告警恢复阈值（%） | 该参数用于表示分区使用率过载紧急级别告警恢复阈值。在触发分区使用率过载紧急级别告警后，只有当分区使用率低于此阈值一段时间后，紧急级别告警才会消除。 |
| 分区使用率过载紧急级别告警上报阈值（%） | 该参数用于表示分区使用率过载紧急级别告警上报阈值。当分区使用率长期高于此阈值时，将会触发分区使用率过载紧急告警。 |
| 分区使用率过载告警的采样窗口 | 该参数用于表示分区使用率过载告警的采样窗口。将会使用该值设定的采样窗口来统计分区的使用率。 |
| 分区空间不足告警上报阈值（MB） | 该参数用于表示分区空间不足告警上报阈值。当分区空间长期低于此阈值时，将会触发分区空间不足告警。 |
| 分区空间不足告警恢复阈值（MB） | 该参数用于表示分区空间不足告警恢复阈值。在触发分区空间不足告警后，只有当分区空间高于此阈值一段时间后，空间不足告警才会消除。 |

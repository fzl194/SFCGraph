## [命令功能](#ZH-CN_MMLREF_0000001173142135)

该命令用于查询本地逻辑接口。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000001173142135)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001173142135)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| NAME | 逻辑接口名称 | 可选必选说明：可选参数<br>参数含义：该参数用于配置逻辑接口名称。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是1~63。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001173142135)

查询逻辑接口名称为：n3if1/1/0的实例：

```
%%LST TWAMPLOGICINF: NAME="n3if1/1/0";%%
RETCODE = 0  操作成功

结果如下
--------
    逻辑接口名称  =  n3if1/1/0
      地址族类型  =  IPV4
逻辑接口IPv4地址  =  10.0.0.0
    逻辑接口掩码  =  255.255.255.255
     VPN实例名称  =  ck
      SHAREDTYPE  =  TRUE
(结果个数 = 1)

---    END
```

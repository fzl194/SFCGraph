## [命令功能](#ZH-CN_MMLREF_0000001610285105)

该命令用于查询证书更新流程参数。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000001610285105)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001610285105)

无

## [使用实例](#ZH-CN_MMLREF_0000001610285105)

查询证书更新流程参数，可以用如下命令：

```
%%LST CERTUPDATPARA:;%%
RETCODE = 0  操作成功

结果如下
------------------------
               单进程每批次重建链路间隔(s)  =  4
                    单进程每批次重建链路数  =  10
                  证书更新时是否重建旧链路  =  TRUE
单证书场景单进程客户端链路允许不更新的数量  =  0
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001610285105)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 单进程每批次重建链路间隔(s) | 该参数用于控制证书开关打开的场景下激活设备证书，每个进程每批次重建链路的间隔时间。 |
| 单进程每批次重建链路数 | 该参数用于控制证书开关打开的场景下激活设备证书，每个进程每批次重建链路数量。 |
| 证书更新时是否重建旧链路 | 该参数用于控制证书开关打开的场景下激活设备证书，是否重新建链。 |
| 单证书场景单进程客户端链路允许不更新的数量 | 该参数用于控制证书开关打开的场景下激活设备证书，每个证书场景在每个进程上，客户端链路更新时允许更新失败的链路数，小于该值时，更新流程继续，超过该值后，更新流程会失败。 |

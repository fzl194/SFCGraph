## [命令功能](#ZH-CN_MMLREF_0000001306289218)

该命令用于查询HTTP消息日志的参数。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000001306289218)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001306289218)

无

## [使用实例](#ZH-CN_MMLREF_0000001306289218)

如果想查询HTTP消息日志参数，可以用如下命令：

```
%%LST HTTPMSGLOGPARA:;%%
RETCODE = 0  操作成功

结果如下
--------
  TCP消息日志开关  =  关闭
停止记录的CPU阈值  =  75
恢复记录的CPU阈值  =  65
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001306289218)

| 输出项名称 | 输出项解释 |
| --- | --- |
| TCP消息日志开关 | 该参数用于配置是否开启TCP消息日志功能。开启该功能后，会记录TCP的SYN、FIN、RST消息及其后续部分消息的报文头内容。 |
| 停止记录的CPU阈值 | 该参数用于配置停止记录消息日志的CPU占用率阈值。当CPU占用率大于或等于该阈值时停止记录消息日志。 |
| 恢复记录的CPU阈值 | 该参数用于配置恢复记录消息日志的CPU占用率阈值。当CPU占用率小于或等于该阈值时恢复记录消息日志。 |

#### [命令功能](#ZH-CN_TOPIC_0243641445)

该命令用于显示NP Fabric OAM链路状态信息。

#### [注意事项](#ZH-CN_TOPIC_0243641445)

- 该命令仅适用于NP卡加速模式场景。

#### [操作用户权限](#ZH-CN_TOPIC_0243641445)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_TOPIC_0243641445)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RESNAME | 源资源名称 | 可选必选说明：必选参数。<br>参数含义：该参数用于指定源资源名称。<br>数据来源：本端规划。<br>取值范围：字符串类型，输入长度范围为1～128。不支持空格，区分大小写。<br>配置原则：FRAME-x-SLOT-y，区分大小写。其中x为框号，y为槽位号，可通过<br>[DSP NPNODE](../../../FEMF/NP状态查询（DSP NPNODE）_18818228.md)<br>查询。例：FRAME-0-SLOT-8表示0框8板。<br>默认值：无。 |

#### [使用实例](#ZH-CN_TOPIC_0243641445)

显示资源名称为FRAME-0-SLOT-8到所有远端资源的NP Fabric OAM链路状态信息：

```
DSP NPFABRICOAMINFO:RESNAME="FRAME-0-SLOT-8";
```

```
RETCODE = 0  操作成功

结果如下
--------
源资源名称      远端资源名称         源节点TB  远端节点TB  链路状态  

FRAME-0-SLOT-8  FRAME-0-SLOT-7-NP-0  1116      1115        up        
FRAME-0-SLOT-8  FRAME-0-SLOT-8-NP-0  1112      1116        up        
(结果个数 = 2)

---    END
```

#### [输出结果说明](#ZH-CN_TOPIC_0243641445)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 源资源名称 | 用于指定源资源名称。 |
| 远端资源名称 | 用于指定远端资源名称。 |
| 源节点TB | 用于指定源节点TB。 |
| 远端节点TB | 用于指定远端的节点TB。 |
| 链路状态 | 用于表示链路当前状态。<br>链路状态枚举类型：<br>- up：链路已链接。<br>- down：链路已中断。<br>- invalid: 链路状态无效。 |

## [操作场景](#ZH-CN_OPI_0169675201)

当在线用户已用带宽过大，超出协商的最大带宽时，设置带宽接入的控制方式。

> **说明**
> 适用于SGW-U、PGW-U、UPF。

## [任务示例](#ZH-CN_OPI_0169675201)

任务描述

本实例中，操作员在 UDG 上进行数据配置实现以下要求：

当在线用户使用带宽超过协商的带宽资源时，上下行采用CAR方式对带宽进行控制，即对超出的带宽数据包进行丢弃。

脚本

//增加APN示例。

```
ADD APN:APN="apn";
```

//配置上下行带宽控制方式。

```
SET APNQOSATTR:APN="apn",CARSHAPESWUL=ENABLE,CARSHAPEUL=CAR,CARSHAPESWDL=ENABLE,CARSHAPEDL=CAR;
```

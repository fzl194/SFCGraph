## [操作场景](#ZH-CN_OPI_0239436118)

在GPRS/UMTS网络无线资源有限的情况下，可以激活IM类业务无线资源管控功能进行无线资源的优化。

> **说明**
> 适用于PGW-U。

## [必备事项](#ZH-CN_OPI_0239436118)

前提条件

- 请仔细阅读[GWFD-020359 IM类业务无线资源管控](../GWFD-020359 IM类业务无线资源管控_39436120.md)。
- 完成[激活PCC基本功能](../GWFD-020351 PCC基本功能/激活PCC基本功能_74096528.md)。
- 完成加载License（如果有需求，请联系华为技术支持工程师处理）。

## [操作步骤](#ZH-CN_OPI_0239436118)

1. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09587387.md)
2. 配置流过滤器。
  [**ADD FLOWFILTER**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/流过滤器/增加流过滤器（ADD FLOWFILTER）_82837361.md)
3. 配置三四层过滤条件。
    a. 配置三四层过滤条件。
      [**ADD FILTER**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/过滤器/增加过滤器（ADD FILTER）_82837348.md)
    b. 配置三四层过滤条件与流过滤器的绑定关系。
      [**ADD FLTBINDFLOWF**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/流过滤器的过滤器绑定/增加流过滤器的过滤器绑定关系（ADD FLTBINDFLOWF）_82837366.md)
    c. 将新配置的Filter置为生效。
      [**SET REFRESHSRV**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/业务刷新/业务刷新（SET REFRESHSRV）_82837355.md)
4. （可选）配置七层协议识别数据。
  特征库用于识别IM类业务，如已加载特征库和解析库，请跳过本步骤。
  [**LOD SIGNATUREDB**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/协议识别/特征库管理/识别特征库/加载协议特征库文件（LOD SIGNATUREDB）_82837716.md)
  [**LOD PARSERDB**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/协议识别/特征库管理/解析特征库/加载协议解析数据库（LOD PARSERDB）_82837719.md)
5. 配置七层过滤条件。
    a. 配置流过滤器与协议绑定。
      [**ADD PROTBINDFLOWF**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/流过滤器的协议绑定/增加流过滤器协议绑定关系（ADD PROTBINDFLOWF）_82837370.md)
    b. 将新配置的七层过滤条件置为生效。
      [**SET REFRESHSRV**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/业务刷新/业务刷新（SET REFRESHSRV）_82837355.md)
      > **说明**
      > 系统自动在60s后将配置的L7Filter置为生效。如果需要将新配置的L7Filter立即生效，可执行本步骤。
6. 配置IM类业务规则。
  **[ADD RULE](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务匹配公共配置/业务规则管理/规则/增加规则（ADD RULE）_82837266.md)**
  > **说明**
  > 如果运营商希望基于流过滤器组设置过滤条件时，配置 “流过滤器或流过滤器组选择(FILTERINGMODE)” 为 “FLOWFILTERGRP” ，并且使用 [**ADD FLOWFILTERGRP**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/流过滤器组/增加流过滤器组（ADD FLOWFILTERGRP）_82837384.md) 命令配置流过滤器组。
7. 配置规则与用户模板的绑定关系 。
    a. 创建UserProfile。
      [**ADD USERPROFILE**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务匹配公共配置/业务规则管理/用户模板/增加用户模板（ADD USERPROFILE）_82837279.md)
    b. 将已配置的rule绑定到UserProfile。
      [**ADD RULEBINDING**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务匹配公共配置/业务规则管理/规则绑定/增加用户模板和规则的绑定关系（ADD RULEBINDING）_82837272.md)
    c. **可选：**禁用Alias Marking功能。
      [**MOD USERPROFILE**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务匹配公共配置/业务规则管理/用户模板/修改用户模板（MOD USERPROFILE）_82837280.md)
      > **说明**
      > DSCP方式下，需要禁用Alias Marking功能。因为该功能使能时，报文ToS域的值不会被改写。

## [任务示例](#ZH-CN_OPI_0239436118)

任务描述

在 UDG 上配置IM类业务的无线资源管控功能。

如下脚本以IM类业务QQ为例，通过DSCP方式实现无线资源管控功能。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV3G5ITSR01",SWITCH=ENABLE;
```

//配置三四层过滤条件。

```
ADD FILTER: FILTERNAME="filter_IM_any", L34PROTTYPE=STRING, L34PROTOCOL=ANY;
ADD FLOWFILTER:FLOWFILTERNAME="flowfilter_IM_01";
ADD FLTBINDFLOWF:FLOWFILTERNAME="flowfilter_IM_01",FILTERNAME="filter_IM_any";
SET REFRESHSRV:REFRESHTYPE=ALL;
```

//配置七层过滤条件。

//通过报文特征库识别协议。

```
LOD SIGNATUREDB:LOADMODE=LATEST;
LOD PARSERDB:LOADMODE=LATEST;
```

//配置流过滤器与协议绑定（此处根据qq_im识别IM类业务“QQ”，其他IM类业务时，PROTOCOLNAME需要根据实际业务规划）。

```
ADD PROTBINDFLOWF:FLOWFILTERNAME="flowfilter_IM_01",PROTOCOLNAME="qq_im";
```

//配置IM类业务使用的规则（将QQ业务映射的DSCP值为AF12，其他IM类业务时，REMARKCLASS需要根据实际业务修改）。

```
ADD RULE:RULENAME="rule_IM_01",POLICYTYPE=REMARK_FPI,
FILTERINGMODE=FLOWFILTER,
FLOWFILTERNAME="flowfilter_IM_01", 
REMARKFPISEL=REMARK,REMARKCFGTYPE=CLASS,REMARKCLASS=AF12,PRIORITY=20;
```

//配置规则与用户模板的绑定关系。

```
ADD USERPROFILE:USERPROFILENAME="up_test";
ADD RULEBINDING:USERPROFILENAME="up_test",RULENAME="rule_IM_01";
```

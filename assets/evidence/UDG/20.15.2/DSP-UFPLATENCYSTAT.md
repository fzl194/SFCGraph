## [命令功能](#ZH-CN_MMLREF_0000001788226712)

该命令用于显示UFP逐包转发时延度量开关。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000001788226712)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001788226712)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| CELLID | Cell ID | 可选必选说明：可选参数<br>参数含义：该参数表示PAE调试消息发送的CELLID，可以通过使用命令<br>[**DSP PAENODE**](../../../服务通信管理/策略查询/显示PAE节点信息（DSP PAENODE）_92520008.md)<br>获取。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~127。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001788226712)

动态查询UFP逐包转发时延度量功能：

```
+++    UEG/*MEID:0 MENAME:UEG_1822*/        2024-03-20 10:55:47
O&M    #69
%%DSP UFPLATENCYSTAT:;%%
RETCODE = 0  操作成功
```

## [输出结果说明](#ZH-CN_MMLREF_0000001788226712)

| 输出项名称 | 输出项解释 |
| --- | --- |
| Cell ID | 该参数表示PAE调试消息发送的CELLID，可以通过使用命令<br>[**DSP PAENODE**](../../../服务通信管理/策略查询/显示PAE节点信息（DSP PAENODE）_92520008.md)<br>获取。 |
| 开关 | 该参数用于显示时延度量开关的实际打开或关闭状态。<br>取值说明：<br>- “ENABLE（开）”：时延度量开关打开<br>- “DISABLE（关）”：时延度量开关关闭 |
| 虚拟局域网ID | 该参数用于显示需要匹配的报文的多个虚拟局域网ID。 |
| 剩余时长 | 该参数用于显示时延度量开关的剩余时长。 |

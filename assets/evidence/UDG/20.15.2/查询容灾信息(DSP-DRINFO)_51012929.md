#### [命令功能](#ZH-CN_CONCEPT_0251012929)

该命令用于查询容灾信息。

#### [注意事项](#ZH-CN_CONCEPT_0251012929)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0251012929)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0251012929)

| **参数标识** | **参数名称** | **参数说明** |
| --- | --- | --- |
| PEERDRINSTID | 对端容灾实例ID | 可选必选说明：可选参数。<br>参数含义：对端容灾实例标识，可以通过<br>**[LST DRINST](查询容灾实例(LST DRINST)_51012924.md)**<br>命令查询本端容灾实例ID，再通过<br>**[LST DRCOMM](查询容灾实例地址(LST DRCOMM)_51012928.md)**<br>命令查询到本端和对端容灾实例ID，从而得知对端容灾实例ID。<br>数据来源：全网规划。<br>取值范围：整型，0~63。<br>默认值：无。 |

#### [使用实例](#ZH-CN_CONCEPT_0251012929)

查询 “对端容灾实例ID” 为 “1” 的容灾信息：

```
DSP DRINFO: PEERDRINSTID=1;
%%DSP DRINFO: PEERDRINSTID=1;%%
RETCODE = 0  操作成功
操作结果如下：
--------------
本端容灾实例ID  =  0
对端容灾实例ID  =  1
  数据备份状态  =  数据实时备份状态
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0251012929)

| **输出项名称** | **输出项解释** |
| --- | --- |
| 本端容灾实例ID | 本端容灾实例标识。 |
| 对端容灾实例ID | 对端容灾实例标识。 |
| 数据备份状态 | 用于返回当前的数据备份状态。<br>取值范围：<br>- 容灾准备状态：容灾配置已下发，但尚未激活，激活可以通过MML命令、软参、License项等。<br>- 数据批量备份状态：初次完成容灾激活后或容灾故障恢复后，系统将自动进行容灾数据同步（即批量备份），同步完成后即进入数据实时备份状态。<br>- 数据实时备份状态：批量备份完成，进入实时备份状态，此时处于稳态。<br>- 容灾通道故障状态：容灾通道故障，备份失效。<br>- 数据备份中断状态：备份停止，可能原因为内存过载，或软参介入中断等原因。<br>- 容灾配置表仍然存在：下发删除容灾实例命令后，容灾配置删除中或删除失败。<br>- 容灾配置表已删除：下发删除容灾实例命令后，容灾配置信息已同步删除完毕。 |

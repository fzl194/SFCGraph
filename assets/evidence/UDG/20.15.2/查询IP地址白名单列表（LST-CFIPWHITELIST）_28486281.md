#### [命令功能](#ZH-CN_CONCEPT_0000203928486281)

**适用NF：PGW-U、UPF**

该命令用于显示配置IP地址。

#### [注意事项](#ZH-CN_CONCEPT_0000203928486281)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000203928486281)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000203928486281)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IPTYPE | IP地址类型 | 可选必选说明：可选参数<br>参数含义：该参数用于设置Ip地址类型。<br>数据来源：本端规划<br>取值范围：<br>- IPV4：IPv4地址类型。<br>- IPV6：IPv6地址类型。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000203928486281)

查询IP地址白名单列表：

```
LST CFIPWHITELIST: IPTYPE=IPV4;
```

```

RETCODE = 0  操作成功。

```

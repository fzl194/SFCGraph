#### [命令功能](#ZH-CN_CONCEPT_0000001549961402)

该命令用来查询当前分片方式。

#### [注意事项](#ZH-CN_CONCEPT_0000001549961402)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549961402)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549961402)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001549961402)

查询当前接口分片方式：

```
LST IFMTUFRAGMODE:;
```

```

RETCODE = 0 操作成功                                                
                                                                                

结果如下                                                        
------------------------                                                        
分片方式  =  标签加IP分片方式                                                                
(结果个数 = 1)                                                                                                             
---    END
```

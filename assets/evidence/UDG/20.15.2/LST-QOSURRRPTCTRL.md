#### [命令功能](#ZH-CN_CONCEPT_0000201414794665)

**适用NF：PGW-U、UPF**

该命令用于查询QoS URR上报的相关参数。

#### [注意事项](#ZH-CN_CONCEPT_0000201414794665)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000201414794665)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000201414794665)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000201414794665)

查询QoS URR上报的相关参数：

```
LST QOSURRRPTCTRL:;
```

```

RETCODE = 0 操作成功。

设置QoS URR上报的的相关参数。
------------------------
   QoS类型URR Stop上报迟滞时间（秒）  =  20
Rule删除时触发QoS URR Stop上报的开关  = 使能
(结果个数 = 1)

---    END
```

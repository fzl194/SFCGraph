# 调测用户面地址分配

- [操作场景](#ZH-CN_OPI_0172547233__1.3.1)
- [必备事项](#ZH-CN_OPI_0172547233__1.3.2)
- [操作步骤](#ZH-CN_OPI_0172547233__1.3.3)

## [操作场景](#ZH-CN_OPI_0172547233)

用户激活采用用户面分配UE IP地址时，需要调测该功能，确保用户可以正常接入。

> **说明**
> 适用于PGW-U、UPF。

## [必备事项](#ZH-CN_OPI_0172547233)

前提条件

- 请仔细阅读[GWFD-010105 用户面地址分配](../GWFD-010105 用户面地址分配_68664420.md)。
- 完成[激活地址分配方式](../GWFD-010104 地址分配方式/激活地址分配方式_71653959.md)。

数据

| 类别 | 参数名称 | 取值样例 | 获取方法 | 说明 |
| --- | --- | --- | --- | --- |
| 用户信息查询 | 用户IMSI（IMSI） | 460000123456789 | 测试终端自带 | - |
| 测试终端使用的APN | APN/DNN实例名（APN） | apn-test1 | 已配置数据中获取 | - |

工具

测试终端

## [操作步骤](#ZH-CN_OPI_0172547233)

1. 测试终端使用“apn-test”APN发起接入网络请求。
2. 执行 [**DSP SESSIONINFO**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/会话信息管理/会话信息查询/显示用户上下文（DSP SESSIONINFO）_86526407.md) 命令，查看 测试终端的IP地址是否在 UDG 本地规划的地址池内 。
  ```
  DSP SESSIONINFO:QUERYTYPE=IMSI,IMSI="460000123456789";
  ```
  ```
  ......

  -------------------------------
                                   IMSI  =  460000123456789
                                       ......              
                               PDP Type  =  IPv4 
                                       ......    
                       IPv4 PDP address  =  10.10.10.1         
                                       ......    
                               APN name  =  apn-test1
                                       ......       
  (Number of results = 1)
  ---    END
  ```
    - 如果测试终端激活成功，且使用的“IPv4 PDP address”在规划的地址池内，测试终端接入成功，调测结束。
    - 如果测试终端激活成功，但使用的“IPv4 PDP address”不在规划的地址池内，执行[步骤 3](#ZH-CN_OPI_0172547233__step4)。
3. 继续查看 [**DSP SESSIONINFO**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/会话信息管理/会话信息查询/显示用户上下文（DSP SESSIONINFO）_86526407.md) 命令，确认测试终端IP地址类型。
  ```
  DSP SESSIONINFO:QUERYTYPE=IMSI,IMSI="460000123456789";
  ```
  ```
  ......

  -------------------------
           APN name  =  apn-test1
                   ......        
  IPv4 Address type  =  UPF ALLOC IP ADDRESS 
                   ......      
  (Number of results = 1)
  ---    END
  ```
    - 如果是本地地址池分配（UPF ALLOC IP ADDRESS），请执行[步骤 4](#ZH-CN_OPI_0172547233__step7)。
    - 如果是外部地址池分配（EXTERNAL ALLOC IP ADDRESS），请返回配置页面重新配置。
4. 收集信息并寻求技术支持。
    a. 在OM Portal上创建用户跟踪任务 并保存报文。
    b. 执行 **[EXP MML](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/配置管理/配置导出管理/导出MML文件（EXP MML）_47200033.md)** 命令将当前配置数据导出为MML脚本文件并保存。
    c. 收集并保存上述所有查询信息。
    d. 收集归纳所有信息并联系华为技术支持解决。

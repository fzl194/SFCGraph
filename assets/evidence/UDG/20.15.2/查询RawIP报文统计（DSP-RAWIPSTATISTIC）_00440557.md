#### [命令功能](#ZH-CN_CONCEPT_0000001600440557)

该命令用于查看RawIP报文统计信息。

RawIP报文统计信息主要分为发送和接收两大部分。

OSPF、ICMP报文封装在RawIP报文中进行发送。因此，例如当进行ping操作时，可以通过查看本机收发RawIP报文的数量，判断是否由于RawIP报文收发不正常而导致网络异常。

#### [注意事项](#ZH-CN_CONCEPT_0000001600440557)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600440557)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600440557)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IPVERSION | IP版本 | 可选必选说明：可选参数<br>参数含义：该参数用于指定RawIP报文统计的IP版本。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- IPv4：IPv4协议族。<br>- IPv6：IPv6协议族。<br>默认值：IPv4 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600440557)

显示当前系统的RawIP报文统计：

```
DSP RAWIPSTATISTIC:;
```

```

        RETCODE = 0  操作成功

        结果如下
        ------------------------
                       接收报文计数  =  22
        接收报文PCB缓存查询失败计数  =  22
                       发送报文计数  =  0
        (结果个数 = 1)
        ---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600440557)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 接收报文计数 | 用于指定接收RawIP报文计数。 |
| 接收报文PCB缓存查询失败计数 | 用于指定接收RawIP报文PCB缓存查询失败计数。 |
| 发送报文计数 | 用于指定发送RawIP报文计数。 |

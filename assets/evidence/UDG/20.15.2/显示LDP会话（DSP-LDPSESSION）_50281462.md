#### [命令功能](#ZH-CN_CONCEPT_0000001550281462)

该命令用于显示LDP会话。

#### [注意事项](#ZH-CN_CONCEPT_0000001550281462)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550281462)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550281462)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| VRFNAME | VPN实例名称 | 可选必选说明：可选参数<br>参数含义：该参数用于表示VPN实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。<br>默认值：无 |
| PEERLSRID | 邻居LDP ID | 可选必选说明：可选参数<br>参数含义：该参数用于表示对等体的LDP标识符，格式为<LSR ID>：<标签空间>。标签空间取值： “0”表示全局标签空间。 “1”表示接口标签空间。<br>数据来源：对端协商<br>取值范围：字符串类型，输入长度范围为1～31。<br>默认值：无 |

显示LDP会话：

```
DSP LDPSESSION:;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001550281462)

| 输出项名称 | 输出项解释 |
| --- | --- |
| VPN实例名称 | 用于表示VPN实例名称。 |
| 邻居LDP ID | 用于表示对等体的LDP标识符，格式为<LSR ID>：<标签空间>。标签空间取值： “0”表示全局标签空间。 “1”表示接口标签空间。 |
| 本地LDP ID | 用于表示本地LDP标识符，格式为<LSR ID>：<标签空间>。标签空间取值： “0”表示全局标签空间。 “1”表示接口标签空间。 |
| TCP源地址 | 用于表示TCP源地址。 |
| TCP目的地址 | 用于表示TCP目的地址。 |
| 会话状态 | 用于表示会话状态。包含空状态、初始化状态、接受init消息状态、发送init消息状态、建立成功状态五种状态。 |
| 会话角色 | 用于表示会话角色。包含PASSIVE和ACTIVE两种角色。 |
| 会话类型 | 用于表示会话类型。包含本地、远端和本地兼远端三种类型。 |
| KA保持定时器协商值（s） | 用于表示KA保持定时器协商值（s）。 |
| KA发送计数 | 用于表示Keepalive发送计数。 |
| KA接收计数 | 用于表示Keepalive接收计数。 |
| 标签发布模式 | 用于表示标签发布模式。在MPLS体系中，由下游LSR决定将标签分配给特定FEC，再通知上游LSR。即标签由下游指定，标签的分配按从下游到上游的方向分发。 具有标签分发邻接关系的上游LSR和下游LSR必须对使用的标签发布方式达成一致。 |
| 邻居标签可用状态 | 用于表示邻居标签可用状态。可用：表示邻居可以正常分配标签。不可用：表示邻居可以不能正常分配标签。 |
| 当前FT标记 | 用于表示当前FT标记。FT为TRUE时，表示LSR具有GR能力。FT为FALSE时，表示LSR不具有GR能力。 |
| 当前MD5标记 | 用于表示当前会话是否配置MD5认证的标记。 |
| reconnect定时器的值（s） | 用于表示reconnect定时器的值。GR Restarter发生主备倒换后，GR Helper检测到和GR Restarter的LDP会话失败，将启动Reconnect定时器，等待LDP会话的重新建立。 |
| recovery定时器的值（s） | 用于表示recovery定时器的值。LDP会话重新建立后，GR Helper启动Recovery定时器，等待LSP的恢复。 |
| session建立的时长（DDDD:HH:MM） | 用于表示session建立的时长（DDDD:HH:MM）。 |
| 会话动态通告能力 | 用于表示是否使能会话的动态通告能力。 |

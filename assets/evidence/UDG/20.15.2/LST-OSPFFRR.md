#### [命令功能](#ZH-CN_CONCEPT_0000001600866329)

该命令用于查询OSPF IP FRR（快速重路由）配置信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600866329)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600866329)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600866329)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | OSPF进程号 | 可选必选说明：可选参数<br>参数含义：OSPF进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600866329)

查询OSPF进程号为1的IP FRR（快速重路由）配置信息：

```
LST OSPFFRR: PROCID=1;
```

```

RETCODE = 0  操作成功。

结果如下
------------------------
                  OSPF进程号  =  1
                 LFA是否使能  =  FALSE
              节点保护优先级  =  40
          最小开销路径优先级  =  20
   LDP联动最大开销路径优先级  =  10
                路由策略类型  =  无策略
                路由策略名称  =  NULL
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600866329)

| 输出项名称 | 输出项解释 |
| --- | --- |
| OSPF进程号 | 该参数用于表示OSPF进程号。 |
| LFA是否使能 | 该参数用于表示LFA是否使能，使能此参数后，OSPF IP FRR功能才会生效，生成无环备份路由。 |
| 节点保护优先级 | 该参数用于表示备份路径节点保护优先级。 |
| 最小开销路径优先级 | 该参数用于表示备份路径最小开销路径优先级。 |
| LDP联动最大开销路径优先级 | 该参数用于表示优选LDP联动通告最大开销路径为备份下一跳的优先级。 |
| 路由策略类型 | 该参数用于表示路由策略的类型，包含两种类型：无策略、路由策略。 |
| 路由策略名称 | 该参数用于表示一个路由策略的名称。 |

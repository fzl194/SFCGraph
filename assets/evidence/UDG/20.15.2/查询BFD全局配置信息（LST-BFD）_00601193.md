#### [命令功能](#ZH-CN_CONCEPT_0000001600601193)

该命令用于查询BFD全局配置信息。可以通过此命令来查询BFD使能情况、BFD延迟协商时长和BFD会话报文优先级配置值。

#### [注意事项](#ZH-CN_CONCEPT_0000001600601193)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600601193)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600601193)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001600601193)

查询当前BFD全局配置：

```
LST BFD:;
```

```

RETCODE = 0  操作成功

结果如下
-------------

             BFD使能标志  =  TRUE
     会话延迟Up时间（s）  =  0
   动态BFD会话报文优先级  =  7
        使能MPLS被动会话  =  FALSE
(结果个数 = 1)
---  END
```

#### [命令功能](#ZH-CN_CONCEPT_0182837225)

**适用NF：SGW-U、PGW-U、UPF**

该命令用于查询路径数。

#### [注意事项](#ZH-CN_CONCEPT_0182837225)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0182837225)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0182837225)

无。

#### [使用实例](#ZH-CN_CONCEPT_0182837225)

在需要查询路径的个数时，使用此命令：

```
DSP PATHNUMBER:;
```

```

RETCODE = 0  操作成功。

结果如下
--------
                最大的路径数  =  630000
          已使用的信令路径数  =  0
          已使用的数据路径数  =  0
                可用的路径数  =  630000
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0182837225)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 最大的路径数 | 用来表示最大的路径数。 |
| 已使用的信令路径数 | 用来表示已使用的信令路径数。 |
| 已使用的数据路径数 | 用来表示已使用的数据路径数。 |
| 可用的路径数 | 用来表示可用的路径数（可用路径数=最大路径数-已用路径数-故障路径数）。 |

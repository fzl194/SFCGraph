#### [命令功能](#ZH-CN_CONCEPT_0000001550121534)

该命令用于查询调度队列维测信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001550121534)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550121534)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550121534)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RUNAME | RU名称 | 可选必选说明：必选参数<br>参数含义：该参数用于表示的RU名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，区分大小写。<br>默认值：无<br>配置原则：使用DSP RU查看RU名称。 |
| THREADID | 线程ID | 可选必选说明：必选参数<br>参数含义：该参数用于表示线程的ID。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～255。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001550121534)

查询线程调度队列维测信息：

```
DSP MSSSCHTHREADQUEUE:THREADID=1,RUNAME="VNODE_VNRS_VNFC_IPU_0066";
```

```

RETCODE = 0  操作成功。

结果如下
--------
                          调度组ID  =  1
                            队列ID  =  0
间隔时间最大/平均值（microsecond）  =  2051/223
处理时间最大/平均值（microsecond）  =  1015/271
                 队列最大/平均深度  =  4/0
                     空调度率（%）  =  97
                       调度率（%）  =  100
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001550121534)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 调度组ID | 用于表示调度组的ID。 |
| 队列ID | 用于表示调度队列的ID。 |
| 间隔时间最大/平均值（microsecond） | 用于表示调度队列调度间隔最大/平均时间，单位是微秒。 |
| 处理时间最大/平均值（microsecond） | 用于表示队列每次得到调度的最大/平均处理时间，单位是微秒。 |
| 队列最大/平均深度 | 用于表示调度队列运行时最大/平均深度。 |
| 空调度率（%） | 用于表示队列调度为空占该队列调度次数的百分比。 |
| 调度率（%） | 用于表示调度队列占总队列调度时间的百分比。 |

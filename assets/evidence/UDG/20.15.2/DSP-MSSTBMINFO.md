#### [命令功能](#ZH-CN_CONCEPT_0000001550121082)

该命令用于查询表管理模块全局信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001550121082)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550121082)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550121082)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RUNAME | RU名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定RU名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，区分大小写。<br>默认值：无<br>配置原则：使用DSP RU查看RU名称。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001550121082)

查询表管理模块全局信息：

```
DSP MSSTBMINFO:RUNAME = "VNODE_VNRS_VNFC_IPU_0064";
```

```

RETCODE = 0  操作成功。

结果如下
--------
                RU名称  =  VNODE_VNRS_VNFC_IPU_0064
            最大表数量  =  128
        已使用的表数量  =  100
已占用内存数量（byte）  =  1078298336
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001550121082)

| 输出项名称 | 输出项解释 |
| --- | --- |
| RU名称 | 用于指定RU名称。 |
| 最大表数量 | 用于表示最大表数量。 |
| 已使用的表数量 | 用于表示已使用的表数量。 |
| 已占用内存数量（byte） | 用于表示已占用内存数量，单位为字节。 |

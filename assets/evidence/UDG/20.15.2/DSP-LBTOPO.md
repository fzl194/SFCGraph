- [参数说明](#ZH-CN_MMLREF_0294730409__1.3.4)

## [命令功能](#ZH-CN_MMLREF_0294730409)

该命令用于查询SDRC中缓存的CSLB的拓扑。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0294730409)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0294730409)

无

## [使用实例](#ZH-CN_MMLREF_0294730409)

查询SDRC中缓存的CSLB拓扑，结果如下：

```
%%DSP LBTOPO:;%%
RETCODE = 0  操作成功

结果如下
--------
High TB  Low TB  TP          组 ID

0       1109   2416123937  4        
0       1121   2416123937  4        
(结果个数 = 2)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0294730409)

| 输出项名称 | 输出项解释 |
| --- | --- |
| High TB | 该参数用于表示虚拟机（裸机）ID的高16位，标识CSLB进程所在的虚拟机（裸机）。 |
| Low TB | 该参数用于表示虚拟机（裸机）ID的低16位，标识CSLB进程所在的虚拟机（裸机）。 |
| TP | 该参数用于表示虚拟端口号，标识CSLB进程占用的端口号。 |
| 组ID | 该参数用于表示CSLB的属组ID。 |

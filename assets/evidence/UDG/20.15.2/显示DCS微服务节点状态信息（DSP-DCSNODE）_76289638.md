## [命令功能](#ZH-CN_MMLREF_0000002176289638)

该命令用于显示DCS微服务节点状态信息。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000002176289638)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000002176289638)

无

## [使用实例](#ZH-CN_MMLREF_0000002176289638)

显示所有实例的节点状态信息。

```
%%DSP DCSNODE:;%%
RETCODE = 0  操作成功

结果如下
------------------------

             节点状态  =  节点状态正常
      内存总容量（MB） =  56500
      内存使用量（MB） =  15665
      磁盘总容量（MB） =  2103
      磁盘使用量（MB） =  612
节点写入速率（Mbit/s） =  0
节点读取速率（Mbit/s） =  0
磁盘写入速率（Mbit/s） =  0
磁盘读取速率（Mbit/s） =  0
跨板写入速率（Mbit/s） =  0
跨板读取速率（Mbit/s） =  0
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000002176289638)

| 输出项名称 | 输出项解释 |
| --- | --- |
| POD名称 | 该参数用于表示POD名称。 |
| 节点状态 | 该参数用于表示节点状态。<br>取值说明：<br>- NODE_STATUS_NORMAL（节点状态正常）<br>- NODE_STATUS_ABNORMAL（节点状态不正常） |
| 内存总容量（MB） | 该参数用于表示配置内存可用最大值。 |
| 内存使用量（MB） | 该参数用于表示当前内存使用量。 |
| 磁盘总容量（MB） | 该参数用于表示磁盘配置可用量。 |
| 磁盘使用量（MB） | 该参数用于表示当前磁盘空间使用量。 |
| 节点写入速率（Mbit/s） | 该参数用于表示节点的回源写入速率。 |
| 节点读取速率（Mbit/s） | 该参数用于表示节点的查询出流速率。 |
| 磁盘写入速率（Mbit/s） | 该参数表示磁盘写入速率。 |
| 磁盘读取速率（Mbit/s） | 该参数表示磁盘读取速率。 |
| 跨板写入速率（Mbit/s） | 该参数用于表示跨板通信数据写入速率。 |
| 跨板读取速率（Mbit/s） | 该参数用于表示跨版通信数据读取速率。 |

#### [命令功能](#ZH-CN_CONCEPT_0000206691417210)

**适用NF：EPSN**

该命令用于显示VPN实例。

#### [注意事项](#ZH-CN_CONCEPT_0000206691417210)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000206691417210)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000206691417210)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| VPNINSTANCE | VPN实例名 | 可选必选说明：可选参数<br>参数含义：该参数用于设置VPN实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000206691417210)

- 显示全部SFIPVPNInst业务配置：
  ```
  LST SFIPVPNINST:;
  ```
  ```

  RETCODE = 0  操作成功

  VPN实例信息
  ------------------------
  VPN实例名          描述信息  

  vpn1               vpn1         
  vpn2               vpn2         
  (结果个数  = 2)

  ---    END
  ```
- 指定VPN实例名显示SFIPVPNInst业务配置：
  ```
  LST SFIPVPNINST: VPNINSTANCE="vpn1";
  ```
  ```

  RETCODE = 0  操作成功

  VPN实例信息
  ------------------------
  VPN实例名  =  vpn1
  描述信息   =  vpn1
  (结果个数  = 1)

  ---    END
  ```

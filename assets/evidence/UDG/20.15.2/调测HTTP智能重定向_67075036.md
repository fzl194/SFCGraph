# 调测HTTP智能重定向

## [操作场景](#ZH-CN_OPI_0167075036)

运营商部署HTTP智能重定向功能后，需要进行HTTP智能重定向功能的调测，保证本功能可以正常使用。

> **说明**
> 适用于PGW-U、UPF。

## [必备事项](#ZH-CN_OPI_0167075036)

前提条件

- HTTP智能重定向激活已经完成，请参见[GWFD-110284 HTTP智能重定向](../GWFD-110284 HTTP智能重定向_67075034.md)。
- 重定向的第三方服务器与UDG的互通配置已经完成。
- 重定向设备上已经安装了第三方抓包工具。

数据

| 类别 | 参数名称 | 取值样例 | 获取方法 | 说明 |
| --- | --- | --- | --- | --- |
| [**LST EXTENDEDFILTER**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/扩展过滤器/查询扩展过滤器（LST EXTENDEDFILTER）_82837382.md) | 扩展过滤器名字（EXTFLTNAME） | filter_http-redirect1<br>filter_http-redirect2 | 已配置数据中获取 | HTTP智能重定向使用的扩展过滤器 |
| [**LST RULE**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务匹配公共配置/业务规则管理/规则/查询规则（LST RULE）_82837270.md) | 规则名称（RULENAME） | rule_test | 已配置数据中获取 | HTTP智能重定向使用的业务规则 |

工具

- 测试终端
- OM Portal跟踪工具
- 第三方抓包工具

## [操作步骤](#ZH-CN_OPI_0167075036)

1. 执行 [**LST LICENSESWITCH**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/查询License配置项开关（LST LICENSESWITCH）_09587900.md) 命令，查询 UDG 上 HTTP重定向 对应的License配置开关是否打开。
  ```
  LST LICENSESWITCH: LICITEM="LKV3G5SHPR01";
  ```
    - 如果“SWITCH”为“ENABLE”，请执行[步骤 2](#ZH-CN_OPI_0167075036__step2)。
    - 如果“SWITCH”为“DISABLE”，则执行[**SET LICENSESWITCH**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09587387.md)命令打开本特性对应的License配置开关。
2. 打开接入侧/PDN侧镜像接口上的抓包工具，准备抓取 测试终端 的出入报文。
3. 测试终端 访问 **www.example.com** 网页。
    - 如果测试终端可以正常访问页面，请执行[步骤 4](#ZH-CN_OPI_0167075036__step4)。
    - 如果测试终端无法正常访问页面，请调测UDG的Web浏览功能。
4. 测试终端 访问 **www.huawei.com** 网页，查看镜像接口的抓包信息。
  **图1** MS/UE发起HTTP Get Request报文

  <br>

  ![](调测HTTP智能重定向_67075036.assets/fig_20_110228_02.png "点击放大")
  **图2** UDG 发送HTTP Response消息给MS/UE

  <br>

  ![](调测HTTP智能重定向_67075036.assets/fig_20_110228_03.png "点击放大")
  > **说明**
  > 对于使用HTTP协议访问URL为以www.huawei.com开头的网页，并且浏览器类型为Mozilla的网页访问，当HTTP服务器返回的错误码大于400时，触发HTTP智能重定向。
  >
  > 如 [图1](#ZH-CN_OPI_0167075036__fig2) 所示， 测试终端 使用Mozilla类型的浏览器访问 **www.huawei.com** 网页，发送HTTP GET请求（消息）。 UDG 将缓存MS/UE的初始请求URL并向HTTP服务器转发MS/UE的HTTP请求（消息）。
  >
  > 由于MS/UE的初始请求URL填写不正确或不存在，HTTP服务器给 UDG 返回失败响应消息，消息携带error code为404。 UDG 对HTTP响应报文进行解析，并对错误码进行判断。由于HTTP响应报文中携带的错误码与 UDG 配置匹配， UDG 构造重定向报文，报文包括：第三方服务器 URL（www.example.com），并根据配置携带初始请求的URL和IMSI。然后 UDG 发送HTTP Response消息给MS/UE，如 [图2](#ZH-CN_OPI_0167075036__fig3) 所示。
  >
  > MS/UE发起HTTP Get Request（消息），消息携带第三方服务器 URL（www.example.com）。
    - 如果测试终端上看到跳转到“www.example.com”页面，接入侧/PDN侧抓包结果与[图1](#ZH-CN_OPI_0167075036__fig2)一致，重定向成功，调测结束。
    - 如果测试终端上仍显示**www.huawei.com**网页，接入侧/PDN侧抓包的目的IP与源IP一样，请执行[步骤 5](#ZH-CN_OPI_0167075036__step5)。
5. 执行 [**LST RULE**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务匹配公共配置/业务规则管理/规则/查询规则（LST RULE）_82837270.md) 、 [**LST SMARTHTTPREDIR**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/重定向控制/Http重定向控制/HTTP智能重定向/查询HTTP重定向配置（LST SMARTHTTPREDIR）_86528733.md) 和 [**LST EXTENDEDFILTER**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务过滤器管理/流过滤器管理/扩展过滤器/查询扩展过滤器（LST EXTENDEDFILTER）_82837382.md) 命令，查询HTTP智能重定向规则是否与规划的一致。
  ```
  LST RULE:RULENAME="rule_test";
  ```
  ```
  LST SMARTHTTPREDIR:;
  ```
  ```
  LST EXTENDEDFILTER: EXTFLTNAME="filter_http-redirect1";
  ```
  ```
  LST EXTENDEDFILTER: EXTFLTNAME="filter_http-redirect2";
  ```
    - 如果是，请执行[步骤 6](#ZH-CN_OPI_0167075036__step6)。
    - 如果不是，请参考[激活HTTP智能重定向](激活HTTP智能重定向_67075035.md)重新配置，并再次执行[步骤 3](#ZH-CN_OPI_0167075036__step3)。
6. 查看 UDG 用户跟踪中的PFCP Session Modification Request消息，检查SMF是否下发正确的HTTP智能重定向规则名称。
    - 如果SMF下发正确业务规则，请执行[步骤 7](#ZH-CN_OPI_0167075036__step8)。
    - 如果SMF下发的规则不正确，请联系SMF工程师修改SMF上的规则配置，并再次执行[步骤 3](#ZH-CN_OPI_0167075036__step3)。
7. 执行 [**PING**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/系统管理/系统维护/Ping和Tracert/Ping/IP Ping（PING）_16499685.md) 命令，查看 UDG 到第三方服务器的链路是否正常连通。
  ```
  PING: IPVERSION=IPv4, DESTIPV4ADDRESS="192.168.200.20";
  ```
    - 如果收到对端网元的响应，说明UDG到第三方服务器连接正常，请执行[步骤 8](#ZH-CN_OPI_0167075036__step9)。
    - 如果连接出现timeout，表明链路不通。请检查本端或第三方服务器是否存在物理接口故障或者路由配置错误，然后重新执行[步骤 4](#ZH-CN_OPI_0167075036__step4)。
8. 收集信息并寻求技术支持。
    a. 在镜像接口或服务器上开启抓包工具，执行 [步骤 4](#ZH-CN_OPI_0167075036__step4) 并保存报文。
    b. 执行 [**EXP MML**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/配置管理/配置导出管理/导出MML文件（EXP MML）_47200033.md) 命令将当前配置数据导出为MML脚本文件并保存。
    c. 收集并保存上述所有查询信息。
    d. 查看并收集对端设备配置及接口状态信息。
    e. 收集归纳所有信息并联系华为技术支持解决。

#### [命令功能](#ZH-CN_CONCEPT_0000201105977152)

**适用NF：UPF**

该命令用于显示UPF上配置的NF实例ID。

#### [注意事项](#ZH-CN_CONCEPT_0000201105977152)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000201105977152)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000201105977152)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000201105977152)

获取当前UPF配置的NF实例ID：

```
LST UPINSTID:;
```

```

RETCODE = 0  操作成功。

结果如下
----------
UUID  =  4947a69a-f61b-4bc1-b9da-47c9c5d14b64
(结果个数 = 1)

---    END
```

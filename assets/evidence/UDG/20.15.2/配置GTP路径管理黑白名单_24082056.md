## [操作场景](#ZH-CN_OPI_0124082056)

当运营商在部署分组交换网，新增 UDG 时，需要配置GTP路径管理黑白名单。 UDG 可以通过GTP路径管理黑白名单区分是否需要对网络中的对端网元进行探测。

> **说明**
> 适用于SGW-U、PGW-U、UPF。

## [操作步骤](#ZH-CN_OPI_0124082056)

1. 开启GTP路径管理的黑白名单开关，配置GTP路径管理名单属性为白名单。
  [**SET UPGTPPATH**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/路径管理/GTP路径管理/GTP协议参数管理/GTP路径全局参数/设置GTP路径相关属性（SET UPGTPPATH）_82837227.md)
2. 配置需要探测的对端网元的GTP信令面或者数据面IP地址段。
  [**ADD ECHOIPLIST**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/路径管理/GTP路径管理/GTP协议参数管理/路径地址列表/配置GTP路径管理IP地址（ADD ECHOIPLIST）_82837219.md)

## [任务示例](#ZH-CN_OPI_0124082056)

任务描述

1. 配置 UDG 向IPv4地址段为192.168.1.10/24、192.168.10.10/24和192.168.100.10/24的对端网元进行echo探测。
2. 配置 UDG 向IPv6地址段为fc00::1/64的对端网元进行echo探测。

脚本

//开启GTP路径管理的黑白名单开关，配置GTP路径管理名单属性为白名单。

```
SET UPGTPPATH:ECHOLISTSWITCH=ENABLE,ECHOLISTTYPE=WHITE;
```

//配置需要探测IPv4地址段。

```
ADD ECHOIPLIST:IPVERSION=IPV4,IPV4ADDR="192.168.1.10",IPV4MASK=24;
```

```
ADD ECHOIPLIST:IPVERSION=IPV4,IPV4ADDR="192.168.10.10",IPV4MASK=24;
```

```
ADD ECHOIPLIST:IPVERSION=IPV4,IPV4ADDR="192.168.100.10",IPV4MASK=24
```

//配置需要探测IPv6地址段。

```
ADD ECHOIPLIST:IPVERSION=IPV6,IPv6ADDR="fc00::1",IPv6MASK=64;
```

#### [命令功能](#ZH-CN_CONCEPT_0000206324472787)

**适用NF：UPF、PGW-U、SGW-U、GGSN**

查询服务器时延统计功能配置。

#### [注意事项](#ZH-CN_CONCEPT_0000206324472787)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000206324472787)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000206324472787)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000206324472787)

查询服务器时延统计功能配置：

```
LST SERVERRTTSTAT:;
```

```

RETCODE = 0  操作成功
 
结果如下
--------
    服务器时延统计开关  =  使能
服务器时延统计协议列表  =  FaceBook&Tiktok&Youtube&Netflix&DNS
    时延统计的开始时间  =  2024-08-05 10:51:54+0800
          时延统计时长  =  1
(结果个数 = 1)
 
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000206324472787)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 时延统计的开始时间 | 表示时延统计的开始时间。 |

其余输出项请参见SET SERVERRTTSTAT的参数说明。

#### [命令功能](#ZH-CN_CONCEPT_0000001549960934)

该命令用于显示报文统计信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001549960934)

- 该命令执行后立即生效。
- 该命令仅适用于非NP卡场景和NP卡非加速模式场景。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549960934)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549960934)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RUNAME | RU名称 | 可选必选说明：可选参数<br>参数含义：资源单元名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。<br>默认值：无<br>配置原则：<br>- 区分大小写，不支持空格。<br>- 使用DSP RU查看RU名称。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549960934)

- 显示报文统计信息：
  ```
  DSP SECTOTALPKTSTAT:;
  ```
  ```
  RETCODE = 0  操作成功

  结果如下
  --------
  RU名称                      总报文数    通过报文数    丢弃报文数

  VNODE_VNRS_VNFC_IPU_0064    56          56            0         
  VNODE_VNRS_VNFC_IPU_0065    0           0             0         
  (结果个数 = 2)
  ---    END
  ```
- 显示指定条件报文统计信息：
  ```
  DSP SECTOTALPKTSTAT:RUNAME="VNODE_VNRS_VNFC_IPU_0064";
  ```
  ```
  RETCODE = 0  操作成功

  结果如下
  -------------------------
      RU名称  =  VNODE_VNRS_VNFC_IPU_0064
    总报文数  =  19578
  通过报文数  =  19578
  丢弃报文数  =  0
  (结果个数 = 1)
  ---    END
  ```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001549960934)

| 输出项名称 | 输出项解释 |
| --- | --- |
| RU名称 | 该参数表示资源单元的名称。 |
| 总报文数 | 该参数表示总报文数量。 |
| 通过报文数 | 该参数表示通过报文数量。 |
| 丢弃报文数 | 该参数表示丢弃报文数量。 |

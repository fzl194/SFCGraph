#### [命令功能](#ZH-CN_CONCEPT_0000001549961438)

该命令用于查询Router LSA生成流程的信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001549961438)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549961438)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549961438)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | OSPF进程号 | 可选必选说明：可选参数<br>参数含义：OSPF进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549961438)

查询OSPF进程1的Router LSA生成流程的信息：

```
DSP OSPFORIGINATELSA:PROCID=1;
```

```

```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001549961438)

| 输出项名称 | 输出项解释 |
| --- | --- |
| OSPF进程号 | 该参数用于表示OSPF进程号。 |
| 区域号 | 该参数用于表示OSPF区域号。 |
| 路由器标识 | 该参数用于表示OSPF路由器标识。 |
| Flush LSA计数 | 该参数用于表示接收到的Flush Router-LSA报文统计计数。 |
| 创建失败计数 | 该参数用于表示创建失败的Router-LSA报文统计计数。 |
| 添加失败计数 | 该参数用于表示添加失败的Router-LSA报文统计计数。 |
| 查找失败计数 | 该参数用于表示查找Router-LSA数据库失败的统计计数。 |
| 接收相同计数 | 该参数用于表示接受到相同Router-LSA报文的统计计数。 |
| 抑制LSA计数 | 该参数用于表示抑制Router-LSA报文生成的统计计数。 |

## [命令功能](#ZH-CN_MMLREF_0000001225912247)

该命令用于查询IKE全局配置。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000001225912247)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001225912247)

无

## [输出结果说明](#ZH-CN_MMLREF_0000001225912247)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 清除分片标记位 | 清除分片标记位。 |
| 加密前分片 | 加密前分片。 |
| 流量方式触发SA过期标记位 | 流量方式触发SA过期。 |
| 流量数 (kbyte) | 流量方式触发SA过期的流量数。 |
| 时间间隔 (s) | 时间方式触发SA过期的时间间隔。 |
| 抗重放 | 抗重放功能开关。 |
| 窗口大小 | 抗重放窗口的大小，超过这个流量的报文会被丢弃。 |
| 本地名称 | 本地的名字。 |
| DPD检查间隔 (s) | DPD间隔。 |
| DPD类型 | DPD类型。 |
| DPD重试间隔 (s) | DPD消息重发的间隔。 |
| 历史信息记录条数 | 历史信息记录条数。 |
| 安全日志阈值 | 安全日志阈值。 |
| NAT保活时间间隔 (s) | 发送NAT保活报文的时间间隔。 |
| IPSEC转发CPU过载告警上报阈值 (%) | IPSEC转发CPU过载告警上报阈值。 |
| IPSEC转发CPU过载告警恢复阈值 (%) | IPSEC转发CPU过载告警恢复阈值。 |
| UIKE主动核查时间 | UIKE主动向AMUP发起核查的时间。 |
| IPsec跟踪的开启流控的CPU阈值（%） | 该参IPsec跟踪的开启流控的CPU阈值。如果IPsec转发使用的CPU的占用超过本参数配置的阈值，将开启跟踪流控。 |
| IPsec跟踪的恢复流控的CPU阈值（%） | 该参数表示IPsec跟踪的恢复流控的CPU阈值。如果IPsec流控在开启状态中，然后IPsec转发使用的CPU占用恢复到本参数配置的阈值以下，将恢复流控。 |

#### [命令功能](#ZH-CN_CONCEPT_0182837706)

**适用NF：SGW-U、PGW-U、UPF**

该命令用于查询是否使能Cos Tos映射策略。

#### [注意事项](#ZH-CN_CONCEPT_0182837706)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0182837706)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0182837706)

无。

#### [使用实例](#ZH-CN_CONCEPT_0182837706)

查询是否使能Cos Tos映射策略：

```
LST COSTOS:;
```

```

RETCODE = 0  操作成功。

结果如下
--------
TOS策略  =  不使能
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0182837706)

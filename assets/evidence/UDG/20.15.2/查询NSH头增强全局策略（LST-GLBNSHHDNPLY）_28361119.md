#### [命令功能](#ZH-CN_CONCEPT_0000203928361119)

**适用NF：PGW-U、UPF**

该命令用于查询NSH头增强全局策略。

#### [注意事项](#ZH-CN_CONCEPT_0000203928361119)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000203928361119)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000203928361119)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000203928361119)

查询NSH头增强全局策略信息：

```
LST GLBNSHHDNPLY:;
```

```

RETCODE = 0  操作成功

全局NSH头增强策略信息
---------------------
    NSH头增强名称  =  nsh
 重定向IP协议版本  =  IPV4
   重定向IPV6地址  =  ::
   重定向IPV4地址  =  10.0.0.0
NSH对端设备端口号  =  6633
(结果个数 = 1)

---    END
```

# 卸载Tethering检测特征库（RMV TETHERDB）

#### [命令功能](#ZH-CN_CONCEPT_0182837455)

**适用NF：PGW-U、UPF**

该命令用来卸载已经加载的Tethering检测特征库。

#### [注意事项](#ZH-CN_CONCEPT_0182837455)

- 该命令执行后立即生效。
- 此命令的生效范围为整机。

#### [操作用户权限](#ZH-CN_CONCEPT_0182837455)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0182837455)

无。

#### [使用实例](#ZH-CN_CONCEPT_0182837455)

假如运营商需卸载已经加载的Tethering检测特征库：

```
RMV TETHERDB:;
```

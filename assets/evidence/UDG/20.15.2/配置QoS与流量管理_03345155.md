## [操作场景](#ZH-CN_OPI_0303345155)

通过配置QoS与流量管理功能，可实现以下需求：

- 为了保证在资源紧缺时，运营商能够为高优先级用户优先分配资源。
- 在资源相对富余时，更充分地发挥网络资源的效益，提高客户满意度。
  > **说明**
  > 适用于SGW-U、PGW-U、UPF。

## [操作步骤](#ZH-CN_OPI_0303345155)

- 配置支持入口报文的优先级由CoS到ToS映射（VLAN到IP网络的优先级映射）。
  ![](配置QoS与流量管理_03345155.assets/notice_3.0-zh-cn.png)

  如果同时配置CoS到ToS的映射和3GPP QoS到DSCP/ToS的映射，CoS到ToS的映射结果会被3GPP QoS到DSCP/ToS的映射结果覆盖。
    1. 配置TunnelMarking是否使能。
      [**SET SRVCOMMONPARA**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务匹配公共配置/业务公共参数管理/业务公共参数/设置业务公共参数（SET SRVCOMMONPARA）_82837309.md)
    2. 定义一个DS域并进入Diff-Serv域实例。
      [**ADD QOSDIFFERSERV**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/安全管理/QoS管理/DS域配置/增加DS域（ADD QOSDIFFERSERV）_00841105.md)
    3. 配置DS域的上行VLAN报文的802.1p的值与 UDG 内部的服务等级的对应关系，并可将报文着色。
      [**SET QOSBA**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/安全管理/QoS管理/BA映射关系/设置QoS BA（SET QOSBA）_00441521.md)
    4. 在接口上绑定DS域，使其支持简单流分类，以便对报文的优先级进行修改,并在该VLAN子接口上配置trust CoS。
      [**ADD QOSIFTRUST**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/安全管理/QoS管理/接口信任信息/增加QoS接口信任（ADD QOSIFTRUST）_50121734.md)
- 配置支持出口报文的优先级由ToS到CoS映射（IP网络到VLAN的优先级映射）。
    1. 配置TunnelMarking是否使能。
      [**SET SRVCOMMONPARA**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务匹配公共配置/业务公共参数管理/业务公共参数/设置业务公共参数（SET SRVCOMMONPARA）_82837309.md)
    2. 定义DS域并创建一个新的DS域。
      [**ADD QOSDIFFERSERV**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/安全管理/QoS管理/DS域配置/增加DS域（ADD QOSDIFFERSERV）_00841105.md)
    3. 配置DS域的下行报文在 UDG 内部的服务等级和报文颜色与外部vlan报文的802.1p值的对应关系。
      [**SET QOSPHB**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/安全管理/QoS管理/PHB映射关系/设置QoS PHB（SET QOSPHB）_00840685.md)
    4. 用来在接口上绑定DS域，使其支持简单流分类，以便对报文的优先级进行修改，并在该VLAN子接口上配置trust CoS。
      [**ADD QOSIFTRUST**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/VNRS功能管理/IP服务/安全管理/QoS管理/接口信任信息/增加QoS接口信任（ADD QOSIFTRUST）_50121734.md)
- 配置QoS CAR功能。
    1. 配置指定类型的用户上下行CAR功能。
      **[SET QOSCAR](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/用户QOS控制/流量管理/QoS CAR配置/设置QosCar配置（SET QOSCAR）_82837668.md)**
    2. 配置基于APN的QoS属性数据。
      **[SET APNQOSATTR](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/用户QOS控制/流量管理/APN的QoS属性配置/设置ApnQosAttr配置（SET APNQOSATTR）_82837665.md)**

## [任务示例](#ZH-CN_OPI_0303345155)

任务描述

- 配置支持入口报文的优先级由CoS到ToS映射（VLAN到IP网络的优先级映射）。
- 配置支持出口报文的优先级由ToS到CoS映射（IP网络到VLAN的优先级映射）。
- 使能QoS CAR功能。

脚本

- 配置支持入口报文的优先级由CoS到ToS映射（VLAN到IP网络的优先级映射）。
  //配置TunnelMarking是否使能。
  ```
  SET SRVCOMMONPARA:TUNNELMARKINGID=ENABLE;
  ```
  //定义DS域并创建一个新的DS域。
  ```
  ADD QOSDIFFERSERV:DSNAME="default";
  ```
  //配置DS域的上行VLAN报文的802.1p的值与 UDG 内部的服务等级的对应关系，并可将报文着色。
  ```
  SET QOSBA:DSNAME="default",BATYPE= 8021p,BAVALUE=3,SERVICECLASS=be,COLOR=red;
  ```
  //用来在接口上绑定DS域，使其支持 简单流分类 ，以便对报文的优先级进行修改，并在该VLAN子接口上配置trust CoS。
  ```
  ADD QOSIFTRUST:DSNAME="default",IFNAME="Ethernet66/0/5";
  ```
- 配置支持出口报文的优先级由Tos到CoS映射（IP到VLAN网络的优先级映射）。
  //配置TunnelMarking是否使能。
  ```
  SET SRVCOMMONPARA:TUNNELMARKINGID=ENABLE;
  ```
  //定义DS域并创建一个新的DS域。
  ```
  ADD QOSDIFFERSERV:DSNAME="default";
  ```
  //配置DS域的下行报文在 UDG 内部的服务等级和报文颜色与外部vlan报文的802.1p值的对应关系。
  ```
  SET QOSPHB:DSNAME="default",PHBTYPE=8021p,PHBVALUE=3,SERVICECLASS=be,COLOR=green;
  ```
  //用来在接口上绑定DS域，使其支持 简单流分类 ，以便对报文的优先级进行修改，并在该VLAN子接口上配置trust CoS。
  ```
  ADD QOSIFTRUST:DSNAME="default",IFNAME="Ethernet66/0/5";
  ```
- 配置QoS CAR功能。
  //指定4G本地用户开启上下行CAR功能。

  ```
  SET QOSCAR: USERTYPE=HOME, ULRAT=eutran-1, DLRAT=eutran-1;
  ```
  //开启基于APN的QoS CAR功能。

  ```
  SET APNQOSATTR: APN="apn1", CARSHAPESWUL=ENABLE, CARSHAPEUL=CAR, CARSHAPESWDL=ENABLE, CARSHAPEDL=CAR;
  ```

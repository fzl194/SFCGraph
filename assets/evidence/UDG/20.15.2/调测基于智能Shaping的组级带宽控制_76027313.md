## [操作场景](#ZH-CN_OPI_0000001576027313)

当部署了基于智能Shaping的组级带宽控制后，对UDG进行调测，确认功能是否生效。

## [必备事项](#ZH-CN_OPI_0000001576027313)

前提条件

已完成 [激活基于智能Shaping的组级带宽控制](激活基于智能Shaping的组级带宽控制_76107785.md) 。

## [操作步骤](#ZH-CN_OPI_0000001576027313)

1. 执行 [**LST LICENSESWITCH**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/查询License配置项开关（LST LICENSESWITCH）_09587900.md) 命令查询UDG上本特性对应的License配置开关是否打开。
  ```
  LST LICENSESWITCH:LICITEM="LKV3G5FSHP01";
  ```
    - 如果“SWITCH”为“ENABLE”，请执行[步骤 2](#ZH-CN_OPI_0000001576027313__step13572103724313)。
    - 如果“SWITCH”为“DISABLE”，则执行[**LST LICENSESWITCH**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/查询License配置项开关（LST LICENSESWITCH）_09587900.md)命令打开本特性对应的License配置开关。
2. 执行LST BCSRVLEVELPL 查询带宽管理控制器业务等级策略是否生效。
  ```
  %%LST BCSRVLEVELPLY:;%%
  RETCODE = 0  Operation succeeded
  BWM Controller Service Level Policy Information
  -----------------------------------------------
  BWM Controller Name  =  bc1
        Service Level  =  2
             CIR Rate  =  50
             ShapRate  =  0
             PIR Rate  =  1000
       MaxPktLostRate  =  10000
   Max Increase Delay  =  NULL
  (Number of results = 1)
  ---    END
  ```
3. 执行LST BWMCONTROLLER查询置智能支持用户级公平调度功能是否生效。

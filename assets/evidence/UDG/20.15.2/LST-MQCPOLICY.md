#### [命令功能](#ZH-CN_CONCEPT_0000001550121150)

该命令用于查询已经创建的流量策略信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001550121150)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550121150)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550121150)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| POLICYNAME | 策略名称 | 可选必选说明：可选参数<br>参数含义：指定流策略的名称，不允许为系统预定义策略default。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。<br>默认值：无<br>配置原则：区分大小写，不支持空格。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001550121150)

查询已经创建的流量策略信息：

```
LST MQCPOLICY:;
```

```

RETCODE = 0  操作成功

结果如下
------------------------
    策略名称  =  p1
统计使能标志  =  去使能
    共享模式  =  非共享模式
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001550121150)

| 输出项名称 | 输出项解释 |
| --- | --- |

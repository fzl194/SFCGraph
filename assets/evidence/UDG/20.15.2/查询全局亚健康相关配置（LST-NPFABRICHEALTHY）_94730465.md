## [命令功能](#ZH-CN_MMLREF_0294730465)

该命令用于查询全局亚健康相关配置：包括亚健康阈值和亚健康检测周期。

> **说明**
> - 该命令执行后立即生效。
> - 该命令仅适用于NP卡加速模式场景。

#### [操作用户权限](#ZH-CN_MMLREF_0294730465)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0294730465)

无

## [使用实例](#ZH-CN_MMLREF_0294730465)

查询全局亚健康相关配置：

```
LST NPFABRICHEALTHY:;
RETCODE = 0  操作成功

结果如下
--------
    亚健康阙值  =  50
亚健康探测周期  =  100
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0294730465)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 亚健康阙值 | NP Fabric亚健康阈值，阈值 = 丢包率（千分比）+5 * 错包率（千分比）。 |
| 亚健康探测周期 | NP Fabric亚健康探测周期，单位为秒。 |

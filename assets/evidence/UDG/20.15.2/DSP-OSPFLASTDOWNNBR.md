#### [命令功能](#ZH-CN_CONCEPT_0000001550121626)

该命令用于显示OSPF断开的邻居信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001550121626)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550121626)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550121626)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | 进程号 | 可选必选说明：可选参数<br>参数含义：OSPF进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001550121626)

显示设备OSPF进程号为1的所有断开的路由信息：

```
DSP OSPFLASTDOWNNBR: PROCID=1;
```


#### [输出结果说明](#ZH-CN_CONCEPT_0000001550121626)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 进程号 | 该参数指定用户创建的OSPF进程号。 |
| 路由器标识 | 该参数指定本地路由器标识。 |
| 邻居区域标识 | 该参数指定邻居所在的区域标识。 |
| 接口名 | 该参数指定本端建立邻居的接口名称。 |
| 邻居IP地址 | 该参数指定邻居IP地址。 |
| 邻居路由器标识 | 该参数指定邻居路由器标识。 |
| 邻居Down的直接原因 | 该参数指定邻居Down的直接原因。 |
| 邻居Down的次要原因 | 该参数指定邻居Down的次要原因。 |
| 邻居Down的时间 | 该参数指定邻居Down的时间点。 |

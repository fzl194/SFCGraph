#### [命令功能](#ZH-CN_CONCEPT_0000203151189266)

**适用NF：PGW-U、UPF**

该命令用于显示配置的全局的内容过滤模板。

#### [注意事项](#ZH-CN_CONCEPT_0000203151189266)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000203151189266)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000203151189266)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000203151189266)

显示配置的全局的内容过滤模板：

```
LST GLBCFTEMPLATE:;
```

```

RETCODE = 0  操作成功

--------------------

(结果个数 = 1)

```

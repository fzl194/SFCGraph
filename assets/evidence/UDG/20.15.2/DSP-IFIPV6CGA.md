#### [命令功能](#ZH-CN_CONCEPT_0000001600441249)

该命令用于显示IPv6 CGA地址状态信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600441249)

- 该命令执行后立即生效。
- 该命令可以在VNRS_VNFC的Ethernet接口，Ethernet子接口上配置。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600441249)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600441249)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IFNAME | 接口名 | 可选必选说明：可选参数<br>参数含义：该参数用于指定IPv6地址CGA动态信息的接口名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。接口名称由接口类型+接口编号组成。<br>默认值：无<br>配置原则：请使用LST INTERFACE命令查看可用接口。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600441249)

显示接口IPv6 CGA信息：

```
DSP IFIPV6CGA:IFNAME="ethernet64/0/3";
```

```

RETCODE = 0  操作成功

结果如下
-------------------------
  接口名  =  Ethernet64/0/3
密钥名称  =  aa
安全级别  =  高
修正地址  =  2001:db8::1
(结果个数 = 1)
---   END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600441249)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 接口名 | 用于指定IPv6地址CGA动态信息的接口名。 |
| 密钥名称 | 用于指定IPv6 CGA密钥信息。 |
| 安全级别 | 用于指定IPv6 CGA安全级别。 |
| 修正地址 | 用于指定IPv6 CGA地址修正值。 |

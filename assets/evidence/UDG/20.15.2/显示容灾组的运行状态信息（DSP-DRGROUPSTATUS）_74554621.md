## [命令功能](#ZH-CN_MMLREF_0000001374554621)

该命令用于显示容灾组的运行状态信息以及容灾控制通道状态。

> **说明**
> - 该命令只用于在UEG-L/UEN网元采用主备（冷备）容灾模式下执行。
> - 该命令只用于在UEG-M/UEG网元采用主备（热备）容灾模式下执行。

#### [操作用户权限](#ZH-CN_MMLREF_0000001374554621)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001374554621)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| DRGROUPID | 容灾组标识 | 可选必选说明：可选参数<br>参数含义：该参数用于表示容灾组标识。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是1~8。<br>默认值：无<br>配置原则：<br>可使用<br>[**LST DRGROUPINFO**](查询容灾组信息（LST DRGROUPINFO）_74835153.md)<br>返回的DRGROUPID作为参数输入。 |

## [使用实例](#ZH-CN_MMLREF_0000001374554621)

显示容灾组的运行状态信息，以及心跳探测结果：

```
%%DSP DRGROUPSTATUS:;%%
RETCODE = 0  操作成功

结果如下
--------
                容灾组标识  =  1
                容灾组名称  =  HafGTnGrp
          本端容灾实例标识  =  222
      本端容灾实例运行状态  =  容灾实例运行状态为备
主备容灾实例之间的心跳状态  =  主备容灾实例之间握手成功，处于连接状态
      最近一次链接建立时间  =  2022-07-31 22:36:05
      最近一次链接断开时间  =  NULL
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001374554621)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 容灾组标识 | 该参数用于表示容灾组标识。 |
| 容灾组名称 | 该参数用于表示容灾组名称。 |
| 本端容灾实例标识 | 该参数用于表示本端容灾实例标识。 |
| 本端容灾实例运行状态 | 该参数用于表示本端容灾实例运行状态。<br>若输出结果为INIT，可使用工程命令DSP DRMODE查询当前环境的容灾模式。当DR MODE为ACTIVE-STANDBY或LOADBALANCE时，表示当前处于初始态；当DR MODE为NULL时，表示未使能容灾功能。<br>取值说明：<br>- INIT（未使能容灾功能或初始态）<br>- ACTIVE（容灾实例运行状态为主）<br>- STANDBY（容灾实例运行状态为备）<br>- IDLE（空闲：在主或备的运行状态时，去使能容灾功能后，处于空闲状态） |
| 主备容灾实例之间的心跳状态 | 该参数用于表示主备容灾实例之间的心跳状态。<br>若输出结果为NULL，可使用工程命令DSP DRMODE查询当前环境的容灾模式。当DRMODE为ACTIVE-STANDBY或LOADBALANCE时，表示当前处于初始态；当DRMODE为NULL时，表示未使能容灾功能。<br>取值说明：<br>- NULL（未使能容灾功能或初始态，其状态无效。）<br>- CONNECTED（主备容灾实例之间握手成功，处于连接状态。）<br>- DISCONNECTED（主备容灾实例之间握手失败，处于断连状态。） |
| 最近一次链接建立时间 | 该参数用于表示最近一次链接建立时间。 |
| 最近一次链接断开时间 | 该参数用于表示最近一次链接断开时间。 |

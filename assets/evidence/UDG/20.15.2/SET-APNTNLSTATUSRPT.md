#### [命令功能](#ZH-CN_CONCEPT_0000202540964602)

**适用NF：SGW-U、PGW-U、UPF**

此命令用于开启或关闭UPF与DN间的隧道链路状态上报功能。

#### [注意事项](#ZH-CN_CONCEPT_0000202540964602)

- 该命令执行后立即生效。
- 系统最多支持配置10000条APNTNLSTATUSRPT。
- 每个APN支持绑定10条探测路径。
- 执行该命令前，必须先使用ADD APN命令配置APN。

#### [操作用户权限](#ZH-CN_CONCEPT_0000202540964602)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000202540964602)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| APN | APN | 可选必选说明：必选参数<br>参数含义：该参数用于指定APN实例名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。只能由“-”、数字、大小写字母和“.”组成，不能以“.”开头且不能出现连续两个“.”。不支持空格及“_”、“#”、“$”、“&”、“%”、“^”、“（”、“）”、“，”、“/”、“;”、“:”、“””、“`”特殊字符，不区分大小写。<br>默认值：无<br>配置原则：该参数使用ADD APN命令配置生成。 |
| SWITCH | 隧道状态上报开关 | 可选必选说明：必选参数<br>参数含义：N4接口上报隧道状态的开关。<br>数据来源：本端规划<br>取值范围：<br>- DISABLE：不使能。<br>- ENABLE：使能。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000202540964602)

当需要设置APN上报隧道状态时，执行：

```
SET APNTNLSTATUSRPT: APN="huawei.com", SWITCH=ENABLE;
```

#### [命令功能](#ZH-CN_CONCEPT_0000203719881178)

**适用NF：PGW-U、UPF**

命令用于显示用户路由可靠性配置。

#### [注意事项](#ZH-CN_CONCEPT_0000203719881178)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000203719881178)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000203719881178)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000203719881178)

显示当前用户路由可靠性配置：

```
LST USERROUTINGSW:;
```

```

RETCODE = 0  操作成功

用户路由可靠性配置
------------------
    告警开关  =  使能
告警上报阈值  =  0
(结果个数 = 1)

---    END
```

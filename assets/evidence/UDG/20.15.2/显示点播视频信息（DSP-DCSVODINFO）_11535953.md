## [命令功能](#ZH-CN_MMLREF_0000002211535953)

该命令用于显示点播视频信息。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000002211535953)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000002211535953)

无

## [使用实例](#ZH-CN_MMLREF_0000002211535953)

显示所有实例的点播视频信息。

```
%%DSP DCSVODINFO:;%%
RETCODE = 0  操作成功

结果如下
------------------------
                      实例ID  =  18378575044282908843
      极热内存区总大小（MB）  =  4960
      极热视频片头大小（MB）  =  15969
            极热视频片头个数  =  4985
        磁盘空间总大小（MB）  =  4990
磁盘空间最大可使用大小（MB）  =  0
  磁盘空间当前使用大小（MB）  =  0
  温热差异区空间总大小（MB）  =  0
    温热差异区视频大小（MB）  =  0
          温热差异区视频个数  =  0
        元数据最大内存（MB）  =  0
元数据当前内存使用大小（MB）  =  0
                  元数据个数  =  0
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000002211535953)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 实例ID | 该参数用于表示SERVER分配的实例ID。 |
| 极热内存区总大小（MB） | 该参数用于表示点播片头内存总大小。 |
| 极热视频片头大小（MB） | 该参数用于表示点播片头缓存大小。 |
| 极热视频片头个数 | 该参数用于表示点播片头缓存个数。 |
| 磁盘空间总大小（MB） | 该参数用于表示磁盘空间总大小。 |
| 磁盘空间最大可使用大小（MB） | 该参数用于表示磁盘可使用最大空间，若使用量超过该阈值，需快速老化磁盘中的点播视频。 |
| 磁盘空间当前使用大小（MB） | 该参数用于表示实际使用磁盘大小。 |
| 温热差异区空间总大小（MB） | 该参数用于表示磁盘中差异区空间总大小。 |
| 温热差异区视频大小（MB） | 该参数用于表示在差异区统计到的视频大小。 |
| 温热差异区视频个数 | 该参数用于表示差异区缓存视频个数。 |
| 元数据最大内存（MB） | 该参数用于表示点播元数据总大小。 |
| 元数据当前内存使用大小（MB） | 该参数用于表示点播元数据当前使用的内存大小。 |
| 元数据个数 | 该参数用于表示当前元数据实际个数。 |

#### [命令功能](#ZH-CN_CONCEPT_0259036438)

该命令用于显示设备驱动信息。

#### [注意事项](#ZH-CN_CONCEPT_0259036438)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0259036438)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0259036438)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RESNAME | 资源名称 | 可选必选说明：必选参数<br>参数含义：该参数表示资源名称，指节点或容器的名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，区分大小写。<br>默认值：无<br>配置原则：使用<br>[**DSP RES**](../../../系统管理/资源管理/资源实例管理/显示资源信息（DSP RES）_59036939.md)<br>查看资源名称。 |

#### [使用实例](#ZH-CN_CONCEPT_0259036438)

```
DSP DEVDRIINFO:RESNAME="OMU1";
```

#### [输出结果说明](#ZH-CN_CONCEPT_0259036438)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 设备名称 | 用于表示设备名称。 |
| 子设备名称 | 用于表示子设备名称。 |
| 设备ID | 用于表示设备ID。 |
| 创建时间 | 用于表示设备的创建时间。 |
| 端口类型 | 用于表示端口类型。<br>- 0：eth类型<br>- 1：veth类型<br>- 2：sub eth类型<br>- 3：local类型 |
| 端口关闭原因 | 用于表示端口关闭原因。<br>- 0：端口未使用<br>- 21：设备打开<br>- 99：设备关闭 |
| 端口状态 | 用于表示端口状态。<br>- 0：关闭。<br>- 1：打开。 |
| 端口ID | 用于表示端口ID。 |
| 已创建 | 用于表示端口设备已创建。<br>- 0：未创建<br>- 1：已创建 |
| 已删除 | 用于表示端口设备已删除。<br>- 0：未删除<br>- 1：已删除 |
| 用户侧设备名称 | 用于表示用户侧设备名称。 |
| 驱动名称 | 用于表示驱动名称。 |
| MAC地址 | 用于表示MAC地址。 |
| 映射状态 | 用于表示端口映射状态。<br>- 0：端口未处于映射状态。<br>- 1：端口处于映射状态。 |
| 关闭 | 用于表示端口是否关闭。<br>- -1：默认值，该输出项未被使用。<br>- 0：关闭。<br>- 1：打开。 |
| 回环 | 用于表示端口是否是回环口。 |
| MTU | 用于表示MTU。<br>- -1：默认值，该输出项未被使用。 |
| 驱动加载类型 | 用于表示驱动加载类型。<br>- 0：netmap类型<br>- 1：dpdk类型 |
| 驱动加载开始时间 | 用于表示驱动加载开始时间。 |
| 驱动加载结束时间 | 用于表示驱动加载结束时间。 |
| 端口状态总变化次数 | 用于表示端口状态总变化次数。 |
| 第一次变化时间 | 用于表示端口状态第一次变化时间。 |
| 第二次变化时间 | 用于表示端口状态第二次变化时间。 |
| 第三次变化时间 | 用于表示端口状态第三次变化时间。 |
| 第四次变化时间 | 用于表示端口状态第四次变化时间。 |
| 第五次变化时间 | 用于表示端口状态第五次变化时间。 |
| 物理端口ID | 用于表示物理端口ID。<br>“-1”<br>为默认值，表示该输出项未被使用。 |
| PAE端口类型 | 用于表示PAE端口类型。 |

| 上次不可用时间 | 用于表示端口上次不可用时间。 |

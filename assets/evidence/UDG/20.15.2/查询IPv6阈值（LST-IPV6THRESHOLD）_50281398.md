#### [命令功能](#ZH-CN_CONCEPT_0000001550281398)

该命令用来查询IPv6整机路由前缀阈值告警的阈值。

#### [注意事项](#ZH-CN_CONCEPT_0000001550281398)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550281398)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550281398)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001550281398)

查询IPv6整机路由前缀阈值告警的阈值：

```
LST IPV6THRESHOLD:;
```

```

RETCODE = 0  操作成功

结果如下
-------------------------
最大百分比（%） =  80
最小百分比（%） =  70
(结果个数 = 1)
---    END
```

## [操作场景](#ZH-CN_OPI_0000001177772003)

为了避免流量突发导致丢包，避免报文重传导致带宽进一步恶化，提高终端用户的业务感受，在 UDG 上配置基于基站对下行用户面流量进行静态整形。

> **说明**
> 适用于 SGW-U、UPF 。

- 完成 **加载license** （如果有需求，请联系华为技术支持工程师处理）。

## [操作步骤](#ZH-CN_OPI_0000001177772003)

1. 打开本特性的License配置开关，使能整形功能。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09587387.md)
2. 配置静态整形带宽。
  [**ADD IPSQMSHAPING**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/IPSQM控制/IPSQM Shaping配置/增加IPSQM整形配置（ADD IPSQMSHAPING）_08873213.md)
3. **可选：**配置整形队列深度。
  [**SET IPSQMQDEPTH**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/IPSQM控制/IPSQM整形队列深度/设置IPSQM缓存队列深度（SET IPSQMQDEPTH）_21392220.md)
4. **可选：**配置整形资源调整的业务处理节点（POD）的最低保护带宽及突发尺寸，缺省使用初始值。
  [**SET IPSQMADJUST**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/IPSQM控制/IPSQM参数调整配置/设置IPSQM调整参数（SET IPSQMADJUST）_08873215.md)
5. **可选：**如果为保障VIP用户的业务体验，可配置VIP用户不做IPSQM流量整形。
  [**ADD IPSQMVIPLIST**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务控制策略/IPSQM控制/IPSQM VIP用户列表/增加IPSQM VIP用户列表（ADD IPSQMVIPLIST）_08873217.md)

## [任务示例](#ZH-CN_OPI_0000001177772003)

任务描述

配置到指定的RAN下行用户面流量的静态整形数据。

脚本

1. 打开本特性的License配置开关，使能整形功能。
  ```
  SET LICENSESWITCH:LICITEM="LKV3G5IPSM01", SWITCH=ENABLE;
  ```
2. 配置静态整形带宽。
  ```
  ADD IPSQMSHAPING: IPTYPE=IPv4, PEERIPV4="192.168.1.235", CIR=100;
  ```
3. 配置静态整形的队列深度。
  ```
  SET IPSQMQDEPTH: DEPTH=256;
  ```

#### [命令功能](#ZH-CN_CONCEPT_0000207394871977)

**适用NF：UPF、PGW-U**

该命令用于增加媒体中继HTTP消息头。

#### [注意事项](#ZH-CN_CONCEPT_0000207394871977)

- 该命令执行后立即生效。
- 该命令最大记录数为4000。
- 对于HEADERVALUE，特殊字符的输入要求：
    - 对于空格必须使用%20代替；例如：hua wei 输入应该是：hua%20wei。
    - 当字符串中出现%20或%25本身时，对于%必须使用%25代替；例如：hua%20wei 输入应该是：hua%2520wei。
    - 对于分号使用\;代替；例如：hua;wei 输入应该是：hua\;wei。
    - 对于双引号使用\"代替；例如：hua"wei 输入应该是：hua\"wei。
    - 对于反斜杠使用\\代替；例如：hua\wei 输入应该是：hua\\wei。
    - 输入“\”只支持上述三种转义字符，不支持其他情况下输入“\”，如果要使用“\”本身，请输入“\\”代替。
    - 不能包含ASCII码值为0x00~0x1F和0x7F的非法字符。

#### [操作用户权限](#ZH-CN_CONCEPT_0000207394871977)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [使用实例](#ZH-CN_CONCEPT_0000207394871977)

增加媒体中继HTTP消息头：

```
ADD RELAYHTTPMHD: HTTPMSGCTRLNAME="http01", MSGTYPE=UEHTTPRSP, HEADERNAME="Age", HEADERVALUE="100";
```

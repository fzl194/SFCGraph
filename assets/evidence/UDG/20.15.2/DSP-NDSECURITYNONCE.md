#### [命令功能](#ZH-CN_CONCEPT_0000001549802290)

该命令用于显示ND安全的随机数。

#### [注意事项](#ZH-CN_CONCEPT_0000001549802290)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549802290)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549802290)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IFNAME | 接口名 | 可选必选说明：必选参数<br>参数含义：该参数用于指定接口名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。<br>默认值：无<br>配置原则：请使用LST INTERFACE命令查看可用接口。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549802290)

显示ND安全随机数：

```
DSP NDSECURITYNONCE:IFNAME="Ethernet65/0/8";
```

```

        RETCODE = 0  操作成功
        结果如下
        -------------------------
        IPv6地址  =  2001:db8::11
          随机数  =  1500
        (结果个数 = 1)
        ---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001549802290)

| 输出项名称 | 输出项解释 |
| --- | --- |
| IPv6地址 | 用于显示IPv6地址。 |
| 随机数 | 用于显示随机数。 |

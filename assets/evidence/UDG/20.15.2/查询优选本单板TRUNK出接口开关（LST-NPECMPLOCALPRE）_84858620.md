#### [命令功能](#ZH-CN_TOPIC_0000002184858620)

该命令用来查询NP优选本单板TRUNK出接口开关。

#### [注意事项](#ZH-CN_TOPIC_0000002184858620)

- 该命令执行后立即生效。
- 该命令仅适用于NP卡加速模式场景。

#### [操作用户权限](#ZH-CN_TOPIC_0000002184858620)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_TOPIC_0000002184858620)

无

#### [使用实例](#ZH-CN_TOPIC_0000002184858620)

查询NP优选本单板TRUNK出接口开关的参数：

```
%%LST NPECMPLOCALPRE:;%%
RETCODE = 0  操作成功

结果如下
--------
关闭优选本单板TRUNK出接口开关  =  否
(结果个数 = 1)

---    END
```

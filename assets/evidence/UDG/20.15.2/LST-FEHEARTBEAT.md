#### [命令功能](#ZH-CN_TOPIC_0000001312463033)

该命令用来查询FE心跳功能开关状态。

#### [注意事项](#ZH-CN_TOPIC_0000001312463033)

- 该命令执行后立即生效。
- 该命令仅适用于NP卡加速模式场景。

#### [操作用户权限](#ZH-CN_TOPIC_0000001312463033)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_TOPIC_0000001312463033)

无

#### [使用实例](#ZH-CN_TOPIC_0000001312463033)

查询FE心跳功能开关状态：

```
LST FEHEARTBEAT:;
RETCODE = 0  操作成功

结果如下
--------
心跳开关  =  ON
(结果个数 = 1)

---    END
```

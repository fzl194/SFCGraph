## [命令功能](#ZH-CN_MMLREF_0000001967609928)

该命令用于查询已有的HTTP状态码判定配置。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000001967609928)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001967609928)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| INDEX | 索引 | 可选必选说明：可选参数<br>参数含义：该参数用于指定HTTPSTATUS配置的索引信息。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是1~1024。<br>默认值：无<br>配置原则：无 |
| STATUS | 状态码 | 可选必选说明：可选参数<br>参数含义：该参数用于指定进行判定的状态码。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是100~699。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001967609928)

查询已有的HTTP状态码判定配置，可以用如下命令：

```
%%LST HTTPSTATUS:;%%
RETCODE = 0  操作成功

结果如下
------------------------
               索引  =  1
               场景  =  ERRNEXTHOP
           配置类型  =  WHITELIST
             状态码  =  604
               描述  =  NULL
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001967609928)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 索引 | 该参数用于指定HTTPSTATUS配置的索引信息。 |
| 场景 | 该参数用于指定特定的场景。 |
| 配置类型 | 该参数用于指定是否执行特定场景的动作。 |
| 状态码 | 该参数用于指定进行判定的状态码。 |
| 描述 | 该参数用于指定HTTPSTATUS配置的描述信息。 |

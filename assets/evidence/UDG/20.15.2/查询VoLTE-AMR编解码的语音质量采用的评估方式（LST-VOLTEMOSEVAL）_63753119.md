#### [命令功能](#ZH-CN_CONCEPT_0000201463753119)

**适用NF：PGW-U**

该命令用于查询VoLTE AMR编解码的语音质量采用的评估方式。

#### [注意事项](#ZH-CN_CONCEPT_0000201463753119)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000201463753119)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000201463753119)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000201463753119)

查询VoLTE AMR编解码的语音质量采用的评估方式：

```
LST VOLTEMOSEVAL:;
```

```

RETCODE = 0 操作成功

显示Mos评估方式
---------------
评估方式 = POLQA_SWB_AMR
(结果个数 = 1)

--- END
```

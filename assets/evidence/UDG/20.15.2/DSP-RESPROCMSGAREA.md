#### [命令功能](#ZH-CN_CONCEPT_0000001157993317)

该命令用于显示进程内的消息分区使用信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001157993317)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001157993317)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001157993317)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RESOURCENAME | 资源名称 | 可选必选说明：可选参数<br>参数含义：用于说明节点名称或容器名称。通过<br>[**DSP RES**](../../../系统管理/资源管理/资源实例管理/显示资源信息（DSP RES）_59036939.md)<br>命令可以查询资源信息。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，区分大小写。<br>默认值：无 |
| PROCID | 进程ID | 可选必选说明：可选参数<br>参数含义：用于说明进程的编号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～4294967295。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001157993317)

显示进程内消息分区的使用情况：

```
DSP RESPROCMSGAREA:RESOURCENAME="OMU1",PROCID=1001;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001157993317)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 进程ID | 用于说明进程的编号。 |
| 资源名称 | 用于说明节点名称或容器名称。通过<br>[**DSP RES**](../../../系统管理/资源管理/资源实例管理/显示资源信息（DSP RES）_59036939.md)<br>命令可以查询资源信息。 |
| 消息分区名 | 用于说明进程内消息分区的名称。 |
| 最大消息单元数目 | 用于说明进程内消息的最大消息单元数目。 |
| 已分配的消息单元数目 | 用于说明此进程内已分配的消息单元数目。 |
| 消息分配百分比（%） | 用于说明进程内消息分配的百分比（单位：%）。 |
| 已使用的消息分区数目 | 用于说明进程内的已经使用的消息分区数。 |
| 消息分区总数 | 用于说明进程内的消息分区的总数目。 |

#### [命令功能](#ZH-CN_CONCEPT_0235373533)

**适用NF：PGW-U、UPF**

该命令用于查看系统的L2TP控制报文统计信息。运营商需要查看控制报文数场景，可以使用该命令。

#### [注意事项](#ZH-CN_CONCEPT_0235373533)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0235373533)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0235373533)

无。

#### [使用实例](#ZH-CN_CONCEPT_0235373533)

假设运营商想查看L2TP控制报文统计信息：

```
DSP L2TPSTATISTICS:;
```

```

RETCODE = 0  操作成功。

结果如下
--------
    收到控制报文数  =  0
收到无效控制报文数  =  0
    发送控制报文数  =  0
发送无效控制报文数  =  0
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0235373533)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 收到控制报文数 | 用于指示收到的控制报文数。 |
| 收到无效控制报文数 | 用于指示收到的无效控制报文数。 |
| 发送控制报文数 | 用于指示发送的控制报文数。 |
| 发送无效控制报文数 | 用于指示发送的无效控制报文数。 |

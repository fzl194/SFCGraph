#### [命令功能](#ZH-CN_CONCEPT_0259103963)

该命令用于显示VNFC系统当前补丁信息，包括补丁包名称、版本号、状态及生效时间。

补丁的加载、激活、删除都会触发补丁状态的变化，通过该命令可及时查询系统当前补丁的状态信息。

#### [注意事项](#ZH-CN_CONCEPT_0259103963)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0259103963)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0259103963)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SERVICEINSTANCE | 服务实例 | 可选必选说明：必选参数<br>参数含义：该参数表示大颗粒服务实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，通过LST VNFC命令获取。<br>默认值：无<br>配置原则：只能填写通过LST VNFC命令查询到的管理代理标识。 |

#### [使用实例](#ZH-CN_CONCEPT_0259103963)

显示补丁信息：

```
DSP PATCHINFO:
SERVICEINSTANCE="vnfc"
;
```

```
RETCODE = 0  操作成功

结果如下:
-------------------------
  补丁单元合集版本号  =  V100R023SPH256
            补丁状态  =  运行

           补丁包名称 =  FENIXV100R023SPH256X8664.PAT
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0259103963)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 补丁单元合集版本号 | 该版本号为制作补丁时若干补丁单元合集的编号，仅用于记录版本补丁的详细信息，用户不用关注。若要查询本产品补丁的版本号，请执行命令<br>[**DSP VERSION**](../../版本信息/显示系统版本信息（DSP VERSION）_59103553.md)<br>。 |
| 补丁状态 | 表示补丁状态，包括空闲、去激活、激活和运行四种状态。 |
| 补丁运行时间 | 表示补丁开始运行的时间点，此输出项仅在补丁为运行状态时存在。 |
| 补丁包名称 | 表示补丁包文件的名称。 |

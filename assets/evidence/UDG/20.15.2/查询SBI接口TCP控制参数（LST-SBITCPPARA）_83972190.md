## [命令功能](#ZH-CN_MMLREF_0000001183972190)

该命令用于查询系统的报文流控阈值数以及TCP连接数。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000001183972190)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001183972190)

无

## [使用实例](#ZH-CN_MMLREF_0000001183972190)

查询SYN包流控阈值和连接数执行如下命令：

```
%%LST SBITCPPARA:;%%
RETCODE = 0 操作成功

结果如下
--------
SYN包流控阈值(pps) = 5000
单个对端IP连接限制数 = 0
(结果个数 = 1)

---   END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001183972190)

| 输出项名称 | 输出项解释 |
| --- | --- |
| SYN包流控阈值(pps) | 该参数用于指定SYN包流控的阈值。 |
| 单个对端IP连接限制数 | 该参数用于当本端作为Server时，单个TCP进程对于单个Client IP地址的连接数限制。 |

#### [命令功能](#ZH-CN_CONCEPT_0000201293531887)

**适用NF：PGW-U、UPF**

此命令用于查询报表消息流控配置。

#### [注意事项](#ZH-CN_CONCEPT_0000201293531887)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000201293531887)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000201293531887)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000201293531887)

假如运营商需要查询报表消息流控配置：

```
%%LST RPTRATELT:;
```

```
%%
RETCODE = 0  操作成功

报表消息流控配置信息
--------------------
        用户信息重发请求消息流控门限  =  100
                用户信息定时重发间隔  =  0
触发UFDR Stats的用户更新消息流控门限  =  300
     SSG主动定时重发用户信息功能开关  =  不使能
     SSG主动定时重发用户信息时间间隔  =  36000
(结果个数 = 1)

---    END
```

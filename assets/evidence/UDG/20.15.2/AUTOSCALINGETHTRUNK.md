#### [命令功能](#ZH-CN_CONCEPT_0000001600841505)

该命令用于查询以太网隧道多虚拟网卡自动化配置模板信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600841505)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600841505)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600841505)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| ETHTRUNKTMPID | 以太Trunk模板ID | 可选必选说明：可选参数<br>参数含义：该参数用来指定以太Trunk模板ID。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～99。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600841505)

查询以太网隧道多虚拟网卡自动化配置模板信息：

```
LST AUTOSCALINGETHTRUNK: ETHTRUNKTMPID=1;
```

```
RETCODE = 0  操作成功

结果如下
-------------------------
以太网Trunk ID  =  1
多个虚拟网卡ID  =  3 4 5
工作模式        =  Lacp
最小激活链路数  = 1
(结果个数 = 1)
---    END
```

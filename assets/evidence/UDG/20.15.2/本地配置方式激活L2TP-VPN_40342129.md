## [操作场景](#ZH-CN_OPI_0240342129)

利用本地配置在 UDG 与企业LNS之间部署L2TP组网。

> **说明**
> 适用于PGW-U、UPF。

## [必备事项](#ZH-CN_OPI_0240342129)

前提条件

- 请仔细阅读 [GWFD-020412 L2TP VPN](../../GWFD-020412 L2TP VPN_40342126.md) 。
- 如果运营商规划采用VPN组网方式，则操作员在执行本操作前应已创建了相应的VPN实例。
- 完成 **加载license** （如果有需求，请联系华为技术支持工程师处理）。

## [操作步骤](#ZH-CN_OPI_0240342129)

1. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09587387.md)
2. 配置VPN实例。
  [**ADD VPNINST**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/DN管理/VPN管理/VPN/增加VPN实例（ADD VPNINST）_82837045.md)
3. 配置L2TP组信息。
  [**ADD L2TPGROUP**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/L2TP隧道管理/L2TP组/创建L2TP组（ADD L2TPGROUP）_35373524.md)
  > **说明**
  > - 创建L2TP Group后，不仅可以在UDG上灵活地配置L2TP各项功能，而且能够方便地实现LAC和LNS之间一对一、一对多的组网应用。L2TP Group在LAC和LNS上独立编号，只需要保证LAC和LNS之间关联的L2TP Group的相关配置保持对应关系即可。
  > - L2TP Group绑定到APN下。当APN下有用户接入时，UDG与该APN绑定的L2TP Group下的LNS中选择LNS建立一条L2TP隧道（主备方式下优先选择主LNS，负荷分担方式下轮选一个LNS），为APN下的每一个用户分别建立一个L2TP会话。每条L2TP隧道承载的L2TP会话个数的上限可通过“MAXSESSIONNUM”参数配置，当L2TP隧道上接入的用户数超过该隧道承载的会话数上限时，UDG与该LNS另外新建一条L2TP隧道。
  > - UDG上可以配置主备LNS，并优先与主LNS创建L2TP隧道，用于传输业务报文。主备模式下最多支持2个LNS服务器，如果配置的LNS服务器大于2个，而又存在优先级不同的LNS服务器时，取优先级最高的两个服务器做主备。主LNS故障时，产生[ALM-81053 LNS无响应](../../../../../../网络运维/故障处理/用户面告警/ALM-81053 LNS无响应_93167101.md)告警，UDG与备LNS建立L2TP隧道。当[ALM-81053 LNS无响应](../../../../../../网络运维/故障处理/用户面告警/ALM-81053 LNS无响应_93167101.md)告警恢复，UDG会尝试将新用户的业务报文发送至主LNS处理。如果主LNS能够处理，则后续新上线用户的业务报文均发送至主LNS处理，备LNS仅处理原有用户的业务报文，直至用户全部下线，备L2TP隧道拆除。
  > - 当L2TP组中有6个以上LNS或对接的LNS自身的地址是IPv6时，需要[**ADD L2TPLNSINFO**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/L2TP隧道管理/L2tp组Lns信息/添加LNS与L2TP组绑定关系（ADD L2TPLNSINFO）_55603814.md)命令配置。
4. 配置L2TP的缺省参数。
  [**SET GLOBALL2TP**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/L2TP隧道管理/L2TP缺省配置/设置L2TP配置（SET GLOBALL2TP）_35373521.md)
  > **说明**
  > - 创建L2TP隧道有两种方式：一是根据鉴权服务器返回的L2TP属性创建；二是根据本地配置的L2TP组创建。
  > - 当PGW-C/SMF下发LNS信息，即AAA Server返回L2TP属性时，PGW-U/UPF根据PGW-C/SMF下发的由鉴权服务器返回的L2TP属性创建L2TP隧道。AAA返回的L2TP属性中不包含Client-Auth-ID（即LAC名称）属性场景，系统将使用[**SET GLOBALL2TP**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/L2TP隧道管理/L2TP缺省配置/设置L2TP配置（SET GLOBALL2TP）_35373521.md)设置的缺省本端名称发起隧道连接；隧道创建后将使用[**SET GLOBALL2TP**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/L2TP隧道管理/L2TP缺省配置/设置L2TP配置（SET GLOBALL2TP）_35373521.md)设置的缺省HELLO报文重发间隔发送HELLO报文以检测隧道的连通性。
  > - 当PGW-C/SMF未下发LNS信息，即AAA Server没有为用户设置L2TP的相关属性，则PGW-U/UPF根据本地配置的LNS信息创建L2TP隧道。如果用户所在APN已使能L2TP功能，并且用户名中携带的域名匹配L2TP组中配置的域名时，将根据本地配置的L2TP组创建L2TP隧道。L2TP组中没有配置本端名称场景，系统将使用[**SET GLOBALL2TP**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/L2TP隧道管理/L2TP缺省配置/设置L2TP配置（SET GLOBALL2TP）_35373521.md)设置的缺省本端名称发起隧道连接。
5. 设置APN的L2TP相关信息。
  [**SET APNL2TPATTR**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/L2TP隧道管理/APN的L2TP属性/设置APN L2TP配置（SET APNL2TPATTR）_35373518.md)
6. 关闭快速流表功能，将Byte671的bit7设置为1。
  [**SET SOFTPARAOFBIT**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/业务匹配策略/业务匹配公共配置/业务软参管理/软件参数比特位/设置软件参数表比特位的值（SET SOFTPARAOFBIT）_82837317.md)
7. 配置Giif接口。
  [**ADD LOGICINF**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/DN管理/逻辑接口管理/逻辑接口配置/接口/增加逻辑接口（ADD LOGICINF）_82835378.md)
8. 将L2TP组绑定指定源端接口。
  [**ADD L2TPCLIENTIP**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/L2TP隧道管理/L2TP组绑定源端接口/增加L2TP绑定的接口（ADD L2TPCLIENTIP）_35373536.md)
9. **可选** ：配置系统进行PPP协商时所使用的参数。
  [**SET PPPCFG**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/L2TP隧道管理/PPP配置/设置PPP配置（SET PPPCFG）_35373544.md)
10. **可选** ：配置指定APN支持PPP鉴权功能。
  [**SET APNPPPACCESS**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/L2TP隧道管理/PPP接入/设置APN PPP配置（SET APNPPPACCESS）_35373547.md)
11. **可选** ：配置L2TP业务加密功能，增强N4接口传输安全。
  [**SET L2TPN4KEY**](../../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/L2TP隧道管理/L2TP N4密码/设置L2TP N4密码配置（SET L2TPN4KEY）_64015280.md)

  > **说明**
  > 对端SMF需同时配置该功能，并且配置相同密钥，否则会导致L2TP业务失败。

## [任务示例](#ZH-CN_OPI_0240342129)

任务描述

- 选择本地配置方式，启用L2TP。
- 将 UDG 配置为LAC。
- 配置LNS相关信息。
- 实现LAC与LNS的互通。
- （可选配置）配置L2TP加密功能。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH:LICITEM="LKV3G5L2TP01",SWITCH=ENABLE;
```

//配置VPN实例。

```
ADD VPNINST:VPNINSTANCE="vpn_l2tp";
```

//配置L2TP组信息。配置L2TP组的组号、域名、本端主机名、鉴权、HELLO报文间隔、magic number协商、L2TP隧道发送窗口大小的上限、L2TP隧道绑定的VPN实例、LNS服务器。

```
ADD L2TPGROUP: GROUPID=1, DOMAINNAME="example.com", LOCALNAME="UPF", AVPHIDDEN=ENABLE, HELLOINTERVALSW=ENABLE, HELLOINTERVAL=120, VPNINSTANCE="vpn_l2tp", LOCALLNSMODE=REDUNDANCY, FIRSTLNSIP="10.10.10.1", FIRSTPWD="0123456", SECONDLNSIP="10.10.10.2", SECONDPWD="0123456", CFMFIRSTPWD="0123456", CFMSECONDPWD="0123456", PPPMAGICNUMBER=ENABLE, MAXSENDWINSIZE=32;
```

//配置L2TP的缺省参数，使能本地配置方式启用L2TP。

```
SET GLOBALL2TP:LOCALNAME="huawei",HELLOINTERVALSW=ENABLE,HELLOINTERVAL=60,RETRYTIMES=3;
```

//设置APN的L2TP相关信息，指定APN下绑定L2TP组。

```
ADD APN: APN="apn-l2tp";
SET APNL2TPATTR:APN="apn-l2tp",L2TPSWITCH=ENABLE,SUPPORTIPV6=DISABLE,L2TPGROUPID=1,ICRQ_CALLINGNO=MSISDN,ICCN_AUTH=ENABLE,IPCP_NEGO=ENABLE,DOMAINNAMEACT=ADD_ENABLE_STRIP_DISABLE,DOMAINNAMEPOS=PREFIX;
```

//关闭快速流表功能。

```
SET SOFTPARAOFBIT: DT2=BYTE, BYTENUM=671, BYTEPOSITION=7, BYTEVALUE=1;
```

//配置Giif接口。

```
ADD LOGICINF:NAME="giif1/0/0",IPVERSION=IPV4,IPV4ADDRESS1="10.8.20.1",IPV4MASK1="255.255.255.255",VPNINSTANCE="vpn_l2tp";
```

//将L2TP组绑定指定源端接口。

```
ADD L2TPCLIENTIP:L2TPGROUPID=1,INTERFACENAME="giif1/0/0";
```

//系统进行PPP协商时所使用的参数。

```
SET PPPCFG:HOSTNAME="UPF",MRU=1500,TIMEOUT=3;
```

//设置APN的PPP相关信息，支持PPP鉴权功能。

```
SET APNPPPACCESS: APN="apn-l2tp", AUTHENTICATION=ENABLE;
```

//（可选配置）设置L2TP业务加密。

```
SET L2TPN4KEY: N4KEYVALUE="*****", CFMN4KEYVALUE="*****";
```

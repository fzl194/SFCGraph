#### [命令功能](#ZH-CN_CONCEPT_0000001549961798)

该命令用于查询OSPFv3区域配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001549961798)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549961798)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549961798)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | OSPFv3进程号 | 可选必选说明：可选参数<br>参数含义：OSPFv3进程号。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |
| AREAID | OSPFv3区域标识 | 可选必选说明：可选参数<br>参数含义：OSPFv3区域标识。<br>数据来源：本端规划<br>取值范围：IPv4地址类型。点分十进制格式。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549961798)

查询OSPFv3区域配置：

```
LST OSPFV3AREA:;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001549961798)

| 输出项名称 | 输出项解释 |
| --- | --- |
| OSPFv3进程号 | 该参数指定用户创建的OSPFv3进程号。 |
| OSPFv3区域标识 | 该参数指定用户创建的OSPFv3区域号。 |
| OSPFv3区域描述 | 该参数指定OSPFv3区域的描述信息。 |
| OSPFv3区域类型 | 该参数指定OSPFv3区域类型。Normal表示普通区域；Stub表示Stub区域，不引入外部路由；Nssa表示Nssa区域，可以引入外部路由并传播到整个NSSA区域，但不会学习外部路由。 |
| 禁止ABR向Stub区域内发送Summary LSA | 该参数指定禁止ABR向Stub区域内发送其他区域的Summary LSA。 |
| 发送到Stub或NSSA区域的Type3缺省路由的开销 | 该参数指定OSPFv3发送到Stub或NSSA区域的Type 3缺省路由的开销。 |
| IPsec SA名称 | 该参数指定区域下IPsec SA的名称。 |
| 产生缺省7类LSA到NSSA区域 | 该参数指定配置在ABR或者ASBR上产生缺省的Type 7 LSA到NSSA区域。 |
| 指定缺省开销值 | 该参数指定是否配置缺省开销值。 |
| 缺省开销值 | 该参数指定缺省开销值。 |
| 指定缺省路由的tag值 | 该参数指定是否配置缺省路由的tag值。 |
| 缺省路由的tag值 | 该参数指定缺省路由的tag值。 |
| 指定缺省路由的类型 | 该参数指定是否配置缺省路由的类型。 |
| 缺省路由类型 | 该参数指定缺省路由类型。 |
| 不引入外部路由 | 该参数指定禁止在NSSA区域引入外部路由。 |
| 禁止ABR向NSSA区域内发送Summary LSA | 该参数指定禁止ABR向NSSA区域内发送其他区域的Summary LSA。 |
| 在DD报文中配置N-bit位 | 该参数指定在DD报文中配置N-bit位的标志。 |
| 指定为转换路由器 | 该参数指定允许将NSSA区域中的多个ABR配置成转换路由器。 |
| 转换路由器的失效时间（s） | 该参数指定转换路由器的失效时间，仅当区域类型配置为NSSA时生效。 |

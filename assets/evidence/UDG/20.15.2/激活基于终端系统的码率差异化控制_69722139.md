## [操作步骤](#ZH-CN_OPI_0169722139)

1. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09587387.md)
2. 配置BWM业务。
    a. 用OSTYPE来定义BWM业务。

UDG 对同一用户组下的IOS系统用户的上行、下行用户面数据流量进行流量监管。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH:LICITEM="LKV3G5RDSC01",SWITCH=ENABLE;
```

//配置BWM业务。

```
ADD BWMSERVICE:BWMSERVICENAME="bs_test",BWMSERVICETYPE=NONTOS,NONTOSSRVTYPE=PROTOCOL,PROTOCOLNAME="http",OSTYPE=IOS;
```

```
ADD BWMCONTROLLER:BWMCNAME="bc-up",CTRLTYPE=CAR,CIR=300,CBS=10000,PIR=600,PBS=12000,GREEN=PASS,YELLOW=PASS,RED=DISCARD;

```
ADD BWMCONTROLLER:BWMCNAME="bc-down",CTRLTYPE=CAR,CIR=300,CBS=10000,PIR=600,PBS=12000,GREEN=PASS,YELLOW=PASS,RED=DISCARD;

```
SET BANDWIDTHMNG:RANGE=SYSTEM;

ADD BWMUSERGROUP:USERGROUPTYPE=SPECIFIC_GROUP,USERGROUPNAME="ug_test",USERGROUPPRI=1,USERGLEVSRVTYPE=NONTOS;

```
ADD BWMRULE:USERGROUPTYPE=SPECIFIC_GROUP,USERGROUPNAME="ug_test",BWMRULETYPE=GROUP_SPECIFIC,BWMRULENAME="br_test",BWMSERVICENAME="bs_test",UPBWMCTRLNAME1="bc-up",DNBWMCTRLNAME1="bc-down",BWMRULEPRI=1;

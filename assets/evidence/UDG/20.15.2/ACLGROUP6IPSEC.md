## [命令功能](#ZH-CN_MMLREF_0000001121361324)

该命令用于查询系统内已经配置的ACL规则组的信息。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000001121361324)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001121361324)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| ACLNAME | 规则组标识 | 可选必选说明：可选参数<br>参数含义：该参数用于指定规则组名称或是编号。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~32。不支持空格，区分大小写。以英文字母a～z或A～Z开始，可以是英文字母、数字、连字符“-”、下划线“_”或中文字符的组合。整数形式，取值范围是3000～3999（高级ACL），但兼容CE版本，允许配置其他数值但不会生效。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001121361324)

查询当前设备上所有的规则组信息，不输入ACLNAME：

```
RETCODE = 0  操作成功。

结果如下
--------
ACL规则组标识    规则组步长    规则组类型    规则的匹配顺序                    规则组描述

3000             5             高级ACL       规则组下的规则按照配置优先排序    NULL    
3005             5             高级ACL       规则组下的规则按照配置优先排序    NULL      
(结果个数 = 4)
---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001121361324)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 规则组标识 | 该参数用于指定规则组名称或是编号。 |
| 规则组步长 | 该参数用于指定规则组规则步长，仅当不指定规则ID时生效。 |
| 规则组类型 | 该参数用于指定规则组类型。如果不输入该参数，当参数ACLNAME为非数字时，该参数默认为高级ACL。 |
| 规则的匹配顺序 | 该参数用于指定规则的匹配顺序是深度优先还是配置优先。 |
| 规则组描述 | 该参数用于指定ACL规则组描述信息。 |

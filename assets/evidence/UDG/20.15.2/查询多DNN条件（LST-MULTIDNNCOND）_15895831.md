#### [命令功能](#ZH-CN_CONCEPT_0000206415895831)

**适用NF：UPF**

该命令用于查询多DNN条件。

#### [注意事项](#ZH-CN_CONCEPT_0000206415895831)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000206415895831)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000206415895831)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000206415895831)

假如运营商需要查询多DNN条件：

```
LST MULTIDNNCOND:;
```

```

RETCODE = 0  操作成功

多DNN条件信息
-------------
DNN条件类型  =  DNN名称以DNNCONDVAL结束
  DNN条件值  =  dnn1
(结果个数 = 1)
```

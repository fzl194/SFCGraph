#### [命令功能](#ZH-CN_CONCEPT_0000207114777349)

**适用NF：PGW-U、UPF**

该命令用于显示媒体中继IP信息。

#### [注意事项](#ZH-CN_CONCEPT_0000207114777349)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000207114777349)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000207114777349)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RELAYIPNAME | 媒体中继IP地址名称 | 可选必选说明：可选参数<br>参数含义：该参数用于表示媒体中继服务IP配置。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。不支持空格，不区分大小写。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000207114777349)

显示媒体中继IP地址信息：

```
LST RELAYIPINFO: RELAYIPNAME="test";
```

```

RETCODE = 0  操作成功。

结果如下
------------------------
媒体中继IP地址名称  =  test
	   IP业务类型  =  组级IP
	      实例ID  =  0
		IPv4地址  =  192.168.1.1
		IPv6地址  =  ::
	   VPN实例名称 = test_vpn
		配置域名称 = testDomain
(结果个数 = 1)

---    END
```

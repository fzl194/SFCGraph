#### [命令功能](#ZH-CN_TOPIC_0255136286)

该命令可以查询已加载的扩展域组件包名称、组件版本、当前组件包状态信息。

#### [注意事项](#ZH-CN_TOPIC_0255136286)

无。

#### [参数说明](#ZH-CN_TOPIC_0255136286)

| **参数标识** | **参数名称** | **参数说明** |
| --- | --- | --- |
| MEID | 网元ID | 可选必选说明：必选参数。<br>参数含义：网元ID，可以通过<br>[**LST ME**](../../系统管理/版本信息/查询网元配置信息（LST ME）_47084797.md)<br>获取。<br>取值范围：0～65535<br>默认值：无。<br>配置原则：无。 |

#### [使用实例](#ZH-CN_TOPIC_0255136286)

查询网元ID为219的网元信息。

```
%%LST COMPKG: MEID=219;%%
RETCODE = 0 操作成功

操作结果如下
------------
网元类型  =  APPTYPE
网元ID    =  219
组件名称  =  APP_TYPE_XXX_V100200300400
组件版本  =  V100200300400
组件状态  =  正常
(结果个数 = 1)

---    END
```

#### [输出结果说明](#ZH-CN_TOPIC_0255136286)

命令执行正常，会返回命令执行成功的提示信息及网元组件信息。输出结果说明如 [表1](#ZH-CN_TOPIC_0255136286__table1331193282114) 所示。

该命令执行异常，会返回对应的错误码。常见错误码如 [表2](#ZH-CN_TOPIC_0255136286__table14686130124019) 所示。

*表1 输出项说明*

| 输出项名称 | 输出项解释 |
| --- | --- |
| 网元类型 | 用于返回查询的网元类型。 |
| 网元ID | 用于返回查询的网元ID。 |
| 组件名称 | 用于返回查询的组件包名称。 |
| 组件版本 | 用于返回查询的组件包版本。 |
| 组件状态 | 用于返回查询的组件包状态。 |

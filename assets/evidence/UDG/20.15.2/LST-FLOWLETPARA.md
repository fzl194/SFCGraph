## [命令功能](#ZH-CN_MMLREF_0000206882901202)

**适用NF：PGW-U、UPF、SGW-U**

该命令用于查询大流优化参数。

> **说明**
> 最多可输入1条记录。

#### [操作用户权限](#ZH-CN_MMLREF_0000206882901202)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000206882901202)

无

## [使用实例](#ZH-CN_MMLREF_0000206882901202)

1. 查询大流优化参数：
  ```
  LST FLOWLETPARA:;
  RETCODE = 0  操作成功
  结果如下：
  ----------
               大流优化功能  =  关闭大流优化
           线程负载更新间隔  =  256
               大流检测速率  =  78000
         大流优化保护报文数  =  64
         大流优化的保护时间  =  10
      启动优化的线程CPU阈值  =  80
  启动优化的线程CPU偏差阈值  =  15
  (结果个数 = 1)
  ---    END
  ```

## [输出结果说明](#ZH-CN_MMLREF_0000206882901202)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 大流优化功能 | 该参数用于配置大流优化功能开关。<br>取值说明：<br>- "DISABLE(关闭大流优化)"：关闭大流优化<br>- "ENABLE(开启大流优化)"：开启大流优化<br>- "FLOWCHK(开启大流检测)"：开启大流检测 |
| 线程负载更新间隔 | 该参数表示线程CPU负载更新周期。 |
| 大流检测速率 | 该参数设置大流检测速率，当某个业务访问的速率超过参数时，会认定为大象流。 |
| 大流优化保护报文数 | 该参数表示启动大流优化后，两次优化间的切换报文数。 |
| 大流优化的保护时间 | 该参数表示启动大流优化后，两次优化间的切换保护时间。 |
| 启动优化的线程CPU阈值 | 该参数用于配置启动大流优化线程CPU阈值。当线程CPU占用率同时满足优化启动阈值和偏差阈值时，启动大流优化。 |
| 启动优化的线程CPU偏差阈值 | 该参数用于配置启动大流优化线程CPU偏差阈值。线程CPU偏差阈值表示线程CPU占用率与进程CPU平均占用率的差值，当线程CPU占用率同时满足优化启动阈值和偏差阈值时，启动大流优化。 |

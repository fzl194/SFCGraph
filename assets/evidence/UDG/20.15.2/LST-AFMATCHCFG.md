#### [命令功能](#ZH-CN_CONCEPT_0186526224)

**适用NF：PGW-U、UPF**

该命令用来查询软参场景欺诈统计功能开关。

#### [注意事项](#ZH-CN_CONCEPT_0186526224)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0186526224)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0186526224)

无。

#### [使用实例](#ZH-CN_CONCEPT_0186526224)

查询软参场景欺诈统计功能配置信息：

```
LST AFMATCHCFG:;
```

```

RETCODE = 0  操作成功。

软参欺诈场景开关状态
--------------------
软参欺诈场景功能开关  =  使能
(结果个数 = 1)
---    END
```

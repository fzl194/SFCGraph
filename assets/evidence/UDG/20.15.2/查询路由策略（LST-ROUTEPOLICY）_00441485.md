#### [命令功能](#ZH-CN_CONCEPT_0000001600441485)

该命令用于查询配置的路由策略。

#### [注意事项](#ZH-CN_CONCEPT_0000001600441485)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600441485)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600441485)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001600441485)

查询所有配置的路由策略信息：

```
LST ROUTEPOLICY:;
```

```

RETCODE = 0  操作成功

结果如下
-------------------------
路由策略名字  =  a
(结果个数 = 1)
---    END
```

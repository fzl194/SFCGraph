#### [命令功能](#ZH-CN_CONCEPT_0235373551)

**适用NF：SGW-U、PGW-U、UPF**

该命令用于查看已设置的前缀分割符和后缀分割符。

#### [注意事项](#ZH-CN_CONCEPT_0235373551)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0235373551)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0235373551)

无。

#### [使用实例](#ZH-CN_CONCEPT_0235373551)

查询已设置的前缀分隔符和后缀分隔符的配置信息：

```
LST DOMAINSEPARATOR:;
```

```

RETCODE = 0  操作成功。

结果如下
--------
前缀域名分隔符  =  @%
后缀域名分隔符  =  #/
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0235373551)

参见SET DOMAINSEPARATOR的参数说明。

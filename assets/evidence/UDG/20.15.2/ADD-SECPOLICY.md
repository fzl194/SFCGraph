#### [命令功能](#ZH-CN_CONCEPT_0000001600866361)

该命令用于添加安全策略。

#### [注意事项](#ZH-CN_CONCEPT_0000001600866361)

- 该命令执行后立即生效。
- 该命令最大记录数为30。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600866361)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600866361)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| POLICYID | 安全策略编号 | 可选必选说明：必选参数<br>参数含义：策略编号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～30。<br>默认值：无 |
| DESCRIPTION | 策略描述 | 可选必选说明：可选参数<br>参数含义：防攻击策略描述。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～64。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600866361)

添加安全策略：

```
ADD SECPOLICY:POLICYID=1;
```

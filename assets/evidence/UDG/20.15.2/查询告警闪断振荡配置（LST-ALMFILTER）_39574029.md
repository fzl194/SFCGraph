## [命令功能](#ZH-CN_MMLREF_0000001139574029)

本命令用于查询系统内告警的闪断振荡配置信息。

> **说明**
> 告警的闪断振荡配置信息只对未屏蔽告警有效。对于被屏蔽的告警，闪断振荡配置信息实际中不起作用。

> **说明**
> 默认查询64条，最大支持1000条告警。

## [参数说明](#ZH-CN_MMLREF_0000001139574029)

| **参数标识** | **参数名称** | **参数说明** |
| --- | --- | --- |
| MEID | 网元ID | 可选必选说明：可选参数。<br>参数含义：用于指示系统需要按照哪个网元来查询。若不输入，则表示查询所有网元的告警。<br>取值范围：0~65535<br>默认值：无。<br>配置原则：网元ID可以通过<br>[**LST ME**](../../系统管理/版本信息/查询网元配置信息（LST ME）_47084797.md)<br>命令查询。 |
| SAID | 起始告警ID | 可选必选说明：可选参数。<br>参数含义：用于指示系统查询时，告警ID的取值下限；若不输入，则表示取值下限<br>**大于等于**<br>**0**<br>且<br>**小于等于**<br>结束告警ID。<br>取值范围：0~999999999<br>默认值：无。<br>配置原则：无。 |
| EAID | 结束告警ID | 可选必选说明：可选参数。<br>参数含义：用于指示系统查询时，告警ID的取值上限；若不输入，则表示取值上限<br>**小于等于999999999**<br>且<br>**大于等于**<br>起始告警ID。<br>取值范围：0~999999999<br>默认值：无。<br>配置原则：无。 |
| CNT | 返回记录数 | 可选必选说明：可选参数。<br>参数含义：用于指示系统返回记录条数的最大值；若不输入，则表示返回记录的最大数目是64。<br>取值范围：1~1000<br>默认值：无。<br>配置原则：无。 |

## [使用实例](#ZH-CN_MMLREF_0000001139574029)

查询告警闪断振荡配置：

```
%%LST ALMFILTER: MEID=0, SAID=5521, EAID=5521, CNT=1;%%
RETCODE = 0  操作成功
```

## [输出结果说明](#ZH-CN_MMLREF_0000001139574029)

命令执行正常，会返回命令执行成功的提示信息，输出结果说明如 [表1](#ZH-CN_MMLREF_0000001139574029__table47787675) 所示。

命令执行异常，会返回提示对应的失败信息。错误码列表如 [表2](#ZH-CN_MMLREF_0000001139574029__table14435102613411) 所示。

*表1 输出项解释*

| **输出项名称** | **输出项解释** |
| --- | --- |
| 网元ID | 用于表示产生告警的网元对应的ID。 |
| 网元名称 | 用于表示产生告警的网元名称。 |
| 告警ID | 用于表示产生告警的ID。 |
| 告警名称 | 用于表示产生告警的名称。 |
| 抑制类型 | 用于表示该告警被抑制的类型。包含未抑制、闪断抑制、振荡抑制、闪断振荡抑制。<br>- “未抑制”：表示系统不抑制告警的上报。<br>- “闪断抑制”：告警闪断指一条告警的产生时间与恢复时间的间隔小于某一特定值（“闪断周期（秒）”），则不在告警页面显示该告警，系统通过“闪断周期（秒）”抑制告警闪断。<br>- “振荡抑制”：告警振荡指在单位时间内（“振荡产生周期（秒）”），同一对象的同一告警（即告警产生对象和告警ID都相同的告警）交替产生/恢复的次数大于某一特定值（“振荡阈值”），则该告警进入振荡抑制，新上报的告警不再显示在告警页面。进入振荡抑制的告警，在指定时间内（“振荡恢复周期（秒）”），不再继续交替产生/恢复，则退出振荡抑制，新上报的告警可显示在告警页面。<br>- “闪断振荡抑制”：系统根据闪断周期、振荡产生周期、振荡恢复周期、振荡阈值等参数对上报的告警进行检查和抑制，检测顺序为先检测振荡告警，不满足则检测闪断告警。 |
| 闪断周期（秒） | 用于指定检测告警是否产生闪断的时间值。闪断周期必须小于振荡产生周期。 |
| 振荡产生周期（秒） | 用于指定检测告警是否产生振荡的时间段。振荡产生周期必须大于闪断周期。 |
| 振荡恢复周期（秒） | 用于指定检测一个告警的振荡是否结束的时间段。 |
| 振荡阈值（次） | 用于指定在振荡产生周期内，判断告警是否产生振荡的故障告警和清除告警交替产生的次数。 |
| 符合查询条件的总记录数 | 用于表示符合查询条件的总记录数。 |
| 实际返回记录数 | 用于表示实际返回记录数。 |

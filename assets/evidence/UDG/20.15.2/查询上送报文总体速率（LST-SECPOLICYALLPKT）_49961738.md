#### [命令功能](#ZH-CN_CONCEPT_0000001549961738)

该命令用来查询上送报文总体速率设置。

#### [注意事项](#ZH-CN_CONCEPT_0000001549961738)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549961738)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549961738)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SECPOLICYID | 安全策略编号 | 可选必选说明：可选参数<br>参数含义：策略编号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～30。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549961738)

查询上送报文总体速率设置：

```
LST SECPOLICYALLPKT:SECPOLICYID=1;
```

```

RETCODE = 0  操作成功

结果如下
-------------------------
  安全策略编号    =  1
上报速率值（pps） =  6000
  上报速率等级    =  高
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001549961738)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 安全策略编号 | 该参数表示策略号。 |
| 上报速率值（pps） | 该参数表示速度值，单位为pps。 |
| 上报速率等级 | 该参数表示速度等级。 |

#### [命令功能](#ZH-CN_CONCEPT_0000203150950414)

**适用NF：PGW-U、UPF**

该命令用于查询内容过滤本地缓存参数。

#### [注意事项](#ZH-CN_CONCEPT_0000203150950414)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000203150950414)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000203150950414)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000203150950414)

查询内容过滤本地缓存参数：

```
LST CFCACHEPARA:;
```

```

RETCODE = 0  操作成功
 
内容过滤本地缓存参数
--------------------
本地缓存过期时间（秒）  =  3600
        缓存开关  =  使能（开启）
(结果个数 = 1)
 
---    END
```

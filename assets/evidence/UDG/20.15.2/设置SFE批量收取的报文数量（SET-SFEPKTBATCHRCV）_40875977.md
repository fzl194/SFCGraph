#### [命令功能](#ZH-CN_TOPIC_0000001840875977)

![](设置SFE批量收取的报文数量（SET SFEPKTBATCHRCV）_40875977.assets/notice_3.0-zh-cn.png)

如果设置的批量收取的报文数量不合理，可能会导致VNRS侧转发性能下降。

该命令用来设置SFE批量收取的报文数量。

#### [注意事项](#ZH-CN_TOPIC_0000001840875977)

- 该命令仅适用于非池化场景。
- 该命令存在系统初始记录，参数的初始设置值如下表：
  | PKTNUM |
  | --- |
  | 128 |

#### [操作用户权限](#ZH-CN_TOPIC_0000001840875977)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_TOPIC_0000001840875977)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PKTNUM | 批量收取报文数量 | 可选必选说明：必选参数<br>参数含义：该参数用于表示SFE批量收取的报文数量信息。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～128。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_TOPIC_0000001840875977)

```
设置SFE批量收取的报文数量为16：
SET SFEPKTBATCHRCV: PKTNUM=16;
```

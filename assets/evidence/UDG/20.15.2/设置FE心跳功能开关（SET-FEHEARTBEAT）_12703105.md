#### [命令功能](#ZH-CN_TOPIC_0000001312703105)

该命令用来设置FE心跳功能开关。

#### [注意事项](#ZH-CN_TOPIC_0000001312703105)

- 该命令执行后立即生效。
- 该命令仅适用于NP卡加速模式场景。
- 系统部署完成后，已经存在初始记录，参数的初始记录值如下表：
  | FEHBSW |
  | --- |
  | ON |

#### [操作用户权限](#ZH-CN_TOPIC_0000001312703105)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_TOPIC_0000001312703105)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| FEHBSW | 心跳开关 | 可选必选说明：必选参数<br>参数含义：该参数用于设置心跳功能开关。<br>数据来源：本端规划<br>取值范围：<br>“OFF”：表示关闭心跳开关。<br>“ON”：表示打开心跳开关。<br>默认值：无。<br>配置原则：建议值为“ON”。 |

#### [使用实例](#ZH-CN_TOPIC_0000001312703105)

设置FE心跳功能开关为启用：

```
SET FEHEARTBEAT: FEHBSW=ON;
```

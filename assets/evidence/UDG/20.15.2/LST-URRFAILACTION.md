#### [命令功能](#ZH-CN_CONCEPT_0242746596)

**适用NF：UPF**

本条命令用于查询计费URR上报失败的动作。

#### [注意事项](#ZH-CN_CONCEPT_0242746596)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0242746596)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0242746596)

无。

#### [使用实例](#ZH-CN_CONCEPT_0242746596)

查询URR上报失败动作：

```
LST URRFAILACTION:;
```

```

RETCODE = 0  操作成功

配额申请失败动作信息
--------------------
        重发间隔  =  60
        重发次数  =  2
重发失败后的动作  =  阻塞
        保持时长  =  30
(结果个数 = 1)
---    END
```

#### [命令功能](#ZH-CN_CONCEPT_0000001600866021)

该命令用于查询IP转发负载分担hash计算模式。

#### [注意事项](#ZH-CN_CONCEPT_0000001600866021)

- 该命令执行后立即生效。
- 该命令仅适用于非NP卡场景和NP卡非加速模式场景。
- 该命令仅支持发往Fabric口方向的报文做负载分担hash。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600866021)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600866021)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001600866021)

查询负载分担hash计算模式：

```
LST SFELBHASH:;
```

```

RETCODE = 0 操作成功。

结果如下
-------------------------
  报文类型  = IP报文
  hash类型  = 三元组
(结果个数 = 1)
--- 结束
```

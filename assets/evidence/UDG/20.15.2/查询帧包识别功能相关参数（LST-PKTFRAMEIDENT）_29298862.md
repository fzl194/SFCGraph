#### [命令功能](#ZH-CN_CONCEPT_0000204929298862)

**适用NF：UPF、PGW-U**

该命令用于显示帧包识别功能相关参数。

#### [注意事项](#ZH-CN_CONCEPT_0000204929298862)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000204929298862)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000204929298862)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000204929298862)

假如运营商希望查看帧包识别相关参数：

```
LST PKTFRAMEIDENT:;
```

```

RETCODE = 0  操作成功
 
帧包识别参数
------------
      帧包识别功能开关  =  使能（开启）
帧包关系学习时间（秒）  =  3
    关键帧识别功能开关  =  不使能（关闭）
(结果个数 = 1)
 
---    END
```

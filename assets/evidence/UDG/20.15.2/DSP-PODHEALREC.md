## [命令功能](#ZH-CN_MMLREF_0263673345)

该命令用于查询POD重建历史记录。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0263673345)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0263673345)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PODNAME | POD名称 | 可选必选说明：可选参数<br>参数含义：该参数用于显示重建的POD名称。如果不输入该输出，则查询系统内所有POD的重建记录。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~128。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0263673345)

查询POD重建历史记录。

```
%%DSP PODHEALREC:;%%
RETCODE = 0  操作成功

结果如下
--------
        序号  =  1
     POD名称  =  vsm-pod-54ddc977b9-4g42g
      堆栈ID  =  524c409a-bc3b-11ea-a186-025565660064
       PodID  =  c2208db2-bc3b-11ea-9d1b-fa163eb94edd
故障容器类型  =  ContainerSm
    故障类型  =  反复故障
故障发生时间  =  2020-07-09 09:06:10
    重建时间  =  2020-07-09 09:08:03+00:00
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0263673345)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 序号 | 该参数用于记录重建POD的序号。同一个POD可能有多次重建历史，仅用PODNAME无法区分，需将SN与PODNAME共同作为主键。 |
| POD名称 | 该参数用于显示重建的POD名称。如果不输入该输出，则查询系统内所有POD的重建记录。 |
| 堆栈ID | 该参数用于显示POD所属的堆栈ID。 |
| PodID | 该参数用于显示PaaS定义的PodID。 |
| 故障容器类型 | 该参数用于显示故障容器的类型。 |
| 故障类型 | 该参数用于显示容器的故障类型。<br>取值说明：<br>- “LOOPFAULT（反复故障）”：反复故障<br>- “CONTINUEFAULT（持续故障）”：持续故障<br>- “STORAGEOVERLOADFAULT（存储过载故障）”：存储过载故障 |
| 故障发生时间 | 该参数用于显示诊断故障发生的时间。 |
| 重建时间 | 该参数用于显示POD重建的时间。 |

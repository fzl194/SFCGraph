# 调测IPv6承载上下文

- [操作场景](#ZH-CN_OPI_0138276177__1.3.1)
- [必备事项](#ZH-CN_OPI_0138276177__1.3.2)

## [操作场景](#ZH-CN_OPI_0138276177)

在部署网络时，需要通过本地IPv6地址分配方式给MS/UE分配下行路由。此时，需要对 UDG 的IPv6承载上下文进行调测。

> **说明**
> 适用于SGW-U、PGW-U、UPF。

## [必备事项](#ZH-CN_OPI_0138276177)

前提条件

- 请仔细阅读[GWFD-020401 IPv6承载上下文](../GWFD-020401 IPv6承载上下文_67107266.md)。
- 完成[激活IPv6承载上下文](激活IPv6承载上下文_38276176.md)。

数据

该操作无需准备数据。

工具

- 测试终端
- OM Portal

## [操作步骤](#ZH-CN_OPI_0138276177)

1. 测试终端 使用“apn-test”APN接入网络，用户IP地址类型为IPv6。
2. 执行 [**DSP SESSIONINFO**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/会话信息管理/会话信息查询/显示用户上下文（DSP SESSIONINFO）_86526407.md) 命令，通过IMSI查看 测试终端 激活后为用户分配的IPv6地址。
  ```
  DSP SESSIONINFO:QUERYTYPE=IMSI,IMSI="460000123456789";
  ```
  ```
  ......

  -------------------------------
                                   IMSI  =  460000123456789
                                       ......              
                               PDP Type  =  IPv6 
                       IPv6 PDP address  =  fc00:1:403:1:0:0:0:0         
                                       ......    
                               APN name  =  apn-test
                                       ......       
  (Number of results = 1)
  ---    END
  ```
    - 如果“IPv6 PDP address”与用户激活时携带的IPv6地址一致，“PDP Type”为IPv6，调测结束。
    - 如果“IPv6 PDP address”与用户激活时携带的IPv6地址不一致，或“PDP Type”不为IPv6，请执行[步骤 3](#ZH-CN_OPI_0138276177__step8)。
3. 收集信息并寻求技术支持。
    a. 执行 [**EXP MML**](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/配置管理/配置导出管理/导出MML文件（EXP MML）_47200033.md) 命令将当前配置数据导出为MML脚本文件并保存。
    b. 查看并收集对端设备配置及接口状态信息。
    c. 收集归纳所有信息并联系华为技术支持解决。

#### [命令功能](#ZH-CN_CONCEPT_0000206691417206)

该命令用于查询第三方app的CPU过载告警阈值。

#### [注意事项](#ZH-CN_CONCEPT_0000206691417206)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000206691417206)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000206691417206)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000206691417206)

查询第三方app的过载告警阈值：

```
LST SFIPCPUALMTHD:;
```

```

RETCODE = 0  操作成功

过载告警阈值
------------
告警产生阈值(%)  =  80
告警恢复阈值(%)  =  60
(结果个数 = 1)

---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000206691417206)

参见SET SFIPCPUALMTHD的参数说明。

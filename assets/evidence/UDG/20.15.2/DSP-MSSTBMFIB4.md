#### [命令功能](#ZH-CN_CONCEPT_0000001549801682)

该命令用于关键字查询第四版互联网协议转发表的结果。

#### [注意事项](#ZH-CN_CONCEPT_0000001549801682)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549801682)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549801682)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RUNAME | RU名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定RU名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，区分大小写。<br>默认值：无<br>配置原则：使用DSP RU查看RU名称。 |
| TABLEID | 表号 | 可选必选说明：必选参数<br>参数含义：该参数用于表示表号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～4294967295。<br>默认值：无 |
| IPV4VPN | 第四版互联网协议虚拟专用网 | 可选必选说明：必选参数<br>参数含义：该参数用于表示第四版互联网协议虚拟专用网。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～4294967295。<br>默认值：无 |
| IPV4ADDRESS | 第四版互联网协议地址 | 可选必选说明：必选参数<br>参数含义：该参数用于表示第四版互联网协议地址。<br>数据来源：本端规划<br>取值范围：IPv4地址类型。<br>默认值：无 |
| MASKLEN | 第四版互联网协议地址掩码长度 | 可选必选说明：必选参数<br>参数含义：该参数用于表示第四版互联网协议地址掩码长度。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～32。<br>默认值：无 |
| MATCHTYPE | 匹配类型 | 可选必选说明：必选参数<br>参数含义：该参数用于表示匹配类型。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- exact：精确匹配。<br>- noexact：模糊匹配。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549801682)

查询使用关键字查询第四版互联网协议转发表的结果：

```
DSP MSSTBMFIB4:TABLEID = 1,IPV4VPN = 1,IPV4ADDRESS = "10.1.1.2",MASKLEN = 24,MATCHTYPE = exact,RUNAME = "VNODE_VNRS_VNFC_IPU_0064";
```

```

RETCODE = 0  操作成功。

结果如下
--------

      表项索引  =  0
        错误码  =  1611661319
(结果个数 = 1)
---    END
```

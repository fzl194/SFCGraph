#### [命令功能](#ZH-CN_CONCEPT_0000001600441165)

该命令用于查询接口震荡抑制配置信息。

若不指定IFNAME参数时，则查询所有接口的震荡抑制；若指定IFNAME参数时，则可以查询指定接口的震荡抑制。

#### [注意事项](#ZH-CN_CONCEPT_0000001600441165)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600441165)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600441165)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IFNAME | 接口名 | 可选必选说明：可选参数<br>参数含义：该参数用于指定震荡抑制的接口名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。接口名称由接口类型+接口编号组成。<br>默认值：无<br>配置原则：请使用LST INTERFACE命令查看可用接口。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600441165)

查询接口震荡抑制：

```
LST IFCONTROLFLAP:IFNAME="ethernet64/0/3";
```

```

RETCODE = 0  操作成功

结果如下
-------------------------
            接口名  =  Ethernet64/0/3
      接口抑制门限  =  1001
      接口重用门限  =  1000

 接口Up半衰期（s）  =  50
接口Down半衰期（s） =  50
(结果个数 = 1)
---   END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600441165)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 接口名 | 用于指定接口抑制的接口名。 |
| 接口抑制门限 | 当接口抑制惩罚值超过此值时，接口被抑制。 |
| 接口重用门限 | 当接口惩罚值小于此值时，接口抑制被解除。 |
| 接口抑制门限最大值 | 当接口抑制惩罚值超过此值时，接口抑制惩罚值不再增加。 |
| 接口Up半衰期（s） | 用于指定当接口UP时，惩罚值半衰期。 |
| 接口Down半衰期（s） | 用于指定当接口Down时，惩罚值半衰期。 |

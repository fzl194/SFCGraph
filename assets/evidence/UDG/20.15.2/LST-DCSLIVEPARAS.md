## [命令功能](#ZH-CN_MMLREF_0000002176289642)

该命令用于查询DCS直播参数。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000002176289642)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000002176289642)

无

## [使用实例](#ZH-CN_MMLREF_0000002176289642)

查询DCS直播参数。

```
%%LST DCSLIVEPARAS:;%%
RETCODE = 0  操作成功

操作结果如下：
------------------------
       读滞后阈值（MB） =  3
     回源超时阈值（ms） =  200
  回源超时检查周期（s） =  30
订阅关系核查周期（Min） =  30
  直播资源老化周期（s） =  30
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000002176289642)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 读滞后阈值（MB） | 该参数用于表示允许直播读取数据落后最新数据的量的最大值，超过该阈值将启动视频读取加速。 |
| 回源超时阈值（ms） | 该参数用于表示回源超时阈值，回源指从源头CDN拉取视频数据。 |
| 回源超时检查周期（s） | 该参数用于表示回源超时检查周期。 |
| 订阅关系核查周期（Min） | 该参数用于表示订阅关系核查周期，避免UE去订阅直播未通知给DCS，导致订阅关系残留。 |
| 直播资源老化周期（s） | 该参数用于表示直播资源老化周期，直播资源老化用于回收已结束的或无人订阅的直播资源。 |

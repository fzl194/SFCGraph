#### [命令功能](#ZH-CN_TOPIC_0251971891)

该命令用来查询LACP配置。

#### [注意事项](#ZH-CN_TOPIC_0251971891)

- 该命令仅适用于NP卡加速模式场景。

#### [操作用户权限](#ZH-CN_TOPIC_0251971891)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_TOPIC_0251971891)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| TRUNKINDEX | Trunk的索引 | 可选必选说明：可选参数<br>参数含义：该参数用于指定Trunk的索引。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～4294967294<br>默认值：无 |

#### [使用实例](#ZH-CN_TOPIC_0251971891)

- 查询LACP配置：
  ```
  LST LACPCFG:;
  ```
  ```
  RETCODE = 0  操作成功

  结果如下
  --------
  Trunk的索引  接口名称    LACP探测速率  LACP优先级  工作模式  

  17           Eth-Trunk1  快            32768       开启      
  18           Eth-Trunk2  快            32768       关闭      
  (结果个数 = 2)

  ---    END
  ```

#### [输出结果说明](#ZH-CN_TOPIC_0251971891)

| 输出项名称 | 输出项解释 |
| --- | --- |
| Trunk的索引 | 用于指定Trunk的索引。 |
| 接口名称 | 用于指定接口名称。 |
| LACP探测速率 | 用于指定LACP探测速率。<br>LACP探测速率枚举类型：<br>- 快：快速LACP探测。<br>- 慢：慢速LACP探测。 |
| LACP优先级 | 用于指定LACP优先级。 |
| 工作模式 | 用于指定工作模式。<br>工作模式枚举类型：<br>- 开启：开启LACP工作模式。<br>- 关闭：关闭LACP工作模式。 |

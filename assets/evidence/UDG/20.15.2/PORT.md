#### [命令功能](#ZH-CN_CONCEPT_0259104204)

该命令用于显示VNFC上的端口信息，包括端口的配置，部署情况以及运行状态。

#### [注意事项](#ZH-CN_CONCEPT_0259104204)

该命令只能显示VNFC上的端口信息，不显示VNFC上的子端口信息。

#### [操作用户权限](#ZH-CN_CONCEPT_0259104204)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0259104204)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PORTNAME | 端口名称 | 可选必选说明：可选参数<br>参数含义：该参数表示端口名称，例如Ethernet64/0/3，其中64代表资源单元编号，0为默认值，3代表端口编号。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，区分大小写。<br>默认值：无 |
| RUNAME | RU名称 | 可选必选说明：可选参数<br>参数含义：该参数表示资源单元名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，区分大小写。<br>默认值：无<br>配置原则：使用<br>[**DSP RU**](../RU管理/显示资源单元信息（DSP RU）_59103857.md)<br>查看RU名称。 |
| SERVICEINSTANCE | 服务实例 | 可选必选说明：必选参数<br>参数含义：该参数表示大颗粒服务实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，通过LST VNFC命令获取。<br>默认值：无<br>配置原则：只能填写通过LST VNFC命令查询到的管理代理标识，但不能填写0，0表示VNFP。 |

#### [输出结果说明](#ZH-CN_CONCEPT_0259104204)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 端口名称 | 表示端口名称，例如Ethernet64/0/3，其中64代表资源单元编号，0为默认值，3代表端口编号。 |
| RU名称 | 表示资源单元名称。 |
| 是否绑定PAE | 表示是否绑定PAE。 |
| 是否对外端口 | 表示是否对外端口。 |
| 网络平面 | 该参数表示网络平面。<br>- 基础业务端口。<br>- 内部管理端口。<br>- 外部管理端口。<br>- 外部连接端口。<br>- Fabric平面子接口。<br>- PAE间通信端口。<br>- 外部linux端口。<br>- NLS外部管理端口。<br>- 主备仲裁扩展端口。<br>- 转发拓扑逻辑端口。<br>- 转发集成逻辑端口。<br>- 内部逻辑服务端口。<br>- 第二基础业务端口。<br>- 第二内部逻辑服务端口。<br>- 内联控制口。<br>- tunnel口。<br>- VNFM服务端口。 |
| 带宽类型 | 表示带宽类型，例如吉比特以太网。 |
| MAC地址 | 表示MAC地址。 |
| 运行状态 | 该参数表示运行状态。<br>- 停止：虚拟端口运行状态为停止。<br>- 运行：虚拟端口运行状态为运行。<br>- 管理停止：虚拟端口运行状态为管理停止。 |
| 上次连接时间 | 表示上次连接时间。 |
| 上次断开时间 | 表示上次断开时间。 |
| 端口在位状态 | 该参数表示端口在位状态。<br>- 不在位：端口为不在位状态。<br>- 在位：端口为在位状态。<br>- 离线：端口为离线状态。 |

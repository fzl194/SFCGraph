## [命令功能](#ZH-CN_MMLREF_0000001215736940)

本命令用于查询网元的实例信息。

> **说明**
> 无。

## [参数说明](#ZH-CN_MMLREF_0000001215736940)

无。

## [使用实例](#ZH-CN_MMLREF_0000001215736940)

查询 UDG 网元的应用实例信息：

```
%%LST APPINFO:;%%
RETCODE = 0  操作成功

操作结果如下：
--------------
      网元ID  =  0
    网元类型  =  
UDG

    应用名称  =  
UDG

    应用版本  =  22.1.0.B220
应用补丁版本  =  NULL
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001215736940)

该命令执行正常，会返回操作成功的提示信息和相关参数，各参数含义请参见 [表1](#ZH-CN_MMLREF_0000001215736940__table8897535143616) 所示。

该命令执行异常，会返回对应的错误码。常见错误码如 [表2](#ZH-CN_MMLREF_0000001215736940__table10759198327) 所示。

*表1 输出项解释*

| 输出项名称 | 输出项解释 |
| --- | --- |
| 网元ID | 网元的ID信息。 |
| 网元类型 | 网元类型。 |
| 应用名称 | 应用名称。 |
| 应用版本 | 应用版本号。 |
| 应用补丁版本 | 应用补丁版本号，用于表示应用的补丁版本。如果该应用没有安装补丁，则显示NULL。 |

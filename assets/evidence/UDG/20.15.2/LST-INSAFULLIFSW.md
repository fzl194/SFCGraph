#### [命令功能](#ZH-CN_CONCEPT_0000206906564543)

**适用NF：PGW-U、UPF**

查询全量智能SA识别开关。

#### [注意事项](#ZH-CN_CONCEPT_0000206906564543)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000206906564543)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000206906564543)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000206906564543)

查询当前全量智能SA识别开关状态：

```
LST INSAFULLIFSW:;
```

```

RETCODE = 0 操作成功。

全量智能SA识别开关信息
------
 
        全量推理开关 = Enable

(结果个数 = 1)
--- END
```

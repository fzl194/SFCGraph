## [命令功能](#ZH-CN_MMLREF_0000001354738721)

该命令用于查询HTTP CCA属性。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0000001354738721)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001354738721)

无

## [使用实例](#ZH-CN_MMLREF_0000001354738721)

如果想查询HTTP CCA属性，可以用如下命令：

```
%%LST HTTPCCACONF:;%%
RETCODE = 0  操作成功

结果如下
--------
     CCA校验开关  =  打开
CCA的有效时长(s)  =  86400
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001354738721)

| 输出项名称 | 输出项解释 |
| --- | --- |
| CCA校验开关 | 该参数用于控制HTTP服务端CCA校验功能是否开启。 |
| CCA的有效时长(s) | 该参数用于配置HTTP客户端生成的CCA的有效时长。 |

#### [命令功能](#ZH-CN_CONCEPT_0000001600441109)

该命令用于查询自动配置开关。

#### [注意事项](#ZH-CN_CONCEPT_0000001600441109)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600441109)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600441109)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001600441109)

查询自动配置开关：

```
LST AUTOCONFIG:;
```

```

RETCODE = 0  操作成功

结果如下
-------------------------
自动配置开关  =  FALSE
(结果个数 = 1)
---    END
```

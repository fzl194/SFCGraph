#### [命令功能](#ZH-CN_CONCEPT_0000001550280946)

该命令用于查询当前系统中以太ACL的配置信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001550280946)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550280946)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550280946)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| ACLNAME | ACL规则组标识 | 可选必选说明：必选参数<br>参数含义：该参数用于指定规则属于哪个规则组。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～32。不支持空格，区分大小写。以英文字母a～z或A～Z开始，可以是英文字母、数字、连字符“-”、下划线“_”或中文字符的组合。整数形式，取值范围是4000～4999。<br>默认值：无 |
| ACLRULENAME | 规则名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定规则名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～32。不支持空格，区分大小写。<br>默认值：无 |

#### [输出结果说明](#ZH-CN_CONCEPT_0000001550280946)

| 输出项名称 | 输出项解释 |
| --- | --- |
| ACL规则组标识 | 该参数用于表示所要创建的规则属于哪个规则组。 |
| 规则名称 | 该参数用于表示所要创建的规则名称。 |
| 规则ID | 该参数用于表示创建的规则的ID。 |
| 规则行为 | 该参数用于表示对符合本规则的报文采取的动作。 |
| 以太帧头协议号 | 该参数用于表示规则要匹配的以太帧协议号。 |
| 以太帧头协议掩码 | 该参数用于表示规则要匹配的以太帧协议号的掩码。 |
| 源MAC地址 | 该参数用于表示规则的源MAC地址。 |
| 源MAC地址掩码 | 该参数用于表示规则的源MAC地址掩码。 |
| 目的MAC地址 | 该参数用于表示规则的目的MAC地址。 |
| 目的MAC地址掩码 | 该参数用于表示规则的目的MAC地址掩码。 |
| VLAN ID | 该参数用于表示规则的VLAN ID。 |
| VLAN ID的掩码 | 该参数用于表示规则的VLAN ID的掩码。 |
| 规则描述 | 该参数用于表示规则描述。 |

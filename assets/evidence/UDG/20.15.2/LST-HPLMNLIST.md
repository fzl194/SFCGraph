#### [命令功能](#ZH-CN_CONCEPT_0244865467)

**适用NF：SGW-U、PGW-U、UPF**

该命令用于查看UPF设备的归属PLMN。

#### [注意事项](#ZH-CN_CONCEPT_0244865467)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0244865467)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0244865467)

无。

#### [使用实例](#ZH-CN_CONCEPT_0244865467)

查询UPF上归属PLMN：

```
LST HPLMNLIST:;
```

```

RETCODE = 0  操作成功。

结果如下
--------
移动国家码  =  470
移动网络号  =  03
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0244865467)

参见ADD HPLMNLIST的参数说明。

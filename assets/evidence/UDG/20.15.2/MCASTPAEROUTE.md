#### [命令功能](#ZH-CN_CONCEPT_0000001550281070)

该命令用于显示组播引流表项信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001550281070)

该命令仅适用于非NP卡场景和NP卡非加速模式场景。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550281070)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550281070)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| ADDRESSFAMILY | 地址族 | 可选必选说明：必选参数<br>参数含义：该参数用于表示地址族类型。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- ipv4unicast：IPv4单播。<br>默认值：无 |
| VRFNAME | VPN实例名称 | 可选必选说明：可选参数<br>参数含义：该参数用于表示VPN实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。<br>默认值：_public_<br>配置原则：通过LST L3VPNINST查看当前已存在的VPN实例。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001550281070)

显示组播引流表项信息：

```
DSP MCASTPAEROUTE:VRFNAME="_public_",ADDRESSFAMILY=ipv4unicast;
```

```
RETCODE = 0 操作成功。

结果如下
------------------------
VPN实例名称    地址族      源地址         组地址       叶子个数    Include叶子个数    表项建立时间      表项标记

_public_       IPv4单播    192.168.0.1    239.0.0.1    1           1                  00:00:23          PAE
_public_       IPv4单播    192.168.0.2    239.0.0.2    2           1                  00:00:52          PAE
(结果个数 = 2)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001550281070)

| 输出项名称 | 输出项解释 |
| --- | --- |
| VPN实例名称 | 用于表示VPN实例名称。 |
| 地址族 | 用于表示地址族类型。 |
| 源地址 | 用于表示组播源地址。 |
| 组地址 | 用于表示组播组地址。 |
| 叶子个数 | 用于表示组播引流路由表项的叶子个数。 |
| Include叶子个数 | 用于表示组播引流路由表项中Include状态的叶子个数。 |
| 表项建立时间 | 用于表示组播引流路由表项的建立时间。 |
| 表项标记 | 用于表示组播引流路由表项的标记，包括：<br>- PAE表示引流消息创建的表项。<br>- PIM表示PIM协议创建的表项。 |

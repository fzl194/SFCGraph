# 恢复TCP统计功能的RTT区间为初始值（RTR TCPSTATRTTRANGE）

- [命令功能](#ZH-CN_CONCEPT_0000201474589219__1.3.1.1)
- [注意事项](#ZH-CN_CONCEPT_0000201474589219__1.3.2.1)
- [操作用户权限](#ZH-CN_CONCEPT_0000201474589219__1.3.3.1)
- [参数说明](#ZH-CN_CONCEPT_0000201474589219__1.3.4.1)
- [使用实例](#ZH-CN_CONCEPT_0000201474589219__1.3.5.1)

#### [命令功能](#ZH-CN_CONCEPT_0000201474589219)

**适用NF：PGW-U、UPF**

该命令用于把TCP统计功能中的UE侧、SP侧RTT区间恢复到初始值。

#### [注意事项](#ZH-CN_CONCEPT_0000201474589219)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000201474589219)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000201474589219)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000201474589219)

如果运营商需要把UE侧、SP侧RTT区间恢复到初始值：

```
RTR TCPSTATRTTRANGE:;
```

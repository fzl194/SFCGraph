# 调测IPv4v6双栈接入

- [操作场景](#ZH-CN_OPI_0138276241__1.3.1)
- [必备事项](#ZH-CN_OPI_0138276241__1.3.2)
- [操作步骤](#ZH-CN_OPI_0138276241__1.3.3)

## [操作场景](#ZH-CN_OPI_0138276241)

支持用户入网时以IPv4v6双栈接入。

> **说明**
> 适用于SGW-U、PGW-U、UPF。

## [必备事项](#ZH-CN_OPI_0138276241)

前提条件

- 请仔细阅读[GWFD-020403 IPv4v6双栈接入](../GWFD-020403 IPv4v6双栈接入_67108868.md)。
- 完成[激活IPv4v6双栈接入](激活IPv4v6双栈接入_38276180.md)。

数据

该操作无需准备数据。

工具

- 测试终端
- OM Portal

## [操作步骤](#ZH-CN_OPI_0138276241)

1. 测试终端使用“apn-test”APN发起接入网络请求。
2. 执行 [**DSP SESSIONINFO**](../../../../../OM参考/命令/UDG MML命令/用户面服务管理/会话管理/会话信息管理/会话信息查询/显示用户上下文（DSP SESSIONINFO）_86526407.md) 命令，通过 测试终端 的IMSI查看 测试终端 是否成功激活。
  ```
  DSP SESSIONINFO:QUERYTYPE=IMSI,IMSI="460000123456789";
  ```
  ```
  ......

  -------------------------------
                                   IMSI  =  460000123456789
                                       ......               
                               PDP Type  =  IPv4v6
                       IPv4 PDP address  =  10.1.1.42     
                       IPv6 PDP address  =  fc00:2000:2000:1600:1000:2300:1000:15
                                       ......                 
                               APN name  =  apn-test         
                                       ......                
     (Number of results = 1)
  ---    END
  ```
    - 如果测试终端激活成功，且“PDP Type”、“IPv4 PDP address”和“IPv6 PDP address”与规划值一致，测试终端接入成功，调测结束。
    - 如果测试终端激活成功，但“PDP Type”、“IPv4 PDP address”和“IPv6 PDP address”与规划值不一致，执行[步骤 3](#ZH-CN_OPI_0138276241__step7)。
3. 收集信息并寻求技术支持。
    a. 在OM Portal上创建用户跟踪任务 并保存报文。
    b. 执行 **[EXP MML](../../../../../OM参考/命令/UDG MML命令/平台服务管理/操作维护/配置管理/配置导出管理/导出MML文件（EXP MML）_47200033.md)** 命令将当前配置数据导出为MML脚本文件并保存。
    c. 收集并保存上述所有查询信息。
    d. 收集归纳所有信息并联系华为技术支持解决。

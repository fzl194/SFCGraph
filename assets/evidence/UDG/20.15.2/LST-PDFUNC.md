#### [命令功能](#ZH-CN_CONCEPT_0000202968723527)

**适用NF：SGW-U、PGW-U、UPF**

该命令用于查询Tx-U接口的下行报文检测功能开关状态。

#### [注意事项](#ZH-CN_CONCEPT_0000202968723527)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000202968723527)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000202968723527)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000202968723527)

查询Tx-U接口的下行报文检测功能开关状态：

```
LST PDFUNC:;
```

```

RETCODE = 0  操作成功

报文检测功能配置
----------------
Tx-U接口下行报文  =  使能
(结果个数 = 1)

---    END
```

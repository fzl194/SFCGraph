#### [命令功能](#ZH-CN_CONCEPT_0237678017)

**适用NF：SGW-U、PGW-U、UPF**

该命令用来查询会话核查配置信息。

#### [注意事项](#ZH-CN_CONCEPT_0237678017)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0237678017)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0237678017)

无。

#### [使用实例](#ZH-CN_CONCEPT_0237678017)

显示会话核查配置：

```
LST SESSCHKFUNC:;
```

```

RETCODE = 0 操作成功。
会话核查配置信息
----------------
        路径断核查开关 = 使能
          每日核查开关 = 使能
        SG恢复核查开关 = 使能
  会话规格限制核查开关 = 使能
      会话创建核查开关 = 使能
      会话创建核查阈值 = 4
(结果个数 = 1)
--- END
```

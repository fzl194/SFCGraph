## [操作场景](#ZH-CN_OPI_0124082259)

为了防止PDN侧网络设备遭受移动用户的DDoS攻击，运营商可以启用 UDG 的防范DDoS攻击功能。在 UDG 上配置报文连接最大速率，当UE向PDN侧网络设备发起TCP连接，如果MS发送的SYN报文连接速率超过配置值， UDG 将丢弃报文，以保护PDN侧的网络设备。

> **说明**
> 适用于SGW-U、PGW-U、UPF。

## [任务示例](#ZH-CN_OPI_0124082259)

任务描述

为了防止PDN侧网络设备遭受移动用户的DDoS攻击，采用防范DDoS攻击功能。

脚本

//设置防范DDoS攻击的流控阈值。

```
SET DDOS:THRESHOLD=10;
```

//使能UserProfile的防范DDoS功能。

```
ADD USERPROFILE:USERPROFILENAME="up_test",DDOSCHECK=ENABLE;
```

## [命令功能](#ZH-CN_MMLREF_0294730408)

该命令用于查询CSLB的状态，服务申请完成则显示为在线状态。

> **说明**
> 无

#### [操作用户权限](#ZH-CN_MMLREF_0294730408)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0294730408)

无

## [使用实例](#ZH-CN_MMLREF_0294730408)

使用如下命令查询CSLB的状态：

```
%%DSP LBSTATUS:;%%
RETCODE = 0  操作成功

结果如下
--------
组 ID  =  4
   状态  =  在线
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0294730408)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 组ID | 该参数用于表示CSLB属组ID。 |
| 状态 | 该参数用于表示CSLB的状态。 |

## [命令功能](#ZH-CN_MMLREF_0000001451081641)

该命令用于查询监控VPN实例。

> **说明**
> 该命令只用于在UEG-M/UEG网元采用主备（热备）容灾模式下执行。

#### [操作用户权限](#ZH-CN_MMLREF_0000001451081641)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001451081641)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| DRGROUPID | 容灾组标识 | 可选必选说明：必选参数<br>参数含义：该参数用于配置容灾组标识。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是1~8。<br>默认值：无<br>配置原则：无 |
| VPNGRPID | VPN组标识 | 可选必选说明：必选参数<br>参数含义：该参数用于配置VPN组标识。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是1~10。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001451081641)

查询监控VPN实例:

```
%%LST MONVPN:DRGROUPID=1,VPNGRPID=1;%%
RETCODE = 0  操作成功

结果如下
--------
 容灾组标识 = 1
  VPN组标识 = 1
VPN实例名称 = 1
是否联动快速隔离 = 否
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001451081641)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 容灾组标识 | 该参数用于配置容灾组标识。 |
| VPN组标识 | 该参数用于配置VPN组标识。 |
| VPN实例名称 | 该参数用于配置VPN实例名称。 |
| 是否联动隔离 | 该参数用于配置VPN实例是否联动隔离。 |

#### [命令功能](#ZH-CN_CONCEPT_0000001549802122)

该命令用于查询BFD会话的运行信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001549802122)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549802122)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549802122)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| LOCALDISCR | 本地标识符 | 可选必选说明：必选参数<br>参数含义：本地描述符。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～49152。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549802122)

查询BFD会话的运行信息：

```
DSP BFDSESSIONRUNNING:LOCALDISCR=1;
```

```

RETCODE = 0  操作成功。

结果如下
----------------
           本地标识符  =  1
             会话状态  =  DOWN
           本地诊断字  =  无
         会话Down原因  =  无

(结果个数 = 1)
--- END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001549802122)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 本地标识符 | 本地描述符。 |
| 会话状态 | 会话的当前状态。包括： UP：BFD会话处于Up状态； DOWN：BFD会话处于Down状态； 管理DOWN：如果对BFD会话执行了管理Down操作，将显示此种状态； INIT：BFD会话处于Init状态。 |
| 本地诊断字 | 最近一次会话Down的本地诊断原因。 |
| 会话Down原因 | 最近一次会话Down的原因。 |

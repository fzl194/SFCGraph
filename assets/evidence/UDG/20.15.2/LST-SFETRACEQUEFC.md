#### [命令功能](#ZH-CN_CONCEPT_0000002098550217)

该命令用来查询VNRS IP消息跟踪的队列流控配置。

#### [注意事项](#ZH-CN_CONCEPT_0000002098550217)

- 该命令执行后立即生效。
- 该命令仅适用于非NP卡场景和NP卡非加速模式场景。

#### [操作用户权限](#ZH-CN_CONCEPT_0000002098550217)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000002098550217)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000002098550217)

查询VNRS IP消息跟踪的队列流控配置：

```
LST SFETRACEQUEFC:;
```

```
RETCODE = 0  操作成功

结果如下
------------------------
       队列流控使能开关  =  是
          起控阈值（%）  =  70
   队列流控自动恢复开关  =  是
          恢复阈值（%）  =  30
自动恢复检测时间（min）  =  5
(结果个数 = 1) 
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000002098550217)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 队列流控使能开关 | 该参数用于指定是否开启VNRS IP消息跟踪的队列流控功能。 |
| 起控阈值（%） | 该参数用于指定触发队列流控的队列使用率阈值。在执行VNRS IP消息跟踪任务期间，如果队列使用率大于等于该参数取值，则触发队列流控。 |
| 队列流控自动恢复开关 | 该参数用于指定触发队列流控后，当满足恢复条件时，能否停止流控且恢复跟踪。 |
| 恢复阈值（%） | 该参数用于指定触发队列流控后，停止流控且恢复跟踪的队列使用率阈值。 |
| 自动恢复检测时间（min） | 该参数用于指定触发队列流控后，对队列使用率的持续检测时间，单位：分钟。如果队列使用率在<br>**“**<br>自动恢复检测时间（min）<br>**”**<br>内持续小于等于“恢复阈值（%）”，则停止流控且恢复跟踪。 |

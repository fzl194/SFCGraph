#### [命令功能](#ZH-CN_CONCEPT_0000001600440761)

该命令用于查询ICMP选项安全配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001600440761)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600440761)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600440761)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001600440761)

查询ICMP选项安全配置：

```
LST ICMPOPTSECURITY:;
```

```

        RETCODE = 0  操作成功

              RU名称  =  VNODE_VNRS_VNFC_IPU_0064
        ICMP安全类型  =  TTL超时报文
        (结果个数 = 1)
        ---    END
```

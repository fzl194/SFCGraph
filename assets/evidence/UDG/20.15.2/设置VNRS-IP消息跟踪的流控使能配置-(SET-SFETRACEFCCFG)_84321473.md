#### [命令功能](#ZH-CN_CONCEPT_0000002084321473)

![](设置VNRS IP消息跟踪的流控使能配置 (SET SFETRACEFCCFG)_84321473.assets/notice_3.0-zh-cn.png)

如果设置流控不使能，可能会导致业务呼损。

当CPU负载超阈值时，该命令用来设置VNRS IP消息跟踪的CPU流控功能是否开启。

#### [注意事项](#ZH-CN_CONCEPT_0000002084321473)

- 该命令执行后立即生效。
- 该命令仅适用于非NP卡场景和NP卡非加速模式场景。
- 该命令存在系统初始记录（NP卡加速模式场景无需关注此初始记录），参数的初始设置值如下表：
  | FLOWCTRLEN |
  | --- |
  | TRUE（是） |

#### [操作用户权限](#ZH-CN_CONCEPT_0000002084321473)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000002084321473)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| FLOWCTRLEN | 流控使能标记 | 可选必选说明：必选参数<br>参数含义：该参数用于指定当CPU负载超阈值时，是否开启VNRS IP消息跟踪的CPU流控功能。<br>数据来源：本端规划<br>取值范围：<br>- “TRUE（是）”：表示CPU流控功能开启。<br>- “FALSE（否）”：表示CPU流控功能关闭。<br>默认值：无<br>配置原则：无。 |

#### [使用实例](#ZH-CN_CONCEPT_0000002084321473)

```
关闭VNRS IP消息跟踪的CPU流控功能：
SET SFETRACEFCCFG: FLOWCTRLEN=FALSE; 
```

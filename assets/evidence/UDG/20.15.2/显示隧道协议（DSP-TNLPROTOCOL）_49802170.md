#### [命令功能](#ZH-CN_CONCEPT_0000001549802170)

该命令用于显示隧道协议。

#### [注意事项](#ZH-CN_CONCEPT_0000001549802170)

只有配置隧道才能使用该命令。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549802170)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549802170)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001549802170)

显示隧道协议：

```
DSP TNLPROTOCOL:;
```

```

RETCODE = 0  操作成功

结果如下
-------------------------
接口名字    隧道类型
Tunnel0     Gre
Tunnel1     Gre
(结果个数 = 2)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001549802170)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 接口名字 | 用于表示接口名字。 |
| 隧道类型 | 用于表示隧道类型。 |

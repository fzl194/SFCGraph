# 删除IPv4 MPAC策略规则（RMV MPACIPV4RULE）

- [注意事项](#ZH-CN_CONCEPT_0000001550281790__1.3.2.1)

#### [命令功能](#ZH-CN_CONCEPT_0000001550281790)

该命令用于删除某策略下的某规则。

#### [注意事项](#ZH-CN_CONCEPT_0000001550281790)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550281790)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550281790)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| POLICYNAME | 策略名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定创建规则的策略名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。大小写敏感，英文字母开头，不支持空格。<br>默认值：无 |
| RULENAME | 规则名字 | 可选必选说明：必选参数<br>参数含义：该参数用于指定规则的名字。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。大小写敏感，英文字母开头，不支持空格。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001550281790)

删除某策略下的某规则：

```
RMV MPACIPV4RULE:POLICYNAME="policyV4",RULENAME="name-test";
```

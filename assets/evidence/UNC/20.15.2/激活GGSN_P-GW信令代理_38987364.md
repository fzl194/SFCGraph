## [操作场景](#ZH-CN_OPI_0000001138987364)

当运营商需要GGSN/P-GW承担Proxy的作用，将漫游用户的激活上下文请求消息路由到归属地的GGSN/P-GW，由归属地GGSN/P-GW直接处理漫游用户的激活请求以及随后的各种业务时，需要配置GGSN/P-GW信令代理特性。

> **说明**
> 适用于 GGSN、 PGW-C

## [对系统的影响](#ZH-CN_OPI_0000001138987364)

性能影响程度：低

开启本特性，将需占用PDP上下文资源，因而将会使整机支持的基本功能PDP上下文数有所下降。并且增加了信令处理负荷。

## [必备事项](#ZH-CN_OPI_0000001138987364)

前提条件

- 请仔细阅读[WSFD-205107 GGSN/P-GW信令代理](WSFD-205107 GGSN_P-GW信令代理特性概述_37468134.md)。
- Proxy GGSN/P-GW与归属地GGSN/P-GW之间的路由可用。
- 获取要进行信令代理的APN、IMSI/MSISN号段规划或计费属性和归属地GGSN/P-GW的IP地址。
- 完成加载license（如果有需求，请联系华为技术支持工程师处理）。

## [操作步骤](#ZH-CN_OPI_0000001138987364)

1. 以下命令在“MML命令行 - UNC”窗口上执行。打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md) :LICITEM=LKV3W9GPSP11,SWITCH=ENABLE;
2. 如需开启指定APN的Proxy 功能，配置基于APN控制是否打开网关Proxy功能。[**ADD APNGWPROXYFUNC**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/GGSN和P-GW Proxy/APN网关Proxy功能/增加APN网关Proxy功能配置（ADD APNGWPROXYFUNC）_42853254.md):APN=APN，PROXYSW=基于APN控制的网关Proxy开关； 如果未配置本步骤，则会使用全局控制是否打开网关Proxy功能。
3. 如需开启所有APN的Proxy 功能，配置基于全局控制是否打开网关Proxy功能。
  [**SET GWPROXYFUNC**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/GGSN和P-GW Proxy/网关Proxy功能/设置网关Proxy功能（SET GWPROXYFUNC）_42853270.md) :PROXYSW=网关Proxy开关；
4. **可选：** 配置将要对其进行信令代理操作IMSI/MSISDN号码段和号码段组。
    a. 配置IMSI/MSISDN号码段。
      [**ADD PRIMSIMSISDNSEG**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/GGSN和P-GW Proxy/代理选择的IMSI_MSISDN号段/增加代理IMSI_MSISDN号段（ADD PRIMSIMSISDNSEG）_42853258.md) :SEGMENTNAME=IMSI/MSISDN号段名称，SEGMENTTYPE=IMSI/MSISDN号段类型，SEGSTART=号段起始字符串，SEGEND=号段结束字符串；
    b. 配置IMSI/MSISDN号码段组。
      [**ADD PRIMSIMSISDNG**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/GGSN和P-GW Proxy/代理选择的IMSI_MSISDN号段组/增加代理IMSI_MSISDN号段组（ADD PRIMSIMSISDNG）_88733217.md) :SEGGROUPNAME=IMSI/MSISDN号段组名称，SEGMENTNAME=IMSI/MSISDN号段名称；
    c. 配置Home group。
      [**ADD HOMEGROUP**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/GGSN和P-GW Proxy/Home Group/增加Home Group（ADD HOMEGROUP）_42853256.md) :HOMEGROUPINDX=home group的编号，IMSIMSISDNGRPNM=home group绑定的用户号码段组名；
5. **可选：** 配置将要对其进行信令代理操作的计费属性组信息。
    a. 配置计费属性组。
      [**ADD CCGROUP**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/GGSN和P-GW Proxy/计费特征组/增加计费特征组（ADD CCGROUP）_88613375.md) :CCGROUPNAME=计费属性组名称；
    b. 配置计费属性值。
      [**ADD CCVALUE**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/GGSN和P-GW Proxy/计费特征值/增加计费特征值（ADD CCVALUE）_88733215.md) :CCGROUPNAME=计费属性组名称, CCVALUE=计费属性值；
    c. 配置Home group。
      [**MOD HOMEGROUP**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/GGSN和P-GW Proxy/Home Group/修改Home Group（MOD HOMEGROUP）_42693474.md) :HOMEGROUPINDX=home group的编号，CCGROUPNAME=计费属性组名称；
6. 在Home group下配置归属P-GW的IP地址。[**ADD HOMEIP**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/GGSN和P-GW Proxy/Home IP/增加Home IP地址（ADD HOMEIP）_88613377.md):HOMEGROUPINDX=组号，HOMEIPID=home-ip的编号，IPVERSION=IP版本,HOMEIPv4=Home IPv4地址/HOMEIPv6=Home IPv6地址，PRODUCTTYPE=产品类型；
7. 将Home group绑定到指定的APN。
  [**ADD HOMEGRPBINDAPN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/GGSN和P-GW Proxy/Home Group绑定APN/增加Home Group和APN的绑定关系（ADD HOMEGRPBINDAPN）_42693466.md) :APN=APN，HOMEGROUPINDX=Home group的编号；

## [任务示例](#ZH-CN_OPI_0000001138987364)

任务描述

本实例中，操作员在UNC上进行数据配置实现以下要求。

- 在UNC上的名称为apn-test的APN上开启信令代理功能。
- 归属地UNC的IP地址为192.168.1.1，以产品形态为P-GW为例。

脚本

1. 打开本特性的License配置开关。
  ```
  SET LICENSESWITCH:LICITEM="LKV3W9GPSP11",SWITCH=ENABLE;
  ```
2. 如需开启指定APN的Proxy 功能，配置基于APN控制是否打开网关Proxy功能。
  ```
  ADD APN: APN="apn-test";
  ```
  ```
  ADD APNGWPROXYFUNC:APN="apn-test",PROXYSW=ENABLE;
  ```
3. 如需开启所有APN的Proxy 功能，配置基于全局控制是否打开网关Proxy功能。
  ```
  SET GWPROXYFUNC:PROXYSW=ENABLE;
  ```
4. 可选：配置将要对其进行信令代理操作IMSI/MSISDN号码段和号码段组。
  ```
  ADD PRIMSIMSISDNSEG:SEGMENTNAME="msisdn-test",SEGMENTTYPE=MSISDN,SEGSTART="138",SEGEND="139";
  ```
  ```
  ADD PRIMSIMSISDNG:SEGGROUPNAME="group-test",SEGMENTNAME="msisdn-test";
  ```
  ```
  ADD HOMEGROUP:HOMEGROUPINDX=1,IMSIMSISDNGRPNM="group-test";
  ```
5. 可选：配置将要对其进行信令代理操作的计费属性组信息。
  ```
  ADD CCGROUP:CCGROUPNAME="cc_group1";
  ```
  ```
  ADD CCVALUE: CCGROUPNAME="cc_group1", CCVALUE="0x0010";
  ```
  ```
  MOD HOMEGROUP:HOMEGROUPINDX=1, CCGROUPNAME="cc_group1";
  ```
6. 在Home group下配置归属P-GW的IP地址。
  ```
  ADD HOMEIP:HOMEGROUPINDX=1,HOMEIPID=1,IPVERSION=IPV4,HOMEIPv4="192.168.1.1",PRODUCTTYPE=PGW;
  ```
7. 将Home group绑定到指定的APN。
  ```
  ADD HOMEGRPBINDAPN:APN="apn-test",HOMEGROUPINDX=1;
  ```

#### [命令功能](#ZH-CN_CONCEPT_0000001283976188)

**适用NF：NCG**

该命令用于显示RU分配信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001283976188)

2345G混合部署的汇聚RU，不会分配业务RU。

#### [本地用户权限](#ZH-CN_CONCEPT_0000001283976188)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_CONCEPT_0000001283976188)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001283976188)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001283976188)

显示汇聚RU信息：

```
DSP RUCONVERGENCE:;
```

```
RETCODE = 0  操作成功。
结果如下:
---------
RU的ID  操作类型  RU角色  客户端数量  当前汇聚RUID  历史汇聚RUID  汇聚RUID状态  汇聚RU的TB值  当前RU的TB值  

64      注册      汇聚RU  30          0             0             正常          0             1033          
65      注册      业务RU  30          64            0             正常          1033          1036          
(结果个数 = 2)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001283976188)

| 输出项名称 | 输出项解释 |
| --- | --- |
| RU的ID | 用于返回当前RU的ID。 |
| 操作类型 | 用于返回当前RU的操作类型。 |
| RU角色 | 用于返回当前RU角色。 |
| 客户端数量 | 用于返回当前RU的客户端数量。 |
| 当前汇聚RUID | 用于返回当前汇聚RUID。 |
| 历史汇聚RUID | 用于返回历史汇聚RUID。 |
| 汇聚RUID状态 | 用于返回汇聚RUID状态。 |
| 汇聚RU的TB值 | 用于返回汇聚RU的TB值。 |
| 当前RU的TB值 | 用于返回当前RU的TB值。 |

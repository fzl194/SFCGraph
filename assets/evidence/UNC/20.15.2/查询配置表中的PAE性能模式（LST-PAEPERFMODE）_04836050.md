## [命令功能](#ZH-CN_MMLREF_0000001904836050)

该命令用于查询配置表中的PAE性能模式。

## [注意事项](#ZH-CN_MMLREF_0000001904836050)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001904836050)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001904836050)

无

## [使用实例](#ZH-CN_MMLREF_0000001904836050)

查询配置表中的PAE性能模式

```
+++    UNC/*MEID:0 MENAME:unc*/        2024-05-20 18:36:11
O&M    #73
%%LST PAEPERFMODE:;%%
RETCODE = 0  操作成功

结果如下
--------
Pod类型      PAEDP性能模式  sdra的性能模式

vusn-pod     低性能模式0    使用默认性能模式
sfpod        低性能模式0    使用默认性能模式
lbpod        低性能模式0    使用默认性能模式
(结果个数 = 3)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001904836050)

| 输出项名称 | 输出项解释 |
| --- | --- |
| Pod类型 | 该参数用于指定POD类型，可以通过使用命令<br>[**DSP POD**](../../../操作维护/Pod管理/POD查询（DSP POD）_69830277.md)<br>获取。 |
| PAE DP侧的性能模式 | 该参数表示PAE DP侧的性能模式。 |
| sdra侧的性能模式 | 该参数表示SDRA侧的性能模式。 |

## [操作场景](#ZH-CN_OPI_0170014705)

本操作指导介绍在运行网络中激活基于SRVCC的数据语音双切换特性的操作过程。

> **说明**
> 适用于MME。

## [必备事项](#ZH-CN_OPI_0170014705)

前提条件

- 请仔细阅读[WSFD-201001 基于SRVCC的数据语音双切换特性概述](特性概述_70014703.md)。
- 已完成加载License。
- 如果数据业务切换目标是GnGp SGSN，目的GnGp SGSN的GTP-C路径版本已经为GTPv1。

数据

| 类别 | 参数 | 取值样例 | 获取方法 | 说明 |
| --- | --- | --- | --- | --- |
| [**SET EMM**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/MM协议参数管理/S1模式MM协议参数/设置S1模式MM协议参数(SET EMM)_72225207.md) | 切换流程资源释放定时器(s)（HOT3） | 2 | 全网规划 | 切换流程资源释放定时器 |

## [操作步骤](#ZH-CN_OPI_0170014705)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. **可选：**设置切换流程资源释放定时器。
  [**SET EMM**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/MM协议参数管理/S1模式MM协议参数/设置S1模式MM协议参数(SET EMM)_72225207.md)

## [任务示例](#ZH-CN_OPI_0170014705)

任务描述

激活基于SRVCC的数据语音双切换。

脚本

// 进入 “MML命令行-UNC” 窗口。

//打开本特性的License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2DVSS01", SWITCH=ENABLE;
```

//设置切换流程资源释放定时器。

```
SET EMM: HOT3=2;
```

## [操作场景](#ZH-CN_OPI_0000001361167329)

本操作指导介绍在运行网络中激活NSA自动开户特性的操作过程。

在5G NSA组网下，用户首次更换了5G终端，可以通过NSA自动开户实现在不换卡、不换号、无需人工操作的场景下开通5G NSA业务功能，更方便快捷地体验5G网络服务，加速5G的快速发展。

> **说明**
> 适用于MME。

## [必备事项](#ZH-CN_OPI_0000001361167329)

前提条件

- 请仔细阅读[WSFD-011522 NSA自动开户特性概述](WSFD-011522 NSA自动开户特性概述_61327113.md)。
- 本功能需要UNC和UDM配合使用，如果需要使用该特性，需要同时激活UDM的“NSA自动开户”特性，详细激活步骤请参考《UDM 20.9.0 产品文档》中“配置NSA自动开户特性”页面。

2. 针对一定范围的用户，配置NSA自动开户的功能。

## [任务示例](#ZH-CN_OPI_0000001361167329)

任务描述

运营商希望对本网用户开启NSA自动开户功能。

脚本

//针对本网用户，开启NSA自动开户功能。

```
SET DMCMPT: UEDCNRCAP=NOT_SUPPORT;
ADD DMCMPTBYIMSI: SUBRANGE=HOME_USER, UEDCNRCAP=SUPPORT;
```

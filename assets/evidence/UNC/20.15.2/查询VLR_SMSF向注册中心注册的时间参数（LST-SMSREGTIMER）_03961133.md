## [命令功能](#ZH-CN_MMLREF_0000001403961133)

**适用NF：SMSF**

该命令用于查询向注册中心注册的时间参数。

## [注意事项](#ZH-CN_MMLREF_0000001403961133)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001403961133)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001403961133)

无

## [使用实例](#ZH-CN_MMLREF_0000001403961133)

运营商希望查询VLR/SMSF向注册中心注册的时间参数，执行如下命令：

```
LST SMSREGTIMER:;
%%LST SMSREGTIMER:;%%
RETCODE = 0  操作成功

结果如下：
------------------------
        SMSF向注册中心注册更新的时间间隔(分钟) =  24
VLR向注册中心注册更新的时间间隔(分钟) =  24
(结果个数 = 1)
```

## [输出结果说明](#ZH-CN_MMLREF_0000001403961133)

| 输出项名称 | 输出项解释 |
| --- | --- |
| SMSF向注册中心注册更新的时间间隔(分钟) | 该参数用于表示SMSF向注册中心注册更新的时间间隔。 |
| VLR向注册中心注册更新的时间间隔(分钟) | 该参数用于表示VLR向注册中心注册更新的时间间隔。 |

## [操作步骤](#ZH-CN_OPI_0277044936)

- 在相关网元均完成本特性的配置后，可以采用以下步骤检查特性工作是否正常：
    1. 进入 “MML命令行-UNC” 窗口。
    2. 显示低优先级流控状态统计。
      [**DSP LOWPRIOFCSTAT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/操作维护/设备管理/流控管理/业务流控管理/低优先级接入流控管理/显示低优先级流控状态统计(DSP LOWPRIOFCSTAT)_72225839.md) :;
      ```
      %%DSP LOWPRIOFCSTAT: QUERYTYPE=ALL;%%
      RETCODE = 0  操作成功。

      输出结果如下
      --------------
       RU名称               进程号  流控告警状态  流控速率（个/秒）  通过量（个）  拒绝量（个）

       USN_SP_RU_0066       1       正常状态      300                0             0           
       USN_SP_RU_0066       2       正常状态      300                0             0           
       USN_SP_RU_0066       3       正常状态      300                0             0           
       USN_SP_RU_0066       4       正常状态      300                0             0           
       USN_SP_RU_0066       0       正常状态      300                0             0         
      (结果个数 = 5)
      ---    END
      ```
      预期结果：流控告警状态为 “正常状态” 或 “告警状态” 。

#### [命令功能](#ZH-CN_CONCEPT_0000001550121638)

该命令用于显示Trunk成员接口状态信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001550121638)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550121638)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550121638)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| TRUNKIFNAME | Trunk接口名 | 可选必选说明：必选参数<br>参数含义：该参数用于指定需要配置Trunk接口名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。接口名称由接口类型+接口编号组成。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001550121638)

显示Eth-Trunk1的成员接口信息：

```
DSP TRUNKMEMBER:TRUNKIFNAME="Eth-Trunk1";
```

```

RETCODE = 0  操作成功

结果如下
--------
Trunk接口名    成员接口名        选中状态    成员状态

Eth-Trunk1     Ethernet64/0/3    接口备选    接口up
Eth-Trunk1     Ethernet64/0/4    接口选中    接口down
(结果个数 = 2)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001550121638)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 选中状态 | 用于显示成员口是否被选中。 接口选中：代表接口被选中，用于转发。 接口未选中：代表接口链路不可用，不能用于转发。 接口备选：代表接口链路可用但当前不是选中口。 |
| 成员状态 | 用于显示成员口状态，是否链路可用。 Down：接口状态为down，并且链路不可用。 Up：接口状态为up，并且链路可用。 |

其余输出项请参见ADD TRUNKMEMBER的参数说明。

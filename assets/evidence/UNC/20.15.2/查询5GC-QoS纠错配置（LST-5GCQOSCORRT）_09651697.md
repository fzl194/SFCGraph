# 查询5GC QoS纠错配置（LST 5GCQOSCORRT）

- [注意事项](#ZH-CN_MMLREF_0209651697__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0209651697__1.3.4)
- [使用实例](#ZH-CN_MMLREF_0209651697__1.3.5)
- [输出结果说明](#ZH-CN_MMLREF_0209651697__1.3.6)

## [命令功能](#ZH-CN_MMLREF_0209651697)

**适用NF：SMF**

该命令用来查询当SMF收到UE、UDM或PCF消息中携带的QoS字段不合法时，使用该配置作为纠错后的5G QoS参数值。

## [注意事项](#ZH-CN_MMLREF_0209651697)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209651697)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209651697)

无

## [使用实例](#ZH-CN_MMLREF_0209651697)

查询5GC QoS纠错配置：

```
%%LST 5GCQOSCORRT:;%%
RETCODE = 0  操作成功

操作结果如下
------------
下行Session AMBR(千比特/秒)  =  20000
上行Session AMBR(千比特/秒)  =  20000
                      5QI值  =  5
                      ARP值  =  3
    上行保证带宽(千比特/秒)  =  4000
    下行保证带宽(千比特/秒)  =  4000
    下行最大带宽(千比特/秒)  =  5000
    上行最大带宽(千比特/秒)  =  5000
                  5QI优先级  =  10
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209651697)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 下行Session AMBR(千比特/秒) | 该参数用于配置下行集合最大比特速率。 |
| 上行Session AMBR(千比特/秒) | 该参数用于配置上行集合的最大比特速率。 |
| 5QI值 | 该参数用于配置5G QoS识别码。 |
| ARP值 | 该参数用于配置ARP的优先级别。 |
| 上行保证带宽(千比特/秒) | 该参数用于配置保证的上行流比特率。 |
| 下行保证带宽(千比特/秒) | 该参数用于配置保证的下行流比特率。 |
| 下行最大带宽(千比特/秒) | 该参数用于配置最大的下行流比特率。 |
| 上行最大带宽(千比特/秒) | 该参数用于配置最大的上行流比特率。 |
| 5QI优先级 | 该参数用于配置5QI优先等级。 |

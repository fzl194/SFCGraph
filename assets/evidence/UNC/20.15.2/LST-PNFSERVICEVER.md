## [命令功能](#ZH-CN_MMLREF_0209653093)

**适用NF：AMF、NSSF、SMF、SMSF、NCG**

该命令用于查询本地配置的对端NF实例支持的服务实例的版本信息。

## [注意事项](#ZH-CN_MMLREF_0209653093)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209653093)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209653093)

无

## [使用实例](#ZH-CN_MMLREF_0209653093)

查询本地配置的对端NF支持的服务实例版本信息。

```
%%LST PNFSERVICEVER:;%%
RETCODE = 0 操作成功

结果如下
------------------------
  NF实例标识 = amf_instance_0
服务实例标识 = Service_Instance_0
APIURI版本号 = v1
 API完整版本 = 1.0.0
（结果个数 = 1）

----    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209653093)

| 输出项名称 | 输出项解释 |
| --- | --- |
| NF实例标识 | 该参数用于指定NF实例标识。 |
| 服务实例标识 | 该参数用于指定服务实例标识。 |
| APIURI版本号 | 该参数用于指定要在URI中用于访问API的服务实例的版本号。 |
| API完整版本 | 该参数用于指定服务访问API的完整版本信息。 |

- [注意事项](#ZH-CN_CONCEPT_0000001983893741__1.3.2.1)

#### [命令功能](#ZH-CN_CONCEPT_0000001983893741)

该命令用于修改、删除、增加网元配置对象的数据。

#### [注意事项](#ZH-CN_CONCEPT_0000001983893741)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001983893741)

G_1，管理员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001983893741)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| DEBUGINFO | 调试信息 | 可选必选说明：可选参数<br>参数含义：维护命令的自定义参数。如果不需要自定义参数，则保持默认值<br>“default”<br>。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1~128。<br>默认值：default<br>配置原则：无 |
| SERVICEINSTANCE | 服务实例 | 可选必选说明：必选参数<br>参数含义：该参数表示大颗粒服务实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，通过LST VNFC命令获取。<br>默认值：无<br>配置原则：只能填写通过LST VNFC命令查询到的管理代理标识。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001983893741)

```
设置调试配置：
SET CFGDEBUG: DEBUGINFO="default", SERVICEINSTANCE="vnfc";
```

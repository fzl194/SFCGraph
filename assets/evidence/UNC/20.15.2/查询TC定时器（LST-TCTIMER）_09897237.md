#### [命令功能](#ZH-CN_CONCEPT_0209897237)

**适用NF：PGW-C、SMF**

该命令用于查询Diameter TC定时器。

#### [注意事项](#ZH-CN_CONCEPT_0209897237)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0209897237)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0209897237)

无。

#### [使用实例](#ZH-CN_CONCEPT_0209897237)

查询Diameter TC定时器，则可以按如下配置：

```
LST TCTIMER:;
```

```

RETCODE = 0  操作成功。

TC定时器配置
------------
TC定时器时长（单位500ms）  =  3
               定时器模式  =  固定方式
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0209897237)

参见SET TCTIMER的参数说明。

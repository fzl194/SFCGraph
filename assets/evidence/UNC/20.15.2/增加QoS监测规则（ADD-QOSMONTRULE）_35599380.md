## [命令功能](#ZH-CN_MMLREF_0000001135599380)

**适用NF：SMF**

该命令用于增加QoS监测规则。

## [注意事项](#ZH-CN_MMLREF_0000001135599380)

- 该命令执行后立即生效。

- 当前版本最多可输入50条记录。
- 此配置当多条规则被匹配到后，需要从已匹配的规则集中选择一条作为生效的规则。执行以下过滤过程：

- 指定单用户的MSISDN优先级高于指定单用户的IMSI，指定单用户的规则，优先级高于未指定单用户的。
- 当以上步骤不能过滤出唯一规则，指定切片的规则，优先级高于未指定切片；
- 当以上步骤不能过滤出唯一规则，指定DNN的规则，优先级高于未指定；
- 当以上步骤不能过滤出唯一规则，指定5QI和ARP的规则，优先级高于未指定；
  5QI和ARP必须同时指定，或者同时不指定。
  当前版本不支持此命令的RESOURCENUM，MONTRATIO参数。

#### [操作用户权限](#ZH-CN_MMLREF_0000001135599380)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [使用实例](#ZH-CN_MMLREF_0000001135599380)

增加IMSI为"460030123456789"的QoS监测：

```
ADD QOSMONTRULE: MATCHMODE=IMSI, IMSI="460030123456789", MONTSWITCH=ENABLE, REQQOSMONT=DL-0&UL-0&RP-1, REPORTFREQ=EVETT-0&PERIO-0&SESRL-1;
```

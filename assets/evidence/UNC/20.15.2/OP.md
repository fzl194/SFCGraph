#### [命令功能](#ZH-CN_CONCEPT_0259036769)

该命令用于查看本地用户。

#### [注意事项](#ZH-CN_CONCEPT_0259036769)

管理级用户可以查看所有本地用户的信息，其他级别的本地用户可以查看自身登录使用的用户的信息。

> **说明**
> 此处查询的本地用户是指 **emscomm** 、 **omuser** 、 **internaluser** 用户。

#### [操作用户权限](#ZH-CN_CONCEPT_0259036769)

G_1，管理员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0259036769)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| USERNAME | 用户名 | 可选必选说明：可选参数<br>参数含义：本地用户的用户名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～32。不区分大小写。<br>默认值：无 |
| USERGROUPNAME | 用户组 | 可选必选说明：可选参数<br>参数含义：本地用户的用户组。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～32。系统缺省定义了十个用户组：manage-ug（管理用户组）、system-ug（系统用户组）、monitor-ug（监视用户组）、visit-ug（参观用户组）、ems-ug（网管用户组）、oper-manage-ug（管理操作组）、oper-system-ug（系统操作组）、time-client-ug（time-client用户组）、mano-ug（mano用户组）、internal-ug（内联用户组），其中oper-manage-ug（管理操作组）、oper-system-ug（系统操作组）、time-client-ug（time-client用户组）和internal-ug（内联用户组）为内部使用用户组，不支持本地用户加入。<br>默认值：无 |

#### [输出结果说明](#ZH-CN_CONCEPT_0259036769)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 用户名 | 本地用户的用户名。 |
| 用户组 | 本地用户的用户组。 |
| 密码 | 本地用户的密码。 |
| 是否修改过密码 | 该用户是否修改过自己的密码。 |
| 密码设置时间 | 当前密码的设置时间。 |
| 密码过期状态 | 密码是否过期。 |
| 用户状态 | 用户的状态。 |
| 是否锁定 | 用户是否被锁定。 |
| 剩余锁定时间 | 距锁定结束剩余时间。 |
| SSH服务 | 用户能否采用SSH方式登录。 |
| HTTP服务 | 用户能否采用HTTP方式登录。 |
| 接入限制数目 | 接入限制数目。 |
| 当前在线的用户数目 | 在线用户占用的会话数目。 |
| 用户老化周期（天） | 用户老化周期。 |
| 密码有效期（天） | 密码有效期。<br>“0”<br>表示永不过期。<br>“NULL”<br>表示65535，此时密码有效期取默认值18250天。 |
| 用户有效结束时间 | 用户有效的截止日期，超出该日期后，用户无效。 |
| 是否限制登录时间 | 是否限制用户只能在配置的允许登录时间范围内登录。 |
| 允许登录起始星期 | 允许该本地用户登录的起始星期。 |
| 允许登录结束星期 | 允许该本地用户登录的结束星期。 |
| 允许登录开始时间 | 允许该本地用户登录的起始时间。 |
| 允许登录结束时间 | 允许该本地用户登录的结束时间。 |
| 上次登录时间 | 最近一次登录的时间。 |
| 二次授权 | 是否允许二次授权。 |
| 重置后首次登录是否强制修改密码 | 重置后该用户首次登录是否强制修改密码 |

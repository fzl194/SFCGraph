## [操作场景](#ZH-CN_OPI_0181812385)

如果某地区因手机DNN参数设置错误，导致用户携带DNN的不被选择的切片支持，并且所选切片中不包含默认DNN，从而造成的PDU会话激活成功率低，建议开启本特性。

- 完成加载License。

## [操作步骤](#ZH-CN_OPI_0181812385)

**以下步骤在AMF上执行**

1. 进入 “MML命令行-UNC” 窗口。
2. 打开license配置开关。
  [**SET LICENSESWITCH**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
  [**SET LICENSESWITCH**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. 配置切片纠正信息。
  **[ADD LOCALNSDNN](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/移动性管理/NF发现和选择管理/网络切片和DNN纠正管理/增加网络切片或DNN纠正配置（ADD LOCALNSDNN）_09652139.md)**
4. **可选：**配置基于DNN配置PDU会话激活处理策略。当运营商需要限制某些特定DNN的用户接入网络，或者PDU会话激活流程中，若签约数据DNN匹配失败，需要AMF拒绝UE的PDU激活请求，可以通过配置本步骤实现。
  **[ADD PDUACTCTRL](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/移动性管理/NF发现和选择管理/PDU会话激活控制参数/增加PDU会话激活控制参数（ADD PDUACTCTRL）_19641130.md)**
5. 配置DNN纠正信息及DNN纠正策略。
  **[ADD LOCALNSDNN](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/移动性管理/NF发现和选择管理/网络切片和DNN纠正管理/增加网络切片或DNN纠正配置（ADD LOCALNSDNN）_09652139.md)**

**以下步骤在SMF上执行**

1. 进入 “MML命令行-UNC” 窗口。
2. 5G用户接入时将AMF携带的SelectedDNN作为用户请求DNN使用。
  **[SET SMFSOFTPARAOFBIT](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/软件参数管理/设置SMF软件参数比特位（SET SMFSOFTPARAOFBIT）_09652696.md)**

## [任务示例](#ZH-CN_OPI_0181812385)

任务描述

为IMSI前缀为12303的用户配置切片纠正信息，网络切片为SST=1，SD=010101。然后，为IMSI前缀为12303的用户配置DNN纠正信息，网络切片为SST=1，SD=010101（eMBB），AMF本地配置的DNN是Huawei，DNN纠正策略为本地配置DNN优先。

脚本

//以下步骤在AMF上执行。

//打开本特性的License配置开关。

```
SET LICENSESWITCH:LICITEM="LKV2DNNC01",SWITCH=ENABLE;
```

//为IMSI前缀为12303的用户配置切片纠正信息，网络切片为SST=1。

```
ADD LOCALNSDNN:SUBRANGE=IMSI_PREFIX,IMSIPRE="12303",CORRECTTYPE=NS_CORRECTION,SST=1,SD="010101";
```

//为IMSI前缀为12303的用户配置DNN纠正信息，网络切片为SST=1，AMF本地配置的DNN是Huawei，DNN纠正策略为本地配置DNN优先。

```
ADD LOCALNSDNN:SUBRANGE=IMSI_PREFIX,IMSIPRE="12303",CORRECTTYPE=DNN_CORRECTION,SST=1,SD="010101",DNN="Huawei",DNNPLCY=Configured_DNN_Preferred;
```

//以下步骤在SMF上执行。

//5G用户接入时将AMF携带的SelectedDNN作为用户请求DNN使用。

```
SET SMFSOFTPARAOFBIT: DT=Dw, DWORDNUM=1039, VALUE=VALUE_1, POSITION=21;
```

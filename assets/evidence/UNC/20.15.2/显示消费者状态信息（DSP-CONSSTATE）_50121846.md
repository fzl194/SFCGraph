#### [命令功能](#ZH-CN_CONCEPT_0000001550121846)

该命令用于显示Diameter消费者状态信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001550121846)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550121846)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550121846)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001550121846)

显示Diameter消费者状态信息：

```
DSP CONSSTATE:;
```

```

RETCODE = 0 操作成功

结果如下
-------------------------
用户PID   状态   发送信息计数 接收信息计数

0x1da4653 激活   3            4
0x2de4651 激活   3            4
(结果个数 = 2)
--- END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001550121846)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 用户PID | 该参数用于表示消费者PID。 |
| 状态 | 该参数用于表示消费者当前状态。 |
| 发送信息计数 | 该参数用于表示Diameter发送到消费者的消息统计。 |
| 接收信息计数 | 该参数用于表示Diameter接收该消费者的消息统计。 |

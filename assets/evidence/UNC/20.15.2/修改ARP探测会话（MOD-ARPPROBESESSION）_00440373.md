#### [命令功能](#ZH-CN_CONCEPT_0000001600440373)

该命令用于修改ARP探测会话的参数。

#### [注意事项](#ZH-CN_CONCEPT_0000001600440373)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600440373)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600440373)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IFTYPE | 接口类型 | 可选必选说明：必选参数<br>参数含义：该参数用于指定ARP探测会话的接口类型。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- EthTrunk：Eth-Trunk接口。<br>默认值：无 |
| SESSIONIP | ARP探测会话IPv4地址 | 可选必选说明：必选参数<br>参数含义：该参数用于指定ARP探测会话的目的IPv4地址。<br>数据来源：本端规划<br>取值范围：IPv4地址类型。IP地址0.0.0.0～255.255.255.255。<br>默认值：无 |
| IFNAME | 接口名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定ARP探测会话的接口名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。<br>默认值：无<br>配置原则：<br>- 该参数必须先由ADD INTERFACE命令定义，才能在此处索引。并且只能为捆绑口或其子接口，子接口必须是vlan-type dot1q类型。<br>- 接口下需要有和ARP探测会话IPv4地址同网段的配置地址，接口地址不支持DHCP动态分配的地址。 |
| INTERVAL | 探测间隔（s） | 可选必选说明：条件可选参数<br>前提条件：该参数在“IFTYPE”配置为“EthTrunk”时为可选参数。<br>参数含义：该参数用于指定ARP探测会话的探测间隔。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～60。<br>默认值：无 |
| FAILTHRESH | 探测失败门限 | 可选必选说明：条件可选参数<br>前提条件：该参数在“IFTYPE”配置为“EthTrunk”时为可选参数。<br>参数含义：该参数用于指定ARP探测会话的探测失败门限，单位为次。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为3～255。<br>默认值：无 |
| RECOVERTHRESH | 探测失败恢复门限 | 可选必选说明：条件可选参数<br>前提条件：该参数在“IFTYPE”配置为“EthTrunk”时为可选参数。<br>参数含义：该参数用于指定ARP探测会话的探测失败恢复门限，单位为次。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为2～255。<br>默认值：无 |
| ISSWAPENABLE | 是否使能激活口切换 | 可选必选说明：条件可选参数<br>前提条件：该参数在“IFTYPE”配置为“EthTrunk”时为可选参数。<br>参数含义：该参数用于指定ARP探测会话的激活口切换使能标记。<br>数据来源：本端规划<br>取值范围：布尔类型，输入格式为“TRUE”或者“FALSE”。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600440373)

修改ARP探测会话参数：

```
MOD ARPPROBESESSION:IFTYPE=EthTrunk,IFNAME="Eth-Trunk1.1",SESSIONIP="10.10.10.10",INTERVAL=3,FAILTHRESH=6,RECOVERTHRESH=6,ISSWAPENABLE=TRUE;
```

#### [命令功能](#ZH-CN_CONCEPT_0000001600841297)

该命令用于显示本设备上建立成功的隧道的基本信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600841297)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600841297)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600841297)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| TUNNELID | 隧道ID | 可选必选说明：可选参数<br>参数含义：该参数用于表示隧道ID。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～20。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600841297)

显示指定的TUNNEL：

```
DSP TUNNELINFO: TUNNELID="0x000000000300000001";
```

```

RETCODE = 0  操作成功

结果如下
-------------------------
        隧道ID =  0x000000000300000001
      隧道类型 =  Gre
      隧道目的 =  10.1.1.1
      隧道状态 =  Down
      隧道Cost =  0
      隧道名字 =  Tunnel1
隧道出接口名字 =  Ethernet65/0/3.101
    隧道下一跳 =  10.1.1.0
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600841297)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 隧道ID | 用于表示隧道ID。 |
| 隧道类型 | 用于表示隧道类型。 |
| 隧道目的 | 用于表示隧道目的地址。 |
| 隧道状态 | 用于表示隧道状态。 |
| 隧道Cost | 用于表示隧道Cost。 |
| 隧道名字 | 用于表示隧道名称。 |
| 隧道出接口名字 | 用来指定隧道的出接口名字。 |
| 隧道下一跳 | 用来指定隧道下一跳IP地址。 |

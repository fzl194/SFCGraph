## [操作场景](#ZH-CN_OPI_0172660317)

本操作指导介绍在运行网络中激活语音业务的位置信息策略控制功能的操作过程。

> **说明**
> 适用于MME。

## [必备事项](#ZH-CN_OPI_0172660317)

前提条件

- 请仔细阅读[特性概述](特性概述_70014751.md)。
- 已完成加载License。
- 请先完成S6a的接口数据配置，请参见[配置到HSS的数据](../../../../初始配置/UNC初始配置与调测/组网对接配置/配置AMF&MME&SGSN/配置MME/配置到HSS的数据_29306465.md)。

数据

该操作无需准备数据。

## [操作步骤](#ZH-CN_OPI_0172660317)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. **可选：**设置MME支持位置信息查询。
    - 针对所有用户设置MME支持位置信息查询。
      [**SET DMCMPT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性配置/设置Diameter兼容性(SET DMCMPT)_26306080.md)
    - 针对部分用户（本网用户、指定外网用户或指定IMSI前缀用户）设置MME支持位置信息查询。
          a. [**LST DMCMPT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性配置/查询Diameter兼容配置(LST DMCMPT)_72345869.md)

  > **说明**
  > 开启该特性后：
  >
  > - 被叫场景下，被叫用户处于CONNECTED状态，查询实时位置信息流程时，本特性立即生效。
  > - 被叫场景下，查询本地位置信息流程时，本特性立即生效。
  > - 被叫场景下，被叫用户处于IDLE状态，查询实时位置信息流程时，需要paging后，通过Service Request再次与系统建立连接后，本特性立即生效。

## [任务示例](#ZH-CN_OPI_0172660317)

任务描述

开启语音业务的位置信息策略控制功能，假设仅本网HSS支持“位置信息重获取”，其他用户的归属HSS都不支持该功能。

脚本

// 进入 “MML命令行-UNC” 窗口。

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2VLPC01", SWITCH=ENABLE;
```

//本网用户以外的其他用户的归属HSS都不支持“位置信息重获取”。

```
SET DMCMPT: SF=STATE_LOCATION_INFO_RETRIEVAL-0;
```

//设置本网HSS支持“位置信息重获取”。

```
ADD DMCMPTBYIMSI: SUBRANGE=HOME_USER, NOID=0, SF=STATE_INFORMATION_RETRIEVAL-1;
```

## [操作步骤](#ZH-CN_OPI_0164009949)

- 独立部署
  已经部署了 [WSFD-102001 VoLTE基础语音业务](../WSFD-102001 VoLTE基础语音业务_67023607.md) 特性基础上，再部署本特性。例如升级局点，升级后叠加部署本特性。
    1. 进入 “MML命令行-UNC” 窗口。
    2. 配置跟踪区群组信息。
          a. 配置跟踪区群组。
            [**ADD TAGP**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/跟踪区管理/跟踪区群组管理/增加TA群组(ADD TAGP)_26145580.md)
          b. 配置跟踪区群组成员。
            [**ADD TAGPMEM**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/跟踪区管理/跟踪区群组成员管理/增加TA群组成员(ADD TAGPMEM)_72225263.md)
    3. 配置跟踪区群组语音策略。
      [**SET IMSVOPS**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/VoPS管理/设置VoPS配置(SET IMSVOPS)_72345711.md)
    4. 打开本特性的License配置开关。
      [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
- 同时部署
  同时部署 [WSFD-102001 VoLTE基础语音业务](../WSFD-102001 VoLTE基础语音业务_67023607.md) 和本特性。例如在新建局点部署VoLTE语音业务，且MME覆盖区域仅有部分E-UTRAN支持VoLTE语音业务。
    1. 进入 “MML命令行-UNC” 窗口。
    2. 配置跟踪区群组信息。
          a. 配置跟踪区群组。
            [**ADD TAGP**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/跟踪区管理/跟踪区群组管理/增加TA群组(ADD TAGP)_26145580.md)
          b. 配置跟踪区群组成员。
            [**ADD TAGPMEM**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/跟踪区管理/跟踪区群组成员管理/增加TA群组成员(ADD TAGPMEM)_72225263.md)
    3. 配置跟踪区群组语音策略。
      [**SET IMSVOPS**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/VoPS管理/设置VoPS配置(SET IMSVOPS)_72345711.md)
    4. 打开本特性的License配置开关。
      [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
    5. 打开VoLTE基础语音业务特性的License开关。
      [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)

## [任务示例](#ZH-CN_OPI_0164009949)

任务描述

运营商允许用户在特定跟踪区内使用VoLTE语音业务，不允许用户在其他区域使用VoLTE语音业务。

脚本

// 进入 “MML命令行-UNC” 窗口。

//配置跟踪区群组信息。

```
ADD TAGP: TAGPID=1, TANAME="TA_1_SUPPORT_IMSVOPS";
```

```
ADD TAGPMEM: TAGPID=1, MCC="123", MNC="03", TAC="112", TACRANGE="350";
```

//配置跟踪区群组语音策略。

```
SET IMSVOPS: MMEHOMO=UNSPECIFIED, IMSVOPSTA=WHITELST, TAGPID=1;
```

//打开本特性的License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2VPCL01", SWITCH=ENABLE;
```

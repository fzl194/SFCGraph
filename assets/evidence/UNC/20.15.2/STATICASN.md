## [命令功能](#ZH-CN_MMLREF_0000001140108905)

**适用NF：PGW-C、SGW-C、GGSN**

该命令用来查看本地配置的SGSN、SGW-C信令地址和ASN映射的表项。

## [注意事项](#ZH-CN_MMLREF_0000001140108905)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001140108905)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001140108905)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SRVNODEIP | Service Node信令IP地址 | 可选必选说明：可选参数<br>参数含义：该参数用于设置SGSN、SGW-C的信令IP。<br>数据来源：本端规划<br>取值范围：IPv4地址类型。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001140108905)

显示所有ASN配置信息：

```
LST STATICASN:;
RETCODE = 0  操作成功。

ASN配置信息
-----------
Service Node信令IP  =  10.13.13.0
            IP掩码  =  255.255.255.0
               ASN  =  1
(结果个数 = 1)
---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001140108905)

| 输出项名称 | 输出项解释 |
| --- | --- |
| Service Node信令IP地址 | 该参数用于设置SGSN、SGW-C的信令IP。 |
| IP掩码 | 该参数用于设置SGSN、SGW-C信令IP的掩码。 |
| ASN | 该参数用于设置SGSN、SGW-C对应的ASN值。 |

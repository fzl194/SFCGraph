# 删除5G用户允许的ARP列表（RMV SMFALLOWEDARPS）

## [命令功能](#ZH-CN_MMLREF_0000001258840355)

**适用NF：SMF**

该命令用于删除5G用户允许的ARP列表。

## [注意事项](#ZH-CN_MMLREF_0000001258840355)

- 命令执行后只对新接入用户生效。

- 删除前请确认和"ISMFDFTQOSCTRL"命令及"SMFNONDFTQOSCTL"命令已不存在引用关系，否则不能删除。

#### [操作用户权限](#ZH-CN_MMLREF_0000001258840355)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001258840355)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| QOSINDEX | 用户QOS索引 | 可选必选说明：必选参数<br>参数含义：该参数表示允许用户接入的ARP列表索引。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~65535。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001258840355)

删除"允许用户接入的ARP列表索引"为1的5G用户允许的ARP列表，执行如下命令:

```
%%RMV SMFALLOWEDARPS: QOSINDEX=2;%%
RETCODE = 0  Operation succeeded

---    END
```

# 显示禁止的NF（DSP INHREGNF）

- [参数说明](#ZH-CN_MMLREF_0209652300__1.3.4)

## [命令功能](#ZH-CN_MMLREF_0209652300)

**适用NF：NRF**

该命令用于在NRF上查询通过INH REGNF命令禁止的NF实例。

## [注意事项](#ZH-CN_MMLREF_0209652300)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209652300)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652300)

无

## [使用实例](#ZH-CN_MMLREF_0209652300)

显示当前NRF上通过INH REGNF命令设置的被禁止的NF实例：

```
DSP INHREGNF:;
%%DSP INHREGNF:;%%
RETCODE = 0  执行成功

结果如下
-------------------------
NF实例标识       

123e4567-e89b-12d3-a456-426655440000     
123e4567-e89b-12d3-a456-426655440001      
123e4567-e89b-12d3-a456-426655440002
(结果个数 = 3)
```

## [输出结果说明](#ZH-CN_MMLREF_0209652300)

| 输出项名称 | 输出项解释 |
| --- | --- |
| NF实例标识 | 该参数表示NRF上指定SUSPENDED状态的NF实例标识。可以通过DSP REGNFINSTANCE命令查询NF实例标识。 |

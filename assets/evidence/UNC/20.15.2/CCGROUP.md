## [命令功能](#ZH-CN_MMLREF_0000001188733219)

**适用NF：PGW-C、GGSN**

该命令用于查询计费特征组。

## [注意事项](#ZH-CN_MMLREF_0000001188733219)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001188733219)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001188733219)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| CCGROUPNAME | 计费特征组名称 | 可选必选说明：可选参数<br>参数含义：该参数用于计费特征组的名字，在系统内唯一。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~31。由英文字母（大小写）、数字、下划线构成，不区分大小写。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001188733219)

查询“计费特征组名称”为“c1”的计费特征组配置：

```
%%LST CCGROUP: CCGROUPNAME="c1";%%
RETCODE = 0  操作成功

结果如下
------------------------
计费特征组名称  =  c1
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001188733219)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 计费特征组名称 | 该参数用于计费特征组的名字，在系统内唯一。 |

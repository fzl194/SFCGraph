## [操作场景](#ZH-CN_OPI_0000001238657599)

本操作用于指导在SMF侧调测5G LAN相关业务，验证5G LAN业务功能是否成功开启。

## [必备事项](#ZH-CN_OPI_0000001238657599)

前提条件

完成 [配置5G LAN业务基本功能](配置5G LAN业务基本功能_93337664.md) 。

数据

| 类别 | 参数名称 | 取值样例 | 获取方法 | 说明 |
| --- | --- | --- | --- | --- |
| 用户信息 | IMSI号码(IMSI) | 123030913121001 | 测试终端自带 | - |
| 测试终端使用的DNN | APN名称（APN） | apn-test | 已配置数据中获取 | 用户激活使用的DNN，可以使用命令<br>**LST APN**<br>进行查询。 |

## [操作步骤](#ZH-CN_OPI_0000001238657599)

1. 创建用户跟踪任务，消息类型选择N4、N10。
2. 使用专用DNN “apn_test” 的5G LAN终端激活PDU会话。
3. 查询用户的PDU会话信息确认会话是否符合预期。
  预期结果：UE的会话创建流程中，选择到IP地址为192.168.126.11的UPF插入了用户会话
  [**DSP PDUSESSION**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/接入管理运维/查询PDP上下文信息/显示PDU会话（DSP PDUSESSION）_09653215.md)

  ```
  DSP PDUSESSION:QUERYTYPE=IMSI, IMSI="123000123456789", DSPINFOTYPE=SIMPLE;
  RETCODE = 0  操作成功  

  结果如下 
  --------------  
              国际移动用户标识 = 123030913121001
                              ...
               用户位置信息TAI = 1230100060001
                              ...
           
  UP的PFCP IPv4地址 = 192.168.126.11

                              ...
                  
  5G LAN组ID = 1-12301-2D

                              ...
  (结果个数 = 1) 
  ---    END
  ```

  ```
  是，执行
  5
  。
  否，执行
  4

  ```
4. 根据 [配置5G LAN业务基本功能](配置5G LAN业务基本功能_93337664.md) 确认配置是否正确。
  ```
  是，执行
  6
  。
  否，修改配置后，并再次执行
  3
  。
  ```
5. 查看用户跟踪。确认UDM给SMF返回的签约信息中是否携带VnGroupData。
  ```
  是，结束。
  否，执行
  6

  ```

    b. 收集并保存上述所有查询信息。
    c. 收集归纳所有信息并联系华为技术支持解决。

# 删除AMF功能实体配置（RMV AMFFUNCTION）

- [命令功能](#ZH-CN_MMLREF_0209652619__1.3.1)
- [注意事项](#ZH-CN_MMLREF_0209652619__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0209652619__1.3.4)
- [使用实例](#ZH-CN_MMLREF_0209652619__1.3.5)

## [命令功能](#ZH-CN_MMLREF_0209652619)

**适用NF：AMF**

该命令用于删除AMF功能实体配置。

## [注意事项](#ZH-CN_MMLREF_0209652619)

- 该命令执行后立即生效。

- 删除该配置可能会导致AMF与北向网管对接异常，请谨慎操作。

#### [操作用户权限](#ZH-CN_MMLREF_0209652619)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652619)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| INSTANCEID | NF实例号 | 可选必选说明：必选参数<br>参数含义：NF实例号。用于AMF与北向网管对接使用，通过NFInstance ID可以实现与北向网管上与网元的话统、告警信息的关联。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~40。<br>默认值：无<br>配置原则：<br>该参数需要根据北向网管的要求来填写，例如，填写为在MANO上创建VNF时的InstanceID。 |

## [使用实例](#ZH-CN_MMLREF_0209652619)

删除InstanceID为“b7b621d82dfb4a009d492491bd9d72a4”的AMF功能实体。

```
RMV AMFFUNCTION: INSTANCEID="b7b621d82dfb4a009d492491bd9d72a4";
```

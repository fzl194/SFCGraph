# 调测透明鉴权接入

- [操作场景](#ZH-CN_OPI_0295351132__1.3.1)
- [必备事项](#ZH-CN_OPI_0295351132__1.3.2)
- [操作步骤](#ZH-CN_OPI_0295351132__1.3.3)

## [操作场景](#ZH-CN_OPI_0295351132)

当运营商需要对透明接入的用户进行Radius鉴权以获取用户属性用于更加灵活的控制用户时，需要 UNC 支持透明鉴权接入功能。此时，需要对 UNC 的透明鉴权接入功能进行调测，确保用户可以正常接入ISP网络及Internet。

> **说明**
> 适用于 GGSN、 PGW-C、SMF。

## [必备事项](#ZH-CN_OPI_0295351132)

前提条件

- 请仔细阅读[WSFD-011305 Radius鉴权接入特性概述](../特性概述_50176278.md)。
- 完成[激活Radius鉴权接入](../激活Radius鉴权接入_50176279.md)。
- AAA Server工作正常，用户的鉴权信息已在AAA Server上配置。

数据

| 类别 | 参数名称 | 取值样例 | 获取方法 | 说明 |
| --- | --- | --- | --- | --- |
| 用户信息 | 国际移动用户识别码（IMSI） | 123000123456789 | 测试终端自带 | - |
| 用户APN信息 | APN名称（APN） | huawei.com | 测试终端自带 | - |
| 鉴权服务器 | Radius Server Group名称（RDSSVRGRPNAME） | isprg | 已配置数据中获取 | 已在<br>[配置RADIUS功能](../../WSFD-011306 Radius功能/激活RADIUS功能/配置RADIUS功能_32909765.md)<br>中配置。 |
| 鉴权服务器 | IPv4地址（SERVERIPV4） | 10.168.10.1 | 已配置数据中获取 | 已在<br>[配置RADIUS功能](../../WSFD-011306 Radius功能/激活RADIUS功能/配置RADIUS功能_32909765.md)<br>中配置。 |
| 鉴权服务器 | 服务器密钥（加密的）（CIPHERKEY） | ispchina | 已配置数据中获取 | 已在<br>[配置RADIUS功能](../../WSFD-011306 Radius功能/激活RADIUS功能/配置RADIUS功能_32909765.md)<br>中配置。 |

工具

测试终端

## [操作步骤](#ZH-CN_OPI_0295351132)

1. 在 UNC OM Portal上建立用户跟踪。
2. 测试终端使用“huawei.com”接入网络。
3. 进入 “MML命令行-UNC” 窗口。
4. 查询用户的PDU会话信息。
  **[DSP PDUSESSION](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/接入管理运维/查询PDP上下文信息/显示PDU会话（DSP PDUSESSION）_09653215.md)**
  ```
  DSP PDUSESSION:QUERYTYPE=IMSI, IMSI="123000123456789", DSPINFOTYPE=DETAILED, WLNETWKTYPE=NW5G, PDUSESSIONID=5, QFI=1;
  RETCODE = 0  操作成功  

  结果如下 
  --------------  
                           IMSI = 123000123456789
                              ...
                     APN或者DNN = huawei.com
                              ...
  (结果个数 = 1) 
  ---    END
  ```
    - 预期结果：用户接入成功并创建会话，且“APN或者DNN”与实际规划值一致，请执行[5](#ZH-CN_OPI_0295351132__step10264021193420)。
    - 非预期结果：用户创建会话失败，或者“APN或者DNN”与实际规划值不一致，执行[6](#ZH-CN_OPI_0295351132__cmd1839812318)。
5. 查看用户跟踪任务。检查 UNC 向AAA Server发送的 “Access Request” 鉴权请求消息中携带的用户名是否与规划值一致且密码加密。
  **图1** Access Request鉴权请求消息

  <br>

  ![](调测透明鉴权接入_95351132.assets/zh-cn_image_0295406985_2.png)

  <br>
    - 如果“user-name”与规划值一致，“user-password”加密，用户透明鉴权接入成功，调测结束。
    - 如果“user-name”与规划值不一致或密码未加密，请执行[6](调测非透明接入_95351131.md#ZH-CN_OPI_0295351131__step18837893116)。
6. 执行 **[LST APNAUTHATTR](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/APN管理/APN鉴权属性/查询APN鉴权属性配置（LST APNAUTHATTR）_28567627.md)** 查询APN鉴权属性。
    - 如果APN的接入模式与规划值一致，请执行[7](#ZH-CN_OPI_0295351132__cmd841512585413)。
    - 如果APN的接入模式与规划值不一致，请参考[激活Radius鉴权接入](../激活Radius鉴权接入_50176279.md)重新配置。
7. 执行 **[LST RDSSVRGRP](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/RADIUS管理/连接管理/Radius服务器组/查询Radius服务器组（LST RDSSVRGRP）_09896733.md)** 查询APN的鉴权服务器配置。
    - 如果Radius服务器组的参数值与规划值一致，请执行[8](#ZH-CN_OPI_0295351132__cmd8911102805110)。
    - 如果Radius服务器组的参数值与规划值不一致，请参考[激活RADIUS功能](../../WSFD-011306 Radius功能/激活RADIUS功能_15542173.md)重新配置后再次执行[2](调测非透明接入_95351131.md#ZH-CN_OPI_0295351131__step19263131225014)。若执行结果仍然错误，请执行[8](调测非透明接入_95351131.md#ZH-CN_OPI_0295351131__step2091112885113)。
8. 收集信息并寻求技术支持。
    a. 执行 **[EXP MML](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/配置管理/配置导出管理/导出MML文件（EXP MML）_47200033.md)** 命令将当前配置数据导出为MML脚本文件并保存。
    b. 收集并保存上述所有查询信息。
    c. 收集归纳所有信息并联系华为技术支持解决。

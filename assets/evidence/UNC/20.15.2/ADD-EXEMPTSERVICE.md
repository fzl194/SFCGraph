## [命令功能](#ZH-CN_MMLREF_0000001181558909)

**适用NF：PGW-C、SMF、GGSN**

该命令用于增加豁免业务。豁免业务（3GPP PS data off Exempt Services）不受3GPP PS data off能力控制，即使3GPP PS data off能力使能，这类业务也不受影响。

## [注意事项](#ZH-CN_MMLREF_0000001181558909)

- 该命令执行后立即生效。

- 当3GPP PS data off功能使能时，即SET PSDATAOFFFUNC的PSDATAOFFSWITCH为ENABLE（使能）时，该命令才生效。
- IMS业务不需要配置，直接为豁免业务。

- 最多可输入1000条记录。

#### [操作用户权限](#ZH-CN_MMLREF_0000001181558909)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001181558909)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| EXEMPTAPN | 豁免APN | 可选必选说明：必选参数<br>参数含义：该参数用于配置豁免业务的APN，APN指真实APN。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~63。只能由“-”、数字、大小写字母和“.”组成，不能以“.”开头且不能出现连续两个“.”。不支持空格及“_”、“#”、“$”、“&”、“%”、“^”、“（”、“）”、“，”、“/”、“;”、“:”、“””、“`”等特殊字符，不区分大小写。<br>默认值：无<br>配置原则：<br>输入的APN名称需要符合APN命名规则。 |

## [使用实例](#ZH-CN_MMLREF_0000001181558909)

当运营商需要增加豁免业务，执行如下命令：

```
ADD EXEMPTSERVICE: EXEMPTAPN="huawei.com";
```

## [操作场景](#ZH-CN_OPI_0276144609)

本操作指导介绍在运行网络中激活CDMA与LTE的非优化切换特性的操作过程。

CDMA与LTE的非优化切换特性是指用户从LTE网络移动到CDMA网络或者从CDMA网络移动到LTE网络时，业务能平滑切换，保证业务的连续性。

> **说明**
> 适用于MME。

## [必备事项](#ZH-CN_OPI_0276144609)

前提条件

- 请仔细阅读[WSFD-104504 CDMA与LTE的非优化切换](../WSFD-104504 CDMA与LTE的非优化切换_76144606.md)。
- 完成加载License。

数据

该操作无需准备数据。

## [操作步骤](#ZH-CN_OPI_0276144609)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. **可选：**设置PDN GW Identity的类型参数。
  根据3GPP 23.402要求：Notify Request只能携带Hostname和IP地址其中一种。由于不能确保所有厂商的网元都按协议要求来填写，如存在PDN GW Identity不一致时，可能会导致后续业务流程失败。因此 UNC 可通过该配置，来选择给HSS上报的是FQDN或者是IP地址。
  [**SET DMCMPT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性配置/设置Diameter兼容性(SET DMCMPT)_26306080.md)
4. **可选：**若需针对不同类型用户（本网用户、指定外网用户以及指定IMSI前缀用户）选择给其归属的HSS上报Hostname或者是IP地址，则执行如下步骤。
    a. [**LST DMCMPT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性配置/查询Diameter兼容配置(LST DMCMPT)_72345869.md)
      查询系统中Diameter兼容性配置

      ![](激活CDMA与LTE的非优化切换_76144609.assets/notice_3.0-zh-cn_2.png)

      命令 [**ADD DMCMPTBYIMSI**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性IMSI号段配置/增加IMSI对应的Diameter兼容性(ADD DMCMPTBYIMSI)_72225977.md) 与 [**SET DMCMPT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性配置/设置Diameter兼容性(SET DMCMPT)_26306080.md) 中除 “用户范围” 、 “运营商标识” 以及 “IMSI前缀” 外的其余参数功能均相同，且若系统中同时存在这两条命令的配置，系统会优先匹配命令 [**ADD DMCMPTBYIMSI**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性IMSI号段配置/增加IMSI对应的Diameter兼容性(ADD DMCMPTBYIMSI)_72225977.md) 的配置。因此在配置 [**ADD DMCMPTBYIMSI**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性IMSI号段配置/增加IMSI对应的Diameter兼容性(ADD DMCMPTBYIMSI)_72225977.md) 前需先执行 [**LST DMCMPT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性配置/查询Diameter兼容配置(LST DMCMPT)_72345869.md) 查询系统中 [**SET DMCMPT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性配置/设置Diameter兼容性(SET DMCMPT)_26306080.md) 的配置记录，确保命令 [**ADD DMCMPTBYIMSI**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性IMSI号段配置/增加IMSI对应的Diameter兼容性(ADD DMCMPTBYIMSI)_72225977.md) 的配置记录中所携带的参数默认值不影响系统已有配置。
    b. 设置PDN GW Identity的类型参数
      [**ADD DMCMPTBYIMSI**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性IMSI号段配置/增加IMSI对应的Diameter兼容性(ADD DMCMPTBYIMSI)_72225977.md)

## [任务示例](#ZH-CN_OPI_0276144609)

任务描述

启用CDMA与LTE的非优化切换。

脚本

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2NOHC01", SWITCH=ENABLE;
```

//设置PDN GW Identity的类型参数。

```
SET DMCMPT: PGWIDTYPE=IP_ADDRESS;
```

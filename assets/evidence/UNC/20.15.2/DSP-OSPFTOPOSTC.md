#### [命令功能](#ZH-CN_CONCEPT_0000001600601069)

该命令用于查询OSPF拓扑的统计信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600601069)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600601069)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600601069)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | OSPF进程号 | 可选必选说明：可选参数<br>参数含义：该参数用于表示OSPF进程号，未指定OSPF进程号时默认查询所有OSPF进程。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600601069)

查询OSPF拓扑的统计信息：

```
DSP OSPFTOPOSTC:PROCID=2;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600601069)

| 输出项名称 | 输出项解释 |
| --- | --- |
| OSPF进程号 | 用于表示OSPF进程号，未指定OSPF进程号时默认查询所有OSPF进程。 |
| OSPF区域号 | 用于表示OSPF区域号。 |
| 路由器标识 | 用于表示路由器标识。 |
| SPT节点总数 | 用于表示SPT节点总数，是SPT路由器节点数量与SPT网络类型节点数量之和。 |
| SPT路由器节点数量 | 用于表示SPT路由器节点数量。 |
| SPT网络类型节点数量 | 用于表示SPT网络类型节点数量。 |
| 参与SPT计算的节点数量 | 用于表示参与SPT计算的节点数量。 |
| 隔离的节点数量 | 用于表示未参与SPT计算的节点数量。 |
| SPT计算最大距离的节点数量 | 用于表示SPT计算最大距离的节点数量。 |
| Link数量 | 用于表示Link总数，是父节点到子节点的Link数量与子节点到父节点的Link数量之和。 |
| 父节点到子节点的Link数量 | 用于表示父节点到子节点的Link数量。 |
| 子节点到父节点的Link数量 | 用于表示子节点到父节点的Link数量。 |
| SPF状态 | 该参数用于表示SPF状态，包括：<br>- 未启动。<br>- 已完成。<br>- 正在运行。 |

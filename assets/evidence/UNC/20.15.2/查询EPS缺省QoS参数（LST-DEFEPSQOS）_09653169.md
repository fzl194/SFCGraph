## [命令功能](#ZH-CN_MMLREF_0209653169)

**适用NF：PGW-C、SGW-C**

该命令用于显示系统配置的EPS缺省QoS参数取值。

## [注意事项](#ZH-CN_MMLREF_0209653169)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209653169)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209653169)

无

## [使用实例](#ZH-CN_MMLREF_0209653169)

显示系统配置的EPS缺省QoS参数取值：

```
%%LST DEFEPSQOS:;%%
RETCODE = 0  操作成功

缺省的EPS QoS配置信息
---------------------
                  QCI值  =  5
                  ARP值  =  3
下行保证带宽(千比特/秒)  =  400
上行保证带宽(千比特/秒)  =  400
下行最大带宽(千比特/秒)  =  500
上行最大带宽(千比特/秒)  =  500
下行APN AMBR(千比特/秒)  =  2000
上行APN AMBR(千比特/秒)  =  2000
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209653169)

| 输出项名称 | 输出项解释 |
| --- | --- |
| QCI值 | 该参数用于配置QoS流量级别。 |
| ARP值 | 该参数用于配置分配/保持优先等级。 |
| 下行保证带宽(千比特/秒) | 该参数用于配置保证的下行比特率。 |
| 上行保证带宽(千比特/秒) | 该参数用于配置保证的上行比特率。 |
| 下行最大带宽(千比特/秒) | 该参数用于配置最大的下行比特率。 |
| 上行最大带宽(千比特/秒) | 该参数用于配置最大的上行比特率。 |
| 下行APN AMBR(千比特/秒) | 该参数用于配置下行集合最大比特速率。 |
| 上行APN AMBR(千比特/秒) | 该参数用于配置上行集合的最大比特速率。 |

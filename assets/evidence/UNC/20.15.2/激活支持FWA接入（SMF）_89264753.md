## [操作场景](#ZH-CN_OPI_0000002089264753)

本操作指导介绍在运行网络中激活支持FWA接入特性的操作过程。

FWA（Fixed Wireless Access）是一种固定宽带的接入技术，指家庭或企业的各种终端设备通过CPE（Customer-Premises Equipment ）直接接入无线网络，提供用户宽带接入功能。根据接入方式可以将固定宽带的接入方式分为有线接入技术和无线接入技术两种，有线接入技术主要包括混合光纤同轴电缆（HFC）接入、铜线（xDSL）接入和光纤（FTTx）接入；无线接入技术主要就是固定无线接入（FWA）。

## [操作步骤](#ZH-CN_OPI_0000002089264753)

1. 进入 “MML命令行-UNC” 窗口。
2. 增加FWA用户匹配规则。SMF根据匹配方式和匹配类型判断当前用户是否是FWA用户。
  **[ADD FWAMATCHRULE](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/FWA用户匹配规则管理/增加FWA用户匹配规则（ADD FWAMATCHRULE）_90774333.md)**
3. 配置N11SMF支持FWA用户的识别。
  **[SET SMFSOFTPARAOFBIT](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/软件参数管理/设置SMF软件参数比特位（SET SMFSOFTPARAOFBIT）_09652696.md)**

## [任务示例](#ZH-CN_OPI_0000002089264753)

任务描述

SMF根据PCF返回的PCF Rule ID判断是否为FWA用户，若PCF返回的PCC Rule ID完全匹配“up_870000174”则SMF判断其为FWA用户。

脚本

//增加FWA用户匹配规则。

```
ADD FWAMATCHRULE: MATCHTYPE=RULE, MATCHFUNC=FULL, RULE="up_87000017";
```

//配置N11SMF支持FWA用户的识别。

```
SET SMFSOFTPARAOFBIT: DT=Dw, DWORDNUM=1040, VALUE=VALUE_1, POSITION=19;
```

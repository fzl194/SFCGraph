#### [命令功能](#ZH-CN_CONCEPT_0000001600865665)

该命令用于显示Trunk工作模式配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001600865665)

- 该命令执行后立即生效。
- 若不指定IFNAME参数时，则显示所有Trunk的工作模式配置；若指定IFNAME参数时，则可以显示特定Trunk的工作模式配置。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600865665)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600865665)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IFNAME | 接口名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定Trunk接口名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。接口名称由接口类型+接口编号组成。<br>默认值：无<br>配置原则：请使用LST INTERFACE命令查看可用接口。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600865665)

查询Eth-Trunk1的工作模式配置：

```
LST TRUNKIF:IFNAME="Eth-Trunk1";
```

```

RETCODE = 0  操作成功

结果如下
------------------------
      接口名称 =  Eth-Trunk1
      工作模式 =  Lacp
最小激活链路数 =  2
  负载分担模式 =  L4哈希算法
(结果个数 = 1)
---   END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600865665)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 接口名称 | 用于指定Trunk接口名称。 |
| 工作模式 | 用于指定Trunk接口工作模式。 |
| 最小激活链路数 | 用于指定Eth-Trunk接口的最小激活链路数。 |
| 负载分担模式 | 用于指定Eth-Trunk接口的负载分担模式。 |

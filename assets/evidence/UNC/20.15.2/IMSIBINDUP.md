## [命令功能](#ZH-CN_MMLREF_0209651580)

**适用NF：SMF、PGW-C、SGW-C、GGSN**

该命令用于查询用户和UPF的绑定关系。

## [注意事项](#ZH-CN_MMLREF_0209651580)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209651580)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209651580)

无

## [使用实例](#ZH-CN_MMLREF_0209651580)

显示所有用户UPF的绑定关系： LST IMSIBINDUP:;

```
%%LST IMSIBINDUP:;%%
RETCODE = 0  操作成功。

结果如下
------------------------

```

## [输出结果说明](#ZH-CN_MMLREF_0209651580)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 起始IMSI | 该参数用于指定IMSI号段的起始IMSI，当只有一个IMSI时，起始IMSI和末位IMSI相同。 |
| 终止IMSI | 该参数用于标识IMSI号段的最后一个IMSI，包含在号段内。当只有一个IMSI时，起始IMSI和末位IMSI相同。 |
| 接入类型 | 该参数用于指定用户的接入类型。 |
| 主锚点UPF实例名称 | 该参数用于指定主锚点UPF实例名称。<br>主锚点UPF为用户接入数据网络使用到的中心UPF锚点。 |
| 辅锚点UPF实例名称 | 该参数用于指定辅锚点UPF实例名称。<br>辅锚点UPF为用户接入数据网络使用到的边缘UPF锚点。 |
| N3/S1-U口UPF实例名称 | 该参数用于指定N3/S1-U口UPF实例名称。<br>N3/S1U口UPF是用户与锚点UPF之间的IUPF。 |
| ULCL部署模式 | 该参数用于表示ULCL部署模式。 |
| 重选到绑定UPF | 该参数用于表示是否允许用户在UP重选的场景下，选择IMSI绑定的UPF。 |
| 重选指向的辅锚点UPF实例名称 | 该参数用于重选指向的辅锚点UPF实例名称。 |
| 重选指向的接入UPF实例名称 | 该参数用于重选指向的接入UPF实例名称。 |
| 重选后的ULCL部署模式 | 该参数用于表示重选后的ULCL部署模式。 |
| 智能分流专用会话锚点UPF实例名称 | 该参数用于指定智能分流专用会话的锚点UPF。 |

# 删除等价PLMN组（RMV EPLMNGRP）

## [命令功能](#ZH-CN_MMLREF_0209651360)

**适用NF：AMF**

该命令用于删除等价PLMN组。

## [注意事项](#ZH-CN_MMLREF_0209651360)

- 该命令执行后立即生效。

- 删除等价PLMN组之前，需要删除该组内的等价PLMN成员。

#### [操作用户权限](#ZH-CN_MMLREF_0209651360)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209651360)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| GRPIDX | 等价PLMN组号 | 可选必选说明：必选参数<br>参数含义：该参数用于在UNC系统内唯一标识一个等价PLMN组。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~127。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209651360)

删除组号为1的等价PLMN组，执行如下命令：

```
RMV EPLMNGRP: GRPIDX=1;
```

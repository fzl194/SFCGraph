# 删除NF或NF服务支持的PLMN（RMV ALLOWEDPLMN）

- [命令功能](#ZH-CN_MMLREF_0209652389__1.3.1)
- [注意事项](#ZH-CN_MMLREF_0209652389__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0209652389__1.3.4)
- [使用实例](#ZH-CN_MMLREF_0209652389__1.3.5)

## [命令功能](#ZH-CN_MMLREF_0209652389)

**适用NF：AMF、SMF、NRF、NSSF、NCG**

该命令用于删除NF或NF服务实例支持的PLMN。当NF或NF服务实例不再支持为某个PLMN服务时，需要对PLMN进行删除。

## [注意事项](#ZH-CN_MMLREF_0209652389)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0209652389)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652389)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| INDEX | 索引标识 | 可选必选说明：必选参数<br>参数含义：该参数用于指定索引标识。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是0~4095。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209652389)

运营商A需要删除索引标识为0的PLMN支持信息。

```
RMV ALLOWEDPLMN: INDEX=0;
```

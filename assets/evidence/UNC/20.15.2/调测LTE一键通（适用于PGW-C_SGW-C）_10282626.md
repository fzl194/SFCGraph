# 调测 LTE一键通 （适用于PGW-C/SGW-C）

- [操作场景](#ZH-CN_OPI_0310282626__1.3.1)
- [必备事项](#ZH-CN_OPI_0310282626__1.3.2)
- [操作步骤](#ZH-CN_OPI_0310282626__1.3.3)

## [操作场景](#ZH-CN_OPI_0310282626)

调测 LTE一键通 。

## [必备事项](#ZH-CN_OPI_0310282626)

前提条件

- 请仔细阅读[WSFD-102601 LTE一键通基础功能（适用于PGW-C/SGW-C）](../WSFD-102601 LTE一键通基础功能（适用于PGW-C_SGW-C）/特性概述_10980736.md)和[WSFD-102602 LTE一键通特性概述（适用于PGW-C/SGW-C）](特性概述_10282604.md)。
- 已完成[激活LTE一键通（适用于PGW-C/SGW-C）](激活LTE一键通（适用于PGW-C_SGW-C）_10282625.md)。

数据

| 类别 | 参数名称 | 取值样例 | 获取方法 |
| --- | --- | --- | --- |
| 用户信息查询 | 国际移动用户识别码（IMSI） | 123000123456789 | 测试终端自带 |
| 测试终端使用的APN | APN名称（APN） | apn1 | 从已配置数据中获取 |

工具

- 测试终端
- OM Portal跟踪

## [操作步骤](#ZH-CN_OPI_0310282626)

1. 查询 LTE一键通 功能对应的License配置开关是否打开。
  [**LST LICENSESWITCH**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/查询License配置项开关（LST LICENSESWITCH）_09651570.md) : LICITEM="LKV2QPPT02";

  - 如果“SWITCH”为“ENABLE”，请执行[2](#ZH-CN_OPI_0310282626__step767223614014)。
    - 如果“SWITCH”为“DISABLE”，则执行[**SET LICENSESWITCH**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)命令打开本特性对应的License配置开关。
2. 在OM Portal上创建用户跟踪任务，在“参数配置”栏输入用户IMSI，其他参数：选择默认值。
3. 测试终端使用“apn1”接入网络。
    - 如果测试终端成功接入网络，请执行[4](#ZH-CN_OPI_0310282626__step446103564119)。
    - 如果测试终端无法接入网络，请调测接入功能。
4. 查看用户跟踪任务结果。检查Create Session Response消息或网络侧下发的PCC规则中携带的QCI值是否为 PTT业务QCI（ 65~67、69~70 ）。
    - 如果是正确携带，则调测完成。
    - 如果没有正确携带，请执行[5](#ZH-CN_OPI_0310282626__step46724361208)。
5. 执行 [**LST APNPTTFUNC**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/QoS/一键通/APN的一键通配置/查询基于APN的一键通功能配置（LST APNPTTFUNC）_06399911.md) 命令，查看APN下的PTT开关是否开启。
    - 如果APN的“LTEPTTSWITCH”与规划值一致，请执行[6](#ZH-CN_OPI_0310282626__step1175027121817)。
    - 如果APN的“LTEPTTSWITCH”与规划值不一致，请参考[激活LTE一键通（适用于PGW-C/SGW-C）](激活LTE一键通（适用于PGW-C_SGW-C）_10282625.md)进行重新配置。
6. 执行 [**LST PTTFUNC**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/QoS/一键通/全局一键通配置/查询一键通功能配置（LST PTTFUNC）_06399916.md) 命令，查看全局的PTT开关是否开启。
    - 如果全局的“LTEPTTSWITCH”与规划值一致，请执行[7](#ZH-CN_OPI_0310282626__step2091112885113)。
    - 如果全局的“LTEPTTSWITCH”与规划值不一致，请参考[激活LTE一键通（适用于PGW-C/SGW-C）](激活LTE一键通（适用于PGW-C_SGW-C）_10282625.md)进行重新配置。
7. 执行 [**LST STDQOSID**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/QoS/扩展QCI功能/标准QoS ID配置/查询标准QoS ID配置（LST STDQOSID）_06399918.md) 命令，查询PTT业务QCI（ 65~67、69~70 ）是否已经在 UNC 上被配置成标准QCI。
    - 如果查询结果包含PTT业务QCI（65~67、69~70），请执行[8](#ZH-CN_OPI_0310282626__cmd8911102805110)。
    - 如果不包含，请参考[激活LTE一键通（适用于PGW-C/SGW-C）](激活LTE一键通（适用于PGW-C_SGW-C）_10282625.md)进行重新配置。
8. 收集信息并寻求技术支持。
    a. 执行 **[EXP MML](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/配置管理/配置导出管理/导出MML文件（EXP MML）_47200033.md)** 命令将当前配置数据导出为MML脚本文件并保存。
    b. 收集并保存上述所有查询信息。
    c. 收集归纳所有信息并联系华为技术支持解决。

# 删除目标NF实例（RMV TNFINS）

- [注意事项](#ZH-CN_MMLREF_0209651333__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0209651333__1.3.4)
- [使用实例](#ZH-CN_MMLREF_0209651333__1.3.5)

## [命令功能](#ZH-CN_MMLREF_0209651333)

**适用NF：AMF、SMF、NSSF、SMSF、NCG**

该命令用于删除目标NF实例的配置。

## [注意事项](#ZH-CN_MMLREF_0209651333)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0209651333)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209651333)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| TNFINSINDEX | 目标NF实例索引 | 可选必选说明：必选参数<br>参数含义：本参数用于指定目标NF实例索引。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~2047。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209651333)

运营商A需要删除索引为1的目标NF实例。

```
RMV TNFINS: TNFINSINDEX=1;
```

#### [命令功能](#ZH-CN_CONCEPT_0251174353)

**适用NF：NCG**

该命令用于显示用户锁定状态。

#### [注意事项](#ZH-CN_CONCEPT_0251174353)

无。

#### [本地用户权限](#ZH-CN_CONCEPT_0251174353)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_CONCEPT_0251174353)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0251174353)

无。

#### [使用实例](#ZH-CN_CONCEPT_0251174353)

显示用户锁定状态：

```
DSP USER:;
```

```
 
RETCODE = 0  操作成功。

结果如下:
---------
用户名    锁定状态    RU名称           PULL任务标识    

BS        未锁定      CG_SP_RU_0064    Distribution_1st
BS        未锁定      CG_SP_RU_0065    Distribution_1st
(结果个数 = 2)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0251174353)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 用户名 | 用于显示用户的名称。 |
| 锁定状态 | 用于显示用户的锁定状态。 |
| RU名称 | 用于显示用户的RU名称。 |
| PULL任务标识 | 用于显示用户的PULL任务标识。 |

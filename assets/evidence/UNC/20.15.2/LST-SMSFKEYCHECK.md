## [命令功能](#ZH-CN_MMLREF_0000001530680374)

**适用NF：SMSF**

该命令用于查询SMSF用户关键信息数据核查扫描速率和周期，以及数据有效时长。

## [注意事项](#ZH-CN_MMLREF_0000001530680374)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001530680374)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001530680374)

无

## [使用实例](#ZH-CN_MMLREF_0000001530680374)

运营商希望查询SMSF用户关键信息数据核查扫描速率和周期，以及数据有效时长，执行如下命令：

```
LST SMSFKEYCHECK:;
%%LST SMSFKEYCHECK:;%%
RETCODE = 0  操作成功

结果如下：
------------------------
       SMSF用户关键信息表核查功能开关       =  打开
       SMSF用户关键信息核查速率(个每秒)        =  70
       SMSF用户关键信息核查周期(分钟)    =  60
       SMSF用户关键信息有效时长开关        =  打开
       SMSF上用户关键信息数据的有效时长(分钟) =  1440
(结果个数 = 1)
```

## [输出结果说明](#ZH-CN_MMLREF_0000001530680374)

| 输出项名称 | 输出项解释 |
| --- | --- |
| SMSF用户关键信息表核查功能开关 | 该参数用于表示SMSF用户关键信息核查功能开关。 |
| SMSF用户关键信息核查速率(个每秒) | 该参数用于表示SMSF用户关键信息核查时的扫描速率。 |
| SMSF用户关键信息核查周期(分钟) | 该参数用于表示SMSF用户关键信息核查扫描周期。 |
| SMSF用户关键信息有效时长开关 | 该参数用于表示SMSF用户关键信息有效时长生效开关。 |
| SMSF上用户关键信息数据的有效时长(分钟) | 该参数用于表示SMSF上用户关键信息数据的有效时长。 |

## [任务示例](#ZH-CN_OPI_0000002209058154)

任务描述

开启基于TA List的寻呼自动流控功能。

脚本

//设置S1 TAI组对象寻呼异常监控功能参数。

```
SET S1TAIPAGMONPARA:SWITCH=ON,PAGSUCRATDEFVAL=90, MINPAGREQTHD=1000, PAGRATEEXPTHD=5, PAGREQEXPTHD=50, PAGEXPTIMESTHD=1, PAGRETRYTIMES=0, IMSPAGINGSW=ON, ALARMSW=ON;
```

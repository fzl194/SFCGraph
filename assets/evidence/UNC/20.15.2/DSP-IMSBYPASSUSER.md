## [命令功能](#ZH-CN_MMLREF_0000002154762937)

**适用NF：SMF、PGW-C**

该命令用于显示进入语音PCF/PCRF故障Bypass状态的用户的IMSI列表。

## [注意事项](#ZH-CN_MMLREF_0000002154762937)

该命令每个进程最多显示10个用户。

#### [操作用户权限](#ZH-CN_MMLREF_0000002154762937)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000002154762937)

无

## [使用实例](#ZH-CN_MMLREF_0000002154762937)

查询进入语音PCF/PCRF故障Bypass状态的用户的IMSI列表：

```
%%DSP IMSBYPASSUSER:;%%
RETCODE = 0  操作成功

结果如下
------------------------
国际移动用户标识  =  123031500010001
         POD名称  =  uncpod-0
    策略接口类型  =  Interface_N7
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000002154762937)

| 输出项名称 | 输出项解释 |
| --- | --- |
| IMSI | 该参数用于显示用户永久标识或者国际移动用户标识。 |
| POD名称 | 该参数用于显示POD名称。 |
| 策略接口类型 | 该参数用于显示策略接口类型。 |

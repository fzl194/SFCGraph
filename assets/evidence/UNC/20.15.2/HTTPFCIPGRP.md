## [命令功能](#ZH-CN_MMLREF_0000001183813632)

该命令用于查询HTTP固定速率流控组的IP地址组信息。

## [注意事项](#ZH-CN_MMLREF_0000001183813632)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001183813632)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001183813632)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| GROUP | 流控组 | 可选必选说明：可选参数<br>参数含义：该参数用于指定HTTP固定速率流控组的唯一标识。一个流控组中可以包括多个IP地址索引。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是0~4095。<br>默认值：无<br>配置原则：无 |
| INDEX | IP地址索引 | 可选必选说明：可选参数<br>参数含义：该参数用于标识具有相同前缀的一组IP地址。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~4095。<br>默认值：无<br>配置原则：无 |
| DESC | 描述 | 可选必选说明：可选参数<br>参数含义：该参数用于填写HTTP固定速率流控地址组的描述信息，建议包含NF类型，实例名等，例如UDM_UDM001。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~255。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001183813632)

查询HTTP固定速率流控地址组信息：

```
LST HTTPFCIPGRP:
%%LST HTTPFCIPGRP:;%%
RETCODE = 0  操作成功

结果如下
------------------------
    流控组  =  1
IP地址索引  =  1
      描述  =  NULL
IP地址类型  =  IPv4
  IPv4地址  =  192.168.4.15
  IPv4掩码  =  255.255.255.0
  IPv6地址  =  ::
  IPv6前缀  =  0
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001183813632)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 流控组 | 该参数用于指定HTTP固定速率流控组的唯一标识。一个流控组中可以包括多个IP地址索引。 |
| IP地址索引 | 该参数用于标识具有相同前缀的一组IP地址。 |
| 描述 | 该参数用于填写HTTP固定速率流控地址组的描述信息，建议包含NF类型，实例名等，例如UDM_UDM001。 |
| IP地址类型 | 该参数用于指定输入的IP地址类型。 |
| IPv4地址 | 该参数用于指定HTTP固定速率流控组内一个或一组IPv4地址。本端发送请求时，使用对端IP与此参数值（一个或一组IPv4地址）进行匹配，成功则进行固定速率流控。本端接收请求时，优先使用本端IP与此参数值进行匹配，成功则进行固定速率流控，失败则再使用对端IP与此参数值进行匹配。 |
| IPv4掩码 | 该参数用于指定流控IPv4地址的掩码。 |
| IPv6地址 | 该参数用于指定HTTP固定速率流控组内一个或一组IPv6地址。本端发送请求时，使用对端IP与此参数值（一个或一组IPv6地址）进行匹配，成功则进行固定速率流控。本端接收请求时，优先使用本端IP与此参数值进行匹配，成功则进行固定速率流控，失败则再使用对端IP与此参数值进行匹配。 |
| IPv6前缀 | 该参数用于指定流控IPv6地址的前缀。 |

#### [命令功能](#ZH-CN_CONCEPT_0000001549802506)

该命令用于查询访问控制列表全局信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001549802506)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549802506)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549802506)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RUNAME | RU名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定RU名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，区分大小写。<br>默认值：无<br>配置原则：使用DSP RU查看RU名称。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549802506)

查询软转发支撑访问控制列表全局信息：

```
DSP MSSACLINFO:RUNAME="VNODE_VNRS_VNFC_IPU_0064";
```

```

RETCODE = 0  操作成功。

结果如下
--------
          访问控制列表模块初始化标识  =  TRUE
                最大访问控制列表个数  =  526466
              已使用访问控制列表个数  =  4
访问控制列表模块内存使用量（K byte）  =  255344
(结果个数 = 1)
---    END
```

#### [命令功能](#ZH-CN_CONCEPT_0209897045)

**适用NF：PGW-C、SMF**

该命令用于查询实时位置上报的开关和本地Trigger的功能。

#### [注意事项](#ZH-CN_CONCEPT_0209897045)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0209897045)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0209897045)

无。

#### [使用实例](#ZH-CN_CONCEPT_0209897045)

查询实时位置上报的开关和本地Trigger：

```
LST CRACtrl:;
```

```

RETCODE = 0  操作成功

CRA控制参数
-----------
CRA控制  =  LOCAL
    TAI  =  使能
   ECGI  =  使能
(结果个数 = 1)

---    END
```

## [命令功能](#ZH-CN_MMLREF_0249644919)

**适用NF：PGW-C、GGSN、SMF**

该命令用于查询基于位置区地址分配的全局开关。

## [注意事项](#ZH-CN_MMLREF_0249644919)

无

#### [操作用户权限](#ZH-CN_MMLREF_0249644919)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0249644919)

无

## [使用实例](#ZH-CN_MMLREF_0249644919)

查询基于位置区地址分配的全局开关：

```
%%LST IPALLOCBYLOCGLBSW:;%%
RETCODE = 0 操作成功

结果如下
-----------
基于位置区地址分配的IPv4全局开关 = 使能
基于位置区地址分配的IPv6全局开关 = 使能
(结果个数 = 1)

---      END
```

## [输出结果说明](#ZH-CN_MMLREF_0249644919)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 基于位置区分配IPv4地址的全局开关 | 该参数用于指定基于位置区分配IPv4地址的开关。 |
| 基于位置区分配IPv6地址的全局开关 | 该参数用于指定基于位置区分配IPv6地址的开关。 |

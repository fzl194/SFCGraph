#### [命令功能](#ZH-CN_CONCEPT_0251174309)

**适用NF：NCG**

该命令用于立即生成话单。

#### [注意事项](#ZH-CN_CONCEPT_0251174309)

该命令执行后立即生效。

#### [本地用户权限](#ZH-CN_CONCEPT_0251174309)

manage-ug；system-ug

#### [网管用户权限](#ZH-CN_CONCEPT_0251174309)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [使用实例](#ZH-CN_CONCEPT_0251174309)

- 显示所有话单生成情况，示例如下：
  ```
  SYN GENERATECDR:;
  RETCODE = 0  操作成功。

  结果如下:
  --------
  模块名    任务状态 

  AP66_1    成功     
  AP67_1    成功     
  (结果个数 = 2)
  ---    END
  ```
- 显示同一个接入网元分组标识的话单生成情况，示例如下：
  ```
  SYN GENERATECDR: AGID="PS_GROUP_1";
  RETCODE = 0  操作成功。

  结果如下
  --------
  模块名     =   AP66_1 
  任务状态   =   成功
  (结果个数 = 1)
  ---    END
  ```

| 任务状态 | 表示立即生成话单的操作结果。 |

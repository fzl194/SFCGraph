## [命令功能](#ZH-CN_MMLREF_0232390458)

**适用NF：SMF**

该命令用于查询选择对端NF时使用的业务服务区。

## [注意事项](#ZH-CN_MMLREF_0232390458)

无

#### [操作用户权限](#ZH-CN_MMLREF_0232390458)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0232390458)

无

## [使用实例](#ZH-CN_MMLREF_0232390458)

查询选择对端NF时使用的业务服务区。

```
LST PNFSLCTSSCOPE:;
RETCODE = 0  操作成功

结果如下
--------
对端NF类型  =  默认的对端NF类型
服务区名称  =  ServingScope1
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0232390458)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 对端NF类型 | 该参数用于指定对端NF类型。 |
| 服务区名称 | 本参数用于指定服务区名称。 |

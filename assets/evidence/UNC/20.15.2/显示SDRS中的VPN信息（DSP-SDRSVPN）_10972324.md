## [命令功能](#ZH-CN_MMLREF_0000001110972324)

该命令用于查询SDRS中的VPN信息。

## [注意事项](#ZH-CN_MMLREF_0000001110972324)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001110972324)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001110972324)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| CELLID | Cell ID | 可选必选说明：必选参数<br>参数含义：该参数用于指定SDR调试消息发送的CELLID，可以通过使用命令<br>[**DSP MSPROCESS**](../../可靠性管理/微服务可靠性管理/显示微服务进程信息（DSP MSPROCESS）_09587887.md)<br>获取。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~127。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001110972324)

显示SDRS中的VPN信息

```
%%DSP SDRSVPN: CELLID="vup-pod-010-104-1-24__103__0";%%
RETCODE = 0  操作成功

结果如下
--------
Cell ID                 = vup-pod-010-104-1-24__103__0
VPN名称                 = _public_
SDRC分配VPN ID          = 1
下行是否经过LB          = 1
VPN地址信息总个数       = 1
VPN有效地址信息个数     = 1
实例ID                  = 281492156580946 0x1000400000452
VPN ID                  = 0
通信地址信息LOW TB      = 1106
通信地址信息TP          = 2416123937
有效通信地址            = TRUE
VPN版本信息             = 6728054339338254048
(结果个数 = 1)
---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001110972324)

| 输出项名称 | 输出项解释 |
| --- | --- |
| Cell ID | 该参数用于指定SDR调试消息发送的CELLID，可以通过使用命令<br>[**DSP MSPROCESS**](../../可靠性管理/微服务可靠性管理/显示微服务进程信息（DSP MSPROCESS）_09587887.md)<br>获取。 |
| VPN名称 | 该参数用于表示VPN的名称。 |
| SDRC分配VPN ID | 该参数用于表示SDRC给该VPN分配的ID。 |
| 下行是否经过LB | 该参数用于表示该VPN下行是否经过LB。 |
| VPN地址信息总个数 | 该参数表示VPN的地址信息总个数。 |
| VPN有效地址信息个数 | 该参数表示有效的地址信息个数。 |
| 实例ID | 该参数表示实例ID。 |
| VPN ID | 该参数表示VPN ID。 |
| 通信地址信息LOW TB | 该参数表示通信地址信息Low Tb(Target Blade)。 |
| 通信地址信息TP | 该参数表示通信地址信息TP(Target Port)。 |
| 有效通信地址 | 该参数表示是否是有效通信地址。<br>取值说明：<br>- TRUE(TRUE)<br>- FALSE(FALSE) |
| VPN版本信息 | 该参数表示VPN的版本信息。 |

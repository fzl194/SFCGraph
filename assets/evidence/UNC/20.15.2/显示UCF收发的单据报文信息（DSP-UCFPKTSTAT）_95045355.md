## [命令功能](#ZH-CN_MMLREF_0000001195045355)

该命令用于显示UCF收发的单据报文信息。

## [注意事项](#ZH-CN_MMLREF_0000001195045355)

执行命令前请确认UCF服务处于上线状态。

#### [操作用户权限](#ZH-CN_MMLREF_0000001195045355)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001195045355)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PODID | UCF POD ID | 可选必选说明：可选参数<br>参数含义：该参数用于请求系统显示UCF指定POD的报文信息；若不输入，则显示环境中所有UCF POD的报文信息。该参数可以通过DSP POD:TYPE=byType, PODTYPE="ucfexec-pod", MEID="网元ID"命令查询得到。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~255。<br>默认值：无<br>配置原则：无 |
| PROTOCOLTYPE | 协议类型 | 可选必选说明：可选参数<br>参数含义：该参数用于请求系统显示指定协议类型的报文信息；若不输入，则显示环境中所有协议类型的信息。该参数可以通过<br>[**LST UCFSVRIP**](../UCF服务器IP/查询UCF报表服务器的接入点IP地址（LST UCFSVRIP）_51253795.md)<br>命令查询得到。<br>数据来源：本端规划<br>取值范围：<br>- TCP（TCP）<br>- SFTP（SFTP）<br>- FTP（FTP）<br>- UDP（UDP）<br>默认值：无<br>配置原则：无 |
| ACCESSNAME | 接入点名称 | 可选必选说明：可选参数<br>参数含义：该参数用于请求系统显示指定接入点的报文信息；若不输入，则显示环境中所有接入点的信息。该参数可以通过<br>[**LST UCFSVRIP**](../UCF服务器IP/查询UCF报表服务器的接入点IP地址（LST UCFSVRIP）_51253795.md)<br>命令查询得到。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~32。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001195045355)

运营商A查询UCF向接入点发送单据报文的信息：

```
%%DSP UCFPKTSTAT:;%%
RETCODE = 0  操作成功

结果如下
--------
UCF POD ID     协议类型  接入点名称  接收报文数  被流控处理报文数  发送成功报文数  发送失败报文数  发送成功文件数  发送失败文件数  被老化处理文件数  非主向主实例发送文件数  
                                                                   
ucfexec-pod-1  TCP       ACCESS1     5120        0                 5120            0               0               0               0                 0                       
ucfexec-pod-0  TCP       ACCESS1     5120        0                 5120            0               0               0               0                 0                       
(结果个数 = 2)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001195045355)

| 输出项名称 | 输出项解释 |
| --- | --- |
| UCF POD ID | 该参数用于请求系统显示UCF指定POD的报文信息；若不输入，则显示环境中所有UCF POD的报文信息。该参数可以通过DSP POD:TYPE=byType, PODTYPE="ucfexec-pod", MEID="网元ID"命令查询得到。 |
| 协议类型 | 该参数用于请求系统显示指定协议类型的报文信息；若不输入，则显示环境中所有协议类型的信息。该参数可以通过<br>[**LST UCFSVRIP**](../UCF服务器IP/查询UCF报表服务器的接入点IP地址（LST UCFSVRIP）_51253795.md)<br>命令查询得到。<br>取值说明：<br>- TCP（TCP）<br>- SFTP（SFTP）<br>- FTP（FTP）<br>- UDP（UDP） |
| 接入点名称 | 该参数用于请求系统显示指定接入点的报文信息；若不输入，则显示环境中所有接入点的信息。该参数可以通过<br>[**LST UCFSVRIP**](../UCF服务器IP/查询UCF报表服务器的接入点IP地址（LST UCFSVRIP）_51253795.md)<br>命令查询得到。 |
| 接收报文数 | 该参数用于返回单个UCFEXEC服务实例接收到的报文总数，不区分接入点（单位：个）。 |
| 被流控处理报文数 | 该参数用于返回单个UCFEXEC服务实例被流控处理的报文总数，不区分接入点（单位：个）。 |
| 发送成功报文数 | 该参数用于返回本UCF向接入点发送成功的报文数（单位：个）。 |
| 发送失败报文数 | 该参数用于返回本UCF向接入点发送失败的报文数（单位：个）。 |
| 发送成功文件数 | 该参数用于返回UCF使用FTP/SFTP协议时向接入点发送成功的文件个数（单位：个）。 |
| 发送失败文件数 | 该参数用于返回UCF使用SFTP协议时向接入点发送失败的文件个数（单位：个）。 |
| 被老化处理文件数 | 该参数用于返回使用FTP/SFTP协议时单个UCFEXEC服务实例老化处理的文件个数，不区分接入点（单位：个）。 |
| 非主向主实例发送文件数 | 该参数用于返回使用FTP协议时，非主UCFEXEC服务实例向主UCFEXEC服务实例发送的文件个数（单位：个）。 |

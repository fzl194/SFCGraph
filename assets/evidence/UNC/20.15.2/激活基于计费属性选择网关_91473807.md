## [操作场景](#ZH-CN_OPI_0191473807)

本操作指导介绍在运行网络中激活基于计费属性选择 网关 特性的操作过程。

基于计费属性选择 网关 ，是指 UNC 在使用 APN （Access Point Name）进行 DNS (Domain Name Service)解析获取网关 IP 地址前，首先通过APN定制方法在原APN中加入用户计费信息（CC），然后再对定制后的APN进行DNS解析从而获取相应的网关地址。

> **说明**
> 适用于 SGSN、 MME。

## [必备事项](#ZH-CN_OPI_0191473807)

前提条件

- 请仔细阅读[WSFD-205002 基于计费属性选择网关特性概述](特性概述_92137769.md)。
- 完成加载License。

- DNS必须配置添加了用户计费属性（CC）的APN所对应的GGSN/P-GW的IP地址，以便解析扩展后的APN。
  > **说明**
  > 用户也可以选择自行配置添加了用户计费属性（CC）的APN所对应的GGSN/P-GW的IP地址（见 [操作步骤3](#ZH-CN_OPI_0191473807__zh-cn_opi_0130428893_step3) ），若两者都没配置将导致基于计费属性选择网关失败。

## [操作步骤](#ZH-CN_OPI_0191473807)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. 增加基于计费属性选择网关的策略。
  [**ADD GWSELPLCY**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/GnGp-GGSN_S5_S8接口管理/GGSN_P-GW选择/增加GGSN_P-GW选择策略（ADD GWSELPLCY）_26145944.md)
  > **说明**
  > - 参数“LABLE”（定制标识类型）选择“CC”。
  > - 参数“SLCTM”（选择方法）选择“CUSTOMIZE_APN-OI_REPLACEMENT”。
  > - 参数“CC”（ 计费属性 ）默认取值为“SAME_AS_GN（一致化处理）”时，采用与Gn接口Create PDP Context Request消息携带的Charging Characteristic信元相同的选择策略，请参考[**ADD CHGGNCCCFG**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/计费管理/Gn接口计费属性选择策略配置/增加Gn接口计费属性选择策略(ADD CHGGNCCCFG)_72344971.md)。
  > - 在不同的位置区内存在多个GGSN/P-GW，且其中既有支持预付费的也有不支持预付费的，需选择与UE在同一位置区且支持预付费的GGSN的场景下，启用基于位置区和计费属性选择网关，“LABLE”选择“CC_RAI”。
  > - 在基于CC选择网关的同时，为具备EPC能力的终端在GU网络接入时就必须选择到融合网关。具体网关选择方式为： 当用户支持EPC能力接入的时候，SGSN会根据EPC能力以及CC值选择到对应的GGSN/P-GW。当用户不支持EPC能力接入的时候，SGSN会根据CC值选择到对应的GGSN。
  >   此时需要启用 [WSFD-205002 基于计费属性选择网关](../WSFD-205002 基于计费属性选择网关_91473806.md) 和 [WSFD-205005 基于UE接入能力选择网关](../WSFD-205005 基于UE接入能力选择网关_63535135.md) ，并且 [**ADD GWSELPLCY**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/GnGp-GGSN_S5_S8接口管理/GGSN_P-GW选择/增加GGSN_P-GW选择策略（ADD GWSELPLCY）_26145944.md) 中 “定制标识类型 ” 选择 “CC_UEACCCAP（计费属性_UE接入能力）” 方式。在APNNI和APNOI中间添加 “CC.EPC标识” 。
4. **可选：**配置添加了用户计费属性（CC）的APN所对应的GGSN/P-GW的IP地址。
  [**ADD IPV4DNSH**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/DNS/DNS Hostfile管理/增加IPV4 DNS Hostfile记录(ADD IPV4DNSH)_26145884.md)

## [任务示例](#ZH-CN_OPI_0191473807)

任务描述

打开基于计费属性选择网关特性的开关，同时添加用户计费属性的APN对应的网关的IP地址。

脚本

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2CCGSN02", SWITCH=ENABLE;
```

//增加基于计费属性选择网关的策略。

```
ADD GWSELPLCY: SUBRANGE=ALL_USER, APNNI="HUAWEI1.com", SLCTM=CUSTOMIZE_APN-OI_REPLACEMENT, LABEL=CC;
```

//可选：配置添加了用户计费属性的APN对应的网关的IP地址。

```
ADD IPV4DNSH: HSINDEX=2, HOSTNAME="HUAWEI1.COM.0100.MNC000.MCC460.GPRS", ADDRSECTION=SECTION1,IPV4ADDR1="10.2.3.4";
```

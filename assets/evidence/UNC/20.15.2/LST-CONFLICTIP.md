## [命令功能](#ZH-CN_MMLREF_0264343878)

**适用NF：PGW-C、GGSN、SMF**

该命令用来查询指定本地地址池中的冲突地址。

## [注意事项](#ZH-CN_MMLREF_0264343878)

无

#### [操作用户权限](#ZH-CN_MMLREF_0264343878)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0264343878)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| POOLNAME | 地址池名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定已配置的地址池的名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~79。<br>默认值：无<br>配置原则：<br>该参数使用ADD ADDRPOOL命令配置生成。 |

## [使用实例](#ZH-CN_MMLREF_0264343878)

查询本地地址池lap中配置的冲突地址，POOLNAME为“lap”： LST CONFLICTIP:POOLNAME="lap";

```
%%LST CONFLICTIP:POOLNAME="lap";%%
RETCODE = 0  操作成功。

结果如下
--------------
地址池名称 =  lap
IP地址类型 =  IPv4
冲突IPv4地址  =  10.1.1.1
(结果个数 = 1)
---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0264343878)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 地址池名称 | 该参数用于指定已配置的地址池的名称。 |
| IP地址类型 | 该参数用于指定IP地址类型。 |
| 冲突IPv4地址 | 该参数用于指定冲突IPv4地址。 |
| 冲突IPv6地址前缀 | 该参数用于指定冲突IPv6地址前缀。 |
| IPv6前缀长度 | 该参数用于指定IPv6前缀长度。当IP地址类型取值为IPv4时，该参数无效。 |

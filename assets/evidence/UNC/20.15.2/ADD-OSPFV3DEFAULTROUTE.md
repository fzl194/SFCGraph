#### [命令功能](#ZH-CN_CONCEPT_0000001550120694)

该命令用于创建OSPFv3默认路由宣告。

#### [注意事项](#ZH-CN_CONCEPT_0000001550120694)

- 该命令执行后立即生效。
- 该命令最大记录数为8000。
- OSPFv3缺省路由发布的方式取决于引入缺省路由的区域类型，该命令仅用于发布缺省路由到普通OSPFv3区域。
- 对于Stub区域、Totally Stub区域，缺省路由自动发布。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550120694)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550120694)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | OSPFv3进程号 | 可选必选说明：必选参数<br>参数含义：OSPFv3进程号。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无<br>配置原则：OSPFv3进程必须已经存在。请使用LST OSPFV3命令查看可用的OSPFv3进程。 |
| TOPOID | 拓扑标识 | 可选必选说明：可选参数<br>参数含义：拓扑标识。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围为0。<br>默认值：0<br>配置原则：目前只支持默认拓扑0。 |
| CONFIGCOST | 使能度量 | 可选必选说明：可选参数<br>参数含义：使能缺省路由度量值。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- FALSE：不使能。<br>- TRUE：使能。<br>默认值：FALSE |
| COST | 外部LSA开销值 | 可选必选说明：条件必选参数<br>前提条件：该参数在“CONFIGCOST”配置为“TRUE”时为必选参数。<br>参数含义：外部LSA开销值。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～16777214。<br>默认值：无<br>配置原则：<br>- 到第一类外部路由的开销值＝本设备到相应的ASBR的开销值＋ASBR到该路由目的地址的开销值。<br>- 到第二类外部路由的开销值＝ASBR到该路由目的地址的开销值。 |
| CONFIGTYPE | 缺省路由使能类型 | 可选必选说明：可选参数<br>参数含义：缺省路由使能类型。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- FALSE：不使能。<br>- TRUE：使能。<br>默认值：FALSE |
| TYPE | 引入外部路由类型 | 可选必选说明：条件必选参数<br>前提条件：该参数在“CONFIGTYPE”配置为“TRUE”时为必选参数。<br>参数含义：引入外部路由类型。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- Type1：外部路由类型1。<br>- Type2：外部路由类型2。<br>默认值：Type2 |
| DELAYTIMER | 延迟发布缺省路由的时间（s） | 可选必选说明：可选参数<br>参数含义：延迟发布缺省路由的时间。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～65535，单位是秒。<br>默认值：无<br>配置原则：如果不输入该参数，则表示不延迟发布默认路由。 |
| ALWAYSFLAG | 默认路由发布标志 | 可选必选说明：可选参数<br>参数含义：配置了always参数，不会计算其他路由器的缺省路由。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- DefRtAdv：默认值。<br>- Always：无论本机是否存在激活的非本OSPFv3进程的缺省路由，都会产生并发布一个描述缺省路由的LSA。<br>- Summary：发布指定缺省路由的Type 3 summary LSA。<br>默认值：DefRtAdv<br>配置原则：配置了always参数的路由器不再计算来自其他路由器的缺省路由。 |
| PERMITCALFLAG | 计算默认路由 | 可选必选说明：可选参数<br>参数含义：计算默认路由。<br>数据来源：本端规划<br>取值范围：布尔类型，输入格式为“TRUE”或者“FALSE”。<br>默认值：FALSE<br>配置原则：本机必须存在激活的非本OSPFv3缺省路由，才会产生并发布一个缺省路由的ASE LSA。 |
| ROUTEPOLICYNAME | 路由策略 | 可选必选说明：可选参数<br>参数含义：路由策略。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～200。<br>默认值：无 |
| CONFIGTAG | 缺省路由使能标签 | 可选必选说明：可选参数<br>参数含义：缺省路由使能标签。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- FALSE：不使能。<br>- TRUE：使能。<br>默认值：FALSE |
| TAG | 路由标签 | 可选必选说明：条件必选参数<br>前提条件：该参数在“CONFIGTAG”配置为“TRUE”时为必选参数。<br>参数含义：路由标签。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～4294967295。<br>默认值：0 |

#### [使用实例](#ZH-CN_CONCEPT_0000001550120694)

发布默认路由到OSPFv3进程1下：

```
ADD OSPFV3DEFAULTROUTE:PROCID=1,TOPOID=0, CONFIGCOST=TRUE, COST=100, CONFIGTYPE=TRUE, TYPE=Type2, DELAYTIMER=1;
```

## [操作场景](#ZH-CN_OPI_0000001822891113)

本操作指导介绍在运行网络中激活SMF支持多服务区特性过程。

> **说明**
> 适用于SMF。

## [必备事项](#ZH-CN_OPI_0000001822891113)

前提条件

- 请仔细阅读[WSFD-205015 SMF支持多服务区特性概述](特性概述_21920373.md)。

*表1 smfInfo里的关键信元与配置的对应关系*

| 信元 | 配置 | 说明 |
| --- | --- | --- |
| sNssaiSmfInfoList | ADD PLMNNS<br>ADD NFNS<br>ADD NSDNN | ADD PLMNNS，增加网络切片。<br>ADD NFNS，增加NF支持的网络切片。<br>ADD NSDNN，增加网络切片支持的DNN。 |
| taiList | **ADD NFTAI** | **ADD NFTAI，**<br>添加NF实例支持的TAI信息。 |

## [任务示例](#ZH-CN_OPI_0000001822891113)

任务描述

用户首次接入时，AMF选择SMF会基于TAI优选SMF，在用户移动时已选定SMF的smfInfoList可匹配该新TAI，从而继续使用选定的SMF为UE提供服务，不需插入I-SMF。具体规划的SMF和服务范围关系如下所示。

*表5 不同SMF服务范围*

| SMF | SMF主服务范围区域(SMF注册时smfInfo携带此服务范围) | SMF次服务范围区域(SMF注册时smfInfoList携带此服务范围) |
| --- | --- | --- |
| SMF1 | TAC1（000001）、TAC3（000003）<br>TAC11~TAC19（000011~000019） | TAC2（000002）、TAC4（000004）<br>TAC21~TAC29（000021~000029） |
| SMF2 | TAC2（000002）、TAC4（000004）<br>TAC21~TAC29（000021~000029） | TAC1（000001）、TAC3（000003）<br>TAC11~TAC19（000011~000019） |

*表6 不同TAC内的SMF优选情况*

| TAC/TAC range | 可提供服务的优选SMF | 可提供服务的次选SMF |
| --- | --- | --- |
| 000001 | SMF1 | SMF2 |
| 000002 | SMF2 | SMF1 |
| TAC11~TAC19（000011~000019） | SMF1 | SMF2 |
| 000003 | SMF1 | SMF2 |
| 000004 | SMF2 | SMF1 |
| TAC21~TAC29（000021~000029） | SMF2 | SMF1 |

脚本

**SMF1上配置脚本如下** ：

//修改NF实例概述信息，确保服务注册消息中smfInfo的priority值小于smfInfoList中smfInfo的priority值。

```
MOD NFPROFILE: NFINSTANCENAME="smf1", PRIORITY=1;
```

//配置SMF实例信息，如SMF的名称，在SMF向NRF注册时会用到本命令的配置数据。

```
ADD SMFINFO: SMFINSTANCENAME="smf1", SMFNAME="smf1.cluster1.net1.smf.5gc", INTERPLMNFQDN="smf1.cluster1.net1.smf.5gc.mnc012.mcc345.3gppnetwork.org", ACCESSTYPE=AccessType3GPP_ACCESS;
```

//配置SMF实例（smf1）支持的TAI信息，在SMF向NRF注册时，smfInfo里面会用到本命令的配置数据。

```
ADD NFTAI: NFINSTANCENAME="smf1", MCC="345", MNC="012", TAC="000001";
ADD NFTAI: NFINSTANCENAME="smf1", MCC="345", MNC="012", TAC="000003";
```

//配置SMF实例（smf1）支持的TAI Range信息，在SMF向NRF注册时，smfInfo里面会用到本命令的配置数据。

```
ADD TACRANGE: INDEX=11, RANGESTART="000011", RANGEEND="000019";
ADD TAIRANGELIST: NFINSTANCENAME="smf1", NFTYPE=NfSMF, MCC="460", MNC="01", TACRANGEINDEX=11;
```

//配置SMF实例（smf1）实例支持的TAI信息，在SMF向NRF注册时，smfInfoList会用到本命令的配置数据。

```
ADD NFTAI: NFINSTANCENAME="smf1", MCC="345", MNC="012", TAC="000002", BINDSMFINFOID="Area2";
ADD NFTAI: NFINSTANCENAME="smf1", MCC="345", MNC="012", TAC="000004", BINDSMFINFOID="Area2";
```

//配置SMF实例（smf1）实例支持的TAI Range信息，在SMF向NRF注册时，smfInfoList会用到本命令的配置数据。

```
ADD TACRANGE: INDEX=21, RANGESTART="000021", RANGEEND="000029";
ADD TAIRANGELIST: NFINSTANCENAME="smf1", NFTYPE=NfSMF, MCC="460", MNC="01", TACRANGEINDEX=21，BINDSMFINFOID="Area2";
```

//增加 **smfInfo扩展** 的信息。一条 **smfInfo扩展** 对应SMF实例中一个smfInfo。

```
ADD SMFINFOEXT: SMFINSTANCENAME="smf1", ID="Area2", PRIORITY=10;
```

//配置SMF支持多smfInfo场景下的会话重建功能（可选）。

```
SET MULSMFPDUREEPOL: MULSMFPDUREESUP=Support;
```

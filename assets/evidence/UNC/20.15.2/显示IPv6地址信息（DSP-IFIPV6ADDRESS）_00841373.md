#### [命令功能](#ZH-CN_CONCEPT_0000001600841373)

该命令用于显示接口的IPv6地址状态信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600841373)

- 该命令执行后立即生效。
- IPv6的地址可以分为两个部分：网络前缀和接口标识。
- 该命令可以在VNRS_VNFC的Ethernet接口，Ethernet子接口，Eth-Trunk接口，Eth-Trunk子接口以及Loopback口，Tunnel口上配置。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600841373)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600841373)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IFNAME | 接口名 | 可选必选说明：可选参数<br>参数含义：该参数用于配置接口IPv6地址的接口名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。接口名称由接口类型+接口编号组成。<br>默认值：无<br>配置原则：请使用LST INTERFACE命令查看可用接口。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600841373)

显示所有接口的IPv6地址：

```
DSP IFIPV6ADDRESS:;
```

```

RETCODE = 0  操作成功

结果如下
-------------------------
接口名            IPv6地址                    IPv6地址前缀    IPv6地址类型    IPv6地址计算类型   IPv6地址状态
Ethernet64/0/3    FE80::207:12FF:FE18:3403    10              链路本地地址    none                 临时的
Ethernet64/0/3    2001:DB8::11                64              全球单播地址    none                 临时的
(结果个数 = 2)
---   END
```

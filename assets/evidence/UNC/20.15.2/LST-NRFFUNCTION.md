## [命令功能](#ZH-CN_MMLREF_0209652771)

**适用NF：NRF**

该命令用于查询特定NRF功能实例的配置信息。

## [注意事项](#ZH-CN_MMLREF_0209652771)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209652771)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652771)

无

## [使用实例](#ZH-CN_MMLREF_0209652771)

查询所有NRF功能实例的配置信息:

```
LST NRFFUNCTION:;
%%LST NRFFUNCTION:;%%
RETCODE = 0 执行成功

结果如下
-------------------------
NRF功能实体号  NRF功能实体描述  管理状态    运行状态      FQDN      最大服务请求次数

Instanceid01   nfdescription01  锁定        运行          fqdn01          1000                                
Instanceid02   nfdescription02  锁定        运行          fqdn02          1000                                

(结果个数 = 2)
        
```

## [输出结果说明](#ZH-CN_MMLREF_0209652771)

| 输出项名称 | 输出项解释 |
| --- | --- |
| NRF功能实体号 | 该参数用于表示特定NRF功能实例号。 |
| NRF功能实体描述 | 该参数用于表示特定NRF功能实体描述。 |
| 管理状态 | 该参数用于表示特定NRF功能实体管理状态。 |
| 运行状态 | 该参数用于表示特定NRF功能实体运行状态。 |
| FQDN | 该参数用于表示特定NRF功能实体FQDN。 |
| 最大服务请求次数 | 该参数用于指定NRF支持处理的最大服务请求次数，系统会根据"NRF每秒事务数"License项自动刷新该参数。 |

## [命令功能](#ZH-CN_MMLREF_0209653161)

**适用NF：AMF**

该命令用于查询5G寻呼优先级策略参数配置。

## [注意事项](#ZH-CN_MMLREF_0209653161)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209653161)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209653161)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PLCYIDX | 策略索引 | 可选必选说明：可选参数<br>参数含义：该参数用于在AMF内唯一标识一条寻呼优先级策略。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~127。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209653161)

- 查询所有5G寻呼优先级策略参数，执行如下命令：
  ```
  %%LST NGPAGINGPRIPLCY:;%%
  RETCODE = 0  操作成功

  结果如下
  --------
        策略索引  =  1
    数据网络名称  =  HUAWEI.COM
       ARP优先级  =  0
  寻呼优先级等级  =  8
  (结果个数 = 1)

  ---    END
  ```
- 查询“策略索引”为2的5G寻呼优先级策略参数，执行如下命令：
  ```
  %%LST NGPAGINGPRIPLCY: PLCYIDX=2;%%
  RETCODE = 0  操作成功

  结果如下
  --------
        策略索引  =  2
    数据网络名称  =  HUAWEI
       ARP优先级  =  0
  寻呼优先级等级  =  3
  (结果个数 = 1)

  ---    END
  ```

## [输出结果说明](#ZH-CN_MMLREF_0209653161)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 策略索引 | 该参数用于在AMF内唯一标识一条寻呼优先级策略。 |
| 数据网络名称 | 该参数用于指定应用寻呼优先级的数据网络名称（DNN）。 |
| ARP优先级 | 该参数用于指定应用寻呼优先级策略的ARP优先级等级。 |
| 寻呼优先级等级 | 该参数用于指定寻呼优先级等级。数值越小，表示NG-RAN寻呼用户的优先等级越高。 |

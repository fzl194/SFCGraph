#### [命令功能](#ZH-CN_CONCEPT_0000001549802650)

该命令用于查询OSPF的NBMA网络邻居路由器配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001549802650)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549802650)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549802650)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | 进程号 | 可选必选说明：可选参数<br>参数含义：OSPF进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |
| IPADDRESS | 邻居路由器IP地址 | 可选必选说明：可选参数<br>参数含义：邻居路由器IP地址。<br>数据来源：对端协商<br>取值范围：IPv4地址类型。点分十进制格式。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549802650)

查询OSPF的NBMA网络邻居路由器配置：

```
LST OSPFPEER:;
```

```

RETCODE = 0  操作成功。

结果如下
------------------------
          进程号  =  1
邻居路由器IP地址  =  10.2.2.2
邻居路由器优先级  =  100
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001549802650)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 进程号 | 该参数指定用户创建的OSPF进程号。 |
| 邻居路由器IP地址 | 该参数指定邻居的IP地址。 |
| 邻居路由器优先级 | 该参数指定邻居的指定路由器优先级。 |

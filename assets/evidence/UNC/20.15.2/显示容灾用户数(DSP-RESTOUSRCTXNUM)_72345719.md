#### [命令功能](#ZH-CN_MMLREF_0000001172345719)

**适用网元：MME**

本命令用于查询系统内备份用户的数量。查询输出结果为查询时刻的备份用户数量。

输出结果分为2个报告，1个报表显示对应RU名称、进程的备份用户数；1个报表显示系统内总的备份用户数。

#### [注意事项](#ZH-CN_MMLREF_0000001172345719)

无

#### [本地用户权限](#ZH-CN_MMLREF_0000001172345719)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172345719)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172345719)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RUNAME | RU名称 | 可选必选说明：可选参数<br>参数含义：本参数显示备份用户处在的RU名称。该参数可以通过<br>[DSP RU](../../../../../../平台服务管理/单体服务公共功能管理/系统管理/资源管理/RU管理/显示资源单元信息（DSP RU）_59103857.md)<br>命令查询。<br>数据来源：本端规划<br>取值范围：1~63字符串<br>默认值：无 |
| PRON | 进程号 | 可选必选说明：可选参数<br>参数含义：本参数表示备份用户处在的进程序号。<br>数据来源：本端规划<br>取值范围：0~20<br>默认值：无 |

#### [输出结果说明](#ZH-CN_MMLREF_0000001172345719)

| 输出项名称 | 输出项解释 |
| --- | --- |
| RU名称 | 参数含义：本参数显示RU名称。<br>取值范围：1~63字符串 |
| 进程号 | 参数含义：本参数备份用户所在的SPP进程号。<br>取值范围：0~20 |
| MM上下文数 | 参数含义：本参数表示备份的MM上下文数量。<br>取值范围：0~4294967295 |
| PDN连接数 | 参数含义：本参数表示备份的PDN上下文数量。<br>取值范围：0~4294967295 |
| 承载上下文数 | 参数含义：本参数表示备份的承载数量。<br>取值范围：0~4294967295 |
| MM签约上下文数 | 参数含义：本参数表示备份的MM签约上下文数量。<br>说明：当用户分离，或者符合容灾APN特征的PDN连接去激活后，S-USN的签约上下文不会立即删除。显示的结果包含了在线备份用户的MM签约上下文数，也包含曾经备份过MM签约上下文，并且当前处于分离态，或者符合容灾APN特征的PDN连接已去激活的用户的MM签约上下文。<br>取值范围：0~4294967295 |
| APN/PDP签约上下文数 | 参数含义：本参数表示备份的APN签约上下文数量。<br>说明：当用户分离，或者符合容灾APN特征的PDN连接去激活后，S-USN的签约上下文不会立即删除。显示的结果包含了在线备份用户的APN/PDP签约上下文数，也包含曾经备份过APN/PDP签约上下文，并且当前处于分离态，或者符合容灾APN特征的PDN连接已去激活的用户的APN/PDP签约上下文。<br>取值范围：0~4294967295 |
| 在线备份用户的MM签约上下文数 | 参数含义：本参数表示在线备份用户的APN/PDP签约上下文数量。“在线备份用户的APN/PDP签约上下文数”小于等于“APN/PDP签约上下文数”。<br>取值范围：0~4294967295 |
| 在线备份用户的APN/PDP签约上下文数 | 参数含义：本参数表示在线备份用户的APN/PDP签约上下文数量。“在线备份用户的APN/PDP签约上下文数”小于等于“APN/PDP签约上下文数”。<br>取值范围：0~4294967295 |

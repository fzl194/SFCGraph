# 激活基于PCF的VoNR业务快速恢复

- [必备事项](#ZH-CN_OPI_0000001190423111__1.3.2)

## [操作场景](#ZH-CN_OPI_0000001190423111)

在 UNC 上激活基于PCF的VoNR业务快速恢复功能，降低P-CSCF服务器故障时，对用户业务的影响。

> **说明**
> 适用于 SMF 。

## [必备事项](#ZH-CN_OPI_0000001190423111)

前提条件

- 请仔细阅读[WSFD-221003 基于PCF的VoNR业务快速恢复特性概述](WSFD-221003 基于PCF的VoNR业务快速恢复特性概述_44303412.md)。
- 完成[激活P-CSCF故障时IMS业务恢复](../WSFD-102202 P-CSCF故障时IMS业务恢复/激活P-CSCF故障时IMS业务恢复_26216211.md)。
- 已完成加载License。

数据

该操作无需准备数据。

## [操作步骤](#ZH-CN_OPI_0000001190423111)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)

## [任务示例](#ZH-CN_OPI_0000001190423111)

任务描述

开启基于PCF的VoNR业务快速恢复功能开关。

脚本

// 进入 “MML命令行-UNC” 窗口。

//打开本特性的License配置开关。

```
SET LICENSESWITCH:ITEM="LKV2FRVNRPCF01",SWITCH=ENABLE;
```

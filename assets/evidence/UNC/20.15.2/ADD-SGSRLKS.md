#### [命令功能](#ZH-CN_CONCEPT_0000001296986429)

**适用NF：SMSF**

此命令用于新增SGS服务端链路集配置。每个SGS MME只能关联一个链路集。

#### [注意事项](#ZH-CN_CONCEPT_0000001296986429)

- 该命令执行后立即生效。
- 整系统最多配置2000个。

#### [本地用户权限](#ZH-CN_CONCEPT_0000001296986429)

manage-ug；system-ug

#### [网管用户权限](#ZH-CN_CONCEPT_0000001296986429)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001296986429)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| LSX | 链路集索引 | 可选必选说明：必选参数<br>参数含义：该参数用于指定链路集的索引。<br>数据来源：本端规划<br>取值范围：0~2000<br>默认值：无<br>配置原则：增加链路集时，链路集索引建议从小到大配置。 |
| MMEN | MME索引 | 可选必选说明：必选参数<br>参数含义：该参数用于指定对端MME提供的索引号。<br>数据来源：整网规划<br>取值范围：0~1999<br>默认值：无<br>配置原则：须先在<br>[ADD SGSMME](../SGS MME实体/增加SGS MME实体(ADD SGSMME)_93164005.md)<br>中配置取值相同的<br>“MMEX”<br>参数。 |
| LSN | 链路集名 | 可选必选说明：可选参数<br>参数说明：该参数用于指定链路集的名称。<br>数据来源：整网规划<br>取值范围：<br>字符串类型，输入长度范围为0~255<br>默认值： 无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001296986429)

1. 添加SGS服务端链路集配置，可以用如下命令：
  ```
  ADD SGSRLKS: LSX=0, MMEN=0, LSN="huawei";
  ```

## [操作场景](#ZH-CN_OPI_0276495251)

当需要对SGSN/MME/SGW-C进行接入控制时，激活SGSN的黑白名单。

> **说明**
> 适用于GGSN、SGW-C、PGW-C。

## [必备事项](#ZH-CN_OPI_0276495251)

前提条件

- 请仔细阅读[WSFD-104509 SGSN的黑白名单特性概述](特性概述_76495250.md)。
- 完成加载License。

## [操作步骤](#ZH-CN_OPI_0276495251)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. 使能黑白名单控制开关并配置SGSN/MME/SGW-C接入控制名单类型。
  **[SET ACCESSLISTFUNC](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/黑白名单控制/设置接入控制名单功能（SET ACCESSLISTFUNC）_72373079.md)**
4. 配置接入控制名单的SGSN/MME/SGW-C的IP地址段。
  **[ADD ACCESSLISTIP](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/黑白名单地址列表/增加黑白名单（ADD ACCESSLISTIP）_72373075.md)**

## [任务示例](#ZH-CN_OPI_0276495251)

任务描述

根据运营商规划，需要禁止IP地址范围在10.20.0.2/27网段内的SGSN/MME/SGW-C接入。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH:LICITEM="LKV3W9SBWL11",SWITCH=ENABLE;
```

//使能黑白名单控制开关并配置接入控制名单类型为黑名单。

```
SET ACCESSLISTFUNC:LISTSWITCH=ENABLE,LISTTYPE=BLACK;
```

//配置接入控制名单的IP地址与掩码。

```
ADD ACCESSLISTIP:IPVERSION=IPV4,SRVNODEIPV4="10.20.0.2",SRVNODEIPV4MASK=27;
```

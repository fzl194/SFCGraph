## [命令功能](#ZH-CN_MMLREF_0000001183653660)

该命令用于查询服务化接口链路集的属性配置信息。

## [注意事项](#ZH-CN_MMLREF_0000001183653660)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001183653660)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001183653660)

无

## [使用实例](#ZH-CN_MMLREF_0000001183653660)

查询服务化接口链路集的属性配置，可以用如下命令：

```
%%LST SBILINKSETCFG:;%%
RETCODE = 0  操作成功

结果如下
--------
链路集老化时间(小时) = 10
FQDN周期性检查开关 = ON
进程内链路集故障阈值 = 50
系统内链路集故障阈值 = 20
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001183653660)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 链路集老化时间(小时) | 该参数用于指定链路集老化时间。 |
| FQDN周期性检查开关 | 该参数用于控制是否开启周期性检查链路集FQDN对应的IP地址的功能。 |
| 进程内链路集故障阈值 | 该参数用于判定链路集在单个HTTP进程中是否为故障，当故障链路数占比大于等于此参数，则认为进程内到对端的链路集故障。 |
| 系统内链路集故障阈值 | 该参数用于判定链路集在系统内是否为故障，当存在链路集故障的进程数占比大于等于此参数，则认为系统内链路集故障，该链路集对应的NF服务不可达。 |

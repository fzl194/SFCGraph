## [命令功能](#ZH-CN_MMLREF_0209652478)

**适用NF：PGW-C、SMF、GGSN**

该命令用于查询配置的策略接口的选择方式。

## [注意事项](#ZH-CN_MMLREF_0209652478)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209652478)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652478)

无

## [使用实例](#ZH-CN_MMLREF_0209652478)

查询策略接口的选择方式。

```
%%LST POLICYMODE:;%%
RETCODE = 0  操作成功

结果如下
```

## [输出结果说明](#ZH-CN_MMLREF_0209652478)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 指定终端和接入类型 | 该参数用于指定终端和接入类型。 |
| 按5GS互操作指示选择策略接口开关 | 该参数用于配置是否按5GS互操作指示的取值来选择策略接口。"5GS互操作指示"指示S11、S5/S8、S2b接口的“5GS Interworking Indication”参数。 |
| 非5G终端4G接入按5GS互操作指示选择策略接口开关 | 该参数用于配置非5G终端4G接入时是否按5GS互操作指示的取值来选择策略接口。"5GS互操作指示"指示S5/S8接口的“5GS Interworking Indication”参数。 |
| 指定策略接口 | 该参数用于指定策略接口。在参数BY5GSIWKI或者BYIWKUEN5GR4G取值为DISABLE时，按本参数的配置确定策略接口。在参数TMACCTYPE配置为RAT2G、RAT3G、RATNBIOT、RATLTEM或者NON_3GPP时，按本参数的配置确定策略接口。 |
| 是否基于PCF实例标识决策策略接口类型 | 该参数用于控制当基于用户的终端和接入类型决策出策略接口类型为“Gx（Gx接口）”时，是否再基于PCF实例标识重新决策最终策略接口类型。 |
| 按5GC无限制接入标识选择策略接口开关 | 该参数用于配置是否按5GC无限制接入标识的取值来选择计费接口。“5GC无限制接入标识”是指S11、S5/S8、S2b接口的“5GC Not Restricted Indication”参数。 |

# 调测基于APN的接入速率控制

- [操作场景](#ZH-CN_OPI_0191285447__1.3.1)
- [必备事项](#ZH-CN_OPI_0191285447__1.3.2)
- [操作步骤](#ZH-CN_OPI_0191285447__1.3.3)

## [操作场景](#ZH-CN_OPI_0191285447)

本操作指导介绍在运行网络中调测基于APN的接入速率控制特性的操作过程。

> **说明**
> 适用于 SGSN 。

## [必备事项](#ZH-CN_OPI_0191285447)

前提条件

- 请仔细阅读[WSFD-106302 基于APN的接入速率控制特性概述](特性概述_91385595.md)。
- 完成[激活基于APN的接入速率控制](激活基于APN的接入速率控制_91285446.md)。

数据

该操作无需准备数据。

工具

该操作无需准备其他工具。

## [操作步骤](#ZH-CN_OPI_0191285447)

1. 进入 “MML命令行-UNC” 窗口。
2. 查询APN的接入速率控制参数。
  [**LST APNCTRLPARA**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/M2M管理/基于APN的信令控制参数/查询基于APN的信令控制参数(LST APNCTRLPARA)_26305594.md) :;
  预期结果： “APNNI组号” 、 “附着请求控制门限” 和 “PDP上下文请求控制门限” 所显示的对应参数与配置相一致。
3. 查询性能对象信息。
  [**LST PERFOBJ**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/操作维护/性能管理/性能对象管理/查询性能对象信息(LST PERFOBJ)_26306000.md) : MOC=APN;
  预期结果：所显示的 “对象名称” 中包含了要测量的APN名称。
4. 添加要进行速率控制的APN到性能统计的测量对象中，在网管上单击 “ 性能管理系统 > 对象类 > APN > 指定APN信令接入速率控制 ” 在测量对象中增加对应APN名称，使性能统计的APN测量对象生效。
5. 显示实时性能统计。
  通过U2020/MAE查看性能指标。
  预期结果： “指定APN信令接入速率控制” 测量单元中的如下指标统计到了相应的值。
    - [117504813 指定APN的信令控制时收到的Attach Request消息次数](../../../../../OM参考/性能指标/UNC业务性能指标/业务性能指标/SGSN_MME业务性能指标/USN性能指标/Gb模式移动管理/指定APN信令接入速率控制/117504813 指定APN的信令控制时收到的Attach Request消息次数_06828056.md)
    - [117504814 指定APN的信令控制时丢弃的Attach Request消息次数](../../../../../OM参考/性能指标/UNC业务性能指标/业务性能指标/SGSN_MME业务性能指标/USN性能指标/Gb模式移动管理/指定APN信令接入速率控制/117504814 指定APN的信令控制时丢弃的Attach Request消息次数_06828057.md)
    - [117504815 指定APN的信令控制时收到的PDP context Request消息次数](../../../../../OM参考/性能指标/UNC业务性能指标/业务性能指标/SGSN_MME业务性能指标/USN性能指标/Gb模式移动管理/指定APN信令接入速率控制/117504815 指定APN的信令控制时收到的PDP context Request消息次数_06828058.md)
    - [117504816 指定APN的信令控制时丢弃的PDP context Request消息次数](../../../../../OM参考/性能指标/UNC业务性能指标/业务性能指标/SGSN_MME业务性能指标/USN性能指标/Gb模式移动管理/指定APN信令接入速率控制/117504816 指定APN的信令控制时丢弃的PDP context Request消息次数_06828059.md)

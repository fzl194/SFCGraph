    a. 配置TAC组。

    a. 配置LAC组。
      **[ADD LACGROUP](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/虚拟APN映射管理/基于LAC位置的虚拟APN映射管理/虚拟APN映射的LAC组/增加LAC组（ADD LACGROUP）_09654412.md)**

## [任务示例](#ZH-CN_OPI_0262638684)

任务描述

按照网络规划，需要将使用APN为huawei.com接入，映射的真实APN为huawei.beijing.com，TAC号段为0x000001~0x000010的5G用户选择到贴近用户位置的部署的P-CSCF，服务器IP地址为10.130.228.70。

脚本

//增加虚拟APN。

```
ADD APN:APN="huawei.com", VIRTUALAPN=ENABLE;
```

//配置TAC组。

```
ADD TACGROUP:TACGROUPNAME="beijing", TACTYPE=N2Tac;
```

//在TAC组内绑定TAC号段。

```
ADD N2TACID:TACSECNUM=2, TACSTARTID="0x000001", TACENDID="0x000010", TACGROUPNAME="beijing";
```

//配置虚拟APN映射策略。

```
ADD VIRTUALAPNRULE: VIRTUALAPN="huawei.com", MAPPINGTYPE=TAC_GROUP, MAPPINGVALUE="beijing", REALAPN="huawei.beijing.com";
```

//配置P-CSCF组。

```
ADD PCSCFGROUP: GROUPNAME="group1", IPVERSION=IPV4, ALLOCTYPE=LOCAL;
```

//配置P-CSCF服务器地址。

```
ADD PCSCFIP: GROUPNAME="group1", IPVERSION=IPV4, PCSCFIPV4="10.130.228.70";
```

//将P-CSCF组绑定APN。

```
ADD PCSCFGRPBNDAPN: APN="huawei.beijing.com", DEFAULTFLAG=DEFAULT, MPCSCFGRPIPV4="group1";
```

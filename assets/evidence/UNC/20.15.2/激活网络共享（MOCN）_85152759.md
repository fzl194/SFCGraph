## [操作场景](#ZH-CN_OPI_0185152759)

本操作指导介绍在运行网络中激活网络共享特性的操作过程。

网络共享（MOCN）指MOCN（Multi-Operator Core Network）方式的网络共享，是指各运营商共享 无线接入网 ，而不共享核心网设备的共享方式，有利于节省运营商的投资成本。

> **说明**
> 适用于 SGSN 。

## [必备事项](#ZH-CN_OPI_0185152759)

前提条件

- 请仔细阅读[WSFD-207001 网络共享（MOCN）特性概述](特性概述_87065742.md)。
- 完成[配置到RNC的数据](../../../../初始配置/UNC初始配置与调测/组网对接配置/配置AMF&MME&SGSN/配置SGSN/配置到RNC的数据_82626756.md)的基本数据配置。
- 完成[配置到CG的数据](../../../../初始配置/UNC初始配置与调测/组网对接配置/配置AMF&MME&SGSN/配置SGSN/配置到CG的数据_82626776.md)的基本数据配置。
- 完成2G Pool和3G Pool的部署。
- 完成加载License。
- 无线侧设置的COMMON PLMN和SGSN上的HPLMN不同时：
    - SGSN需要将COMMON PLMN配置为HPLMN。因此需要同时部署[WSFD-104401 支持多HPLMN功能](../../组网功能/WSFD-104401 支持多HPLMN功能_63530664.md)特性。
    - 在对端SGSN上，需要增加COMMON PLMN对应的DNS解析数据，便于对端SGSN根据COMMON PLMN寻址本端SGSN。

## [操作步骤](#ZH-CN_OPI_0185152759)

1. 进入 “MML命令行-UNC” 窗口。
2. 设置License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. 修改NSE信息。
    - 当NSE是静态配置或动态手工配置的时候，
      [**MOD NSE**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/Gb接口管理/信令实体管理/修改信令实体(MOD NSE)_26305838.md)
    - 当NSE是动态自动配置的时候，
      [**MOD GBNSECFGPARA**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/Gb接口管理/Gb自动配置管理/NSE属性模板管理/修改NSE属性模板(MOD GBNSECFGPARA)_26305814.md)
4. 确认无线侧设置的COMMON PLMN和SGSN上的HPLMN是否相同。
    - 相同，处理结束。
    - 不同，执行[5](#ZH-CN_OPI_0185152759__zh-cn_opi_0130429072_step4)。
5. 配置RNC信息。
    a. 增加COMMON PLMN对应的RNC信息。
      [**ADD RNC**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/Iu接口管理/Iu接口RNC信息/增加Iu接口RNC信息(ADD RNC)_26146040.md)
    b. 增加COMMON PLMN对应的寻呼信息。
      [**ADD IUPAGING**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/Iu接口管理/Iu接口寻呼数据/增加Iu接口寻呼数据(ADD IUPAGING)_26305844.md)
6. 参见 [一个为一个拥有多PLMN ID的MNO服务。](../../组网功能/WSFD-104401 支持多HPLMN功能/激活支持多HPLMN功能（适用于SGSN_MME）_70150791.md#ZH-CN_OPI_0170150791__cmd1000064495180640) 的步骤1和步骤2增加COMMON PLMN为HPLMN。
7. 配置Non-supporting UE的计费信息。
    a. 设置Non-supporting UE的PLMN获取策略。
          - 设置Gn接口上传输的或话单中使用的PLMN为Serving PLMN，然后执行 [b](#ZH-CN_OPI_0185152759__zh-cn_opi_0130429072_cmd297651773180641) 。
            [**SET CHGCDR**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/计费管理/计费控制/设置计费CDR参数（SET CHGCDR）_26145372.md)
          - 设置Gn接口上传输的或话单中使用的PLMN为COMMON PLMN。处理结束。
            [**SET CHGCDR**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/计费管理/计费控制/设置计费CDR参数（SET CHGCDR）_26145372.md)
    b. 为Non-supporting UE指定Serving PLMN。
      [**ADD HNOSRVPLMN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/归属网络运营商管理/归属网络Serving PLMN管理/增加归属网络Serving PLMN信息(ADD HNOSRVPLMN)_72345655.md)

## [任务示例](#ZH-CN_OPI_0185152759)

任务描述

打开MOCN功能开关。RAN侧设置的COMMON PLMN为46123，SGSN上的HPLMN为12323。RNC的相关信息如 [表1](#ZH-CN_OPI_0185152759__zh-cn_opi_0130429072_table_0357ED36) 所示。

*表1 RNC相关信息*

| 参数 | 取值 |
| --- | --- |
| 网络指示语 | 国内备用网 |
| 信令点编码 | 0xb01 |
| 协议版本 | R6 |
| 位置区标识 | 461232301 |
| 路由区编码 | 0x21 |

脚本

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2MOCN02", SWITCH=ENABLE;
```

//修改NSE信息。

```
MOD NSE: NSEI=0, MOCN=YES;
MOD NSE: NSEI=1, MOCN=YES;
MOD NSE: NSEI=2, MOCN=YES;
```

//增加COMMON PLMN对应的RNC信息。

```
ADD RNC: RNCX=1, RNCMCC="461", RNCMNC="23", RNCID=0, NI=NATB, SPC="0xb01";
```

//增加COMMON PLMN对应的寻呼信息。

```
ADD IUPAGING: LAI="461232301", RAC="0x21", RNCINDEX=1;
```

//增加COMMON PLMN为HPLMN。

```
SET LICENSESWITCH: LICITEM="LKV2HPLMN02", SWITCH=ENABLE;
ADD HPLMN: MCC="461", MNC="23", CC="86";
```

//设置Non-supporting UE的PLMN获取策略。

```
SET CHGCDR: NONSUPPUEPLMN=SRV_PLMN;
```

//为Non-supporting UE指定Serving PLMN。

```
ADD HNOSRVPLMN: NOID=0, MCC="123", MNC="23";
```

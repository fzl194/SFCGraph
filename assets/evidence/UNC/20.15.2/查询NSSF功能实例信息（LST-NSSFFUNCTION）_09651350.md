## [命令功能](#ZH-CN_MMLREF_0209651350)

**适用NF：NSSF**

本命令用于查询NSSF功能实例的信息。

## [注意事项](#ZH-CN_MMLREF_0209651350)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209651350)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209651350)

无

## [使用实例](#ZH-CN_MMLREF_0209651350)

查询所有NSSF功能实例的配置信息。

```
LST NSSFFUNCTION;
```

## [输出结果说明](#ZH-CN_MMLREF_0209651350)

| 输出项名称 | 输出项解释 |
| --- | --- |
| NSSF功能实例号 | NSSF功能实例号。 |
| NSSF功能实例描述 | NSSF功能实例描述。 |
| 管理状态 | NSSF功能实例的管理状态。 |
| 运行状态 | NSSF功能实例的运行状态。 |
| FQDN | NSSF功能实例的FQDN。 |
| 最大切片选择次数 | 该参数用于指定NSSF支持的每秒最大切片选择次数，系统会根据“NSSF每秒事务数”License项值自动刷新该参数。 |

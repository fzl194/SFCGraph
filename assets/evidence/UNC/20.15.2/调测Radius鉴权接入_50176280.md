# 调测Radius鉴权接入

- [操作场景](#ZH-CN_OPI_0250176280__1.3.1)
- [必备事项](#ZH-CN_OPI_0250176280__1.3.2)
- [操作步骤](#ZH-CN_OPI_0250176280__1.3.3)

## [操作场景](#ZH-CN_OPI_0250176280)

当运营商部署Radius鉴权接入功能时，需要对本功能进行调测，确保本功能可以正常使用。

> **说明**
> 适用于 GGSN、 PGW-C、SMF。

## [必备事项](#ZH-CN_OPI_0250176280)

前提条件

- 请仔细阅读[WSFD-011305 Radius鉴权接入特性概述](特性概述_50176278.md)。
- 完成[激活Radius鉴权接入](激活Radius鉴权接入_50176279.md)。

数据

| 类别 | 参数名称 | 取值样例 | 获取方法 | 说明 |
| --- | --- | --- | --- | --- |
| 用户信息 | 国际移动用户识别码（IMSI） | 123000123456789 | 测试终端自带 | - |
| 用户APN信息 | APN名称（APN） | huawei.com | 测试终端自带 | - |

工具

测试终端

## [操作步骤](#ZH-CN_OPI_0250176280)

1. 终端使用APN信息“huawei.com”接入网络，创建PDU会话标识为1的PDU会话。
2. 进入 “MML命令行-UNC” 窗口。
3. 查询用户的PDU会话信息。
  **[DSP PDUSESSION](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/接入管理运维/查询PDP上下文信息/显示PDU会话（DSP PDUSESSION）_09653215.md)**
  预期结果：用户接入成功。

  ```
  DSP PDUSESSION:QUERYTYPE=IMSI, IMSI="123000123456789", DSPINFOTYPE=DETAILED, WLNETWKTYPE=NW5G, PDUSESSIONID=1, QFI=1;
  RETCODE = 0  操作成功  

  结果如下 
  --------------  
              国际移动用户标识 = 123000123456789
                              ...
                           APN = huawei.com
                              ...
  (结果个数 = 1) 
  ---    END
  ```
4. 查询APN鉴权属性。
  **[LST APNAUTHATTR](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/APN管理/APN鉴权属性/查询APN鉴权属性配置（LST APNAUTHATTR）_28567627.md)**
    - 如果APN的接入模式与规划值一致，调测结束。
    - 如果APN的接入模式与规划值不一致，请参考[激活Radius鉴权接入](激活Radius鉴权接入_50176279.md)重新配置。

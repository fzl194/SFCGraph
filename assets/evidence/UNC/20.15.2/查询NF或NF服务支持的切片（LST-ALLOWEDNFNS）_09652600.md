## [命令功能](#ZH-CN_MMLREF_0209652600)

**适用NF：AMF、SMF、NRF、NSSF、NCG**

该命令用于查询NF或NF服务实例支持的服务切片信息。

## [注意事项](#ZH-CN_MMLREF_0209652600)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209652600)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652600)

无

## [使用实例](#ZH-CN_MMLREF_0209652600)

运营商A需要查询所有的服务切片支持信息。

```
%%LST ALLOWEDNFNS:;%%
RETCODE = 0  操作成功

结果如下
--------
     索引标识  =  0
     注册类型  =  RegService
   NF实例名称  =  SMF_Instance_0
 服务实例标识  =  Service_Instance_0
切片/服务类型  =  1
   切片区分码  =  000001
(结果个数 = 1)

---    END
```

| 注册类型 | 本参数用于指定注册类型。 |

| 服务实例标识 | 本参数用于指定服务实例标识。 |

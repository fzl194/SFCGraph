## [操作场景](#ZH-CN_OPI_0000002014101265)

本操作指导介绍在运行网络中激活基于用户群的QoS控制特性的操作过程，对 **本网非本省用户群** 的QoS进行单独配置，从而实现灵活的QoS控制策略。

> **说明**
> 适用于SMF。

## [对系统的影响](#ZH-CN_OPI_0000002014101265)

该操作对系统正常运行无影响。

- 完成加载License。

## [操作步骤](#ZH-CN_OPI_0000002014101265)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  **[**SET LICENSESWITCH**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)**
3. （可选）针对使用语音业务的用户，如果这些用户不受基于漫入用户的QoS控制，可将其加入白名单。
  **[**ADD QOSCAPWHITEAPNS**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/QoS/5GC QoS配置/基于IMSI号段的QoS管理/不受SMFQOSCAP配置控制的APN白名单列表/增加不受SMFQOSCAP配置控制的APN白名单列表。（ADD QOSCAPWHITEAPNS）_26639421.md)**
4. 添加UDM ID配置。
  **[ADD LOCALUDMINSTID](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/QoS/5GC QoS配置/基于IMSI号段的QoS管理/本省UDM实例标识/增加本省UDM实例标识（ADD LOCALUDMINSTID）_77419796.md)**
5. 添加QosFlow QoS限制的配置。
  [**ADD SMFQOSCAP**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/QoS/5GC QoS配置/基于IMSI号段的QoS管理/Qos Flow QoS限制配置/增加QosFlow QoS限制的配置（ADD SMFQOSCAP）_26279737.md)
6. **可选：**（可选）当需要基于5QI粒度进一步限制QoS时，添加基于5QI的QosFlow QoS限制配置。
  [**ADD QOSCAPBY5QI**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/QoS/5GC QoS配置/基于IMSI号段的QoS管理/基于5QI的Qos Flow QoS限制配置/增加基于5QI的QosFlow QoS限制配置（ADD QOSCAPBY5QI）_76079620.md)

## [任务示例](#ZH-CN_OPI_0000002014101265)

任务描述

增加一条QoS限制配置，对于本网漫入用户，将上行APN AMBR限制为20000000，下行APN AMBR限制为16600000，QOS5QI为9，5QI的优先级别为10，ARP的优先级别为5，ARP的抢占能力为MAY_PREEMPT，ARP的被抢占能力为NOT_PREEMPTABLE，QoS协商动作为RESTRICT。

脚本

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2QCBISM01", SWITCH=ENABLE;
```

//（可选）针对使用语音业务的用户，如果这些用户不受漫入用户的QoS控制，可将其加入白名单。

```
ADD QOSCAPWHITEAPNS:APN="ims",SUBRANGE=ALL_USER;
```

//添加UDM ID的配置。

```
ADD LOCALUDMINSTID:UDMINSTID= "UDM_Instance_0";
```

//添加QosFlow QoS限制的配置。

```
ADD SMFQOSCAP:SUBRANGE=HOME_NOLOCAL_USER,QOSIDTYPE=NONGBR, AMBRUL=20000000, AMBRDL=16600000,QOS5QI=9,QOS5QIPL=10,ARPPRL=5,ARPPCI=MAY_PREEMPT,ARPPVI=NOT_PREEMPTABLE,QOSACTION=RESTRICT;
```

//（可选）当需要基于5QI粒度进一步限制QoS时，添加基于5QI的QosFlow QoS限制配置。

```
ADD QOSCAPBY5QI:SUBRANGE=HOME_NOLOCAL_USER, QOS5QI=9,QOS5QIPL=10,ARPPRL=5,ARPPCI=MAY_PREEMPT,ARPPVI=NOT_PREEMPTABLE;
```

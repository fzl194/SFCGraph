## [命令功能](#ZH-CN_MMLREF_0000001088248950)

**适用NF：SMF**

该命令用于查询切片地址池性能统计对象。

## [注意事项](#ZH-CN_MMLREF_0000001088248950)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001088248950)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001088248950)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IDX | 索引 | 可选必选说明：可选参数<br>参数含义：该参数用于指定切片地址池性能统计对象的索引。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~499。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001088248950)

查询所有的切片地址池性能统计对象： LST PERFSLCADDRPOOL:;

```
%%LST PERFSLCADDRPOOL:;%%
RETCODE = 0  操作成功。

结果如下
------------------------
        索引  =  1
切片业务类型  =  1
切片细分标识  =  010101
  地址池名称  =  pool1
(结果个数 = 1)
```

## [输出结果说明](#ZH-CN_MMLREF_0000001088248950)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 索引 | 该参数用于指定切片地址池性能统计对象的索引。 |
| 切片业务类型 | 该参数用于指定切片的业务类型，如eMBB（1）、URLLC（2）、MIoT（3）等协议定义的标准SST，或者运营商自定义的SST。 |
| 切片细分标识 | 该参数用于指定，根据服务群体对某类网络业务切片的进一步细分。 |
| 地址池名称 | 该参数用于指定已配置的地址池的名称。 |

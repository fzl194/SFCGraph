## [操作场景](#ZH-CN_OPI_0193260161)

融合计费会话过程中，对于各种异常场景，可以在CHF本地设置一些处理动作对融合计费流程进行干预，以确保业务的正常进行。这里的异常场景，主要是指CHF响应SMF请求消息的状况，及响应消息中携带的返回码值。处理动作有三种大小不同的粒度：整个用户的所有计费过程、PDU会话、RG。

> **说明**
> 适用于SMF。

## [操作步骤](#ZH-CN_OPI_0193260161)

1. 进入“MML命令行-UNC”窗口。
2. **可选：** 配置当SMF与CHF交互时，SMF超时未收到CHF返回的ChargingDataResponse消息时的动作。
  ****SET FAILHANDLING****
3. **可选：** 配置当SMF收到CHF的PDU级异常返回码时的动作。
  ****ADD PDUSCACT****
4. **可选：** 配置当SMF收到CHF的响应的RG级异常返回码时的动作。
  ****ADD RGRCACT****
5. **可选：** 配置忽略CHF响应消息的信元列表。
  ****SET CNVRGDCHGPARA****
  > **说明**
  > ****SET CNVRGDCHGPARA**** 命令的 “IGNORERSPAVP” 参数的取值范围是位域类型，支持如下参数单独配置或同时配置：
  >
  > - 当“IGNORERSPAVP”参数配置为“INVOCATIONNUM”时，表示忽略CHF响应消息中的Invocation Sequence Number，不检查是否跟请求消息一致。
  > - 当“IGNORERSPAVP”参数配置为“FUI”时，表示当CHF返回RG级异常返回码和FUI时，忽略CHF响应消息中的FinalUnitIndication，按照对应****ADD RGRCACT****配置的“RCACT”动作执行。当**ADD RGRCACT**未配置时，执行默认动作。
  > - 当“IGNORERSPAVP”参数配置为“SESSIONFAILOVER”时，表示忽略CHF响应消息中的SessionFailover，使用对应CCT模板通过****SET FAILHANDLING****配置的“FAILOVERSUP”动作，“FAILOVERSUP”表示主CHF故障时，向备CHF发送计费消息进行处理。
  > - 当“IGNORERSPAVP”参数配置为“FAILUREHANDLING”时，表示忽略CHF响应消息中的FailureHandling，使用对应CCT模板通过**SET FAILHANDLING**配置的“FHACTION”动作。

## [任务示例](#ZH-CN_OPI_0193260161)

任务描述

配置SMF超时未收到ChargingDataResponse消息、收到PDU级/RG级异常返回码时的动作。具体要求如下：

1、设置名为“cct-test”的CCT融合计费模板，FH处理动作为CONTINUE，定时器时长20秒，用户保持时长30分钟，并使能Failback开关、在线计费自动恢复开关。

2、增加PDU异常返回码动作，配置的CCT名称为“cct-test”，指定异常返回码为502，动作为FAILOVER。

3、增加RG异常返回码动作，配置的CCT名称为“cct-test”，指定异常返回码为“QUOTAMNOTAPPL”，动作为阻塞业务。

4、配置CHF下发RG级异常返回码和FinalUnitIndication时，SMF忽略CHF下发的FinalUnitIndication。

脚本

//配置SMF超时未收到ChargingDataResponse消息时的动作。

```
SET FAILHANDLING: CCTMPLTNAME="cct-test", FHACTION=CONTINUE, FAILOVERSUP=ENABLE, AUTOFAILBACK=ENABLE, TXTIMER=5, HOLDINGTIME=10;
```

//配置SMF收到PDU级异常返回码时的动作。

```
ADD PDUSCACT: CCTMPLTNAME="cct-test", CODETYPE=VALUE, STATUSCODE=502, SCACT=FAILOVER;
```

//配置SMF收到RG级异常返回码时的动作。

```
ADD RGRCACT: CCTMPLTNAME="cct-test", RCCODE=QUOTAMNOTAPPL, RCACT=BLCK_IMMED_RPT, BLKTIMER=30;
```

//配置CHF下发RG级异常返回码和FinalUnitIndication时，SMF忽略CHF下发的FinalUnitIndication。

```
SET CNVRGDCHGPARA: IGNORERSPAVP=FUI, BADRSPACT=TERMINATE;
```

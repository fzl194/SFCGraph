## [操作场景](#ZH-CN_OPI_0170014693)

本操作指导介绍在运行网络中激活VoLTE紧急呼叫特性的操作过程。

> **说明**
> 适用于MME。

## [必备事项](#ZH-CN_OPI_0170014693)

前提条件

- 请仔细阅读[WSFD-102101 VoLTE紧急呼叫](特性概述_70014691.md)。
- 已完成加载License。
- 部署VoLTE紧急呼叫特性需要在MME、SGW-C、PGW-C、MSC、eNodeB等网元上首先完成基本业务的数据配置。
- 运营商已经通过 [**ADD HPLMN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/归属网络运营商管理/MNO管理/MNO网络配置表/增加本地PLMN(ADD HPLMN)_26146074.md) 、 [**ADD MVNONET**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/归属网络运营商管理/MVNO管理/MVNO网络标识配置表/增加MVNO网络配置信息(ADD MVNONET)_72345663.md) 或 [**ADD CONNECTPLMN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/互联PLMN管理/增加互联PLMN(ADD CONNECTPLMN)_26305852.md) 配置了运营商标识（MNOID/MVNOID/NOID）。
- VOLTE紧急呼叫号码下发会影响终端存储的紧急呼叫号码列表，当终端回落到2/3G发起紧急呼叫时，需要MSC配置紧急呼叫类型和对应号码的映射关系，详细配置请咨询MSC技术支持。

## [操作步骤](#ZH-CN_OPI_0170014693)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. 配置紧急号码信息。
  ![](激活VoLTE紧急呼叫（适用于MME）_70014693.assets/notice_3.0-zh-cn_2.png)

  紧急呼叫号码必须是运营商已经规划的号码，需要正确配置。配置错误可能导致UE的紧急呼叫接入错误，例如：

  - 将非紧急呼叫号码配置为紧急呼叫号码，用户被接入至政府部门。
    - 紧急服务分类配置错误，用户被错误接入其他政府部门。
    - VOLTE紧急呼叫号码下发会影响终端存储的紧急呼叫号码列表，当终端回落到2/3G发起紧急呼叫时，需要MSC配置紧急呼叫类型和对应号码的映射关系，详细配置请咨询MSC技术支持。
  [**SET MMFUNC**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/MM扩展功能管理/设置移动性管理扩展功能(SET MMFUNC)_26145512.md)
  [**ADD EMGCNUM**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/紧急呼叫配置/紧急呼叫号码配置/增加紧急号码配置信息(ADD EMGCNUM)_26305314.md)
4. 配置下发紧急号码。
  [**SET SOFTPARAOFBIT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/操作维护/软件参数管理/软件参数比特位/设置软件参数表比特位(SET SOFTPARAOFBIT)_72345783.md)
    - 需要在GPRS/UMTS网络下发紧急号码时，设置软参DWORD_EX6 BIT10为0。
    - 需要在EPS网络下发紧急号码时：
          - 如果仅对支持VoLTE语音业务的用户下发紧急呼叫号码，则将DWORD_EX6 BIT11配置为0，将DWORD_EX12 BIT16配置为1。
          - 如果对EPS网络下的所有用户下发紧急呼叫号码，则将DWORD_EX6 BIT11配置为0。
5. 配置紧急呼叫功能参数。
  [**ADD EMGCFG**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/紧急呼叫配置/紧急呼叫功能配置/增加运营商紧急呼叫功能配置（ADD EMGCFG）_26305316.md)
6. **可选：**配置E-STN-SR和PGW-C关系。
  [**ADD PGWESTNSR**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/紧急呼叫配置/E-STN-SR和PGW关系配置表/增加E-STN-SR和PGW关系配置表(ADD PGWESTNSR)_72345105.md)
  > **说明**
  > 与华为MSC设备对接才需要配置此步骤。
7. **可选：**配置优化的寻呼参数。
  [**SET EMM**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/MM协议参数管理/S1模式MM协议参数/设置S1模式MM协议参数(SET EMM)_72225207.md)
  > **说明**
  > 紧急呼叫的寻呼重发采用优化的寻呼参数，详情参见特性 [WSFD-102001 VoLTE基础语音业务](../WSFD-102001 VoLTE基础语音业务_67023607.md) 。
8. **可选：**获取紧急呼叫用户IMEI信息。
  [**ADD S1IMEICFG**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/设备检查管理/S1模式IMEI配置/增加S1模式IMEI配置(ADD S1IMEICFG)_26305448.md)
  > **说明**
  > 为保证 “有USIM卡的受限用户” 在鉴权失败的场景下紧急呼叫的SRVCC切换成功，建议MME开启获取IMEI信息功能。
9. **可选：**配置PLMN的用户禁止使用VoLTE紧急呼叫业务。
  如果某PLMN的用户在PS域无法完成VoLTE紧急呼叫，需要配置此步骤使该类用户在CS域重新发起紧急呼叫业务。否则跳过此步骤。

  - 如果是归属地PLMN的用户，则执行如下命令配置。

## [任务示例](#ZH-CN_OPI_0170014693)

任务描述

某运营商已接入VoLTE网络，且已通过 [**ADD HPLMN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/归属网络运营商管理/MNO管理/MNO网络配置表/增加本地PLMN(ADD HPLMN)_26146074.md) / [**ADD MVNONET**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/归属网络运营商管理/MVNO管理/MVNO网络标识配置表/增加MVNO网络配置信息(ADD MVNONET)_72345663.md) / [**ADD CONNECTPLMN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/互联PLMN管理/增加互联PLMN(ADD CONNECTPLMN)_26305852.md) 命令为该运营商增加了配置信息，指定运营商的国家码为“123”，运营商的标识（NOID）为128。现在为该运营商启用火警VoLTE紧急呼叫业务，配置火警紧急号码为119。

脚本

// 进入 “MML命令行-UNC” 窗口。

//配置紧急号码信息。

```
SET MMFUNC: EMNUM=YES;
ADD EMGCNUM: MCC="123", ESC=POLICE-0&AMBULANCE-0&FIREBRIGADE-1&MARINEGUARD-0&MOUNTAIN-0, NUM="119";
```

//配置在GPRS/UMTS下发紧急呼叫号码列表。

```
SET SOFTPARAOFBIT: DT=DWORD_EX, PARANUM=6, VALUE=VALUE_0, POSITION=10;
```

//配置在EPC网络下发紧急号码，并且 EPS网络下仅对支持VoLTE语音业务的用户下发紧急呼叫号码。

```
SET SOFTPARAOFBIT: DT=DWORD_EX, PARANUM=6, VALUE=VALUE_0, POSITION=11;
SET SOFTPARAOFBIT: DT=DWORD_EX, PARANUM=12, VALUE=VALUE_1, POSITION=16;
```

//配置紧急呼叫功能参数。

```
ADD EMGCFG: NOID=128, EMGMODE=ALLUE,EMGAPN="EMG", ULAPNAMBR=10000, DLAPNAMBR=10000, QCI=5, ARPPRL=1, ARPPCI=YES, ARPPVI=NO; 
```

//（可选）配置E-STN-SR和PGW-C关系。

```
ADD PGWESTNSR: IPT=IPV4, PGWIPV4="10.10.10.10", IPV4MSK="255.255.255.255", ESTNSR="11111";
```

//（可选）配置优化的寻呼参数。

```
SET EMM: HPT3413=2, HPN3413=4, HPPAGINGDELTA=0;
```

//（可选）获取紧急呼叫用户IMEI信息。

```
ADD S1IMEICFG: SUBRANGE=IMSI_PREFIX, IMSIPRE="123456", GETIMEIPLC=IMEISV;
```

//配置PLMN的用户禁止使用VoLTE紧急呼叫业务。

//（可选）配置归属地PLMN的用户禁止使用VoLTE紧急呼叫。 执行该命令前请先通过 [**LST HPLMN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/归属网络运营商管理/MNO管理/MNO网络配置表/查询本地PLMN(LST HPLMN)_72345675.md) 查询当前是否已增加归属地PLMN配置信息。 根据查询结果，选择是否配置该参数。

```
MOD HPLMN: MCC="123", MNC="00", EMCBS=NO;
```

//（可选）配置互联PLMN的用户禁止使用VoLTE紧急呼叫。 执行该命令前请先通过 **[LST CONNECTPLMN](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/互联PLMN管理/查询互联PLMN(LST CONNECTPLMN)_72225723.md)** 查询当前是否已增加互联PLMN配置信息。 根据查询结果，选择是否配置该参数。

```
MOD CONNECTPLMN: MCC="123", MNC="00", EMCBS=NO;
```

//（可选）配置MVNO网络的用户禁止使用VoLTE紧急呼叫。 执行该命令前请先通过 [**LST MVNONET**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/归属网络运营商管理/MVNO管理/MVNO网络标识配置表/查询MVNO网络配置信息(LST MVNONET)_72225743.md) 查询当前是否已增加MVNO网络配置信息。 根据查询结果，选择是否配置该参数。

```
RMV MVNONET: MVNOID=1;
ADD MVNONET: MVNOMCC="123", MVNOMNC="00", MATCHIMSI="111", MVNOCC="860", MVNOID=1, EMCBS=NO;
```

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2VLEC01", SWITCH=ENABLE;
```

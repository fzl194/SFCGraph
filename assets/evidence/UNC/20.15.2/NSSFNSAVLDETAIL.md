## [命令功能](#ZH-CN_MMLREF_0000001156834605)

**适用NF：NSSF**

显示某个AMF下所有动态上报的切片可用性信息。

## [注意事项](#ZH-CN_MMLREF_0000001156834605)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001156834605)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001156834605)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| AMFID | AMF实例标识 | 可选必选说明：必选参数<br>参数含义：该参数表示AMF实例标识，通过该参数配置去查询指定的AMF上报的可用性信息详情。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~40。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001156834605)

运营商想要查询切片可用性详细信息时，执行此命令。

```
DSP NSSFNSAVLDETAIL: AMFID="0004c7d76079f5a24517a4e03d90e484975c";
%%DSP NSSFNSAVLDETAIL: AMFID="0004c7d76079f5a24517a4e03d90e484975c";%%
RETCODE = 0  操作成功

结果如下
--------
AMF实例标识                           AMF集合的标识  移动国家码  移动网号  跟踪区域码  切片服务类型  切片细分标识  切片可用性信息是否上报

0004c7d76079f5a24517a4e03d90e484975c  460-03-01-001  460         03        290102      1             010101        TRUE
0004c7d76079f5a24517a4e03d90e484975c  460-03-01-001  125         236       290102      1             010101        TRUE
(结果个数 = 2)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001156834605)

| 输出项名称 | 输出项解释 |
| --- | --- |
| AMF实例标识 | 该参数表示AMF实例标识，通过该参数配置去查询指定的AMF上报的可用性信息详情。 |
| AMF集合的标识 | 该参数用于描述AMF集合的标识。 |
| 移动国家码 | 该参数用于描述移动国家码。 |
| 移动网号 | 该参数用于描述移动网号。 |
| 跟踪区域码 | 该参数用于描述跟踪区域码。 |
| 切片服务类型 | 该参数用于描述切片服务类型。 |
| 切片细分标识 | 该参数用于表示切片细分标识。 |
| 切片可用性信息是否上报 | 该参数表示该切片可用性信息是否从本NSSF上报。（TRUE表示从本NSSF上报，FALSE表示从容灾的另一个NSSF上报）。<br>取值说明：<br>- TRUE(TRUE)<br>- FALSE(FALSE) |

## [命令功能](#ZH-CN_MMLREF_0246243299)

该命令用于查询多连接开关的配置。

## [注意事项](#ZH-CN_MMLREF_0246243299)

无

#### [操作用户权限](#ZH-CN_MMLREF_0246243299)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0246243299)

无

## [使用实例](#ZH-CN_MMLREF_0246243299)

查询多连接开关的配置：

```
%%LST MCSWITCH:;%%
RETCODE = 0  操作成功

结果如下
--------
多连接开关  =  开启
```

## [输出结果说明](#ZH-CN_MMLREF_0246243299)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 多连接开关 | 该参数用于表示多连接开关。 |

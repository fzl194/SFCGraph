# 删除PRA位置区成员（RMV NGPRAMEM）

- [注意事项](#ZH-CN_MMLREF_0244007664__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0244007664__1.3.4)

## [命令功能](#ZH-CN_MMLREF_0244007664)

**适用NF：AMF**

该命令用于从指定的PRA删除位置区成员。

## [注意事项](#ZH-CN_MMLREF_0244007664)

- 该命令执行后立即生效。

- 如果仅输入PRA ID，那么执行本命令将删除该PRA内的所有位置区成员。

#### [操作用户权限](#ZH-CN_MMLREF_0244007664)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0244007664)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PRAID | PRA标识 | 可选必选说明：必选参数<br>参数含义：该参数用于表示位置区成员所归属的PRA。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是8388608~16777215。<br>默认值：无<br>配置原则：无 |
| LOCTYPE | 位置区类型 | 可选必选说明：可选参数<br>参数含义：该参数用于表示PRA位置区成员的类型。<br>数据来源：全网规划<br>取值范围：<br>- “TAI（5G跟踪区标识）”：5G跟踪区标识<br>- “GNBID（gNodeB ID）”：gNodeB ID<br>- “NRCELLID（5G小区标识）”：5G小区标识<br>默认值：无<br>配置原则：无 |
| MCC | MCC | 可选必选说明：可选参数<br>参数含义：该参数用于表示组成PRA区域的位置区成员的移动国家码，与无线接入网的MCC一致。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度是3。只允许输入十进制数字（0-9）。<br>默认值：无<br>配置原则：无 |
| MNC | MNC | 可选必选说明：可选参数<br>参数含义：该参数用于表示组成PRA区域的位置区成员的移动网号，与无线接入网的MNC一致。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是2~3。只允许输入十进制数字（0-9）。<br>默认值：无<br>配置原则：无 |
| TACBEGIN | 跟踪区起始编码 | 可选必选说明：该参数在"LOCTYPE"配置为"TAI"时为条件必选参数。<br>参数含义：该参数用于表示跟踪区编码起始值。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度是6。TAC编码为16进制数，按照字符串格式输入，字符串长度为6，只能由数字（0-9），字母（A-F、a-f）组成。字母大小写不敏感。<br>默认值：无<br>配置原则：<br>- 如果组成PRA的跟踪区编码是连续的，则可通过本参数以及“跟踪区结束编码”联合表示；如果跟踪区编码是离散的，那么仅使用本参数表示该跟踪区。<br>- 不同记录间的TAC区间不允许存在重叠。 |
| GNBIDLEN | gNodeB标识长度 | 可选必选说明：该参数在"LOCTYPE"配置为"GNBID"时为条件必选参数。<br>参数含义：该参数用于指定gNodeB标识的长度（比特位数）。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是22~32。<br>默认值：无<br>配置原则：无 |
| GNBIDBEGIN | gNodeB起始标识 | 可选必选说明：该参数在"LOCTYPE"配置为"GNBID"时为条件必选参数。<br>参数含义：该参数用于表示gNodeB标识起始值。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是1~4294967295。<br>默认值：无<br>配置原则：<br>如果组成PRA的gNodeB标识是连续的，则可通过本参数以及“gNodeB结束标识”联合表示；如果gNodeB标识是离散的，那么仅使用本参数表示该gNodeB。<br>不同记录间的gNodeB ID区间不允许存在重叠。<br>可以通过DSP NGAPLINK获取gNodeB标识。 |
| NGCIDBEGIN | 5G小区起始标识 | 可选必选说明：该参数在"LOCTYPE"配置为"NRCELLID"时为条件必选参数。<br>参数含义：该参数用于表示5G小区标识起始值。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度是9。采用十六进制表示（无须输入“0x”前缀），只能由数字（0-9），字母（A-F、a-f）组成。字母大小写不敏感。<br>默认值：无<br>配置原则：<br>如果组成PRA的5G小区标识是连续的，则可通过本参数以及“5G小区结束标识”联合表示；如果5G小区标识是离散的，那么仅使用本参数表示该小区。<br>不同记录间的小区区间不允许存在重叠。 |

## [使用实例](#ZH-CN_MMLREF_0244007664)

删除标识为16777211的PRA内的所有位置区成员，执行如下命令：

```
RMV NGPRAMEM: PRAID=16777211;
```

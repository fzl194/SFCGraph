## [命令功能](#ZH-CN_MMLREF_0209651654)

**适用NF：AMF**

该命令用于查询N2接口固定速率流控信息。

## [注意事项](#ZH-CN_MMLREF_0209651654)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209651654)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209651654)

无

## [使用实例](#ZH-CN_MMLREF_0209651654)

查询N2接口固定速率流控信息，执行如下命令：

```
%%LST N2FIXEDFC:;%%
RETCODE = 0  操作成功

结果如下
--------
       固定速率流控开关  =  开启
    接收速率门限(个/秒)  =  10000
    发送速率门限(个/秒)  =  100000
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209651654)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 固定速率流控开关 | 用于开启或关闭N2接口固定速率流控功能。 |
| 接收速率门限(个/秒) | 用于设置N2接口接收速率的上限。 |
| 发送速率门限(个/秒) | 用于设置N2接口发送速率的上限。 |

## [命令功能](#ZH-CN_MMLREF_0000001669803421)

**适用NF：AMF**

该命令用于查询内统配置参数。

## [注意事项](#ZH-CN_MMLREF_0000001669803421)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001669803421)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001669803421)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| STATPRIORITY | 内统优先级 | 可选必选说明：可选参数<br>参数含义：该参数用于设置内统优先级。<br>数据来源：本端规划<br>取值范围：<br>- DEFAULTPRIORITY（默认优先级）<br>- LOWPRORITY（低优先级）<br>- HIGHPRORITY（高优先级）<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001669803421)

查询内统配置参数，执行如下命令：

```
%%LST INNERSTAT:;%%
RETCODE = 0  操作成功

结果如下
------------------------
内统优先级  内统的日志打印周期  内统的日志打印周期失效时间  

默认优先级  5分钟               0
低优先级    5分钟      	        0
高优先级    5分钟               0
(结果个数 = 3)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001669803421)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 内统优先级 | 该参数用于设置内统优先级。 |
| 内统的日志打印周期 | 该参数用于设置内统的日志打印周期。 |
| 内统的日志打印周期失效时间 | 该参数用于设置内统的日志打印周期失效时间。当内统的日志打印时间超过失效时间时，内统日志打印周期恢复为5min。 |

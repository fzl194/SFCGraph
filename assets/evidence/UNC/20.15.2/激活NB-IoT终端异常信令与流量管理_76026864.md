## [操作场景](#ZH-CN_OPI_0276026864)

本操作指导介绍在运行网络中激活NB-IoT终端异常信令与流量管理的操作过程。

本特性用于在NB-IoT终端行为异常情况下保护核心网免受异常信令冲击。

多数行业用户的NB-IoT终端的单次数传的通信包数（包含数据包和短消息），处于连接态的持续时长，单位周期内的通信次数，通信时段以及通信时段内的通信次数等其中一种或几种一般是比较规律的。本特性可根据终端的上述规律行为提前设定阈值，当终端行为出现异常，异常程度超过一定阈值时MME会主动触发释放S1连接或者下发Back off timer给终端，从而达到保护核心网的目的。

> **说明**
> 适用于MME。

## [必备事项](#ZH-CN_OPI_0276026864)

前提条件

- 请仔细阅读[WSFD-215401 NB-IoT终端异常信令与流量管理特性概述](特性概述_75808673.md)。
- 由于不同类型终端的行为有差异，需要结合行业用户终端行为规划相应的配置，以免影响正常业务。
- 已完成加载License。

## [任务示例](#ZH-CN_OPI_0276026864)

任务描述

**示例1** ：某行业用户，用户终端的IMSI前缀为12303000，行业APN为HANGYE1.COM，单次通信的数据包个数为3，通信持续时长约为100s，未知该行业用户终端异常行为情况，需要对其进行观察。

**示例2** ：某行业用户，用户终端的IMSI前缀为47003000，行业APN为HANGYE2.COM，单次通信的数据包个数为3，通信持续时长约为100s，该终端正常情况下每小时需要上报2次数据。

**示例3** ：某行业用户，用户终端的IMSI前缀为48003000，行业APN为HANGYE3.COM，单次通信的数据包个数为3，通信持续时长约为100s，每天的22:00～23:00上报一次日报表。

脚本

示例1：

//为最大限度降低对行业用户正常业务的影响，参数规划需考虑一定的容错率，具体需以现网调研情况为准。本示例中通信包数限制值取4，通信持续时长取110s，配置脚本如下，配置完成后通过性能指标117466827～117466829和117467227～117467229观察对终端异常行为的统计：

//配置信令控制策略

```
ADD M2MSIGCTRL: CFGINDEX=1, CTRLSW=OFF, COMPKGSW=ON, COMPKG=4, COMTIMESW=ON, COMTIME=110;
```

//打开信令控制开关。

```
ADD M2MPLCY: SUBRANGE=IMSI_PREFIX, IMSIPRE="12303000", APNNI="HANGYE1.COM", SIGCTRLSW=ON, SIGCTRLINDEX=1;
```

示例2：

//为最大限度降低对行业用户正常业务的影响，参数规划需考虑一定的容错率，具体需以现网调研情况为准。本示例中通信包数限制值取4，通信持续时长取110s，每周期允许通信次数取3，配置脚本如下：

//配置信令控制策略

```
ADD M2MSIGCTRL: CFGINDEX=2, CTRLSW=ON, COMPKGSW=ON, COMPKG=4, COMTIMESW=ON, COMTIME=110, COMPERIODSW=ON, COMPERIODUNIT=HOUR, COMPERIOD=1, COMPERIODSERTIME=3;
```

//打开信令控制开关。

```
ADD M2MPLCY: SUBRANGE=IMSI_PREFIX, IMSIPRE="47003000", APNNI="HANGYE2.COM", SIGCTRLSW=ON, SIGCTRLINDEX=2;
```

//打开本功能的License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2PTST01", SWITCH=ENABLE;
```

示例3：

//为最大限度降低对行业用户正常业务的影响，参数规划需考虑一定的容错率，具体需以现网调研情况为准。本示例中通信包数限制值取4，通信持续时长取110s，通信时段内允许的通信次数取2，配置脚本如下：

//配置信令控制策略

```
ADD M2MSIGCTRL: CFGINDEX=3, CTRLSW=ON, COMPKGSW=ON, COMPKG=4, COMTIMESW=ON, COMTIME=110, COMTIMEINTERVALSW=ON, COMFRE=DAY, BEGINTIME=22&00&00, ENDTIME=23&00&00, SERVICETIME=1;
```

//打开信令控制开关。

```
ADD M2MPLCY: SUBRANGE=IMSI_PREFIX, IMSIPRE="48003000",  APNNI="HANGYE3.COM", SIGCTRLSW=ON, SIGCTRLINDEX=3;
```

//打开本功能的License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2PTST01", SWITCH=ENABLE;
```

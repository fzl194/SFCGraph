## [操作场景](#ZH-CN_OPI_0276869184)

本操作指导介绍在运行网络中激活LTE和WiFi网络之间的紧急呼叫切换特性的操作过程。

LTE和WiFi（Wireless Fidelity） 网络之间的紧急呼叫切换是指用户在使用紧急呼叫时，从LTE网络移动到WiFi网络或者从WiFi网络移动到LTE网络时，业务能平滑切换，保证业务的连续性。

> **说明**
> 适用于MME。

## [必备事项](#ZH-CN_OPI_0276869184)

前提条件

- 请仔细阅读[WSFD-104505 LTE和WiFi 网络之间的切换](../WSFD-104505 LTE和WiFi 网络之间的切换_76144612.md)。
- 完成加载License。

数据

该操作无需准备数据。

## [操作步骤](#ZH-CN_OPI_0276869184)

配置MME数据

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. 设置PGW-C Identity的类型参数。
  根据3GPP 23402要求：Notify Request只能携带Hostname和IP地址其中一种。由于不能确保所有厂商的网元都按协议要求来填写，如存在PGW-C Identity不一致时，可能会导致后续业务流程失败。因此 UNC 可通过该配置，来选择给HSS上报的是FQDN或者是IP地址。
  [**SET DMCMPT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性配置/设置Diameter兼容性(SET DMCMPT)_26306080.md)
4. 在UE从GSM/UMTS网络初始接入，切换到LTE网络后再切换到WiFi网络场景下需要配置。用于保证切换到WiFi后仍能选到原来的网关。
  [**SET DMCMPT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性配置/设置Diameter兼容性(SET DMCMPT)_26306080.md)
5. 配置MME支持紧急呼叫能力。
  [**SET DMCMPT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性配置/设置Diameter兼容性(SET DMCMPT)_26306080.md)
6. 配置漫游/无卡用户的紧急呼叫PGW-C，可以基于FQDN或者IP进行相应配置。
  **[ADD EMGCFG](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/紧急呼叫配置/紧急呼叫功能配置/增加运营商紧急呼叫功能配置（ADD EMGCFG）_26305316.md)**

## [任务示例](#ZH-CN_OPI_0276869184)

任务描述

启用LTE和WiFi网络之间的切换特性，并支持紧急呼叫。针对本网用户，MME给HSS发送的PGW-C ID类型为FQDN，并且将切换的PDN承载的PGW-C ID信息通知HSS，配置MME支持紧急呼叫能力。其他用户MME给HSS发送的PGW-C ID类型为IP_ADDRESS，切换后的PDN承载的PGW-C ID信息不通知HSS，需要配置静态紧急呼叫PGW-C。

脚本

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2WIFI01", SWITCH=ENABLE;
```

//设置从GSM/UMTS切换到WiFi后PGW-C Identity更新策略以及PGW-C Identity的类型参数。

```
SET DMCMPT: PGWIDTYPE=IP_ADDRESS, UPDATEPGWID=NOUPDATE;
ADD DMCMPTBYIMSI: SUBRANGE=HOME_USER, NOID=0, PGWIDTYPE=FQDN, UPDATEPGWID=UPDATE;
```

//配置MME支持紧急呼叫能力。

```
SET DMCMPT:SF2=FLSTID2_BIT24-1;
```

//配置漫游或无卡用户紧急呼叫PGW。

```
ADD EMGCFG:NOID=128,EMGMODE=ALLUE,EMGAPN="EMG",ULAPNAMBR=10000,DLAPNAMBR=10000,QCI=5,ARPPRL=1,ARPPCI=YES,ARPPVI=NO, EMGPGWIDTYPE=FQDN, FQDN="HUAWEI1.COM.APN.EPC.MNC123.MCC123.3GPPNETWORK.ORG";
```

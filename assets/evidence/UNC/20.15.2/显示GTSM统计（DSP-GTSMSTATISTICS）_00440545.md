#### [命令功能](#ZH-CN_CONCEPT_0000001600440545)

该命令用于显示设备GTSM各协议统计计数。

#### [注意事项](#ZH-CN_CONCEPT_0000001600440545)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600440545)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600440545)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RUNAME | RU名称 | 可选必选说明：可选参数<br>参数含义：该参数用于查询指定资源单元的GTSM统计。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。通过DSP RU命令可以查询资源单元信息。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600440545)

- 显示指定资源单元的GTSM统计信息：
  ```
  DSP GTSMSTATISTICS: RUNAME="VNODE_VNRS_VNFC_IPU_0064";
  ```
  ```

  DSP GTSMSTATISTICS:;
  ```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600440545)

| 输出项名称 | 输出项解释 |
| --- | --- |
| RU名称 | 用于查询指定资源单元的GTSM统计。 |
| 协议名 | 用于显示协议类型。 |
| 报文总数 | 用于显示报文总数。 |
| 丢弃报文总数 | 用于显示丢弃报文总数。 |

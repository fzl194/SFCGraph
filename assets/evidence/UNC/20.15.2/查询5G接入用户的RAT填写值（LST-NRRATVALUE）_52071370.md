- [参数说明](#ZH-CN_MMLREF_0252071370__1.3.4)

## [命令功能](#ZH-CN_MMLREF_0252071370)

**适用NF：SMF、PGW-C**

查询5G接入用户的RAT填写值。

## [注意事项](#ZH-CN_MMLREF_0252071370)

无

#### [操作用户权限](#ZH-CN_MMLREF_0252071370)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0252071370)

无

## [使用实例](#ZH-CN_MMLREF_0252071370)

查询5G接入用户的RAT填写值。

```
LST NRRATVALUE:;
RETCODE = 0  操作成功

结果如下
------------------------
                  和OCS交互使用的RAT值  =  RAT取值为51
                   和CG交互使用的RAT值  =  RAT取值为51
        和AAA计费服务器交互使用的RAT值  =  RAT取值为51
(结果个数 = 1)
```

## [输出结果说明](#ZH-CN_MMLREF_0252071370)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 和OCS交互使用的RAT值 | 该参数用于设置和OCS交互使用的RAT值。 |
| 和CG交互使用的RAT值 | 该参数用于设置和CG交互使用的RAT值。 |
| 和AAA计费服务器交互使用的RAT值 | 该参数用于设置和AAA计费服务器交互使用的RAT值。 |

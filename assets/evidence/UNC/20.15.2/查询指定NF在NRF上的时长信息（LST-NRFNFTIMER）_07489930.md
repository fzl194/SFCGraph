## [命令功能](#ZH-CN_MMLREF_0000001307489930)

**适用NF：NRF**

该命令用于查询指定NF在NRF上的时长信息。

## [注意事项](#ZH-CN_MMLREF_0000001307489930)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001307489930)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001307489930)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| NFINSTANCEID | NF实例标识 | 可选必选说明：可选参数<br>参数含义：该参数用于表示NF实例标识。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~36。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001307489930)

在NRF上查询实例标识为"88888888-6666-1234-5678-123456789ABC"的时长信息，执行如下命令。

```
LST NRFNFTIMER: NFINSTANCEID="88888888-4444-1234-5678-123456789ABC";
%%LST NRFNFTIMER: NFINSTANCEID="88888888-4444-1234-5678-123456789ABC";%%
RETCODE = 0  操作成功

结果如下
------------
	      NF实例标识  =  88888888-4444-1234-5678-123456789abc
服务发现缓存有效时长(秒)  =  120
        订阅有效时长(秒)  =  120
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001307489930)

| 输出项名称 | 输出项解释 |
| --- | --- |
| NF实例标识 | 该参数用于表示NF实例标识。 |
| 服务发现缓存有效时长(秒) | 该参数用于表示系统对服务发现结果的缓存有效时长。在缓存有效时长内，该系统中的NF实例再次服务发现相同NF实例时，可以直接使用缓存中的结果，不需要再去NRF上处理。超过缓存有效时间，服务发现结果缓存数据将被清除。系统中的NF实例再次服务发现相同NF实例时需要重新向NRF发起服务发现请求。 |
| 订阅有效时长(秒) | 该参数用于表示NF服务订阅有效时长。在订阅有效时长内，订阅方NF订阅的内容生效且能正常收到被订阅NF的变化通知。超过此订阅有效时长，NF订阅的信息不再生效，被订阅方NF的变化将不再通知给订阅方NF。 |

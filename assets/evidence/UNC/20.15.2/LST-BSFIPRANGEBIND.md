# 查询BSF实例与IPRANGE之间的绑定关系（LST BSFIPRANGEBIND）

- [命令功能](#ZH-CN_MMLREF_0209651737__1.3.1)
- [注意事项](#ZH-CN_MMLREF_0209651737__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0209651737__1.3.4)
- [使用实例](#ZH-CN_MMLREF_0209651737__1.3.5)
- [输出结果说明](#ZH-CN_MMLREF_0209651737__1.3.6)

## [命令功能](#ZH-CN_MMLREF_0209651737)

**适用NF：SMF**

该命令用于查询BSF（Binding Support Function）所管辖的IP地址范围。

## [注意事项](#ZH-CN_MMLREF_0209651737)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0209651737)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209651737)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| BSFINSTANCENAME | BSF实例名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定BSF的实例名称。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是1~50。<br>默认值：无<br>配置原则：<br>该参数需要在ADD NFUUID中事先配置，可执行LST NFUUID进行查看。 |
| IPRANGENAME | IPRANGE名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定该BSF实例所管辖的IP地址范围的名称。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是1~64。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209651737)

查询所有的BSF实例所管辖的IP地址范围：

```
%%LST BSFIPRANGEBIND:;%%
RETCODE = 0  操作成功

结果如下
--------
         BSF实例名称  =  BSF_Instance_0
         IPRANGE名称  =  range1
         IPRANGE类型  =  IPv4地址类型
IPv4地址段范围起始值  =  192.168.2.1
IPv4地址段范围结束值  =  192.168.2.10
  IPv4地址段归属的域  =  NULL
IPv6前缀段范围起始值  =  ::
IPv6前缀段范围结束值  =  ::
             APN名称  =  NULL
       绑定的APN组名  =  NULL
  绑定的IPDOMAIN组名  =  NULL
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209651737)

| 输出项名称 | 输出项解释 |
| --- | --- |
| BSF实例名称 | 该参数用于指定BSF的实例名称。 |
| IPRANGE名称 | 该参数用于指定该BSF实例所管辖的IP地址范围的名称。 |
| IPRANGE类型 | 该参数用于指定该BSF实例所管辖的IP地址的地址类型。 |
| IPv4地址段范围起始值 | 该参数用于指定IPv4地址段范围起始值。 |
| IPv4地址段范围结束值 | 该参数用于指定IPv4地址段范围结束值。 |
| IPv4地址段归属的域 | 根据运营商的规划，如果为IPv4类型的地址段划分了域，则需要通过该参数指定。 |
| IPv6前缀段范围起始值 | 该参数用于指定IPv6前缀段范围起始值。 |
| IPv6前缀段范围结束值 | 该参数用于指定IPv6前缀段范围结束值。 |
| APN名称 | 该参数用于指定该IP地址范围所支持的APN实例名。 |
| 绑定的APN组名 | 该参数用于指定绑定的APN组名。 |
| 绑定的IPDOMAIN组名 | 该参数用于指定绑定的IPDOMAIN组名。 |

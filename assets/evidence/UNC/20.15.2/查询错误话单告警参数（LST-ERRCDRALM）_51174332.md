#### [命令功能](#ZH-CN_CONCEPT_0251174332)

**适用NF：NCG**

该命令用于查询NCG上报错误话单告警时对错误话单采用的监控模式。

#### [注意事项](#ZH-CN_CONCEPT_0251174332)

无。

#### [本地用户权限](#ZH-CN_CONCEPT_0251174332)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_CONCEPT_0251174332)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0251174332)

无。

#### [使用实例](#ZH-CN_CONCEPT_0251174332)

查询NCG对错误话单的监控模式：

```
LST ERRCDRALM:;
```

```
RETCODE = 0  操作成功。

结果如下:
------------
监控模式  =  周期性
(结果个数 = 1)

---    END
```

## [命令功能](#ZH-CN_MMLREF_0000002101166942)

该命令用于查询POD的GID、TB、NPTB信息。

## [注意事项](#ZH-CN_MMLREF_0000002101166942)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000002101166942)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

| 参数标识 | 参数名称 | 参数说明 |

## [使用实例](#ZH-CN_MMLREF_0000002101166942)

假如操作员想查询POD的GID信息，可以调用以下命令：

```
%%DSP GIDINFO: SCENE=PODNAME, PODNAME="a-pod-0";%%
RETCODE = 0  操作成功

结果如下
--------
 POD名称  =  a-pod-0
     GID  =  6000
      TB  =  2049
NPTB列表  =  1/2/3
(结果个数 = 1)

---    END
```

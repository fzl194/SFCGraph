## [命令功能](#ZH-CN_MMLREF_0000001883159234)

**适用NF：AMF**

在部署感知的场景下，通过LST SFCPARA查询SFC的参数。

## [注意事项](#ZH-CN_MMLREF_0000001883159234)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001883159234)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001883159234)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SFCINSTANCEID | SFC实例ID | 可选必选说明：可选参数<br>参数含义：该参数用于指定SFC网元的实例ID。<br>数据来源：对端协商<br>取值范围：字符串类型，输入长度范围是1~64。只允许输入十进制数字（0-9），除0之外不能以0开头。对应十进制数取值范围是0~4294967295。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001883159234)

若运营商想查询SFC实例ID为1的SFC参数信息，可以用如下命令：

```
%%LST SFCPARA: SFCINSTANCEID="1";%%
RETCODE = 0  操作成功

结果如下
------------------------
SFC实例ID  =  1
    IP类型  =  IPv4
  IPv4地址  =  192.168.4.15
  IPv6地址  =  ::
    端口号  =  80
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001883159234)

| 输出项名称 | 输出项解释 |
| --- | --- |
| SFC实例ID | 该参数用于指定SFC网元的实例ID。 |
| IP类型 | 该参数用于指定SFC的地址的IP类型。 |
| IPv4地址 | 该参数用于指定SFC的IPv4地址。 |
| IPv6地址 | 该参数用于指定SFC的IPv6地址。 |
| 端口号 | 该参数用于指定SFC的端口号。 |

#### [命令功能](#ZH-CN_MMLREF_0000001172225457)

**适用网元：MME**

该命令用于查询S1模式eMTC Mode B用户的MM协议参数。

#### [注意事项](#ZH-CN_MMLREF_0000001172225457)

该命令执行后立即生效。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172225457)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172225457)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172225457)

无。

#### [使用实例](#ZH-CN_MMLREF_0000001172225457)

查询S1模式下CE MODE B用户MM协议参数：

```
LST EMTCBEMM:;
```

```
%%LST EMTCBEMM:;%%
RETCODE = 0  操作成功。

操作结果如下
------------
    T3422 （s）  =  48
N3422 （times）  =  5
    T3450 （s）  =  48
N3450 （times）  =  5
    T3460 （s）  =  48
N3460 （times）  =  5
    T3470 （s）  =  48
N3470 （times）  =  5
    T3413 （s）  =  48
N3413 （times）  =  5
SUB_T3413 （s）  =  48
(结果个数 = 1)

---    END
```

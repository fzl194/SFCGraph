## [命令功能](#ZH-CN_MMLREF_0209651496)

**适用NF：AMF**

本命令用于查询特定AMF功能实体的服务列表。

## [注意事项](#ZH-CN_MMLREF_0209651496)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209651496)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209651496)

无

## [使用实例](#ZH-CN_MMLREF_0209651496)

查询AMF功能实体服务列表：

```
LST AMFSERVICELIST:;
%%LST AMFSERVICELIST:;%%
RETCODE = 0  操作成功

结果如下
------------------------
AMF服务名称  =  namfComm
(结果个数 = 1)
```

## [输出结果说明](#ZH-CN_MMLREF_0209651496)

| 输出项名称 | 输出项解释 |
| --- | --- |
| AMF服务名称 | 特定AMF功能实例的服务名称。 |

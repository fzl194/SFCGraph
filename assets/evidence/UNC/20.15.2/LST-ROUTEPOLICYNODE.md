#### [命令功能](#ZH-CN_CONCEPT_0000001550121758)

该命令用于查询路由策略节点信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001550121758)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550121758)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550121758)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| POLICYNAME | 路由策略名字 | 可选必选说明：可选参数<br>参数含义：该参数用来指定路由策略的名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～200。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001550121758)

查询路由策略节点信息：

```
LST ROUTEPOLICYNODE:;
```

```

RETCODE = 0  操作成功

(结果个数 = 2)
---    END
```

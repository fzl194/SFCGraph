## [命令功能](#ZH-CN_MMLREF_0000001336880621)

此命令用于查询ucfexec-pod的磁盘空间大小。

## [注意事项](#ZH-CN_MMLREF_0000001336880621)

执行命令前请确认UCF服务处于上线状态。

#### [操作用户权限](#ZH-CN_MMLREF_0000001336880621)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001336880621)

无

## [使用实例](#ZH-CN_MMLREF_0000001336880621)

查询ucfexec-pod的磁盘空间：

```
%%DSP UCFDISKSIZE: ;%%
RETCODE = 0  操作成功

结果如下
--------
UCF Pod Id      UCF可用磁盘总空间(GB)

ucfexec-pod-0   64         
ucfexec-pod-1   64    
(结果个数 = 2)       

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001336880621)

| 输出项名称 | 输出项解释 |
| --- | --- |
| UCF Pod Id | 该参数用于表示UCF服务的Pod Id。 |
| UCF可用磁盘总空间(GB) | 该参数用于表示UCF服务当前可用磁盘总空间，单位为GB。 |

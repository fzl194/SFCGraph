#### [命令功能](#ZH-CN_CONCEPT_0000001549802026)

查询WLR组件与TNLM交互的统计信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001549802026)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549802026)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549802026)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001549802026)

查询WLR组件与TNLM交互的统计信息：

```
DSP WLRTNLMSTC:;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001549802026)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 总数 | 用于表示隧道总数。 |
| 添加数 | 用于表示隧道添加数。 |
| 删除数 | 用于表示隧道删除数。 |
| 更新数 | 用于表示隧道更新数。 |
| 序列号错误数 | 用于表示序列号错误数。 |
| 对帐数 | 用于表示WLR和TNLM的对账次数。 |
| 批量数 | 用于表示上一次批量更新Tunnel的个数。 |
| 平滑数 | 用于表示WLR和TNLM的平滑次数。 |
| 最后原因 | 该参数用于表示上一次平滑原因：<br>- 无：未平滑。<br>- 平滑请求：收到平滑请求消息。<br>- 对账请求：收到对账请求消息。<br>- 序列号错误：收到序列号错误消息。 |
| 最后开始时间 | 用于表示上一次平滑开始时间。 |
| 最后结束时间 | 用于表示上次一平滑结束时间。 |
| 生产者状态机 | 该参数用于表示生产者平滑状态：<br>- 初始化：初始化状态。<br>- 平滑请求：平滑开始化状态。<br>- 平滑中：平滑中化状态。<br>- 平滑结束：平滑结束化状态。<br>- 正常：状态机运行中等待下次平滑化状态。 |
| 隧道状态机 | 该参数用于表示隧道管理平滑状态<br>- 初始化：初始化状态。<br>- 批量添加生产者开始：批量添加生产者开始化状态。<br>- 批量添加生产者：批量添加生产者化状态。<br>- 批量添加生产者结束：批量添加生产者结束化状态。<br>- 正常：状态机运行中等待下次平滑化状态。 |

## [操作场景](#ZH-CN_OPI_0193181080)

本操作指导介绍在运行网络中调测漫游用户QoS限制特性的操作过程。

## [必备事项](#ZH-CN_OPI_0193181080)

前提条件

- 请仔细阅读[WSFD-105002 漫游用户QoS限制特性概述](特性概述_93181077.md)。
- 完成[激活漫游用户QoS限制](激活漫游用户QoS限制_93181079.md)。

数据

该操作无需准备数据。

工具

- 测试终端
- OM Portal跟踪

## [操作步骤](#ZH-CN_OPI_0193181080)

在相关网元均完成本特性的配置后，可以采用以下步骤检查特性工作是否正常：

> **说明**
> 此处以 [任务示例](激活漫游用户QoS限制_93181079.md#ZH-CN_OPI_0193181079__example) 配置成功为例进行验证方法说明。

1. 用户（本网异地）开机，发起Attach请求。
2. 用户进行PDP激活，触发PDP激活流程。
3. 查看用户的PDP上下文中的“上行最大速率”和“下行最大速率”是否受限于 [**ADD IMSISMCHAR**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/QoS管理/Pre-R8 QoS/QoS协商控制/QoS协商参数管理/增加QoS协商参数(ADD IMSISMCHAR)_26306040.md) 配置的“上行最大速率”和“下行最大速率”。
  进入 “MML命令行-UNC” 窗口。
  [**DSP SMCTX**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/系统管理/用户数据库管理/显示承载上下文(DSP SMCTX)_72226033.md) : QUERYOPT=查询方式, IMSI="IMSI", IDTYPE=显示类型, CTXID=上下文标识;
  ```
  %%DSP SMCTX:QUERYOPT=BYIMSI, IMSI="123031100000007", IDTYPE=BYCTXID, CTXID=6;%%
  RETCODE = 0  操作成功

  结果如下:
  -------------------------
                          ............

                          上行最大速率  =  568kbps
                          下行最大速率  =  568kbps

                         ............
  (结果个数 = 1)

  ---    结束
  ```
  预期结果：用户的PDP上下文中的 “上行最大速率” 和 “下行最大速率” 受限于 [**ADD IMSISMCHAR**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/QoS管理/Pre-R8 QoS/QoS协商控制/QoS协商参数管理/增加QoS协商参数(ADD IMSISMCHAR)_26306040.md) 配置的 “上行最大速率” 和 “下行最大速率” 。

## [命令功能](#ZH-CN_MMLREF_0000002229906473)

**适用NF：SMF、SGW-C、PGW-C、GGSN**

该命令用于查询锁定白名单记录。

## [注意事项](#ZH-CN_MMLREF_0000002229906473)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000002229906473)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000002229906473)

无

## [使用实例](#ZH-CN_MMLREF_0000002229906473)

查询锁定白名单记录：

```
%%LST LCKWHITELST:;%%
RETCODE = 0  操作成功

结果如下
--------
白名单类型  IMSI             MSISDN           

IMSI        123456789012345  000000000000000  
MSISDN      000000000000000  123456789012345  
(结果个数 = 2)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000002229906473)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 白名单类型 | 该参数用于指定白名单的类型。 |
| IMSI | 该参数用于指定国际移动用户标识。 |
| MSISDN | 该参数用于指定移动台国际ISDN号码。 |

#### [命令功能](#ZH-CN_CONCEPT_0209897296)

**适用NF：PGW-C、SMF**

此命令用于查询所有DRA或者指定DRA的链接状态。

#### [注意事项](#ZH-CN_CONCEPT_0209897296)

- 未配置Diameter链路组的DRA不显示。
- 如果显示“No Link”记录，则说明此DRA未配置Diameter链路。

#### [操作用户权限](#ZH-CN_CONCEPT_0209897296)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0209897296)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| HOSTNAME | DRA主机名 | 可选必选说明：可选参数<br>参数含义：该参数用于表示查询DRA的主机名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～127。不支持空格，由软参BIT 150控制是否区分大小写。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0209897296)

- 查询指定主机名的DRA的状态信息：
  ```
  DSP DRASTATUS:HOSTNAME="dra1";
  ```
  ```

  RETCODE = 0  操作成功。
  ```

- 查询所有DRA的状态信息：
  ```
  DSP DRASTATUS:;
  ```

#### [输出结果说明](#ZH-CN_CONCEPT_0209897296)

| 输出项名称 | 输出项解释 |
| --- | --- |
| DRA主机名 | 用于表示查询DRA的主机名称。 |
| POD名称 | 用于表示DRA所在资源单元的名称。 |
| 本端地址 | 用于表示与DRA连接的本地IP地址。 |
| 本端子地址 | 用于表示与DRA连接的本地子IP地址。 |
| 对端地址 | 用于表示指定DRA的IP地址。 |
| 正常链路 | 用于表示本端与DRA的链路状态正常。 |
| 异常链路 | 用于表示本端与DRA的链路状态异常。 |
| 无链路 | 用于表示本端与DRA之间没有建立链路。 |
| 本端主机名 | 用于表示本端主机名。 |

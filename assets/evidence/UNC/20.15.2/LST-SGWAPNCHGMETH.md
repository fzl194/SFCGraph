#### [命令功能](#ZH-CN_CONCEPT_0209896993)

**适用NF：SGW-C**

LST SGWAPNCHGMETH命令查询APN下的用户是否产生S-GW话单。

#### [注意事项](#ZH-CN_CONCEPT_0209896993)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0209896993)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0209896993)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| APN | APN名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定APN实例名。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格以及特殊字符：“_”、“#”、“$”、“&”等。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0209896993)

查询APN实例aa的用户是否打开SGW的离线计费开关，是否生成SGW话单：

```
LST SGWAPNCHGMETH:APN="aa";
```

```

RETCODE = 0  操作成功。

SGW计费方式
-----------
        APN名称  =  aa
APN离线计费开关  =  禁止
(结果个数 = 1)
---    END
```

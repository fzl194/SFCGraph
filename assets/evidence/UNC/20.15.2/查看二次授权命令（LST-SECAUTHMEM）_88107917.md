#### [命令功能](#ZH-CN_TOPIC_0188107917)

查询需要二次授权的命令列表。

#### [注意事项](#ZH-CN_TOPIC_0188107917)

网元ID必须在系统中存在。

#### [参数说明](#ZH-CN_TOPIC_0188107917)

| **参数标识** | **参数名称** | **参数说明** |
| --- | --- | --- |
| MEID | 网元ID | 可选必选说明：可选参数<br>参数含义：网元ID。<br>取值范围：0～65535。<br>默认值：无。<br>配置原则：无。 |
| COMMAND | 命令名称 | 可选必选说明：可选参数<br>参数含义：需要查询的MML命令名称。<br>取值范围：长度不超过80的英文字符串。<br>默认值：无。<br>配置原则：无。 |

#### [使用实例](#ZH-CN_TOPIC_0188107917)

查看二次授权命令：

```
%%LST SECAUTHMEM:;%%
RETCODE = 0  操作成功

操作结果如下
------------
网元ID  命令名称     提示信息  

0       LST CONNECT  NULL              
0       LST WKSP     NULL         
(结果个数 = 2)

---    END
```

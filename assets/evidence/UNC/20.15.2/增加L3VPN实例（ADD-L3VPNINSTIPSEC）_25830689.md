## [命令功能](#ZH-CN_MMLREF_0000001225830689)

该命令用于增加L3VPN实例。

## [注意事项](#ZH-CN_MMLREF_0000001225830689)

- 该命令执行后立即生效。

- 不能增加VPN实例 _public_、__mpp_vpn_inner__ 。

- 最多可输入1001条记录。

- 系统部署完成后，已经存在初始记录，参数的初始记录值如下表：

| VRFNAME |
| --- |
| _public_ |

#### [操作用户权限](#ZH-CN_MMLREF_0000001225830689)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001225830689)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| VRFNAME | VPN实例名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定用户需要增加的L3VPN实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~31。<br>默认值：无<br>配置原则：<br>不支持空格。 |

## [使用实例](#ZH-CN_MMLREF_0000001225830689)

新建名称为“vrf1”的VPN实例：

```
ADD L3VPNINSTIPSEC: VRFNAME="vrf1";
```

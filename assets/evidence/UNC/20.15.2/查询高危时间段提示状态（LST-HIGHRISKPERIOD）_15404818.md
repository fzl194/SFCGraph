## [命令功能](#ZH-CN_MMLREF_0000001315404818)

该命令用于查询高危时间段提示状态、高危时间段和高危命令提示信息。

## [注意事项](#ZH-CN_MMLREF_0000001315404818)

无。

## [参数说明](#ZH-CN_MMLREF_0000001315404818)

无。

## [使用实例](#ZH-CN_MMLREF_0000001315404818)

查询高危时间段提示状态：

```
%%LST HIGHRISKPERIOD:;%% 
RETCODE = 0  操作成功

操作结果如下 
------------ 
高危时间段提示状态  =  开启           
          起始时间  =  09:36           
          结束时间  =  23:36           
          提示信息  =  SFWEFGSDF 
(结果个数 = 1)
 
---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001315404818)

命令执行正常，会返回当前系统高危时间段提示状态。输出结果说明如 [表1](#ZH-CN_MMLREF_0000001315404818__table47787675) 所示。

命令执行异常，请联系华为技术支持处理。输出错误码说明如 [表2](#ZH-CN_MMLREF_0000001315404818__table365213262475) 所示。

*表1 输出项解释*

| **输出项名称** | **输出项解释** |
| --- | --- |
| 高危时间段提示状态 | 高危时间段提示功能的状态。<br>- 关闭：关闭高危时间段提示。<br>- 开启：开启高危时间段提示。 |
| 起始时间 | 高危时间段的具体起始时间。 |
| 结束时间 | 高危时间段的具体结束时间。 |
| 提示信息 | 高危时间段执行高危命令的提示信息。 |

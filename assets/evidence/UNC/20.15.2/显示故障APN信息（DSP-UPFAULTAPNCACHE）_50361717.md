## [命令功能](#ZH-CN_MMLREF_0000001450361717)

**适用NF：PGW-C、SMF、GGSN**

该命令用于显示用户面故障APN信息。

## [注意事项](#ZH-CN_MMLREF_0000001450361717)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001450361717)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001450361717)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| UPINSTANCEID | UPF实例标识 | 可选必选说明：可选参数<br>参数含义：该参数用于指定UPF实例名称。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是0~50。不区分大小写。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001450361717)

查询所有用户面故障APN信息：

```
%%DSP UPFAULTAPNCACHE:;%%
RETCODE = 0  操作成功

结果如下
------------------------
         UPF实例标识  =  upf_instance_1
         UPF描述名称  =  NULL
             APN名称  =  huawei.com
            故障时间  =  2022-12-12T11:37:22+08:00
(Number of results = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001450361717)

| 输出项名称 | 输出项解释 |
| --- | --- |
| UPF实例标识 | 该参数用于指定UPF实例名称。 |
| UPF描述名称 | 该参数用于表示NF实例的描述信息。 |
| APN名称 | 该参数用于指示APN名称。 |
| 故障时间 | 该参数用于表示UPF最后一次上报APN故障时间。 |

## [命令功能](#ZH-CN_MMLREF_0209653623)

**适用NF：AMF**

该命令用于增加等价PLMN（Equivalent PLMN）组。在同一组内的PLMN互为等价PLMN。等价PLMN也叫对等PLMN，是指与UE当前选择的PLMN（即Registered PLMN，RPLMN）享有同等地位的PLMN。在PLMN选择过程中，RPLMN及其等价PLMN将被高优先选择。

## [注意事项](#ZH-CN_MMLREF_0209653623)

- 该命令执行后立即生效。

- 当指定用户群组时，等价PLMN列表对群组内的用户生效；否则针对所有用户都生效。
- 当指定跟踪区域组ID时，等价PLMN列表仅针对位置在此跟踪区域组内的用户生效；否则针对所有位置的用户都生效。
- 对于从指定的Serving PLMN接入的同一用户组和跟踪区域组标识，只允许在AMF上为其配置一个等价PLMN组；其中用户组通过ADD NGUSRGRP命令进行配置。跟踪区域组通过ADD EPLMNAREAMEM命令进行配置。
- 对于从指定的Serving PLMN接入的不同用户组或跟踪区域组标识，可在AMF为其分别配置不同的等价PLMN组。
- EPLMN匹配策略优先选择指定用户群组标识和跟踪区域组ID都有效的配置。

- 最多可输入128条记录。

#### [操作用户权限](#ZH-CN_MMLREF_0209653623)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209653623)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| GRPIDX | 等价PLMN组号 | 可选必选说明：必选参数<br>参数含义：该参数用于在UNC系统内唯一标识一个等价PLMN组。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~127。<br>默认值：无<br>配置原则：无 |
| PLMNIDX | PLMN索引 | 可选必选说明：必选参数<br>参数含义：该参数表示配置等价PLMN列表的Serving PLMN的索引信息，需要通过ADD NGSRVPLMN提前配置。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~127。<br>默认值：无<br>配置原则：无 |
| USRGRPID | 用户群组标识 | 可选必选说明：可选参数<br>参数含义：该参数用于标识EPLMN适用的用户群信息，需要通过ADD NGUSRGRP提前配置。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是0~4294967294。<br>默认值：无<br>配置原则：无 |
| TACGRPID | 跟踪区域组ID | 可选必选说明：可选参数<br>参数含义：该参数用于标识等价PLMN适用的区域，需要通过ADD EPLMNAREAMEM提前配置。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是0~4294967294。<br>默认值：无<br>配置原则：<br>如果该参数没有配置，则表示不需要根据区域来限制用户使用指定的等价PLMN列表。<br>当TACGRPID未输入取值时，系统会为此参数赋无效值4294967295(0xFFFFFFFF)。 |
| DESC | 描述信息 | 可选必选说明：可选参数<br>参数含义：该参数用于对等价PLMN组的描述，在运维中起助记的作用。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~32。<br>默认值：无<br>配置原则：<br>输入单空格将删除该参数已有配置项。 |

## [使用实例](#ZH-CN_MMLREF_0209653623)

为PLMN为12345的Serving PLMN配置等价PLMN组，其中Serving PLMN的索引为0，用户群组标识为1，跟踪区域组ID为3，则执行如下命令：

```
ADD EPLMNGRP: GRPIDX=0, PLMNIDX=0, USRGRPID=1, TACGRPID=3, DESC="for ServingPLMN 12345";
```

## [命令功能](#ZH-CN_MMLREF_0000001188343780)

灰度升级阶段，执行此命令，用于显示灰度升级正在进行的阶段。

## [注意事项](#ZH-CN_MMLREF_0000001188343780)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001188343780)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001188343780)

无

## [使用实例](#ZH-CN_MMLREF_0000001188343780)

显示灰度升级阶段：

```
%%LST 5GGUMINFO:;%%
RETCODE = 0  操作成功

结果如下
--------
阶段  =  PreUpgrade
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001188343780)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 阶段 | 该参数用于标识灰度升级所处的阶段。 |

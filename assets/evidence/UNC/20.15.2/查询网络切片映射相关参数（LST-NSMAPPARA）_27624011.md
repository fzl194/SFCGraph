## [命令功能](#ZH-CN_MMLREF_0000001227624011)

**适用NF：AMF**

该命令用于查询网络切片映射相关参数。

## [注意事项](#ZH-CN_MMLREF_0000001227624011)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001227624011)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001227624011)

无

## [使用实例](#ZH-CN_MMLREF_0000001227624011)

查询切片映射相关参数，执行如下命令：

```
%%LST NSMAPPARA:;%%
RETCODE = 0  操作成功

结果如下
--------
           本地切片映射开关  =  打开
漫游用户CFG切片增强功能开关  =  打开
           标准切片映射开关  =  打开
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001227624011)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 本地切片映射开关 | 该参数用于控制AMF是否为UE提供本地切片映射服务。 |
| 漫游用户CFG切片增强功能开关 | 该参数用于控制当漫游用户在Registration Request消息中Requested NSSAI信元携带的切片映射关系与核心网不一致或者未携带映射关系时，AMF在Registration Accept消息中是否下发Configured NSSAI。 |
| 标准切片映射开关 | 该参数用于控制AMF是否支持标准切片映射功能。<br>标准切片定义：SST为0~127，SD为空或者0xFFFFFF。 |

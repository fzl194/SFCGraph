## [命令功能](#ZH-CN_MMLREF_0000001870462561)

**适用NF：AMF**

该命令用于查询RRC Inactive参数。

## [注意事项](#ZH-CN_MMLREF_0000001870462561)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001870462561)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001870462561)

无

## [使用实例](#ZH-CN_MMLREF_0000001870462561)

查询RRC Inactive参数配置，执行如下命令：

```
%%LST RRCINACTTAI:;%%
RETCODE = 0  操作成功

输出结果如下
--------------
位置获取超时  =  失败响应
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001870462561)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 位置获取超时 | 该参数用于指定因UE进入RRC Inactive状态导致的获取位置信息超时，给周边网元的位置信息。 |

## [命令功能](#ZH-CN_MMLREF_0000001504291629)

该命令用于查询HTTP是否支持FQDN。

## [注意事项](#ZH-CN_MMLREF_0000001504291629)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001504291629)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001504291629)

无

## [使用实例](#ZH-CN_MMLREF_0000001504291629)

查询HTTP是否支持FQDN，可以用如下命令：

```
%%LST HTTPFQDNSW:;%%
RETCODE = 0  操作成功

结果如下
--------
        全局FQDN开关  =  关闭
间接路由场景FQDN开关  =  关闭
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001504291629)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 全局FQDN开关 | 该参数用于全局控制HTTP是否使用响应报文中Location字段携带的FQDN。 |
| 间接路由场景FQDN开关 | 该参数用于控制间接路由场景下HTTP是否使用响应报文中Location字段携带的FQDN。 |

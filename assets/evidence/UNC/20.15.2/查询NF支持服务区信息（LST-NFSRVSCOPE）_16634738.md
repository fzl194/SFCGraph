## [命令功能](#ZH-CN_MMLREF_0216634738)

**适用NF：AMF、SMF、NRF、NSSF、SMSF、NCG**

该命令用于查询NF实例支持的服务区信息。

## [注意事项](#ZH-CN_MMLREF_0216634738)

无

#### [操作用户权限](#ZH-CN_MMLREF_0216634738)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0216634738)

无

## [使用实例](#ZH-CN_MMLREF_0216634738)

运营商A需要查询NF实例支持的服务区信息。

```
%%LST NFSRVSCOPE:;%%
RETCODE = 0  操作成功

结果如下
--------
NF实例名称  =  AMF_Instance_0
服务区名称  =  CityA
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0216634738)

| 输出项名称 | 输出项解释 |
| --- | --- |
| NF实例名称 | 本参数用于指定NF实例名称。 |
| 服务区名称 | 本参数用于指定服务区名称。 |

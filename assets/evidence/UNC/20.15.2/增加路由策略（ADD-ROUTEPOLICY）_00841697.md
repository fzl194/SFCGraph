#### [命令功能](#ZH-CN_CONCEPT_0000001600841697)

该命令用于添加路由策略。

#### [注意事项](#ZH-CN_CONCEPT_0000001600841697)

- 该命令执行后立即生效。
- 该命令最大记录数为65535。
- 添加该路由策略时，要保证该路由策略未添加过。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600841697)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600841697)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| POLICYNAME | 路由策略名字 | 可选必选说明：必选参数<br>参数含义：该参数用于指定路由策略的名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～200。<br>默认值：无<br>配置原则：不支持输入空格，区分大小写。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600841697)

添加一个名字为a的路由策略：

```
ADD ROUTEPOLICY: POLICYNAME="a";
```

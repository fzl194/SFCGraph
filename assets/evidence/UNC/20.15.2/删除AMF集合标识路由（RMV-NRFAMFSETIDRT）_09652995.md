# 删除AMF集合标识路由（RMV NRFAMFSETIDRT）

## [命令功能](#ZH-CN_MMLREF_0209652995)

**适用NF：NRF**

该命令用于删除目标AMF的集合标识路由信息。

## [注意事项](#ZH-CN_MMLREF_0209652995)

- 立即生效

- 主备或双活组网的场景下，如果需要配置此命令，则两个NRF上均需执行此命令，且配置参数一致。

#### [操作用户权限](#ZH-CN_MMLREF_0209652995)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652995)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| AMFSETID | AMF集合标识 | 可选必选说明：必选参数<br>参数含义：该参数用于表示AMF集合标识。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度是3。按照十六进制输入，输入时不带0x，不足三位时从左边补0，取值范围0~3ff。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209652995)

运营商网络为三层网络，最高层PLMN-NRF，中间层H-NRF，最底层L-NRF。L-NRF1归属于H-NRF1，H-NRF1归属于PLMN-NRF。 在H-NRF1和PLMN-NRF上分别执行如下命令，删除AMF集合标识为086的路由信息。

```
RMV NRFAMFSETIDRT:AMFSETID="086";
```

#### [命令功能](#ZH-CN_CONCEPT_0000001550121798)

该命令用于显示本设备上IPv6静态路由使用的IID信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001550121798)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550121798)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550121798)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| AFTYPE | 地址族 | 可选必选说明：必选参数<br>参数含义：该参数用于表示IPv6静态路由。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- ipv6unicast：IPv6单播。<br>默认值：无 |
| INDIRECTID | 直连IID | 可选必选说明：可选参数<br>参数含义：该参数用于表示静态路由IID。<br>数据来源：本端规划<br>取值范围：十六进制整数类型，取值范围为0x0～0xFFFFFFFF。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001550121798)

显示本设备上IPv6静态路由使用的IID信息：

```
DSP SRIPV6IIDINFO:AFTYPE=ipv6unicast,INDIRECTID="E6000071";
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001550121798)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 直连IID | 用于表示静态路由IID。 |
| VPID | 用于表示迭代结果时使用的虚拟PID。 |
| 源下一跳 | 用于表示静态路由用户配置下一跳。 |
| 配置接口名字 | 用于表示静态路由配置出接口名称。 |
| VPN名字 | 用于表示静态路由前缀所在VPN名称。 |
| 下一跳所在VPN | 用于表示静态路由下一跳所在VPN。 |
| 前缀所在表 | 用于表示静态路由所在表。 |
| 标记 | 该参数用于表示静态路由标记：<br>- DG：表示IID已经下发。<br>- B：黑洞标记。 |
| 参考计数 | 用于表示静态路由当前引用此IID的活跃数。 |
| 迭代类型 | 该参数用于表示静态路由迭代类型：<br>- NO Relay：不迭代。<br>- IP Relay：迭代IP。<br>- TNL Relay：迭代隧道。 |
| 迭代深度 | 用于表示静态路由迭代深度。 |
| 上一次分配IID时间 | 用于表示静态路由上一次分配IID时间。 |
| 上一次回收IID时间 | 用于表示静态路由上一次回收IID时间。 |
| 上一次Up的时间 | 用于表示静态路由上一次Up的时间。 |
| 上一次Down的时间 | 用于表示静态路由上一次Down的时间。 |
| 本机地址 | 用于表示静态路由本机地址，::表示无。 |
| 远端地址 | 用于表示静态路由P2P接口远端地址，::表示无。 |
| 接口状态 | 用于表示静态路由接口状态包括Down和Up。 |
| 迭代下一跳 | 用于表示静态路由迭代下一跳，::表示无。 |
| 接口名字 | 用于表示静态路由迭代接口名称。 |
| 隧道ID | 用于表示静态路由隧道ID值。 |
| 抑制计数 | 用于表示静态路由循环迭代抑制当前计数。 |
| 抑制剩余时间（s） | 用于表示静态路由循环迭代抑制当前剩余时间。 |

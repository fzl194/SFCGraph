## [操作场景](#ZH-CN_OPI_0166313822)

本操作指导介绍在运行网络中激活基于权重/优先级的Diameter负荷分担特性的操作过程。

基于权重/优先级的Diameter负荷分担是指 UNC 在通过Diameter路由选择对端设备时，优选路由组中优先级最高Diameter路由，当高优先级的路由故障时，则选择次高的路由。相同优先级的路由之间可以通过预设的权重进行负荷分担。

> **说明**
> 适用于 SGSN、 MME。

## [操作步骤](#ZH-CN_OPI_0166313822)

1. 打开本特性的License配置开关。 进入 “MML命令行-UNC” 窗口。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
2. 配置到HSS的Diameter路由基于权重/优先级负荷分担
  配置步骤参考 [配置到HSS的数据](../../../../初始配置/UNC初始配置与调测/组网对接配置/配置AMF&MME&SGSN/配置MME/配置到HSS的数据_29306465.md) ，其中配置Diameter主机路由和Diameter域路由时，采用的选路模式为优先级权重，如下。
    - Diameter主机路由的选路模式为优先级权重
      [**ADD DMHOSTRT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter主机路由/增加Diameter主机路由(ADD DMHOSTRT)_26146272.md)
    - Diameter域路由的选路模式为优先级权重
      [**ADD DMRT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter路由/增加Diameter域路由配置(ADD DMRT)_26306100.md)
3. 配置到DRA的Diameter路由基于权重/优先级负荷分担
  配置步骤参考 [配置到DRA的数据](../../../../初始配置/UNC初始配置与调测/组网对接配置/配置AMF&MME&SGSN/配置MME/配置到DRA的数据_29346433.md) ，其中配置Diameter域路由时，采用的选路模式为优先级权重，即RSELMODE=SELMODE_PRIORITY_WEIGHT，如下。
  Diameter域路由的选路模式为优先级权重
  [**ADD DMRT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter路由/增加Diameter域路由配置(ADD DMRT)_26306100.md)

## [任务示例](#ZH-CN_OPI_0166313822)

任务描述

- 场景一
  主机路由基于权重的Diameter负荷分担
- 场景二
  域路由基于优先级的Diameter负荷分担
- 场景三
  主机路由和域路由混合组网，基于优先级的Diameter负荷分担

脚本

- 场景一脚本
  //两个主机路由，基于权重的负荷分担。
  ```
  //打开License配置开关。
  ```
  ```
  SET LICENSESWITCH: LICITEM="LKV2DMLS01", SWITCH=ENABLE;
  ```
  //配置主机路由索引0和1，选路模式为优先级权重，优先级为0，权重分别是1和2。
  ```
  ADD DMHOSTRT: ROUTEIDX=0, APPNAM=S6A/S6D, RSELMODE=SELMODE_PRIORITY_WEIGHT, PEERSEL=PEER_INDEX, PEERIDX=0, PRIORITY=0, WEIGHT=1;
  ADD DMHOSTRT: ROUTEIDX=1, APPNAM=S6A/S6D, RSELMODE=SELMODE_PRIORITY_WEIGHT, PEERSEL=PEER_INDEX, PEERIDX=1, PRIORITY=0, WEIGHT=2;
  ```
  //将主机路由配置到相同索引的路由组中。
  ```
  ADD DMRTGRP: GRPIDX=0, RTMODE=HOST_ROUTE, ROUTEIDX=0;
  ADD DMRTGRP: GRPIDX=0, RTMODE=HOST_ROUTE, ROUTEIDX=1;
  ```
  //在路由组中配置到IMS与IHSS的映射关系。
  ```
  ADD IMSIHSS: IMSIPRE="123031501", HSSRLM="hss1501.huawei03.com", GRPIDX=0;
  ```
- 场景二脚本
  //两个域路由，基于优先级的负荷分担
  //打开License配置开关。
  ```
  SET LICENSESWITCH: LICITEM="LKV2DMLS01", SWITCH=ENABLE;
  ```
  //配置同一域路由索引中配置两个peer，选路模式为优先级权重，优先级分别是0和1，权重为1。
  ```
  ADD DMRT: ROUTEIDX=0, APPNAM=S6A/S6D, ISDEFAULT=NO, RSELMODE=SELMODE_PRIORITY_WEIGHT, REALMNAME="hss1501.huawei03.com", PEERIDX=0, PRIORITY=0, WEIGHT=1;
  ADD DMRT: ROUTEIDX=0, APPNAM=S6A/S6D, ISDEFAULT=NO, RSELMODE=SELMODE_PRIORITY_WEIGHT, REALMNAME="hss1501.huawei03.com", PEERIDX=1, PRIORITY=1, WEIGHT=1;
  ```
  //在路由组中配置到IMS与IHSS的映射关系。
  ```
  ADD IMSIHSS: IMSIPRE="123031501", HSSRLM="hss1501.huawei03.com";
  ```
- 场景三脚本
  //主机路由和域路由混合组网，基于优先级的负荷分担。
  //打开License配置开关。
  ```
  SET LICENSESWITCH: LICITEM="LKV2DMLS01", SWITCH=ENABLE;
  ```
  //配置主机路由索引0和1，选路模式为优先级权重，优先级分别是0和1，权重为1。
  ```
  ADD DMHOSTRT: ROUTEIDX=0, APPNAM=S6A/S6D, RSELMODE=SELMODE_PRIORITY_WEIGHT, PEERSEL=PEER_INDEX, PEERIDX=0, PRIORITY=0, WEIGHT=1;
  ADD DMHOSTRT: ROUTEIDX=1, APPNAM=S6A/S6D, RSELMODE=SELMODE_PRIORITY_WEIGHT, PEERSEL=PEER_INDEX, PEERIDX=1, PRIORITY=1, WEIGHT=1;
  ```
  //配置同一域路由索引中配置两个peer，选路模式为优先级权重，优先级分别是0和1，权重为1。
  ```
  ADD DMRT: ROUTEIDX=0, APPNAM=S6A/S6D, ISDEFAULT=NO, RSELMODE=SELMODE_PRIORITY_WEIGHT, REALMNAME="hss1501.huawei03.com", PEERIDX=2, PRIORITY=0, WEIGHT=1;
  ADD DMRT: ROUTEIDX=0, APPNAM=S6A/S6D, ISDEFAULT=NO, RSELMODE=SELMODE_PRIORITY_WEIGHT, REALMNAME="hss1501.huawei03.com", PEERIDX=3, PRIORITY=1, WEIGHT=1;
  ```
  //将主机路由配置到相同索引的路由组中，并选择优选主机路由模式。
  ```
  ADD DMRTGRP: GRPIDX=0, RTMODE=HOST_ROUTE, RTPRIMODE=HOST_ROUTE_PREFER, ROUTEIDX=0;
  ADD DMRTGRP: GRPIDX=0, RTMODE=HOST_ROUTE, RTPRIMODE=HOST_ROUTE_PREFER, ROUTEIDX=1;
  ```
  //将域路由配置到相同索引的路由组中，并选择优选主机路由模式。
  ```
  ADD DMRTGRP: GRPIDX=0, RTMODE=REALM_ROUTE, RTPRIMODE=HOST_ROUTE_PREFER, ROUTEIDX=0;
  ```
  //在路由组中配置到IMS与IHSS的映射关系。
  ```
  ADD IMSIHSS: IMSIPRE="123031501", HSSRLM="hss1501.huawei03.com", GRPIDX=0;
  ```

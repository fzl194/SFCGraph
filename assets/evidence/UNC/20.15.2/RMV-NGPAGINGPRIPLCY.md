# 删除5G寻呼优先级策略参数配置（RMV NGPAGINGPRIPLCY）

- [命令功能](#ZH-CN_MMLREF_0209652645__1.3.1)
- [注意事项](#ZH-CN_MMLREF_0209652645__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0209652645__1.3.4)
- [使用实例](#ZH-CN_MMLREF_0209652645__1.3.5)

## [命令功能](#ZH-CN_MMLREF_0209652645)

**适用NF：AMF**

该命令用于删除5G寻呼优先级策略配置。

## [注意事项](#ZH-CN_MMLREF_0209652645)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0209652645)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652645)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PLCYIDX | 策略索引 | 可选必选说明：必选参数<br>参数含义：该参数用于在AMF内唯一标识一条寻呼优先级策略。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~127。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209652645)

将策略索引为1的5G寻呼优先级配置删除，执行如下命令：

```
RMV NGPAGINGPRIPLCY: PLCYIDX=1;
```

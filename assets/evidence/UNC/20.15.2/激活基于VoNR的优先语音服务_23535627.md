## [操作场景](#ZH-CN_OPI_0000001223535627)

本操作指导介绍在运行网络中激活基于VoNR的优先语音服务的操作过程。

> **说明**
> 适用于SMF。

## [必备事项](#ZH-CN_OPI_0000001223535627)

前提条件

- 请仔细阅读[WSFD-102705 基于VoNR的优先语音服务特性概述](特性概述_23614189.md)。
- 已部署VoNR基础语音业务，详细请参考[WSFD-102701 VoNR基础语音业务](../WSFD-102701 VoNR基础语音业务_11845316.md)。
- AMF上已根据PPI配置不同的寻呼定时器、重发次数和重呼间隔时间，生成差异化的寻呼策略，为语音业务配置高优先级的寻呼策略。详细请参考 [差异化寻呼](../../流控功能/WSFD-206001 精准寻呼/WSFD-206001 精准寻呼（5G）/实现原理/差异化寻呼_59289453.md) 。

## [操作步骤](#ZH-CN_OPI_0000001223535627)

1. 进入 “MML命令行-UNC” 窗口。
2. **可选：**通过 **[ADD APN](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/APN管理/APN/增加APN配置（ADD APN）_09653747.md)** 命令开启PPI功能。
3. 通过 **[ADD APN](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/APN管理/APN/增加APN配置（ADD APN）_09653747.md)** 命令中继承 **[SET SMFFUNC](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/会话协议参数管理/会话管理拓展功能/5GC会话管理拓展功能/设置SMF扩展功能（SET SMFFUNC）_09653731.md)** 命令中的全局配置开启PPI功能。

## [任务示例](#ZH-CN_OPI_0000001223535627)

任务描述

按照网络规划，需要将使用APN为ims的业务，在SMF上使能该APN的PPI功能开关，后续AMF可以根据PPI识别语音业务，实现语音的优先寻呼。

脚本

//脚本一：直接通过ADD APN命令开启PPI功能。

```
ADD APN:APN="ims", PPISWITCH=ENABLE;
```

//脚本二：通过ADD APN命令中继承SET SMFFUNC命令中的全局配置开启PPI功能。

```
SET SMFFUNC:PPISWITCH=ENABLE;
ADD APN:APN="ims", PPISWITCH=INHERIT;
```

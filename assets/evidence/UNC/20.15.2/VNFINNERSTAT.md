## [命令功能](#ZH-CN_MMLREF_0000002136685937)

该命令用于显示网元的内部状态。

## [注意事项](#ZH-CN_MMLREF_0000002136685937)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000002136685937)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000002136685937)

无

## [使用实例](#ZH-CN_MMLREF_0000002136685937)

当运营商需要对网元进行操作（如升级、补丁等）时，可以使用该命令查询网元内部状态，提前识别风险。

```
%%DSP VNFINNERSTAT:;%%
RETCODE = 0  操作成功

结果如下
--------
分类      状态  备注  

POD状态   正常  NULL  
服务状态  正常  NULL  
规格状态  正常  NULL  
(结果个数 = 3)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000002136685937)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 分类 | 状态分类。<br>取值说明：<br>- “PODSTAT（POD状态）”：代表网元内的POD状态，由SCF感知到的POD内容器状态、心跳监测等状态汇集。<br>- “SRVSTAT（服务状态）”：代表网元内服务平面上的服务状态。<br>- “SPECSTAT（规格状态）”：代表网元规格是否处于稳态。 |
| 状态 | 状态。<br>取值说明：<br>- “Normal（正常）”：状态正常<br>- “Abnormal（异常）”：状态异常 |
| 备注 | 备注说明。 |

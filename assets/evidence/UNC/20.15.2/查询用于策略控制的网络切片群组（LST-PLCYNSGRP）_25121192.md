# 查询用于策略控制的网络切片群组（LST PLCYNSGRP）

- [参数说明](#ZH-CN_MMLREF_0225121192__1.3.4)

## [命令功能](#ZH-CN_MMLREF_0225121192)

**适用NF：AMF**

该命令用于查询应用于AM策略或者UE策略控制的网络切片群组。

## [注意事项](#ZH-CN_MMLREF_0225121192)

无

#### [操作用户权限](#ZH-CN_MMLREF_0225121192)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0225121192)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| NSGRPID | 网络切片群组标识 | 可选必选说明：可选参数<br>参数含义：该参数用于标识应用于AM策略或者UE策略控制参数的网络切片群组。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是1~16。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0225121192)

- 查询“网络切片群组标识”为“1”的用于AM策略或者UE策略控制的网络切片群组，执行如下命令：
  ```
  %%LST PLCYNSGRP: NSGRPID=1;%%
  RETCODE = 0  操作成功

  结果如下
  --------
  网络切片群组标识   =  1
           描述信息  =  group1
  (结果个数 = 1)

  ---    END
  ```
- 查询系统内当前配置的用于AM策略或者UE策略控制的网络切片群组，执行如下命令：
  ```
  %%LST PLCYNSGRP:;%%
  RETCODE = 0  操作成功

  结果如下
  --------
  网络切片群组标识   =  1
           描述信息  =  group1
  (结果个数 = 1)

  ---    END
  ```

## [输出结果说明](#ZH-CN_MMLREF_0225121192)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 网络切片群组标识 | 该参数用于标识应用于AM策略或者UE策略控制参数的网络切片群组。 |
| 描述信息 | 该参数是对用于AM策略或者UE策略控制的网络切片群组的描述信息，在运维中起助记作用。 |

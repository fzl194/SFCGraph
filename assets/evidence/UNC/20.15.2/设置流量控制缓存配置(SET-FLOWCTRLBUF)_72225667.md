#### [命令功能](#ZH-CN_MMLREF_0000001172225667)

**适用网元：SGSN**

该命令用于修改BSSGP流控缓存参数配置。

#### [注意事项](#ZH-CN_MMLREF_0000001172225667)

- 系统初次运行时，会执行系统初始设置值。
- 该命令执行后立即生效。
- 该命令需要华为技术支持人员指导下才能执行，请慎重使用。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172225667)

manage-ug；system-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172225667)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [使用实例](#ZH-CN_MMLREF_0000001172225667)

修改BSSGP流控缓存配置：

SET FLOWCTRLBUF: MSSIGQL=16, MSDATQL=100, MSCONTHD=60, MSCONRTHD=50, BVCSIGQL=200, BVCDATQL=500, BVCCONTHD=60, BVCCONRTHD=50;

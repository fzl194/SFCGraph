## [命令功能](#ZH-CN_MMLREF_0000001244540604)

**适用NF：SMSF**

该命令用于查询NF实例支持的群组信息。

## [注意事项](#ZH-CN_MMLREF_0000001244540604)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001244540604)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001244540604)

无

## [使用实例](#ZH-CN_MMLREF_0000001244540604)

查询NF支持的群组信息。

```
%%LST NFGROUPID:;%%
RETCODE = 0  操作成功

结果如下
--------
NF实例名称  =  SMSF_Instance_0
  群组标识  =  GROUP01
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001244540604)

| 输出项名称 | 输出项解释 |
| --- | --- |
| NF实例名称 | 该参数用于指定NF实例名称。 |
| 群组标识 | 该参数用于指定NF服务的群组标识。 |

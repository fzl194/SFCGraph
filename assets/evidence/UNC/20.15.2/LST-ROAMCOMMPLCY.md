## [命令功能](#ZH-CN_MMLREF_0000001179048182)

**适用NF：AMF、SMF**

该命令用于显示漫游通信策略信息。

## [注意事项](#ZH-CN_MMLREF_0000001179048182)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001179048182)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001179048182)

无

## [使用实例](#ZH-CN_MMLREF_0000001179048182)

运营商需要查询漫游通信策略；

```
%%LST ROAMCOMMPLCY:;%%
RETCODE = 0  操作成功

结果如下
------------------------
漫游路由模式 = 通过SEPP通信
信令汇聚模式 = Http Proxy
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001179048182)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 漫游路由模式 | 该参数用于指定漫游路由模式。 |
| 信令汇聚模式 | 该参数用于指定漫游信令汇聚模式。仅在ROUTEMODE设置为SEPP或SCP时生效。 |

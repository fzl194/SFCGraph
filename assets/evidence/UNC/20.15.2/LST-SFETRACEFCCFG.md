#### [命令功能](#ZH-CN_CONCEPT_0000002048203926)

该命令用来查询当CPU负载超阈值时，VNRS IP消息跟踪的CPU流控功能是否开启。

#### [注意事项](#ZH-CN_CONCEPT_0000002048203926)

- 该命令执行后立即生效。
- 该命令仅适用于非NP卡场景和NP卡非加速模式场景。

#### [操作用户权限](#ZH-CN_CONCEPT_0000002048203926)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000002048203926)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000002048203926)

查询VNRS IP消息跟踪的CPU流控功能开关状态：

```
LST SFETRACEFCCFG:;
```

```
RETCODE = 0  操作成功

结果如下
------------------------
流控使能标记  =  否
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000002048203926)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 流控使能标记 | 该参数用于指定当CPU负载超阈值时，是否开启VNRS IP消息跟踪的CPU流控功能。 |

## [命令功能](#ZH-CN_MMLREF_0000001870382321)

**适用NF：SMF**

该命令用于查询SMF是否支持多SMFInfo场景下的会话重建功能。

## [注意事项](#ZH-CN_MMLREF_0000001870382321)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001870382321)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001870382321)

无

## [使用实例](#ZH-CN_MMLREF_0000001870382321)

查询SMF是否支持多SMFInfo场景下的会话重建功能。

```
%%LST MULSMFPDUREEPOL;%%
RETCODE = 0  操作成功

结果如下
--------
是否支持多SMFInfo下PDU会话重建 = 不支持
（结果个数 = 1）
---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001870382321)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 是否支持多SMFInfo下会话重建 | 该参数用于指定SMF是否支持多SMFInfo场景下的会话重建功能。 |

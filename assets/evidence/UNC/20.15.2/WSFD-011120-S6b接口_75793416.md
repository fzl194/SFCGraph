#### [适用NF](#ZH-CN_CONCEPT_0275793416)

PGW-C

#### [定义](#ZH-CN_CONCEPT_0275793416)

S6b接口是PGW-C和3GPP AAA Server之间用于移动性认证的接口，仅在Non-3GPP接入及CL 、WL 互操作网络中使用。S6b接口用于获取及请求移动性参数，在不支持动态PCC的网络中为Non-3GPP接入UE提供静态的QoS信息。

#### [客户价值](#ZH-CN_CONCEPT_0275793416)

| 受益方 | 受益描述 |
| --- | --- |
| 运营商 | UNC<br>的S6b接口基于3GPP标准实现，可以实现PGW-C与不同厂家的3GPP AAA Server设备对接。 |
| 用户 | 用户不感知该特性。 |

#### [应用场景](#ZH-CN_CONCEPT_0275793416)

当需要作为PGW-C的 UNC 与AAA Server对接实现移动性认证时，需要配置本接口。

#### [原理概述](#ZH-CN_CONCEPT_0275793416)

**S6b接口协议栈**

S6b接口协议基于Diameter协议栈，是Diameter协议应用之一，通常叫做NASREQ（Network Access Server Requirements），定义在RFC4005 Diameter Network Access Server Application中。

**S6b接口业务流程**

S6b接口涉及的流程如下：

- PGW-C发起的会话终结流程
- 3GPP AAA发起的会话终结流程
- 业务鉴权更新流程

由于单链路处理能力有限， UNC 能够支持单host多IP的组网方式和多host多IP的组网方式。

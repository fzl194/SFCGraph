## [命令功能](#ZH-CN_MMLREF_0000001124796822)

**适用NF：NRF**

该命令用于查询NRF作为归属地NRF时对指定拜访地PLMN的跨PLMN请求处理策略。

## [注意事项](#ZH-CN_MMLREF_0000001124796822)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001124796822)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001124796822)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| MCC | 移动国家码 | 可选必选说明：可选参数<br>参数含义：该参数用于表示拜访地PLMN的移动国家码。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度是3。<br>默认值：无<br>配置原则：无 |
| MNC | 移动网号 | 可选必选说明：可选参数<br>参数含义：该参数用于表示拜访地PLMN的移动网号。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是2~3。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001124796822)

- 查询所有指定归属地PLMN的拜访地策略；
  ```
  %%LST NRFPLMNHOMEPLY:;%%
  RETCODE = 0  操作成功

  结果如下
  ------------------------
  移动国家码  移动网号  是否支持跨PLMN订阅    信息精简策略  

  460         02        打开                  IP   
  466         03        打开                  IP   
  (结果个数 = 2)

  ---    END
  ```
- 查询移动国家码为460，移动网号为03的指定归属地PLMN的拜访地策略。
  ```
  %%LST NRFPLMNHOMEPLY: MCC="460", MNC="02";%%
  RETCODE = 0  操作成功

  结果如下
  ------------------------
          移动国家码  =  460
            移动网号  =  02
  是否支持跨PLMN订阅  =  打开
        信息精简策略  =  IP
  (结果个数 = 1)

  ---    END
  ```

## [输出结果说明](#ZH-CN_MMLREF_0000001124796822)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 移动国家码 | 该参数用于表示拜访地PLMN的移动国家码。 |
| 移动网号 | 该参数用于表示拜访地PLMN的移动网号。 |
| 是否支持跨PLMN订阅 | 该参数用于表示本NRF作为归属地NRF时，是否处理指定的拜访地PLMN的NF发起的对本PLMN内的NF订阅以及订阅更新请求。开关为FUNC_ON表示处理，开关为FUNC_OFF表示不处理，并返回403错误码。订阅更新请求仅最终处理的NRF会按照此开关进行控制，中间层NRF仅做透传处理。 |
| 信息精简策略 | 该参数用于表示NRF作为归属地NRF时，对于收到指定的拜访地PLMN的跨PLMN服务发现和通知中精简的NF Profile字段信息。<br>选择某一项表示该项被精简，不在消息中携带，不选择表示不精简。 |

# 调测基于TA的网络切片选择

- [操作场景](#ZH-CN_OPI_0245188370__1.3.1)
- [必备事项](#ZH-CN_OPI_0245188370__1.3.2)

## [操作场景](#ZH-CN_OPI_0245188370)

本操作指导介绍在运行网络中调测支持基于TA选择切片的操作过程。

> **说明**
> 适用于AMF 、NSSF 。

## [必备事项](#ZH-CN_OPI_0245188370)

前提条件

- 请仔细阅读[WSFD-110004 基于TA的网络切片选择特性概述](特性概述_45188367.md)。

- 完成[激活基于TA的网络切片选择](激活基于TA的网络切片选择_45188369.md)。

数据

该操作无需准备数据。

工具

该操作无需准备其他工具。

## [操作步骤](#ZH-CN_OPI_0245188370)

1. 进入 “MML命令行-UNC” 窗口。
2. 执行 **[LST NSSFFUNCPARA](../../../../../OM参考/命令/UNC MML命令/业务服务管理/NSSF业务及策略管理/NSSF功能参数配置/查询NSSF数据源以及切片选择流程返回信元（LST NSSFFUNCPARA）_44007076.md)** 命令，查询NSSF在业务流程中所使用的数据源。
  预期结果：NSSF在业务流程中所使用的数据源已经配置为本地配置可用性信息。

  ```
  %%LST NSSFFUNCPARA:;%% 
  RETCODE = 0  执行成功  
  结果如下 
  -------------------------
             数据源设置  =  本地配置可用性信息
   切片选择返回信源设置  =  targetAmfSet
  (结果个数 = 1)

  ---    END
  ```

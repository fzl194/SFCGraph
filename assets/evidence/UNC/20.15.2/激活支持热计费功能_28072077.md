## [操作场景](#ZH-CN_OPI_0228072077)

本操作指导介绍在运行网络中激活支持热计费功能的操作过程。

热计费是一种计费方式，具有普通计费的所有功能，只是话单产生速度比普通计费话单更快，保证运营商对此类用户及时收费。

> **说明**
> 适用于 GGSN、 SGW-C、PGW-C。

## [操作步骤](#ZH-CN_OPI_0228072077)

1. 热计费功能的配置与离线计费的配置类似，请参见 [激活离线计费](../../WSFD-011201 支持离线计费/激活离线计费_25768943.md) 。
  > **说明**
  > 热计费与普通离线计费的不同点在于：
  >
  > - 热计费功能只有在离线计费下启用，所以请开启离线计费开关。
  > - 热计费功能话单产生速度稍快些，配置热计费话单产生参数时，参数阈值比普通离线计费话单要小，配置时应注意。

## [任务示例](#ZH-CN_OPI_0228072077)

任务描述

激活热计费功能。

脚本

//配置使能离线计费开关和热计费属性。

```
SET CHARGECTRL:HOMEOFFLINE=ENABLE,VISITOFFLINE=ENABLE;
```

```
ADD CHARGECHAR: CCNAME="testchgcha", HOME="0x100", ROAM="0x100", VISIT="0x100", HOMESGSN=DISABLE, ROAMSGSN=DISABLE, VISITSGSN=DISABLE;
```

//配置计费参数。

```
ADD OFCTEMPLATE:OFCTEMPLATENAME="offlinecharg-test",GCDRVERSION=R7V740_EGCDR,TQM=QCT,QCTVALUE=30;
```

```
SET OFCTHRESHOLD:OFCTEMPLATENAME="offlinecharg-test",THRESHINHERIT=NON_INHERIT,CONDITIONCHANGE=3,VOLUMETHRESHOLD=1000,TIMETHRESHOLD=30,SERVINGNODECHNG=3;
```

//绑定到APN下。

```
ADD APN:APN="apn-test";
```

```
SET APNCHARGECTRL: APN="apn-test", ONLINESW=DISABLE, OFFLINESW=ENABLE, PGWOFCTEMPLATE="offlinecharg-test", SGWOFCTEMPLATE="offlinecharg-test", GGSNOFCTEMPLATE="offlinecharg-test", CCNAME="testchgcha";
```

#### [命令功能](#ZH-CN_MMLREF_0000001172345955)

**适用网元：SGSN、MME**

该命令用于查看系统内附着的用户数和激活的用户承载数等信息。

#### [注意事项](#ZH-CN_MMLREF_0000001172345955)

该命令执行后立即生效，如果输入进程号，必须同时输入RU名称。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172345955)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172345955)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172345955)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| DT | 查询方式 | 可选必选说明：可选参数<br>参数含义：该参数用于指定查询方式。<br>取值范围：<br>“PROCESS_NO(进程标识)”<br>默认值：无 |
| RUNAME | RU名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定<br>SPU<br>资源单元名。该参数可以通过<br>[DSP RU](../../../../../平台服务管理/单体服务公共功能管理/系统管理/资源管理/RU管理/显示资源单元信息（DSP RU）_59103857.md)<br>命令查询。<br>取值范围：0~63位字符串<br>默认值：无 |
| PN | 进程号 | 可选必选说明：可选参数<br>参数含义：该参数用于指定需要查看用户信息对应SPP进程的进程号。<br>取值范围：0~20<br>默认值：无 |

#### [输出结果说明](#ZH-CN_MMLREF_0000001172345955)

| 输出项名称 | 输出项解释 |
| --- | --- |
| RU名称 | 显示<br>SPU<br>资源单元名。<br>取值范围：1~63位字符串 |
| 进程号 | 显示业务进程所属的SPP进程序号。<br>取值范围：<br>- 参数取值范围为0-11。 |
| 静态用户数 | 系统中保存的静态MM上下文数量（包含签约数据等）。用户分离后，用户的签约数据不会马上删除。用户签约数据删除的时间由<br>[**SET SDBTMR**](../../移动性管理/用户数据资源核查/设置核查定时器配置（SET SDBTMR）_72225265.md)<br>中<br>“PURGTMR（用户分离后签约数据保留时间）”<br>参数控制，默认设置为24小时。<br>取值范围：0~4294967294 |
| APN签约上下文数 | 显示系统中附着的用户APN签约上下文总数。<br>取值范围：0~4294967294 |
| 2G在线用户数 | 显示系统中附着的2G用户总数。<br>取值范围：0~4294967294 |
| 3G在线用户数 | 显示系统中附着的3G用户总数。<br>取值范围：0~4294967294 |
| 4G在线用户数 | 显示系统中附着的4G用户总数，即处于ECM-CONNECTED和ECM-IDLE态的用户数。<br>取值范围：0~4294967294<br>说明：4G用户总数中不包含NB-IoT用户数。 |
| NB-IoT在线用户数 | 显示系统中附着的NB-IoT用户总数，即处于ECM-CONNECTED和ECM-IDLE态的用户数。<br>取值范围：0~4294967294 |
| 2G激活PDP个数 | 显示系统中2G用户激活的PDP总数。<br>取值范围：0~4294967294 |
| 3G激活PDP个数 | 显示系统中3G用户激活的PDP总数。<br>取值范围：0~4294967294 |
| 4G EPS承载数 | 显示系统中4G用户激活的EPS承载总数。<br>取值范围：0~4294967294<br>说明：4G用户激活的EPS承载总数中不包含NB-IoT用户激活的EPS承载数。 |
| NB-IoT承载数 | 显示系统中NB-IoT用户激活的EPS承载总数。<br>取值范围：0~4294967294 |
| NB-IoT服务用户数 | 显示系统中24小时内服务过的NB-IoT用户数。<br>取值范围：0~4294967294 |
| NR在线用户数 | 显示系统中附着的具备NR（New Radio）能力且签约NR业务的用户总数，即处于ECM-CONNECTED和ECM-IDLE态的用户数。<br>取值范围：0~4294967294 |
| NR承载数 | 显示系统中附着的具备NR（New Radio）能力且签约NR业务的用户激活的承载总数。<br>取值范围：0~4294967294 |
| HSS Bypass状态用户数 | 显示系统中处于HSS Bypass状态的用户数。<br>取值范围：0~4294967294 |
| POD ID | 该参数用于标识用户所在系统中的USN POD ID。非灰度升级期间，该参数不显示。 |
| POD版本号信息 | 该参数用于标识用户所在系统中的USN POD版本号。非灰度升级期间，该参数不显示。 |

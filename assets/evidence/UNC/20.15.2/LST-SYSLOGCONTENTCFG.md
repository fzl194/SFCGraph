## [命令功能](#ZH-CN_MMLREF_0000001951800805)

本命令用于查询系统上报给第三方Syslog服务器的日志内容配置。

日志内容配置可通过执行 [**SET SYSLOGCONTENTCFG**](设置日志内容配置（SET SYSLOGCONTENTCFG）_24521974.md) 命令进行操作。

## [注意事项](#ZH-CN_MMLREF_0000001951800805)

无。

## [参数说明](#ZH-CN_MMLREF_0000001951800805)

无。

## [使用实例](#ZH-CN_MMLREF_0000001951800805)

```
%%LST SYSLOGCONTENTCFG:;%%
RETCODE = 0  操作成功

操作结果如下
------------
报文头APP-NAME  =  日志类型
      报文头IP  =  -
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001951800805)

该命令执行正常，会返回命令执行成功的提示信息和相关参数，各参数含义请参见 [**SET SYSLOGCONTENTCFG**](设置日志内容配置（SET SYSLOGCONTENTCFG）_24521974.md) 命令的参数说明。

该命令执行异常，会返回对应的错误码。常见的错误码如 [表1](#ZH-CN_MMLREF_0000001951800805__table19755332194617) 所示。

## [命令功能](#ZH-CN_MMLREF_0000001823782754)

**适用NF：SMF**

该命令用于查询V-SMF和H-SMF之间是否进行Roaming Charging Profile的协商。

## [注意事项](#ZH-CN_MMLREF_0000001823782754)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001823782754)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001823782754)

无

## [使用实例](#ZH-CN_MMLREF_0000001823782754)

查询漫游参数协商开关：

```
%%LST HVNEGRCPSW:;%%
RETCODE = 0  操作成功

结果如下
------------------------
V-SMF漫游参数协商开关 =  不使能
H-SMF漫游参数协商开关 =  不使能
```

## [输出结果说明](#ZH-CN_MMLREF_0000001823782754)

| 输出项名称 | 输出项解释 |
| --- | --- |
| V-SMF漫游参数协商开关 | 该参数用于设置V-SMF是否和H-SMF进行Roaming Charging Profile协商。 |
| H-SMF漫游参数协商开关 | 该参数用于设置H-SMF是否和V-SMF进行Roaming Charging Profile协商。 |

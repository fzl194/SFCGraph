#### [命令功能](#ZH-CN_MMLREF_0000001172345883)

**适用网元：SGSN、MME**

该命令用于查询当前的Diameter本地实体信息。

#### [注意事项](#ZH-CN_MMLREF_0000001172345883)

该命令执行后立即生效。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172345883)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172345883)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172345883)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| LOINDEX | 本地实体索引 | 可选必选说明：可选参数<br>参数含义：该参数用于指定待查询的本地实体索引。<br>取值范围：0～31<br>默认值：无<br>说明：不设置该参数，表示查询所有记录。 |

#### [使用实例](#ZH-CN_MMLREF_0000001172345883)

查询本地实体状态：

DSP DMLE:;

```
%%DSP DMLE:;%%
RETCODE = 0  操作成功。

输出结果如下
--------------
本地实体索引  =  0
  本地实体名  =  DMLE
    实体状态  =  正常
(结果个数 = 1)
```

#### [输出结果说明](#ZH-CN_MMLREF_0000001172345883)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 本地实体索引 | 本地实体的索引，该参数用来在系统中唯一标识一个本地实体。 |
| 本地实体名 | 本地实体的名称，标识本地实体。 |
| 本地实体状态 | 查询的本地实体的状态。<br>取值说明：<br>- 正常：表示MME与HSS之间的通信正常。<br>- 故障：表示MME与HSS之间的通信中断，需要检查网络状况，分析故障原因。详细操作参见**ALM-80591 Diameter链路故障**告警处理。<br>- 拥塞：表示MME与HSS之间的通信在数据量达到链路负荷门限。需要调整组网或扩容。详细操作参见**ALM-80591 Diameter链路故障**告警处理。 |

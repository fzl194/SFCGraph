#### [命令功能](#ZH-CN_CONCEPT_0000001600440441)

该命令用于查询ARP防欺骗功能。

#### [注意事项](#ZH-CN_CONCEPT_0000001600440441)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600440441)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600440441)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001600440441)

查询ARP防欺骗配置信息：

```
LST ARPANTISPOOFING:;
```

```

RETCODE = 0  操作成功。

结果如下
--------
    ARP防欺骗使能  =  去使能
      ARP探测次数  =  5
ARP防欺骗告警使能  =  去使能
      ARP冲突阈值  =  30
    老化时间（s）  =  1200
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600440441)

| 输出项名称 | 输出项解释 |
| --- | --- |
| ARP防欺骗使能 | 该参数用于指定ARP防欺骗是否使能。 |
| ARP探测次数 | 该参数用于指定ARP冲突探测次数。 |
| ARP防欺骗告警使能 | 该参数用于指定ARP防欺骗告警是否使能。 |
| ARP冲突阈值 | 该参数用于指定ARP冲突阈值。 |
| 老化时间（s） | 该参数用于指定ARP动态黑名单老化时间。 |

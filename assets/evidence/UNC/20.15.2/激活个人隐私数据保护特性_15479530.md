# 激活 个人隐私数据保护 特性

- [操作场景](#ZH-CN_OPI_0215479530__1.3.1)
- [必备事项](#ZH-CN_OPI_0215479530__1.3.2)
- [操作步骤](#ZH-CN_OPI_0215479530__1.3.3)

## [操作场景](#ZH-CN_OPI_0215479530)

为保证个人隐私数据的安全性，建议运营商遵从所在国家的相关法律开启假名化/匿名化功能，对跟踪、日志、CHR中使用的用户个人标识进行保护。

> **说明**
> 适用于 SGSN、 MME、 GGSN、 SGW_C、PGW_C、AMF、SMF 、NRF 、NSSF 。

## [必备事项](#ZH-CN_OPI_0215479530)

前提条件

请仔细阅读 [WSFD-010305 个人隐私数据保护特性概述](特性概述_65743439.md) 。

数据

该操作无需准备数据。

## [操作步骤](#ZH-CN_OPI_0215479530)

- 配置跟踪匿名化功能。
  | Option | Description |
  | --- | --- |
  | **通过网管** | 详细操作步骤请参见<br>U2020/MAE<br>产品文档的“假名化策略管理”。<br>须知：<br>- U2020/MAE上假名化/匿名化开关默认开启，建议运营商遵从所在国家的相关法律。<br>- 为更好地保护用户个人隐私，请定期更新并妥善保存HMAC密钥，避免被他人获取。密钥的长度建议符合安全要求，提升安全性。 |
  | **通过OM Portal** | - 当UNC未连接U2020/MAE之前，根据OM Portal跟踪“下载”界面的“匿名化”选项状态判断是否做匿名化处理。<br>- 当UNC连接U2020/MAE之后，同步保存U2020/MAE上配置的假名化/匿名化功能状态及HMAC密钥信息：- 如果U2020/MAE上配置匿名化功能为开启，则强制做匿名化处理；<br>- 如果U2020/MAE上配置匿名化功能为关闭，则根据OM Portal跟踪“下载”界面的“匿名化”选项状态判断是否做匿名化处理。 |
- 配置日志匿名化功能。
  通过MML命令 [**SET COMMONSOFTPARAOFBIT**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/软件参数管理/设置公共软件参数比特位（SET COMMONSOFTPARAOFBIT）_26494601.md) 设置软件参数 [DWORD1 BIT1](../../../../../OM参考/软件参数/UNC软件参数/业务软件参数/软件参数（5G业务公共）/公共软参/DWORD1 BIT1 控制敏感信息匿名化_10872586.md) 为0。
  通过MML命令 [**SET FWSOFTPARABITS**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/软参配置管理/设置软件参数比特位（SET FWSOFTPARABITS）_37580503.md) 设置软件参数 [DWORD1601 BIT1](../../../../../OM参考/软件参数/UNC软件参数/平台软件参数/HTTP软参/DWORD1601 BIT1 控制HTTP敏感信息匿名化_90970504.md) 为0。
  ```
  SET COMMONSOFTPARAOFBIT: DT=DwCom, DWORDCOMNUM=1, VALUE=0, POSITION=1;
  ```
  ```
  SET FWSOFTPARABITS: PARATYPE=DWORD, DWORDNUM=1601, VALUE=VALUE_0, POSITION=1;
  ```
- 配置CHR假名化功能。
  | Option | Description |
  | --- | --- |
  | **通过网管** | 详细操作步骤请参见<br>U2020/MAE<br>产品文档的“假名化策略管理”。<br>须知：<br>- U2020/MAE上假名化/匿名化开关默认开启，建议运营商遵从所在国家的相关法律。<br>- 为更好地保护用户个人隐私，请定期更新并妥善保存HMAC密钥，避免被他人获取。密钥的长度建议符合安全要求，提升安全性。 |
  | **通过MML** | 执行MML命令[**SET PSEUDONYPOLICY**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/操作维护/软件参数管理/假名化策略/设置CHR假名化本地策略（SET PSEUDONYPOLICY）_51175645.md)开启本地CHR假名化功能，并配置假名化口令。假名化口令为密码类型，长度必须为偶数（最小16位，最大30位），且只能输入0-9、A-F、a-f。<br>`SET PSEUDONYPOLICY: LOCALPOLICY=ON, KEY="*****";`<br>- 当UNC未连接U2020/MAE，或已连接U2020/MAE但U2020/MAE上未进行假名化/匿名化功能设置时，根据UNC上MML命令配置判断是否做假名化处理。<br>- 当UNC已连接U2020/MAE，且U2020/MAE上已有假名化/匿名化功能配置时，根据U2020/MAE上配置的假名化/匿名化功能状态判断是否做假名化处理。 |

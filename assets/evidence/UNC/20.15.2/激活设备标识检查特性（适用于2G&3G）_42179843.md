## [操作步骤](#ZH-CN_OPI_0242179843)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md) :
3. 设置GERAN、UTRAN、E-UTRAN网络获取和检查IMEI的策略。
  [**MOD GBIMEICFG**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/设备检查管理/Gb模式IMEI配置/修改Gb模式IMEI配置(MOD GBIMEICFG)_26305444.md)
  [**MOD IUIMEICFG**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/设备检查管理/Iu模式IMEI配置/修改Iu模式IMEI配置(MOD IUIMEICFG)_72345233.md)
4. **可选：**增加指定IMSI号段用户获取和检查IMEI的策略。指定IMSI号段的用户采用对应的IMEI策略，未配置的用户采用默认的IMEI策略。
  IMSI号段有两种形式：一种为IMSI前缀形式，另一种为起始IMSI和终止IMSI形式。这两种形式同时只允许配置一种。
    a. 配置IMSI前缀形式的号段用户在GERAN网络获取和检查IMEI的策略。
      [**ADD GBIMEICFG**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/设备检查管理/Gb模式IMEI配置/增加Gb模式IMEI配置(ADD GBIMEICFG)_26145632.md)
    b. 配置起始IMSI和终止IMSI形式的号段用户在GERAN网络获取和检查IMEI的策略。
      [**ADD GBIMEICFG**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/设备检查管理/Gb模式IMEI配置/增加Gb模式IMEI配置(ADD GBIMEICFG)_26145632.md)
  > **说明**
  > 激活IMEI检查特性，参数 “GETIMEIPLC” （是否获取用户的IMEI）选择 “IMEI” （获取IMEI）或 “IMEISV” （获取IMEISV）；参数 “CHKIMEIPLC” （是否检查用户的IMEI）选择 “IMEI” （检查IMEI）或 “IMEISV” （检查IMEISV）。

## [任务示例](#ZH-CN_OPI_0242179843)

任务描述

在UTRAN网络中启用IMEI检查功能，配置IMEI获取策略为优先获取IMEISV，实现对所有用户使用Gf接口进行IMEI检查。IMEI获取失败允许接入，IMEI检查超时允许接入，允许灰名单用户接入，不允许黑名单用户接入，允许EIR返回用户原因失败时接入，允许EIR返回设备原因失败时接入，EIR响应超时时长为5。对IMSI前缀为12303的用户在UTRAN网络中优先获取IMEISV，但不检查。

脚本

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2EIR02", SWITCH=ENABLE;
```

//设置UTRAN网络获取和检查IMEI的策略。

```
MOD IUIMEICFG: SUBRANGE=ALL_USER, GETIMEIPLC=IMEISV, CHKIMEIPLC=IMEI, ITTYPE=Gf, IMEIACCESS=YES, CTFLAG=YES, GRALLOW=YES, BLALLOW=NO, USERERR=YES, PROVIDERR=YES, EIROUTTIMER=5;
```

//设置UTRAN网络IMSI前缀为12303的用户获取和检查IMEI的策略。

```
ADD IUIMEICFG: SUBRANGE=IMSI_PREFIX, IMSIPRE="12303", GETIMEIPLC=IMEISV, CHKIMEIPLC=NO;
```

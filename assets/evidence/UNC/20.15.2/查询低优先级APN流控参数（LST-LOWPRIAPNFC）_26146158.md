#### [命令功能](#ZH-CN_MMLREF_0000001126146158)

**适用网元：MME**

该命令用于查询低优先级APN流控功能的相关参数。

#### [注意事项](#ZH-CN_MMLREF_0000001126146158)

该命令执行后立即生效。

#### [本地用户权限](#ZH-CN_MMLREF_0000001126146158)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126146158)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126146158)

无。

#### [使用实例](#ZH-CN_MMLREF_0000001126146158)

查询低优先级APN流控参数：

LST LOWPRIAPNFC:;

```
%%LST LOWPRIAPNFC:;%%
RETCODE = 0  操作成功。

输出结果如下
------------
   低优先级APN流控功能开关  =  开启
                 APNNI组号  =  1
   流控速率最小值（个/秒）  =  10
   流控速率最大值（个/秒）  =  300
Back off timer最小值（秒）  =  900
Back off timer最大值（秒）  =  1800
                  测试IMSI  =  12300000
(结果个数 = 1)
---    END
```

## [操作场景](#ZH-CN_OPI_0295923455)

用户进行在线/离线计费的方式可以通过其他对接网元下发，也可以在GGSN/PGW-C本地配置，本文描述在GGSN/PGW-C上如何配置在线/离线计费的开关。

- GGSN/PGW-C的离线计费方式选择有PCRF下发、本地配置，本地配置有三种粒度：UserProfile、APN和计费属性CC，这些方式的优先级依次降低。如[图1](#ZH-CN_OPI_0295923455__fig517555792110)所示为GGSN/PGW-C的离线计费方式选择图。
  **图1** 离线计费方式选择图

  <br>

  ![](配置离线_在线计费方式（GGSN_PGW-C）_95923455.assets/zh-cn_image_0292153500_2.png)
- GGSN/PGW-C的在线计费方式选择有PCRF下发、RADIUS Server下发、本地配置，本地配置有三种粒度：UserProfile、APN和计费属性CC，如[图2](#ZH-CN_OPI_0295923455__fig188082038205017)所示为GGSN/PGW-C的在线计费方式选择图。
  **图2** 在线计费方式选择图

  <br>

  ![](配置离线_在线计费方式（GGSN_PGW-C）_95923455.assets/zh-cn_image_0292153501_2.png)
    - 如果网络中部署了PCRF控制在线计费方式，则PCRF在CCA-I消息中携带Online AVP指示IP-CAN会话是否使能在线计费。
    - 如果网络中未部署PCRF，且RADIUS Server支持下发OCS-ID，则RADIUS Server在鉴权响应消息Access-Accept消息中携带OCS-ID，且该OCS在GGSN/PGW-C本地已配置，则使能在线计费。
    - 如果网络中未部署PCRF，且RADIUS Server不支持下发OCS-ID，则使用本地配置的在线计费方式。

其中，根据用户归属属性配置的开关 **SET CHARGECTRL** 是控制在线/离线计费的总开关，是进行相应方式计费的前提和基础，它控制GGSN/PGW-C本地配置的计费开关和RADIUS server下发的计费方式是否有效，PCRF下发计费方式不受此控制。即当总开关使能了在线/离线计费，其他开关的配置才能生效。默认情况下，总开关同时使能在线/离线计费。

基于此，在线/离线计费选择开关的优先级从高到低依次是：PCRF下发的参数–〉GGSN/PGW-C本地总开关–〉RADIUS下发的参数–〉UserProfile下的配置–〉APN下的配置-〉用户计费属性下的配置。对于GGSN/PGW-C本地配置的各级开关来说，如果高级别的参数没有配置，则依次向下继承低一级别的参数取值。

> **说明**
> 适用于PGW-C 、GGSN 。

## [操作步骤](#ZH-CN_OPI_0295923455)

1. 基于用户归属属性（本地用户、漫游用户、拜访用户）使能GGSN/PGW-C的离线/在线计费方式。
  **SET CHARGECTRL**
  > **说明**
  > 针对同一个用户，不同属性下规划的计费方式要一致，否则用户计费会出错。例如，针对CC取值为NORMAL的本地用户，如果设置NORMAL用户进行离线计费，又设置本地用户进行在线计费，则最终用户计费将出错。
2. 基于UserProfile配置计费方式。
  **SET USRPROFCHARGE**
3. 基于APN使能GGSN/PGW-C的离线/在线计费方式。
  **SET APNCHARGECTRL**
4. 基于用户计费属性使能GGSN/PGW-C的离线/在线计费方式。
  **ADD CHARGEMETHOD**
5. 配置免费业务的计费方式。
    a. 配置免费业务的计费模式。
      **ADD URR**
      > **说明**
      > 离线计费：参数 “USAGERPTMODE” 设置为 “OFFIINE” ， “OFFMETERINGTYPE” 设置为 “FREE” 。
      >
      > 在线计费：参数 “USAGERPTMODE” 设置为 “ONLINE” ， “ONLMETERINGTYPE” 设置为 **“FREE”** 。
    b. 配置UserProfile免费业务的计费方式。
      **ADD USERPROFILE**
      > **说明**
      > 对于UserProfile中计费属性的 “FREESERONLCHG” 或 “FREESEROFFCHG” 配置为 “ENABLE” 的业务即免费业务，如果要想在CG离线话单或在线计费配额上报消息中观察到这些业务的计费信息，则必须在其所在的UserProfile下用 **ADD USERPROFILE** 命令使能offline或online计费，系统缺省已经使能online和offline计费。

## [任务示例](#ZH-CN_OPI_0295923455)

任务描述

1. 使能GGSN/PGW-C的离线计费。
    - 任务一：配置基于UserProfile的离线计费方式。
    - 任务二：配置基于APN使能GGSN/PGW-C的离线计费方式。
    - 任务三：配置基于用户计费属性使能GGSN/PGW-C的离线计费方式。
2. 使能GGSN/PGW-C的在线计费。
    - 任务一：配置基于UserProfile的在线计费方式。
    - 任务二：配置基于APN的在线计费方式。
    - 任务三：配置基于用户计费属性的在线计费方式。
3. 使能GGSN/PGW-C的离线和在线计费。
    - 任务一：配置基于UserProfile的离线和在线计费方式。
    - 任务二：配置基于APN的离线和在线计费方式。
    - 任务三：配置基于用户计费属性的离线和在线计费方式。

脚本

1. 使能GGSN/PGW-C的离线计费。
  //任务一：配置基于UserProfile的离线计费方式。

  ```
  SET CHARGECTRL: HOMEOFFLINE=ENABLE,VISITOFFLINE=ENABLE,ROAMOFFLINE=ENABLE;
  ```

  ```
  ADD USERPROFILE: USERPROFILENAME="up-test", UPTYPE=PCC;
  ```

  ```
  SET USRPROFCHARGE: USRPROFNAME="up-test", OFFLINESW=ENABLE;
  ```
  //任务二：配置基于APN使能GGSN/PGW-C的离线计费方式。

  ```
  SET CHARGECTRL: HOMEOFFLINE=ENABLE,VISITOFFLINE=ENABLE,ROAMOFFLINE=ENABLE;
  ```

  ```
  ADD APN: APN="apn-test";
  ```

  ```
  SET APNCHARGECTRL: APN="apn-test",OFFLINESW=ENABLE;
  ```
  //任务三：配置基于用户计费属性使能GGSN/PGW-C的离线计费方式。

  ```
  SET CHARGECTRL: HOMEOFFLINE=ENABLE,VISITOFFLINE=ENABLE,ROAMOFFLINE=ENABLE;
  ```

  ```
  ADD CHARGEMETHOD: CHARGECHARACT=SPECIFIC_VALUE, CCVALUE="0x0400", MASK="0x0400", PRIORITY=2, OFFLINE=ENABLE;
  ```
2. 使能GGSN/PGW-C的在线计费。
  //任务一：配置基于UserProfile的在线计费方式。

  ```
  SET CHARGECTRL: HOMEONLINE=ENABLE,VISITONLINE=ENABLE,ROAMONLINE=ENABLE;
  ```

  ```
  ADD USERPROFILE: USERPROFILENAME="up-test", UPTYPE=PCC;
  ```

  ```
  SET USRPROFCHARGE: USRPROFNAME="up-test", ONLINESW=ENABLE;
  ```
  //任务二：配置基于APN的在线计费方式。

  ```
  SET CHARGECTRL: HOMEONLINE=ENABLE,VISITONLINE=ENABLE,ROAMONLINE=ENABLE;
  ```

  ```
  ADD APN: APN="apn-test";
  ```

  ```
  SET APNCHARGECTRL: APN="apn-test", ONLINESW=ENABLE;
  ```
  //任务三：配置基于用户计费属性的在线计费方式。

  ```
  SET CHARGECTRL: HOMEONLINE=ENABLE,VISITONLINE=ENABLE,ROAMONLINE=ENABLE;
  ```

  ```
  ADD CHARGEMETHOD: CHARGECHARACT=SPECIFIC_VALUE, CCVALUE="0x0400", MASK="0x0400", PRIORITY=2, ONLINE=ENABLE;
  ```
3. 使能GGSN/PGW-C的离线和在线计费。
  //任务一：配置基于UserProfile的离线和在线计费方式。

  ```
  SET CHARGECTRL: HOMEONLINE=ENABLE, HOMEOFFLINE=ENABLE, VISITONLINE=ENABLE, VISITOFFLINE=ENABLE, ROAMONLINE=ENABLE, ROAMOFFLINE=ENABLE;
  ```

  ```
  ADD USERPROFILE: USERPROFILENAME="up-test", UPTYPE=PCC;
  ```

  ```
  SET USRPROFCHARGE: USRPROFNAME="up-test", ONLINESW=ENABLE, OFFLINESW=ENABLE;
  ```
  //任务二：配置基于APN使能GGSN/PGW-C的离线和在线计费方式。

  ```
  SET CHARGECTRL: HOMEONLINE=ENABLE, HOMEOFFLINE=ENABLE, VISITONLINE=ENABLE, VISITOFFLINE=ENABLE, ROAMONLINE=ENABLE, ROAMOFFLINE=ENABLE;
  ```

  ```
  ADD APN: APN="apn-test";
  ```

  ```
  SET APNCHARGECTRL: APN="apn-test",ONLINESW=ENABLE, OFFLINESW=ENABLE;
  ```
  //任务三：配置基于用户计费属性使能GGSN/PGW-C的离线和在线计费方式。

  ```
  SET CHARGECTRL: HOMEONLINE=ENABLE, HOMEOFFLINE=ENABLE, VISITONLINE=ENABLE, VISITOFFLINE=ENABLE, ROAMONLINE=ENABLE, ROAMOFFLINE=ENABLE;
  ```

  ```
  ADD CHARGEMETHOD: CHARGECHARACT=SPECIFIC_VALUE, CCVALUE="0x0400", MASK="0x0400", PRIORITY=2, ONLINE=ENABLE, OFFLINE=ENABLE;
  ```

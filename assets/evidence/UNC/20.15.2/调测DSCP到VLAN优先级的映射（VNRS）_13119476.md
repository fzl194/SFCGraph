# 调测DSCP到VLAN优先级的映射 （VNRS）

- [操作场景](#ZH-CN_OPI_0213119476__1.3.1)
- [必备事项](#ZH-CN_OPI_0213119476__1.3.2)
- [操作步骤](#ZH-CN_OPI_0213119476__1.3.3)

## [操作场景](#ZH-CN_OPI_0213119476)

- 用于DSCP到VLAN优先级的映射是否正常运行。
- 用于修改DSCP到VLAN优先级的映射。

## [必备事项](#ZH-CN_OPI_0213119476)

前提条件

- 操作人员已经登录网络管理系统NMS（Network Management System）。
- 硬件安装已经完成。

数据

无

## [操作步骤](#ZH-CN_OPI_0213119476)

1. 查询DSCP到VLAN优先级的映射。
  在 “MML命令行-UNC” 窗口上执行：
  [**LST DSCPMAP**](../../../../../../../OM参考/命令/UNC MML命令/平台服务管理/VNRS功能管理/IP服务/安全管理/QoS管理/DSCP映射VLAN优先级/查询DSCP值到VLAN优先级的映射（LST DSCPMAP）_50121842.md) :;
    - **预期结果：**查询结果与配置结果一致。
      ```
      LST DSCPMAP:
      RETCODE = 0  操作成功

      结果如下
      -------------------------
      DSCP值   优先级类型   优先级数值 
       62      exp          5 
       63      8021p        7 
      (结果个数 = 2)
      ---    END
      ```
      **异常结果：** 查询不到配置信息，请重新配置。
2. 修改DSCP到VLAN优先级的映射。
  在 “MML命令行-UNC” 窗口上执行：
  [**MOD DSCPMAP**](../../../../../../../OM参考/命令/UNC MML命令/平台服务管理/VNRS功能管理/IP服务/安全管理/QoS管理/DSCP映射VLAN优先级/修改DSCP值到VLAN优先级的映射（MOD DSCPMAP）_00841565.md) : DSCP=62, TYPE=8021p, VALUE=5;
    - **预期结果：**查询结果与修改结果一致。
      ```
      LST DSCPMAP:
      RETCODE = 0  操作成功

      结果如下
      -------------------------
          DSCP值 = 62
      优先级类型 = exp
        先级数值 = 5
      (结果个数 = 1)
      ---    END
      ```
      **异常结果：** 查询不到配置信息，请重新配置。

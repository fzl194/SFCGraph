## [命令功能](#ZH-CN_MMLREF_0000001357496925)

**适用NF：AMF**

该命令用于查询AMF的AUSF故障处理策略。

## [注意事项](#ZH-CN_MMLREF_0000001357496925)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001357496925)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001357496925)

无

## [使用实例](#ZH-CN_MMLREF_0000001357496925)

查询AMF的AUSF故障处理策略，执行如下命令：

```
%%LST AMFAUSFRESET:;%%
RETCODE = 0  操作成功

结果如下
--------
recoveryTime变化时重选AUSF开关  =  打开
                      扫描模式  =  默认扫描模式
               扫描速率(个/秒)  =  2
              单位扫描时间(秒)  =  3
                  单位扫描个数  =  2
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001357496925)

| 输出项名称 | 输出项解释 |
| --- | --- |
| recoveryTime变化时重选AUSF开关 | 该参数用于指定AUSF的recoveryTime变化时是否重选AUSF。AUSF的recoveryTime改变时，会发NFUpdate消息给NRF，携带改变后的recoveryTime信元，然后NRF发NFStatusNotify消息给AMF。 |
| 扫描模式 | 该参数表示扫描模式。 |
| 扫描速率(个/秒) | 该参数用于指定每个DS每秒扫描多少个用户，对符合重选条件用户在后续需要鉴权的流程中重选AUSF并进行鉴权。 |
| 单位扫描时间(秒) | 该参数用于表示AUSF故障后AMF对用户扫描的时间间隔。 |
| 单位扫描个数 | 该参数用于表示每个DS每隔INTERVAL设置的时间扫描多少个用户，对符合重选条件用户在后续需要鉴权的流程中重选AUSF并进行鉴权。 |

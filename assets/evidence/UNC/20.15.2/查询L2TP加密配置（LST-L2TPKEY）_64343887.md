## [命令功能](#ZH-CN_MMLREF_0264343887)

**适用NF：PGW-C、SMF、GGSN**

该命令用于查询L2TP加密配置。

## [注意事项](#ZH-CN_MMLREF_0264343887)

无

#### [操作用户权限](#ZH-CN_MMLREF_0264343887)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0264343887)

无

## [使用实例](#ZH-CN_MMLREF_0264343887)

查询L2TP加密配置

```
%%LST L2TPKEY:;%%
RETCODE = 0  操作成功

结果如下
--------
  加密开关  =  不使能
  密钥口令  =  *****
  密钥变更  =  不变更
时长(分钟)  =  1
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0264343887)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 加密开关 | 该参数用于控制是否启用加密功能。 |
| 密钥口令 | 该参数用于指定生成加解密密钥所需的口令。 |
| 密钥变更 | 该参数用于控制是否定期变更密钥。 |
| 时长(分钟) | 该参数用于指定变更密钥的触发条件，当当前所使用的密钥满足配置的变更条件时，系统进行密钥变更。 |

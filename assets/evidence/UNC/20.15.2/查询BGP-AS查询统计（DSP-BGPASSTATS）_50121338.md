#### [命令功能](#ZH-CN_CONCEPT_0000001550121338)

该命令用来用于显示收到的以前缀为key查询AS-Query消息和查询响应消息的统计计数。

#### [注意事项](#ZH-CN_CONCEPT_0000001550121338)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550121338)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550121338)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001550121338)

查询BGP AS查询统计：

```
DSP BGPASSTATS:;
```

```

RETCODE = 0  操作成功。

结果如下
-------------------------
收取消息计数   =  5678
发送消息计数   =  5678
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001550121338)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 收取消息计数 | 用于表示BGP收到的查询消息个数。 |
| 发送消息计数 | 用于表示BGP发送的查询消息响应个数。 |

#### [命令功能](#ZH-CN_MMLREF_0000001126146286)

**适用网元：SGSN、MME**

该命令用于查询Diameter对端实体配置。

#### [注意事项](#ZH-CN_MMLREF_0000001126146286)

该命令执行后立即生效。

#### [本地用户权限](#ZH-CN_MMLREF_0000001126146286)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126146286)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126146286)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PEERIDX | 对端实体索引 | 可选必选说明：可选参数<br>参数含义：该参数用于指定Diameter对端实体索引。<br>取值范围：0～639<br>默认值：无<br>说明：如果不输入，表示查询所有配置记录。 |

#### [使用实例](#ZH-CN_MMLREF_0000001126146286)

查询所有Diameter对等端实体配置：

LST DMPE:;

```
%%LST DMPE:;%%
RETCODE = 0  操作成功。

操作结果如下
------------
对端实体索引    对端主机名                   对端实体名    节点支持的应用    对端拥塞避免开关

0               exapmle.com                   NULL          S6a/S6d           关闭            
1               example01.com                 NULL          S6a/S6d           关闭            
(结果个数 = 2)
---    END
```

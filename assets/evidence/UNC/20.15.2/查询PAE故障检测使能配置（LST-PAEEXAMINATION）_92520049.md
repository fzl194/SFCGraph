#### [命令功能](#ZH-CN_CONCEPT_0192520049)

该命令用于查询PAE故障检测功能使能状态。

#### [注意事项](#ZH-CN_CONCEPT_0192520049)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0192520049)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0192520049)

无

#### [使用实例](#ZH-CN_CONCEPT_0192520049)

查询所有微服务的PAE故障检测功能使能状态：

```
LST PAEEXAMINATION:;
```

```
RETCODE = 0  操作成功。

结果如下
---------
使能标志  =  是      
(结果个数 = 1)
---    END
```

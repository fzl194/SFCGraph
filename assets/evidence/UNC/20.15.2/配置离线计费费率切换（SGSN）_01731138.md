## [操作场景](#ZH-CN_OPI_0301731138)

本操作指导介绍在运行网络中激活支持节假日、星期、费率时段设置特性的操作过程。

SGSN支持节假日、星期、费率时段及对应费率的灵活设置，可以记录用户在不同的费率时段的流量，增强话单的计费信息收集功能。

- 计费节假日表用于指定国家的法定节假日以及对应的计费属性。
- 计费星期表用于指定一周中某天的费率类型为工作日费率还是周休日费率以及对应的计费属性。
- 费率时段表与计费节假日表和计费星期表一起使用，用于指定节假日或周休日使用优惠费率的时间段，可以指定多个时间段，但是具有相同计费属性的时间段不允许重叠。

> **说明**
> 适用于 SGSN 。

## [操作步骤](#ZH-CN_OPI_0301731138)

1. 进入 “MML命令行-UNC” 窗口。
2. **可选：**增加节假日配置。
  **ADD CHGHOLI**
3. **可选：**设置星期配置。
  **SET CHGWKDY**
4. **可选：**增加费率时段配置。
  **ADD CHGTARI**

## [任务示例](#ZH-CN_OPI_0301731138)

任务描述

设置2010年10月1日为普通计费属性用户的节假日，实时计费属性用户周五的费率属性是休息日，18：00~21：30为预付费计费属性用户工作日的费率时段。

脚本

//增加一条普通计费属性的节假日配置：2010年10月1日。

```
ADD CHGHOLI: YEAR=2010, MONTH=10, DAY=1, CC=NORMAL;
```

//设置一条热计费属性用户的星期配置：将热计费属性用户的星期五的费率类型改为周休日。

```
SET CHGWKDY: WKDAY=FRI, CC=HOTBILLING, TT=WEEKEND;
```

//增加一条预付费计费属性的费率时段记录：费率类型为工作日，起始时间为18点，结束时间为21点30分。

```
ADD CHGTARI: TT=WORK, ST=18&00&00, ET=21&30&00, CC=PREPAID;
```

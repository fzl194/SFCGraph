## [命令功能](#ZH-CN_MMLREF_0000001607669721)

该命令用于增加预共享密钥信息。

## [注意事项](#ZH-CN_MMLREF_0000001607669721)

- 该命令执行后立即生效。

- 该命令参数“预共享密钥值”的内容是密钥信息，执行“[**EXP MML**](../../../../操作维护/配置管理/配置导出管理/导出MML文件（EXP MML）_47200033.md):;”命令导出配置时，“预共享密钥值”参数的值是匿名化后的内容，因而该命令不支持MML导出后直接导入，必须手动下发。

- 最多可输入128条记录。

#### [操作用户权限](#ZH-CN_MMLREF_0000001607669721)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001607669721)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PSKIDX | 索引 | 可选必选说明：必选参数<br>参数含义：该参数用于指定TLS预共享密钥的记录索引。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是1~128。<br>默认值：无<br>配置原则：无 |
| PSKGRPIDX | 预共享密钥组索引 | 可选必选说明：必选参数<br>参数含义：该参数用于指定TLS预共享密钥组的索引。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是1~128。<br>默认值：无<br>配置原则：<br>与<br>[**ADD TLSPSKGRP**](../HTTP TLS预共享密钥组管理/增加TLS预共享密钥组（ADD TLSPSKGRP）_07789673.md)<br>中配置的预共享密钥组索引保持一致。 |
| MODE | 模式 | 可选必选说明：必选参数<br>参数含义：该参数用于指定PSK信息在TLS的客户端或服务端使用。<br>数据来源：全网规划<br>取值范围：<br>- “Client（客户端模式）”：Client指的是客户端模式<br>- “Server（服务端模式）”：Server指的是服务端模式<br>默认值：无<br>配置原则：无 |
| PSKID | 预共享密钥标识 | 可选必选说明：必选参数<br>参数含义：该参数用于指定预共享密钥的密钥ID。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是1~128。<br>默认值：无<br>配置原则：<br>该参数与“HTTP模式”参数组合，唯一标识一个预共享密钥记录，不能重复配置，而且大小写敏感。 |
| PSKTYPE | 预共享密钥内容类型 | 可选必选说明：必选参数<br>参数含义：该参数用于控制预共享密钥值类型。<br>数据来源：全网规划<br>取值范围：<br>- “HEXADECIMAL（16进制ASCII码）”：TLS预共享密钥内容类型为16进制ASCII码。<br>默认值：无<br>配置原则：无 |
| PSKVALUE | 预共享密钥值 | 可选必选说明：必选参数<br>参数含义：该参数用于标识一条预共享密钥的密钥值，通过密钥ID可以找到对应的预共享密钥。<br>数据来源：全网规划<br>取值范围：Pwd，取值范围是16~128。<br>默认值：无<br>配置原则：<br>- 预共享密钥区分大小写。<br>- 如果“预共享密钥内容类型”为“HEXADECIMAL(十六进制ASCII码)”，需要满足下列条件：长度最短16个字符，最长128个字符。长度必须偶数。必须为十六进制字符，包括数字0~9、英文大写字母、英文小写字母。 |
| DESC | 描述 | 可选必选说明：可选参数<br>参数含义：该参数用于对TLS预共享密钥进行描述。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~255。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001607669721)

若需要添加一个预共享密钥，索引为1，预共享密钥组索引为1，模式为“Client”，预共享密钥标识为“psktest”，预共享密钥内容类型为“HEXADECIMAL”，预共享密钥为“0123456789ABCDEF”，可执行如下命令：

```
ADD TLSPSK: PSKIDX=1, PSKGRPIDX=1, MODE=Client, PSKID="psktest", PSKTYPE=HEXADECIMAL, PSKVALUE="0123456789ABCDEF";
```

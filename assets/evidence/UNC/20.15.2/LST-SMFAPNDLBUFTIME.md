# 查询APN的下行报文缓存时长配置（LST SMFAPNDLBUFTIME）

- [注意事项](#ZH-CN_MMLREF_0296805491__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0296805491__1.3.4)

## [命令功能](#ZH-CN_MMLREF_0296805491)

**适用NF：SGW-C**

该命令用来查询APN的下行报文缓存时长配置。

## [注意事项](#ZH-CN_MMLREF_0296805491)

无

#### [操作用户权限](#ZH-CN_MMLREF_0296805491)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0296805491)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| APN | APN名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定APN实例名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~63。只能由“-”、数字、大小写字母和“.”组成，不能以“.”开头且不能出现连续两个“.”。不支持空格及“_”、“#”、“$”、“&”、“%”、“^”、“（”、“）”、“，”、“/”、“;”、“:”、“””、“`”等特殊字符，不区分大小写。<br>默认值：无<br>配置原则：<br>该参数使用ADD APN命令配置生成。 |

## [使用实例](#ZH-CN_MMLREF_0296805491)

查询huawei.com的下行报文缓存时长配置：

```
%%LST SMFAPNDLBUFTIME: APN="huawei.com";%%
RETCODE = 0  操作成功

结果如下
--------
                       APN名称  =  huawei.com
  普通用户下行报文缓存时长(秒)  =  9
NB-IoT用户下行报文缓存时长(秒)  =  6
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0296805491)

| 输出项名称 | 输出项解释 |
| --- | --- |
| APN名称 | 该参数用于指定APN实例名。 |
| 普通用户下行报文缓存时长(秒) | 该参数用于指定APN下的普通用户下行报文缓存时长。 |
| NB-IoT用户下行报文缓存时长(秒) | 该参数用于指定APN下的NB-IoT用户下行报文缓存时长。 |

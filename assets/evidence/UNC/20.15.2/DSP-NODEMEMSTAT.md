## [命令功能](#ZH-CN_MMLREF_0209587380)

该命令用于查询所有节点或指定节点内存信息。

## [注意事项](#ZH-CN_MMLREF_0209587380)

- 该命令执行后立即生效。
- 该命令仅在虚机场景下支持。

#### [操作用户权限](#ZH-CN_MMLREF_0209587380)

G_1，管理员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209587380)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| NODETYPE | 节点类型 | 可选必选说明：可选参数<br>参数含义：本参数用于指定节点类型。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~100。<br>默认值：无<br>配置原则：<br>虚拟机类型，例如OMU。 |
| NODEID | 节点ID | 可选必选说明：可选参数<br>参数含义：本参数用于指定节点ID。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~100。<br>默认值：无<br>配置原则：无 |
| MEID | 网元 ID | 可选必选说明：可选参数<br>参数含义：该参数用于指定网元来查询node的内存信息。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~100。<br>默认值：无<br>配置原则：<br>本命令不加参数可以查询所有支持的网元ID。 |

## [使用实例](#ZH-CN_MMLREF_0209587380)

- 查询所有节点内存信息。
  ```
  %%DSP NODEMEMSTAT:;%%
  RETCODE = 0  操作成功

  结果如下
  ------------------------
  节点类型    节点ID        网元ID  内存总量(MB)      内存使用量(MB)  内存使用率(%)

  SGU_A      192.168.55.114   0   31995             8498          26.56
  IPU        192.168.15.39    0   23931            11558          48.30
  SGU_A      192.168.35.107   0   31995             9653          30.17
  UPU        192.168.55.84    0   31995             5143          16.07
  (结果个数 = 4)
  ```
- 查询节点类型为PBU_A的所有节点内存信息。
  ```
  %%DSP NODECPUSTAT: NODETYPE="PBU_A";%%
  RETCODE = 0  操作成功

  结果如下
  ------------------------
  节点类型    节点ID       网元ID 内存总量(MB)      内存使用量(MB)  内存使用率(%)

  PBU_A      192.168.15.11  0    31995             10361         32.39
  PBU_A      192.168.55.34  0    31995             10334         32.30
  (结果个数 = 2)
  ```
- 查询节点id为192.168.15.11的节点内存信息。
  ```
  %%DSP NODEMEMSTAT: NODEID="192.168.15.11";%%
  RETCODE = 0  操作成功

  结果如下
  ------------------------
       节点类型   =  PBU_A
         节点ID   =  192.168.15.11
         网元ID   =  0
    内存总量(MB)  =  31995
  内存使用量(MB)  =  13606
  内存使用率(%)   =  42.53
  (结果个数 = 1)
  ```
- 查询节点类型为PBU_A，且节点id为192.168.15.11的节点内存信息。
  ```
  %%DSP NODEMEMSTAT: NODETYPE="PBU_A", NODEID="192.168.15.11";%%
  RETCODE = 0  操作成功

  结果如下
  ------------------------
       节点类型   =  PBU_A
         节点ID   =  192.168.15.11
         网元ID  =  0
    内存总量(MB)  =  31995
  内存使用量(MB)  =  13606
  内存使用率(%)   =  42.53
  (结果个数 = 1)
  ```

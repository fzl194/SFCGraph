## [命令功能](#ZH-CN_MMLREF_0249644923)

**适用NF：PGW-C、GGSN、SMF**

该命令用于查询全局地址分配规则。

## [注意事项](#ZH-CN_MMLREF_0249644923)

无

#### [操作用户权限](#ZH-CN_MMLREF_0249644923)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0249644923)

无

## [使用实例](#ZH-CN_MMLREF_0249644923)

查询全局地址分配规则：

```
%%LST IPALLOCRULE:;%%
RETCODE = 0 操作成功

结果如下
---------------------
IPv4第一级规则开关 = 使能
    IPv4第一级规则 = APN
IPv4第二级规则开关 = 去使能
    IPv4第二级规则 = NULL
IPv4第三级规则开关 = 去使能
    IPv4第三级规则 = NULL
IPv6第一级规则开关 = 使能
    IPv6第一级规则 = APN
IPv6第二级规则开关 = 去使能
    IPv6第二级规则 = NULL
IPv6第三级规则开关 = 去使能
    IPv6第三级规则 = NULL
(结果个数 = 1)

---    END
```

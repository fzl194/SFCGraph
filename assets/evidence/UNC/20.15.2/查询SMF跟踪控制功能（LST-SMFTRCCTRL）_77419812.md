## [命令功能](#ZH-CN_MMLREF_0000001977419812)

**适用NF：SGW-C、PGW-C、SMF、GGSN**

该命令用于查询SMF跟踪功能控制参数。

## [注意事项](#ZH-CN_MMLREF_0000001977419812)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001977419812)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001977419812)

无

## [使用实例](#ZH-CN_MMLREF_0000001977419812)

查询SMF跟踪控制功能，执行如下命令：

```
LST SMFTRCCTRL:;
```

## [输出结果说明](#ZH-CN_MMLREF_0000001977419812)

| 输出项名称 | 输出项解释 |
| --- | --- |
| E2E信元忽略开关 | 该参数用于指定系统是否忽略E2E跟踪信元。当忽略E2E跟踪信元时，系统不会上报E2E跟踪消息，也不会向下游网元传递跟踪信元。 |
| E2E跟踪不上报开关 | 该参数用于指定系统是否不上报E2E跟踪消息。 |
| E2E跟踪信元不传递开关 | 该参数用于指定系统是否向下游网元传递跟踪信元。 |
| 接口跟踪不上报开关 | 该参数用于指定系统是否不上报接口跟踪消息。 |
| 用户跟踪不上报开关 | 该参数用于指定系统是否不上报用户跟踪消息。 |
| 随机用户跟踪不上报开关 | 该参数用于指定系统是否不上报随机跟踪消息。 |
| 5G LAN组会话跟踪不上报开关 | 该参数用于指定系统是否不上报5G LAN组会话跟踪消息。 |
| 5G广播跟踪不上报开关 | 该参数用于指定系统是否不上报5G广播跟踪消息。 |
| 上报跟踪流控的消息开关 | 该参数用于指定系统是否上报跟踪流控消息。当开关为DISABLE时，不上报跟踪流控消息。 |

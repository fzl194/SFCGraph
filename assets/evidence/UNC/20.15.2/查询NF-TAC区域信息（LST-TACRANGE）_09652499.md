- [参数说明](#ZH-CN_MMLREF_0209652499__1.3.4)

## [命令功能](#ZH-CN_MMLREF_0209652499)

**适用NF：AMF、SMF、NRF、NSSF、NCG、SMSF**

该命令用于查询NF支持的TAC区域信息。

## [注意事项](#ZH-CN_MMLREF_0209652499)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209652499)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652499)

无

## [使用实例](#ZH-CN_MMLREF_0209652499)

运营商A需要查询配置的TAC区域信息。

```
%%LST TACRANGE:;%%
RETCODE = 0  操作成功

结果如下
--------
     索引标识  =  1
TAC区域起点值  =  000001
TAC区域结束值  =  000100
     描述信息  =  NULL
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209652499)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 索引标识 | 本参数用于指定索引标识，用于标识TAC区域。 |
| TAC区域起点值 | 本参数用于指定TAC区域起点值。 |
| TAC区域结束值 | 本参数用于指定TAC区域结束值。 |
| 描述信息 | 该参数表示对TAC区域信息的描述，在运维中起助记作用。 |

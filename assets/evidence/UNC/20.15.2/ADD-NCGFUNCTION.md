## [命令功能](#ZH-CN_MMLREF_0245110911)

**适用NF：NCG**

该命令用于增加NCG功能实例信息。

## [注意事项](#ZH-CN_MMLREF_0245110911)

- 该命令执行后立即生效。

- 最多可输入1条记录。

#### [操作用户权限](#ZH-CN_MMLREF_0245110911)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0245110911)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| INSTANCEID | NF实例号 | 可选必选说明：必选参数<br>参数含义：该参数用于指定NF实例号。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~40。<br>默认值：无<br>配置原则：无 |
| CDRPROCCAP | 话单处理容量 | 可选必选说明：必选参数<br>参数含义：该参数用于指定话单处理容量。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~2147483647，单位是条。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0245110911)

增加NF实例号为“a”，话单处理容量为“900”的NCG功能实例信息：

```
ADD NCGFUNCTION: INSTANCEID="a", CDRPROCCAP=900;
```

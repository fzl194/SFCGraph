#### [命令功能](#ZH-CN_CONCEPT_0000001600840757)

该命令用于查询指定组ID和计数器ID的计数器信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600840757)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600840757)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600840757)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RUNAME | RU名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定RU名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，区分大小写。<br>默认值：无<br>配置原则：使用DSP RU查看RU名称。 |
| GROUPID | 计数器组ID | 可选必选说明：必选参数<br>参数含义：该参数用于表示计数器组ID。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～4294967295。<br>默认值：无 |
| COUNTERID | 计数器ID | 可选必选说明：必选参数<br>参数含义：该参数用于表示计数器ID。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～4294967295。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600840757)

查询指定counter ID的计数器信息：

```
DSP MSSCOUNTERINFO:RUNAME = "VNODE_VNRS_VNFC_IPU_0064",GROUPID = 1,COUNTERID = 0;
```

```

RETCODE = 0  操作成功。

结果如下
--------
计数器组ID  =  1
  计数器ID  =  0
计数器类型  =  ATOMIC_COUNTER_64
 计数器值0  =  0
 计数器值1  =  NULL
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600840757)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 计数器组ID | 用于表示计数器组ID。 |
| 计数器ID | 用于表示计数器ID。 |
| 计数器类型 | 用于表示计数器类型，例如32位计数器、64位计数器、32位线程计数器、64位线程计数器、32位线程双计数器、64位线程双计数器。 |
| 计数器值0 | 用于表示计数器值0。 |
| 计数器值1 | 用于表示计数器值1。 |

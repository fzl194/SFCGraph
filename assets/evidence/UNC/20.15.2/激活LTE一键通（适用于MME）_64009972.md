> **说明**
> 用户从LTE网络移动到GSM/UMTS网络场景下：
>
> - 部署本功能前，65、66、69和70为扩展QCI，UNC需要将扩展QCI转换为标准QCI（[**ADD QCICONV**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/QoS管理/EPS QoS/扩展QCI转换关系/增加扩展QCI转换关系(ADD QCICONV)_26306024.md)），然后转换为Pre-R8 QoS（[**SET R8QOSMAP**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/QoS管理/QoS兼容性管理/EPS QoS参数到Pre-R8 QoS参数映射/设置EPS QoS参数到Pre-R8 QoS参数映射规则（SET R8QOSMAP）_26146234.md)）。
> - 部署本功能后，65、66、69和70为PTT专用QCI，UNC直接将PTT专用QCI转换为Pre-R8 QoS（[**SET R8QOSMAP**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/QoS管理/QoS兼容性管理/EPS QoS参数到Pre-R8 QoS参数映射/设置EPS QoS参数到Pre-R8 QoS参数映射规则（SET R8QOSMAP）_26146234.md)）。
>
> 因此，建议运营商在部署本功能之前，需要规划好PTT专用QCI和Pre-R8 QoS的映射关系。

## [操作步骤](#ZH-CN_OPI_0164009972)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开 LTE一键通 功能特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. 配置PTT APN以及PTT业务寻呼优先级。
  [**ADD APNPAGINGPLCY**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/会话管理/基于APN的寻呼策略配置/增加APN寻呼策略参数配置(ADD APNPAGINGPLCY)_72225389.md)
4. 增加PTT业务QoS参数到Pre-R8 QoS参数映射规则。
  [**SET R8QOSMAP**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/QoS管理/QoS兼容性管理/EPS QoS参数到Pre-R8 QoS参数映射/设置EPS QoS参数到Pre-R8 QoS参数映射规则（SET R8QOSMAP）_26146234.md)
5. 配置PTT业务QCI和标准QCI的映射规则。
  [**ADD QCICONV**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/QoS管理/EPS QoS/扩展QCI转换关系/增加扩展QCI转换关系(ADD QCICONV)_26306024.md)
6. 配置MME间移动时用户的QCI传递策略。
  [**SET COMPATIBILITY**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/QoS管理/QoS兼容性管理/QoS兼容性配置/设置QoS兼容性配置(SET COMPATIBILITY)_72345835.md)

## [任务示例](#ZH-CN_OPI_0164009972)

任务描述

激活 LTE一键通 功能特性。

脚本

// 进入 “MML命令行-UNC” 窗口。

//打开 LTE一键通 功能特性的License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2QPPT01", SWITCH=ENABLE;
```

//配置PTT APN以及PTT业务寻呼优先级。

```
ADD APNPAGINGPLCY: APNNI="huawei.com", PAGINGPRIORITY=7;
```

//增加PTT业务QoS参数到Pre-R8 QoS参数映射规则。

```
SET R8QOSMAP: QCI=65, TC=CC, MAXSDU=150, DO=NORDER, RBER=E_RESIDUAL_BIT_ERR_RATIO_7, DESDU=NOT_DETECT, SDUER=SDUER1, TD=10;
```

```
SET R8QOSMAP: QCI=66, TC=CC, MAXSDU=150, DO=NORDER, RBER=E_RESIDUAL_BIT_ERR_RATIO_7, DESDU=NOT_DETECT, SDUER=SDUER1, TD=10;
```

```
SET R8QOSMAP: QCI=69, TC=IC, MAXSDU=150, DO=NORDER, RBER=E_RESIDUAL_BIT_ERR_RATIO_7, DESDU=NOT_DETECT, SDUER=SDUER4, THPRI=THPRI1;
```

```
SET R8QOSMAP: QCI=70, TC=IC, MAXSDU=150, DO=NORDER, RBER=E_RESIDUAL_BIT_ERR_RATIO_7, DESDU=NOT_DETECT, SDUER=SDUER4, THPRI=THPRI1;
```

//配置PTT业务QCI和标准QCI的映射规则。

```
ADD QCICONV: QCIEXT=65, QCISTEP=1, QCISTD=1;
```

```
ADD QCICONV: QCIEXT=69, QCISTD=5;
```

```
ADD QCICONV: QCIEXT=70, QCISTD=6;
```

//配置MME间移动时用户的QCI传递策略。

```
SET COMPATIBILITY: GTPV2EXTQCI=SUPPORT;
```

前提条件

- 请仔细阅读[WSFD-206008 5G UE信令控制特性概述](特性概述_72052698.md)。

## [任务示例](#ZH-CN_OPI_0272052700)

任务描述

为了减轻由于UE的异常行为对网络侧的影响，有效地减少网络中的异常信令，提升网络可用性和可靠性，对5G用户异常反复发送的初始注册请求，服务请求、PDU会话建立请求消息、4G到5G的移动注册更新请求或者4G到5G的切换请求进行抑制，需要对用户手机采用抑制唤醒规则最终使其恢复正常业务。配置执行抑制激活规则（特定原因值拒绝、Parking DNN激活、网络侧去注册、丢弃请求）来消除用户异常行为。

脚本

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2SAUSC01", SWITCH=ENABLE;
```

//开启用户异常信令抑制功能。

```
SET NGSMARTPARA: CTRLSW=ON;
```

//设置异常识别参数。

```
SET NGSMARTPARA: CTRLSW=ON, REGTHRESH=30, REGMEPERIOD=3600,SRVREQTHRESH=30, SRVREQPERIOD=3600,PDUREQTHRESH=30, PDUREQPERIOD=3600;
```

//设置抑制时长。

```
SET NGSMARTPARA: CTRLSW=ON, REGDURATION=60; 
```

//设置抑制措施。

```
SET NGSMARTPARA: CTRLSW=ON, REGCTRLRULE=SPECCAUSEREJ&CNDEREG-1&DISCARD-1, REGREJCAUSE=15,SRVREQCTRLRULE=SPECCAUSEREJ-1&CNDEREG-1&DISCARD-1,SRVREJCAUSE=15, PDUREQCTRLRULE=SPECCAUSEREJ-1&PARKDNNACT-1&CNDEREG-1&DISCARD-1, PDUREQREJCAUSE=65, DEREGISTCAUSE=15; 
```

//配置Parking DNN参数以及Parking DNN的抑制唤醒措施。

```
SET NGSMARTPARA: CTRLSW=ON, PARKINGDNN="PARKING",PARKDNNWAKEUP=CNDEACT;
```

//配置4G到5G初始注册流程异常信令抑制开关。

```
SET NGSMARTPARA: CTRLSW=ON, LTETONRINITREGSW=ON;
```

//配置4G到5G重选流程的异常信令抑制功能相关参数。

```
SET NGSMARTPARA: CTRLSW=ON, LTETONRRSLSW=ON, LTETONRRSLTHRESH=50, LTETONRRSLMEPERIOD=3600, LTETONRRSLDURATION=60, LTETONRREGCTRLRULE=SPECCAUSEREJ-1, LTETONRRSLREJCAUSE=15;
```

//配置4G到5G切换流程的异常信令抑制功能相关参数。

```
SET NGSMARTPARA: CTRLSW=ON, LTETONRHOSW=ON, LTETONRHOTHRESH=50, LTETONRHOMEPERIOD=3600, LTETONRHODURATION=60, LTETONRHOCTRLRULE=SPECCAUSEREJ-1, LTETONRHOREJCAUSE=81;
```

//（可选）配置AMF对于Paging触发的Service Request流程是否进行信令抑制判断。

```
SET AMFSOFTPARAOFBIT: DT=Dw, DWORDNUM=72, VALUE=VALUE_0, POSITION=3;
```

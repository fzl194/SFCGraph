#### [命令功能](#ZH-CN_MMLREF_0000001126307106)

**适用网元：MME**

本命令用于查询SDAP(Sdup Application Protocol)实体的重启计数。

#### [注意事项](#ZH-CN_MMLREF_0000001126307106)

无

#### [本地用户权限](#ZH-CN_MMLREF_0000001126307106)

manage-ug；system-ug；monitor-ug；visit-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126307106)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组；G_4，来宾级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126307106)

无。

#### [输出结果说明](#ZH-CN_MMLREF_0000001126307106)

| 输出项名称 | 输出项解释 |
| --- | --- |
| RU名称 | 参数含义：该参数用于指定资源单元名称。<br>取值范围：1~63位字符串 |
| 进程号 | 参数含义：本参数显示SDAP实体所在的SAP进程号。<br>取值范围：0～20 |
| 重启计数 | 参数含义：本参数显示SDAP实体重启计数。<br>取值范围：0～255 |
| 文件状态 | 参数含义：本参数显示SDAP实体重启计数存储文件的状态。<br>若从文件读取结果成功，输出结果为<br>“正常”<br>。<br>若从文件读取结果失败，输出结果为<br>“异常”<br>。<br>取值范围：<br>- 正常<br>- 异常<br>说明：- 如果“文件状态”是“正常”，表明“重启计数”可信。<br>- 如果“文件状态”是“异常”，此时“重启计数”会取一个随机值。 |

## [操作场景](#ZH-CN_OPI_0184683735)

本操作指导介绍在运行网络中激活支持热计费功能的操作过程。

热计费是一种计费方式，具有普通计费的所有功能，只是话单产生速度比普通计费话单更快，保证运营商对此类用户及时收费。

> **说明**
> 适用于 SGSN 。

## [操作步骤](#ZH-CN_OPI_0184683735)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. 设置计费属性为热计费，并配置相关参数。
  [**SET CHGCHAR**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/计费管理/计费属性参数配置/设置计费属性参数(SET CHGCHAR)_26145368.md)
  > **说明**
  > 参数 “CC” （计费属性）选择 “HOTBILLING” (实时计费)。

## [任务示例](#ZH-CN_OPI_0184683735)

任务描述

设置实时计费用户的S-CDR生成周期(min)为2分钟。

脚本

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2HBILL02", SWITCH=ENABLE;
```

//计费属性设为“实时计费”，生成S-CDR设为“YES”，周期生成S-CDR设为“PERMIT”，S-CDR生成周期(min)设为“2”。

```
SET CHGCHAR: CC=HOTBILLING, SP=YES, SPP=PERMIT, SPL=2;
```

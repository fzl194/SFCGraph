## [操作场景](#ZH-CN_OPI_0000002048518810)

运营商需要UNC支持外部QoS能力开放服务器通过私有接口查询用户信息，需要激活QoS能力开放的功能。

> **说明**
> 适用于 GGSN、 PGW-C。

## [必备事项](#ZH-CN_OPI_0000002048518810)

前提条件

- 请仔细阅读 [PCC基本功能（2G/3G/4G，基于Gx接口）](../../PCC基本功能（2G_3G_4G，基于Gx接口）_29056156.md) 。
- 操作员了解UNC各类接口的类型及命名规范，具体内容请参见[SGW-C、PGW-C、GGSN](../../../../../../../快速入门/网元介绍/UNC产品描述/接口与协议标准/SGW-C、PGW-C、GGSN_17837431.md)。
- 操作员了解UNC与对端网元互通时可以采用的各种组网方式，以及每种组网方式的实现及特点等信息。
- 如果运营商规划采用VPN组网方式，则操作员在执行本操作前应已创建了相应的VPN实例。
- 需要配置逻辑接口的PBU正常运行，且PBU上无用户存在。当PBU还未启动或正在启动过程中，将无法在该PBU上配置逻辑接口。

## [操作步骤](#ZH-CN_OPI_0000002048518810)

1. 配置外联口IP及路由数据，详细请参见 [组网路由配置](../../../../../../初始配置/UNC初始配置与调测/组网路由配置_24779181.md) 。
2. 进入 “MML命令行-UNC” 窗口。
3. 创建一个VPN实例。
  [**ADD VPNINST**](../../../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/计费和策略接口管理/VPN/增加VPN实例（ADD VPNINST）_09653755.md)
4. 配置Restif逻辑接口地址。
  [**ADD LOGICIP**](../../../../../../../OM参考/命令/UNC MML命令/平台服务管理/IP管理/IP配置/增加逻辑IP地址（ADD LOGICIP）_09587376.md)
5. 配置HTTP本端端点组信息。
  **[ADD HTTPLEGRP](../../../../../../../OM参考/命令/UNC MML命令/平台服务管理/HTTP功能管理/HTTP管理/HTTP本端实体组管理/增加HTTP本端实体组（ADD HTTPLEGRP）_29213277.md)**
6. 配置HTTP本端端点信息。
  **[ADD HTTPLE](../../../../../../../OM参考/命令/UNC MML命令/平台服务管理/HTTP功能管理/HTTP管理/HTTP本端实体管理/增加HTTP本端实体（ADD HTTPLE）_84132094.md)**
7. 配置服务化接口本端实体。
  **[ADD SBIAPLE](../../../../../../../OM参考/命令/UNC MML命令/平台服务管理/HTTP功能管理/SBI管理/服务化接口本端实体管理/增加服务化接口本端实体（ADD SBIAPLE）_28971835.md)**
8. 使能HTTP1.1开关。
  **[SET HTTPCONF](../../../../../../../OM参考/命令/UNC MML命令/平台服务管理/HTTP功能管理/HTTP管理/HTTP属性管理/设置HTTP属性（SET HTTPCONF）_83972196.md)**
9. 开启QoS能力开放功能开关。
  **[SET SMCOMMFUNC](../../../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/会话协议参数管理/会话管理拓展功能/通用会话管理拓展功能/设置通用会话拓展功能（SET SMCOMMFUNC）_08433263.md)**

## [任务示例](#ZH-CN_OPI_0000002048518810)

任务描述

本实例中，操作员在UNC上进行数据配置实现以下要求：

- UNC使用逻辑接口与SBC连通。
- UNC使用VPN缺省路由与SBC建立起正常的IP连接。

脚本

1. 创建一个VPN实例。
  ```
  ADD VPNINST:VPNINSTANCE="
  vpn_restif
  ";
  ```
2. 配置即Restif逻辑接口地址（HTTP本端端点的逻辑地址信息）。
  ```
  ADD LOGICIP:IPVERSION=
  IPv6
  , LOGICIPV6="
  fe80:8029:5008:6B::247
  ", VPNINSTNAME="
  vpn_restif
  ", DESC="Restful_IPv6_Client";
  ```
  ```
  ADD LOGICIP:IPVERSION=
  IPv6
  , LOGICIPV6="
  fe80:8029:5008:6B::248
  ", VPNINSTNAME="
  vpn_restif
  ", DESC="Restful_IPv6_Server";
  ```
3. 配置HTTP本端端点组信息。
  ```
  ADD HTTPLEGRP: INDEX=3, DESC="BSF";
  ```
4.
  配置HTTP本端端点信息（包括服务端和客户端）。

  ```
  ADD HTTPLE: INDEX=7, HTTPLEGRPIDX=3, LETYPE=CLIENT, DESC="BSF Client", IPTYPE=IPv6, IPV6="
  fe80:8029:5008:6B::247
  ", VPNNAME="
  vpn_restif
  ";
  ```

  ```
  ADD HTTPLE: INDEX=8, HTTPLEGRPIDX=3, LETYPE=SERVER, PORT=80, TLSFLG=NO, DESC="BSF Server", IPTYPE=IPv6, IPV6="
  fe80:8029:5008:6B::248
  ", VPNNAME="
  vpn_restif
  "; 
  ```
5. 配置服务化接口本端实体。
  ```
  ADD SBIAPLE: INDEX=1, HTTPLEGRPIDX=3, NFTYPE=NFTypeBSF, DESCRIPTION="BSF";
  ```
6.
  使能HTTP1.1开关。

  ```
  SET HTTPCONF: HTTPVERSIONSW=TRUE;
  ```
7. 开启QoS能力开放功能。
  ```
  SET SMCOMMFUNC: PCRFQRYSW=ENABLE
  ```

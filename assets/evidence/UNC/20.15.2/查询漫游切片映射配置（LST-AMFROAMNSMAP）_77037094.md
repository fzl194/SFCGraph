## [命令功能](#ZH-CN_MMLREF_0000001177037094)

**适用NF：AMF**

该命令用于查询漫游切片映射配置。

## [注意事项](#ZH-CN_MMLREF_0000001177037094)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001177037094)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001177037094)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| HMCC | 归属地移动国家码 | 可选必选说明：可选参数<br>参数含义：该参数用于描述归属地移动国家码。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度是3。3位十进制数。<br>默认值：无<br>配置原则：无 |
| HMNC | 归属地移动网号 | 可选必选说明：可选参数<br>参数含义：该参数用于描述归属地移动网号。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是2~3。2位或者3位十进制数。<br>默认值：无<br>配置原则：无 |
| HSST | 归属地切片服务类型 | 可选必选说明：可选参数<br>参数含义：该参数用于描述归属地切片服务类型。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是0~255。<br>默认值：无<br>配置原则：无 |
| HSD | 归属地切片细分标识 | 可选必选说明：可选参数<br>参数含义：该参数用于表示归属地切片细分标识。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是1~6。该参数输入长度为6或1。长度为6时，只能由字母（A-F或者a-f）、数字（0-9）组成，且大小写不敏感，转大写处理。长度为1时，只能为“*”，表示通配符，任意SD值都可以匹配。<br>默认值：无<br>配置原则：<br>如果该参数为空请填入FFFFFF。 |
| VMCC | 拜访地移动国家码 | 可选必选说明：可选参数<br>参数含义：该参数用于描述拜访地移动国家码。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度是3。3位十进制数。<br>默认值：无<br>配置原则：无 |
| VMNC | 拜访地移动网号 | 可选必选说明：可选参数<br>参数含义：该参数用于描述拜访地移动网号。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是2~3。2位或者3位十进制数。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001177037094)

查询漫游切片映射配置，执行如下命令：

```
%%LST AMFROAMNSMAP:;%%
RETCODE = 0  操作成功

结果如下
------------------------
  归属地移动国家码  =  234
    归属地移动网号  =  38
归属地切片服务类型  =  1
归属地切片细分标识  =  FFFFFF
  拜访地移动国家码  =  123
    拜访地移动网号  =  45
拜访地切片服务类型  =  1
拜访地切片细分标识  =  FFFFFF
(结果个数 = 1)

---    END
```

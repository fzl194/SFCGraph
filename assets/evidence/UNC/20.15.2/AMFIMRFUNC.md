## [命令功能](#ZH-CN_MMLREF_0244006790)

**适用NF：AMF**

该命令用于查询用户消息统计功能配置。

## [注意事项](#ZH-CN_MMLREF_0244006790)

无

#### [操作用户权限](#ZH-CN_MMLREF_0244006790)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0244006790)

无

## [使用实例](#ZH-CN_MMLREF_0244006790)

查询所有的AMFIMRFUNC记录：

```
%%LST AMFIMRFUNC:;%%
RETCODE = 0  操作成功

结果如下
--------
用户范围  =  所有用户
接口类型  =  Nas
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0244006790)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 用户范围 | 该参数用于指定用户范围。 |
| 接口类型 | 该参数用于指定接口类型。 |

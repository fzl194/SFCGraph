# 查询OM IP (LST OMIP)

- [命令功能](#ZH-CN_TOPIC_0276163349__1.3.1.1)
- [注意事项](#ZH-CN_TOPIC_0276163349__1.3.2.1)

#### [命令功能](#ZH-CN_TOPIC_0276163349)

用于查询OM网络配置。

#### [注意事项](#ZH-CN_TOPIC_0276163349)

无。

#### [参数说明](#ZH-CN_TOPIC_0276163349)

无。

#### [使用实例](#ZH-CN_TOPIC_0276163349)

查询OMIP：

```
%%LST OMIP:;%%
RETCODE = 0  操作成功
IPv4浮动IP信息如下
------------------
  IP地址  =  10.144.246.179
    端口  =  31943
子网掩码  =  255.255.255.0
默认网关  =  10.144.246.254
(结果个数 = 1)

IPv6浮动IP信息如下
------------------
  IP地址  =  2001:db8:0000:4202:0156:0110:0202:0511
    端口  =  31943
前缀长度  =  64
默认网关  =  2001:db8:0000:4202:0156:0110:0202:0254
(结果个数 = 1)

物理IP信息如下
--------------
主机名  IP地址                         

host1   2001:db8:0:4202:156:110:202:191  
host2   10.144.246.202                
(结果个数 = 2)

---    END
```

#### [输出结果说明](#ZH-CN_TOPIC_0276163349)

命令执行正常，会返回命令执行成功的提示信息。输出结果说明如 [表1](#ZH-CN_TOPIC_0276163349__table47787675) 所示。

该命令执行异常，请联系华为技术支持处理。错误码说明如 [表2](#ZH-CN_TOPIC_0276163349__table698716490471) 所示。

*表1 输出项解释*

| **输出项名称** | **输出项解释** |
| --- | --- |
| IP地址 | 登录OM Portal的浮动IPv4地址。 |
| 端口 | 登录OM Portal的IPv4端口。 |
| 默认网关 | 登录OM Portal的浮动IPv4地址的网关。 |
| 子网掩码 | 登录OM Portal的IPv4子网掩码。 |
| IP地址 | 登录OM Portal的IPv6浮动IP地址。 |
| 端口 | 登录OM Portal的IPv6端口。 |
| 默认网关 | 登录OM Portal的浮动IPv6地址的网关。 |
| 前缀长度 | IPv6 地址的前缀长度。 |
| 主机名 | 物理主机名。 |
| IP地址 | 物理IP地址。 |

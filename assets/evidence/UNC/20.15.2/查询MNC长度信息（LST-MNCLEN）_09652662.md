## [命令功能](#ZH-CN_MMLREF_0209652662)

**适用NF：SMF**

该命令用于查看MCC号对应的MNC长度。

## [注意事项](#ZH-CN_MMLREF_0209652662)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209652662)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652662)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| MCC | 移动国家码 | 可选必选说明：可选参数<br>参数含义：该参数用于指定移动国家码。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度是3。取值范围000～999。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209652662)

查看当前UNC所属的MCC的MNC长度：

```
%%LST MNCLEN:;%%
RETCODE = 0  操作成功

结果如下
--------
      移动国家码  =  233
对应MCC的MNC长度  =  2
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209652662)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 移动国家码 | 该参数用于指定移动国家码。 |
| 对应MCC的MNC长度 | 该参数用于指定MCC值所对应的MNC长度。 |

#### [命令功能](#ZH-CN_CONCEPT_0209897002)

**适用NF：SGW-C、PGW-C、SMF**

命令用来查看配置的缓存话单文件的有效期。

#### [注意事项](#ZH-CN_CONCEPT_0209897002)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0209897002)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0209897002)

无。

#### [使用实例](#ZH-CN_CONCEPT_0209897002)

查询缓存话单文件的超期时间信息：

```
LST CDRSTORAGECTRL:;
```

```

RETCODE = 0  操作成功。

话单存储控制参数
----------------
话单缓存超期天数  =  3
话单缓存超期周数  =  1
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0209897002)

参见SET CDRSTORAGECTRL的参数说明。

# 删除UPF和规则的绑定关系（RMV UPFBINDSELRULE）

- [注意事项](#ZH-CN_MMLREF_0000001875823004__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0000001875823004__1.3.4)
- [使用实例](#ZH-CN_MMLREF_0000001875823004__1.3.5)

## [命令功能](#ZH-CN_MMLREF_0000001875823004)

**适用NF：SMF、PGW-C、SGW-C**

该命令用于删除UPF和规则的绑定关系。

## [注意事项](#ZH-CN_MMLREF_0000001875823004)

该命令执行后只对新激活用户生效。

#### [操作用户权限](#ZH-CN_MMLREF_0000001875823004)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001875823004)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| INDEX | 索引 | 可选必选说明：必选参数<br>参数含义：该参数用于指定UPF和规则的绑定关系的索引。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~4999。<br>默认值：无<br>配置原则：<br>该参数取值不能重复，建议从0开始顺序取值。 |

## [使用实例](#ZH-CN_MMLREF_0000001875823004)

删除UPF和规则的绑定关系，UPF实例名称为"UPF1"，规则名称为"rulename1"：

```
RMV UPFBINDSELRULE: INDEX=0;
```

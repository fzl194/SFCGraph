## [命令功能](#ZH-CN_MMLREF_0244529800)

**适用NF：PGW-C、SGW-C**

该命令用来增加区域性能统计对象和TAC段的绑定关系。当用户从某TAC接入时，系统会使用对应的区域进行指定区域或指定区域和APN的性能统计。

## [注意事项](#ZH-CN_MMLREF_0244529800)

- 该命令执行后立即生效。

- 最多可输入1000条记录。
- 相同的TAC段只能绑定到一个区域，不能同时绑定到多个区域。

#### [操作用户权限](#ZH-CN_MMLREF_0244529800)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0244529800)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| REGNAME | 区域名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定与TAC段绑定的区域名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~31。不支持空格，不区分大小写。<br>默认值：无<br>配置原则：<br>该参数使用ADD PERFREGION命令配置生成。 |
| TACSEGNAME | TAC段名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定与区域绑定的TAC段名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~31。不支持空格，不区分大小写。<br>默认值：无<br>配置原则：<br>该参数使用ADD PERFS1TACSEG命令配置生成。 |

## [使用实例](#ZH-CN_MMLREF_0244529800)

当运营商需要配置一个区域性能统计对象绑定的TAC号段时，执行如下命令：

```
ADD PERFREGTAC: REGNAME="beijing", TACSEGNAME="changping";
```

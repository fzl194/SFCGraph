## [操作场景](#ZH-CN_OPI_0170150790)

随着5G网络部署的普及，越来越多的用户将会选择5G UE，运营商将会面对5G签约用户数量激增的场景，若单一HPLMN内的用户号段资源无法满足激增的5G签约用户数量，则需要开启本特性， UNC 为拥有多个HPLMN的运营商服务，使运营商具备为多个PLMN内用户提供归属地业务服务的能力，从而扩展运营商的用户号段资源。本特性可以帮助运营商配置签署漫游协议的互联网络，同时为互联网络中的用户提供服务。

> **说明**
> 适用于AMF。

## [操作步骤](#ZH-CN_OPI_0170150790)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License开关。
  **[**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)**
3. 查询MNO信息， UNC 在初配过程中已使用 [**ADD MNO**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/归属网络运营商管理/MNO管理/MNO配置表/增加MNO配置信息(ADD MNO)_72345671.md) 命令添加了一条缺省MNO记录，通过本步骤可获取该缺省MNO的相关参数信息。
  **[**LST NGMNO**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/运营商管理/5G 移动网络运营商管理/查询5G模式移动网络运营商信息（LST NGMNO）_09652431.md)**
4. 通过本步骤在查询到的MNO上增加HPLMN记录，使得该MNO可以接入新增加的HPLMN网络用户。
  **[**ADD NGHPLMN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/运营商管理/Home PLMN信息管理/增加5G Home PLMN（ADD NGHPLMN）_09651693.md)**
5. **可选：**增加互联PLMN配置信息。若运营商需要为互联网络中的漫游用户提供接入服务，则需要该步骤来增加该MNO上可服务的互联网络信息。
  **[**ADD NGCONNECTPLMN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/运营商管理/互联PLMN管理/增加5G Connect PLMN（ADD NGCONNECTPLMN）_09651402.md)**

## [任务示例](#ZH-CN_OPI_0170150790)

任务描述

一个 UNC 为一个拥有2个PLMN ID的MNO服务，该MNO的MNO ID为01，两个HPLMN的信息分别如下：

- PLMN1：移动国家码为123，移动网号为002；
- PLMN2：移动国家码为123，移动网号为003。

脚本

//打开本特性的License开关。

```
SET LICENSESWITCH: LICITEM="LKV2MHAM01", SWITCH=ENABLE;
```

//添加两条HPLMN记录。

```
ADD NGHPLMN: NOID=0, MCC="123", MNC="002", DESC="00";
ADD NGHPLMN: NOID=0, MCC="123", MNC="003", DESC="01";
```

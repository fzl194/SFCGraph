## [命令功能](#ZH-CN_MMLREF_0230518083)

**适用NF：SMF**

该命令用于显示UP故障处理的系统参数。

## [注意事项](#ZH-CN_MMLREF_0230518083)

无

#### [操作用户权限](#ZH-CN_MMLREF_0230518083)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0230518083)

无

## [使用实例](#ZH-CN_MMLREF_0230518083)

查询UP故障处理的系统参数： LST UPFAULTOPERPARA:;

```
%%LST UPFAULTOPERPARA:;%%
RETCODE = 0  操作成功

结果如下
------------------------
故障迁移速率(个/秒)  =  1000
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0230518083)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 故障迁移速率(个/秒) | 该参数用于控制副锚点UPF故障时，系统批量迁移或者批量删除分流业务的速率。 |

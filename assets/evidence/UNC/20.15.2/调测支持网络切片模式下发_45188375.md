# 调测 支持网络切片模式下发

- [操作场景](#ZH-CN_OPI_0245188375__1.3.1)
- [必备事项](#ZH-CN_OPI_0245188375__1.3.2)

## [操作场景](#ZH-CN_OPI_0245188375)

本操作指导介绍在运行网络中调测 支持网络切片模式下发 特性的操作过程。

> **说明**
> 适用于AMF。

## [必备事项](#ZH-CN_OPI_0245188375)

前提条件

- 请仔细阅读[WSFD-110005 支持网络切片模式下发特性概述](特性概述_45188372.md)。

- 完成[激活支持网络切片模式下发](激活支持网络切片模式下发_45188374.md)。

数据

该操作无需准备数据。

工具

该操作无需准备其他工具。

## [操作步骤](#ZH-CN_OPI_0245188375)

1. 进入 “MML命令行-UNC” 窗口。
2. 执行 [**LST NGMMSUBDATA**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/移动性管理/签约数据管理/查询用户移动性管理参数（LST NGMMSUBDATA）_09651400.md) 命令，查询3GPP接入方式下的网络切片包含模式和网络切片包含模式策略。
  预期结果：3GPP接入方式下的网络切片包含模式已配置为模式A，网络切片包含模式策略为标准模式。

  ```
  %%LST NGMMFUNC:;%% 
  RETCODE = 0  操作成功  

  结果如下 
  --------
                        AMF重选方式  =  重路由方式
                        最近访问TAI  =  是
                   RRC Inactive功能  =  是
                       发送网络信息  =  否
                      支持无N26接口  =  不支持
                支持通过NAS传短消息  =  不支持
                网络切片包含模式策略 = 标准模式
   3GPP接入方式下的网络切片包含模式  =  网络切片包含模式A
                是否允许紧急呼叫业务 =   NR接入的紧急呼叫回落
     基于UDM的P-CSCF Restoration功能 = 否 
  (结果个数 = 1)  

  ---    END
  ```

## [命令功能](#ZH-CN_MMLREF_0209587911)

此命令用于查询集群选举实例的信息，竞选实例组成集群，从中选出主和备。

## [注意事项](#ZH-CN_MMLREF_0209587911)

手动复位hafetcd服务的主时，至少延迟10秒再使用该命令查询。

#### [操作用户权限](#ZH-CN_MMLREF_0209587911)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209587911)

无

## [使用实例](#ZH-CN_MMLREF_0209587911)

获取集群选举实例的位置信息、主从信息、服务状态等：

```
%%DSP ELECTION:;%%
RETCODE = 0  操作成功
 
结果如下
------------------------
节点标识      Pod名称            ETCD实例标识       IP地址/端口号          角色      运行状态       开源版本号       数据存储量       递交时延

192.168.0.1  hafetcd-pod-2     b35a5665f835ac8c  192.168.1.54:2379     主节点     正常          3.3.6           332 kB          1.925483ms
192.168.0.2  hafetcd-pod-0     b10a1908d6d9f545  192.168.1.48:2379     从节点     正常          3.3.6           332 kB          12.376402ms
192.168.0.3  hafetcd-pod-1     675b998e579d3b37  192.168.1.51:2379     从节点     正常          3.3.6           332 kB          3.204205ms
(结果个数 = 3)
 
---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209587911)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 节点标识 | 该参数用于表示集群选举实例所在节点的编号。 |
| Pod名称 | 该参数用于表示集群选举实例所在Pod的名称。 |
| ETCD实例标识 | 该参数用于表示集群选举实例的编号。 |
| IP地址/端口号 | 该参数用于表示集群选举实例所在节点的IP地址/端口号。 |
| 角色 | 该参数用于表示集群选举实例的主备角色。<br>取值说明：<br>- Leader（主节点）<br>- NotLeader（从节点） |
| 运行状态 | 该参数用于表示集群选举实例的运行状态。<br>取值说明：<br>- Normal（正常）<br>- Fault（故障） |
| 开源版本号 | 该参数用于表示集群选举实例所用的开源版本号。 |
| 数据存储量 | 该参数用于表示集群选举实例的数据存储量。 |
| 递交时延 | 该参数用于表示集群选举实例的数据递交时延。 |
| Raft索引 | 该参数用于表示集群选举实例之间日志索引值。 |
| 选举任期 | 该参数用于表示集群选举实例的选举任期。 |

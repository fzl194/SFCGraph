## [命令功能](#ZH-CN_MMLREF_0209654416)

**适用NF：AMF**

该命令用于为5G区域群增加一条区域群成员记录，同一个区域群中的成员具有相同的IP地址分配策略。

## [注意事项](#ZH-CN_MMLREF_0209654416)

- 命令执行后只对新接入用户生效。

- 不同区域群中的TAC不能同属于一个TAI LIST，否则AMF无法实时感知用户的位置变化。
- 输入的TAC或TAC段不允许与该对象其它记录定义的TAC范围相互交叉、包含或重合。

- 最多可输入2048条记录。

#### [操作用户权限](#ZH-CN_MMLREF_0209654416)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209654416)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| AREAID | 区域群标识 | 可选必选说明：必选参数<br>参数含义：该参数用于指定区域群标识。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是1~64。此AREAID要与NGIPAREAGRP中的AREAID保持一致，受NGIPAREAGRP中的IPAREASW开关控制；。<br>默认值：无<br>配置原则：无 |
| TAC | 跟踪区域码 | 可选必选说明：必选参数<br>参数含义：该参数用于指定跟踪区域码。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度是6。TAC编码为16进制数，按照字符串格式输入，字符串长度为6，只能由数字（0-9），字母（A-F、a-f）组成。字母大小写不敏感。<br>默认值：无<br>配置原则：无 |
| TACEND | 跟踪区域结束值 | 可选必选说明：可选参数<br>参数含义：该参数用于指定跟踪区域结束值。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度是6。TACEND编码为16进制数，按照字符串格式输入，字符串长度为6，只能由数字（0-9），字母（A-F、a-f）组成。字母大小写不敏感。<br>默认值：无<br>配置原则：<br>如果未输入TACEND或输入值等于TAC，则表示某个固定TAC。<br>输入的TACEND要大于或等于TAC。 |
| DESC | 描述信息 | 可选必选说明：可选参数<br>参数含义：该参数是对组成IP区域的某个TAC的描述信息。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~32。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209654416)

增加一条IP区域群成员记录，区域群标识为SomeCity，跟踪区域码为0x123456，描述信息为SomeTown：

```
ADD NGIPAREAGRPMEM: AREAID="SomeCity", TAC="123456", Desc="SomeTown";
```

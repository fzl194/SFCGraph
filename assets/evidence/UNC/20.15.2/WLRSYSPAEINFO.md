#### [命令功能](#ZH-CN_CONCEPT_0000001600440977)

该命令用于显示系统PAE信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600440977)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600440977)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600440977)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001600440977)

显示系统PAE信息：

```
DSP WLRSYSPAEINFO:;
```

```

RETCODE = 0  操作成功

结果如下
-------------------------
FEI PID     Partner状态      TB high    TB low    TP      状态     剩余时间      PAE延时定时器

0x7f4661    Avail            0          64        4098    S        0              FALSE
0x7f4666    Avail            0          65        4098    S        0              FALSE
(结果个数 = 2)
---    END
```

## [命令功能](#ZH-CN_MMLREF_0000001823782746)

**适用NF：MME、AMF**

该命令用于查询拨测用户配置。

## [注意事项](#ZH-CN_MMLREF_0000001823782746)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001823782746)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001823782746)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| BEGINIMSI | 起始IMSI | 可选必选说明：可选参数<br>参数含义：该参数用于配置拨测用户的起始IMSI。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是14~15。只允许输入十进制数字（0-9）。不同记录的起始IMSI不能相同。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001823782746)

查询一条拨测用户配置，起始IMSI为460001111111111，执行如下命令：

```
%%LST AMFDIALTEST: BEGINIMSI="460001111111111";%%
RETCODE = 0  操作成功。

操作结果如下
-------------------------
起始IMSI  =  460001111111111
终止IMSI  =  460001111111112
(结果个数 = 1)
---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001823782746)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 起始IMSI | 该参数用于配置拨测用户的起始IMSI。 |
| 终止IMSI | 该参数用于配置拨测用户的终止IMSI。 |

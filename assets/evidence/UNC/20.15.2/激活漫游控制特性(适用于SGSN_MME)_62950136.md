## [操作场景](#ZH-CN_OPI_0162950136)

本操作指导介绍在运行网络中激活漫游控制特性的操作过程。

漫游控制，是指 UNC 提供互联PLMN表的配置，只有用户的归属PLMN属于互联PLMN表才能够接入本 UNC 。漫游控制特性使网络具有限制用户移动范围和业务访问的能力。

> **说明**
> 适用于 SGSN、 MME。

## [对系统的影响](#ZH-CN_OPI_0162950136)

该操作对系统正常运行没有影响。

## [操作步骤](#ZH-CN_OPI_0162950136)

1. 进入 “MML命令行-UNC” 窗口。
2. 增加互联的PLMN，并配置允许的业务。
  [**ADD CONNECTPLMN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/互联PLMN管理/增加互联PLMN(ADD CONNECTPLMN)_26305852.md)
3. 设置漫游PLMN用户的计费属性参数。
  [**SET CHGPLMNCHAR**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/计费管理/PLMN计费属性参数配置/设置PLMN的计费属性参数(SET CHGPLMNCHAR)_26305204.md)
  > **说明**
  > 参数 “PLMN” （PLMN类型）选择 “VPLMN” （拜访PLMN）。

## [任务示例](#ZH-CN_OPI_0162950136)

任务描述

增加一个互联的PLMN，PLMN ID为46000，国家码是86， 允许这个互连PLMN用户使用纠正短消息中心业务，纠正后的短消息中心为861395678956778， 并配置该VPLMN用户不生成LCS-NI-CDR话单。

脚本

//增加互联的PLMN，其移动国家码为“460”，移动网号为“00”，国家码为“86”，是否允许纠正短消息中心选择“YES”，纠正后的短消息中心为“861395678956778”，其他参数默认。

```
ADD CONNECTPLMN: MCC="460", MNC="00", CC="86", SMS=YES, SMSCR=YES, SMSCT="861395678956778";
```

//设置漫游用户的话单控制属性，配置其不生成LCS-NI-CDR，其他参数默认。

```
SET CHGPLMNCHAR: PLMN=VPLMN, LCSNIP=NO;
```

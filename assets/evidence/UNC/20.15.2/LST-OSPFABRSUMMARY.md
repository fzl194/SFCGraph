#### [命令功能](#ZH-CN_CONCEPT_0000001549961502)

该命令用于查询区域内路由聚合配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001549961502)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549961502)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549961502)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | OSPF进程号 | 可选必选说明：可选参数<br>参数含义：OSPF进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |
| AREAID | 区域ID | 可选必选说明：可选参数<br>参数含义：区域ID。<br>数据来源：对端协商<br>取值范围：IPv4地址类型。点分十进制格式。<br>默认值：无 |
| IPADDRESS | IP地址 | 可选必选说明：可选参数<br>参数含义：IP地址。<br>数据来源：本端规划<br>取值范围：IPv4地址类型。点分十进制格式。<br>默认值：无 |
| ADDRESSMASK | IP地址的掩码 | 可选必选说明：可选参数<br>参数含义：IP地址的掩码。<br>数据来源：本端规划<br>取值范围：IPv4地址类型。点分十进制格式。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549961502)

查询区域内路由聚合配置：

```
LST OSPFABRSUMMARY:;
```


#### [输出结果说明](#ZH-CN_CONCEPT_0000001549961502)

| 输出项名称 | 输出项解释 |
| --- | --- |
| OSPF进程号 | 该参数指定用户创建的OSPF进程号。 |
| 区域ID | 该参数指定用户创建的OSPF区域ID。 |
| 拓扑标识 | 该参数指定用户创建的OSPF拓扑标识。 |
| IP地址 | 该参数指定聚合路由的IP地址。 |
| IP地址的掩码 | 该参数指定聚合路由的地址掩码。 |
| 发布路由标识 | 该参数指定是否发布这条聚合路由。缺省时发布聚合路由。 |
| 使能开销配置 | 该参数指定是否配置聚合路由的开销值。 |
| 聚合路由的开销 | 该参数用于配置聚合路由的开销，当使能开销配置为true时生效。 |
| 继承最小cost | 该参数指定是否继承最小开销值。默认继承的就是最小开销值。 |
| 生成路由黑洞 | 该参数用于配置生成黑洞路由，用来防止路由环路。 |

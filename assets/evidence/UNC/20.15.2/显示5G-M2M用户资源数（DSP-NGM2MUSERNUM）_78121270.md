## [命令功能](#ZH-CN_MMLREF_0000001578121270)

**适用NF：AMF**

该命令用于查询AMF系统内M2M用户资源的统计结果。

## [注意事项](#ZH-CN_MMLREF_0000001578121270)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001578121270)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001578121270)

无

## [使用实例](#ZH-CN_MMLREF_0000001578121270)

查看AMF上的M2M用户资源统计结果，执行如下命令：

```
%%DSP NGM2MUSERNUM:;%%
RETCODE = 0  操作成功

结果如下
------------------------
5G RedCap注册态用户数  5G RedCap PDU会话数  POD ID    

10                     12                   usn-pod-0 0  
10                     12                   total            
(结果个数 = 2)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001578121270)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 5G RedCap注册态用户数 | 该参数用于标识系统POD中处于注册态的5G RedCap用户数。 |
| 5G RedCap PDU会话数 | 该参数用于标识系统POD中保存的5G RedCap PDU会话数量。 |
| POD ID | 该参数用于标识系统中的POD ID。 |

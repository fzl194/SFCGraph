## [命令功能](#ZH-CN_MMLREF_0000001171436535)

**适用NF：PGW-C、SMF**

该命令用于查询融合计费惯性运行参数。

## [注意事项](#ZH-CN_MMLREF_0000001171436535)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001171436535)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001171436535)

无

## [使用实例](#ZH-CN_MMLREF_0000001171436535)

查询融合计费惯性运行参数：

```
%%LST CHGIOPARA:;%%
RETCODE = 0  操作成功
结果如下
--------
                   N40用量上报开关  =  使能
             惯性运行FINAL上报开关  =  不使能
        惯性运行期间VT触发上报开关  =  使能
惯性运行期间CHF REAUTH触发上报开关  =  使能
(结果个数 = 1)
---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001171436535)

| 输出项名称 | 输出项解释 |
| --- | --- |
| N40用量上报开关 | 该参数用于控制SMF惯性运行期间是否计费。 |
| 惯性运行FINAL上报开关 | 该参数用于控制SMF进入惯性运行时是否向CHF上报所有RG和QF的FINAL用量容器。 |
| 惯性运行期间VT触发上报开关 | 该参数用于控制SMF惯性运行期间VT定时器超时是否上报CHF。 |
| 惯性运行期间CHF REAUTH触发上报开关 | 该参数用于控制SMF惯性运行期间收到CHF REAUTH请求时是否上报CHF。 |

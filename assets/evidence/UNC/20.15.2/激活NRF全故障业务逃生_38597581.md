## [操作步骤](#ZH-CN_OPI_0000001538597581)

1. 进入 “MML命令行-UNC” 窗口。
2. 配置缓存开关。
  [**SET NFDISCPLCY**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/服务化接口管理/注册与服务发现/本地NRF功能管理/NF发现策略管理/设置NF的服务发现策略（SET NFDISCPLCY）_09651764.md)
3. 订阅目标NF。
  [**ADD SUBCONDITION**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/服务化接口管理/注册与服务发现/NRF管理/NRF参数管理/注册与订阅管理/增加NF订阅条件（ADD SUBCONDITION）_09651488.md)

## [任务示例](#ZH-CN_OPI_0000001538597581)

任务描述

开启缓存开关，增加NF类型为NfAMF的订阅信息。

脚本

//开启缓存功能。

```
SET NFDISCPLCY: CACHESWITCH=ON;
```

//增加NF类型为NFAMF的订阅信息，订阅条件为NfInstanceId，通知事件为NF_REGISTERED。

```
ADD SUBCONDITION: NFTYPE=NfAMF, CONDITION=NfInstanceId-1&NFType-0&ServiceName-0&AmfRegionId-0&AmfSetId-0&GuamiList-0&PlmnId-0&NetworkSlice-0&NfGroup-0&Rev1-0&Rev2-0&Rev3-0, REQNOTIFEVENT=NF_REGISTERED-1&NF_DEREGISTERED-1&NF_PROFILE_CHANGED-1;
```

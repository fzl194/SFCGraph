## [命令功能](#ZH-CN_MMLREF_0209651659)

**适用NF：AMF、SMF、NRF、NSSF、SMSF、NCG、CBCF**

该命令用于查询NRF实例地址信息。

## [注意事项](#ZH-CN_MMLREF_0209651659)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209651659)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209651659)

无

## [使用实例](#ZH-CN_MMLREF_0209651659)

该命令用于查询NRF实例地址信息。

```
%%LST NRFADDR:;%%
RETCODE = 0  操作成功

结果如下
--------
NRF实例名称  =  NRF_Instance_0
 IP地址类型  =  IPTypeV4
   IPV4地址  =  192.168.0.1
   IPV6地址  =  ::
       端口  =  8081
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209651659)

| 输出项名称 | 输出项解释 |
| --- | --- |
| NRF实例名称 | 该参数用于指定NRF的实例名称。 |
| IP地址类型 | 该参数用于指定NRF的IP地址类型。 |
| IPV4地址 | 该参数用于指定IPV4类型地址。 |
| IPV6地址 | 该参数用于指定IPV6类型地址。 |
| 端口 | 该参数用于指定NRF的端口号。 |

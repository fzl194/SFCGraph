## [操作场景](#ZH-CN_OPI_0276652607)

当需要保证静态IP地址网段内的用户选择到UPF的可靠性时，激活静态地址用户路由冗余功能。

> **说明**
> 适用于GGSN、PGW-C、SMF。

## [对系统的影响](#ZH-CN_OPI_0276652607)

该操作对系统正常运行没有影响。

## [必备事项](#ZH-CN_OPI_0276652607)

前提条件

- 请仔细阅读[WSFD-107021 静态地址用户路由冗余特性概述](WSFD-107021 静态地址用户路由冗余特性概述_76652606.md)。
- 本特性与UDG配合使用生效，要求UNC上配置的用户静态地址网段信息必须在包含在UDG上配置的地址池内，请参考UDG中的**GWFD-010104 地址分配方式**。

## [任务示例](#ZH-CN_OPI_0276652607)

任务描述

某企业用户接入网络时，为用户分配的IP地址采用静态IP地址分配方式。配置UPF名称为UPF_Instance_1的UPF1为主用UPF，配置UPF名称为UPF_Instance_2的UPF2为备用UPF，当主用UPF故障时，可以为用户选择到备用UPF进行业务，提高业务可靠性。

UPF_Instance_1和UPF_Instance_2上已经配置包含网段10.0.0.0~10.1.2.3的地址池信息，需要配置静态IP地址段10.0.0.0~10.1.2.3到UPF_Instance_1和UPF_Instance_2的关系，在主用UPF正常时，为用户选择到主用UPF，在主用UPF故障时，为用户选择到备用UPF。

脚本

//创建本地地址池。

```
ADD ADDRPOOL: POOLNAME="pool01", IPVERSION=IPv4, POOLTYPE=UDM, CHECKIPVALID=Enable, HASVPN=Enable, VPNINSTANCE="VRF-A";
```

//配置地址池与APN的绑定关系。

```
ADD POOLBINDAPN:APN="huawei.com",POOLNAME="pool01";
```

//配置本地地址池里的IPv4地址段。

```
ADD SECTION: POOLNAME="pool01", SECTIONNUM=1, IPVERSION=IPv4, V4STARTIP="10.0.0.0", V4ENDIP="10.1.2.3";
```

//配置地址池组名称。

```
ADD ADDRPOOLGRP: POOLGRPNAME="poolgroup1", POOLGRPTYPE=UDM;
```

//配置地址池与地址池组绑定。

```
ADD POOLBINDGRP: POOLGRPNAME="poolgroup1", POOLNAME="pool01";
```

//配置静态地址用户使用的主、备UPF。

```
ADD UPNODE: NFINSTANCENAME="UPF_Instance_1", ADDRALLOCMODE=INHERIT;
ADD UPNODE: NFINSTANCENAME="UPF_Instance_2", ADDRALLOCMODE=INHERIT;
```

//配置静态地址用户使用的UPF组。

```
ADD ADDRUPGROUP: UPFGRPNAME="upfgroup1", UPFGRPTYPE=UDM;
```

//配置主、备UPF与UPF组绑定。

```
ADD UPFBINDGRP: UPFGRPNAME="upfgroup1", UPFID="UPF_Instance_1", PRIORITY=0;
ADD UPFBINDGRP: UPFGRPNAME="upfgroup1", UPFID="UPF_Instance_2", PRIORITY=1;
```

//配置地址池组与UPF组绑定。

```
ADD POOLGRPMAP: MAPPINGNAME="mapping01", POOLGRPNAME="poolgroup1", UPFGRPNAME="upfgroup1";
```

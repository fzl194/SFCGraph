# 查询5G模式移动网络运营商信息（LST NGMNO）

- [注意事项](#ZH-CN_MMLREF_0209652431__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0209652431__1.3.4)
- [使用实例](#ZH-CN_MMLREF_0209652431__1.3.5)

## [命令功能](#ZH-CN_MMLREF_0209652431)

**适用NF：SGW-C、PGW-C、AMF、SMF、NRF、NSSF**

该命令用于查询移动网络运营商的基本信息。

## [注意事项](#ZH-CN_MMLREF_0209652431)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209652431)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652431)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| NOID | 运营商标识 | 可选必选说明：可选参数<br>参数含义：该参数用于在系统内唯一标识移动网络运营商。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209652431)

- 查询“运营商标识”为“0”的移动网络运营商的基本信息，执行如下命令：
  ```
  %%LST NGMNO: NOID=0;%%
  RETCODE = 0  操作成功

  结果如下
  --------
  运营商标识  =  0
  运营商全称  =  NULL
  运营商简称  =  NULL
    描述信息  =  NULL
  (结果个数 = 1)

  ---    END
  ```
- 查询移动网络运营商的基本信息，执行如下命令：
  ```
  %%LST NGMNO:;%%
  RETCODE = 0  操作成功

  结果如下
  --------
  运营商标识  =  0
  运营商全称  =  NULL
  运营商简称  =  NULL
    描述信息  =  NULL
  (结果个数 = 1)

  ---    END
  ```

## [输出结果说明](#ZH-CN_MMLREF_0209652431)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 运营商标识 | 该参数用于在系统内唯一标识移动网络运营商。 |
| 运营商全称 | 该参数用于指定移动网络运营商的全称。AMF发送给UE的Configuration Update Command消息中携带的“Full name for network”信元值来源于本参数。 |
| 运营商简称 | 该参数用于指定移动网络运营商的简称。AMF下发给UE的Configuration Update Command消息中携带的“Short name for network”信元值来源于本参数。 |
| 描述信息 | 该参数表示对移动网络运营商的描述信息，在运维过程中起到助记的作用。 |

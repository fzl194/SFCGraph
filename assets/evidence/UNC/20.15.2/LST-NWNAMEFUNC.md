#### [命令功能](#ZH-CN_TOPIC_0000002045191233)

**适用NF：MME**

本命令用于查询运营商名称信息下发功能。该命令对应的业务功能暂未实现。

#### [注意事项](#ZH-CN_TOPIC_0000002045191233)

无。

#### [本地用户权限](#ZH-CN_TOPIC_0000002045191233)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_TOPIC_0000002045191233)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_TOPIC_0000002045191233)

无。

#### [使用实例](#ZH-CN_TOPIC_0000002045191233)

查询运营商名称信息下发功能，可以用如下命令：

```
%%LST NWNAMEFUNC:;%%
RETCODE = 0  操作成功  
操作结果如下 ------------              
    使用HSS网元标识不发送运营商名称开关  =  ON
使用HSS主机名或域名不发送运营商名称开关  =  OFF
         NB-IoT用户不发送运营商名称开关  =  OFF
       5G到4G业务流程发送运营商名称开关  =  OFF
 
(结果个数 = 1)  
---    结束
```

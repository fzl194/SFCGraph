- [注意事项](#ZH-CN_MMLREF_0249644921__1.3.2)

## [命令功能](#ZH-CN_MMLREF_0249644921)

**适用NF：PGW-C、SMF、GGSN**

该命令用于查询基于UPF地址分配的全局开关。

## [注意事项](#ZH-CN_MMLREF_0249644921)

无

#### [操作用户权限](#ZH-CN_MMLREF_0249644921)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0249644921)

无

## [使用实例](#ZH-CN_MMLREF_0249644921)

查询基于UPF地址分配的全局开关：

```
%%LST IPALLOCBYUPFGLBSW:;%%
RETCODE = 0 操作成功

结果如下
-----------------------------------------------
基于UPF分配IPv4地址的全局开关  =  使能
基于UPF分配IPv6地址的全局开关  =  使能
(结果个数 = 1)

--- END
```

## [输出结果说明](#ZH-CN_MMLREF_0249644921)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 基于UPF分配IPv4地址的全局开关 | 该参数用于指定基于UPF分配IPv4地址的全局开关。 |
| 基于UPF分配IPv6地址的全局开关 | 该参数用于指定基于UPF分配IPv6地址的全局开关。 |

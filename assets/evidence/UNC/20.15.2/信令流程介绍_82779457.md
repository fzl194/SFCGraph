| 业务流程场景 | 概述 |
| --- | --- |
| 已签约专网业务的UE在专网业务区域内激活PDU会话创建流程。 | 用户在专网业务区域外用自己的手机号码通过专网业务提供商网站入口完成自助签约，用户进入专网业务区域范围内时发起数据业务，激活PDU会话，详细流程如<br>[图1](信令流程介绍/PDU会话创建使能专网业务流程_85233681.md#ZH-CN_TOPIC_0000001185233681__fig1123514171632)<br>所示。 |
| 已签约专网业务的UE从专网业务区域外移动至区域内，或者用户直接在专网业务区域内完成专网业务签约，触发PDU会话重建流程。 | - 用户在专网业务区域外用自己的手机号码通过专网业务提供商网站入口完成自助签约，用户在专网业务区域外激活PDU会话后又进入到专网业务区域范围内，触发活PDU会话重建，详细流程如[图1](信令流程介绍/PDU会话重建使能专网业务流程_37553658.md#ZH-CN_TOPIC_0000001137553658__fig1123514171632)所示。<br>- 用户直接在专网业务区域范围内通过扫码等方式完成对专网业务的签约，触发PDU会话重建，详细流程如[图1](信令流程介绍/PDU会话重建使能专网业务流程_37553658.md#ZH-CN_TOPIC_0000001137553658__fig1123514171632)所示。 |
| 用户主动取消签约专网业务或原已签约的专网业务超时失效。 | 用户不再使用专网业务，主动注销签约触发PDU会话重建，详细流程如<br>[图1](信令流程介绍/专网业务去使能流程_39072900.md#ZH-CN_TOPIC_0000001139072900__fig1123514171632)<br>所示。 |

## [命令功能](#ZH-CN_MMLREF_0000001171516431)

**适用NF：AMF**

该命令用于查询AMF的PCF故障处理策略。

## [注意事项](#ZH-CN_MMLREF_0000001171516431)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001171516431)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001171516431)

无

## [使用实例](#ZH-CN_MMLREF_0000001171516431)

查询AMF的PCF故障处理策略，执行如下命令：

```
%%LST AMFPCFRESET:;%%
RETCODE = 0  操作成功

结果如下
--------
 PCF故障重选开关  =  打开
 扫描速率(个/秒)  =  5
        扫描模式  =  自定义扫描模式
单位扫描时间(秒)  =  3
    单位扫描个数  =  4

(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001171516431)

| 输出项名称 | 输出项解释 |
| --- | --- |
| PCF故障重选开关 | 该参数用于指定AMF和PCF的接口故障或者PCF的状态更新为去注册时，AMF是否重选PCF。 |
| 扫描模式 | 该参数表示AMF对于用户的扫描模式。 |
| 扫描速率(个/秒) | 该参数用于指定每个DS每秒扫描多少个用户，扫描到后为符合重选PCF条件的用户重新选择正常的PCF进行注册。 |
| 单位扫描时间(秒) | 该参数用于表示PCF故障后DS对用户扫描的时间间隔。 |
| 单位扫描个数 | 该参数用于表示每个DS每隔INTERVAL设置的时间扫描多少个用户，扫描到后为符合重选PCF条件的用户重新选择正常的PCF进行注册。 |

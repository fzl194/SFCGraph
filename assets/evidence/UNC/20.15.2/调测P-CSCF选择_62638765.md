# 调测P-CSCF选择

- [操作场景](#ZH-CN_OPI_0262638765__1.3.1)
- [必备事项](#ZH-CN_OPI_0262638765__1.3.2)
- [操作步骤](#ZH-CN_OPI_0262638765__1.3.3)

## [操作场景](#ZH-CN_OPI_0262638765)

当运营商需要部署P-CSCF选择功能时，需要对本功能进行调测，确保本功能可以正常使用。

> **说明**
> 适用于SMF。

## [必备事项](#ZH-CN_OPI_0262638765)

前提条件

- 请仔细阅读[WSFD-107019 P-CSCF选择特性概述](WSFD-107019 P-CSCF选择特性概述_62638683.md)。
- 完成[激活P-CSCF选择](激活P-CSCF选择_62638684.md)。

数据

| 类别 | 参数名称 | 取值样例 | 获取方法 | 说明 |
| --- | --- | --- | --- | --- |
| 用户信息 | 国际移动用户识别码（IMSI） | 123000123456789 | 测试终端自带 | - |
| 用户位置信息 | 用户位置信息（TAC） | 0x000001 | 测试终端自带 | - |

## [操作步骤](#ZH-CN_OPI_0262638765)

1. 终端使用位置信息“0x000001”接入网络，创建PDU会话标识为1的PDU会话。
2. 进入 “MML命令行-UNC” 窗口。
3. 查询用户的PDU会话信息。
  **[DSP PDUSESSION](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/接入管理运维/查询PDP上下文信息/显示PDU会话（DSP PDUSESSION）_09653215.md)**
  预期结果：PDU会话信息中的P-CSCF是根据用户位置信息为终端选择的IP地址为"10.130.228.70"的P-CSCF。

  ```
  DSP PDUSESSION:QUERYTYPE=IMSI, IMSI="123000123456789", DSPINFOTYPE=DETAILED, WLNETWKTYPE=NW5G, PDUSESSIONID=1, QFI=1;
  RETCODE = 0  操作成功  

  结果如下 
  --------------  
              国际移动用户标识 = 123000123456789
                              ...
          当前使用的P-CSCF地址 = 10.130.228.70
                              ...
  (结果个数 = 1) 
  ---    END
  ```

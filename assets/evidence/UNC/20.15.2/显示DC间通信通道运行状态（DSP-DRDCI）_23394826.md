## [命令功能](#ZH-CN_MMLREF_0000001323394826)

该命令用于查询DC间通信通道运行状态。

## [注意事项](#ZH-CN_MMLREF_0000001323394826)

- 该命令只用于在UEG-L/UEN网元采用主备（冷备）容灾模式下执行。
- 该命令只用于在UEG-M/UEG网元采用主备（热备）容灾模式下执行。

#### [操作用户权限](#ZH-CN_MMLREF_0000001323394826)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001323394826)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| DRGROUPID | 容灾组标识 | 可选必选说明：可选参数<br>参数含义：该参数用于指定容灾组标识。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是1~8。<br>默认值：无<br>配置原则：<br>可使用<br>[**LST DRGROUPINFO**](查询容灾组信息（LST DRGROUPINFO）_74835153.md)<br>返回的DRGROUPID作为参数输入。 |

## [使用实例](#ZH-CN_MMLREF_0000001323394826)

查询DC间通信通道运行状态。

```
%%DSP DRDCI:;%%
RETCODE = 0  操作成功
结果如下
--------
容灾组标识      隧道组索引     DC间通信通道状态
1               1              主备容灾实例之间握手成功，处于连接状态                                                              
1               2              主备容灾实例之间握手失败，处于断连状态                                                                                                                         
(结果个数 = 2)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001323394826)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 容灾组标识 | 该参数用于指定容灾组标识。 |
| 隧道组索引 | 该参数用于指定隧道组索引。 |
| DC间通信通道状态 | 该参数用于指定DC间通信通道状态。<br>若输出结果为NULL，可使用工程命令DSP DRMODE查询当前环境的容灾模式。当DRMODE为ACTIVE-STANDBY时，表示当前处于初始态；当DRMODE为LOADBALANCE或NULL时，表示未使能容灾功能。<br>取值说明：<br>- NULL（未使能容灾功能或初始态，其状态无效。）<br>- CONNECTED（主备容灾实例之间握手成功，处于连接状态。）<br>- DISCONNECTED（主备容灾实例之间握手失败，处于断连状态。） |

## [命令功能](#ZH-CN_MMLREF_0000001565233886)

**适用NF：SGSN、MME、AMF**

该命令用于查询用户黑名单接入限制功能开关以及拒绝原因值。

## [注意事项](#ZH-CN_MMLREF_0000001565233886)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001565233886)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001565233886)

无

## [使用实例](#ZH-CN_MMLREF_0000001565233886)

查询用户黑名单接入限制功能记录，执行如下命令：

```
%%LST USRBLACKLSTFUN:;%%
RETCODE = 0  操作成功。

查询结果如下
-------------------------
        接入限制开关  =  是
    N2模式拒绝原因值  =  15
	S1模式拒绝原因值  =  15
	IU模式拒绝原因值  =  7
	GB模式拒绝原因值  =  14
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001565233886)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 接入限制开关 | 该参数用于控制系统是否支持用户黑名单接入限制功能。<br>黑名单接入限制的用户列表通过ADD USRBLACKLST命令配置。 |
| N2模式拒绝原因值 | 该参数用于设置用户由于黑名单接入限制导致流程失败时，下发给UE的原因值。 |
| S1模式拒绝原因值 | 该参数用于设置用户由于黑名单接入限制导致流程失败时，下发给UE的原因值。 |
| IU模式拒绝原因值 | 该参数用于设置用户由于黑名单接入限制导致流程失败时，下发给UE的原因值。 |
| GB模式拒绝原因值 | 该参数用于设置用户由于黑名单接入限制导致流程失败时，下发给UE的原因值。 |

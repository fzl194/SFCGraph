## [任务示例](#ZH-CN_OPI_0000001182619543)

任务描述

针对IMSI前缀为123456的用户开通访问某本地专网业务，当用户进入专网业务区域范围进行专网业务（DNN=dnn-ar）时，网络针对用户请求DNN为Huawei.com的PDU会话进行会话重建。

脚本

//配置专网业务适用的用户群。

```
ADD NGUSRGRP: USRGRPID=20, DESC="for roaming user";
ADD NGUSRGRPMEM: USRGRPID=20, IMSIPRE="123456", DESC="for roaming user";
```

//配置用户群的AM策略。

```
ADD AMUEPLCYCTRL: SUBRANGE=IMSI_PREFIX, IMSIPRE="123456", ISUEASSOC=YES;
```

//配置AM-PCF的选择策略。

```
ADD PCFSELPLCY: SUBRANGE=USER_GROUP, USRGRPID=20,CTRLMODE=WHITELIST, SSSW=YES, SERVINGSCOPE="YES", DESC="for roaming user";
```

//配置允许进行专网业务的DNN。

```
ADD ALLOWDNN: REQDNN="Huawei.com",REPLDNN="dnn-ar";
```

## [操作场景](#ZH-CN_OPI_0276026862)

在部署NB-IoT业务时，为实现 PCRF免升级 即可支持NB-IoT用户接入，需要开启该功能。

> **说明**
> 适用于PGW-C。

## [必备事项](#ZH-CN_OPI_0276026862)

前提条件

- 请仔细阅读 [WSFD-215302 PCRF免升级特性概述](特性概述_75804173.md) 。
- 完成加载License。

数据

| 类别 | 参数 | 取值样例 | 获取方法 | 说明 |
| --- | --- | --- | --- | --- |
| [**SET NBIOTRATVALUE**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/计费管理/计费控制/NB-IoT用户RAT值/设置NB-IoT用户的RAT值（SET NBIOTRATVALUE）_09896820.md) | 和PCRF交互使用的RAT值（PCRF） | EUTRAN | 本端规划 | PCRF免升级功能 |

## [操作步骤](#ZH-CN_OPI_0276026862)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. 配置开启PCRF免升级功能。
  [**SET NBIOTRATVALUE**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/计费管理/计费控制/NB-IoT用户RAT值/设置NB-IoT用户的RAT值（SET NBIOTRATVALUE）_09896820.md)

## [任务示例](#ZH-CN_OPI_0276026862)

任务描述

在 UNC 上配置 PCRF免升级 功能，配置 UNC 给PCRF发送消息时，RAT信元携带的值为EUTRAN。

脚本

```
SET LICENSESWITCH:LICITEM="LKV3WNPUNN11",SWITCH=ENABLE;
```

```
SET NBIOTRATVALUE: PCRF=EUTRAN;
```

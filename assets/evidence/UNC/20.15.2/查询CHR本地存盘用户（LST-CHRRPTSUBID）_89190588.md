## [命令功能](#ZH-CN_MMLREF_0000001089190588)

**适用NF：SMF、PGW-C、SGW-C、GGSN**

该命令用于显示所有指定的本地存储CHR表单的用户。

## [注意事项](#ZH-CN_MMLREF_0000001089190588)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001089190588)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001089190588)

无

## [使用实例](#ZH-CN_MMLREF_0000001089190588)

显示IMSI指定用户本地存储CHR表单的信息：

```
LST CHRRPTSUBID:;

RETCODE = 0  操作成功。

CHR本地存盘用户信息
-------------------
                               IMSI  =  122222222222222
                 流程模板索引  =  1
(结果个数 = 1)
---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001089190588)

| 输出项名称 | 输出项解释 |
| --- | --- |
| IMSI | 该参数用于指示用户的IMSI。 |
| 流程模板索引 | 该参数用于指示流程控制模板索引。 |

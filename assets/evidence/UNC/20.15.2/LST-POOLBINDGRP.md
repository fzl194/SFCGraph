## [命令功能](#ZH-CN_MMLREF_0232232819)

**适用NF：PGW-C、SMF、GGSN**

该命令用于查询地址池与地址池组的绑定关系。

## [注意事项](#ZH-CN_MMLREF_0232232819)

无

#### [操作用户权限](#ZH-CN_MMLREF_0232232819)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0232232819)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| POOLGRPNAME | 地址池组名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定已配置的地址池组的名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~79。<br>默认值：无<br>配置原则：<br>该参数使用ADD ADDRPOOLGRP命令配置生成。 |

## [使用实例](#ZH-CN_MMLREF_0232232819)

查询地址池与地址池组的绑定关系：

```
LST POOLBINDGRP:;

结果如下
--------
地址池组名称  =  spoolgrp1
  地址池名称  =  spool1
地址池优先级  =  15
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0232232819)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 地址池组名称 | 该参数用于指定已配置的地址池组的名称。 |
| 地址池名称 | 该参数用于指定已配置的地址池的名称。 |
| 地址池优先级 | 该参数用于指定地址池优先级。生效范围为地址池组内相同VPN的地址池。优先级数值越小，优先级越高。对于配置优先级相同的地址池，选择地址池的顺序不做要求。 |

#### [命令功能](#ZH-CN_CONCEPT_0259037219)

该命令用于显示虚拟端口的信息。用户可使用本命令查看资源的端口的网络平面、MAC地址、运行状态等信息。

#### [注意事项](#ZH-CN_CONCEPT_0259037219)

该命令执行后显示虚拟端口信息，而不是虚拟子端口信息。

#### [操作用户权限](#ZH-CN_CONCEPT_0259037219)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0259037219)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| VPORTNAME | 端口名称 | 可选必选说明：可选参数<br>参数含义：该参数表示虚拟端口名称，例如Ethernet64/0/3，其中64代表逻辑资源编号（虚拟机场景下也代表逻辑虚拟机编号），0为默认值，3代表端口编号。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，区分大小写。<br>默认值：无 |
| VMNAME | 资源名称 | 可选必选说明：可选参数<br>参数含义：该参数表示资源名称，指节点或容器的名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，区分大小写。<br>默认值：无<br>配置原则：无。 |

#### [输出结果说明](#ZH-CN_CONCEPT_0259037219)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 端口名称 | 表示虚拟端口名称，例如Ethernet64/0/3，其中64代表逻辑资源编号（虚拟机场景下也代表逻辑虚拟机编号），0为默认值，3代表端口编号。 |
| 资源名称 | 表示资源名称，指节点或容器的名称。 |
| 是否绑定PAE | 该参数表示是否绑定PAE。<br>- 是。<br>- 否。 |
| 是否对外端口 | 该参数表示是否对外端口。<br>- 是。<br>- 否。 |
| 网络平面 | 该参数表示网络平面。<br>- 基础业务端口。<br>- 内部管理端口。<br>- 外部管理端口。<br>- 外部连接端口。<br>- Fabric平面子接口。<br>- PAE间通信端口。<br>- 外部linux端口。<br>- NLS外部管理端口。<br>- 主备仲裁扩展端口。<br>- 转发拓扑逻辑端口。<br>- 转发集成逻辑端口。<br>- 内部逻辑服务端口。<br>- 第二基础业务端口。<br>- 第二内部逻辑服务端口。<br>- 内联控制口。<br>- tunnel口。<br>- VNFM服务端口。 |
| 带宽类型 | 表示带宽类型，目前只支持吉比特以太网。 |
| MAC地址 | 表示虚拟端口的MAC地址。 |
| 运行状态 | 该参数表示虚拟端口的运行状态。<br>- 停止：虚拟端口为关闭状态。<br>- 运行：虚拟端口为打开状态。<br>- 管理停止：用户主动设置端口为关闭状态。 |
| 上次连接时间 | 表示上次连接时间。显示为NULL时表示端口未曾打开过。 |
| 上次断开时间 | 表示上次断开时间，显示为NULL时表示端口未曾关闭过。 |
| 虚拟端口在位状态 | 该参数表示虚拟端口在位状态。<br>- 不在位：资源不注册或者资源注册但资源上不存在该端口会显示为不在位状态。<br>- 在位：资源注册同时资源上存在该端口会显示为在位状态。 |

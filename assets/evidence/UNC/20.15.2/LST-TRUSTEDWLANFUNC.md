## [命令功能](#ZH-CN_MMLREF_0000002214117605)

**适用NF：PGW-C、SMF**

该命令用于查询可信Non-3GPP接入开关。

## [注意事项](#ZH-CN_MMLREF_0000002214117605)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000002214117605)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000002214117605)

无

## [使用实例](#ZH-CN_MMLREF_0000002214117605)

查询可信Non-3GPP接入开关：

```
%%LST TRUSTEDWLANFUNC:;%%
RETCODE = 0  操作成功

结果如下
--------
可信Non-3GPP接入开关  =  不使能
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000002214117605)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 可信Non-3GPP接入开关 | 本参数用于控制是否使能可信Non-3GPP接入功能。 |

## [命令功能](#ZH-CN_MMLREF_0209654437)

**适用NF：SMF**

该命令用于查询NF实例的TAI信息。

## [注意事项](#ZH-CN_MMLREF_0209654437)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209654437)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209654437)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| NFINSTANCENAME | NF实例名称 | 可选必选说明：可选参数<br>参数含义：本参数用于指定对应的NF实例名称。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是0~50。<br>默认值：无<br>配置原则：<br>本参数需要与ADD NFUUID命令中的NFINSTANCENAME值保持一致。 |

## [使用实例](#ZH-CN_MMLREF_0209654437)

运营商A需要查询标识为SMF_Instance_0的NF实例支持的TAI。

```
%%LST NFTAI:;%%
RETCODE = 0  操作成功

结果如下
--------
        NF实例名称  =  SMF_Instance_0
      移动国家代码  =  460
          移动网号  =  01
        跟踪区域码  =  000001
  绑定的SMFINFO ID  =  null
绑定的NWDAFINFO ID  =  null
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209654437)

| 输出项名称 | 输出项解释 |
| --- | --- |
| NF实例名称 | 本参数用于指定对应的NF实例名称。 |
| 移动国家代码 | 本参数用于指定TAI的移动国家代码。 |
| 移动网号 | 本参数用于指定TAI的移动网号。 |
| 跟踪区域码 | 本参数用于指定跟踪区域码。 |
| 绑定的SMFINFO ID | 该参数用于指定绑定的SMFINFOEXT记录。 |
| 绑定的NWDAFINFO ID | 该参数用于指定绑定的NWDAFINFOEXT记录。 |

## [操作场景](#ZH-CN_OPI_0227675786)

华为 UNC 支持DCNR用户使用扩展QoS接入基本功能，可以保证签约的DCNR用户扩展QoS接入网络并使用业务。

> **说明**
> 适用于SGW-C、PGW-C。

## [操作步骤](#ZH-CN_OPI_0227675786)

1. 进入 “MML命令行-UNC” 窗口。
2. 配置PGW-CDR/SGW-CDR容器携带扩展QoS参数。
  [**SET OFCCDRPARA**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/计费管理/离线计费/离线计费基础参数/离线公共参数/配置离线计费话单参数（SET OFCCDRPARA）_09896905.md)
3. 设置Diameter字典加载路径。
  [**MOD DIAMDICTPATH**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/计费和策略接口管理/Diameter管理/Diameter字典管理/加载路径/修改Diameter字典加载路径（MOD DIAMDICTPATH）_09897248.md)
  加载Diameter字典。
  [**LOD DIAMDICT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/计费和策略接口管理/Diameter管理/Diameter字典管理/加载字典/加载Diameter字典（LOD DIAMDICT）_09897254.md)
  > **说明**
  > UNC 主机软件加载时，会自动加载Diameter控制文件。如果文件不存在，或者文件加载失败，也可以通过命令 [**LOD DIAMDICT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/计费和策略接口管理/Diameter管理/Diameter字典管理/加载字典/加载Diameter字典（LOD DIAMDICT）_09897254.md) 手工加载Diameter控制文件。
4. 配置PCRF的域信息，并使能动态协商参数及扩展QoS功能。
  [**ADD PCRF**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/PCC管理/PCRF Diameter连接/PCRF/增加PCRF（ADD PCRF）_09897101.md)
5. 基于IMSI/MSISDN号段或全局配置Realm，并使能动态协商参数及扩展QoS功能。
  [**ADD GLBDIAMREALM**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/计费和策略接口管理/Diameter管理/Diameter Realm/全局Realm/增加全局Diameter域（ADD GLBDIAMREALM）_09897280.md)
6. 配置APN对应的Diameter域信息，并使能动态协商参数及扩展QoS功能。
  [**ADD REALMBINDAPN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/计费和策略接口管理/Diameter管理/Diameter Realm/Realm绑定APN/增加APN与Diameter Realm关联关系（ADD REALMBINDAPN）_09897285.md)

## [任务示例](#ZH-CN_OPI_0227675786)

任务描述

激活NSA 用户QoS管理功能。

脚本

//配置PGW-CDR/SGW-CDR容器携带扩展QoS参数。

```
SET OFCCDRPARA:SGWLOTVQOSEXT=ENABLE,PGWLOTVQOSEXT=ENABLE,PGWLOSDQOSEXT=ENABLE;
```

//设置Diameter字典加载路径，加载Diameter字典。

```
MOD DIAMDICTPATH:APPLICATION=GX,DICTNO=1,DICTPATH=CUSTOM1;
```

```
LOD DIAMDICT:DICTFILESELECT=ALL;
```

//配置PCRF的域信息，并使能动态协商参数及扩展QoS功能。

```
ADD PCRF:HOSTNAME="pcrf1",REALMNAME="pcrf.huawei.com",FEATURELIST=EXTENDED_BW_NR-1&RELEASE8-1,SUPFEANEGOSW=ENABLE;
```

//基于IMSI/MSISDN号段或全局配置Realm，并使能动态协商参数及扩展QoS功能。

```
ADD GLBDIAMREALM:APPLICATION=GX,CONSTBYIMSISW=DISABLE,REALM="pcrf.huawei.com",SUPFTNEGOSW=ENABLE,FEATURELIST=EXTENDED_BW_NR-1&RELEASE8-1;
```

//配置APN对应的Diameter域信息，并使能动态协商参数及扩展QoS功能。

```
ADD REALMBINDAPN:APN="isp",APPLICATION=GX,CONSTBYIMSISW=DISABLE,REALMNAME="pcrf.huawei.com",SUPFTNEGOSW=ENABLE,FEATURELIST=EXTENDED_BW_NR-1&RELEASE8-1;
```

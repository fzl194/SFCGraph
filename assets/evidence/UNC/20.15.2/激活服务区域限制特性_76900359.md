## [操作步骤](#ZH-CN_OPI_0276900359)

1. 进入 “MML命令行-UNC” 窗口。
2. 配置服务区域限制的功能开关和非标开关。
  [**SET NGMMFUNC**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/移动性管理/MM协议参数管理/5G移动性管理/设置5G移动性管理功能（SET NGMMFUNC）_09653748.md)
3. 配置由于服务区域限制而拒绝UE的业务流程（如服务请求、PDU会话建立）时，下发给UE的原因值。
  [**SET NGMMPROCTRL**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/移动性管理/MM流程管理/5G移动性管理流程控制参数/设置5G移动性管理流程控制参数（SET NGMMPROCTRL）_09652386.md)
4. （可选）配置某个区域信息，以及AreaCode与TAI之间的映射关系。
    - 配置区域编码
      [**ADD AREACODE**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/通用配置管理/通用区域编码管理/增加区域编码（ADD AREACODE）_44006351.md)
    - 为指定的区域增加跟踪区成员
      [**ADD AREAMEM**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/通用配置管理/通用区域成员管理/增加跟踪区成员（ADD AREAMEM）_44006352.md)
5. （可选）配置本地服务区域限制。
  **[**ADD SRVAREARES**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/移动性管理/接入限制/AMF本地服务区域限制/增加本地服务区域限制（ADD SRVAREARES）_58735671.md)**

## [任务示例](#ZH-CN_OPI_0276900359)

任务描述

启用服务区域限制的功能开关和非标开关，并配置AREACODE为"MyTown"和TAI之间的映射关系。

脚本

//配置服务区域限制的功能开关和非标开关。

```
SET NGMMFUNC：SARSW=YES, SARNSSW=YES;
```

//配置由于服务区域限制而拒绝UE的业务流程的原因值。

```
SET NGMMPROCTRL: PROT=OTHER_PROC, SARST=0;
```

//（可选）配置某个区域信息，以及AreaCode与TAI之间的映射关系。

```
ADD AREACODE：AREACODE="MyTown";
ADD AREAMEM：AREACODE="MyTown", MCC="123", MNC="456", BGNTAC="010101", ENDTAC="010102";
```

//（可选）配置本地服务区域限制。

```
ADD SRVAREARES: SUBRANGE=ALL_USER, SARPRIORITY=CFG_FIRST, AREACODE="SomeCity",RESTYPE=NOT_ALLOWED_AREAS;
```

## [命令功能](#ZH-CN_MMLREF_0273321234)

**适用NF：PGW-C、SMF**

该命令用于查询M2M服务器组配置。

## [注意事项](#ZH-CN_MMLREF_0273321234)

无

#### [操作用户权限](#ZH-CN_MMLREF_0273321234)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0273321234)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| GROUPNAME | M2M服务器组名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定M2M服务器组的名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~31。字符串类型，输入长度范围为1～31。不区分大小写，不支持空格。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0273321234)

显示组名为m2msrvgroup01的M2M服务器组配置信息：

```
%%LST M2MSERVERGRP: GROUPNAME="m2msrvgroup01";%%
RETCODE = 0  操作成功

M2M服务器组信息：
-----------------
    M2M服务器组名称  =  m2msrvgroup01
M2M服务器IP地址类型  =  IPv4
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0273321234)

| 输出项名称 | 输出项解释 |
| --- | --- |
| M2M服务器组名称 | 该参数用于指定M2M服务器组的名称。 |
| M2M服务器IP地址类型 | 该参数用于指定M2M服务器的IP地址类型。 |

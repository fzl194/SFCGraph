## [操作场景](#ZH-CN_OPI_0193971024)

本操作指导介绍在运行网络中调测支持基于终端能力选择用户面特性的操作过程。

> **说明**
> 适用于MME。

## [必备事项](#ZH-CN_OPI_0193971024)

前提条件

- 请仔细阅读[WSFD-104601 支持基于终端能力选择用户面特性概述](特性概述_93971022.md)。
- 完成[激活支持基于终端能力选择用户面](激活支持基于终端能力选择用户面_93971023.md)。

数据

该操作无需准备数据。

工具

- 测试终端
- OM Portal跟踪

## [操作步骤](#ZH-CN_OPI_0193971024)

在相关网元均完成本特性的配置后，可以采用以下步骤检查特性工作是否正常：

1. 在 UNC 上执行 **[**DSP SMCTX**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/系统管理/用户数据库管理/显示承载上下文(DSP SMCTX)_72226033.md)** ，以显示指定用户的承载上下文信息。
2. 用户发起附着，观察支持DCNR能力的终端用户获取网关过程。
  预期结果： UNC 为支持DCNR能力的终端用户选择支持DCNR的高速网关。
  ```
  %%DSP SMCTX:QUERYOPT=BYIMSI, IMSI="46000", IDTYPE=BYBEARID/NSAPI, CTXID=5;%%
  RETCODE = 0  操作成功

  输出结果如下:
  -------------------------
                          ............

                          S-GW address of control plane(S11) = 10.2.3.11

                          P-GW address of control plane(S5/S8) = 10.2.3.10

                          ............

                          Select a High-Speed Gateway = Yes

                          ............

  (结果个数 = 1)

  ---    结束
  ```

#### [命令功能](#ZH-CN_MMLREF_0000001126305938)

**适用网元：SGSN、MME**

该命令已废弃。

该命令用于显示主备网元之间的容灾业务通道状态和网元的主备状态。

#### [注意事项](#ZH-CN_MMLREF_0000001126305938)

无。

#### [本地用户权限](#ZH-CN_MMLREF_0000001126305938)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126305938)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126305938)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| CHNID | 容灾业务通道ID | 参数含义：该参数用于指定需要显示的业务容灾通道的序号。<br>取值范围：0~0<br>默认值：无。 |

#### [使用实例](#ZH-CN_MMLREF_0000001126305938)

查询主备网元之间的容灾业务通道状态和网元的主备状态。

DSP ASRCHN:;

```
%%DSP ASRCHN:;%%
RETCODE = 0  执行成功

-------------------------
容灾业务通道ID   = 0
通道状态         = 正常
本端网元运行状态 = 正常
对端网元运行状态 = 正常

仍有后续报告输出...
---    END

%%DSP ASRCHN:;%%
RETCODE = 0  执行成功
-------------------------
容灾业务通道本端IP    容灾业务通道本端端口    容灾业务通道对端IP    容灾业务通道对端端口    路径状态
10.10.27.100          18100                   10.10.28.100         18100                   路径正常
10.10.27.100          18100                   10.10.28.101         18100                   路径正常
10.10.27.101          18100                   10.10.28.100         18100                   路径正常
10.10.27.101          18100                   10.10.28.101         18100                   路径正常
(结果个数 = 5)
共有2个报告
---    END
```

#### [输出结果说明](#ZH-CN_MMLREF_0000001126305938)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 容灾业务通道ID | 查询到的容灾业务通道的序号信息 |
| 通道状态 | 查询到的容灾业务通道状态信息 |
| 本端网元运行状态 | 查询到的容灾业务通道的本端网元运行状态 |
| 对端网元运行状态 | 查询到的容灾业务通道的对端网元运行状态 |

| 容灾业务通道本端端口 | 查询到的容灾业务通道的本端端口号 |
| 容灾业务通道对端IP | 查询到的容灾业务通道的对端IP地址 |
| 容灾业务通道对端端口 | 查询到的容灾业务通道的对端端口号 |
| 路径状态 | 查询到的容灾业务通道的路径状态 |

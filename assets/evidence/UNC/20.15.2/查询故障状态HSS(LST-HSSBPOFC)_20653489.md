#### [命令功能](#ZH-CN_CONCEPT_0000001320653489)

**适用网元：MME**

该命令用于查询配置的故障状态HSS配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001320653489)

无。

#### [本地用户权限](#ZH-CN_CONCEPT_0000001320653489)

manage-ug;system-ug;monitor-ug

#### [网管用户权限](#ZH-CN_CONCEPT_0000001320653489)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001320653489)

无

#### [使用实例](#ZH-CN_CONCEPT_0000001320653489)

查询已经配置的所有故障状态HSS数据。

```
%%LST HSSBPOFC:;%% 
RETCODE = 0  操作成功 
The result is as follows 
------------------------ 
HOSTNAME  =  huawei 
(结果个数 = 1)  
---    END
```

## [命令功能](#ZH-CN_MMLREF_0209652356)

**适用NF：AMF、SMF、NSSF、SMSF、NCG**

该命令用于查询目标NF实例的配置。

## [注意事项](#ZH-CN_MMLREF_0209652356)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209652356)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652356)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| TNFTYPE | 目标NF类型 | 可选必选说明：可选参数<br>参数含义：本参数用于指定目标NF类型。<br>数据来源：全网规划<br>取值范围：<br>- “CHF（CHF）”：CHF<br>- “UPF（UPF）”：UPF<br>- “PCSCF（PCSCF）”：PCSCF<br>- “PCF（PCF）”：PCF<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209652356)

运营商A需要查询配置的所有目标NF实例。

```
%%LST TNFINS:;%%
RETCODE = 0  操作成功

结果如下
--------
    目标NF类型  =  CHF
目标NF实例索引  =  1
目标NF实例名称  =  target_chf_0
      服务名称  =  CHF提供的Nchf_Converged_Charging服务
  服务实例标识  =  service_instance_0
      协议模式  =  http
          域名  =  NULL
    NF描述名称  =  NULL
(结果个数 = 1)

---    END
```

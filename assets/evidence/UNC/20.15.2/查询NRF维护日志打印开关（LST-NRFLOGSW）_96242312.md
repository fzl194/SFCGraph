## [命令功能](#ZH-CN_MMLREF_0296242312)

**适用NF：NRF**

该命令用于查询NRF的维护日志打印开关的值。

## [注意事项](#ZH-CN_MMLREF_0296242312)

无

#### [操作用户权限](#ZH-CN_MMLREF_0296242312)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0296242312)

无

## [使用实例](#ZH-CN_MMLREF_0296242312)

当维护人员希望查看当前NRF维护日志打印开关的值时，执行如下命令：

```
%%LST NRFLOGSW:;%%
RETCODE = 0  操作成功

结果如下
--------
       服务发现日志打印开关  =  打开
         NF检索日志打印开关  =  关闭
   NF注册和更新日志打印开关  =  打开
         NF订阅日志打印开关  =  关闭
           通知日志打印开关  =  关闭
服务发现日志打印速率(次/秒)  =  5
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0296242312)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 服务发现日志打印开关 | 该参数用于控制服务发现维护日志打印。开关打开时打印维护日志，开关关闭不打印维护日志。 |
| NF检索日志打印开关 | 该参数用于控制NF检索维护日志打印。开关打开时打印维护日志，开关关闭不打印维护日志。 |
| NF注册和更新日志打印开关 | 该参数用于控制NF注册和更新维护日志打印。开关打开时打印维护日志，开关关闭不打印维护日志。 |
| NF订阅日志打印开关 | 该参数用于控制NF订阅维护日志打印。开关打开时打印维护日志，开关关闭不打印维护日志。 |
| 通知日志打印开关 | 该参数用于控制状态通知维护日志打印。开关打开时打印维护日志，开关关闭不打印维护日志。 |
| 服务发现日志打印速率(次/秒) | 该参数表示服务发现流程中，单进程每秒可以打印的最大维护日志次数，1秒内对于超出该参数值的服务发现维护日志将不会打印。 |

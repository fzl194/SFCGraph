#### [命令功能](#ZH-CN_CONCEPT_0000201616858407)

**适用NF：SGW-C、PGW-C、SMF**

该命令用来添加一个新的Gx UPF组。配置Gx UPF组后，可以在该Gx UPF组下绑定UPF的HostName。

#### [注意事项](#ZH-CN_CONCEPT_0000201616858407)

- 该命令执行后立即生效。
- 该命令最大记录数为300。

#### [操作用户权限](#ZH-CN_CONCEPT_0000201616858407)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000201616858407)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| UPFGRPNAME | Gx UPF组名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定UPF组名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～32。不支持空格及特殊字符“#”、“$”和“&”等，由“_”、“-”、数字、大小写字母和“.”组成，不能以“.”开头且不能出现连续两个“.”，不区分大小写。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000201616858407)

配置名称为"upfgrp"的Gx UPF组：

```
ADD GXUPFGROUP: UPFGRPNAME="upfgrp";
```

# 删除SCTP端点（RMV SCTPENDPOINT）

- [注意事项](#ZH-CN_CONCEPT_0209897323__1.3.2.1)

#### [命令功能](#ZH-CN_CONCEPT_0209897323)

**适用NF：PGW-C、SMF**

此命令用于删除SCTP端点信息。

#### [注意事项](#ZH-CN_CONCEPT_0209897323)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0209897323)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0209897323)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| ENDPOINTNAME | 端点名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定SCTP端点名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，不区分大小写。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0209897323)

根据网络规划，需要删除一个SCTP端点，则可以按如下配置：

```
RMV SCTPENDPOINT: ENDPOINTNAME="sctp_ep1";
```

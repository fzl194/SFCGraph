#### [命令功能](#ZH-CN_CONCEPT_0000001292075657)

**适用NF：SMSF**

该命令用于查询配置的VLROPC本局信令点。

#### [注意事项](#ZH-CN_CONCEPT_0000001292075657)

无。

#### [网管用户权限](#ZH-CN_CONCEPT_0000001292075657)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001292075657)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| OPX | VLR信令点索引 | 可选必选说明：可选参数<br>参数含义：该参数用于指定VLR本局信令点索引。<br>数据来源：本端规划<br>取值范围：0~1<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001292075657)

查询已经配置的所有的VLROPC记录。

```
%%LST VLROPC:;%%
RETCODE = 0  操作成功 
The result is as follows 
------------------------ 
VLR信令点索引     网络指示语     本局信令点编码     本局VLR号        本局VLR名称    
0                  国内网         0x17701           861390211101     noname
1                  国际网         0x1               861              noname          
(结果个数 = 2)  
```

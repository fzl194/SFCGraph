# 查询基于用户属性的DDN消息优先级（LST DDNUSERTYPEPRI）

- [命令功能](#ZH-CN_MMLREF_0304284712__1.3.1)
- [注意事项](#ZH-CN_MMLREF_0304284712__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0304284712__1.3.4)
- [使用实例](#ZH-CN_MMLREF_0304284712__1.3.5)
- [输出结果说明](#ZH-CN_MMLREF_0304284712__1.3.6)

## [命令功能](#ZH-CN_MMLREF_0304284712)

**适用NF：SGW-C、SMF**

该命令用于查询本地用户、漫游用户、拜访用户的DDN消息的优先级。

## [注意事项](#ZH-CN_MMLREF_0304284712)

无

#### [操作用户权限](#ZH-CN_MMLREF_0304284712)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0304284712)

无

## [使用实例](#ZH-CN_MMLREF_0304284712)

查询本地用户、漫游用户、拜访用户的DDN消息的优先级：

```
%%LST DDNUSERTYPEPRI:;%%
RETCODE = 0  操作成功

基于用户属性的DDN消息优先级
---------------------------
       本地用户优先级  =  高
       漫游用户优先级  =  低
       拜访用户优先级  =  最低
DDN用户类型优先级开关  =  使能
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0304284712)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 本地用户优先级 | 该参数用于配置本地用户的DDN消息的优先级。 |
| 漫游用户优先级 | 该参数用于配置漫游用户的DDN消息的优先级。 |
| 拜访用户优先级 | 该参数用于配置拜访用户的DDN消息的优先级。 |
| DDN用户类型优先级开关 | 该参数用于设置是否使能基于用户的本地漫游拜访属性识别DDN消息的优先级。 |

## [命令功能](#ZH-CN_MMLREF_0000001582861629)

**适用NF：SMF**

该命令用于查询全局终端接入的eDRX模式。

## [注意事项](#ZH-CN_MMLREF_0000001582861629)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001582861629)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001582861629)

无

## [使用实例](#ZH-CN_MMLREF_0000001582861629)

假如用户需要对全局终端接入的eDRX模式进行查询，则使用该实例：

```
%%LST GLBEDRXATTR:;%%
RETCODE = 0  操作成功

结果如下
------------------------
              支持eDRX模式开关  =  使能
        下行包缓存数获取优先级  =  从DLBUFFPKTCNT获取
                  下行包缓存数  =  10
            下行包缓存额外时长  =  10
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001582861629)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 支持eDRX模式开关 | 该参数用于指定是否使能NR和NR RedCap终端接入时的eDRX模式。 |
| 下行包缓存数获取优先级 | 该参数用于指定下行包缓存数获取优先级。 |
| 下行包缓存数 | 该参数用于指定下行包缓存数。 |
| 下行包缓存额外时长 | 该参数用于指定SMF在AMF下发的下行包缓存时长的基础上额外增加的时长。 |

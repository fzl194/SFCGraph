## [操作场景](#ZH-CN_OPI_0168358227)

本操作指导介绍在运行网络中激活别名APN特性的操作过程。

别名APN是为了网络业务扩展需要，将一个特定的APN映射为多个不同的别名APN，或为了网络业务合并的需要，将多个特定的APN映射为一个指定的别名APN，使用别名APN替代协商的APN发起DNS解析，以便屏蔽用户APN的差异以及降低用户关于APN的配置要求。

> **说明**
> 适用于 SGSN、 MME。

## [操作步骤](#ZH-CN_OPI_0168358227)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. 添加别名映射表的记录。
  > **说明**
  > 根据实际情况选择添加别名映射表记录。
  [**ADD ALIASAPN**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/会话管理/别名APN管理/增加别名APN配置(ADD ALIASAPN)_72345281.md)

## [任务示例](#ZH-CN_OPI_0168358227)

任务描述

添加别名APN映射表的记录，将APN(example.com)映射为别名APN（huawei.com）。

脚本

//设置别名APN License开关为“ENABLE”。

```
SET LICENSESWITCH: LICITEM="LKV2ALIASAPN02", SWITCH=ENABLE;
```

//添加别名APN映射表的记录，将APN(example.com)映射为别名APN（huawei.com）。

```
ADD ALIASAPN: SUBRANGE=IMSI_PREFIX, IMSIPRE="12345", OLDAPN="example.com", NEWAPN="huawei.com";
```

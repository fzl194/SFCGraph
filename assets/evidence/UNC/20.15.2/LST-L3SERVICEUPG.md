## [命令功能](#ZH-CN_MMLREF_0000001188502328)

灰度升级中，执行此命令，用于查询服务升级进度。

## [注意事项](#ZH-CN_MMLREF_0000001188502328)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001188502328)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001188502328)

无

## [使用实例](#ZH-CN_MMLREF_0000001188502328)

查询服务AM的升级进度：

```
%%LST L3SERVICEUPG:;%%
RETCODE = 0  操作成功

结果如下
--------
L3Service  =  AM
 Position  =  Service
     Mode  =  Gray
   Status  =  InUpgrade
 Progress  =  20
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001188502328)

| 输出项名称 | 输出项解释 |
| --- | --- |
| L3Service | 该参数用于标识一个微服务。 |
| Position | 该参数用于标识位于哪一层。 |
| Mode | 该参数用于标识模式。 |
| Status | 该参数用于标识状态。 |
| Progress | 该参数用于标识进度。 |

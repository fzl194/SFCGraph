## [命令功能](#ZH-CN_MMLREF_0248290737)

**适用NF：SMF**

该命令用于查询UDM的选择策略。

## [注意事项](#ZH-CN_MMLREF_0248290737)

无

#### [操作用户权限](#ZH-CN_MMLREF_0248290737)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0248290737)

无

## [使用实例](#ZH-CN_MMLREF_0248290737)

查询当前配置的UDM选择策略，执行如下命令：

```
%%LST SMFSELUDM:;%%
RETCODE = 0  操作成功

结果如下
------------------------
               用户范围  =  Foreign Subscriber
           是否使用GPSI  =  YES
       是否使用网络切片  =  NO
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0248290737)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 用户范围 | 该参数用于指定应用UDM选择策略的用户范围。 |
| 是否使用GPSI | 该参数用于指定是否使用用户的GPSI作为目标UDM的选择条件。 |
| 是否使用网络切片 | 该参数用于指定是否使用用户的可用网络切片作为目标UDM的选择条件。 |

#### [命令功能](#ZH-CN_MMLREF_0000001172345959)

**适用网元：SGSN、MME**

该命令用于查询USN扩展包状态以及USN与其他VNFC网元扩展包状态同步结果。

#### [注意事项](#ZH-CN_MMLREF_0000001172345959)

当前只实现与CSLB状态同步结果查询

#### [本地用户权限](#ZH-CN_MMLREF_0000001172345959)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172345959)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172345959)

无。

#### [使用实例](#ZH-CN_MMLREF_0000001172345959)

显示扩展包运行状态：

DSP PLGPKGSTATE:;

```
RETCODE = 0  操作成功。

查询结果如下
------------------------
 USN_VNFC扩展包生效状态  =  生效
CSLB_VNFC扩展包生效状态  =  生效
                  同步时间  =   2015-11-19 14:18:46
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_MMLREF_0000001172345959)

| 输出项名称 | 输出项解释 |
| --- | --- |
| USN_VNFC扩展包生效状态 | 表示USN_VNFC当前扩展包的生效状态。 |
| CSLB_VNFC扩展包生效状态 | 表示CSLB_VNFC最后一次同步给USN的扩展包的生效状态。 |
| 同步时间 | 表示最近一次发起扩展包状态同步的时间。 |

#### [命令功能](#ZH-CN_CONCEPT_0209897348)

**适用NF：SGW-C、PGW-C、SMF**

该命令用来查询话单池空间不足告警阈值。

#### [注意事项](#ZH-CN_CONCEPT_0209897348)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0209897348)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0209897348)

无。

#### [使用实例](#ZH-CN_CONCEPT_0209897348)

查询话单池空间不足告警的上报和恢复阈值信息：

```
LST CDRPOOLTHRSH:;
```

```

RETCODE = 0  操作成功。

话单池告警门限
--------------
告警产生阈值（%）  =  80
告警恢复阈值（%）  =  70
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0209897348)

参见SET CDRPOOLTHRSH的参数说明。

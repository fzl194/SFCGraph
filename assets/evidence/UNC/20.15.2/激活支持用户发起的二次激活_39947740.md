## [操作场景](#ZH-CN_OPI_0000001139947740)

本操作指导介绍在运行网络中激活支持终端侧发起二次PDP上下文激活特性的操作过程。

> **说明**
> 适用于GGSN。

## [必备事项](#ZH-CN_OPI_0000001139947740)

前提条件

- 请仔细阅读[WSFD-205104 支持用户发起的二次激活特性概述](WSFD-205104 支持用户发起的二次激活特性概述_39788502.md)
- 完成加载License。

数据

该操作无需准备数据。

## [操作步骤](#ZH-CN_OPI_0000001139947740)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. 打开UE发起二次激活开关。
  [**ADD APN**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/APN管理/APN/增加APN配置（ADD APN）_09653747.md)

## [任务示例](#ZH-CN_OPI_0000001139947740)

任务描述

激活支持终端侧发起二次PDP上下文激活特性。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV3W9SACT11", SWITCH=ENABLE;
```

//打开UE发起二次激活开关

```
ADD APN: APN="huawei.com", USRINTSECACTSW=ENABLE;
```

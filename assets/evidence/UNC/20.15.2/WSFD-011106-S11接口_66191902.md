#### [适用NF](#ZH-CN_TOPIC_0166191902)

MME、SGW-C

#### [定义](#ZH-CN_TOPIC_0166191902)

S11接口是 MME 与S-GW/SGW-C间的信令面接口，主要用于MME和S-GW/SGW-C之间传递承载创建、更新及删除请求消息，以及UE处于ECM-IDLE态时，用于S-GW/SGW-C通知MME寻呼UE，恢复S1承载。

S11-U接口是MME与S-GW/SGW-U间的用户面接口，用于传递MME与S-GW/SGW-U间的上下行用户面数据，实现物联网业务。

- GTP-C（GPRS Tunnelling Protocol for the control plane）：利用隧道协议在骨干网的GSN间传输信令。主要信令包括路径管理、隧道管理、移动管理、位置管理，实现骨干网通道维护、会话建立/修改/删除、用户信息交互。
- GTP-U（GPRS Tunnelling Protocol for the user plane）：利用隧道协议在骨干网的GSN间传输用户数据，使用隧道协议传输数据，屏蔽上层协议影响，同时提供会话级的双向快速传输通道。GTP协议用户面除数据包外，还包括数据传输出错指示、Echo消息、支持的扩展头列表等用户面传输辅助信令。
- GTP-C版本为GTPv2。
- GTP-U版本为GTPv1。

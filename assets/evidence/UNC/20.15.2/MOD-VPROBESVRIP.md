# 修改vProbe报表服务器的接入点IP地址（MOD VPROBESVRIP）

## [命令功能](#ZH-CN_MMLREF_0000001439402653)

该命令用于修改vProbe报表服务器的接入点IP地址。

## [注意事项](#ZH-CN_MMLREF_0000001439402653)

- 该命令执行后立即生效。

- 执行命令前请确认vProbe服务处于上线状态，可通过DSP FUNCTIONSETINFO命令查询确认。
- IP和端口必须同时配置。
- 当前版本不支持参数“协议类型”取值为“UDP(UDP)”。
- 当前版本不支持参数“服务器认证方式”取值为“PUBLICKEY(公钥认证)”。

#### [操作用户权限](#ZH-CN_MMLREF_0000001439402653)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001439402653)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| ACCESSNAME | 接入点名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定vProbe报表服务器的接入点名称。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是0~32。<br>默认值：无<br>配置原则：无 |
| VPROBESVRNAME | 报表服务器名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定vProbe报表服务器名称。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是0~32。<br>默认值：无<br>配置原则：无 |
| IPTYPE | IP类型 | 可选必选说明：可选参数<br>参数含义：该参数用于指定vProbe报表服务器的IP类型。<br>数据来源：全网规划<br>取值范围：<br>- IPv4（IPv4）<br>- IPv6（IPv6）<br>默认值：无<br>配置原则：无 |
| IPV6 | 报表服务器IPv6地址 | 可选必选说明：该参数在"IPTYPE"配置为"IPv6"时为条件必选参数。<br>参数含义：该参数用于指定vProbe报表服务器的IPv6地址。<br>数据来源：全网规划<br>取值范围：IPv6地址类型。<br>默认值：无<br>配置原则：<br>IPv6地址必须是全球单播地址，不能为::、FFFF:FFFF:FFFF:FFFF:FFFF:FFFF:FFFF:FFFF、环回地址(::1)、链路本地地址(FE80::/10)和组播地址(FF00::/8)。<br>IPv6地址不能和vProbe本地IP地址资源池中的IPv6地址在同一网段。 |
| IPV4 | 报表服务器IPv4地址 | 可选必选说明：该参数在"IPTYPE"配置为"IPv4"时为条件必选参数。<br>参数含义：该参数用于指定vProbe报表服务器的IPv4地址。<br>数据来源：全网规划<br>取值范围：IPv4地址类型。<br>默认值：无<br>配置原则：<br>IPv4地址必须是A、B或者C类地址，不能为环回地址（127.x.y.z）、组播地址（240.x.y.z）或（255.0.0.0）。<br>IPv4地址不能和vProbe本地IP地址资源池中的IPv4地址在同一网段。 |
| SVRPORT | 报表服务器端口号 | 可选必选说明：可选参数<br>参数含义：该参数用于指定vProbe报表服务器端口号。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是0~65534。<br>默认值：无<br>配置原则：无 |
| PROTOCOLTYPE | 协议类型 | 可选必选说明：可选参数<br>参数含义：该参数用于指定协议类型。<br>数据来源：全网规划<br>取值范围：<br>- TCP（TCP）<br>- SFTP（SFTP）<br>- UDP（UDP）<br>默认值：无<br>配置原则：无 |
| USER | 报表服务器用户名 | 可选必选说明：该参数在"PROTOCOLTYPE"配置为"SFTP"时为条件必选参数。<br>参数含义：该参数用于指定vProbe报表服务器用户名。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是0~64。<br>默认值：无<br>配置原则：无 |
| PWD | 报表服务器密码 | 可选必选说明：该参数在"PROTOCOLTYPE"配置为"SFTP"时为条件必选参数。<br>参数含义：该参数用于指定vProbe报表服务器密码。<br>数据来源：全网规划<br>取值范围：Pwd，取值范围是1~432。<br>默认值：无<br>配置原则：无 |
| AUTHMODE | 报表服务器认证方式 | 可选必选说明：该参数在"PROTOCOLTYPE"配置为"SFTP"时为条件必选参数。<br>参数含义：该参数用于指定vProbe报表服务器认证方式。<br>数据来源：本端规划<br>取值范围：<br>- “NOAUTH（无认证）”：在密钥协商阶段服务器认证方式不使用公钥认证，使用用户名密码认证。<br>- “PUBLICKEY（公钥认证）”：在密钥协商阶段服务器认证方式使用密钥认证。<br>默认值：无<br>配置原则：无 |
| SVRPATH | 报表服务器上传路径 | 可选必选说明：该参数在"PROTOCOLTYPE"配置为"SFTP"时为条件必选参数。<br>参数含义：该参数用于指定vProbe报表服务器上传路径。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是0~512。<br>默认值：无<br>配置原则：<br>服务器上传路径不能包含特殊字符\|;&$<>，且不能包含点点（..）。 |

## [使用实例](#ZH-CN_MMLREF_0000001439402653)

运营商A因IP规划发生变更，将1号VPROBE报表服务器（VPROBESVR1）对应的接入点1（ACCESS1）对应的IP地址由10.180.211.254更改为10.180.211.253，端口号继续保持为默认端口10500：

```
MOD VPROBESVRIP: ACCESSNAME="ACCESS1", VPROBESVRNAME="VPROBESVR1", IPTYPE=IPv4, IPV4="10.180.211.253", SVRPORT=10500;
```

## [命令功能](#ZH-CN_MMLREF_0000001432892345)

**适用NF：SMF**

该命令用于查询MB-SMF组播广播会话上下文的相关信息。

## [注意事项](#ZH-CN_MMLREF_0000001432892345)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001432892345)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001432892345)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| QUERYTYPE | 查询方式 | 可选必选说明：必选参数<br>参数含义：该参数用于指定查询方式。<br>数据来源：本端规划<br>取值范围：<br>- TMGI（TMGI）<br>默认值：无<br>配置原则：无 |
| MCC | 移动国家代码 | 可选必选说明：该参数在"QUERYTYPE"配置为"TMGI"时为条件必选参数。<br>参数含义：该参数用于指定移动国家代码。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度是3。仅支持0-9的数字。<br>默认值：无<br>配置原则：无 |
| MNC | 移动网号 | 可选必选说明：该参数在"QUERYTYPE"配置为"TMGI"时为条件必选参数。<br>参数含义：该参数用于指定移动网号。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是2~3。仅支持0-9的数字。<br>默认值：无<br>配置原则：无 |
| MBSSERVICEID | Mbs Service ID | 可选必选说明：该参数在"QUERYTYPE"配置为"TMGI"时为条件必选参数。<br>参数含义：该参数用于指定Mbs Service ID。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度是6。<br>默认值：无<br>配置原则：<br>Mbs Service ID编码为16进制数，按照字符串格式输入，字符串长度为6，只能由数字（0-9），字母（A-F、a-f）组成。 |

## [使用实例](#ZH-CN_MMLREF_0000001432892345)

当需要查询MCC为123，MNC为03，MBSSERVICEID为000001的TMGI广播会话上下文信息时，执行如下命令：

```
%%DSP SMFMBSSESSION: QUERYTYPE=TMGI, MCC="123", MNC="03", MBSSERVICEID="000001";%%
RETCODE = 0  操作成功
```

## [输出结果说明](#ZH-CN_MMLREF_0000001432892345)

| 输出项名称 | 输出项解释 |
| --- | --- |
| TMGI | 该参数用于标识广播会话的TMGI，其组成格式为MCC-MNC-MBSSERVICEID。 |
| 会话激活时间 | 该参数用于表示会话激活时系统上的时间信息，可通过LST TZ查看系统的时区信息。 |
| QoSFlow标识 | 该参数用于表示QoS流ID。 |
| 5QI的Qos类别标识 | 该参数用于表示5QI。 |
| QoSFlow 5QI优先级 | 该参数用于表示QoS流的5QI优先级。 |
| QoS ARP优先级 | 该参数用于表示QoS流ARP优先级。 |
| QoS ARP抢占能力 | 该参数用于表示QoS流ARP抢占能力。 |
| QoS ARP被抢占能力 | 该参数用于表示QoS流ARP被抢占能力。 |
| QoS最大下行比特速率 | 该参数用于表示QoS流最大下行比特速率。 |
| QoS可保证的下行比特速率 | 该参数用于表示QoS流可保证的下行比特速率。 |
| 会话 AMBR 下行比特速率 | 该参数用于表示会话 AMBR 下行比特速率。 |
| 平均窗口大小 | 该参数用于表示计算GFBR/MFBR的时间窗口。 |
| 最大数据峰值 | 该参数用于指示Delay-critical类型的GBR QoS Flow（应用于uRLLC业务）要求空口在时延预算内传输的最大数据包长度。 |
| 请求TAI | 该参数用于表示会话请求TAI区域。最多显示10个TAI。 |
| AMF服务网络功能标识 | 该参数用于表示AMF服务网络功能标识。最多显示5个AMF服务网络标识。 |
| UPF服务网络功能标识 | 该参数用于表示UPF ID。最多显示10个UPF ID。 |

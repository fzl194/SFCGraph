## [命令功能](#ZH-CN_MMLREF_0000001144888613)

**适用NF：NRF**

该命令用于查询服务发现、检索响应及通知请求中NF的私有信息字段格式。

## [注意事项](#ZH-CN_MMLREF_0000001144888613)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001144888613)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001144888613)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| NFTYPE | 网元类型 | 可选必选说明：可选参数<br>参数含义：该参数用于表示设置NF私有信息格式的目标网元类型。<br>数据来源：本端规划<br>取值范围：<br>- BSF（BSF）<br>默认值：无<br>配置原则：<br>当前只支持设置BSF的私有信息格式。 |

## [使用实例](#ZH-CN_MMLREF_0000001144888613)

查询BSF的私有信息格式。

```
LST NRFNFINFOFMT:;
%%LST NRFNFINFOFMT:;%%
RETCODE = 0  操作成功

结果如下
------------------------
网元类型  =  BSF
NF私有信息格式  =  ARRAY
(结果个数 = 1)
```

## [输出结果说明](#ZH-CN_MMLREF_0000001144888613)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 网元类型 | 该参数用于表示设置NF私有信息格式的目标网元类型。 |
| NF私有信息格式 | 该参数用于表示服务发现、检索响应及通知请求中NF的私有信息字段格式类型。 |

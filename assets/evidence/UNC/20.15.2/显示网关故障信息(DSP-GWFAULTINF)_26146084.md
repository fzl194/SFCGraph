#### [命令功能](#ZH-CN_MMLREF_0000001126146084)

**适用网元：MME**

该命令用于查看系统内记录的网关故障信息，包含故障类型、故障时间、故障IP地址。

#### [注意事项](#ZH-CN_MMLREF_0000001126146084)

无。

#### [本地用户权限](#ZH-CN_MMLREF_0000001126146084)

manage-ug；system-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126146084)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126146084)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| DT | 查询方式 | 可选必选说明：可选参数<br>参数含义：该参数用于指定查询方式。<br>取值范围：<br>“PROCESS_NO(进程标识)”<br>默认值：无<br>说明：如果不设置本参数，则表示查询所有业务进程上故障信息的数目。 |
| RUNAME | RU名称 | 可选必选说明：条件必选参数<br>参数含义：该参数用于指定节点的RU名称。该参数可以通过<br>[DSP RU](../../../../../平台服务管理/单体服务公共功能管理/系统管理/资源管理/RU管理/显示资源单元信息（DSP RU）_59103857.md)<br>命令查询。<br>前提条件: 该参数在<br>“查询方式”<br>参数配置为<br>“PROCESS_NO(进程标识)”<br>后生效。<br>取值范围：0～63位字符串<br>默认值：无 |
| PN | 进程号 | 可选必选说明：条件必选参数<br>参数含义：该参数用于指定需要查看SPP进程的进程号。<br>前提条件: 该参数在<br>“查询方式”<br>参数配置为<br>“PROCESS_NO(进程标识)”<br>后生效。<br>取值范围：0~20<br>默认值：无 |

#### [使用实例](#ZH-CN_MMLREF_0000001126146084)

1. 显示所有SPP进程上的故障信息：
  DSP GWFAULTINF:;
  ```
  %%DSP GWFAULTINF:;%%
  RETCODE = 0  操作成功。

  操作结果如下
  ------------
  RU名称           进程号  接口故障个数  网关重启个数

  USN_SP_RU_0064   1       0             0           
  USN_SP_RU_0064   0       0             0           
  (结果个数 = 2)

  ---    END
  ```
2. 显示指定SPP进程上的故障信息：
  DSP GWFAULTINF:DT=PROCESS_NO, RUNAME="USN_SP_RU_0064", PN=0:;
  ```
  %%DSP GWFAULTINF: DT=PROCESS_NO, RUNAME="USN_SP_RU_0064", PN=0;%%
  RETCODE = 0  操作成功。

  接口故障信息
  ------------
    接口类型  =  S11
    故障时间  =  2014-11-25 21:07:52
  本端IP地址  =  192.168.24.48
  对端IP地址  =  10.10.10.10
  (结果个数 = 1)

  ---    END
  ```

#### [输出结果说明](#ZH-CN_MMLREF_0000001126146084)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 接口故障个数 | 参数含义：显示接口故障个数。<br>取值范围：0~512 |
| 网关重启个数 | 参数含义：显示网关重启个数。<br>取值范围：0~512 |
| 接口类型 | 参数含义：显示故障接口类型（S11或S5接口）。<br>取值范围：<br>- S11(S11)<br>- S5(S5) |
| 网关类型 | 参数含义：显示重启网关类型（S-GW或P-GW重启）。<br>取值范围：<br>- SGW(S-GW)<br>- PGW(P-GW) |
| 故障时间 | 参数含义：显示故障发生的时间。<br>取值范围：日期时间字符串。 |
| 本端IP地址 | 参数含义：此参数用于显示本端IP地址。<br>当S11接口故障或S-GW重启时，为MME的S11接口地址；<br>当S5接口故障或P-GW重启时，为S-GW的S11接口地址。<br>取值范围：0~64位字符串 |
| 对端IP地址 | 参数含义：此参数用于显示对端IP地址。<br>当S11接口故障或S-GW重启时，为S-GW的S11接口地址；<br>当S5接口故障或P-GW重启时，为P-GW的S5接口地址。<br>取值范围：0~64位字符串 |

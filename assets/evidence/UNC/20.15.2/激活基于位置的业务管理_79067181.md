## [操作场景](#ZH-CN_OPI_0279067181)

当运营商期望根据用户所在位置采取差异化策略（基于用户的位置分配IP地址或性能统计数据上报）时，可以配置基于位置的业务管理功能。

> **说明**
> 适用于 GGSN、 PGW-C、SMF。

## [必备事项](#ZH-CN_OPI_0279067181)

- 请仔细阅读[WSFD-109105 基于位置的业务管理](../WSFD-109105 基于位置的业务管理_79067178.md)。
- 完成加载License（如果有需求，请联系华为技术支持工程师处理）。

2. 打开本特性的License配置开关。

  > [步骤 3](#ZH-CN_OPI_0279067181__step15536230103518) ～ [步骤 6](#ZH-CN_OPI_0279067181__step1638043213516) 几个步骤之间相互独立，可以单个参数作为位置信息，也可以多个参数的组合作为位置信息。请根据实际规划配置。
3. UNC 作为GGSN时，配置位置信息（RAC作为位置信息）。
    a. 增加RAC组。
      [**ADD RACGROUP**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/虚拟APN映射管理/基于RAC位置的虚拟APN映射管理/虚拟APN映射的RAC组/增加RAC组（ADD RACGROUP）_09652702.md)

## [任务示例](#ZH-CN_OPI_0279067181)

任务描述

配置位置信息与虚拟APN的映射规则，以及虚拟APN与真实APN的映射规则。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV3WPLBSM11", SWITCH=ENABLE;
```

//配置位置信息（RAC作为位置信息）。

```
ADD RACGROUP: RACGROUPNAME="CityA";
ADD RACID: RACSECNUM=0, RACSTARTID="0x11", RACENDID="0x20", RACGROUPNAME="CityA";
```

//配置位置信息（LAC作为位置信息）。

```
ADD LACGROUP: LACGROUPNAME="CityA";
ADD LACID: LACSECNUM=1, LACSTARTID="0x0121", LACENDID="0x0130", LACGROUPNAME="CityA";
ADD LACGROUP: LACGROUPNAME="CityB";
ADD LACID: LACSECNUM=2, LACSTARTID="0x0131", LACENDID="0x0135", LACGROUPNAME="CityB";
```

//配置位置信息（TAC作为位置信息）。

```
ADD TACGROUP: TACGROUPNAME="CityB", TACTYPE=S1Tac;
ADD S1TACID: TACSECNUM=3, TACSTARTID="0x0136", TACENDID="0x0140", TACGROUPNAME="CityB";
ADD TACGROUP: TACGROUPNAME="CityC", TACTYPE=S1Tac;
ADD S1TACID: TACSECNUM=4, TACSTARTID="0x0141", TACENDID="0x0150", TACGROUPNAME="CityC";
```

//配置位置信息（服务节点IP作为位置信息）。

```
ADD SRVNODEGROUP: GROUPNAME="CityD";
ADD SRVNODEIP: GROUPNAME="CityD", IPSECTIONID=1, IPVERSION=IPV4, IPV4ADDRSTART="192.168.1.10", IPV4ADDREND="192.168.1.120";
```

//配置虚拟APN。

```
ADD APN: APN="example.com", VIRTUALAPN=ENABLE;
```

//配置虚拟APN和真实APN的映射规则。

```
ADD VIRTUALAPNRULE: VIRTUALAPN="example.com", MAPPINGTYPE=RAC_GROUP, PRIORITY=1, MAPPINGVALUE="CityA", REALAPN="huawei.com";
ADD VIRTUALAPNRULE: VIRTUALAPN="example.com", MAPPINGTYPE=LAC_GROUP, PRIORITY=2, MAPPINGVALUE="CityA", REALAPN="huawei.com";
ADD VIRTUALAPNRULE: VIRTUALAPN="example.com", MAPPINGTYPE=LAC_GROUP, PRIORITY=2, MAPPINGVALUE="CityB", REALAPN="huawei.com";
ADD VIRTUALAPNRULE: VIRTUALAPN="example.com", MAPPINGTYPE=TAC_GROUP, PRIORITY=3, MAPPINGVALUE="CityB", REALAPN="huawei.com";
ADD VIRTUALAPNRULE: VIRTUALAPN="example.com", MAPPINGTYPE=TAC_GROUP, PRIORITY=3, MAPPINGVALUE="CityC", REALAPN="huawei.com";
ADD VIRTUALAPNRULE: VIRTUALAPN="example.com", MAPPINGTYPE=SERVING_GROUP, PRIORITY=4, MAPPINGVALUE="CityD", REALAPN="huawei.com";
```

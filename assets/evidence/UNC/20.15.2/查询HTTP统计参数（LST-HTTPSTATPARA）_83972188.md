## [命令功能](#ZH-CN_MMLREF_0000001183972188)

该命令用于查询进行统计的HTTP对端地址。

## [注意事项](#ZH-CN_MMLREF_0000001183972188)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001183972188)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001183972188)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PEERIPTYPE | 对端IP地址类型 | 可选必选说明：可选参数<br>参数含义：该参数用于指定对端IP地址类型。<br>数据来源：全网规划<br>取值范围：<br>- “IPv4（IPv4）”：IPv4地址<br>- “IPv6（IPv6）”：IPv6地址<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001183972188)

查询HTTP统计的对端IP地址，执行命令如下：

```
%%LST HTTPSTATPARA:;%%
RETCODE = 0  操作成功

结果如下
--------
对端IP地址类型  对端IPv4地址  对端IPv6地址        

IPv4            10.2.3.4      ::
IPv4            10.2.3.5      ::
IPv4            10.2.3.6      ::
IPv4            10.2.3.7      ::
IPv4            10.2.3.8      ::
IPv4            10.2.3.9      ::
IPv4            10.2.3.10     ::
(结果个数 = 7)

---    END
```

| 对端IPv6地址 | 该参数用于指定对端IPv6地址。 |

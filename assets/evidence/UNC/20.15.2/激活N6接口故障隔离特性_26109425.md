## [操作场景](#ZH-CN_OPI_0000001426109425)

当网络中存在UPF/PGW-U/GGSN-U与DNN/APN数据网络之间的接口（N6接口、SGi或者Gi接口）故障时，可以激活N6接口故障隔离特性，使SMF/PGW-C/GGSN-C在选择UPF/PGW-U/GGSN-U时屏蔽N6/SGi/Gi接口故障的用户面网元。激活N6接口故障隔离特性，需要在UPF/PGW-U/GGSN-U侧和SMF/PGW-C/GGSN-C侧分别进行配置。SMF/PGW-C/GGSN-C侧配置请参考本章节进行操作。UPF/PGW-U/GGSN-U侧配置请在SMF/PGW-C/GGSN-C侧配置完成后参考UPF/PGW-U/GGSN-U的产品文档进行配置。

> **说明**
> 适用于 GGSN、 PGW-C、SMF。

## [任务示例](#ZH-CN_OPI_0000001426109425)

任务描述

在SMF侧增加支持故障联动的APN信息，使SMF选择UPF时能够过滤N6接口故障的网元，实现UPF故障隔离功能。

脚本

//配置检测到N6接口故障后进行联动的APN。

```
SET APNFAULT: APN="huawei.com",UPFAULTMODE=DETECTION,RESTOREFAULT=ENABLE;
```

//配置用户面链路故障老化时间。

```
SET PFCPPARA: UPFAULTAGETIME=5;
```

//设置APN故障任务扫描用户的速率。

```
SET UPCSCANRATE: SCANTYPE=APNFAULT, RATE=20;
```

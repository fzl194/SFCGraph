#### [命令功能](#ZH-CN_CONCEPT_0000001150812805)

**适用NF：SGSN、MME、AMF**

该命令用于查询系统在为SCTP分配缓冲区时使用的参数来源。

#### [注意事项](#ZH-CN_CONCEPT_0000001150812805)

无。

#### [本地用户权限](#ZH-CN_CONCEPT_0000001150812805)

manage-ug;system-ug;monitor-ug

#### [网管用户权限](#ZH-CN_CONCEPT_0000001150812805)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001150812805)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001150812805)

查询SCTP缓冲区参数源类型：

```
LST SCTPBUFFCFG:;
```

```
查询结果如下
-------------------------
 SCTP缓冲区参数源类型  =  自定义
(结果个数 = 1)

---    END
```

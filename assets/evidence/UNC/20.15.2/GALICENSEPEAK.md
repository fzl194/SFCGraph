#### [命令功能](#ZH-CN_TOPIC_0000002081572754)

**适用NF：NCG**

该命令用于显示GA接口话务高峰信息。

#### [注意事项](#ZH-CN_TOPIC_0000002081572754)

无。

#### [本地用户权限](#ZH-CN_TOPIC_0000002081572754)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_TOPIC_0000002081572754)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_TOPIC_0000002081572754)

无。

#### [使用实例](#ZH-CN_TOPIC_0000002081572754)

显示GA接口话务高峰信息，示例如下：

```
DSP GALICENSEPEAK:;
```

```
RETCODE = 0  操作成功  
已使用的峰值License记录 
-----------------------
      使用日期  =  2024-10-08 至 2024-10-10
 过载License项  =  4G Bearers数,5G Bearers数
(结果个数 = 1)  

---    END
```

#### [输出结果说明](#ZH-CN_TOPIC_0000002081572754)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 使用日期 | 用于返回GA接口License峰值的日期区间。 |
| 过载License项 | 用于返回出现峰值的GA接口License项。 |

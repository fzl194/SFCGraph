#### [命令功能](#ZH-CN_CONCEPT_0209896705)

**适用NF：PGW-C、SMF**

此命令用于查询集中点的部署模式。

#### [注意事项](#ZH-CN_CONCEPT_0209896705)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0209896705)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0209896705)

无。

#### [使用实例](#ZH-CN_CONCEPT_0209896705)

查询集中点部署模式：

```
LST CONCENPOINT:;
```

```

RETCODE = 0  操作成功。

集中点信息
----------
 Ga集中点模式  =  单连接
 Gy集中点模式  =  本端IP和Diameter对端主机名
 Gx集中点模式  =  本端IP和Diameter对端主机名
 S6b集中点模式  =  本端IP和Diameter对端主机名
 Ga每进程端口数  =  1
(结果个数 = 1)
---    END
```

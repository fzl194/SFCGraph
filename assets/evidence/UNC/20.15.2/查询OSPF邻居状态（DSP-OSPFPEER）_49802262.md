#### [命令功能](#ZH-CN_CONCEPT_0000001549802262)

该命令用于显示OSPF邻居信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001549802262)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549802262)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549802262)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| NBRROUTERID | 邻居路由器标识 | 可选必选说明：可选参数<br>参数含义：邻居路由器标识。<br>数据来源：对端协商<br>取值范围：IPv4地址类型。点分十进制格式。<br>默认值：无 |
| PROCID | 进程号 | 可选必选说明：可选参数<br>参数含义：OSPF进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |
| NBRIPADDR | 邻居IP地址 | 可选必选说明：可选参数<br>参数含义：邻居IP地址。<br>数据来源：对端协商<br>取值范围：IPv4地址类型。点分十进制格式。<br>默认值：无 |
| IFNAME | 接口名 | 可选必选说明：可选参数<br>参数含义：接口名称。<br>数据来源：对端协商<br>取值范围：字符串类型，输入长度范围为1～63。<br>默认值：无<br>配置原则：请使用LST INTERFACE命令查看可用接口。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549802262)

显示设备OSPF进程号为1的所有节点邻居信息：

```
DSP OSPFPEER: PROCID=1;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001549802262)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 邻居路由器标识 | 该参数指定邻居路由器标识。 |
| 进程号 | 该参数指定用户创建的OSPF进程号。 |
| 区域号 | 该参数指定用户创建的OSPF区域号。 |
| 邻居IP地址 | 该参数指定邻居IP地址。 |
| 接口名 | 该参数指定本端建立邻居的接口名称。 |
| 邻居状态 | 该参数指定邻居状态。Down：该状态为邻居的初始状态。表示没有邻居收到任何信息。在NBMA网络上，Hello报文在该状态时仍然可以收发，周期为Poll Interval（Hello Interval<Poll Interval）；Attempt：该状态只存在于NBMA网络上，表明正在尝试建立邻居关系。此时，Hello报文的收发周期为Hello Interval（Hello Interval<Poll Interval）；Init：该状态表明已经接收到了从邻居发送来的Hello报文；2-Way：该状态表明已经接收到了从邻居发送过来的Hello报文，并且该Hello报文的Neighbor List中包含本地Router ID，即双方可以互通；ExStart：该状态为建立Adjacency的第一步，进行主从关系、DD Sequence Number的协商；Exchange：从该状态开始，进行LSDB同步操作，交互的报文有DD报文、LSR报文、LSU报文；Loading：LSDB正在进行同步操作，交互的报文有LSR报文和LSU报文；Full：该状态说明，邻居的LSDB已经同步完成，双方建立了Full邻接关系。 |
| 邻居模式 | 该参数指定DD交换过程中邻居的角色是Master或Slave。 |
| 邻居优先级 | 该参数指定邻居的指定路由器优先级。 |
| 指定路由器 | 该参数指定邻居关系中的指定路由器。 |
| 备份路由器 | 该参数指定邻居关系中的备份指定路由器。 |
| 邻居接口MTU值 | 该参数指定邻居接口MTU的值。 |
| 邻居认证序列号 | 该参数指定邻居接口的认证加密的序列号。 |
| 邻居UP时间 | 该参数指定邻居被激活的时间。 |
| GR状态 | 该参数指定GR状态，可分为正常模式、正在GR、Helper模式、NA四个状态。 |
| 失效时间（s） | 该参数指定邻居距离失效的时间。 |
| 邻居重传间隔（s） | 该参数指定重传LSA的时间间隔。 |
| 主机名 | 该参数指定邻居的主机名。 |

## [操作场景](#ZH-CN_OPI_0230951488)

本操作指导介绍在运行网络中激活漫游控制特性的操作过程。

漫游控制，是指 UNC 提供互连PLMN表的配置，只有配置允许互连的PLMN用户才能够接入本 UNC 。漫游控制特性使网络具有限制用户移动范围和业务访问的能力。对于非本网的且不在互联PLMN表中的用户，将拒绝其接入，拒绝的原因值可灵活配置。

> **说明**
> 适用于AMF。

## [对系统的影响](#ZH-CN_OPI_0230951488)

该操作对系统正常运行没有影响。

## [操作步骤](#ZH-CN_OPI_0230951488)

1. 进入 “MML命令行-UNC” 窗口。
2. 增加互联的PLMN，并配置允许的业务。
  [**ADD NGCONNECTPLMN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/运营商管理/互联PLMN管理/增加5G Connect PLMN（ADD NGCONNECTPLMN）_09651402.md)
3. 设置漫游受限拒绝原因值。
  [**SET NGMMPROCTRL**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/移动性管理/MM流程管理/5G移动性管理流程控制参数/设置5G移动性管理流程控制参数（SET NGMMPROCTRL）_09652386.md)

## [任务示例](#ZH-CN_OPI_0230951488)

任务描述

增加一个互联的PLMN，PLMN ID为12345，运营商标识为0。

脚本

//PLMN为12345的运营商A与本运营商签署了漫游协议，移动国家码为123，移动网号为45。为了支持该运营商的用户注册到本运营商网络，新增互联PLMN配置如下：

```
ADD NGCONNECTPLMN: NOID=0, MCC="123", MNC="45", DESC="for MNO A";
```

//当非本网的且不在互联PLMN表中的用户注册，将拒绝其接入。拒绝原因值设置为0，表示发生该异常时系统将下发#15（No suitable cells in tracking area）原因值。

```
SET NGMMPROCTRL: PROT=OTHER_PROC,ROAMRST=0;
```

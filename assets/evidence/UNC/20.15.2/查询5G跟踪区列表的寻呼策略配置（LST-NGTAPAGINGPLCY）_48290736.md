## [命令功能](#ZH-CN_MMLREF_0248290736)

**适用NF：AMF**

该命令用于查询基于5G跟踪区列表的寻呼策略配置数据。

## [注意事项](#ZH-CN_MMLREF_0248290736)

无

#### [操作用户权限](#ZH-CN_MMLREF_0248290736)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0248290736)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| TALISTID | 跟踪区列表标识 | 可选必选说明：可选参数<br>参数含义：该参数用于标识一个跟踪区列表，一个跟踪区列表由一个或若干个跟踪区组成。一个跟踪区列表最多可包含16个跟踪区。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~65534。<br>默认值：无<br>配置原则：<br>本参数配置前须先在ADD NGTALST中配置。 |

## [使用实例](#ZH-CN_MMLREF_0248290736)

- 查询所有5G跟踪区列表的寻呼策略配置，执行如下命令：
  ```
  %%LST NGTAPAGINGPLCY:;%%
  RETCODE = 0  操作成功

  结果如下
  --------
  跟踪区列表标识  是否关闭精准寻呼  

  1               是                
  2               否                
  (结果个数 = 2)

  ---    END
  ```
- 查询“跟踪区列表标识”为“1”的5G跟踪区列表的寻呼策略配置，执行如下命令：
  ```
  %%LST NGTAPAGINGPLCY: TALISTID=1;%%
  RETCODE = 0  操作成功

  结果如下
  --------
    跟踪区列表标识  =  1
  是否关闭精准寻呼  =  是
  (结果个数 = 1)

  ---    END
  ```

## [输出结果说明](#ZH-CN_MMLREF_0248290736)

| --- | --- |
| 跟踪区列表标识 | 该参数用于标识一个跟踪区列表，一个跟踪区列表由一个或若干个跟踪区组成。一个跟踪区列表最多可包含16个跟踪区。 |
| 是否关闭精准寻呼 | 该参数用来配置指定的5G跟踪区列表是否关闭精准寻呼。 |

#### [命令功能](#ZH-CN_MMLREF_0000001172225255)

**适用网元：SGSN**

此命令用于增加Iu模式实时小区位置订阅，根据用户当前所在的位置，控制是否订阅该用户的实时小区位置并上报网络运维网元，例如vSCEF。

#### [注意事项](#ZH-CN_MMLREF_0000001172225255)

- 此命令执行后立即生效。
- 此命令最大记录数为32。
- 此命令配置后，UNC会发起扫描任务，根据用户当前所在位置和本命令的配置记录来判断是否针对该用户发起实时小区位置订阅流程。在用户量大时，用户的实时小区订阅流程发起时间会有一定延时。
- 此命令配置后会触发RAN侧通过Location Report流程上报用户的实时小区位置，若要将用户的实时小区位置上报vSCEF等CloudUDN，还需要通过CHR的相关命令[**SET CHRCFG**](../../CHR管理/CHR配置/设置CHR配置（SET CHRCFG）_72225295.md)来订阅Location Report流程的CHR单据。
- 执行命令前请使用[**DSP LICENSE**](../../../../../../平台服务管理/操作维护/License管理/显示License(DSP LICENSE)_00360098.md)命令确认对应特性license是否得到授权，执行[**LST LICENSESWITCH**](../../../../../../平台服务管理/操作维护/License管理/查询License配置项开关（LST LICENSESWITCH）_09651570.md)命令确认特性开关状态为“打开”。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172225255)

manage-ug；system-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172225255)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172225255)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| AREA | 区域范围 | 可选必选说明：必选参数<br>参数含义：该参数用于指定区域范围。<br>数据来源：整网规划<br>取值范围：<br>- “RA(指定路由区)”：表示该区域范围类型为指定路由区。<br>默认值：无<br>说明：- 区域范围表示进行实时位置订阅所适用的位置区域。<br>- 区域范围目前只支持路由区粒度的范围。<br>- 相同粒度区域范围的各记录的区域码区段不能交叠。<br>- 只有在配置记录位置区段中用户，并且订阅开关打开，才会开启实时位置区域订阅流程。 |
| MCC | 移动国家码 | 可选必选说明：条件必选参数<br>参数含义：该参数用于指定移动国家代码。<br>前提条件：该参数在“AREA(区域范围)”设置为“RA(指定路由区)”时，才需要配置。<br>数据来源：整网规划<br>取值范围：3位的十进制数字<br>默认值：无 |
| MNC | 移动网号 | 可选必选说明：条件必选参数<br>参数含义：该参数用于指定移动网号。<br>前提条件：该参数在“AREA(区域范围)”设置为“RA(指定路由区)”时，才需要配置。<br>数据来源：整网规划<br>取值范围：2～3位的十进制数字<br>默认值：无 |
| LAC | 位置区域码 | 可选必选说明：条件必选参数<br>参数含义：该参数用于指定位置区域码。<br>前提条件：该参数在“AREA(区域范围)”设置为“RA(指定路由区)”时，才需要配置。<br>数据来源：整网规划<br>取值范围：0x0000～0xFFFF<br>默认值：无 |
| RAC | 路由区域码 | 可选必选说明：条件必选参数<br>参数含义：该参数用于指定路由区域码。<br>前提条件：该参数在“AREA(区域范围)”设置为“RA(指定路由区)”时，才需要配置。<br>数据来源：整网规划<br>取值范围：0x00～0xFF<br>默认值：无<br>说明：- 各条记录的RAC区段不能交叠。 |
| RACRANGE | 路由区域码范围 | 可选必选说明：条件可选参数<br>参数含义：该参数用于指定路由区域码范围。<br>前提条件：该参数在“AREA(区域范围)”设置为“RA(指定路由区)”时，才需要配置。<br>数据来源：整网规划<br>取值范围：0x00～0xFF<br>默认值：无<br>配置原则：该参数的取值需要大于或等于“RAC(路由区域码)”。<br>说明：- 该参数与“RAC(路由区域码)”参数构成一个RAC区段，方便客户配置连续的路由区域。<br>- 如果不输入，表示配置单个RAC。 |
| SUBSWITCH | 订阅开关 | 可选必选说明：必选参数<br>参数含义：该参数用于指定是否对该范围内的用户发起实时位置订阅。<br>数据来源：整网规划<br>取值范围：<br>- “ON(打开)”<br>- “OFF(关闭)”<br>默认值：无 |

#### [使用实例](#ZH-CN_MMLREF_0000001172225255)

增加一条Iu模式实时小区位置订阅记录：“区域范围”为“RA”，“移动国家码”为“111”，“移动网号”为“222”，“位置区域码”为“0x3333”，“路由区域码”为“0x44”，“路由区域码范围”为“0x55”，“订阅开关”为“ON”：

ADD IULOCRPT: AREA=RA, MCC="111", MNC="222", LAC="0x3333", RAC="0x44", RACRANGE="0x55", SUBSWITCH=ON;

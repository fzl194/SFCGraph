## [命令功能](#ZH-CN_MMLREF_0000001192780526)

该命令用于查询POD内存阈值。

## [注意事项](#ZH-CN_MMLREF_0000001192780526)

- 该命令执行后立即生效。
- 该命令仅在FusionStage裸机容器下支持。

#### [操作用户权限](#ZH-CN_MMLREF_0000001192780526)

G_1，管理员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001192780526)

无

## [使用实例](#ZH-CN_MMLREF_0000001192780526)

查询POD内存告警阈值。

```
%%LST PODMEMTHD:;%%
RETCODE = 0  操作成功

结果如下
------------------------
告警上报门限  =  90
告警恢复门限  =  80
(结果个数 = 1)
```

## [输出结果说明](#ZH-CN_MMLREF_0000001192780526)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 告警上报门限 | 该参数用于指定告警上报门限。 |
| 告警恢复门限 | 该参数用于指定告警恢复门限。 |

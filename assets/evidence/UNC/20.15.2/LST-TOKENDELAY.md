## [命令功能](#ZH-CN_MMLREF_0000001278965049)

该命令用于查询当前Token延时迁移时间。

## [注意事项](#ZH-CN_MMLREF_0000001278965049)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001278965049)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001278965049)

无

## [使用实例](#ZH-CN_MMLREF_0000001278965049)

查询当前Token延时迁移时间。

```
%%LST TOKENDELAY:;%%
RETCODE = 0  操作成功

结果如下
--------
延时迁移时间(秒) = 0
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001278965049)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 延时迁移时间(秒) | 该参数用于表示服务实例故障恢复后，服务实例延迟承担业务的时长。 |

## [命令功能](#ZH-CN_MMLREF_0000001124956624)

**适用NF：AMF、MME、SGSN**

该命令用于显示UAM GTP-C路径数量。

## [注意事项](#ZH-CN_MMLREF_0000001124956624)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001124956624)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001124956624)

无

## [使用实例](#ZH-CN_MMLREF_0000001124956624)

DSP COMGTPCPATHNUM:;

```
%%DSP COMGTPCPATHNUM:;%%
RETCODE = 0  操作成功

结果如下
------------------------
     AMF路径数量  =  4
MME/SGSN路径数量  =  0
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001124956624)

| 输出项名称 | 输出项解释 |
| --- | --- |
| AMF路径数量 | 该参数用于显示AMF路径数量。 |
| MME/SGSN路径数量 | 该参数用于显示MME/SGSN路径数量。 |

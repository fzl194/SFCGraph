# 删除M2M关联的UPF组（RMV M2MUPGROUP）

- [命令功能](#ZH-CN_MMLREF_0296242806__1.3.1)
- [注意事项](#ZH-CN_MMLREF_0296242806__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0296242806__1.3.4)

## [命令功能](#ZH-CN_MMLREF_0296242806)

**适用NF：PGW-C、SMF**

该命令用于删除M2M关联的UPF组。

## [注意事项](#ZH-CN_MMLREF_0296242806)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0296242806)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0296242806)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| UPFGRPNAME | UPF组名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定UPF组名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~31。不区分大小写。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0296242806)

删除M2M关联的UPF组，组名为upfgrp1：

```
RMV M2MUPGROUP: UPFGRPNAME="upfgrp1";
```

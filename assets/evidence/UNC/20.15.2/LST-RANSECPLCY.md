## [命令功能](#ZH-CN_MMLREF_0209653115)

**适用NF：SMF**

该命令用于查询RAN侧用户面安全策略。

## [注意事项](#ZH-CN_MMLREF_0209653115)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209653115)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209653115)

无

## [使用实例](#ZH-CN_MMLREF_0209653115)

用户需要查看配置的用户安全策略参数时，执行如下命令:

```
%%LST RANSECPLCY:;%%
RETCODE = 0  操作成功

结果如下
------------------------
             完整性保护策略  =  优先从签约数据获取
         本地完整性保护指示  =  优选
          完整性保护PDU策略  =  限速 
               加密保护策略  =  优先从签约数据获取
           本地加密保护指示  =  优选
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209653115)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 完整性保护策略 | 该参数用于指定RAN侧用户面完整性保护策略。 |
| 本地完整性保护指示 | 该参数是本地配置，用于指示RAN侧是否要执行用户面完整性保护策略。 |
| 完整性保护PDU策略 | 该参数用于指定完整性保护指示为“REQUIRED”，但是会话AMBR大于UE支持的完整性保护最大速率时的处理策略。 |
| 加密保护策略 | 该参数用于指定RAN侧用户面加密保护策略。 |
| 本地加密保护指示 | 该参数是本地配置，用于指示RAN侧是否要执行用户面加密保护策略。 |

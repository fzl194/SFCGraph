#### [命令功能](#ZH-CN_MMLREF_0000001126146096)

**适用网元：SGSN、MME**

此命令用于查询迁移配置信息。

#### [注意事项](#ZH-CN_MMLREF_0000001126146096)

无。

#### [本地用户权限](#ZH-CN_MMLREF_0000001126146096)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126146096)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126146096)

无。

#### [使用实例](#ZH-CN_MMLREF_0000001126146096)

查询迁移配置信息：

LST OFFLOADINF:;

```
%%LST OFFLOADINF:;%%
RETCODE = 0  操作成功。

迁移配置信息
------------
      迁移模式周期路由更新定时器(s)  =  60
              第一阶段迁移时间(min)  =  1
                         非广播 RAI  =  111111111111
                         POOL区标识  =  0
                         NULL NRI值  =  9
                           迁移速率  =  高速
 迁移时是否设置Force To Standby信元  =  是
迁移时是否设置Follow On Proceed信元  =  是
          根据UE无线能力选择USN开关  =  关闭
          根据UE无线能力选择UNC开关  =  关闭
(结果个数 = 1)
---    END
```

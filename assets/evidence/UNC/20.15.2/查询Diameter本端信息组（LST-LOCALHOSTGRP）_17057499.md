#### [命令功能](#ZH-CN_CONCEPT_0000201617057499)

**适用NF：PGW-C、SMF**

此命令用于查询Diameter本端主机组名称。

#### [注意事项](#ZH-CN_CONCEPT_0000201617057499)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000201617057499)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000201617057499)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000201617057499)

查询Diamter本端主机组记录：

```
LST LOCALHOSTGRP:;
```

```

-------------------

(结果个数 = 1)

```

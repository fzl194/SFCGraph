## [命令功能](#ZH-CN_MMLREF_0000001287520116)

**适用NF：NCG**

该命令用于查询cdf token权重。

## [注意事项](#ZH-CN_MMLREF_0000001287520116)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001287520116)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001287520116)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PODNAME | pod 名称 | 可选必选说明：可选参数<br>参数含义：该参数用于描述pod名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~255。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001287520116)

当查询pod名称为cgfb-pod-0上的cdf token权重：

```
+++    UNC/*MEID:0 MENAME:UNC_VNF_ncg001*/        2022-06-11 15:27:17+8:00
O&M    #3549
%%LST CDFTOKENWEIGHT: PODNAME="cgfb-pod-0";%%
RETCODE = 0  操作成功

结果如下
--------
 pod 名称  =  cgfb-pod-0
token权重  =  80
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001287520116)

| --- | --- |
| pod 名称 | 该参数用于描述pod名称。 |
| token权重 | 该参数用于描述token权重。 |

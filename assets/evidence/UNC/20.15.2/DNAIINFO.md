## [命令功能](#ZH-CN_MMLREF_0000001211085190)

**适用NF：SMF、PGW-C**

该命令用于查询DNAI信息。

## [注意事项](#ZH-CN_MMLREF_0000001211085190)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001211085190)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001211085190)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| DNAI | 数据网络访问标识 | 可选必选说明：可选参数<br>参数含义：该参数用于指定数据网络访问标识。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是1~63。不支持空格，不区分大小写。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001211085190)

查询DNAI信息

```
LST DNAIINFO: DNAI="cmnet_dnai";
```

## [输出结果说明](#ZH-CN_MMLREF_0000001211085190)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 数据网络访问标识 | 该参数用于指定数据网络访问标识。 |
| 分流方向 | 该参数用于控制分流的方向。 |
| 分流位置 | 该参数用于指示DNAI的分流位置。 |
| Always分流开关 | 该参数用于控制DNAI是否支持Always分流。 |

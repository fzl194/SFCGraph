## [操作场景](#ZH-CN_OPI_0163535136)

本操作指导介绍在运行网络中激活基于UE能力选择 网关 特性的操作过程。

基于UE接入能力选择网关，是指 UNC 在使用 APN （Access Point Name）进行 DNS (Domain Name Service)解析获取网关 IP 地址前，将UE接入能力信息加入到APN-NI和APN-OI之间，然后将扩展的APN发送到DNS服务器解析，获取合建的GGSN/P-GW的IP地址。

> **说明**
> 适用于 SGSN、 MME。

## [对系统的影响](#ZH-CN_OPI_0163535136)

该操作对系统正常运行没有影响。

## [必备事项](#ZH-CN_OPI_0163535136)

前提条件

- 请仔细阅读[WSFD-205005 基于UE接入能力选择网关特性概述](特性概述_70166788.md)。

- DNS必须配置添加了UE接入能力属性（UEACCCAP）的APN所对应的GGSN/P-GW的IP地址，以便解析扩展后的APN。
  > **说明**
  > 用户也可以选择自行配置添加了UE接入能力属性（UEACCCAP）的APN所对应的GGSN/P-GW的IP地址（见 [操作步骤4](#ZH-CN_OPI_0163535136__step3) ），若DNS或用户都没有配置添加UE接入能力属性（UEACCCAP）的APN所对应的GGSN/P-GW的IP地址，将导致基于UE接入能力选择网关失败。
- 完成加载License。

## [操作步骤](#ZH-CN_OPI_0163535136)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. 增加基于UE接入能力选择网关的策略。
  [**ADD GWSELPLCY**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/GnGp-GGSN_S5_S8接口管理/GGSN_P-GW选择/增加GGSN_P-GW选择策略（ADD GWSELPLCY）_26145944.md)
  > **说明**
  > - 参数“SELPLCY”（选择策略）指定在用户附着流程中UNC同时选择S-GW和P-GW时采用S-GW优选策略还是P-GW优选策略，不设置则默认为S-GW优选。该参数在[**SET SMFUNC**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/会话管理/SM扩展功能管理/设置会话管理扩展功能(SET SMFUNC)_26145684.md)命令的“拓扑选择开关”配置为“YES”时生效。
  > - 参数“LABEL”（定制标识类型）选择“UEACCCAP”。
  > - 参数“SLCTM”（选择方法）选择“CUSTOMIZE_APN-OI_REPLACEMENT”。
4. **可选：**为具备GUL能力的UE设置统一使用.gprs后缀的APN DNS域名。
  [**ADD APNDNS**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/GnGp-GGSN_S5_S8接口管理/APN DNS域名策略/增加APN DNS域名策略(ADD APNDNS)_26145932.md)
  > **说明**
  > 当UE从E-UTRAN接入GUL网络时，需要执行此命令确定使用.gprs后缀的DNS域名进行DNS查询。
  >
  > - 参数“UEACCCAP”（UE接入能力）选择“GERAN/UTRAN/EUTRAN_UE”。
  > - 参数“DNSPOLICY”（DNS域名策略）选择“GPRS”。
5. **可选：**配置添加了UE接入能力属性的APN所对应的GGSN/P-GW的IP地址。
  > **说明**
  > 若DNS中已经配置添加了UE接入能力属性的APN所对应的GGSN/P-GW的IP地址，则不需要执行此步骤。
  [**ADD IPV4DNSH**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/DNS/DNS Hostfile管理/增加IPV4 DNS Hostfile记录(ADD IPV4DNSH)_26145884.md)

## [任务示例](#ZH-CN_OPI_0163535136)

任务描述

打开基于UE接入能力选择网关开关，同时添加UE接入能力属性的APN对应的网关的IP地址。

脚本

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2GSUE02", SWITCH=ENABLE;
```

//增加基于UE接入能力选择网关的策略。

```
ADD GWSELPLCY: SUBRANGE=ALL_USER, APNNI="HUAWEI1.com", SLCTM=CUSTOMIZE_APN-OI_REPLACEMENT, LABEL=UEACCCAP;
```

//可选：为具备GUL能力的UE设置统一使用.gprs后缀的APN DNS域名。

```
ADD APNDNS: APNNI="HUAWEI1.com", UEACCCAP=GERAN/UTRAN/EUTRAN_UE, DNSPOLICY=GPRS;
```

//可选：配置添加了UE接入能力属性的APN对应的网关的IP地址。

```
ADD IPV4DNSH: HSINDEX=2, HOSTNAME="HUAWEI1.com.EPC.MNC123.MCC123.GPRS", ADDRSECTION=SECTION1, IPV4ADDR1="10.2.3.4";
```

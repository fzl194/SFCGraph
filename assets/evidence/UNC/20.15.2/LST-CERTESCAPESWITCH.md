## [使用实例](#ZH-CN_MMLREF_0000001606110022)

查询证书过期逃生开关状态。

```
%%LST CERTESCAPESWITCH:;%% 
RETCODE = 0  操作成功
  
操作结果如下 
------------ 
证书过期逃生开关状态  =  关 
(结果个数 = 1)  
---    END
```

## [操作场景](#ZH-CN_OPI_0000001962486537)

两个负荷分担关系的UPF对接同一对VxLAN交换机/DCGW或对接不同对VxLAN交换机/DCGW时，本特性支持通过UPF间的N19接口为不同UPF下层二终端组的成员之间提供以太类型的通信服务。

> **说明**
> - 两个UPF是负荷分担关系，且连接到同一对DCGW/VxLAN交换机时，同一5G LAN组内成员跨UPF的互访可通过UPF间的VxLAN隧道实现或通过UPF间的N19接口实现。
> - 两个UPF是负荷分担关系，且连接到不同对DCGW/VxLAN交换机时，同一5G LAN组内成员跨UPF的互访可通过UPF与DCGW之间的VxLAN隧道实现或通过UPF间的N19接口实现。

## [任务示例](#ZH-CN_OPI_0000001962486537)

任务描述

某行业用户签约了5G LAN业务，该用户涉及多个分支机构，地理范围跨越了两个UPF的范围，两个分支机构的终端基于两个UPF的N19接口实现数据转发互通。该用户需要开启组播和N6侧上行广播功能。

脚本

//开启5G LAN核查功能。

```
SET NGLANFUNC: UECHKSW=Support, GROUPCHKSW=Support;
```

//配置UPF间基于N19通信，开启组播、N6侧上行广播和报文增殖消除功能。

```
ADD GRPNGLANFUNC: GROUPID="A0000001-460-003-A000000001", N19SUPPORT=Support, MULTICAST=Support, N6BROADCASTSW=Support, PKTELIMINATESSW=Support; 
```

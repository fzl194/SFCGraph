#### [命令功能](#ZH-CN_MMLREF_0000001126146148)

**适用网元：SGSN、MME**

该命令用于查询HTR局向的流控统计信息。

#### [注意事项](#ZH-CN_MMLREF_0000001126146148)

当执行此命令时，最多输出系统缓存的100个周期的数据，每个周期时长为20秒。命令的输出结果分为两部分，第一部分为最多9条“局向周期数据”；第二部分为最多91条“历史流控数据”。每个20秒统计周期结束时，系统会将上周期的数据刷新保存在“局向周期数据”中；如果上周期的数据是流控状态的，也会同时刷新保存在“历史流控数据”中。命令的显示结果会去除“局向周期数据”和“历史流控数据”中的重复数据。

#### [本地用户权限](#ZH-CN_MMLREF_0000001126146148)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126146148)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126146148)

无。

#### [输出结果说明](#ZH-CN_MMLREF_0000001126146148)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 局向索引 | 参数含义：局向索引。<br>取值范围：0~254 |
| 局向名称 | 参数含义：局向名称。<br>取值范围：1~127位字符串 |
| 统计周期序号 | 参数含义：当前统计周期的序号。<br>取值范围：0~65535 |
| 周期开始时间 | 参数含义：当前统计周期的起始日期时间。 |
| 发出请求数 | 参数含义：当前周期未被流控的请求数。<br>取值范围：0~4294967294 |
| 接收应答数 | 参数含义：当前周期处理的应答数。<br>取值范围：0~4294967294<br>说明：由于本周期统计的“接收应答数”可能是上一周期发出的请求返回的，因此可能出现<br>“接收应答数”<br>大于<br>“发出请求数”<br>的场景；另外由于本周期的请求同样也可能要在下一周期才会收到应答，因此总体统计上是相对均匀的。 |
| 被流控的请求数 | 参数含义：当前周期被流控的请求数。<br>取值范围：0~4294967294 |
| 周期WAL | 参数含义：当前周期的WAL(Window Access Limit)值。WAL是指系统每周期允许的<br>“发出请求数”<br>。<br>取值范围：0~4294967294 |
| 周期成功率 | 参数含义：当前周期的成功率(%)，指<br>“接收应答数”<br>/<br>“发出请求数”<br>的值。<br>取值范围：0~100 |
| 流程平均时延 | 参数含义：当前周期采样的流程平均时延（单位：毫秒），指“注册流程”在系统中被处理的时长。<br>取值范围：0~20000 |

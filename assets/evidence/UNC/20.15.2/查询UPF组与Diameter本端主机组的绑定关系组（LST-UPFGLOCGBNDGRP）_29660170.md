#### [命令功能](#ZH-CN_CONCEPT_0000201429660170)

**适用NF：PGW-C、SMF**

此命令用于查询UPF组与Diameter本端主机组的绑定关系组。

#### [注意事项](#ZH-CN_CONCEPT_0000201429660170)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000201429660170)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000201429660170)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000201429660170)

查询UPF组与Diameter本端主机组的绑定关系组记录：

```
LST UPFGLOCGBNDGRP:;
```

```

RETCODE = 0 操作成功。

-----------------------------

(结果个数 = 1)
---    END
```

## [操作场景](#ZH-CN_OPI_0164082494)

本操作指导介绍在运行网络中激活用户接入控制功能的操作过程。

用户接入控制功能是指 UNC 支持先根据IMSI号段将用户进行分类，再对于每一类用户按照用户属性进行区分而控制用户接入 GERAN、 UTRAN和 E-UTRAN网络。 UNC 按以下三种方式划分用户属性并实施用户接入控制：

- 根据签约的ARD（Access Restriction Data）控制用户接入
- 根据签约的APN控制用户接入
- 根据卡类型控制用户接入（只适用于GERAN/UTRAN接入用户）

> **说明**
> 适用于 SGSN、 MME。

## [操作步骤](#ZH-CN_OPI_0164082494)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. 配置用户接入控制功能的控制策略。
    - 增加控制用户接入GERAN网络的接入限制参数。
      [**ADD GBARD**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/接入限制管理/Gb模式接入限制参数/增加Gb模式接入限制参数(ADD GBARD)_26305284.md)
    - 增加控制用户接入UTRAN网络的接入限制参数。
      [**ADD IUARD**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/接入限制管理/Iu模式接入限制参数/增加Iu模式接入限制参数(ADD IUARD)_72345073.md)
    - 增加控制用户接入E-UTRAN网络的接入限制参数。
      [**ADD S1ARD**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/接入限制管理/S1模式接入限制参数/增加S1模式接入限制参数(ADD S1ARD)_26145478.md)
  > **说明**
  > 当只使用根据签约的ARD来控制用户接入时，如果 “APNNI” 设置为 “*” ，此时 “控制类型” 最好选择 “允许” 。

## [任务示例](#ZH-CN_OPI_0164082494)

任务描述

启用用户接入控制功能，限制条件如下：

- 所有用户拒绝接入E-UTRAN网络。
- IMSI前缀为12303001，此号段内拒绝SIM卡用户接入UTRAN网络。
- IMSI前缀为12303001，此号段内只允许SIM卡用户接入GERAN网络。
- IMSI前缀为12303001，此号段内只允许APNNI为wcdma的用户接入E-UTRAN网络。
- IMSI前缀为12303001，此号段内拒绝APNNI为wcdma的用户接入E-UTRAN网络。
- IMSI前缀为12303001，此号段内根据签约ARD允许用户接入E-UTRAN网络。
- IMSI前缀为12303001，此号段内不根据签约ARD，只根据控制类型允许用户接入E-UTRAN网络。

脚本

//打开用户接入控制功能的License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2ARD02", SWITCH=ENABLE;
```

- 脚本一
  //所有用户拒绝接入E-UTRAN网络。
  ```
  ADD S1ARD: SUBRANGE=ALL_USER, APNNI="*", ARD=NO, CTRLTYPE=REJECT, CAUSE=EPS_SERVICE_NOT_ALLOWED;
  ```
- 脚本二
  //IMSI前缀为12303001，此号段内拒绝SIM卡用户接入UTRAN网络。
  ```
  ADD IUARD: SUBRANGE=SPECIAL_IMSIPRE, IMSIPRE="12303001", APNNI="*", CARDTYPE=SIM-1, ARD=NO, CTRLTYPE=REJECT, CAUSE=GPRS_SERVICE_NOT_ALLOWED;
  ```
- 脚本三
  //IMSI前缀为12303001，此号段内只允许SIM卡用户接入GERAN网络。
  ```
  ADD GBARD: SUBRANGE=SPECIAL_IMSIPRE, IMSIPRE="12303001", APNNI="*", CARDTYPE=SIM-1, ARD=NO, CTRLTYPE=ALLOW;
  ```
- 脚本四
  //IMSI前缀为12303001，此号段内只允许APNNI为wcdma的用户接入E-UTRAN网络。
  ```
  ADD S1ARD: SUBRANGE=SPECIAL_IMSIPRE, IMSIPRE="12303001", APNNI="wcdma", ARD=NO, CTRLTYPE=ALLOW;
  ```
- 脚本五
  //IMSI前缀为12303001，此号段内拒绝APNNI为wcdma的用户接入E-UTRAN网络。
  ```
  ADD S1ARD: SUBRANGE=SPECIAL_IMSIPRE, IMSIPRE="12303001", APNNI="wcdma", ARD=NO, CTRLTYPE=REJECT, CAUSE=EPS_SERVICE_NOT_ALLOWED;
  ```
- 脚本六
  //IMSI前缀为12303001，此号段内根据签约ARD允许用户接入E-UTRAN网络。
  ```
  ADD S1ARD: SUBRANGE=SPECIAL_IMSIPRE, IMSIPRE="12303001", APNNI="*", ARD=YES, CTRLTYPE=ALLOW;
  ```
- 脚本七
  //IMSI前缀为12303001，此号段内不根据签约ARD，只根据控制类型允许用户接入E-UTRAN网络。
  ```
  ADD S1ARD: SUBRANGE=SPECIAL_IMSIPRE, IMSIPRE="12303001", APNNI="*", ARD=NO, CTRLTYPE=ALLOW;
  ```

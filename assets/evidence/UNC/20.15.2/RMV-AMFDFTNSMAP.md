# 删除漫游切片默认映射配置（RMV AMFDFTNSMAP）

## [命令功能](#ZH-CN_MMLREF_0000001658572693)

**适用NF：AMF**

该命令用于删除漫游切片默认映射配置。

## [注意事项](#ZH-CN_MMLREF_0000001658572693)

- 该命令执行后只对新激活用户生效。

- 1.删除漫游切片映射配置时，对已注册用户不会立即生效。如果需要立即生效，请先执行STR NGUSRBATCHDEREG命令对漫游用户启动批量去注册任务，并指定去注册类型为需要重新注册，触发用户重新注册，获取最新切片映射关系。

#### [操作用户权限](#ZH-CN_MMLREF_0000001658572693)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001658572693)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |

| VMNC | 拜访地移动网号 | 可选必选说明：该参数在"CTRLMODE"配置为"VPLMN"时为条件必选参数。<br>参数含义：该参数用于指定拜访地移动网号。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是2~3。<br>默认值：无<br>配置原则：无 |
| HMCC | 归属地移动国家码 | 可选必选说明：该参数在"CTRLMODE"配置为"HPLMN"时为条件必选参数。<br>参数含义：该参数用于指定归属地移动国家码。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度是3。<br>默认值：无<br>配置原则：无 |
| HMNC | 归属地移动网号 | 可选必选说明：该参数在"CTRLMODE"配置为"HPLMN"时为条件必选参数。<br>参数含义：该参数用于指定归属地移动网号。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是2~3。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001658572693)

如果需要删除“控制模式”为“拜访地PLMN”、“拜访地移动国家码”为“245”、“拜访地移动网号”为“38”的漫游切片默认映射配置，执行如下命令：

```
RMV AMFDFTNSMAP: CTRLMODE=VPLMN, VMCC="123", VMNC="45";
```

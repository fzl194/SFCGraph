## [命令功能](#ZH-CN_MMLREF_0000001101480778)

**适用NF：SMF**

该命令用于查询I-SMF支持的DNAI。

## [注意事项](#ZH-CN_MMLREF_0000001101480778)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001101480778)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001101480778)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| DNAI | 数据网络访问标识符 | 可选必选说明：可选参数<br>参数含义：该参数用于指定I-SMF支持的DNAI。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是1~63。不支持空格，不区分大小写。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001101480778)

查询DNAI为“huawei.com.dnai”的I-SMF DNAI，运行以下命令：

```
%%LST ISMFDNAI: DNAI="huawei.com.dnai";%%
RETCODE = 0  操作成功

结果如下
------------------------
            数据网络访问标识符  =  huawei.com.dnai
                      DNAI类型  =  DNN级别
                           DNN  =  huawei.com
仅使用DNAI查询辅锚点的控制开关  =  使能
            (结果个数 = 1)

            ---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001101480778)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 数据网络访问标识符 | 该参数用于指定I-SMF支持的DNAI。 |
| DNAI类型 | 该参数用于指定DNAI的类型。 |
| DNN | 该参数用于指定该DNAI关联的DNN，即用户请求的DNN。 |
| 仅使用DNAI查询辅锚点的控制开关 | 该参数用于控制基于当前DNAI查询辅锚点时，是否支持仅使用DNAI来查询。该开关开启时，I-SMF查询辅锚点时，仅使用该DNAI来查询。开关关闭时，I-SMF查询辅锚点时，使用DNN+DNAI来查询。 |

## [操作场景](#ZH-CN_OPI_0304444260)

用户面业务通道保活是指，企业园区场景下，当园区gNodeB、UPF跟园区外部署的AMF、SMF失联时，即N2/N4接口不可用但N3接口可用时，弱移动性（不跨gNodeB切换）连接态用户的数传业务能够继续进行（惯性运行）。

> **说明**
> 适用于AMF。

## [操作步骤](#ZH-CN_OPI_0304444260)

1. 进入 “MML命令行-UNC” 窗口。
2. 配置DNN群组。
  [**ADD DNNGRP**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/通用配置管理/DNN群组标识管理/增加DNN群组（ADD DNNGRP）_64343822.md)
3. 为DNN群组添加DNN成员。
  [**ADD DNNGRPMEM**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/通用配置管理/DNN群组成员管理/增加DNN群组成员（ADD DNNGRPMEM）_64343823.md)
4. 增加区域编码。
  [**ADD AREACODE**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/通用配置管理/通用区域编码管理/增加区域编码（ADD AREACODE）_44006351.md)
5. 为指定的区域增加跟踪区成员。
  [**ADD AREAMEM**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/通用配置管理/通用区域成员管理/增加跟踪区成员（ADD AREAMEM）_44006352.md)
6. **可选：**增加园区策略，设置园区惯性运行开关。
  **[ADD NGPARKPLCY](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/移动性管理/园区策略管理/增加园区策略（ADD NGPARKPLCY）_06399906.md)**

## [任务示例](#ZH-CN_OPI_0304444260)

任务描述

开启业务园区惯性运行开关，当N2和N4接口同时故障时，连接态用户惯性运行。

脚本

//配置惯性运行DNN和区域。

```
ADD DNNGRP: DNNGRPID="BIG_GROUP";
ADD DNNGRPMEM: DNNGRPID="BIG_GROUP", DNN="DNNa";
ADD AREACODE: AREACODE="huawei";
ADD AREAMEM: AREACODE="huawei", MCC="460", MNC="03", BGNTAC="120101", ENDTAC="999999";
```

//设置园区惯性运行开关。

```
ADD NGPARKPLCY: PARKID="park", AREACODE="huawei", DNNGRPID="BIG_GROUP", INERTIA=ON;
```

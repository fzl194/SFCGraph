## [命令功能](#ZH-CN_MMLREF_0000001439242821)

该命令用于查询vProbe与报表服务器之间的链路状态。

## [注意事项](#ZH-CN_MMLREF_0000001439242821)

执行命令前请确认vProbe服务处于上线状态，可通过DSP FUNCTIONSETINFO命令查询确认。

#### [操作用户权限](#ZH-CN_MMLREF_0000001439242821)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001439242821)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| VPROBESVRNAME | vProbe报表服务器名称 | 可选必选说明：可选参数<br>参数含义：该参数用于请求系统显示到指定vProbe报表服务器的链路状态；若不输入，则显示本端到所有vProbe报表服务器的链路状态信息。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是0~32。<br>默认值：无<br>配置原则：无 |
| PROTOCOLTYPE | 协议类型 | 可选必选说明：可选参数<br>参数含义：该参数用于请求系统显示指定协议类型的链路状态；若不输入，则显示环境中所有协议类型的信息。<br>数据来源：本端规划<br>取值范围：<br>- TCP（TCP）<br>- SFTP（SFTP）<br>- UDP（UDP）<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001439242821)

运营商A查询vProbe本端与报表服务器的链路状态信息：

```
%%DSP VPROBELNKSTAT:;%%
RETCODE = 0  操作成功

结果如下
--------
vProbe Pod ID     vProbe报表服务器名称  本端IP类型  本端IPv4地址   本端IPv6地址  本端端口  对端IP类型  对端IPv4地址    对端IPv6地址  对端端口  协议类型  链路状态  

vprobeexec-pod-1  VPROBESVR1            IPv4        10.185.23.253  ::            6001      IPv4        10.180.211.253  ::            10500     TCP       正常      
vprobeexec-pod-0  VPROBESVR1            IPv4        10.185.23.253  ::            6000      IPv4        10.180.211.253  ::            10500     TCP       正常      
(结果个数 = 2)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001439242821)

| 输出项名称 | 输出项解释 |
| --- | --- |
| vProbe Pod ID | 该参数用于返回VPROBEEXEC服务的Pod ID。 |
| vProbe报表服务器名称 | 该参数用于请求系统显示到指定vProbe报表服务器的链路状态；若不输入，则显示本端到所有vProbe报表服务器的链路状态信息。 |
| 本端IP类型 | 该参数用于返回vProbe本端的IP地址类型。<br>取值说明：<br>- IPv4（IPv4）<br>- IPv6（IPv6） |
| 本端IPv4地址 | 该参数用于返回vProbe本端的IPv4地址。 |
| 本端IPv6地址 | 该参数用于返回vProbe本端的IPv6地址。 |
| 本端端口 | 该参数用于返回vProbe本端的端口号。 |
| 对端IP类型 | 该参数用于返回vProbe报表服务器的IP地址类型。<br>取值说明：<br>- IPv4（IPv4）<br>- IPv6（IPv6） |
| 对端IPv4地址 | 该参数用于返回vProbe报表服务器的IPv4地址。 |
| 对端IPv6地址 | 该参数用于返回vProbe报表服务器的IPv6地址。 |
| 对端端口 | 该参数用于返回vProbe报表服务器的端口号。 |
| 协议类型 | 该参数用于请求系统显示指定协议类型的链路状态；若不输入，则显示环境中所有协议类型的信息。<br>取值说明：<br>- TCP（TCP）<br>- SFTP（SFTP）<br>- UDP（UDP） |
| 链路状态 | 该参数用于返回链路的状态。<br>取值说明：<br>- UP（正常）<br>- Down（中断） |

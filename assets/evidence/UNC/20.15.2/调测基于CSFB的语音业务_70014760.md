# 调测基于CSFB的语音业务

- [操作场景](#ZH-CN_OPI_0170014760__1.3.1)
- [必备事项](#ZH-CN_OPI_0170014760__1.3.2)

## [操作场景](#ZH-CN_OPI_0170014760)

本操作指导用于验证基于CSFB的语音业务功能。

## [必备事项](#ZH-CN_OPI_0170014760)

前提条件

已完成特性激活配置。

数据

该操作无需准备数据。

## [操作步骤](#ZH-CN_OPI_0170014760)

验证基于CSFB的主叫语音业务。

1. UE使用联合附着接入调测MME。
2. 查询用户MM上下文的用户状态信息。
  进入 “MML命令行-UNC” 窗口。 执行如下命令。
  [**DSP COMMMCTX**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/融合接入业务管理/融合用户数据库管理/显示移动性管理上下文的相关信息（DSP COMMMCTX）_58365337.md) : QUERYOPT=BYIMSI, IMSI="IMSI";
  预期结果： “MM状态” 为 “ECM-CONNECTED” ； “VLR可靠性” 为 “可靠” ； “SGs关联状态” 为 “关联已建立” 。
3. 被测UE主动发起语音业务，语音业务执行成功。

验证基于CSFB的被叫语音业务。

4. UE使用联合附着接入调测MME。
5. 查询用户MM上下文的用户状态信息。
  进入 “MML命令行-UNC” 窗口。 执行如下命令。
  [**DSP COMMMCTX**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/融合接入业务管理/融合用户数据库管理/显示移动性管理上下文的相关信息（DSP COMMMCTX）_58365337.md) : QUERYOPT=BYIMSI, IMSI="IMSI";
  预期结果： “MM状态” 为 “ECM-CONNECTED” ； “VLR可靠性” 为 “可靠” ； “SGs关联状态” 为 “关联已建立” 。
6. 被叫UE接收语音业务，被叫语音业务执行成功。

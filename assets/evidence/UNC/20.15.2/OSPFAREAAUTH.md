#### [命令功能](#ZH-CN_CONCEPT_0000001549961934)

该命令用于查询OSPF区域认证配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001549961934)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549961934)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549961934)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | OSPF进程号 | 可选必选说明：可选参数<br>参数含义：OSPF进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |
| AREAID | 区域号 | 可选必选说明：可选参数<br>参数含义：区域号。<br>数据来源：全网规划<br>取值范围：IPv4地址类型。点分十进制格式。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549961934)

查询OSPF区域认证配置：

```
LST OSPFAREAAUTH:;
```

```

RETCODE = 0  操作成功。

结果如下
------------------------
                   OSPF进程号  =  1
                       区域号  =  0.0.0.0
                     认证模式  =  HMAC-SHA256

             密文验证字标识符  =  100
                 KeyChain名称  =  NULL
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001549961934)

| 输出项名称 | 输出项解释 |
| --- | --- |
| OSPF进程号 | 该参数指定用户创建的OSPF进程号。 |
| 区域号 | 该参数指定用户创建的OSPF区域号。 |
| 认证模式 | 该参数指定OSPF认证模式。 |
| 密文口令类型 | 密文口令类型，当认证模式配置为HMAC-SHA256、Simple、MD5或HMAC-MD5时生效。 |
| MD5/HMAC-MD5/HMAC-SHA256 密码 | MD5/HMAC-MD5/HMAC-SHA256 密码，当认证模式配置为HMAC-SHA256，MD5或HMAC-MD5时生效。 |
| 简单认证密码 | 简单认证密码，当认证模式配置为Simple时生效。 |
| 密文验证字标识符 | 密文验证字标识符，当认证模式配置为HMAC-SHA256，MD5或HMAC-MD5时生效。 |
| KeyChain名称 | KeyChain名称，当认证模式配置为KeyChain时生效。 |

## [命令功能](#ZH-CN_MMLREF_0264343877)

**适用NF：PGW-C**

该命令用于查询APN速率控制配置信息。

## [注意事项](#ZH-CN_MMLREF_0264343877)

无

#### [操作用户权限](#ZH-CN_MMLREF_0264343877)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0264343877)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| APN | APN名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定APN实例名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~63。只能由“-”、数字、大小写字母和“.”组成，不能以“.”开头且不能出现连续两个“.”。不支持空格及“_”、“#”、“$”、“&”、“%”、“^”、“（”、“）”、“，”、“/”、“;”、“:”、“””、“`”等特殊字符，不区分大小写。<br>默认值：无<br>配置原则：<br>该参数使用<br>[**ADD APN**](../../../APN管理/APN/增加APN配置（ADD APN）_09653747.md)<br>命令配置生成。 |

## [使用实例](#ZH-CN_MMLREF_0264343877)

- 查询APN名称为test的APN速率控制配置信息：
  ```
  %%LST APNRATECTRL: APN="test";%%
  RETCODE = 0  操作成功

  结果如下
  --------
              APN  =  test
  APN速率控制开关  =  不使能
     上行时间单位  =  不限制
     最大上行速率  =  0
     下行时间单位  =  不限制
     最大下行速率  =  0
  (结果个数 = 1)

  ---    END
  ```
- 查询所有的APN速率控制配置信息：
  ```
  %%LST APNRATECTRL:;%%
  RETCODE = 0  操作成功

  结果如下
  --------
  APN                   APN速率控制开关  上行时间单位  最大上行速率  下行时间单位  最大下行速率  

  0168apn1.com          不使能           不限制        0             不限制        0             
  0168apn2.com          不使能           不限制        0             不限制        0             
  (结果个数 = 2)

  ---    END
  ```

## [输出结果说明](#ZH-CN_MMLREF_0264343877)

| 输出项名称 | 输出项解释 |
| --- | --- |
| APN名称 | 该参数用于指定APN实例名。 |
| APN速率控制开关 | 该参数用于控制开启和关闭APN速率控制功能。 |
| 上行时间单位 | 该参数用于表示APN速率控制的上行时间单位。 |
| 最大上行速率 | 该参数用于表示单位时间内可发送的最大上行数据包数。 |
| 下行时间单位 | 该参数用于表示APN速率控制的下行时间单位。 |
| 最大下行速率 | 该参数用于表示单位时间内可发送的最大下行数据包数。 |

## [操作场景](#ZH-CN_OPI_0168260818)

本操作指导介绍在运行网络中激活基于LTE的网络共享（GWCN）特性的操作过程。

> **说明**
> 适用于MME。

## [必备事项](#ZH-CN_OPI_0168260818)

前提条件

- 请仔细阅读[WSFD-207004 基于LTE的网络共享（GWCN）特性概述](特性概述_68358158.md)。
- 当UNC被多PLMN共享时，UNC需要和多个PLMN的核心网设备（例如S-GW/P-GW、MSC、HSS等）连接，故在部署本特性之前，需要完成和其他核心网网元的对接配置。
- 到eNodeB的基本数据已经配置完成，配置方法见[配置到eNodeB的数据](../../../../初始配置/UNC初始配置与调测/组网对接配置/配置AMF&MME&SGSN/配置MME/配置到eNodeB的数据_82626680.md)。
- [WSFD-104401 支持多HPLMN功能](../../组网功能/WSFD-104401 支持多HPLMN功能_63530664.md)和[WSFD-207006 基于LTE的MVNO](../WSFD-207006 基于LTE的MVNO_68260820.md)中至少需要激活一个特性。
- 已完成加载License。

## [操作步骤](#ZH-CN_OPI_0168260818)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开基于LTE的网络共享（GWCN）功能开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. **可选：**配置“支持多HPLMN”，请参见 [激活支持多HPLMN功能（适用于SGSN/MME）](../../组网功能/WSFD-104401 支持多HPLMN功能/激活支持多HPLMN功能（适用于SGSN_MME）_70150791.md) 。
  > **说明**
  > 在如下情况下，需要执行本步骤：
  >
  > - UNC被多个运营商共享。
  > - UNC由一个运营商部署，但是该运营商拥有多个PLMN。
4. **可选：**配置 UNC 支持MVNO，请参见 [激活基于LTE的MVNO](../WSFD-207006 基于LTE的MVNO/激活基于LTE的MVNO_68260841.md) 。
  > **说明**
  > 当 UNC 被租用给拥有独立PLMN的虚拟运营商时，需要执行本步骤。
5. 配置共享MME的PLMN ID。eNodeB需要在系统广播消息中下发此PLMN，此PLMN可以是HPLMN，也可以是MVNO的PLMN。MME侧必须配置该命令，否则可能导致MME不能识别eNodeB发来的S1 Setup Request消息中携带的PLMN，从而返回S1 Setup Failure。
  [**ADD MMESHAREPLMN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/MME POOL区管理/MME共享管理/增加MME的共享PLMN(ADD MMESHAREPLMN)_26146086.md)
6. **可选：**增加SGs接口配置，为每个PLMN（LAI）配置对应的VLR。
  > **说明**
  > 当eNodeB支持多HPLMN时，需要执行本步骤。
    a. 添加LAI与VLR号对应关系。
      [**ADD LAIVLR**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/电路域联合业务/LAI与VLR号对应关系/增加LAI与VLR号对应关系(ADD LAIVLR)_72345015.md)
    b. 添加TAI与LAI的对应关系。
      **[ADD TAILAI](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/电路域联合业务/TAI与LAI对应关系/增加TAI与LAI对应关系(ADD TAILAI)_72345017.md)**

## [任务示例](#ZH-CN_OPI_0168260818)

任务描述

运营商网络中未部署语音业务， UNC 被Operator A和Operator B共享。Operator A的MNO ID为0，PLMN ID为12321。Operater B的MNO ID为128，PLMN ID为46122。（WSFD-104401 支持多HPLMN功能特性的License已经开启。）

脚本

//开启GWCN功能。

```
SET LICENSESWITCH: LICITEM="LKV2GWCN03", SWITCH=ENABLE;
```

//增加MNO配置信息。对于Operator A，系统初始化的时候已经默认存在MNOID为0的记录，不需要单独在此特性中配置。

```
ADD MNO: MNOID=128;
```

//打开运营商网络名称下发开关，当4G终端接入时，系统允许将运营商网络名称携带给终端。

```
SET MMFUNC: MMINFO=S1_MODE-1, EMMINFOIEPLY=NETWORK_NAME-1;
```

//增加本地PLMN信息。

```
ADD HPLMN: MCC="123", MNC="21", MNOID=0;
ADD HPLMN: MCC="461", MNC="22", MNOID=128;
```

//增加MME的共享信息。

```
ADD MMESHAREPLMN: MCC="123", MNC="21";
ADD MMESHAREPLMN: MCC="461", MNC="22";
```

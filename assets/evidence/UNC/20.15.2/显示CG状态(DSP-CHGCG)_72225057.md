#### [命令功能](#ZH-CN_MMLREF_0000001172225057)

**适用网元：SGSN**

该命令用于查询对端CG与本端SGSN的通信状态。

#### [注意事项](#ZH-CN_MMLREF_0000001172225057)

- 该命令执行后立即生效。
- 如果输入的RU名称中不存在，将会提示没有满足条件的记录

#### [本地用户权限](#ZH-CN_MMLREF_0000001172225057)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172225057)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172225057)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RUNAME | RU名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定要查询的资源单元名称。该参数可以通过<br>[DSP RU](../../../../平台服务管理/单体服务公共功能管理/系统管理/资源管理/RU管理/显示资源单元信息（DSP RU）_59103857.md)<br>命令查询。<br>数据来源：整网规划<br>取值范围：0 ~ 63位字符串<br>默认值：无 |

#### [使用实例](#ZH-CN_MMLREF_0000001172225057)

查询CDP进程对应的所有CG状态：

DSP CHGCG:;

```
%%DSP CHGCG:;%%
RETCODE = 0  操作成功。

输出结果如下
------------
RU名称            进程号    CG的IP地址      CG状态      GTP承载协议    CG接收端口号
      
USN_SP_RU_0067    0         192.168.253.1   通信正常    UDP            4007        
USN_SP_RU_0066    0         192.168.253.1   通信正常    UDP            4007       
(结果个数 = 2)
---    END
```

#### [输出结果说明](#ZH-CN_MMLREF_0000001172225057)

| 输出项名称 | 输出项解释 |
| --- | --- |
| RU名称 | 指定需要显示CG通信状态的资源单元。<br>取值说明：1 ~ 63位字符串 |
| CG的IP地址 | 标识与本端SGSN连接的CG的IPv4地址。<br>说明：“0.0.0.0”和“255.255.255.255”是无效的IP地址。 |
| GTP承载协议 | 标识CG支持的GTP'承载协议。<br>取值说明：<br>- UDP<br>- TCP<br>说明：- UDP表示SGSN和CG之间通过UDP/IP进行通讯。<br>- TCP表示SGSN和CG之间通过TCP/IP进行通讯。 |
| CG接收端口号 | 标识SGSN消息所要发往的端口号（亦即UDP类型CG的接收端口号）或TCP类型CG的侦听端口号。<br>说明：与对端CG网元的设置保持一致。 |
| 进程号 | 用于标识进程号。 |
| CG状态 | 用于标识CG状态。<br>取值说明：<br>- 空闲<br>- 通信中断：参考**ALM-80663 SGSN和CG通讯中断**进行处理。<br>- 通信正常<br>- 正在删除 |

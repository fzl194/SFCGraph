# 删除服务支持的切片实例（RMV SLICEINSTINFO）

- [命令功能](#ZH-CN_MMLREF_0247815994__1.3.1)
- [注意事项](#ZH-CN_MMLREF_0247815994__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0247815994__1.3.4)

## [命令功能](#ZH-CN_MMLREF_0247815994)

**适用NF：SMF、AMF、NRF、NSSF、NCG**

该命令用于删除服务支持的切片实例。

## [注意事项](#ZH-CN_MMLREF_0247815994)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0247815994)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0247815994)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| NSIINFO | 网络分片实例信息 | 可选必选说明：必选参数<br>参数含义：该参数用于指定网络分片实例信息。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~40。本参数的构成字符只能是字母A～Z或a～z、数字0～9和中划线"-"，且不能以中划线“-”开头或结尾。<br>默认值：无<br>配置原则：无 |
| NSSIID | 网络分片子网实例标识 | 可选必选说明：必选参数<br>参数含义：该参数用于指定网络分片子网实例标识。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~40。本参数的构成字符只能是字母A～Z或a～z、数字0～9和中划线"-"，且不能以中划线“-”开头或结尾。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0247815994)

删除一个网络分片实例信息为“NSSI-C-001-HDBNJ-NSSMF-01-A-HW”，网络分片子网实例标识为“NSSI-C-001-HDBNJ-NSSMF-01-A-HW01”的服务支持的切片实例：

```
RMV SLICEINSTINFO：NSIINFO="NSSI-C-001-HDBNJ-NSSMF-01-A-HW"； NSSIID="NSSI-C-001-HDBNJ-NSSMF-01-A-HW01"；
```

# 删除NF局向性能统计对象（RMV PERFPEERNF）

- [命令功能](#ZH-CN_MMLREF_0209653833__1.3.1)
- [注意事项](#ZH-CN_MMLREF_0209653833__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0209653833__1.3.4)
- [使用实例](#ZH-CN_MMLREF_0209653833__1.3.5)

## [命令功能](#ZH-CN_MMLREF_0209653833)

**适用NF：AMF、SMF、SMSF、NCG**

该命令用于删除NF局向性能统计对象。

## [注意事项](#ZH-CN_MMLREF_0209653833)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0209653833)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209653833)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| NFINSTANCEID | NF实例号 | 可选必选说明：必选参数<br>参数含义：该参数表示对端局向NF实例号。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是1~40。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209653833)

删除“NF实例号”为“AMF_INSTANCE_001”的NF局向性能统计对象：

```
RMV PERFPEERNF: NFINSTANCEID="AMF_INSTANCE_001";
```

#### [命令功能](#ZH-CN_CONCEPT_0000001550281250)

该命令用来设置ND快回使能配置。

大量客户端发送ND请求报文会造成CPU负载过高。为了降低CPU负载，可通过配置此命令，使设备接收到ND请求报文后不上送CPU处理，直接转发ND响应报文给客户端。

#### [注意事项](#ZH-CN_CONCEPT_0000001550281250)

- 该命令执行后立即生效。
- 该命令仅适用于非NP卡场景和NP卡非加速模式场景。
- 该命令存在系统初始记录，参数的初始设置值如下表：

| FASTRLYEN |
| --- |
| TRUE |

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550281250)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550281250)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| FASTRLYEN | 使能标记 | 可选必选说明：必选参数<br>参数含义：使能标记。<br>数据来源：本端规划<br>取值范围：布尔类型，输入格式为“TRUE”或者“FALSE”。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001550281250)

设置ND快回使能配置：

```
SET SECND:FASTRLYEN=TRUE;
```

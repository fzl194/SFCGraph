## [命令功能](#ZH-CN_MMLREF_0000001089951698)

本命令用于查询安全日志。

该命令获取日志与 OM Portal 界面 “ 安全>日志审计>安全日志 ” 相同。

## [注意事项](#ZH-CN_MMLREF_0000001089951698)

最大限度为1000条。如果超过最大限度，则返回满足条件的最新的1000条。

## [参数说明](#ZH-CN_MMLREF_0000001089951698)

| **参数标识** | **参数名称** | **参数说明** |
| --- | --- | --- |

| LEVEL | 级别 | 可选必选说明：可选参数。<br>参数含义：用于命令按照操作级别来查询安全日志；若不输入，则表示查询前1000条安全日志。<br>取值范围：<br>- Minor(次要)<br>- Important(重要)<br>- Urgent(紧急)<br>默认值：无。<br>配置原则：无。 |
| RESULT | 操作结果 | 可选必选说明：可选参数。<br>参数含义：用于命令按照操作结果状态来查询安全日志；若不输入，则表示查询前1000条安全日志。<br>取值范围：<br>- Successful(成功)<br>- Failed(失败)<br>- PartiallySuccessful(部分成功)<br>- Executing(执行中)<br>默认值：无。<br>配置原则：无。 |
| SDT | 开始时间 | 可选必选说明：可选参数。<br>参数含义：用于命令按照安全日志开始时间来查询安全日志；若不输入，则表示查询前1000条安全日志。<br>取值范围：1990/01/01 00:00:00-2037/12/31 23:59:59。<br>默认值：无。<br>配置原则：无。 |
| EDT | 结束时间 | 可选必选说明：可选参数。<br>参数含义：用于命令按照安全日志结束时间来查询安全日志；若不输入，则表示查询前1000条安全日志。<br>取值范围：1990/01/01 00:00:00-2037/12/31 23:59:59。<br>默认值：无。<br>配置原则：无。 |
| OPERATIONNAME | 操作名称 | 可选必选说明：可选参数。<br>参数含义：用于命令按照操作名称来查询安全日志；若不输入，则表示查询前1000条安全日志。<br>取值范围：字符串类型，输入长度范围为0~100。<br>默认值：无。<br>配置原则：无。 |

## [使用实例](#ZH-CN_MMLREF_0000001089951698)

查询安全日志：

```
%%LST SECLOG:;%%
RETCODE = 0  操作成功

结果信息 
-------- 
操作名称                 级别   操作用户    用户类型    网元名称   开始时间              操作源IP      服务名称      操作结果    接入方式       详细信息

查询时间控制策略         重要   admin       本地用户    faircat    2020-04-17 00:58:58   10.0.0.0       Privilege    成功        本地终端       查询登录时间策略。 
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001089951698)

该命令执行异常，会返回对应的错误码。常见的错误码如 [表2](#ZH-CN_MMLREF_0000001089951698__table0350152692312) 所示。

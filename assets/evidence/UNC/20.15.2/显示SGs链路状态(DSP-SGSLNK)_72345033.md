#### [命令功能](#ZH-CN_MMLREF_0000001172345033)

**适用网元：MME**

此命令用于查看SGs链路状态。

#### [注意事项](#ZH-CN_MMLREF_0000001172345033)

无

#### [本地用户权限](#ZH-CN_MMLREF_0000001172345033)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172345033)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172345033)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SRT | 查询方式 | 可选必选说明：可选参数<br>参数含义：该参数用于指定查询SGs链路状态的方式。<br>取值范围：<br>- “ALL(所有)”：查询所有SGs链路的状态。<br>- “LNK(链路)”：通过指定链路索引查询SGs链路的状态。<br>- “LSX(链路集)”：通过指定链路集查询SGs链路的状态。<br>- “VLR(VLR号)”：通过指定VLR号查询SGs链路的状态。<br>默认值：无 |
| LNK | 链路索引 | 可选必选说明：条件可选参数<br>参数含义：待查询链路的索引。<br>前提条件：该参数在<br>“查询方式”<br>设置为<br>“LNK(链路)”<br>时生效。<br>取值范围：0~511<br>默认值：无 |
| LSX | 链路集索引 | 可选必选说明：条件可选参数<br>参数含义：待查询链路所属链路集的索引。<br>前提条件：该参数在<br>“查询方式”<br>设置为<br>“LSX(链路集)”<br>时生效。<br>取值范围：0~255<br>默认值：无 |
| VN | VLR号 | 可选必选说明：条件可选参数<br>参数含义：待查询链路的VLR号。<br>前提条件：该参数在<br>“查询方式”<br>设置为<br>“VLR(VLR号)”<br>时生效。<br>取值范围：1~15位十进制数字<br>默认值：无 |

#### [使用实例](#ZH-CN_MMLREF_0000001172345033)

1. 查询系统内所有的SGs链路状态信息：
  DSP SGSLNK: SRT=ALL;

  ```
  RETCODE = 0  操作成功。

  操作结果如下

  USN_SP_RU_0065    0              6             6                NULL        86139064209    故障           
  USN_SP_RU_0064    0              5             5                NULL        86139064208    故障           
  USN_SP_RU_0065    1              4             4                NULL        86139064207    故障           
  USN_SP_RU_0064    1              3             3                NULL        86139064206    故障

  ```
2. 查询指定VLR号的SGs链路状态信息：
  DSP SGSLNK: SRT=VLR, VN="1230301";
  ```
  %%DSP SGSLNK: SRT=VLR, VN="1230301";%%
  RETCODE = 0  操作成功。

  操作结果如下
  --------------
          RU名称  =  USN_SP_RU_0064
          进程号  =  1

  ---    END
  ```

#### [输出结果说明](#ZH-CN_MMLREF_0000001172345033)

| 输出项名称 | 输出项解释 |
| --- | --- |
| RU名称 | SGs链路的资源单位名称。<br>取值说明：1～63位字符串 |
| 进程号 | SGs链路的进程号。<br>取值说明：0~11 |
| 链路索引 | SGs链路的索引。<br>取值说明：0~511 |
| 链路集索引 | SGs链路所属链路集的索引。<br>取值说明：0~255 |
| VPN名称 | SGs链路的VPN名称。<br>取值说明：1～31位字符串 |
| VLR号 | SGs链路的VLR号。<br>取值说明：1~15位十进制数字 |
| SGs链路状态 | SGs链路所处状态。<br>取值说明：<br>- 正常：表示UNC与MSC/VLR之间的通信在该链路上正常。<br>- 故障：表示UNC与MSC/VLR之间的通信在该链路上中断，需要检查网络状况，分析故障原因。详细操作参见SGs链路故障告警处理。<br>- 去激活：表示该SGs链路处于去激活状态。 |

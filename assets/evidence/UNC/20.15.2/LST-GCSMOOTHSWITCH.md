## [命令功能](#ZH-CN_MMLREF_0000001143960919)

该命令用于查询GC（Garbage Collection）平滑开关，决定流控模块计算CPU时是否需要刨除GC。

## [注意事项](#ZH-CN_MMLREF_0000001143960919)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001143960919)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001143960919)

无

## [使用实例](#ZH-CN_MMLREF_0000001143960919)

查询GC平滑开关状态。

```
LST GCSMOOTHSWITCH:;
RETCODE = 0  操作成功

结果如下
------------------------
GC平滑开关  =  打开GC平滑开关
令牌刷新周期 = 60
特定周期内最大GC刨除次数 = 10

(结果个数 = 1)
```

## [输出结果说明](#ZH-CN_MMLREF_0000001143960919)

| 输出项名称 | 输出项解释 |
| --- | --- |
| GC平滑开关 | 该参数用于标识是否需要打开GC（Garbage Collection）平滑开关。 |
| 令牌刷新周期 | 该参数用于设置GC（Garbage Collection）平滑采样的周期，单位秒。 |
| 特定周期内最大GC刨除次数 | 该参数用于表示特定周期内最大GC（Garbage Collection）刨除次数。 |

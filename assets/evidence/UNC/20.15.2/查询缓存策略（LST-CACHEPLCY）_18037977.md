## [命令功能](#ZH-CN_MMLREF_0000001318037977)

**适用NF：AMF、SMF、SMSF、NCG、NSSF**

该命令用于查询缓存策略。

## [注意事项](#ZH-CN_MMLREF_0000001318037977)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001318037977)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001318037977)

无

## [使用实例](#ZH-CN_MMLREF_0000001318037977)

查询缓存策略。

```
%%LST CACHEPLCY:;%%
RETCODE = 0  操作成功

结果如下
------------------------
        缓存策略  =  全部缓存模式
快速老化时间(秒)  =  600
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001318037977)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 缓存策略 | 该参数用于设置缓存策略。 |
| 快速老化时间(秒) | 该参数用于指定快速老化时间。当缓存策略选择“非白名单尽力而为模式”，且缓存规格达到上限时，持续未被发现时间达到快速老化时间的不在白名单中的异网NF数据将会从缓存中删除。 |

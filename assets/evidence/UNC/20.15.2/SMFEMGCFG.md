# 查询运营商紧急呼叫会话功能配置（LST SMFEMGCFG）

- [注意事项](#ZH-CN_MMLREF_0000001090836434__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0000001090836434__1.3.4)
- [使用实例](#ZH-CN_MMLREF_0000001090836434__1.3.5)

## [命令功能](#ZH-CN_MMLREF_0000001090836434)

**适用NF：SMF**

该命令用于查询指定的MNO或MVNO对应的紧急呼叫会话功能配置数据。

## [注意事项](#ZH-CN_MMLREF_0000001090836434)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001090836434)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001090836434)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| NOID | 运营商标识 | 可选必选说明：可选参数<br>参数含义：该参数用于指定运营商标识。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围为0。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001090836434)

查询表中全部紧急呼叫会话功能配置的数据，执行如下命令：

```
%%LST SMFEMGCFG: NOID=0;%%
RETCODE = 0  操作成功

结果如下
--------
                            运营商标识  =  0
   紧急会话上行Session AMBR(千比特/秒)  =  20000
   紧急会话下行session AMBR(千比特/秒)  =  40000
                 紧急会话ARP优先级级别  =  1
   紧急会话ARP的Pre-emption Capability  =  不抢占
紧急会话ARP的Pre-emption Vulnerability  =  不可抢占
                      紧急会话创建模式  =  仅限合法用户
                       紧急会话标准5QI  =  5
                 紧急会话5QI的优先级别  =  0
               紧急会话注册UDM功能开关  =  不使能
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001090836434)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 运营商标识 | 该参数用于指定运营商标识。 |
| 紧急会话上行Session AMBR(千比特/秒) | 该参数用于指定紧急会话的上行Session AMBR比特率。 |
| 紧急会话下行session AMBR(千比特/秒) | 该参数用于指定紧急会话的下行Session AMBR比特率。 |
| 紧急会话ARP优先级级别 | 该参数用于指定紧急会话的ARP优先级级别。 |
| 紧急会话ARP的Pre-emption Capability | 该参数用于指定紧急会话ARP的Pre-emption Capability。 |
| 紧急会话ARP的Pre-emption Vulnerability | 该参数用于指定紧急会话ARP的Pre-emption Vulnerability。 |
| 紧急会话创建模式 | 该参数用于指定紧急会话创建模式。 |
| 紧急会话标准5QI | 该参数用于指定紧急会话的默认QoS Flow的5QI，协议定义IMS信令承载推荐的5QI是5。 |
| 紧急会话5QI的优先级别 | 该参数用于指定紧急会话的5QI优先级别。 |
| 紧急会话注册UDM功能开关 | 该参数用于指定本地合法用户是否支持向UDM注册紧急呼叫会话信息。 |

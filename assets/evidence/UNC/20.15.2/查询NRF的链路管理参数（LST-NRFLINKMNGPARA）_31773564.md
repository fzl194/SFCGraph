## [命令功能](#ZH-CN_MMLREF_0231773564)

**适用NF：NRF**

该命令用于查询NRF的链路管理参数。

## [注意事项](#ZH-CN_MMLREF_0231773564)

无

#### [操作用户权限](#ZH-CN_MMLREF_0231773564)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0231773564)

无

## [使用实例](#ZH-CN_MMLREF_0231773564)

查询NRF的链路管理参数：

```
LST NRFLINKMNGPARA:;
%%LST NRFLINKMNGPARA:;%%
RETCODE = 0  操作成功

结果如下
------------------------
  逻辑链路刷新间隔(秒)  =  300
      单次下发地址个数  =  20
链路信息老化时长(分钟)  =  1440
(结果个数 = 1)
```

## [输出结果说明](#ZH-CN_MMLREF_0231773564)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 逻辑链路刷新间隔(秒) | 该参数用于表示NRF重新从sbilink服务获取逻辑链路标识的时间间隔。 |
| 单次下发地址个数 | 该参数用于表示NRF在3秒定时周期内向sbilink服务下发地址消息的个数上限，为了平滑处理，避免短时间过多消息。 |
| 链路信息老化时长(分钟) | 该参数用于表示NRF上管理一些动态链路的老化时长，比如NRF分层转发迭代模式下的重定向的地址信息。 |

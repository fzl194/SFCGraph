## [命令功能](#ZH-CN_MMLREF_0000001397301786)

**适用NF：SMF**

该命令用于显示TMGI使用率。

## [注意事项](#ZH-CN_MMLREF_0000001397301786)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001397301786)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001397301786)

无

## [使用实例](#ZH-CN_MMLREF_0000001397301786)

当需要显示TMGI使用率时，执行如下命令：

```
%%DSP TMGIUSAGE:;%%
RETCODE = 0  操作成功

结果如下
--------
已分配的TMGI数量  =  0
        TMGI总数  =  2
      TMGI使用率  =  0.00%
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001397301786)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 已分配的TMGI数量 | 该参数用于表示已分配的TMGI数量。 |
| TMGI总数 | 该参数用于表示TMGI总数量。 |
| TMGI使用率 | 该参数用于表示TMGI使用率。 |

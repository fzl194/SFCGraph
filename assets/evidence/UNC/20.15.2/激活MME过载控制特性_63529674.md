## [操作场景](#ZH-CN_OPI_0163529674)

本操作指导介绍在运行网络中激活MME过载控制特性的操作过程。

UNC 设备作为MME逻辑网元运行时，当 UNC 过载时向eNodeB发送Overload Start消息，通知eNodeB拒绝UE新建连接，从而减少网络侧信令冲击。当 UNC 从过载状态恢复正到常状态后则向eNodeB发送Overload Stop消息，通知eNodeB允许UE接入，继续为UE提供服务。

> **说明**
> 适用于MME。

## [任务示例](#ZH-CN_OPI_0163529674)

任务描述

设置CPU阈值和过载控制策略。

脚本

//设置MME过载控制License开关为“ENABLE”。

```
SET LICENSESWITCH: LICITEM="LKV2POOLOC02", SWITCH=ENABLE;
```

//设置过载控制策略。

```
SET S1OVERLOAD: LV1=REJECT_ALL_NON_EMERGENCY,LV2=REJECT_ALL_NON_EMERGENCY,LV3=PERMIT_EMERGENCY_ONLY, LV4=PERMIT_EMERGENCY_ONLY, DLYTM=30, REJRATEIND=NO;
```

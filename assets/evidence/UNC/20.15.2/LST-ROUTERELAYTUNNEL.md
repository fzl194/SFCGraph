#### [命令功能](#ZH-CN_CONCEPT_0000001550281618)

该命令用于查询路由迭代隧道功能开关。

#### [注意事项](#ZH-CN_CONCEPT_0000001550281618)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550281618)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550281618)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001550281618)

查询路由迭代隧道开关：

```
LST ROUTERELAYTUNNEL:;
```

```

RETCODE = 0  操作成功

结果如下
-------------------------
迭代隧道使能开关 =  TRUE
前缀列表名       =  a
隧道策略名       =  a
(结果个数 = 1)
---    END
```

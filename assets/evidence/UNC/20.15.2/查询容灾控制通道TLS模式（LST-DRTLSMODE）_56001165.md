## [命令功能](#ZH-CN_MMLREF_0000002156001165)

该命令用于查询容灾控制通道TLS模式。

## [注意事项](#ZH-CN_MMLREF_0000002156001165)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000002156001165)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000002156001165)

无

## [使用实例](#ZH-CN_MMLREF_0000002156001165)

查询网元容灾控制通道TLS模式。

```
%%LST DRTLSMODE:;%%
RETCODE = 0  操作成功

结果如下
--------
TLS模式  =  开启
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000002156001165)

| 输出项名称 | 输出项解释 |
| --- | --- |
| TLS模式 | 该参数用于指定容灾控制通道TLS模式。 |

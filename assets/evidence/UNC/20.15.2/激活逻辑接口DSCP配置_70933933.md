## [操作场景](#ZH-CN_OPI_0170933933)

本操作指导介绍在运行网络中激活逻辑接口DSCP配置特性的操作过程。

逻辑接口DSCP配置特性是指UNC在组网的时候针对不同的逻辑接口可设置不同的DSCP值。

> **说明**
> 适用于 SGSN、 MME。

## [必备事项](#ZH-CN_OPI_0170933933)

前提条件

- 请仔细阅读[WSFD-105004 逻辑接口DSCP配置](../WSFD-105004 逻辑接口DSCP配置_62745883.md)。
- 完成加载License。

## [操作步骤](#ZH-CN_OPI_0170933933)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开特性的License配置开关。
  **[SET LICENSESWITCH](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)**
3. **可选：**配置输出IP报文的DSCP值。
  > **说明**
  > DSCP值是输出IP报文的必选字段，根据端到端网络规划和对端网元的QoS调度要求，执行下列步骤，可以调整输出IP报文的DSCP值后发送给对端网元，以便于对端网元进行QoS调度。
    - 修改输出信令报文的DSCP值。
      **[SET IFDSCP](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/QoS管理/DSCP/接口DSCP管理/设置接口DSCP配置(SET IFDSCP)_26306022.md)**
    - 透传输出数据报文的DSCP值。
      **[SET IFDSCP](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/QoS管理/DSCP/接口DSCP管理/设置接口DSCP配置(SET IFDSCP)_26306022.md)**
    - 修改输出数据报文的DSCP值。
      **[SET IFDSCP](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/QoS管理/DSCP/接口DSCP管理/设置接口DSCP配置(SET IFDSCP)_26306022.md)**
4. **可选：**配置输入报文DSCP的映射优先级。
  > **说明**
  > DSCP映射优先级，是VM接收的IP报文入队列的依据，不同的DSCP报文按照优先级表入不同的优先级队列，高优先级的业务建议配置入高优先级的队列，以便系统优先处理。
  >
  > 一般情况推荐使用系统建议值，不需要执行本步骤，即默认按照业务类型入队列。根据网络规划和设计，可以按照下面步骤调整不同DSCP值的映射优先级。
    - 修改DSCP的映射优先级
      [**MOD DSCPPRI**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/QoS管理/DSCP/DSCP优先级映射管理/修改DSCP映射优先级配置表(MOD DSCPPRI)_26146208.md)
    - 增加非默认DSCP的映射优先级
      [**ADD DSCPPRI**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/QoS管理/DSCP/DSCP优先级映射管理/增加DSCP映射优先级配置表(ADD DSCPPRI)_26306018.md)

## [任务示例](#ZH-CN_OPI_0170933933)

任务描述

修改S6a接口的DSCP值为58，满足对端网元高优先级调度的要求。

脚本

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2LIDC01", SWITCH=ENABLE;
```

//设置S6a接口的DSCP值为58。

```
SET IFDSCP: MSGCLS=SIGNALLING, INTFTP_SHOW=Diameter_PROTOCOL, DSCPV=58;
```

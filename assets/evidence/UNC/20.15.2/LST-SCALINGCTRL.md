#### [命令功能](#ZH-CN_CONCEPT_0000001172345465)

**适用网元：MME**

该命令用于查询扩缩容业务控制参数配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001172345465)

无。

#### [本地用户权限](#ZH-CN_CONCEPT_0000001172345465)

manage-ug;system-ug;monitor-ug

#### [网管用户权限](#ZH-CN_CONCEPT_0000001172345465)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组。

#### [参数说明](#ZH-CN_CONCEPT_0000001172345465)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001172345465)

查询扩缩容业务控制参数：

LST SCALINGCTRL:;

```
%%LST SCALINGCTRL:;%%
RETCODE = 0  操作成功。

输出结果如下
--------------
          暂态业务保护时长（秒） =  5
           SCTP链路自动均衡开关  =  开启
            GTP路径自动均衡开关  =  开启
(结果个数 = 1)

---    END
```

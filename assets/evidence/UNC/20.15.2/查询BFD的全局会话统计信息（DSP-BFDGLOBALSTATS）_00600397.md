#### [命令功能](#ZH-CN_CONCEPT_0000001600600397)

该命令用于查询BFD全局会话的统计信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600600397)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600600397)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600600397)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001600600397)

查询BFD全局的统计信息：

```
DSP BFDGLOBALSTATS:;
```

```

RETCODE = 0  操作成功。

结果如下
-------------------------
       BFD组件HA状态  =  正常工作
            IP会话数  =  1
         BFD会话总数  =  1
            Up会话数  =  0
          Down会话数  =  1
          静态会话数  =  1
          动态会话数  =  0
    静态自协商会话数  =  0
    已用会话标识符数  =  1
    (结果个数 = 1)
    ---  END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600600397)

| 输出项名称 | 输出项解释 |
| --- | --- |
| BFD组件HA状态 | 表示BFD组件HA状态，比如初始化、批量备份、备组件就绪、内部平滑、FEI平滑、FES平滑、应用平滑、正常工作。 |
| IP会话数 | 表示IP类型的BFD会话个数。 |
| BFD会话总数 | 表示BFD会话总个数。 |
| Up会话数 | 表示BFD会话状态为Up的个数。 |
| Down会话数 | 表示BFD会话状态为Down的个数。 |
| 静态会话数 | 表示静态BFD会话个数。 |
| 动态会话数 | 表示动态BFD会话个数。 |
| 静态自协商会话数 | 表示静态自协商BFD会话个数。 |
| 已用会话标识符数 | 表示已经使用的会话标识符个数。 |

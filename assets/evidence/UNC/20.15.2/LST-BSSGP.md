#### [命令功能](#ZH-CN_MMLREF_0000001126145988)

**适用网元：SGSN**

该命令用于查询BSSGP层系统参数。

#### [注意事项](#ZH-CN_MMLREF_0000001126145988)

无。

#### [本地用户权限](#ZH-CN_MMLREF_0000001126145988)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126145988)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126145988)

无。

#### [使用实例](#ZH-CN_MMLREF_0000001126145988)

查询BSSGP参数:

LST BSSGP:;

```
%%LST BSSGP:;%%
RETCODE = 0  操作成功。

操作结果如下
---------------
        保证Reset的时钟值(ms)  =  30000
             MS流控有效值(ms)  =  6000
          BVC复位消息重发次数  =  3
     BVC流控发送的最小间隔(s)  =  1
        创建PFC监控定时器(ms)  =  3000
          创建PFC消息重发次数  =  5
             流控缓存时间(ms)  =  12000
   PDU生存时间(centi-seconds)  =  700
                 小区拥塞门限  =  85
             小区拥塞恢复门限  =  75
闭塞小区的删除定时器时长(min)  =  0
        BSSID与NSEI的对应关系  =  NSEI
         是否关闭BSSGP小区流控 = 是
(结果个数 = 1)

---    END
```

## [操作场景](#ZH-CN_OPI_0168260841)

本操作指导介绍在运行网络中激活基于LTE的MVNO特性的操作过程。

MVNO（Mobile Virtual Network Operator）是指利用 MNO （Mobile Network Operator）授权的网络资源，提供业务（包括MNO提供的业务以及MVNO自己定制的业务），并对授权的资源进行维护的虚拟移动运营商。本特性是指MNO授权LTE网络资源。

> **说明**
> 适用于MME

[WSFD-207005 MVNO](../WSFD-207005 MVNO_85152766.md) 和 [WSFD-207006 基于LTE的MVNO](../WSFD-207006 基于LTE的MVNO_68260820.md) 可以共同部署，也可以独立部署。

- UNC作为融合网元时，应[共同部署](../WSFD-207005 MVNO/激活MVNO_85152767.md#ZH-CN_OPI_0185152767__zh-cn_opi_0130429108_cmd1058708162180641)**MVNO**和**基于LTE的MVNO**特性。
- 当UNC仅作为MME时，独立部署[WSFD-207006 基于LTE的MVNO](../WSFD-207006 基于LTE的MVNO_68260820.md)即可，操作步骤参见[未部署 **MVNO** 的情况下，部署 **基于LTE的MVNO**](#ZH-CN_OPI_0168260841__sep_one)。
- 当UNC已经作为SGSN部署了**MVNO**的情况下，UNC新增MME的功能，升级为融合网元，需要部署[WSFD-207006 基于LTE的MVNO](../WSFD-207006 基于LTE的MVNO_68260820.md)，操作步骤参见[部署 **MVNO** 的情况下，部署 **基于LTE的MVNO**](#ZH-CN_OPI_0168260841__sep_two)。

## [必备事项](#ZH-CN_OPI_0168260841)

前提条件

- 请仔细阅读[WSFD-207006 基于LTE的MVNO特性概述](特性概述_68358164.md)。
- 本特性只有获得了License许可后才能获得该特性的服务，对应的License控制项为“基于LTE的MVNO”。
- 当UNC被共享时，UNC需要和其他核心网设备（例如S-GW/P-GW、MSC、HSS等）连接，故在部署本特性之前，需要完成和其他核心网网元的对接配置。

## [操作步骤](#ZH-CN_OPI_0168260841)

- 进入 “MML命令行-UNC” 窗口。
- 未部署 **MVNO** 的情况下，部署 **基于LTE的MVNO**
    1. 打开 [WSFD-207006 基于LTE的MVNO](../WSFD-207006 基于LTE的MVNO_68260820.md) 特性的License配置开关。
      [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
    2. 增加运营商网络标识。
          a. 打开运营商网络名称下发开关，当4G终端接入时，系统允许将运营商网络名称携带给终端。
            [**SET MMFUNC**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/MM扩展功能管理/设置移动性管理扩展功能(SET MMFUNC)_26145512.md)
          b.
            增加运营商网络名称。

            > **说明**
            > 若系统中同时存在命令 [**ADD MVNO**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/归属网络运营商管理/MVNO管理/MVNO配置表/增加MVNO配置信息(ADD MVNO)_72225747.md) 和 [**ADD USRMMINFO**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/归属网络运营商管理/网络名称管理/增加网络名称(ADD USRMMINFO)_26146058.md) 的配置记录，系统会优先将命令 [**ADD USRMMINFO**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/归属网络运营商管理/网络名称管理/增加网络名称(ADD USRMMINFO)_26146058.md) 配置的运营商网络名称下发给终端。
                  - 增加MVNO标识。
                    [**ADD MVNO**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/归属网络运营商管理/MVNO管理/MVNO配置表/增加MVNO配置信息(ADD MVNO)_72225747.md)
                  - **可选：**
                    当需要特定IMSI范围的用户增加特定网络名称时执行以下步骤。
                    [**ADD USRMMINFO**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/归属网络运营商管理/网络名称管理/增加网络名称(ADD USRMMINFO)_26146058.md)
    3. 增加MVNO网络配置信息。
      [**ADD MVNONET**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/归属网络运营商管理/MVNO管理/MVNO网络标识配置表/增加MVNO网络配置信息(ADD MVNONET)_72345663.md)
    4. 增加MVNO资源配置信息。
      [**ADD MVNORES**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/归属网络运营商管理/MVNO管理/MVNO资源配置表/增加MVNO资源配置信息(ADD MVNORES)_72345665.md)
    5. **可选：**当MVNO签订了漫游协议时，增加互联PLMN配置信息。
      [**ADD CONNECTPLMN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/互联PLMN管理/增加互联PLMN(ADD CONNECTPLMN)_26305852.md)
      > **说明**
      > 此处的 “NOID（运营商标识）” 是指MVNO标识。
    6. **可选：**配置本端标识的选择规则。
      > **说明**
      > 当MVNO和MNO使用不同本端标识时，需要执行本步骤。
      本端标识是Diameter

任务描述

**示例1** ：MVNO通过租用IMSI号段方式向用户提供业务，MVNO和MNO使用相同的本端标识。MVNO相关数据收集如 [表1](#ZH-CN_OPI_0168260841__MVNO) 所示。

*表1 MVNO数据（租用IMSI号段）*

| 数据 | 取值 |
| --- | --- |
| MVNO的标识 | 1 |
| MVNO的名称 | MVNO1 |
| 属于该MVNO的IMSI号段 | 12302111 |
| MVNO的国家码 | 86 |
| 允许的4G建立承载数 | 10,000 |
| 允许的4G用户数 | 10,000 |

**示例2** ：MVNO拥有独立的PLMN，通过租用MME向用户提供业务，MVNO和MNO使用不同的本端标识。MVNO相关数据收集如 [表2](#ZH-CN_OPI_0168260841__table_03B5F385) 所示。

脚本

- **示例1** ：MVNO通过租用IMSI号段方式向用户提供业务，MVNO和MNO使用相同的本端标识。
  //打开License配置开关。
  ```
  SET LICENSESWITCH: LICITEM="LKV2MVNO03", SWITCH=ENABLE;
  ```
  //打开运营商网络名称下发开关，当4G终端接入时，系统允许将运营商网络名称携带给终端。
  ```
  SET MMFUNC: MMINFO=S1_MODE-1, EMMINFOIEPLY=NETWORK_NAME-1;
  ```
  //增加MVNO标识。
  ```
  ADD MVNO: MVNOID=1, MVNON="MVNO1";
  ```
  //增加MVNO网络配置信息。
  ```
  ADD MVNONET: MVNOMCC="123", MVNOMNC="02", MATCHIMSI="111", MVNOCC="86", MVNOID=1;
  ```
  //增加MVNO资源配置信息。
  ```
  ADD MVNORES: MVNOID=1, SAESNUM=10000, SBRNUM=10000;
  ```
- **示例2** ：MVNO拥有独立的PLMN，通过租用MME向用户提供业务，MVNO和MNO使用不同的本端标识。
  //打开License配置开关。
  ```
  SET LICENSESWITCH: LICITEM="LKV2MVNO03", SWITCH=ENABLE;
  ```
  //打开运营商网络名称下发开关，当4G终端接入时，系统允许将运营商网络名称携带给终端。
  ```
  SET MMFUNC: MMINFO=S1_MODE-1, EMMINFOIEPLY=NETWORK_NAME-1;
  ```
  //增加MVNO标识。
  ```
  ADD MVNO: MVNOID=1, MVNON="MVNO1";
  ```
  //增加MVNO网络配置信息。
  ```
  ADD MVNONET: MVNOMCC="461", MVNOMNC="23", MVNOCC="86", MVNOID=1;
  ```
  //增加MVNO资源配置信息。
  ```
  ADD MVNORES: MVNOID=1, SAESNUM=10000, SBRNUM=10000;
  ```
  //配置本端标识的选择规则。
  ```
  ADD HNOINFO: NOID=1, LOINDEX=1;
  ```

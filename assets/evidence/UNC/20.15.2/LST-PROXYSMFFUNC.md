## [命令功能](#ZH-CN_MMLREF_0000001823622954)

**适用NF：SMF**

该命令用于查询PROXYSMFFUNC参数。

## [注意事项](#ZH-CN_MMLREF_0000001823622954)

该命令参数NRFPRISW已弃用。

#### [操作用户权限](#ZH-CN_MMLREF_0000001823622954)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001823622954)

无

## [使用实例](#ZH-CN_MMLREF_0000001823622954)

查询proxy SMF控制配置。

```
%%LST PROXYSMFFUNC:;%%
RETCODE = 0 操作成功
结果如下
--------
                               归属地SMF实例标识的查询 = IMSI优先
                   是否优先选择支持互操作能力的归属地SMF =  开启
             是否选择相同DNN和不同S-NSSAI会话的归属地SMF = 关闭
                            是否优先向NRF发现归属地SMF = 关闭
                          是否根据接入类型过滤归属地SMF = 关闭
                                            控制类型 = 允许
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001823622954)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 归属地SMF实例标识的查询 | 该参数用于控制Proxy SMF特性中根据本地IMSI或者MSISDN查找归属地SMF实例标识。 |
| 是否优先选择支持互操作能力的归属地SMF | 该参数用于控制Proxy SMF是否优先选择支持互操作能力的归属地SMF。 |
| 是否选择相同DNN和不同S-NSSAI会话的归属地SMF | 该参数用于控制Proxy SMF是否选择相同DNN和不同S-NSSAI会话的归属地SMF。 |
| 是否优先向NRF发现归属地SMF | 该参数用于控制Proxy SMF是否优先向NRF发现归属地SMF。 |
| 是否根据接入类型过滤归属地SMF | 该参数用于控制Proxy SMF是否根据接入类型过滤归属地SMF。 |
| 控制类型 | 该参数用于控制Proxy SMF特性接入列表功能中IMSI和MSISDN同时匹配时是否允许接入。 |

## [命令功能](#ZH-CN_MMLREF_0000001088422284)

本命令用于查询网络亚健康参数。

> **说明**
> 该命令仅在Full-stack虚机场景下支持。

## [注意事项](#ZH-CN_MMLREF_0000001088422284)

无。

## [参数说明](#ZH-CN_MMLREF_0000001088422284)

无。

## [使用实例](#ZH-CN_MMLREF_0000001088422284)

1. 查询网络亚健康参数。
  ```
  %%LST NETSUBHEALTH:;%%
  RETCODE = 0  操作成功

  操作结果如下
  ------------
              网络亚健康检测开关  =  开启
          网络亚健康故障自愈开关  =  开启
               上报告警阈值（%）  =  2
               清除告警阈值（%）  =  0.7
          一个错包数等价的丢包数  =  5
  网络亚健康故障检测周期（分钟）  =  5
  (结果个数 = 1)

  ---    END
  ```

## [输出结果说明](#ZH-CN_MMLREF_0000001088422284)

命令执行正常，会返回命令执行成功的提示信息和相关参数。输出结果说明如 [表1](#ZH-CN_MMLREF_0000001088422284__table819976173217) 所示。

命令执行异常，会返回提示没有对应的查询结果。错误码列表如 [表2](#ZH-CN_MMLREF_0000001088422284__table1996105120425) 所示。

*表1 输出项说明*

| 输出项名称 | 输出项解释 |
| --- | --- |
| 网络亚健康检测开关 | 用于返回网络亚健康检测开关的开关状态。 |
| 网络亚健康故障自愈开关 | 用于返回网络亚健康故障自愈开关的开关状态 |
| 上报告警阈值（%） | 用于返回上报告警阈值。 |
| 清除告警阈值（%） | 用于返回清除告警阈值。 |
| 一个错包数等价的丢包数 | 用于返回一个错包数等价的丢包数值。 |
| 网络亚健康故障检测周期（分钟） | 用于返回网络亚健康故障检测周期。 |

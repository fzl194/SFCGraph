#### [命令功能](#ZH-CN_CONCEPT_0251174298)

**适用NF：NCG**

该命令用于查询UDP校验和校验开关。

#### [注意事项](#ZH-CN_CONCEPT_0251174298)

无。

#### [本地用户权限](#ZH-CN_CONCEPT_0251174298)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_CONCEPT_0251174298)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0251174298)

无。

#### [使用实例](#ZH-CN_CONCEPT_0251174298)

查询UDP校验和校验开关：

```
LST CHKSUM:;
```

```
RETCODE = 0  操作成功。

结果如下:
--------
             网络用途  =  Ga
接收UDP校验和校验开关  =  关闭
发送UDP校验和校验开关  =  关闭
(结果个数 = 1)
---    END
```

# 删除pbuf重用检测开关设置（RMV PBUFREUSE5GC）

- [命令功能](#ZH-CN_MMLREF_0000001999779937__1.3.1)
- [注意事项](#ZH-CN_MMLREF_0000001999779937__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0000001999779937__1.3.4)
- [使用实例](#ZH-CN_MMLREF_0000001999779937__1.3.5)

## [命令功能](#ZH-CN_MMLREF_0000001999779937)

此命令用于删除MSS的pbuf重用检测开关设置，恢复未设置时的默认状态。

## [注意事项](#ZH-CN_MMLREF_0000001999779937)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0000001999779937)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001999779937)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PODTYPE | pod类型 | 可选必选说明：必选参数<br>参数含义：该参数用于指定pod类型。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~127。<br>默认值：无<br>配置原则：无 |
| POOLNAME | 内存池名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定内存池名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~31。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001999779937)

通过如下命令删除podtype为aa的pod内PAE内存池的pbuf重用检测开关设置：

```
RMV PBUFREUSE5GC:PODTYPE="aa",POOLNAME="PAE";
```

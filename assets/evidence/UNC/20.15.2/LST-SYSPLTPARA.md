#### [命令功能](#ZH-CN_MMLREF_0000001172345949)

**适用网元：SGSN、MME**

该命令用于查询产品平台相关的控制参数

#### [注意事项](#ZH-CN_MMLREF_0000001172345949)

无。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172345949)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172345949)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172345949)

无。

#### [使用实例](#ZH-CN_MMLREF_0000001172345949)

查询产品平台参数：

LST SYSPLTPARA:;

```
%%LST SYSPLTPARA:;%%
RETCODE = 0  操作成功

操作结果如下
--------------
        VNFC故障检测阈值（次数）  =  20
                  VNFC探测开关  =  打开
               主动关闭链路开关  =  打开
             非期望报文转发开关  =  打开
           海量SCTP消息防护开关  =  打开
   海量SCTP消息识别门限（包/秒）  =  1000
       异常eNodeB屏蔽时长（秒）  =  600
	  
(结果个数 = 1)

---    END
```

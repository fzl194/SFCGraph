# 删除Pod自愈黑名单（RMV PODBLACKLIST）

- [注意事项](#ZH-CN_MMLREF_0209587924__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0209587924__1.3.4)
- [使用实例](#ZH-CN_MMLREF_0209587924__1.3.5)

## [命令功能](#ZH-CN_MMLREF_0209587924)

该命令用于删除Pod自愈黑名单中指定的Pod类型。从Pod自愈黑名单中删除的Pod支持自愈功能。

## [注意事项](#ZH-CN_MMLREF_0209587924)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0209587924)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209587924)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PODTYPE | Pod类型 | 可选必选说明：必选参数<br>参数含义：该参数用于指示Pod自愈黑名单的Pod类型。可通过<br>[**DSP PODINFO**](../../编排管理/POD管理/显示已部署的Pod实例信息（DSP PODINFO）_09587375.md)<br>查询。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~32。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209587924)

将pod类型"sbim-pod"从Pod自愈黑名单中删除。

```
RMV PODBLACKLIST: PODTYPE="sbim-pod";
```

## [操作场景](#ZH-CN_OPI_0164082507)

本操作指导介绍在运行网络中激活支持 ODB 特性的操作过程。

支持ODB（Operator Determined Barring）特性，即运营商决定的闭锁，运营商能够通过设置ODB参数限制一些用户的业务和服务。

> **说明**
> 适用于 SGSN、 MME。

## [必备事项](#ZH-CN_OPI_0164082507)

前提条件

- 请仔细阅读[WSFD-106005 支持ODB（适用于SGSN、MME）特性概述](特性概述_68358182.md)。
- 需要对端网元HLR支持ODB特性。
- 已完成加载License，对应的License项为“82206154 LKV2ODB02 ODB（Operator Determined Barring）”。

数据

该操作无需准备数据。

## [操作步骤](#ZH-CN_OPI_0164082507)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. 开启 SGSN 支持ODB功能开关。
  [**SET SDCFG**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/签约数据管理/签约数据信息/设置签约数据配置(SET SDCFG)_72225433.md)
  > **说明**
  > - 使用[**SET SDCFG**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/签约数据管理/签约数据信息/设置签约数据配置(SET SDCFG)_72225433.md)命令设置禁止漫游用户通过HPLMN的接入点接入时，应确保[**ADD GWSELPLCY**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/GnGp-GGSN_S5_S8接口管理/GGSN_P-GW选择/增加GGSN_P-GW选择策略（ADD GWSELPLCY）_26145944.md)命令中“VAA”参数取值是“STANDARD_MODE”。
  > - 参数选择“YES”即开启SGSN支持该ODB功能开关，所有参数默认取值为“YES”。
  > - 在PS域中，禁用呼出指禁止用户发送短消息。
  > - 若参数“IOG”（禁止所有国际呼出）或“HOG”（禁止所有非归属国家呼出）选择了“YES”，则需配置对应的判断方案，即执行[步骤4](#ZH-CN_OPI_0164082507__cmd1083551435180642)。
4. **可选：**设置 SMS ODB选择方案，配置禁止所有国际呼出功能或禁止所有非归属国家呼出的判断方案。
  [**SET SMS**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/短消息/短消息配置/设置SMS配置信息(SET SMS)_72345329.md)

## [任务示例](#ZH-CN_OPI_0164082507)

任务描述

配置SGSN支持禁止所有呼出功能和支持禁止所有国际呼出功能，并将SMS ODB限制方案设置为根据目的号码限制。

脚本

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2ODB02", SWITCH=ENABLE;
```

//配置SGSN支持禁止所有呼出功能和支持禁止所有国际呼出功能。

```
SET SDCFG: AOG=YES, IOG=YES;
```

//SMS ODB限制方案设置为根据目的号码限制。

```
SET SMS: SMSODBSCHEME=DESTNUM;
```

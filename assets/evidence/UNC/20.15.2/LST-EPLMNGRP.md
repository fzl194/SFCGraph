## [命令功能](#ZH-CN_MMLREF_0209651354)

**适用NF：AMF**

该命令用于查询等价PLMN组。

## [注意事项](#ZH-CN_MMLREF_0209651354)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209651354)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209651354)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| GRPIDX | 等价PLMN组号 | 可选必选说明：可选参数<br>参数含义：该参数用于在UNC系统内唯一标识一个等价PLMN组。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~127。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209651354)

- 查询系统中“等价PLMN组号”为“0”的等价PLMN组信息，执行如下命令：
  ```
  %%LST EPLMNGRP: GRPIDX=0;%%
  RETCODE = 0  操作成功

  结果如下
  --------
  等价PLMN组号  =  0
      PLMN索引  =  0
  用户群组标识  =  0
  跟踪区域组ID  =  3
      描述信息  =  for ServingPLMN 12345
  (结果个数 = 1)

  ---    END
  ```
- 查询系统中配置的等价PLMN组信息，执行如下命令：
  ```
  %%LST EPLMNGRP:;%%
  RETCODE = 0  操作成功

  结果如下
  --------
  等价PLMN组号  =  0
      PLMN索引  =  0
  用户群组标识  =  0
  跟踪区域组ID  =  3
      描述信息  =  for ServingPLMN 12345
  (结果个数 = 1)

  ---    END
  ```

## [输出结果说明](#ZH-CN_MMLREF_0209651354)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 等价PLMN组号 | 该参数用于在UNC系统内唯一标识一个等价PLMN组。 |
| PLMN索引 | 该参数表示配置等价PLMN列表的Serving PLMN的索引信息，需要通过ADD NGSRVPLMN提前配置。 |
| 用户群组标识 | 该参数用于标识EPLMN适用的用户群信息，需要通过ADD NGUSRGRP提前配置。 |
| 跟踪区域组ID | 该参数用于标识等价PLMN适用的区域，需要通过ADD EPLMNAREAMEM提前配置。 |
| 描述信息 | 该参数用于对等价PLMN组的描述，在运维中起助记的作用。 |

## [任务示例](#ZH-CN_OPI_0242981541)

任务描述

部署SMSF 1+1互备容灾业务。

脚本

// 进入 “MML命令行-UNC” 窗口。

**//配置AMF数据。**

//AMF需要对接两个不同优先级的SMSF。

```
ADD PNFPROFILE:NFINSTANCEID="SMSF_Instance_0", NFTYPE=NfSMSF, NFSTATUS=Registered, IPADDRESSTYPE=IPTypeV4, IPV4ADDRESS1="10.2.180.15", PORT=8080, PRIORITY=0;
ADD PNFPROFILE:NFINSTANCEID="SMSF_Instance_1", NFTYPE=NfSMSF, NFSTATUS=Registered, IPADDRESSTYPE=IPTypeV4, IPV4ADDRESS1="10.2.180.16", PORT=8080, PRIORITY=1;
```

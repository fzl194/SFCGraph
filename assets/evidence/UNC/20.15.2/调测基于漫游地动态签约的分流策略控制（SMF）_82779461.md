## [操作场景](#ZH-CN_OPI_0000001182779461)

本操作指导介绍在运行网络中调测基于漫游地动态签约的分流策略控制特性的操作过程。

## [必备事项](#ZH-CN_OPI_0000001182779461)

前提条件

- 请仔细阅读[WSFD-223003 基于漫游地动态签约的分流策略控制特性概述](../WSFD-223003 基于漫游地动态签约的分流策略控制特性概述_42136946.md)特性原理概述。
- 完成[部署基于漫游地动态签约的分流策略控制（SMF）](../特性部署/部署基于漫游地动态签约的分流策略控制（SMF）_82779459.md)。

数据

| 类别 | 参数名称 | 取值样例 | 说明 |
| --- | --- | --- | --- |
| 用户信息 | 国际移动用户识别码（IMSI） | 123000123456789 | - |
| 专网DNN信息 | DNN名称（DNN） | “dnn-ar” | 专网业务的DNN |
| ULCL UPF IP地址 | IP地址 | 192.168.126.11 | 分流UPF的IP地址 |

## [操作步骤](#ZH-CN_OPI_0000001182779461)

1. 用户A已签约DNN为dnn-ar的专网业务，从归属地移动到漫游地，在专网区域内激活会话，创建PDU会话标识为5的PDU会话。
2. 进入 “MML命令行-UNC” 窗口。
3. 查询用户的PDU会话信息。
  **[DSP PDUSESSION](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/接入管理运维/查询PDP上下文信息/显示PDU会话（DSP PDUSESSION）_09653215.md)**
  预期结果：UE的会话创建流程中，选择到IP地址为192.168.126.11的ULCL UPF插入了用户会话。

  ```
  DSP PDUSESSION:QUERYTYPE=IMSI, IMSI="123000123456789", DSPINFOTYPE=DETAILED, WLNETWKTYPE=NW5G, PDUSESSIONID=5, QFI=1;
  RETCODE = 0  操作成功  

  结果如下 
  --------------  
              国际移动用户标识 = 123000123456789
                              ...
               用户位置信息TAI = 123010006
                              ...
             UP的PFCP IPv4地址 = 192.168.126.11
                              ...
                      ULCL标识 = True
                              ...
  (结果个数 = 1) 
  ---    END
  ```

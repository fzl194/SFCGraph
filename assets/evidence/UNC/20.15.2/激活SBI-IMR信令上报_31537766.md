## [操作场景](#ZH-CN_OPI_0000002131537766)

本操作指导介绍在运行网络中激活SBI IMR信令上报过程。

SBI IMR信令上报是指网元支持将SBI接口解密后的原始信令使用IMR（initial message record）机制发送给单据服务器（如CloudUDN），单据服务器将原始信令通过xDR流北向接口发送给服务探针/SEQ系统/第三方服务器等。

- 当已实例化的UNC不带vProbe服务时，需要上线vProbe，具体流程参见《UNC软件安装》手册 > 安装vProbe服务。
- 完成UNC上IP路由数据的配置，具体流程参见《UNC初始配置与调测》手册>配置VNF侧IP路由数据。
- 安装SBI信令软采扩展包，具体流程参见《支持SBI信令软采扩展包安装指导书》手册 > 安装SBI信令软采扩展包。
- (可选)安装部署IPSec，建议部署IPSec保证传输通道安全、数据安全。
- 配置vProbe与单据服务器对接数据。

## [操作步骤](#ZH-CN_OPI_0000002131537766)

1. 准备阶段：动态上线vProbe，以及配置UNC和单据服务器IP路由数据。
  **非SDN组网** ：
    a. 动态上线vProbe，具体流程参见《UNC软件安装》手册>基于VNF LCM安装UNC（非SDN）>[动态上线VPROBE服务（非SDN）](../../../../软件安装/基于NFVO安装UNC（SDN）/安装可选服务/安装VPROBE服务_05616886.md#ZH-CN_OPI_0000001405616886)。
    b. 完成UNC上IP路由数据的配置，具体流程参见[《UNC初始配置与调测》手册](../../../../初始配置/UNC初始配置与调测/了解组网架构与原理_43045513.md)>配置VNF侧IP路由数据（非SDN）。
  **SDN组网** ：
    a. 动态上线vProbe，具体流程参见《UNC软件安装》手册>基于NFVO安装UNC(SDN组网)>[动态上线VPROBE服务（SDN）](../../../../软件安装/基于NFVO安装UNC（SDN）/安装可选服务/安装VPROBE服务_05616886.md#ZH-CN_OPI_0000001405616886)。
    b. 完成UNC上IP路由数据的配置，具体流程参见[《UNC初始配置与调测》手册](../../../../初始配置/UNC初始配置与调测/了解组网架构与原理_43045513.md)>配置VNF侧IP路由数据(SDN)。
2. 安装SBI信令软采扩展包。
  流程参见《支持SBI信令软采扩展包安装指导书》手册>安装SBI信令软采扩展包。
3. (可选)安装部署IPSec。
    a. 安装IPSec服务，具体流程参见《UNC软件安装》手册 > 安装可选服务 > 安装IPSec服务。
    b. 配置IPSec，具体参见《UNC安全配置》手册 > 可选配置 > 网络安全配置 > 配置IPSec。
4. 配置vProbe和单据服务器对接数据。
    a. 进入“MML命令行-UNC”窗口
    b. 配置vProbe报表本地IP资源池。
      [**ADD VPROBEIP**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/vProbe管理/vProbe本地IP/增加vProbe报表本地IP资源池（ADD VPROBEIP）_89042700.md)
    c. 配置vProbe报表服务器接入点IP地址等信息。
      [**ADD VPROBESVRIP**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/vProbe管理/vProbe服务器IP/增加vProbe报表服务器的接入点IP地址（ADD VPROBESVRIP）_39402649.md)
    d. 查询vProbe与报表服务器之间的链路状态。
      **[DSP VPROBELNKSTAT](../../../../../OM参考/命令/UNC MML命令/平台服务管理/vProbe管理/vProbe服务器链路状态/显示vProbe链路状态（DSP VPROBELNKSTAT）_39242821.md)**
    e. vProbe部署完成，且链路正常后，打开本特性的License配置开关。
      [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)

## [任务示例](#ZH-CN_OPI_0000002131537766)

任务描述

激活SBI信令上报功能，对所有用户的HTTPS消息解密，并封装在IMR消息里发送给单据服务器。

脚本

//配置vProbe报表本地IP资源池。

```
ADD VPROBEIP: INDEX=1, PROTOCOLTYPE=TCP, IPTYPE=IPv4, IPV4="10.185.23.253", PORTSTART=6000, PORTEND=6031;
```

//配置报表服务器接入点IP地址等信息。

```
ADD VPROBESVRIP: PROTOCOLTYPE=TCP, ACCESSNAME="ACCESS1", VPROBESVRNAME="VPROBESVR1", IPTYPE=IPv4, IPV4="10.180.211.254", SVRPORT=10500;
```

//查询vProbe本端与报表服务器的链路状态信息。


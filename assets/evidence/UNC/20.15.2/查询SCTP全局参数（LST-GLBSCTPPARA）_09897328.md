#### [命令功能](#ZH-CN_CONCEPT_0209897328)

**适用NF：PGW-C、SMF**

此命令用于查询SCTP全局参数。

#### [注意事项](#ZH-CN_CONCEPT_0209897328)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0209897328)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0209897328)

无。

#### [使用实例](#ZH-CN_CONCEPT_0209897328)

查询SCTP全局参数：

```
LST GLBSCTPPARA:;
```

```

RETCODE = 0  操作成功

SCTP全局参数
------------
           SCTP路径选择模式  =  仅平行路径为可用路径
RTO重发超时的最小值（毫秒）  =  500
RTO重发超时的最大值（毫秒）  =  1500
RTO重发超时的初始值（毫秒）  =  500
(结果个数 = 1)

---    END
```

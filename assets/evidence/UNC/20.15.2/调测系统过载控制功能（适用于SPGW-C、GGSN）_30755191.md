# 调测系统过载控制功能（适用于SPGW-C、GGSN）

- [操作场景](#ZH-CN_OPI_0230755191__1.3.1)
- [必备事项](#ZH-CN_OPI_0230755191__1.3.2)
- [操作步骤](#ZH-CN_OPI_0230755191__1.3.3)

## [操作场景](#ZH-CN_OPI_0230755191)

支持当网络中出现CPU占用率达到了系统流控的阈值时，系统开始对相应进程进行流控。

> **说明**
> 适用于 GGSN、 SGW-C、PGW-C。

## [必备事项](#ZH-CN_OPI_0230755191)

前提条件

- 请仔细阅读[WSFD-010801系统过载控制功能特性概述（适用于SPGW-C、GGSN）](WSFD-010801系统过载控制功能特性概述（适用于SPGW-C、GGSN）_30755109.md)。
- 完成[激活系统过载控制功能（适用于SPGW-C、GGSN）](激活系统过载控制功能（适用于SPGW-C、GGSN）_30755190.md)。

数据

该操作无需准备数据。

工具

该操作无需准备其他工具。

## [操作步骤](#ZH-CN_OPI_0230755191)

1. 进入 “MML命令行-UNC” 窗口。
2. 查询基于CPU过载流控开关状态。
  **[LST FCSWITCH](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/服务通信管理/流控管理/查询流控开关（LST FCSWITCH）_09587891.md)**
  预期结果： “开关” 取值和配置一致。
3. 查询APN接入速率。
  [**LST APNACCESSWAL**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/APN管理/APN接入速率/查询APN接入速率（LST APNACCESSWAL）_09651641.md)
  预期结果： “用户的接入速率” 取值和配置一致。
4. 查询GTPC接口固定速率流控参数。
  **[LST GTPCFIXEDFC](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/GTP-C接口配置管理/GTP-C固定速率流控管理/查询指定消息类型固定速率流控信息（LST GTPCFIXEDFC）_88377440.md)**
  预期结果：
    - 指定消息类型的“固定速率流控开关”显示为“开启”。
    - “流控速率门限（个/秒）”查询输出的数值与配置一致。
5. 设置信令抑制功能开关以及老化定时器时长。
  **[LST OVERLOADCTRL](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/接入管理运维/流控管理/信令抑制/查询过载控制的配置信息（LST OVERLOADCTRL）_35803152.md)**
  预期结果：
    - “信令抑制功能开关”显示为“开启”。
    - “信令抑制记录的老化定时器时长(秒)”查询输出的数值与配置一致。
    - “**OCS CCR-I去活是否开启信令抑制**”和“**ACCT去活是否开启信令抑制**”和配置一致。
6. 查看过载控制记录信息。
  **[DSP OVERLOAD](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/接入管理运维/过载控制信息查询/显示过载控制记录信息（DSP OVERLOAD）_35803150.md)**
  预期结果：信令抑制中的用户的剩余老化时长和预期一致。

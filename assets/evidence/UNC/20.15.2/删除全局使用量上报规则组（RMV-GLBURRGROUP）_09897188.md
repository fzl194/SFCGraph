# 删除全局使用量上报规则组（RMV GLBURRGROUP）

- [命令功能](#ZH-CN_CONCEPT_0209897188__1.3.1.1)
- [注意事项](#ZH-CN_CONCEPT_0209897188__1.3.2.1)
- [操作用户权限](#ZH-CN_CONCEPT_0209897188__1.3.3.1)
- [参数说明](#ZH-CN_CONCEPT_0209897188__1.3.4.1)
- [使用实例](#ZH-CN_CONCEPT_0209897188__1.3.5.1)

#### [命令功能](#ZH-CN_CONCEPT_0209897188)

**适用NF：PGW-C、SMF**

该命令用于PDP用户将全局使用量上报规则组恢复成默认记录。

#### [注意事项](#ZH-CN_CONCEPT_0209897188)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0209897188)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0209897188)

无。

#### [使用实例](#ZH-CN_CONCEPT_0209897188)

假如用户要删除全局使用量上报规则组：

```
RMV GLBURRGROUP:;
```

#### [命令功能](#ZH-CN_MMLREF_0000001126145566)

**适用网元：SGSN、MME**

此命令用于查询区域码记录。区域码用于定义允许或者禁止用户漫游的区域，借助一个完全的区域码列表，运营商可以确定用户是否能在此区域漫游。

#### [注意事项](#ZH-CN_MMLREF_0000001126145566)

此命令执行后立即生效。

#### [本地用户权限](#ZH-CN_MMLREF_0000001126145566)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126145566)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126145566)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| ZC | 区域码 | 可选必选说明：可选参数<br>参数含义：待查询的区域码信息。<br>取值范围：0x0000～0xFFFF<br>默认值：无 |
| IDTYPE | 标识类型 | 可选必选说明：可选参数<br>参数含义：待查询的区域标识类型。<br>取值范围：<br>- “LA(位置区)”：表示该区域标识类型为位置区。<br>- “RA(路由区)”：表示该区域标识类型为路由区<br>- “SA(服务区)”：表示该区域标识类型为服务区<br>- “TA(跟踪区)”：表示该区域标识类型为跟踪区。<br>默认值：无 |
| MCC | 移动国家码 | 可选必选说明：可选参数<br>参数含义：待查询的移动国家码的信息。<br>取值范围：3位的十进制数字<br>默认值：无 |
| MNC | 移动网号 | 可选必选说明：可选参数<br>参数含义：待查询的移动网号的信息。<br>取值范围：2～3位的十进制数字<br>默认值：无 |
| LAC | 位置区域码 | 可选必选说明：可选参数<br>参数含义：待查询的位置区域码的信息。<br>前提条件：该参数在<br>“IDTYPE(标识类型)”<br>设置为<br>“LA(位置区)”<br>、<br>“RA(路由区)”<br>、<br>“SA(服务区)”<br>时生效。<br>取值范围：0x0000～0xFFFF<br>默认值：无 |
| RAC | 路由区域码 | 可必选说明：可选参数<br>参数含义：待查询的路由区域码的信息。<br>前提条件：该参数在<br>“IDTYPE(标识类型)”<br>设置为<br>“RA(路由区)”<br>时生效。<br>取值范围：0x00～0xFF<br>默认值：无 |
| SAC | 服务区域码 | 可选必选说明：可选参数<br>参数含义：待查询的服务区域码的信息。<br>前提条件：该参数在<br>“IDTYPE(标识类型)”<br>设置为<br>“SA(服务区)”<br>时生效。<br>取值范围：0x0000～0xFFFF<br>默认值：无 |
| TAC | 跟踪区域码 | 可选必选说明：可选参数<br>参数含义：待查询的跟踪区域码的信息。<br>前提条件：该参数在<br>“IDTYPE(标识类型)”<br>设置为<br>“TA(跟踪区)”<br>时生效。<br>取值范围：0x0000～0xFFFF<br>默认值：无 |

#### [使用实例](#ZH-CN_MMLREF_0000001126145566)

1. 查询所有区域码记录：
  LST ZC:;
  ```
  %%LST ZC:;%%
  RETCODE = 0  操作成功。

  输出结果如下
  --------------
   区域码  标识类型  移动国家码  移动网号  位置区域码  位置区域码范围  路由区域码  路由区域码范围  服务区域码  服务区域码范围  跟踪区域码  跟踪区域码范围  区域名称

   0x1234  位置区    123         03        0x01        0x01            NULL        NULL            NULL        NULL            NULL        NULL            NULL    
   0x12    路由区    123         03        0x10        0x10            0x01        0x01            NULL        NULL            NULL        NULL            NULL    
  (结果个数 = 2)

  ---    END
  ```
2. 查询区域码为0x12的记录：
  LST ZC: ZC="0x12";
  ```
  %%LST ZC: ZC="0x12";%%
  RETCODE = 0  操作成功。

  输出结果如下
  --------------
          区域码  =  0x0012
        标识类型  =  路由区
      移动国家码  =  123
        移动网号  =  03
      位置区域码  =  0x10
  位置区域码范围  =  0x10
      路由区域码  =  0x01
  路由区域码范围  =  0x01
      服务区域码  =  NULL
  服务区域码范围  =  NULL
      跟踪区域码  =  NULL
  跟踪区域码范围  =  NULL
        区域名称  =  NULL
  (结果个数 = 1)

  ---    END
  ```

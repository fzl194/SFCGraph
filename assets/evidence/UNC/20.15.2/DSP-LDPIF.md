#### [命令功能](#ZH-CN_CONCEPT_0000001600601417)

该命令用于显示LDP接口状态。

#### [注意事项](#ZH-CN_CONCEPT_0000001600601417)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600601417)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600601417)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| VRFNAME | VPN实例名称 | 可选必选说明：可选参数<br>参数含义：该参数用于表示VPN实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。<br>默认值：无 |
| IFNAME | 接口名 | 可选必选说明：可选参数<br>参数含义：该参数用于表示接口名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。<br>默认值：无<br>配置原则：请使用LST INTERFACE命令查看可用接口。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600601417)

显示LDP接口状态：

```
DSP LDPIF:;
```

```

RETCODE = 0  操作成功。

结果如下
--------
       VPN实例名称  =  _public_
            接口名  =  Ethernet64/0/3
          接口状态  =  激活的
      标签发布模式  =  DU
协商的hello hold值  =  15
   接口有效的MTU值  =  1500
      自动创建标识  =  默认值
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600601417)

| 输出项名称 | 输出项解释 |
| --- | --- |
| VPN实例名称 | 用于表示VPN实例名称。 |
| 接口名 | 用于表示接口名。 |
| 接口状态 | 用于表示接口状态，包括“激活的”和“非激活的”两种。 |
| 标签发布模式 | 用于表示标签发布模式。 |
| 协商的hello hold值 | 用于表示协商的hello hold值。 |
| 接口有效的MTU值 | 用于表示接口有效的MTU值，决定MPLS转发报文的大小。 |
| 自动创建标识 | 用于表示自动创建标识。包含Default、ISIS、OSPF、ISIS-OSPF。目前暂不支持ISIS和ISIS-OSPF。 |

## [操作场景](#ZH-CN_OPI_0000002185120025)

本操作指导介绍在运行网络中激活小区位置信息上报功能的操作过程。

小区位置信息上报功能是指MME基于S-GW/P-GW下发的策略向RAN侧发起位置订阅，当用户的ECGI发生变化时，MME将最新的用户位置信息上报给S-GW/P-GW，并经S-GW/P-GW上报给PCRF。PCRF可综合用户位置、使用流量、网络状态等进行策略决策并发起策略更新。

> **说明**
> 适用于MME

## [对系统的影响](#ZH-CN_OPI_0000002185120025)

性能影响程度：中

该特性会增加S1-MME接口、S11接口上信令数目，降低系统性能，并且频繁的位置更新上报也会大量增加Gx接口的信令负载，因此不能对所有用户同时开启该特性，应该区分区域和用户进行开启。

对系统性能的影响程度取决于开启该特性的用户数和相应的话务模型。请联系华为技术支持获取服务。

## [操作步骤](#ZH-CN_OPI_0000002185120025)

1. 打开本特性的License配置开关。
  在 “MML命令行-UNC” ：
  [**SET LICENSESWITCH**](../../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md) : LICITEM="LKV2LCR01"，SWITCH=ENABLE；
2. 配置特定的GGSN/P-GW允许使用该功能。
  在“MML命令行-UNC”窗口上执行：
  [**ADD LOCRPTCFG**](../../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/GnGp-GGSN_S5_S8接口管理/P-GW属性/增加位置上报配置信息(ADD LOCRPTCFG)_26305746.md) :RANGE=对端设备范围，IPT=IP地址类型，GWIPV4=GGSN/P-GW的信令面地址，MASKV4=掩码，LOCRPT=支持位置上报功能；
3. **可选：**设置实时位置上报最小间隔。
  在“MML命令行-UNC”窗口上执行：
  [**SET MMFUNC**](../../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/MM扩展功能管理/设置移动性管理扩展功能(SET MMFUNC)_26145512.md) :LOCRPTTIMER=实时位置上报最小间隔。

## [验证方法](#ZH-CN_OPI_0000002185120025)

如 [图1](#ZH-CN_OPI_0000002185120025__fig2) 所示，MME所在PLMN无线覆盖区域为Cell1、Cell2和Cell3。

**图1** “基于实时位置的策略控制”的调测场景举例

<br>![](激活小区位置信息上报功能（S11接口）_85120025.assets/zh-cn_image_0000002149887814_2.png)<br>

MME上报实时ECGI信息给P-GW。

- 4G场景
    1. 在UNC操作维护台上创建用户跟踪、S1-MME接口跟踪、S11接口跟踪。
    2. 用户A在Cell1发起激活，PCRF要求上报的位置信息类型为ECGI。
      预期结果：
          - S11接口跟踪里P-GW回复的Create Session Response消息中会携带信元Change Reporting Action。
          - S1-MME接口跟踪里MME会向eNodeB下发Location Reporting Control消息，其中携带信元Event和Report Area。
    3. 用户A进行数传业务，并从Cell1移动到Cell2。
      预期结果：
          - S1-MME接口跟踪里eNodeB会向MME上报Location Report消息，其中携带新的ECGI。
          - S11接口跟踪里MME向P-GW发送Change Notification Request消息，携带信元消息中携带用户信息（IMSI）、用于指示PDN连接的LBI和更新后的User Location Information信元。

## [任务示例](#ZH-CN_OPI_0000002185120025)

任务描述

开启小区位置信息上报功能，网关IP地址为“10.0.0.1”。

脚本

```
//打开小区位置信息上报功能。
SET LICENSESWITCH: LICITEM="LKV2LCR01", SWITCH=ENABLE;

//配置特定的GGSN/P-GW允许使用该功能。
ADD LOCRPTCFG: RANGE=SPECIFIC_GGSN/P-GW, IPT=IPV4, GWIPV4="10.0.0.1", MASKV4="255.255.255.0", LOCRPT=YES; 
```

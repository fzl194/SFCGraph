#### [适用NF](#ZH-CN_CONCEPT_0276948719)

SGW-C、PGW-C

#### [定义](#ZH-CN_CONCEPT_0276948719)

华为 UNC 支持标准定义的WLAN AN/GERAN/UTRAN/E-UTRAN无线网络接入，实现WLAN与GSM/UMTS/LTE网络用户同时接入 UNC 。

#### [客户价值](#ZH-CN_CONCEPT_0276948719)

| 受益方 | 受益描述 |
| --- | --- |
| 运营商 | - 节省投资成本，提高设备利用率。<br>- 减少维护成本，便于集中维护管理。 |
| 用户 | 降低网络切换时延，改善用户体验。 |

#### [应用场景](#ZH-CN_CONCEPT_0276948719)

当需要作为PGW-C的 UNC 同时为WLAN和GSM/UMTS/LTE接入网络提供服务，实现通过两个不同的PDN连接进行业务访问。

#### [可获得性](#ZH-CN_CONCEPT_0276948719)

**涉及NF**

| 涉及NF | 版本支持 | 功能说明 |
| --- | --- | --- |
| MS/UE | 无特殊要求 | - |
| BSS/<br>RAN<br>/eNodeB | 无特殊要求 | - |
| HSS/HLR | 无特殊要求 | - |
| MME/SGSN | UNC 20.5.0及后续版本 | - |
| ePDG | 无特殊要求 | - |
| GGSN/PGW-C/SGW-C | UNC 20.5.0及后续版本 | - |
| 3GPP AAA Server | 无特殊要求 | - |

**License支持**

本特性只有获得了License许可后才能获得该特性的服务，对应的License控制项为 “82209419 LKV3WPWLAN11 支持WLAN与GSM/UMTS/LTE双流并发” 。

#### [与其他特性的交互](#ZH-CN_CONCEPT_0276948719)

本特性不涉及与其他特性的交互。

#### [对系统的影响](#ZH-CN_CONCEPT_0276948719)

性能影响程度：低

支持WLAN与GSM/UMTS/LTE双流并发时， UNC 只进行简单信元解析和路由转发，对系统无影响。

#### [应用限制](#ZH-CN_CONCEPT_0276948719)

- 不支持WLAN与GSM/UMTS网络间的切换。
- 同一用户使用相同的APN同时建立WLAN和LTE两个PDN连接时，不支持WLAN网络和LTE网络间的相互切换。

#### [原理概述](#ZH-CN_CONCEPT_0276948719)

**系统实现**

随着3GPP技术的发展， UNC 支持作为PGW-C节点，同时为WLAN和GSM/UMTS/LTE接入网络提供服务。满足终端用户通过相同或者不同的APN，通过两个不同的PDN连接进行业务访问。如 [图1](#ZH-CN_CONCEPT_0276948719__fig1) 所示，WLAN与GSM/UMTS/LTE同时接入的组网部署。

*表1 支持WLAN与GSM/UMTS/LTE双流并发组网网络中主要功能单元*

| 功能单元 | 描述 |
| --- | --- |
| MS/UE | 3GPP终端，支持GSM/UMTS/LTE功能。<br>WiFi终端，支持WiFi网络接入功能。 |
| BSS/RAN/eNodeB | GSM/UMTS/LTE用户接入时，为MS/UE提供无线资源及承载。若MS/UE未携带SGSN/MME信息，为MS/UE进行SGSN/MME选择。 |
| SGSN/MME | 为GSM/UMTS/LTE接入用户提供附着、去附着等移动性管理功能。为MS/UE选择接入的SGW-C和GGSN/PGW-C。<br>在LTE网络中和SGW-C配合，为UE提供缺省承载建立、删除、更新等会话管理功能。 |
| SGW-C | 过S1-U接口和eNodeB互通，通过S11接口和MME互通。和PGW-C配合，为UE提供缺省承载建立、删除、更新等会话管理功能。 |
| GGSN/PGW-C | 为MS/UE接入外部数据网提供基本承载功能。接收用户发送的数据，选路到相应的外部网络；或接收外部网络的数据，根据其目的地址选择网络中的传输通道。 |
| 3GPP AAA Server | MS/UE接入网络时，为其提供静态QoS信息。 |
| WLAN AN | 与ePDG配合，为UE提供网络侧的附着、去附着等移动性管理功能。 |
| ePDG（Evolved Packet Data Gateway） | 能够连接到3GPP AAA Server和PGW-C，为MS/UE提供信令和数据业务通道。 |

#### [命令功能](#ZH-CN_MMLREF_0000001172225293)

**适用网元：SGSN、MME**

该命令用于查询CHR传输策略。

#### [注意事项](#ZH-CN_MMLREF_0000001172225293)

- 该命令执行后立即生效。
- 如果不输入任何参数，执行该命令会显示所有记录。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172225293)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172225293)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172225293)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| TYPE | CHR传输策略类型 | 可选必选说明：可选参数<br>参数含义：该参数用于指定CHR传输策略类型。<br>数据来源：本端规划<br>取值范围：<br>- “APNNI（签约APNNI）”：表示签约数据指定的APNNI的用户的CHR单据才会传输给CloudUDN。<br>默认值：无<br>配置原则：当前仅支持配置“签约APNNI” |

#### [使用实例](#ZH-CN_MMLREF_0000001172225293)

查询CHR传输策略：

LST CHRSNDPLCY:;

```
%%LST CHRSNDPLCY:;%%
RETCODE = 0  操作成功。

操作结果如下
--------------
CHR传输策略类型  =  签约APNNI
    APN网络标识  =  HUAWEI.COM
(结果个数 = 1)

---    END
```

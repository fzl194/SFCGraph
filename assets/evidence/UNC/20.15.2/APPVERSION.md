## [命令功能](#ZH-CN_MMLREF_0209587389)

此命令用于显示系统版本号。

## [注意事项](#ZH-CN_MMLREF_0209587389)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209587389)

G_1，管理员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209587389)

无

## [使用实例](#ZH-CN_MMLREF_0209587389)

DSP APPVERSION:;

```
%%DSP APPVERSION:;%%
RETCODE = 0  操作成功
结果如下
------------------------
版本号  =  xxx 20.0.0.B080
(结果个数 = 1)
---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209587389)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 版本号 | 该参数用于表示网元版本号。 |

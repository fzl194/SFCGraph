#### [命令功能](#ZH-CN_CONCEPT_0209896961)

**适用NF：PGW-C、SMF**

此命令用来查询指定的OCS组。

#### [注意事项](#ZH-CN_CONCEPT_0209896961)

- 如果查询一个OCS组，需要输入OCS组的名称。
- 如果查询所有的OCS组，则不需要输入参数。

#### [操作用户权限](#ZH-CN_CONCEPT_0209896961)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0209896961)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| OCSGRPNAME | Ocs组名称 | 可选必选说明：可选参数<br>参数含义：ocs组的名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，不区分大小写。<br>默认值：无<br>配置原则：如果不配置则是要查询所有的OCS组。 |

#### [使用实例](#ZH-CN_CONCEPT_0209896961)

查询名称为“ocsgroup1”的OCS组：

```
LST OCSGROUP:;
```

```

RETCODE = 0  操作成功。

Ocs组信息
---------
Ocs组名称  =  ocsgroup1
(结果个数 = 1)
---    END
```

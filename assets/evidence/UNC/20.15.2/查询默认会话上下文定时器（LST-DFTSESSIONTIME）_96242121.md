## [命令功能](#ZH-CN_MMLREF_0296242121)

**适用NF：SGW-C、PGW-C、SMF、GGSN**

该命令用于查询SessionTimeout的全局配置。

## [注意事项](#ZH-CN_MMLREF_0296242121)

无

#### [操作用户权限](#ZH-CN_MMLREF_0296242121)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0296242121)

无

## [使用实例](#ZH-CN_MMLREF_0296242121)

查询SessionTimeout的全局配置：

```
%%LST DFTSESSIONTIME:;%%
RETCODE = 0  操作成功

结果如下
--------
会话时长开关  =  使能
    会话时长  =  120
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0296242121)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 会话时长开关 | 该参数用于设置会话时长开关。 |
| 会话时长 | 该参数用于指定最大会话时长。 |

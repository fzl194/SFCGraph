#### [命令功能](#ZH-CN_CONCEPT_0000001600440617)

该命令用于显示OSPF BFD信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600440617)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600440617)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600440617)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | 进程号 | 可选必选说明：可选参数<br>参数含义：OSPF进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |
| NBRROUTERID | 邻居路由器标识 | 可选必选说明：可选参数<br>参数含义：邻居路由器标识。<br>数据来源：对端协商<br>取值范围：IPv4地址类型。点分十进制格式。<br>默认值：无 |
| AREAID | 区域号 | 可选必选说明：可选参数<br>参数含义：区域号。<br>数据来源：本端规划<br>取值范围：IPv4地址类型。点分十进制格式。<br>默认值：无 |
| IFNAME | 接口名 | 可选必选说明：可选参数<br>参数含义：接口名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。<br>默认值：无<br>配置原则：请使用LST INTERFACE命令查看可用接口。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600440617)

在设备上建立OSPF进程1，建立邻居并配置BFD，然后显示：

```
DSP OSPFBFD: PROCID=1;
```


#### [输出结果说明](#ZH-CN_CONCEPT_0000001600440617)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 进程号 | 该参数指定用户创建的OSPF进程号。 |
| 邻居路由器标识 | 该参数指定邻居路由器标识。 |
| 区域号 | 该参数指定用户创建的OSPF区域号。 |
| 本地IP地址 | 该参数指定BFD会话本地IP地址。 |
| 远端IP地址 | 该参数指定BFD会话远端IP地址。 |
| 接口名 | 该参数指定BFD会话的接口名。 |
| BFD状态 | 该参数指定BFD会话状态。Unknown：BFD通知OSPF receive admindown，OSPF BFD会话状态会变为Unknown；Up：OSPF BFD会话UP；Down：OSPF BFD会话Down，会通知BFD删除会话。 |
| BFD最小接收报文间隔（ms） | 该参数指定期望从对端接收BFD报文的最小接收间隔，单位是毫秒。 |
| BFD最小发送报文间隔（ms） | 该参数指定向对端发送BFD报文的最小发送间隔，单位是毫秒。 |
| 检测倍数 | 该参数指定会话检测倍数，如果BFD会话在配置的检测周期内没有收到对端发来的BFD报文，则认为链路发生了故障，BFD会话的状态将会置为Down。 |

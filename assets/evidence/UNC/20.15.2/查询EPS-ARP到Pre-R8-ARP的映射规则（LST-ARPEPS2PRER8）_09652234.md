- [参数说明](#ZH-CN_MMLREF_0209652234__1.3.4)

## [命令功能](#ZH-CN_MMLREF_0209652234)

**适用NF：SGW-C、PGW-C、GGSN**

该命令用于查询Gn接口和Gx接口之间ARP的映射值。

## [注意事项](#ZH-CN_MMLREF_0209652234)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209652234)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652234)

无

## [使用实例](#ZH-CN_MMLREF_0209652234)

查询Gn接口和Gx接口之间ARP的映射值：

```
%%LST ARPEPS2PRER8:;%%
RETCODE = 0  操作成功

ARP的映射值配置信息
-------------------
ARP高优先级上限  =  5
ARP中优先级上限  =  10
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209652234)

| 输出项名称 | 输出项解释 |
| --- | --- |
| ARP高优先级上限 | 该参数用于指定ARP优先级正常时（ARP值为2）的映射值。 |
| ARP中优先级上限 | 该参数用于指定ARP优先级低时（ARP值为3）的映射值。 |

- [注意事项](#ZH-CN_TOPIC_0188031787__1.3.2.1)

#### [命令功能](#ZH-CN_TOPIC_0188031787)

查询所有支持二次授权的用户名。

#### [注意事项](#ZH-CN_TOPIC_0188031787)

无。

#### [参数说明](#ZH-CN_TOPIC_0188031787)

无。

#### [使用实例](#ZH-CN_TOPIC_0188031787)

查看二次授权用户：

```
%%LST USRSECAUTH:;%%
RETCODE = 0  操作成功

操作结果如下
------------
用户名    
  
admin     
test01   
(结果个数 = 2)

---    END
```

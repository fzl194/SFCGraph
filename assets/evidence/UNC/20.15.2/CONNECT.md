#### [命令功能](#ZH-CN_TOPIC_0181901697)

本命令用于查询Soap服务建立的长连接信息，本命令不需要输入任何参数。

#### [注意事项](#ZH-CN_TOPIC_0181901697)

无。

#### [参数说明](#ZH-CN_TOPIC_0181901697)

无。

#### [使用实例](#ZH-CN_TOPIC_0181901697)

1. 查询网元长连接信息（网管网元建立长连接）。
  ```
  %%LST CONNECT:;%% 
  RETCODE = 0  操作成功  
  操作结果如下 
  ------------ 
  节点IP       链接ID       序号  状态    
  127.0.0.1  *****956*****  1     2      
  127.0.0.1  *****956*****  2     2      
  127.0.0.1  *****956*****  3     2      
  127.0.0.1  *****956*****  4     2      
  127.0.0.1  *****956*****  5     2      
  127.0.0.1  *****956*****  6     2      
  (结果个数 = 6)
  
  ---    END
  ```
2. 查询网元长连接信息（网管网元没有建立长连接）。
  ```
  %%LST CONNECT:;%% 
  RETCODE = 0  操作成功  

  没有查到相应结果 
  ---    END
  ```

#### [输出结果说明](#ZH-CN_TOPIC_0181901697)

命令执行正常，会返回命令执行成功的提示信息，输出结果说明如 [表1](#ZH-CN_TOPIC_0181901697__table47787675) 所示。

命令执行异常，请联系华为技术支持处理。输出结果说明如 [表2](#ZH-CN_TOPIC_0181901697__table698716490471) 所示。

*表1 输出项解释*

| **输出项名称** | **输出项解释** |
| --- | --- |
| 节点IP | Soap服务容器IP。 |
| 链接ID | 匿名化后的链接ID。 |
| 序号 | 链接序号，每套网管环境和Soap服务建立6条链接。 |
| 状态 | 对应链接的状态码，其中0表示要建立长链接状态；1表示未建立长链接状态；2表示已建立长链接状态；3表示要断开长链接；4表示要建立新长连接状态；5表示无效状态。 |

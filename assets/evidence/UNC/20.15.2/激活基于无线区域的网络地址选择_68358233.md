## [操作场景](#ZH-CN_OPI_0168358233)

本操作指导介绍在运行网络中激活基于无线区域的网络地址选择特性的操作过程。

UNC 被多个运营商共享情况下， UNC 选择用户接入时的无线网络区域所属的运营商及该运营商对应的网络地址为用户提供服务。

> **说明**
> 适用于MME。

## [操作步骤](#ZH-CN_OPI_0168358233)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. 配置应用本特性的无线区域。
  [**SET MMFUNC**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/MM扩展功能管理/设置移动性管理扩展功能(SET MMFUNC)_26145512.md)
  > **说明**
  > 如果因为漫游协议、计费结算等原因，不希望用户通过 UNC 内跨运营商的 RAU/ TAU业务流程完成业务迁移，而要求用户在跨运营商网络时通过重新Attach接入时，需要将 “NOIDCHG（允许跨共享运营商RAU/TAU）” 设置为 “NO” ，关闭 UNC 内两个运营商之间的 RAU/ TAU操作。
4. 确认是否针对服务运营商配置相应的本端信息。
  [**LST HNOINFO**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/归属网络运营商管理/归属网络信息管理/查询归属网络信息(LST HNOINFO)_72225733.md)
    - 是，处理结束。
    - 否，执行[步骤 5](#ZH-CN_OPI_0168358233__ADD_HNOINFO)。
5. 配置服务运营商的本端信息。
  [**ADD HNOINFO**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/归属网络运营商管理/归属网络信息管理/增加归属网络信息(ADD HNOINFO)_26305862.md)

## [任务示例](#ZH-CN_OPI_0168358233)

任务描述

组网如 [图1](#ZH-CN_OPI_0168358233__fig01) 所示。MNO-A和MNO-B共享 UNC ，它们分别有自己独立的无线网络。

**图1** 组网示意图

<br>

![](激活基于无线区域的网络地址选择_68358233.assets/zh-cn_image_0168621333_2.png "点击放大")

脚本

1. 激活多HPLMN特性。
  //打开License配置开关。
  ```
  SET LICENSESWITCH: LICITEM="LKV2HPLMN02", SWITCH=ENABLE;
  ```
  //增加MNO信息。
  ```
  ADD MNO: MNOID=254, FULLNAME="MNO-A";
  ADD MNO: MNOID=128, FULLNAME="MNO-B";
  ```
  //添加MNO-A和MNO-B的HPLMN记录。
  ```
  ADD HPLMN: MCC="123", MNC="27", CC="86", MNOID=254, SM=YES, SMS=YES, SMSCR=NO;
  ADD HPLMN: MCC="123", MNC="17", CC="86", MNOID=128, SM=YES, SMS=YES, SMSCR=NO;
  ```
  //配置MNO-A和MNO-B的归属网络信息。
  ```
  ADD HNOINFO: NOID=254;
  ADD HNOINFO: NOID=128;
  ```
2. 配置区域漫游限制。
  //开启区域漫游限制功能开关。
  ```
  SET LICENSESWITCH: LICITEM="LKV2RRR02", SWITCH=ENABLE;
  ```
  //位置区域划分。
  ```
  ADD AREAGP: AREAID=2, AREAN="LA_46027";
  ADD AREAGPMEM: AREAID=2, IDTYPE=LA, MCC="123", MNC="27", LAC="0x0000", LACRANGE="0xFFFF";
  ```
  //跟踪区域划分。
  ```
  ADD AREAGP: AREAID=3, AREAN="TA_46027";
  ADD AREAGPMEM: AREAID=3, IDTYPE=TA, MCC="123", MNC="27", TAC="0x0000", TACRANGE="0xFFFF";
  ```
  //针对不同的用户群设置对应的漫游控制策略。
  ```
  ADD GBACCAREALST: AREA=AG, AREAID=2, SUBRANGE=SPECIAL_IMSIPRE, IMSIPRE="12317", CTRLTYPE=REJECT, CAUSE=NO_SUITABLE_CELLS_IN_LA;
  ADD IUACCAREALST: AREA=AG, AREAID=2, SUBRANGE=SPECIAL_IMSIPRE, IMSIPRE="12317", CTRLTYPE=REJECT, CAUSE=NO_SUITABLE_CELLS_IN_LA;
  ADD S1ACCAREALST: AREA=AG, AREAID=3, SUBRANGE=SPECIAL_IMSIPRE, IMSIPRE="12317", CTRLTYPE=REJECT, CAUSE=NO_SUITABLE_CELLS_IN_TA;
  ```
3. 激活基于无线区域网络地址选择。
  //打开License配置开关。
  ```
  SET LICENSESWITCH: LICITEM="LKV2NSBR01", SWITCH=ENABLE;
  ```
  //配置应用本特性的无线区域。
  ```
  SET MMFUNC: RANBASEDSHARE=GERAN-1&UTRAN-1&E-UTRAN-1, NOIDCHG=NO;
  ```

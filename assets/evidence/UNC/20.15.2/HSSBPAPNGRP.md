#### [命令功能](#ZH-CN_CONCEPT_0000001263701290)

**适用网元：MME**

此命令用于查询HSS BYPASS最小APN签约数据群组配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001263701290)

无

#### [本地用户权限](#ZH-CN_CONCEPT_0000001263701290)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_CONCEPT_0000001263701290)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001263701290)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| APNGRPID | APN群组ID | 可选必选说明：可选参数<br>参数含义：该参数用于指定APN本地配置签约群组ID。<br>数据来源：本端规划<br>取值范围：0~511<br>默认值：无<br>配置原则：无 |
| APNSUBIDX | APN本地签约索引 | 可选必选说明：可选参数<br>参数含义：该参数用于指定APN群组包含的APN本地签约数据索引。<br>数据来源：本端规划<br>取值范围：0~255<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001263701290)

查询HSS BYPASS最小APN签约数据群组配置，可以用如下命令：

LST HSSBPAPNGRP:;

```
%%LST HSSBPAPNGRP:;%%
RETCODE = 0  操作成功。

操作结果如下
------------
      APN群组ID  =  0
APN本地签约索引  =  12
    是否默认APN  =  否
(结果个数 = 1)

---    END
```

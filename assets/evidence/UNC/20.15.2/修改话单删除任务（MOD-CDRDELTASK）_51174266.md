#### [命令功能](#ZH-CN_CONCEPT_0251174266)

**适用NF：NCG**

该命令用于修改已经存在的话单删除任务规则。

执行任务之前，可执行 [**LST CDRDELTASK**](查询话单删除任务（LST CDRDELTASK）_51174267.md) 命令查询当前系统中需要修改的话单删除任务情况，找到对应的删除任务标识。

#### [注意事项](#ZH-CN_CONCEPT_0251174266)

- 此命令可以动态生效。
- 对于“话单删除类型”配置为“按时间删除”的任务，NCG只支持对“RU的ID”、“话单压缩天数”、“话单保存天数”参数的修改，不支持对“话单删除目录”、“话单压缩源目录”、“话单压缩目的目录”参数的修改。若需要修改目录，可先执行[**RMV CDRDELTASK**](删除话单删除任务（RMV CDRDELTASK）_51174265.md)命令删除话单删除任务，再执行[**ADD CDRDELTASK**](增加话单删除任务（ADD CDRDELTASK）_51174264.md)命令重新配置删除任务。
- 不支持对“话单删除类型”参数的修改。
- 对于“话单删除类型”配置为“按文件名删除”的任务，NCG只支持对“待删除话单前缀”、“待删除话单后缀”和“话单删除时间间隔”参数的修改，不支持对“话单删除目录”参数的修改。

#### [本地用户权限](#ZH-CN_CONCEPT_0251174266)

manage-ug；system-ug

#### [网管用户权限](#ZH-CN_CONCEPT_0251174266)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0251174266)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| CDRDELTASKID | 删除任务标识 | 可选必选说明：必选参数<br>参数含义：话单删除任务标识，全局唯一。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～32。<br>默认值：无<br>配置原则：不能输入的特殊字符请参考“<br>[**特殊字符表**](../话单分发/增加话单分发（ADD CDRDISTR）_51174254.md#ZH-CN_CONCEPT_0251174254__table_0365FEF0)<br>”。 |
| RUID | RU的ID | 可选必选说明：可选参数<br>参数含义：RU的ID。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为64～4294967294。<br>默认值：无<br>配置原则：该值需要执行<br>[**LST SERVICERUSTATE**](../../../../../平台服务管理/单体服务编排功能管理/系统管理/资源管理/RU信息/查询RU的信息(LST SERVICERUSTATE)_29626965.md)<br>命令，查询出存在的RU ID进行填写。 |
| CDRDELTYPE | 话单删除类型 | 可选必选说明：必选参数<br>参数含义：该参数用于指定话单的删除类型。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- DELETEBYTIME：按时间删除。<br>- DELETEBYFILENAME：按文件名删除。<br>默认值：无<br>配置原则：无 |
| STOREDAYS | 话单保存天数 | 可选必选说明：条件可选参数<br>前提条件：该参数在“CDRDELTYPE”配置为“DELETEBYTIME”时为条件可选参数。<br>参数含义：自定义目录下话单文件的保存天数，当配置为非压缩删除任务时，NCG将自动删除“话单删除目录”下超过此期限的话单文件；当配置为压缩删除任务时，NCG将自动删除“话单压缩目的目录”下超过此期限的压缩文件。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～180。<br>默认值：无<br>配置原则：<br>- 同一话单删除任务的“话单保存天数”应大于“话单压缩天数”（同时为0除外）。<br>- 此参数配置为“0”时，表示不删除。<br>- 在判断是否超过期限时，只判断话单文件修改日期到系统当天日期是否超过期限。 |
| COMPRESSDAYS | 话单压缩天数 | 可选必选说明：条件可选参数<br>前提条件：该参数在“CDRDELTYPE”配置为“DELETEBYTIME”时为条件可选参数。<br>参数含义：当选择“是否压缩话单”为“YES(是)”时，此参数有效。 自定义目录下话单文件的压缩天数，NCG自动将“话单压缩源目录”下超过此期限的话单文件进行压缩，并删除源文件。压缩后的文件日期和源文件保持一致，保存在“话单压缩目的目录”下。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～180。<br>默认值：无<br>配置原则：<br>- 当修改的任务为非压缩删除任务时，此参数配置无效。<br>- 此参数配置为“0”时，表示不压缩也不删除。<br>- 在判断是否超过期限时，只判断话单文件修改日期到系统当天日期是否超过期限。 |
| CDRDELPREFIX | 待删除话单文件名前缀 | 可选必选说明：条件可选参数<br>前提条件：该参数在“CDRDELTYPE”配置为“DELETEBYFILENAME”时为条件可选参数。<br>参数含义：该参数用于指定待删除话单的前缀。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～128。<br>默认值：无<br>配置原则：<br>- 长度不超过128的字符串，且只能为大小写英文字母、数字、字符“-”及字符“_”。<br>- 当同时配置了“待删除话单文件名前缀”和“待删除话单文件名后缀”时，会删除同时包含前缀和后缀的话单文件。<br>- 当未配置“待删除话单文件名前缀”和“待删除话单文件名后缀”时，会删除归档路径下所有话单文件。 |
| CDRDELSUFFIX | 待删除话单文件名后缀 | 可选必选说明：条件可选参数<br>前提条件：该参数在“CDRDELTYPE”配置为“DELETEBYFILENAME”时为条件可选参数。<br>参数含义：该参数用于指定待删除话单的后缀。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～128。<br>默认值：无<br>配置原则：<br>- 长度不超过128的字符串，且只能为大小写英文字母、数字、字符“-”及字符“_”。<br>- 当同时配置了“待删除话单文件名前缀”和“待删除话单文件名后缀”时，会删除同时包含前缀和后缀的话单文件。<br>- 当未配置“待删除话单文件名前缀”和“待删除话单文件名后缀”时，会删除归档路径下所有话单文件。 |
| CDRDELINTERVAL | 话单删除时间间隔 | 可选必选说明：条件可选参数<br>前提条件：该参数在“TIMETYPE”配置为“SECOND”时为可选参数。<br>参数含义：该参数用于指定执行完一次删除任务后到下一次删除任务开始的时间间隔，单位为秒。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～3600，单位是秒。<br>默认值：无<br>配置原则：无 |
| TIMETYPE | 话单删除时间类型 | 可选必选说明：条件可选参数<br>前提条件：该参数在“CDRDELTYPE”配置为“DELETEBYFILENAME”时为可选参数。<br>参数含义：该参数用于指定删除重命名话单文件时的时间类型。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- SECOND：话单保存的时间单位为秒。<br>- DAY：话单保存的时间单位为天。<br>默认值：无<br>配置原则：无 |
| RENMSTOREDAYS | 重命名话单保存天数 | 可选必选说明：条件可选参数<br>前提条件：该参数在“TIMETYPE”配置为“DAY”时为可选参数。<br>参数含义：自定义目录下重命名话单文件的保存天数，NCG将自动删除“话单删除目录”下超过此期限的重命名话单文件。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～180。<br>默认值：无<br>配置原则：<br>- 参数配置为“0”时，表示不删除。<br>- 参数填写不为“0”时，删除自定义目录及其一级子目录下超过保存天数的重命名话单文件。 |
| CDRDELSUBLEV | 话单删除子目录层级 | 可选必选说明：可选参数<br>参数含义：该参数用于指定话单删除子目录层级。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～32。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0251174266)

修改删除任务标识为“cdr_deltask2”的话单删除任务，将“话单保存天数”修改为“90”，将“话单压缩天数”修改为“10”：

```
MOD CDRDELTASK: CDRDELTASKID="cdr_deltask2", CDRDELTYPE=DELETEBYTIME, STOREDAYS=90, COMPRESSDAYS=10;
```

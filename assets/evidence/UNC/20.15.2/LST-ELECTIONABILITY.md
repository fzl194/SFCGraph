## [命令功能](#ZH-CN_MMLREF_0242938063)

该命令已废弃。

该命令用于查询业务进程选举能力的开关。

## [注意事项](#ZH-CN_MMLREF_0242938063)

无

#### [操作用户权限](#ZH-CN_MMLREF_0242938063)

G_1，管理员级别命令组

## [参数说明](#ZH-CN_MMLREF_0242938063)

无

## [使用实例](#ZH-CN_MMLREF_0242938063)

查询客户端选举能力使能状态。

```
LST ELECTIONABILITY;
RETCODE = 0  操作成功

结果如下
------------------------
业务进程选举能力使能开关  =  开启
(结果个数 = 1)
```

## [输出结果说明](#ZH-CN_MMLREF_0242938063)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 业务进程选举能力使能开关 | 该参数表示客户端选举能力的使能状态。 |

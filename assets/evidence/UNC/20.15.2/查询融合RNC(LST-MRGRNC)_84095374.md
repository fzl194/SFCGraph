#### [命令功能](#ZH-CN_TOPIC_0000002184095374)

**适用网元：SGSN**

该命令用于查询融合RNC。

#### [注意事项](#ZH-CN_TOPIC_0000002184095374)

- 此命令执行后立即生效。

#### [本地用户权限](#ZH-CN_TOPIC_0000002184095374)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_TOPIC_0000002184095374)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_TOPIC_0000002184095374)

无。

#### [使用实例](#ZH-CN_TOPIC_0000002184095374)

查询融合RNC。

LST MRGRNC:;

```
%%LST MRGRNC:;%%
RETCODE = 0  操作成功。

输出结果如下
--------------
  RNC索引 =  0
  RNC名称 =  HUAWEI
(结果个数 = 1)

---    END
```

#### [命令功能](#ZH-CN_CONCEPT_0000001600866737)

该命令用于显示OSPF动态主机名的信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600866737)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600866737)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600866737)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | 进程号 | 可选必选说明：可选参数<br>参数含义：OSPF进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600866737)

查看OSPF进程1的动态主机名信息：

```
DSP OSPFHOSTTABLE:PROCID=1;
```

```

RETCODE = 0  操作成功。

结果如下
-------------------------
            进程号  =  1
            区域号  =  0.0.0.0
        动态主机名  =  HostName
        路由器标识  =  192.168.7.1
自治域洪泛的主机名  =  区域范围
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600866737)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 进程号 | 该参数表示OSPF进程号。 |
| 区域号 | 该参数表示OSPF区域号。 |
| 动态主机名 | 该参数表示OSPF动态主机名。 |
| 路由器标识 | 该参数表示OSPF路由器标识。 |
| 自治域洪泛的主机名 | 主机名的洪泛范围，包括路径范围，区域范围和自治域范围。 |

#### [命令功能](#ZH-CN_MMLREF_0000001126145780)

**适用网元：MME**

该命令用于设置S1模式CE Mode B（一种eMTC终端）用户的SM协议参数。

#### [注意事项](#ZH-CN_MMLREF_0000001126145780)

该命令执行后立即生效。

#### [本地用户权限](#ZH-CN_MMLREF_0000001126145780)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126145780)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126145780)

无。

#### [使用实例](#ZH-CN_MMLREF_0000001126145780)

查询S1模式下CE MODE B用户SM协议参数：

LST EMTCBESM:;

```
%%LST EMTCBESM:;%%
RETCODE = 0  操作成功。

操作结果如下
------------
    T3485 （s）  =  48
N3485 （times）  =  5
    T3486 （s）  =  48
N3486 （times）  =  5
    T3489 （s）  =  48
N3489 （times）  =  5
    T3495 （s）  =  48
N3495 （times）  =  5
(结果个数 = 1)

---    END
```

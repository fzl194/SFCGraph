#### [命令功能](#ZH-CN_CONCEPT_0000001550281374)

该命令用于显示DHCPv4客户端状态信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001550281374)

该命令在Ethernet接口上配置执行。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550281374)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550281374)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IFNAME | 接口名称 | 可选必选说明：可选参数<br>参数含义：该参数用于显示接口名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。<br>默认值：无<br>配置原则：请使用LST INTERFACE命令查看可用接口。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001550281374)

显示DHCPv4客户端状态信息：

```
DSP DHCPCLIENTSTAT: IFNAME="Ethernet64/0/4";
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001550281374)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 接口名称 | 用于显示接口名称。 |
| MAC地址 | 用于显示接口的MAC地址。 |
| IPv4地址 | 用于显示获取的IP地址。 |
| IPv4地址掩码 | 用于显示获取的IP地址掩码。 |
| 服务器IPv4地址 | 用于显示DHCPv4服务器地址。 |
| 状态 | 用于显示DHCPv4客户端状态。 |
| 获取时间 | 用于显示地址获取的时间。 |
| 租期时间 | 用于显示地址超期的时间。 |
| 续租时间 | 用于显示地址续租的时间。 |
| 重绑定时间 | 用于显示地址重新绑定的时间。 |
| 中止原因 | 用于显示地址申请失败的原因。 |
| 租期时长 | 用于租期时长，租期为无限时才有值。 |

## [命令功能](#ZH-CN_MMLREF_0209653843)

**适用NF：PGW-C**

该命令用来查看指定APN实例的P-CSCF地址优先级信息。

## [注意事项](#ZH-CN_MMLREF_0209653843)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209653843)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209653843)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| APN | APN名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定APN实例名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~63。只能由“-”、数字、大小写字母和“.”组成，不能以“.”开头且不能出现连续两个“.”。不支持空格及“_”、“#”、“$”、“&”、“%”、“^”、“（”、“）”、“，”、“/”、“;”、“:”、“””、“`”特殊字符，不区分大小写。<br>默认值：无<br>配置原则：<br>该参数使用ADD APN命令配置生成。 |

## [使用实例](#ZH-CN_MMLREF_0209653843)

- 显示APN实例huawei.com下的P-CSCF地址优先级信息：
  ```
  LST APNPCSCFSRVPRI:APN="huawei.com";
  RETCODE = 0  操作成功

  结果如下
  --------------------------
                    APN名称  =  huawei.com
  IPv4 第一优先级服务器属性  =  dhcp
  IPv4 第二优先级服务器属性  =  local
  IPv6 第一优先级服务器属性  =  dhcp
  IPv6 第二优先级服务器属性  =  local
  (结果个数 = 1)
  ---    END
  ```
- 查询整机APN的P-CSCF服务器优先级信息：
  ```
  LST APNPCSCFSRVPRI:;
  RETCODE = 0  操作成功。

  结果如下
  --------------------------
  APN名称       IPv4 第一优先级服务器属性    IPv4 第二优先级服务器属性    IPv6 第一优先级服务器属性    IPv6 第二优先级服务器属性
                   
  huawei.com       dhcp                         local                        dhcp                         local                                       
  test          dhcp                         local                        dhcp                         local                    
  (结果个数 = 2)
  ---    END
  ```

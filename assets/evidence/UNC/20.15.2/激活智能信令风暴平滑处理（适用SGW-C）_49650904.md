## [操作步骤](#ZH-CN_OPI_0249650904)

1. 进入 “MML命令行-UNC” 窗口。
2. 使能 “DDN消息无响应时进行用户延时删除处理” 功能，并设置延时时长。
  **[SET DDNTIMEOUTCTRL](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/接入管理运维/流控管理/DDN消息无响应处理策略/设置DDN定时器超时参数（SET DDNTIMEOUTCTRL）_46314493.md)**

## [任务示例](#ZH-CN_OPI_0249650904)

任务描述

运营商希望开启DDN消息无响应延时删除用户功能，配置用户延时删除定时器时长区间值为40分钟至70分钟。

脚本

//使能 “DDN消息无响应时进行用户延时删除处理” 功能，并设置延时时长。

```
SET DDNTIMEOUTCTRL: DELAYSUBDELSW=ENABLE, MINDELAYTIMER=40, MAXDELAYTIMER=70;
```

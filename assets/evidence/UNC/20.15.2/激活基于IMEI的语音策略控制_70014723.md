## [操作步骤](#ZH-CN_OPI_0170014723)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
  > **说明**
  > 本特性和“ [1.2-WSFD-201002 用户群语音策略控制](../WSFD-201002 用户群语音策略控制_70014708.md) ”特性交互部署存在以下方式，运营商可以根据实际情况选择：
  >
  > - [同时部署](../WSFD-201002 用户群语音策略控制/激活用户群语音策略控制_64009940.md#ZH-CN_OPI_0164009940__cmd775893272180640)。
  > - 先部署“[1.2-WSFD-201002 用户群语音策略控制](../WSFD-201002 用户群语音策略控制_70014708.md)”特性，再部署本特性。过程分别参见[独立部署](../WSFD-201002 用户群语音策略控制/激活用户群语音策略控制_64009940.md#ZH-CN_OPI_0164009940__cmd235429820180640)和本特性操作步骤。
3. 增加IMEI群组信息。
    a. 增加IMEI群组。
      [**ADD IMEIGP**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/用户终端管理/IMEI群组管理/增加IMEI群组(ADD IMEIGP)_72225435.md)
    b. 增加IMEI群组成员。
      [**ADD IMEIGPMEM**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/用户终端管理/IMEI群组成员管理/增加IMEI群组成员(ADD IMEIGPMEM)_26305568.md)
4. 启用基于IMEI的语音策略。
  [**MOD VOICEDEPLOY**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/语音业务管理/修改语音部署配置(MOD VOICEDEPLOY)_26145762.md)

## [任务示例](#ZH-CN_OPI_0170014723)

任务描述

运营商A（NO ID=0）只允许特定型号终端的本网用户使用IMS VoPS业务，且优先使用VoLTE语音业务，不允许外网用户使用VoLTE语音业务。对于Voice Centric类型的终端需要考虑UE的SRVCC能力。

脚本

**部署方式一** ：先部署用户群语音策略控制，再部署本特性。

1. 进入 “MML命令行-UNC” 窗口。
2.
  **用户群语音策略控制** 特性配置。

  a. 打开 **用户群语音策略控制** 特性的License配置开关。
      ```
      SET LICENSESWITCH: LICITEM="LKV2VPCU01", SWITCH=ENABLE;
      ```
    b. 配置用户群语音策略。
      //配置本网用户的语音策略。
      ```
      ADD VOICEDEPLOY: SUBRANGE=HOME_USER, NOID=0, IMSVOPS=YES, APNNI="IMS", UETYPE=VOICE_CENTRIC-1&DATA_CENTRIC-0, CSFBNOTPREIND=YES;
      ```
      //配置外网用户的语音策略。
      ```
      ADD VOICEDEPLOY: SUBRANGE=FOREIGN_USER, NOID=0, IMSVOPS= NO;
      ```
3.
  **基于IMEI的语音策略控制** 特性配置。

  a. 打开 **基于IMEI的语音策略控制** 特性的License配置开关。
      ```
      SET LICENSESWITCH: LICITEM="LKV2VPCI01", SWITCH=ENABLE;
      ```
    b. 增加IMEI群组信息。
      ```
      ADD IMEIGP: IMEIGPID=1, IMEIGPN="Phone_Type1";
      ```
      ```
      ADD IMEIGPMEM: IMEIGPID=1, IMEITAC="12345678", UEDESC="Phone_Type11";
      ```
      ```
      ADD IMEIGPMEM: IMEIGPID=1, IMEITAC="87654321", UEDESC="Phone_Type12";
      ```
    c. 启用基于IMEI的语音策略。
      ```
      MOD VOICEDEPLOY: SUBRANGE=HOME_USER, NOID=0, IMSVOPS=YES, IMEICTRL=WHITELIST, IMEIGPID=1;
      ```

**部署方式二** ：本特性和用户群语音策略控制同时部署。

1. 进入 “MML命令行-UNC” 窗口。
2. 增加IMEI群组信息。
  ```
  ADD IMEIGP: IMEIGPID=1, IMEIGPN="Phone_Type1";
  ```
  ```
  ADD IMEIGPMEM: IMEIGPID=1, IMEITAC="12345678", UEDESC="Phone_Type11";
  ```
  ```
  ADD IMEIGPMEM: IMEIGPID=1, IMEITAC="87654321", UEDESC="Phone_Type12";
  ```
3. 增加语音策略。
  //配置本网用户的语音策略。
  ```
  ADD VOICEDEPLOY: SUBRANGE=HOME_USER, NOID=0, IMSVOPS=YES, APNNI="IMS", UETYPE=VOICE_CENTRIC-1&DATA_CENTRIC-0, CSFBNOTPREIND=YES, IMEICTRL=WHITELIST, IMEIGPID=1;
  ```
  //配置外网用户的语音策略。
  ```
  ADD VOICEDEPLOY: SUBRANGE=FOREIGN_USER, NOID=0, IMSVOPS= NO;
  ```
4. 打开 **用户群语音策略控制** 特性的License配置开关。
  ```
  SET LICENSESWITCH: LICITEM="LKV2VPCU01", SWITCH=ENABLE;
  ```
5. 打开 **基于IMEI的语音策略控制** 特性的License配置开关。
  ```
  SET LICENSESWITCH: LICITEM="LKV2VPCI01", SWITCH=ENABLE;
  ```

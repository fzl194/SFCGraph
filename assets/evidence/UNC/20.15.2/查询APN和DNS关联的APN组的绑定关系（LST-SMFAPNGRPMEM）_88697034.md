## [命令功能](#ZH-CN_MMLREF_0000001088697034)

**适用NF：PGW-C、SMF、GGSN**

该命令用于查询APN和DNS关联的APN组的绑定关系。

## [注意事项](#ZH-CN_MMLREF_0000001088697034)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001088697034)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001088697034)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| GRPNAME | APN组名 | 可选必选说明：可选参数<br>参数含义：该参数用于指定APN组名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~31。不区分大小写。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001088697034)

查询APN和DNS关联的APN组的绑定关系：

```
LST SMFAPNGRPMEM:GRPNAME="grp1";
```

## [输出结果说明](#ZH-CN_MMLREF_0000001088697034)

| 输出项名称 | 输出项解释 |
| --- | --- |
| APN组名 | 该参数用于指定APN组名。 |
| APN名称 | 该参数用于指定配置到APN组内的APN名称。 |

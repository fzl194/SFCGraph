## [操作场景](#ZH-CN_OPI_0276669503)

本特性支持NB-IoT（NarrowBand Internet of Things）制式终端通过NB-IoT RAN接入MME，实现对NB-IoT终端的移动性管理与会话管理。

> **说明**
> 适用于MME。

## [必备事项](#ZH-CN_OPI_0276669503)

前提条件

- 请仔细阅读[WSFD-011601 NB-IoT终端标准接入特性概述（适用于MME）](特性概述_76669502.md)。
- 已完成[激活S-GW/P-GW拓扑选择特性](../../../网关选择功能/WSFD-205004 S-GW_P-GW拓扑选择/激活S-GW_P-GW拓扑选择特性_63535133.md)和[配置到MSC的数据（SGs接口）](../../../../../初始配置/UNC初始配置与调测/组网对接配置/配置AMF&MME&SGSN/配置MME/配置到MSC的数据（SGs接口）_82467084.md)。

## [操作步骤](#ZH-CN_OPI_0276669503)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. **可选：**配置S6a接口兼容性策略。
  [**SET DMCMPT**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性配置/设置Diameter兼容性(SET DMCMPT)_26306080.md)
  > **说明**
  > - 当对端HSS不支持NB-IoT RAT时，则将参数“NBIOTRAT（是否支持NBIoT RAT）”设置为“NOT_SUPPORT”，此时系统对终端的接入控制将不考虑从HSS获取的签约数据。
  > - “HSS免升级”特性（特性编号： **WSFD-111412** ，License部件编码：81202653）的相关License授权并开启后，若发现系统中不存在 **[ADD DMCMPTBYIMSI](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性IMSI号段配置/增加IMSI对应的Diameter兼容性(ADD DMCMPTBYIMSI)_72225977.md)** 或者 **[MOD DMCMPTBYIMSI](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性IMSI号段配置/修改IMSI对应的Diameter兼容性(MOD DMCMPTBYIMSI)_26306110.md)** 命令对该参数的配置记录，该参数配置生效。

## [任务示例](#ZH-CN_OPI_0276669503)

任务描述

假设对端HSS能识别NB-IoT RAT，配置本特性。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2NBPS01", SWITCH=ENABLE;
```

//配置用户为NB-IoT第一档用户。

```
SET LICENSESWITCH: LICITEM="LKV2NBFQ01", SWITCH=ENABLE;
```

//配置用户为NB-IoT第二档用户。

```
SET LICENSESWITCH: LICITEM="LKV2NBFQ02", SWITCH=ENABLE;
```

//配置S6a接口兼容性策略 。

```
SET DMCMPT: NBIOTRAT=SUPPORT;
```

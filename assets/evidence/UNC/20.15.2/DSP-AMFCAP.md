## [命令功能](#ZH-CN_MMLREF_0000001230167479)

**适用NF：AMF**

该命令用于查询AMF相对容量信息。

## [注意事项](#ZH-CN_MMLREF_0000001230167479)

如果使用DSP AMFCAP查询的AMF相对容量信息有效（无效值为0xFFFF），则系统实际生效的AMF相对容量以该命令输出结果为准。否则以LST AMFINFO输出的相对容量信息为准。

#### [操作用户权限](#ZH-CN_MMLREF_0000001230167479)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001230167479)

无

## [使用实例](#ZH-CN_MMLREF_0000001230167479)

查询AMF相对容量信息，执行如下命令：

```
%%DSP AMFCAP:;%%
RETCODE = 0  操作成功

结果如下
--------
 AMF名称  =  AMF1.CLUSTER1.NET2.AMF.5GC.MNC003.MCC460.3GPPNETWORK.ORG
相对容量  =  0
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001230167479)

| 输出项名称 | 输出项解释 |
| --- | --- |
| AMF名称 | 该参数用于在运营商网络中唯一标识本AMF实例。 |
| 相对容量 | 该参数表示AMF下发给基站的相对容量信息。 |

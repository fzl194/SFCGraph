## [操作场景](#ZH-CN_OPI_0251453007)

本操作指导介绍在运行网络中，UE进行跨区域移动，所在区域的（R）AN超出了原来UPF的服务范围的场景下，保证UE会话业务连续性的操作过程。

> **说明**
> 适用于SMF。

## [操作步骤](#ZH-CN_OPI_0251453007)

1. 进入 “MML命令行-UNC” 窗口。
2. 配置规划的UPF为SMF提供的服务区域。
  **[ADD PNFSMFSERAREA](../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/服务化接口管理/注册与服务发现/本地NRF功能管理/对端SMF服务区管理/增加对端NF的SMF服务区域信息（ADD PNFSMFSERAREA）_09653019.md)**
  > **说明**
  > UNC 支持规划某些UPF仅作为分流UPF使用，不支持当前特性，因此配置规划此特性中的UPF时，需要执行 **[LST UPNODE](../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/PFCP接口管理/UP管理/UP节点管理/查询UPF节点（LST UPNODE）_09652974.md)** 命令，确保当前规划的UPF对应的 “是否仅支持作为分流UPF” 取值为 “不使能” 。
3. 配置此UPF的服务区。
  **[ADD UPAREA](../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/PFCP接口管理/UP管理/UP跟踪区管理/UP区域管理/增加UPF服务区（ADD UPAREA）_09652457.md)**
4. 配置此UPF服务区服务的TAI范围。
  **[ADD UPAREABINDN2TAI](../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/PFCP接口管理/UP管理/UP跟踪区管理/TAI绑定UP区域/增加UPF服务区名称绑定的5G TAI范围（ADD UPAREABINDN2TAI）_09653098.md)**
5. 配置I-UPF选择失败处理方式。
  **[SET IUPFSELFAIL](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/UPF选择管理/I-UPF选择失败处理/设置I-UPF选择失败处理方式（SET IUPFSELFAIL）_59000295.md)**

## [任务示例](#ZH-CN_OPI_0251453007)

任务描述

5G UE在UPF1范围作为本地接入进行正常的5G PDU会话连接，后UE进行跨区域移动，移出UPF1范围移动到UPF2范围内时，需要确保UE的PDU会话连接连续性，其中：

UPF1和UPF2均为SMF提供服务区域。如果当SMF选择UPF2出现失败时，希望能保留当前用户的会话不释放。

UPF1的NF Instance ID为：12e4567-e89b-12d3-a456-112233440012，为SMF提供的服务区域为area01，此区域提供的5G TAI范围为：12345000001~12345123456。

UPF2的NF Instance ID为：56e4567-e89b-12d3-a456-667788990012，为SMF提供的服务区域为area02，此区域提供的5G TAI范围为：67891000001~67891123456。

脚本

//配置指定UPF1和UPF2为SMF提供的服务区域。

```
ADD PNFSMFSERAREA: NFINSTANCEID="12e4567-e89b-12d3-a456-112233440012", SMFSERVINGAREA="area01";
ADD PNFSMFSERAREA: NFINSTANCEID="56e4567-e89b-12d3-a456-667788990012", SMFSERVINGAREA="area02";
```

//配置指定UPF的为5G服务。

```
ADD UPAREA: AREANAME="area01", AREATYPE=N2TAI;
ADD UPAREA: AREANAME="area02", AREATYPE=N2TAI;
```

//配置指定UPF为SMF提供的5G服务区域TAI范围。

```
ADD UPAREABINDN2TAI: AREANAME="area01", N2BEGINTAI="12345000001", N2ENDTAI="12345123456";
ADD UPAREABINDN2TAI: AREANAME="area02", N2BEGINTAI="67891000001", N2ENDTAI="67891123456";
```

//配置I-UPF选择失败处理方式。

```
SET IUPFSELFAIL: SWITCH=ENABLE, FAILPROC=KEEP_SESSION;
```

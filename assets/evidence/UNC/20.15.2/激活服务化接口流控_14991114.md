## [任务示例](#ZH-CN_OPI_0000001614991114)

任务描述

**任务1：** 以NRF作为HTTP服务端为例，在NRF上配置链路粒度的基于IP流控组的固定速率流控。

**任务2：** 以NRF作为HTTP服务端为例，在NRF上配置整机粒度的基于IP流控组的固定速率流控。

**任务3：** 以SMF收到AMF发送的Nsmf_PDUSession_CreateSMContext Request和Nsmf_PDUSession_RetrieveSMContext Request消息为例 ，在SMF上配置指定消息类型的固定速率流控。

**任务4：** 以SMF到AMF的局向为例 ，SMF作为HTTP服务端，在SMF上配置基于接口类型的固定速率流控。

**任务5：** 配置全局固定带宽流控。

**任务6：** 以本端作为HTTP客户端为例，配置基于ReqCB资源的流控。

**任务7：** 配置基于内存资源的流控。

脚本

**任务1：** 以NRF作为HTTP服务端为例，在NRF上配置链路粒度的基于IP流控组的固定速率流控。

//配置流控组信息。

```
ADD HTTPFCIPGRP: INDEX=1, GROUP=1, IPTYPE=IPv4, IPV4ADDR="192.168.100.1", IPV4MASK="255.255.255.0", DESC="AMF_instance1"; 
ADD HTTPFCIPGRP: INDEX=2, GROUP=1, IPTYPE=IPv4, IPV4ADDR="192.168.101.2", IPV4MASK="255.255.255.0", DESC="AMF_instance2"; 
```

//配置流控速率。

```
ADD HTTPFIXEDFC: GROUP=1, GROUPTYPE=SERVER, RECVTHD=1000, STATUSCODE=TooManyRequests, RETRYAFTERFLAG=NO, DESC="LINK";  
```

**任务2** ：以NRF作为HTTP服务端为例，在NRF上配置整机粒度的基于IP流控组的固定速率流控。

//配置流控组信息。

```
ADD HTTPFCIPGRP: INDEX=3, GROUP=2, IPTYPE=IPv4, IPV4ADDR="192.168.200.1", IPV4MASK="255.255.255.0", DESC="AMF_instance1"; 
```

//配置流控速率。

```
ADD HTTPFIXEDFC: GROUP=2, GROUPTYPE=SERVER, GLBRECVTHD=2000, STATUSCODE=TooManyRequests, RETRYAFTERFLAG=NO, DESC="GLOBAL";  
```

**任务3** ：以SMF收到AMF发送的Nsmf_PDUSession_CreateSMContext Request和Nsmf_PDUSession_RetrieveSMContext Request消息为例 ，在SMF上配置指定消息类型的固定速率流控。

//配置指定消息类型的固定速率流控。

```
SET HTTPFIXEDFCMSG: MSGTYPE=NSMF_PDUSESSION_SMCONTEXTSCREATE, FCSWITCH=ON, THRESHOLD=1200;
```

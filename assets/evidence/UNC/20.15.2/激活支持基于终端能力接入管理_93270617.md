## [任务示例](#ZH-CN_OPI_0193270617)

任务描述

配置 UNC ，使得具有DCNR能力的终端能够接入网络。

脚本

//增加NSA控制策略。

```
ADD NSACTRLPLCY: SUBRANGE=FOREIGN_USER, NOID=2, ISDCNR=YES, ISSECRATRPTSGW=YES, ISSECRATRPT=YES, ISGUUPSEL=YES;
```

//配置GTP-C V2协议兼容性。

```
ADD GTPCV2CMPT: IDX=0, MSGCLS=TM, TMMSGTYPE=CRT_SES_REQ, CRTSESREQ=UPSELIND, UPSELFLAG=DCNR-1;
```

//配置S1接口兼容性。

```
SET S1CMPT: NRR=YES;
```

//配置S6a接口兼容性策略。

```
ADD DMCMPTBYIMSI: SUBRANGE=IMSI_PREFIX, IMSIPRE="460123", SF2=FLSTID2_BIT27-1, HSIMSVOPS=BY_SYS_CAP, NODCNR=YES;
```

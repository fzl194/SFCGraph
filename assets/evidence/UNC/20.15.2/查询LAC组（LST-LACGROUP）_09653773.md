## [命令功能](#ZH-CN_MMLREF_0209653773)

**适用NF：GGSN**

该命令用来查看指定LAC组或者所有LAC组的配置信息。

## [注意事项](#ZH-CN_MMLREF_0209653773)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209653773)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209653773)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| LACGROUPNAME | LAC组名 | 可选必选说明：可选参数<br>参数含义：该参数用于指定TAC组名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~32。不支持空格，不区分大小写。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209653773)

假设运营商需要去查询指定“LAC组名”为“beijing”的LAC组：

```
LST LACGROUP: LACGROUPNAME="beijing";
RETCODE = 0  操作成功。

结果如下
--------
指定LAC组名  =  beijing
(结果个数 = 1)
---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209653773)

| 输出项名称 | 输出项解释 |
| --- | --- |
| LAC组名 | 该参数用于指定TAC组名。 |

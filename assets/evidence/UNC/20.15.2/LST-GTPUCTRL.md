#### [命令功能](#ZH-CN_MMLREF_0000001172345439)

**适用网元：SGSN、MME**

该命令用于显示用户面控制参数信息。

#### [注意事项](#ZH-CN_MMLREF_0000001172345439)

无。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172345439)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172345439)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172345439)

无。

#### [使用实例](#ZH-CN_MMLREF_0000001172345439)

查询用户面控制参数：

LST GTPUCTRL:;

```
%%LST GTPUCTRL:;%%
RETCODE = 0  操作成功。

输出结果如下
-------------------------
                   新SGSN开始下行时间(100ms)  =  36
                     停止接收切换数据时间(s)  =  3
                GTP-U路径断后多久去活PDP (s)  =  0
   流量更新时是否将PDP信息上报到用户跟踪开关  =  关闭
                    2G支持特殊业务类型扩展头  =  不支持
                    3G透传特殊业务类型扩展头  =  透传
(结果个数 = 1)

---    END
```

#### [输出结果说明](#ZH-CN_MMLREF_0000001172345439)

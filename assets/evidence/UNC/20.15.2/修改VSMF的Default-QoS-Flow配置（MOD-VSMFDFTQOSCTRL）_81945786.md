## [命令功能](#ZH-CN_MMLREF_0000001181945786)

**适用NF：SMF**

该命令用来修改V-SMF的Default QoS Flow配置。

## [注意事项](#ZH-CN_MMLREF_0000001181945786)

命令执行后只对新接入用户生效。

#### [操作用户权限](#ZH-CN_MMLREF_0000001181945786)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001181945786)

| 参数标识 | 参数名称 | 参数说明 |

| MCC | 移动国家码 | 可选必选说明：必选参数<br>参数含义：该参数用于表示组成QoS控制的移动国家码信息。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度是3。<br>默认值：无<br>配置原则：无 |

| DFTQOSTYPE | 缺省QoS类型 | 可选必选说明：可选参数<br>参数含义：该参数用于表示组成QoS控制的缺省QoS类型。<br>数据来源：全网规划<br>取值范围：<br>- NONGBR（Non-GBR）<br>- GBR（GBR）<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001181945786)

将“移动国家码”为“460”，“移动网号”为“00”，控制类型为GLOBAL_LEVEL的QoS配置修改为“ARP的优先级别”为“1”，“ARP的抢占能力”为NOT_PREEMPT，“ARP的被抢占能力”为PREEMPTABLE。

```
MOD VSMFDFTQOSCTRL: MCC="460", MNC="00", CTRLTYPE=GLOBAL_LEVEL, ARPPL=1, ARPPCI=NOT_PREEMPT, ARPPVI=PREEMPTABLE;
```

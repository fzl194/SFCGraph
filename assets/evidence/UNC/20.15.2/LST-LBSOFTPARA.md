#### [命令功能](#ZH-CN_CONCEPT_0129627112)

该命令用于查询CSLB系统软件参数信息。

#### [注意事项](#ZH-CN_CONCEPT_0129627112)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0129627112)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0129627112)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PARAID | 参数标识 | 可选必选说明：可选参数<br>参数含义：软件参数ID<br>数据来源：本端规划<br>取值范围：0~1023<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0129627112)

查询CSLB软件调试参数表中参数标识为1023的参数值:

LST LBSOFTPARA: PARAID=1023;

```
%%LST LBSOFTPARA: PARAID=1023;%%
RETCODE = 0  操作成功。

操作结果如下：
--------------
参数标识  =  1023
  参数值  =  2
(结果个数 = 1)
---    END
```

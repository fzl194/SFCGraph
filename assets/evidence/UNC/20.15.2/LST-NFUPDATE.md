## [命令功能](#ZH-CN_MMLREF_0209653014)

**适用NF：AMF、SMF、NRF、NSSF、SMSF、NCG**

该命令用于查询NF信息更新到NRF时的方式。

## [注意事项](#ZH-CN_MMLREF_0209653014)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209653014)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209653014)

无

## [使用实例](#ZH-CN_MMLREF_0209653014)

查询当前NF信息更新到NRF时采用的方式。

```
%%LST NFUPDATE:;%%
RETCODE = 0  操作成功

结果如下
--------
      Patch更新缓存最大数量  =  5
Patch更新缓存定时器时长(分钟)  =  2
( 结果个数= 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209653014)

| 输出项名称 | 输出项解释 |
| --- | --- |
| Patch更新缓存最大数量 | 当Patch更新缓存数量达到该参数表示的值，会向NRF推送缓存的更新请求。 |
| Patch更新缓存定时器时长(分钟) | Patch更新缓存的定时推送任务会按照该参数表示的时间间隔向NRF推送缓存中的更新信息。 |

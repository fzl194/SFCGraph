#### [命令功能](#ZH-CN_MMLREF_0000001126145698)

**适用网元：SGSN、MME**

该命令用来查询APNNI组成员信息。

#### [注意事项](#ZH-CN_MMLREF_0000001126145698)

该命令执行后立即生效。

#### [本地用户权限](#ZH-CN_MMLREF_0000001126145698)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126145698)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126145698)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| GRPID | APNNI组号 | 可选必选说明：可选参数<br>参数含义：该参数用于指定APNNI组号。<br>数据来源：本端规划<br>取值范围：0～15<br>默认值：无 |

#### [使用实例](#ZH-CN_MMLREF_0000001126145698)

查询APNNI组成员信息：

LST APNNI:;

```
%%LST APNNI:;%%
输出结果如下
--------------
APNNI组号  =  2
    APNNI  =  HUAWEI.COM
     描述  =  noname
(结果个数 = 1)

---    END
```

#### [输出结果说明](#ZH-CN_MMLREF_0000001126145698)

| 输出项名称 | 输出项解释 |
| --- | --- |
| APNNI组号 | 参数含义：该参数用于指定APNNI组ID号<br>取值范围：0～15 |
| APNNI | 参数含义：该参数用来设置APNNI信息<br>取值范围：1～62位字符串 |
| 描述 | 参数含义：APNNI描述信息<br>取值范围：0～256位字符串 |

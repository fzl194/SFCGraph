#### [命令功能](#ZH-CN_CONCEPT_0259103553)

该命令用于显示VNFC的版本信息。

在日常的维护和版本升级时，可使用本命令显示系统当前的基础包、补丁包版本信息。

#### [注意事项](#ZH-CN_CONCEPT_0259103553)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0259103553)

G_1，管理员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0259103553)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SERVICEINSTANCE | 服务实例 | 可选必选说明：必选参数<br>参数含义：该参数表示大颗粒服务实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，通过LST VNFC命令获取。<br>默认值：无<br>配置原则：只能填写通过LST VNFC命令查询到的管理代理标识。 |

#### [使用实例](#ZH-CN_CONCEPT_0259103553)

显示VNFC的版本信息：

```
DSP VERSION:
SERVICEINSTANCE="vnfc"
;
```

```
RETCODE = 0  操作成功

结果如下:
-------------------------
      包类型  =  版本包
    版本类型  =  基础版本
      版本号  =  V100R005C00
版本应用类型  =  BaseApp
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0259103553)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 包类型 | 表示包的类型，包括版本包，风险排查包和App软件包。 |
| 版本类型 | 表示版本类型，包括基础版本、冷补丁、热补丁。 |
| 版本号 | 表示软件包的版本号。 |
| 版本应用类型 | 表示版本应用类型。 |

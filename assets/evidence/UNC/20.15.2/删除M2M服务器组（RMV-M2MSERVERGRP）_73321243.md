# 删除M2M服务器组（RMV M2MSERVERGRP）

- [命令功能](#ZH-CN_MMLREF_0273321243__1.3.1)
- [注意事项](#ZH-CN_MMLREF_0273321243__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0273321243__1.3.4)
- [使用实例](#ZH-CN_MMLREF_0273321243__1.3.5)

## [命令功能](#ZH-CN_MMLREF_0273321243)

**适用NF：PGW-C、SMF**

该命令用于删除M2M服务器配置，以及M2M服务器组下的M2M服务器IP配置。

## [注意事项](#ZH-CN_MMLREF_0273321243)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0273321243)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0273321243)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| GROUPNAME | M2M服务器组名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定M2M服务器组的名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~31。字符串类型，输入长度范围为1～31。不区分大小写，不支持空格。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0273321243)

删除组名为m2msrvgroup01的M2M服务器组配置信息：

```
RMV M2MSERVERGRP:GROUPNAME="m2msrvgroup01";
```

## [命令功能](#ZH-CN_MMLREF_0000001113337934)

**适用NF：SGW-C**

该命令用来查看语音IMS的DDN消息流控时是否上报CHR功能的开关状态。

## [注意事项](#ZH-CN_MMLREF_0000001113337934)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001113337934)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001113337934)

无

## [使用实例](#ZH-CN_MMLREF_0000001113337934)

显示语音IMS的DDN消息流控时上报CHR功能开关状态：

```
%%LST IMSDDNFLOWCTRLCHR:;%%
RETCODE = 0  操作成功

结果如下
--------
IMS DDN流量控制上报CHR配置开关  =  使能
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001113337934)

| 输出项名称 | 输出项解释 |
| --- | --- |
| IMS DDN流量控制上报CHR配置开关 | 该参数用来设置IMS DDN流量控制上报CHR配置开关。 |

## [操作场景](#ZH-CN_OPI_0277897405)

当运营商需要部署基于延迟定时器的信令拥塞控制功能时，需要对本功能进行配置，确保本功能可以正常使用。

> **说明**
> 适用于PGW-C。

- 完成加载License。

## [操作步骤](#ZH-CN_OPI_0277897405)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  **[SET LICENSESWITCH](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)**
3. 设置全局Back-Off时长。
  **[SET GBACKOFFTIME](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/接入管理运维/流控管理/全局Back-off Time信息/设置全局Back-off Time信息（SET GBACKOFFTIME）_76686936.md)**
4. 配置APN的 Back-off时长。
  [**SET BACKOFFTIME**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/接入管理运维/流控管理/Back-off Time信息/设置Back-off Time信息（SET BACKOFFTIME）_09652083.md)

## [任务示例](#ZH-CN_OPI_0277897405)

任务描述

系统过载时使用APN的Back-off time功能。

脚本

// 打开本特性的License配置开关。

```
SET LICENSESWITCH:LICITEM="LKV3WMTSBT11",SWITCH=ENABLE;
```

//设置全局Back-Off时长。

```
SET GBACKOFFTIME:BACKOFFTIMER=600;
```

//配置APN粒度的PGW-C Back-off time功能。

```
SET BACKOFFTIME:APNNAME="apn-op", OVERLOADSW=ENABLE, BACKOFFTIMER=900;
```

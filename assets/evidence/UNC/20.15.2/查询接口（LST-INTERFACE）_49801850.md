#### [命令功能](#ZH-CN_CONCEPT_0000001549801850)

该命令用于查询设备上接口的信息。设备接口包括逻辑接口及物理接口。物理接口是真实存在的接口。逻辑接口是指能够实现数据交换功能但物理上不存在、需要通过配置建立的接口。

不指定IFNAME参数时，查询设备上所有接口的信息；当指定IFNAME参数时，可以查询指定接口的配置信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001549801850)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549801850)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549801850)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IFNAME | 接口名 | 可选必选说明：可选参数<br>参数含义：该参数用于指定接口名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。接口名称由接口类型+接口编号组成。接口编号是十进制自然数，当编号为0时填写为“0”，其他情况不得以“0”开头。<br>默认值：无 |
| IFCLASS | 接口类别 | 可选必选说明：可选参数<br>参数含义：该参数用于设置接口的主接口或子接口信息。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- mainInterface：主接口。<br>- subInterface：子接口。<br>默认值：无 |
| IFPHYTYPE | 接口类型 | 可选必选说明：可选参数<br>参数含义：该参数用于设置接口的物理类型。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- Ethernet：以太接口。<br>- Eth_trunk：Eth-Trunk接口。<br>- Tunnel：Tunnel接口。<br>- Loopback：Loopback接口。<br>- Null：Null接口。<br>- IMEth：内联管理接口。<br>- Fabric-Tunnel：Fabric-Tunnel接口。<br>- ISMeth：内联服务接口。<br>- IPMEth：内联物理管理接口。<br>- ISPMeth：内联服务物理接口。<br>- 400GE：400GE接口。<br>- 200GE：200GE接口。<br>- 100GE：100GE接口。<br>- 25GE：25GE接口。<br>默认值：无 |
| VRFNAME | VPN实例名称 | 可选必选说明：可选参数<br>参数含义：该参数用于设置接口的绑定的VPN实例名字。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～50。<br>默认值：无 |

| 接口名 | 用于表示接口名。 |
| 接口类别 | 用于显示接口的主接口或子接口信息。 |

| 接口编号 | 用于显示接口的编号。 |

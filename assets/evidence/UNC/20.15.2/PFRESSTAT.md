#### [命令功能](#ZH-CN_MMLREF_0000001172225533)

**适用网元：SGSN、MME**

该命令用于查询资源状态表，这些资源包括PDP数状态、QoS带宽状态、静态带宽状态、动态带宽状态。

#### [注意事项](#ZH-CN_MMLREF_0000001172225533)

无

#### [本地用户权限](#ZH-CN_MMLREF_0000001172225533)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172225533)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172225533)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RUNAME | RU名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定资源单元名称。该参数可以通过<br>**DSP RU**<br>命令查询。<br>取值范围：0~63位字符串<br>默认值：无 |
| PN | 进程号 | 可选必选说明：可选参数<br>参数含义：该参数用于指定UPP进程的序号。<br>取值范围：0~20<br>默认值：无 |

#### [使用实例](#ZH-CN_MMLREF_0000001172225533)

查询资源单元为USN_SP_RU_0064上0号UPP进程的资源状态：

LST PFRESSTAT: RUNAME="USN_SP_RU_0064", PN=0;

```
%%LST PFRESSTAT: RUNAME="USN_SP_RU_0064", PN=0;%%
RETCODE = 0  操作成功。

输出结果如下
--------------
                 RU名称  =  USN_SP_RU_0064
                 进程号  =  0
        转发面PDP数状态  =  正常
  CONV(QoS)带宽使用状态  =  正常
STREAM(QoS)带宽使用状态  =  正常
INTACT(QoS)带宽使用状态  =  正常
    BG(QoS)带宽使用状态  =  正常
           动态带宽状态  =  正常
           静态带宽状态  =  正常
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_MMLREF_0000001172225533)

| 输出项名称 | 输出项解释 |
| --- | --- |
| RU名称 | 资源单元名称。 |
| 进程号 | 待查询的UPP进程的序号。 |
| 转发面PDP数状态 | PDP上下文表的使用状态。<br>取值说明：<br>- 正常：PDP数小于忙门限时的状态。<br>- 忙：PDP数大于等于忙门限，小于过载门限时的状态。<br>- 过载：PDP数大于等于过载门限，小于拥塞门限时的状态。<br>- 拥塞：PDP数大于等于拥塞门限时的状态。<br>说明：- 过载和拥塞门限可以通过命令[**LST USRPDPCAP**](../PDP资源管理/用户面PDP规格管理/查询用户面PDP规格表(LST USRPDPCAP)_72345453.md)获得。<br>- 忙门限计算方法：- 如果过载恢复门限>20，忙恢复门限=过载恢复门限-10。否则，忙恢复门限=过载恢复门限/2。（如果过载恢复门限设置为1，则忙恢复门限为0，此时将无法恢复成正常状态。）<br>- 如果过载门限>20，忙门限=过载门限-10。否则，忙门限=过载门限/2。 （忙恢复门限大于0的情况下，如果忙门限等于忙恢复门限，则把忙门限加1） |

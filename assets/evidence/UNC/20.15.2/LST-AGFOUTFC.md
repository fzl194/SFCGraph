## [命令功能](#ZH-CN_MMLREF_0000001232847667)

**适用NF：NCG**

该命令用于查询AGF出局流控参数。

## [注意事项](#ZH-CN_MMLREF_0000001232847667)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001232847667)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001232847667)

无

## [使用实例](#ZH-CN_MMLREF_0000001232847667)

查询AGF出局流控参数

```
+++    UNC/*MEID:0 MENAME:UNC_VNF_ncg2*/        2021-12-12 18:48:24+8:00
O&M    #1886
%%LST AGFOUTFC:;%%
RETCODE = 0  操作成功

结果如下
--------
                                出局流控模式  =  会话级和全局流控模式
                            全局流控阈值(次)  =  20000
                        全局流控恢复阈值(次)  =  4000
                            消息间隔条数(条)  =  10
                    会话级流控退出时长(分钟)  =  120
                会话级流控状态退避时长(分钟)  =  60
                            全局统计周期(秒)  =  60
                   是否统计TOPO发现失败次数   =  true
            全局流控状态最大退避时长(秒)      = 600
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001232847667)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 出局流控模式 | 该参数用于控制流控模式。取值说明:<br>值为ALL时，默认开启会话级流控处理模式和OCS全局流控处理模式。<br>值为CLOSE时，关闭流控处理功能。<br>值为SESSION时，仅开启会话级流控处理模式。<br>值为GLOBAL时，仅开启OCS全局拥塞流控处理模式。<br>值为ISOLATE_ALL时，开启隔离全部对端网元，对所有在线计费消息直接代应答模式。<br>值为MSG_BY_STEP时，开启在线计费消息按照条数间隔转发给OCS模式。 |
| 全局流控阈值(次) | 该参数用于调整在设置OCS全局流控模式下，在参数GFCPERIOD设置的时间内，拥塞统计次数达到该值后，切换至拥塞状态，默认取值20000次。如当参数GFCPERIOD设置为60s,该参数设置为10000时，表示60s内转发给某个OCS的计费消息出现10000次拥塞统计后切换至拥塞状态。 |
| 全局流控恢复阈值(次) | 该参数用于调整在设置OCS全局流控模式下，在参数GFCPERIOD设置的时间内，拥塞计数次数少于该值后，切换至正常状态，默认取值4000次。如当参数GFCPERIOD设置为60s,该参数设置为2000时，表示60s内转发给某个OCS的计费消息出现少于2000次拥塞计数后，由待决状态切换至正常状态。 |
| 消息间隔条数(条) | 该参数用于调整在在线计费消息按照条数间隔转发给OCS模式下，用于调整消息条数间隔。比如该参数值为10时，表示每隔10条在线计费消息转发一次给OCS。 |
| 会话级流控退出时长(分钟) | 该参数用于调整在会话级流控模式下，调整会话级别待决状态切换至正常状态的无代应答时长，单位分钟。比如该参数设置为60时，表示处于会话级待决状态下，超过60分钟没有出现代应答，则切换至会话级正常状态。 |
| 会话级流控状态最大退避时长(分钟) | 该参数用于调整会话级流控模式下，会话级拥塞状态切换至待决状态的时间的退避上限，单位为分钟，默认60分钟。比如该参数设置为120时，表示会话级拥塞状态最大持续时间为120分钟，当超过120分钟，切换为待决状态。 |
| 全局统计周期(秒) | 该参数用于调整在设置OCS全局流控模式下，设置全局流控统计周期，单位秒。如该参数设置为30秒，表示统计转发给某个OCS的计费消息出现代应答次数的周期为30秒。30秒过后，统计次数清零，进行新的30秒统计。 |
| 是否统计TOPO发现失败次数 | 该参数用于调整会话级流控模式下，是否统计TOPO发现失败次数。比如该参数设置为false时，表示会话级定时器时，出现TOPO发现失败时，也不会进入会话级流控状态。 |
| 全局流控状态最大退避时长(秒) | 该参数用于调整在设置OCS全局流控模式下，全局拥塞状态切换至待决状态的时间的退避上限，单位为分钟，默认600s。比如该参数设置为600时，表示全局拥塞状态最大持续时间为600s，当超过600s，切换为待决状态。 |

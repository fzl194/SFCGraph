#### [命令功能](#ZH-CN_CONCEPT_0000001126305758)

**适用网元：SGSN、MME**

该命令用于查询基于CC选择GGSN/P-GW配置记录。

#### [注意事项](#ZH-CN_CONCEPT_0000001126305758)

无。

#### [本地用户权限](#ZH-CN_CONCEPT_0000001126305758)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_CONCEPT_0000001126305758)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001126305758)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| APNNI | APNNI | 可选必选说明：可选参数<br>参数含义：该参数用于指定APNNI。<br>数据来源：全网规划<br>取值范围：1~62位字符串<br>默认值：无 |
| SUBCC | 签约CC值 | 可选必选说明：可选参数<br>参数含义：该参数用于指定签约CC值。<br>数据来源：全网规划<br>取值范围：0x0000~0xFFFF<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001126305758)

查询所有基于IMEI选择P-GW/GGSN配置：

LST GWSELBYCC:;

```
%%LST GWSELBYCC:;%%
RETCODE = 0  操作成功。

操作结果如下
------------
   APNNI  =  HUAWEI.COM
签约CC值  =  0x1234
(结果个数 = 1)
---    END
```

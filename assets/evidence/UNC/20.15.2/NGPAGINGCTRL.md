## [命令功能](#ZH-CN_MMLREF_0209652258)

**适用NF：AMF**

此命令用于设置NG寻呼策略，比如可基于最近访问RAN、邻接RAN、TA、邻接TA、TA List进行分级寻呼，可设置RAN的学习开关，可设置移动性检查开关等。

## [注意事项](#ZH-CN_MMLREF_0209652258)

- 该命令执行后立即生效。

- 系统部署完成后，已经存在初始记录，参数的初始记录值如下表：

## [使用实例](#ZH-CN_MMLREF_0209652258)

- 将寻呼策略中的邻接GNB的开关打开，执行如下命令：
  ```
  SET NGPAGINGCTRL:NEIBGNBSW=ON;
  ```
- 将寻呼策略中的邻接TA学习开关和邻接TA开关打开，并设置邻接TA寻呼超时时长为3秒，邻接TA寻呼超时重发次数为0次，邻接TA老化时间为360小时，执行如下命令：
  ```
  SET NGPAGINGCTRL: NEIBTASW=ON, NEIBTAT3513=3, NEIBTAN3513=0, NEIBTALRNSW=ON, TAEXPIRETIME=360;
  ```

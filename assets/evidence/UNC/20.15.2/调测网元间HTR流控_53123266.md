## [操作场景](#ZH-CN_OPI_0000001553123266)

本操作指导介绍在运行网络中对 网元间HTR流控 功能进行调测的操作过程，下面的调测过程以AMF和SMF到UDM/PCF的HTR流控过程为例。

## [必备事项](#ZH-CN_OPI_0000001553123266)

前提条件

- 请仔细阅读[WSFD-206009 网元间HTR流控特性概述](WSFD-206009 网元间HTR流控特性概述_04122089.md)。
- 完成[调测网元间HTR流控](调测网元间HTR流控_53123266.md)。

数据

该操作无需准备数据。

工具

OM Portal

2. 进行业务测试。
  **场景1：AMF与UDM之间链路拥塞触发HTR流控** 。
    a. 创建UAM随机用户跟踪，过滤条件为：“接入类型”选择“NGRAN”，“流程类型”选择“Initial Registration Procedure”，“原因值类型”选择“Protocol Cause”，“协议原因值”选择“Congestion (22)”，“筛选条件类型”选择“No Filter”。
    b. 构造AMF与UDM之间链路拥塞，消息失败率达到配置的HTR流控阈值。
    c. 在告警监控界面检查是否上报了**[ALM-100576 HTTP HTR流控](../../../../../网络运维/故障处理/UNC告警处理/平台告警/ALM-100576 HTTP HTR流控_01513825.md)**告警。
      预期结果：网元上报 **[ALM-100576 HTTP HTR流控](../../../../../网络运维/故障处理/UNC告警处理/平台告警/ALM-100576 HTTP HTR流控_01513825.md)** 告警。
    d. 查看UAM随机用户跟踪中是否存在原因值为22的Registration Reject消息。
      ![](调测网元间HTR流控_53123266.assets/zh-cn_image_0000001588266794_2.png "点击放大")
      预期结果：UAM随机用户跟踪中存在原因值为22的Registration Reject消息。
  **场景2：SMF与UDM之间链路拥塞触发HTR流控。**
    a. 创建USM随机用户跟踪，过滤条件为：“接口类型”选择“N1NAS(SMF)”，“流程类型”选择“N1NAS-UE initiated PDU Session Establishment”，“协议原因值”选择“NAS Request rejected, unspecified”。
    b. 构造SMF与UDM之间链路拥塞，消息失败率达到配置的HTR流控阈值。
    c. 在告警监控界面检查是否上报了**[ALM-100576 HTTP HTR流控](../../../../../网络运维/故障处理/UNC告警处理/平台告警/ALM-100576 HTTP HTR流控_01513825.md)**告警。
      预期结果：网元上报 **[ALM-100576 HTTP HTR流控](../../../../../网络运维/故障处理/UNC告警处理/平台告警/ALM-100576 HTTP HTR流控_01513825.md)** 告警。
    d. 查看USM随机用户跟踪中是否存在“ProtocolCause: Nas24501Cause”为“RequestRejectedUnspecified (CauseValue: 31)”的EMS_SIGNLING_ABNORMAL消息。
      ![](调测网元间HTR流控_53123266.assets/zh-cn_image_0000001638012241_2.png)
      预期结果：USM随机用户跟踪中存在 “ProtocolCause: Nas24501Cause” 为 “RequestRejectedUnspecified (CauseValue: 31)” 的EMS_SIGNLING_ABNORMAL消息。
  **场景3：SMF与PCF之间链路拥塞触发HTR流控（以PDU Session创建流程为例）。**
    a. 创建USM随机用户跟踪，过滤条件为：“接口类型”选择“N1NAS(SMF)”，“流程类型”选择“N1NAS-UE initiated PDU Session Establishment”，“协议原因值”选择“Network Failure”。
    b. 构造SMF与PCF之间链路拥塞，消息失败率达到配置的HTR流控阈值。
    c. 在告警监控界面检查是否上报了**[ALM-100576 HTTP HTR流控](../../../../../网络运维/故障处理/UNC告警处理/平台告警/ALM-100576 HTTP HTR流控_01513825.md)**告警。
      预期结果：网元上报 **[ALM-100576 HTTP HTR流控](../../../../../网络运维/故障处理/UNC告警处理/平台告警/ALM-100576 HTTP HTR流控_01513825.md)** 告警。
    d. 查看USM随机用户跟踪中是否存在“ProtocolCause: Nas24501Cause”为“NetworkFailure (CauseValue: 38)”的EMS_SIGNLING_ABNORMAL消息。
      ![](调测网元间HTR流控_53123266.assets/zh-cn_image_0000001592050910_2.png)
      预期结果：USM随机用户跟踪中存在 “ProtocolCause: Nas24501Cause” 为 “NetworkFailure (CauseValue: 38)” 的EMS_SIGNLING_ABNORMAL消息。

# 查询RAC组内绑定的RAC号段（LST RACID）

- [注意事项](#ZH-CN_MMLREF_0209652289__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0209652289__1.3.4)
- [使用实例](#ZH-CN_MMLREF_0209652289__1.3.5)
- [输出结果说明](#ZH-CN_MMLREF_0209652289__1.3.6)

## [命令功能](#ZH-CN_MMLREF_0209652289)

**适用NF：GGSN**

该命令用来查询指定RAC号段与RAC组的绑定关系或者配置的所有RAC号段和RAC组的绑定关系。

## [注意事项](#ZH-CN_MMLREF_0209652289)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209652289)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652289)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RACSECNUM | RAC段编号 | 可选必选说明：可选参数<br>参数含义：该参数用于指定RAC段编号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~2047。<br>默认值：无<br>配置原则：无 |
| RACGROUPNAME | RAC组名 | 可选必选说明：可选参数<br>参数含义：该参数用于指定RAC组名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~32。不支持空格，不区分大小写。<br>默认值：无<br>配置原则：<br>该RACGROUPNAME必须已经由ADD RACGROUP配置过。 |

## [使用实例](#ZH-CN_MMLREF_0209652289)

假设运营商需要查看指定RAC号段与RAC组的绑定关系：“RAC组名”为“beijing”，“RAC号段”为2：

```
LST RACID:;
RETCODE = 0  操作成功。

RAC号段配置信息
---------------
  RAC段编号  =  2
  RAC起始ID  =  0x01
  RAC截止ID  =  0x10
  RAC组名  =  beijing
(结果个数 = 1)
---    END
```

| 输出项名称 | 输出项解释 |
| --- | --- |

| RAC组名 | 该参数用于指定RAC组名。 |
| RAC起始ID | 该参数用于指定RAC起始ID。 |
| RAC截止ID | 该参数用于指定RAC截止ID。 |

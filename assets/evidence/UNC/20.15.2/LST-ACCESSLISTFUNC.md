## [命令功能](#ZH-CN_MMLREF_0296242026)

**适用NF：GGSN、SGW-C、PGW-C**

该命令用来查询接入控制名单功能。

## [注意事项](#ZH-CN_MMLREF_0296242026)

无

#### [操作用户权限](#ZH-CN_MMLREF_0296242026)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0296242026)

无

## [使用实例](#ZH-CN_MMLREF_0296242026)

显示接入控制名单功能：

```
LST ACCESSLISTFUNC
RETCODE = 0  操作成功。

结果如下
--------
设置系统是否支持接入控制名单的功能  =  使能
                      黑白名单类型  =  白名单
(结果个数 = 1)
---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0296242026)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 设置系统是否支持接入控制名单的功能 | 本参数用于控制打开或关闭接入控制名单的功能。 |
| 黑白名单类型 | 本参数用于控制接入控制名单类型。 |

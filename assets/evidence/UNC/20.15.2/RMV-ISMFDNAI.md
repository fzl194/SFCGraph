# 删除I-SMF支持的DNAI（RMV ISMFDNAI）

- [注意事项](#ZH-CN_MMLREF_0000001147600749__1.3.2)

## [命令功能](#ZH-CN_MMLREF_0000001147600749)

**适用NF：SMF**

该命令用于删除I-SMF支持的DNAI。

## [注意事项](#ZH-CN_MMLREF_0000001147600749)

该命令执行后只对新接入该设备且把该设备作为I-SMF的PDU会话生效。

#### [操作用户权限](#ZH-CN_MMLREF_0000001147600749)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001147600749)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| DNAI | 数据网络访问标识符 | 可选必选说明：必选参数<br>参数含义：该参数用于指定I-SMF支持的DNAI。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是1~63。不支持空格，不区分大小写。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001147600749)

删除DNAI为“huawei.com.dnai”的I-SMF DNAI，执行如下命令：

```
RMV ISMFDNAI: DNAI="huawei.com.dnai";
```

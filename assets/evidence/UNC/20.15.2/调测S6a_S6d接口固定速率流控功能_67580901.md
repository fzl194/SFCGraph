# 调测S6a/S6d接口固定速率流控功能

- [操作场景](#ZH-CN_OPI_0167580901__1.3.1)
- [必备事项](#ZH-CN_OPI_0167580901__1.3.2)
- [操作步骤](#ZH-CN_OPI_0167580901__1.3.3)

## [操作场景](#ZH-CN_OPI_0167580901)

S6a/S6d/DRA存在过载风险时，微服务USN基于配置的流控速率阈值，控制Attach、Inter TAU/RAU、Intra TAU(GERAN/UTRAN->LTE)、Intra RAU(LTE->GERAN/UTRAN)流程首次到HSS/DRA的取鉴权集请求或位置更新请求的速率，对于超过阈值的请求，给UE返回Attach Reject或TAU/RAU Reject消息，并携带相应失败原因值。

> **说明**
> 适用于 SGSN、 MME。

## [必备事项](#ZH-CN_OPI_0167580901)

前提条件

该操作无需具备特殊条件。

- 请仔细阅读[S6a/S6d接口固定速率流控功能特性概述](WSFD-010802 周边网元过载保护特性概述（S6a_S6d接口固定速率流控功能）_61608270.md)。
- 完成[激活S6a/S6d接口固定速率流控功能](激活S6a_S6d接口固定速率流控功能_61609826.md)。

数据

该操作无需准备数据。

工具

OM Portal跟踪

    - 性能指标“S1模式EPS附着失败次数（#111 Protocol error, unspecified）”：在用户大量接入时，由于系统未进行S6a/S6d接口固定速率流控，造成S6a/S6d接口拥塞或HSS/DRA设备过载，HSS无法处理AIR请求，附着失败次数陡增。

    - 性能指标“S1模式EPS附着失败次数（拥塞）”：在用户大量接入时，S6a/S6d接口固定速率流控功能防止S6a/S6d接口拥塞或HSS/DRA设备过载，控制用户接入，附着失败次数增加，性能指标曲线升高；随着用户不断接入，待附着用户数减少，被流控拒绝的用户数相应减少，性能指标曲线降低。
    - 性能指标“S1模式平均承载数”、“S1模式最大附着用户”、“S1模式平均附着用户数”：随用户不断接入，性能指标曲线平稳升高，大部分用户成功接入。
3. 查看系统上报的告警 **：** 预期结果：
    - 告警台上存在ALM-10753 S6a/S6d接口固定速率流控起控告警指示。
    - 大部分用户成功接入后，系统向HSS发送的消息（Update-Location-Request/Authentication-Information-Request）数小于[**SET FIXEDFC**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/操作维护/设备管理/流控管理/业务流控管理/固定速率流控管理/设置固定速率流控信息(SET FIXEDFC)_26146142.md)设置的“固定流控速率”阈值，发往HSS的请求消息不再被丢弃，告警自动恢复。
4. 查询流控系统参数 **。**
  进入 “MML命令行-UNC” 窗口。
  [**LST FIXEDFC**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/操作维护/设备管理/流控管理/业务流控管理/固定速率流控管理/查询固定速率流控信息(LST FIXEDFC)_72225821.md)
  预期结果：
    - “固定速率流控开关”显示为“开启”。
    - “固定流控速率（个/秒）”查询输出的数值与配置一致。
5. 显示流控系统参数。
  进入 “MML命令行-UNC” 窗口。
  [**DSP FIXEDFC**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/操作维护/设备管理/流控管理/业务流控管理/固定速率流控管理/显示固定速率流控信息(DSP FIXEDFC)_26305952.md)
  预期结果：
  “固定流控速率（个/秒）” 所显示的数值为实际的流控速率。

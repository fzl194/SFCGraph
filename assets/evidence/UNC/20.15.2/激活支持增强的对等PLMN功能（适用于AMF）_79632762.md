## [操作场景](#ZH-CN_OPI_0279632762)

本操作指导介绍在运行网络中激活支持增强的对等PLMN功能的操作过程。

对等PLMN（Equivalent PLMN）是指可向用户提供和本网相同服务的PLMN。对等PLMN列表中各PLMN，在用户进行PLMN重选、小区重选、切换等流程中完全等价。支持增强的对等PLMN功能，是指 UNC 可以向终端下发“对等PLMN列表”，允许终端根据这个列表记录，自行选择网络资源来为其提供服务。

> **说明**
> 适用于AMF。

## [任务示例](#ZH-CN_OPI_0279632762)

任务描述

将用户按照IMSI前缀配置群组，在运营商A的 UNC 上配置等价PLMN组并标识等价PLMN适用的用户群信息和跟踪区域组信息，并在等价PLMN组中添加运营商B的PLMN作为等价PLMN组成员。等价PLMN群组适用用户为“IMSI前缀为123456”的用户群组，跟踪区域组标识为“3”，并将HPLMN_0和EPLMN_1配置在同一等价PLMN组中，这样HPLMN_0的签约用户就可以选择从运营商B的EPLMN_1中接入。

![](激活支持增强的对等PLMN功能（适用于AMF）_79632762.assets/zh-cn_image_0279634205_2.png)

脚本

//配置用户群组。

```
ADD NGUSRGRP: USRGRPID=20;
```

//为用户群组配置对应IMSI前缀。

```
ADD NGUSRGRPMEM: USRGRPID=20, IMSIPRE="123456";
```

//配置跟踪区域组成员。

```
ADD EPLMNAREAMEM: TACGRPID=3, BGNTAC="120101", ENDTAC="120102";
```

//增加等价PLMN组并标识等价PLMN适用的用户群信息和跟踪区域组信息。

```
ADD EPLMNGRP:GRPIDX=1,PLMNIDX=0,USRGRPID=20,TACGRPID=3;
```

//为等价PLMN组添加组成员。

```
ADD EPLMNGRPMEM: GRPIDX=1, MCC="134", MNC="56";
```

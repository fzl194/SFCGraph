## [命令功能](#ZH-CN_MMLREF_0209652446)

**适用NF：AMF、SMF、NSSF、SMSF、NCG**

该命令用于查询本地配置的对端NF实例支持的群组信息。

## [注意事项](#ZH-CN_MMLREF_0209652446)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209652446)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652446)

无

## [使用实例](#ZH-CN_MMLREF_0209652446)

查询本地配置的对端NF支持的群组信息。

```
%%LST PNFGROUPID:;%%
RETCODE = 0 操作成功

结果如下
------------------------
  NF实例标识 = udm_instance_0
    群组标识 = GROUP01
（结果个数 = 1）

----    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209652446)

| 输出项名称 | 输出项解释 |
| --- | --- |
| NF实例标识 | 该参数用于指定NF实例标识。 |
| 群组标识 | 该参数用于指定NF服务的群组标识。 |

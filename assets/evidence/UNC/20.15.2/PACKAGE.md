#### [命令功能](#ZH-CN_CONCEPT_0259036083)

该命令用于下载升级软件包。版本升级一般先后执行下载，加载和激活命令。

#### [注意事项](#ZH-CN_CONCEPT_0259036083)

- 该命令执行后立即生效。
- VNFP支持VNF级和VNFC级下载路径，VNFC支持VNFC级下载路径。
- 如果是VNF级路径，命令执行后通过[**DSP SOFTWARE**](../../../系统管理/系统维护/软件仓软件/显示软件仓软件（DSP SOFTWARE）_59036808.md)查询是否下载成功。如果是VNFC级路径，命令执行后通过[**DSP PACKAGE**](../../../../单体服务公共功能管理/系统管理/版本信息/显示系统软件信息（DSP PACKAGE）_59103342.md)查询是否下载成功。
- 如果软件仓中存在与目标下载的软件包同版本号，同VNFC类型的软件包，执行[**LOD PACKAGE**](../../../../单体服务公共功能管理/操作维护/系统调测/升级维护/加载软件包（LOD PACKAGE）_59104019.md)命令，加载的是软件仓中的软件包。

#### [操作用户权限](#ZH-CN_CONCEPT_0259036083)

G_1，管理员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0259036083)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IPVERSION | 服务器IP版本 | 可选必选说明：可选参数<br>参数含义：该参数用于指定服务器地址族。<br>数据来源：对端协商<br>取值范围：枚举类型。<br>- **IPv4**：IPv4地址。<br>- **IPv6**：IPv6地址。<br>默认值：<br>**IPv4** |
| SERVERIP | 服务器IPv4地址 | 可选必选说明：条件必选参数，该参数在<br>“IPVERSION”<br>配置为<br>“IPv4”<br>时为必选参数。<br>参数含义：该参数用于指定服务器端的IP地址。<br>数据来源：对端协商<br>取值范围：IPv4地址类型。<br>默认值：无 |
| SERVERPORT | 服务器端口号 | 可选必选说明：必选参数<br>参数含义：该参数用于指定服务器端的服务端口。<br>数据来源：对端协商<br>取值范围：整数类型，取值范围为1～65535。<br>默认值：无 |
| USERNAME | 服务器用户名 | 可选必选说明：必选参数<br>参数含义：该参数用于指定服务器端的用户名。<br>数据来源：对端协商<br>取值范围：字符串类型，输入长度范围为1～63。<br>默认值：无 |
| PASSWORD | 服务器密码 | 可选必选说明：必选参数<br>参数含义：该参数用于指定服务器端的密码。<br>数据来源：对端协商<br>取值范围：密码类型，输入长度范围为1～127。<br>默认值：无 |
| UPDOBJECT | 升级对象 | 可选必选说明：必选参数<br>参数含义：该参数用于指定升级对象。<br>数据来源：对端协商<br>取值范围：枚举类型。<br>- **BaseSoftware**：只升级基础版本。<br>- **Patch**：只升级补丁。<br>- **All**：升级基础版本和补丁。<br>默认值：无 |
| BASICPKGDIR | 基础软件包路径 | 可选必选说明：条件必选参数，该参数在<br>“UPDOBJECT”<br>配置为<br>“BaseSoftware”<br>或<br>“All”<br>时为必选参数。<br>参数含义：该参数用于指定服务器端基础软件包存放路径，VNFP支持VNF级路径和VNFC级路径两种模式，VNFC支持VNFC级路径。<br>数据来源：对端协商<br>取值范围：字符串类型，输入长度范围为1～255。<br>默认值：无 |
| PATCHDIR | 补丁路径 | 可选必选说明：条件必选参数，该参数在<br>“UPDOBJECT”<br>配置为<br>“Patch”<br>或<br>“All”<br>时为必选参数。<br>参数含义：该参数用于指定服务器端补丁包存放路径，VNFP支持VNF级路径和VNFC级路径两种模式，VNFC支持VNFC级路径。<br>数据来源：对端协商<br>取值范围：字符串类型，输入长度范围为1～255。<br>默认值：无 |
| TRANSFERPRTL | 传输协议 | 可选必选说明：可选参数<br>参数含义：该参数用于指定下载软件包使用的传输协议。<br>数据来源：对端协商<br>取值范围：枚举类型。<br>- **SFTP**：使用SFTP协议下载文件。<br>默认值：<br>**SFTP** |
| SERVERIPV6 | 服务器IPv6地址 | 可选必选说明：条件必选参数，该参数在<br>“IPVERSION”<br>配置为<br>“IPv6”<br>时为必选参数。<br>参数含义：该参数用于指定服务器IPv6地址，服务器需要提前放置新的软件包。<br>数据来源：对端协商<br>取值范围：IPv6地址类型。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0259036083)

下载升级软件包：

```
DLD PACKAGE: SERVERIP="192.168.0.15",SERVERPORT=22,USERNAME="HUAWEI-123",PASSWORD="*****",UPDOBJECT=Patch,PATCHDIR="/home/patch",TRANSFERPRTL=SFTP;
```

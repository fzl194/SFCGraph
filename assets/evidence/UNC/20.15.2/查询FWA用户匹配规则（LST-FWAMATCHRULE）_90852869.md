## [命令功能](#ZH-CN_MMLREF_0000002090852869)

**适用NF：SMF**

该命令用于查询FWA用户匹配规则。

## [注意事项](#ZH-CN_MMLREF_0000002090852869)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000002090852869)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000002090852869)

无

## [使用实例](#ZH-CN_MMLREF_0000002090852869)

显示FWA用户匹配规则。

```
%%LST FWAMATCHRULE:;%%
RETCODE = 0 操作成功
结果如下
匹配类型        匹配方式        PCC规则        APN名称
```

## [输出结果说明](#ZH-CN_MMLREF_0000002090852869)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 匹配类型 | 该参数用于指定匹配是否为FWA用户的类型，当前支持按照PCC Rule和APN进行匹配。 |
| 匹配方式 | 该参数用于指定采用全匹配还是只需要包含配置的PCC RULE或者APN的方式判断是否是FWA用户。 |
| PCC规则 | 该参数用于指定用于判断是否是FWA用户的PCC规则。 |
| APN名称 | 该参数用于指定用于判断是否是FWA用户的APN。 |

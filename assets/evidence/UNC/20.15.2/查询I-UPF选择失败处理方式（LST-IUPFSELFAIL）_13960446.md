## [命令功能](#ZH-CN_MMLREF_0000001213960446)

**适用NF：SMF**

该命令用于查询当SMF/I-SMF在Service Request流程中选择I-UPF失败时的处理方式。

## [注意事项](#ZH-CN_MMLREF_0000001213960446)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001213960446)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001213960446)

无

## [使用实例](#ZH-CN_MMLREF_0000001213960446)

查询所有IUPFSELFAIL记录：

```
%%LST IUPFSELFAIL:;%%
RETCODE = 0  操作成功

结果如下
--------
                     开关  =  不使能
             失败处理方式  =  释放会话
释放会话时携带的NAS原因值  =  #38 网络侧失败
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001213960446)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 开关 | 该参数用于表示是否开启I-UPF选择失败处理功能。 |
| 失败处理方式 | 该参数用于指定SMF/I-SMF在Service Request流程中选择I-UPF失败时的处理方式。 |
| 释放会话时携带的NAS原因值 | 该参数用于指定I-UPF选择失败并需要释放PDU会话时所携带的NAS原因值。 |

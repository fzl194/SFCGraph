# 删除用户的SMSF关键信息（RMV SMSFKEYDATA）

## [命令功能](#ZH-CN_MMLREF_0000001531000046)

**适用NF：SMSF**

该命令用于删除用户的SMSF关键信息。当功能调测或拨测过程中需要手动删除用户的SMSF关键信息时，使用该命令。

## [注意事项](#ZH-CN_MMLREF_0000001531000046)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0000001531000046)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001531000046)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SUPI | SUPI | 可选必选说明：必选参数<br>参数含义：该参数用于指定用户的SUPI，在本命令中，可以按指定SUPI删除对应用户的SMSF关键信息。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是5~15。只允许输入十进制数字（0-9）。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001531000046)

操作员在功能调测过程中手动删除值为“460023500100001”的SUPI对应的用户的SMSF用户关键信息，执行如下命令：

```
RMV SMSFKEYDATA: SUPI="460023500100001";
```

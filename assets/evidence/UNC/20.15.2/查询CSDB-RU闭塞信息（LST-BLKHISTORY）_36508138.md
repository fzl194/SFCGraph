#### [命令功能](#ZH-CN_CONCEPT_0136508138)

该命令用于查询闭塞/解闭CSDB RU结果。

#### [注意事项](#ZH-CN_CONCEPT_0136508138)

- 该命令执行后，可立刻查询到结果。

#### [操作用户权限](#ZH-CN_CONCEPT_0136508138)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0136508138)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SCALEGROUP | 物理资源组名称 | 可选必选说明：可选。<br>参数含义：该参数用于指定唯一一个物理资源组。<br>数据来源：该物理资源组名称可以通过<br>**[LST SERVICERUSTATE](../../../单体服务编排功能管理/系统管理/资源管理/RU信息/查询RU的信息(LST SERVICERUSTATE)_29626965.md)**<br>命令查询获取，对应<br>**ScaleGroup的名字**<br>。<br>取值范围：字符串类型，长度为1～63。<br>默认值：无。 |

#### [使用实例](#ZH-CN_CONCEPT_0136508138)

查询闭塞和解闭历史操作结果:

```
LST BLKHISTORY:;
%%LST BLKHISTORY:;%%
RETCODE = 0  操作成功。
操作结果如下：
-------------------------
闭塞次序       闭塞类型    开始时间               结束时间               物理资源组名称        RU列表        闭塞结果      备注信息
1              闭塞        2018-10-15 9:57:9      2018-10-18 6:28:17     SG1_CSDB_ForCommon    64,65         处理成功       NULL
2              解闭        2018-10-15 9:57:14     2018-10-15 9:57:14     SG1_CSDB_ForCommon    64,65         处理成功       NULL
(结果个数 = 2)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0136508138)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 闭塞次序 | 闭塞/解闭命令执行的先后次序。<br>取值范围：0~4294967295。 |
| 闭塞类型 | 闭塞类型。<br>取值范围：<br>- **闭塞**：闭塞操作。<br>- **解闭**：解闭操作。 |
| 开始时间 | 命令开始执行时间。<br>取值范围：字符串类型，长度1~31。 |
| 结束时间 | 命令执行完成时间。<br>取值范围：字符串类型，长度1~31。 |
| 物理资源组名称 | 物理资源组名称。<br>取值范围：字符串类型，长度为1~63。 |
| RU列表 | RU的RU ID组成的列表。<br>取值范围：字符串类型，长度为1~255。 |
| 闭塞结果 | 闭塞/解闭命令执行的结果。<br>取值范围：<br>- **处理中**：操作正在处理中。<br>- **处理成功**：操作处理成功。<br>- **处理失败**：操作处理失败。 |
| 备注信息 | 备注信息。<br>取值范围：字符串类型，长度0~255。 |

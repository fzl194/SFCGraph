## [命令功能](#ZH-CN_MMLREF_0000001261307350)

**适用NF：AMF**

该命令用于查询AMF热备容灾的控制参数。

## [注意事项](#ZH-CN_MMLREF_0000001261307350)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001261307350)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001261307350)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SUBRANGE | 用户范围 | 可选必选说明：可选参数<br>参数含义：该参数用于指定热备容灾的用户范围。<br>数据来源：全网规划<br>取值范围：<br>- “ALL_USER（所有用户）”：所有用户<br>- “HOME_USER（本网用户）”：本网用户<br>- “FOREIGN_USER（外网用户）”：外网用户<br>- “IMSI_PREFIX（IMSI前缀）”：IMSI前缀<br>- “MSISDN_PREFIX（MSISDN前缀）”：MSISDN前缀<br>默认值：无<br>配置原则：无 |
| IMSIPRE | IMSI前缀 | 可选必选说明：该参数在"SUBRANGE"配置为"IMSI_PREFIX"时为条件可选参数。<br>参数含义：该参数用于指定热备容灾用户的IMSI前缀。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是5~15。只允许输入十进制数字（0-9）。<br>默认值：无<br>配置原则：无 |
| MSISDNPRE | MSISDN前缀 | 可选必选说明：该参数在"SUBRANGE"配置为"MSISDN_PREFIX"时为条件可选参数。<br>参数含义：该参数用于指定热备容灾用户的MSISDN前缀。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是5~15。只允许输入十进制数字（0-9）。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001261307350)

查询 所有用户的AMF热备容灾控制参数，执行如下命令：

```
%%LST AMFDRBKCTRL:;%%
RETCODE = 0  操作成功

结果如下
--------
    用户范围  =  所有用户
    IMSI前缀  =  NULL
  MSISDN前缀  =  NULL
特征条件组合  =  不指定特征条件
数据网络名称  =  NULL
切片业务类型  =  NULL
切片细分标识  =  FFFFFF
  匹配优先级  =  255
    备份开关  =  是
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001261307350)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 用户范围 | 该参数用于指定热备容灾的用户范围。 |
| IMSI前缀 | 该参数用于指定热备容灾用户的IMSI前缀。 |
| MSISDN前缀 | 该参数用于指定热备容灾用户的MSISDN前缀。 |
| 特征条件组合 | 该参数用于指示特征条件的组合。 |
| 数据网络名称 | 该参数用于指定热备容灾用户的DNN。 |
| 切片业务类型 | 该参数用于指定应用热备容灾功能的切片业务类型特征。如eMBB（1）、URLLC（2）、MIoT（3）等协议定义的标准SST，或者运营商自定义的SST。 |
| 切片细分标识 | 该参数用于指定应用热备容灾功能的切片细分标识特征。 |
| 匹配优先级 | 该参数用于热备容灾控制规则的匹配优先级。 |
| 备份开关 | 该参数用于指定是否支持热备容灾功能。 |

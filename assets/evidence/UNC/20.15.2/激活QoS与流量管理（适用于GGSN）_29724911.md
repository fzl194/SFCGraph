## [操作场景](#ZH-CN_OPI_0229724911)

通过配置QoS与流量管理，可实现以下需求：

- 为了保证在资源紧缺时，运营商能够为高优先级用户优先分配资源。
- 在资源相对富余时，更充分地发挥网络资源的效益，提高客户满意度。

QoS与流量管理指 UNC 支持3GPP QoS，按照QoS等级对用户报文进行分类，还可以提供DiffServ，从而支持流量监管、拥塞管理、拥塞避免等多种QoS实现技术。

## [任务示例](#ZH-CN_OPI_0229724911)

任务描述

- 配置基于全局的3GPP QoS到DSCP/ToS的映射关系。
- 配置基于APN的3GPP QoS到DSCP/ToS的映射关系。

脚本

- 配置基于全局的3GPP QoS到DSCP/ToS的映射关系。
  //配置整机使能用户的上下行Remark功能。
  ```
  SET QOSREMARK:DOWNLINK=ENABLE,UPLINK=ENABLE;
  ```
  //配置PreR8 QoS参数到IP报文头中的DSCP/ToS的映射规则。
  ```
  LST QOSGLOBAL:;
  ADD PRER8REMARK:QOSPROFILENAME="globalqos",SERVICELEVEL=CONVERSATIONAL,USERPRIORITY=HIGH,REMARKTYPE=DSCP,DSCP=EF;
  ```

- 配置基于APN的3GPP QoS到DSCP/ToS的映射关系。
  //配置整机使能用户的上下行Remark功能。
  ```
  SET QOSREMARK:DOWNLINK=ENABLE,UPLINK=ENABLE;
  ```
  //配置QosProfile，并进入QoS实例。
  ```
  ADD QOSPROFILE:QOSPROFILENAME="gp1";
  ```
  //配置PreR8 QoS参数到IP报文头中的DSCP/ToS的映射规则。
  ```
  ADD PRER8REMARK:QOSPROFILENAME="gp1",SERVICELEVEL=CONVERSATIONAL,USERPRIORITY=HIGH,REMARKTYPE=DSCP,DSCP=EF;
  ```
  //查找APN实例。
  ```
  LST APN:;
  ```
  //为该APN实例绑定QOSPROFILE。
  ```
  SET APNQOSATTR:APN="apn",HASQOSPROFILE=ENABLE,QOSPROFILENAME="gp1";
  ```

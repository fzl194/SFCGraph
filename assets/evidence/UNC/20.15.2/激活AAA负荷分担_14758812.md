## [操作场景](#ZH-CN_OPI_0214758812)

部署多台负荷分担的AAA服务器，解决网络中AAA鉴权和计费的拥塞问题。

> **说明**
> 适用于 GGSN、 PGW-C、SMF。

## [必备事项](#ZH-CN_OPI_0214758812)

前提条件

- 请仔细阅读[WSFD-011308 AAA负荷分担特性概述](特性概述_14758811.md)。
- 如果运营商规划采用VPN组网方式，则操作员在执行本操作前应已创建了相应的VPN实例。
- 完成[增加APN配置](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/APN管理/APN/增加APN配置（ADD APN）_09653747.md)。

## [操作步骤](#ZH-CN_OPI_0214758812)

1. 进入 “MML命令行-UNC” 窗口。
2. 配置AAA服务器组。
  [**ADD RDSSVRGRP**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/RADIUS管理/连接管理/Radius服务器组/增加Radius服务器组（ADD RDSSVRGRP）_09896730.md)
3. 配置AAA鉴权服务器及计费服务器。
    a. 配置AAA鉴权服务器。
      [**ADD RDSSVR**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/RADIUS管理/连接管理/Radius服务器/增加RADIUS服务器（ADD RDSSVR）_09896756.md)
    b. 配置AAA计费服务器。
      [**ADD RDSSVR**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/RADIUS管理/连接管理/Radius服务器/增加RADIUS服务器（ADD RDSSVR）_09896756.md)
4. 配置APN下绑定AAA服务器组。
  [**ADD APNRDSSVRGRP**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/RADIUS管理/连接管理/APN Radius服务器/设置APN Radius服务器组（ADD APNRDSSVRGRP）_09896735.md)

## [任务示例](#ZH-CN_OPI_0214758812)

任务描述

激活AAA负荷分担。

脚本

//配置AAA服务器组。

```
ADD RDSSVRGRP: RDSSVRGRPNAME="isprg", MODE=ROUND_ROBIN, SIGOPTACCTMSG=DISABLE;
```

//配置AAA鉴权服务器及计费服务器，VPNinst实例为"vpn_Gi"。

```
ADD RDSSVR: RDSSVRGRPNAME="isprg", SERVERTYPE=AUTHENTICATION, IPVERSION=IPV4, SERVERIPV4="192.168.7.12", PORT=1812, VPNINSTANCE="vpn_Gi", CIPHERKEY="*****", PRIFLAG=PRIMARY, CIPHERKEYCNFM="*****";
```

```
ADD RDSSVR: RDSSVRGRPNAME="isprg", SERVERTYPE=ACCOUNTING, IPVERSION=IPV4, SERVERIPV4="192.168.7.14", PORT=1813, VPNINSTANCE="vpn_Gi", CIPHERKEY="*****", PRIFLAG=PRIMARY, CIPHERKEYCNFM="*****";
```

//配置APN实例"apn-test"下绑定AAA服务器组。

```
ADD APNRDSSVRGRP:APN="apn-test",RDSSVRGRPNAME="isprg";
```

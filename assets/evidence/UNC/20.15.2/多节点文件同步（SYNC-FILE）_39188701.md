## [命令功能](#ZH-CN_MMLREF_0000001139188701)

将多个节点的文件进行同步，使得整个文件系统的数据保持一致。

## [注意事项](#ZH-CN_MMLREF_0000001139188701)

无。

## [参数说明](#ZH-CN_MMLREF_0000001139188701)

无。

## [使用实例](#ZH-CN_MMLREF_0000001139188701)

同步文件系统的异常文件：

```
%%SYNC FILE:;%%
RETCODE = 0  操作成功

进度报告
--------
已完成 = 0%
(结果个数 = 1)

---    END

%%SYNC FILE:;%%
RETCODE = 0  操作成功

进度报告
--------
已完成 = 100%
(结果个数 = 1)

---    END

%%SYNC FILE:;%%
RETCODE = 0  操作成功

同步结果
--------
删除文件  =  0
同步文件  =  0
(结果个数 = 1)

共有3个报告
---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001139188701)

该命令执行正常，会返回命令执行成功的提示信息。

| **输出项名称** | **输出项解释** |
| --- | --- |
| 删除文件 | 用于返回存活实例删除文件的总个数。 |
| 同步文件 | 用于返回存活实例同步文件的总个数。 |

#### [命令功能](#ZH-CN_CONCEPT_0000001621904504)

该命令用于查询CSLB隧道IP-in-IP功能开关配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001621904504)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001621904504)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001621904504)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001621904504)

查询CSLB隧道IP-in-IP功能开关的配置，命令如下：

```
%%LST LBTNIPINIPSWITCH:;%%
RETCODE = 0  操作成功

操作结果如下：
--------------
CSLB隧道IP-in-IP配置开关  =  CSLB隧道IP-in-IP开启
(结果个数 = 1)

---    END
```

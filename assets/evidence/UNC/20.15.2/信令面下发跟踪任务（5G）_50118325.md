UDM作为初始网元

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本功能的License配置开关。
  [**SET LICENSESWITCH**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. 配置AMF是否向UDM查询或者订阅TraceData。
  [**SET NGMMFUNC**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/移动性管理/MM协议参数管理/5G移动性管理/设置5G移动性管理功能（SET NGMMFUNC）_09653748.md)

  > **说明**
  > 当网络中有任意一个UDM不支持Trace Data时，需要通过 [**LST NGMMFUNC**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/移动性管理/MM协议参数管理/5G移动性管理/查询5G移动性管理功能（LST NGMMFUNC）_09653016.md) 命令检查 **全网跟踪开关** 是否为 “否” ，若开关开启，需要通过 [**SET NGMMFUNC**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/移动性管理/MM协议参数管理/5G移动性管理/设置5G移动性管理功能（SET NGMMFUNC）_09653748.md) 命令设置 **E2ESW** 参数为 “否” ，以关闭AMF向UDM查询或者订阅TraceData的开关，避免造成业务流程异常。
4. 配置SMSF是否向UDM单独获取和订阅Trace Data。
  [**SET COMMONSOFTPARAOFBIT**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/软件参数管理/设置公共软件参数比特位（SET COMMONSOFTPARAOFBIT）_26494601.md)
5. 配置SMF是否向UDM单独获取和订阅Trace Data。
  **[SET SMFSOFTPARAOFBIT](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/软件参数管理/设置SMF软件参数比特位（SET SMFSOFTPARAOFBIT）_09652696.md)**
6. （可选）配置跟踪消息上报速率控制（建议使用默认值。如果修改，请联系华为技术支持）。
  [**SET FWSOFTPARABITS**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/软参配置管理/设置软件参数比特位（SET FWSOFTPARABITS）_37580503.md)

AMF作为初始网元

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本功能的License配置开关。
  [**SET LICENSESWITCH**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. 关闭SMSF向UDM单独获取和订阅Trace Data的软参开关（软参值设置为0）。
  [**SET COMMONSOFTPARAOFBIT**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/软件参数管理/设置公共软件参数比特位（SET COMMONSOFTPARAOFBIT）_26494601.md)
4. 关闭SMF向UDM单独获取和订阅Trace Data可通过软参控制（软参值设置为0）。
  **[SET SMFSOFTPARAOFBIT](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/软件参数管理/设置SMF软件参数比特位（SET SMFSOFTPARAOFBIT）_09652696.md)**
5. （可选）配置跟踪消息上报速率控制（建议使用默认值。如果修改，请联系华为技术支持）。
  [**SET FWSOFTPARABITS**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/软参配置管理/设置软件参数比特位（SET FWSOFTPARABITS）_37580503.md)

## [任务示例](#ZH-CN_OPI_0000001350118325)

任务描述

**信令传播下发，AMF作为初始网元，且存在UDM不支持Trace Data的场景**

1. 开启端到端用户跟踪特性，在网管上创建端到端跟踪任务，选择AMF网元。
2. 在无I-SMF（SMF直连AMF）场景下，关闭SMF向UDM单独获取和订阅Trace Data（即软参DWORD1001中BIT31=0）。
3. 在SMSF不需要上报消息到端到端用户跟踪时，关闭SMSF向UDM单独获取和订阅Trace Data（即软参DWORD16中BIT5=0）。

脚本

**信令传播下发，AMF作为初始网元，且存在UDM不支持Trace Data的场景**

//打开端到端用户跟踪的License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2ESTAM01", SWITCH=ENABLE;
SET LICENSESWITCH: LICITEM="LKV3ESTSM02", SWITCH=ENABLE;
SET LICENSESWITCH: LICITEM="LKV2ESTAM03", SWITCH=ENABLE;
```

//在无I-SMF（SMF直连AMF）场景下，关闭SMF向UDM单独获取和订阅Trace Data。

```
SET SMFSOFTPARAOFBIT: DT=Dw, DWORDNUM=1001, VALUE=VALUE_0, POSITION=31;
```

//关闭SMSF向UDM单独获取和订阅Trace Data。

```
SET COMMONSOFTPARAOFBIT: DT=DwCom, DWORDCOMNUM=16, VALUE=VALUE_0, POSITION=5;
```

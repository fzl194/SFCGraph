## [操作步骤](#ZH-CN_OPI_0230285492)

1. 进入 “MML命令行-UNC” 窗口。
2. 设置基于CPU过载流控开关。
  **[SET FCSWITCH](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/服务通信管理/流控管理/设置流控开关（SET FCSWITCH）_09587940.md)**
3. 设置APN下用户接入速率。
  [**SET APNACCESSWAL**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/APN管理/APN接入速率/设置APN接入速率（SET APNACCESSWAL）_09652087.md)
4. 设置N4接口指定消息类型固定流控速率。
  **[SET N4FIXEDFC](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/PFCP接口管理/UP管理/N4接口固定速率/设置指定消息类型固定速率流控信息（SET N4FIXEDFC）_35374751.md)**
5. 设置服务化接口上指定消息类型的固定速率。
  **[SET HTTPFIXEDFCMSG](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/HTTP功能管理/HTTP流控管理/HTTP指定消息类型固定速率流控管理/设置HTTP指定消息类型固定速率流控信息（SET HTTPFIXEDFCMSG）_84132110.md)**
6. 设置信令抑制功能开关以及老化定时器时长。
  **[SET OVERLOADCTRL](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/接入管理运维/流控管理/信令抑制/设置信令抑制功能开关以及老化定时器时长（SET OVERLOADCTRL）_82122531.md)**
7. 设置N4接口上指定PFCP偶联消息类型固定速率流控信息。
  **[SET PFCPFIXEDFC](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/PFCP接口管理/PFCP流控管理/设置指定PFCP消息类型固定速率流控信息（SET PFCPFIXEDFC）_20015330.md)**

## [任务示例](#ZH-CN_OPI_0230285492)

任务描述

系统中度或者重度过载时，启动自保流控。

脚本

//设置基于CPU过载流控开关。

```
SET FCSWITCH: SWITCH=ENABLE;
```

//设置APN接入速率。

```
SETAPNACCESSWAL:APN="huawei.com",WALNUMBER=300;
```

//设置N4接口接收消息流控功能。

```
SET N4FIXEDFC:MSGTYPE=DLDATA, FCSWITCH=ON, THRESHOLD=2000;
```

//设置服务化接口上指定消息类型的固定速率流控参数。

```
SET HTTPFIXEDFCMSG:MSGTYPE=NSMF_PDUSESSION_SMCONTEXTSCREATE,FCSWITCH=ON,THRESHOLD=4000;
```

//设置信令抑制功能开关以及老化定时器时长。

```
SET OVERLOADCTRL: CTRLENABLE=ENABLE,AGINGTIME=34,SRVTRIGCCRI=ENABLE,SRVTRIGACCT=ENABLE;
```

//开启N4接口PFCP偶联消息固定速率流控功能。

```
SET PFCPFIXEDFC:MSGTYPE=HEARTBEATREQ, FCSWITCH=ON, THRESHOLD=500;
```

## [操作场景](#ZH-CN_OPI_0163535142)

本操作指导介绍在运行网络中激活本地P-GW的PDN重建的操作过程。

UE在移动过程中，可能会出现S-GW与P-GW不在同一区域（例如，不同省份）的情况，导致S-GW和P-GW之间的迂回流量增加，业务时延也会增加。本地P-GW的PDN重建特性可以识别S-GW与P-GW不在同一区域的PDN连接，并在对用户业务无影响的前提下，为UE重新选择与S-GW同区域的本地P-GW进行业务。

> **说明**
> 适用于MME。

## [必备事项](#ZH-CN_OPI_0163535142)

前提条件

- 运营商网络中Host Name的规划满足以下条件：
    - Host Name使用“topon”开头。
    - 能够根据Host Name中的Canonical Node Name区分不同网元设备之间的拓扑关系远近。
- 请仔细阅读[WSFD-205007 本地P-GW的PDN重建特性概述](特性概述_70166794.md)。
- 完成加载License。
- 本特性的依赖特性“[WSFD-205004 S-GW/P-GW拓扑选择](../WSFD-205004 S-GW_P-GW拓扑选择_63535132.md)”已经激活。有关如何激活“[WSFD-205004 S-GW/P-GW拓扑选择](../WSFD-205004 S-GW_P-GW拓扑选择/特性概述_70166785.md)”，请参见[激活S-GW/P-GW拓扑选择特性](../WSFD-205004 S-GW_P-GW拓扑选择/激活S-GW_P-GW拓扑选择特性_63535133.md)。
- 本特性在使用过程中，如果需要保证用户从GnGp SGSN迁移到MME上时，UNC作为新侧MME能够使用本特性，则需要首先激活“[WSFD-205006 基于P-GW锚点选择S-GW](../WSFD-205006 基于P-GW锚点选择S-GW_63535138.md)”特性。 有关如何激活“[WSFD-205006 基于P-GW锚点选择S-GW](../WSFD-205006 基于P-GW锚点选择S-GW_63535138.md)”，请参见[激活基于P-GW锚点选择S-GW](../WSFD-205006 基于P-GW锚点选择S-GW/激活基于P-GW锚点选择S-GW特性_63535139.md)。

## [操作步骤](#ZH-CN_OPI_0163535142)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. 配置不同类型业务（APN）的本地P-GW重选策略。
    a. 针对IMS语音业务，配置根据承载状态进行本地P-GW的重选。即采用当UE没有QCI为1（语音）或2（视频）的EPS承载时，系统发起P-GW重选过程。
      [**ADD PGWRESEL**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/会话管理/本地P-GW重选功能配置/增加本地P-GW重选策略(ADD PGWRESEL)_26305522.md)
    b. 针对PS数据业务，配置根据指定时间进行本地P-GW的重选。即系统在每天的固定时段发起P-GW重选过程。
      [**ADD PGWRESEL**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/会话管理/本地P-GW重选功能配置/增加本地P-GW重选策略(ADD PGWRESEL)_26305522.md)
4. 设置重建用户部分PDN连接时Deactivate Bearer Request消息携带原因值为“#39: reactivation requested”。
  [**SET SOFTPARAOFBIT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/操作维护/软件参数管理/软件参数比特位/设置软件参数表比特位(SET SOFTPARAOFBIT)_72345783.md)

## [任务示例](#ZH-CN_OPI_0163535142)

任务描述

启用本地P-GW的PDN重建。

针对IMS语音和PS数据业务分别配置不同的本地P-GW重选策略。

- 针对IMS语音业务，配置根据承载状态进行本地P-GW的重选。
- 针对PS数据业务，配置根据指定时间进行本地P-GW的重选：在每天凌晨1点到2点，UE进入ECM-IDLE状态60秒后，系统发起P-GW重选，为UE选择本地的P-GW。

脚本

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2PRLP01", SWITCH=ENABLE;
```

//针对IMS语音业务，配置根据承载状态进行本地P-GW的重选。

```
ADD PGWRESEL: APNNI="IMS", NBEGIN=7, NEND=8, TRIGGER=BEARER;
```

//针对PS数据业务，配置根据指定时间进行本地P-GW的重选。

```
ADD PGWRESEL: APNNI="Internet", NBEGIN=7, NEND=8, TRIGGER=TIME, ST=01&00&00, ET=02&00&00, ECMIDLETIME=60;
```

//设置重建用户部分PDN连接时Deactivate Bearer Request消息携带原因值为“#39: reactivation requested”。

```
SET SOFTPARAOFBIT: DT=DWORD_EX, PARANUM=12, VALUE=VALUE_1, POSITION=30;
```

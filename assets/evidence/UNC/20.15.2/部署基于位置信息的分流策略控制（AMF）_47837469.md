## [任务示例](#ZH-CN_OPI_0000001147837469)

任务描述

运营商需要为某企业用户提供基于小区粒度用户位置信息的UL CL分流方案，覆盖该企业的小区范围为900001~900003，PRA ID为8388616。

脚本

//AMF配置PRA区域（校园）基本信息。

```
ADD NGPRA: PRAID=8388616, DESC="for EastLake Campus";
```

//AMF在指定的PRA区域配置位置信息。

```
ADD NGPRAMEM: PRAID=8388616, LOCTYPE=NRCELLID, MCC="123", MNC="45", NGCIDBEGIN="000900001", NGCIDEND="000900003";
```

//配置感兴趣区域订阅失败后的处理策略。

```
SET NGMMFUNC: SUBLOCRPTSW=YES;
```

//设置NGAP兼容性参数。

```
SET NGAPCMPT: AOILIST=NO;
```

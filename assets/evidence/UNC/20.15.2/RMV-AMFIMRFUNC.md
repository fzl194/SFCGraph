# 删除用户消息统计功能配置（RMV AMFIMRFUNC）

- [命令功能](#ZH-CN_MMLREF_0244007528__1.3.1)
- [注意事项](#ZH-CN_MMLREF_0244007528__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0244007528__1.3.4)
- [使用实例](#ZH-CN_MMLREF_0244007528__1.3.5)

## [命令功能](#ZH-CN_MMLREF_0244007528)

**适用NF：AMF**

该命令用于删除用户消息统计功能配置。

## [注意事项](#ZH-CN_MMLREF_0244007528)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0244007528)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0244007528)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SUBRANGE | 用户范围 | 可选必选说明：必选参数<br>参数含义：该参数用于指定用户范围。<br>数据来源：本端规划<br>取值范围：<br>- “ALL_USER（所有用户）”：所有用户<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0244007528)

删除“用户范围”是“所有用户”的记录：

```
RMV AMFIMRFUNC:SUBRANGE=ALL_USER;
```

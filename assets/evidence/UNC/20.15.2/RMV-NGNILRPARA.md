# 删除NI-LR功能参数（RMV NGNILRPARA）

- [命令功能](#ZH-CN_MMLREF_0244007658__1.3.1)
- [注意事项](#ZH-CN_MMLREF_0244007658__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0244007658__1.3.4)

## [命令功能](#ZH-CN_MMLREF_0244007658)

**适用NF：AMF**

该命令用于基于运营商删除NI-LR功能的参数。

## [注意事项](#ZH-CN_MMLREF_0244007658)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0244007658)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0244007658)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| NOID | 运营商标识 | 可选必选说明：必选参数<br>参数含义：该参数用于在UNC系统内唯一标识移动网络运营商。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围为0。<br>默认值：无<br>配置原则：<br>该参数取值必须和ADD NGMNO中配置的“NOID”参数取值相同。 |

## [使用实例](#ZH-CN_MMLREF_0244007658)

删除NOID为0的运营商的NI-LR功能参数，执行如下命令:

```
RMV NGNILRPARA:NOID=0;
```

# 删除运营商紧急呼叫会话功能配置（RMV SMFEMGCFG）

## [命令功能](#ZH-CN_MMLREF_0000001091284506)

**适用NF：SMF**

该命令用于删除指定的MNO或MVNO对应的紧急呼叫会话功能配置数据。

## [注意事项](#ZH-CN_MMLREF_0000001091284506)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0000001091284506)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001091284506)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| NOID | 运营商标识 | 可选必选说明：必选参数<br>参数含义：该参数用于指定运营商标识。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围为0。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001091284506)

删除“运营商标识”为“0”的运营商紧急呼叫会话功能的配置，执行如下命令：

```
RMV SMFEMGCFG:NOID=0;
```

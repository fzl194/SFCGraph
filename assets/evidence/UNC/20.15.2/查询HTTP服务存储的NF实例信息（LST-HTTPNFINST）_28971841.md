## [命令功能](#ZH-CN_MMLREF_0000001228971841)

该命令用于查询HTTP服务存储的NF实例信息。

## [注意事项](#ZH-CN_MMLREF_0000001228971841)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001228971841)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001228971841)

无

## [使用实例](#ZH-CN_MMLREF_0000001228971841)

维护人员需要查询HTTP服务存储的NF实例信息时，可使用如下命令：

```
LST HTTPNFINST:;
%%LST HTTPNFINST:;%%
RETCODE = 0  操作成功

结果如下
------------------------
NF类型       NF实例名称        NF实例标识

NFTypeNRF    NRF_Instance_0    00000000-0000-0000-0000-000000000013
NFTypeAMF    AMF_Instance_0    00000000-0000-0000-0000-000000000011
NFTypeSMF    SMF_Instance_0    00000000-0000-0000-0000-000000000012
NFTypeNSSF   NSSF_Instance_0   00000000-0000-0000-0000-000000000014
(结果个数 = 4)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001228971841)

| 输出项名称 | 输出项解释 |
| --- | --- |
| NF类型 | 本参数用于指定NF类型。 |
| NF实例名称 | 本参数用于指定NF实例名称。 |
| NF实例标识 | 本参数用于指定NF实例标识。 |

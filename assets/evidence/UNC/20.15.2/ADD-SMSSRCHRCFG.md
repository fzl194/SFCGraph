# 增加SMS小范围CHR上报规则配置（ADD SMSSRCHRCFG）

## [命令功能](#ZH-CN_MMLREF_0000001404041265)

**适用NF：SMSF**

该命令用于增加SMS小范围CHR上报规则配置。

## [注意事项](#ZH-CN_MMLREF_0000001404041265)

- 该命令执行后立即生效。

- 最多可输入1000000条记录。

#### [操作用户权限](#ZH-CN_MMLREF_0000001404041265)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001404041265)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SUBRANGE | 用户范围 | 可选必选说明：必选参数<br>参数含义：该参数用于指定配置小范围CHR的用户范围。<br>数据来源：本端规划<br>取值范围：<br>- “SPECIFIC_IMSI（指定用户IMSI）”：表示指定用户IMSI。<br>默认值：无<br>配置原则：无 |
| IMSIPRE | IMSI前缀 | 可选必选说明：该参数在"SUBRANGE"配置为"SPECIFIC_IMSI"时为条件必选参数。<br>参数含义：该参数用于指定IMSI前缀。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是5~15。只允许输入十进制数字（0-9）。<br>默认值：无<br>配置原则：无 |
| TMPLIDX | 流程控制模板索引 | 可选必选说明：必选参数<br>参数含义：该参数用于指定CHR上报流程控制模板索引。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~127。<br>默认值：无<br>配置原则：<br>TMPLIDX参数依赖于ADD SMSCHRPRCTMPL命令的TMPLIDX参数。 |

## [使用实例](#ZH-CN_MMLREF_0000001404041265)

运营商希望增加“用户范围”为“指定用户IMSI”，“IMSI前缀”为“12303120010”，“流程控制模板索引”为“1”的SMS小范围CHR上报规则配置，执行如下命令：

```
ADD SMSSRCHRCFG: SUBRANGE=SPECIFIC_IMSI, IMSIPRE="12303120010", TMPLIDX=1;
```

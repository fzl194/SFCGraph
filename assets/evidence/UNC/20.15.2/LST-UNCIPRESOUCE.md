## [命令功能](#ZH-CN_MMLREF_0000001517146834)

**适用NF：SGSN、MME、SGW-C、PGW-C、AMF、SMF、NRF、NSSF、GGSN、SMSF、NCG**

该命令用于查询已经上报至网管北向的UNC网元业务和管理接口的IP地址。

## [注意事项](#ZH-CN_MMLREF_0000001517146834)

当UNC各接口不存在有效IP地址信息时，将生成IP地址为0.0.0.0的无效默认记录，这些默认记录在查询命令结果中不显示。

#### [操作用户权限](#ZH-CN_MMLREF_0000001517146834)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001517146834)

无

## [使用实例](#ZH-CN_MMLREF_0000001517146834)

查询已经上报至网管北向的UNC网元业务和管理接口的IP地址，执行以下命令:

```
%%LST UNCIPRESOUCE:;%%
RETCODE = 0  操作成功

结果如下
------------------------
接口类型  IP地址类型  本端IP地址               

N4              IPv4             10.1.3.11                      
N4              IPv6             2001:0DB8::800:200C:510A
(结果个数 = 2)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001517146834)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 接口类型 | 该参数用于指定UNC网元的接口名称。其中，Mgt为网元和网管的对接接口；DnsQry为SGSN、MME、AMF网元的DNS本端实体与DNS服务器进行通信的接口；Bx为NCG网元与计费账务域交互的文件接口；其他为协议定义的接口。 |
| IP地址类型 | 该参数用于指定IP地址的类型。 |
| 本端IP地址 | 该参数用于表示对应网元接口的本端IP地址。 |

## [命令功能](#ZH-CN_MMLREF_0209651736)

**适用NF：GGSN**

该命令用来查询整机RAT类型的优先级。

## [注意事项](#ZH-CN_MMLREF_0209651736)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209651736)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209651736)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| NODETYPE | 网元类型 | 可选必选说明：可选参数<br>参数含义：该参数表示当前配置的网元类型。<br>数据来源：本端规划<br>取值范围：<br>- GGSN（GGSN）<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209651736)

查询整机RAT类型的优先级：

```
%%LST SRVNODERATPRI:;%%
RETCODE = 0  操作成功

结果如下
--------
           网元类型  =  GGSN
RAT类型的第一优先级  =  RAT
RAT类型的第二优先级  =  SGSNIP
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209651736)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 网元类型 | 该参数表示当前配置的网元类型。 |
| RAT类型的第一优先级 | 该参数用于指定获取RAT类型的两种方式的第一优先级。 |
| RAT类型的第二优先级 | 该参数用于指定获取RAT类型的两种方式的第二优先级。 |

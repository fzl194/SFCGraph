# 调测NSA用户QoS管理

- [操作场景](#ZH-CN_OPI_0227675787__1.3.1)
- [必备事项](#ZH-CN_OPI_0227675787__1.3.2)
- [操作步骤](#ZH-CN_OPI_0227675787__1.3.3)

## [操作场景](#ZH-CN_OPI_0227675787)

当运营商部署NSA 用户QoS管理功能时，需对该功能进行调测，确保功能可用。

> **说明**
> 适用于SGW-C、PGW-C。

## [必备事项](#ZH-CN_OPI_0227675787)

前提条件

- 请仔细阅读[WSFD-011521 NSA 用户QoS管理特性概述](特性概述_27675785.md)。
- 完成 [激活NSA用户QoS管理](激活NSA用户QoS管理_27675786.md) 。

数据

该操作无需准备数据。

工具

- 测试终端
- OM Portal跟踪

## [操作步骤](#ZH-CN_OPI_0227675787)

在相关网元均完成本特性的配置后，可以采用以下步骤检查特性工作是否正常：

1. 用户附着到MME，激活用户，并跟踪该用户消息。
2. 检查消息跟踪中Gx接口的Credit Control Request-Initial，Credit Control Answer-Initial消息中是否有Extended-APN-AMBR-DL、Extended-APN-AMBR-UL、Extended-GBR-DL、Extended-GBR-UL、Extended-Max-Requested-BW-DL、Extended-Max-Requested-BW-UL和Supported-feature-list-id2信元。
    - 如果有，则调测结束。
    - 如果没有，请执行[步骤3](#ZH-CN_OPI_0227675787__substep424463515237)。
3. 进入 “MML命令行-UNC” 窗口。 执行 [**LST OFCCDRPARA**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/计费管理/离线计费/离线计费基础参数/离线公共参数/显示离线计费话单参数（LST OFCCDRPARA）_09896906.md) ，查看PGW-CDR/SGW-CDR容器携带扩展Qos参数的配置是否使能。
  ```
  LST OFCCDRPARA:;
  ```
  ```
  RETCODE = 0  操作成功。

  离线话单参数配置
  ----------------
                                                ......
        PGW-CDR话单List of ServiceData携带Serving PLMN Rate Control  =  不使能
               PGW-CDR话单的List of ServiceData携带APN Rate Control  =  不使能
   SGW-CDR话单的List of Traffic Volumes携带ServingPLMN Rate Control  =  不使能
  (结果个数 = 1)
  ---    END
  ```
    - 如果“PGW-CDR话单List of Service Data携带Serving PLMN Rate Control”、“PGW-CDR话单的List of Service Data携带APN Rate Control”和“SGW-CDR话单的List of Traffic Volumes携带Serving PLMN Rate Control”取值均为“使能”，请执行 [步骤 4](#ZH-CN_OPI_0227675787__substep728215413284) 。
    - 如果“PGW-CDR话单List of Service Data携带Serving PLMN Rate Control”、“PGW-CDR话单的List of Service Data携带APN Rate Control”和“SGW-CDR话单的List of Traffic Volumes携带Serving PLMN Rate Control”取值为“不使能”，请重新配置。
4. 进入 “MML命令行-UNC” 窗口。 执行 [**LST PCRF**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/PCC管理/PCRF Diameter连接/PCRF/查询PCRF（LST PCRF）_09897104.md) 、 [**LST GLBDIAMREALM**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/计费和策略接口管理/Diameter管理/Diameter Realm/全局Realm/查询全局Diameter域（LST GLBDIAMREALM）_09897283.md) 、 [**LST REALMBINDAPN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/计费和策略接口管理/Diameter管理/Diameter Realm/Realm绑定APN/查询APN与Diameter Realm关联关系（LST REALMBINDAPN）_09897288.md) 查看Feature列表是否为RELEASE8。
  ```
  LST PCRF:HOSTNAME="pcrf1";
  ```
  ```
  RETCODE = 0 操作成功。

  PCRF信息
  --------
                      PCRF主机名  = pcrf1

                         VPN实例  = vpn1

  ```
  ```
  LST GLBDIAMREALM: APPLICATION=GX;
  ```
  ```

     根据IMSI构造归属地Realm开关  = 不使能
                    Diameter域名  = pcrf.huawei.com
  Supported-Features动态协商开关  =  使能
                     Feature列表  =  扩展带宽特性&3GPPRel-8 Gx功能
  (结果个数 = 1)

  ```
  ```
  LST REALMBINDAPN: APN="isp";
  ```
  ```

                         APN名称  = isp

  ```
    - 是，请执行 [步骤5](#ZH-CN_OPI_0227675787__cmd745473011425) 。
    - 否，请重新配置。
5. 收集信息并寻求技术支持。
    - 执行**[EXP MML](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/配置管理/配置导出管理/导出MML文件（EXP MML）_47200033.md)**命令将当前配置数据导出为MML脚本文件并保存。
    - 收集并保存上述所有查询信息。
    - 收集归纳所有信息并联系华为技术支持解决。

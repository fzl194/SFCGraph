## [操作场景](#ZH-CN_OPI_0250287962)

本操作指导介绍配置 UNC 和CHR服务器的对接，以及激活CHR功能的操作过程。

- 在激活CHR功能之前，当已实例化的UNC不带UCF服务时，需要上线UCF，具体流程参见《UNC软件安装》手册>安装UCF服务。
- 完成UNC上IP路由数据的配置，具体流程参见《UNC初始配置与调测》手册>配置VNF侧IP路由数据。
- (可选)安装部署IPsec，建议部署IPSec保证传输通道安全、数据安全。
- 配置UCF与CHR服务器对接数据。
- 开启CHR功能，配置CHR服务器类型以及发送策略。操作过程

## [操作步骤](#ZH-CN_OPI_0250287962)

准备阶段：动态上线UCF，以及配置UNC和CHR服务器IP路由数据。

**非SDN组网** ：

1. 动态上线UCF，具体流程参见《UNC软件安装》手册>基于VNF LCM安装UNC（非SDN）>[安装UCF服务](../../../../../软件安装/基于NFVO安装UNC（SDN）/安装可选服务/安装UCF服务_67742476.md)或《UNC软件安装》手册>基于NFVO安装UNC（非SDN）>[安装UCF服务](../../../../../软件安装/基于NFVO安装UNC（SDN）/安装可选服务/安装UCF服务_67742476.md)。
2. 完成UNC上IP路由数据的配置，具体流程参见[《UNC初始配置与调测》手册](../../../../../初始配置/UNC初始配置与调测/组网路由配置/配置VNF侧IP路由数据（非SDN）_75096860.md)>配置VNF侧IP路由数据（非SDN）。

**SDN组网** ：

1. 动态上线UCF，具体流程参见《UNC软件安装》手册>基于NFVO安装UNC(SDN组网)>[安装UCF服务](../../../../../软件安装/基于NFVO安装UNC（SDN）/安装可选服务/安装UCF服务_67742476.md)。
2. 完成UNC上IP路由数据的配置，具体流程参见[《UNC初始配置与调测》手册](../../../../../初始配置/UNC初始配置与调测/组网路由配置/配置VNF侧IP路由数据(SDN)_45606309.md)>配置VNF侧IP路由数据(SDN)。

步骤一：（可选）安装部署IPsec

## [任务示例](#ZH-CN_OPI_0250287962)

任务描述

步骤一：配置UCF和CHR服务器对接数据。

步骤二：激活CHR功能，配置CHR服务器类型以及发送策略。

- 打开CHR功能License，并设置CHR服务器。
  > **说明**
  > CHR License仅控制整系统的正常流程CHR单据。
- 设置具体流程上报CHR单据。

步骤三：（可选，适用于MME/SGSN）配置CHR存储策略。

步骤四：（可选）配置小范围CHR上报策略。

脚本

**此处以AMF网元为例。**

步骤一：配置UCF和CHR服务器对接数据。

//配置UCF报表本地IP资源池。

```
ADD UCFIP: PROTOCOLTYPE=TCP, IPTYPE=IPv4, IPV4="10.181.23.253", PORTSTART=6000, PORTEND=6031, INDEX=1;
```

> **说明**
> 请确保配置ADD UCFIP的IP地址不与LST SERVICEIP和LST SERVICEIPMCR的IP地址重复，否则会导致链路概率性不通。

//配置CHR服务器接入点IP地址等信息。

```
ADD UCFSVRIP: PROTOCOLTYPE=TCP, ACCESSNAME="ACCESS1", UCFSVRNAME="UCFSVR1", IPTYPE=IPv4, IPV4="10.80.128.1", SVRPORT=10500;
```

步骤二：激活CHR功能，配置CHR服务器类型以及发送策略。

//打开CHR功能License。

```
SET LICENSESWITCH: LICITEM="LKV2CHR03", SWITCH=ENABLE;
```

//设置初始注册流程成功和失败时均上报CHR单据，以及Nsmf_PDUSession_UpdateSMContext事件成功和失败时均上报CHR单据。

```
ADD NGACCCHRPRCTMPL: TMPLIDX=10, NGAPCHRSUCCPRC=INITREG-1, NGAPCHRFAILPRC=INITREG-1, NGAPKEYSUCCSIG=UPDSMCTX-1, NGAPKEYFAILSIG=UPDSMCTX-1;
```

//设置CHR服务器，将高性能CHR服务器上报流程控制模板索引修改为的10。

```
SET NGACCCHRCFG: HISRVPRCTMPLIDX=10;
```

//根据现网实际规划配置DWORD1 BIT13，控制AMF CHR采用不同版本单据进行上报。

```
SET AMFSOFTPARAOFBIT:DT=Dw,DWORDNUM=1,VALUE=VALUE_0,POSITION=13;
```

> **说明**
> 请根据protoid的取值来配置DWORD1 BIT13的取值，若有疑问，请联系华为技术支持。
>
> 0：采用protoid大于100的chr proto，代表新版本（单据版本0x01）。
>
> 1：采用protoid小于100的chr proto，代表旧版本（单据版本0xFE）。

步骤三：（可选）配置小范围CHR上报策略。

//配置小范围CHR的过滤条件为IMSI为“12303000100”的用户，流程控制模板索引为0。

```
ADD NGSRCHRCFG: SUBRANGE=SPECIFIC_IMSI, IMSIPRE="12303000100", TMPLIDX=0;
```

//打开小范围CHR单据上报开关，并配置小范围CHR保存到OMU。

```
SET NGCHRSTORCFG: CHRSPESUBSAVE=ENABLE, CHRSAVEPATH=SAVE_OMU;
```

**此处以SGSN/MME网元为例。**

步骤一：配置UCF和CHR服务器对接数据。

//配置UCF报表本地IP资源池。

```
ADD UCFIP: PROTOCOLTYPE=TCP, IPTYPE=IPv4, IPV4="10.181.23.253", PORTSTART=6000, PORTEND=6031, INDEX=1;
```

> **说明**
> 请确保配置ADD UCFIP的IP地址不与LST SERVICEIP和LST SERVICEIPMCR的IP地址重复，否则会导致链路概率性不通。

//配置CHR服务器接入点IP地址等信息。

```
ADD UCFSVRIP: PROTOCOLTYPE=TCP, ACCESSNAME="ACCESS1", UCFSVRNAME="UCFSVR1", IPTYPE=IPv4, IPV4="10.80.128.1", SVRPORT=10500;
```

步骤二：激活CHR功能，配置CHR服务器类型以及发送策略。

//打开CHR功能License。

```
SET LICENSESWITCH: LICITEM="LKV2CHR03", SWITCH=ENABLE;
```

//设置Gb/Iu/S1模式下ATTACH流程成功时上报CHR，Gb/Iu模式下ATTACH和RAU流程失败时上报CHR，S1模式下ATTACH和TAU流程失败时上报CHR，CHR单据设备号填充方式为SGSN_NUMBER。

```
SET CHRCFG: GERANSUCC=ATTACH-1, GERANFAIL=ATTACH-1&RAU-1,UTRANSUCC=ATTACH-1, UTRANFAIL=ATTACH-1&RAU-1, EPSSUCC=ATTACH-1, EPSFAIL=ATTACH-1&TAU-1, RPTDEVNO=SGSN_NUMBER;  
```

//（可选）勾选上报至低性能CloudUDN的业务。

```
 SET CHRLOWPERFSVRSUB: GERANSUCC=ATTACH-1, GERANFAIL=ATTACH-1&RAU-1,UTRANSUCC=ATTACH-1;  
```

//（可选）设置CHR传输策略为自定义传输方式，对未获取到签约数据的CHR单据也传输。

```
SET CHRSNDPLCYCFG: CHRSNDPLCYSELECT=CUSTOMIZE, NOSUBSWITCH=ON;  
```

//（可选）增加根据签约APNNI的匹配策略。

```
 ADD CHRSNDPLCY: TYPE=APNNI, APNNI="huawei.com";   
```

步骤三：（可选）配置CHR存储策略。

//配置CHR本地存储开关（请根据现网实际情况来选择CHR本地存储开关）

```
SET CHRSTORECFG:STOREALLFLAG=ON,STOREALLTYPE=STORE_AND_SEND;
```

> **说明**
> 如果需要转发第三方服务器，需要将“存储类型”设置为“STORE_AND_SEND(存储和转发)”。

步骤四：（可选）配置小范围CHR上报策略。

//增加指定签约APNNI和QCI的小范围CHR生成规则，指定签约APNNI为“Huawei.com”，QCI为“2”；

```
ADD SRANGECHRCFG: SUBRANGE=E_SPECIAL_APNQCI, APNNI="Huawei.com", QCI=2;
```

# 查询默认空闲上下文定时器配置（LST DFTIDLETIME）

- [注意事项](#ZH-CN_MMLREF_0209653130__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0209653130__1.3.4)

## [命令功能](#ZH-CN_MMLREF_0209653130)

**适用NF：SGW-C、PGW-C、SMF、GGSN**

该命令用于查询默认空闲上下文定时器开关和时长配置。

## [注意事项](#ZH-CN_MMLREF_0209653130)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209653130)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209653130)

无

## [使用实例](#ZH-CN_MMLREF_0209653130)

查询系统配置的默认空闲上下文定时器开关和时长参数：

```

RETCODE = 0  操作成功

         I-SMF/V-SMF空闲上下文核查开关  =  使能
```

## [输出结果说明](#ZH-CN_MMLREF_0209653130)

| 输出项名称 | 输出项解释 |
| --- | --- |
| SGW-C空闲上下文核查开关 | 该参数用于指定SGW-C空闲上下文的使能状态。 |
| PGW-C和SGW-C/PGW-C网元的上下文核查开关 | 该参数用于指定PGW-C或者SGW-C/PGW-C合一部署时空闲上下文的使能状态。 |
| GGSN空闲上下文核查开关 | 该参数用于指定GGSN空闲上下文的使能状态。 |
| H-SMF空闲上下文核查开关 | 该参数用于指定H-SMF空闲上下文的使能状态。 |
| H-SMF不活动上下文核查开关 | 该参数用于指定H-SMF不活动上下文的使能状态。 |
| I-SMF/V-SMF空闲上下文核查开关 | 该参数用于指定I-SMF/V-SMF空闲上下文的使能状态。 |
| I-SMF/V-SMF不活动上下文核查开关 | 该参数用于指定I-SMF/V-SMF不活动上下文的使能状态。 |
| GUL承载级别参数 | 该参数用于指定定时器级别，仅对2G、3G、4G用户生效。 |
| 缺省承载和默认GBR的定时器 | 该参数用于指定缺省承载和默认GBR的空闲上下文定时器时长，仅对4G用户生效。 |
| 承载定时器时长(min) | 该参数用于指定承载级空闲定时器时长，仅对2G、3G、4G用户生效。 |
| 会话定时器时长(min) | 该参数用于指定会话级空闲定时器时长，仅对2G、3G、4G用户生效。 |
| SMF会话空闲定时器时长(min) | 该参数用于指定SMF的PDU会话的空闲定时器时长，仅对5G用户生效。 |
| SMF会话不活动定时器时长(min) | 该参数用于指定SMF的PDU会话的不活动定时器时长，仅对5G用户生效。 |
| 空闲超时发送更新消息 | 该参数用于指定当用户持续没有流量的时长超过配置时长后是否发送更新消息，仅对2G、3G、4G用户生效。 |
| H-SMF空闲上下文核查级别 | 该参数用于指定H-SMF空闲上下文核查级别，仅对5G用户生效。 |
| 专有QoS Flow空闲定时器时长(min) | 该参数用于指定SMF的专有QoS Flow空闲定时器时长，仅对5G用户生效。 |
| 缺省QoS Flow空闲定时器时长(min) | 该参数用于指示SMF的缺省QoS Flow空闲定时器时长，仅对5G用户生效。当该参数设置为0时，表示不开启缺省QoS Flow空闲上下文核查。 |
| GGSN一次激活上下文定时器 | 该参数用于指定GGSN一次激活上下文空闲定时器时长。 |
| Proxy-SMF S8空闲上下文核查开关 | 该参数用于指定Proxy-SMF S8空闲上下文的使能状态。 |
| Proxy-SMF 空闲上下文检查开关 | 该参数用于指定Proxy-SMF 空闲上下文的使能状态。 |

#### [命令功能](#ZH-CN_CONCEPT_0000001600841105)

该命令用来在接口应用DS（Differ-Server）域，定义一个DS域。DS域中描述了报文优先级与服务等级和颜色之间的映射关系。

#### [注意事项](#ZH-CN_CONCEPT_0000001600841105)

- 该命令执行后立即生效。
- 该命令最大记录数为8。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600841105)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600841105)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| DSNAME | DS域名 | 可选必选说明：必选参数<br>参数含义：该参数用于指定DS域的名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。<br>默认值：无<br>配置原则：区分大小写，不支持空格。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600841105)

创建一个新的DS域d1：

```
ADD QOSDIFFERSERV:DSNAME="d1";
```

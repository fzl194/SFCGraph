## [命令功能](#ZH-CN_MMLREF_0212701672)

**适用NF：SMF、PGW-C、SGW-C、GGSN**

该命令用于查询UPC DS粒度N4请求等待超时异常的告警阈值。

## [注意事项](#ZH-CN_MMLREF_0212701672)

当前版本不支持此命令。

#### [操作用户权限](#ZH-CN_MMLREF_0212701672)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0212701672)

无

## [使用实例](#ZH-CN_MMLREF_0212701672)

查询UPC DS粒度N4请求等待超时异常的告警阈值：

```
%%LST UPCALMVAL:;%%
RETCODE = 0  操作成功

结果如下
--------
UPC告警上报阈值(秒)  =  10
(结果个数 = 1)

  -----    END
```

## [输出结果说明](#ZH-CN_MMLREF_0212701672)

| 输出项名称 | 输出项解释 |
| --- | --- |
| UPC告警上报阈值(秒) | 该参数用于UPC告警上报阈值。 |

## [操作步骤](#ZH-CN_OPI_0193971023)

1. 进入 “MML命令行-UNC” 窗口。
2. 配置NSA用户选择用户面策略。
    a. 设置会话管理扩展功能。
      [**SET SMFUNC**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/会话管理/SM扩展功能管理/设置会话管理扩展功能(SET SMFUNC)_26145684.md)
    b. **可选：**设置DNS运行参数。
      [**SET DNS**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/DNS/DNS运行参数/设置DNS运行参数(SET DNS)_72345495.md)
    c. 增加GTP-C V2协议兼容性配置。
      [**ADD GTPCV2CMPT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/GTP-C协议管理/GTP-C V2协议兼容性/增加GTP-C V2协议兼容性(ADD GTPCV2CMPT)_26305734.md)
3. **可选：**配置支持DCNR的高速网关的本地Hostfile记录。如果运营商在DNS Server上已配置该对应关系，则不需要配置本步骤。
    a. 增加IPV4 DNS Hostfile的A解析记录。
      [**ADD IPV4DNSH**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/DNS/DNS Hostfile管理/增加IPV4 DNS Hostfile记录(ADD IPV4DNSH)_26145884.md)
    b. 增加DNS Hostfile的NAPTR解析记录。
      [**ADD DNSN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/DNS/DNS NAPTR管理/增加DNS NAPTR记录(ADD DNSN)_72225569.md)
4. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)

## [任务示例](#ZH-CN_OPI_0193971023)

任务描述

完成支持DCNR能力的终端用户网关选择配置，同时添加对应的定制NAPTR/A记录，在需要为用户选择网关时为NSA用户选择到支持DCNR的高速S-GW、P-GW，以匹配高速用户的业务。

脚本

//设置会话管理扩展功能。

```
SET SMFUNC: LTENSAUPSELPLCY=UECAP_ARD;
```

//可选：设置DNS运行参数。

```
SET DNS: NONNSASEL=NO;
```

//增加GTP-C V2协议兼容性配置。

```
ADD GTPCV2CMPT: IDX=1, MSGCLS=TM, TMMSGTYPE=CRT_SES_REQ, CRTSESREQ=UPSELIND, UPSELFLAG=DCNR-1;
```

//增加 UNC 用于选择高速的S-GW/P-GW的A解析记录。

```
ADD IPV4DNSH: HSINDEX=1, HOSTNAME="TOPON.S11.SGW.GW1.NODES.EPC.MNC003.MCC460.3GPPNETWORK.ORG", ADDRSECTION=SECTION1, IPV4ADDR1="10.2.3.10", PRIORITY1=10, WEIGHT1=100; 
ADD IPV4DNSH: HSINDEX=2, HOSTNAME="TOPON.S5.PGW.GW1.NODES.EPC.MNC003.MCC460.3GPPNETWORK.ORG", ADDRSECTION=SECTION1, IPV4ADDR1="10.2.3.10", PRIORITY1=10, WEIGHT1=100;
```

//增加 UNC 用于选择高速的S-GW/P-GW的NAPTR解析记录。

```
ADD DNSN: FQDN="TAC-LB01.TAC-HB11.TAC.EPC.MNC003.MCC460.3GPPNETWORK.ORG", HSINDEX=1, ENTITY=SGW, INTYPE=S11, DCNR=YES, PRIORITY=10, WEIGHT=101; 
ADD DNSN: FQDN="HUAWEI1.COM.APN.EPC.MNC003.MCC460.3GPPNETWORK.ORG", HSINDEX=2, ENTITY=PGW, INTYPE=S5, DCNR=YES, PRIORITY=10, WEIGHT=101;
```

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2UNCUP01", SWITCH=ENABLE;
```

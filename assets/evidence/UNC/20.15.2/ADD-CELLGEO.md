#### [命令功能](#ZH-CN_MMLREF_0000001126305598)

**适用网元：SGSN**

此命令用于增加小区标识CID与地理位置经纬度坐标的对应关系。SGSN利用这个对应关系完成CID与地理位置经纬度坐标之间的转换，提供给LCS MT/NI流程使用。

#### [注意事项](#ZH-CN_MMLREF_0000001126305598)

- 此命令执行后立即生效。
- 此命令的最大记录数是50,000。
- 该命令只针对2.5G的LCS业务，3G业务中，服务区与地理位置经纬度坐标的对应关系在[**ADD SAIGEO**](../SAI和地理坐标对照表/增加服务区标识与地理坐标对照关系(ADD SAIGEO)_72345391.md)中设置。

#### [本地用户权限](#ZH-CN_MMLREF_0000001126305598)

manage-ug；system-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126305598)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126305598)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| CID | 小区标识 | 可选必选说明：必选参数<br>参数含义：该参数用于指定小区的标识，CID = MCC + MNC + LAC + RAC + CI。<br>数据来源：整网规划<br>取值范围：15~16位十六进制数<br>默认值：无 |
| LAT | 纬度类型 | 可选必选说明：必选参数<br>参数含义：该参数用于指定位置坐标的纬度类型。<br>数据来源：整网规划<br>取值范围：<br>- “NLA(北纬)”<br>- “SLA(南纬)”<br>默认值：无<br>说明：SOUTH_LATITUDE，南纬；NOUTH_LATITUDE，北纬。 |
| LAD | 纬度（度） | 可选必选说明：必选参数<br>参数含义：该参数用于指定位置坐标的纬度的度值。<br>数据来源：整网规划<br>取值范围：0度～90度<br>默认值：无 |
| LAM | 纬度（分） | 可选必选说明：必选参数<br>参数含义：该参数用于指定位置坐标的纬度的分的值。<br>数据来源：整网规划<br>取值范围：0分～59分<br>默认值：无<br>说明：当LAD取值为90度时，LAM只能为0分。 |
| LAS | 纬度（秒） | 可选必选说明：必选参数<br>参数含义：该参数用于指定位置坐标的纬度的秒的值。<br>数据来源：整网规划<br>取值范围：0秒～59秒<br>默认值：无<br>说明：当LAD取值为90度时，LAS只能为0秒。 |
| LOT | 经度类型 | 可选必选说明：必选参数<br>参数含义：该参数用于指定位置坐标的经度类型。<br>数据来源：整网规划<br>取值范围：<br>- “ELO(东经)”<br>- “WLO(西经)”<br>默认值：无<br>说明：EAST_LONGITUDE，东经；WEST_LONGITUDE，西经。 |
| LOD | 经度（度） | 可选必选说明：必选参数<br>参数含义：该参数用于指定位置坐标的经度的度值。<br>数据来源：整网规划<br>取值范围：0度～180度<br>默认值：无 |
| LOM | 经度（分） | 可选必选说明：必选参数<br>参数含义：该参数用于指定位置坐标的经度的分的值。<br>数据来源：整网规划<br>取值范围：0分～59分<br>默认值：无<br>说明：当LOD取值为180度时，LOM只能为0分。 |
| LOS | 经度（秒） | 可选必选说明：必选参数<br>参数含义：该参数用于指定位置坐标的经度的秒的值。<br>数据来源：整网规划<br>取值范围：0秒～59秒<br>默认值：无<br>说明：当LOD取值为180度时，LOS只能为0秒。 |
| RAD | 小区半径（m） | 可选必选说明：必选参数<br>参数含义：该参数用于指定小区半径的值，就是该小区覆盖的半径。<br>数据来源：整网规划<br>取值范围：0m～1800000m<br>默认值：无<br>说明：在LCS MT/NI流程中当只能获得被定位用户的小区标识时，小区半径就是该用户坐标偏离该小区中心坐标的误差。 |

#### [使用实例](#ZH-CN_MMLREF_0000001126305598)

增加CID为123090755123400与北纬5度15分25秒西经35度45分55秒，小区半径为5500米的地理位置坐标的对应关系：

ADD CELLGEO: CID="123090755123400", LAT=NLA, LAD=5, LAM=15, LAS=25, LOT=WLO, LOD=35, LOM=45, LOS=55, RAD=5500;

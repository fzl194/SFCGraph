## [命令功能](#ZH-CN_MMLREF_0000002080872788)

**适用NF：SMF、PGW-C**

该命令用于查询智能双N7会话特性是否使能。

## [注意事项](#ZH-CN_MMLREF_0000002080872788)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000002080872788)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000002080872788)

无

## [使用实例](#ZH-CN_MMLREF_0000002080872788)

查询智能双N7会话特性是否使能：

```
%%LST IPEN7SESSFUNC:;%%
RETCODE = 0  操作成功

结果如下
--------
智能N7会话特性开关  =  使能
(结果个数 = 1)

---    END
```

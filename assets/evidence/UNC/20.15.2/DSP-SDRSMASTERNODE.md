# 显示HAF向SDRS推送的主节点信息（DSP SDRSMASTERNODE）

- [注意事项](#ZH-CN_MMLREF_0000001105225906__1.3.2)

- [输出结果说明](#ZH-CN_MMLREF_0000001105225906__1.3.6)

## [命令功能](#ZH-CN_MMLREF_0000001105225906)

该命令用于显示HAF向SDRS推送的主节点信息。

## [注意事项](#ZH-CN_MMLREF_0000001105225906)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001105225906)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001105225906)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| CELLID | Cell ID | 可选必选说明：必选参数<br>参数含义：该参数用于指定SDR调试消息发送的CELLID，可以通过使用命令<br>[**DSP MSPROCESS**](../../可靠性管理/微服务可靠性管理/显示微服务进程信息（DSP MSPROCESS）_09587887.md)<br>获取。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~127。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001105225906)

使用如下命令查询指定CELL ID上HAF向SDRS推送的主节点信息：

```
%%DSP DSP SDRSMASTERNODE: CELLID="vup-pod-010-104-1-24__103__0";%%
RETCODE = 0  操作成功

结果如下
--------
Cell ID   = vup-pod-010-104-1-24__103__0
app类型   = 102
主实例ID    = 11879281905165310720 0xa4dbaff4201a9b00
Group Id  = 999
(结果个数 = 1)
---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001105225906)

| 输出项名称 | 输出项解释 |
| --- | --- |
| Cell ID | 该参数用于指定SDR调试消息发送的CELLID，可以通过使用命令<br>[**DSP MSPROCESS**](../../可靠性管理/微服务可靠性管理/显示微服务进程信息（DSP MSPROCESS）_09587887.md)<br>获取。 |
| app类型 | 该参数表示主节点所属的APP类型。 |
| Group Id | 该参数表示主节点所属的Group Id。 |
| 主实例ID | 该参数用于表示主节点的实例ID。 |

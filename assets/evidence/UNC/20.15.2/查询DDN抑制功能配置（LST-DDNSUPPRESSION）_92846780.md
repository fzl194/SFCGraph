## [命令功能](#ZH-CN_MMLREF_0000001192846780)

**适用NF：SGW-C**

该命令用来查询DDN抑制功能配置信息。

## [注意事项](#ZH-CN_MMLREF_0000001192846780)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001192846780)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001192846780)

无

## [使用实例](#ZH-CN_MMLREF_0000001192846780)

查询DDN抑制功能配置：

```
%%LST DDNSUPPRESSION:;%%
RETCODE = 0  操作成功

输出结果如下
------------------------
DDN抑制功能开关  =  不使能
   监控时长(分)  =  5
    DDN失败次数  =  10
   抑制时长(分)  =  5
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001192846780)

| 输出项名称 | 输出项解释 |
| --- | --- |
| DDN抑制功能开关 | 该参数用于控制DDN抑制功能是否开启。 |
| 监控时长(分) | 该参数用于配置DDN抑制的监控时长。 |
| DDN失败次数 | 该参数用于配置DDN失败次数。 |
| 抑制时长(分) | 该参数用于配置DDN抑制的时长。 |

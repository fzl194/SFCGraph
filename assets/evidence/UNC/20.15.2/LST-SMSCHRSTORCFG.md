## [命令功能](#ZH-CN_MMLREF_0000001353321870)

**适用NF：SMSF**

该命令用于查询SMS小范围CHR存储配置。

## [注意事项](#ZH-CN_MMLREF_0000001353321870)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001353321870)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001353321870)

无

## [使用实例](#ZH-CN_MMLREF_0000001353321870)

运营商希望查询SMS小范围CHR存储配置，执行如下命令：

```
LST SMSCHRSTORCFG:;
%%LST SMSCHRSTORCFG:;%%
RETCODE = 0  操作成功

结果如下：
------------------------
小范围CHR使能开关 =  打开
小范围CHR存储路径  =  保存到OMU
       
(结果个数 = 1)
```

## [输出结果说明](#ZH-CN_MMLREF_0000001353321870)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 小范围CHR使能开关 | 该参数用于表示小范围CHR使能开关。 |
| 小范围CHR存储路径 | 该参数用于表示小范围CHR存储路径。 |

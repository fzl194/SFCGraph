## [命令功能](#ZH-CN_MMLREF_0000001276431669)

![](设置故障快速检测配置（SET FAULTCHECKBYFABRIC）_76431669.assets/notice_3.0-zh-cn_2.png)

该命令仅适用于单域模式下的指定网元，使用前需明确该功能是否适用于当前网元，请谨慎使用并联系华为技术支持协助操作。

该命令用于设置基于Fabric链路状态的故障快速检测参数配置。

## [注意事项](#ZH-CN_MMLREF_0000001276431669)

- 该命令执行后立即生效。

- 系统部署完成后，已经存在初始记录，参数的初始记录值如下表：

| ENABLE | PROTECTDURATION | WAITRPTDURATION |
| --- | --- | --- |
| OFF | 10 | 3000 |

#### [操作用户权限](#ZH-CN_MMLREF_0000001276431669)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001276431669)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| ENABLE | 故障快速检测开关 | 可选必选说明：必选参数<br>参数含义：该参数用于设置故障快速检测开关状态。<br>数据来源：本端规划<br>取值范围：<br>- “ON（开启）”：故障快速检测开关打开。<br>- “OFF（关闭）”：故障快速检测开关关闭。<br>默认值：无。<br>配置原则：无 |
| PROTECTDURATION | 防震荡保护时长 (分) | 可选必选说明：该参数在"ENABLE"配置为"ON"时为条件必选参数。<br>参数含义：该参数用于设置防震荡保护时长，即当检测到故障后，在此时长内不再做快速故障判定。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是1~1440，单位是分钟。<br>默认值：10。执行命令并不输入该参数时，该参数保持系统当前配置不变，可通过LST FAULTCHECKBYFABRIC查询当前参数配置值。<br>配置原则：无 |
| WAITRPTDURATION | 等待上报时长 (毫秒) | 可选必选说明：该参数在"ENABLE"配置为"ON"时为条件必选参数。<br>参数含义：该参数用于设置Fabric链路状态收集的时间窗口，仅该时间窗口内的链路状态参与快速故障判定。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是100~10000，单位是毫秒。<br>默认值：3000。执行命令并不输入该参数时，该参数保持系统当前配置不变，可通过LST FAULTCHECKBYFABRIC查询当前参数配置值。<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001276431669)

- 关闭故障快速检测功能。
  ```
  SET FAULTCHECKBYFABRIC: ENABLE=OFF;
  ```
- 打开故障快速检测功能，设置防震荡保护时长为20分钟，故障上报等待时长为4秒。
  ```
  SET FAULTCHECKBYFABRIC: ENABLE=ON, PROTECTDURATION=20, WAITRPTDURATION=4000;
  ```

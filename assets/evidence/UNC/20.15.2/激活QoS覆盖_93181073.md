## [操作场景](#ZH-CN_OPI_0193181073)

本操作指导介绍在运行网络中激活QoS覆盖特性的操作过程。

QoS覆盖是将指定用户签约的QoS速率改写成特定值，在HLR向SGSN插入签约数据时，SGSN根据定义的QoS转换规则把Rel5或者Rel99版本的QoS数据转换为Rel7版本的QoS。

> **说明**
> 适用于 SGSN 。

## [任务示例](#ZH-CN_OPI_0193181073)

任务描述

- 任务1：增加所有用户QoS覆盖记录。增加签约Rel99格式的QoS用户，将Rel99格式的QoS转换为Rel7格式的QoS。签约的最大上行速率和保证上行速率是18Mbit/s，最大下行速率和保证下行速率是48Mbit/s，原始QoS扩展下行最大速率和扩展下行保证速率为0Mbit/s，将以上速率转换为最大上行扩展速率和保证上行扩展速率是118Mbit/s，最大下行扩展速率和保证下行扩展速率是148Mbit/s到Rel7格式的QoS。
- 任务2：增加指定用户QoS覆盖记录。增加IMSI前缀为12300755，签约Rel99格式的QoS用户，将Rel99格式的QoS转换为Rel7格式的QoS。签约的最大上行速率和保证上行速率是18Mbit/s，最大下行速率和保证下行速率是48Mbit/s，原始QoS扩展下行最大速率和扩展下行保证速率为0Mbit/s，将以上速率转换为最大上行扩展速率和保证上行扩展速率是118Mbit/s，最大下行扩展速率和保证下行扩展速率是148Mbit/s到Rel7格式的QoS。

脚本

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2QOSOW02", SWITCH=ENABLE;
```

- 脚本1：
  //增加所有用户QoS覆盖数据。
  ```
  ADD QOSTRANS: SUBRANGE=ALL_USER, MBRULKO=18, GBRULKO=18, MBRDLKO=48, GBRDLKO=48, MBRDLKEXO=0, MGRDLKEXO=0, MBRULKEXT=118, GBRUPLKEXT=118, MBRDLKEXT=148, GBRDLKEXT=148;
  ```
- 脚本2：
  //增加指定用户QoS覆盖数据。
  ```
  ADD QOSTRANS: SUBRANGE=SPECIAL_USER, IMSIPRE="12300755", MBRULKO=18, GBRULKO=18, MBRDLKO=48, GBRDLKO=48, MBRDLKEXO=0, MGRDLKEXO=0, MBRULKEXT=118, GBRUPLKEXT=118, MBRDLKEXT=148, GBRDLKEXT=148;
  ```

## [操作场景](#ZH-CN_OPI_0295923602)

OCS的选择满足以下要求：

1. 基于计费属性CC（Charge Characteristics）区分Postpaid和Prepaid用户。
2. 基于用户号段进行OCS的选择。

> **说明**
> 适用于 GGSN、 PGW-C。

## [操作步骤](#ZH-CN_OPI_0295923602)

1. 配置在线计费中GGSN/PGW-C的设备标识。
  > **说明**
  > UNC 的Gx应用和Gy应用的本端主机名可以相同，但是同时要求 “REALMNAME” 参数和 “PRODUCTNAME” 也必须相同。
  **ADD DIAMLOCINFO**
2. 配置基于用户号段进行OCS负荷分担。
    a. 配置OCS信息。重复执行本步骤可增加多个OCS信息。
      **ADD OCS**
    b. 配置IMSI及MSISDN号码段。
      **ADD IMSIMSISDNSEG**
    c. 配置IMSI及MSISDN号码段组。
      **ADD SUBSCRIBERIDSEGGRP**
    d. 配置OCS组。
      **ADD OCSGROUP**
    e. 配置基于用户号段IMSI/MSISDN选择在线计费系统的OCS。
      **ADD OCSBINDING**
3. 配置主从OCS组。
  **ADD DCCTEMPLATE**
4. 配置虚拟APN。
    a. 使能虚拟APN功能。
      **ADD APN**
    b. 配置虚拟APN映射规则。
      **ADD VIRTUALAPNRULE**
5. 配置真实APN。
    a. 配置真实APN实例。
      **ADD APN**
    b. 绑定DCC模板。
      **SET APNCHARGECTRL**

## [任务示例](#ZH-CN_OPI_0295923602)

任务描述

如 [图1](#ZH-CN_OPI_0295923602__fig193171816181711) 所示，GGSN/PGW-C首先基于CC进行真实APN的映射，进而获得真实APN下绑定的OCS组，在OCS组内基于用户号段进行OCS的选择，即最终实现基于CC+用户号段选择OCS的方案。

- CC（Postpaid）+用户号段（13900100100～13900100199）可选择的OCS：OCS3（主）、OCS1（备）
- CC（Postpaid）+用户号段（13900100200～13900100299）可选择的OCS：OCS4（主）、OCS2（备）
- CC（Prepaid）+用户号段（13900100100～13900100199）可选择的OCS：OCS1（主）、OCS3（备）
- CC（Prepaid）+用户号段（13900100200～13900100299）可选择的OCS：OCS2（主）、OCS4（备）

脚本

1. 配置GGSN/PGW-C设备标识。
  ```
  ADD DIAMLOCINFO: HOSTNAME="pgw1",REALMNAME="huawei.com",PRODUCTNAME="unc@huawei.com";
  ```
2. 配置基于用户号段进行OCS负荷分担。
  //配置OCS设备标识。
  ```
  ADD OCS:OCSHOSTNAME="ocs_host1",REALMNAME="ocs_realm";
  ```
  ```
  ADD OCS:OCSHOSTNAME="ocs_host2",REALMNAME="ocs_realm";
  ```
  ```
  ADD OCS:OCSHOSTNAME="ocs_host3",REALMNAME="ocs_realm";
  ```
  ```
  ADD OCS:OCSHOSTNAME="ocs_host4",REALMNAME="ocs_realm";
  ```
  ```
  ADD DIAMPEERADDR:HOSTNAME="ocs_host1",ADDRTYPE=IPv4,IPV4ADDRESS="192.168.40.10";
  ```
  ```
  ADD DIAMPEERADDR:HOSTNAME="ocs_host2",ADDRTYPE=IPv4,IPV4ADDRESS="192.168.40.11";
  ```
  ```
  ADD DIAMPEERADDR:HOSTNAME="ocs_host3",ADDRTYPE=IPv4,IPV4ADDRESS="192.168.40.12";
  ```
  ```
  ADD DIAMPEERADDR:HOSTNAME="ocs_host4",ADDRTYPE=IPv4,IPV4ADDRESS="192.168.40.13";
  ```
  //配置号段，此处以MSISDN为例，如果需要也可以使用IMSI。
  ```
  ADD IMSIMSISDNSEG:SEGMENTNAME="testmsisdn1",SEGMENTTYPE=MSISDN,SEGSTART="13900100100",SEGEND="13900100199";
  ```
  ```
  ADD SUBSCRIBERIDSEGGRP: SEGGROUPNAME="testmsisdngr1", SEGMENTNAME="testmsisdn1", SEGMENTTYPE=IMSIMSISDN;
  ```
  ```
  ADD IMSIMSISDNSEG:SEGMENTNAME="testmsisdn2",SEGMENTTYPE=MSISDN,SEGSTART="13900100200",SEGEND="13900100299";
  ```
  ```
  ADD SUBSCRIBERIDSEGGRP: SEGGROUPNAME="testmsisdngr2", SEGMENTNAME="testmsisdn2", SEGMENTTYPE=IMSIMSISDN;
  ```
  //基于用户号段将OCS info绑定到OCS group，这样同一OCS group的用户会基于用户号段进行负荷分担。
  ```
  ADD OCSGROUP:OCSGRPNAME="ocs-gra";
  ```
  ```
  ADD OCSGROUP:OCSGRPNAME="ocs-grb";
  ```
  ```
  ADD OCSBINDING:OCSGRPNAME="ocs-gra",OCSHOSTNAME="ocs_host1",SEGGROUPNAME="testmsisdngr1",IMSIPRIORITY=30;
  ```
  ```
  ADD OCSBINDING:OCSGRPNAME="ocs-gra",OCSHOSTNAME="ocs_host2",SEGGROUPNAME="testmsisdngr2",IMSIPRIORITY=40;
  ```
  ```
  ADD OCSBINDING:OCSGRPNAME="ocs-grb",OCSHOSTNAME="ocs_host3",SEGGROUPNAME="testmsisdngr1",IMSIPRIORITY=30;
  ```
  ```
  ADD OCSBINDING:OCSGRPNAME="ocs-grb",OCSHOSTNAME="ocs_host4",SEGGROUPNAME="testmsisdngr2",IMSIPRIORITY=40;
  ```
3. 配置主从OCS组。
  ```
  ADD DCCTEMPLATE:DCCTMPLTNAME="dcctemp01",PRIOCSGRPNAME="ocs-gra",SECOCSGRPNAME="ocs-grb";
  ```
  ```
  ADD DCCTEMPLATE:DCCTMPLTNAME="dcctemp02",PRIOCSGRPNAME="ocs-grb",SECOCSGRPNAME="ocs-gra";
  ```
4. 配置真实APN。
  ```
  ADD APN:APN="apn2",RESTORPGWSWITCH=INHERIT;
  ```
  ```
  SET APNCHARGECTRL:APN="apn2",DCCTEMPLATE="dcctemp01";
  ```
  ```
  ADD APN:APN="apn3",RESTORPGWSWITCH=INHERIT;
  ```
  ```
  SET APNCHARGECTRL:APN="apn3",DCCTEMPLATE="dcctemp02";
  ```
5. 配置虚拟APN。
  ```
  ADD APN:APN="apn1",VIRTUALAPN=ENABLE,RESTORPGWSWITCH=INHERIT;
  ```
  //计费属性为 “PREPAID” 的映射到真实APN apn2，计费属性为 “NORMAL” 的映射到真实APN apn3。
  ```
  ADD VIRTUALAPNRULE:VIRTUALAPN="apn1",MAPPINGTYPE=CHARGE_CHAR,CCMODE=Prepaid,REALAPN="apn2";
  ```
  ```
  ADD VIRTUALAPNRULE:VIRTUALAPN="apn1",MAPPINGTYPE=CHARGE_CHAR,CCMODE=Normal,REALAPN="apn3";
  ```

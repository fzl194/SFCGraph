## [操作场景](#ZH-CN_OPI_0170437186)

当要对使用同一APN接入的用户实施不同的业务策略时，需配置虚拟APN功能。

> **说明**
> 适用于 GGSN、 PGW-C、SMF。

## [必备事项](#ZH-CN_OPI_0170437186)

前提条件

- 请仔细阅读 [WSFD-205102 虚拟APN](../WSFD-205102 虚拟APN_66765163.md)。
- 完成配置APN[**ADD APN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/APN管理/APN/增加APN配置（ADD APN）_09653747.md)或别名APN配置，别名APN参考[别名APN（适用于GGSN/PGW-C/SMF）](../WSFD-106203 别名APN/别名APN（适用于GGSN_PGW-C_SMF）_34797878.md)。
- 完成加载license（如果有需求，请联系华为技术支持工程师处理）。

## [操作步骤](#ZH-CN_OPI_0170437186)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md) :
3. 配置APN，开启虚拟APN功能。
  [**ADD APN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/APN管理/APN/增加APN配置（ADD APN）_09653747.md)
4. （可选）配置APN纠错。
  [**SET ANONYMOUSAPN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/APN管理/匿名APN/设置匿名APN配置（SET ANONYMOUSAPN）_09651574.md)
5. 配置虚拟APN与真实APN的映射规则。
  [**ADD VIRTUALAPNRULE**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/虚拟APN映射管理/虚拟APN映射策略/增加虚拟APN映射策略配置（ADD VIRTUALAPNRULE）_09652380.md)
6. （可选）配置映射规则为缺省APN。
  [**ADD VIRTUALAPNRULE**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/虚拟APN映射管理/虚拟APN映射策略/增加虚拟APN映射策略配置（ADD VIRTUALAPNRULE）_09652380.md)
  > **说明**
  > 开启虚拟APN功能后，如果用户没有匹配到映射规则，则使用缺省APN激活，如果缺省APN没有配置，则用户激活失败。
  >
  > - 当配置虚拟APN映射方式为**PCO**时，还需要通过[**SET DOMAINSEPARATOR**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/接入控制/域名分隔符/设置域名前后缀分隔符（SET DOMAINSEPARATOR）_09654169.md)设置前缀分隔符或后缀分隔符，将从PCO信元中解析出的域名作为真实的APN。

## [任务示例](#ZH-CN_OPI_0170437186)

任务描述

新增一个名为“example.com”的虚拟APN，并添加虚拟APN映射规则，映射方式为MSISDN，将MSISDN以139开头的用户映射到APN“email”。

脚本

1. 开启虚拟APN功能License。
  ```
  SET LICENSESWITCH:LICITEM="LKV3W9VAPN12",SWITCH=ENABLE;
  ```
2. 新增虚拟APN“example.com”，使能Virtual APN功能。
  ```
  ADD APN:APN="example.com",VIRTUALAPN=ENABLE;
  ```
3.
  配置真实APN“email”。

  ```
  ADD APN:APN="email",VIRTUALAPN=DISABLE;
  ```
4. 添加虚拟APN映射规则，映射方式MSISDN，映射值MAPPINGVALUE为“139”，映射的真实APN名称为“email”。
  ```
  ADD VIRTUALAPNRULE:VIRTUALAPN="example.com",MAPPINGTYPE=MSISDN,MAPPINGVALUE="139",REALAPN="email";
  ```

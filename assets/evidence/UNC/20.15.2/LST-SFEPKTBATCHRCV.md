#### [命令功能](#ZH-CN_TOPIC_0000001794196890)

该命令用来查询SFE批量收取的报文数量。

#### [注意事项](#ZH-CN_TOPIC_0000001794196890)

- 该命令执行后立即生效。
- 该命令仅适用于非池化场景。

#### [操作用户权限](#ZH-CN_TOPIC_0000001794196890)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_TOPIC_0000001794196890)

无。

#### [使用实例](#ZH-CN_TOPIC_0000001794196890)

查询SFE批量收取的报文数量：

```
LST SFEPKTBATCHRCV:;
```

```
RETCODE = 0  操作成功

结果如下
------------------------
批量收取报文数量  =  16
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_TOPIC_0000001794196890)

参见SET SFEPKTBATCHRCV的参数说明。

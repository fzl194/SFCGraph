## [命令功能](#ZH-CN_MMLREF_0000001100360098)

该命令用于请求License文件状态信息、控制项信息以及销售项信息。

## [注意事项](#ZH-CN_MMLREF_0000001100360098)

- 当系统中不存在激活的License时，激活License文件、产品名称、产品版本等信息，显示为“NULL”。
- 查询“销售项信息”时，如果销售项在授权截止日期内，则正常显示；如果销售项超过授权截止日期，则不显示。

## [参数说明](#ZH-CN_MMLREF_0000001100360098)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| DSPBBOM | 显示类型 | 可选必选说明：可选参数。<br>参数含义：用于指示系统需要按照哪种类型来显示对应的信息。<br>取值范围：<br>- “CONTROLITEM(控制项信息)”：表示查询控制项的信息。<br>- “SALESITEM(销售项信息)”：表示查询销售项的信息。- 2.0版本的License文件无销售项信息，不支持显示销售项信息。<br>- 3.0版本的License文件是否显示销售项信息，由网元自定义控制。<br>- 该参数中国区暂不支持。<br>默认值：“CONTROLITEM(控制项信息)”。<br>配置原则：无。 |

## [输出结果说明](#ZH-CN_MMLREF_0000001100360098)

该命令执行正常，会返回命令执行成功的提示信息，输出项说明如 [表1](#ZH-CN_MMLREF_0000001100360098__table714812389314) 所示。

该命令执行异常，会返回对应的错误码及错误码解释信息。常见错误码列表如 [表2 错误码列表](#ZH-CN_MMLREF_0000001100360098__table16741512233) 所示。

*表1 输出项解释*

| **输出项名称** | **输出项解释** |
| --- | --- |
| 当前状态 | License运行状态。<br>- 未激活：还未激活License文件，或者License文件/某特征段过了试用天数，License配置调整为缺省默认值，并且发送License限制业务告警。此时，用户需要尽快申请新的License文件。<br>- 试用期：系统试用期默认打开的功能项可用，所有资源项按照License最大缺省值控制。有60天试用期，过期之后所有功能项和资源项都用最小缺省值控制。<br>- 商用已激活/调测已激活：已激活了License文件，处于正常阶段，用授权值限制业务。<br>- 已启动紧急：用最大值限制业务，一个大版本可以开启3次，每次可使用7天，发送License开启紧急告警，以及剩余天数。<br>- 进入宽限期： License文件/某特征段已过期，但是还有60天的宽限期，继续用授权值限制业务，发送License进入宽限期告警，以及剩余天数。 |
| 已激活License文件 | 正在运行的License文件名称。 |
| 激活时间 | 激活License的时间。 |
| 设备序列号 | 该License绑定的硬件设备序列号或唯一的软件设备序列号。 |
| 版本 | License版本号，2.0为dat（DAT）格式，3.0为xml（XML）格式。 |
| 宽限期详情 | 宽限期可用60天，已进入宽限期后，提示剩余天数。 |
| 试用期详情 | 试用期可用60天，已进入试用期后，提示剩余天数。 |
| License失效码 | 根据License文件的序列号生成的失效码，失效码可以用于申请新的License文件。 |
| 失效时间 | License文件的失效时间。 |
| 启动紧急详情 | 一个大版本可以开启3次，每次可使用7天。 |
| 序列号 | License文件的序列号，用于唯一确定一个License文件。 |
| 生成时间 | License文件生成时间。 |
| 产品名称 | 产品软件名称。 |
| 产品版本 | 产品软件版本。 |
| 网元名称 | 网元信息NodeInfo中Node字段。 |
| 顺序号 | 网元信息NodeInfo，标识该License文件是该网元的第几个License文件。当网元License进行升级、扩容、ESN变更、调整、修改用户名等操作时，该顺序号会递增。 |
| 控制项信息 | 使用的License支持的各资源信息。<br>- 2.0版本显示：控制项、控制项描述、当前授权值、授权截止日期。<br>- 3.0版本显示：销售项、销售项描述、 控制项 、控制项描述、当前授权值、授权截止日期。<br>当前授权值：当为默认状态时用默认值控制，已激活或者进入宽限期时用授权值控制，已启动紧急后最大值控制。 |
| 销售项信息 | - 2.0版本：不支持，无销售项信息。<br>- 3.0版本：销售项、销售项描述 、当前控制值 、当前使用值 、使用率、授权截止日期。说明：销售项包含一个或多个控制项，控制项又分为功能项和资源项。如果当前激活的License文件中的某个销售项包含的控制项只有功能项，或者用于统计该销售项使用信息的控制项未使用时，那么该销售项的<br>“当前使用值”<br>和<br>“使用率”<br>显示为<br>“NULL”<br>。 |

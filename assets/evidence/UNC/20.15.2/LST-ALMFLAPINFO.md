#### [命令功能](#ZH-CN_CONCEPT_0259103455)

该命令用于查询告警震荡信息。

#### [注意事项](#ZH-CN_CONCEPT_0259103455)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0259103455)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0259103455)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| AID | 告警ID | 可选必选说明：可选参数<br>参数含义：告警ID。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～4294967295。<br>默认值：无 |
| SERVICEINSTANCE | 服务实例 | 可选必选说明：必选参数<br>参数含义：该参数表示大颗粒服务实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，通过LST VNFC命令获取。<br>默认值：无<br>配置原则：只能填写通过LST VNFC命令查询到的管理代理标识。 |

#### [使用实例](#ZH-CN_CONCEPT_0259103455)

查询告警ID为135602186的告警震荡信息，可通过如下命令查询：

```
LST ALMFLAPINFO:AID=135602186
,SERVICEINSTANCE="ACS"
;
```

```
RETCODE = 0  操作成功

结果如下:
------------------------
                     告警ID  =  135602186
                   告警名称  =  数据库恢复失败
     告警震荡产生周期（秒）  =  400
     告警震荡恢复周期（秒）  =  600
               告警震荡阈值  =  3
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0259103455)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 告警ID | 告警ID。 |
| 告警名称 | 告警名称。 |
| 告警震荡产生周期（秒） | 告警震荡产生周期，单位是秒，产生周期必须大于告警抑制时间。在告警震荡产生周期内，告警上报次数（重传不算）达到告警震荡阈值，则进入告警震荡状态，反之退出检测周期。 |
| 告警震荡恢复周期（秒） | 告警震荡恢复周期，单位是秒，恢复周期必须大于告警抑制时间。告警进入震荡状态，若告警震荡恢复周期内没有告警产生与恢复上报，退出震荡状态，若上报恢复告警，则从上报开始计时，当时间达到告警震荡恢复周期，且期间无告警产生上报，则恢复告警上报终端，退出告警震荡状态，若恢复告警上报之后有告警产生上报，则重新计时，直到计时达到告警震荡恢复周期，告警退出震荡状态。 |
| 告警震荡阈值 | 告警震荡阈值，在告警震荡产生周期内，若上报次数达到震荡阈值（重传不算），告警进入震荡状态。 |

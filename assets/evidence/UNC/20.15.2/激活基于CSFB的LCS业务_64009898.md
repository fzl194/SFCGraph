## [操作步骤](#ZH-CN_OPI_0164009898)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. **可选：**配置SGs和Sv接口合一选择MSC。
  > **说明**
  > 当网络中同时部署VoLTE和CSFB场景下，执行如下步骤，保证IMS语音通话过程中进行LCS业务的场景下，语音和LCS业务的正常。
  >
  > 此种场景下，组网上要求同时支持SGs和Sv接口的MSC需要具备IWF（Interworking Function）。
    a. 查询VLR配置。
      [**LST VLR**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/电路域联合业务/VLR管理/查询VLR配置信息(LST VLR)_72225125.md)
          - “Sv/SGs合一”等于“CSFB触发合一”，操作结束。
          - “Sv/SGs合一”等于“不合一”，执行[步骤 3.b](#ZH-CN_OPI_0164009898__MOD_VLR)。
    b. 修改VLR配置。
      [**MOD VLR**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/电路域联合业务/VLR管理/修改VLR配置信息(MOD VLR)_26145444.md)
    c. 增加SGs接口MSC的A解析记录。
      [**ADD IPV4DNSH**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/DNS/DNS Hostfile管理/增加IPV4 DNS Hostfile记录(ADD IPV4DNSH)_26145884.md)

## [任务示例](#ZH-CN_OPI_0164009898)

任务描述

运营商同时部署了VoLTE和CSFB业务，需要保证在进行VoLTE语音过程中进行LCS业务场景下，语音和LCS业务正常。组网如 [图1 SGs和Sv接口组网示意图](#ZH-CN_OPI_0164009898__fig320922015441) 所示，MSC_1和MSC_2组成pool 1，pool 1覆盖LAI 1和LAI 2。MSC_3组成pool 2，pool 2覆盖LAI 3。

**图1** SGs和Sv接口组网示意图

<br>

![](激活基于CSFB的LCS业务_64009898.assets/zh-cn_image_0170785184_2.png "点击放大")

脚本

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2CSFBEP01", SWITCH=ENABLE;
```

//修改VLR配置。

```
MOD VLR: VN="86139027", SVSGS=CONSISTENT_SELECTION, MSCHOSTNAME="topon.sv.eth1.mnc308.mcc001.org";
MOD VLR: VN="86130027", SVSGS=CONSISTENT_SELECTION, MSCHOSTNAME="topon.sv.eth2.mnc308.mcc001.org";
MOD VLR: VN="86130028", SVSGS=CONSISTENT_SELECTION, MSCHOSTNAME="topon.sv.eth3.mnc308.mcc001.org";
```

//增加SGs接口MSC的A解析记录。

```
ADD IPV4DNSH: HSINDEX=1, HOSTNAME="topon.sv.eth1.mnc308.mcc001.org", ADDRSECTION=SECTION1, IPV4ADDR1="10.49.49.206", IPV4ADDR2="10.49.49.207";
ADD IPV4DNSH: HSINDEX=2, HOSTNAME="topon.sv.eth2.mnc308.mcc001.org", ADDRSECTION=SECTION1, IPV4ADDR1="10.50.49.206", IPV4ADDR2="10.50.49.207";
ADD IPV4DNSH: HSINDEX=3, HOSTNAME="topon.sv.eth3.mnc308.mcc001.org", ADDRSECTION=SECTION1, IPV4ADDR1="10.51.49.206", IPV4ADDR2="10.51.49.207";
```

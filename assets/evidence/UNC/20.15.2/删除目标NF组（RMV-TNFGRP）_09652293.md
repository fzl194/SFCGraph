# 删除目标NF组（RMV TNFGRP）

- [命令功能](#ZH-CN_MMLREF_0209652293__1.3.1)
- [注意事项](#ZH-CN_MMLREF_0209652293__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0209652293__1.3.4)
- [使用实例](#ZH-CN_MMLREF_0209652293__1.3.5)

## [命令功能](#ZH-CN_MMLREF_0209652293)

**适用NF：AMF、SMF、NRF、NSSF、NCG、SMSF**

该命令用于删除目标NF组的配置。

## [注意事项](#ZH-CN_MMLREF_0209652293)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0209652293)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652293)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| TNFGRPINDEX | 目标NF组索引 | 可选必选说明：必选参数<br>参数含义：本参数用于指定目标NF组索引。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是0~511。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209652293)

运营商A需要删除TNFGRPINDEX为0的目标NF组信息。

```
RMV TNFGRP: TNFGRPINDEX=0;
```

# 删除CP流过滤器（RMV CPFLOWFILTER）

- [注意事项](#ZH-CN_MMLREF_0296805497__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0296805497__1.3.4)

## [命令功能](#ZH-CN_MMLREF_0296805497)

**适用NF：SMF、PGW-C、SGW-C、GGSN**

该命令用于删除SMF和UPF之间的消息流过滤器。

## [注意事项](#ZH-CN_MMLREF_0296805497)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0296805497)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0296805497)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| FLOWFILTERNAME | 流过滤器名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定SMF和UPF之间的消息流过滤器名称。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是1~63。不支持空格，不区分大小写。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0296805497)

删除SMF和UPF之间的IPv6 RS消息流过滤器：

```
RMV CPFLOWFILTER: FLOWFILTERNAME="RS";
```

#### [命令功能](#ZH-CN_MMLREF_0000001126305866)

**适用网元：SGSN**

此命令用于查询归属网络Serving PLMN信息。

#### [注意事项](#ZH-CN_MMLREF_0000001126305866)

- 此命令执行后立即生效。

#### [本地用户权限](#ZH-CN_MMLREF_0000001126305866)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126305866)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126305866)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| NOID | 运营商标识 | 可选必选说明：可选参数<br>参数含义：该参数用于指定运营商标识<br>数据来源：整网规划<br>取值范围：0～64，128～254<br>默认值：无 |
| MCC | 移动国家码 | 可选必选说明：可选参数<br>参数含义：该参数用于指定该运营商的Serving PLMN的移动国家号码。<br>数据来源：整网规划<br>取值范围：3位十进制数字<br>默认值：无 |
| MNC | 移动网号 | 可选必选说明：可选参数<br>参数含义：该参数用于指定该运营商的Serving PLMN的移动网号码。<br>数据来源：整网规划<br>取值范围：位数为2或3的十进制数字<br>默认值：无 |

#### [使用实例](#ZH-CN_MMLREF_0000001126305866)

场景参见 [**ADD HNOSRVPLMN**](增加归属网络Serving PLMN信息(ADD HNOSRVPLMN)_72345655.md) 的命令使用实例。

- 查询归属网络Serving PLMN信息：
  LST HNOSRVPLMN:;
  ```
  %%LST HNOSRVPLMN:;%%
  RETCODE = 0  操作成功。
  输出结果如下
  -------------------------
      运营商标识  =  0
      移动国家码  =  123
        移动网号  =  03
      运营商名称  =  noname
  (结果个数 = 1)
  ---    END
  ```

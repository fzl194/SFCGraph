## [命令功能](#ZH-CN_MMLREF_0000002013939885)

**适用NF：SMSF**

在SMSF/VLR计费场景下，通过本命令查询到短消息计费CHF组。

## [注意事项](#ZH-CN_MMLREF_0000002013939885)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000002013939885)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000002013939885)

无

## [使用实例](#ZH-CN_MMLREF_0000002013939885)

查询SMSF/VLR使用的计费CHF组：

```
LST SMSCHFGRP:;
%%LST SMSCHFGRP:;%%
RETCODE = 0  操作成功

结果如下：
------------------------
 主CHF组的名称  =  SMS CHF Group1
 备CHF组的名称  =  SMS CHF Group2
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000002013939885)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 主CHF组的名称 | 该参数用于设置主CHF组的名称。 |
| 备CHF组的名称 | 该参数用于设置备CHF组的名称。 |

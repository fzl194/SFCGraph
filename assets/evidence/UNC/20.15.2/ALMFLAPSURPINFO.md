#### [命令功能](#ZH-CN_CONCEPT_0259103606)

该命令用于显示告警震荡抑制信息以及当前处于震荡的告警实例信息。

#### [注意事项](#ZH-CN_CONCEPT_0259103606)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0259103606)

G_1，管理员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0259103606)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| AID | 告警ID | 可选必选说明：必选参数<br>参数含义：告警ID。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～4294967295。<br>默认值：无 |
| SERVICEINSTANCE | 服务实例 | 可选必选说明：必选参数<br>参数含义：该参数表示大颗粒服务实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，通过LST VNFC命令获取。<br>默认值：无<br>配置原则：只能填写通过LST VNFC命令查询到的管理代理标识。 |

#### [使用实例](#ZH-CN_CONCEPT_0259103606)

显示警ID为135602186的震荡告警抑制信息，可通过如下命令显示：

```
DSP ALMFLAPSURPINFO:AID=135602186
,SERVICEINSTANCE="ACS"
;
```


#### [输出结果说明](#ZH-CN_CONCEPT_0259103606)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 告警ID | 告警ID。 |
| 告警名称 | 告警名称。 |
| 告警震荡状态 | 告警震荡状态。 |
| 告警震荡产生周期（秒） | 告警震荡产生周期，单位是秒，产生周期必须大于告警抑制时间。在告警震荡产生周期内，告警上报次数（重传不算）达到告警震荡阈值，则进入告警震荡状态，反之退出检测周期。 |
| 告警震荡恢复周期（秒） | 告警震荡恢复周期，单位是秒，恢复周期必须大于告警抑制时间。告警进入震荡状态，若告警震荡恢复周期内没有告警产生与恢复上报，退出震荡状态，若上报恢复告警，则从上报开始计时，当时间达到告警震荡恢复周期，且期间无告警产生上报，则恢复告警上报终端，退出告警震荡状态，若恢复告警上报之后有告警产生上报，则重新计时，直到计时达到告警震荡恢复周期，告警退出震荡状态。 |
| 告警震荡阈值 | 告警震荡阈值，在告警震荡产生周期内，若上报次数达到震荡阈值（重传不算），告警进入震荡状态。 |
| 当前告警震荡产生周期（秒） | 当前告警震荡产生周期，单位是秒，告警震荡进入检测状态后：（1）当前告警震荡产生周期未达到告警震荡产生周期，告警上报次数（重传不算）达到告警震荡阈值，则告警进入震荡状态。（2）当前告警震荡产生周期等于告警震荡产生周期，告警上报次数（重传不算）未达到告警震荡阈值，退出告警震荡检测状态。 |
| 当前告警震荡恢复周期（秒） | 当前告警震荡恢复周期。单位是秒。告警进入震荡状态，当前告警震荡恢复周期从0开始随时间递增：（1）当前告警震荡恢复周期达到告警震荡恢复周期，期间未有告警上报，退出告警震荡状态。（2）若上报恢复告警，当前告警震荡恢复周期从0开始计算，达到告警震荡恢复周期未有告警上报（重传不算），恢复告警上报终端，退出告警震荡检测状态，若上报恢复后有告警产生上报，则告警恢复被抵消，当前告警震荡恢复周期从0开始计算，一直按条件走（1）（2）流程，直到结束为止。 |
| 当前震荡阈值 | 告警进入震荡检测状态后，动态统计告警上报次数（告警重传不算），在告警震荡产生周期内，出现当前震荡阈值大于告警震荡阈值，告警立即进入震荡状态。 |

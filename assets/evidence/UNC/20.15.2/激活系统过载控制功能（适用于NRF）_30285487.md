## [操作步骤](#ZH-CN_OPI_0230285487)

1. 进入 “MML命令行-UNC” 窗口。
2. 添加具体实例的流控参数。
  **[ADD FCPARAM](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/服务通信管理/流控管理/增加流控参数（ADD FCPARAM）_09587901.md)**
3. 设置基于CPU过载流控开关。
  **[SET FCSWITCH](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/服务通信管理/流控管理/设置流控开关（SET FCSWITCH）_09587940.md)**
4. 开启针对管理类消息和发现类消息的自保流控功能。
  **[SET NRFFUNCSW](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/NRF业务及策略管理/NRF业务参数/NRF功能开关/设置NRF功能开关（SET NRFFUNCSW）_09651606.md)**
5. 设置不同流控级别对应的流控响应。
  **[SET NRFFCPARA](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/NRF业务及策略管理/NRF业务参数/NRF流控参数/设置流控等级对应的流控参数（SET NRFFCPARA）_09652224.md)**
6. 设置NF变更事件合并发送通知相关阈值。
  **[SET NRFNOTIFYFCPARA](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/NRF业务及策略管理/NRF业务参数/NRF流控参数/设置通知流控参数（SET NRFNOTIFYFCPARA）_09653165.md)**
7. 设置服务化接口上指定消息类型的固定速率。
  **[SET HTTPFIXEDFCMSG](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/HTTP功能管理/HTTP流控管理/HTTP指定消息类型固定速率流控管理/设置HTTP指定消息类型固定速率流控信息（SET HTTPFIXEDFCMSG）_84132110.md)**

## [任务示例](#ZH-CN_OPI_0230285487)

任务描述

系统过载时，启动自保流控。

脚本

//添加具体实例的流控参数，增加微服务类型为UesmCtrlSvc、微服务实例ID为999、流控套餐ID为0的流控参数，最小令牌数为30，最大令牌数为5000。

```
ADD FCPARAM: SERVICETYPE="UesmCtrlSvc", SERVICEID=999, CASEID=0, PARAMCAT=GENERAL, MINTOKEN=30, MAXTOKEN=5000;
```

//设置基于CPU过载流控开关。

```
SET FCSWITCH: SWITCH=ENABLE;
```

//开启针对管理类消息和发现类消息的自保流控功能。

```
SET NRFFUNCSW: NFMFCSWITCH=FUNC_ON, DISCFCSWITCH=FUNC_ON;
```

//设置不同流控级别对应的流控响应。

```
SET NRFFCPARA: FCLEVEL=LOW, FCRSP=TooManyRequest_429;
SET NRFFCPARA: FCLEVEL=MEDIUM, FCRSP=ServiceUnavailable_503;
SET NRFFCPARA: FCLEVEL=HIGH, FCRSP=NoResponse;
```

//设置NF变更事件合并发送通知相关阈值，50s内NF再次发生变更，则将此NF在900s内的变更事件合并发送，减少NRF发送变更通知的数量。

```
SET NRFNOTIFYFCPARA:NFCHGCACHETHD=900, NFCHGCACHETIME=50;
```

//设置服务化接口上指定消息类型的固定速率流控参数。

```
SET HTTPFIXEDFCMSG:MSGTYPE=NNRF_NFDISCOVERYSERVICE_SEARCHNFINSTANCES,FCSWITCH=ON,THRESHOLD=4000;
```

## [操作场景](#ZH-CN_OPI_0276015639)

本操作指导介绍在运行网络中激活 Non-IP数据传输 的操作过程。

本特性支持NB-IoT（Narrow Band Internet of Things）制式UE通过NB-IoT RAN接入MME后，建立Non-IP的PDN上下文，并采用Non-IP进行数据传输，从而减小传送数据包大小，提高传输效率和空口容量。

> **说明**
> 适用于MME。

## [必备事项](#ZH-CN_OPI_0276015639)

前提条件

- 请仔细阅读[WSFD-215103 Non-IP数据传输特性概述（MME）](特性概述_75839310.md)。
- 已按照[配置到S-GW的数据](../../../../../初始配置/UNC初始配置与调测/组网对接配置/配置AMF&MME&SGSN/配置MME/配置到S-GW的数据_82467048.md)完成S11-U接口的配置。
- 完成加载License。

## [操作步骤](#ZH-CN_OPI_0276015639)

1. 进入 “MML命令行-UNC” 窗口。
2. 增加M2M策略参数。
  [**ADD M2MPLCY**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/M2M管理/M2M策略参数配置/增加M2M策略参数(ADD M2MPLCY)_72225443.md)
3. （可选）设置Diameter兼容性。
  [**SET DMCMPT**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性配置/设置Diameter兼容性(SET DMCMPT)_26306080.md)
4. （可选）当HSS中未给用户签约Non-IP签约数据且需要使用Non-IP业务时，需要增加Non-IP APNNI配置。
  [**ADD NONIPSUBCTRL**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/M2M管理/Non-IP APNNI配置/增加Non-IP APNNI配置(ADD NONIPSUBCTRL)_72225451.md)
  > **说明**
  > - 当开启[WSFD-215301 HSS免升级](../../WSFD-215301 HSS免升级_75808644.md)时，[**ADD NONIPSUBCTRL**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/M2M管理/Non-IP APNNI配置/增加Non-IP APNNI配置(ADD NONIPSUBCTRL)_72225451.md)生效。
  > - 当“SUPAPNNI”选为“SPECIAL_APNNI”时，需要先通过[**ADD APNNIGROUP**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/会话管理/APNNI信息管理/APNNI组管理/增加APNNI组(ADD APNNIGROUP)_26305508.md)和[**ADD APNNI**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/会话管理/APNNI信息管理/APNNI管理/增加APNNI(ADD APNNI)_26305506.md)来绑定APNNI组。
  > - 签约数据匹配时，如果UE未携带请求的APN NI，系统将使用签约的Non-IP Default APN作为该PDN的APN NI。
5. （可选）当用户发生Inter MME移动，且目标MME支持Non-IP数据传输时，需要增加MME属性配置信息。
  [**ADD MMECHARACT**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/GnGp-SGSN_S10_S16_S3接口管理/MME属性/增加MME属性配置信息（ADD MMECHARACT）_26305766.md)
6. 打开本功能的License配置开关。
  [**SET LICENSESWITCH**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)

## [任务示例](#ZH-CN_OPI_0276015639)

任务描述

某行业用户APNNI为"huawei.com"，运营商为其分配了IMSI前缀为"12303"的号段，配置本特性时，需要进行以下操作。

脚本

//增加M2M策略参数。

```
ADD M2MPLCY: SUBRANGE=IMSI_PREFIX, IMSIPRE="12303", APNNI="huawei.com", CPOPSW=SUPPORT;
```

//（可选）设置Diameter兼容性。

```
SET DMCMPT: SF2=FLSTID2_BIT21-1,NBIOTRAT=SUPPORT;
```

//（可选）当HSS中未给用户签约Non-IP签约数据且需要使用Non-IP业务时，需要增加Non-IP APNNI配置。

```
ADD NONIPSUBCTRL: SUBRANGE=IMSI_PREFIX, IMSIPRE="12303", SUPAPNNI=ALL_APNNI; 
```

//（可选）当用户发生Inter MME移动，需要目标MME支持Non-IP数据传输时，需要增加MME属性配置信息。

```
ADD MMECHARACT: RANGE=ALL_MME, CIOTIND=SGNIPDN-1;
```

//打开本功能的License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2NOIP01", SWITCH=ENABLE;
```

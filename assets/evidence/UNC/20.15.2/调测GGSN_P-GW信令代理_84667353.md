# 调测GGSN/P-GW信令代理

- [操作场景](#ZH-CN_OPI_0000001184667353__1.3.1)
- [对系统的影响](#ZH-CN_OPI_0000001184667353__1.3.2)
- [必备事项](#ZH-CN_OPI_0000001184667353__1.3.3)
- [操作步骤](#ZH-CN_OPI_0000001184667353__1.3.4)

## [操作场景](#ZH-CN_OPI_0000001184667353)

当运营商完成GGSN/P-GW信令代理功能配置后，需要通过调试手段检查归属地GGSN/P-GW以及漫游地SGSN/S-GW是否能够正常交互。

> **说明**
> 适用于 GGSN、 PGW-C

## [对系统的影响](#ZH-CN_OPI_0000001184667353)

该操作对系统正常运行无影响。

## [必备事项](#ZH-CN_OPI_0000001184667353)

前提条件

- 请仔细阅读[WSFD-205107 GGSN/P-GW信令代理](WSFD-205107 GGSN_P-GW信令代理特性概述_37468134.md)。
- 完成[激活GGSN/P-GW信令代理](激活GGSN_P-GW信令代理_38987364.md)。

数据

| 类别 | 参数名称 | 取值样例 | 获取方法 |
| --- | --- | --- | --- |
| 用户信息查询 | IMSI | 460000123456789 | 测试终端自带 |
| 测试终端使用的APN | APN | apn-test | 从已配置数据中获取 |

工具

- 测试终端
- OM Portal

## [操作步骤](#ZH-CN_OPI_0000001184667353)

1. 以下命令在“MML命令行-UNC”窗口执行。查询License中是否允许使用GGSN/P-GW信令代理功能。
  [**LST LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/查询License配置项开关（LST LICENSESWITCH）_09651570.md) :LICITEM="LKV3W9GPSP11";
    - 如果“SWITCH”为“ENABLE”，请执行[步骤2](#ZH-CN_OPI_0000001184667353__step35844617562)。
    - 如果“SWITCH”为“DISABLE”，则当前设备不支持GGSN/P-GW信令代理功能，请重新申请License后继续调测本功能。
2. 在OM Portal上 [创建用户跟踪](../../../../../网络运维/日常维护/UNC基础运维操作/创建消息跟踪/创建用户跟踪/用户跟踪_72467812.md) ，在“参数配置”栏输入用户IMSI，准备跟踪测试终端的出入报文。
3. 测试终端使用APN“apn-test”接入网络，用户访问业务。
4. 查看消息跟踪，观察以下四条消息，如 [图1](#ZH-CN_OPI_0000001184667353__fig95676103414) 和 [图2](#ZH-CN_OPI_0000001184667353__fig1874751516343) 所示。
  **图1** 用户跟踪任务结果—GGSN信令代理场景

  <br>

  ![](调测GGSN_P-GW信令代理_84667353.assets/zh-cn_image_0000001149058134_2.png)
  **图2** 用户跟踪任务结果—P-GW信令代理场景

  <br>

  ![](调测GGSN_P-GW信令代理_84667353.assets/zh-cn_image_0000001195097759_2.png)
  以上 [图1](#ZH-CN_OPI_0000001184667353__fig95676103414) 和 [图2](#ZH-CN_OPI_0000001184667353__fig1874751516343) 中，消息从上到下分别对应以下的消息a、消息b、消息c、消息d。
    a. SGSN/S-GW发给UNC的激活请求消息；
    b. UNC转发给归属地GGSN/P-GW的激活请求消息；
    c. 归属地GGSN/P-GW发给UNC的激活响应消息；
    d. UNC发给SGSN/S-GW的激活响应消息；
    - 如果消息a异常：联系SGSN/S-GW的维护人员处理SGSN/S-GW的故障。
    - 如果消息b或者消息d异常：执行[步骤 5](#ZH-CN_OPI_0000001184667353__step821519435811)。
    - 如果消息c异常，执行[步骤 12](#ZH-CN_OPI_0000001184667353__step846217271667)。
    - 如果消息a～d都正常，则功能正常，调测结束。
5. 检查UNC上的配置，执行 [**LST APNGWPROXYFUNC**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/GGSN和P-GW Proxy/APN网关Proxy功能/查询APN网关Proxy功能配置（LST APNGWPROXYFUNC）_42693468.md) 命令，观察指定APN的信令代理功能是否开启。
  ```
  LST APNGWPROXYFUNC:APN="apn-test";
  ```
  ```
  APN网关Proxy功能参数 
  ------------------------------------                  
                     APN  =  apn-test          
            Proxy功能开关 =  使能 
  (结果个数 = 1) 
  ---    END
  ```
    - Proxy功能开关值为ENABLE：执行[步骤 6](#ZH-CN_OPI_0000001184667353__step126305315816)。
    - Proxy功能开关值为DISABLE：参照激活GGSN/P-GW信令代理开启信令代理功能。
6. 检查UNC上的配置，执行 [**LST GWPROXYFUNC**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/GGSN和P-GW Proxy/网关Proxy功能/查询网关Proxy功能配置（LST GWPROXYFUNC）_42693470.md) 命令，观察信令代理功能是否开启。
  ```
  LST GWPROXYFUNC;
  ```
  ```
  网关Proxy功能参数 
  -------------------------------- 
  Proxy功能开关  =  不使能
  ```
    - 是：执行[步骤 7](#ZH-CN_OPI_0000001184667353__step194761041458)。
    - 否：参照激活GGSN/P-GW信令代理开启信令代理功能。
7. 执行 [**LST HOMEGRPBINDAPN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/GGSN和P-GW Proxy/Home Group绑定APN/查询Home Group和APN的绑定关系（LST HOMEGRPBINDAPN）_88733221.md) 命令查看指定APN下是否绑定了home-group。
  ```
  LST HOMEGRPBINDAPN:APN="apn-test";
  ```
  ```
  Home Group绑定APN配置信息： 
  -------------------------------             
  APN = apn-test Home group的编号 = 1 
  (结果个数 = 1) 
  --- END
  ```
8. 执行 [**LST HOMEGROUP**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/GGSN和P-GW Proxy/Home Group/查询Home Group（LST HOMEGROUP）_88613381.md) 命令查看home-group相关配置是否正确。
  ```
  LST HOMEGROUP: HOMEGROUPINDX=1;
  ```
  ```
  Home Group信息
  ----------------------                    
                             Home Group Index  =  1                 
                       Home Group Priority  =  1      
                 IMSI/MSISDN Segment Group Name  =  group-test 
  Charging Characteristics Group Name  =  cc_group1 
  (Number of results = 1) 
  ---    END
  ```
9. **可选：** 执行 [**LST PRIMSIMSISDNG**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/GGSN和P-GW Proxy/代理选择的IMSI_MSISDN号段组/查询代理IMSI_MSISDN号段组（LST PRIMSIMSISDNG）_42693472.md) 命令，查看IMSI/MSISDN号码段组配置是否正常。
  ```
  LST PRIMSIMSISDNG: SEGGROUPNAME="group-test";
  ```
  ```
  IMSI或MSISDN号段组信息 
  ---------------------------------------- 
  IMSI/MSISDN号段组名称  =  huawei   
  IMSI/MSISDN号段名称  =  imsi 
  (结果个数 = 1) 
  ---    END
  ```
    - 是：执行[步骤 10](#ZH-CN_OPI_0000001184667353__step4890141715811)。
    - 否：参考激活GGSN/P-GW信令代理重新配置。
10. **可选：** 执行 [**LST PRIMSIMSISDNSEG**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/GGSN和P-GW Proxy/代理选择的IMSI_MSISDN号段/查询代理IMSI_MSISDN号段（LST PRIMSIMSISDNSEG）_88613383.md) 命令，查看号码段和规划值是否一致。
  ```
  LST PRIMSIMSISDNSEG: SEGMENTNAME="msisdn-test";
  ```
  ```
  IMSI或MSISDN号段信息 
  ---------------------------------- 
  IMSI/MSISDN号段名称  =  msisdn-test 
  IMSI/MSISDN号段类型  =  MSISDN       
  号段起始字符串  =  138       
  号段结束字符串  =  139       
  通配号段字符串  =  NULL (结果个数 = 1) 
  ---    END
  ```
    - 是：执行[步骤 12](#ZH-CN_OPI_0000001184667353__step846217271667)。
    - 否：参考激活GGSN/P-GW信令代理重新配置。
11. **可选：** 执行 [**LST CCVALUE**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/GGSN和P-GW Proxy/计费特征值/查询计费特征值（LST CCVALUE）_42853260.md) 命令，查看UNC上的计费属性信息和规划值是否一致。
  ```
  LST CCVALUE: CCGROUPNAME="cc_group1";
  ```
  ```
  Charging Characteristics Value Configuration 
  -------------------------------------------- 
  Charging Characteristics Group Name    Charging Characteristic Value  
  cc_group1                               0x0010  
  (Number of results = 1) 
  --- END
  ```
    - 是：执行[步骤 12](#ZH-CN_OPI_0000001184667353__step846217271667)。
    - 否：参考激活GGSN/P-GW信令代理重新配置。
12. 执行 **[**SET GTPCFUNCPARA**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/GTP-C接口配置管理/GTP-C功能参数/设置GTP-C功能参数（SET GTPCFUNCPARA）_09653666.md)** 命令，探测归属地GGSN/P-GW状态是否正常
    - 如果没有告警“ALM-80610 GTPC路径故障”产生：执行[步骤 13](#ZH-CN_OPI_0000001184667353__step847173213618)。
    - 如果有告警“ALM-80610 GTPC路径故障”产生：请归属地GGSN/P-GW维护人员处理。
13. 收集信息并寻求技术支持。
    a. 执行**[**EXP MML**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/配置管理/配置导出管理/导出MML文件（EXP MML）_47200033.md)**命令将当前配置数据导出为MML脚本文件并保存。
    b. 收集并保存上述所有查询信息。
    c. 查看并收集对端设备配置及接口状态信息。
    d. 收集归纳所有信息并联系华为技术支持解决。

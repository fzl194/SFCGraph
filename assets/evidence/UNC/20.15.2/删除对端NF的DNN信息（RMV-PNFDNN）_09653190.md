# 删除对端NF的DNN信息（RMV PNFDNN）

- [命令功能](#ZH-CN_MMLREF_0209653190__1.3.1)
- [注意事项](#ZH-CN_MMLREF_0209653190__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0209653190__1.3.4)
- [使用实例](#ZH-CN_MMLREF_0209653190__1.3.5)

## [命令功能](#ZH-CN_MMLREF_0209653190)

**适用NF：AMF、SMF、NSSF、SMSF、NCG、SGW-C、PGW-C、GGSN**

该命令用于删除本地配置的对端NF实例支持的DNN及其归属切片的信息。

## [注意事项](#ZH-CN_MMLREF_0209653190)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0209653190)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209653190)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| INDEX | 索引 | 可选必选说明：必选参数<br>参数含义：该参数用于指定DNN的索引。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~4294967295。<br>默认值：无<br>配置原则：<br>该参数取值不能重复，建议从0开始顺序取值。 |

## [使用实例](#ZH-CN_MMLREF_0209653190)

删除对端NF的DNN信息，索引为1。

```
RMV PNFDNN: INDEX=1;
```

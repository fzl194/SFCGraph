## [命令功能](#ZH-CN_MMLREF_0000001171436531)

**适用NF：NRF**

此命令用于查询手动向NF发送的订阅通知的发送结果。

## [注意事项](#ZH-CN_MMLREF_0000001171436531)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001171436531)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001171436531)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| CALLBACKURI | 通知地址 | 可选必选说明：可选参数<br>参数含义：该参数表示通知发往的目的NF回调URI。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是1~512。<br>默认值：无<br>配置原则：无 |
| SUBID | 订阅标识 | 可选必选说明：可选参数<br>参数含义：该字段表示订阅者的订阅标识。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~48。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001171436531)

显示手动发送的订阅通知结果。

```
%%DSP NRFMANUSENDNTY:;%%
RETCODE = 0  操作成功

结果如下
------------------------
  通知地址  =  http://10.70.1.118:5164/nnrf-nfm/v1/nf-instances/ff02-1
  订阅标识  =  000200050b16cec9bd65491681cb88775ca9ded0
通知响应码  =  204
(Number of results = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001171436531)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 通知地址 | 该参数表示通知发往的目的NF回调URI。 |
| 订阅标识 | 该字段表示订阅者的订阅标识。 |
| 通知响应码 | 该参数表示NRF向NF发送通知后收到的响应码。 |

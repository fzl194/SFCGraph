## [操作场景](#ZH-CN_OPI_0000001601717985)

该操作用于指导在运行网络中激活 磁盘故障隔离 功能。

## [必备事项](#ZH-CN_OPI_0000001601717985)

前提条件

请仔细阅读 [WSFD-209002 磁盘故障隔离特性概述](WSFD-209002 磁盘故障隔离特性概述_50999118.md) 。

数据

该操作无需准备数据。

## [操作步骤](#ZH-CN_OPI_0000001601717985)

1. 设置节点自动进入BYPASS开关，该开关默认开启。
  **[SET BMBYPASSSWITCH](../../../../../OM参考/命令/UNC MML命令/平台服务管理/设备管理/Bypass开关管理/设置裸机节点自动进入BYPASS开关（SET BMBYPASSSWITCH）_58120294.md)**

## [任务示例](#ZH-CN_OPI_0000001601717985)

任务描述

配置节点在存储故障时自动进行Bypass状态。

脚本

//设置节点自动进入BYPASS开关，该开关默认开启。

```
SET BMBYPASSSWITCH: SSW_BYPASS_SWITCH=ON;
```

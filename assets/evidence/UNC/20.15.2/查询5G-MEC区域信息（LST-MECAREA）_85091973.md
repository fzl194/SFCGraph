## [命令功能](#ZH-CN_MMLREF_0000001585091973)

**适用NF：AMF**

该命令用于查询5G MEC区域信息。

## [注意事项](#ZH-CN_MMLREF_0000001585091973)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001585091973)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001585091973)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| AREAID | 区域标识 | 可选必选说明：可选参数<br>参数含义：该参数用于唯一标识AMF服务的某个区域（例如：商场）。其服务范围可以通过ADD MECAREATAI或ADD MECAREAGNB进行配置。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是1~1024。<br>默认值：无<br>配置原则：<br>当通过RMV MECAREA命令删除“AREAID”区域成员时，请执行LST MECAREATAI和LST MECAREAGNB命令，确保中“AREAID”未被索引。 |

## [使用实例](#ZH-CN_MMLREF_0000001585091973)

查询5G MEC区域信息，执行如下命令：

```
%%LST MECAREA:;%%
RETCODE = 0  操作成功

结果如下
--------
    区域标识  =  1
    服务范围  =  West Lake
    区域类型  =  公网区域
    描述信息  =  West Lake
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001585091973)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 区域标识 | 该参数用于唯一标识AMF服务的某个区域（例如：商场）。其服务范围可以通过ADD MECAREATAI或ADD MECAREAGNB进行配置。 |
| 服务范围 | 该参数用于描述目标PCF的服务范围。 |
| 区域类型 | 该参数用于标识当前区域的区域类型。 |
| 描述信息 | 该参数用于描述区域信息。 |

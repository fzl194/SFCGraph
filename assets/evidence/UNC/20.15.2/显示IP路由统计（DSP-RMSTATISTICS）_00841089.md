#### [命令功能](#ZH-CN_CONCEPT_0000001600841089)

该命令用于显示IP路由表的统计信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600841089)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600841089)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600841089)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| ADDRESSFAMILY | 地址族 | 可选必选说明：必选参数<br>参数含义：该参数用于表示路由所属VPN实例的地址族信息。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- ipv4unicast：IPv4单播。<br>- ipv6unicast：IPv6单播。<br>默认值：无 |
| VRFNAME | VPN实例名称 | 可选必选说明：可选参数<br>参数含义：该参数用于表示VPN实例的名字。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。<br>默认值：无<br>配置原则：公网需要输入_public_，默认查询所有VPN，不支持空格。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600841089)

显示IP路由表的统计信息：

```
DSP RMSTATISTICS: ADDRESSFAMILY=ipv4unicast;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600841089)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 路由协议名字 | 用于表示路由协议的名字。 |
| 路由总数 | 用于表示路由总数量。 |
| 活跃路由数 | 用于表示激活路由的数量。 |
| 添加路由数 | 用于表示增加的路由数量，包含激活和未激活的路由。 |
| 删除路由数 | 用于表示删除的路由数量。 |
| 释放路由数 | 用于表示永久删除的路由数目。 |

其余输出项请参见SET PREFIXLIMIT的参数说明。

#### [命令功能](#ZH-CN_CONCEPT_0000002071887721)

该命令用于查询base亚健康配置信息。

#### [注意事项](#ZH-CN_CONCEPT_0000002071887721)

该命令执行后立即生效。

#### [操作用户权限DSP RU](#ZH-CN_CONCEPT_0000002071887721)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000002071887721)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SERVICEINSTANCE | 服务实例 | 可选必选说明：必选参数<br>参数含义：该参数表示大颗粒服务实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，通过LST VNFC命令获取。<br>默认值：无<br>配置原则：只能填写通过LST VNFC命令查询到的管理代理标识。 |

#### [使用实例](#ZH-CN_CONCEPT_0000002071887721)

查询base亚健康配置信息：

```
LST BASESUBHEALTH:
SERVICEINSTANCE="vnfc"
;
```

```
RETCODE = 0  操作成功

结果如下:

       检测周期 （s）  =  1
       统计周期 （s）  =  30
亚健康阈值 （千分比）  =  50
             使能标志  =  去使能
(结果个数 = 1)
---    END
```

## [操作场景](#ZH-CN_OPI_0277396886)

本操作指导介绍在运行网络中激活 eMTC低优先级接入控制 功能的操作过程。

随着物联网的广泛应用，eMTC终端被广泛应用在多种行业中，例如水、电表的上报、物流、智能停车等。

目前运营商对于eMTC终端业务的接入控制较为薄弱，当大量的eMTC终端在很短的时间内同时发起业务（例如指定日期的水表上报），可能会造成系统过载并影响普通终端用户的业务接入。

本特性通过识别eMTC终端业务为低优先级业务，限制eMTC终端业务的接入量，来保证普通终端用户业务的成功率，预防或缓解系统拥塞。

> **说明**
> 适用于MME。

## [操作步骤](#ZH-CN_OPI_0277396886)

1. 进入 “MML命令行-UNC” 窗口。
2. 设置低优先级接入控制参数。
  [**SET LOWPRIOFC**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/操作维护/设备管理/流控管理/业务流控管理/低优先级接入流控管理/设置低优先级流控参数(SET LOWPRIOFC)_72345759.md)
3. 打开本功能的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)

## [任务示例](#ZH-CN_OPI_0277396886)

任务描述

网络存在eMTC终端业务与普通终端业务共用一套 UNC ，当出现大量的eMTC业务情况时，可能会影响在网普通终端用户的业务，此时开启低优先级接入控制，自动调节低优先级业务的接入消息允许处理速率。

脚本

//设置低优先级接入控制参数。

```
SET LOWPRIOFC: FCSWITCH=ON, MINTHD=10, MAXTHD=300;
```

//打开本功能的License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2LPAC02", SWITCH=ENABLE;
```

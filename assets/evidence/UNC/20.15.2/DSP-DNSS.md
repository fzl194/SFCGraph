#### [命令功能](#ZH-CN_MMLREF_0000001172345499)

**适用网元：SGSN、MME** **、AMF**

该命令用于查看DNS服务器的状态。

#### [注意事项](#ZH-CN_MMLREF_0000001172345499)

- 该命令执行后立即生效。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172345499)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172345499)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172345499)

无。

#### [使用实例](#ZH-CN_MMLREF_0000001172345499)

查询DNS服务器信息：

DSP DNSS:;

```
%%DSP DNSS:;%%
RETCODE = 0  操作成功。

操作结果如下
-------------------------
 服务器组ID    IP地址类型     IP地址           域名服务器优先级   DNS服务器名称     服务器状态   是否支持递归
 0            IPV4           192.168.179.1   优先级1           NULL              正常        不支持
 1            IPV4           192.168.179.2   优先级1           NULL              故障        不支持
(结果个数 = 2)

---    END
```

#### [输出结果说明](#ZH-CN_MMLREF_0000001172345499)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 服务器组ID | DNS服务器组ID。<br>取值说明：0~37 |
| IP地址类型 | DNS服务器IP地址类型。 |
| IP地址 | DNS服务器IPv4地址。 |
| IPv6地址 | DNS服务器IPv6地址。 |
| 域名服务器优先级 | DNS服务器的优先级。<br>取值说明：<br>- “PRI1”<br>- “PRI2”<br>- “PRI3”<br>- “PRI4”<br>- “PRI5”<br>- “PRI6”<br>说明：优先级从PRI1到PRI6依次递减。 |
| DNS服务器名称 | DNS服务器名称。 |
| 服务器状态 | DNS服务器状态。<br>取值说明：<br>- 正常<br>- 故障<br>- 未知<br>说明：DNS服务器的状态可以是<br>“正常”<br>、<br>“故障”<br>、<br>“未知”<br>，当没有配置DNS本端实体时，DNS服务器状态为<br>“未知”<br>。 |
| 是否支持递归 | 是否支持递归。<br>取值说明：<br>- 不支持<br>- 支持<br>- 未知 |

## [操作场景](#ZH-CN_OPI_0193181079)

本操作指导介绍在运行网络中激活漫游用户QoS限制特性的操作过程。

漫游用户QoS限制特性根据用户IMSI和用户所属HLR的HLR号码对用户属地进行识别，并对不同用户属地的用户设置不同的QoS优先级，以合理利用资源，优先保证本网本地用户的业务体验。用户属地包括外网、本网，本网进一步区分为本网本地和本网异地。其中，漫游用户指外网用户和本网异地用户。

## [操作步骤](#ZH-CN_OPI_0193181079)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. 配置本网本地用户所归属的HLR号码列表，用于进一步判断本网用户是本网本地用户还是本网异地用户。
  [**ADD LOCALHLR**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/LOCALHLR管理/增加本地HLR(ADD LOCALHLR)_72225755.md)
4. 配置不同属地用户的QoS协商参数，用于为不同属地的用户设置不同的本地配置QoS。
  [**ADD IMSISMCHAR**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/QoS管理/Pre-R8 QoS/QoS协商控制/QoS协商参数管理/增加QoS协商参数(ADD IMSISMCHAR)_26306040.md)
  > **说明**
  > 不同属地类别的参数选择有所不同，此处以本网用户为例。
  >
  > 参数 “NTYPE” （协商方式）必须选择 “网络侧协商QoS” ，否则漫游用户QoS限制特性无法生效。

## [任务示例](#ZH-CN_OPI_0193181079)

任务描述

在GPRS/UMTS网络中启用漫游用户QoS限制，满足如下漫游用户QoS限制需求，以实现对漫游用户进行限制，优先保证本网本地用户的业务体验。

- 指定归属HLR的HLR号码为8613900477的用户为本网本地用户，归属HLR的HLR号码为其他的用户为本网异地用户。
- 对于某运营商GPRS网络的外网用户，SGSN本地配置QoS为上行最大速率64kbit/s、下行最大速率64kbit/s。
- 对于某运营商GPRS网络中的本网异地用户，SGSN本地配置QoS为上行最大速率64kbit/s、下行最大速率64kbit/s。
- 对于某运营商GPRS网络中的本网本地用户，SGSN本地配置QoS为上行最大速率128kbit/s、下行最大速率128kbit/s。

脚本

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2RSQR01", SWITCH=ENABLE;
```

//指定归属HLR的HLR号码为8613900477的用户为本网本地用户，归属HLR的HLR号码为其他的用户为本网异地用户。

```
ADD LOCALHLR: HLRIDX=1, HLRNUM="8613900477", HLRNAME="Operator A";
```

//对于GPRS网络的外网用户，设置SGSN本地配置QoS为上行最大速率64kbit/s、下行最大速率64kbit/s。

```
ADD IMSISMCHAR: SUBRANGE=FOREIGN_USER, NOID=60, APNNI="*", UEACCCAP=ALL_UE, NTYPE=NET_NEGO_QOS, QOS2G=YES, MBRUPLK2G=64, MBRDNLK2G=64;
```

//对于GPRS网络中的本网异地用户，设置SGSN本地配置QoS为上行最大速率64kbit/s、下行最大速率64kbit/s。

```
ADD IMSISMCHAR: SUBRANGE=HOME_USER, NOID=60, USERSUBTYPE=HOME_UNLOCAL_USER, APNNI="*", UEACCCAP=ALL_UE, NTYPE=NET_NEGO_QOS, QOS2G=YES, MBRUPLK2G=64, MBRDNLK2G=64;
```

//对于GPRS网络中的本网本地用户，设置SGSN本地配置QoS为上行最大速率128kbit/s、下行最大速率128kbit/s。

```
ADD IMSISMCHAR: SUBRANGE=HOME_USER, NOID=60, USERSUBTYPE=HOME_LOCAL_USER, APNNI="*", UEACCCAP=ALL_UE, NTYPE=NET_NEGO_QOS, QOS2G=YES, MBRUPLK2G=72, MBRDNLK2G=72;
```

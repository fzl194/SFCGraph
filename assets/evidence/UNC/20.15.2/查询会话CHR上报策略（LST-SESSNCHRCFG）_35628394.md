## [命令功能](#ZH-CN_MMLREF_0235628394)

**适用NF：SGW-C、PGW-C、SMF、GGSN**

该命令用于查询会话CHR单据采集及订阅流程和上报CHR单据的UNC产品设备号。

## [注意事项](#ZH-CN_MMLREF_0235628394)

无

#### [操作用户权限](#ZH-CN_MMLREF_0235628394)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0235628394)

无

## [使用实例](#ZH-CN_MMLREF_0235628394)

查询会话CHR采集及订阅流程：

```
LST SESSNCHRCFG:;
RETCODE = 0 操作成功。

结果如下
------------------------
高性能服务器上报流程控制模板索引  =  0
低性能服务器上报流程控制模板索引  =  1
(结果个数= 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0235628394)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 高性能服务器上报流程控制模板索引 | 控制网元向高性能CHR服务器发送的会话CHR单据的流程列表，此索引必须已经通过ADD SESSNCHRPRCTMPL命令配置。 |
| 低性能服务器上报流程控制模板索引 | 控制网元向低性能CHR服务器发送的会话CHR单据的流程列表，此索引必须已经通过ADD SESSNCHRPRCTMPL命令配置。<br>该参数功能待后续版本实现。 |
| 上报CHR单据的UNC产品设备号 | 该参数用于指定上报CHR单据的UNC产品设备号。 |

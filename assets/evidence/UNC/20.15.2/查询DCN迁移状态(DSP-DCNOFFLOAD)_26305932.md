#### [命令功能](#ZH-CN_MMLREF_0000001126305932)

**适用网元：MME**

该命令用于查询DCN迁移的进度信息。

#### [注意事项](#ZH-CN_MMLREF_0000001126305932)

此命令对类型为“IMSI(IMSI)”的迁移任务不生效。针对类型为“IMSI(IMSI)”的迁移任务，可通过用户跟踪等手段查看迁移进度。

#### [本地用户权限](#ZH-CN_MMLREF_0000001126305932)

manage-ug；system-ug；monitor-ug；visit-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126305932)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组；G_4，来宾级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126305932)

无。

#### [使用实例](#ZH-CN_MMLREF_0000001126305932)

1. 查询DCN迁移的进度信息：DSP DCNOFFLOAD:;
  ```
  %%DSP DCNOFFLOAD:;%%
  RETCODE = 0  执行成功

  输出结果如下：
  ------------
           迁移类型  =  全部用户
       指定IMSI前缀  =  NULL
           迁移状态  =  结束完成
     已经迁移用户数  =  1
  剩余迁移时间(min)  =  0
  (结果个数 = 1)

  ---    END
  ```

#### [输出结果说明](#ZH-CN_MMLREF_0000001126305932)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 迁移用户范围 | 该参数用来显示DCN迁移的用户范围。 |
| 指定IMSI前缀 | 该参数用来显示IMSI前缀。 |
| 迁移状态 | 该参数用来显示迁移任务当前状态，取值范围：<br>- “DCN迁移中”<br>- “结束完成”<br>- “异常中断”<br>- “DCN迁移任务未启动” |
| 已经迁移用户数 | 该参数用于显示迁移任务中执行专用核心网重选的用户数。<br>说明：当<br>[**SET DCNCTRL**](../../业务安全管理/DCN管理/DCN控制参数/设置DCN控制参数(SET DCNCTRL)_26305634.md)<br>配置的<br>“重选策略”<br>为<br>“DELAY(延缓)”<br>时，该参数表示系统判定需要迁移的用户数，该类用户在空闲态发送Attach Request或者TAU Request消息时MME进行专用核心网重选。 |
| 剩余迁移时间(min) | 该参数用于显示迁移任务中根据迁移速率（每SPP进程4个/秒）和用户数得出的剩余迁移时间的估计值。 |

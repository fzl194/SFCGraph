#### [命令功能](#ZH-CN_CONCEPT_0209896774)

**适用NF：PGW-C、SMF**

该命令用于查询流量备份配置的信息。

#### [注意事项](#ZH-CN_CONCEPT_0209896774)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0209896774)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0209896774)

无。

#### [使用实例](#ZH-CN_CONCEPT_0209896774)

查询流量备份配置：

```
LST TRAFFICBACKUP:;
```

```

RETCODE = 0  操作成功。

流量备份配置
------------
   流量备份开关  =  允许
备份间隔 (分钟)  =  30
(结果个数 = 1)
---    END
```

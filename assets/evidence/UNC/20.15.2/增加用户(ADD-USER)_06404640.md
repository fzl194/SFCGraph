#### [命令功能](#ZH-CN_TOPIC_0206404640)

本命令用于创建OM Portal的操作用户。该用户可以通过OM Portal页面登录系统。

- 账号策略：对于账号的一些特定要求，可以通过OM Portal的“ 安全 > 安全策略 ”查询。
- 密码策略：对于密码的一些特定要求，可以通过OM Portal的“ 安全 > 安全策略 ”查询。
- 默认用户：系统缺省已配置的用户，可以通过OM Portal的“ 安全 > 用户管理 ”查询。

#### [注意事项](#ZH-CN_TOPIC_0206404640)

无。

#### [参数说明](#ZH-CN_TOPIC_0206404640)

| **参数标识** | **参数名称** | **参数说明** |
| --- | --- | --- |
| USRNAME | 用户名称 | 可选必选说明：必选参数。<br>参数含义：用户名称。<br>取值范围：仅支持中英文、数字、下划线、中划线，不支持生僻字。且控制在6~32个字符，输入长度根据<br>“安全 > 安全策略 > 账号策略”<br>中的<br>“账号名最小字符个数”<br>值确定。<br>默认值：无。<br>配置原则：无。 |
| USERTYPE | 用户类型 | 可选必选说明：必选参数。<br>参数含义：用户类型。<br>取值范围：<br>- COMMON(普通用户)：用于登录OM Portal的用户。<br>- SYSTEM(系统对接用户)：用于对接网管等系统的用户。<br>默认值：COMMON(普通用户)。<br>配置原则：无。 |
| NPWD | 密码 | 可选必选说明：必选参数。<br>参数含义：新增用户对应的密码。<br>取值范围：<br>- 至少包括一个大写字母、一个小写字母和一个数字字符，且不能包含用户名及用户名的倒序。<br>- 输入长度范围10~64个字符。<br>- 用户密码中可包含特殊字符： !"#$%&'()*+,-./\:;<=>?@[]^`{_\|}~。<br>- 用户密码中不能出现三位以上（包括三位）的重复数字、字母与特殊字符，例如“111”、“aaa”、“\\\”。<br>- 用户密码中不能出现三位以上（包括三位）的连续数字、字母，例如“123”、“abc”、“aBc”。<br>默认值：无。<br>配置原则：无。 |
| CPWD | 确认密码 | 可选必选说明：必选参数。<br>参数含义：确认密码。<br>取值范围：<br>- 至少包括一个大写字母、一个小写字母和一个数字字符，且不能包含用户名及用户名的倒序。<br>- 输入长度范围10~64个字符。<br>- 用户密码中可包含特殊字符： !"#$%&'()*+,-./\:;<=>?@[]^`{_\|}~。<br>- 用户密码中不能出现三位以上（包括三位）的重复数字、字母与特殊字符，例如“111”、“aaa”、“\\\”。<br>- 用户密码中不能出现三位以上（包括三位）的连续数字、字母，例如“123”、“abc”、“aBc”。<br>默认值：无。<br>配置原则：无。 |
| DESC | 描述 | 可选必选说明：可选参数。<br>参数含义：用户的描述信息。<br>取值范围：长度不超过127个字符。<br>默认值：无。<br>配置原则：无。 |

#### [使用实例](#ZH-CN_TOPIC_0206404640)

增加普通用户test01：

```
%%ADD USER: USRNAME="test01", USERTYPE=COMMON, NPWD="*****", CPWD="*****";%%
RETCODE = 0  操作成功

---    END
```

#### [命令功能](#ZH-CN_CONCEPT_0000001550280882)

该命令用于显示BGP IPv4对等体统计信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001550280882)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550280882)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550280882)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| VRFNAME | VPN实例名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定用户所配置的VPN实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。<br>默认值：无<br>配置原则：使用LST L3VPNINST命令查看可用VPN。 |
| AFTYPE | 地址族类型 | 可选必选说明：可选参数<br>参数含义：该参数用于指定VRF的地址族类型。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- ipv4uni：IPv4地址族。<br>- ipv4vpn：VPNv4地址族。<br>- ipv6uni：IPv6地址族。<br>- ipv6vpn：VPNv6地址族。<br>默认值：ipv4uni |
| SOURCEIFNAME | 多源接口名称 | 可选必选说明：可选参数<br>参数含义：该参数用来给定多源对等体的接口名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。<br>默认值：无<br>配置原则：仅支持Ethernet及其子接口类型，不区分大小写。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001550280882)

```
DSP BGPPEERSTATISTICSINFO:;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001550280882)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 对等体地址 | 用于指定路由器对等体的对端地址或地址段。 |
| 多源接口名称 | 用来给定多源对等体的接口名称。 |
| 可达路由数量 | 用于指定可达路由数量。 |
| 不可达路由数量 | 用于指定不可达路由数量。 |
| 过策略前路由数量 | 用于指定过策略前路由数量。 |
| 过策略后路由数量 | 用于指定过策略后路由数量。 |
| 最优路由数量 | 用于指定最优路由数量。 |
| 策略拒绝路由数量 | 用于指定策略拒绝路由数量。 |
| AS环路路由数量 | 用于指定AS环路路由数量。 |
| AS联盟环路路由数量 | 用于指定AS联盟环路路由数量。 |
| 簇环路路由数量 | 用于指定簇环路路由数量。 |
| 无效源路由器ID路由数量 | 用于指定无效源路由器ID路由数量。 |

其余输出项请参见ADD BGPMULTIPEER、ADD FILTERPOLICY的参数说明。

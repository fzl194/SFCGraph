## [命令功能](#ZH-CN_MMLREF_0000001405054701)

**适用NF：SMSF**

该命令用于查询SMSF的UDM全故障Bypass功能开关。

## [注意事项](#ZH-CN_MMLREF_0000001405054701)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001405054701)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001405054701)

无

## [使用实例](#ZH-CN_MMLREF_0000001405054701)

运营商希望查询SMSF的UDM全故障Bypass功能开关，执行如下命令：

```
LST SMSFUDMBYPASS:;
%%LST SMSFUDMBYPASS:;%%
RETCODE = 0  操作成功

结果如下：
------------------------
UDM全故障Bypass开关 = 打开
退出Bypass状态恢复动作 = 补充缺失的UDM流程
故障探测速率(个/秒) = 5
扫描时间间隔(秒) = 20
上报异常CHR开关 = 打开

(结果个数 = 1)
```

## [输出结果说明](#ZH-CN_MMLREF_0000001405054701)

| 输出项名称 | 输出项解释 |
| --- | --- |
| UDM全故障Bypass开关 | 该参数用于控制开启和关闭UDM全故障Bypass特性。 |
| 退出Bypass状态恢复动作 | 该参数用于设置UDM全故障Bypass状态恢复动作。 |
| 故障探测速率(个/秒) | 该参数用于指定故障探测速率，即每个DS每秒扫描多少个用户，扫描到后对符合条件的用户进行探测。 |
| 扫描时间间隔(秒) | 该参数用于表示UDM Bypass后，允许用户连续两次向UDM发试探消息的最小时间间隔。 |
| 上报异常CHR开关 | 该参数用于设置当用户注册到SMSF且进入UDM Bypass的流程时，是否上报异常CHR单据。是否上报单据取决于ADD SMSCHRPRCTMPL配置。 |

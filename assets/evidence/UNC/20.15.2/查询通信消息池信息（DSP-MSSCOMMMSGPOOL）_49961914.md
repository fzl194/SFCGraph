#### [命令功能](#ZH-CN_CONCEPT_0000001549961914)

该命令用于查询MSS的通信消息池信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001549961914)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549961914)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549961914)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RUNAME | RU名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定RU名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。<br>默认值：无<br>配置原则：使用DSP RU查看RU名称。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549961914)

查询通信消息池信息：

```
DSP MSSCOMMMSGPOOL:RUNAME = "VNODE_VNRS_VNFC_IPU_0064";
```

```

RETCODE = 0  操作成功。

结果如下
-------------------------
消息池ID   单元大小（byte）   单元数量   已用数量    申请次数   释放次数   泄露数量
0          4096               4096       1           1123       1123       0
1          512                4096       0           1024       1024       0
(结果个数 = 2)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001549961914)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 消息池ID | 用于表示消息池ID。 |
| 单元大小（byte） | 用于表示消息单元大小，单位为字节。 |
| 单元数量 | 用于表示消息单元总数量。 |
| 已用数量 | 用于表示已用消息单元数量。 |
| 申请次数 | 用于表示消息单元成功申请次数。 |
| 释放次数 | 用于表示消息单元成功释放次数。 |
| 泄露数量 | 用于表示消息单元泄露数量。 |

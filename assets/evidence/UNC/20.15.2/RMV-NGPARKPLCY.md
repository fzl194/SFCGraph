# 删除园区策略（RMV NGPARKPLCY）

- [注意事项](#ZH-CN_MMLREF_0306399926__1.3.2)

## [命令功能](#ZH-CN_MMLREF_0306399926)

**适用NF：AMF**

该命令用于删除园区策略。

## [注意事项](#ZH-CN_MMLREF_0306399926)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0306399926)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0306399926)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PARKID | 园区标识 | 可选必选说明：必选参数<br>参数含义：该参数用于标识一个园区。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是1~32。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0306399926)

删除园区标识为“park”的园区策略，执行如下命令：

```
RMV NGPARKPLCY: PARKID="park";
```

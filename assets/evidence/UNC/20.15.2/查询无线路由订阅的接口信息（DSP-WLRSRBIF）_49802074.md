#### [命令功能](#ZH-CN_CONCEPT_0000001549802074)

该命令用于查询无线路由订阅的接口名称和索引对应关系。

#### [注意事项](#ZH-CN_CONCEPT_0000001549802074)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549802074)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549802074)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001549802074)

查询无线路由订阅的接口名称和索引对应关系：

```
DSP WLRSRBIF:;
```

```

RETCODE = 0  操作成功

结果如下
------------------------
  接口名称  =  Tunnel1
  接口索引  =  21
  引用计数  =  0
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001549802074)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 接口名称 | 用来表示无线路由订阅的接口名称。 |
| 接口索引 | 用来表示无线路由订阅的接口索引。 |
| 引用计数 | 用来表示接口被引流表引用的计数。 |

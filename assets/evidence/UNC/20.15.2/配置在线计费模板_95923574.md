## [操作场景](#ZH-CN_OPI_0295923574)

在线计费模板是用来绑定在线计费本地配置参数的，在线计费DCC模板分为全局通用参数模板dcc-global和普通的模板DCC Template，前者用于配置整机通用的在线计费参数，后者用于配置特殊场景下的在线计费参数。

在线计费参数中，除了OCS的基本信息是基于全局配置的之外，其他的参数，如要使用的OCS服务器、CCR触发场景、DCC处理动作、CCR消息中携带的参数等都可以选择基于整机或特殊场景生效。

GGSN/PGW-C支持以下几种粒度范围的在线计费参数配置：

- 支持配置整机通用参数。
- 支持根据用户计费属性Charging Characteristics配置在线计费参数。
- 支持根据APN配置在线计费参数。
- 支持根据UserProfile配置在线计费参数。

上述几种粒度范围的优先级从高到低依次是：UserProfile下的配置–〉APN下的配置–〉计费属性下的配置–〉整机通用参数。如果高级别的参数没有配置或者配置inherit，则依次向下继承低一级别的参数取值。整机通用参数绑定在dcc-global模板下，其他几种粒度都绑定在DCC Template下，并进而绑定其生效的范围：某一计费属性、APN、或者UserProfile。

在线计费参数的配置粒度，建议与 [配置离线/在线计费方式（GGSN/PGW-C）](../../WSFD-011201 支持离线计费/激活离线计费/配置离线_在线计费方式（GGSN_PGW-C）_95923455.md) 的配置粒度保持一致，即如果基于APN开启在线计费功能，同样基于APN配置在线计费参数，这样可以避免各配置参数生效范围冲突和不一致。一旦确定了整个系统的在线计费参数配置粒度，则所有在线计费参数的配置原则上都必须统一使用该粒度。

> **说明**
> 适用于 GGSN、 PGW-C。

## [操作流程](#ZH-CN_OPI_0295923574)

- 配置全局通用参数DCCGlobal模板，请执行[步骤 1](#ZH-CN_OPI_0295923574__step2)。
- 配置计费属性下的DCC Template模板，请执行[步骤 1](#ZH-CN_OPI_0295923574__step2)，[步骤 2](#ZH-CN_OPI_0295923574__step3)。
- 配置APN下的DCC Template模板，请执行[步骤 1](#ZH-CN_OPI_0295923574__step2)，[步骤 3](#ZH-CN_OPI_0295923574__step4)。
- 配置UserProfile下的DCC Template模板，请执行[步骤 1](#ZH-CN_OPI_0295923574__step2)，[步骤 4](#ZH-CN_OPI_0295923574__step5)。

## [操作步骤](#ZH-CN_OPI_0295923574)

1. 配置DCC Template模板。重复执行本步骤可增加多个DCC Template模板。
    a. 配置全局通用参数DCC模板，DCC模板名称。 “DCCTMPLTNAME” = “global”
      **MOD DCCTEMPLATE**
    b. 配置DCC Template模板，以及触发CCR的各参数、CCR消息中携带的参数。
      **ADD DCCTEMPLATE**
    c. 配置各计费参数，请见在线计费章节的后面小节。
          - [配置CCR触发场景](配置在线计费模板/配置CCR触发场景_95923545.md)
          - [配置CCR消息中携带的参数](配置在线计费模板/配置CCR消息中携带的参数_95923469.md)
          - [配置在线计费自动回切功能](配置在线计费模板/配置在线计费自动回切功能_95923555.md)
2. 配置DCC Template模板绑定到计费属性。
  **ADD GLBDCCTEMPLATE**
3. 配置DCC Template模板绑定APN实例。
  **SET APNCHARGECTRL**
4. 配置DCC Template模板绑定UserProfile实例。
    a. 指定UserProfile实例。
      **ADD USERPROFILE**
    b. 配置DCC Template模板绑定UserProfile实例。
      **SET USRPROFCHARGE**
    c. 配置UserProfileGroup。
      **ADD USRPROFGROUP**
    d. 将UserProfile绑定到UserProfileGroup。
      **ADD UPBINDUPG**
    e. 将UserProfileGroup绑定到APN实例下。
      **ADD APNUSRPROFG**

## [任务示例](#ZH-CN_OPI_0295923574)

任务描述

任务一：配置全局通用参数DCCGlobal模板。

任务二：配置计费属性下的DCC Template模板。

任务三：配置APN下的DCC Template模板。

任务四：配置UserProfile下的DCC Template模板。

脚本

//任务一：配置全局通用参数DCCGlobal模板。

```
MOD DCCTEMPLATE:DCCTMPLTNAME="global";
```

//任务二：配置计费属性下的DCC Template模板。

```
ADD DCCTEMPLATE:DCCTMPLTNAME="dcc-test";
```

```
ADD GLBDCCTEMPLATE:CHARGECHARACT=CHARGE_CHARACT,SPECIFICCCVALUE="0x0800",DCCTEMPLATENAME ="dcc-test";
```

//任务三：配置APN下的DCC Template模板。

```
ADD DCCTEMPLATE:DCCTMPLTNAME="dcc-test";
```

```
ADD APN: APN="apn-test";
```

```
SET APNCHARGECTRL:APN="apn-test",DCCTEMPLATE="dcc-test";
```

//任务四：配置UserProfile下的DCC Template模板。

```
ADD DCCTEMPLATE:DCCTMPLTNAME="dcc-test";
```

```
ADD USERPROFILE: USERPROFILENAME="up-test", UPTYPE=PCC;
```

```
SET USRPROFCHARGE: USRPROFNAME="up-test", DCCTEMPLATE="dcc-test";
```

```
ADD USRPROFGROUP:USERPROFGNAME="upg-test";
```

```
ADD UPBINDUPG:USERPROFGNAME="upg-test",UPBINDTYPE=DEFAULT,USERPROFILENAME="up-test";
```

```
ADD APNUSRPROFG:APN="apn-test",USERPROFGNAME="upg-test";
```

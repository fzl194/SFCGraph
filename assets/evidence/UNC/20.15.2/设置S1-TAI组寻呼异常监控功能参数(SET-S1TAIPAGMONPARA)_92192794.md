#### [命令功能](#ZH-CN_CONCEPT_0000002192192794)

![](设置S1 TAI组寻呼异常监控功能参数(SET S1TAIPAGMONPARA)_92192794.assets/notice_3.0-zh-cn_2.png)

检测参数设置不当，可能导致寻呼消息误流控或影响寻呼风暴场景下的流控效果。

**适用网元：MME**

该命令用于设置S1 TAI组对象寻呼异常监控功能的参数。针对局部区域（某个或某些TAI）下行寻呼风暴或因基站侧故障不回寻呼响应触发大量寻呼消息重发，导致寻呼成功率下降场景，MME业务侧进行寻呼流控，减少寻呼风暴持续时长，快速恢复寻呼成功率。

#### [注意事项](#ZH-CN_CONCEPT_0000002192192794)

- 此命令执行后立即生效。
- 被监控的S1 TAI组对象可以通过**[ADD PERFOBJRULE](../../../../../性能管理/性能对象规则管理/增加性能对象规则(ADD PERFOBJRULE)_72225873.md)**命令添加，也可以通过开启话统对象自动生成开关后自动生成。

#### [本地用户权限](#ZH-CN_CONCEPT_0000002192192794)

manage-ug；system-ug

#### [网管用户权限](#ZH-CN_CONCEPT_0000002192192794)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000002192192794)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SWITCH | TAI寻呼异常监控开关 | 可选必选说明：必选参数<br>参数含义：该参数用来设置是否开启S1 TAI寻呼异常监控功能。<br>数据来源：本端规划<br>取值范围：<br>“OFF（关闭）”：关闭<br>“ON（开启）”：开启<br>默认值：OFF（关闭）<br>配置原则：<br>当希望开启S1 TAI寻呼异常监控功能时，设置为"ON"；当希望关闭S1 TAI寻呼异常监控功能时，设置为"OFF"。 |
| PAGSUCRATDEFVAL | S1模式TAI组寻呼成功率初始值（%） | 可选必选说明：该参数在"SWITCH"配置为"ON"时为条件可选参数。<br>参数含义：该参数用于设置系统初始状态下S1模式TAI组寻呼成功率基准值。后续根据本初始值自学习S1模式TAI组寻呼成功率基准值。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是20~100，单位是百分比。<br>默认值：90<br>配置原则：<br>建议将系统稳态运行时“指定TAI组的S1模式寻呼成功率”指标值作为“S1模式TAI组寻呼成功率初始值”。 |
| MINPAGREQTHD | S1模式TAI组寻呼请求次数起始监控阈值 | 可选必选说明：该参数在"SWITCH"配置为"ON"时为条件可选参数。<br>参数含义：该参数用来设置最小业务量。在统计周期内，当实际业务量小于最小业务量时，TAI寻呼流控检测功能失效。业务量是指“指定TAI组的S1模式寻呼请求次数”的性能指标值。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是100~100000。单位：个/5分钟。<br>默认值：1000<br>执行命令并不输入该参数时，该参数保持系统当前配置不变，可通过<br>**[LST S1TAIPAGMONPARA](查询S1 TAI组寻呼异常监控功能参数(LST S1TAIPAGMONPARA)_27678301.md)**<br>查询当前参数配置值。<br>配置原则：<br>该参数配置为系统峰值运行情况下，话统指标“指定TAI组的S1模式寻呼请求次数”*1.5与1000取最大值。 |
| PAGRATEEXPTHD | S1模式TAI组寻呼成功率降幅阈值（%） | 可选必选说明：该参数在"SWITCH"配置为"ON"时为条件可选参数。<br>参数含义：该参数用于设置S1模式TAI组寻呼成功率降幅阈值。该参数用于与“S1模式TAI组二次寻呼请求次数增幅阈值”联合判断是否发生寻呼风暴。TAI组寻呼成功率降幅和TAI组二次寻呼请求次数增幅均超过阈值，则MME判定为发生寻呼风暴。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是1~80，单位是百分比。<br>默认值：5<br>配置原则：<br>参考系统稳态运行时，连续3天内同时一话统周期“S1模式TAI组寻呼成功率”指标降幅与5%的最大值，进行配置。 |
| PAGREQEXPTHD | S1模式TAI组二次寻呼请求次数增幅阈值（%） | 可选必选说明：该参数在"SWITCH"配置为"ON"时为条件可选参数。<br>参数含义：该参数用于设置S1模式TAI组二次寻呼请求次数增幅阈值。该参数用于与“S1模式TAI组寻呼成功率降幅阈值”联合判断是否发生寻呼风暴。TAI组寻呼成功率降幅和TAI组二次寻呼请求次数增幅均超过阈值，则MME判定为发生寻呼风暴。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是1~1000。<br>默认值：50<br>配置原则：<br>参考系统稳态运行时，连续3天内同时一话统周期“S1模式TAI组二次寻呼请求次数”指标增幅与50%的最大值，进行配置。 |
| PAGEXPTIMESTHD | TAI寻呼消息启控连续异常周期个数 | 可选必选说明：该参数在"SWITCH"配置为"ON"时为条件可选参数。<br>参数含义：该参数用于指定TAI对象实例连续寻呼异常周期个数。当异常周期数大于等于本参数值时，MME启动寻呼流控。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是1~5。<br>默认值：1<br>配置原则：<br>该参数表示几个连续5分钟检测周期，值越大越不容易触发将TAI组对象检测为异常，建议保持默认值配置。 |
| PAGRETRYTIMES | 寻呼最大重试次数 | 可选必选说明：该参数在"SWITCH"配置为"ON"时为条件可选参数。<br>参数含义：该参数用于指定在TAI对象处于寻呼风暴情况下，寻呼消息最大重试次数。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~1。<br>默认值：0<br>配置原则：<br>该参数值越小对指定TAI组的S1模式寻呼流控效果越明显，对指定TAI组的S1模式寻呼成功率影响越大。 |
| IMSPAGINGSW | 语音业务TAI寻呼流控开关 | 可选必选说明：该参数在"SWITCH"配置为"ON"时为条件可选参数。<br>参数含义：该参数用于设置在TAI组对象处于寻呼风暴情况下，是否对语音寻呼进行流控。<br>数据来源：本端规划<br>取值范围：<br>“OFF（关闭）”：关闭<br>“ON（开启）”：开启<br>默认值：OFF（关闭）<br>配置原则：<br>如果存在因语音寻呼导致的寻呼风暴场景，则开启该功能。否则关闭该功能。 |
| ALARMSW | 告警开关 | 可选必选说明：该参数在"SWITCH"配置为"ON"时为条件可选参数。<br>参数含义：该参数用于设置在触发TAI组对象寻呼流控时，是否上报告警。<br>数据来源：本端规划<br>取值范围：<br>“OFF（关闭）”：关闭<br>“ON（开启）”：开启<br>默认值：ON（开启）<br>配置原则：无。 |

#### [使用实例](#ZH-CN_CONCEPT_0000002192192794)

设置S1 TAI组寻呼异常监控开关为开启状态。

SET S1TAIPAGMONPARA: SWITCH=ON;

## [命令功能](#ZH-CN_MMLREF_0248332253)

该命令用于查询租约时长。

## [注意事项](#ZH-CN_MMLREF_0248332253)

无

#### [操作用户权限](#ZH-CN_MMLREF_0248332253)

G_1，管理员级别命令组

## [参数说明](#ZH-CN_MMLREF_0248332253)

无

## [使用实例](#ZH-CN_MMLREF_0248332253)

查询租约时长

```
LST MSETCDTTL;
RETCODE = 0  操作成功

结果如下
------------------------
租约时长  =  3
(结果个数 = 1)
```

## [输出结果说明](#ZH-CN_MMLREF_0248332253)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 租约时长 | 该参数用于指定主服务与HAFETCD服务之间的心跳超时时间，当多个服务使用HAFETCD服务选出的主服务与HAFETCD服务之间的心跳超过该设置的值时，则其他备服务中的任意一个服务切为主服务。 |

## [操作场景](#ZH-CN_OPI_0276015637)

本操作指导介绍在运行网络中激活基于信令面的数据传输的操作过程。

IoT终端数据收发频率低、数据包较小且多数情况为单向发送（例如智能抄表）。本特性是3GPP协议针对此类业务定义的控制面优化功能CP-CIoT（Control Plane Cellular Internet of Thing），实现了数据包在终端和MME之间通过NAS消息传递，在MME和SGW-U间通过S11-U口传递，上、下行数传过程中无需频繁建立E-RAB，从而提升数传效率，节省不必要的信令开销。

> **说明**
> 适用于MME。

## [操作步骤](#ZH-CN_OPI_0276015637)

1. 进入 “MML命令行-UNC” 窗口。
2. 配置CP-CIoT功能。
    a. 查询某指定用户范围是否配置了CP-CIoT功能。
      [**LST M2MPLCY**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/M2M管理/M2M策略参数配置/查询M2M策略参数(LST M2MPLCY)_26145766.md)
    b. **可选：**如果 [步骤a](#ZH-CN_OPI_0276015637__step_a) 未查询到该指定用户范围的相关配置，则执行此步骤，为该用户范围配置CP-CIoT功能。
      [**ADD M2MPLCY**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/M2M管理/M2M策略参数配置/增加M2M策略参数(ADD M2MPLCY)_72225443.md)
    c. **可选：**如果 [步骤a](#ZH-CN_OPI_0276015637__step_a) 查询到系统已存在该指定用户范围的相关配置，则执行此步骤，为该用户范围添加CP-CIoT功能。
      [**MOD M2MPLCY**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/M2M管理/M2M策略参数配置/修改M2M策略参数(MOD M2MPLCY)_72345365.md)
3. **可选：**配置某指定TAI下eNodeB的CP-CIoT能力。
    a. 查询某指定TAI下相关配置。
      [**LST TAICIOT**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/M2M管理/CIoT能力配置/查询CIoT能力配置(LST TAICIOT)_72225461.md)
    b. **可选：**如果 [步骤 3.a](#ZH-CN_OPI_0276015637__step_a1) 未查询到该指定TAI下的相关配置，则执行此步骤，为该TAI下的eNodeB配置CP-CIoT能力。
      [**ADD TAICIOT**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/M2M管理/CIoT能力配置/增加CIoT能力配置(ADD TAICIOT)_26305590.md)
    c. **可选：**如果 [步骤 3.a](#ZH-CN_OPI_0276015637__step_a1) 查询到系统已存在该指定TAI下的相关配置，且不支持CP-CIoT能力，则执行此步骤，为该TAI下的eNodeB添加CP-CIoT能力。
      [**MOD TAICIOT**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/M2M管理/CIoT能力配置/修改CIoT能力配置(MOD TAICIOT)_26145782.md)
4. **可选：**配置Control Plane Service Request流程CHR单据的上报。
  系统支持通过命令 [**SET CHRCFG**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/CHR管理/CHR配置/设置CHR配置（SET CHRCFG）_72225295.md) 和 [**SET CHRLOWPERFSVRSUB**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/低性能CHR服务器的单据订阅条件/设置低性能CHR服务器的单据订阅条件（SET CHRLOWPERFSVRSUB）_72345083.md) 配置在Control Plane Service Request流程成功或失败时是否上报CHR单据，配置过程参见 [激活CHR功能](../../../操作维护功能/WSFD-202001 CHR功能/WSFD-202001 CHR功能（适用于SGSN_MME_AMF）/激活CHR功能_50287962.md) 。
5. 打开本功能的License配置开关。
  [**SET LICENSESWITCH**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)

## [任务示例](#ZH-CN_OPI_0276015637)

任务描述

假设某行业用户APNNI为"HUAWEI1.com"，运营商为其分配了IMSI前缀为"123038101"的号段，配置本特性。

脚本

//配置CP-CIoT功能。

```
ADD M2MPLCY: SUBRANGE=IMSI_PREFIX, IMSIPRE="123038101", APNNI="HUAWEI1.com", LONGRAUTAUTMR=0,  CPOPSW=SUPPORT;
```

//配置eNodeB的CP-CIoT能力。

```
ADD TAICIOT: BEGINTAI="123037301", ENDTAI="123037302", NBDATATRANSSUPPORT=SUPPORT_CP-1;
```

//打开本功能的License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2DNAS01", SWITCH=ENABLE;
```

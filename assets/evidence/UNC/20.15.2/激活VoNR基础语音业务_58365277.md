## [操作场景](#ZH-CN_OPI_0000001158365277)

VoNR即Voice over NR，UE在NR覆盖下，采用VoNR技术直接在NR上拨打电话，专用QoS Flow也在NR上建立。通话过程中，如果UE移动到LTE网络覆盖下，EPC和5GC会负责把整个IMS DNN平滑到LTE中，并确保给终端分配的IP地址以及P-CSCF的地址不变，使得IMS并不感知。所以只要在NR上发起呼叫，无论通话中终端是否移动到LTE网络，均称为VoNR呼叫。

> **说明**
> 适用于AMF、SMF。

## [任务示例](#ZH-CN_OPI_0000001158365277)

任务描述

激活VONR基础语音业务特性。

脚本

**以下步骤适用于AMF。**

// 进入 “MML命令行-UNC” 窗口。

//校验漫游外网用户语音DNN，并配置对应的IMS语音支持策略，DNN为"IMS-123"的语音用户配置支持IMS语音业务。

```
ADD NGVOICEDEPLOY: SUBRANGE=FOREIGN_USER, AMFHOMO=SUPPORT, DCVOPS=SUPPORT, UES1MODE=ON, VOICEDNN="IMS-123", IWKEPSIND=ON;
```

//对外网用户以外的其他用户，配置支持IMS语音业务。

```
SET NGIMSVOPS: AMFHOMO=SUPPORT,DCVOPS=SUPPORT;
```

**以下步骤适用于SMF。**

进入 “MML命令行-UNC” 窗口。

//配置IMS网络APN。

```
ADD APN: APN="ims";
```

//修改IMS APN的参数。

```
SET APNIMSATTR: APN="ims", IMSSWITCH=ENABLE;
```

//添加P-CSCF组配置。

```
ADD PCSCFGROUP: GROUPNAME="vonr", IPVERSION=IPV4, ALLOCTYPE=LOCAL;
```

//添加P-CSCF地址配置。

```
ADD PCSCFIP: GROUPNAME="vonr", IPVERSION=IPV4, PCSCFIPV4="10.28.236.11", WEIGHT=10, ALLOCTYPE=LOCAL;
```

//增加IMSI和MSISDN号段。

```
ADD PCSCFIMSISDNSEG: SEGMENTNAME="vonr", SEGMENTTYPE=IMSI, SEGSTART="123031200100026", SEGEND="123031200100027";
```

//将APN绑定到P-CSCF组。

```
ADD PCSCFGRPBNDAPN: APN="ims", DEFAULTFLAG=DEFAULT, MPCSCFGRPIPV4="vonr";
```

//在APN下将某指定号段绑定到P-CSCF组。

```
ADD PCSCFGRPBNDAPN: APN="ims", DEFAULTFLAG=IMSI_MSISDN_SEG, IMSIMSISDNSEG="vonr", PRIORITY=1, MPCSCFGRPIPV4="vonr";
```

//设置EPS Fallback保护定时器时长，该参数配置时应大于基站侧EPS Fallback的保护定时器时长，并小于IMS的AAR定时器时长。

```
SET SMTIMER: TEPSFB=5;
```

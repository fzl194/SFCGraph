#### [命令功能](#ZH-CN_MMLREF_0000001126145498)

**适用网元：SGSN**

此命令用于查看2G分离非活动用户的配置参数。当用户通过附着或路由区更新流程接入到本SGSN后，如果在指定时长(大于 “非活动用户分离定时器(分)” 时长)没有PDP（Packet Data Protocol）激活，则认为该用户为非活动用户对用户进行分离操作；当系统将用户判断为非活动用户进行分离后，如果用户马上重新附着且与上次分离的时间间隔不超过配置的 “永久在线识别定时器长(秒)” 时长， 则该用户被定义为永久在线用户，后续系统不会对该用户进行分离非活动用户操作。SGSN发起分离非活动用户的流程，可以释放这些用户的空闲资源，以支持更多的用户。

#### [注意事项](#ZH-CN_MMLREF_0000001126145498)

该命令执行后立即生效。

#### [本地用户权限](#ZH-CN_MMLREF_0000001126145498)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126145498)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126145498)

无。

#### [使用实例](#ZH-CN_MMLREF_0000001126145498)

查询2G分离非活动用户配置参数：

LST GBDETACH:;

```
%%LST GBDETACH:;%%
RETCODE = 0  操作成功。

输出结果如下
--------------
                        非活动用户分离  =  否
              非活动用户分离定时器(分)  =  360
                             保留参数1  =  否
                             保留参数2  =  0
              永久在线识别定时器长(秒)  =  60
                          永久在线识别  =  否
          是否在分离响应超时后分离用户  =  是
(结果个数 = 1)

---    END
```

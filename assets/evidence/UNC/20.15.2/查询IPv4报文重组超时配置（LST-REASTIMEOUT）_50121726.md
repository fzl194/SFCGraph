#### [命令功能](#ZH-CN_CONCEPT_0000001550121726)

该命令用于查询IPv4分片报文的重组超时时间。

#### [注意事项](#ZH-CN_CONCEPT_0000001550121726)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550121726)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550121726)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001550121726)

查询IPv4报文重组超时配置：

```
LST REASTIMEOUT:;
```

```

        RETCODE = 0  操作成功

        结果如下
        -------------------------
        IPv4报文重组超时时间（s）  =  100
        (结果个数 = 1)
        ---   END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001550121726)

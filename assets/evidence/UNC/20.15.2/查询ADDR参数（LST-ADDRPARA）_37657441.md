## [命令功能](#ZH-CN_MMLREF_0000001237657441)

**适用NF：PGW-C、SMF、GGSN**

该命令用于查询ADDR参数，包括核查速率、去活间隔等参数。

## [注意事项](#ZH-CN_MMLREF_0000001237657441)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001237657441)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001237657441)

无

## [使用实例](#ZH-CN_MMLREF_0000001237657441)

该命令用于查询ADDRPARA配置： LST ADDRPARA:;

```
%%LST ADDRPARA:;%%
RETCODE = 0  操作成功

结果如下
--------------------
同步地址使用情况数据时间间隔(秒)  =  300
        转发地址申请最大次数(次)  =  2
     地址使用情况核查速率(个/秒)  =  2
 地址子段分配情况核查速率(个/秒)  =  2
             租约核查速率(个/秒)  =  2
             地址回收速率(个/秒)  =  6
            地址使用情况核查开关  =  是
        地址子段分配情况核查开关  =  是
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001237657441)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 同步地址使用情况数据时间间隔(秒) | 该参数用来设置地址分配管理节点实例向地址子段管理节点实例同步地址使用数据时间间隔。 |
| 转发地址申请最大次数(次) | 该参数用来设置地址分配管理节点实例间地址申请最大转发次数。 |
| 地址使用情况核查速率(个/秒) | 该参数用来设置每个DS每秒核查地址使用情况的地址池个数。 |
| 地址子段分配情况核查速率(个/秒) | 该参数用来设置每个DS每秒核查地址子段分配情况的地址池个数。 |
| 租约核查速率(个/秒) | 该参数用来设置每个DS的租约核查速率。 |
| 地址回收速率(个/秒) | 该参数用来设置每个DS每秒回收地址的个数。 |
| 地址使用情况核查开关 | 该参数用来控制是否开启会话管理节点向地址分配管理节点的地址核查功能。 |
| 地址子段分配情况核查开关 | 该参数用来控制是否开启地址分配管理节点向地址子段管理节点的地址子段核查功能。 |

#### [命令功能](#ZH-CN_MMLREF_0000001172226021)

![](设置信令网属性(SET SIGATTR)_72226021.assets/notice_3.0-zh-cn_2.png)

如果设置信令网属性后未立即复位相关进程，在后续的系统运行中如果发生部分进程的复位，将会导致系统内进程运行的信令网属性表现不同，导致业务中断。比较 [**DSP SIGATTR**](显示信令网属性状态(DSP SIGATTR)_72345943.md) 和 [**LST SIGATTR**](查询信令网属性(LST SIGATTR)_26306154.md) 命令查询的查询结果的一致性，可以排查SPP、SGP、LLP上实际运行的信令网属性是否异常。

**适用网元：SGSN、MME、SMSF**

此命令用于设置和修改信令网属性，信令网属性定义了本局支持的信令特性。

#### [注意事项](#ZH-CN_MMLREF_0000001172226021)

- 系统初次运行时，会执行系统初始设置值。
- 此命令执行后需要通过执行[**RST PROCESS**](../../../../../平台服务管理/单体服务维护功能管理/操作维护/系统调测/进程管理/复位进程(RST PROCESS)_29626893.md)重启所有SPP进程和所有SGP进程才能生效。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172226021)

#### [网管用户权限](#ZH-CN_MMLREF_0000001172226021)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [使用实例](#ZH-CN_MMLREF_0000001172226021)

设置国际网为14位，国内网为24位，支持BSSAP+功能，BSSAP+、MAP所在的信令网为国内网：

```
SET SIGATTR: IF=LABEL14, NF=LABEL24, BSSAPPF=ENABLE, BSSAPPNI=NAT, MAPNI=NAT;
```

## [命令功能](#ZH-CN_MMLREF_0000001146692903)

**适用NF：AMF、SMF**

该命令用于查询服务化接口负载控制参数。

## [注意事项](#ZH-CN_MMLREF_0000001146692903)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001146692903)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001146692903)

无

## [使用实例](#ZH-CN_MMLREF_0000001146692903)

查询服务化接口负载控制参数：

```
%%LST SBILCIPARA:;%%
RETCODE = 0  操作成功

结果如下
--------
服务化接口负载控制开关  =  关
       负载通知阈值(%)  =  80
       负载通知步长(%)  =  5
负载获取间隔时长(分钟)  =  5
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001146692903)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 服务化接口负载控制开关 | 该参数表示服务化接口负载控制的开关。 |
| 负载通知阈值(%) | 该参数表示负载通知的阈值，当服务端的服务化接口负载超过阈值，会通知客户端。 |
| 负载通知步长(%) | 该参数表示当服务端的服务化接口负载超过阈值通知客户端后，再次通知的步长。 |
| 负载获取间隔时长(分钟) | 该参数表示系统获取实时负载间隔，当网元注册成功后，会首次获取本网元实时负载，后续按照该参数的时间间隔周期性获取。 |

## [命令功能](#ZH-CN_MMLREF_0209652373)

**适用NF：AMF**

该命令用于查询AMF系统内各种用户状态的统计结果。

## [注意事项](#ZH-CN_MMLREF_0209652373)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209652373)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652373)

无

## [使用实例](#ZH-CN_MMLREF_0209652373)

查看AMF上的用户数据统计结果，执行如下命令：

```
%%DSP NGUSERNUM:;%%
RETCODE = 0  操作成功

结果如下
------------------------
POD ID    静态用户数  动态用户数  PDU会话数  POD版本号信息

usn-pod-0 0           0           0          22.2.0.B060  
total     0           0           0          -            
(结果个数 = 2)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209652373)

| 输出项名称 | 输出项解释 |
| --- | --- |
| POD ID | 该参数用于标识系统中的POD ID。 |
| 静态用户数 | 该参数用于标识系统POD中保存的静态MM上下文数量（包含签约数据等）。用户去注册后，用户的签约数据不会马上删除。用户签约数据删除的时间由SET NGMMPARA中“PURGTMR（用户分离后签约数据保留时间）”参数控制，默认设置为24小时。 |
| 动态用户数 | 该参数用于标识系统POD中处于注册态的用户数。 |
| PDU会话数 | 该参数用于标识系统POD中保存的PDU会话的数量。 |
| POD版本号信息 | 该参数用于指定pod版本号。非灰度升级期间，该参数不显示。 |

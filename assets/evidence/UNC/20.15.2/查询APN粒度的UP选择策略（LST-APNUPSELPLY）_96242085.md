## [命令功能](#ZH-CN_MMLREF_0296242085)

**适用NF：SMF、SGW-C、PGW-C、GGSN**

该命令用于查询APN粒度下的UP选择策略。

## [注意事项](#ZH-CN_MMLREF_0296242085)

无

#### [操作用户权限](#ZH-CN_MMLREF_0296242085)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0296242085)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| APN | APN名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定APN实例名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~63。不区分大小写。<br>默认值：无<br>配置原则：<br>该参数使用ADD APN命令配置生成。 |

## [使用实例](#ZH-CN_MMLREF_0296242085)

查询APN为test下的UP选择策略。

```
%%LST APNUPSELPLY: APN="test";%%
RETCODE = 0  操作成功

结果如下
--------
                            APN  =  test
 基于ULIForSGW位置选择SGW-U开关  =  关
基于位置条件优选主锚点UPF的开关  =  关
      分流场景下共享UPF优选开关  =  关
          基于优先级优选UPF开关  =  关
           合一与优先级优选策略  =  合一优先
        选择合一UPF的优先级策略  =  按TAI/TAIRANGE/SMFSERVINGAREA优先级优选
    位置区S11口与优先级优选策略  =  优先级优先
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0296242085)

| 输出项名称 | 输出项解释 |
| --- | --- |
| APN名称 | 该参数用于指定APN实例名。 |
| 基于ULI For SGW位置选择SGW-U开关 | 该参数用于控制SMF在进行SGW-U选择时，在ULI信元为空时是否优先使用ULI For SGW进行选择。 |
| 基于位置条件优选主锚点UPF的开关 | 该参数用于指示APN粒度下基于位置优选主锚点UPF开关。<br>当此开关为ENABLE时，SMF优先选择支持位置与当前用户所在位置相匹配的UPF作为主锚点。<br>主锚点包括PSA UPF、PGW-U、GGSN。 |
| 分流场景下共享UPF优选开关 | 该参数用于指示在该APN下，SMF在分流场景下是否优选共享UPF。该优选策略属于合一优先选择，生效顺位与合一优先选择相同。 |
| 基于优先级优选UPF开关 | 该参数用于指示该APN下SMF基于优先级选择UPF的功能是否开启。 |
| 合一与优先级优选策略 | 该参数用于指示UNC选择UPF时优先选择合一部署UPF还是优选优先级最高的UPF。 |
| 选择合一UPF的优先级策略 | 该参数用于指示在该命令“APN”参数所指定的APN条件下，UNC优选主锚点与接入点合一UPF的优先级策略。 |
| 位置区S11口与优先级优选策略 | 该参数用于指示UNC选择UPF时优先选择位置区\S11口UPF还是优选优先级高的UPF。 |

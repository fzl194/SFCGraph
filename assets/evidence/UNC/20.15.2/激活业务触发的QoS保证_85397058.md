## [操作步骤](#ZH-CN_OPI_0285397058)

1. 打开本特性的License配置开关。

## [任务示例](#ZH-CN_OPI_0285397058)

任务描述

SMF支持业务触发的QoS保证。指定业务的业务特征在PGW-U/UPF上配置， UNC 上需要进行如下配置（数据规划原则及含义请参见 **必备事项** ），才能将相应的预定义规则的Rule下发给PGW-U/UPF，并根据PGW-U/UPF上报的PFCP Session Report Report（QoS）触发专有QoS Flow/专有承载创建。QoS Flow/专有承载创建后，PGW-U/UPF通过业务识别，将指定的业务使用专有QoS Flow/专有承载传输，从而进行QoS保证。

网络中存在PCRF/PCF时，UPF、SMF、PCRF/PCF上配置的预定义规则名称必须一致，否则不能达到预期效果。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH:LICITEM="LKV3W9STQE11",SWITCH=ENABLE;
```

//配置QoS URR。

```
ADD URR: URRNAME="URR_qos_01", URRID=1005, USAGERPTMODE=QOS;
```

//（5G）配置业务特定的QoS信息。

```
ADD QOSPROP: QOSPROPNAME="qosprop_501", GBRUPLKVALUE=600, GBRDNLKVALUE=6000, MBRUPLKVALUE=1000, MBRDNLKVALUE=10000, QOSTYPE=QOS_FLOW_PARA, ARPVALUE=5, FQI=2, QOSURRNAME="URR_qos_01";
```

//（2/3/4G）配置业务特定的QoS信息。

```
ADD QOSPROP: QOSPROPNAME="qosprop_401", GBRUPLKVALUE=600, GBRDNLKVALUE=6000, MBRUPLKVALUE=1000, MBRDNLKVALUE=10000, QOSTYPE=QOS_BEARER_PARA, QCIVALUE=66, ARPVALUE=5, QOSURRNAME="URR_qos_01";
```

//（方式一，通过PCC规则实现）配置QoS属性、PCC策略组和规则的绑定关系。

```
ADD PCCPOLICYGRP:PCCPOLICYGRPNM="pccpolicy_group",QOSPROPNAME="qosprop_501";
ADD RULE:RULENAME="rule_test",POLICYTYPE=PCC,POLICYNAME="pccpolicy_group";
```

//（方式二，通过QoS规则实现）配置QoS属性与规则的绑定关系。

```
ADD RULE:RULENAME="rule_test",POLICYTYPE=QOS,POLICYNAME="qosprop_501";
```

//（无可用PCRF/PCF或预定义规则组时使用）配置UserProfile并绑定Rule。

```
ADD USERPROFILE:USERPROFILENAME="up_test",UPTYPE=PCC;
ADD RULEBINDING:USERPROFILENAME="up_test",RULENAME="rule_test",POLICYTYPE=QOS;
```

//（无可用的PCRF/PCF时使用）配置UserProfile组并和APN绑定。

```
ADD USRPROFGROUP:USERPROFGNAME="upg_test";
ADD UPBINDUPG:USERPROFGNAME="upg_test",UPBINDTYPE=DEFAULT,USERPROFILENAME="up_test";
ADD APNUSRPROFG:APN="huawei.com",USERPROFGNAME="upg_test";
```

//（可选）系统初始值不符合要求时，修改指定APN下的专有QoS Flow级别的空闲定时器时长。

```
SET APNIDLETIME: APN="huawei.com", INHERIT=NO, HSMFTIMERLEVEL=QOSFLOW, DEDQFIDLETIMER=600;
```

//（可选）UE无线连接丢失（NGAPCAUSEVALUE=21）时，配置基于APN去活用户面专有QoS Flow策略。

```
ADD APNDEACTQFPLCY: APN="huawei.com", NGAPCAUSEGROUP=RADIO_NETWORK, NGAPCAUSEVALUE=21, QFPLCY=DELAY_RELEASE, DELAYTIMER=60;
```

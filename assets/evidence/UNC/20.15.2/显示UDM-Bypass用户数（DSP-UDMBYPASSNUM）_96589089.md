## [命令功能](#ZH-CN_MMLREF_0000001296589089)

**适用NF：AMF**

该命令用于查询AMF系统内处于UDM Bypass状态的用户数。

## [注意事项](#ZH-CN_MMLREF_0000001296589089)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001296589089)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001296589089)

无

## [使用实例](#ZH-CN_MMLREF_0000001296589089)

查看AMF系统内处于UDM Bypass状态的用户数，执行如下命令：

```
%%DSP UDMBYPASSNUM:;%%
RETCODE = 0  操作成功

结果如下
------------------------
POD ID    UDM Bypass用户数

uncpod-0  10
total     10
(结果个数 = 2)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001296589089)

| 输出项名称 | 输出项解释 |
| --- | --- |
| POD ID | 该参数用于标识系统中的POD ID。 |
| UDM Bypass用户数 | 该参数用于标识系统POD中处于UDMBYPASS的用户数。 |

## [命令功能](#ZH-CN_MMLREF_0000001428093181)

**适用NF：PGW-C、GGSN**

该命令用来显示全局的QoS信息，包括漫游以及拜访用户的级别限制功能，以及限制级别。

## [注意事项](#ZH-CN_MMLREF_0000001428093181)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001428093181)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001428093181)

无

## [使用实例](#ZH-CN_MMLREF_0000001428093181)

查询整机的拜访以及漫游用户的级别：

```
LST USERPRIORARP:;

RETCODE = 0  操作成功。

拜访或漫游用户的级别信息
------------------------
限制漫游用户级别  =  使能
漫游用户最高级别  =  中
限制拜访用户级别  =  使能
拜访用户最高级别  =  中
(结果个数 = 1)
---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001428093181)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 漫游用户ARP级别限制 | 该参数用于表示漫游用户的ARP级别限制（将漫游用户的ARP级别限制为某一固定值，并非最高级别）。 |
| 拜访用户ARP级别限制 | 该参数用于表示拜访用户的ARP级别限制（将拜访用户的ARP级别限制为某一固定值，并非最高级别）。 |
| 漫游用户级别限制开关 | 该参数用于表明系统是否支持漫游用户级别限制。 |
| 拜访用户级别限制开关 | 该参数用于表明系统是否支持拜访用户级别限制。 |

# 删除支持的S-NSSAI（RMV CFGSNSSAI）

## [命令功能](#ZH-CN_MMLREF_0209654364)

**适用NF：NSSF**

该命令用于删除NSSF支持的S-NSSAI列表。

## [注意事项](#ZH-CN_MMLREF_0209654364)

- 该命令执行后立即生效。

- 主备或双活组网的场景下，如果需要配置此命令，则两个NSSF上均需执行此命令，且配置参数一致。

#### [操作用户权限](#ZH-CN_MMLREF_0209654364)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209654364)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| INDEX | 索引 | 可选必选说明：必选参数<br>参数含义：该参数用于描述命令的索引值。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~4294967295。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209654364)

假如运营商希望删除INDEX为1的记录，执行下列命令。

```
RMV CFGSNSSAI: INDEX=1;
```

## [命令功能](#ZH-CN_MMLREF_0000001823782758)

**适用NF：SMF**

该命令用于查询层二规则与层二规则组的绑定关系。

## [注意事项](#ZH-CN_MMLREF_0000001823782758)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001823782758)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001823782758)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| L2RULEGROUPNAME | 层二规则组名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定层二规则组名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~63。不支持空格，不区分大小写。<br>默认值：无<br>配置原则：<br>该参数使用ADD L2RULEGROUP命令配置生成。 |

## [使用实例](#ZH-CN_MMLREF_0000001823782758)

- 查询层二规则组名称为“rulegrp1”的绑定关系：
  ```
  LST L2RULEBINDGRP: L2RULEGROUPNAME="rulegrp1";
  RETCODE = 0  操作成功

  结果如下
  --------
  层二规则组名称  =  rulegrp1
  层二规则名称  =  rule1
  (结果个数 = 1)

  ---    END
  ```
- 查询系统中所有的层二规则与层二规则组的绑定关系：
  ```
  LST L2RULEBINDGRP:;
  RETCODE = 0  操作成功

  结果如下
  --------
  层二规则组名称  =  rulegrp1
  层二规则名称  =  rule1
  (结果个数 = 1)

  ---    END
  ```

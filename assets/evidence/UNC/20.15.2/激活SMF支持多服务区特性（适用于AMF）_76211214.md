> 适用于AMF。

任务描述

打开开关，AMF发现SMF时，支持使用的smfInfoList信元匹配。根据AMF对SMF的服务发现，是经由NRF进行的，还是基于本地服务进行的，分为两种场景进行配置举例。

脚本

**场景一. 经由NRF发现SMF**

//打开开关，AMF发现SMF时，支持使用的smfInfoList信元匹配。

```
SET NFDISCPLCY: SMFINFOLISTPLCY=RESTRICTED;
```

**场景二. 本地配置目标SMF**

SMF1、SMF2均支持TAC1（345012000001）、TAC2（345012000002）、TAC3（345012000003）、TAC4（345012000004）、TAIrange11（345012000011~345012000019）、TAIrange21（345012000021~345012000029），新用户从TAC1、TAC3、TAIrange11接入时优选SMF1，从TAC2、TAC4、TAIrange21接入时优选SMF2。

//打开开关，AMF发现SMF时，支持使用的smfInfoList信元匹配。

```
SET NFDISCPLCY: CFGTAIPRISW=ON;
```

//配置SMF1相关信息。

```
ADD PNFPROFILE: NFINSTANCEID="smf1", NFTYPE=NfSMF, NFSTATUS=Registered, PRIORITY=1;
ADD PNFTAI: NFINSTANCEID="smf1", TAI="345012000001", PRISWITCH=INHERIT, CAPSWITCH=INHERIT;
ADD PNFTAI: NFINSTANCEID="smf1", TAI="345012000003", PRISWITCH=INHERIT, CAPSWITCH=INHERIT;
ADD PNFTAI: NFINSTANCEID="smf1", TAI="345012000002", PRISWITCH=SPECIFIC, PRIORITY=10, CAPSWITCH=INHERIT;
```

# 修改MNC长度信息（MOD MNCLEN）

- [注意事项](#ZH-CN_MMLREF_0209651551__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0209651551__1.3.4)

## [命令功能](#ZH-CN_MMLREF_0209651551)

**适用NF：SMF**

该命令用于修改指定MCC号对应的MNC长度。

## [注意事项](#ZH-CN_MMLREF_0209651551)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0209651551)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209651551)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| MCC | 移动国家码 | 可选必选说明：必选参数<br>参数含义：该参数用于指定移动国家码。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度是3。取值范围000～999。<br>默认值：无<br>配置原则：无 |
| MNCLEN | 对应MCC的MNC长度 | 可选必选说明：必选参数<br>参数含义：该参数用于指定MCC值所对应的MNC长度。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是2~3。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209651551)

修改当前UNC所属的MCC为460，MNC长度为3：

```
MOD MNCLEN:MCC="460",MNCLEN=3;
```

# 删除基于ARP的Paging消息在流控期间放通的优先级（RMV NGPAGINGARPPRI）

## [命令功能](#ZH-CN_MMLREF_0000001198627535)

**适用NF：AMF**

该命令用于删除基于ARP的Paging消息在流控期间放通的优先级。删除后该ARP对应的Paging消息变为低优先级。

## [注意事项](#ZH-CN_MMLREF_0000001198627535)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0000001198627535)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001198627535)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| ARP | ARP数值 | 可选必选说明：必选参数<br>参数含义：该参数用于指定ARP。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是1~15。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001198627535)

删除ARP为1的Paging消息在流控期间放通的优先级，执行如下命令：

```
RMV NGPAGINGARPPRI: ARP=1;
```

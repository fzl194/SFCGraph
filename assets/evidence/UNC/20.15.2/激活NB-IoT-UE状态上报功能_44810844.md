## [操作场景](#ZH-CN_OPI_0000001444810844)

本操作指导介绍在运行网络中激活NB-IoT UE状态上报功能的操作过程。

> **说明**
> 适用于MME。

## [对系统的影响](#ZH-CN_OPI_0000001444810844)

该操作对系统正常运行无影响。

## [必备事项](#ZH-CN_OPI_0000001444810844)

前提条件

- 操作人员已经登录U2020\MAE-Access。
- eNodeB的订阅已经具备，具体请参见该特性页面的应用场景部分描述。
- 已经加载支持该特性的License，对应的License项为“82208412 LKV2USRF01 NB-IoT UE状态上报功能”。
- 请仔细阅读[WSFD-011603 NB-IoT UE状态上报功能特性概述](WSFD-011603 NB-IoT UE状态上报功能特性概述_95011277.md)。

## [操作步骤](#ZH-CN_OPI_0000001444810844)

1. 配置支持NB-IoT UE状态上报功能特性。
    a. 查询系统中Diameter兼容性配置
      ![](激活NB-IoT UE状态上报功能_44810844.assets/notice_3.0-zh-cn_2.png)

      命令 [**ADD DMCMPTBYIMSI**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性IMSI号段配置/增加IMSI对应的Diameter兼容性(ADD DMCMPTBYIMSI)_72225977.md) 与 [**SET DMCMPT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性配置/设置Diameter兼容性(SET DMCMPT)_26306080.md) 中除 “用户范围” 、 “运营商标识” 以及 “IMSI前缀” 外的其余参数功能均相同，且若系统中同时存在这两条命令的配置，系统会优先匹配命令 [**ADD DMCMPTBYIMSI**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性IMSI号段配置/增加IMSI对应的Diameter兼容性(ADD DMCMPTBYIMSI)_72225977.md) 的配置。因此在配置 [**ADD DMCMPTBYIMSI**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性IMSI号段配置/增加IMSI对应的Diameter兼容性(ADD DMCMPTBYIMSI)_72225977.md) 前需先执行 [**LST DMCMPT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性配置/查询Diameter兼容配置(LST DMCMPT)_72345869.md) 查询系统中 [**SET DMCMPT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性配置/设置Diameter兼容性(SET DMCMPT)_26306080.md) 的配置记录，确保命令 [**ADD DMCMPTBYIMSI**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性IMSI号段配置/增加IMSI对应的Diameter兼容性(ADD DMCMPTBYIMSI)_72225977.md) 的配置记录中所携带的参数默认值不影响系统已有配置。
      [**LST DMCMPT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性配置/查询Diameter兼容配置(LST DMCMPT)_72345869.md) : ;
    b. 根据HSS是否支持本特性，配置NB-IoT UE状态上报功能的兼容性。
          - 所有HSS都支持此特性：
            [**SET DMCMPT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性配置/设置Diameter兼容性(SET DMCMPT)_26306080.md) : SF2=特性列表, NBIOTSTAT=支持上报的状态列表, T6SF=T6A接口特性列表;
          - 本网所有的HSS支持此特性，外网所有HSS不支持此特性：
            [**ADD DMCMPTBYIMSI**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性IMSI号段配置/增加IMSI对应的Diameter兼容性(ADD DMCMPTBYIMSI)_72225977.md) : SUBRANGE=HOME_USER, NOID=运营商标识, SF2=特性列表, NBIOTSTAT=支持上报的状态列表, T6SF=T6A接口特性列表;
          - 本网所有的HSS不支持此特性，外网所有HSS支持此特性：
            [**ADD DMCMPTBYIMSI**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性IMSI号段配置/增加IMSI对应的Diameter兼容性(ADD DMCMPTBYIMSI)_72225977.md) : SUBRANGE=FOREIGN_USER, NOID=运营商标识, SF2=特性列表, NBIOTSTAT=支持上报的状态列表, T6SF=T6A接口特性列表;
          - 特定IMSI前缀的用户注册的HSS支持此特性：
            [**ADD DMCMPTBYIMSI**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性IMSI号段配置/增加IMSI对应的Diameter兼容性(ADD DMCMPTBYIMSI)_72225977.md) : SUBRANGE=IMSI_PREFIX, IMSIPRE=IMSI前缀, SF2=特性列表, NBIOTSTAT=支持上报的状态列表, T6SF=T6A接口特性列表;
2. （可选）若对端MME支持NB-IoT能力开放订阅信息信元传递，需要配置对端MME的属性信息。
  [**ADD MMECHARACT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/GnGp-SGSN_S10_S16_S3接口管理/MME属性/增加MME属性配置信息（ADD MMECHARACT）_26305766.md) : RANGE=对端设备范围, IPTYPE=IP地址类型, IPV4=MME IPv4信令面地址, MASKV4=IPv4掩码, MNTREVTINFO=NB-IoT状态上报订阅信息;
3. 打开本功能的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md) : LICITEM="LKV2USRF01", SWITCH=ENABLE;

## [任务示例](#ZH-CN_OPI_0000001444810844)

任务描述

假设本网HSS支持本特性，其他用户的归属HSS不支持本特性。部署本特性。

脚本

//本网用户以外的其他用户的归属HSS都不支持本特性。

```
SET DMCMPT: SF2=FLSTID2_BIT19-0;
```

//查询系统中Diameter兼容性配置。

```
LST DMCMPT:;
```

//设置本网HSS支持NB-IoT UE状态上报，支持上报配置中的所有上报列表，需确保命令ADD DMCMPTBYIMSI中其他参数所携带的默认值不影响系统已有的Diameter兼容性配置。

```
ADD DMCMPTBYIMSI: SUBRANGE=HOME_USER, NOID=0, SF2=FLSTID2_BIT19-1, NBIOTSTAT=UEREACH-1&UELOSTCONN-1&UECOMMFAIL-1&AVAILAFTDDNFAIL-1, T6SF=T6FLID_BIT0-1;
```

//可选，设置对端支持NB-IoT 能力上报信息信元。

```
ADD MMECHARACT: RANGE=SPECIAL_MME, IPTYPE=IPV4, IPV4="10.22.168.12", MASKV4="255.255.255.255", MNTREVTINFO=YES;
```

//打开本功能的License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2USRF01", SWITCH=ENABLE;
```

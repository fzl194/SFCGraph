# 删除AMF服务列表（RMV AMFSERVICELIST）

- [注意事项](#ZH-CN_MMLREF_0209652985__1.3.2)

## [命令功能](#ZH-CN_MMLREF_0209652985)

**适用NF：AMF**

本命令删除特定AMF功能实体服务名。本命令执行影响北向网管对接功能。

## [注意事项](#ZH-CN_MMLREF_0209652985)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0209652985)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652985)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SERVICENAME | AMF服务名称 | 可选必选说明：必选参数<br>参数含义：特定AMF功能实例的服务名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~64。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209652985)

删除指定的AMF实例服务名称：

```
RMV AMFSERVICELIST: SERVICENAME="namfComm";
```

# 删除EPS签约QoS配置（RMV EPSSUBQOS）

- [注意事项](#ZH-CN_MMLREF_0209653729__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0209653729__1.3.4)
- [使用实例](#ZH-CN_MMLREF_0209653729__1.3.5)

## [命令功能](#ZH-CN_MMLREF_0209653729)

**适用NF：SGW-C、PGW-C**

该命令用来删除用户的签约QoS属性的相关配置。

## [注意事项](#ZH-CN_MMLREF_0209653729)

该命令执行后只对新激活用户生效。

#### [操作用户权限](#ZH-CN_MMLREF_0209653729)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209653729)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SUBQOSINDEX | 用户QoS索引 | 可选必选说明：必选参数<br>参数含义：该参数表示用户QoS索引。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~255。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209653729)

删除“SUBQOSINDEX”为“123”的记录：

```
RMV EPSSUBQOS: SUBQOSINDEX=123;
```

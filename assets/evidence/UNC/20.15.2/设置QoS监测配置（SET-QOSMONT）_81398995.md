## [命令功能](#ZH-CN_MMLREF_0000001181398995)

**适用NF：SMF**

该命令用于设置QoS监测配置。

## [注意事项](#ZH-CN_MMLREF_0000001181398995)

- 该命令执行后立即生效。

- 系统部署完成后，已经存在初始记录，参数的初始记录值如下表：

| MONTSWITCH |
| --- |
| DISABLE |

#### [操作用户权限](#ZH-CN_MMLREF_0000001181398995)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001181398995)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| MONTSWITCH | QoS监测开关 | 可选必选说明：必选参数<br>参数含义：该参数用于配置QoS监测开关。<br>数据来源：本端规划<br>取值范围：<br>- DISABLE（不使能）<br>- ENABLE（使能）<br>默认值：无。<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001181398995)

QoS监测开关配置为ENABLE:

```
SET QOSMONT: MONTSWITCH=ENABLE;
```

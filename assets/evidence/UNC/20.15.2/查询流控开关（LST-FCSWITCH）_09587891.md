## [命令功能](#ZH-CN_MMLREF_0209587891)

该命令用于查询整系统的流控功能禁用/启用状态。

## [注意事项](#ZH-CN_MMLREF_0209587891)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209587891)

G_1，管理员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209587891)

无

## [使用实例](#ZH-CN_MMLREF_0209587891)

查询流控功能是否启用。

```
LST FCSWITCH:;
RETCODE = 0  操作成功

结果如下
------------------------
开关  =  启用
(结果个数 = 1)
```

## [输出结果说明](#ZH-CN_MMLREF_0209587891)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 开关 | 该参数用于配置流控功能的禁用/启用状态。 |

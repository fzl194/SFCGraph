## [命令功能](#ZH-CN_MMLREF_0263673349)

该命令用于显示UCF文件管理的规格配置信息。

## [注意事项](#ZH-CN_MMLREF_0263673349)

执行命令前请确认UCF服务处于上线状态。

#### [操作用户权限](#ZH-CN_MMLREF_0263673349)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0263673349)

无

## [使用实例](#ZH-CN_MMLREF_0263673349)

运营商A查询UCF的文件管理规格配置信息：

```
%%LST UCFCONFIG:;%%
RETCODE = 0  操作成功

结果如下
--------
缓存文件存储空间大小(GB)  =  20
    文件最长写入时间(秒)  =  300
    文件最大写入大小(MB)  =  100
    文件最长存留时间(天)  =  7
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0263673349)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 缓存文件存储空间大小(GB) | UCF本地缓存文件存储的最大磁盘空间（单位：GB），SFTP、FTP、TCP协议对应的缓存文件共用该磁盘空间。 |
| 文件最长写入时间(秒) | UCF本地缓存文件最长写入时间（单位：秒），该参数仅对SFTP和FTP协议类型的缓存文件生效。 |
| 文件最大写入大小(MB) | UCF本地缓存文件最大写入大小（单位：MB），该参数仅对SFTP和FTP协议类型的缓存文件生效。当缓存文件使用FTP协议类型时，该参数设置大于50时，实际该参数生效为50；小于50时，实际输入参数值生效。 |
| 文件最长存留时间(天) | UCF本地缓存文件最长存留时间（单位：天）。 |

#### [相关概念](#ZH-CN_TOPIC_0193181078)

- 外网用户：不属于本运营商的用户。例如，运营商A的用户使用运营商B的网络，对运营商B来说，此用户就是外网用户。
- 本网用户：属于本运营商的用户。例如，运营商A的用户使用运营商A的网络，对运营商A来说，此用户就是本网用户。本网用户包括本网异地用户和本网本地用户。
  > **说明**
  > 在MVNO场景下，虚拟运营商用户也属于本网用户。
- 本网异地用户：属于本运营商但来自其他地区的用户。例如，来自地区1的运营商A的用户使用属于运营商A但是在地区2的网络，对地区2的运营商A网络来说，此用户就是本网异地用户。
- 本网本地用户：属于本运营商且本地区的用户。例如，运营商A地区1的用户使用运营商A地区1的网络，对运营商A地区1的网络来说，此用户就是本网本用户。
- 漫游用户: 漫游用户包括外网用户和本网异地用户。
- HLR号码：用于标示一个HLR实体。
  HLR号码采用E.164（International PSTN/ISDN numbers）编码方式，由CC（Country Code）、NDC（National Destination Code）、LSP（Locally Significant Part）组成。其中，CC为国家码，例如中国为86；NDC为国内目的码，即电话号码的前缀码；LSP为本地有效部分，为运营商规划在消息中使用的、代表HLR的号码。

#### [用户属地识别](#ZH-CN_TOPIC_0193181078)

UNC 进行用户属地识别的具体流程如下：

1. UNC 根据用户的IMSI获取用户所属的PLMN，判断用户所属网络。
  UNC 判断用户所属的PLMN（MCC + MNC）是否为 UNC 通过命令 [**ADD HPLMN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/归属网络运营商管理/MNO管理/MNO网络配置表/增加本地PLMN(ADD HPLMN)_26146074.md) 配置的MCC和MNC，或者通过命令 [**ADD MVNONET**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/归属网络运营商管理/MVNO管理/MVNO网络标识配置表/增加MVNO网络配置信息(ADD MVNONET)_72345663.md) 配置的MCC和MNC。
    - 是 => 判断该用户为本网用户，执行步骤[2](#ZH-CN_TOPIC_0193181078__step2)以进一步对用户属地进行识别。
    - 否 => 判断该用户为外网用户。
2.
  UNC 查看是否在用户附着时获取到了用户所属HLR的HLR号码。

  > **说明**
  > 在用户附着的时候， UNC 通过Update Location流程向用户所属的HLR请求获取HLR号码，用于此处进行用户属地的识别。
    - 是 => 执行步骤[3](#ZH-CN_TOPIC_0193181078__step-3)，进一步对用户归属进行识别。
    - 否 => 无法根据用户所属HLR的HLR号码进行进一步的用户归属判断，仅判断该用户为本网用户。
3. UNC 对用户属地进行进一步识别，查看用户所属HLR的HLR号码是否在命令 [**ADD LOCALHLR**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/LOCALHLR管理/增加本地HLR(ADD LOCALHLR)_72225755.md) 配置的HLR号码列表中。
    - 是 => 判断该用户为本网本地用户。
    - 否 => 判断该用户为本网异地用户。

#### [本地配置QoS的选择过程](#ZH-CN_TOPIC_0193181078)

UNC 在获得用户属地信息后，根据用户属地选择不同的本地配置QoS进行QoS协商，以根据用户归属进行精细化的QoS控制。

不同粒度的本地配置QoS的优先级从高到低顺序如下：指定IMSI号段QoS、指定用户属地QoS、所有用户QoS。即，如果用户同时匹配了多个粒度的本地配置QoS，则选择优先级最高的本地配置QoS参与QoS协商。

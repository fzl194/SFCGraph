#### [命令功能](#ZH-CN_CONCEPT_0000001600866665)

该命令用于查询LSPV全局属性。

#### [注意事项](#ZH-CN_CONCEPT_0000001600866665)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600866665)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600866665)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001600866665)

查询LSPV全局配置：

```
LST LSPVGLOBALCFG:;
```

```

RETCODE = 0  操作成功。

结果如下
--------
     LSPV响应端  =  使能
最大速率（pps）  =  1000
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600866665)

| 输出项名称 | 输出项解释 |
| --- | --- |
| LSPV响应端 | LSPV响应端是否使能。 |
| 最大速率（pps） | LSPV报文防攻击的限速值。 |

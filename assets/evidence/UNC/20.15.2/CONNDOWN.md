#### [命令功能](#ZH-CN_CONCEPT_0000001600601185)

该命令用于显示Diameter连接中断信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600601185)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600601185)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600601185)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001600601185)

显示Diameter连接中断信息：

```
DSP CONNDOWN:;
```

```

RETCODE = 0 操作成功

结果如下
------------------------
本地角色  本端地址         远端地址          连接ID          断连原因            断连时间               创建时间

服务端    10.1.1.10        10.1.1.9          3               客户端删除          2017-01-24 03:00:56    2017-01-24 02:55:50
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600601185)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 本地角色 | 该参数用于标识Diameter角色，取值是server或client。 |
| 本端地址 | 该参数用于标识本地IP地址。 |
| 远端地址 | 该参数表示当本地为client时的远端服务器地址。 |
| 连接ID | 该参数用于标识连接ID。 |
| 断连原因 | 该参数用于表示连接down的原因。包括：serverDelete：服务端删除；clientDelete：客户端删除；keepaliveTimeout：保活超时。 |
| 断连时间 | 该参数表示down事件产生的时间。 |
| 创建时间 | 该参数表示连接创建时间。 |

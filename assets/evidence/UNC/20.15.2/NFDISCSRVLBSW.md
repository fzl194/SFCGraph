## [命令功能](#ZH-CN_MMLREF_0000001357337229)

**适用NF：AMF、SMF、NSSF、SMSF、NCG**

该命令用于查询服务发现时的NF Service负载均衡功能。

## [注意事项](#ZH-CN_MMLREF_0000001357337229)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001357337229)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001357337229)

无

## [使用实例](#ZH-CN_MMLREF_0000001357337229)

查询服务发现时的NF Service负载均衡功能。

```
%%LST NFDISCSRVLBSW:;%%
RETCODE = 0  操作成功

结果如下
------------------------
对端NF类型    =  NfUDM
负载均衡开关  =  ON
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001357337229)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 对端NF类型 | 该参数用于指定对端NF类型。 |
| 负载均衡开关 | 该参数用于指定是否开启NF Service负载均衡功能。 |

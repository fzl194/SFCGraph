## [命令功能](#ZH-CN_MMLREF_0000001135636457)

**适用NF：PGW-C、SMF、GGSN**

该命令用于查询位置区域分配地址用户白名单。

## [注意事项](#ZH-CN_MMLREF_0000001135636457)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001135636457)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001135636457)

无

## [使用实例](#ZH-CN_MMLREF_0000001135636457)

查询位置区域分配地址用户白名单：

```
LST SMFADDRLOCWL:;
RETCODE = 0  操作成功。

结果如下
----------------------
MSISDN  =  123456
(结果个数 = 1)
---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001135636457)

| 输出项名称 | 输出项解释 |
| --- | --- |
| MSISDN | 该参数用于指定用户的MSISDN信息。 |

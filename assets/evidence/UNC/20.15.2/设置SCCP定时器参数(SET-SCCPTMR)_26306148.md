#### [命令功能](#ZH-CN_MMLREF_0000001126306148)

**适用网元：SGSN、MME、SMSF**

此命令用于设置SCCP定时器表中各个定时器的值。

#### [注意事项](#ZH-CN_MMLREF_0000001126306148)

- 系统初次运行时，会执行系统初始设置值。
- 此命令执行后立即生效。
- TIAS和TIAR两个参数需要与RNC协商，否则会出现用户被去活情况。具体规则：
    - 远端Tias的最小值要小于本端Tiar。
    - 远端Tiar的最小值要大于本端Tias。

#### [本地用户权限](#ZH-CN_MMLREF_0000001126306148)

manage-ug；system-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126306148)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126306148)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| TCONNEST | 连接建立定时器（s） | 可选必选说明：可选参数<br>参数含义：该参数用于指定连接建立定时器，用于等待连接确认消息。<br>数据来源：整网规划。<br>取值范围：60s～120s<br>系统初始设置值：90s |
| TIAS | 不活动性发送定时器（s） | 可选必选说明：可选参数<br>参数含义：该参数用于指定不活动性发送定时器，用于在连接会话中，没有消息可发送时，延迟发送消息。<br>数据来源：对端协商。<br>取值范围：60s～600s<br>系统初始设置值：450s |
| TIAR | 不活动性接收定时器（s） | 可选必选说明：可选参数<br>参数含义：该参数用于指定不活动性接收定时器，用于在连接会话中等待接收消息。<br>数据来源：对端协商。<br>取值范围：180s～1260s<br>系统初始设置值：1000s |
| TREL | 连接释放定时器（s） | 可选必选说明：可选参数<br>参数含义：该参数用于指定连接释放定时器，用于等待释放完成消息。<br>数据来源：整网规划。<br>取值范围：10s～20s<br>系统初始设置值：15s |
| TREPEATREL | 连接重复释放定时器（s） | 可选必选说明：可选参数<br>参数含义：该参数用于指定连接重复释放定时器，用于等待释放完成消息，或者在初始TREL定时器超时后重新发送释放消息。<br>数据来源：整网规划。<br>取值范围：10s～20s<br>系统初始设置值：15s |
| TINTERVAL | 连接释放外围定时器（s） | 可选必选说明：可选参数<br>参数含义：该参数用于指定连接释放外围定时器，用于等待释放完成消息，或者用于释放连接资源、冻结LRN，以及在初始TREL定时器超时后产生维护功能告警。<br>数据来源：整网规划。<br>取值范围：0s～60s<br>系统初始设置值：60s |
| TGUARD | 保护定时器（s） | 可选必选说明：可选参数<br>参数含义：该参数用于指定保护定时器，用于在重启进程时等待恢复临时连接会话。<br>数据来源：整网规划。<br>取值范围：1380s～1500s<br>系统初始设置值：1440s |
| TRESET | 复位定时器（s） | 可选必选说明：可选参数<br>参数含义：该参数用于指定复位定时器，用于等待释放临时连接会话，或者在复位请求消息发送后产生维护功能告警。<br>数据来源：整网规划。<br>取值范围：10s～20s<br>系统初始设置值：15s |
| TREASSEMBLY | 重装定时器（s） | 可选必选说明：可选参数<br>参数含义：该参数用于指定重装定时器，用于在接收消息的第一个分片后等待接收该消息的所有其余分片。<br>数据来源：整网规划。<br>取值范围：10s～20s<br>系统初始设置值：15s |
| TSTATINFO | 子系统测试定时器（s） | 可选必选说明：可选参数<br>参数含义：该参数用于指定子系统测试定时器。<br>数据来源：整网规划。<br>取值范围：5s～1200s<br>系统初始设置值：60s |
| TCOORDCHG | 协调定时器（s） | 可选必选说明：可选参数<br>参数含义：该参数用于指定协调定时器。<br>数据来源：整网规划。<br>取值范围：60s～120s<br>系统初始设置值：90s |
| TIGNORESST | 忽略子系统测试定时器（s） | 可选必选说明：可选参数<br>参数含义：该参数用于指定忽略子系统测试定时器。<br>数据来源：整网规划。<br>取值范围：5s～3600s<br>系统初始设置值：60s |
| TFREEZEN | 资源冻结定时器（s） | 可选必选说明：可选参数<br>参数含义：该参数用于指定资源冻结定时器。<br>数据来源：整网规划。<br>取值范围：4s～120s<br>系统初始设置值：30s |
| TEA | EA定时器（s） | 可选必选说明：可选参数<br>参数含义：该参数用于指定EA定时器。<br>数据来源：整网规划。<br>取值范围：1s～60s<br>系统初始设置值：10s |
| TLRN | LRN冻结定时器（s） | 可选必选说明：可选参数<br>参数含义：该参数用于指定LRN冻结定时器。<br>数据来源：整网规划。<br>取值范围：4s～60s<br>系统初始设置值：15s |
| TRESOURCE | 资源核查定时器（s） | 可选必选说明：可选参数<br>参数含义：该参数用于指定资源核查定时器。<br>数据来源：整网规划。<br>取值范围：60s～86400s<br>系统初始设置值：3600s |

#### [使用实例](#ZH-CN_MMLREF_0000001126306148)

设置SCCP定时器表：

SET SCCPTMR: TCONNEST=61, TIAS=301, TIAR=660, TREL=11, TREPEATREL=12, TINTERVAL=2, TGUARD=1381, TRESET=11, TREASSEMBLY=11, TSTATINFO=6, TCOORDCHG=61, TIGNORESST=6, TFREEZEN=6, TEA=2, TLRN=5, TRESOURCE=60;

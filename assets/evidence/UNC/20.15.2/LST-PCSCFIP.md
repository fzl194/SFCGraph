## [命令功能](#ZH-CN_MMLREF_0209653781)

**适用NF：SMF、PGW-C、GGSN**

该命令用于查询P-CSCF地址信息。

## [注意事项](#ZH-CN_MMLREF_0209653781)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209653781)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209653781)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| GROUPNAME | P-CSCF组名 | 可选必选说明：可选参数<br>参数含义：该参数用于指定P-CSCF组的名字，在系统内唯一。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~63。只允许输入字母、数字、“.”、“_”和“-”。字母会被转换为小写字母进行存储和处理。<br>默认值：无<br>配置原则：<br>该参数使用ADD PCSCFGROUP命令配置生成。 |

## [使用实例](#ZH-CN_MMLREF_0209653781)

查询指定P-CSCF组的P-CSCF地址配置记录:

```
LST PCSCFIP: GROUPNAME="mygroup";
RETCODE = 0   操作成功

结果如下
------------------------
     P-CSCF组名  =  mygroup
     IP地址版本  =  IPV6
       IPv4地址  =  255.255.255.255
       IPv6地址  =  2001:DB8::1
           权重  =  1
 P-CSCF获取方式  =  本地分配
           锁定  =  FALSE
          结对ID = 1
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209653781)

| 输出项名称 | 输出项解释 |
| --- | --- |
| P-CSCF组名 | 该参数用于指定P-CSCF组的名字，在系统内唯一。 |
| IP地址版本 | 该参数用于指定P-CSCF地址类型。 |
| IPv4地址 | 该参数用于指定IPv4类型的P-CSCF地址。 |
| IPv6地址 | 该参数用于指定IPv6类型的P-CSCF地址。 |
| 权重 | 该参数用于指定该P-CSCF的能力，权重越大说明性能越高，被选中的概率就越大。 |
| P-CSCF获取方式 | 该参数用于指定p-cscf服务器地址分配方式。 |
| 锁定 | 该参数用于指定P-CSCF组或者P-CSCF IP是否锁定。 |
| 结对ID | 该参数用于指定P-CSCF IP的结对ID，结对ID相同的P-CSCF互为容灾。 |

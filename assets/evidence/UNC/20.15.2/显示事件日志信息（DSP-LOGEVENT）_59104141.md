#### [命令功能](#ZH-CN_CONCEPT_0259104141)

该命令用于显示事件日志信息。

#### [注意事项](#ZH-CN_CONCEPT_0259104141)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0259104141)

G_1，管理员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0259104141)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SERVICEINSTANCE | 服务实例 | 可选必选说明：必选参数<br>参数含义：该参数表示大颗粒服务实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，通过LST VNFC命令获取。<br>默认值：无<br>配置原则：只能填写通过LST VNFC命令查询到的管理代理标识。 |

#### [使用实例](#ZH-CN_CONCEPT_0259104141)

显示事件日志信息：

```
DSP LOGEVENT:
SERVICEINSTANCE="vnfc"
;
```


#### [输出结果说明](#ZH-CN_CONCEPT_0259104141)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 事件序列号 | 用于表示事件序列号。 |
| 事件ID | 用于表示事件ID。 |
| 事件级别 | 用于表示事件级别。 紧急：事件级别为紧急。 警报：事件级别为警报。 严重：事件级别为严重。 错误：事件级别为错误。 警告：事件级别为警告。 通知：事件级别为通知。 提示：事件级别为提示。 调试：事件级别为调试。 |
| 事件产生时间 | 用于表示事件产生时间。 |
| 事件描述信息 | 用于表示事件描述信息。 |

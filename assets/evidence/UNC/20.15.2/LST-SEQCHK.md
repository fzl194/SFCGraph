#### [命令功能](#ZH-CN_MMLREF_0000001172345633)

**适用网元：SGSN、MME**

此命令用于查询GTP序号检查开关的状态。

#### [注意事项](#ZH-CN_MMLREF_0000001172345633)

无。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172345633)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172345633)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172345633)

无。

#### [使用实例](#ZH-CN_MMLREF_0000001172345633)

查询GTP的序号检查开关：

LST SEQCHK:;

```
%%LST SEQCHK:;%%
RETCODE = 0  执行成功。

输出结果如下
------------------
 序号检查功能开关  =  关
(结果个数 = 1)

---    END
```

#### [输出结果说明](#ZH-CN_MMLREF_0000001172345633)

请参考命令 [**SET SEQCHK**](设置序号检查信息表(SET SEQCHK)_26305842.md) 的参数标识。

#### [命令功能](#ZH-CN_CONCEPT_0209897024)

**适用NF：SGW-C、PGW-C、SMF**

此命令用于查询CGIP测量对象。

#### [注意事项](#ZH-CN_CONCEPT_0209897024)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0209897024)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0209897024)

无。

#### [使用实例](#ZH-CN_CONCEPT_0209897024)

查询CGIP测量对象：

```
LST CGIP:;
```

```

RETCODE = 0  操作成功。

CGIP地址
--------
IP地址  =  192.168.1.12
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0209897024)

| 输出项名称 | 输出项解释 |
| --- | --- |

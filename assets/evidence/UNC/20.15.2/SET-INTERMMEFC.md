#### [命令功能](#ZH-CN_MMLREF_0000001172345763)

![](设置Inter-MME流控参数(SET INTERMMEFC)_72345763.assets/notice_3.0-zh-cn_2.png)

- 此命令需要华为技术支持人员指导下才能执行，请慎重使用。
- 该命令主要用于Inter-MME用户的接入控制，配置不当可能导致业务受损，建议保持系统初始设置值。

**适用网元：MME**

该命令用于设置Inter-MME接入流控功能的相关参数。

MME组Pool时，如果某个MME故障，则故障MME上的用户会接入到Pool内正常的MME上，或非Pool组网下，大量用户同时从其他区域移动到某些MME覆盖区域时，均可能对这些MME造成较大冲击，导致其过载。为了防止这些场景下MME间的新接入用户影响在网用户的正常业务， UNC 系统能够基于当前CPU负荷，自动调节非本MME用户的接入消息允许处理速率。这些消息包括：Attach Request、Service Request消息。 如果这些消息是针对“WSFD- 102004 基于VoLTE的优先语音服务”或者“WSFD- 102101 VoLTE紧急呼叫”，则不对该消息进行流控。

#### [参数说明](#ZH-CN_MMLREF_0000001172345763)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| FCSWITCH | Inter-MME接入流控功能开关 | 可选必选说明：可选参数<br>参数含义：该参数用于设置是否开启Inter-MME接入流控功能。<br>数据来源：整网规划<br>取值范围：<br>- OFF(关闭)<br>- ON(开启)<br>系统初始设置值：<br>“ON(开启)”<br>。<br>配置原则：推荐系统初始设置值，如需修改，请联系华为技术支持。 |
| MINTHD | 流控速率最小值 | 可选必选说明：条件可选参数<br>参数含义：用于将用户接入速率控制在一定范围内，该参数用于控制每个SPP进程上每秒允许通过的Inter-MME接入消息量的最小值。最大值通过参数“MAXTHD”设置。<br>前提条件：该参数在“Inter-MME接入流控功能开关”参数配置为“ON(开启)”时生效。<br>数据来源：整网规划<br>取值范围：0～5000<br>系统初始设置值：10。<br>说明：- 流控速率是基于Attach流程评估的能力，并非真正的流程消息通过量。 |
| MAXTHD | 流控速率最大值 | 可选必选说明：条件可选参数<br>参数含义：用于将用户接入速率控制在一定范围内，该参数用于控制每个SPP进程上每秒允许通过的Inter-MME接入消息量的最大值。最小值通过参数“MINTHD”设置。<br>前提条件：该参数在“Inter-MME接入流控功能开关”参数配置为“ON(开启)”时生效。<br>数据来源：整网规划<br>取值范围：0～5000<br>系统初始设置值：500。<br>配置原则：该参数的取值必须大于等于“流控速率最小值”的取值。<br>说明：- 流控速率是基于Attach流程评估的能力，并非真正的流程消息通过量。 |

#### [使用实例](#ZH-CN_MMLREF_0000001172345763)

当POOL内某个MME故障时，会有故障MME上的用户会接入到Pool内正常的MME上，对正常MME造成冲击。此时需要设置“Inter-MME接入流控功能开关(FCSWITCH)”参数为“ON(开启)”、“流控速率最小值(MINTHD)”参数为10、“流控速率最大值(MAXTHD)”参数为200，配置格式如下：

SET INTERMMEFC: FCSWITCH=ON, MINTHD=10, MAXTHD=200;

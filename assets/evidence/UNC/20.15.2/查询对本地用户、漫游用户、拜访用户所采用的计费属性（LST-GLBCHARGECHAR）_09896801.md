#### [命令功能](#ZH-CN_CONCEPT_0209896801)

**适用NF：PGW-C、SMF**

该命令用来查询配置对本地用户、漫游用户、拜访用户所采用的计费属性。

#### [注意事项](#ZH-CN_CONCEPT_0209896801)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0209896801)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0209896801)

无。

#### [使用实例](#ZH-CN_CONCEPT_0209896801)

查询配置对本地用户、漫游用户、拜访用户所采用的计费属性：

```
LST GLBCHARGECHAR:;
```


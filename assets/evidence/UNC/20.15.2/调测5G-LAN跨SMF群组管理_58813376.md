# 调测5G LAN跨SMF群组管理

- [操作场景](#ZH-CN_OPI_0000001558813376__1.3.1)
- [必备事项](#ZH-CN_OPI_0000001558813376__1.3.2)

## [操作场景](#ZH-CN_OPI_0000001558813376)

本操作用于指导在SMF侧调测5G LAN跨SMF群组管理，验证该功能是否成功开启。

## [必备事项](#ZH-CN_OPI_0000001558813376)

前提条件

- 请仔细阅读[WSFD-225002 5G LAN跨SMF群组管理特性概述](WSFD-225002 5G LAN跨SMF群组管理特性概述_88411854.md)。
- 完成[激活5G LAN跨SMF群组管理](激活5G LAN跨SMF群组管理_09652117.md)。

数据

| 类别 | 参数名称 | 取值样例 | 获取方法 | 说明 |
| --- | --- | --- | --- | --- |
| 用户信息查询 | 用户IMSI号（IMSI） | 12300123456789 | 测试终端自带 | - |
| 用户访问的DNN | APN名称（DNN） | data | 测试终端自带 | - |
| 地址池名称 | 地址池名称（IPDomain） | Ipdomain-a | 已配置数据中获取 | - |

工具

- 测试终端
- OM Portal

## [操作步骤](#ZH-CN_OPI_0000001558813376)

1. 测试终端使用“data”DNN发起接入网络请求。
2. 执行 ****DSP PDUSESSION**** 命令，通过测试终端的IMSI查看测试终端是否成功激活。
    - 如果测试终端激活成功，且使用的“IPv4 PDP address”与规划值一致，测试终端接入成功，调测结束。
    - 如果测试终端激活成功，但使用的“IPv4 PDP address”与规划值不一致，执行 [步骤 3](#ZH-CN_OPI_0000001558813376__step177831404618) 。
3. 执行 ****LST ADDRESSATTR**** 检查地址分配方式是否正确。
    - 如果地址分配方式与规划值一致，请执行[5](#ZH-CN_OPI_0000001558813376__cmd186381634132315)。
    - 如果地址分配方式与规划值不一致，再次进行配置[激活5G LAN跨SMF群组管理](激活5G LAN跨SMF群组管理_09652117.md)后仍无法解决问题请执行[5](#ZH-CN_OPI_0000001558813376__cmd186381634132315)。
4. 执行 ****LST SECTION**** 查询地址池信息是否正确
    - 如果地址池的地址配置与规划值一致，请执行 [5](#ZH-CN_OPI_0000001558813376__cmd186381634132315) 。
    - 如果地址池的地址配置与规划值不一致，再次进行配置 [激活5G LAN跨SMF群组管理](激活5G LAN跨SMF群组管理_09652117.md) 后仍无法解决问题请执行 [5](#ZH-CN_OPI_0000001558813376__cmd186381634132315) 。
5. 收集信息并寻求技术支持。
    a. 在OM Portal上创建用户跟踪任务并保存报文。
    b. 执行 **EXP MML** 命令将当前配置数据导出为MML脚本文件并保存。
    c. 收集并保存上述所有查询信息。
    d. 收集归纳所有信息并联系华为技术支持解决。

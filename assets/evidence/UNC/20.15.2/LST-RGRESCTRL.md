## [命令功能](#ZH-CN_MMLREF_0296242438)

**适用NF：PGW-C、SMF**

该命令用于查询RG老化功能以及RG超出规格后的处理动作。

## [注意事项](#ZH-CN_MMLREF_0296242438)

无

#### [操作用户权限](#ZH-CN_MMLREF_0296242438)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0296242438)

无

## [使用实例](#ZH-CN_MMLREF_0296242438)

查询RG资源控制配置。

```
LST RGRESCTRL:;
RETCODE = 0  操作成功

结果如下
------------------------
                   在线计费RG老化控制  =  使能
              在线计费RG老化时长 (分)  =  20
在线计费全局业务阻塞处理时间间隔 (分)  =  0
                   离线计费RG老化控制  =  使能
              离线计费RG老化时长 (分)  =  20
             超出业务最大规格处理动作  =  阻塞对应业务
(结果个数 = 1)
```

## [输出结果说明](#ZH-CN_MMLREF_0296242438)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 在线计费RG老化控制 | 该参数用于控制在线计费RG老化功能。 |
| 在线计费RG老化时长 (分) | 该参数用于控制在线计费RG老化时长。 |
| 在线计费全局业务阻塞处理时间间隔 (分) | 该参数用于配置全局业务阻塞或重定向的时长，从阻塞或重定向开始经过这段时间以后，进行RG老化。 |
| 离线计费RG老化控制 | 该参数用于控制离线计费RG老化功能。 |
| 离线计费RG老化时长 (分) | 该参数用于控制离线计费RG老化时长。 |
| 超出业务最大规格处理动作 | 该参数用于控制新业务上报时，超过规格控制时的处理方式。 |

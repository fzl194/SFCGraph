#### [命令功能](#ZH-CN_CONCEPT_0209896943)

**适用NF：PGW-C、SMF**

该命令用于查询MSCC层异常返回码的处理动作配置。

#### [注意事项](#ZH-CN_CONCEPT_0209896943)

仅输入在线计费模板名称参数，查询结果为该在线计费模板下的所有MSCC层异常返回码处理动作。

#### [操作用户权限](#ZH-CN_CONCEPT_0209896943)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0209896943)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| DCCTMPLTNAME | 在线计费模板名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定在线计费模板名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。<br>默认值：无<br>配置原则：该参数使用ADD DCCTEMPLATE命令配置生成。 |
| MSCCRC | MSCC层异常返回码 | 可选必选说明：可选参数<br>参数含义：该参数用于指定MSCC层异常返回码。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～4。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0209896943)

- 查询全局在线计费模板的所有MSCC层异常返回码处理动作：
  ```
  LST MSCCRCACT: DCCTMPLTNAME="global";
  ```
  ```

  RETCODE = 0  操作成功

  MSCC层异常返回码动作
  --------------------
                在线计费模板名称  =  global
                MSCC层异常返回码  =  5012
            MSCC层异常返回码动作  =  重定向
          MSCC层阻塞处理时间间隔  =  NULL
  MSCC层重定向处理重定向IPv4地址  =  192.168.10.16
  MSCC层重定向处理重定向IPv6地址  =  NULL
              MSCC层重新激活请求  =  禁止
  (结果个数 = 1)

  ---    END
  ```
- 查询全局在线计费模板中MSCC层异常返回码为1000的处理动作：
  ```
  LST MSCCRCACT: DCCTMPLTNAME="global",MSCCRC="1000";
  ```
  ```

  RETCODE = 0 操作成功。

  MSCC层异常返回码动作
  --------------------
  在线计费模板名称 = global
  MSCC层异常返回码 = 1000
  MSCC层异常返回码动作 = 阻断后trigger触发上报
  MSCC层阻塞处理时间间隔 = NULL
  MSCC层重定向处理重定向IPv4地址 = NULL
  MSCC层重定向处理重定向IPv6地址 = NULL
  (结果个数 = 1)
  --- END
  ```

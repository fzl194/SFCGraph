#### [命令功能](#ZH-CN_CONCEPT_0251174248)

**适用NF：NCG**

该命令用于显示向第三方服务器/UDN服务器或者是本地备份话单文件时，备份任务的状态。该命令执行后，系统返回相应的备份任务状态值。

在系统新安装时、系统调测或者话单备份异常时，可以使用该命令进行检查。

查询方式有4种：

- 不输入参数，查询所有话单备份任务。
- 输入“备份任务标识”参数，查询指定备份任务标识所涉及话单备份任务。
- 输入“接入网元分组标识”参数，查询指定接入网元分组上所有话单备份任务。
- 输入“模块名”参数，查询指定模块上所有话单备份任务。

#### [注意事项](#ZH-CN_CONCEPT_0251174248)

该命令在单个RU上最多支持查询大约2400条链路信息，超过会出现“操作超时”，建议查询时输入“备份任务标识”参数。

#### [本地用户权限](#ZH-CN_CONCEPT_0251174248)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_CONCEPT_0251174248)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0251174248)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| BACKUPID | 备份任务标识 | 可选必选说明：可选参数<br>参数含义：备份任务标识，全局唯一。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～32。<br>默认值：无<br>配置原则：无 |
| AGID | 接入网元分组标识 | 可选必选说明：可选参数<br>参数含义：用于区分不同域的接入网元。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～32。<br>默认值：无<br>配置原则：无 |
| MNAME | 模块名 | 可选必选说明：可选参数<br>参数含义：用于表示一个业务模块对象，全局唯一。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～15。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0251174248)

显示所有备份任务状态。示例如下：

```
DSP CDRBACKUP: BACKUPID="CDRbakid_1";
```

#### [输出结果说明](#ZH-CN_CONCEPT_0251174248)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 通道列表 | 备份任务所对应的备份通道名称列表。这里是“scdr，gcdr”。 |
| 本端IP地址 | 备份任务所对应的CG侧IP地址。 |
| 第三方服务器IP地址 | 备份任务所对应的第三方服务器IP地址。 |
| 第三方服务器端口号 | 备份任务所对应的第三方服务器端口号。 |
| 任务状态 | 查询到的CG到第三方服务器的FTP/SFTP备份任务状态：<br>- 正常：表示当前有话单正在备份，备份任务正常。<br>- 故障：表示备份任务有故障发生，或者链路连接失败。<br>- 等待：表示备份链路正常。当前无话单正在备份。<br>- 停止：表示一次性备份任务的所有话单备份完成。 |
| 日志稽核功能状态 | 查询日志稽核功能的状态：<br>- 正常：表示日志稽核功能正常。<br>- 故障：表示日志稽核模板内容无效，日志稽核功能故障。<br>- 未启用：表示当前任务没有启用日志稽核功能。 |

其余输出项请参见 [**ADD CDRBACKUP**](增加话单备份（ADD CDRBACKUP）_51174244.md) 的参数说明。

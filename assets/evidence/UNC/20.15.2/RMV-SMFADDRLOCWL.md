# 删除位置区域地址分配用户白名单（RMV SMFADDRLOCWL）

- [注意事项](#ZH-CN_MMLREF_0000001135636463__1.3.2)

## [命令功能](#ZH-CN_MMLREF_0000001135636463)

**适用NF：PGW-C、SMF、GGSN**

该命令用于删除按照区域地址分配用户白名单。

## [注意事项](#ZH-CN_MMLREF_0000001135636463)

该命令执行后只对新激活用户生效。

#### [操作用户权限](#ZH-CN_MMLREF_0000001135636463)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001135636463)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| MSISDN | MSISDN | 可选必选说明：必选参数<br>参数含义：该参数用于指定用户的MSISDN信息。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~15。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001135636463)

删除用户MSISDN为123456的区域地址分配白名单：

```
RMV SMFADDRLOCWL: MSISDN="123456";
```

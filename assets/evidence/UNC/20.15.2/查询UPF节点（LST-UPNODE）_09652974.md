## [命令功能](#ZH-CN_MMLREF_0209652974)

**适用NF：SMF、SGW-C、PGW-C、GGSN**

该命令用于查询指定实例名称的UPF节点特征。

## [注意事项](#ZH-CN_MMLREF_0209652974)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209652974)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652974)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| NFINSTANCENAME | UPF实例名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定UPF实例名称。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是0~255。不区分大小写。<br>默认值：无<br>配置原则：<br>本参数取值与ADD PNFPROFILE命令中的“NF实例标识”参数取值保持一致时，关联关系生效。 |
| NFDESCNAME | UPF描述名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定UPF实例描述名称。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是0~255。区分大小写。<br>默认值：无<br>配置原则：<br>本参数取值与ADD PNFPROFILE命令中的“NF描述名称”参数取值保持一致。 |

## [输出结果说明](#ZH-CN_MMLREF_0209652974)

| 输出项名称 | 输出项解释 |
| --- | --- |
| UPF实例名称 | 该参数用于指定UPF实例名称。 |
| UPF位置特征 | 该参数用于指定UPF节点的位置特征。 |
| UPF功能 | 该参数用于指定UPF节点功能。 |
| UPF锁 | 该参数用于标识UPF是否被锁定，SMF不会选择到已锁定的UPF。 |
| 辅锚点UPF故障迁移功能开关 | 该参数用于标识辅锚点UPF是否具备冗余功能。该开关打开情况下，辅锚点UPF支持故障迁移和故障重选功能。 |
| 地址分配属性 | 该参数用于配置UPF的UE动态地址分配属性。 |
| UPF共享开关 | 该参数用于表示是否支持作为共享UPF使用。 |
| VPN名称 | 该参数用于指定VPN名称。 |
| 是否仅支持作为分流UPF | 该参数用于标识是否仅支持作为分流UPF。 |
| 是否支持作为ATSSS UPF | 该参数用于标识是否支持作为ATSSS UPF。 |
| IPv6 ND信息携带方式 | 该参数用于标识IPv6 Network Discovery信息携带方式。 |
| 惯性运行开关 | 该参数用于表示UPF是否支持惯性运行功能。 |
| UPF描述名称 | 该参数用于指定UPF实例描述名称。 |
| 园区UPF开关 | 该参数用于控制是否作为园区UPF。 |
| ULCL UPF间创建转发隧道开关 | 该参数用于控制位置移动触发替换ULCL UPF的场景下是否支持源侧ULCL UPF与目标侧ULCL UPF间创建转发隧道，来保证切换过程中数据的完整性。 |
| 是否支持UPF无N9逻辑接口地址时作为主锚点激活 | 该参数用于指定是否支持UPF无N9逻辑接口地址时作为主锚点激活。<br>当UPF不配N9逻辑接口地址，PDUESTWITHOUTN9设置为ENABLE时，不支持GGSN或者PGW-C形态。<br>当UPF不配N9逻辑接口地址，PDUESTWITHOUTN9设置为ENABLE时，不支持EPS Fallback场景。语音会话场景下，主锚点UPF必须配置N9逻辑接口。<br>该参数仅在UPF未规划配置N9地址的场景下生效。 |
| 是否支持5G LAN | 该参数用于控制是否支持5G LAN。 |
| 主锚点与分流节点合设功能 | 该参数用于控制UPF是否支持主锚点与分流节点合设。 |
| 是否支持组播广播业务 | 该参数用于控制是否支持组播广播业务。 |
| 是否支持智能业务 | 该参数用于设置UPF是否支持智能业务。 |
| 路径迁移开关 | 该参数用于设置UPF是否支持PFCP会话路径迁移功能。如果UPF支持路径迁移功能，并且SMF与UPF之间存在有多条路径，当一条路径发生故障时，允许SMF将这条路径上的会话迁移到另一条可用路径上。 |
| 超时触发路径迁移开关 | 该参数用于设置UPF是否支持PFCP会话业务消息超时触发路径迁移功能。如果UPF支持路径迁移功能，并且SMF与UPF之间存在有多条路径，当SMF接收到N4响应消息超时时，允许将该会话从当前路径迁移到另一条可用路径上。 |
| 路径迁移响应等待时间(100毫秒) | 该参数用于设置路径迁移响应等待时间。在该时间内没有收到对应的响应，则判断本次消息通信失败。 |
| 是否支持UE IP地址NAT转换功能 | 该参数用于表示UPF是否支持UE IP地址NAT转换功能。在通用DNN漫游分流场景下，专网SMF会选择支持UE IP地址NAT转换功能的专网UPF。 |
| UPF作为Proxy UPF对接PGW-U开关 | UPF是否支持作为Proxy UPF对接PGW-U。 |
| 质差分析开关 | 该参数用于设置UPF是否支持质差分析。 |
| 是否支持作为智家随行场景To Home会话主锚点 | 该参数用于设置UPF是否支持作为智家随行场景To Home会话主锚点。 |
| 信令触发静默路径迁移开关 | 该参数用于指定当UPG代理路径全部故障后，由于PFCP会话信令交互超时或者当前判断路径已经故障的情况下，是否支持将PFCP会话直接迁移到状态正常的静默路径。 |
| 路径迁移扩展信元携带开关 | 该参数用于设置SMF将故障路径上的会话迁移到另一条可用路径上时是否携带扩展信元。 |

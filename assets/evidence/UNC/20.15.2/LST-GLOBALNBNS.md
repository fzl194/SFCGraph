## [命令功能](#ZH-CN_MMLREF_0000001222556857)

**适用NF：PGW-C、SMF、GGSN**

该命令用来查看系统的NBNS属性。

## [注意事项](#ZH-CN_MMLREF_0000001222556857)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001222556857)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001222556857)

无

## [使用实例](#ZH-CN_MMLREF_0000001222556857)

查询系统的默认的NBNS属性：

```
%%LST GLOBALNBNS:;%%
RETCODE = 0  操作成功

结果如下
--------
        NBNS功能开关  =  不使能
      主NBNS服务器IP  =  10.1.1.1
      备NBNS服务器IP  =  10.2.2.2
第一优先级服务器属性  =  dhcp
第二优先级服务器属性  =  radius
第三优先级服务器属性  =  local
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001222556857)

| 输出项名称 | 输出项解释 |
| --- | --- |
| NBNS功能开关 | 该参数用于控制是否使能NBNS功能。 |
| 主NBNS服务器IP | 该参数用于指定主NBNS服务器IP。 |
| 备NBNS服务器IP | 该参数用于指定备NBNS服务器IP。 |
| 第一优先级服务器属性 | 该参数用于指定IPv4第一优先级属性。 |
| 第二优先级服务器属性 | 该参数用于指定IPv4第二优先级属性。 |
| 第三优先级服务器属性 | 该参数用于指定IPv4第三优先级属性。 |

#### [命令功能](#ZH-CN_CONCEPT_0209897007)

**适用NF：SGW-C、PGW-C、SMF**

该命令用来查看话单缓存目录状态信息。

#### [注意事项](#ZH-CN_CONCEPT_0209897007)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0209897007)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0209897007)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PODNAME | POD名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定POD名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0209897007)

查看PODNAME为uncpod-011-30-0-217下的话单缓存目录状态信息：

```
LST CDRSTRGSTATUS:PODNAME="uncpod-011-30-0-217";
```

```

RETCODE = 0  操作成功

话单缓存目录状态
----------------
     POD名称  =  uncpod-011-30-0-217
话单缓存目录  =  /charge1
    目录状态  =  锁定
(结果个数 = 1)

---    END
```

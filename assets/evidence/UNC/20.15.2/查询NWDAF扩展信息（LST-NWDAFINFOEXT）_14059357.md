## [命令功能](#ZH-CN_MMLREF_0000002014059357)

该命令用以查询NWDAF实例的扩展信息。

## [注意事项](#ZH-CN_MMLREF_0000002014059357)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000002014059357)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000002014059357)

无

## [使用实例](#ZH-CN_MMLREF_0000002014059357)

查询NWDAF实例的扩展信息：

```
%%LST NWDAFINFOEXT:;%%
RETCODE = 0  操作成功

结果如下
--------
NWDAF实例名称  =  NWDAF_Instance_0
           ID  =  central
NWDAF事件类型  =  QOS分析
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000002014059357)

| 输出项名称 | 输出项解释 |
| --- | --- |
| NWDAF实例名称 | 该参数用于唯一指定某一个NWDAF实例。 |
| ID | 该参数用于唯一标识NWDAF实例中的某个NwdafInfo。 |
| NWDAF事件类型 | 该参数用于指定NWDAF支持的分析事件类型。 |

# 删除SMS CHR流程控制模板（RMV SMSCHRPRCTMPL）

- [命令功能](#ZH-CN_MMLREF_0000001353481542__1.3.1)
- [注意事项](#ZH-CN_MMLREF_0000001353481542__1.3.2)

## [命令功能](#ZH-CN_MMLREF_0000001353481542)

**适用NF：SMSF**

该命令用于删除SMS CHR流程控制模板。

## [注意事项](#ZH-CN_MMLREF_0000001353481542)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0000001353481542)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001353481542)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| TMPLIDX | 流程控制模板索引 | 可选必选说明：必选参数<br>参数含义：该参数用于表示流程控制模板索引。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~127。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001353481542)

运营商希望删除“流程控制模板索引”为“1”的SMS CHR流程控制模板，执行如下命令：

```
RMV SMSCHRPRCTMPL: TMPLIDX=1;
```

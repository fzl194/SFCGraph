#### [命令功能](#ZH-CN_CONCEPT_0000001550120646)

该命令用于查询COMM规则匹配统计计数信息。

COMM消息、报文的正常流程或者异常流程的规则匹配开关默认关闭，当开关关闭时，查询信息无统计计数。

例如，当发现丢包时，可打开规则匹配开关，并设置匹配规则，然后执行该命令查询COMM规则匹配统计计数，判断是否是COMM丢包。

#### [注意事项](#ZH-CN_CONCEPT_0000001550120646)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550120646)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550120646)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RUNAME | RU名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定RU名称，执行DSP RU查看RU名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，区分大小写。<br>默认值：无<br>配置原则：使用DSP RU查看RU名称。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001550120646)

查询规则匹配统计信息：

```
DSP MSSCOMMSTAT:RUNAME="VNODE_VNRS_VNFC_IPU_0066";
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001550120646)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 服务类型 | 用于表示服务类型，包括packet、packet-discard、message和message-discard，分别是正常报文、丢弃报文、正常消息和丢弃消息。 |
| 状态 | 用于表示状态，包括open、close和self-closing，self-closing为开关超时自动关闭后的状态，open和self-closing状态能显示后面的字段信息。 |

| 目的功能块ID | 用于表示目的功能块ID。 |
| 目的实例ID | 用于表示目的实例ID。 |

| 匹配个数 | 用于表示消息或者报文匹配个数。 |
| 回调方式通信匹配计数 | 用于表示以回调方式通信的报文、消息匹配计数。 |

## [命令功能](#ZH-CN_MMLREF_0000001146661889)

该命令用于查询系统当前License紧急状态信息。

## [注意事项](#ZH-CN_MMLREF_0000001146661889)

无。

## [参数说明](#ZH-CN_MMLREF_0000001146661889)

无。

## [使用实例](#ZH-CN_MMLREF_0000001146661889)

License处于紧急状态时：

```
%%DSP LICCTL:;%%
RETCODE = 0  操作成功

License紧急信息
---------------
           License紧急状态  =  打开
距离自动关闭启动紧急的天数  =  6
          剩余启动紧急次数  =  2
(结果个数 = 1)

---    END 
```

License处于非紧急状态时：

```
%%DSP LICCTL:;%%
RETCODE = 0  操作成功

License紧急信息
---------------
           License紧急状态  =  关闭
距离自动关闭启动紧急的天数  =  NULL
          剩余启动紧急次数  =  2  
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001146661889)

该命令执行正常，会返回命令执行成功的提示信息，输出项说明如 [表1](#ZH-CN_MMLREF_0000001146661889__table714812389314) 所示。

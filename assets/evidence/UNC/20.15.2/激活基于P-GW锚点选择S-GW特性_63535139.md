## [操作场景](#ZH-CN_OPI_0163535139)

本操作指导介绍在运行网络中激活基于P-GW锚点选择S-GW的操作过程。

UE从MME切换到GnGp SGSN时，MME需要将Co-located GGSN-PGW FQDN信元传递给GnGp SGSN。这样，当UE再从GnGp SGSN切换到MME时，MME可以根据GnGp SGSN带回来的Co-located GGSN-PGW FQDN选择拓扑最近的S-GW。

如果GnGp SGSN不支持Co-located GGSN-PGW FQDN信元，MME还支持通过GGSN IP查询P-GW的主机名，选择拓扑最近的S-GW。

> **说明**
> 适用于 SGSN、 MME。

## [操作步骤](#ZH-CN_OPI_0163535139)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. **可选：** 在Inter USN 2/3G切换4G场景下，增加Hostfile文件的A解析记录。
  [**ADD IPV4DNSH**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/DNS/DNS Hostfile管理/增加IPV4 DNS Hostfile记录(ADD IPV4DNSH)_26145884.md)
4. **可选：** 在Inter USN 2/3G切换4G场景下，增加NAPTR解析记录。
  [**ADD DNSN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/DNS/DNS NAPTR管理/增加DNS NAPTR记录(ADD DNSN)_72225569.md)
5. **可选：** 在Inter USN 2/3G切换4G场景下，增加DNS查询控制参数设置。
  [**ADD DNSQ**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/DNS/DNS查询管理/增加DNS查询控制参数(ADD DNSQ)_26305702.md)
6. **可选：**增加GTP-C V0/V1协议兼容性配置。
  > **说明**
  > - “MMMSGTYPE”设置为“SGSN_CONTEXT_RESP”或者“FWD_RLC_REQ”。
  > - UNC系统默认携带Co-located GGSN-PGW FQDN信元。可通过[**LST GTPCCMPT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/GTP-C协议管理/GTP-C V0 V1协议兼容性/查询GTP-C V0_V1协议兼容性配置(LST GTPCCMPT)_26145924.md)命令查看当前配置，如果携带方式被设置为不携带Co-located GGSN-PGW FQDN信元时则需要重新配置。
  [**ADD GTPCCMPT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/GTP-C协议管理/GTP-C V0 V1协议兼容性/增加GTP-C V0_V1协议兼容性配置(ADD GTPCCMPT)_72225601.md)
7. 打开3G到4G切换S-GW拓扑选择增强开关。
  [**SET SMFUNC**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/会话管理/SM扩展功能管理/设置会话管理扩展功能(SET SMFUNC)_26145684.md)

## [任务示例](#ZH-CN_OPI_0163535139)

任务描述

启用基于P-GW锚点选择S-GW，设置 UNC 将Co-located GGSN-PGW FQDN信元通过SGSN Context Response消息传递到对端，并且当GnGp SGSN不支持通过Co-located GGSN-PGW FQDN信元选择S-GW时，通过GGSN IP地址查询P-GW的主机名，选择拓扑最近的S-GW。

脚本

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2SELGW04", SWITCH=ENABLE;
```

// **可选：** 在Inter USN 2/3G切换4G场景下，增加Hostfile文件的A解析记录。

```
ADD IPV4DNSH: HSINDEX=1, HOSTNAME="topon.pgwggsn1.pudong.shanghai.example.com", ADDRSECTION=SECTION1, IPV4ADDR1="10.2.3.4";
```

// **可选：** 在Inter USN 2/3G切换4G场景下，增加NAPTR解析记录。

```
ADD DNSN: FQDN="HUAWEI1.COM.MNC003.MCC123.GPRS", HSINDEX=1, ENTITY=PGW, INTYPE=Gn, PRIORITY=0, WEIGHT=255;
```

// **可选：** 在Inter USN 2/3G切换4G场景下，增加DNS查询控制参数设置。

```
ADD DNSQ: SUBRANGE=SPECIFY, IMSIPRE="12302", DNSUF="HUAWEI1.COM.MNC003.MCC123.GPRS", DNSQUERYMODE=NAPTR;
```

// **可选：** 增加GTP-C V0/V1协议兼容性配置。

```
ADD GTPCCMPT: IDX=1, MSGCLS=MM, MMMSGTYPE=SGSN_CONTEXT_RESP, CTXRSP=CO_LOC_GGSN_PGW_FQDN, INCLUDE=PRO_DEFINE;
```

//打开3G到4G切换S-GW拓扑选择增强开关。

```
SET SMFUNC: SGWTOPO=YES;
```

## [任务示例](#ZH-CN_OPI_0000001110539042)

任务描述

运营商要为某企业园区内的用户提供UL CL业务，对于进入园区的用户访问园区内的服务时，数据流就近在园区内DC卸载。

该园区内部署了一套“UL CL UPF+辅锚点UPF”形态的UPF，为了确保行业用户的业务可靠性，运营商在园区外部署了一套主、辅锚点和UL CL UPF合一形态的UPF，和园区内的UPF形成冗余组网的容灾组网模式。

SMF侧基于位置区优选策略来配置和园区内UPF以及园区外UPF的对接，确保进去MEC区域的用户能够优先选择园区内UPF进行UL CL分流，当园区内UPF故障后能够选择园区外UPF来继续为用户提供UL CL分流服务。

假定UPF实例标识分别为UPF_Instance_0和UPF_Instance_1，对应的IP地址为192.168.126.11和192.168.126.12。

脚本

// **SMF本端添加UPF** 。

```
ADD PNFPROFILE:NFINSTANCEID="
UPF_Instance_0
", NFTYPE=NfUPF, NFSTATUS=Registered,FQDN="upf00.node.mnc003.mcc123.3gppnetwork.org",IPADDRESSTYPE=IPTypeV4, IPV4ADDRESS1="192.168.126.11";
ADD PNFPROFILE:NFINSTANCEID="
UPF_Instance_1
", NFTYPE=NfUPF, NFSTATUS=Registered,FQDN="upf01.node.mnc003.mcc123.3gppnetwork.org",IPADDRESSTYPE=IPTypeV4, IPV4ADDRESS1="192.168.126.12";
```

// **配置园区外UPF辅锚点的虚拟DNAI** 。

```
ADD VIRTUALUPFID:UPFINSTANCEID="
UPF_Instance_0
",VUPFINSTANCEID="UPF_Instance_0_v1";
```

```
ADD VUPFIDBINDDNAI:VUPFINSTANCEID="UPF_Instance_0_v1",DNAI="testdnai1";
```

// **配置用户面支持的服务切片信息** 。

```
ADD PNFNS: INDEX=4, NFINSTANCEID="
UPF_Instance_1

```

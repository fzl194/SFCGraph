#### [命令功能](#ZH-CN_CONCEPT_0277661137)

该命令用于查询CSDB使用者信息。

#### [注意事项](#ZH-CN_CONCEPT_0277661137)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0277661137)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0277661137)

| **参数标识** | **参数名称** | **参数说明** |
| --- | --- | --- |
| LOGICGRPID | 逻辑组ID | 可选必选说明：可选参数<br>参数含义：逻辑组标识，如果不输入该参数，系统会查询CSDB的所有使用者信息。<br>数据来源：本端规划<br>取值范围：整型，0~4294967295。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0277661137)

查询所有使用者信息：

```
%%DSP DBUSERINFO:;%%
RETCODE = 0  操作成功

操作结果如下:
-------------------------
         逻辑组ID  =  0
     业务服务类型  =  1062
       业务服务ID  =  999
 业务服务类型名称  =  NRF
         子实例ID  =  7
     容灾使用状态  =  开启容灾
         备份组ID  =  4294967295
   持久化使用状态  =  开启持久化
         部署模式  =  默认部署
(结果个数 = 1)

---    END
```

查询 “逻辑组ID” 为 “0” 的使用者信息：

```
%%DSP DBUSERINFO: LOGICGRPID=0;%%
RETCODE = 0  操作成功

操作结果如下:
-------------------------
         逻辑组ID  =  0
     业务服务类型  =  1062
       业务服务ID  =  999
 业务服务类型名称  =  NRF
         子实例ID  =  7
     容灾使用状态  =  开启容灾
         备份组ID  =  4294967295
   持久化使用状态  =  开启持久化
         部署模式  =  默认部署
(结果个数 = 1)

---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0277661137)

| **输出项名称** | **输出项解释** |
| --- | --- |
| 逻辑组ID | 逻辑组标识。 |
| 业务服务类型 | 业务服务类型。 |
| 业务服务ID | 业务服务标识。 |
| 业务服务类型名称 | 业务服务类型名称。 |
| 子实例ID | CSDB分配的子实例标识。 |
| 容灾使用状态 | 是否使用容灾。<br>- 开启容灾：使用者使用地理容灾功能。<br>- 关闭容灾：使用者不使用地理容灾功能。 |
| 备份组ID | 灰度升级的备份组标识。 |
| 持久化使用状态 | 是否使用持久化。<br>- 开启持久化：使用者指定数据表写入磁盘。<br>- 关闭持久化：使用者所有数据表不写入磁盘。 |
| 部署模式 | 子集群部署模式。<br>- 共享子集群：使用者可以和多个使用者共用一个子集群。<br>- 独占子集群：使用者独占一个子集群。<br>- 默认部署：根据配置文件决定共享子集群还是独占子集群。 |

# 删除切片地址池性能统计对象（RMV PERFNSADDRPOOL）

- [命令功能](#ZH-CN_MMLREF_0212701644__1.3.1)
- [注意事项](#ZH-CN_MMLREF_0212701644__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0212701644__1.3.4)
- [使用实例](#ZH-CN_MMLREF_0212701644__1.3.5)

## [命令功能](#ZH-CN_MMLREF_0212701644)

**适用NF：SMF**

该命令用于删除切片地址池性能统计对象。

## [注意事项](#ZH-CN_MMLREF_0212701644)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0212701644)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0212701644)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IDX | 索引 | 可选必选说明：必选参数<br>参数含义：该参数用于指定切片地址池性能统计对象的索引。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~499。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0212701644)

删除索引为1的性能统计对象：

```
RMV PERFNSADDRPOOL: IDX=1;
```

# 删除LAC组内绑定的LAC号段（RMV LACID）

- [命令功能](#ZH-CN_MMLREF_0209652348__1.3.1)
- [注意事项](#ZH-CN_MMLREF_0209652348__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0209652348__1.3.4)
- [使用实例](#ZH-CN_MMLREF_0209652348__1.3.5)

## [命令功能](#ZH-CN_MMLREF_0209652348)

**适用NF：GGSN**

该命令用来删除LAC组内绑定的LAC号段。

## [注意事项](#ZH-CN_MMLREF_0209652348)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0209652348)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652348)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| LACSECNUM | LAC段编号 | 可选必选说明：必选参数<br>参数含义：该参数用于指定LAC段编号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~23999。<br>默认值：无<br>配置原则：无 |
| LACGROUPNAME | LAC组名 | 可选必选说明：必选参数<br>参数含义：该参数用于指定LAC组名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~32。不支持空格，不区分大小写。<br>默认值：无<br>配置原则：<br>该LACGROUPNAME必须已经由ADD LACGROUP配置过。 |

## [使用实例](#ZH-CN_MMLREF_0209652348)

假设运营商需要删除指定LAC组绑定的某个LAC号段，“LAC组名”为“beijing”，“LAC号段”为2，执行该命令：

```
RMV LACID:LACGROUPNAME="beijing",LACSECNUM=2;
```

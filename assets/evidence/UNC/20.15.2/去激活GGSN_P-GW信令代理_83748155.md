## [操作场景](#ZH-CN_OPI_0000001183748155)

运营商无需GGSN/P-GW承担Proxy的作用，需要去激活GGSN/P-GW信令代理特性。

> **说明**
> 适用于 GGSN、 PGW-C

## [对系统的影响](#ZH-CN_OPI_0000001183748155)

去激活GGSN/P-GW信令代理特性时，如果有用户正在使用该功能，则这些用户将无法激活，导致业务处理异常。所以建议在无用户使用该功能时进行该操作。

## [必备事项](#ZH-CN_OPI_0000001183748155)

前提条件

- 请仔细阅读[WSFD-205107 GGSN/P-GW信令代理](WSFD-205107 GGSN_P-GW信令代理特性概述_37468134.md)。
- 完成[激活GGSN/P-GW信令代理](激活GGSN_P-GW信令代理_38987364.md)
- 已在本地备份配置文件。

## [操作步骤](#ZH-CN_OPI_0000001183748155)

1. 以下命令在“MML命令行 - UNC”窗口上执行。关闭本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md) :LICITEM=LKV3W9GPSP11,SWITCH=DISABLE;
2. 删除Home group与APN的绑定关系。[**RMV HOMEGRPBINDAPN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/GGSN和P-GW Proxy/Home Group绑定APN/删除Home Group和APN的绑定关系（RMV HOMEGRPBINDAPN）_42853268.md):APN=APN，HOMEGROUPINDX=Home group的编号；
3. 关闭网关的Proxy功能<br>[**SET GWPROXYFUNC**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/GGSN和P-GW Proxy/网关Proxy功能/设置网关Proxy功能（SET GWPROXYFUNC）_42853270.md):PROXYSW=网关Proxy开关；
4. 删除在Home group下配置的归属P-GW的IP地址。
  [**RMV HOMEIP**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/GGSN和P-GW Proxy/Home IP/删除Home IP地址（RMV HOMEIP）_42693478.md) :HOMEGROUPINDX=组号；
5. 删除Home group。
  [**RMV HOMEGROUP**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/GGSN和P-GW Proxy/Home Group/删除Home Group（RMV HOMEGROUP）_88733227.md) :HOMEGROUPINDX=home group的编号；
6. **可选：** 删除将要对其进行信令代理操作IMSI/MSISDN号码段和号码段组。
    a. 删除IMSI/MSISDN号码段。
      [**RMV PRIMSIMSISDNSEG**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/GGSN和P-GW Proxy/代理选择的IMSI_MSISDN号段/删除代理IMSI_MSISDN号段（RMV PRIMSIMSISDNSEG）_88733229.md) :SEGMENTNAME=IMSI/MSISDN号段名称；
    b. 删除IMSI/MSISDN号码段组。
      [**RMV PRIMSIMSISDNG**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/GGSN和P-GW Proxy/代理选择的IMSI_MSISDN号段组/删除代理IMSI_MSISDN号段组（RMV PRIMSIMSISDNG）_88613389.md) :SEGGROUPNAME=IMSI/MSISDN号段组名称；
7. **可选：** 删除将要对其进行信令代理操作的计费属性信息。
    a. 删除指定计费属性组的计费属性值。
      [**RMV CCVALUE**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/GGSN和P-GW Proxy/计费特征值/删除计费特征值（RMV CCVALUE）_88613387.md) : CCGROUPNAME=计费特征组名称;
    b. 删除计费属性组。
      [**RMV CCGROUP**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/GGSN和P-GW Proxy/计费特征组/删除计费特征组（RMV CCGROUP）_42693476.md) : CCGROUPNAME=计费特征组名称;
8. 关闭APN的proxy功能。<br>[**RMV APNGWPROXYFUNC**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/GGSN和P-GW Proxy/APN网关Proxy功能/删除APN网关Proxy功能配置（RMV APNGWPROXYFUNC）_88733225.md):APN=APN，PROXYSW=网关Proxy开关；

## [任务示例](#ZH-CN_OPI_0000001183748155)

任务描述

去激活指定APN的GGSN/P-GW信令代理功能。

脚本

1. 关闭本特性的License开关。
  ```
  SET LICENSESWITCH:LICITEM=LKV3W9GPSP11,SWITCH=DISABLE;
  ```
2. 删除Home group与APN的绑定关系。
  ```
  RMV HOMEGRPBINDAPN:APN="apn-test",HOMEGROUPINDX=1;
  ```
3. 关闭网关的proxy功能。
  ```
  SET GWPROXYFUNC:PROXYSW=DISABLE;
  ```
4. 删除在Home group下配置的归属P-GW的IP地址。
  ```
  RMV HOMEIP:HOMEGROUPINDX=1;
  ```
5. 删除Home group。
  ```
  RMV HOMEGROUP:HOMEGROUPINDX=1;
  ```
6. 可选：删除将要对其进行信令代理操作IMSI/MSISDN号码段和号码段组。
  ```
  RMV PRIMSIMSISDNSEG:SEGMENTNAME="msisdn-test";
  ```
  ```
  RMV PRIMSIMSISDNG:SEGGROUPNAME="group-test";
  ```
7. 可选：删除将要对其进行信令代理操作的计费属性信息。
  ```
  RMV CCVALUE:CCGROUPNAME="cc_group1";
  ```
  ```
  RMV CCGROUP:CCGROUPNAME="cc_group1";
  ```
8. 关闭APN的proxy功能。
  ```
  RMV APNGWPROXYFUNC:APN="apn-test",PROXYSW=DISABLE;
  ```

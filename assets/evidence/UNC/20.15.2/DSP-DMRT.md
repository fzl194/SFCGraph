#### [命令功能](#ZH-CN_MMLREF_0000001126306102)

**适用网元：SGSN、MME**

该命令用户查询Diameter域路由状态。

#### [注意事项](#ZH-CN_MMLREF_0000001126306102)

该命令执行后立即生效。

#### [本地用户权限](#ZH-CN_MMLREF_0000001126306102)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126306102)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126306102)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| ROUTEIDX | 路由索引 | 可选必选说明：可选参数<br>参数含义：该参数用于指定待查询的Diameter域路由索引。<br>取值范围：0~2047<br>默认值：无<br>说明：如果不输入，表示查询系统内所有Diameter域路由的状态。 |

#### [使用实例](#ZH-CN_MMLREF_0000001126306102)

1. 不输入查询参数，查询所有Diameter域路由信息：
  DSP DMRT:;
  ```
  %%DSP DMRT:;%%
  RETCODE = 0  操作成功。

  输出结果如下
  --------------
   路由索引  路由名称  路由状态

   0         TO-HSS0   正常    
   1         TO-HSS1   故障    
  (结果个数 = 2)
  ---    END
  ```
2. 输入Diameter域路由索引，查询指定的Diameter域路由状态信息：
  DSP DMRT: ROUTEIDX=0;
  ```
  %%DSP DMRT: ROUTEIDX=0;%%
  RETCODE = 0  操作成功。

  输出结果如下
  --------------
  路由索引  =  0
  路由名称  =  TO-HSS0
  路由状态  =  正常
  (结果个数 = 1)
  ---    END
  ```

#### [输出结果说明](#ZH-CN_MMLREF_0000001126306102)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 路由索引 | Diameter域路由的索引，该参数用来在系统范围内部标识一条Diameter域路由。 |
| 路由名称 | Diameter域路由的名称。 |
| 路由状态 | 查询Diameter域路由状态的结果。<br>取值说明：<br>- “正常”：表示UNC与HSS、EIR、DRA或GMLC之间的该条域路由正常。<br>- “故障”：表示UNC与HSS、EIR、DRA或GMLC之间的该条域路由中断。<br>- “拥塞”：表示UNC与HSS、EIR、DRA或GMLC之间的该条域路由上达到链路负荷门限。<br>说明：查询Diameter域路由信息非正常，则需要按照如下方式进行排查：<br>- 通过命令[**LST DMLKS**](../Diameter链路集/查询Diameter链路集配置(LST DMLKS)_26146280.md)查询该对端实体属于哪一个链路集；<br>- 通过命令[**LST DMLKS**](../Diameter链路集/查询Diameter链路集配置(LST DMLKS)_26146280.md)来查询哪条链路属于该链路集；<br>- 通过[**DSP DMLKS**](../Diameter链路集/显示Diameter链路集状态(DSP DMLKS)_72225959.md)来查看该链路的状态信息。 |

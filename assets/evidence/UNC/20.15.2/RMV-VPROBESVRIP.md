# 删除vProbe报表服务器的接入点IP地址（RMV VPROBESVRIP）

## [命令功能](#ZH-CN_MMLREF_0000001388723160)

该命令用于删除vProbe报表服务器的接入点IP地址。

## [注意事项](#ZH-CN_MMLREF_0000001388723160)

- 该命令执行后立即生效。

- 执行命令前请确认vProbe服务处于上线状态，可通过DSP FUNCTIONSETINFO命令查询确认。
- 如果删除系统中唯一的一个接入点，则会影响报表上传。

#### [操作用户权限](#ZH-CN_MMLREF_0000001388723160)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001388723160)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| ACCESSNAME | 接入点名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定vProbe报表服务器的接入点名称。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是0~32。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001388723160)

运营商A不再使用1号报表服务器（VPROBESVR1）下的1号接入点（ACCESS1），将此接入点删除：

```
RMV VPROBESVRIP: ACCESSNAME="ACCESS1";
```

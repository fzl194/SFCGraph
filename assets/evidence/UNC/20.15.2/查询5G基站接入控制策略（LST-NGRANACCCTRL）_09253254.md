## [命令功能](#ZH-CN_MMLREF_0000001609253254)

**适用NF：AMF**

该命令用于查询5G基站接入控制策略。

## [注意事项](#ZH-CN_MMLREF_0000001609253254)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001609253254)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001609253254)

无

## [使用实例](#ZH-CN_MMLREF_0000001609253254)

查询5G基站接入控制策略，执行如下命令：

```
%%LST NGRANACCCTRL:;%%
RETCODE = 0  操作成功

结果如下
--------
      基于位置区基站接入限制开关  =  开启
                  跟踪区群组标识  =  1
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001609253254)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 基于位置区基站接入限制开关 | 该参数用于设置是否开启基于位置区基站接入限制功能。 |
| 跟踪区群组标识 | 该参数用于指定跟踪区群组标识。 |

## [命令功能](#ZH-CN_MMLREF_0209653265)

**适用NF：AMF、SMF、NRF、NSSF、NCG**

该命令用于查询NF或NF服务实例支持的PLMN信息。

## [注意事项](#ZH-CN_MMLREF_0209653265)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209653265)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209653265)

无

## [使用实例](#ZH-CN_MMLREF_0209653265)

运营商A需要查询所有的PLMN支持信息。

```
%%LST ALLOWEDPLMN:;%%
RETCODE = 0  操作成功

结果如下
--------
    索引标识  =  0
    注册类型  =  RegService
  NF实例名称  =  SMF_Instance_0
服务实例标识  =  Service_Instance_0
移动国家代码  =  460
    移动网号  =  01
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209653265)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 索引标识 | 该参数用于指定索引标识。 |
| 注册类型 | 该参数用于指定注册类型。 |
| NF实例名称 | 该参数用于指定NF实例名称。 |
| 服务实例标识 | 该参数用于指定服务实例标识。 |
| 移动国家代码 | 该参数用于指定移动国家代码。 |
| 移动网号 | 该参数用于指定移动网号。 |

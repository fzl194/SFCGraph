#### [命令功能](#ZH-CN_CONCEPT_0000001600441505)

该命令用于查询OSPF引入路由配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001600441505)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600441505)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600441505)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | OSPF进程号 | 可选必选说明：可选参数<br>参数含义：OSPF进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |
| PROTOCOL | 协议分类 | 可选必选说明：可选参数<br>参数含义：用于指定OSPF引入外部路由的路由类型。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- Direct：直连路由。<br>- OSPF：OSPF路由。<br>- Static：静态路由。<br>- BGP：BGP路由。<br>- wlr：无线路由。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600441505)

查询OSPF引入路由配置：

```
LST OSPFIMPORTROUTE:;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600441505)

| 输出项名称 | 输出项解释 |
| --- | --- |
| OSPF进程号 | 该参数指定用户创建的OSPF进程号。 |
| 拓扑标识 | 该参数指定拓扑标识。 |
| 协议分类 | 该参数指定OSPF引入外部路由的协议类型。 |
| 协议进程号 | 该参数指定引入路由协议的进程号，当协议分类配置为OSPF时生效。 |
| 引入路由开销值配置 | 该参数指定允许配置引入的外部路由的开销值。 |
| 引入路由标签配置 | 该参数指定允许配置引入的外部路由的标签。 |
| 引入路由类型配置 | 该参数指定允许配置引入的外部路由的类型。 |
| 路径Cost值 | 该参数指定自治系统外部路径的开销，当引入路由开销值配置为true时生效。 |
| 标签 | 该参数指定外部LSA中的标记，当引入路由标签配置为true时生效。 |
| 引入路由类型 | 该参数指定外部LSA的度量类型，当引入路由Type使能标志配置为true时生效。 |
| 路由策略名称 | 该参数指定引入时依据的路由策略名称。 |
| 允许BGP配置 | 该参数指定允许使能引入IBGP路由，当协议分类配置为BGP时生效。 |

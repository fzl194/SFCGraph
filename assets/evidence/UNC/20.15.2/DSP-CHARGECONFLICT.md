#### [命令功能](#ZH-CN_CONCEPT_0209897020)

**适用NF：PGW-C、SMF**

命令用来查询当前系统中是否存在与业务有冲突的配置。

#### [注意事项](#ZH-CN_CONCEPT_0209897020)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0209897020)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0209897020)

无。

#### [使用实例](#ZH-CN_CONCEPT_0209897020)

查询系统中的计费配置冲突：

```
DSP CHARGECONFLICT:;
```

```

结果如下
------------------------
计费属性 = 0x0800
用户模板名称 = up1
APN名称 = apn2
(结果个数 = 1)

--- END
```

| 用户模板名称 | UserProfile名称。 |

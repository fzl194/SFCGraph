## [命令功能](#ZH-CN_MMLREF_0000001183813636)

该命令用于查询服务化接口链路自动控制功能是否开启。

## [注意事项](#ZH-CN_MMLREF_0000001183813636)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001183813636)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001183813636)

无

## [使用实例](#ZH-CN_MMLREF_0000001183813636)

运营商想要查询服务化接口链路自动控制功能是否开启。

```
LST SBILNKCTRLSW:;

%%LST SBILNKCTRLSW:;%%
RETCODE = 0  操作成功

结果如下
------------------------
       链路控制开关  =  OFF
             起始位  =  9
             结束位  =  24
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001183813636)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 链路控制开关 | 该参数用于设置服务化接口的链路自动控制功能开关。当用户打开该功能，系统会按默认的建链规则建立到对端设备的链路。 |
| 起始位 | 该参数用于设置系统中自动识别本端设备和对端设备的互联情况时比对的起始比特位。网络中对自治管理的NF的实例ID的按UUID格式编值，并自定义了其中的NODE的48bit，各个不同的运营商的自定义可能存在差异，可通过该参数和结束位参数配合修改进行调整。 |
| 结束位 | 该参数用于设置系统中自动识别本端设备和对端设备的互联情况时比对的结束比特位。网络中对自治管理的NF的实例ID按UUID格式编值，并自定义了其中的NODE的48bit，各不同的运营商的自定义可能存在差异，可通过该参数和起始位参数配合修改进行调整。 |

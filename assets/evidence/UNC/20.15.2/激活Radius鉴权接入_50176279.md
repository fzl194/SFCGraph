## [操作步骤](#ZH-CN_OPI_0250176279)

1. 进入 “MML命令行-UNC” 窗口。
2. 配置APN鉴权属性。
  **[SET APNAUTHATTR](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/APN管理/APN鉴权属性/设置APN鉴权属性配置（SET APNAUTHATTR）_28567656.md)**

## [任务示例](#ZH-CN_OPI_0250176279)

任务描述

配置APN为huawei.com的鉴权属性，接入模式为透明鉴权接入，公用的用户名和密码分别为huawei123和123。当用户以该APN接入时， UNC 直接使用该APN下配置的公用的用户名和密码为用户的用户名和密码，到Radius服务器进行鉴权。

脚本

//配置APN鉴权属性。

```
SET APNAUTHATTR: APN="huawei.com", ACCESSMODE=TRANS_AUTH, COMMONUSERNAME="huawei123", COMMONUSERPASS="123", CFMCOMMUSERPASS="123";
```

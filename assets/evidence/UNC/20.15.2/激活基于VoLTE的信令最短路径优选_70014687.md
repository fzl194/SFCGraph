## [操作步骤](#ZH-CN_OPI_0170014687)

1. 进入 “MML命令行-UNC” 窗口。
2. 设置T-ADS查询结果与用户IMS PDN连接相关联。
  [**SET DMCMPT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性配置/设置Diameter兼容性(SET DMCMPT)_26306080.md)
3. **可选：**若仅针对部分用户使用T-ADS查询功能时，则可通过如下步骤设置这部分用户T-ADS查询结果与用户IMS PDN连接的关联性。
    a. 查询系统中Diameter兼容性配置。
      [**LST DMCMPT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性配置/查询Diameter兼容配置(LST DMCMPT)_72345869.md)
    b. 设置T-ADS查询结果与用户IMS PDN连接相关联。
      [**ADD DMCMPTBYIMSI**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性IMSI号段配置/增加IMSI对应的Diameter兼容性(ADD DMCMPTBYIMSI)_72225977.md)
4. **可选：**配置IMSI对应的HLR/HSS接口。
  [**MOD IMSIHLRHSS**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/签约数据管理/接口选择/修改IMSI对应的HLR_HSS接口（MOD IMSIHLRHSS）_26305562.md)

## [任务示例](#ZH-CN_OPI_0170014687)

任务描述

启用T-ADS域选功能，并针对IMSI前缀为“40060”用户以外的其他用户，设置T-ADS查询结果与IMS PDN连接相关联。

脚本

// 进入 “MML命令行-UNC” 窗口。

//设置T-ADS查询结果与PDN连接相关联。

```
SET DMCMPT: TADSIMSPDN=YES;
```

//针对IMSI前缀为“40060”用户，T-ADS查询结果与用户IMS PDN连接无关联。

```
ADD DMCMPTBYIMSI: SUBRANGE=IMSI_PREFIX, IMSIPRE="40060", TADSIMSPDN=NO;
```

//修改IMSI对应的HLR/HSS接口。

```
MOD IMSIHLRHSS: IMSIRANGE=ALL_IMSI, INTRASINGLEREG=ENABLE, INTRAFORCE=ULR, INTERSINGLEREG=ENABLE, INTERFORCE=ULR; 
```

# 删除对端NF实例网络切片标识（RMV PNFNSILST）

- [命令功能](#ZH-CN_MMLREF_0209653278__1.3.1)
- [注意事项](#ZH-CN_MMLREF_0209653278__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0209653278__1.3.4)
- [使用实例](#ZH-CN_MMLREF_0209653278__1.3.5)

## [命令功能](#ZH-CN_MMLREF_0209653278)

**适用NF：AMF、SMF、NSSF、SMSF、NCG**

该命令用于删除本地配置的对端NF实例支持的NSI信息。

## [注意事项](#ZH-CN_MMLREF_0209653278)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0209653278)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209653278)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| NFINSTANCEID | NF实例标识 | 可选必选说明：必选参数<br>参数含义：该参数用于指定对端NF实例标识。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是0~50。NFINSTANCEID参数建议满足以下约束规则：1.如果输入为uuid格式（格式例如：a6a61c6f-0d3a-4221-b1da-424eda3ccf67）只能为A-F、a-f、0-9和“-”的字符。2.如果输入不为uuid格式，长度不能超过18且不允许输入只包含0-9和“.”的字符串，例如：1.2.3.4、不允许输入只包含十六进制数（A-F、a-f、0-9）和“:”的字符串，例如：1::2、FBFF:FFFF:FFFF:FFFF:FFFF:FFFF:FFFF:FFFF。<br>默认值：无<br>配置原则：<br>本参数取值与ADD PNFPROFILE命令中的“NF实例标识”参数取值保持一致时，关联关系生效。 |
| NSI | NF切片实例标识 | 可选必选说明：必选参数<br>参数含义：该参数用于指定对端NF的网络切片实例标识。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是0~50。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209653278)

删除对端NF的NSI信息，NF实例标识为SMF_Instance_0，NSI为10001。

```
RMV PNFNSILST:NFINSTANCEID="SMF_Instance_0",NSI="10001";
```

## [任务示例](#ZH-CN_OPI_0000002094212361)

任务描述

在UPF主备容灾场景下，主用UPF故障恢复后，将备用UPF上的5G LAN会话去活后重新在主UPF上激活。

脚本

//配置软参DWORD1066 BIT29值为1，主UPF故障恢复后，删除备UPF上的5G LAN会话。

```
SET SMFSOFTPARAOFBIT: DT=Dw, DWORDNUM=1066, VALUE=VALUE_1, POSITION=29;
```

//配置主UPF故障恢复处理模式。

```
ADD APNUPRECOVER: APN="apn1", MASTERUPFID="upf1", HANDLINGMODE=BACKMIGRATION, BACKUPFID="upf2";
```

//配置SMF上故障恢复回迁任务的控制扫描速率和扫描间隔。

```
SET UPCSCANRATE: SCANTYPE=BACKMIGRATION, RATE=10, INTERVAL=5;
```

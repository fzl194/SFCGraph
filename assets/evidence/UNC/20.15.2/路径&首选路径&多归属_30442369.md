# 路径&首选路径&多归属

#### [属性](#ZH-CN_TOPIC_0230442369)

| 名称 | 缩略语 | 同义词 |
| --- | --- | --- |
| 路径 | - | Path |
| 首选路径 | - | Primary Path |
| 多归属 | - | Multi-homing |

#### [定义](#ZH-CN_TOPIC_0230442369)

- 路径是一个端点将SCTP分组发送到对端的端点特定目的传送地址的路由。本端传送地址和对端传送地址之间的连接构成一条路径。
- 首选路径是SCTP分组的默认路径。如果可以使用多个目的地址作为到一个端点的目的地址，则这个SCTP端点为多归属节点。如果发出SCTP分组的端点属于多归属节点，定义了目的地址、源地址，就能够更好控制响应数据块返回的路径和数据包被发送的接口。
- 一个SCTP偶联的两个SCTP端点都可以配置多个IP地址，这样一个偶联的两个端点之间具有多条路径，这就是SCTP偶联的多归属。一个偶联可以包括多条路径，但只有一个首选路径。

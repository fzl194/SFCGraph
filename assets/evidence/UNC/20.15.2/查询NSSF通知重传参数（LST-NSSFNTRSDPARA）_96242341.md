## [命令功能](#ZH-CN_MMLREF_0296242341)

**适用NF：NSSF**

该命令用于查询NSSF通知重传参数信息。

## [注意事项](#ZH-CN_MMLREF_0296242341)

无

#### [操作用户权限](#ZH-CN_MMLREF_0296242341)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0296242341)

无

## [使用实例](#ZH-CN_MMLREF_0296242341)

运营商希望查询NSSF通知重传参数信息，执行如下命令。

```
LST NSSFNTRSDPARA:;
%%LST NSSFNTRSDPARA:;%%
RETCODE = 0  操作成功

结果如下
------------------------

      最大通知重传次数 = 1000
通知重传执行周期(分钟) = 50
        重传响应码列表 = 429.500.503.504

(结果个数 = 1)
```

## [输出结果说明](#ZH-CN_MMLREF_0296242341)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 最大通知重传次数 | 该参数用于表示执行通知任务时，通知失败后进行通知重传的最大次数。 |
| 通知重传执行周期(分钟) | 该参数用于表示重传通知任务时两次通知的时间间隔。 |
| 重传响应码列表 | 该参数用于表示需要进行通知重传的响应码列表。 |

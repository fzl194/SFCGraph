# 删除Pre-R8签约QoS配置（RMV PRER8SUBQOS）

- [命令功能](#ZH-CN_MMLREF_0209653173__1.3.1)
- [注意事项](#ZH-CN_MMLREF_0209653173__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0209653173__1.3.4)
- [使用实例](#ZH-CN_MMLREF_0209653173__1.3.5)

## [命令功能](#ZH-CN_MMLREF_0209653173)

**适用NF：GGSN**

该命令用于删除用户的签约QoS属性信息。

## [注意事项](#ZH-CN_MMLREF_0209653173)

命令执行后只对新接入用户生效。

#### [操作用户权限](#ZH-CN_MMLREF_0209653173)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209653173)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SUBQOSINDEX | 用户QoS索引 | 可选必选说明：必选参数<br>参数含义：该参数用于指定用户QoS索引值。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~255。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209653173)

删除SUBQOSINDEX为1的QoS签约信息：

```
RMV PRER8SUBQOS: SUBQOSINDEX=1;
```

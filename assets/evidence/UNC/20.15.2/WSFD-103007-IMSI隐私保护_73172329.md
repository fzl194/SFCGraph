#### [适用NF](#ZH-CN_CONCEPT_0173172329)

AMF

#### [定义](#ZH-CN_CONCEPT_0173172329)

5G网络中，SUPI（Subscription Permanent Identifier）是UE的永久身份标识，包含IMSI或者NAI（Network Access Identifier）。UE通过归属网络的公共密钥加密SUPI生成SUCI（Subscription Concealed Identifier），从而实现IMSI隐私保护。

#### [客户价值](#ZH-CN_CONCEPT_0173172329)

| 受益方 | 受益描述 |
| --- | --- |
| 运营商 | 对用户身份提供保密性措施，提高用户满意度。 |
| 用户 | 在业务流程中使用加密的永久标识，防止用户身份被窃取，提高用户身份信息的安全性。 |

#### [应用场景](#ZH-CN_CONCEPT_0173172329)

- 当UE进行初始注册时，在初始注册请求消息中携带SUCI。
- 当UE收到AMF发送的身份识别请求时，在身份响应消息中携带SUCI。

#### [原理概述](#ZH-CN_CONCEPT_0173172329)

4G时代，在核心网没有给UE分配GUTI前，UE通过IMSI标识自己的身份。安全上下文建立前，IMSI在空口明文传输。5G网络中，UE使用归属网络的公共密钥加密SUPI中的非路由信息，生成SUCI。在初始注册流程（UE还没有被分配5G-GUTI）或者UE收到网络侧发送的身份识别请求（网络侧无法通过5G-GUTI识别UE身份）时，UE在NAS消息中发送SUCI。获取到SUCI后，UDM使用私钥将SUCI解密为SUPI，获取用户的真实身份及签约信息。

**图2** SUCI结构

<br>

![](WSFD-103007 IMSI隐私保护_73172329.assets/zh-cn_image_0173172408_2.png)

- SUPI Type：SUPI的类型，取值0-7。SUPI为IMSI时，取值为0；SUPI为NAI时，取值为1。
- Home Network Identifier：用户归属网络标识。当SUPI Type为IMSI时，包括MCC和MNC。
- Routing Indicator：路由指示符，由归属网络运营商分配并存储在USIM中，与Home Network Identifier一起用于寻址AUSF和UDM。
- Protection Scheme ID：SUPI加密为SUCI过程中使用的加密算法标识，取值0-15。
- Home Network Public Key ID：SUPI加密为SUCI过程中使用的归属网络公钥标识，取值0-255。
- Scheme Output：SUPI经过加密算法加密后的内容，格式取决于所使用的加密算法。

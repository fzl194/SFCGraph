## [命令功能](#ZH-CN_MMLREF_0000001364670737)

**适用NF：NSSF**

该命令用于增加默认AMF集所支持的S-NSSAI范围。

## [注意事项](#ZH-CN_MMLREF_0000001364670737)

- 该命令执行后立即生效。

- 该命令配置错误会影响切片选择功能。
- 主备或双活组网的场景下，如果需要配置此命令，则两个NSSF上均需执行此命令，且配置参数一致。

- 最多可输入10000条记录。

#### [操作用户权限](#ZH-CN_MMLREF_0000001364670737)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [使用实例](#ZH-CN_MMLREF_0000001364670737)

假如运营商希望增加MCC为"460"、MNC为"03"、AMFREGIONID为"01"、AMFSETID为"001"的AMFSET作为默认AMFSET支持TAIMCC为"460"、TAIMNC为"20"、SST为1、SD区间为"010101"到"01FFFF"的切片区间，则需要执行下列命令。

```
ADD NSSFDFTAMFSET: INDEX=1, TAIMCC="460", TAIMNC="20", SST=1, SDSTART="010101", SDEND="01FFFF", MCC="460", MNC="03", AMFREGIONID="01", AMFSETID="001";
```

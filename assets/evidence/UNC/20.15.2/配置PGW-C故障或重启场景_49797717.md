## [操作场景](#ZH-CN_OPI_0249797717)

PGW-C故障场景下，SGW-C本地删除承载，并且通知MME网元PGW-C故障。

> **说明**
> 适用于SGW-C。

## [必备事项](#ZH-CN_OPI_0249797717)

前提条件

- 请仔细阅读[WSFD-201204 VoLTE承载故障快速恢复特性概述](../../特性概述_86652960.md)。
- 完成配置普通APN。
- 完成 [激活IMS功能](../../../WSFD-201103 IMS功能/激活IMS功能_91473678.md)

## [操作步骤](#ZH-CN_OPI_0249797717)

1. 进入 “MML命令行-UNC” 窗口。
2. **可选：**配置T3N3时间。
  [**SET GTPCFUNCPARA**](../../../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/GTP-C接口配置管理/GTP-C功能参数/设置GTP-C功能参数（SET GTPCFUNCPARA）_09653666.md)
  > **说明**
  > MME上配置的T3-RESPONSE(s)*N3-REQUEST(times)的值需大于SGW-C该值的配置，否则PGW-C重选功能无法生效。
3. 配置基于整机的防闪断定时器时长。
  [**SET FASTRECOVERY**](../../../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/网络管理/业务快速恢复/全局参数/设置全局业务快速恢复配置（SET FASTRECOVERY）_31453525.md)
4. 基于整机配置开启PGW-C restoration功能。
  [**SET GTPCFUNCPARA**](../../../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/GTP-C接口配置管理/GTP-C功能参数/设置GTP-C功能参数（SET GTPCFUNCPARA）_09653666.md)
5. **可选：**配置SGW-C中Echo消息的重发时间间隔和次数。
  [**SET GTPCFUNCPARA**](../../../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/GTP-C接口配置管理/GTP-C功能参数/设置GTP-C功能参数（SET GTPCFUNCPARA）_09653666.md)
  > **说明**
  > 如果SGW-C的T3N3时长大于MME的超时时长，可能导致MME的PGW-C重选功能失效，需要单独配置来规避这种情况。

## [任务示例](#ZH-CN_OPI_0249797717)

任务描述

使能PGW-C故障后的VoLTE承载故障快速恢复功能。

脚本

// 进入 “MML命令行-UNC” 窗口。

//配置T3N3时间。

```
SET GTPCFUNCPARA:T3=3,N3=5;
```

//基于整机的防闪断定时器时长为120秒。

```
SET FASTRECOVERY:PRTFRQBRKTIME=120;
```

//基于整机配置开启PGW-C restoration功能。

```
SET GTPCFUNCPARA:V2ECHOSW=ON,PRNSWITCH=ON;
```

//可选：配置SGW-C中Echo消息的T3N3值。

```
SET GTPCFUNCPARA:T3=10,N3=5;
```

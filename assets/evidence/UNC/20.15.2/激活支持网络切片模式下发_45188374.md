## [操作场景](#ZH-CN_OPI_0245188374)

本操作指导介绍在运行网络中激活 支持网络切片模式下发 的操作过程。

AMF支持基于PLMN（Public Land Mobile Network）配置NSSAI Inclusion Mode信元，来决定AMF控制UE在创建RRC（Radio Resource Control）连接时是否携带网络切片信息及携带此信息的方式。

> **说明**
> 适用于AMF。

## [操作步骤](#ZH-CN_OPI_0245188374)

1. 进入 “MML命令行-UNC” 窗口。
2. 配置网络切片包含模式。
  **[ADD NGMMSUBDATA](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/移动性管理/签约数据管理/增加用户移动性管理参数（ADD NGMMSUBDATA）_09651362.md)**
3. 配置下发NIM给UE的判断策略。
  **[SET NGMMFUNC](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/移动性管理/MM协议参数管理/5G移动性管理/设置5G移动性管理功能（SET NGMMFUNC）_09653748.md)**

## [任务示例](#ZH-CN_OPI_0245188374)

任务描述

启用支持网络切片模式下发功能，配置网络切片包含模式为模式A，并使用标准模式向UE下发NIM的判断策略。

脚本

//配置网络切片包含模式。

```
ADD NGMMSUBDATA: SUBRANGE=HOME_USER, NIM=MODE_A;
```

//配置下发NIM给UE的判断策略。

```
SET NGMMFUNC: NIMPLCY=STANDARD_MODE;
```

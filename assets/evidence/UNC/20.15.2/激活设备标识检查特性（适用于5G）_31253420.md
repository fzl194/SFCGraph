- 完成AMF与5G-EIR对接，AMF与5G-EIR的对接配置请参考[SBI接口（N8/N11/N12/N14/N15/N17/N20/N22/N50/NLs/NLg/N11mb）配置](../../../../../初始配置/UNC初始配置与调测/组网对接配置/配置AMF&MME&SGSN/配置AMF/配置SBI接口_82467016.md)或[配置和NF直连](../../../../../初始配置/UNC初始配置与调测/组网对接配置/配置SMF&PGW-C&SGW-C&GGSN/配置SMF/配置和NF直连_31443845.md)。
- 5G-EIR已激活设备标识检查功能。
- 完成加载License。

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md) :
3. 配置5G网络PEI获取和检查策略。
  [**ADD NGPEIPLCY**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/业务安全管理/PEI策略管理/增加5G PEI策略（ADD NGPEIPLCY）_09651346.md)

## [任务示例](#ZH-CN_OPI_0000001131253420)

任务描述

在5G网络中针对IMSI前缀为12303的用户检查IMEISV，且支持EIR重选或重定向。不允许黑名单用户接入，但允许用户在如下场景接入：PEI获取失败或检查超时、EIR发现失败或返回失败、灰名单用户、紧急注册用户的PEI检查结果为禁止。

脚本

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2EIR02", SWITCH=ENABLE;
```

//设置5G网络IMSI前缀为12303的用户的PEI检查策略。

```
ADD NGPEIPLCY: SUBRANGE=IMSI_PREFIX, IMSIPRE="12303", GETIMEIPLC=IMEISV, CHKIMEIPLC=IMEISV, PROTTYPE=INIT_REG-1&MOBL_INTER_REG-1&REG_AFT_INTERHO-1&IDLE_SYSCHG_REG-1&CONN_SYSCHG_REG-1, PEIACCESS=YES, CTFLAG=YES, GRALLOW=YES, BALLOW=NO, EMERGENCYSW=YES, DISCFAIL=YES, CHKFAIL=YES, RETRYSW=YES, REDIRECTSW=YES;
```

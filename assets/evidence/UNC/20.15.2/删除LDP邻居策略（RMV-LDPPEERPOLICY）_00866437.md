- [注意事项](#ZH-CN_CONCEPT_0000001600866437__1.3.2.1)

#### [命令功能](#ZH-CN_CONCEPT_0000001600866437)

该命令用于删除LDP邻居策略。

#### [注意事项](#ZH-CN_CONCEPT_0000001600866437)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600866437)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600866437)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| VRFNAME | VPN实例名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定VPN实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。<br>默认值：_public_ |
| PEERID | 对等体的LSR ID | 可选必选说明：必选参数<br>参数含义：该参数用于指定对等体的LSR ID。<br>数据来源：本端规划<br>取值范围：IPv4地址类型。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600866437)

删除LDP邻居策略：

```
RMV LDPPEERPOLICY:VRFNAME="_public_",PEERID="192.168.1.1";
```

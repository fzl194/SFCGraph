当对全局所有业务采用同样的计费参数时，可以使用全局CCT模板进行计费。

## [任务示例](#ZH-CN_OPI_0193029782)

任务描述

SMF通过NRF发现选择CHF，使用全局CCT模板“global”进行计费，修改全局CCT模板的QHT参数为5，上下行都使用在线计费、且费率标识为（RG，SID）=（10,20）的URR。SMF在建立PDU会话时发送ChargingDataRqueset与CHF交互，不预申请配额。

脚本

//配置在线计费、且费率标识为（RG，SID）=（10,20）的URR，并将该URR设置为全局上下行URR。

```
ADD URR: URRNAME="urr1", URRID=1, USAGERPTMODE=ONLINE, ONLCOMPOUNDTYPE=RGSID, ONLINERG=10, ONLINESID=20, ONLMETERINGTYPE=VOLUME; 
SET GLBURRGROUP:UPONLURRNAME = "urr1", DNONLURRNAME = "urr1";
```

//修改全局通用参数CCT模板的QHT参数值为5。

```
MOD CCT:CCTMPLTNAME="global",QHT=5;
```

//配置建立PDU会话时发送ChargingDataRqueset与CHF交互。

```
SET CHFINIT: CCTMPLTNAME="global", CHFINIT=SENDREQ, CCRINITRGNUM=10;
```

//配置Session级/PDU级Trigger。

```
ADD PDUTRIGGER: CCTMPLTNAME="global", QOSCHG=IMMEDIATE, SRVNDCHG=DEFERRED, RATCHG=IMMEDIATE;
ADD RGTRIGGER: CCTMPLTNAME="global", ULCHG=NONREPORT, PRACHG=IMMEDIATE, PSDATAOFFCHG=IMMEDIATE;
```

//配置SMF超时未收到ChargingDataResponse消息时的动作。

```
SET FAILHANDLING: CCTMPLTNAME="global", FHACTION=RETRY_AND_TERM, FAILOVERSUP=ENABLE, AUTOFAILBACK=ENABLE, TXTIMER=5, HOLDINGTIME=10;
```

//配置SMF收到PDU级异常返回码时的动作。

```
ADD PDUSCACT: CCTMPLTNAME="global", CODETYPE=VALUE, STATUSCODE=404, SCACT=CONTINUE;
```

//配置SMF收到RG级异常返回码时的动作。

```
ADD RGRCACT: CCTMPLTNAME="global", RCCODE=QUOTAMNOTAPPL, RCACT=BLCK_IMMED_RPT, BLKTIMER=30;
```

#### [命令功能](#ZH-CN_TOPIC_0000001953585048)

使用CPU节能功能时，通过此命令可以查询全局的调频和休眠策略。

> **说明**
> 该命令仅在Full-stack虚机场景下支持。

#### [注意事项](#ZH-CN_TOPIC_0000001953585048)

- 网元ID必须在系统中存在。

#### [参数说明](#ZH-CN_TOPIC_0000001953585048)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| MEID | 网元ID | 可选必选说明：必选参数。<br>参数含义：网元ID，可以通过<br>[**LST ME**](../../系统管理/版本信息/查询网元配置信息（LST ME）_47084797.md)<br>获取。<br>取值范围：0～65535<br>默认值：无。<br>配置原则：无。 |

#### [使用实例](#ZH-CN_TOPIC_0000001953585048)

查询 “网元ID” 为 “222” 的网元的CPU节能策略。

```
%%LST CPUECOPOLICY: MEID=222;%%
RETCODE = 0  操作成功

操作结果如下
------------
CPU休眠策略  =  浅休眠
CPU调频策略  =  开启
(结果个数 = 1)

---    END
```

#### [输出结果说明](#ZH-CN_TOPIC_0000001953585048)

命令执行正常，会返回命令执行成功的提示信息及当前网元的CPU节能策略，输出结果项信息项目如 [表1](#ZH-CN_TOPIC_0000001953585048__table819976173217) 。

命令执行失败则返回相应的错误，参见 [表2](#ZH-CN_TOPIC_0000001953585048__table18477416112710) ；命令执行异常，请联系华为技术支持处理。

*表1 输出项说明*

| 输出项名称 | 输出项解释 |
| --- | --- |
| CPU休眠策略 | 用于返回查询的全局CPU休眠策略。<br>取值说明：<br>- 关闭：CPU休眠策略为关闭。<br>- 浅休眠：CPU休眠策略为浅休眠。<br>- 深休眠：CPU休眠策略为深休眠。<br>- 未配置：未配置CPU休眠策略。 |
| CPU调频策略 | 用于返回查询的全局CPU调频策略。<br>取值说明：<br>- 关闭：CPU调频策略为关闭。<br>- 开启：CPU调频策略为开启。<br>- 未配置：未配置CPU调频策略。 |

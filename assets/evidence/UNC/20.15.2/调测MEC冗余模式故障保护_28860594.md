# 调测MEC冗余模式故障保护

- [操作场景](#ZH-CN_OPI_0228860594__1.3.1)
- [必备事项](#ZH-CN_OPI_0228860594__1.3.2)
- [操作步骤](#ZH-CN_OPI_0228860594__1.3.3)

## [操作场景](#ZH-CN_OPI_0228860594)

本操作指导介绍在运行网络中调测MEC冗余模式故障保护特性的操作过程。

## [必备事项](#ZH-CN_OPI_0228860594)

前提条件

- 完成[配置“主锚点UPF双主负荷分担部署模式”](配置“主锚点UPF双主负荷分担部署模式”_28860593.md)。

数据

| 类别 | 参数名称 | 取值样例 | 说明 |
| --- | --- | --- | --- |
| 用户信息 | 国际移动用户识别码（IMSI） | 123000123456789 | - |
| 用户DNN信息 | DNN名称（DNN） | Huawei.com | MEC区域内的DNN |
| 用户位置信息 | 用户位置信息TAI（用户位置信息TAI） | 123010006 | MEC覆盖区域对应的TA |
| ULCL UPF+辅锚点UPF1<br>ULCL UPF+辅锚点UPF2 | IP地址 | 192.168.126.11<br>192.168.126.12 | 负荷分担组网模式下两个UPF的IP地址 |

## [操作步骤](#ZH-CN_OPI_0228860594)

1. 用户进入MEC覆盖区域，发起PDU会话建立流程。
2. 进入 “MML命令行-UNC” 窗口。
3. 查询用户的PDU会话信息。
  **[DSP PDUSESSION](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/接入管理运维/查询PDP上下文信息/显示PDU会话（DSP PDUSESSION）_09653215.md)**
  预期结果：选择到IP地址为192.168.126.11的 **ULCL UPF+辅锚点UPF1** 。

  ```
  DSP PDUSESSION:QUERYTYPE=IMSI, IMSI="123000123456789", DSPINFOTYPE=DETAILED, WLNETWKTYPE=NW5G, PDUSESSIONID=5, QFI=1;
  RETCODE = 0  操作成功  

  结果如下 
  --------------  
              国际移动用户标识 = 123000123456789
                              ...
               用户位置信息TAI = 123010006
                              ...
             UP的PFCP IPv4地址 = 192.168.126.11
                              ...
                      ULCL标识 = support ULCL
                              ...
  (结果个数 = 1) 
  ---    END
  ```
4. 构造故障场景：断开SMF和 **ULCL UPF+辅锚点UPF1** 的连接。
5. 查询用户的PDU会话信息。
  **[DSP PDUSESSION](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/接入管理运维/查询PDP上下文信息/显示PDU会话（DSP PDUSESSION）_09653215.md)**
  预期结果：选择到IP地址为192.168.126.12的 **ULCL UPF+辅锚点UPF2** 。

  ```
  DSP PDUSESSION:QUERYTYPE=IMSI, IMSI="123000123456789", DSPINFOTYPE=DETAILED, WLNETWKTYPE=NW5G, PDUSESSIONID=5, QFI=1;
  RETCODE = 0  操作成功  

  结果如下 
  --------------  
              国际移动用户标识 = 123000123456789
                              ...
               用户位置信息TAI = 123010006
                              ...
             UP的PFCP IPv4地址 = 192.168.126.12
                              ...
                      ULCL标识 = support ULCL
                              ...
  (结果个数 = 1) 
  ---    END
  ```

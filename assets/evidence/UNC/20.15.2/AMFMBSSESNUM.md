## [命令功能](#ZH-CN_MMLREF_0000001388107262)

**适用NF：AMF**

该命令用于查询AMF组播广播会话数。

## [注意事项](#ZH-CN_MMLREF_0000001388107262)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001388107262)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001388107262)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SESSIONTYPE | 会话类型 | 可选必选说明：可选参数<br>参数含义：该参数用于指定查询的会话类型。<br>数据来源：本端规划<br>取值范围：<br>- BROADCAST（广播）<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001388107262)

查看AMF上的广播会话数，执行如下命令：

```
%%DSP AMFMBSSESNUM: SESSIONTYPE=BROADCAST;%%
RETCODE = 0  操作成功

结果如下
------------------------
POD ID     MBS会话数  

gtp-pod-0  2                       
total      2                 
(结果个数 = 2)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001388107262)

| 输出项名称 | 输出项解释 |
| --- | --- |
| POD ID | 该参数用于标识系统中的POD ID。 |
| MBS会话数 | 该参数表示指定会话类型的MBS会话数。 |

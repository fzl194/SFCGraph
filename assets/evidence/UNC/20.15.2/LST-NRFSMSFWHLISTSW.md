## [命令功能](#ZH-CN_MMLREF_0000001271463514)

**适用NF：NRF**

该命令用于查询SMSF白名单功能是否开启。

## [注意事项](#ZH-CN_MMLREF_0000001271463514)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001271463514)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001271463514)

无

## [使用实例](#ZH-CN_MMLREF_0000001271463514)

查询SMSF白名单开关是否开启。

```
LST NRFSMSFWHLISTSW:;
%%LST NRFSMSFWHLISTSW:;%%
RETCODE = 0  操作成功

结果如下
---------
SMSF白名单开关 =  打开
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001271463514)

| 输出项名称 | 输出项解释 |
| --- | --- |
| SMSF白名单开关 | 该参数用于表示SMSF白名单功能是否打开。 当开关设置为FUNC_ON时，NF基于NFtype发现SMSF，NRF仅会与SMSF白名单中的网元进行匹配；当开关设置为FUNC_OFF时，NRF将会正常匹配SMSF。 |

#### [命令功能](#ZH-CN_CONCEPT_0129627089)

该命令用于查询隧道组信息，辅助定位隧道转发失败的问题。

#### [注意事项](#ZH-CN_CONCEPT_0129627089)

该命令批量下发可能导致执行超时。

manage-ug；system-ug；monitor-ug

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [操作用户权限](#ZH-CN_CONCEPT_0129627089)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0129627089)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SRVVNFCID | 业务VNFCID | 可选必选说明：可选参数<br>参数含义：该参数用于指定业务VNFC的唯一标识，通过在业务VNFC下执行<br>**[LST NODE](../../../../单体服务公共功能管理/系统管理/基础参数/查询节点信息/查询节点信息（LST NODE）_59103764.md)**<br>获得，所得NODE ID（节点ID）即为业务VNFCID。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0~4294967295。<br>默认值：无<br>配置原则：无 |
| TNGRPID | 隧道组ID | 可选必选说明：可选参数<br>参数含义：该参数用于表示业务VNFC定义的隧道组的标识。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0~4294967295。<br>默认值：无<br>配置原则：无 |
| TNGRPNAME | 隧道组名称 | 可选必选说明：可选参数<br>参数含义：该参数用于表示业务VNFC定义的隧道组名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围0~15。<br>默认值：无<br>配置原则：使用DSP LBTN命令查看可用的隧道组名称 |

#### [使用实例](#ZH-CN_CONCEPT_0129627089)

查询隧道组。

DSP LBTNGROUP:;

```
%%DSP LBTNGROUP:;%%
RETCODE = 0  操作成功。

操作结果如下：
--------------
  隧道组名称    =  tnc_grp1
  隧道组ID      =  1
  业务VNFCID    =  5
  是否共享隧道  =  共享
  隧道传输QOS   =  1
  探测开关状态  =  打开
  上报状态开关  =  打开
  心跳间隔时长  =  1
  心跳重传次数  =  3
  隧道组状态    =  正常
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0129627089)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 隧道组名称 | 参数含义：该参数用于表示业务VNFC定义的隧道组名称。<br>取值范围：字符串类型，输入长度范围为0~15。 |
| 隧道组ID | 参数含义：该参数用于表示业务VNFC定义的隧道组的标识。<br>取值范围：整数类型，取值范围为0~4294967295。 |
| 业务VNFCID | 参数含义：该参数用于指定业务VNFC的唯一标识。<br>取值范围：整数类型，取值范围为0~4294967295。 |
| 是否共享隧道 | 参数含义：该参数用于表示是否共享隧道。<br>取值范围：<br>- “共享”<br>- “非共享” |
| 隧道传输QoS | 参数含义：该参数用于表示隧道传输QoS。<br>取值范围：整数类型，取值范围为0~63。 |
| 探测开关状态 | 参数含义：该参数用于表示业务VNFC是否要求CSLB发起隧道探测。<br>取值范围：<br>- “打开”<br>- “关闭” |
| 上报状态开关 | 参数含义：该参数用于表示业务VNFC是否要求CSLB上报隧道组状态。<br>取值范围：<br>- “打开”<br>- “关闭” |
| 心跳间隔时长 | 参数含义：该参数用于表示业务VNFC下发隧道组信息时配置的隧道组下所有隧道探测的心跳间隔时长。<br>取值范围：整数类型，取值范围为5~100，单位100毫秒。 |
| 心跳重传次数 | 参数含义：该参数用于表示业务VNFC下发隧道组信息时配置的隧道组下所有隧道探测的心跳重传次数。<br>取值范围：整数类型，取值范围为1~60。 |
| 隧道组状态 | 参数含义：该参数用于表示隧道组状态。<br>取值范围：<br>- “初始态”：隧道组下没有正常状态的隧道且有初始态的隧道，则该隧道组状态为初始态。<br>- “正常”：隧道组下只要有一条隧道状态为正常，则该隧道组的状态为正常。<br>- “故障”：隧道组下所有隧道的状态均为故障，则该隧道组的状态为故障。 |

## [命令功能](#ZH-CN_MMLREF_0000001232102615)

**适用NF：PGW-C、SMF、GGSN**

该命令用于查询DHCP服务器组与地址池组的关联。

## [注意事项](#ZH-CN_MMLREF_0000001232102615)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001232102615)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001232102615)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| POOLGRPNAME | 地址池组名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定地址池组名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~79。不支持空格及特殊字符“_”、“#”、“$”和“&”等，由“-”、数字、大小写字母和“.”组成，不能以“.”开头且不能出现连续两个“.”，不区分大小写。<br>默认值：无<br>配置原则：<br>该参数使用ADD ADDRPOOLGRP命令配置生成。 |

## [使用实例](#ZH-CN_MMLREF_0000001232102615)

- 当需要查看远端地址池组与DHCP服务器组的绑定关系时，使用该命令：
  ```
  %%LST DHCPBINDPOOLGRP:POOLGRPNAME="poolgrp1";%%
  RETCODE = 0  操作成功

  结果如下
  --------
      地址池组名称  =  poolgrp1
  DHCP服务器组名称  =  group1
  (结果个数 = 1)

  ---    END
  ```
- 当需要查看所有远端地址池组和DHCP服务器的绑定关系时，使用该命令：
  ```
  %%LST DHCPBINDPOOLGRP:;%%
  RETCODE = 0  操作成功

  结果如下
  --------
  地址池组名称       DHCP服务器组名称

  poolgrp1           group1         
  poolgrp2           group2         
  (结果个数 = 2)

  ---    END
  ```

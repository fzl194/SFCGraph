## [命令功能](#ZH-CN_MMLREF_0000001144461347)

**适用NF：NSSF**

该命令用于查询NSSF响应切片最大长度信息。

## [注意事项](#ZH-CN_MMLREF_0000001144461347)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001144461347)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001144461347)

无

## [使用实例](#ZH-CN_MMLREF_0000001144461347)

若运营商希望查询所有的数据，执行下列命令。

```
LST NSSFRTNSMAXNUM:;
%%LST NSSFRTNSMAXNUM:;%%
RETCODE = 0 执行成功

结果如下
------------------------
Allowed切片最大数量  =  8
(结果个数 = 1)
```

## [输出结果说明](#ZH-CN_MMLREF_0000001144461347)

| 输出项名称 | 输出项解释 |
| --- | --- |
| Allowed切片最大数量 | 该参数用于控制切片选择流程响应中Allowed切片最大数量。 |

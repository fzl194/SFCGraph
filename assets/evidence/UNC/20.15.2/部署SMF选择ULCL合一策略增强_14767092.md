## [操作步骤](#ZH-CN_OPI_0000001314767092)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开ALWAYS分流开关。
  ```
  ADD DNAIINFO:DNAI="
  huawei
  ",ALWAYSSHUNTSW=ENABLE;
  ```
3. 增加指定DNAI服务区域的UL CL部署模式。
  ```
  ADD ULCLAREADPMODE:APN="
  huawei.com
  ",DNAI="
  huawei
  ",DNAREANAME="
  dnarea1
  ",ULCLDEPLOYMODE=
  AUXSHUNTPREFER
  ;
  ```
4. 增加指定DNAI的UL CL部署模式。
  ```
  ADD ULCLDNAIDPMODE:APN="
  huawei.com
  ",DNAI="
  huawei
  ",ULCLDEPLOYMODE=
  PSASHUNTMUST
  ;
  ```
5. 增加指定APN的UL CL部署模式。
  ```
  ADD ULCLAPNDPMODE:APN="
  huawei.com
  ",ULCLDEPLOYMODE=
  PSASHUNTMUST
  ;
  ```

## [任务示例](#ZH-CN_OPI_0000001314767092)

任务描述

SMF为某一校园用户配置DNAI服务区域，DNAI，DNN三种粒度的UL CL分流节点部署策略。在DNAI服务区域内优先选择辅锚点作为分流ULCL，APN为huawei.com，DNAI名称为huawei，DNAI服务区域为dnarea1。在DNAI内必须选择主锚点作为分流ULCL，APN为huawei.com，DNAI名称为huawei。在DNN内必须选择主锚点作为分流ULCL，APN为huawei.com。

脚本

```
//打开ALWAYS分流开关。
ADD DNAIINFO: DNAI="huawei", ALWAYSSHUNTSW=ENABLE;
//增加指定DNAI服务区域的UL CL部署模式。
ADD ULCLAREADPMODE:APN="huawei.com",DNAI="huawei",DNAREANAME="dnarea1",ULCLDEPLOYMODE=AUXSHUNTPREFER;
//增加指定DNAI的UL CL部署模式。
ADD ULCLDNAIDPMODE:APN="huawei.com",DNAI="huawei",ULCLDEPLOYMODE=PSASHUNTMUST;
//增加指定DNN的UL CL部署模式。
ADD ULCLAPNDPMODE:APN="huawei.com",ULCLDEPLOYMODE=PSASHUNTMUST;
```

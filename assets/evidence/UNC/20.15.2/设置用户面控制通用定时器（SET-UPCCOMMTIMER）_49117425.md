## [命令功能](#ZH-CN_MMLREF_0000001149117425)

**适用NF：SMF、SGW-C、PGW-C、GGSN**

该命令用于设置用户面控制的通用定时器的时长。

## [注意事项](#ZH-CN_MMLREF_0000001149117425)

- 该命令执行后立即生效。

- 系统部署完成后，已经存在初始记录，参数的初始记录值如下表：

| BUFFINATIMER | DELAYRELTIMER |
| --- | --- |
| 3 | 1000 |

#### [操作用户权限](#ZH-CN_MMLREF_0000001149117425)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001149117425)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| BUFFINATIMER | 转发隧道不活动定时器(秒) | 可选必选说明：可选参数<br>参数含义：该参数用于设置SMF源侧ULCL UPF与目标侧ULCL UPF间的N9转发隧道的不活动定时器的时长。该定时器小于SMCOMMTIMER设置的Service Request（TREDIRECTREL参数）、Xn切换（TXNREL参数）、N2切换（TINDFWDTNL参数）流程释放阶段定时器时生效。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是1~16，单位是秒。<br>默认值：无。执行命令并不输入该参数时，该参数保持系统当前配置不变，可通过LST UPCCOMMTIMER查询当前参数配置值。<br>配置原则：无 |
| DELAYRELTIMER | 延迟释放定时器(毫秒) | 可选必选说明：可选参数<br>参数含义：该参数用于设置UPF通知SMF流量乱序需延迟释放N4会话时，延迟释放定时器的时长。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~2000，单位是毫秒。<br>默认值：无。执行命令并不输入该参数时，该参数保持系统当前配置不变，可通过LST UPCCOMMTIMER查询当前参数配置值。<br>配置原则：<br>该值为0时表示延迟释放不生效。 |

## [使用实例](#ZH-CN_MMLREF_0000001149117425)

设置转发隧道不活动定时器时长为10秒，延迟释放定时器为2秒：

```
SET UPCCOMMTIMER: BUFFINATIMER=10, DELAYRELTIMER=2000;
```

## [操作场景](#ZH-CN_OPI_0166313816)

本操作指导介绍在运行网络中激活基于用户群的灵活鉴权特性的操作过程。

通过基于用户群的灵活鉴权功能， UNC 可以针对不同的用户群的属性使用差异化的安全策略。

> **说明**
> 适用于 SGSN、 MME。

## [必备事项](#ZH-CN_OPI_0166313816)

前提条件

- 请仔细阅读[WSFD-103005 基于用户群的灵活鉴权特性概述](特性概述_66289237.md)。

- 安全功能被启用：
    - 在Gb模式下，将[**ADD GBAUTHCIPH**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/用户安全管理/Gb模式用户安全参数/增加Gb模式用户安全参数（ADD GBAUTHCIPH）_26145642.md)命令中的“SECPLC”参数设置为“AUTHANDPROTECTED”或“AUTHONLY”。
    - 在Iu模式下，将[**ADD IUAUTHCIPH**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/用户安全管理/Iu模式用户安全参数/增加Iu模式用户安全参数(ADD IUAUTHCIPH)_72225327.md)命令中的“SECPLC”参数设置为“AUTHANDPROTECTED”或“AUTHONLY”。
    - 在S1模式下，将[**ADD S1USRSECPARA**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/用户安全管理/S1模式用户安全参数/增加S1模式用户安全配置(ADD S1USRSECPARA)_26305460.md)命令中的“SECPLC”参数设置为“AUTHANDPROTECTED”；“OPTIONAL”参数设置为“YES”。

- 完成加载License。

## [任务示例](#ZH-CN_OPI_0166313816)

任务描述

- 场景一：S1模式下，启用不同PLMN的灵活鉴权功能。
  添加IMSI前缀为“30801”，安全策略为“既鉴权又保护”，鉴权事件为“ATTACH-1”，鉴权事件上限为“3”的HOME PLMN的灵活鉴权参数。
  添加IMSI前缀为“30802”，安全策略为“既鉴权又保护”，鉴权事件为“ATTACH-1&INTRA_TAU-1”，鉴权事件上限为“2”的CONNECT PLMN的灵活鉴权参数。
- 场景二：启用特定用户群的灵活鉴权功能。
  Gb模式下，添加IMSI前缀为“308010001”，安全策略为“既鉴权又保护”，鉴权事件为“PTMSI_ATTACH-1&IMSI_ATTACH-1”，鉴权事件上限为“3”的特定用户群的灵活鉴权参数。
  Iu模式下，添加IMSI前缀为“308010000000001”，安全策略为“既鉴权又保护”，鉴权事件为“INTRA_RAU-1&IMSI_ATTACH-1”，鉴权事件上限为“2”的特定用户的灵活鉴权参数。

脚本

- 脚本一
  //开启基于用户群的灵活鉴权功能开关。
  ```
  SET LICENSESWITCH: LICITEM="LKV2FLEXAUTH02", SWITCH=ENABLE;
  ```
  //添加HOME PLMN的灵活鉴权参数。
  ```
  ADD S1USRSECPARA:IMSIPRE="30801",SECPLC=AUTHANDPROTECTED,OPTIONAL=YES,AUTHEVENT=ATTACH-1,AUTHEVENTTHRESHOLD=3; 
  ```
  //添加CONNECT PLMN的灵活鉴权参数。
  ```
  ADD S1USRSECPARA:IMSIPRE="30802",SECPLC=AUTHANDPROTECTED,OPTIONAL=YES,AUTHEVENT=ATTACH-1&INTRA_TAU-1,AUTHEVENTTHRESHOLD=2; 
  ```

- 脚本二
  //开启基于用户群的灵活鉴权功能开关。
  ```
  SET LICENSESWITCH: LICITEM="LKV2FLEXAUTH02", SWITCH=ENABLE;
  ```
  //Gb模式下，添加特定用户群的灵活鉴权参数。
  ```
  ADD GBAUTHCIPH:IMSIPRE="308010001",SECPLC=AUTHANDPROTECTED,AUTHEVENT=PTMSI_ATTACH-1&IMSI_ATTACH-1,AUTHEVENTTHRESHOLD=3;
  ```
  //Iu模式下，添加特定用户的灵活鉴权参数。

  ```
  ADD IUAUTHCIPH:IMSIPRE="308010000000001",SECPLC=AUTHANDPROTECTED,AUTHEVENT=INTRA_RAU-1&IMSI_ATTACH-1,AUTHEVENTTHRESHOLD=2;
  ```

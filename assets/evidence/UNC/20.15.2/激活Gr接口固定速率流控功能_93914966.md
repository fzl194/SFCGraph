## [操作场景](#ZH-CN_OPI_0193914966)

本操作指导介绍在运行网络中激活Gr接口固定速率流控功能的操作过程。

Gr接口固定流速控制功能是指在Gr接口拥塞或HLR/STP过载时， UNC 通过配置Gr接口的流控速率阈值，控制Attach、Inter RAU、Intra RAU(LTE->GERAN/UTRAN)流程首次到HLR/STP的取鉴权集或位置更新请求的速率，防止Gr接口拥塞或HLR/STP过载，以确保整网接入成功率；对于超过流控速率阈值的业务请求， UNC 将会给UE返回Attach Reject或RAU Reject消息，此消息中携带 “#22（Congestion）” 原因值，UE在收到此消息后一段时间，将再次尝试重新接入。

> **说明**
> 适用于 SGSN 。

## [操作步骤](#ZH-CN_OPI_0193914966)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开固定速率流控开关。
  [**SET SOFTPARAOFBIT**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/操作维护/软件参数管理/软件参数比特位/设置软件参数表比特位(SET SOFTPARAOFBIT)_72345783.md)
3. 设置固定流控速率。
  [**SET SOFTPARA**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/操作维护/软件参数管理/软件参数/设置软件参数表(SET SOFTPARA)_26146182.md)
  > **说明**
  > 固定流控速率的取值参照 [Gr接口固定速率流控功能](../Gr接口固定速率流控功能_93914965.md) 。

## [任务示例](#ZH-CN_OPI_0193914966)

任务描述

升级过程中系统重启，预计会有大量用户发送请求到HLR的读取数据，因此系统需要在重启前开启Gr接口固定速率流控功能并设置固定流控速率，AIR、ULR消息将会按照预设的速率接入。

脚本

//打开固定速率流控开关，计算并设置相应固定流控速率。

```
SET SOFTPARAOFBIT: DT=BYTE, PARANUM=27, VALUE=VALUE_1, POSITION=4;
SET SOFTPARA: DT=BYTE, PARANUM=45, VALUE="250";
```

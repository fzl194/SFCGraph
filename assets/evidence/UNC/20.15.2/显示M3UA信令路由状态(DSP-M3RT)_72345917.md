#### [命令功能](#ZH-CN_MMLREF_0000001172345917)

**适用网元：SGSN、MME、SMSF**

该命令用于查询M3UA信令路由状态，M3UA信令路由信息描述了从本地实体出发去往其它目的实体的信令路由。

#### [注意事项](#ZH-CN_MMLREF_0000001172345917)

系统会自行选取可用的信令进程（SPP/SGP），因此，执行该操作的时候要确保系统中有可用的信令进程。否则，操作将不能正常执行。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172345917)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172345917)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172345917)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RUNAME | RU名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定SPU资源单元名称。该参数可以通过<br>[DSP RU](../../../../../../平台服务管理/单体服务公共功能管理/系统管理/资源管理/RU管理/显示资源单元信息（DSP RU）_59103857.md)<br>命令查询。<br>取值范围：1~63位字符串<br>默认值：无 |
| PT | 进程类型 | 可选必选说明：可选参数<br>参数含义：该参数用于指定进程的进程类型。<br>取值范围：<br>- “SPP(SPP)”<br>- “SGP(SGP)”<br>默认值：无 |
| PN | 进程号 | 可选必选说明：可选参数<br>参数含义：该参数用于指定进程号。<br>取值范围：0~11<br>默认值：无 |
| RTX | 路由索引 | 可选必选说明：可选参数<br>参数含义：该参数用于指定准备查询的路由的索引。<br>取值范围：0~1279<br>默认值：无 |
| DEX | 目的实体索引 | 可选必选说明：可选参数<br>参数含义：该参数用于指定该M3UA路由的目的实体。<br>取值范围：0~1279<br>默认值：无 |
| LSX | 链路集索引 | 可选必选说明：可选参数<br>参数含义：该参数用于指定到达该的目的实体的M3UA链路集。<br>取值范围：0~1279<br>默认值：无 |
| RTN | 信令路由名 | 可选必选说明：可选参数<br>参数含义：该参数用于指定M3UA路由名称，标识M3UA路由。<br>取值范围：长度不超过32的字符串<br>默认值：无 |

#### [使用实例](#ZH-CN_MMLREF_0000001172345917)

查询路由索引为0的M3UA信令路由状态：

DSP M3RT: RTX=0;

```
%%DSP M3RT: RTX=0;%%
RETCODE = 0  操作成功。

查询M3UA路由状态
----------------
         路由索引  =  0
     目的实体索引  =  0
       链路集索引  =  0
         路由状态  =  不可用
SG和SEP间路由禁止  =  否
SG和SEP间路由受限  =  否
     链路集不可用  =  是
              SLS  =  NULL
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_MMLREF_0000001172345917)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 路由索引 | 准备查询的路由的索引。 |
| 目的实体索引 | 该M3UA路由的目的实体。 |
| 链路集索引 | 到达该的目的实体的M3UA链路集。 |
| 路由状态 | 指定M3UA路由的当前状态，有可用、不可用两种状态。正常情况下为可用。<br>取值说明：<br>- “可用”<br>- “不可用”<br>说明：路由不可用时，参考<br>**ALM-10336 M3UA路由不可用**<br>进行处理。 |
| SG和SEP间路由禁止 | SG和SEP间路由是否禁止，有是、否两种状态。正常情况下应该为否。<br>取值说明：<br>- “否”<br>- “是” |
| SG和SEP间路由受限 | SG和SEP间路由是否受限，有是、否两种状态。正常情况下应该为否。<br>取值说明：<br>- “否”<br>- “是” |
| 链路集不可用 | 指定M3UA路由对应的链路集是否可用，有是、否两种状态。正常情况下应该为否。取值说明：<br>- “否”<br>- “是” |
| SLS | 信令链路选择码，有数值、NULL两种状态。正常情况下应该为有效数值，该数值与链路的负荷分担情况相关。路由不可用时为NULL。 |

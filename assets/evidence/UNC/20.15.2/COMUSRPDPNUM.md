## [命令功能](#ZH-CN_MMLREF_0000001155639265)

**适用NF：MME、SGSN、AMF**

该命令用于查询系统内各种用户状态的统计结果。

## [注意事项](#ZH-CN_MMLREF_0000001155639265)

5G用户、234G用户和NB-IoT用户状态的统计结果也可以通过DSP NGUSERNUM和DSP USRPDPNUM命令来查询。

#### [操作用户权限](#ZH-CN_MMLREF_0000001155639265)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001155639265)

无

## [使用实例](#ZH-CN_MMLREF_0000001155639265)

查询系统内各种用户状态的统计结果，执行如下命令：

```
%%DSP COMUSRPDPNUM:;%%
RETCODE = 0  操作成功

结果如下
--------
       234G静态用户数  =  0
         5G静态用户数  =  0
      APN签约上下文数  =  0
         2G在线用户数  =  0
         3G在线用户数  =  0
         4G在线用户数  =  0
         5G在线用户数  =  0
     NB-IoT在线用户数  =  0
        2G激活PDP个数  =  0
        3G激活PDP个数  =  0
         4G EPS承载数  =  0
          5GPDU会话数  =  0
         NB-IoT承载数  =  0
     NB-IoT服务用户数  =  0
             NR承载数  =  0
         NR在线用户数  =  0
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001155639265)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 234G静态用户数 | 该参数用于标识系统中保存的通过MME和SGSN接入的用户的静态MM上下文数量（包含签约数据等）。用户分离后，用户的签约数据不会马上删除。用户签约数据删除的时间由SET SDBTMR中“PURGTMR（用户分离后签约数据保留时间）”参数控制，默认设置为24小时。 |
| 5G静态用户数 | 该参数用于标识系统POD中保存的5G用户的静态MM上下文数量（包含签约数据等）。用户去注册后，用户的签约数据不会马上删除。用户签约数据删除的时间由SET NGMMPARA中“PURGTMR（用户分离后签约数据保留时间）”参数控制，默认设置为24小时。 |
| APN签约上下文数 | 该参数用于标识系统中通过MME和SGSN接入并附着的用户的APN签约上下文总数。 |
| 2G在线用户数 | 该参数用于标识系统中附着的2G用户总数。 |
| 3G在线用户数 | 该参数用于标识系统中附着的3G用户总数。 |
| 4G在线用户数 | 该参数用于标识系统中附着的4G用户总数。 |
| 5G在线用户数 | 该参数用于标识系统POD中处于注册态的5G用户数。 |
| NB-IoT在线用户数 | 该参数用于标识系统中附着的NB-IoT用户总数，即处于ECM-CONNECTED和ECM-IDLE态的用户数。 |
| 2G激活PDP个数 | 该参数用于标识系统中2G用户激活的PDP总数。 |
| 3G激活PDP个数 | 该参数用于标识系统中3G用户激活的PDP总数。 |
| 4G EPS承载数 | 该参数用于标识系统中4G用户激活的PDP总数。 |
| 5G PDU会话数 | 该参数用于标识系统POD中保存的5GPDU会话的数量。 |
| NB-IoT承载数 | 该参数用于标识系统中NB-IoT用户激活的EPS承载总数。 |
| NB-IoT服务用户数 | 该参数用于标识系统中24小时内服务过的NB-IoT用户数。 |
| NR承载数 | 该参数用于标识系统中附着的具备NR（New Radio）能力且签约NR业务的用户在E-UTRAN激活的承载总数。 |
| NR在线用户数 | 该参数用于标识系统中附着的具备NR（New Radio）能力且签约NR业务的用户从E-UTRAN接入的总数，即处于ECM-CONNECTED和ECM-IDLE态的用户数。 |

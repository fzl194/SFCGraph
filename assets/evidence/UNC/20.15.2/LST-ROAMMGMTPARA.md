## [命令功能](#ZH-CN_MMLREF_0000001606385409)

**适用NF：AMF、SMF**

该命令用于查询漫游管理参数。

## [注意事项](#ZH-CN_MMLREF_0000001606385409)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001606385409)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001606385409)

无

## [使用实例](#ZH-CN_MMLREF_0000001606385409)

查询漫游管理参数。

```
%%LST ROAMMGMTPARA:;%%
RETCODE = 0 操作成功

结果如下
------------------------
识别关口局NF开关 = OFF
避免网元混用开关 = OFF
（结果个数=1）

---- END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001606385409)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 识别关口局NF开关 | 该参数用于指定漫游场景下是否根据对端NF的IP/FQDN识别是否为关口局NF。 |
| 避免网元混用开关 | 该参数用于指定是否避免混用本网网元和异网网元。 |

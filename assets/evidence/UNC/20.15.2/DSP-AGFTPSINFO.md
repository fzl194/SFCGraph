## [命令功能](#ZH-CN_MMLREF_0245110913)

**适用NF：NCG**

该命令用于显示AGF每个pod的计费消息速率。

## [注意事项](#ZH-CN_MMLREF_0245110913)

无

#### [操作用户权限](#ZH-CN_MMLREF_0245110913)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0245110913)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PODNAME | Pod名称 | 可选必选说明：可选参数<br>参数含义：该参数用于描述pod名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~64，单位是个。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0245110913)

显示AGF每个pod的计费消息速率。

```
DSP AGFTPSINFO:;
RETCODE = 0  操作成功

结果如下
--------
          Pod名称  =  ccp-pod
计费消息速率(TPS)  =  666
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0245110913)

| 输出项名称 | 输出项解释 |
| --- | --- |
| Pod名称 | 该参数用于描述pod名称。 |
| 计费消息速率(TPS) | 该参数用于描述计费消息速率(TPS) 。 |

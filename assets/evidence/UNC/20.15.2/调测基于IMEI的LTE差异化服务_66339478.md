# 调测基于IMEI的LTE差异化服务

## [操作场景](#ZH-CN_OPI_0166339478)

本操作指导介绍在运行网络中调测基于IMEI的LTE差异化服务的操作过程。

> **说明**
> 假设与MME对接的eNodeB设备为支持“Masked IMEISV”信元的华为eNodeB设备。

## [必备事项](#ZH-CN_OPI_0166339478)

前提条件

- 请仔细阅读[WSFD-011303 基于IMEI的LTE差异化服务特性概述](特性概述_66339474.md)。
- 完成[激活基于IMEI的LTE差异化服务](激活基于IMEI的LTE差异化服务_66339477.md)。

数据

该操作无需准备数据。

工具

OM Portal跟踪

## [操作步骤](#ZH-CN_OPI_0166339478)

1. 在 UNC 操作维护台上创建用户跟踪任务和S1接口跟踪任务。
2. 用户开机附着到LTE网络。
  预期结果：用户附着成功。
3. 观察用户消息跟踪和S1接口消息跟踪。
  预期结果：用户消息跟踪和S1接口跟踪中都可以观察到MME向eNodeB发送的Initial Context Setup Request消息中携带“Masked IMEISV”信元。

# 增加重选UPF的故障原因值（ADD RESELECTUPCAUSE）

- [注意事项](#ZH-CN_MMLREF_0306399908__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0306399908__1.3.4)

## [命令功能](#ZH-CN_MMLREF_0306399908)

**适用NF：SGW-C、PGW-C、SMF、GGSN**

该命令用于增加重选UPF的故障原因值。在SMF对接多个UPF时，当PFCP会话建立失败，UPF在PFCP会话建立响应消息中携带故障原因值。如果已通过本命令配置对应的故障原因值，SMF支持根据故障原因值触发UPF重选。

## [注意事项](#ZH-CN_MMLREF_0306399908)

- 该命令执行后立即生效。

- 最多可输入255条记录。

#### [操作用户权限](#ZH-CN_MMLREF_0306399908)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0306399908)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| FAULTCAUSE | 故障原因值 | 可选必选说明：必选参数<br>参数含义：该参数用于指定UPF重选的故障原因值。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是1~255。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0306399908)

增加重选UPF的故障原因值75。

```
ADD RESELECTUPCAUSE: FAULTCAUSE=75;
```

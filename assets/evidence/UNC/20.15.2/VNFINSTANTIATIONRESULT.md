## [命令功能](#ZH-CN_MMLREF_0227570469)

该命令用于查询实例化结果。

## [注意事项](#ZH-CN_MMLREF_0227570469)

无

#### [操作用户权限](#ZH-CN_MMLREF_0227570469)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0227570469)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| MEID | 网元ID | 可选必选说明：可选参数<br>参数含义：该参数用于标识网元ID。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~40。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0227570469)

查询实例化结果:

```
DSP VNFINSTANTIATIONRESULT:;
```

## [输出结果说明](#ZH-CN_MMLREF_0227570469)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 实例化处理进度 | 该参数用于表示实例化进度。 |
| 实例化处理结果 | 该参数用于表示实例化处理结果。<br>取值说明：<br>- Processing（处理中）<br>- OK（成功）<br>- Nok（失败）<br>- PartlyOK（部分成功） |
| 实例化结果说明 | 该参数用于说明实例化结果，仅在InstResult为Nok或者PartlyOK时有效。 |

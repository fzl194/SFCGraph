## [命令功能](#ZH-CN_MMLREF_0000001751027637)

该命令用于查询HTTP服务端故障链路信息。

## [注意事项](#ZH-CN_MMLREF_0000001751027637)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001751027637)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001751027637)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| FAULTLINKTYPE | 故障链路类型 | 可选必选说明：必选参数<br>参数含义：该参数用于指定查询的故障链路类型。<br>数据来源：本端规划<br>取值范围：<br>- “FAULTTLSLINK（TLS故障链路）”：TLS故障链路<br>默认值：无<br>配置原则：无 |
| PODNAME | POD名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定HTTP链路所在的POD名称。该POD名称可通过<br>[**DSP HTTPPROCESS**](../HTTP进程状态管理/显示HTTP进程信息（DSP HTTPPROCESS）_29053327.md)<br>命令查询获取。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~255。<br>默认值：无<br>配置原则：无 |

## [输出结果说明](#ZH-CN_MMLREF_0000001751027637)

| 输出项名称 | 输出项解释 |
| --- | --- |
| POD名称 | 该参数用于指定HTTP链路所在的POD名称。该POD名称可通过<br>[**DSP HTTPPROCESS**](../HTTP进程状态管理/显示HTTP进程信息（DSP HTTPPROCESS）_29053327.md)<br>命令查询获取。 |
| 进程标识 | 该参数用于显示进程标识。 |
| 本端IPv4地址 | 该参数用于显示HTTP服务端故障链路的本端IPv4地址。 |
| 对端IPv4地址 | 该参数用于显示HTTP服务端故障链路的对端IPv4地址。 |
| 本端IPv6地址 | 该参数用于显示HTTP服务端故障链路的本端IPv6地址。 |
| 对端IPv6地址 | 该参数用于显示HTTP服务端故障链路的对端IPv6地址。 |
| 时间戳 | 该参数用于显示HTTP服务端故障链路记录产生的时间戳。 |
| 错误码 | 该参数用于显示HTTP服务端链路故障的错误码。 |

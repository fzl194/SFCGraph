## [操作场景](#ZH-CN_OPI_0170635541)

本操作指导介绍在运行网络中控制EPS QoS各项功能的启动和调整的操作过程。

在E-UTRAN网络和GERAN/UTRAN网络互操作过程中， UNC 支持EPS（Evolved Packet System） QoS参数和Pre-Rel-8 QoS参数之间的映射，实现网络间的切换和重选。 UNC 支持承载级别、APN级别、终端级别的EPS QoS控制。

QoS与流量管理指 UNC 支持3GPP QoS，按照QoS等级对用户报文进行分类，还可以提供DiffServ，从而支持流量监管、拥塞管理、拥塞避免等多种QoS实现技术。

## [操作步骤](#ZH-CN_OPI_0170635541)

1. 进入 “MML命令行-UNC” 窗口。
2. 增加EPS QoS参数到Pre-Rel-8 QoS参数映射规则。
  [**SET R8QOSMAP**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/QoS管理/QoS兼容性管理/EPS QoS参数到Pre-R8 QoS参数映射/设置EPS QoS参数到Pre-R8 QoS参数映射规则（SET R8QOSMAP）_26146234.md)
3. 设置QoS兼容性（EPS ARP、Pre-Rel-8 ARP转换参数和EPS QoS Extended-2信元控制参数等）。
  [**SET COMPATIBILITY**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/QoS管理/QoS兼容性管理/QoS兼容性配置/设置QoS兼容性配置(SET COMPATIBILITY)_72345835.md)

## [任务示例](#ZH-CN_OPI_0170635541)

任务描述

增加一条QCI为1的EPS QoS参数到Pre-Rel-8 QoS参数的映射规则，当EPS QoS参数中的QCI为1时，映射到Pre-Rel-8 QoS参数的流量等级为Conversational class，最大SDU长度为150字节，发送次序存在，不检查发送错误SDU，保留BER为5×10 <sup>-2</sup> ，SDU误码率为1×10 <sup>-4</sup> ，QoS传递时延为10ms。同时设置EPS ARP和Pre-Rel-8 ARP转换参数H值为5，M值为10，EPS QoS Extended-2信元控制参数设置为关闭状态。

脚本

//增加EPS QoS参数到Pre-Rel-8 QoS参数的映射规则。

```
SET R8QOSMAP: QCI=1, TC=CC, MAXSDU=150, DO=ORDER, DESDU=NOT_DETECT, RBER=E_RESIDUAL_BIT_ERR_RATIO_1, SDUER=SDUER4, TD=10;
```

//设置QoS兼容性（EPS ARP、Pre-Rel-8 ARP转换参数和EPS QoS Extended-2信元控制参数等）。

```
SET COMPATIBILITY: HARP=5, MARP=10, EPSQOSEXT2=NO;
```

## [命令功能](#ZH-CN_MMLREF_0000001232382545)

**适用NF：PGW-C、SMF、GGSN**

该命令用于查询DHCP服务器组配置。

## [注意事项](#ZH-CN_MMLREF_0000001232382545)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001232382545)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001232382545)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| GROUPNAME | DHCP服务器组名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定DHCP服务器组的名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~31。不支持空格，不区分大小写。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001232382545)

- 查询一个DHCP服务器组的信息，GROUPNAME为testgrp：
  ```
  %%LST DHCPSERVERGRP:GROUPNAME="testgrp";%%
  RETCODE = 0  操作成功

  结果如下
  --------
          DHCP服务器组名称  =  testgrp
                    IP类型  =  IPV4
            租约时间(小时)  =  0
  DHCP消息超时重发时间(秒)  =  3
              DHCP重发次数  =  3
                   绑定VPN  =  不使能
                 VPN实例名  =  NULL
  (结果个数 = 1)

  ---    END
  ```
- 查询所有的DHCP服务器的信息：
  ```
  %%LST DHCPSERVERGRP:;%%
  RETCODE = 0  操作成功

  结果如下
  --------
  DHCP服务器组名称    IP类型    租约时间(小时)    DHCP消息超时重发时间(秒)    DHCP重发次数    绑定VPN    VPN实例名

  testgrp             IPV4      0                 3                           3               不使能     NULL     
  testgrp2            IPV4      0                 3                           3               不使能     NULL     
  (结果个数 = 2)

  ---    END
  ```

## [输出结果说明](#ZH-CN_MMLREF_0000001232382545)

| 输出项名称 | 输出项解释 |
| --- | --- |
| DHCP服务器组名称 | 该参数用于指定DHCP服务器组的名称。 |
| IP类型 | 该参数用于指定IP地址类型。 |
| 租约时间(小时) | 该参数用于指定DHCP服务器组的地址租期。 |
| DHCP消息超时重发时间(秒) | 该参数用于指定向DHCP服务器发送消息的超时时间。 |
| DHCP重发次数 | 该参数用于指定向DHCP服务器发送消息的重发次数。 |
| 绑定VPN | 该参数用于指定是否绑定VPN实例。 |
| VPN实例名 | 该参数用于指定VPN实例名。 |

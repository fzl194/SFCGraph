#### [命令功能](#ZH-CN_MMLREF_0000001126305554)

**适用网元：MME**

该命令用于查询S1模式下的用户信令状态。

#### [注意事项](#ZH-CN_MMLREF_0000001126305554)

- 该命令执行后立即生效。
- 该命令查询的请求次数显示为当前统计周期内的请求次数，统计周期为1小时。

#### [本地用户权限](#ZH-CN_MMLREF_0000001126305554)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126305554)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126305554)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IMSI | IMSI | 可选必选说明：必选参数<br>参数含义：该参数用于指定用户的国际移动用户标识。<br>是否必选：必选<br>数据来源：本端规划<br>取值范围：1～15位十进制数字字符串。<br>默认值：无 |

DSP S1SMARTINFO: IMSI="123036500000002";

#### [输出结果说明](#ZH-CN_MMLREF_0000001126305554)

| 输出项名称 | 输出项解释 |
| --- | --- |
| IMSI | 显示用户的国际移动用户标识。 |
| 终端类型 | 显示用户的终端类型，由ADD IMEILIB命令配置。 |
| 附着抑制类型 | 显示用户的附着流程抑制状态。 |
| 附着请求次数 | 显示当前统计周期内用户的附着请求次数。 |
| 服务请求抑制类型 | 显示用户的服务请求流程抑制状态。 |
| 服务请求次数 | 显示当前统计周期内用户的服务请求次数。 |
| APN个数 | 显示用户的PDN连接流程使用的APN个数。 |
| APN标识 | 显示用户的PDN连接流程使用的APN网络标识。 |
| PDN连接抑制类型 | 显示用户的APN粒度的PDN连接抑制状态。 |
| PDN连接请求次数 | 显示当前统计周期内用户的APN粒度的PDN连接请求次数。 |

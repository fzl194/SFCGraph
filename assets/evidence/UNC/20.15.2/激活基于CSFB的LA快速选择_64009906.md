## [操作场景](#ZH-CN_OPI_0164009906)

本操作指导介绍在运行网络中激活基于CSFB的LA快速选择的操作过程。

> **说明**
> 适用于MME。

## [必备事项](#ZH-CN_OPI_0164009906)

前提条件

已完成加载License。

数据

该操作无需准备数据。

## [操作步骤](#ZH-CN_OPI_0164009906)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. 设置S1接口支持Register LAI信元。
  [**SET S1CMPT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/S1接口管理/S1接口兼容性/设置S1接口兼容性(SET S1CMPT)_72345837.md)

## [任务示例](#ZH-CN_OPI_0164009906)

任务描述

启用基于CSFB的LA快速选择。

脚本

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2CSFBEP01", SWITCH=ENABLE;
```

//设置S1接口支持Register LAI信元。

```
SET S1CMPT: REGLAI=SUPPORT;
```

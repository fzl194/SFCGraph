## [命令功能](#ZH-CN_MMLREF_0000001232224049)

**适用NF：PGW-C、SMF、GGSN**

该命令用于查询DHCP服务器信息。

## [注意事项](#ZH-CN_MMLREF_0000001232224049)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001232224049)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001232224049)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| GROUPNAME | DHCP服务器组名称 | 可选必选说明：可选参数<br>参数含义：该参数指定DHCP服务器组名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~31。不支持空格，不区分大小写。<br>默认值：无<br>配置原则：<br>该参数使用ADD DHCPSERVERGRP命令配置生成。 |

## [使用实例](#ZH-CN_MMLREF_0000001232224049)

当需要查看DHCP服务器组group1的DHCP服务器信息时，使用该命令：

```
%%LST DHCPSERVER:GROUPNAME="group1";%%
RETCODE = 0  操作成功

结果如下
--------
  DHCP服务器组名称  =  group1
是否是主DHCP服务器  =  使能
        IP地址类型  =  IPV4
DHCPv4服务器IP地址  =  10.1.1.1
DHCPv6服务器IP地址  =  ::
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001232224049)

| 输出项名称 | 输出项解释 |
| --- | --- |
| DHCP服务器组名称 | 该参数指定DHCP服务器组名称。 |
| 是否是主DHCP服务器 | 该参数指定是否为主DHCP服务器。 |
| IP地址类型 | 该参数指定DHCP服务器的IP地址类型。 |
| DHCPv4服务器IP地址 | 该参数指定DHCP服务器的IPv4地址。 |
| DHCPv6服务器IP地址 | 该参数指定DHCP服务器的IPv6地址。 |
| WAL值 | 该参数表示整机（UNC）每秒发送给该DHCP服务器的最大消息数。 |

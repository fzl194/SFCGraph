## [命令功能](#ZH-CN_MMLREF_0000001188262395)

**适用NF：PGW-C**

该命令用于查询全局的PGW的计费暂停功能。

## [注意事项](#ZH-CN_MMLREF_0000001188262395)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001188262395)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001188262395)

无

## [使用实例](#ZH-CN_MMLREF_0000001188262395)

显示全局计费暂停配置：

```
%%LST GLBPGWCHGPAUSE:;%%
RETCODE = 0  操作成功

结果如下
--------
计费暂停开关  =  不使能
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001188262395)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 计费暂停开关 | 该参数用于表示计费暂停功能开关。 |

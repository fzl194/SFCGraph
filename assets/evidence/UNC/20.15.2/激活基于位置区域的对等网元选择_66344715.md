## [操作场景](#ZH-CN_OPI_0166344715)

本操作指导介绍在运行网络中激活基于位置区域的对等网元选择特性的操作过程。

UNC 在Attach/RAU/TAU等流程中查询源侧网元节点SGSN，在Handover/Relocation等流程中查询目标侧网元节点SGSN/MME，以及在SRVCC流程中查询目标MSC的过程均为对等网元选择，基于LAI/RAI/TAI定制DNS域名进行对等网元查询称为基于位置区域的对等网元选择。

> **说明**
> 适用于 SGSN、 MME。

## [操作步骤](#ZH-CN_OPI_0166344715)

1. 进入 “MML命令行-UNC” 窗口。
2. **可选：**增加Hostfile的A解析记录。
  [**ADD IPV4DNSH**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/DNS/DNS Hostfile管理/增加IPV4 DNS Hostfile记录(ADD IPV4DNSH)_26145884.md)
  > **说明**
  > 如果运营商在DNS Server上已配置该对应关系，则不需要配置本步骤。
3. **可选：**增加Hostfile的NAPTR解析记录。
  [**ADD DNSN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/DNS/DNS NAPTR管理/增加DNS NAPTR记录(ADD DNSN)_72225569.md)
  > **说明**
  > 如果运营商在DNS Server上已配置该对应关系，则不需要配置本步骤。
4. 增加基于位置区域的DNS域名定制策略。
    - 选择SGSN
      [**ADD AREADNS**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/GnGp-SGSN_S10_S16_S3接口管理/位置域名管理/增加位置区域DNS域名(ADD AREADNS)_72345559.md)
    - 选择MME
      [**ADD AREADNS**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/GnGp-SGSN_S10_S16_S3接口管理/位置域名管理/增加位置区域DNS域名(ADD AREADNS)_72345559.md)
    - 选择MSC
      [**ADD AREADNS**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/GnGp-SGSN_S10_S16_S3接口管理/位置域名管理/增加位置区域DNS域名(ADD AREADNS)_72345559.md)

## [任务示例](#ZH-CN_OPI_0166344715)

任务描述

- 任务1：选择SGSN过程中，配置基于RAI区域集合(映射到区域名称ZONE)选择SGSN的DNS域名定制方式，对应的位置区域为"0x0001"，路由区集合为"0x01"~"0x05"。
- 任务2：选择MME过程中，配置基于TAI区域集合(映射到起始TAI)选择MME的DNS域名定制方式，对应的跟踪区集合为"0x0001"~"0x0005"。
- 任务3：选择MSC过程中，配置基于LAI区域集合(映射到区域名称ZONE)选择MSC的DNS域名定制方式，对应的位置区集合为"0x0001"~"0x0005"。

脚本

- 任务1：
  //增加Hostfile的A解析记录。
  ```
  ADD IPV4DNSH: HSINDEX=1, HOSTNAME="ZONE.MNC123.MCC123.GPRS", ADDRSECTION=SECTION1, IPV4ADDR1="10.2.3.10";
  ```
  //增加基于位置区域的DNS域名定制策略。
  ```
  ADD AREADNS: DNTYPE=SGSN_RAI, LAC="0x0001", RAC="0x01", RACRANGE="0x05", ZONESW=YES, ZONENAME="ZONE";
  ```
- 任务2：
  //增加Hostfile的A解析记录。
  ```
  ADD IPV4DNSH: HSINDEX=2, HOSTNAME="TOPON.MME-S10.MMEC02.MMEGI8001.MME.EPC.MNC000.MCC123.3GPPNETWORK.ORG", ADDRSECTION=SECTION1, IPV4ADDR1="10.2.3.11";
  ```
  //增加Hostfile的NAPTR解析记录。
  ```
  ADD DNSN: FQDN="TAC-LB01.TAC-HB00.TAC.EPC.MNC123.MCC123.3GPPNETWORK.ORG", HSINDEX=2, ENTITY=MME, INTYPE=S10;
  ```
  //增加基于位置区域的DNS域名定制策略。
  ```
  ADD AREADNS: DNTYPE=MME_TAI, TAC="0x0001", TACRANGE="0x0005";
  ```
- 任务3：
  //增加Hostfile的A解析记录。
  ```
  ADD IPV4DNSH: HSINDEX=3, HOSTNAME="NODES.EPC.ENVID68.MNC003.MCC123.3GPPNETWORK.ORG", ADDRSECTION=SECTION1, IPV4ADDR1="10.2.3.12";
  ```
  //增加Hostfile的NAPTR解析记录。
  ```
  ADD DNSN: FQDN="ZONE.EPC.MNC003.MCC123.3GPPNETWORK.ORG", HSINDEX=3, ENTITY=MSC, INTYPE=Sv;
  ```
  //增加基于位置区域的DNS域名定制策略。
  ```
  ADD AREADNS: DNTYPE=MSC_LAI, LAC="0x0001", LACRANGE="0x0005", ZONESW=YES, ZONENAME="ZONE";
  ```

## [命令功能](#ZH-CN_MMLREF_0254491178)

本命令用于请求系统显示NTP服务器的运行状态。

## [注意事项](#ZH-CN_MMLREF_0254491178)

无。

## [参数说明](#ZH-CN_MMLREF_0254491178)

| **参数标识** | **参数名称** | **参数说明** |
| --- | --- | --- |
| SVRNAME | NTP服务器名 | 可选必选说明：可选参数。<br>参数含义：用于请求系统显示哪个名称的NTP服务器的运行状态；若不输入，则表示所有的NTP服务器。<br>取值范围：长度不超过32个字符。<br>默认值：无。<br>配置原则：无。 |

## [使用实例](#ZH-CN_MMLREF_0254491178)

1. 显示所有的NTP服务器的运行状态：
  LST NTPSVR:;

  ```
  %%LST NTPSVR:;%% 
  RETCODE = 0  操作成功  

  操作结果如下 
  ------------ 
  NTP服务器名  NTP协议版本  类型        地址族  IP地址                                   身份验证标志  身份验证密钥号  密钥类型   时钟状态  参考时间戳                  
  server1      NTPV4        服务器类型  IPv4    192.168.156.12                           是            1314            SHA256       未同步    03:13:19.019 Aug 14 2020   
  server2      NTPV3        服务器类型  IPv6    2001:0db8:86a3:08d3:1319:8a2e:0370:7344  否            NULL            NULL         未同步    03:13:19.019 Aug 14 2020   
  (结果个数 = 2)
  ---    END
  ```
2. 显示名为“NtpSvrName”的NTP服务器的运行状态：
  LST NTPSVR: SVRNAME="NtpSvrName";

  ```
  %%LST NTPSVR: SVRNAME="NtpSvrName";%%
  RETCODE = 0  操作成功  

  操作结果如下

        时钟状态  =  未同步

  ---    END
  ```

## [输出结果说明](#ZH-CN_MMLREF_0254491178)

命令执行正常，会返回命令执行成功的提示信息。输出结果说明如 [表1](#ZH-CN_MMLREF_0254491178__table152998433919) 所示。

*表1 输出项解释*

| **输出项名称** | **输出项解释** |
| --- | --- |
| NTP服务器名 | 用于返回NTP服务器名。 |
| NTP协议版本 | 用于返回NTP服务器的协议版本。 |
| 类型 | 用于返回NTP服务器的类型：服务器类型或客户端类型。 |
| 地址族 | 用于返回NTP服务器IP地址所采用的协议族：IPV4或者IPV6。 |
| IP地址 | 用于返回NTP服务器的IP地址。 |
| 身份验证标志 | 用于返回NTP服务器是否做身份验证：是或否。 |
| 身份验证密钥号 | 用于返回NTP服务器身份验证的密钥号：数字[1~65533]或者NULL。 |
| 密钥类型 | 用于返回NTP服务器身份验证的密钥类型：SHA256或SHA1。 |
| 时钟状态 | 用于返回NTP服务器的时钟状态。<br>“同步”<br>或<br>“未同步” |
| 参考时间戳 | 用于返回当前时间戳。 |

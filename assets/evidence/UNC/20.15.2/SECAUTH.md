#### [命令功能](#ZH-CN_TOPIC_0188029947)

![](设置二次授权功能状态（SET SECAUTH）_88029947.assets/notice_3.0-zh-cn_2.png)

- 执行本命令将开启或关闭二次授权功能，需要创建二次授权用户，对高危操作进行二次授权认证。
- 打开免二次授权时间段开关后，支持从开始日期到结束日期内，每天的开始时间和结束时间范围内免二次授权。

打开或关闭二次授权功能。

#### [注意事项](#ZH-CN_TOPIC_0188029947)

- 该命令仅限具有Administrators角色的用户可以执行。在扩展域，仅限系统管理员可以执行（默认为关闭状态）。
- 该命令存在系统初始记录，参数的初始设置值， 其它参数默认是NULL，如下：
  | STATUS |
  | --- |
  | OFF（关闭二次授权功能） |
- 设置的“开始日期”不能大于“结束日期”。
- 设置“开始日期”和“结束日期”相等时，“开始时间”不能大于“结束时间”。
- 设置日期跨天时，“开始时间”允许大于“结束时间”。
- 如果要设置一秒生效时间，请将“开始时间”和“结束时间”设置相等，例如：2024/08/06 22:00:00 - 2024/08/06 22:00:00 或者 2024/08/06 22:00:00 - 2024/08/07 22:00:00。
- 如果要设置二十四小时生效时间：
    - 当天场景，设置“开始时间”小于“结束时间”，例如：2024/08/06 00:00:00 - 2024/08/06 23:59:59。
    - 跨天场景，设置“开始时间”大于“结束时间”一秒，例如：2024/08/06 22:00:00 - 2024/08/07 21:59:59。
- 此命令时间精度是秒，可能存在一秒误差。

#### [使用实例](#ZH-CN_TOPIC_0188029947)

设置二次授权功能：

```
%%SET SECAUTH: STATUS=ON, TIMESWITCH=NO;%%
RETCODE = 0  操作成功

---    END
```

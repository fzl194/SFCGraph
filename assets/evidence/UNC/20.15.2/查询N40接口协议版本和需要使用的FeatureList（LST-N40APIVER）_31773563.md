## [命令功能](#ZH-CN_MMLREF_0231773563)

**适用NF：PGW-C、SMF、GGSN**

该命令用于查询N40接口协议版本、需要使用的FeatureList和是否携带指定消息属性。

## [注意事项](#ZH-CN_MMLREF_0231773563)

无

#### [操作用户权限](#ZH-CN_MMLREF_0231773563)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0231773563)

无

## [使用实例](#ZH-CN_MMLREF_0231773563)

查询N40接口协议版本、需要使用的FeatureList和是否携带指定消息属性：

```
%%LST N40APIVER:;%%
RETCODE = 0  操作成功

结果如下
--------
 API接口版本  =  F30
        特性  =  NODEFUNC
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0231773563)

| 输出项名称 | 输出项解释 |
| --- | --- |
| API接口版本 | 该参数用于指定SMF支持的N40接口API版本（3GPP TS 32291的协议版本）。 |
| 特性 | 该参数用于指定SMF支持的N40接口增强的功能集。 |

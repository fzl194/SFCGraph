## [命令功能](#ZH-CN_MMLREF_0000001372273645)

**适用NF：AMF**

该命令用于查询位置区对应的时区和夏令时信息。

## [注意事项](#ZH-CN_MMLREF_0000001372273645)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001372273645)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001372273645)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| TZID | 时区标识 | 可选必选说明：可选参数<br>参数含义：该参数用于指定待增加时区的索引信息。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是1~24。<br>默认值：无<br>配置原则：无 |
| TZONE | 时区 | 可选必选说明：可选参数<br>参数含义：该参数用于指定设备所处时区。<br>数据来源：本端规划<br>取值范围：<br>- E0000（GMT+00:00）<br>- E0100（GMT+01:00）<br>- E0200（GMT+02:00）<br>- E0300（GMT+03:00）<br>- E0330（GMT+03:30）<br>- E0400（GMT+04:00）<br>- E0430（GMT+04:30）<br>- E0500（GMT+05:00）<br>- E0530（GMT+05:30）<br>- E0545（GMT+05:45）<br>- E0600（GMT+06:00）<br>- E0630（GMT+06:30）<br>- E0700（GMT+07:00）<br>- E0800（GMT+08:00）<br>- E0845（GMT+08:45）<br>- E0900（GMT+09:00）<br>- E0930（GMT+09:30）<br>- E1000（GMT+10:00）<br>- E1030（GMT+10:30）<br>- E1100（GMT+11:00）<br>- E1130（GMT+11:30）<br>- E1200（GMT+12:00）<br>- E1245（GMT+12:45）<br>- E1300（GMT+13:00）<br>- E1400（GMT+14:00）<br>- W0100（GMT-01:00）<br>- W0200（GMT-02:00）<br>- W0300（GMT-03:00）<br>- W0330（GMT-03:30）<br>- W0400（GMT-04:00）<br>- W0430（GMT-04:30）<br>- W0500（GMT-05:00）<br>- W0600（GMT-06:00）<br>- W0700（GMT-07:00）<br>- W0800（GMT-08:00）<br>- W0900（GMT-09:00）<br>- W0930（GMT-09:30）<br>- W1000（GMT-10:00）<br>- W1100（GMT-11:00）<br>- W1200（GMT-12:00）<br>默认值：无<br>配置原则：<br>选择列表包括GMT+TIME和GMT-TIME两种形式，“+”代表进入东时区，“-”代表进入西时区。例如：GMT+14:00代表进入东时区后，时区与世界标准时区提前14小时；GMT-01:00代表进入西时区后，时区与世界标准时区延缓1小时。 |

## [使用实例](#ZH-CN_MMLREF_0000001372273645)

- 查询所有的时区和夏令时配置信息，执行如下命令：
  ```
  %%LST NGTZLST:;%%
  RETCODE = 0  操作成功。

  输出结果如下
  --------------
   时区标识  时区描述  时区       是否实行夏令时  开始日期输入方式  开始月份  开始日  第几个星期(开始)  开始星期  开始时间  结束日期输入方式  结束月份  结束日  第几个星期(结束)  结束星期  结束时间  时间偏移量(小时)

   1         NULL      GMT+00:00  否              日期方式          NULL      0       第一个            星期日    00:00:00  日期方式          NULL      0       第一个            星期日    00:00:00  0            
   2         NULL      GMT+04:30  是              星期方式          3月       0       第三个            星期三    14:42:00  星期方式          4月       0       第三个            星期六    02:51:00  1               
  (结果个数 = 2)

  ---    END
  ```
- 查询单条时区和夏令时配置信息，执行如下命令：
  ```
  %%LST NGTZLST: TZID=2;%%
  RETCODE = 0  操作成功。

  输出结果如下

              时区  =  GMT+04:30
  ```

## [输出结果说明](#ZH-CN_MMLREF_0000001372273645)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 时区标识 | 该参数用于指定待增加时区的索引信息。 |
| 时区描述 | 该参数用于描述时区信息。 |
| 时区 | 该参数用于指定设备所处时区。 |
| 是否实行夏令时 | 该参数用于指定是否实行夏令时。 |
| 开始日期输入方式 | 该参数用于指定夏令时开始日期的输入方式。 |
| 开始月份 | 该参数用于指定开始实施夏令时的具体月份值。 |
| 开始日 | 该参数用于指定夏令时的开始日。 |
| 第几个星期(开始) | 该参数用于指定开始实施夏令时的星期值在夏令时开始月中为第几个。 |
| 开始星期 | 该参数用于指定开始实施夏令时的具体星期值。 |
| 开始时间 | 该参数用于指定开始实施夏令时的具体时间，该时间为非夏令时时间，输入格式为“HH&MM&SS”，“HH”代表时，“MM”代表分，“SS”代表秒。 |
| 结束日期输入方式 | 该参数用于指定夏令时结束日期的输入方式。 |
| 结束月份 | 该参数用于指定结束实施夏令时的具体月份值。 |
| 结束日 | 该参数用于指定夏令时的结束日。 |
| 第几个星期(结束) | 该参数用于指定结束实施夏令时的星期值在夏令时结束月中为第几个。 |
| 结束星期 | 该参数用于指定结束实施夏令时的具体星期值。 |
| 结束时间 | 该参数用于指定结束实施夏令时的具体时间，该时间为非夏令时时间，输入格式为“HH&MM&SS”，“HH”代表时，“MM”代表分，“SS”代表秒。 |
| 时间偏移量(小时) | 该参数用于指定夏令时相对于标准时的时间偏移量。 |

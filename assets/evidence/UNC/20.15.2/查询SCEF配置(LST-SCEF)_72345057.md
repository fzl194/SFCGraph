#### [命令功能](#ZH-CN_MMLREF_0000001172345057)

**适用网元：MME**

该命令用于查询SCEF配置。

#### [注意事项](#ZH-CN_MMLREF_0000001172345057)

该命令执行后立即生效。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172345057)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172345057)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172345057)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SCEFRLM | SCEF域名 | 可选必选说明：可选参数<br>参数含义：该参数用于指定SCEF域名。<br>数据来源：对端协商<br>取值范围：1~127位字符串<br>默认值：无 |
| SCEFHTN | SCEF主机名 | 可选必选说明：可选参数<br>参数含义：该参数用于指定SCEF主机名。<br>数据来源：对端协商<br>取值范围：1~127位字符串<br>默认值：无 |

#### [使用实例](#ZH-CN_MMLREF_0000001172345057)

查询SCEF配置：

LST SCEF:;

```
%%LST SCEF:;%%
RETCODE = 0  操作成功。

操作结果如下
--------------
  SCEF域名  =  huawei01.com
SCEF主机名  =  scef0701.huawei01.com
(结果个数 = 1)

---    END
```

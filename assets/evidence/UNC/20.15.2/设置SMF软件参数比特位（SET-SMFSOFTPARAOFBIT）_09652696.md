## [命令功能](#ZH-CN_MMLREF_0209652696)

**适用NF：SMF**

该命令用于设置SMF软件参数比特位。

## [注意事项](#ZH-CN_MMLREF_0209652696)

- 该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0209652696)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [使用实例](#ZH-CN_MMLREF_0209652696)

- 设置双字类型1号软参Bit1值为0:
  ```
  SET SMFSOFTPARAOFBIT: DT=Dw, DWORDNUM=1, VALUE=VALUE_0, POSITION=1;
  ```
- 设置字节类型1号软参Bit1值为0:
  ```
  SET SMFSOFTPARAOFBIT: DT=Byte, VALUE=VALUE_0, BYTEPOSITION=1, BYTENUM=1;
  ```

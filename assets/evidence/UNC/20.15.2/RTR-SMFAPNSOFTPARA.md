# 恢复SMF APN软参记录（RTR SMFAPNSOFTPARA）

- [命令功能](#ZH-CN_MMLREF_0225121208__1.3.1)
- [注意事项](#ZH-CN_MMLREF_0225121208__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0225121208__1.3.4)

## [命令功能](#ZH-CN_MMLREF_0225121208)

**适用NF：SMF**

该命令用于删除APN下软件参数信息。

## [注意事项](#ZH-CN_MMLREF_0225121208)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0225121208)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0225121208)

## [使用实例](#ZH-CN_MMLREF_0225121208)

当需要删除当前索引为8的APN huawei.com BYTE类软参值为默认值时：

```
RTR SMFAPNSOFTPARA: APN="huawei.com", DT=Byte, BYTENUM=8;
```

## [命令功能](#ZH-CN_MMLREF_0000001870382341)

**适用NF：PGW-C、GGSN**

该命令用于查询字符串类型的Monitoring-Key。

## [注意事项](#ZH-CN_MMLREF_0000001870382341)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001870382341)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001870382341)

无

## [使用实例](#ZH-CN_MMLREF_0000001870382341)

查询当前配置的stringmk：

```
LST STRINGMK:;
RETCODE = 0  操作成功。

结果如下
--------
字符串Monitoring-Key    监控属性值

test                     100       
test1                    200       
(结果个数 = 2)
---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001870382341)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 字符串Monitoring-Key | 该参数用于配置字符串类型的Monitoring-Key。 |
| 监控属性值 | 设置监控属性的ID。 |

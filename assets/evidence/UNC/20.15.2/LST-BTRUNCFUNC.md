#### [命令功能](#ZH-CN_CONCEPT_0000001341460225)

**适用NF：MME**

此命令用于查询宽带集群系统扩展功能配置参数。

#### [注意事项](#ZH-CN_CONCEPT_0000001341460225)

无。

#### [本地用户权限](#ZH-CN_CONCEPT_0000001341460225)

manage-ug;system-ug;monitor-ug

#### [网管用户权限](#ZH-CN_CONCEPT_0000001341460225)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001341460225)

无

#### [使用实例](#ZH-CN_CONCEPT_0000001341460225)

查询宽带集群系统扩展功能配置参数，执行命令：

```
LST BTRUNCFUNC:;
RETCODE = 0  操作成功。 
```

```
查询结果如下
-------------------------
    是否支持集群业务  =  否

(结果个数 = 1)

---    END
```

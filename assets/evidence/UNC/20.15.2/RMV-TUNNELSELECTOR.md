#### [命令功能](#ZH-CN_CONCEPT_0000001600601169)

该命令用于删除隧道选择器。

#### [注意事项](#ZH-CN_CONCEPT_0000001600601169)

- 该命令执行后立即生效。
- 删除该隧道选择器时，要保证该隧道选择器被添加过。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600601169)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600601169)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| TNLSELECTORNAME | 隧道选择器名字 | 可选必选说明：必选参数<br>参数含义：该参数用于指定隧道选择器名字。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～40。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600601169)

删除名称为a的隧道选择器：

```
RMV TUNNELSELECTOR:TNLSELECTORNAME="a";
```

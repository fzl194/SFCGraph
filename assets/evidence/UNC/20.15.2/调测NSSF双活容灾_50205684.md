## [操作场景](#ZH-CN_OPI_0250205684)

本操作指导介绍在运行网络中调测NSSF双活容灾的操作过程。

> **说明**
> 适用于AMF 、NSSF 。

## [必备事项](#ZH-CN_OPI_0250205684)

前提条件

- 请仔细阅读[WSFD-110012 NSSF双活容灾特性概述](特性概述_50205681.md)。

- 完成[激活NSSF双活容灾](激活NSSF双活容灾_50205683.md)。

数据

| 类别 | 参数名称 | 取值样例 | 获取方法 | 说明 |
| --- | --- | --- | --- | --- |
| **需要查询的容灾实例信息** | 容灾实例标识（DRINSTID） | 0<br>1 | 与对端协商 | 通过命令<br>**ADD DRINST**<br>进行配置，可以使用命令<br>**LST DRINST**<br>查询。 |

工具

该操作无需准备其他工具。

## [操作步骤](#ZH-CN_OPI_0250205684)

1. 进入 “MML命令行-UNC” 窗口。
2. 执行 [**LST LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/查询License配置项开关（LST LICENSESWITCH）_09651570.md) 命令，查询本特性对应的License配置开关是否打开。
    - 如果“SWITCH”为“ENABLE”，请参考步骤3。
    - 如果“SWITCH”为“DISABLE”，则执行[**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)命令打开本特性对应的License配置开关。
3. 执行 **[LST DRINST](../../../../../OM参考/命令/UNC MML命令/平台服务管理/CSDB功能管理/CSDB管理/容灾管理/查询容灾实例(LST DRINST)_51012924.md)** 命令，查询容灾实例信息是否配置成功。
  预期结果：以其中一个NSSF为例，查询结果显示本端容灾实例标识对应的容灾实例名称已经配置正确。
  ```
  %%LST DRINST: DRINSTID=0;%%
  RETCODE = 0  操作成功。
 
  结果如下
  ------------------------  
   容灾实例标识 = 0
   容灾实例名称 = DR1
  (结果个数 = 1)

  ---    END
  ```
4. 执行 **[LST DRCOMM](../../../../../OM参考/命令/UNC MML命令/平台服务管理/CSDB功能管理/CSDB管理/容灾管理/查询容灾实例地址(LST DRCOMM)_51012928.md)** 命令，查询容灾实例通信地址信息是否配置成功。
  预期结果：以其中一个NSSF为例，查询结果显示已正确配置其容灾通信地址信息。
  ```
  %%LST DRCOMM: DRINSTID=0, IPVERSION=COMM_IPV4; %%
  RETCODE = 0  操作成功。
 
  结果如下
  ------------------------  
   容灾实例标识 = 0
       IP版本号 = IPV4
   通信IPv4地址 = 10.5.5.1
    VPN实例名称 = DR_VPN
  (结果个数 = 1)

  ---    END
  ```
5. 执行 **[DSP DRINFO](../../../../../OM参考/命令/UNC MML命令/平台服务管理/CSDB功能管理/CSDB管理/容灾管理/查询容灾信息(DSP DRINFO)_51012929.md)** 命令，在本端NSSF上查询容灾实例信息以及容灾间通道状态。
  预期结果：NSSF间通道正常，数据实时备份。

  ```
  DSP DRINFO: PEERDRINSTID=0; 
  RETCODE = 0  操作成功。
 
  结果如下
  ------------------------  
   本端容灾标识 = 1
   对端容灾标识 = 0
   数据备份状态 = 实时备份
  (结果个数 = 1)

  ---    END
  ```
6. 若以上步骤不能解决问题，请收集信息并寻求技术支持。
    a. 在 “MML命令行-UNC” 窗口上执行命令 **[EXP MML](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/配置管理/配置导出管理/导出MML文件（EXP MML）_47200033.md)** ，将当前配置数据导出为MML脚本文件并保存。
    b. 收集并保存上述所有查询信息。
    c. 收集归纳所有信息并联系华为技术支持解决。

#### [命令功能](#ZH-CN_MMLREF_0000001172225283)

**适用网元：SGSN、MME**

该命令用于查询IP区域群记录。

#### [注意事项](#ZH-CN_MMLREF_0000001172225283)

- 此命令执行后立即生效。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172225283)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172225283)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172225283)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| AREAID | 区域群标识 | 可选必选说明：可选参数<br>参数含义：该参数用于标识待查询的区域群。<br>数据来源：整网规划<br>取值范围：1~256<br>默认值：无 |

#### [使用实例](#ZH-CN_MMLREF_0000001172225283)

1. 查询标识为1的IP区域群记录：
  LST IPAREAGP: AREAID = 1;
  ```
  %%LST IPAREAGP: AREAID=1;%%
  RETCODE = 0  操作成功。

  输出结果如下
  --------------       
        区域群标识  =  1
        IP区域开关  =  关闭
      漫游用户开关  =  关闭
  本网异地用户开关  =  关闭
        区域群名称  =  noname
  (结果个数 = 1)

  ---    END
  ```

#### [命令功能](#ZH-CN_MMLREF_0000001172225371)

**适用网元：SGSN**

该命令用于查询Gb模式SM流程控制参数。

#### [注意事项](#ZH-CN_MMLREF_0000001172225371)

该命令执行后立即生效。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172225371)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172225371)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172225371)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROT | 流程类型 | 可选必选说明：可选参数<br>参数含义：该参数用于指定流程类型，要根据场景来配置相应的流程。<br>取值范围：<br>- “ACT_PROC(激活流程)”<br>- “MOD_PROC(修改流程)”<br>- “PFC_PROC(PFC流程)”<br>- “HSS_INIT_SUB_QOS_MOD_PROC(HSS发起的签约QoS修改流程)”<br>默认值：无<br>说明：若不输入流程类型，则将显示所有流程类型的原因值。 |

#### [使用实例](#ZH-CN_MMLREF_0000001172225371)

查询 “流程类型” 为 “修改流程” 的Gb模式SM流程控制参数：

LST GBSMPROCTRL: PROT=MOD_PROC;

```
%%LST GBSMPROCTRL: PROT=MOD_PROC;%%
RETCODE = 0  操作成功。

操作结果如下
--------------
修改拒绝原因值（GGSN拒绝）  =  3
(结果个数 = 1)

---    END
```

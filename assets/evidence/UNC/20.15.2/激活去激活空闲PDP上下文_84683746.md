## [操作场景](#ZH-CN_OPI_0184683746)

本操作指导介绍在运行网络中激活去激活空闲 PDP上下文 特性的操作过程。

去激活空闲PDP上下文是指在一个可控的时间内，如果一个PDP上下文没有任何流量， SGSN 将会去激活这个空闲PDP上下文。

> **说明**
> 适用于 SGSN 。

## [操作步骤](#ZH-CN_OPI_0184683746)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. 为相应计费属性的PDP上下文配置去激活空闲PDP上下文的时长。
  [**ADD CHGBEHA**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/计费管理/计费行为参数配置/增加计费行为参数(ADD CHGBEHA)_26145366.md)
  > **说明**
  > 参数 “BA” （行为属性）选择 “PLT” （关闭空闲 PDP 长定时器）或者 “PST” （关闭空闲PDP短定时器）。

## [任务示例](#ZH-CN_OPI_0184683746)

任务描述

增加一个计费行为，计费行为B1，行为属性为关闭空闲PDP长定时器，关闭空闲PDP定时器时长为30分钟，支持的计费属性包括普通计费属性、预付费计费属性、包月制计费属性和实时计费属性。

脚本

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2DAPDP02", SWITCH=ENABLE;
```

//计费行为设为“B1”，行为属性设为“PLT”，关闭空闲PDP定时器时长(min)设为“30分钟”，支持的计费属性设为“HOTBILLING-1&FLATRATE-1&PREPAID-1&NORMAL-1”。

```
ADD CHGBEHA: CB=B1, BA=PLT, PURGELEN=30, ACC=HOTBILLING-1&FLATRATE-1&PREPAID-1&NORMAL-1;
```

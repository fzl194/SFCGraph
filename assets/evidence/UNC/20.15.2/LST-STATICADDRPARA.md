## [命令功能](#ZH-CN_MMLREF_0232232821)

**适用NF：PGW-C、SMF、GGSN**

该命令用于查询静态地址相关参数。

## [注意事项](#ZH-CN_MMLREF_0232232821)

无

#### [操作用户权限](#ZH-CN_MMLREF_0232232821)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0232232821)

无

## [使用实例](#ZH-CN_MMLREF_0232232821)

查询静态地址相关参数：

```
LST STATICADDRPARA:;
RETCODE = 0  操作成功。

结果如下
------------------------
扫描任务开关  =  不使能
扫描开始时间  =  00:00
扫描结束时间  =  00:00
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0232232821)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 扫描任务开关 | 该参数用于指定静态用户扫描任务的开关。 |
| 扫描开始时间 | 该参数用于指定静态地址用户扫描开始UTC时间。 |
| 扫描结束时间 | 该参数用于指定静态地址用户扫描结束UTC时间。 |

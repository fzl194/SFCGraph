## [命令功能](#ZH-CN_MMLREF_0209653069)

**适用NF：SGW-C、PGW-C、AMF、GGSN**

该命令用于查询GTP-C本地实体组信息。

## [注意事项](#ZH-CN_MMLREF_0209653069)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209653069)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209653069)

无

## [使用实例](#ZH-CN_MMLREF_0209653069)

查询GTP-C本地实体组信息。

```
%%LST GTPCLEGRP:;%%
RETCODE = 0  操作成功

结果如下
--------
组号  描述信息  

0     s11          
1     s5_s         
2     s5_p         
3     n26 
(结果个数 = 4)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209653069)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 组号 | 该参数用于指定GTP-C本地实体所属的组。 |
| 描述信息 | 该参数用于指定GTP-C本地实体组的描述信息。 |

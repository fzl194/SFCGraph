## [命令功能](#ZH-CN_MMLREF_0000001240101321)

**适用NF：SMF、GGSN、PGW-C**

该命令用来查询UP重选相关的功能配置。

## [注意事项](#ZH-CN_MMLREF_0000001240101321)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001240101321)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001240101321)

无

## [使用实例](#ZH-CN_MMLREF_0000001240101321)

查看触发UP重选的场景：

```
%%LST UPRESELECTFUNC:;%%
RETCODE = 0  操作成功

结果如下
--------
UP重选场景  =  SMF分配地址失败
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001240101321)

| 输出项名称 | 输出项解释 |
| --- | --- |
| UP重选场景 | 该参数用来配置触发UP重选的场景。 |

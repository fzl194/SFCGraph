## [操作场景](#ZH-CN_OPI_0310243675)

AAA Server上的用户信息达到运营商设置的阈值条件时，AAA Server可以通过Gi接口下发CoA Request消息给GGSN/PGW-C/SMF，由GGSN/PGW-C/SMF控制修改用户的业务策略以及带宽控制。

> **说明**
> 适用于 GGSN、 PGW-C、SMF。

## [必备事项](#ZH-CN_OPI_0310243675)

前提条件

- 请仔细阅读[WSFD-109005 支持CoA功能](../WSFD-109005 支持CoA功能_10243673.md)。
- 完成**加载license**（如果有需求，请联系华为技术支持工程师处理）。

## [操作步骤](#ZH-CN_OPI_0310243675)

1. 打开本特性的License开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
2. 配置COA功能使能开关。
    a. 指定APN实例名称。
      [**ADD APN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/APN管理/APN/增加APN配置（ADD APN）_09653747.md)
    b. 配置CoA功能使能开关。
      [**SET APNAUTHATTR**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/APN管理/APN鉴权属性/设置APN鉴权属性配置（SET APNAUTHATTR）_28567656.md)
      > **说明**
      > [**SET APNAUTHATTR**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/APN管理/APN鉴权属性/设置APN鉴权属性配置（SET APNAUTHATTR）_28567656.md) 用于需要RADIUS鉴权或本地鉴权时，对APN的鉴权属性进行配置。当COA功能未使能时，收到COA Request消息时，直接响应COA NAK。
3. 配置QoS-ID进行用户QoS的协商控制，主动调整带宽。
    a. 配置用户的签约QoS属性。
      [**ADD PRER8SUBQOS**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/QoS/PreR8 QoS配置/本地PreR8 QoS/增加Pre-R8签约QoS配置（ADD PRER8SUBQOS）_09653616.md)
      **[**ADD EPSSUBQOS**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/QoS/EPS QoS配置/本地EPS QoS/增加EPS签约QoS配置（ADD EPSSUBQOS）_09653648.md)**
      **[**ADD 5GCSUBQOS**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/QoS/5GC QoS配置/本地5GC QoS/增加5GC签约QoS配置（ADD 5GCSUBQOS）_09652601.md)**
      > **说明**
      > 当APN下使能COA功能时，RADIUS鉴权服务器会下发用户QoS索引，GGSN/PGW-C/SMF根据索引匹配到subscriber-qos配置，用来进行用户QoS的协商控制。
    b. 配置AAA和PCRF共同协商QoS。
      **[SET APNQOSATTR](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/QoS/APN的QoS属性/设置指定APN的QoS属性配置信息（SET APNQOSATTR）_09651825.md)**

## [任务示例](#ZH-CN_OPI_0310243675)

任务描述

在apn-test的APN下使能CoA功能。

脚本

//打开本特性的License开关。

```
SET LICENSESWITCH: LICITEM="LKV2UNCCOA1", SWITCH=ENABLE;
```

//配置COA功能使能开关。

```
ADD APN:APN="apn-test";
SET APNAUTHATTR:APN="apn-test",COASWITCH=ENABLE;
```

//配置用户的签约QoS属性，使能AAA和PCRF共同协商QoS。

```
ADD PRER8SUBQOS: SUBQOSINDEX=1, HIGHESTTC=STREAMING, STRMGBRDL=50, STRMGBRUL=100, STRMMBRDL=150, STRMMBRUL=200, ARP=HIGH;
ADD EPSSUBQOS: SUBQOSINDEX=2, QCI=5, ARPPL=10, AMBRDL=564110, AMBRUL=300000;
ADD 5GCSUBQOS:SUBQOSINDEX=3, FIVEQI=5, QOS5QIPL=2, ARPPL=2, AMBRDL=1111, AMBRUL=2222;
SET APNQOSATTR:APN="apn-test",RDSANDPCCNEGO=ENABLE;
```

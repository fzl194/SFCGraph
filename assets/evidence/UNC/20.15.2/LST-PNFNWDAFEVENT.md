## [命令功能](#ZH-CN_MMLREF_0000002092262949)

**适用NF：AMF、SMF、NSSF、SMSF、NCG**

该命令用于查询本地配置的对端NWDAF的信息。

## [注意事项](#ZH-CN_MMLREF_0000002092262949)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000002092262949)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000002092262949)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| NFINSTANCEID | NF实例标识 | 可选必选说明：可选参数<br>参数含义：该参数用于唯一指定某一个NWDAF实例。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是1~38。本参数的构成字符只能是字母A~Z或a~z、数字0~9、下划线“_”和中划线“-”，例如，NWDAF_Instance_0。<br>默认值：无<br>配置原则：无 |
| NWDAFINFOID | NwdafInfo标识 | 可选必选说明：可选参数<br>参数含义：该参数用于唯一标识NWDAF实例中的某个NwdafInfo。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~32。该参数大小写不敏感。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000002092262949)

查询对端NWDAF信息。

```
%%LST PNFNWDAFEVENT:;%%
RETCODE = 0  操作成功

结果如下
--------
       NF实例标识  =  nwdaf_instance_1
    NwdafInfo标识  =  central
NWDAF数据分析事件  =  QOS分析
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000002092262949)

| 输出项名称 | 输出项解释 |
| --- | --- |
| NF实例标识 | 该参数用于唯一指定某一个NWDAF实例。 |
| NwdafInfo标识 | 该参数用于唯一标识NWDAF实例中的某个NwdafInfo。 |
| NWDAF数据分析事件 | 该参数用于指定NWDAF支持的分析事件类型。 |

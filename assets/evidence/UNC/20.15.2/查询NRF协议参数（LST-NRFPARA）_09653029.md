## [命令功能](#ZH-CN_MMLREF_0209653029)

**适用NF：AMF、SMF、NRF、NSSF、SMSF、NCG、CBCF**

该命令用于查询NRF协议相关的配置信息。

## [注意事项](#ZH-CN_MMLREF_0209653029)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209653029)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209653029)

无

## [使用实例](#ZH-CN_MMLREF_0209653029)

查询实例名称为NRF_Instance_0的NRF的参数相关的配置信息。

```
%%LST NRFPARA:;%%
RETCODE = 0  操作成功

结果如下
--------
         NRF实例名称  =  NRF_Instance_0
            等待时长  =  5
        最大重传次数  =  3
            心跳时长  =  60
心跳是否携带Load信息  =  否
        订阅流程开关  =  OFF
      OAUTH2鉴权开关  =  OFF
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209653029)

| 输出项名称 | 输出项解释 |
| --- | --- |
| NRF实例名称 | 该参数用于指定NRF实例名称。 |
| 等待时长 | 该参数用于指定NF向NRF发起请求消息后，等待响应的最大时长。 |
| 最大重传次数 | 该参数用于指定NF等待NRF响应超时后，重新发起请求消息的最大次数。 |
| 心跳时长 | 向NRF建议的心跳时长，实际心跳以NRF决策为准。 |
| 心跳是否携带Load信息 | 该参数用于指定是否在心跳消息中携带NF的Load信息。 |
| 订阅流程开关 | 该参数用于指定NF是否能发起订阅流程。 |
| OAUTH2鉴权开关 | 该参数用于指定NF是否开启Oauth2鉴权。 |

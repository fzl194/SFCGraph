## [命令功能](#ZH-CN_MMLREF_0212701671)

**适用NF：SMF、PGW-C、SGW-C、GGSN**

该命令用于查询UPC DS粒度的N4请求等待超时异常的告警恢复阈值。

## [注意事项](#ZH-CN_MMLREF_0212701671)

当前版本不支持此命令。

#### [操作用户权限](#ZH-CN_MMLREF_0212701671)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0212701671)

无

## [使用实例](#ZH-CN_MMLREF_0212701671)

查询UPC DS粒度N4请求等待超时异常的告警恢复阈值：

```
%%LST UPCALMRSTVAL:;%%
RETCODE = 0  操作成功

结果如下
--------
UPC告警恢复阈值(秒)  =  10
(结果个数 = 1)

  -----    END
```

## [输出结果说明](#ZH-CN_MMLREF_0212701671)

| 输出项名称 | 输出项解释 |
| --- | --- |
| UPC告警恢复阈值(秒) | UPC DS粒度N4请求等待超时异常的恢复告警阈值。 |

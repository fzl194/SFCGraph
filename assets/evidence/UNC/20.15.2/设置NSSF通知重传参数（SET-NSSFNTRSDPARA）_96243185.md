## [命令功能](#ZH-CN_MMLREF_0296243185)

**适用NF：NSSF**

该命令用于设置NSSF通知重传参数信息。

## [注意事项](#ZH-CN_MMLREF_0296243185)

- 该命令执行后立即生效。

- 最大通知重传次数默认值为0，即通知重传关闭。

- 系统部署完成后，已经存在初始记录，参数的初始记录值如下表：

| MAXRESENDTIMES | RESENDPERIOD | RESENDCODELIST |
| --- | --- | --- |
| 0 | 20 | 429.500.503.504.602 |

#### [操作用户权限](#ZH-CN_MMLREF_0296243185)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0296243185)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| MAXRESENDTIMES | 最大通知重传次数 | 可选必选说明：可选参数<br>参数含义：该参数用于表示执行通知任务时，通知失败后进行通知重传的最大次数。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是0~10080，单位是次。<br>默认值：无。执行命令并不输入该参数时，该参数保持系统当前配置不变，可通过LST NSSFNTRSDPARA查询当前参数配置值。<br>配置原则：无 |
| RESENDPERIOD | 通知重传执行周期(分钟) | 可选必选说明：可选参数<br>参数含义：该参数用于表示重传通知任务时两次通知的时间间隔。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~10080，单位是分钟。<br>默认值：无。执行命令并不输入该参数时，该参数保持系统当前配置不变，可通过LST NSSFNTRSDPARA查询当前参数配置值。<br>配置原则：无 |
| RESENDCODELIST | 重传响应码列表 | 可选必选说明：可选参数<br>参数含义：该参数用于表示需要进行通知重传的响应码列表。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~256。不支持通知重传的响应码：204、404。<br>默认值：无。执行命令并不输入该参数时，该参数保持系统当前配置不变，可通过LST NSSFNTRSDPARA查询当前参数配置值。<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0296243185)

当NSSF网元在执行通知任务时，可能会因为各种偶发性因素导致通知失败，此时可以通过设置NSSF通知重传参数信息来规避这种偶发性因素对业务造成的影响。 此处设置最大通知重传次数设置为1000次，通知重传执行周期设置为50分钟，重传响应码列表设置为"429.500.503.504"。

```
SET NSSFNTRSDPARA: MAXRESENDTIMES=1000, RESENDPERIOD=50, RESENDCODELIST="429.500.503.504";
```

# 删除NF TAC区域信息（RMV TACRANGE）

- [注意事项](#ZH-CN_MMLREF_0209653266__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0209653266__1.3.4)
- [使用实例](#ZH-CN_MMLREF_0209653266__1.3.5)

## [命令功能](#ZH-CN_MMLREF_0209653266)

**适用NF：AMF、SMF、NRF、NSSF、NCG、SMSF**

该命令用于删除TAC区域信息。

## [注意事项](#ZH-CN_MMLREF_0209653266)

- 该命令执行后立即生效。

- 当该命令记录的INDEX被TAIRANGELIST引用时，无法进行删除操作。

#### [操作用户权限](#ZH-CN_MMLREF_0209653266)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209653266)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| INDEX | 索引标识 | 可选必选说明：必选参数<br>参数含义：本参数用于指定索引标识，用于标识TAC区域。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是0~2047。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209653266)

运营商A需要删除索引标识为1的TAC区域信息。

```
RMV TACRANGE: INDEX=1;
```

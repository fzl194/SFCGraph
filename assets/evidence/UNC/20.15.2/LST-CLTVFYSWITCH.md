## [命令功能](#ZH-CN_MMLREF_0000002119797985)

此命令用于查询双向认证开关的开关状态。

## [注意事项](#ZH-CN_MMLREF_0000002119797985)

无。

## [参数说明](#ZH-CN_MMLREF_0000002119797985)

无。

## [使用实例](#ZH-CN_MMLREF_0000002119797985)

查询双向认证开关：

```
LST CLTVFYSWITCH;
```

```
%%LST CLTVFYSWITCH:;%%
RETCODE = 0  操作成功  
操作结果如下 
------------ 
端口  开关    
6443  关      
9000  关      
(结果个数 = 2)

---    END
```

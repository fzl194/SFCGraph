## [操作场景](#ZH-CN_OPI_0000001588959889)

本操作指导用于核心网所有网元及gNodeB都支持RedCap功能的场景下，激活RedCap用户接入。

> **说明**
> 适用于AMF和SMF。

## [对系统的影响](#ZH-CN_OPI_0000001588959889)

该操作对系统正常运行没有影响。

## [操作步骤](#ZH-CN_OPI_0000001588959889)

1. 进入 “MML命令行-UNC” 窗口。
2. 增加AMF兼容性策略：
  [**SET AMFREDCAPFUNC**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G M2M管理/RedCap功能管理/设置AMF RedCap功能（SET AMFREDCAPFUNC）_17306402.md)
3. 设置SMF扩展功能：
  [**SET SMFFUNC**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/会话协议参数管理/会话管理拓展功能/5GC会话管理拓展功能/设置SMF扩展功能（SET SMFFUNC）_09653731.md)
4. 设置RedCap终端接入RAT填写值：
  [**SET REDCAPRATVALUE**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/计费管理/计费控制/RedCap用户RAT值/设置RedCap终端接入RAT填写值（SET REDCAPRATVALUE）_68425885.md)
5. 设置RedCap终端接入Qos控制：
  **[SET QOSCTRL](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/QoS/全局QoS功能配置/全局QoS控制功能/设置QoS控制配置（SET QOSCTRL）_09653269.md)**

## [任务示例](#ZH-CN_OPI_0000001588959889)

任务描述

在所有网元和gNodeB均支持RedCap的场景下，配置RedCap用户接入核心网。

脚本

//打开AMF上的RedCap功能开关，RatType所有周边网元兼容性开关，配置AMF向空闲态PDU会话关联的SMF通知RatType的策略。

```
SET AMFREDCAPFUNC:REDCAPSW=YES,RATCMPTSW=UDM-1&PCF-1&SMSF-1&SMF-1,NTYSMFPLCY=IMMEDIATELY_NOTIFY_ON_CHANGE-1&CHANNEL_ASSOCIATED_NOTIFY-0;
```

//打开SMF上的RedCap功能开关。

```
SET SMFFUNC: REDCAPSW=Support;
```

//配置SMF向所有周边网元传递RatType的值为NR_REDCAP。

```
SET REDCAPRATVALUE: CHF=NR_REDCAP,PCF=NR_REDCAP,UPF=NR_REDCAP,AAAACCT=NR_REDCAP,AAAAUTH=NR_REDCAP,N16SMF=NR_REDCAP,N16ASMF=NR_REDCAP;
```

//配置SMF上的RedCap制式的Qos控制功能开关。

```
SET QOSCTRL: USERTYPE=HOME, RATTYPE=REDCAP-1;
SET QOSCTRL: USERTYPE=VISITING, RATTYPE=REDCAP-1;
SET QOSCTRL: USERTYPE=ROAMING, RATTYPE=REDCAP-1;
```

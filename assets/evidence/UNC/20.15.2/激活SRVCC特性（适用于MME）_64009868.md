## [操作场景](#ZH-CN_OPI_0164009868)

本操作指导介绍在运行网络中部署MME的SRVCC基本业务过程。

## [必备事项](#ZH-CN_OPI_0164009868)

前提条件

- 已完成加载License。
- MME、MSC Server、eNodeB等网元基本业务的数据配置已完成。
- Sv接口对接数据已经配置完成，请参见[Sv接口配置指导](../../../../初始配置/UNC初始配置与调测/组网对接配置/配置AMF&MME&SGSN/配置MME/配置到MSC的数据（Sv接口）_82626704.md)。
- 用户终端需要支持SRVCC业务，并已在HSS签约相应数据。

## [操作步骤](#ZH-CN_OPI_0164009868)

1. 进入 “MML命令行-UNC” 窗口。
2. 设置EPS Fallback回落后，TAU过程携带UE的SRVCC能力。
  **[SET SOFTPARAOFBIT](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/操作维护/软件参数管理/软件参数比特位/设置软件参数表比特位(SET SOFTPARAOFBIT)_72345783.md)**
3. 设置协议兼容性。
  [**ADD GTPCV2CMPT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/GTP-C协议管理/GTP-C V2协议兼容性/增加GTP-C V2协议兼容性(ADD GTPCV2CMPT)_26305734.md)
4. 设置Diameter兼容性。
  [**SET DMCMPT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/Diameter管理/Diameter协议接口兼容性配置/设置Diameter兼容性(SET DMCMPT)_26306080.md)
5. **可选：**设置SRVCC切换准备定时器。
  [**SET T3N3**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/GTP-C协议管理/GTP-C协议参数配置/设置GTP-C T3_N3参数配置(SET T3N3)_26305730.md)
6. **可选：**设置SRVCC切换完成定时器。
  [**SET T3N3**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/GTP-C协议管理/GTP-C协议参数配置/设置GTP-C T3_N3参数配置(SET T3N3)_26305730.md)
7. 核对协议版本。
  [**SET SYS**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/系统管理/系统参数管理/设置系统参数(SET SYS)_72345947.md)

## [任务示例](#ZH-CN_OPI_0164009868)

任务描述

某局点已部署VoLTE业务，需要开启SRVCC功能。

脚本

进入 “MML命令行-UNC” 窗口。

//设置EPS Fallback回落后，TAU过程携带UE的SRVCC能力。

```
SET SOFTPARAOFBIT: DT=BYTE_EX_B, PARANUM=330, VALUE=VALUE_1, POSITION=2;
```

//设置协议兼容性。

```
ADD GTPCV2CMPT: IDX=0, MSGCLS=SV, SVMSGTYPE=PS_TO_CS_REQ, PSTOCSREQ=STN-SR, STNSR=CARRY;
```

//设置Diameter兼容性。

```
SET DMCMPT: UESRVCC=SUPPORT;
```

//设置SRVCC切换准备定时器。

```
SET T3N3: GTPVER=GTPv2, V2MSGCLS=MM, V2SMMMSGTYPE=SRVCC_PS_CS_REQ, T3RES=7, N3REQ=1;
```

//设置SRVCC切换完成定时器。

```
SET T3N3: GTPVER=GTPv2, V2MSGCLS=MM, V2SMMMSGTYPE=SRVCC_PS_CS_COMPLETE_NOTIFY, T3RES=7, N3REQ=1;
```

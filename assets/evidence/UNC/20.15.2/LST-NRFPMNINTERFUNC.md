## [命令功能](#ZH-CN_MMLREF_0000001225448787)

**适用NF：NRF**

此命令用于查询基于对端PLMN的国际漫游功能参数。

## [注意事项](#ZH-CN_MMLREF_0000001225448787)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001225448787)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001225448787)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| MCC | 移动国家码 | 可选必选说明：可选参数<br>参数含义：该参数用于表示指定对端PLMN的移动国家码。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度是3。<br>默认值：无<br>配置原则：无 |
| MNC | 移动网号 | 可选必选说明：可选参数<br>参数含义：该参数用于表示指定对端PLMN的移动网号。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是2~3。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001225448787)

查询基于对端PLMN国际漫游功能参数：NA

```
%%LST NRFPMNINTERFUNC:;%%
RETCODE = 0  操作成功

结果如下
--------
                  移动国家码  =  233
                    移动网号  =  13
本NRF是否是国际漫游关口局NRF  =  是
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001225448787)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 移动国家码 | 该参数用于表示指定对端PLMN的移动国家码。 |
| 移动网号 | 该参数用于表示指定对端PLMN的移动网号。 |
| 本NRF是否是国际漫游关口局NRF | 该参数用于表示本NRF基于指定的PLMN是否是漫游关口局NRF。 |
| 关口局NF选择策略 | 该参数用于表示本NRF作为关口局NRF收到漫游服务发现请求时，对于关口局NF的选择策略。 |

## [命令功能](#ZH-CN_MMLREF_0209654148)

**适用NF：AMF、SMF、SGW-C、PGW-C、NRF、NSSF、SGW-U、PGW-U、NCG、SMSF**

该命令用于查询系统内部运行信息，不涉及用户敏感数据，方便问题定位。

## [注意事项](#ZH-CN_MMLREF_0209654148)

- 该命令只允许在华为工程师指导下执行。
- 该命令用于查询产品内部模块的运行状态是否正常。
- 该命令执行后立即生效。
- 输出的内部运行信息随着系统运行的时间和流程会有所不同。

#### [操作用户权限](#ZH-CN_MMLREF_0209654148)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

```

RETCODE = 0  Operation succeeded

No matching result is found
---    END
```

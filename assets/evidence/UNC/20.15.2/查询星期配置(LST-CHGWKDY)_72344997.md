#### [命令功能](#ZH-CN_MMLREF_0000001172344997)

**适用网元：SGSN**

该命令用于查询计费星期表配置信息。星期表与 [**ADD CHGTARI**](../计费费率时段配置/增加费率时段配置(ADD CHGTARI)_26305208.md) 、 [**ADD CHGHOLI**](../计费节假日配置/增加节假日配置(ADD CHGHOLI)_26145382.md) 相结合，实现灵活的费率配置。

#### [注意事项](#ZH-CN_MMLREF_0000001172344997)

如果有输入参数，则显示与输入参数均匹配的星期表记录；如果没有输入参数，则显示所有星期表记录。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172344997)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172344997)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172344997)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| WKDAY | 星期序号 | 可选必选说明：可选参数<br>参数含义：该参数用于指定一个星期中的某一天。<br>取值范围：<br>- “MON(星期一)”<br>- “TUES(星期二)”<br>- “WED(星期三)”<br>- “THURS(星期四)”<br>- “FRI(星期五)”<br>- “SAT(星期六)”<br>- “SUN(星期日)”<br>默认值：无 |
| CC | 计费属性 | 可选必选说明：可选参数<br>参数含义：该参数用于指定该天是对普通计费属性用户、预付费计费属性用户、包月制计费属性用户还是对实时计费属性用户有效。<br>取值范围：<br>- “NORMAL(普通计费)”：表示普通计费属性，按照此种方式计费的用户按照常规的方式支付费用<br>- “PREPAID(预付费)”：表示预付费计费属性，按照此种方式计费的用户在获取某种服务之前需要预支付一定的费用。<br>- “FLATRATE(包月制)”：表示包月制计费属性，按照此种方式计费的用户在一个月内的收费是固定的。<br>- “HOTBILLING(实时计费)”：表示实时计费属性，按照此种方式计费的用户将在短时间或流量达到某个值时及时生成话单，保证运营商对此类用户及时收费。<br>默认值：无 |
| TT | 费率类型 | 可选必选说明：可选参数<br>参数含义：该参数用于指定该天的费率类型。<br>取值范围：<br>- “WORK(工作日)”<br>- “WEEKEND(周休日)”<br>默认值：无 |

#### [使用实例](#ZH-CN_MMLREF_0000001172344997)

1. 查询预付费计费属性的星期一的配置信息，配置格式为：
  LST CHGWKDY: WKDAY=MON, CC=PREPAID;
  ```
  %%LST CHGWKDY: WKDAY=MON, CC=PREPAID;%%
  RETCODE = 0  操作成功。

  输出结果如下
  ------------
  星期序号  =  星期一
  计费属性  =  预付费
  费率类型  =  工作日
  (结果个数 = 1)
  ---    END
  ```
2. 查询预付费计费属性的所有的星期表配置信息，配置格式为：
  LST CHGWKDY: CC=PREPAID;
  ```
  %%LST CHGWKDY: CC=PREPAID;%%
  RETCODE = 0  操作成功。

  输出结果如下

  星期六      预付费      周休日  
  (结果个数 = 7)
  ---    END
  ```

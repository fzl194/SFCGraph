## [命令功能](#ZH-CN_MMLREF_0000001181558911)

**适用NF：SMF**

该命令用于查询QoS监测配置。

## [注意事项](#ZH-CN_MMLREF_0000001181558911)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001181558911)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001181558911)

无

## [使用实例](#ZH-CN_MMLREF_0000001181558911)

查询QoS监测配置，执行如下命令：

```
%%LST QOSMONT:;%%
RETCODE = 0  操作成功

结果如下
--------
           QoS监测开关 = 使能
(结果个数 = 1)

---    END
```

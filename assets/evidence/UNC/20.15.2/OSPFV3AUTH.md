#### [命令功能](#ZH-CN_CONCEPT_0000001549961174)

该命令用于查询OSPFv3进程认证配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001549961174)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549961174)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549961174)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | OSPFv3进程号 | 可选必选说明：可选参数<br>参数含义：OSPFv3进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549961174)

查询OSPFv3进程认证配置：

```
LST OSPFV3AUTH:;
```

```

RETCODE = 0  操作成功。

结果如下
------------------------
     OSPFv3进程号  =  1
     验证字标识符  =  1000
         认证类型  =  HMAC-SHA256
HMAC-SHA256 密码   =  *****
(结果个数 = 1)
---    END
```

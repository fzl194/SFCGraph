# 删除用户黑名单限制列表（RMV USRBLACKLST）

- [注意事项](#ZH-CN_MMLREF_0000001615553257__1.3.2)

## [命令功能](#ZH-CN_MMLREF_0000001615553257)

**适用NF：SGSN、MME、AMF**

该命令用于删除黑名单限制的用户列表。

## [注意事项](#ZH-CN_MMLREF_0000001615553257)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0000001615553257)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001615553257)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IMSI | IMSI | 可选必选说明：必选参数<br>参数含义：该参数用于指定限制用户黑名单接入的IMSI。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度是15。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001615553257)

删除一条黑名单限制记录，执行如下命令：

```
RMV USRBLACKLST: IMSI="123456789012345";
```

# 查询虚拟APN映射策略配置（LST VIRTUALAPNRULE）

## [命令功能](#ZH-CN_MMLREF_0209653739)

**适用NF：SMF、PGW-C、GGSN**

该命令用来查询指定虚拟APN的匹配规则。

## [注意事项](#ZH-CN_MMLREF_0209653739)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209653739)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209653739)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| VIRTUALAPN | 虚拟APN名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定APN实例为虚拟APN。该APN必须是系统已经配置的APN，并且该APN下的VIRTUALAPN字段是使能的。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~63。不区分大小写。不支持空格及部分特殊字符。可以支持的特殊字符有“.”和“-”，“.”不可以是第一个字符且不可以连续出现。<br>默认值：无<br>配置原则：<br>该参数的取值必须在ADD APN中进行配置。 |

## [输出结果说明](#ZH-CN_MMLREF_0209653739)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 虚拟APN名称 | 该参数用于指定APN实例为虚拟APN。该APN必须是系统已经配置的APN，并且该APN下的VIRTUALAPN字段是使能的。 |
| 映射方式 | 该参数用于表示虚拟APN的映射规则。 |
| 优先级 | 该参数用于指定配置的映射规则的优先级。同一虚拟APN下，值越小优先级越高，除LOAD_BALANCE和REDUNDANCY映射规则外，不允许指定相同的优先级。如果PRIORITY参数没有配置，表明优先级最低。 |
| 计费属性模式 | 该参数用于表示按照Charging Characteristics(计费属性)信元的取值进行虚拟APN的映射。 |
| 映射值 | 该参数用于指定所选映射值。 |
| 计费属性值 | 该参数用于指定配置的其他计费特征的值。 |
| 计费特征掩码和优先级开关 | 该参数用于指定是否配置计费特征值掩码和优先级。 |
| 计费特征掩码 | 该参数用于指定计费属性的掩码，通过配置掩码可以实现全匹配CC值，还是只匹配部分bit位。 |
| 计费特征优先级 | 该参数用于表示设置优先级，值越小优先级越高。配置CCMASK时必须指定优先级，同一虚拟APN下不允许指定相同的优先级。PRIORITY不配置时，才会根据该参数配置的优先级进行CC的虚拟APN映射。 |
| 无线接入类型 | 该参数用于表示根据用户RAT（无线接入类型）类型来匹配真实APN。 |
| 漫游状态 | 该参数用于表示根据用户漫游状态来匹配真实APN。 |
| 真实APN开关 | 该参数用于指定是否配置真实APN。 |
| IMSI | 该参数用于表示匹配真实APN时使用的IMSI号段。 |
| MSISDN | 该参数用于表示匹配真实APN时使用的MSISDN号段。 |
| IMEI | 该参数用于表示匹配真实APN时使用的IMEI号段。 |
| LAC组名 | 该参数用于表示匹配真实APN时使用的LAC组。 |
| TAC组名 | 该参数用于表示匹配真实APN时使用的TAC组。 |
| RAC组名 | 该参数用于表示匹配真实APN时使用的RAC组。 |
| Serving Node组名称 | 该参数用于表示匹配真实APN时使用的Serving Node组。 |
| PLMN | 该参数用于表示匹配真实APN时使用的PLMN。 |
| PCO开关 | 该参数用于表示是否根据PCO匹配真实APN。 |
| 域名 | 该参数用于表示匹配真实APN时使用的PCO信元中解析出的域名。 |
| 选择模式 | 该参数用于表示匹配真实APN时使用的Selection Mode。 |
| 切片业务类型 | 该参数表示匹配真实APN的切片的业务类型，如eMBB（1）、URLLC（2）、MIoT（3）等协议定义的标准SST，或者运营商自定义的SST。 |
| 切片细分标识 | 该参数用于表示匹配真实APN使用的切片细分标识。切片细分标识表示根据服务群体，对某类网络业务切片的进一步细分。 |
| 真实APN名称 | 该参数用于表示配置的规则对应的APN。该APN必须是系统已经配置的APN。 |

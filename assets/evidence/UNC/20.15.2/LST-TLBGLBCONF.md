## [命令功能](#ZH-CN_MMLREF_0000001915834601)

该命令用于查询HTTP服务端TCP链路整系统负载均衡（TLB）的功能开关及相关参数。

## [注意事项](#ZH-CN_MMLREF_0000001915834601)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001915834601)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001915834601)

无

## [使用实例](#ZH-CN_MMLREF_0000001915834601)

如果想查询HTTP服务端TCP链路整系统负载均衡（TLB）的功能开关及相关参数，可以用如下命令：

```
%%LST TLBGLBCONF:;%%
RETCODE = 0  操作成功

结果如下
--------
                           TLB全局开关  =  打开
                              均衡模式  =  链路数控制
                   重均衡偏差阈值（%）  =  10
                  重均衡最小链路数阈值  =  5
              重均衡最小时间间隔（秒）  =  30
    接收TCP SYN报文的流控阈值（包/秒）  =  2000
上报直通TCP SYN报文的流控阈值（包/秒）  =  200
            临时五元组策略有效期（秒）  =  5
                        五元组核查开关  =  打开
                五元组核查周期（分钟）  =  30
                         五元组老化阈值 =  5
              TLB表项流控阈值（个/秒）  =  1000
                暂停TLB功能的阈值（%）  =  95
                恢复TLB功能的阈值（%）  =  85
                SYN包转发时延（毫秒）   =  20
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001915834601)

| 输出项名称 | 输出项解释 |
| --- | --- |
| TLB全局开关 | 该参数用于设置TLB功能全局开关。如果打开开关，则控制所有HTTP服务端TCP链路的分布，使所有HTTP进程间负载均衡。 |
| 均衡模式 | 该参数用于指定均衡控制模式。 |
| 重均衡偏差阈值（%） | 该参数用于设置服务端链路重均衡的负载偏差门限值。如果某个HTTP进程的负载超过平均负载的百分比数大于或等于该参数的值，则记录该进程满足重均衡的一个条件。 |
| 重均衡最小链路数阈值 | 该参数用于设置服务端链路重均衡的最小链路数的门限值。如果某个HTTP进程的链路数大于该参数的值，则记录该进程满足重均衡的一个条件。 |
| 重均衡最小时间间隔（秒） | 该参数用于设置服务端链路重均衡的最小时间间隔。启动重均衡检测时，如果距离上一次均衡调整结束的时长未达到该参数设置的最小间隔时长，则不启动重均衡。 |
| 接收TCP SYN报文的流控阈值（包/秒） | 该参数用于设置TLB控制器接收TCP SYN报文的速率流控阈值。TCP SYN报文数量过大，可能导致系统负荷过高，无法接收正常的TCP连接请求。设置该参数可以预防开启TLB功能后的TCP SYN Flood攻击，保护系统正常运行。 |
| 上报直通TCP SYN报文的流控阈值（包/秒） | 该参数用于设置HTTP服务上报非预期的TCP SYN报文的速率流控阈值。对于开启负载均衡控制的链路，如果TCP SYN报文直接发到HTTP服务，则认为是非预期的TCP SYN报文，上报给TLB控制器处理，并且按照该参数的值限制上报的报文速率。 |
| 临时五元组策略有效期（秒） | 该参数用于设置服务端链路五元组临时策略的有效期。TLB控制器接收到TCP SYN后，生成五元组的临时策略，如果在该参数设置的时长内未成功建链，则删除临时策略。 |
| 五元组核查开关 | 该参数用于设置服务端链路五元组信息核查开关。 |
| 五元组核查周期（分钟） | 该参数用于设置服务端链路五元组信息核查周期时长。 |
| 五元组老化阈值 | 该参数用于设置服务端链路五元组信息老化阈值。TLB控制器周期核查5元组与物理链路的一致性，如果连续多个周期都没有查到五元组的物理链路信息，且周期数超过该参数设置的值，则删除五元组信息。 |
| TLB表项流控阈值（个/秒） | 该参数用于设置TLB策略表项操作速率的流控阈值。 |
| 暂停TLB功能的阈值（%） | 该参数用于设置暂停TLB功能的五元组占用率阈值，当五元组数量占用系统规格的比例超过该阈值后，暂停TLB的功能，即不做链路均衡控制。 |
| 恢复TLB功能的阈值（%） | 该参数用于设置恢复TLB功能的五元组占用率阈值，当五元组数量占用系统规格的比例低于该阈值后，恢复TLB的功能。 |
| SYN包转发时延（毫秒） | 当参数“TLB全局开关”打开时，该参数用于设置服务端链路建链过程中SYN包的转发延迟时间，以保障建链时TLB策略已生效，避免建链失败。在NP场景下建议配置为50毫秒。 |

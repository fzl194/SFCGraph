## [命令功能](#ZH-CN_MMLREF_0000002082282718)

**适用NF：SMF、PGW-C**

该命令用于查询全局默认智能PCF组。

## [注意事项](#ZH-CN_MMLREF_0000002082282718)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000002082282718)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000002082282718)

无

## [使用实例](#ZH-CN_MMLREF_0000002082282718)

查询全局智能PCF组配置：

```
%%LST DFTIPEPCFGRP:;%%
RETCODE = 0  操作成功

结果如下
--------
主智能PCF组  =  NULL
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000002082282718)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 主智能PCF组 | 该参数用于设置主智能PCF组的名称。 |

#### [命令功能](#ZH-CN_MMLREF_0000001172226031)

**适用网元：SGSN**

该命令用于查询BSC下的用户数。

#### [注意事项](#ZH-CN_MMLREF_0000001172226031)

无。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172226031)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172226031)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172226031)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| TYPE | 查询类型 | 可选必选说明：可选参数<br>参数含义：该参数用于指定查询类型。<br>数据来源：整网规划<br>取值范围：<br>- “SYSTEM(整系统)”<br>- “BYBSC(基于BSC查询)”<br>默认值：<br>“SYSTEM(整系统)” |
| BSSID | BSS编号 | 可选必选说明：条件必选参数<br>参数含义：该参数用于指定BSS编号。<br>前提条件：此参数在<br>“查询类型”<br>设置为<br>“BYBSC(基于BSC查询)”<br>后生效。<br>数据来源：整网规划<br>取值范围： 0-65534<br>默认值： 无。 |

#### [使用实例](#ZH-CN_MMLREF_0000001172226031)

查询BSC下所有用户数：

```
%%DSP BSCUSRNUM:TYPE=SYSTEM;%%
RETCODE = 0  操作成功。

BSC用户数信息
-------------
附着用户数  =  200
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_MMLREF_0000001172226031)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 附着用户数 | 参数含义：显示当前BSC的附着用户数。 |

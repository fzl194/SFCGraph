## [命令功能](#ZH-CN_MMLREF_0244529805)

**适用NF：SGW-C、PGW-C**

该命令用来查询指定区域性能统计对象的配置信息。

## [注意事项](#ZH-CN_MMLREF_0244529805)

无

#### [操作用户权限](#ZH-CN_MMLREF_0244529805)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0244529805)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| REGNAME | 区域名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定区域名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~31。不支持空格，不区分大小写。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0244529805)

当运营商需要查询名为“beijing”的区域性能统计对象时，执行如下命令：

```
LST PERFREGION: REGNAME="beijing";
%%LST PERFREGION: REGNAME="beijing";%%
RETCODE = 0  操作成功

结果如下
--------
区域名称  =  beijing
(结果个数 = 1)

---    END
```

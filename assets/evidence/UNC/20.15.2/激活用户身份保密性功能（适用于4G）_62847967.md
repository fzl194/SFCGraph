## [操作场景](#ZH-CN_OPI_0162847967)

本操作指导介绍在运行网络中激活用户身份保密特性的操作过程。

用户身份保密性功能是通过GUTI的分配来实现。分配GUTI（Globally Unique Temporary UE Identity）的目的是为入网的用户提供一个临时标识，同时这个标识不会展现用户的永久性标识。

> **说明**
> 适用于MME

## [操作步骤](#ZH-CN_OPI_0162847967)

1. 配置GUTI重分配策略。
  进入 “MML命令行-UNC” 窗口。
  [**SET EMM**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/MM协议参数管理/S1模式MM协议参数/设置S1模式MM协议参数(SET EMM)_72225207.md)
2. 配置GUTI分配方式。
  > **说明**
  > 系统默认采用增强TMSI分配方式，只有当增强算法存在无法解决的问题时，才使用非增强算法。为确保GUTI分配方式在各业务VM上的一致性，当执行此步骤修改GUTI分配方式后，需立即整系统重启。
  进入 “MML命令行-UNC” 窗口。
  [**SET SYS**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/系统管理/系统参数管理/设置系统参数(SET SYS)_72345947.md)

## [任务示例](#ZH-CN_OPI_0162847967)

任务描述

配置在Attach或TAU流程中使用增强TMSI算法重分配GUTI，重分配定时器时长和最大间隔均为1小时。

脚本

//配置GUTI重分配策略。

```
SET EMM:GUTITMR=1,GUTIREALLOC=YES,GUTIREALLOCTMR=1;
```

//配置GUTI分配方式。

```
SET SYS: TMSIALLOC=ENHAN;
```

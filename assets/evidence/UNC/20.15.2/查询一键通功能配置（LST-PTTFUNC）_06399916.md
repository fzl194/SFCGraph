## [命令功能](#ZH-CN_MMLREF_0306399916)

**适用NF：PGW-C、SGW-C**

该命令用于查询整机一键通功能相关的配置信息。

## [注意事项](#ZH-CN_MMLREF_0306399916)

无

#### [操作用户权限](#ZH-CN_MMLREF_0306399916)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0306399916)

无

## [使用实例](#ZH-CN_MMLREF_0306399916)

显示全局一键通功能配置：

```
%%LST PTTFUNC:;%%
RETCODE = 0  操作成功

结果如下
--------
LTE一键通功能开关  =  不使能
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0306399916)

| 输出项名称 | 输出项解释 |
| --- | --- |
| LTE一键通功能开关 | 该参数用于全局控制开启和关闭LTE一键通功能。 |

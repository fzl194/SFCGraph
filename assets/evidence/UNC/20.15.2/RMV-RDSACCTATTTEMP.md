# 删除计费消息属性模板（RMV RDSACCTATTTEMP）

#### [命令功能](#ZH-CN_CONCEPT_0209896784)

**适用NF：PGW-C、SMF**

该命令用来删除计费消息属性模板。

#### [注意事项](#ZH-CN_CONCEPT_0209896784)

- 该命令执行后立即生效。
- 若该模板已被RADIUS抄送服务器绑定，则无法删除。

#### [操作用户权限](#ZH-CN_CONCEPT_0209896784)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0209896784)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| ATTRTEMPNAME | 计费消息属性模板名字 | 可选必选说明：必选参数<br>参数含义：该参数用于指定计费消息属性模板名字。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0209896784)

删除名为“RDSACCTATTTEMP”的计费消息属性模板：

```
RMV RDSACCTATTTEMP: ATTRTEMPNAME="RDSACCTATTTEMP";
```

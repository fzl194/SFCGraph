## [命令功能](#ZH-CN_MMLREF_0000001226032197)

该命令用于查询设备上VPN实例下配置的地址族及参数。

## [注意事项](#ZH-CN_MMLREF_0000001226032197)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001226032197)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001226032197)

无

## [使用实例](#ZH-CN_MMLREF_0000001226032197)

查询设备上所有VPN实例的地址族，其中VPN实例_public_为公网实例，表示相对于私网的公网概念：

```
LST VPNINSTAFIPSEC:;

 RETCODE = 0  操作成功。

        结果如下
        -------------------------
        VPN实例名称   vrf1    
        地址族类型  IPv4 address family                     
        (结果个数 = 1)
         ---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001226032197)

| 输出项名称 | 输出项解释 |
| --- | --- |
| VPN实例名称 | 该参数用于指定用户增加地址族的L3VPN实例名称。 |
| 地址族类型 | 该参数用于设置指定VPN实例下的地址族。 |

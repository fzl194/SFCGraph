#### [命令功能](#ZH-CN_CONCEPT_0000001550121418)

该命令用于显示配置的Ipv4静态路由的详细信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001550121418)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550121418)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550121418)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| AFTYPE | 地址族 | 可选必选说明：可选参数<br>参数含义：该参数用来指定VPN实例的地址族信息。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- ipv4unicast：IPv4单播。<br>默认值：ipv4unicast |
| VRFNAME | VPN实例名称 | 可选必选说明：可选参数<br>参数含义：该参数用于表示路由前缀所属VPN实例的名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。<br>默认值：无<br>配置原则：不输入默认查询公网。 |
| PREFIX | 路由前缀 | 可选必选说明：可选参数<br>参数含义：该参数用于表示路由前缀信息。<br>数据来源：本端规划<br>取值范围：IPv4地址类型。<br>默认值：无 |
| MASKLENGTH | 路由掩码长度 | 可选必选说明：可选参数<br>参数含义：该参数用于表示路由的掩码长度。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～32。<br>默认值：无<br>配置原则：不可单独以掩码查询路由。掩码需要和前缀一起配置。 |

#### [输出结果说明](#ZH-CN_CONCEPT_0000001550121418)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 地址族 | 用来指定VPN实例的地址族信息。 |
| VPN实例名称 | 用于表示路由前缀所属VPN实例的名称。 |
| 路由前缀 | 用于表示路由前缀信息。 |
| 路由掩码长度 | 用于表示路由的掩码长度。 |
| 路由接口名字 | 用于表示路由的出接口名称。 |
| 下一跳VPN名字 | 用于表示路由下一跳所属VPN实例的名称。 |
| 路由下一跳 | 用于指定静态路由的下一跳IP地址。 |
| 路由优先级 | 用于指定路由的优先级。 |
| 路由Tag值 | 用于表示路由的tag属性值。 |
| BFD检测结果 | 该参数用于标示静态路由所使能的BFD的会话建立状态：<br>- Disable：未使能BFD会话。<br>- UP：BFD会话建立成功。<br>- Down：BFD会话建立失败。<br>- UNKNOWN：检测结果未知。<br>- UNKNOWN(AdminStatus=Down)：检测结果未知，管理状态为Down，可以下发路由表项。<br>- UNKNOWN(TimeOut)：检测结果未知，由于超时不可以下发路由表项。 |
| BFD类型 | 用于表示静态路由所使能的BFD类型。 |
| 路由状态 | 该参数用于标示静态路由的选路结果：<br>- Active Primary：活跃的主路由。<br>- Active Backup：活跃的备路由。<br>- Inactive Valid：参加选路但是没有优选的路由。<br>- Inactive Invalid：不能参加优选的路由。 |
| 路由迭代下一跳 | 用于表示静态路由的迭代下一跳。 |
| IID | 用于表示路由的下一跳分离关键字。 |
| 路由接口状态 | 用于表示路由的出接口状态。 |
| 路由迭代出接口 | 用于表示静态路由迭代出接口的名称。 |
| BFD会话名 | 用于表示为静态路由绑定的静态BFD的会话名。 |
| 路由开销值 | 用于表示路由的开销值。 |
| 路由本地地址 | 用于表示本地接口IP。 |
| 路由远程地址 | 用于表示P2P接口远端IP。 |
| 继承迭代路由cost | 用于表示静态路由是否继承迭代路由Cost。 |
| 隧道ID | 用于表示静态路由迭代隧道ID。 |
| 是否永久发布 | 用于表示静态路由是否永久发布。 |
| 标签 | 用于表示静态路由的标签。 |
| EFM接口名称 | 用于表示静态路由EFM的接口名称。 |
| EFM探测结果 | 用于表示静态路由EFM检测结果。 |

## [命令功能](#ZH-CN_MMLREF_0296805381)

**适用NF：AMF**

该命令用于查询AMF的SMF故障处理策略。

## [注意事项](#ZH-CN_MMLREF_0296805381)

无

#### [操作用户权限](#ZH-CN_MMLREF_0296805381)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0296805381)

无

## [使用实例](#ZH-CN_MMLREF_0296805381)

查询AMF的SMF故障处理策略，执行如下命令：

```
%%LST AMFSMFRESET:;%%
RETCODE = 0  操作成功

结果如下
--------
   SMF故障重选开关  =  打开
自定义扫描速率开关  =  打开
   扫描速率(个/秒)  =  2
  单位扫描时间(秒)  =  3
      单位扫描个数  =  4
  语音业务恢复开关  =  打开
       语音业务DNN  =  HUAWEI.COM

(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0296805381)

| 输出项名称 | 输出项解释 |
| --- | --- |
| SMF故障重选开关 | 该参数用于指定AMF和SMF的接口故障时，是否通过扫描任务对在该SMF上有语音业务的用户进行去注册，达到重选SMF恢复语音业务的效果。 |
| 自定义扫描速率开关 | 该参数表示扫描速率方式。<br>当取值为“ON”时，使用通过SCANNUM除以INTERVAL得到的自定义扫描速率；当取值为“OFF”时，使用RATE设置的扫描速率。 |
| 扫描速率(个/秒) | 该参数用于指定每个DS每秒扫描多少个用户，扫描到后对符合去注册条件的用户立即去注册。 |
| 单位扫描时间(秒) | 该参数用于表示SMF故障后AMF对用户扫描的时间间隔。 |
| 单位扫描个数 | 该参数用于表示每个DS每隔INTERVAL设置的时间扫描多少个用户，扫描到后对符合去注册条件的用户立即去注册。 |
| 语音业务恢复开关 | 该参数用于指定AMF感知到SMF故障时，在该SMF上有语音业务的用户进行数据业务请求时是否恢复语音业务。 |
| 语音业务DNN | 该参数用于指定语音业务DNN，SMF故障时针对该DNN的语音业务会话根据配置策略进行恢复。 |

## [操作场景](#ZH-CN_OPI_0249797715)

MME故障或者重启场景SGW-C保留会话，并在有下行数据和信令时触发恢复操作。

> **说明**
> 适用于SGW-C。

- 完成配置APN。
- 完成 [激活IMS功能](../../../WSFD-201103 IMS功能/激活IMS功能_91473678.md) 。

## [操作步骤](#ZH-CN_OPI_0249797715)

1. 进入 “MML命令行-UNC” 窗口。
2. 配置基于整机的防闪断定时器时长和保留承载时长。
  [**SET FASTRECOVERY**](../../../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/网络管理/业务快速恢复/全局参数/设置全局业务快速恢复配置（SET FASTRECOVERY）_31453525.md)
3. 基于整机配置SGW-C发送给MME的Echo Request消息中携带支持NTSR功能的指示。
  [**SET GTPCFUNCPARA**](../../../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/GTP-C接口配置管理/GTP-C功能参数/设置GTP-C功能参数（SET GTPCFUNCPARA）_09653666.md)
4. 配置MME POOL实例。
  [**ADD MMEPOOL**](../../../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/网络管理/业务快速恢复/MME Pool/增加MME POOL（ADD MMEPOOL）_31453510.md)
5. 配置与SGW-C对接的MME POOL。
  [**ADD MMEIPTOMMEPOOL**](../../../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/网络管理/业务快速恢复/MME IP/增加MME IP（ADD MMEIPTOMMEPOOL）_31453389.md)
  > **说明**
  > 如果是IPv6组网，需要配置IPv6地址。
6. 设置承载具备全局支持NTSR的能力。
  [**SET FASTRECOVERY**](../../../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/网络管理/业务快速恢复/全局参数/设置全局业务快速恢复配置（SET FASTRECOVERY）_31453525.md)
  > **说明**
  > 当NTSRSWITCH参数配置为开启的情况下可通过软参 [BIT769](../../../../../../../OM参考/软件参数/UNC软件参数/业务软件参数/软件参数（SMF_GGSN_SGW-C_PGW-C）/会话管理/BIT769 控制MME故障恢复时在DDN消息中是否携带TAI信元_10873890.md) 配置在MME故障场景发送的DDN消息中是否使用私有扩展信元携带用户的TAI。
7. 配置指定APN的NTSR功能。
  [**ADD APN**](../../../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/APN管理/APN/增加APN配置（ADD APN）_09653747.md)
  > **说明**
  > - 推荐普通语音APN和紧急语音APN下配置该步骤，数据APN下不推荐配置。
  > - SGW-C上面配置NTSR功能，MME上也需要配置NTSR功能。

## [任务示例](#ZH-CN_OPI_0249797715)

任务描述

使能MME故障后的VoLTE承载故障快速恢复功能。

脚本

// 进入 “MML命令行-UNC” 窗口。

//基于整机的防闪断定时器时长为120秒，保留承载时长为61分钟。

```
SET FASTRECOVERY:PRTFRQBRKTIME=120,HOLDONTIME=61;
```

//基于整机配置SGW-C发送给MME的Echo Request消息中携带支持NTSR功能的指示。

```
SET GTPCFUNCPARA:V2ECHOSW=ON, NTSRSWITCH=ON, REQPEERNTSR=ON;
```

//配置MME POOL实例。

```
ADD MMEPOOL:MMEPOOLNAME="mmepool",BACKUPSWITCH=ENABLE;
```

//配置与SGW-C对接的MME POOL。

```
ADD MMEIPTOMMEPOOL:MMEPOOLNAME="mmepool",IPVERSION=IPV4,IPV4ADDRESS="10.10.10.10",BACKUP=TRUE;
```

```
ADD MMEIPTOMMEPOOL:MMEPOOLNAME="mmepool",IPVERSION=IPV4,IPV4ADDRESS="10.10.10.11",BACKUP=TRUE;
```

//设置承载具备全局支持NTSR的能力。

```
SET FASTRECOVERY:NTSRSWITCH=ENABLE;
```

//设置指定APN下的NTSR功能。

```
ADD APN:APN="ims",NTSRSWITCH=ENABLE;
```

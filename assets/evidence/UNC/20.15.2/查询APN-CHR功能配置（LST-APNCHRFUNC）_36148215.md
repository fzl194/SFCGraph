## [命令功能](#ZH-CN_MMLREF_0000001136148215)

**适用NF：GGSN、SGW-C、PGW-C、SMF**

该命令用于查询基于APN的CHR功能配置。

## [注意事项](#ZH-CN_MMLREF_0000001136148215)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001136148215)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001136148215)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| APN | APN | 可选必选说明：可选参数<br>参数含义：该参数用于指定APN实例名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~63。只能由“-”、数字、大小写字母和“.”组成，不能以“.”开头且不能出现连续两个“.”。不支持空格及“_”、“#”、“$”、“&”、“%”、“^”、“（”、“）”、“，”、“/”、“;”、“:”、“””、“`”等特殊字符，不区分大小写。<br>默认值：无<br>配置原则：<br>该参数必须已经通过命令ADD APN配置。 |

## [使用实例](#ZH-CN_MMLREF_0000001136148215)

查询“APN”为“HUAWEI.COM”的CHR功能配置：

```
%%LST APNCHRFUNC: APN="HUAWEI.COM";%%
RETCODE = 0  操作成功

结果如下
--------
            APN  =  huawei.com
APN CHR上报开关  =  不使能
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001136148215)

| 输出项名称 | 输出项解释 |
| --- | --- |
| APN | 该参数用于指定APN实例名。 |
| CHR上报开关 | 该参数用于基于APN控制开启和关闭CHR上报功能。 |

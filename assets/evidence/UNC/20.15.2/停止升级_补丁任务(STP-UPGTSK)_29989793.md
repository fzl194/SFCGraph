## [命令功能](#ZH-CN_MMLREF_0000001729989793)

![](停止升级_补丁任务(STP UPGTSK)_29989793.assets/notice_3.0-zh-cn_2.png)

- “快速失败”参数为空时，执行该命令会中止当前正在正常执行的升级/补丁任务，并重启后台升级工具服务，请谨慎操作。
- “快速失败”参数非空时，执行该命令会打开/关闭原子快速失败能力。
- 该命令会中止当前正在执行的升级/补丁任务，升级工具服务会重启（快速失败参数取值为ON/OFF时不会重启），请谨慎操作。
- 升级/补丁过程中执行mml命令时，若mml命令执行依赖的微服务状态异常、节点复位等阶段可能会导致命令不生效或者失败，执行命令时请尽量避开这些阶段。

- “快速失败”参数为空时，该命令用于停止正在执行的升级/补丁任务，并且在升级界面弹窗提示，用户可在界面上根据提示选择“继续”或“回退”。
- “快速失败”参数非空时，该命令仅用于打开/关闭原子快速失败能力。

## [注意事项](#ZH-CN_MMLREF_0000001729989793)

无。

## [参数说明](#ZH-CN_MMLREF_0000001729989793)

| **参数标识** | **参数名称** | **参数说明** |
| --- | --- | --- |
| ATOMPAUSE | 快速失败 | 可选必选说明：必选参数。<br>参数含义：表示执行停止升级/补丁任务或者执行开启/关闭原子快速失败能力。<br>取值范围：<br>- NULL<br>- ON(开)<br>- OFF(关)<br>默认值：NULL。<br>配置原则：无。 |
| ATOMLIST | 原子列表 | 可选必选说明：该参数在<br>“快速失败”<br>为<br>“ON(开)”<br>时，为条件必选参数。<br>参数含义：表示手动输入需要快速失败的原子列表，当前输入仅支持数字/字母/":"/"\|"/" "/"-"/"_"。<br>取值范围：字符串长度范围0~4000。<br>默认值：无。<br>配置原则：部件1:原子1\|部件2:原子2\|...\|部件X:原子X。 |
| ADDMODE | 模式 | 可选必选说明：该参数在<br>“快速失败”<br>为<br>“ON(开)”<br>时，为条件必选参数。<br>参数含义：表示快速失败原子执行模式。<br>- CLEAR(清除)表示原子失败列表按照本次输入执行。<br>- ADD(追加)表示在已有原子失败列表基础上追加本次需要快速失败的原子列表。<br>取值范围：<br>- CLEAR(清除)。<br>- ADD(追加)。<br>默认值：CLEAR(清除)。<br>配置原则：无。 |

## [使用实例](#ZH-CN_MMLREF_0000001729989793)

1. 停止升级/补丁任务
  ```
  %%STP UPGTSK:;%%
  RETCODE = 0  操作成功
  ---    END
  ```
2. 使用原子快速失败能力
  ```
  %%STP UPGTSK: ATOMPAUSE=ON, ATOMLIST="部件名1:原子名1|部件名2:原子名2|部件名N:原子名N", ADDMODE=CLEAR;%%
  RETCODE = 0  操作成功
  ---    END
  ```
3. 关闭原子快速失败能力
  ```
  %%STP UPGTSK: ATOMPAUSE=OFF;%%
  RETCODE = 0  操作成功
  ---    END
  ```

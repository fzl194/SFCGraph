## [命令功能](#ZH-CN_MMLREF_0000001719125360)

该命令用于显示防呆开关状态。

## [注意事项](#ZH-CN_MMLREF_0000001719125360)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001719125360)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001719125360)

无

## [使用实例](#ZH-CN_MMLREF_0000001719125360)

假如操作员想了解防呆开关的当前状态，可以调用以下命令显示所有防呆开关的状态。

```
%%DSP FOOLPROOFSWITCH:;%%
RETCODE = 0  操作成功

结果如下
--------
名称  =  ONECLICKDEPLOY
状态  =  打开
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001719125360)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 名称 | 该参数用于表示防呆开关名称。 |
| 状态 | 该参数用于表示防呆开关状态。<br>取值说明：<br>- “ON（打开）”：防呆开关打开<br>- “OFF（关闭）”：防呆开关关闭 |

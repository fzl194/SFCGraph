#### [命令功能](#ZH-CN_MMLREF_0000001725231842)

**适用网元：MME**

该命令用于配置根据UE无线能力选择USN的参数，在指定用户范围内并且UE无线能力信元长度小于指定阈值的用户，在存储UE无线能力内存的使用率超过阈值后开始自动迁移用户到USN（具体指USN9810/CloudUSN）网元。

指定用户范围的用户的UE无线能力长度小于指定阈值的用户在 [**STR OFFLOADBYENODEB**](../../../网络管理/迁移控制/启动eNodeB迁移任务(STR OFFLOADBYENODEB)_26305902.md) 和 [**STR OFFLOADBYMME**](../../../网络管理/迁移控制/启动MME迁移任务（STR OFFLOADBYMME）_72345693.md) 迁移用户的过程中迁移用户到USN网元。

#### [注意事项](#ZH-CN_MMLREF_0000001725231842)

- 该命令执行后立即生效。
- 最大配置记录数是1024条。

#### [本地用户权限](#ZH-CN_MMLREF_0000001725231842)

manage-ug；system-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001725231842)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001725231842)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SUBRANGE | 用户范围 | 可选必选说明：必选参数<br>参数含义：该参数用于指定待进行网元重选的用户范围。<br>数据来源：整网规划<br>取值范围：<br>- ALL_USER（所有用户）：表示用户范围为系统内所有用户。<br>- HOME_USER（本网用户）：表示用户范围为本网用户。<br>- FOREIGN_USER（外网用户）：表示用户范围为外网用户。<br>- IMSI_PREFIX（指定IMSI前缀）：表示用户范围通过IMSI前缀指定。<br>- SPECIFIC_IMSI（指定IMSI）：表示用户范围通过IMSI指定。<br>默认值：无<br>配置原则：USN重选策略匹配优先级高到低为：<br>“SPECIFIC_IMSI（指定IMSI）”<br>，<br>“IMSI_PREFIX（指定IMSI前缀）”<br>，<br>“FOREIGN_USER(外网用户)”<br>或<br>“HOME_USER(本网用户)”<br>，<br>“ALL_USER(所有用户)”<br>。 系统优先匹配高优先级的配置记录，如果匹配不到，再匹配低优先级的配置记录。 |
| NOID | 运营商标识 | 可选必选说明：条件必选参数<br>参数含义：该参数用于指定运营商标识。<br>前提条件：该参数在<br>“用户范围”<br>参数配置为<br>“HOME_USER(本网用户)”<br>或<br>“FOREIGN_USER(外网用户)”<br>后生效。<br>数据来源：整网规划<br>取值范围：0～64，128～254<br>默认值：无<br>配置原则：<br>- 若该参数需要配置为0或128～254之间的值时，须先在[**ADD MNO**](../../../网络管理/归属网络运营商管理/MNO管理/MNO配置表/增加MNO配置信息(ADD MNO)_72345671.md)中配置取值相同的“MNOID”参数。<br>- 若该参数需要配置为1～64之间的值时，须先在中配置取值相同的“MVNOID”参数。 |
| IMSIPRE | IMSI前缀 | 可选必选说明：条件必选参数<br>参数含义：该参数用于指定IMSI前缀以区分不同的用户群。<br>前提条件：该参数在<br>“用户范围”<br>参数配置为<br>“指定IMSI前缀”<br>后生效。<br>数据来源：整网规划<br>取值范围：5~14位十进制数字字符串<br>默认值：无 |
| IMSI | IMSI | 可选必选说明：条件必选参数<br>参数含义：该参数用于指定IMSI以区分不同的用户群。<br>前提条件：该参数在<br>“用户范围”<br>参数配置为<br>“指定IMSI”<br>后生效。<br>数据来源：整网规划<br>取值范围：15位十进制数字字符串<br>默认值：无 |
| USNSELSW | USN重选开关 | 可选必选说明：必选参数<br>参数含义：该参数用于指定是否开启USN重选功能。<br>数据来源：整网规划<br>取值范围：<br>- OFF：关闭<br>- ON：开启<br>默认值：无 |
| UERCMEM | 存储内存占用率阈值 | 可选必选说明：条件必选参数<br>参数含义：该参数用于指定根据UE无线能力重选USN的内存使用率阈值，UE无线能力长度小于<br>“长度阈值(KB)”<br>的用户在存储UE无线能力内存使用率超过该阈值后开始自动迁移到USN网元。<br>前提条件：该参数在<br>“USN重选开关”<br>参数配置为<br>“ON(开启)”<br>后生效。<br>数据来源：整网规划<br>取值范围：1~100<br>默认值：无 |
| UERCLEN | 长度阈值(KB) | 可选必选说明：条件必选参数<br>参数含义：该参数用于指定根据UE无线能力重选USN的UE无线能力长度， UE无线能力长度小于<br>“长度阈值（KB）”<br>的用户可以迁移到USN网元。<br>前提条件：该参数在<br>“USN重选开关”<br>参数配置为<br>“ON(开启)”<br>后生效。<br>数据来源：整网规划<br>取值范围：1~63<br>默认值：无 |
| TMMEGPIDX | 目标MME组索引 | 可选必选说明：条件必选参数<br>参数含义：该参数用于指定根据UE无线能力进行USN选择时的目标MME群组的标识。<br>前提条件：该参数在<br>“USN重选开关”<br>参数配置为<br>“ON(开启)”<br>后生效。<br>数据来源：整网规划<br>取值范围：0~63的整数类型。<br>默认值：无<br>配置原则：该参数必须已经在<br>[**ADD MMEGP**](../../MME群组管理/MME群组配置/增加MME群组(ADD MMEGP)_72225301.md)<br>配置。 |
| USRTHD | 迁移用户数百分比 | 可选必选说明：条件可选参数<br>参数含义：该参数用于指定SPP进程上实时用户数占SPP进程用户规格的百分比阈值，UNC上的用户数百分比低于本阈值后则停止自动迁移，防止USN9810/CloudUSN上用户过载。UNC上的用户数百分比高于本阈值则可以迁移。<br>前提条件：该参数在<br>“USN重选开关”<br>参数配置为<br>“ON(开启)”<br>后生效。<br>数据来源：整网规划<br>取值范围：0~100，0表示所有用户都可以迁移，100表示所有用户都不迁移。<br>默认值：0<br>配置原则：该参数需要根据MME POOL中的各个MME的规格配置，并且需要给USN9810/CloudUSN升级预留冗余资源。 |

#### [使用实例](#ZH-CN_MMLREF_0000001725231842)

增加一条 “用户范围” 为 “所有用户” ， “USN重选开关” 为 “ON（开启）” ， “存储内存占用率阈值” 为 “10” ， “长度阈值(KB)” 为 “3” ， “目标MME组索引” 为 “0” 的记录；

```
ADD UERCAPUSNSEL: SUBRANGE=ALL_USER, USNSELSW=ON, UERCMEM=10, UERCLEN=3, TMMEGPIDX=0;
```

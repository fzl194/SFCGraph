# 清除计费配置冲突（CLR CHARGECONFLICT）

#### [命令功能](#ZH-CN_CONCEPT_0209897022)

**适用NF：PGW-C、SMF**

该命令用来清除当前系统中所有保存的计费业务冲突配置的信息并恢复告警。

#### [注意事项](#ZH-CN_CONCEPT_0209897022)

- 该命令执行后立即生效。
- 当系统中有计费业务冲突的配置时，系统提示告警（ALM-81033 配置与业务冲突）信息，执行该命令后，系统的冲突计费配置信息立即清除，告警恢复。

#### [操作用户权限](#ZH-CN_CONCEPT_0209897022)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0209897022)

无。

#### [使用实例](#ZH-CN_CONCEPT_0209897022)

清除计费配置冲突：

```
CLR CHARGECONFLICT:;
```

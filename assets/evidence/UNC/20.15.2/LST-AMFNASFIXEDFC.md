# 查询指定消息类型的AMF NAS固定速率流控信息（LST AMFNASFIXEDFC）

## [命令功能](#ZH-CN_MMLREF_0000001106990498)

**适用NF：AMF**

该命令用于查询指定消息类型的AMF NAS固定速率流控信息。

## [注意事项](#ZH-CN_MMLREF_0000001106990498)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001106990498)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001106990498)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| MSGTYPE | 流控消息类型 | 可选必选说明：可选参数<br>参数含义：该参数用于设置AMF被流控的NAS消息类型。<br>数据来源：全网规划<br>取值范围：<br>- PDUSESSIONEST（PDU Session Establishment Request）<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001106990498)

查询所有流控消息类型流控信息，执行如下命令：

```
%%LST AMFNASFIXEDFC:;%%
RETCODE = 0  操作成功

结果如下
------------------------
             流控消息类型  =  PDUSESSIONEST
         固定速率流控开关  =  打开
      流控速率门限(个/秒)  =  1000
         流控阈值作用范围  =  单usn-pod
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001106990498)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 流控消息类型 | 该参数用于设置AMF被流控的NAS消息类型。 |
| 固定速率流控开关 | 该参数用于控制AMF指定NAS消息类型固定速率流控功能。 |
| 流控速率门限(个/秒) | 该参数用于设置AMF指定NAS消息的接收流控速率上限。 |
| 流控阈值作用范围 | 该参数用于设置AMF NAS消息流控阈值的作用范围。 |

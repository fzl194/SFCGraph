#### [命令功能](#ZH-CN_CONCEPT_0000001549961790)

该命令用于查询OSPFv3过滤策略配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001549961790)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549961790)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549961790)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | OSPFv3进程号 | 可选必选说明：可选参数<br>参数含义：OSPFv3进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |
| FILTERTYPE | 过滤方向 | 可选必选说明：可选参数<br>参数含义：该参数用于指定过滤方向是import还是export。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- import：引入方向的过滤。<br>- export：发布方向的过滤。<br>默认值：无 |
| TYPE | 过滤规则类型 | 可选必选说明：可选参数<br>参数含义：该参数用于指定过滤类型。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- acl_name：ACL名称。<br>- ip_prefix：IPv6前缀过滤策略名称。<br>- route_policy：路由策略名。<br>- acl_num：ACL号。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549961790)

查询OSPFv3过滤策略配置：

```
LST OSPFV3FILTERPOLICY:;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001549961790)

| 输出项名称 | 输出项解释 |
| --- | --- |
| OSPFv3进程号 | 该参数指定用户创建的OSPFv3进程号。 |
| 拓扑标识 | 该参数指定拓扑标识。 |
| 过滤方向 | 该参数指定过滤方向是import还是export。import表示引入方向的过滤，export表示发布方向的过滤。 |
| 过滤规则类型 | 该参数指定过滤规则类型，acl_name-ACL名称；ip_prefix-IPv6前缀过滤策略名称；route_policy-路由策略名；acl_num-ACL号。 |
| ACL名称或ACL号 | 该参数指定ACL规则组名称或是编号，当过滤规则类型为acl_name或acl_num时生效。 |
| IPv6前缀过滤策略名称 | 该参数指定IPv6前缀过滤策略名称，当过滤规则类型为ip_prefix时生效。 |
| 路由策略名称 | 该参数指定路由策略名称，当过滤规则类型为route_policy时生效。 |
| 协议号 | 该参数指定对某一种协议的路由进行过滤，当过滤方向为export时生效。 |
| 协议进程号 | 该参数指定对某一进城的路由进行过滤，当协议号为OSPFv3时生效，当协议号为其他协议时，由于其他协议不区分进程号，此字段显示为1，并且无意义。 |

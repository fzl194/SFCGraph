#### [命令功能](#ZH-CN_CONCEPT_0209896773)

**适用NF：PGW-C、SMF**

该命令用于设置AAA计费不使能时是否将承载的上下行流量进行备份处理。

#### [注意事项](#ZH-CN_CONCEPT_0209896773)

- 该命令执行后立即生效。
- 该命令最大记录数为1。
- 当前版本不支持此命令。
- 该命令存在系统初始记录，参数的初始设置值如下表：

| 参数标识 | TRAFFICBACKUPSW | BACKUPINTERVAL |
| --- | --- | --- |
| 初始值 | DISABLE | 30 |

#### [操作用户权限](#ZH-CN_CONCEPT_0209896773)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0209896773)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| TRAFFICBACKUPSW | 流量备份开关 | 可选必选说明：可选参数<br>参数含义：该参数用于标识在AAA计费不使能时是否将承载的上下行流量进行备份处理。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- DISABLE：否。<br>- ENABLE：是。<br>默认值：无<br>配置原则：无 |
| BACKUPINTERVAL | 备份间隔 (分钟) | 可选必选说明：条件可选参数<br>前提条件：该参数在“TRAFFICBACKUPSW”配置为“ENABLE”时为可选参数。<br>参数含义：该参数用于指定上下行流量备份时的备份间隔。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为5～60，单位是分钟。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0209896773)

设置流量备份配置：

```
SET TRAFFICBACKUP: TRAFFICBACKUPSW=ENABLE, BACKUPINTERVAL=30;
```

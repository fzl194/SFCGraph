#### [命令功能](#ZH-CN_CONCEPT_0192520004)

![](设置PBUF重用检测开关（SET MSSPBUFREUSE）_92520004.assets/notice_3.0-zh-cn_2.png)

本命令用于使能PBUF重用检测，开启后会降低性能且在24小时之后会自动去使能，关闭后会恢复性能。此命令仅在SET PBUFREUSE5GC未设置时生效，可以通过LST PBUFREUSE5GC查询是否有设置。

此命令用于设置MSS的PBUF重用检测开关，重用检测功能用来检测多个进程是否同时在使用同一片PBUF。

用户打开开关后，系统收集运行信息，导致转发面性能下降。打开开关后24小时后会自动关闭。

#### [注意事项](#ZH-CN_CONCEPT_0192520004)

- 该命令执行后立即生效。
- 本命令用于使能PBUF重用检测，开启后会降低性能且在24小时之后会自动去使能，关闭后会恢复性能。

#### [操作用户权限](#ZH-CN_CONCEPT_0192520004)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [使用实例](#ZH-CN_CONCEPT_0192520004)

打开类型为aa的微服务bb内PBUF重用检测开关：

```
SET MSSPBUFREUSE:CELLTYPE="aa",CELLINSTANCE="bb",POOLNAME="PAE",REUSESWITCH=TRUE;
```

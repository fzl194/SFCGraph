## [命令功能](#ZH-CN_MMLREF_0000001298868305)

**适用NF：SMF**

该命令用于查询5G LAN特性相关的功能控制设置参数。

## [注意事项](#ZH-CN_MMLREF_0000001298868305)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001298868305)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001298868305)

无

## [使用实例](#ZH-CN_MMLREF_0000001298868305)

查询所有NGLANFUNC记录：

```
%%LST NGLANFUNC:;%%
RETCODE = 0  操作成功

结果如下
--------
      是否向UDM请求获取5G LAN组ID  =  支持
                 是否支持N19通信  =  不支持
         以太会话锚点重定位功能开关  =  不支持
              是否创建组播PDR规则  =  不支持
是否发起SMF内5G Lan组内会话数量核查  =  支持
         是否发起向UPF的组会话核查  =  支持
优选5G Lan组内已有的主锚点UPF的开关  =  不支持
             是否支持N6侧上行广播  =  不支持
             是否支持报文增殖消除  =  不支持
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001298868305)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 是否向UDM请求获取5G LAN组ID | 该参数用于控制5G LAN会话是否向UDM请求获取组ID。 |
| 是否支持N19通信 | 该参数用于控制5G LAN会话是否支持通过UPF间的N19接口进行通信，用于5G LAN会话的跨园区互通场景。 |
| 以太会话锚点重定位功能开关 | 该参数用于控制SMF是否支持以太会话的锚点重定位即主锚点重选功能。 |
| 是否创建组播PDR规则 | 该参数用于控制是否创建组播PDR规则，以支持5G LAN组通过N6接口或N19接口进行组播通信。 |
| 是否发起SMF内5G Lan组内会话数量核查 | 该参数用于控制SMF内不同模块间5G LAN组会话上下文数量一致性核查功能是否启用。 |
| 是否发起向UPF的组会话核查 | 该参数用于控制SMF是否向UPF发起组会话的上下文核查。 |
| 优选5G Lan组内已有的主锚点UPF的开关 | 该参数用于控制是否支持优选5G LAN组内已建立组会话的锚点UPF。例如在UPF容灾场景下，主用UPF发生故障时，组会话迁移至备用UPF，当主用UPF故障恢复后，如果仍希望SMF选择备用UPF建立会话，保证企业的所有用户始终在同一个UPF上获取5G LAN服务，此时将参数设置为“Support”。 |
| 是否支持N6侧上行广播 | 该参数用于控制是否为5G LAN PDU会话创建N6上行广播规则，以支持UE与存量网络通过N6接口进行上行广播。 |
| 是否支持报文增殖消除 | 该参数用于控制标准5G LAN是否支持报文增殖消除方案，避免报文环回。 |

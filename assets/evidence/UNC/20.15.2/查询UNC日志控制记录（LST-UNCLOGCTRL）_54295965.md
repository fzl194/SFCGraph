## [命令功能](#ZH-CN_MMLREF_0000001954295965)

**适用NF：SMF、AMF、NRF、NSSF、SMSF、NCG**

该命令用于查询UNC日志的控制记录。

## [注意事项](#ZH-CN_MMLREF_0000001954295965)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001954295965)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001954295965)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| INDEX | 索引 | 可选必选说明：可选参数<br>参数含义：记录索引。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是1~255。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001954295965)

如下命令用于查询记录

```
%%LST UNCLOGCTRL: INDEX=1;%%
RETCODE = 0  操作成功

结果如下
--------
                索引  =  1
              CS名称  =  NULL
        文件或路径名  =  NULL
            代码行号  =  0
              函数名  =  NULL
              关键字  =  NULL
      是否打印调用栈  =  否
          白名单模式  =  否
设置匹配的日志的级别  =  INVALID
                描述  =  NULL
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001954295965)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 索引 | 记录索引。 |
| CS名称 | 创建logger时的CS名称。 |
| 文件或路径名 | 文件名或路径。 |
| 代码行号 | 代码行号。 |
| 函数名 | 函数名。 |
| 关键字 | 日志关键字，可以有多个，以空格分隔。 |
| 是否打印调用栈 | 是否打印调用栈。 |
| 白名单模式 | 是否开启白名单模式。 |
| 设置匹配的日志的级别 | 日志级别。 |
| 描述 | 记录描述。 |

# 删除故障隔离接口（RMV DRSEPINTERFACE）

- [注意事项](#ZH-CN_MMLREF_0000001535390226__1.3.2)

## [命令功能](#ZH-CN_MMLREF_0000001535390226)

该命令用于在免交换组网下组成热备容灾关系的网元间DCI通道变化时，删除需要关闭/开启的逻辑接口。

## [注意事项](#ZH-CN_MMLREF_0000001535390226)

- 该命令执行后立即生效。

- 该命令只用于在UEG-M/UEG网元采用主备（热备）容灾模式下执行。

#### [操作用户权限](#ZH-CN_MMLREF_0000001535390226)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001535390226)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| DRGROUPID | 容灾组标识 | 可选必选说明：必选参数<br>参数含义：该参数用于配置容灾组标识。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是1~8。<br>默认值：无<br>配置原则：无 |
| IFNAME | 接口名 | 可选必选说明：必选参数<br>参数含义：该参数用于配置接口名称。可通过<br>[**LST DRSEPINTERFACE**](查询快速隔离接口（LST DRSEPINTERFACE）_86255389.md)<br>命令查询获取。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~63。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001535390226)

删除故障隔离接口。

```
RMV DRSEPINTERFACE: DRGROUPID=1, IFNAME="itf1"; 
```

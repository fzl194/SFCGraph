## [命令功能](#ZH-CN_MMLREF_0000001262337370)

**适用NF：AMF、SMF、SMSF、NCG、NSSF**

该命令用于查询从NRF中获得的远端NF信息缓存数据规格。

## [注意事项](#ZH-CN_MMLREF_0000001262337370)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001262337370)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001262337370)

无

## [使用实例](#ZH-CN_MMLREF_0000001262337370)

运营商A需要查询从NRF中获得的远端NF信息缓存数据规格：

```
%%LST CACHESPECS:;%%
RETCODE = 0  操作成功

结果如下
------------------------
           网元缓存规格  =  4000                    
     SUPI及GPSI缓存规格  =  4000000
       TAIRANGE缓存规格  =  1500000
            DNN缓存规格  =  3000000
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001262337370)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 网元缓存规格 | 该参数用于指定网元缓存规格。 |
| SUPI及GPSI缓存规格 | 该参数用于指定SUPI及GPSI缓存规格。 |
| TAIRANGE缓存规格 | 该参数用于指定TAIRANGE缓存规格。 |
| DNN缓存规格 | 该参数用于指定DNN缓存规格。 |

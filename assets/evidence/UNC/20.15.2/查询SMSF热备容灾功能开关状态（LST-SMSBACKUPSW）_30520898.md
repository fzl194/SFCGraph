## [命令功能](#ZH-CN_MMLREF_0000001530520898)

**适用NF：SMSF**

该命令用于查询SMSF热备容灾功能开关状态。

## [注意事项](#ZH-CN_MMLREF_0000001530520898)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001530520898)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001530520898)

无

## [使用实例](#ZH-CN_MMLREF_0000001530520898)

运营商希望查询SMSF热备容灾功能开关状态时，执行如下命令：

```
LST SMSBACKUPSW:;
%%LST SMSBACKUPSW:;%%
RETCODE = 0  操作成功

结果如下：
--------------
 SMSF热备容灾功能开关  =  打开	

(结果个数 = 1)
```

## [输出结果说明](#ZH-CN_MMLREF_0000001530520898)

| 输出项名称 | 输出项解释 |
| --- | --- |
| SMSF热备容灾功能开关 | 该参数用于表示SMSF热备容灾功能开关。 |

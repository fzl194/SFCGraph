- [参数说明](#ZH-CN_MMLREF_0209652534__1.3.4)

## [命令功能](#ZH-CN_MMLREF_0209652534)

**适用NF：AMF、SMF、NSSF、SMSF、NCG、PGW-C、SGW-C**

该命令用于查询本地配置的对端NF实例的切片信息。

## [注意事项](#ZH-CN_MMLREF_0209652534)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209652534)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652534)

无

## [使用实例](#ZH-CN_MMLREF_0209652534)

查询本地配置的对端NF实例支持的切片信息。

```
%%LST PNFNS:;%%
RETCODE = 0 操作成功

结果如下
------------------------
         索引 = 1
   NF实例标识 = udm_instance_0
切片/服务类型 = 1
   切片分类器 = 010101
     切片描述 = NULL
（结果个数 = 1）

----    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209652534)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 索引 | 该参数用于指定切片的索引。 |
| NF实例标识 | 该参数用于指定对端NF实例标识。 |
| 切片/服务类型 | 该参数用于指定对端NF的切片/服务类型。 |
| 切片分类器 | 该参数用于指定对端NF的切片区分码。如果切片需要进一步细化子切片可配置该参数。 |
| 切片描述 | 该参数用于指定切片描述信息。 |

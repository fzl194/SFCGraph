## [操作场景](#ZH-CN_OPI_0184683758)

本操作指导介绍在运行网络中激活IM类业务无线资源管控功能的操作过程。

IM （Instant Messaging）类业务 无线资源 管控是指在无线资源有限的情况下，网络侧识别出IM类业务（例如QQ），并通过GTP-U（GPRS Tunneling Protocol-User plane）和 BSSGP （Base Station Subsystem GPRS Protocol）协议头的扩展字段将业务参数传递给无线侧，进而使无线侧对IM类业务进行管理，实现无线资源优化的功能。

> **说明**
> 适用于 SGSN 。

## [操作步骤](#ZH-CN_OPI_0184683758)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开 SGSN 功能控制开关。
  [**SET GTPUCTRL**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/数据转发管理/GTP-U/GTP-U参数管理/设置用户面控制参数(SET GTPUCTRL)_26305648.md)
  > **说明**
  > 参数 “SPTSPCSRVEXT” （2G支持特殊业务类型扩展头）固定选择 “SUPPORT” （支持）。
3. 打开PCU功能控制开关。
  [**MOD NSE**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/Gb接口管理/信令实体管理/修改信令实体(MOD NSE)_26305838.md)
  > **说明**
  > 参数 “SPECSRV” （是否支持特殊业务类型）固定选择 “YES” （是）。
4. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)

## [任务示例](#ZH-CN_OPI_0184683758)

任务描述

在2G网络中，SGSN支持将IM类业务信息传递给无线侧。设置SGSN支持特殊业务类型扩展头，NSE标识为1的信令实体支持特殊业务类型。

脚本

//打开功能开关。

```
SET GTPUCTRL: SPTSPCSRVEXT=SUPPORT;
MOD NSE: NSEI=1, SPECSRV=YES;
```

//打开本特性的License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2RRMIM01", SWITCH=ENABLE;
```

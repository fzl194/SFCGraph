## [命令功能](#ZH-CN_MMLREF_0209652273)

**适用NF：SMF**

该参数用于查看MEP_SMF联动功能开关。

## [注意事项](#ZH-CN_MMLREF_0209652273)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209652273)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652273)

无

## [使用实例](#ZH-CN_MMLREF_0209652273)

该命令用来查看当前MEP_SMF联动控制开关。 LST MEPINTERACT:;

```
%%LST MEPINTERACT:;%%
RETCODE = 0  操作成功

结果如下
------------------------
功能开关  =  使能
(结果个数 = 1)

----    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209652273)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 功能开关 | 该参数用于设置SMF是否与MEP联动。 |

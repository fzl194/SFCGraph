#### [命令功能](#ZH-CN_CONCEPT_0000002229958289)

**适用网元：MME**

该命令用于查询S1 TAI对象寻呼流控状态信息。

#### [注意事项](#ZH-CN_CONCEPT_0000002229958289)

无

#### [本地用户权限](#ZH-CN_CONCEPT_0000002229958289)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_CONCEPT_0000002229958289)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000002229958289)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000002229958289)

查询系统中的所有S1 TAI对象寻呼流控状态，执行如下命令：

```
%%DSP S1TAIDETECTINFO:;%%
RETCODE = 0  操作成功。

输出结果如下
-----------------------
            任务ID =  monitor_task_5 
     检测任务状态  =  正常
    TAI组对象数量  =  0 
异常TAI组对象数量  =  0
(结果个数 = 1)

---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000002229958289)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 任务ID | 该参数用于显示检测任务ID。 |
| 检测任务状态 | 该参数用于标识检测任务状态。<br>取值说明：<br>- “NORMAL（正常）”：TAI组对象监控任务状态正常<br>- “ABNORMAL（异常）”：TAI组对象监控任务状态异常 |
| TAI组对象数量 | 该参数用于显示当前注册的待检测TAI对象总数。 |
| 异常TAI组对象数量 | 该参数用于显示异常TAI组对象数量。 |

# 删除AMF重选控制策略（RMV AMFRESELPLCY）

## [命令功能](#ZH-CN_MMLREF_0000001898493196)

**适用NF：AMF**

该命令用于删除AMF重选功能控制策略。

## [注意事项](#ZH-CN_MMLREF_0000001898493196)

该命令执行后只对新激活用户生效。

#### [操作用户权限](#ZH-CN_MMLREF_0000001898493196)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001898493196)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SUBGRPID | 用户群组标识 | 可选必选说明：必选参数<br>参数含义：该参数用于指定应用AMF重选功能的用户群组。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是0~4294967294。该用户群组标识通过ADD NGUSRGRP进行添加。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001898493196)

删除用户群组1的AMF重选控制策略记录，执行命令如下：

```
RMV AMFRESELPLCY: SUBGRPID=1;
```

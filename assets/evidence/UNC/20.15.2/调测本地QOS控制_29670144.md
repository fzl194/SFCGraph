# 调测 本地QOS控制

- [操作场景](#ZH-CN_OPI_0229670144__1.3.1)
- [必备事项](#ZH-CN_OPI_0229670144__1.3.2)
- [操作步骤](#ZH-CN_OPI_0229670144__1.3.3)

## [操作场景](#ZH-CN_OPI_0229670144)

当运营商需要控制用户接入QoS时，需对 UNC 的接入QoS控制功能进行调测，确保本功能可以正常使用。

> **说明**
> 适用于 GGSN、 SGW-C、PGW-C、SMF。

## [必备事项](#ZH-CN_OPI_0229670144)

前提条件

- 请仔细阅读 [WSFD-109203 本地QOS控制特性概述](特性概述_28258571.md) 。
- 完成[激活本地QOS控制（适用于GGSN）](激活本地QOS控制（适用于GGSN）_28258572.md)。
- 完成[激活本地QOS控制（适用于SGW-C/PGW-C）](激活本地QOS控制（适用于SGW-C_PGW-C）_28258573.md)。
- 完成[激活本地QOS控制（SMF）](激活本地QOS控制（SMF）_77795473.md)。
- 完成配置普通APN [**ADD APN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/APN管理/APN/增加APN配置（ADD APN）_09653747.md) 。

数据

| 类别 | 参数名称 | 取值样例 | 获取方法 |
| --- | --- | --- | --- |
| 用户信息查询 | 国际移动用户识别码（IMSI） | 123000123456789 | 测试终端自带 |
| 测试终端使用的APN | APN名称（APN） | apn1 | 从已配置数据中获取 |
| 承载上下文QoS映射 | 标准QCI（QCI） | 3 | 从已配置数据中获取 |
| 承载上下文QoS映射 | DSCP（DSCP） | EF | 从已配置数据中获取 |

工具

- 测试终端
- 接入侧/PDN侧镜像接口已安装第三方抓包工具

## [操作步骤](#ZH-CN_OPI_0229670144)

1. 测试终端发起接入网络请求。
2. 进入 “MML命令行-UNC” 窗口。
3. 通过测试终端的IMSI查看测试终端是否成功激活。
  [**DSP SMCTX**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/系统管理/用户数据库管理/显示承载上下文(DSP SMCTX)_72226033.md)
  ```
  %%DSP SMCTX: QUERYOPT=BYIMSI, IMSI="123000123456789", IDTYPE=BYCTXID;%% 
  RETCODE = 0  操作成功。
  
  结果如下 -------------------------
                            .....
      实际使用的APN网络标识  =  apn1
          实际使用的PDN类型  =  IPv4
         实际使用的IPv4地址  =  10.141.149.100
                            .....
  (结果个数 = 1)

  ---    END 
  ```
    - 如果测试终端激活成功，且使用的“APN name”与规划值一致，则执行下一步。
    - 如果测试终端激活成功，但使用“APN name”与规划值不一致，或测试终端激活失败，请执行[步骤 5](#ZH-CN_OPI_0229670144__step1822204765920)。
4. 进入 “MML命令行-UNC” 窗口。 执行 [**LST LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/查询License配置项开关（LST LICENSESWITCH）_09651570.md) ，查询License中是否允许使用本地QoS控制。
  ```
  %%LST LICENSESWITCH: LICITEM="LKV3W9LQOS12";%% 
  RETCODE = 0  操作成功
  
  操作结果如下 
  ------------
     License项  =  LKV3W9LQOS12
   License名称  =  Local QoS Control
          开关  =  开
   (结果个数 = 1)
  
  ---    END
  ```
    - 如果License的“开关”为“不使能”，则设备不支持本地QoS控制功能，请重新申请License后继续调测本功能。
    - 如果License的“开关”为“使能”，设备支持本地QoS控制功能。
      > **说明**
      > 当 UNC 为SMF时，无License控制，不需要完成加载。
5. 收集并保存上述所有查询信息，并联系华为技术支持解决。
    - 执行[**EXP MML**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/配置管理/配置导出管理/导出MML文件（EXP MML）_47200033.md)命令将当前配置数据导出为MML脚本文件并保存。
    - 收集并保存上述所有查询信息。
    - 收集归纳所有信息并联系华为技术支持解决。

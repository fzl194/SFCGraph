## [命令功能](#ZH-CN_MMLREF_0000001823782766)

**适用NF：SGW-C**

该命令用以查询配置的SGW-C Home PLMN。

## [注意事项](#ZH-CN_MMLREF_0000001823782766)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001823782766)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001823782766)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| NOID | 运营商标识 | 可选必选说明：可选参数<br>参数含义：该参数用于在UNC上唯一标识某个运营商。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0。<br>默认值：无<br>配置原则：无 |
| MCC | 移动国家码 | 可选必选说明：可选参数<br>参数含义：该参数用于表示组成SGW-C上Home PLMN的移动国家码信息。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度是3。只允许输入十进制数字（0-9）。<br>默认值：无<br>配置原则：无 |
| MNC | 移动网号 | 可选必选说明：可选参数<br>参数含义：该参数用于表示组成SGW-C上Home PLMN的移动网号信息。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是2~3。只允许输入十进制数字（0-9）。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001823782766)

- 查询指定运营商的SGW-C Home PLMN列表，执行如下命令：
  ```
  %%LST NGSGWPLMN: NOID=0, MCC="123", MNC="210";%%
  RETCODE = 0  操作成功

  结果如下
  --------
  运营商标识  =  0
  移动国家码  =  123
    移动网号  =  210
    描述信息  =  NULL
  (结果个数 = 1)

  ---    END
  ```
- 查询系统配置的运营商SGW-C Home PLMN列表，执行如下命令：
  ```
  %%LST NGSGWPLMN:;%%
  RETCODE = 0  操作成功

  结果如下
  ------------------------
  运营商标识  移动国家码  移动网号  描述信息  

  0           123         03        NULL
  0           123         210       NULL
  (结果个数 = 2)

  ---    END
  ```

## [输出结果说明](#ZH-CN_MMLREF_0000001823782766)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 运营商标识 | 该参数用于在UNC上唯一标识某个运营商。 |
| 移动国家码 | 该参数用于表示组成SGW-C上Home PLMN的移动国家码信息。 |
| 移动网号 | 该参数用于表示组成SGW-C上Home PLMN的移动网号信息。 |
| 描述信息 | 该参数是对配置的SGW-C上Home PLMN的描述信息，在运维过程中起助记作用。 |

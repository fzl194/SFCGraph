## [操作场景](#ZH-CN_OPI_0193400212)

融合计费模板CCT（Converged Charging Template）中包含本地配置的融合计费参数，如计费的时间/流量门限等。UNC上默认配置一个名为global的模板，即全局通用参数模板CCT-global，用于配置整机通用的融合计费参数。除了CCT-global外，还可以配置其他普通CCT模板，用于配置以下更细粒度的融合计费参数。

- UserProfile粒度：根据UserProfile配置CCT，将CCT绑定到特性UserProfile下。
- DNN粒度：根据DNN配置CCT，将CCT绑定到特定DNN下。
- CC粒度：根据用户的计费属性Charging Characteristics配置CCT，将CCT绑定到特定CC下。

上述几种粒度范围的优先级从高到低依次是： UserProfile下的配置 > DNN下的配置 > 计费属性Charging Characteristics下的配置 > 整机通用参数。如果高级别的参数没有配置，则依次向下继承低一级别的参数取值。

一旦确定了整个系统的融合计费参数配置粒度，则原则上所有融合计费参数的配置都必须统一使用该粒度。

> **说明**
> 适用于SMF。

## [操作步骤](#ZH-CN_OPI_0193400212)

- 配置全局通用CCT模板。
    1. 修改系统默认配置的全局通用参数CCT模板，该模板名称为 “global” 。
      **MOD CCT**
- 配置CC粒度的CCT。
    1. 配置CCT模板。
      **ADD CCT**
    2. 配置计费属性。
      请参见 [配置计费属性CC](配置计费属性CC_90776700.md) 。
    3. 配置CCT模板绑定CC。
      **ADD SELECTCCTBYCC**
- 配置DNN粒度的CCT。
    1. 配置CCT模板。
      **ADD CCT**
    2. 配置DNN。
      **ADD APN**
    3. 配置CCT模板绑定DNN。
      **SET APNCHARGECTRL**
- 配置UserProfile粒度的CCT。
    1. 配置CCT模板。
      **ADD CCT**
    2. 配置UserProfile，将CCT绑定到UserProfile上。
          a. 配置UserProfile。
            **ADD USERPROFILE**
          b. 将CCT绑定到UserProfile上。
            **SET USRPROFCHARGE**
    3. 配置UserProfile组及DNN实例。
          a. 配置UserProfile组，将UserProfile绑定到UserProfile组。
            **ADD USRPROFGROUP**
            **ADD UPBINDUPG**
          b. 配置DNN实例，将UserProifle组绑定到DNN实例。
            **ADD APN**
            **ADD APNUSRPROFG**
            每个DNN实例只能绑定一个UserProfile组。如UserProfile组和DNN实例已配置且已绑定，本步骤只需执行 **ADD UPBINDUPG** 命令，将UserProfile绑定到UserProfile组即可。

## [任务示例](#ZH-CN_OPI_0193400212)

任务描述

任务一：修改全局通用CCT模板额QHT值为 “5” 。

任务二：配置CC粒度的CCT，绑定值为 “0x0800” 的CC上。

任务三：配置DNN粒度的CCT，绑定在名为 “apn-test” 的DNN实例上。

任务四：配置UserProfile粒度额CCT，绑定在名为up_test的UserProfile上。

脚本

//任务一：修改全局通用CCT模板额QHT值为 “5” 。

```
MOD CCT: CCTMPLTNAME="global", QHT=5;
```

//任务二：配置CC粒度的CCT，绑定值为 “0x0800” 的CC上。

```
ADD CCT: CCTMPLTNAME="cct_test", QHT=0, VQT=20, TQT=20, UQT=0, VT=30, MAXNUMOFCCC=3, PDUVOLUMELIMIT=500, PDUTIMELIMIT=30, RGVOLUMELIMIT=500, RGTIMELIMIT=30, FUATERMINATE=TERM_SERVICE, TIMEFORMAT=LocalTime, MAXSVCCONTAINER=100, SECRUTHRESHOLD=100;
ADD SELECTCCTBYCC: CCTYPE=VALUE, CCVALUE="0x0800", CCTMPLTNAME="cct_test";
```

//任务三：配置DNN粒度的CCT，绑定在名为 “apn-test” 的DNN实例上。

```
ADD APN: APN="apn-test", HASVPN=DISABLE, HASVPNIPV6=DISABLE, RESTORPGWSWITCH=INHERIT;
SET APNCHARGECTRL: APN="apn-test", CCTEMPLATE="cct_test";
```

//任务四：配置UserProfile粒度额CCT，绑定在名为up_test的UserProfile上。

```
ADD USERPROFILE: USERPROFILENAME="up_test", UPTYPE=PCC;
SET USRPROFCHARGE: USRPROFNAME="up_test", CCTEMPLATE="cct_test";
```

//配置UserProfile组，将UserProfile绑定到UserProfile组。

```
ADD USRPROFGROUP: USERPROFGNAME="upg_test";
ADD UPBINDUPG: USERPROFGNAME="upg_test", UPBINDTYPE=DEFAULT, USERPROFILENAME="userprofile001";
```

//配置DNN实例，将UserProfile组绑定到DNN实例。

```
ADD APN: APN="apn-test", HASVPN=DISABLE, HASVPNIPV6=DISABLE, RESTORPGWSWITCH=INHERIT;
ADD APNUSRPROFG: APN="apn-test", USERPROFGNAME="upg_test";
```

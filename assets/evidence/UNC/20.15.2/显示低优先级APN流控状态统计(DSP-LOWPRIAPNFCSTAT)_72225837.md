#### [命令功能](#ZH-CN_MMLREF_0000001172225837)

**适用网元：MME**

该命令用于显示低优先级APN流控功能的相关状态和统计信息。

#### [注意事项](#ZH-CN_MMLREF_0000001172225837)

无

#### [本地用户权限](#ZH-CN_MMLREF_0000001172225837)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172225837)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172225837)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| QUERYTYPE | 查询方式 | 可选必选说明：可选参数<br>参数含义：该参数用于指定低优先级APN流控状态统计信息的查询方式。<br>数据来源：本端规划<br>取值范围：<br>- “ALL(整系统查询)”<br>- “PROC(按进程查询)”<br>默认值：<br>“ALL(整系统查询)” |
| RUNAME | RU名称 | 可选必选说明：条件必选参数<br>参数含义：该参数用于指定SPU资源单元名称。该参数可以通过<br>[DSP RU](../../../../../../../../平台服务管理/单体服务公共功能管理/系统管理/资源管理/RU管理/显示资源单元信息（DSP RU）_59103857.md)<br>命令查询。<br>前提条件：该参数在<br>“查询方式”<br>配置为<br>“PROC(按进程查询)”<br>后生效。<br>数据来源：本端规划<br>取值范围：1～63位字符串<br>默认值：无 |
| PROCNO | 进程号 | 可选必选说明：条件必选参数<br>参数含义：该参数用于指定需要查看流控状态信息对应SPP进程的进程号。<br>前提条件：该参数在<br>“查询方式”<br>配置为<br>“PROC(按进程查询)”<br>后生效。<br>数据来源：本端规划<br>取值范围：0～20<br>默认值：无 |

#### [使用实例](#ZH-CN_MMLREF_0000001172225837)

按照“查询方式为ALL(整系统查询)”显示低优先级APN流控状态统计：

DSP LOWPRIAPNFCSTAT: QUERYTYPE=ALL;

```
%%DSP LOWPRIAPNFCSTAT: QUERYTYPE=ALL;%%
RETCODE = 0  操作成功。

输出结果如下

```

#### [输出结果说明](#ZH-CN_MMLREF_0000001172225837)

| 输出项名称 | 输出项解释 |
| --- | --- |
| RU名称 | 该参数用于显示SPP进程所属的资源单元名称。 |
| 进程号 | 该参数用于显示SPP进程序号。 |
| 流控告警状态 | 参数含义：该参数用于显示低优先级APN流控告警状态。<br>取值范围：<br>- 功能关闭：表示低优先级APN流控功能未开启。<br>- 正常状态：表示功能开启，但非低优先级APN流控告警状态。<br>- 告警状态：表示处于低优先级APN流控告警状态。 |
| 流控速率（个/秒） | 参数含义：该参数用于显示系统每个SPP进程允许通过的低优先级APN业务消息的速率。<br>取值范围：0～2000 |
| 通过量（个） | 参数含义：该参数用于显示低优先级APN业务通过的消息包数总量累计值，超出取值范围后会重新计数。<br>取值范围：0～4294967295 |
| 拒绝量（个） | 参数含义：该参数用于显示低优先级APN业务拒绝的消息包数总量累计值，超出取值范围后会重新计数。<br>取值范围：0～4294967295 |

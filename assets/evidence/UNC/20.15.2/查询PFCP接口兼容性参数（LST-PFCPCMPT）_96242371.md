## [命令功能](#ZH-CN_MMLREF_0296242371)

**适用NF：SGW-C、PGW-C、SMF、GGSN**

该命令用于查询PFCP接口兼容性参数。

## [注意事项](#ZH-CN_MMLREF_0296242371)

无

#### [操作用户权限](#ZH-CN_MMLREF_0296242371)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0296242371)

无

## [使用实例](#ZH-CN_MMLREF_0296242371)

查询PFCP接口兼容性配置，执行如下命令：

```
%%LST PFCPCMPT:;%%
RETCODE = 0  操作成功

结果如下
--------
     是否支持BAR信元  =  不支持
是否携带地址池占位符  =   否
         是否填写VPN  =   否
(结果个数 = 3)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0296242371)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 是否支持BAR信元 | 该参数用于控制PFCP接口是否支持BAR（Buffering Action Rule）信元。 |
| 是否携带地址池占位符 | 该参数用于指定PFCP接口消息是否支持携带地址池占位符。UNC向UDG申请双栈地址的时候，如果只有IPv6类型的地址池，则将IPv4地址池的位置用*代替，即为占位符。<br>以免对端将IPv6类型的地址池识别成IPv4类型。 |
| 路由实例是否填写VPN | 该参数用于控制PFCP接口中N6口PDR的Network Instance信元是否填写VPN。 |

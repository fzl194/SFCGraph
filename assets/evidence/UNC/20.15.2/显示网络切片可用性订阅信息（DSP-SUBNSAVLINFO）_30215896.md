## [命令功能](#ZH-CN_MMLREF_0000001330215896)

**适用NF：AMF**

该命令用于显示NSSF向AMF返回的网络切片可用性的订阅信息。

## [注意事项](#ZH-CN_MMLREF_0000001330215896)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001330215896)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001330215896)

无

## [使用实例](#ZH-CN_MMLREF_0000001330215896)

显示NSSF向AMF返回的网络切片可用性的订阅信息，执行如下命令：

```
DSP SUBNSAVLINFO:;
```

## [输出结果说明](#ZH-CN_MMLREF_0000001330215896)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 跟踪区标识 | 该参数用于表示NSSF向AMF返回的订阅成功的跟踪区标识。TAI由PLMN ID和TAC组成。 |
| 订阅ID | 该参数用于表示NSSF向AMF返回的网络切片可用性信息订阅ID。 |
| 订阅失效时间 | 该参数用于表示NSSF向AMF返回的网络切片可用性信息的订阅失效时间。 |

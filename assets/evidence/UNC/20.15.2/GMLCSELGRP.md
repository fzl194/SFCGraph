#### [命令功能](#ZH-CN_MMLREF_0000001172345411)

**适用网元：MME**

该命令用于查询GMLC选择策略组。

#### [注意事项](#ZH-CN_MMLREF_0000001172345411)

- 无。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172345411)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172345411)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172345411)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| GMLCGRPID | GMLC选择策略组索引 | 可选必选说明：可选参数<br>参数含义：该参数在系统内唯一标识一个GMLC选择策略组。<br>数据来源：本端规划<br>取值范围：0~191<br>默认值：无 |

#### [使用实例](#ZH-CN_MMLREF_0000001172345411)

1. 查询所有GMLC选择策略组。
  LST GMLCSELGRP:;
  ```
  %%LST GMLCSELGRP:;%%
  RETCODE = 0  操作成功。

  输出结果如下
  --------------
  GMLC选择策略组索引  =  0
                描述  =  noname
  (结果个数 = 1)

  ---    END
  ```

#### [命令功能](#ZH-CN_CONCEPT_0000001600440453)

该命令用于查询IP可达性检测自动化配置模板。

#### [注意事项](#ZH-CN_CONCEPT_0000001600440453)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600440453)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600440453)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SERVICENAME | 服务名称 | 可选必选说明：可选参数<br>参数含义：该参数用来表示接口自动化配置服务模板名称。该参数由ADD AUTOSCALINGSERVICE命令配置获得。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。不支持空格和中文。<br>默认值：无 |
| IPVERSION | IP版本 | 可选必选说明：可选参数<br>参数含义：该参数用来表示IP版本。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- IPv4：IPv4地址族。<br>- IPv6：IPv6地址族。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600440453)

查询IP可达性检测自动化配置模板：

```
LST AUTOSCALINGIPREACH:;
```

```

RETCODE = 0  操作成功。

```

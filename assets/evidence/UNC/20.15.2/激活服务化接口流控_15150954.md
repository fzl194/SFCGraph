## [任务示例](#ZH-CN_OPI_0000001615150954)

任务描述

**任务1：** 以AMF作为HTTP客户端为例，在AMF上配置链路粒度的固定速率流控。

**任务2：** 以AMF作为HTTP客户端为例，在AMF上配置整机粒度的固定速率流控。

**任务3：** 以AMF到UDM的局向为例 ，AMF作为HTTP客户端，在AMF上配置基于接口类型的固定速率流控。

脚本

**任务1：** 以AMF作为HTTP客户端为例，在AMF上配置链路粒度的固定速。

//配置流控组信息。

```
ADD HTTPFCIPGRP: INDEX=1, GROUP=1, IPTYPE=IPv4, IPV4ADDR="192.168.100.1", IPV4MASK="255.255.255.0", DESC="UDM_instance1"; 
ADD HTTPFCIPGRP: INDEX=2, GROUP=1, IPTYPE=IPv4, IPV4ADDR="192.168.100.2", IPV4MASK="255.255.255.0", DESC="UDM_instance2"; 
```

//配置流控速率。

```
ADD HTTPFIXEDFC: GROUP=1, GROUPTYPE=CLIENT, SENDTHD=1000, DESC="LINK"; 
```

**任务2** ：以AMF作为HTTP客户端为例，在AMF上配置整机粒度的固定速率流控。

//配置流控组信息。

```
ADD HTTPFCIPGRP: INDEX=3, GROUP=2, IPTYPE=IPv4, IPV4ADDR="192.168.200.1", IPV4MASK="255.255.255.0", DESC="UDM_instance1"; 
```

//配置流控速率。

```
ADD HTTPFIXEDFC: GROUP=2, GROUPTYPE=CLIENT, GLBSENDTHD=2000, DESC="GLOBAL"
```

**任务3** ：以AMF到UDM的局向为例，AMF作为HTTP客户端，在AMF上配置基于接口类型的固定速率流控。

//增加基于接口类型的固定速率流控生效的HTTP局向。

```
ADD HTTPOFC: OFCIDX=0, OFCTYPE=NFTYPE, PEERNFTYPE=NFTypeUDM, NFITEM=HTRINTF-0&FIXSPDFCINTF-1, LOCALNFTYPE=NFTypeAMF, OFCNAME="AMF2UDMFIXSPDFCINTF";
```

//配置基于接口类型的固定速率流控。

```
 ADD HTTPFIXEDFCINF: OFCIDX=0, LETYPE=CLIENT, SENDTHD=1000;
```

#### [命令功能](#ZH-CN_MMLREF_0000001172225733)

**适用网元：SGSN、MME**

该命令用于查询归属网络信息。

#### [注意事项](#ZH-CN_MMLREF_0000001172225733)

该命令执行后立即生效。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172225733)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172225733)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172225733)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| NOID | 运营商标识 | 可选必选说明：可选参数<br>参数含义：该参数用于指定运营商标识<br>取值范围：0～64，128～254<br>默认值：无 |

#### [使用实例](#ZH-CN_MMLREF_0000001172225733)

查看所有的HNOINFO:

LST HNOINFO:;

```
%%LST HNOINFO:;%%
RETCODE = 0  操作成功。

输出结果如下
--------------
     运营商标识  =  0
     本局SGSN号  =  123
   本地实体索引  =  0
(结果个数 = 1)

---    END
```

#### [命令功能](#ZH-CN_MMLREF_0000001126145730)

**适用网元：SGSN**

此命令用于查询SMSC转换记录。

#### [注意事项](#ZH-CN_MMLREF_0000001126145730)

此命令执行后立即生效。

#### [本地用户权限](#ZH-CN_MMLREF_0000001126145730)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126145730)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126145730)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| REQADDR | 请求SMSC地址 | 可选必选说明：可选参数<br>参数含义：该参数用于指定请求SMSC地址。<br>“请求SMSC地址”<br>是手机短消息中携带的请求SMSC地址。<br>取值范围：1～16位十进制字符串<br>默认值：无 |

#### [使用实例](#ZH-CN_MMLREF_0000001126145730)

查询SMSC转换配置记录：

LST SMSCCONVERT:;

```
%%LST SMSCCONVERT:;%%
RETCODE = 0  操作成功。

输出结果如下
--------------
请求SMSC地址  =  8613951701
转换SMSC地址  =  8613951996
        描述  =  FOR MSC1
(结果个数 = 1)

---    END
```

#### [命令功能](#ZH-CN_CONCEPT_0000001549801814)

该命令用于查询免费ARP过滤配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001549801814)

- 该命令执行后立即生效。
- 该命令仅适用于非NP卡场景和NP卡非加速模式场景。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549801814)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549801814)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IFNAME | 接口名称 | 可选必选说明：可选参数<br>参数含义：接口名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～47。<br>默认值：无<br>配置原则：使用LST INTERFACE命令查看可用接口。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549801814)

- 查询免费ARP过滤配置：
  ```
  LST SECARPGRAT:;
  ```
  ```
  RETCODE = 0  操作成功

  结果如下
  -------------------------
  接口名称  =  Ethernet66/0/3
  使能标记  =  TRUE
  (结果个数 = 1)
  ---    END
  ```
- 查询指定条件的免费ARP过滤配置：
  ```
  LST SECARPGRAT:IFNAME="Ethernet66/0/3";
  ```
  ```
  RETCODE = 0  操作成功

  结果如下
  -------------------------
  接口名称  =  Ethernet66/0/3
  使能标记  =  TRUE
  (结果个数 = 1)
  ---    END
  ```

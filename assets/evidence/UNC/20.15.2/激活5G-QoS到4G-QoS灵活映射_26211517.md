## [操作步骤](#ZH-CN_OPI_0000001326211517)

1. 进入 “MML命令行-UNC” 窗口。
2. 配置5QI到标准QCI的映射。
  [**ADD QOS5QI2QCIMAP**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/QoS/QoS映射/5QI到QCI映射/增加5QI到标准QCI的映射（ADD QOS5QI2QCIMAP）_40119805.md)
3. 配置APN-AMBR带宽速率控制。
  [**ADD APNAMBRCTRL**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/QoS/QoS映射/APN-AMBR控制/增加APN-AMBR控制的配置（ADD APNAMBRCTRL）_39919385.md)
4. 配置GBR承载带宽速率控制。
  [**ADD GBRBWCTRL**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/QoS/QoS映射/GBR带宽控制/增加GBR承载带宽速率控制配置（ADD GBRBWCTRL）_87360188.md)

## [任务示例](#ZH-CN_OPI_0000001326211517)

任务描述

配置5G QoS到4G QoS的灵活映射。

脚本

//配置5QI到标准QCI的映射。

```
ADD QOS5QI2QCIMAP:CTRLTYPE=APN_LEVEL,APN="huawei.com",QOS5QISTART=1,QOS5QIEND=2,QCI=1;
```

//配置APN-AMBR带宽速率控制。

```
ADD APNAMBRCTRL:CTRLTYPE=APN_LEVEL,APN="huawei.com",AMBRUL=1000,AMBRDL=1000;
```

//配置GBR承载带宽速率控制。

```
ADD GBRBWCTRL:CTRLTYPE=APN_LEVEL,APN="huawei.com",MBRUL=1000,MBRDL=1000,GBRUL=1000,GBRDL=1000;
```

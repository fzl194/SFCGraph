#### [命令功能](#ZH-CN_CONCEPT_0000001600600413)

该命令用来查询安全策略URPF配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001600600413)

- 该命令执行后立即生效。
- 该命令仅适用于非NP卡场景和NP卡非加速模式场景。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600600413)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600600413)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001600600413)

查询安全策略中的URPF配置：

```
LST SECDEVURPF:;
```

```
RETCODE = 0  操作成功

结果如下
------------------------
        策略编号  =  1
安全URPF检查类型  =  严格检查
是否允许缺省路由  =  FALSE
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600600413)

| 输出项名称 | 输出项解释 |

#### [命令功能](#ZH-CN_MMLREF_0000001172225679)

**适用网元：SGSN**

此命令用于显示本端端口号选择范围。

#### [注意事项](#ZH-CN_MMLREF_0000001172225679)

此命令执行后立即生效。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172225679)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172225679)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172225679)

无。

#### [使用实例](#ZH-CN_MMLREF_0000001172225679)

显示本端端口号选择范围：

LST GBLOCPORTRGE:;

```
LST GBLOCPORTRGE:;%%
RETCODE = 0  操作成功。

操作结果如下
--------------
  起始端口号  =  1024
  结束端口号  =  65535
起始预置端口  =  2000
  预置端口数  =  128
    描述信息  =  DEFAULT
(结果个数 = 1)

---    END
```

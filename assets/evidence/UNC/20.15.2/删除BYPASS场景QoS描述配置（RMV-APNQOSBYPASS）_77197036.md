# 删除BYPASS场景QoS描述配置（RMV APNQOSBYPASS）

- [注意事项](#ZH-CN_MMLREF_0000001177197036__1.3.2)

## [命令功能](#ZH-CN_MMLREF_0000001177197036)

**适用NF：SMF**

该命令用于删除APNQOSBYPASS的配置信息。

## [注意事项](#ZH-CN_MMLREF_0000001177197036)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0000001177197036)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001177197036)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| APN | APN名称 | 可选必选说明：必选参数<br>参数含义：该参数用来指定APN名称。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是1~63。<br>默认值：无<br>配置原则：<br>该参数使用ADD APN命令配置生成。 |

## [使用实例](#ZH-CN_MMLREF_0000001177197036)

删除APN为"huawei.com"的配置信息：

```
RMV APNQOSBYPASS:APN="huawei.com";
```

#### [命令功能](#ZH-CN_CONCEPT_0000001600440649)

该命令用于查询OSPFv3默认路由宣告配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001600440649)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600440649)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600440649)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | OSPFv3进程号 | 可选必选说明：可选参数<br>参数含义：OSPFv3进程号。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600440649)

查询OSPFv3默认路由宣告配置：

```
LST OSPFV3DEFAULTROUTE:;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600440649)

| 输出项名称 | 输出项解释 |
| --- | --- |
| OSPFv3进程号 | 该参数指定用户创建的OSPFv3进程号。 |
| 拓扑标识 | 该参数指定用户创建的OSPFv3拓扑标识。 |
| 使能度量 | 该参数指定是否使能缺省路由度量值。 |
| 外部LSA开销值 | 该参数指定自治系统外部LSA的开销值，仅当使能度量为true时生效。 |
| 缺省路由使能类型 | 该参数指定是否使能缺省路由类型。 |
| 引入外部路由类型 | 该参数指定引入外部路由类型，仅当缺省路由使能类型为true时生效。 |
| 延迟发布缺省路由的时间（s） | 该参数用于配置发布缺省路由的延时定时器，延迟发布缺省路由的时间。 |
| 默认路由发布标志 | 该参数指定默认路由发布标志。 |
| 计算默认路由 | 该参数指定是否计算默认路由。 |
| 路由策略 | 该参数指定路由策略名称，通过路由策略，实现在路由表中有匹配的非OSPFv3产生的缺省路由表项时，按路由策略所配置的参数发布缺省路由。 |
| 缺省路由使能标签 | 该参数指定是否使能缺省路由标签。 |
| 路由标签 | 该参数指定路由标签名称。 |

# 激活支持不同PLMN切片能力差异化

- [操作场景](#ZH-CN_OPI_0000001252527766__1.3.1)

## [操作场景](#ZH-CN_OPI_0000001252527766)

本操作指导介绍在运行网络中激活支持不同PLMN下网络切片能力差异化功能的操作过程。运营商需根据实际情况来确定及规划多PLMN下的网络切片配置。

AMF、SMF支持多个PLMN，不同PLMN下的网络切片支持的能力可能不同，在业务识别时需要对不同PLMN下的网络切片进行区分，以满足运营商支持不同PLMN下网络切片能力差异化的需求。

> **说明**
> 适用于AMF、SMF、NRF。

## [操作步骤](#ZH-CN_OPI_0000001252527766)

1. 进入 “MML命令行-UNC” 窗口。
2. 开启本端网元向NRF注册及更新时携带perPlmnSnssaiList信元的功能。
  [**SET NFNRFMGMTPARA**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/服务化接口管理/注册与服务发现/NRF管理/NRF参数管理/NF与NRF间全局管理参数管理/设置NF与NRF间的全局管理参数（SET NFNRFMGMTPARA）_32041712.md)
3. 配置控制本端网元缓存服务发现时支持根据perPlmnSnssaiList匹配发现参数。
  [**SET NFDISCPLCY**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/服务化接口管理/注册与服务发现/本地NRF功能管理/NF发现策略管理/设置NF的服务发现策略（SET NFDISCPLCY）_09651764.md)
4. 增加运营商的PLMN支持的网络切片（例如，运营商有PLMN1和PLMN2，PLMN1支持切片1，PLMN2支持切片2和3）。
  **增加运营商的PLMN1支持的网络切片1。**
    a. 增加运营商的Home PLMN的信息。
      [**ADD NGHPLMN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/运营商管理/Home PLMN信息管理/增加5G Home PLMN（ADD NGHPLMN）_09651693.md)
    b. 增加运营商Serving PLMN的信息。

## [任务示例](#ZH-CN_OPI_0000001252527766)

任务描述

运营商配置SMF支持2个PLMN，配置PLMN1支持的网络切片1，PLMN2支持的网络切片2、3。SMF上配置不同PLMN下网络切片能力差异，满足不同PLMN下可用网络切片差异化的需求。

脚本

//配置SMF向NRF注册及更新时携带perPlmnSnssaiList信元。

```
SET NFNRFMGMTPARA: REGPERPLMNNSSW=ON;
```

//配置AMF缓存服务发现时支持根据perPlmnSnssaiList匹配发现参数。

```
SET NFDISCPLCY: PERPLMNSELSW=ON;
```

//增加运营商的PLMN1支持的网络切片1。

```
ADD NGHPLMN: NOID=0, MCC="123", MNC="68";
ADD NGSRVPLMN: PLMNIDX=1, NOID=0, MCC="123", MNC="68";
ADD H2SRVPLMN: HMCC="123", HMNC="68", SRVPLMNIDX=1;
ADD PLMNNS: NSIDX=1, PLMNIDX=1, SST=1, SD="010101", STATE=ACTIVE, NSSCOPE=GLOBALNW;
```

//增加运营商的PLMN2支持的网络切片2、3。

```
ADD NGHPLMN: NOID=0, MCC="123", MNC="69";
ADD NGSRVPLMN: PLMNIDX=1, NOID=0, MCC="123", MNC="69";
ADD H2SRVPLMN: HMCC="123", HMNC="69", SRVPLMNIDX=2;
ADD PLMNNS: NSIDX=2, PLMNIDX=2, SST=1, SD="010111", STATE=ACTIVE, NSSCOPE=GLOBALNW;
ADD PLMNNS: NSIDX=3, PLMNIDX=2, SST=2, SD="FFFFFF", STATE=ACTIVE, NSSCOPE=GLOBALNW;
```

//为SMF增加其所支持的网络切片1、2、3。

```
ADD NFNS: NFTYPE=NfSMF, NSIDX=1;
ADD NFNS: NFTYPE=NfSMF, NSIDX=2;
ADD NFNS: NFTYPE=NfSMF, NSIDX=3;
```

# 查询基于APN的DDN消息优先级配置（LST DDNPRIORITYAPN）

- [注意事项](#ZH-CN_MMLREF_0304284709__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0304284709__1.3.4)

## [命令功能](#ZH-CN_MMLREF_0304284709)

**适用NF：SGW-C、SMF**

该命令用于查询基于APN的DDN消息优先级配置。

## [注意事项](#ZH-CN_MMLREF_0304284709)

无

#### [操作用户权限](#ZH-CN_MMLREF_0304284709)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0304284709)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| APN | APN名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定APN。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~63。只能由“-”、数字、大小写字母和“.”组成，不能以“.”开头且不能出现连续两个“.”。不支持空格及“_”、“#”、“$”、“&”、“%”、“^”、“（”、“）”、“，”、“/”、“;”、“:”、“””、“`”特殊字符，不区分大小写。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0304284709)

查询APN的DDN消息优先级配置：

```
%%LST DDNPRIORITYAPN:;%%
RETCODE = 0  操作成功

基于APN的DDN消息优先级
----------------------
APN名称       优先级                 

0168apn1.com  低  
huawei.com    低  
(结果个数 = 2)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0304284709)

| 输出项名称 | 输出项解释 |
| --- | --- |
| APN名称 | 该参数用于指定APN。 |
| 优先级 | 该参数用于指定DDN优先级。 |

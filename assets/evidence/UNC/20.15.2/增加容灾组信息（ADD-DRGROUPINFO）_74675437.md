## [命令功能](#ZH-CN_MMLREF_0000001374675437)

该命令用于增加容灾组信息。

## [注意事项](#ZH-CN_MMLREF_0000001374675437)

- 该命令执行后立即生效。

- 该命令只用于在UEG-L/UEN网元采用主备（冷备）容灾模式下执行。
- 该命令只用于在UEG-M/UEG网元采用主备（热备）容灾模式下执行。
- 在两个有容灾关系的网元上配置容灾组信息时，容灾组标识、容灾组名称要相同，本端与对端的容灾实例标识要相对应，否则容灾功能无法使用。
- 在同一个网元下，不允许增加具有相同容灾组标识的容灾组信息，对于错误的容灾组信息，应先进行删除操作，再重新增加正确的容灾组信息。

- 最多可输入1条记录。

#### [操作用户权限](#ZH-CN_MMLREF_0000001374675437)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001374675437)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| DRGROUPID | 容灾组标识 | 可选必选说明：必选参数<br>参数含义：该参数用于标识一条容灾组信息，不同容灾组信息的该标识不能相同。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是1~8。<br>默认值：无<br>配置原则：无 |
| DRGROUPNAME | 容灾组名称 | 可选必选说明：必选参数<br>参数含义：该参数用于表示容灾组的名称。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是1~32。只支持字母和数字，字母区分大小写，不能包含空格和特殊字符。<br>默认值：无<br>配置原则：无 |
| LDRINSTID | 本端容灾实例标识 | 可选必选说明：必选参数<br>参数含义：该参数用于表示容灾组中本端的网元标识。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是0~255。<br>默认值：无<br>配置原则：无 |
| DFTMODE | 本端默认运行模式 | 可选必选说明：必选参数<br>参数含义：该参数用于配置默认的运行模式，容灾功能运行后，会根据该默认的运行模式，与对端协商真实使用的运行模式。<br>数据来源：全网规划<br>取值范围：<br>- “ACTIVE（配置主）”：配置默认运行模式为主角色<br>- “STANDBY（配置备）”：配置默认运行模式为备角色<br>默认值：无<br>配置原则：无 |
| PDRINSTID | 对端容灾实例标识 | 可选必选说明：必选参数<br>参数含义：该参数用于表示容灾组中对端的网元标识。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是0~255。<br>默认值：无<br>配置原则：无 |
| IPVERSION | 对端容灾控制通道IP地址版本号 | 可选必选说明：可选参数<br>参数含义：该参数用于用户指定对端容灾控制通道IP地址类型。<br>数据来源：全网规划<br>取值范围：<br>- IPV4（IPv4类型地址）<br>- IPV6（IPv6类型地址）<br>默认值：无<br>配置原则：无 |
| PGRCIP | 对端容灾控制通道IPv4地址 | 可选必选说明：该参数在"IPVERSION"配置为"IPV4"时为条件必选参数。<br>参数含义：该参数用于配置对端容灾控制通道的IPv4地址。<br>数据来源：全网规划<br>取值范围：IPv4地址类型。<br>默认值：无<br>配置原则：无 |
| PGRCIPV6 | 对端容灾控制通道IPv6地址 | 可选必选说明：该参数在"IPVERSION"配置为"IPV6"时为条件必选参数。<br>参数含义：该参数用于配置对端容灾控制通道的IPv6地址。<br>数据来源：全网规划<br>取值范围：IPv6地址类型。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001374675437)

增加一条容灾组信息：

```
%%ADD DRGROUPINFO: DRGROUPID=1, DRGROUPNAME="vnf1", LDRINSTID=1, DFTMODE=ACTIVE, PDRINSTID=2;%%
RETCODE = 0  操作成功

---    END
```

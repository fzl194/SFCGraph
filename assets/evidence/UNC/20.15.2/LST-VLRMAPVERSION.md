## [命令功能](#ZH-CN_MMLREF_0000001353321874)

**适用NF：SMSF**

该命令用于查询VLR对接HLR业务中使用的MAP接口版本号。

## [注意事项](#ZH-CN_MMLREF_0000001353321874)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001353321874)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001353321874)

无

## [使用实例](#ZH-CN_MMLREF_0000001353321874)

运营商希望查询VLR对接HLR业务中使用的MAP接口版本号，执行如下命令：

```
LST VLRMAPVERSION:;
%%LST VLRMAPVERSION:;%%
RETCODE = 0  操作成功

结果如下：
------------------------
        MAP 版本号 =  V3
(结果个数 = 1)
```

## [输出结果说明](#ZH-CN_MMLREF_0000001353321874)

| 输出项名称 | 输出项解释 |
| --- | --- |
| MAP版本号 | 该参数用于表示VLR对接HLR业务中使用的MAP接口版本号。 |

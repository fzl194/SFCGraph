## [命令功能](#ZH-CN_MMLREF_0296805487)

**适用NF：PGW-C、SMF**

该命令用于查询N40接口配额的控制参数。

## [注意事项](#ZH-CN_MMLREF_0296805487)

无

#### [操作用户权限](#ZH-CN_MMLREF_0296805487)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0296805487)

无

## [使用实例](#ZH-CN_MMLREF_0296805487)

查询N40接口配额的控制参数：

```
%%LST N40QUOTACTRL:;%%
RETCODE = 0  操作成功

结果如下
--------
       CHF下发零配额的动作  =  正常处理
   CHF下发零时长配额的动作  =  去活会话
RG级阻塞处理时间间隔(分钟)  =  0
      在线计费上报统计类型  =  缺省
              费率切换模式  =  缺省
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0296805487)

| 输出项名称 | 输出项解释 |
| --- | --- |
| CHF下发零配额的动作 | 该参数用于指定CHF下发零配额时的处理动作。 |
| CHF下发零时长配额的动作 | 该参数用于控制CHF响应消息中GrantedUnit中携带零时长配额的动作。 |
| RG级阻塞处理时间间隔(分钟) | 该参数用于指定RG级阻塞动作的业务阻塞时间。从阻塞开始经过配置时长以后，业务再来时可以重新触发配额申请。 |
| 在线计费上报统计类型 | 该参数用于控制CHF仅下发在线计费RG时长或流量配额时，是否同时上报时长和流量的使用情况给CHF。 |
| 费率切换模式 | 该参数用于指定零点时是否强制发生费率切换。 |

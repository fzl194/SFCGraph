## [命令功能](#ZH-CN_MMLREF_0000001608719037)

本命令用于查询存储故障时，裸机节点自动进入BYPASS开关的状态。

> **说明**
> 该命令仅在Full-stack裸机场景下支持。

## [注意事项](#ZH-CN_MMLREF_0000001608719037)

无。

## [参数说明](#ZH-CN_MMLREF_0000001608719037)

无。

## [使用实例](#ZH-CN_MMLREF_0000001608719037)

1. 查询自动进入BYPASS开关成功。
  ```
  %%DSP BMBYPASSSWITCH:;%% 
  RETCODE = 0  操作成功  
  操作结果如下 
  ------------ 
  节点名称                           节点IP       自动进入BYPASS开关
  e23d6419-cie-qb8it-dbf4467df-6vpwh XX.XX.XX.XX  开启
  e23d6419-cie-qb8it-dbf4467df-mx6cj XX.XX.XX.XX  开启
  e23d6419-cie-qb8it-dbf4467df-6vpwh XX.XX.XX.XX  开启             
  (结果个数 = 3) 
  ---    END
  ```
2. 查询自动进入BYPASS开关部分失败。
  ```
  %%DSP BMBYPASSSWITCH:;%% 
  RETCODE = 20026  部分失败  
  操作结果如下 
  ------------ 
  节点名称                           节点IP       自动进入BYPASS开关
  e23d6419-cie-qb8it-dbf4467df-6vpwh XX.XX.XX.XX  未知
  e23d6419-cie-qb8it-dbf4467df-mx6cj XX.XX.XX.XX  开启
  e23d6419-cie-qb8it-dbf4467df-6vpwh XX.XX.XX.XX  开启             
  (结果个数 = 3) 
  ---    END
  ```

## [输出结果说明](#ZH-CN_MMLREF_0000001608719037)

命令执行正常，会返回命令执行成功的提示信息。输出结果说明如 [表1](#ZH-CN_MMLREF_0000001608719037__table149221250144917) 所示。

命令执行异常，会返回提示对应的失败信息。错误码列表如 [表2](#ZH-CN_MMLREF_0000001608719037__table698716490471) 所示。

| 节点名称 | 用于返回节点名称。 |

| 自动进入BYPASS开关 | 用于返回裸机节点进入BYPASS开关状态。<br>取值范围：<br>- “开启”：裸机节点自动进入BYPASS开关处于打开状态。<br>- “关闭”：裸机节点自动进入BYPASS开关处于关闭状态。<br>- “未知”：查询裸机节点自动进入BYPASS开关失败。 |

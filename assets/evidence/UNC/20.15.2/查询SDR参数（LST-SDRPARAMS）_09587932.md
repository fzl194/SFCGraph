## [命令功能](#ZH-CN_MMLREF_0209587932)

该命令用于查询SDR服务的相关参数。SDR主要功能是为微服务间提供分布式透明通信。

## [注意事项](#ZH-CN_MMLREF_0209587932)

- SDR参数ID为0的功能该版本不支持；
- SDR参数ID为4的功能，其对应的PARAVALUE1表示Fabric平面可靠传输的校验和开关。修改PARAVALUE1的值后，请全量复位POD，否则会上报系统命令配置修改未生效告警。

#### [操作用户权限](#ZH-CN_MMLREF_0209587932)

G_1，管理员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209587932)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PARAID | 参数ID | 可选必选说明：可选参数<br>参数含义：该参数用于指定SDR命令标识。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~10。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209587932)

查询SDR参数：

```
%%LST SDRPARAMS: PARAID=1;%%
RETCODE = 0  操作成功

结果如下
--------
      参数ID  =  1
第一个参数值  =  0
第二个参数值  =  0
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209587932)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 参数ID | 该参数用于指定SDR命令标识。 |
| 第一个参数值 | 该参数用于SDR命令。<br>当SDR参数ID为0时，该参数值用于SDR可靠传输开关控制，0表示关闭base和fabric可靠传输开关，1表示只开启可靠base传输开关，2表示只开启可靠fabric传输开关，3表示开启base和fabric可靠传输开关。<br>当SDR参数ID为1时，该参数值用于Token通信场景Fabric链路的故障检测和业务隔离能力功能开关，0表示关闭，1表示开启。<br>当SDR参数ID为2时，该参数值用于DTP功能开关，0表示关闭，1表示开启。<br>当SDR参数ID为3时，该参数值用于SDRS内多线程消息处理转组件线程消息处理功能开关，0表示关闭，1表示开启。<br>当SDR参数ID为4时，该参数值用于Fabric可靠传输校验和开关，0表示关闭，1表示开启。<br>当SDR参数ID为5时，该参数值用于SDR话统数据上报开关，0表示关闭，1表示开启。<br>当SDR参数ID为6时，该参数值用于SDRS与SDRC之间的策略核查开关，0表示关闭，1表示开启。<br>当SDR参数ID为7时，该参数值用于SDR pbuf流控功能开关，0表示关闭，1表示开启。<br>当SDR参数ID为8时，该参数值用于无状态通信场景Fabric链路的故障检测和业务隔离能力功能开关，0表示关闭，1表示开启。<br>当SDR参数ID为9时，该参数值用于SDR内统日志打印功能开关，0表示关闭，1表示开启。<br>当SDR参数ID为10时，该参数值用于SDR系统命令配置修改未生效告警功能开关，0表示关闭，1表示开启。 |
| 第二个参数值 | 该参数用于SDR命令值扩展，类型是字符串。当SDR参数ID为6时，该参数值用于表示SDRS与SDRC之间的策略核查周期，单位为分钟。<br>当SDR参数ID为7时，该参数值用于表示pbuf流控功能起控阈值和pbuf使用率刷新周期，刷新周期单位为毫秒。<br>当SDR参数ID为10时，该参数值用于表示系统命令配置修改未生效告警检测周期，单位为分钟。 |

# 调测终端二次鉴权

- [操作场景](#ZH-CN_OPI_0000001374143640__1.3.1)
- [必备事项](#ZH-CN_OPI_0000001374143640__1.3.2)
- [操作步骤](#ZH-CN_OPI_0000001374143640__1.3.3)

## [操作场景](#ZH-CN_OPI_0000001374143640)

终端二次鉴权功能激活后，调测终端二次鉴权功能。

> **说明**
> 适用于SMF。

## [必备事项](#ZH-CN_OPI_0000001374143640)

前提条件

- 请仔细阅读[终端二次鉴权特性概述](WSFD-108007 终端二次鉴权特性概述_73984008.md)。
- 完成[激活RADIUS功能](../../组网功能/WSFD-011306 Radius功能/激活RADIUS功能_15542173.md)。
- 完成[激活Radius鉴权接入](../../组网功能/WSFD-011305 Radius鉴权接入/激活Radius鉴权接入_50176279.md)。
- 完成[激活终端二次鉴权](激活终端二次鉴权_24743509.md)。

数据

| 类别 | 参数 | 取值样例 | 获取方法 | 说明 |
| --- | --- | --- | --- | --- |
| DSP RDSPFCPSESSINFO | 查询类型（QUERYTYPE） | PEERTEID | 全网规划 | 指定查询类型。<br>取值范围：<br>IPTYPE（Radius客户端IP类型）PEERTEID（RADIUS中转UPF会话中UPF分配的N4口TEID）。 |
| DSP RDSPFCPSESSINFO | 对端TEID（PEERTEID） | 2148532224 | 全网规划 | Radius中转UPF会话中UPF分配的N4口TEID。该参数在"QUERYTYPE"配置为"PEERTEID"时为条件必选参数。<br>取值范围：整数类型，取值范围是0~4294967295。 |

## [操作步骤](#ZH-CN_OPI_0000001374143640)

1. 测试终端使用“huawei.com”接入网络。
2. 进入 “MML命令行-UNC” 窗口。
3. 执行 **[DSP RDSPFCPSESSINFO](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/RADIUS管理/RADIUS维护/RADIUS中转UPF会话信息/显示RADIUS中转UPF会话信息（DSP RDSPFCPSESSINFO）_51714013.md)** 查询UPF与SMF间中转Radius的会话信息。
  ```
  %%DSP RDSPFCPSESSINFO:QUERYTYPE=PEERTEID,PEERTEID=2148532224;%%
  RETCODE = 0  操作成功。

  结果如下
  ------------------------
              查询类型  =  PEERTEID
                IP类型  =  IPv4 Address
  Radius客户端IPv4地址  =  10.2.102.17
  Radius客户端IPv6地址  =  ::
              本端TEID  =  8348532224
              对端TEID  =  2148532224
          本端IPv4地址  =  192.168.208.11
          本端IPv6地址  =  ::
          对端IPv4地址  =  192.168.48.11
          对端IPv6地址  =  ::
  (结果个数 = 1)

  ---    END
  ```
    - 预期结果：“Radius客户端IPv4地址”或“Radius客户端IPv6地址”与实际规划值一致，调测结束。
    - 非预期结果：“Radius客户端IPv4地址”或“Radius客户端IPv6地址”与实际规划值不一致，请参考[激活终端二次鉴权](激活终端二次鉴权_24743509.md)重新配置或执行步骤[4](#ZH-CN_OPI_0000001374143640__zh-cn_opi_0000001150149112_cmd8911102805110)。
4. 收集信息并寻求技术支持。
    a. 执行 **[EXP MML](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/配置管理/配置导出管理/导出MML文件（EXP MML）_47200033.md)** 命令将当前配置数据导出为MML脚本文件并保存。
    b. 收集并保存上述所有查询信息。
    c. 收集归纳所有信息并联系华为技术支持解决。
      > **说明**
      > 对应的Radius功能调测请参考： [WSFD-011306 Radius功能](../../组网功能/WSFD-011306 Radius功能_14755851.md) 和 [WSFD-011305 Radius鉴权接入](../../组网功能/WSFD-011305 Radius鉴权接入_50176277.md) 章节。

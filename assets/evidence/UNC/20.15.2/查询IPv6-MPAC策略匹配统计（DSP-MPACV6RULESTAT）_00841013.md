#### [命令功能](#ZH-CN_CONCEPT_0000001600841013)

该命令用于显示当前规则的匹配统计信息。

不指定RULENAME参数时，查询所有规则名字的统计信息；当指定RULENAME参数时，可以查询指定规则名字的统计信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600841013)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600841013)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600841013)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| POLICYNAME | 策略名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定策略名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。<br>默认值：无 |
| RULENAME | 规则名字 | 可选必选说明：可选参数<br>参数含义：该参数用于指定规则名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600841013)

显示当前规则的匹配统计信息：

```
DSP MPACV6RULESTAT:POLICYNAME="policyV6";
```

```

        RETCODE = 0  操作成功

        结果如下
        -------------------------
        策略名称  =  policyV6
        规则名字  =  name1
        匹配计数  =  0
        (结果个数 = 1)
        ---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600841013)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 策略名称 | 用于指定策略名称。 |
| 规则名字 | 用于指定规则名称。 |
| 匹配计数 | 用于显示匹配规则的报文计数。 |

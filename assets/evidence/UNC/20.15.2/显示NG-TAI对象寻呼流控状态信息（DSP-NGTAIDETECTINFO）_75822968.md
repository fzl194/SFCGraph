## [命令功能](#ZH-CN_MMLREF_0000001875822968)

**适用NF：AMF**

该命令用于查询NG TAI对象寻呼流控状态信息。

## [注意事项](#ZH-CN_MMLREF_0000001875822968)

- 如果选择“报告输出”，且当前TAI检测对象数量较多时，该命令执行时间较长。
- 若连续多次执行该命令可能导致OMU虚机CPU使用率突增。

#### [操作用户权限](#ZH-CN_MMLREF_0000001875822968)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001875822968)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| OUTPUTTYPE | 输出类型 | 可选必选说明：必选参数<br>参数含义：该参数用于指定NG TAI对象寻呼流控状态查询的输出类型。<br>数据来源：本端规划<br>取值范围：<br>- “Summary（统计信息）”：统计信息<br>- “Screen（报告输出）”：报告输出<br>默认值：无<br>配置原则：无 |
| TAIGROUPNAME | TAI组名 | 可选必选说明：该参数在"OUTPUTTYPE"配置为"Screen"时为条件可选参数。<br>参数含义：该参数用于表示待查询的NG TAI组对象名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~255。<br>默认值：无<br>配置原则：<br>查询的对象名称必须先通过ADD PERFNGTAIGROUP进行配置。 |

## [使用实例](#ZH-CN_MMLREF_0000001875822968)

查询系统中的所有NG TAI对象寻呼流控状态，执行如下命令：

```
%%DSP NGTAIDETECTINFO: OUTPUTTYPE=Summary;%%
RETCODE = 0  操作成功

结果如下
------------------------
           任务ID  =  monitor_task_5
     检测任务状态  =  正常
    TAI组对象数量  =  0
异常TAI组对象数量  =  0
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001875822968)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 索引 | 该参数用户显示索引信息。 |
| TAI组名 | 该参数用于表示待查询的NG TAI组对象名称。 |
| TAI组对象状态 | 该参数用于显示TAI组对象状态。<br>取值说明：<br>- “NORMAL（正常）”：TAI组对象状态正常<br>- “ABNORMAL（异常）”：TAI组对象状态异常 |
| TAI组对象状态更新时间 | 该参数用于显示TAI组对象状态最后更新时间。 |
| TAI组对象数量 | 该参数用于显示当前注册的待检测TAI对象总数。 |
| 异常TAI组对象数量 | 该参数用于显示异常TAI组对象数量。 |
| 检测任务状态 | 该参数用于标识检测任务状态。<br>取值说明：<br>- “NORMAL（正常）”：TAI组对象监控任务状态正常<br>- “ABNORMAL（异常）”：TAI组对象监控任务状态异常 |
| 任务ID | 该参数用于显示检测任务ID。 |

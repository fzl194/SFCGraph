#### [命令功能](#ZH-CN_CONCEPT_0000001550280598)

该命令用于显示OSPF的错误信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001550280598)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550280598)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550280598)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | OSPF进程号 | 可选必选说明：可选参数<br>参数含义：OSPF进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |

#### [输出结果说明](#ZH-CN_CONCEPT_0000001550280598)

| 输出项名称 | 输出项解释 |
| --- | --- |
| OSPF进程号 | 该参数表示OSPF进程号。 |
| 相同IP地址数据包计数 | 该参数表示错误报文数量，错误类型是收到自己接口发出的报文。 |
| 错误包 | 该参数表示错误报文数量，错误类型是报文解析错误，错误包括长度域的校验和等。 |
| 错误版本号 | 该参数表示错误报文数量，错误类型是收到OSPF版本号错误。 |
| 错误校验和 | 该参数表示错误报文数量，错误类型是收到OSPF校验和错误。 |
| 错误区域ID | 该参数表示错误报文数量，错误类型是收到的报文中的Area ID与本地不匹配（Vlink可以收到Area0的报文，其他情况的Area ID不一致时都算错误）。 |
| 无编号接口丢包 | 该参数表示错误报文数量，错误类型是接收报文的接口是Unnumbered，但不是P2P类型（Unnumbered接口必须是P2P类型）。 |
| 错误虚拟连接 | 该参数表示错误报文数量，错误类型是Vlink收到非法报文。 |
| 错误验证类型 | 该参数表示错误报文数量，错误类型是认证类型错误。 |
| 错误验证值 | 该参数表示错误报文数量，错误类型是认证密钥错误。 |
| 数据包过大 | 该参数表示错误报文数量，错误类型是OSPF报文长度大于IP报文所允许的长度。 |
| 数据包过小 | 该参数表示错误报文数量，错误类型是收到的报文长度与IP报文头长度加上报文长度之和不相等。 |
| 传输错误 | 该参数表示错误报文数量，错误类型是向Socket发送报文失败。 |
| 接口down | 该参数表示错误报文数量，错误类型是OSPF接口Down。 |
| 未知邻居 | 该参数表示错误报文数量，错误类型是对于NBMA、sham-link和Vlink网络而言，收到非OSPF邻居发来的OSPF报文。 |
| 错误认证序列号 | 该参数表示错误报文数量，错误类型是报文认证序列号错误。 |
| 掩码不匹配 | 该参数表示错误的Hello报文数量，错误类型是地址掩码与本地不匹配。 |
| HelloInterval不匹配 | 该参数表示错误的Hello报文数量，错误类型是HelloInterval与本地不一致。 |
| DeadInterval不匹配 | 该参数表示错误的Hello报文数量，错误类型是DeadInterval与本地不一致。 |
| 可选项不匹配 | 该参数表示错误的Hello报文数量，错误类型是扩展属性与本地不一致的Hello报文数量。 |
| Hello报文路由器ID冲突 | 该参数表示错误的Hello报文数量，错误类型是路由器ID冲突的Hello报文数量。 |
| 虚连接邻居未知 | 该参数表示错误的Hello报文数量，错误类型是Router ID与Vlink配置的邻居Router ID不一致。 |
| NBMA邻居未知 | 该参数表示错误的Hello报文数量，错误类型是NBMA邻居的状态不是Active时报错。 |
| 无效源 | 该参数表示错误的Hello报文数量，错误类型是无效源地址。 |
| 路由器ID | 该参数指定OSPF路由器ID。 |
| MTU值不匹配 | 该参数表示错误的DD报文数量，错误类型是使能了OSPF接口的MTU检查时，收到的DD报文的MTU值大于接收接口的MTU值。 |
| DD可选项不匹配 | 该参数表示错误的DD报文数量，错误类型是DD报文可选项不匹配。 |
| DD报文路由器ID冲突 | 该参数表示错误的DD报文数量，错误类型是DD报文路由器ID冲突。 |
| DD报文LSA类型未知 | 该参数表示错误的DD报文数量，错误类型是DD报文LSA类型未知。 |
| 错误确认 | 该参数表示收到错误的ACK报文的次数。 |
| DD数 | 该参数表示DD报文重传超次发生次数统计。 |
| LSR数 | 该参数表示Request报文重传超次发生次数统计。 |
| LSU数 | 该参数表示Update报文重传超次发生次数统计。 |
| 无效LSA数 | 该参数表示无效LSA数量。 |
| 策略过滤LSA数 | 该参数表示被策略拒绝的LSA数量。 |
| 时限无效LSA数 | 该参数表示错误周期LSA数量。 |
| 隧道开销值错误 | 该参数表示OSPF Tunnel接口的cost值小于1的次数统计。每计算一次OSPF Tunnel接口的cost值小于1时，该计数器加1，然后OSPF将此cost值计算为1。 |
| 重复确认 | 该参数表示收到重复ACK报文的次数。 |
| LSAck包LSA类型未知 | 该参数表示错误的LS ACK报文数量，错误类型是LSA类型未知。 |
| 空请求 | 该参数表示错误的LSR报文数量，错误类型是空的请求报文。 |
| 错误请求 | 该参数表示错误的LSR报文数量，错误类型是错误请求。 |
| LSA校验和错误 | 该参数表示错误的LSU报文数量，错误类型是LSA校验和错误。 |
| LSU包LSA类型错误 | 该参数表示错误的LSU报文数量，错误类型是LSU包LSA类型错误。 |
| 新收到自生成LSA | 该参数表示错误的LSU报文数量，错误类型是新收到自生成LSA。 |
| 收到超时LSA | 该参数表示错误的LSU报文数量，错误类型是收到超时LSA。 |
| 9类LSA越界 | 该参数表示超出了洪泛范围的Type-9 LSA数量。 |
| 10类LSA越界 | 该参数表示超出了洪泛范围的Type-10 LSA数量。 |
| 11类LSA越界 | 该参数表示超出了洪泛范围的Type-11 LSA数量。 |
| 越界TLV统计 | 该参数表示越界TLV统计。 |
| 邻居接口类型不匹配 | 邻居接口的网络类型不一致。 |
| 邻居状态低不能接收DD报文 | 邻居状态低不能接收DD报文。 |
| 邻居状态低不能接收LS ACK报文 | 邻居状态低不能接收LS ACK报文。 |
| 邻居状态低不能接收LS REQ报文 | 邻居状态低不能接收LS REQ报文。 |
| 邻居状态低不能接收LS UPD报文 | 邻居状态低不能接收LS UPD报文。 |
| 最小LSA接收间隔 | 该参数表示错误的LSU报文数量，错误类型是在LSA到达时间间隔内收到LSA。 |

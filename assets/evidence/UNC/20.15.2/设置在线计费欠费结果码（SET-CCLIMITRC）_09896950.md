#### [命令功能](#ZH-CN_CONCEPT_0209896950)

**适用NF：PGW-C、SMF**

该命令用于配置判断欠费用户的OCS返回码。

#### [注意事项](#ZH-CN_CONCEPT_0209896950)

- 该命令执行后立即生效。
- 该命令最大记录数为1。
- 在大规模流量统付业务中，当需要根据返回码判断用户是否欠费时，使用此命令。
- 该命令配置后，如果OCS返回的消息中携带FUI（Final-Unit-Indication）信元，即使OCS返回的响应码与本命令配置的不同，也判断为该用户欠费。
- 当前版本不支持此命令。

#### [操作用户权限](#ZH-CN_CONCEPT_0209896950)

G_1，管理员级别命令组；G_2，操作员级别命令组

配置在线计费欠费结果码：

```

```

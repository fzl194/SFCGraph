## [命令功能](#ZH-CN_MMLREF_0000001400761578)

该命令用于查询是否使能热备容灾组。

## [注意事项](#ZH-CN_MMLREF_0000001400761578)

该命令只用于在UEG-M/UEG网元采用主备（热备）容灾模式下执行。

#### [操作用户权限](#ZH-CN_MMLREF_0000001400761578)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001400761578)

无

## [使用实例](#ZH-CN_MMLREF_0000001400761578)

查询是否使能热备容灾组:

```
%%LST DRGROUPABLEMENT:;%%
RETCODE = 0  操作成功

结果如下
--------
是否使能热备容灾组  =  去使能热备容灾功能
      是否自动升主  =  否
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001400761578)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 是否使能热备容灾组 | 该参数用于是否使能热备容灾组。 |
| 是否自动升主 | 该参数用于配置在热备容灾模式首次协商持续失败场景下是否自动升为运行主状态。 |

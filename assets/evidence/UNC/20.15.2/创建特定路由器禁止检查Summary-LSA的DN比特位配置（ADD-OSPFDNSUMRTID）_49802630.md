#### [命令功能](#ZH-CN_CONCEPT_0000001549802630)

该命令用于配置特定路由器禁止对Summary LSA的DN位检查。

#### [注意事项](#ZH-CN_CONCEPT_0000001549802630)

- 该命令执行后立即生效。
- 该命令最大记录数为48000。
- 只有在执行ADD OSPF配置了OSPF进程后才能使用此命令。
- 仅支持在OSPF私网进程下配置，并且只在PE上生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549802630)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549802630)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | OSPF进程号 | 可选必选说明：必选参数<br>参数含义：OSPF进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无<br>配置原则：OSPF进程必须已经存在。请使用LST OSPF命令查看可用的OSPF进程。 |
| ROUTERID | 路由器标识 | 可选必选说明：必选参数<br>参数含义：路由器标识。<br>数据来源：全网规划<br>取值范围：IPv4地址类型。点分十进制格式。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549802630)

配置10.1.1.1的路由器禁止对Summary LSA的DN位检查：

```
ADD OSPFDNSUMRTID:PROCID=1,ROUTERID="10.1.1.1";
```

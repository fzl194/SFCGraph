## [命令功能](#ZH-CN_MMLREF_0228567650)

**适用NF：SGW-C、PGW-C**

该命令用于查询Non-IP功能相关的配置信息。

## [注意事项](#ZH-CN_MMLREF_0228567650)

无

#### [操作用户权限](#ZH-CN_MMLREF_0228567650)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0228567650)

无

## [使用实例](#ZH-CN_MMLREF_0228567650)

显示全局Non-IP功能配置：

```
%%LST NONIPFUNC:;%%
RETCODE = 0  操作成功

结果如下
--------
Non-IP功能开关  =  不使能
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0228567650)

| 输出项名称 | 输出项解释 |
| --- | --- |
| Non-IP功能开关 | 该参数用于控制开启和关闭Non-IP功能。 |

## [操作场景](#ZH-CN_OPI_0276015638)

本操作指导介绍在运行网络中激活 NB-IoT用户面传输通道保活 的操作过程。

NB-IoT终端数据收发频率低、数据包较小且多数情况为单向发送（例如智能抄表），建立、释放S1–U连接和数据无线承载信令开销大，需要一种高效的小数据传输解决方案。3GPP协议针对此类业务定义了用户面优化功能UP-CIoT（User Plane CIoT EPS Optimization）。UP-CIoT的方式是当终端进入ECM-IDLE态时，UE、eNodeB和MME保存用于恢复连接的S1-AP关联、UE上下文和承载上下文数据。当需要恢复数传时，无需进行频繁的E-RAB重建，提升数传效率，节省信令开销。

> **说明**
> 适用于MME。

## [操作步骤](#ZH-CN_OPI_0276015638)

1. 进入 “MML命令行-UNC” 窗口。
2. 配置UP-CIoT功能。
    a. 查询某指定用户范围的相关配置。
      [**LST M2MPLCY**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/M2M管理/M2M策略参数配置/查询M2M策略参数(LST M2MPLCY)_26145766.md)
    b. **可选：**如果 [步骤 2.a](#ZH-CN_OPI_0276015638__step_a) 未查询到该指定用户范围的相关配置，则执行此步骤，为该用户范围配置UP-CIoT功能。
      [**ADD M2MPLCY**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/M2M管理/M2M策略参数配置/增加M2M策略参数(ADD M2MPLCY)_72225443.md)
    c. **可选：**如果 [步骤 2.a](#ZH-CN_OPI_0276015638__step_a) 查询到系统已存在该指定用户范围的相关配置，且未配置UP-CIoT功能，则执行此步骤，为该用户范围添加UP-CIoT功能。
      [**MOD M2MPLCY**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/M2M管理/M2M策略参数配置/修改M2M策略参数(MOD M2MPLCY)_72345365.md)
3. **可选：**配置某指定TAI下eNodeB的UP-CIoT能力。
    a. 查询某指定TAI下相关配置。
      [**LST TAICIOT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/M2M管理/CIoT能力配置/查询CIoT能力配置(LST TAICIOT)_72225461.md)
    b. **可选：**如果 [步骤 3.a](#ZH-CN_OPI_0276015638__step_a1) 未查询到该指定TAI下的相关配置，则执行此步骤，为该TAI下的eNodeB配置UP-CIoT能力。
      [**ADD TAICIOT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/M2M管理/CIoT能力配置/增加CIoT能力配置(ADD TAICIOT)_26305590.md)
    c. **可选：**如果 [步骤 3.a](#ZH-CN_OPI_0276015638__step_a1) 查询到系统已存在该指定TAI下的相关配置，且不支持UP-CIoT能力，则执行此步骤，为该TAI下的eNodeB添加UP-CIoT能力。
      [**MOD TAICIOT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/M2M管理/CIoT能力配置/修改CIoT能力配置(MOD TAICIOT)_26145782.md)
4. 打开本功能的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)

## [任务示例](#ZH-CN_OPI_0276015638)

任务描述

假设某行业用户APNNI为"HUAWEI1.com"，运营商为其分配了IMSI前缀为"123037301"的号段，配置本特性。

脚本

//配置该用户的UP-CIoT能力。

```
ADD M2MPLCY: SUBRANGE=IMSI_PREFIX, IMSIPRE="123037301", APNNI="HUAWEI1.com", S1UCAPA=SUPPORT_S1UDATA_AND_UP;
```

//配置eNodeB的UP-CIoT能力。

```
ADD TAICIOT: BEGINTAI="123037301", ENDTAI="123037301", NBDATATRANSSUPPORT=SUPPORT_UP-1&SUPPORT_S1U-1;
```

//打开本功能的License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2UPTH01", SWITCH=ENABLE;
```

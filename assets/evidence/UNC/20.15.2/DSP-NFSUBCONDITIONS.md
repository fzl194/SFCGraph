## [命令功能](#ZH-CN_MMLREF_0209652567)

**适用NF：AMF、SMF、NRF、NSSF、NCG、SMSF**

该命令用于查询当前NF的所有订阅信息。

## [注意事项](#ZH-CN_MMLREF_0209652567)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209652567)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652567)

无

## [使用实例](#ZH-CN_MMLREF_0209652567)

查询当前NF的所有订阅信息。

```
%%DSP NFSUBCONDITIONS:;%%
RETCODE = 0  操作成功

结果如下
--------
      订阅标识  =  SubscribeId_1
      订阅条件  =  {Sst:460, Sd:AA}
  通知事件类型  =  {1}
       位置信息 = 10.107.240.216:81/SubSnssaiLocation
      失败次数  =  0
下次重订阅时间  = NULL
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209652567)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 订阅标识 | 该参数显示订阅标识，由NRF返回，NRF内唯一标识一条订阅记录。 |
| 订阅条件 | 该参数显示订阅条件信息。 |
| 通知事件类型 | 该参数显示通知事件类型。 |
| 位置信息 | 该参数显示NRF的位置描述信息。 |
| 失败次数 | 该参数显示向NRF连续创建订阅失败的次数。 |
| 下次重订阅时间 | 该参数显示触发重订阅的时间。 |

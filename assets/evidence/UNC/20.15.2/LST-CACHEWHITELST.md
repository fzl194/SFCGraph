## [命令功能](#ZH-CN_MMLREF_0000001317997837)

**适用NF：AMF、SMF、NSSF、SMSF、NCG**

该命令用于查询异网白名单的PLMN信息。

## [注意事项](#ZH-CN_MMLREF_0000001317997837)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001317997837)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001317997837)

无

## [使用实例](#ZH-CN_MMLREF_0000001317997837)

查询异网白名单的PLMN信息，MCC为460，MNC为03。

```
%%LST CACHEWHITELST:;%%
RETCODE = 0  操作成功

结果如下
------------------------
移动国家码  =  460
  移动网号  =  03
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001317997837)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 移动国家码 | 该参数用于指定异网缓存白名单中的移动国家码。 |
| 移动网号 | 该参数用于指定异网白名单中的移动网号。 |

#### [命令功能](#ZH-CN_CONCEPT_0000001250268010)

**适用NF：SMSF**

此命令用于查询SGS服务端链路集配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001250268010)

无

#### [本地用户权限](#ZH-CN_CONCEPT_0000001250268010)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_CONCEPT_0000001250268010)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001250268010)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| LSX | 链路集索引 | 可选必选说明：必选参数<br>参数含义：该参数用于指定链路集的索引。<br>数据来源：本端规划<br>取值范围：0~2000<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001250268010)

1. 查询关联索引为0的注册中心链路集配置，可以用如下命令：
  LST SGSRLKS: LSX=0;
  ```
  %%LST SGSRLKS: LSX=0;%%
  RETCODE = 0  操作成功。

  操作结果如下
  ------------
  链路集索引  =  0
     MME索引  =  0
    链路集名  =  noname
  (结果个数 = 1)

  ---    END
  ```

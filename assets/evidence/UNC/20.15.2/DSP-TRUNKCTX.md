#### [命令功能](#ZH-CN_CONCEPT_0000001368520854)

**适用NF：MME**

该命令用于查看MME和TSN之间的建立信息以及UE当前在TSN上的状态。

#### [注意事项](#ZH-CN_CONCEPT_0000001368520854)

无

#### [本地用户权限](#ZH-CN_CONCEPT_0000001368520854)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_CONCEPT_0000001368520854)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001368520854)

| 参数ID | 参数名称 | 参数说明 |
| --- | --- | --- |
| IMSI | IMSI | 可选必选说明：必选参数<br>参数说明：该参数用于指定国际移动用户标识。<br>取值范围：1～15位数字<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001368520854)

1. 获取宽带集群系统上下文：
  ```
  DSP TRUNKCTX: IMSI="123031601000001";
  ```
  ```
  查询结果如下
  ------------------------
   UE与TSN之间是否建立了连接  =  否
    UE与TSN之间连接的TSN标识  =  NULL
   UE与TSN之间连接的建立时间  =  2000-01-01 08:00:00+08:00
            集群终端协议版本  =  版本1
            UE当前的遥开状态  =  遥开
  (结果个数 = 1)
  ---    END
  ```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001368520854)

| 输出项名称 | 输出项解释 |
| --- | --- |
| UE与TSN之间是否建立了连接 | 结果枚举如下：<br>- 是<br>- 否 |
| UE与TSN之间连接的TSN标识 | 输出与UE连接的TSN的域名例如：TSN1.EPC.3GPPNETWORK.ORG |
| UE与TSN之间连接的建立时间 | UE与TSN之间连接的建立时间 |
| 集群用户协议版本 | 结果枚举如下：<br>- 版本1<br>- 版本2 |
| UE当前的遥开状态 | 结果枚举如下：<br>- 遥开<br>- 遥晕 |

## [命令功能](#ZH-CN_MMLREF_0000001225328709)

**适用NF：SMF**

该命令用于查询UNC作为V-SMF时的计费模式和CHF选择参数。

## [注意事项](#ZH-CN_MMLREF_0000001225328709)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001225328709)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001225328709)

无

## [使用实例](#ZH-CN_MMLREF_0000001225328709)

查询UNC作为V-SMF时的计费模式和CHF选择参数：

```
%%LST VSMFCHGCTRL:;%%
RETCODE = 0  操作成功

结果如下
--------
         计费模式  =  QBC计费
      CHF选择模式  =  Local
拜访用户主用CHF组  =  CHFGRP1
拜访用户备用CHF组  =  CHFGRP2
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001225328709)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 计费模式 | 该参数用于设置V-SMF上的计费方式。 |
| CHF选择模式 | 该参数用于设置V-SMF的CHF选择模式。 |
| 拜访用户主用CHF组 | 该参数用于设置拜访用户主CHF组的名称。 |
| 拜访用户备用CHF组 | 该参数用于设置拜访用户备用CHF组的名称。 |

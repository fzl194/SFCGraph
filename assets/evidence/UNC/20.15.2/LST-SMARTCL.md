#### [命令功能](#ZH-CN_CONCEPT_0000001327191013)

**适用网元：MME**

该命令用于查询智能分流功能参数。

#### [注意事项](#ZH-CN_CONCEPT_0000001327191013)

无。

#### [本地用户权限](#ZH-CN_CONCEPT_0000001327191013)

manage-ug;system-ug;monitor-ug

#### [网管用户权限](#ZH-CN_CONCEPT_0000001327191013)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001327191013)

无

#### [使用实例](#ZH-CN_CONCEPT_0000001327191013)

查询智能分流功能参数：

```
LST SMARTCL:;
```

```
查询结果如下
-------------------------
                 智能分流开关  =  打开
            APN智能分流关键字  =  multidomain
APN智能分流关键字定制标识位置  =  BEFORE
(结果个数 = 1)

---    END
```

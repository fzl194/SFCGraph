## [操作场景](#ZH-CN_OPI_0248043372)

本操作指导介绍在运行网络中激活IPv4v6双栈接入特性的操作过程。

UNC 支持创建IPv4v6双栈，通过IPv4v6地址为UE提供业务，使UE可以用IPv4v6地址进行数据传输。

> **说明**
> 适用于AMF、SMF。

## [必备事项](#ZH-CN_OPI_0248043372)

前提条件

- 请仔细阅读[WSFD-104002 IPv4v6双栈接入特性概述（5G）](特性概述_48043373.md)。
- 完成加载License。
- 支持IPv4v6的用户已经接入到网络。

数据

该操作无需准备数据。

## [操作步骤](#ZH-CN_OPI_0248043372)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开IPv4v6双栈接入特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)

## [任务示例](#ZH-CN_OPI_0248043372)

任务描述

开启IPv4v6双栈接入功能。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2IPDSAM01",SWITCH=ENABLE;
SET LICENSESWITCH: LICITEM="LKV2IPDSSM01",SWITCH=ENABLE;
```

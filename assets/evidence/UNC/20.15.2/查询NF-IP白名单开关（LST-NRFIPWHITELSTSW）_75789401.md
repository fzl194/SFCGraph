## [命令功能](#ZH-CN_MMLREF_0000001175789401)

**适用NF：NRF**

该命令用于查询NF IP白名单功能是否开启。

## [注意事项](#ZH-CN_MMLREF_0000001175789401)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001175789401)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001175789401)

无

## [使用实例](#ZH-CN_MMLREF_0000001175789401)

查询NF IP白名单开关，执行如下命令：

```
LST NRFIPWHITELSTSW:;
%%LST NRFIPWHITELSTSW:;%%
RETCODE = 0  操作成功

结果如下
------------------------
NF IP白名单开关  =  打开
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001175789401)

| 输出项名称 | 输出项解释 |
| --- | --- |
| NF IP白名单开关 | 该参数用于表示NF IP白名单功能是否打开。开关设置为“FUNC_ON”，NRF将基于NF的客户端IP对IP白名单内外的NF做区别处理，正常处理配置在IP白名单内NF的注册、去注册、更新及心跳业务；不在IP白名单内的NF上述业务将不能被处理。开关设置为“FUNC_OFF”，NRF不对IP白名单内外NF区别处理，正常处理所有NF的注册、去注册、更新及心跳业务。 |

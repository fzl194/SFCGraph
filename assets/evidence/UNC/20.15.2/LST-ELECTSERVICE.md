#### [命令功能](#ZH-CN_CONCEPT_0000001143286257)

该命令用于查询ETCD仲裁服务的客户端相关配置，包括ETCD仲裁的开关和ETCD灯塔特性的开关

#### [注意事项](#ZH-CN_CONCEPT_0000001143286257)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001143286257)

G_1，管理员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001143286257)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SERVICEINSTANCE | 服务实例 | 可选必选说明：必选参数<br>参数含义：该参数表示大颗粒服务实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，通过LST VNFC命令获取。<br>默认值：无<br>配置原则：只能填写通过LST VNFC命令查询到的管理代理标识。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001143286257)

查询仲裁服务打开/关闭配置：

```
LST ELECTSERVICE:
SERVICEINSTANCE="vnfc"
;
```

```
RETCODE = 0  操作成功

结果如下:
-------------------------
       仲裁服务开关配置 =  使能
       ETCD灯塔开关配置 =  使能
(结果个数 = 1)
 ---    END
```

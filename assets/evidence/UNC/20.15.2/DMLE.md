#### [命令功能](#ZH-CN_MMLREF_0000001126306094)

**适用网元：SGSN、MME**

该命令用于查询当前的Diameter本端实体信息。

#### [注意事项](#ZH-CN_MMLREF_0000001126306094)

无。

#### [本地用户权限](#ZH-CN_MMLREF_0000001126306094)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126306094)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126306094)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| LOINDEX | 本地实体索引 | 可选必选说明：可选参数<br>参数含义：该参数用于指定待查询的Diameter本地实体的索引。<br>取值范围：0～31<br>默认值：无 |

#### [使用实例](#ZH-CN_MMLREF_0000001126306094)

查询Diameter本地实体信息：

LST DMLE: LOINDEX=0;

```
%%LST DMLE: LOINDEX=0;%%
RETCODE = 0  操作成功。

操作结果如下
--------------
  本地实体索引  =  0
本地实体主机名  =  mme.epc.mnc123.mcc123.3gppnetwork.org
  本地实体域名  =  epc.mnc123.mcc123.3gppnetwork.org
        产品名  =  MME
    本地实体名  =  DMLE
(结果个数 = 1)
---    END
```

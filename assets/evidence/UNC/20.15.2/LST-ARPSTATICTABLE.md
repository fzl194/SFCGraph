#### [命令功能](#ZH-CN_CONCEPT_0000001600600701)

该命令用于查询ARP静态表项。

当设备上创建了ARP静态表项，可用该命令查询已创建的ARP静态表项信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600600701)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600600701)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600600701)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| VRFNAME | VPN实例名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定ARP表项的VPN名称。<br>数据来源：本端规划<br>取值范围：字符串类型，取值范围是1～31。<br>默认值：无 |
| IPADDR | IP地址 | 可选必选说明：可选参数<br>参数含义：该参数用于显示指定（目的）IPv4地址的ARP表项。<br>数据来源：本端规划<br>取值范围：IPv4地址类型。IP地址0.0.0.0～255.255.255.255。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600600701)

不指定参数时，查询VNFC上所有ARP静态表项：

```
LST ARPSTATICTABLE:;
```

```

RETCODE = 0  操作成功。

结果如下
--------
     IP地址  =  10.1.1.2
  VLAN ID值  =  0
    MAC地址  =  00E0-FC12-3456
VPN实例名称  =  _public_
   接口名称  =  Ethernet64/0/3
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600600701)

| 输出项名称 | 输出项解释 |
| --- | --- |

| VLAN ID值 | 该参数表示VLAN值。 |
| MAC地址 | 该参数表示MAC地址。 |
| VPN实例名称 | 该参数表示VPN实例名称。 |
| 接口名称 | 该参数表示接口名称。 |

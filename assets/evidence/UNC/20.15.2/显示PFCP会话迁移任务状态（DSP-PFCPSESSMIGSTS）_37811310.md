## [命令功能](#ZH-CN_MMLREF_0000001537811310)

**适用NF：SGW-C、PGW-C、SMF、GGSN**

该命令用于查询PFCP会话迁移任务状态。

## [注意事项](#ZH-CN_MMLREF_0000001537811310)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001537811310)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001537811310)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| MIGTYPE | 迁移类型 | 可选必选说明：可选参数<br>参数含义：该参数用于指定迁移类型。<br>数据来源：本端规划<br>取值范围：<br>- UPF（指定UPF）<br>- IMSISEG（指定IMSI号段）<br>默认值：UPF<br>配置原则：无 |
| NFINSTANCENAME | UPF实例名称 | 可选必选说明：该参数在"MIGTYPE"配置为"UPF"时为条件可选参数。<br>参数含义：该参数用于指定UPF实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~36。不区分大小写。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001537811310)

- 当希望查询upf1的搬迁状态时，使用如下命令：
  ```
  %%DSP PFCPSESSMIGSTS:MIGTYPE=UPF,NFINSTANCENAME="upf";%%
  RETCODE = 0  操作成功

  结果如下
  ------------------------
         执行阶段  =  当前迁移任务待启动
      迁移进度(%)  =  0
   剩余迁移会话数  =  0
     迁移开始时间  =  NULL
     迁移结束时间  =  NULL
       迁移源路径  =  192.168.0.1
     迁移目的路径  =  192.168.0.2
     待启动的列表  =  NULL
       完成的列表  =  NULL
       运行的列表  =  NULL
       停止的列表  =  NULL
  (结果个数 = 1)

  ---    END
  ```
- 当希望查询全部upf的搬迁状态时，使用如下命令：
  ```
  %%DSP PFCPSESSMIGSTS:MIGTYPE=UPF;%%
  RETCODE = 0  操作成功

  结果如下
  ------------------------
         执行阶段  =  当前迁移任务待启动
      迁移进度(%)  =  0
   剩余迁移会话数  =  0
     迁移开始时间  =  NULL
     迁移结束时间  =  NULL
       迁移源路径  =  0.0.0.0
     迁移目的路径  =  NULL
     待启动的列表  =  upf1, upf2
       完成的列表  =  NULL
       运行的列表  =  NULL
       停止的列表  =  NULL
  (结果个数 = 1)

  ---    END
  ```

## [输出结果说明](#ZH-CN_MMLREF_0000001537811310)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 执行阶段 | 该参数用于指定迁移命令执行阶段。<br>取值说明：<br>- Invalid（当前未执行相关批量迁移命令）<br>- Wait（当前迁移任务待启动）<br>- Migrating（当前正在执行迁移任务）<br>- Stop（当前执行的迁移任务被强制终止）<br>- Finish（当前执行迁移任务已完成）<br>- TimeoutStop（当前执行的迁移任务超时终止） |
| 迁移进度(%) | 该参数用于表示当前迁移任务的进度。 |
| 剩余迁移会话数 | 该参数用于表示当前迁移任务的剩余迁移会话数。 |
| 迁移开始时间 | 该参数用于表示当前迁移任务的开始时间。 |
| 迁移结束时间 | 该参数用于表示当前迁移任务的结束时间。 |
| 迁移源路径 | 该参数用于表示当前迁移任务源路径的对端IP地址。<br>在查询单个UPF迁移状态时显示。 |
| 迁移目的路径 | 该参数用于表示当前迁移任务目的路径的对端IP地址。<br>在查询单个UPF迁移状态时显示。 |
| 待启动的列表 | 该参数用于表示待启动迁移任务的列表。<br>在查询整体迁移状态时显示。 |
| 完成的列表 | 该参数用于表示完成迁移任务的列表。<br>在查询整体迁移状态时显示。 |
| 运行的列表 | 该参数用于表示正在运行的迁移任务的列表。<br>在查询整体迁移状态时显示。 |
| 停止的列表 | 该参数用于表示迁移任务异常停止的列表。<br>在查询整体迁移状态时显示。 |

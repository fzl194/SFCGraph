## [操作场景](#ZH-CN_OPI_0243355961)

本操作指导介绍在激活支持EDGE功能（仅用于Gb 模式）的操作过程。

EDGE（Enhanced Data Rate for GSM Evolution）是一种能够进一步提高移动数据业务传输速率和从GSM向3G过渡中的重要技术，以其数据服务速率高、能节约网络投资、可以充分满足未来无线多媒体应用的带宽需求等特点，成为接近于第三代移动通信系统的一项数据通信技术。

> **说明**
> 适用于 SGSN 。

## [必备事项](#ZH-CN_OPI_0243355961)

前提条件

- 请仔细阅读[WSFD-106002 支持EDGE功能（仅用于Gb 模式）特性概述](特性概述_85005563.md)。
- 完成加载License。

数据

该操作无需准备数据。

## [操作步骤](#ZH-CN_OPI_0243355961)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)

## [任务示例](#ZH-CN_OPI_0243355961)

任务描述

激活支持EDGE功能（仅用于Gb 模式）。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2EDGE02", SWITCH=ENABLE;
```

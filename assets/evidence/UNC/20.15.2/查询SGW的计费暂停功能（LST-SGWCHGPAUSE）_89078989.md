## [命令功能](#ZH-CN_MMLREF_0000001189078989)

**适用NF：SGW-C**

该命令用于查询SGW计费暂停功能的参数配置。

## [注意事项](#ZH-CN_MMLREF_0000001189078989)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001189078989)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001189078989)

无

## [使用实例](#ZH-CN_MMLREF_0000001189078989)

查询SGW中计费暂停功能参数配置：

```
%%LST SGWCHGPAUSE:;%%
RETCODE = 0  操作成功

结果如下
------------------------
 S1 Release触发计费暂停开关  =  DISABLE
 DDN寻呼失败触发计费暂停开关  =  DISABLE
下行报文阈值触发计费暂停开关  =  DISABLE
               下行报文阈值  =  500
  计费暂停时下行报文处理动作  =  buffer
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001189078989)

| 输出项名称 | 输出项解释 |
| --- | --- |
| S1 Release触发计费暂停开关 | 该参数用于表示原因值为Abnormal Release of Radio Link的S1 Release流程触发计费暂停功能开关。 |
| DDN寻呼失败触发计费暂停开关 | 该参数用于表示DDN寻呼失败触发计费暂停功能开关。 |
| 下行报文阈值触发计费暂停开关 | 该参数用于表示下行报文阈值触发计费暂停功能开关。 |
| 下行报文丢包阈值 | 该参数用于控制IDLE状态用户触发计费暂停功能的下行报文个数阈值。 |
| 计费暂停时下行报文处理动作 | 该参数用于控制开始计费暂停功能后下行报文处理动作。 |

#### [适用NF](#ZH-CN_CONCEPT_0168358176)

SGSN、 MME

#### [定义](#ZH-CN_CONCEPT_0168358176)

用户接入控制功能是指 UNC 支持先根据IMSI号段将用户进行分类，再对于每一类用户按照用户属性进行区分而控制用户接入 GERAN、 UTRAN和 E-UTRAN网络。 UNC 按以下三种方式划分用户属性并实施用户接入控制：

- 根据签约的ARD（Access Restriction Data）控制用户接入
- 根据签约的APN控制用户接入
- 根据卡类型控制用户接入（只适用于GERAN/UTRAN接入用户）

#### [客户价值](#ZH-CN_CONCEPT_0168358176)

| 受益方 | 受益描述 |
| --- | --- |
| 运营商 | 根据用户属性，灵活的设置接入控制策略，方便管理网络资源。 |
| 用户 | 用户不感知该特性。 |

#### [应用场景](#ZH-CN_CONCEPT_0168358176)

- 若运营商希望拒绝某个号段接入GERAN、UTRAN或E-UTRAN，则可以启用根据签约ARD控制用户接入特性。
- 若运营商希望控制某类用户是否允许接入GERAN、UTRAN或E-UTRAN，但HLR/HSS不支持ARD时，则可以启用根据签约APN控制用户接入特性。
- 若运营商已经强制要求对GERAN用户使用SIM卡，UTRAN用户使用USIM卡，并希望对它们进行灵活的接入控制，就需要启用按SIM卡/USIM卡控制用户接入特性。

**根据签约的ARD控制用户接入功能原理**

ARD：接入限制数据。3GPP R6协议中在HLR/HSS签约数据中加入了ARD信息，该信息中指示了允许用户使用的RAT（Radio Access Technologies），用于表明是否拒绝用户接入GERAN、UTRAN或E-UTRAN。

ARD信息在HLR/HSS和SGSN/MME之间通过Insert Subscriber Data Request消息或Update Location Answer消息中的AccessRestrictionData信元传送。AccessRestrictionData信元包括bit0~bit7八个bit位，bit0、bit1、bit4分别代表了是否允许用户接入UTRAN、GERAN或E-UTRAN，bit6代表是否允许NB-IoT用户接入网络。其中bit位取值为1表示拒绝接入，为0表示允许接入。

> **说明**
> 如果 UNC 支持ARD特性，但是没有从HLR/HSS收到RAT消息， UNC 将假定用户签约时不需要限制RAT。

若 UNC 启动了支持ARD功能，它就可以在接收到Insert Subscriber Data Request消息或Update Location Answer消息中的ARD信息后，结合当前用户请求登录的无线接入系统，来判断是允许还是拒绝该用户的接入请求。如果拒绝该用户的接入请求， UNC 根据命令 [**ADD GBARD**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/接入限制管理/Gb模式接入限制参数/增加Gb模式接入限制参数(ADD GBARD)_26305284.md) 、 [**ADD IUARD**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/接入限制管理/Iu模式接入限制参数/增加Iu模式接入限制参数(ADD IUARD)_72345073.md) 或 [**ADD S1ARD**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/接入限制管理/S1模式接入限制参数/增加S1模式接入限制参数(ADD S1ARD)_26145478.md) 下的参数 “原因值” 的设置，返回给用户合适的拒绝原因值。

**根据签约的APN控制用户接入功能原理**

在HLR/HSS中为用户签约一个特殊的APNNI，此APNNI用来控制用户是否允许接入GERAN、UTRAN或E-UTRAN。用户附着时 UNC 可以据此来判断用户登录的无线接入系统是GERAN、UTRAN还是E-UTRAN，从而进行接入控制。

**根据卡类型控制用户接入功能原理（只适用于GERAN/UTRAN接入用户）**

UNC 通过分析用户签约数据中的鉴权信息是三元组还是五元组来确定是SIM卡还是USIM卡，从而区分GERAN用户和UTRAN用户，实现接入控制。

> **说明**
> - SIM（Subscriber Identity Module）卡是用户识别模块，也称为智能卡、用户身份识别卡，卡中包括了用户接入所注册的网络的相关信息，用于进行用户身份识别。SIM卡在GSM网络/UMTS网络均可以使用。
> - USIM（UMTS Subscriber Identity Module）卡是UMTS用户识别模块。USIM卡除了包括用户接入所注册的网络的相关信息之外，还在安全性方面对算法进行了升级，并增加了卡对网络的认证功能，这种双向认证可以有效防止黑客对卡片的攻击。USIM卡在GSM网络/UMTS网络/LTE网络均可以使用。

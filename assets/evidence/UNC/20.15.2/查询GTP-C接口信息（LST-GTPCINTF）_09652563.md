## [命令功能](#ZH-CN_MMLREF_0209652563)

**适用NF：SGW-C、PGW-C、AMF、GGSN**

该命令用于查询GTP-C接口信息。

## [注意事项](#ZH-CN_MMLREF_0209652563)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209652563)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652563)

无

## [使用实例](#ZH-CN_MMLREF_0209652563)

查询GTP-C接口信息。

```
%%LST GTPCINTF:;%%
RETCODE = 0  操作成功

结果如下
--------
接口类型     组号  

S11接口      0     
N26接口      3     
S5-S接口     1     
S5-P/GN接口  2     
(结果个数 = 4)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209652563)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 接口类型 | 该参数用于指定GTP-C接口类型。 |
| 组号 | 该参数用于指定GTP-C本地实体的组号。 |

#### [命令功能](#ZH-CN_MMLREF_0000001172225585)

**适用网元：SGSN**

此命令用于查询配置的手工恢复GGSN地址。

#### [注意事项](#ZH-CN_MMLREF_0000001172225585)

无。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172225585)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172225585)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172225585)

无。

#### [使用实例](#ZH-CN_MMLREF_0000001172225585)

查询配置的手工恢复GGSN地址列表：

LST USRCTLGGSN:;

```
%%LST USRCTLGGSN:;%%
RETCODE = 0  操作成功。

输出结果如下
----------------
 IP地址类型  =  IPv4
   IPv4地址  =  192.168.123.123
       描述  =  123
(结果个数 = 1)

---    END
```

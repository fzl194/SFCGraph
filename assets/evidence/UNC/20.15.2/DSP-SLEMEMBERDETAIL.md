#### [命令功能](#ZH-CN_CONCEPT_0000001181883493)

该命令用于显示仲裁成员详细信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001181883493)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001181883493)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001181883493)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RUNAME | RU名称 | 可选必选说明：必选参数<br>参数含义：该参数用于显示RU名称。通过<br>[**DSP SLEMEMBER**](显示仲裁成员列表信息（DSP SLEMEMBER）_35678578.md)<br>命令可以查询所有仲裁成员对应的资源单元信息。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。<br>默认值：无<br>配置原则：<br>- 只能填写实际存在的资源单元或资源名称。<br>- 当本命令在VNFP上使用时，需要先使用[**DSP RES**](../../../../单体服务平台功能管理/系统管理/资源管理/资源实例管理/显示资源信息（DSP RES）_59036939.md)查到“资源名称”，然后将“资源名称”的取值配置到本参数。<br>- 当本命令在VNFC上使用时，需要先使用[**DSP RU**](显示资源单元信息（DSP RU）_59103857.md)查到“RU名称”，然后将“RU名称”的取值配置到本参数。 |
| SERVICEINSTANCE | 服务实例 | 可选必选说明：必选参数<br>参数含义：该参数表示大颗粒服务实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，通过LST VNFC命令获取。<br>默认值：无<br>配置原则：只能填写通过LST VNFC命令查询到的管理代理标识。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001181883493)

显示仲裁成员详细信息：

```
DSP SLEMEMBERDETAIL:RUNAME="VNODE_VNRS_VNFC_OMU_0001"
,SERVICEINSTANCE="vnfc"
;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001181883493)

| 输出项名称 | 输出项解释 |
| --- | --- |
| RU名称 | 用于显示RU名称。通过<br>[**DSP SLEMEMBER**](显示仲裁成员列表信息（DSP SLEMEMBER）_35678578.md)<br>命令可以查询所有仲裁成员对应的资源单元信息。 |
| 是否候选者 | 该参数表示是否资源单元主节点候选者。<br>- TRUE：是资源单元主节点候选者。<br>- FALSE：不是资源单元主节点候选者。 |
| 角色 | 该参数表示主从角色。<br>- 未知：表示当前节点尚未决策出主从角色。<br>- 主节点：表示当前节点角色为主。<br>- 从节点：表示当前节点角色为备。 |
| 工作模式 | 该参数表示工作模式。<br>- 非防脑裂：表示当前节点工作在非防脑裂模式，即OMU节点自己决策主备角色的模式。<br>- 防脑裂：表示当前节点工作在防脑裂模式，即由若干成员节点（通常5个节点）投票决策主备节点的模式。<br>- 未知：表示当前节点尚未参与仲裁的模式。 |
| 区域ID | 表示区域ID。 |
| 主节点ID | 表示主节点ID。 |
| 仲裁区域节点总数 | 表示仲裁区域节点总数。 |
| 心跳连接数 | 表示心跳连接数。 |
| 通讯方式 | 该参数表示通讯方式。<br>- MAC模式。<br>- IP模式。 |
| 消息队列出错数 | 表示消息队列出错数。 |
| 申请内存出错数 | 表示申请内存出错数。 |
| 消息发送出错数 | 表示消息发送出错数。 |
| 消息接收出错数 | 表示消息接收出错数。 |
| 仲裁启动时间 | 表示仲裁启动时间。 |
| 软仲裁选主容忍时间（ms） | 表示仲裁选主容忍时间。 |
| 软仲裁停主容忍时间（ms） | 表示仲裁停主容忍时间。 |

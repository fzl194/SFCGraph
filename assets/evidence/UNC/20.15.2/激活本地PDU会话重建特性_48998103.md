## [操作场景](#ZH-CN_OPI_0248998103)

本操作指导介绍在运行网络中激活本地PDU会话重建操作过程。AMF和SMF上分别实现了本地PDU会话重建功能，AMF和SMF上完成一种配置即可实现此特性的激活任务，如果AMF上和SMF上都进行了配置，系统默认使用SMF的实现。

> **说明**
> 适用于AMF、SMF。

## [必备事项](#ZH-CN_OPI_0248998103)

前提条件

- AMF侧使用NF Instance ID识别非本地SMF的PDU会话连接方式，运营商网络中NF Instance ID的规划满足以下条件：
    - NF Instance ID命名规则采用通用唯一标识符(UUID) 4版本格式。
    - NF Instance ID的node部分使用固定的字符表示指定的区域。
- 请仔细阅读[WSFD-205011 本地PDU会话重建特性概述](特性概述_48998101.md)。
- 完成加载License。

## [操作步骤](#ZH-CN_OPI_0248998103)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. 配置PDU会话重选策略。
  > **说明**
  > - [3.a](#ZH-CN_OPI_0248998103__substep20930005426)和[3.b](#ZH-CN_OPI_0248998103__substep083405255216)只能配置其中一种方式。
  > - [3.a](#ZH-CN_OPI_0248998103__substep20930005426)、[3.b](#ZH-CN_OPI_0248998103__substep083405255216)和[3.c](#ZH-CN_OPI_0248998103__cmd240064164218)分别为AMF和SMF侧的实现，配置时根据需要AMF侧和SMF侧选择一侧配置即可。如果AMF和SMF都进行了配置，系统默认使用SMF的实现。
    a. **可选：**AMF侧通过比较AMF和SMF的NF Instance ID方式配置PDU会话重建。
      **[ADD SMFRESEL](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/移动性管理/NAS传输管理/本地SMF的PDU会话重建功能管理/增加本地SMF重选策略（ADD SMFRESEL）_45495622.md)**
    b. **可选：**AMF侧通过判断I-SMF插入/修改方式配置PDU会话重建。
      **[ADD SMFRESEL](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/移动性管理/NAS传输管理/本地SMF的PDU会话重建功能管理/增加本地SMF重选策略（ADD SMFRESEL）_45495622.md)**
    c. **可选：**SMF侧配置PDU会话重建。
      **[ADD SMFPDUREACT](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/跨区域PDU会话管理/跨区域PDU会话重建/增加跨区域PDU会话重建策略（ADD SMFPDUREACT）_64343866.md)**
4. 配置AMF重选时选择同区域SMF。
  > **说明**
  > 以下场景至少选择一种，达到AMF重选选择同区域SMF的目的。
    a. **可选：**配置AMF根据UE位置选择SMF。
      [**ADD SMFSELPLCY**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/移动性管理/NF发现和选择管理/SMF选择策略管理/增加SMF选择策略（ADD SMFSELPLCY）_09653765.md)
    b. **可选：**配置AMF根据位置优选SMF。
      **[ADD NFLOC](../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/服务化接口管理/注册与服务发现/NF优选区域管理/增加目标NF发现和选择的位置匹配信息（ADD NFLOC）_09652578.md)**
5. **可选：**当现网不存在NRF时 ，可以在AMF上直接配置同区域SMF的信息。
  **[ADD PNFPROFILE](../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/服务化接口管理/注册与服务发现/本地NRF功能管理/对端NF域名信息管理_34319795.md)**

## [任务示例](#ZH-CN_OPI_0248998103)

任务描述

任务1：AMF侧使用比较AMF和SMF的NF Instance ID方式判断本地PDU会话重建。UE移动后，处于CM-IDLE态且满足PDU会话重建条件之后，AMF要求UE发起PDU会话重建，选择本地SMF。

PDU会话重建条件：UE处于CM-IDLE态达到60秒，且时间在00:02:00~00:04:00之间。

任务2：SMF侧启用本地PDU会话重建。UE移动后，只存在缺省QoSFlow时，SMF要求UE发起PDU会话重建，择本地SMF。

脚本

任务1：

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2AMFLS01", SWITCH=ENABLE;
```

//AMF侧启用PDU会话重建。

```
ADD SMFRESEL:DNN="IMS",ISNSSAI=NO,RESELCOND=NFINSTANCEID,NBEGIN=5,NEND=6,CMIDLETIME=60,ST=00:02:00,ET=00:04:00;
```

//配置AMF基于UE当前TAI选择同区域的SMF。

```
ADD SMFSELPLCY:SUBRANGE=ALL_USER, TAISW=YES;
```

任务2：

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2SMFLS01", SWITCH=ENABLE;
```

//SMF侧启用PDU会话重建。

```
ADD SMFPDUREACT:DNN="IMS",ISNSSAI=NO,NBEGIN=5,NEND=6;
```

//SMF侧启用国际漫游场景下的PDU会话重建，执行SET SMFSOFTPARAOFBIT命令，设置 **DWORD1059 BIT29** 软参值为1 **，** 打开本地PDU会话重建开关。

```
SET SMFSOFTPARAOFBIT:DT=Dw,DWORDNUM=1059,VALUE=VALUE_1,POSITION=29;
```

//配置AMF基于UE当前TAI选择同区域的SMF。

```
ADD SMFSELPLCY:SUBRANGE=ALL_USER, TAISW=YES;
```

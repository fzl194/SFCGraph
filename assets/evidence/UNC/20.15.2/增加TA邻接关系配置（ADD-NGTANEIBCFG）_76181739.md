## [命令功能](#ZH-CN_MMLREF_0276181739)

**适用NF：AMF**

此命令用于手动添加TA的邻接关系。当激活“精准寻呼”特性时，整网所有TA邻接关系的学习是一个漫长的过程，通过此命令可以避免等待系统自动学习邻接关系的大量时间消耗。

## [注意事项](#ZH-CN_MMLREF_0276181739)

- 该命令执行后立即生效。

- 此命令仅用于测试、演示目的。在特性使用时，TA邻接关系通过移动性注册更新流程学习获得，而不能通过此命令添加。
- 通过此命令配置的TA邻接关系不会自动老化，若想删除TA邻接关系，请执行RMV命令。
- 该命令仅能添加中心TA与邻接TA列表中各TA的单向邻接关系，使用该命令建立TA邻接关系后，还需使用该命令建立邻接TA列表中各个TA与中心TA的邻接关系。

- 最多可输入10条记录。

#### [操作用户权限](#ZH-CN_MMLREF_0276181739)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0276181739)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| TAI | TA标识 | 可选必选说明：必选参数<br>参数含义：该参数用于指定中心跟踪区域的标识。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是11~12。TAI由MCC、MNC和TAC组成。MCC为3位十进制数字，MNC为2位或者3位十进制数字，填写时请遵循实际长度。TAC编码为十六进制数，长度固定为6位；若不足则左起用0补足6位。<br>默认值：无<br>配置原则：无 |
| NEIBTALIST | 邻接TA列表 | 可选必选说明：必选参数<br>参数含义：该参数用于指定邻接TA列表，建立中心TA及其周边TA的邻接关系。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是11~207。<br>默认值：无<br>配置原则：<br>每个中心TA的邻接TA最大个数为16个。<br>TAI之间有且只有一个“&”用以分隔。<br>每条记录内的邻接TA标识不能与该记录的中心TA标识相同。<br>每条记录内的邻接TA标识不能重复。<br>每条记录内的邻接TA标识位数必须与该记录的中心TA标识位数相同。 |
| DESC | 规则描述 | 可选必选说明：可选参数<br>参数含义：该参数用于指定TA邻接关系的描述信息。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~32。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0276181739)

在测试邻接TA范围的精准寻呼特性时，手动添加一条TA邻接关系，中心TA的标识TAI为"12345100010"，其邻接TA的标识分别为"12345100011","12345100012"，执行如下命令：

```
ADD NGTANEIBCFG: TAI="12345100010",NEIBTALIST="12345100011&12345100012";
```

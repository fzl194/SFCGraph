## [任务示例](#ZH-CN_OPI_0000001299103732)

任务描述

任务1：开启端到端用户跟踪特性，并设置系统支持对SGs、Sv接口消息的跟踪及上报。

任务2：手动添加端到端跟踪任务。

脚本

任务1：开启端到端用户跟踪特性，并设置系统支持对SGs、Sv接口消息的跟踪及上报。

```
//打开License配置开关。
SET LICENSESWITCH: LICITEM="LKV2ESTAM01", SWITCH=ENABLE;
SET LICENSESWITCH: LICITEM="LKV3ESTSM02", SWITCH=ENABLE;
```

```
//打开SGs、Sv接口消息跟踪控制开关。
SET E2ETRCCFG: SGS=YES, SV=YES;   
```

任务2：手动增加端到端跟踪任务，配置IMSI为460030912121001，TRCID为12345-000064，eNodeB配置为上报，接口使用S1-MME，跟踪深度为1，跟踪任务有效时长为1天。

```
ADD E2ETRCTSK: IMSI="460030912121001", TRCID="12345-000064", ENB=RPT, ENBINTERFACE=S1MME-1, ENBTRCDPTH=1,TRCVALIDTM=1;
```

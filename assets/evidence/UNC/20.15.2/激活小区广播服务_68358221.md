## [操作场景](#ZH-CN_OPI_0168358221)

本操作指导介绍在运行网络中激活小区广播服务功能特性的操作过程。

小区广播服务是通过移动网络及时广播公共告警信息给终端用户，用于避险，如地震、海啸预警等。告警信息来自政府部门的告警发布部门，告警消息由政府部门发送到核心网的小区广播中心（Cell Broadcast Center， CBC）或小区广播中心功能（Cell Broadcast Center Function，CBCF）， UNC 作为MME或AMF实体支持小区广播服务，接收来自小区广播中心下发的告警信息，并广播给指定的eNodeB或gNodeB。

## [必备事项](#ZH-CN_OPI_0168358221)

前提条件

- 请仔细阅读[WSFD-106201 小区广播服务特性概述](特性概述_68358218.md)。
- 已完成加载License，对应的License项为“82207709 LKV2CBS02 小区广播服务”和**“82209076 LKV2PWSR01 PWS故障快速恢复”**。
- EPC场景请先完成SBc的接口数据配置，请参见[配置到CBC的数据](../../../../../初始配置/UNC初始配置与调测/组网对接配置/配置AMF&MME&SGSN/配置MME/配置到CBC的数据_82626736.md)。
- 5GC场景请先完成N50接口数据配置，请参见[SBI接口配置](../../../../../初始配置/UNC初始配置与调测/组网对接配置/配置AMF&MME&SGSN/配置AMF/配置SBI接口_82467016.md)。

数据

该操作无需准备数据。

## [操作步骤](#ZH-CN_OPI_0168358221)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. **可选：**EPC场景需要配置EMM information消息的下发策略。
  [**SET MMFUNC**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/MM扩展功能管理/设置移动性管理扩展功能(SET MMFUNC)_26145512.md)
  > **说明**
  > 小区广播服务功能需要MME在周期性TAU等流程中通过EMM Information消息下发当前网络时间给UE，确保UE和网络中的时间保持一致。
4. **可选：**EPC场景配置SBc接口大包处理功能。
  **[SET SOFTPARAOFBIT](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/操作维护/软件参数管理/软件参数比特位/设置软件参数表比特位(SET SOFTPARAOFBIT)_72345783.md)**
  [**SET SCTPBUFFERMODE**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/SCTP管理/设置SCTP缓冲区模式(SET SCTPBUFFERMODE)_04252670.md)
  **[SET SBCFUNC](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/SBc接口管理/SBc参数/设置小区广播功能(SET SBCFUNC)_26306190.md)**
  **[SET BIGSCTPRXNUM](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/SCTP管理/设置大端模式SCTP接收缓冲区参数(SET BIGSCTPRXNUM)_26238016.md)**
  > **说明**
  > 支持处理大包功能需要：
  >
  > - 开启**[SET SBCFUNC](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/SBc接口管理/SBc参数/设置小区广播功能(SET SBCFUNC)_26306190.md)**中的“大包处理功能开关”。
  > - 执行[**SET BIGSCTRRXNUM**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/SCTP管理/设置大端模式SCTP接收缓冲区参数(SET BIGSCTPRXNUM)_26238016.md)命令配合使用，该命令执行后需要重启SGP进程才能生效，需要配合设置SCTP缓冲区模式[**SET SCTPBUFFERMODE**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/信令传输管理/SCTP管理/设置SCTP缓冲区模式(SET SCTPBUFFERMODE)_04252670.md)使用。
  > - 将软参[BYTE_EX_B338 BIT2](../../../../../../OM参考/软件参数/UNC软件参数/业务软件参数/软件参数（SGSN_MME）/参数说明/S1接口管理/BYTE_EX_B338 BIT2 控制MME在给eNodeB发送包含E-CGI的WRITE-REPL_fa1d6467_05451121.md)设置为1，仅支持过滤S1接口上的E-CGI，不支持E-CGI以外的TAI（Tracking Area ID）和EAI（Emergency Area ID）过滤。
  >
  > 无法解析MME和CBC之间的消息跟踪（SBc接口跟踪）超过20K字节的部分。
5. **可选：**5GC场景需要设置小区广播服务功能参数。
  **[SET NGCBSFUNC](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/移动性管理/5G小区广播管理/设置小区广播功能（SET NGCBSFUNC）_51856024.md)**
  建议AMF将基站响应的预警结果仅发送给下发预警任务的CBCF，参数 “NOTIFYPLCY” 设置为 “SPECIAL_CBCF(特定的CBCF)” 。

## [任务示例](#ZH-CN_OPI_0168358221)

任务描述

打开小区广播服务开关，用于发送小区广播服务消息。

脚本

//打开本特性的License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2CBS02", SWITCH=ENABLE;
SET LICENSESWITCH: LICITEM="LKV2PWSR01", SWITCH=ENABLE;
```

//（可选）EPC场景需要配置EMM information消息的下发策略。

```
SET MMFUNC: MMINFO=S1_MODE-1,EMMINFOFIRST=S1_MODE-1,EMMINFOSENDPLY=PERIOD_TAU-1;
```

//（可选）EPC场景配置SBc接口大包处理功能。

```
SET SOFTPARAOFBIT: DT=BYTE_EX_B, PARANUM=338, VALUE=VALUE_1, POSITION=2;
SET SCTPBUFFERMODE: RXMODE=SHARE;
SET SBCFUNC: SWITCH=ON, DISTIME=10, ENBNUM=32;
SET BIGSCTPRXNUM:  BIGSHAMAXBFNUM=372;
```

//（可选）5GC场景需要打开AMF向CBCF反馈功能开关。

```
SET NGCBSFUNC: INDPWS=YES, NOTIFYPLCY=SPECIAL_CBCF;
```

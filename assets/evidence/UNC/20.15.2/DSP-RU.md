#### [命令功能](#ZH-CN_CONCEPT_0259103857)

该命令用于显示VNFC上的资源块。

#### [注意事项](#ZH-CN_CONCEPT_0259103857)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0259103857)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0259103857)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RUNAME | RU名称 | 可选必选说明：可选参数<br>参数含义：该参数表示资源单元名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。区分大小写。<br>默认值：无<br>配置原则：只能填写实际存在的资源单元。 |
| SERVICEINSTANCE | 服务实例 | 可选必选说明：必选参数<br>参数含义：该参数表示大颗粒服务实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，通过LST VNFC命令获取。<br>默认值：无<br>配置原则：只能填写通过LST VNFC命令查询到的管理代理标识，但不能填写0，0表示VNFP。 |

#### [输出结果说明](#ZH-CN_CONCEPT_0259103857)

| 输出项名称 | 输出项解释 |
| --- | --- |
| RU编号 | 表示资源单元编号。 |
| RU名称 | 表示资源单元名称。 |
| RU类型 | 表示资源单元类型名称。 |
| HA角色 | 该参数表示HA角色。<br>- 无：表示该资源单元无HA角色。<br>- 主：表示该资源单元HA角色为主。<br>- 备：表示该资源单元HA角色为备。<br>- 冷备：表示该资源单元HA角色为冷备。 |
| 虚拟机名称 | 表示虚拟机名称。<br>说明：Full-stack裸机场景下，虚拟机名称为NULL。 |
| 运行状态 | 该参数表示资源单元的运行状态。<br>- 正常：资源单元运行状态正常。<br>- 不正常：资源单元运行状态异常。<br>- 离线：资源单元未激活。<br>- 停止：资源单元运行状态停止。 |
| 注册状态 | 该参数表示注册状态。<br>- 注册态：注册状态是注册态。<br>- 未注册态：注册状态是未注册态。 |
| 内存资源（MB） | 表示内存资源（MB）。表示该资源单元理论上可使用的最大内存资源。各个资源单元的内存使用过程中会动态变化，内存使用具体值是最大内存*内存占用率，节点内各资源单元共同使用整个节点的内存资源，实际使用的内存大小总和不会超过节点的总内存。 |
| 存储资源（GB） | 表示存储资源（GB）。 |
| CPU资源 | 表示CPU资源。 |
| 内存利用率（%） | 表示内存利用率。 |
| CPU利用率（%） | 表示系统CPU利用率。 |
| HA组 | 表示资源单元的HA组。在同一HA组的资源单元需要部署在不同的物理机上。显示为NULL时表示没有指定HA组，不受该限制。OMU天然具有反亲和属性，需要部署在不同的物理机上。 |
| 亲和性组 | 表示资源单元的亲和性组。如果两个资源单元在同一个亲和性组里，则这两个资源单元会尽量部署在同一个物理机上。显示为NULL表示没有指定亲和性组，资源单元部署时不受该限制。 |
| 扩展组 | 表示资源单元的扩展组。每个扩展组会独立判断Scale out/in（扩容/减容）的条件是否满足。 |
| 主机位置 | 表示主机位置。 |
| RU扩展信息 | 表示RU扩展信息。 |
| 主机名称 | 表示主机名称。 |
| 位置信息 | 表示位置信息。 |
| 父资源名称 | 表示父资源名称。 |
| CPU架构类型 | 该参数表示资源单元的CPU架构类型。<br>- X86：CPU架构类型是X86。<br>- ARM：CPU架构类型是ARM。 |

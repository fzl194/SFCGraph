## [命令功能](#ZH-CN_MMLREF_0000001404121609)

**适用NF：SMSF**

该命令用于查询VLR的SGs接口自保流控参数。

## [注意事项](#ZH-CN_MMLREF_0000001404121609)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001404121609)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001404121609)

无

## [使用实例](#ZH-CN_MMLREF_0000001404121609)

运营商希望查询VLR的SGs接口自保流控参数，执行如下命令：

```
LST VLRSGSSELFFC:;
%%LST VLRSGSSELFFC:;%%
RETCODE = 0  操作成功

结果如下：
------------------------
 流控开关 =  开启
位置更新流控响应 =  拥塞拒绝
MO流控响应 =  临时不可达
(结果个数 = 1)
```

## [输出结果说明](#ZH-CN_MMLREF_0000001404121609)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 流控开关 | 该参数用于表示VLR SGs接口消息流控开关状态。 |
| 位置更新流控响应 | 该参数用于表示系统过载时在位置更新流程中，VLR收到SGs接口的SGsAP-LOCATION-UPDATE-REQUEST消息的响应原因值。 |
| MO流控响应 | 该参数用于表示系统过载时在MO短消息流程中，VLR收到SGs接口的SGsAP-UPLINK-UNITDATA消息的响应原因值。 |

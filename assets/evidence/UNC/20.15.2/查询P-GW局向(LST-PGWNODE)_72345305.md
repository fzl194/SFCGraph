#### [命令功能](#ZH-CN_MMLREF_0000001172345305)

**适用网元：SGSN、MME**

该命令用于查询P-GW局向。

#### [注意事项](#ZH-CN_MMLREF_0000001172345305)

该命令执行后立即生效。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172345305)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172345305)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172345305)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| GRPID | P-GW组号 | 可选必选说明：可选参数<br>参数含义：该参数用于指定P-GW组号。<br>数据来源：本端规划<br>取值范围：0～15<br>默认值：无 |

#### [使用实例](#ZH-CN_MMLREF_0000001172345305)

查询P-GW组号为2的局向信息：

LST PGWNODE: GRPID=2;

```
%%LST PGWNODE: GRPID=2;%%
RETCODE = 0  操作成功。

操作结果如下
--------------
         P-GW组号  =  2
P-GW Nodename后缀  =  GW01.CITY-B.PROVINCE-A.NODE.EPC.MNC000.MCC460.3GPPNETWORK.ORG
             描述  =  noname
(结果个数 = 1)

---    END
```

#### [输出结果说明](#ZH-CN_MMLREF_0000001172345305)

| 输出项名称 | 输出项解释 |
| --- | --- |
| P-GW组号 | 参数含义：该参数用于指定P-GW组号<br>取值范围：0～15 |
| P-GW Nodename后缀 | 参数含义：该参数用来设置P-GW的Nodename后缀，用于匹配P-GW的Nodename识别P-GW局向。<br>取值范围：1～255位字符串 |
| 描述 | 参数含义：P-GW局向描述信息<br>取值范围：0～256位字符串 |

## [命令功能](#ZH-CN_MMLREF_0000001954415745)

**适用NF：NWDAF、SMF、SPF**

该命令用于显示设备档案信息。

## [注意事项](#ZH-CN_MMLREF_0000001954415745)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001954415745)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001954415745)

无

## [使用实例](#ZH-CN_MMLREF_0000001954415745)

查询设备档案信息

```
%% LST DEVICEINFO:;%%
RETCODE = 0  操作成功

结果如下
--------
       网元类型  =  XXX
       	EOS时间  =  2027-12-31
   网元激活时间  =  2024-02-26
       网元版本  =  25.0.0
   网元补丁版本  =  25.0.0.1
       网元名称  =  XXX
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001954415745)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 网元类型 | 该参数用于表示网元的类型。 |
| EOS时间 | 该参数用于表示停止支持服务的时间。 |
| 网元激活时间 | 该参与用于表示网元的首次激活时间。 |
| 网元版本 | 该参数用于表示网元的版本。 |
| 网元补丁版本 | 该参数用于表示网元的补丁版本。 |
| 网元名称 | 该参数用于表示网元的名称。 |

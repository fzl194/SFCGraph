- 已完成加载License。

## [操作步骤](#ZH-CN_OPI_0275848137)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. 配置HTTP动态头域压缩字典上报功能，以及上报周期。
  [**SET HTTPINFORPT**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/HTTP功能管理/HTTP管理/HTTP CHR上报管理/设置HTTP连接信息上报参数（SET HTTPINFORPT）_29213293.md)

## [任务示例](#ZH-CN_OPI_0275848137)

任务描述

开启HTTP动态头压缩字典上报功能，且上报周期为60秒。

脚本

//打开AMF和SMF的HTTP动态头压缩字典上报功能的License。

```
SET LICENSESWITCH: LICITEM="LKV2HTTPCMPHD01", SWITCH=ENABLE;
SET LICENSESWITCH: LICITEM="LKV3HTTPCMPHD02", SWITCH=ENABLE;
```

//配置HTTP动态头域压缩字典上报功能，以及上报周期。

```
SET HTTPINFORPT: HTTPLNKINFOLST=HTTPDYNCOMPHDIDXRPT-1, DYNHDRPTMR=60;
```

## [命令功能](#ZH-CN_MMLREF_0000001977579560)

**适用NF：SMF**

该命令用于查询SMF向AMF下发事件订阅时的老化策略。

## [注意事项](#ZH-CN_MMLREF_0000001977579560)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001977579560)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001977579560)

无

## [使用实例](#ZH-CN_MMLREF_0000001977579560)

查询SMF向AMF下发事件订阅时的老化策略：

```
%%LST SMAMEVTEXPPLY:;%%
RETCODE = 0  操作成功

结果如下
------------------------
 下发老化时间开关 =  使能
老化时间时长(min) =  1440
```

## [输出结果说明](#ZH-CN_MMLREF_0000001977579560)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 下发老化时间开关 | 该参数用于指定SMF向AMF下发事件订阅时，请求消息是否携带老化时间。 |
| 老化时间时长（min） | 该参数用于指定SMF向AMF下发事件订阅时携带老化时间的时长。 |

## [操作步骤](#ZH-CN_OPI_0277768572)

1. 进入 “MML命令行-UNC” 窗口。
2. 设置NRF每秒处理NFPROFILE变更事件的流控功能。
  **[SET NRFNOTIFYFCPARA](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/NRF业务及策略管理/NRF业务参数/NRF流控参数/设置通知流控参数（SET NRFNOTIFYFCPARA）_09653165.md)**
3. 配置NRF状态通知失败时重传的参数。
  **[SET NOTIFYRSDPARA](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/NRF业务及策略管理/NRF业务参数/NRF通知失败重传开关/设置状态通知失败重传参数（SET NOTIFYRSDPARA）_25121212.md)**
4. 设置NRF平滑通知参数，合并变更通知。
  **[SET NRFNOTIFYFCPARA](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/NRF业务及策略管理/NRF业务参数/NRF流控参数/设置通知流控参数（SET NRFNOTIFYFCPARA）_09653165.md)**

## [任务示例](#ZH-CN_OPI_0277768572)

任务描述

系统过载时，启动自保流控。

脚本

//设置基于CPU过载流控开关。

```
SET FCSWITCH: SWITCH=ENABLE;
```

//设置NRF每秒处理NFPROFILE变更事件的流控功能。

```
SET NRFNOTIFYFCPARA:FCSWITCH=FUNC_ON, FCTHD=200, RSDFCSWITCH=FUNC_OFF;
```

//配置NRF状态通知失败时重传的参数。

```
SET NOTIFYRSDPARA:NOTIFYRSDSW=FUNC_OFF;
```

//配置NRF平滑通知参数，抑制合并变更通知。

```
SET NRFNOTIFYFCPARA:NFCHGCACHETHD=900, NFCHGCACHETIME=50;
```

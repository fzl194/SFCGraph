#### [命令功能](#ZH-CN_CONCEPT_0209897185)

**适用NF：PGW-C、SMF**

该命令用来查询全局的所有初始请求URR组的计费属性。

#### [注意事项](#ZH-CN_CONCEPT_0209897185)

当前版本不支持此命令。

#### [操作用户权限](#ZH-CN_CONCEPT_0209897185)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0209897185)

无。

#### [使用实例](#ZH-CN_CONCEPT_0209897185)

查询全局的所有初始请求URR组的计费属性：

```
LST GLBCTXREQRATE:;
```

```

RETCODE = 0  操作成功。

全局初始请求计费绑定关系信息
----------------------------
初始请求URR组名称  =  urrgrp1
(结果个数 = 1)
---    END
```

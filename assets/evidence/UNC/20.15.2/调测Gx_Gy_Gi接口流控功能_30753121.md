# 调测Gx/Gy/Gi接口流控功能

- [操作场景](#ZH-CN_OPI_0230753121__1.3.1)
- [必备事项](#ZH-CN_OPI_0230753121__1.3.2)
- [操作步骤](#ZH-CN_OPI_0230753121__1.3.3)

## [操作场景](#ZH-CN_OPI_0230753121)

UNC 的周边网元存在过载风险时， UNC 能够基根据本地配置控制向各网元发送消息的速率，从而保护周边网元。

> **说明**
> 适用于 GGSN、 PGW-C。

## [必备事项](#ZH-CN_OPI_0230753121)

前提条件

- 请仔细阅读[统一HTR流控功能特性概述](../统一HTR流控功能/WSFD-010802 周边网元过载保护特性概述（统一HTR流控功能特性概述）_67584274.md)。
- 完成[激活Gx/Gy/Ga/Gi接口流控功能](激活Gx_Gy_Ga_Gi接口流控功能_30753120.md)。

数据

该操作无需准备数据。

工具

该操作无需准备其他工具。

## [操作步骤](#ZH-CN_OPI_0230753121)

1. UNC 上大量用户附着的场景下，统计 UNC 每秒发送给PCRF的CCR-I和CCR-U的消息数，检查其是否超过如下命令中wal值。
  **[LST PCRF](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/PCC管理/PCRF Diameter连接/PCRF/查询PCRF（LST PCRF）_09897104.md)**
2. UNC 上大量用户附着的场景下，统计 UNC 每秒发送给OCS的CCR-I和CCR-U的消息数，检查其是否超过如下命令中wal值。
  **[LST OCS](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/计费管理/在线计费/OCS Diameter连接/OCS Server/查询OCS（LST OCS）_09896957.md)**
3. UNC 上大量用户附着的场景下，统计 UNC 每秒发送给CG的话单数，检查其是否超过如下命令中wal值。
  **[LST CG](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/计费管理/离线计费/GTPP信令/CG管理/查询CG（LST CG）_09896848.md)**
4. UNC 上大量用户附着的场景下，统计 UNC 每秒发送给AAA计费服务器的Accounting Start和Accounting Interim数，检查其是否超过如下命令中计费服务器对应的wal值。
  **[LST RDSSVR](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/RADIUS管理/连接管理/Radius服务器/查询RADIUS服务器（LST RDSSVR）_09896759.md)**
5. UNC 上大量用户附着的场景下，软参Byte863值非0，统计 UNC 的Gx接口集中点每秒发送给PCRF的CCR消息数，检查当Gx集中点每秒发送的CCR消息数超过软参BYTE863配置的阈值后，是否上报 [ALM-81057 接口拥塞告警](../../../../../../网络运维/故障处理/UNC告警处理/业务告警/SMF&GW-C&GGSN/ALM-81057 接口拥塞_74835235.md) 。
  **[LST SMFSOFTPARA](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/软件参数管理/查询SMF软参（LST SMFSOFTPARA）_09652535.md)**

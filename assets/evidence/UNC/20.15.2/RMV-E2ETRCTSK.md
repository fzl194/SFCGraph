# 删除端到端跟踪任务（RMV E2ETRCTSK）

- [注意事项](#ZH-CN_MMLREF_0209653046__1.3.2)

## [命令功能](#ZH-CN_MMLREF_0209653046)

**适用NF：AMF、MME**

该命令用于删除端到端跟踪任务。

## [注意事项](#ZH-CN_MMLREF_0209653046)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0209653046)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209653046)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IMSI | IMSI | 可选必选说明：必选参数<br>参数含义：该参数用于指定用户的IMSI。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是5~15。只允许输入十进制数字（0-9）。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209653046)

删除端到端跟踪任务，删除“SUPI”为“460123456789000”的端到端跟踪任务，执行如下命令：

```
RMV E2ETRCTSK: IMSI="460123456789000";
```

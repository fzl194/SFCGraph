#### [命令功能](#ZH-CN_CONCEPT_0000001281288845)

**适用NF：NCG**

该命令用于更新FTP PULL分发任务的密码为不可逆加密方式存储。

#### [注意事项](#ZH-CN_CONCEPT_0000001281288845)

- 该命令执行后立即生效。

#### [本地用户权限](#ZH-CN_CONCEPT_0000001281288845)

manage-ug；system-ug

#### [网管用户权限](#ZH-CN_CONCEPT_0000001281288845)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001281288845)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| INPUT | 输入 | 可选必选说明：必选参数<br>参数含义：该参数用于指定输入参数<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～128<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001281288845)

执行UPL CGFTPPWD更新FTP PULL分发任务的密码为不可逆加密方式存储：

```
UPL CGFTPPWD: INPUT="ncg";
```

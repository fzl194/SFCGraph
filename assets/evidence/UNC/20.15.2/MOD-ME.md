## [命令功能](#ZH-CN_MMLREF_0000001100124818)

![](修改网元信息（MOD ME）_00124818.assets/notice_3.0-zh-cn_2.png)

该命令用于修改网元信息，如果涉及修改网元接口ID，请确认网管侧是否已安装对应版本的网管适配层，若已安装，网管对接会断连后重连；否则网管对接会断连。

本命令用于对配置数据库中的一个网元实例的某些参数进行修改和调整。在本命令中，系统通过“网元ID”来唯一标识一个网元实例。

## [注意事项](#ZH-CN_MMLREF_0000001100124818)

执行此命令会重新加载适配包，处理耗时较长，频繁执行可能会导致Soap服务CPU升高，进而上报“ALM-5525 容器资源过载”告警 。建议至少间隔2min再次执行此命令。

## [参数说明](#ZH-CN_MMLREF_0000001100124818)

| **参数标识** | **参数名称** | **参数说明** |
| --- | --- | --- |
| MEID | 网元ID | 可选必选说明：必选参数。<br>参数含义：用于指定需要对哪一个网元对象实例的参数进行修改或调整。<br>取值范围：0~65535。<br>默认值：无。<br>配置原则：可以使用<br>[**LST ME**](查询网元配置信息（LST ME）_47084797.md)<br>命令查询获得。 |
| MELN | 网元本地名称 | 可选必选说明：可选参数。<br>参数含义：用于标识网元的本地名称，可自定义。<br>取值范围：字符串类型，输入最大长度为80字符，且不支持特殊字符%%、+++、"、=、;、'、,。<br>默认值：无。<br>配置原则：根据实际情况定义网元的本地名称，建议取用有说明性质的本地名称。例如：网元所在地理位置信息（省/地市/机房等）。若不输入，默认不做修改。 |
| MENAME | 网元名称 | 可选必选说明：可选参数。<br>参数含义：用于标识网元的名称，可自定义。<br>取值范围：<br>字符串类型，输入最大长度为64字符。<br>默认值：无。<br>配置原则：<br>- 根据网元的实际情况命名，建议定义具有说明性质的名称。若不输入，默认不做修改。<br>- 该参数由数字、字母“A～Z”、“a～z”和特殊字符“-”、“_”、“.”组成。<br>- 该参数必须以字母“A～Z”、“a～z”开头。<br>- 该参数不能以“.”结尾。<br>- 该参数不允许输入目录符号“./”、“..”。 |
| MECC | 网元国家（地区）码 | 可选必选说明：可选参数。<br>参数含义：用于标识网元的国家（地区）码，枚举类型。<br>取值范围：国际规定的全球通用的国家地区码 。<br>默认值：无。<br>配置原则：根据网元所处地域进行选择，若不选择，默认不做修改。 |
| MEIID | 网元接口ID | 可选必选说明：可选参数。<br>参数含义：用于标识网元的网元接口ID，可自定义。<br>取值范围：长度不超过64的字符串。<br>默认值：无。<br>配置原则：<br>- 根据实际情况定义网元的网元接口ID，修改网元接口ID后，需要使用对应版本的网管适配层重新对接网管，否则可能会导致网管断链，若不输入，默认不做修改。<br>- 该参数由数字、字母“A～Z”、“a～z”和特殊字符“-”、“_”、“.”组成。<br>- 该参数必须以数字或者字母“A～Z”、“a～z”开头。<br>- 该参数不允许输入目录符号“..”。<br>- 网管适配层的网元接口ID是由appType和网元接口ID拼接而来，该参数修改时无需添加appType。 |

## [使用实例](#ZH-CN_MMLREF_0000001100124818)

修改网元信息：

```
%%MOD ME: MEID=20, MELN="new name", MENAME="new meName", MECC=CN, MEIID="new meIId";%% 
RETCODE = 0  操作成功  

---    END
```

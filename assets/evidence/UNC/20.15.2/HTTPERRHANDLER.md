## [命令功能](#ZH-CN_MMLREF_0000001216816700)

**适用NF：SMF**

该命令用于查询间接路由代理故障重选配置。

## [注意事项](#ZH-CN_MMLREF_0000001216816700)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001216816700)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001216816700)

无

## [使用实例](#ZH-CN_MMLREF_0000001216816700)

查询间接路由代理故障重选配置。

```
%%LST HTTPERRHANDLER:;%%
RETCODE = 0  操作成功
结果如下
--------
            对端网络功能类型  =  UDM
                      状态码  =  500
     间接路由代理故障处理开关 = 使能
---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001216816700)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 对端网络功能类型 | 该参数用于指定对端网元类型。 |
| 状态码 | 该参数用于指定状态码。 |
| 间接路由代理故障处理开关 | 该参数用于控制开启和关闭间接路由代理故障处理功能。 |

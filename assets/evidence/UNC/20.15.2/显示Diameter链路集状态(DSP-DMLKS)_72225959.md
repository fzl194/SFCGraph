#### [命令功能](#ZH-CN_MMLREF_0000001172225959)

**适用网元：SGSN、MME**

该命令用于查看Diameter链路集状态。

#### [注意事项](#ZH-CN_MMLREF_0000001172225959)

该命令执行后立即生效。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172225959)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172225959)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172225959)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| LINKSIDX | Diameter链路集索引 | 可选必选说明：可选参数<br>参数含义：该参数用于指定待查询的Diameter链路集的索引。<br>取值范围：0～639<br>默认值：无<br>说明：如果不输入，表示查询系统内所有Diameter链路集的状态。 |

#### [使用实例](#ZH-CN_MMLREF_0000001172225959)

1. 不输入查询参数，查询所有Diameter链路状态信息：
  DSP DMLKS:;
  ```
  %%DSP DMLKS:;%%
  RETCODE = 0  操作成功。

  输出结果如下
  --------------
   Diameter链路集索引  链路集名称  链路集状态

   0                   To-HSS0     正常      
   1                   To-HSS1     故障      
  (结果个数 = 2)
  ---    END
  ```
2. 输入Diameter链路集索引，查询指定的Diameter链路集状态信息：
  DSP DMLKS: LINKSIDX=0;
  ```
  %%DSP DMLKS: LINKSIDX=0;%%
  RETCODE = 0  操作成功。

  输出结果如下
  --------------
  Diameter链路集索引  =  0
          链路集名称  =  To-HSS0
          链路集状态  =  正常
  (结果个数 = 1)
  ---    END
  ```

#### [输出结果说明](#ZH-CN_MMLREF_0000001172225959)

| 输出项名称 | 输出项解释 |
| --- | --- |
| Diameter链路集索引 | Diameter链路集的索引，该参数用来在系统范围内部唯一标识一条Diameter链路集。 |
| 链路集名称 | Diameter链路集名称。 |
| 链路集状态 | 查询Diameter链路集状态的查询结果。<br>取值说明：<br>- “正常”：表示MME与HSS、EIR、DRA或GMLC之间的通信在该链路集上正常。<br>- “故障”：表示MME与HSS之间的通信在该链路集上中断，该链路集下所有链路的状态均为故障（可以通过[**LST DMLNK**](../Diameter链路/查询Diameter链路配置(LST DMLNK)_26146276.md)命令查看该链路集下的链路索引，[**DSP DMLNK**](../Diameter链路/显示Diameter链路状态(DSP DMLNK)_72225955.md)命令查看链路状态。），需要检查网络状况，分析故障原因。详细操作参见**ALM-80591 Diameter链路故障**告警处理。<br>- “拥塞”：表示MME与HSS之间的通信在该链路集上的数据量达到链路集负荷门限。该链路集下所有链路状态均不正常（可以通过[**LST DMLNK**](../Diameter链路/查询Diameter链路配置(LST DMLNK)_26146276.md)命令查看该链路集下的链路索引，[**DSP DMLNK**](../Diameter链路/显示Diameter链路状态(DSP DMLNK)_72225955.md)命令查看链路状态。），需要调整组网或扩容。详细操作参见**ALM-80591 Diameter链路故障**告警处理。 |

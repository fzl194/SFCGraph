#### [命令功能](#ZH-CN_CONCEPT_0000002194695664)

**适用网元：MME**

该命令用于查询S1 TAI对象寻呼流控状态详细信息。

#### [注意事项](#ZH-CN_CONCEPT_0000002194695664)

无。

#### [本地用户权限](#ZH-CN_CONCEPT_0000002194695664)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_CONCEPT_0000002194695664)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000002194695664)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| TAIGROUPNAME | TAI组名 | 可选必选说明：可选参数。<br>参数含义：该参数用于指定待查询的S1 TAI组对象名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~255。<br>默认值：无<br>配置原则：<br>查询的对象名称可以通过<br>**[LST PERFOBJ](../../../../../性能管理/性能对象管理/查询性能对象信息(LST PERFOBJ)_26306000.md)**<br>进行查询。 |

#### [使用实例](#ZH-CN_CONCEPT_0000002194695664)

查询group_1 TAI组对象寻呼流控状态详细信息，执行如下命令：

```
%%DSP S1TAIDETECTDETAILS, TAIGROUPNAME="group_1";%%
RETCODE = 0  操作成功。

输出结果如下
-----------------------
                 索引  =  1
              TAI组名  =  group_1
        TAI组对象状态  =  正常
TAI组对象状态更新时间  =  2025-03-06 19:35:21+08:00 
(结果个数 = 1)

---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000002194695664)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 索引 | 该参数用于显示索引信息。 |
| TAI组名 | 该参数用于显示待查询的S1 TAI组对象名称。 |
| TAI组对象状态 | 该参数用于显示TAI组对象状态。<br>取值说明：<br>- “NORMAL（正常）”：TAI组对象状态正常<br>- “ABNORMAL（异常）”：TAI组对象状态异常 |
| TAI组对象状态更新时间 | 该参数用于显示TAI组对象状态最后更新时间。 |

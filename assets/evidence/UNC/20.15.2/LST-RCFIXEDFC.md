## [命令功能](#ZH-CN_MMLREF_0000001353801278)

**适用NF：SMSF**

该命令用于查询SMSF/VLR向注册中心发送请求的固定速率流控。

## [注意事项](#ZH-CN_MMLREF_0000001353801278)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001353801278)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001353801278)

无

## [使用实例](#ZH-CN_MMLREF_0000001353801278)

运营商希望查询SMSF/VLR向注册中心发送请求的固定速率流控参数，执行如下命令：

```
LST RCFIXEDFC:;
%%LST RCFIXEDFC:;%%
RETCODE = 0  操作成功

结果如下：
----------
            SMSF/VLR向注册中心发送注册请求的固定速率流控开关  =  开启
            SMSF/VLR向注册中心发送查询请求的固定速率流控开关  =  开启			
SMSF/VLR向注册中心发送注册请求的固定速率流控的起控阈值(个/秒) =  10000 
SMSF/VLR向注册中心发送查询请求的固定速率流控的起控阈值(个/秒) =  10000
                                                阈值计算策略  =  整系统
(结果个数 = 1)
```

## [输出结果说明](#ZH-CN_MMLREF_0000001353801278)

| 输出项名称 | 输出项解释 |
| --- | --- |
| SMSF/VLR向注册中心发送注册请求的固定速率流控开关 | 该参数用于表示SMSF/VLR向注册中心发送注册请求的固定速率流控开关。 |
| SMSF/VLR向注册中心发送查询请求的固定速率流控开关 | 该参数用于表示SMSF/VLR向注册中心发送查询请求的固定速率流控开关。 |
| SMSF/VLR向注册中心发送注册请求的固定速率流控的起控阈值(个/秒) | 该参数用于表示SMSF/VLR向注册中心发送注册请求的固定速率流控的起控阈值。 |
| SMSF/VLR向注册中心发送查询请求的固定速率流控的起控阈值(个/秒) | 该参数用于表示SMSF/VLR向注册中心发送查询请求的固定速率流控的起控阈值。 |
| 阈值计算策略 | 该参数用于控制“SMSF/VLR向注册中心发送注册请求的固定速率流控的起控阈值(个/秒)”和“SMSF/VLR向注册中心发送查询请求的固定速率流控的起控阈值(个/秒)”的计算策略。 |

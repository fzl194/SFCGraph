## [命令功能](#ZH-CN_MMLREF_0000001872196070)

该命令用于查询节点磁盘部署规格。

## [注意事项](#ZH-CN_MMLREF_0000001872196070)

- 该命令执行后立即生效。
- 该命令仅在虚机场景下支持。

#### [操作用户权限](#ZH-CN_MMLREF_0000001872196070)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001872196070)

无

## [使用实例](#ZH-CN_MMLREF_0000001872196070)

查询所有节点的磁盘部署规格。

```
%%DSP DISKINFO:;%%
RETCODE = 0  操作成功

结果如下
------------------------
节点类型    节点名称     磁盘资源总量（MB）

SGU_A       10.0.0.0     31372.41
```

## [输出结果说明](#ZH-CN_MMLREF_0000001872196070)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 节点类型 | 本参数用于指定节点类型。 |
| 节点名称 | 该参数用于指定节点IP地址。 |
| 磁盘资源总量（MB） | 该参数用于指定磁盘资源总量。 |

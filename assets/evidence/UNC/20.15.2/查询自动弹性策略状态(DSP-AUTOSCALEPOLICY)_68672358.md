## [命令功能](#ZH-CN_MMLREF_0000002068672358)

该命令用于查询自动弹性策略状态。

> **说明**
> 该命令仅在Full-stack裸机场景下支持。

## [注意事项](#ZH-CN_MMLREF_0000002068672358)

无。

## [参数说明](#ZH-CN_MMLREF_0000002068672358)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| MEID | 网元ID | 可选必选说明：可选参数。<br>参数含义：网元ID，可以通过<br>[**LST ME**](../../系统管理/版本信息/查询网元配置信息（LST ME）_47084797.md)<br>获取。<br>取值范围：0～65535<br>默认值：无。<br>配置原则：无。 |

## [使用实例](#ZH-CN_MMLREF_0000002068672358)

查询自动弹性策略状态：

```
%%DSP AUTOSCALEPOLICY: MEID=0;%% 
RETCODE = 0  操作成功  
操作结果如下 
------------     
弹性策略名称  =  csdb-pod-autopolicy 
弹性策略是否激活  =  是 
弹性策略执行阶段  =  采集
弹性策略执行状态  =  执行中 

 (结果个数 = 1)  ---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000002068672358)

命令执行正常，会返回弹性策略名称、弹性策略是否激活、弹性策略执行阶段和弹性策略执行状态信息，输出结果项信息项目如 [表1 输出项说明](#ZH-CN_MMLREF_0000002068672358__table819976173217) 。

命令执行异常，会返回对应的错误码。输出错误码说明如 [表2](#ZH-CN_MMLREF_0000002068672358__table319821018114) 所示。

*表1 输出项说明*

| 输出项名称 | 输出项解释 |
| --- | --- |
| 弹性策略名称 | 用于返回查询的弹性策略名称。 |
| 弹性策略是否激活 | 用于标识弹性策略是否激活。<br>取值说明：<br>- 是：弹性策略激活。<br>- 否：弹性策略未激活。 |
| 弹性策略执行阶段 | 用于标识该弹性策略对应弹性任务所处阶段。<br>取值说明：<br>- 采集：采集阶段。<br>- 弹性前检查：弹性前检查阶段。<br>- 人工确认：人工确认阶段。<br>- 弹性执行：执行弹性扩缩容任务阶段。<br>- NA：任务未启动。 |
| 弹性策略执行状态 | 用于标识该弹性任务不同阶段的执行状态。<br>取值说明：<br>- 执行中：任务正在执行。<br>- 成功：任务执行成功。<br>- 失败：任务执行失败。<br>- NA：任务未启动，暂无状态。 |

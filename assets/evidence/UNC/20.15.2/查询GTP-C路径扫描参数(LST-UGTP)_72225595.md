#### [命令功能](#ZH-CN_MMLREF_0000001172225595)

**适用网元：SGSN、MME**

该命令用于查询GTP信息表。该信息表包含了SPP和UPP进程的运行控制参数。

#### [注意事项](#ZH-CN_MMLREF_0000001172225595)

无。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172225595)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172225595)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172225595)

无。

#### [使用实例](#ZH-CN_MMLREF_0000001172225595)

查询GTP信息表：

LST UGTP:;

```
%%LST UGTP:;%%
RETCODE = 0  操作成功。

操作结果如下
-------------------------
    空闲计数器扫描间隔(秒)  =  180
      SM信令路径空闲计数器  =  60
      MM信令路径空闲计数器  =  60
      GTPMAP路径空闲计数器  =  60
路径版本探测扫描周期(分钟)  =  60
(结果个数 = 1)

---    END
```

#### [命令功能](#ZH-CN_CONCEPT_0000001167848008)

**适用NF：SMSF**

该命令用于查询SMSF对接SMSC网元时，SCCP层主叫地址中携带的SSN号。

#### [注意事项](#ZH-CN_CONCEPT_0000001167848008)

- 该命令执行后立即生效。

#### [参数说明](#ZH-CN_CONCEPT_0000001167848008)

无

#### [使用实例](#ZH-CN_CONCEPT_0000001167848008)

查询SMSF的主叫地址的子系统号：

```
LST SMSFSSN:;
```

```
%%LST SMSFSSN:;%%
RETCODE = 0  操作成功

输出结果如下
------------
子系统号  =  VLR(7)
(结果个数 = 1)

---    END
```

#### [命令功能](#ZH-CN_CONCEPT_0209896841)

**适用NF：SGW-C、PGW-C、SMF**

该命令用来查询全局绑定的费率切换组。

#### [注意事项](#ZH-CN_CONCEPT_0209896841)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0209896841)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0209896841)

无。

#### [使用实例](#ZH-CN_CONCEPT_0209896841)

查询全局绑定的费率切换组，命令为：

```
LST GLBTARIFFGROUP:;
```

```

RETCODE = 0  操作成功。

全局费率组
----------

(结果个数 = 1)
---    END
```

#### [适用NF](#ZH-CN_CONCEPT_0169313567)

SGSN、 GGSN、 MME、SGW-C、PGW-C、AMF、SMF

#### [定义](#ZH-CN_CONCEPT_0169313567)

支持多HPLMN（Home Public Land Mobile Network）特性指 UNC 可以同时为多个PLMN提供服务。 UNC 通过本特性可以配置多个HPLMN，用于判断接入用户的漫游属性。

#### [客户价值](#ZH-CN_CONCEPT_0169313567)

| **受益方** | **受益描述** |
| --- | --- |
| 运营商 | - 对于单个运营商而言，该运营商可以通过本特性在不同国家或同一国家的不同地区运营不同PLMN的网络，既增加了运营商网络的灵活性和可扩展性，又降低了设备成本。<br>- 在2G、3G、4G网络中，不同运营商可以通过本特性共享一套UNC设备，有利于节约网络建设成本。 |
| 用户 | 用户不感知该特性。 |

#### [应用场景](#ZH-CN_CONCEPT_0169313567)

- 当单一PLMN内的用户号段数量无法满足签约用户数量时，运营商可以选择开启本特性来运营多个PLMN，从而为更多签约用户提供服务。
- 在2G、3G、4G网络中，如果运营商要对不同的PLMN内的签约用户采取不同的计费和策略控制等处理，则运营商可通过本特性在UNC上配置多个HPLMN，保证对签约用户进行精准的计费和业务控制。

#### [原理概述](#ZH-CN_CONCEPT_0169313567)

**相关概念**

- PLMN
  PLMN（Public Land Mobile Network）指公用陆地移动网，用于区分一个国家或地区不同的移动通信运营商。一个服务区可由一个或若干个公用陆地移动通信网组成。PLMN ID是一个PLMN网络的唯一标识，它由移动国家号码MCC和移动网号码MNC两部分组成。
- HPLMN
  HPLMN（Home Public Land Mobile Network ）指归属公共陆地移动网，如果UE IMSI/SUPI/SUCI中的MCC、MNC与某个PLMN匹配时，该PLMN便是这个UE的HPLMN。
- MNO
  MNO（Mobile Network Operator）指移动网络运营商，一个MNO可以拥有一个或多个PLMN。
- 本端设备标识
  本端设备标识可以是以下几种之一，用于 UNC 与对端设备通讯时寻址，在 UNC 共享的情况下，不同的MNO使用不同的本端设备标识。
    - SGSN号码，例如Gr接口的本端设备标识是SGSN网元设备号码。
    - GTP-C/GTP-U IP。例如Gn/Gp接口的本端标识是GTP-C/GTP-U IP。
    - Diameter本地实体。例如S6a/S6d接口的本端标识是本端域名或者本端主机名。

**系统实现**

运营商可以通过本特性同时为多个HPLMN签约的用户提供本地服务，也可将签署了漫游协议的PLMN添加到互联PLMN列表并为其提供漫游服务，此时，对于非本网签约且不在互联PLMN列表中的用户， UNC 将拒绝其接入请求。

对于漫游用户而言， UNC 将根据其签约数据信息判断其漫游类型为Home Routed或Local Breakout。若漫游类型为Home Routed，则该用户使用HPLMN内 GGSN/ PGW-C/UPF接入网络，否则该用户使用VPLMN内 GGSN/ PGW-C/UPF接入网络。

**多HPLMN中的业务控制**

2G、3G、 4G网络中，在多HPLMN情况下， UNC 可以通过配置，分别为不同的HPLMN指定以下信息：是否允许SM业务、最大承载数、是否允许SMS业务、是否允许纠正短消息中心、纠正后的短消息中心、是否允许LCS业务、协议类型、运营商名称等。

在 UNC 支持多HPLMN情况下，MNO和设备标识的选择方式如下：

- MNO的选择
  UE接入 UNC 后， UNC 可以选择为其服务MNO。

  - 对于MNO本网用户，UNC直接选择为UE分配IMSI的MNO。
    - 对于签署了漫游协议的外网用户，可以指定关联到对应的MNO。

  > **说明**
  > 对于仅和一个MNO签署了漫游协议的专属外网用户，指定的是唯一关联的MNO；对于和多个MNO签署了漫游协议的非专属外网用户，指定的是优先关联的MNO。优先关联的MNO可通过 [**ADD CONNECTPLMN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/互联PLMN管理/增加互联PLMN(ADD CONNECTPLMN)_26305852.md) 命令中的 “EXCLUSIVEPLMN” 参数设定。
- 本端设备标识的选择
  本端设备标识的选择通过配置实现，按照配置粒度，分为三个配置层次。系统优先使用第一层配置，如果没有配置，再考虑第二层配置，依次类推。
  第一层配置指根据IMSI号段配置，第二层配置指根据MNO/MVNO ID配置，第三层配置是轮选系统中所有的缺省配置。

#### [计费与话单](#ZH-CN_CONCEPT_0169313567)

- 用户可以根据PLMN配置S-CDR生成策略和CG的选择。
- 话单中携带由MCC和MNC号码，计费中心可以通过PLMN ID区分过滤话单。

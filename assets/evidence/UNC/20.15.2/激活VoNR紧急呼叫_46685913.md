## [操作步骤](#ZH-CN_OPI_0000001446685913)

在AMF上配置VoNR紧急呼叫业务

1. 打开本特性的License配置开关。
  **[SET LICENSESWITCH](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)**
2. 配置是否允许紧急呼叫业务。
  **[SET NGMMFUNC](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/移动性管理/MM协议参数管理/5G移动性管理/设置5G移动性管理功能（SET NGMMFUNC）_09653748.md)**
3. 添加紧急呼叫号码。
  **[ADD NGEMGCNUM](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/5G 语音业务管理/紧急呼叫业务管理/紧急呼叫号码配置/增加紧急号码配置信息（ADD NGEMGCNUM）_09652453.md)**
4. 配置本网用户5G紧急呼叫业务。

## [任务示例](#ZH-CN_OPI_0000001446685913)

任务描述

激活VoNR紧急呼叫特性。

脚本

//该功能AMF和SMF的License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2VONRECMM01", SWITCH=ENABLE;
SET LICENSESWITCH: LICITEM="LKV2VONRECSM01", SWITCH=ENABLE;
```

//使能紧急呼叫功能。

//在SMF上创建APN实例，使能紧急呼叫功能。

```
ADD APN:APN="sos",EMERGENCYSWITCH=ENABLE;
```

//在SMF上使能APN的PCC功能。

```
SET APNPCCFUNC:APN="sos", HOMEPCCSWITCH=ENABLE, ROAMPCCSWITCH=ENABLE, VISITPCCSWITCH=ENABLE, PCFSELECTMODE=DNN-1&GPSI-0&IMSI-0&NFLOC-0&PLMN-0&SERVINGSCOPE-1&SNSSAIS-0;
```

//在SMF上配置APN的IMS属性

```
SET APNIMSATTR:APN="sos", IMSSWITCH=ENABLE;
```

//在SMF上配置P-CSCF组名。

```
ADD PCSCFGROUP: GROUPNAME="pcscf_group1", IPVERSION=IPV6, ALLOCTYPE=LOCAL;
```

//在SMF上将APN下的某指定号段绑定到P-CSCF组。

```
ADD PCSCFGRPBNDAPN:APN="sos", DEFAULTFLAG=DEFAULT, MPCSCFGRPIPV6="pcscf_group1";
```

//在AMF上配置允许VoNR紧急呼叫业务。

```
SET NGMMFUNC: EMG=EMCNR-1;
```

//在AMF上添加紧急呼叫号码。

```
ADD NGEMGCNUM: MCC="123", ESC=FIREBRIGADE, NUM="119";
```

//在AMF上配置本网用户5G紧急呼叫业务。

```
SET NGEMGSRVFUNC:HOMEEMG=EMFNR-0&EMCNR-1,HOMEEMGCNUMSW=YES;
```

//在AMF上配置紧急会话创建模式，需要配置支持紧急业务的切片业务类型SST和切片细分标识SD。

```
ADD NGEMGCFG: NOID=0, EMGMODE=VALIDUE,DNN="sos",SST=0,SD="FFFFFF";
```

//在SMF上配置运营商紧急呼叫会话相关功能。

```
ADD SMFEMGCFG: NOID=0, ARPPL=1, ARPPCI=NOT_PREEMPT, ARPPVI=NOT_PREEMPTABLE, ULSESSIONAMBR=20000, DLSESSIONAMBR=40000, EMGMODE=VALIDUE, FIVEQI=5, FIVEQIPL=0, UDMREG=DISABLE;
```

// 在SMF上配置紧急呼叫结束后释放缺省承载。

```
SET APNIDLETIME:APN="sos", INHERIT=NO, SCTXCHKSW=ENABLE, PCTXCHKSW=ENABLE, GCTXCHKSW=ENABLE, HCTXCHKSW=ENABLE, ICTXCHKSW=ENABLE, GULTIMERLEVEL=BEARER, DFTBEARPOLICY=ONEDAY, BEARERTIMER=60, SESSIONTIMER=120;
```

## [命令功能](#ZH-CN_MMLREF_0000001360209801)

**适用NF：NRF**

该命令用于查询NRF上的通知策略。

## [注意事项](#ZH-CN_MMLREF_0000001360209801)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001360209801)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001360209801)

无

## [使用实例](#ZH-CN_MMLREF_0000001360209801)

运营商希望查询当前NRF上通知策略，执行如下命令。

```
LST NRFNOTIFYPLY:;
%%LST NRFNOTIFYPLY;%%
RETCODE = 0  操作成功

结果如下
---------
 NRF通知策略  =  NFINSTANCEIDNOT 
内部通知策略  =  DEFAULT
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001360209801)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 通知策略 | 该参数用于设置NRF通知策略。 |
| 内部通知策略 | 该参数用于设置NRF内部通知策略，仅在SCP和NRF联合部署场景下生效。 |

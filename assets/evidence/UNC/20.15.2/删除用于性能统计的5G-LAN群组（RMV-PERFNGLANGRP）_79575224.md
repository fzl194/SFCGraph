# 删除用于性能统计的5G LAN群组（RMV PERFNGLANGRP）

- [注意事项](#ZH-CN_MMLREF_0000001179575224__1.3.2)

## [命令功能](#ZH-CN_MMLREF_0000001179575224)

**适用NF：SMF**

该命令用于删除可用于性能统计的5G LAN群组。

## [注意事项](#ZH-CN_MMLREF_0000001179575224)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0000001179575224)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001179575224)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| GROUPID | 5G LAN组ID | 可选必选说明：必选参数<br>参数含义：该参数用于指定5G LAN群组的ID。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是18~37。字母大小写不敏感且全局唯一。<br>默认值：无<br>配置原则：<br>GROUPID以连字号分为4段，形式为GroupServiceID-MCC-MNC-LocalGroupID。其中，GroupServiceID长度为8，只能输入数字或者范围为A-F或a-f的字母；MCC长度为3，只能输入数字；MNC长度为2~3，只能输入数字；LocalGroupID长度为2~20的偶数，只能输入数字或者范围为A-F或a-f的字母。例如，A0000001-460-003-01，A0000001-460-003-A000000001。 |

## [使用实例](#ZH-CN_MMLREF_0000001179575224)

删除一条群组编号为“A0000001-460-003-01”的5G LAN群组，用于性能统计，执行如下命令：

```
RMV PERFNGLANGRP: GROUPID="A0000001-460-003-01";
```

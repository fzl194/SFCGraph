## [命令功能](#ZH-CN_MMLREF_0000002120679418)

该命令用于显示PAE染色跟踪信息。

## [注意事项](#ZH-CN_MMLREF_0000002120679418)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000002120679418)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000002120679418)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| CELLID | Cell ID | 可选必选说明：可选参数<br>参数含义：该参数表示PAE调试消息发送的CELL ID，可以通过使用命令DSP PAENODE获取。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~127。<br>默认值：无<br>配置原则：无 |

## [输出结果说明](#ZH-CN_MMLREF_0000002120679418)

| 输出项名称 | 输出项解释 |
| --- | --- |
| Cell ID | 该参数表示PAE调试消息发送的CELL ID，可以通过使用命令DSP PAENODE获取。 |
| 染色全流控状态 | 该参数用于显示染色跟踪全流控状态。<br>取值说明：<br>- “TRUE（是）”：染色跟踪全流控已激活<br>- “FALSE（否）”：染色跟踪全流控未激活 |
| CPU负载 (‰) | 该参数用于显示CPU负载比例，千分比形式。 |
| 最大批收满比 (‰) | 该参数用于显示最大批收满比例，千分比形式。 |
| 染色消耗CPU比 (‰) | 该参数用于显示染色模块消耗CPU比例，千分比形式。 |

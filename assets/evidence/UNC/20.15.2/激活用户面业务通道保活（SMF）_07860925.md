## [操作场景](#ZH-CN_OPI_0307860925)

用户面业务通道保活是指，企业园区场景下，当园区gNodeB、UPF跟园区外部署的AMF、SMF失联时，即N2/N4接口不可用但N3接口可用时，弱移动性（不跨gNodeB切换）连接态用户的数传业务能够继续进行（惯性运行）。

> **说明**
> 适用于SMF。

## [操作步骤](#ZH-CN_OPI_0307860925)

1. 进入 “MML命令行-UNC” 窗口。
2. **可选：**增加园区策略，设置园区惯性运行开关。
  **[MOD UPNODE](../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/PFCP接口管理/UP管理/UP节点管理/修改UPF节点（MOD UPNODE）_09652705.md)**
3. 设置SMF和园区UPF间链路惯性运行定时器。
  **[SET PFCPPARA](../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/PFCP接口管理/PFCP路径管理/PFCP路径参数管理/设置PFCP参数（SET PFCPPARA）_09652597.md)**
4. 设置控制面扫描用户的速率。
  **[SET UPCSCANRATE](../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/PFCP接口管理/用户面控制扫描速率管理/设置用户面控制扫描速率（SET UPCSCANRATE）_04284727.md)**

## [任务示例](#ZH-CN_OPI_0307860925)

任务描述

开启业务园区惯性运行开关，当N2和N4接口同时故障时，连接态用户惯性运行。

脚本

//设置园区惯性运行开关。

```
MOD UPNODE: NFINSTANCENAME="upf1", IOSWITCH=ENABLE, CAMPUSSW=ENABLE;
```

//设置SMF和园区UPF间链路惯性运行定时器。

```
SET PFCPPARA: IOINTERVAL=60;
```

//设置控制面扫描用户的速率。

```
SET UPCSCANRATE: SCANTYPE=INERTIALOPERATION, RATE=30;
```

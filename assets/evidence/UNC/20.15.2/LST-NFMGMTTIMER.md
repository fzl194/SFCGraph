## [命令功能](#ZH-CN_MMLREF_0000001601006689)

**适用NF：NRF、SMF、AMF、NSSF、NCG、SMSF**

该命令用于查询NF管理模块的定时器配置。

## [注意事项](#ZH-CN_MMLREF_0000001601006689)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001601006689)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001601006689)

无

## [使用实例](#ZH-CN_MMLREF_0000001601006689)

运营商A需要查询NF管理模块的定时器配置。

```
%%LST NFMGMTTIMER:;%%
RETCODE = 0  操作成功

结果如下
--------
URI地址表老化周期  =  168
URI地址表刷新间隔  =  10
URI地址表扫描间隔  =  60
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001601006689)

| 输出项名称 | 输出项解释 |
| --- | --- |
| URI地址表老化周期 | 该命令用于设置URI地址表的老化周期。老化周期内业务未进行查询的地址表将在老化周期后失效。 |
| URI地址表刷新间隔 | 该命令用于设置URI地址表的老化时间戳刷新间隔。 |
| URI地址表扫描间隔 | 该命令用于设置URI地址表扫描任务的间隔。此值越低，超出老化周期的地址记录就会越快老化。 |

## [命令功能](#ZH-CN_MMLREF_0000001089030926)

**适用NF：SGW-C、PGW-C、SMF、GGSN**

该命令用于查询CHR存储配置。

## [注意事项](#ZH-CN_MMLREF_0000001089030926)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001089030926)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001089030926)

无

## [使用实例](#ZH-CN_MMLREF_0000001089030926)

查询当前CHR存储方式设置情况：

```
LST SESSNCHRSTORECFG:;
RETCODE = 0  操作成功。

CHR存储配置
-----------
    异常CHR存盘开关  =  使能
所有CHR单据存储开关  =  不使能
(结果个数 = 1)
---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001089030926)

| 输出项名称 | 输出项解释 |
| --- | --- |
| UCF保存异常CHR开关 | 该参数用于控制UCF是否保存异常CHR单据。本参数和CHRALLSAVE参数中任意一个使能时异常CHR单据均可以保存在UCF中。 |
| UCF保存整系统CHR开关 | 该参数用于控制UCF是否保存整系统CHR单据。本参数和CHRFAILSAVE参数中任意一个使能时异常CHR单据均可以保存在UCF中。 |
| 小范围CHR使能开关 | 该参数用于控制是否开启小范围CHR功能。 |
| 小范围CHR存储路径 | 该参数用于控制小范围CHR单据的存储路径。 |

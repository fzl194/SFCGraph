#### [命令功能](#ZH-CN_CONCEPT_0209896990)

**适用NF：SGW-C**

LST SGWCHARGECFG命令用来查询SGW计费配置。

#### [注意事项](#ZH-CN_CONCEPT_0209896990)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0209896990)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0209896990)

无。

#### [使用实例](#ZH-CN_CONCEPT_0209896990)

当需要查询SGW计费配置，即本地用户、漫游用户、拜访用户的离线计费开关是否打开时，将命令配置为：

```
LST SGWCHARGECFG:;
```

```

RETCODE = 0  操作成功。

SGW计费控制
-----------
本地用户离线计费开关  =  禁止
漫游用户离线计费开关  =  允许
拜访用户离线计费开关  =  允许
       SGW CG IP选择  =  使用本地CG IP
(结果个数 = 1)
---    END
```

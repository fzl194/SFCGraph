## [命令功能](#ZH-CN_MMLREF_0000001276551421)

该命令用于查询基于Fabric链路状态的故障快速检测参数配置。

## [注意事项](#ZH-CN_MMLREF_0000001276551421)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001276551421)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001276551421)

无

## [使用实例](#ZH-CN_MMLREF_0000001276551421)

查询故障快速检测配置。

```
%%LST FAULTCHECKBYFABRIC:;%%
RETCODE = 0  操作成功

结果如下
------------------------
   故障快速检测开关  =  开启
防震荡保护时长 (分)  =  20
等待上报时长 (毫秒)  =  4000
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001276551421)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 故障快速检测开关 | 该参数用于设置故障快速检测开关状态。 |
| 防震荡保护时长 (分) | 该参数用于设置防震荡保护时长，即当检测到故障后，在此时长内不再做快速故障判定。 |
| 等待上报时长 (毫秒) | 该参数用于设置Fabric链路状态收集的时间窗口，仅该时间窗口内的链路状态参与快速故障判定。 |

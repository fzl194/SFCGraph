## [命令功能](#ZH-CN_MMLREF_0000001382452582)

**适用NF：SMF**

该命令用于查询MB-SMF组播广播功能参数。

## [注意事项](#ZH-CN_MMLREF_0000001382452582)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001382452582)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001382452582)

无

## [使用实例](#ZH-CN_MMLREF_0000001382452582)

当需要查询MB-SMF 5G组播广播功能参数时，执行如下命令：

```
%%LST SMFMBSFUNC:;%%
RETCODE = 0  操作成功

结果如下
--------
                   5G MBMS功能控制开关  =  不使能
                     TMGI失效时长(min)  =  1440
N11mb接口等待基站侧响应消息最大时长(s)  =  120
       N4mb接口等待响应消息最大时长(s)  =  180
                      N3mb接口传输方式  =  单播传输
           广播会话部分AMF故障处理方式  =  保留会话
        广播会话部分MB-UPF故障处理方式  =  保留会话
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001382452582)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 5G MBS功能控制开关 | 该参数用于控制MB-SMF是否开启MBS广播功能。 |
| TMGI失效时长(min) | 该参数用于指定TMGI失效时长。该时间比MB-SMF返回给AF的TMGI过期时间长5分钟。 |
| N11mb接口等待基站侧响应消息最大时长(s) | 该参数用于指定N11mb接口等待基站侧响应消息最大时长。对应Namf_MBSBroadcast_ContextCreate Request和Namf_MBSBroadcast_ContextUpdate Request消息中的maxResponseTime字段。 |
| N4mb接口等待响应消息最大时长(s) | 该参数用于指定N4mb接口等待响应消息最大时长。 |
| N3mb接口传输方式 | 该参数用于指定N3mb接口传输方式。 |
| 广播会话部分AMF故障处理方式 | 该参数用于指定在广播会话的非创建流程中，当部分AMF故障时，MB-SMF处理广播会话的方式。 |
| 广播会话部分MB-UPF故障处理方式 | 该参数用于指定在广播会话的非创建流程中，当部分MB-UPF故障时，MB-SMF处理广播会话的方式。 |
| 广播会话超期协商功能开关 | 该参数用于设置广播会话超期协商的功能开关。 |
| 广播会话超期间隔(分钟) | 该参数用于设置广播会话的默认超期间隔。当前时间与本参数设置的超期间隔之和作为广播会话的超期时间，在广播会话服务响应消息中带给AMF。 |

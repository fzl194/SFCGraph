## [操作场景](#ZH-CN_OPI_0228361026)

当需要将用户接入属性映射到话单信息中时，需要激活接入用户属性反射功能。

> **说明**
> 适用于 GGSN、 PGW-C、SMF。

## [操作步骤](#ZH-CN_OPI_0228361026)

1. 进入 “MML命令行-UNC” 窗口。
2. 配置获取RAT Type的两种方式的优先级。
  [**SET SRVNODERATPRI**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/接入控制/获取RAT管理/获取RAT方式的优先级/设置获取RAT Type的优先级（SET SRVNODERATPRI）_09652385.md)
3. 配置SGSN/S-GW IP与RAT类型间的映射关系表。
  [**ADD SRVNODERAT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/接入控制/获取RAT管理/IP地址映射RAT/增加SGSN_SGW IP与RAT类型间的映射关系（ADD SRVNODERAT）_09653221.md)
4. 配置SGSN/S-GW默认RAT类型。
  [**ADD DFTSRVNODERAT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/接入控制/获取RAT管理/全局RAT配置/设置默认RAT类型（SET DFTSRVNODERAT）_09653166.md)
  > **说明**
  > SGSN/S-GW IP地址段与RAT类型的映射表配有默认项，当SGSN/S-GW IP在表中没有匹配项时，会用默认的RAT类型作为它对应的RAT类型。当要配置默认表项时，使用 [**ADD DFTSRVNODERAT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/接入控制/获取RAT管理/全局RAT配置/设置默认RAT类型（SET DFTSRVNODERAT）_09653166.md) 命令。
5. 配置话单中携带的字段。
  [**ADD CDRFIELDTEMP**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/计费管理/离线计费/话单字段控制/话单字段模板/增加话单字段模板（ADD CDRFIELDTEMP）_09896890.md)
6. 配置APN实例，设置是否允许此APN下的用户通过SGSN/S-GW-IP映射方式来获取RAT Type。
  [**ADD APN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/APN管理/APN/增加APN配置（ADD APN）_09653747.md)
7. 配置APN实例绑定话单字段模板。
  [**SET APNCHARGECTRL**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/计费管理/计费控制/APN计费配置/设置APN的计费配置（SET APNCHARGECTRL）_09896817.md)

## [任务示例](#ZH-CN_OPI_0228361026)

任务描述

配置用户通过RAT TYPE获取接入类型，并将用户的接入属性映射到话单中。

脚本

//配置获取RAT Type的两种方式的优先级。

```
SET SRVNODERATPRI:FIRSTPRIORITY=RAT,SECONDPRIORITY=SGSN_IP,NODETYPE=GGSN;
```

//配置SGSN/S-GW IP与RAT类型间的映射关系表。

```
ADD SRVNODERAT:IPVERSION=IPV4, SRVNODESTARTV4="192.168.1.1", SRVNODEENDV4="192.168.2.2", RATTYPE=UTRAN;
```

//配置SGSN/S-GW默认RAT类型。

```
SET DFTSRVNODERAT:RATTYPE=UTRAN,NODETYPE=GGSN;
```

//配置话单中携带的字段。

```
ADD CDRFIELDTEMP:TEMPLATENAME="cdr-field-test",RATTYPE=ENABLE,ULI=ENABLE,MSTIMEZONE=ENABLE;
```

//配置APN实例。

```
ADD APN:APN="apn-test",SERVINGNODERAT=ENABLE;
```

//配置APN实例绑定话单字段模板。

```
SET APNCHARGECTRL:APN="apn-test", GCDRTEMPLATE="cdr-field-test", PGWCDRTEMPLATE="cdr-field-test", SGWCDRTEMPLATE="cdr-field-test";
```

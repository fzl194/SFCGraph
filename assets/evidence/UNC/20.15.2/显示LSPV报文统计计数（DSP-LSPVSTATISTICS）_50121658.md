#### [命令功能](#ZH-CN_CONCEPT_0000001550121658)

该命令用于显示LSPV报文统计计数。

#### [注意事项](#ZH-CN_CONCEPT_0000001550121658)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550121658)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550121658)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001550121658)

显示LSPV报文统计计数：

```
DSP LSPVSTATISTICS:;
```

```

RETCODE = 0  操作成功。

结果如下
--------
                    发包总数  = 0
                    收包总数  = 0
            发送请求报文总数  = 0
            接收请求报文总数  = 0
            发送应答报文总数  = 0
            接收应答报文总数  = 0
接口下协议去使能丢弃报文总数  = 0
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001550121658)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 发包总数 | 用于显示发包总数。 |
| 收包总数 | 用于显示收包总数。 |
| 发送请求报文总数 | 用于显示发送请求报文总数。 |
| 接收请求报文总数 | 用于显示接收请求报文总数。 |
| 发送应答报文总数 | 用于显示发送应答报文总数。 |
| 接收应答报文总数 | 用于显示接收应答报文总数。 |
| 接口下协议去使能丢弃报文总数 | 用于显示接口下协议去使能丢弃报文总数。 |

# 删除IPv6扩展头选项安全配置（RMV IPV6EXTHSECALL）

#### [命令功能](#ZH-CN_CONCEPT_0000001549802466)

该命令用于删除IPv6扩展头选项安全配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001549802466)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549802466)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549802466)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SWITCHOP | IPv6过滤规则 | 可选必选说明：必选参数<br>参数含义：该参数表示允许或拒绝对应选项IPv6报文的处理能力。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- permit：允许。<br>- deny：拒绝。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549802466)

删除IPv6扩展头选项安全配置：

```
RMV IPV6EXTHSECALL:SWITCHOP=deny;
```

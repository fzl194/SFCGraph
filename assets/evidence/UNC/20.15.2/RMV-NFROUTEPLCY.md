# 删除NF路由策略（RMV NFROUTEPLCY）

- [注意事项](#ZH-CN_MMLREF_0000001247358580__1.3.2)

## [命令功能](#ZH-CN_MMLREF_0000001247358580)

**适用NF：SMF、AMF、SMSF、NCG、NSSF**

该命令用于删除对端NF路由策略。

## [注意事项](#ZH-CN_MMLREF_0000001247358580)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0000001247358580)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001247358580)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| INDEX | 索引 | 可选必选说明：必选参数<br>参数含义：该参数用于指定索引。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是0~65535。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001247358580)

删除到对端NF的路由策略，索引为1。

```
RMV NFROUTEPLCY: INDEX=1;
```

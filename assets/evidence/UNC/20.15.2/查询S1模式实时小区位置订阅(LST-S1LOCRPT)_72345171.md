#### [命令功能](#ZH-CN_MMLREF_0000001172345171)

**适用网元：MME**

此命令用于查询S1模式实时小区位置订阅记录。

#### [注意事项](#ZH-CN_MMLREF_0000001172345171)

- 此命令执行后立即生效。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172345171)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172345171)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172345171)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| AREA | 区域范围 | 可选必选说明：可选参数<br>参数含义：该参数用于指定区域范围。<br>数据来源：整网规划<br>取值范围：<br>- “TA(指定跟踪区)”：表示该区域范围类型为指定跟踪区。<br>默认值：无 |
| MCC | 移动国家码 | 可选必选说明：条件可选参数<br>参数含义：该参数用于指定移动国家代码。<br>前提条件：该参数在“AREA(区域范围)”设置为“TA(指定跟踪区)”时，才需要配置。<br>数据来源：整网规划<br>取值范围：3位的十进制数字<br>默认值：无 |
| MNC | 移动网号 | 可选必选说明：条件可选参数<br>参数含义：该参数用于指定移动网号。<br>前提条件：该参数在“AREA(区域范围)”设置为“TA(指定跟踪区)”时，才需要配置。<br>数据来源：整网规划<br>取值范围：2～3位的十进制数字<br>默认值：无 |
| TAC | 跟踪区域码 | 可选必选说明：条件可选参数<br>参数含义：该参数用于指定跟踪区域码。<br>前提条件：该参数在“AREA(区域范围)”设置为“TA(指定跟踪区)”时，才需要配置。<br>数据来源：整网规划<br>取值范围：0x0000～0xFFFF。<br>默认值：无<br>说明：- 查询命令支持“TAC(跟踪区域码)”的范围匹配，即如果用户输入某TAC，则将查询对应TAC区段的配置记录。 |

#### [使用实例](#ZH-CN_MMLREF_0000001172345171)

查询所有S1模式实时小区位置订阅记录：

LST S1LOCRPT:;

```
%%LST S1LOCRPT:;%%
RETCODE = 0  操作成功。

操作结果如下
--------------
      区域范围  =  指定跟踪区
    移动国家码  =  456
      移动网号  =  12
    跟踪区域码  =  0x1234
跟踪区域码范围  =  0x1234
      订阅开关  =  打开
(结果个数 = 1)

---    END
```

## [命令功能](#ZH-CN_MMLREF_0000001875822980)

**适用NF：AMF**

在部署感知的场景下，通过LST SFGAPLE查询SFGAP本端实体信息。

## [注意事项](#ZH-CN_MMLREF_0000001875822980)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001875822980)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001875822980)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SFGAPLEIDX | SFGAP本端实体索引 | 可选必选说明：可选参数<br>参数含义：该参数用于指定SFGAP本端实体的索引，该索引作为SFGAP本端实体的唯一标识。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~1023。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001875822980)

输入SFGAP本端实体标识，查询指定的SFGAP本端实体，执行如下命令：

```
%%LST SFGAPLE:;%%
RETCODE = 0  操作成功

结果如下
------------------------
 SFGAP本端实体索引  =  1
SCTP本端实体组标识  =  1
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001875822980)

| 输出项名称 | 输出项解释 |
| --- | --- |
| SFGAP本端实体索引 | 该参数用于指定SFGAP本端实体的索引，该索引作为SFGAP本端实体的唯一标识。 |
| SCTP本端实体组标识 | 该参数用于指定SFGAP本端实体引用的SCTP本端实体组索引，一个SFGAP本端实体最多可引用32个SCTP本端实体。该本端实体组索引下最多配置32个本端实体。通过ADD SCTPLEGRP配置SCTP本端实体组。 |

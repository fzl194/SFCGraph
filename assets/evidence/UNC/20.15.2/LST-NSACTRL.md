#### [命令功能](#ZH-CN_MMLREF_0000001172345733)

**适用网元：MME**

该命令用于查询NSA组网场景下的整系统控制参数。

#### [注意事项](#ZH-CN_MMLREF_0000001172345733)

无。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172345733)

manage-ug；system-ug；monitor-ug。

#### [网管用户权限](#ZH-CN_MMLREF_0000001172345733)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组。

#### [参数说明](#ZH-CN_MMLREF_0000001172345733)

无。

#### [使用实例](#ZH-CN_MMLREF_0000001172345733)

查询NSA控制参数：

LST NSACTRL:;

```
%%LST NSACTRL:;%%
RETCODE = 0  操作成功。

输出结果如下
--------------
                       NR流量上报  =  否
DNS查询对等网元时是否携带NCNR标志  =  否
                         流量区分  =  否
    MME通知HSS是否为NSA用户的方式  =  功能关闭
            PSCell ID消息清除选项  =  NULL
(结果个数 = 1)

---    END
```

#### [输出结果说明](#ZH-CN_MMLREF_0000001172345733)

#### [命令功能](#ZH-CN_CONCEPT_0209896824)

**适用NF：PGW-C、SMF**

该命令用于显示每个POD上，当在线计费用户转离线后，每秒最大离线转回在线的用户数。

#### [注意事项](#ZH-CN_CONCEPT_0209896824)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0209896824)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0209896824)

无。

#### [使用实例](#ZH-CN_CONCEPT_0209896824)

查询计费速率控制：

```
LST CHGRATECTRL:;
```

```

RETCODE = 0  操作成功

计费速率控制
------------
在线计费恢复速率  =  500
融合计费恢复速率  =  500
(结果个数 = 1)

--- END
```

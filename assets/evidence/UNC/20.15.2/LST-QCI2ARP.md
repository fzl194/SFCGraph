# 查询标准QCI到ARP的映射规则（LST QCI2ARP）

## [命令功能](#ZH-CN_MMLREF_0209653066)

**适用NF：PGW-C、SMF**

该命令用于查询ARP和标准QCI的对应关系，从而通过标准QCI来对应出相应的承载ARP取值。

## [注意事项](#ZH-CN_MMLREF_0209653066)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209653066)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209653066)

无

## [使用实例](#ZH-CN_MMLREF_0209653066)

查询ARP和标准QCI的对应关系：

```
%%LST QCI2ARP:;%%
RETCODE = 0  操作成功
```

## [输出结果说明](#ZH-CN_MMLREF_0209653066)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 标准QCI | 该参数用于指定标准QCI。 |
| ARP的可抢占能力 | 该参数用于指示资源限制时，业务是否可以抢占更低优先级别的业务资源。 |
| ARP的优先级别 | 该参数表示ARP的优先级别。 |
| ARP的被抢占能力 | 该参数表示资源限制时，业务资源是否可以被更高优先级别的业务抢占。 |

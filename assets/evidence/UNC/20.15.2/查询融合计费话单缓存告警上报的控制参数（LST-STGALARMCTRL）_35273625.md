## [命令功能](#ZH-CN_MMLREF_0000001135273625)

**适用NF：SMF、PGW-C**

该命令用于查询融合计费话单缓存告警上报的控制参数。

## [注意事项](#ZH-CN_MMLREF_0000001135273625)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001135273625)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001135273625)

无

## [使用实例](#ZH-CN_MMLREF_0000001135273625)

查询告警产生门限，告警监控时长：

```
%%LST STGALARMCTRL:;%%
RETCODE = 0  操作成功

结果如下
--------
告警产生门限  =  10240
告警监控时长  =  120
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001135273625)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 告警产生门限(KB) | 该参数用于控制SMF上报缓存告警（ALM-81025）的缓存文件占用空间阈值。缓存文件超过该阈值时，上报告警；缓存文件全部被回放后，清除告警。 |
| 告警监控时长(分) | 当SMF检查到缓存文件存在且不超过占用空间阈值时，超过告警监控时长后，SMF上报缓存告警（ALM-81025）。 |

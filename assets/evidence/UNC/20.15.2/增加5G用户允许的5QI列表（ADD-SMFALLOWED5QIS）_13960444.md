## [命令功能](#ZH-CN_MMLREF_0000001213960444)

**适用NF：SMF**

该命令用于配置5G用户允许接入的5QI列表。当产品作为V-SMF/I-SMF形态时，通过该配置可以实现QoS管控功能。

## [注意事项](#ZH-CN_MMLREF_0000001213960444)

- 命令执行后只对新接入用户生效。

- 5QI可配置的范围为：1~9、65~67、69~76、79~80、82~86。
- 同一个“用户QOS索引”下配多个“5QI范围起始值”和"5QI范围结束值"，算多条记录。
- 同一个“用户QOS索引”下的“5QI范围起始值”和"5QI范围结束值"一共最多配置2048条。

- 最多可输入2048条记录。

#### [操作用户权限](#ZH-CN_MMLREF_0000001213960444)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001213960444)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| QOSINDEX | 用户QOS索引 | 可选必选说明：必选参数<br>参数含义：该参数表示允许用户接入的5QI列表索引。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~65535。<br>默认值：无<br>配置原则：无 |
| QOS5QISTART | 5QI范围起始值 | 可选必选说明：必选参数<br>参数含义：该参数用于指定允许接入的5QI范围的起始值。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是1~255。<br>默认值：无<br>配置原则：无 |
| QOS5QIEND | 5QI范围结束值 | 可选必选说明：可选参数<br>参数含义：该参数用于指定允许接入的5QI范围的结束值。<br>数据来源：全网规划<br>取值范围：整数类型，取值范围是1~255。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001213960444)

如果想增加"用户QOS索引"为1,"5QI范围起始值"为1,"5QI范围结束值"为2和"用户QOS索引"为1,"5QI范围起始值"为5,"5QI范围结束值"为9的5G用户允许的5QI列表，执行如下命令:

```
ADD SMFALLOWED5QIS:QOSINDEX=1,QOS5QISTART=1,QOS5QIEND=2;
ADD SMFALLOWED5QIS:QOSINDEX=1,QOS5QISTART=5,QOS5QIEND=9;
```

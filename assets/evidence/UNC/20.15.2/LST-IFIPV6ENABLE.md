#### [命令功能](#ZH-CN_CONCEPT_0000001600600793)

该命令用于查询使能了IPv6的接口的IPv6接口最大传输单元和自动生成链路层地址标志。

#### [注意事项](#ZH-CN_CONCEPT_0000001600600793)

- 该命令执行后立即生效。
- 该命令在VNRS_VNFC上的Ethernet接口，Ethernet子接口以及Loopback口，Tunnel口上配置执行。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600600793)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600600793)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IFNAME | 接口名 | 可选必选说明：可选参数<br>参数含义：该参数用于配置IPv6使能的接口名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。接口名称由接口类型+接口编号组成。<br>默认值：无<br>配置原则：请使用LST INTERFACE命令查看可用接口。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600600793)

查询接口的IPv6使能信息：

```
LST IFIPV6ENABLE:;
```

```
RETCODE = 0  操作成功

结果如下
-------------------------
接口名                  IPv6使能标志    IPv6接口最大传输单元（byte）    自动生成链路层地址标志
Ethernet64/0/3          使能            1500                            FALSE
Ethernet64/0/4          使能            1500                            FALSE
(结果个数 = 2)
---     END
```

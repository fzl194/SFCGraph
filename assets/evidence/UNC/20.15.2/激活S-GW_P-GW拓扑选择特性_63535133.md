## [操作场景](#ZH-CN_OPI_0163535133)

本操作指导介绍在运行网络中激活S-GW/P-GW拓扑选择特性的操作过程。

S-GW/P-GW拓扑选择特性即 UNC 按照不同的优选策略在手机附着等流程时选择合一的S-GW/P-GW（Serving Gateway/PDN Gateway），或拓扑最近的S-GW、P-GW节点，便于节约S5/S8接口流量，减少信令传输的时延。

> **说明**
> 适用于 SGSN、 MME。

## [操作步骤](#ZH-CN_OPI_0163535133)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)

  > **说明**
  > 此步骤在为不同范围的用户配置不同的GGSN/P-GW选择方法和机制的时候执行，需根据特定的业务调整参数。采用默认S-GW优选时，可不用配置选择S-GW优选或者P-GW优选策略。

## [任务示例](#ZH-CN_OPI_0163535133)

任务描述

MME上配置了本网的S5/S8协议类型为GTP。存在两个本网用户A、B。现网中存在S11口地址为10.2.3.10、S5口支持GTP协议的S-GW网元；存在S5口地址为10.11.1.10、S5口支持GTP的P-GW网元；不存在地址为10.11.1.11的P-GW网元或该网元故障。

**图2** Attach流程时选择S-GW/P-GW实例

<br>

![](激活S-GW_P-GW拓扑选择特性_63535133.assets/zh-cn_image_0205454413_2.png)

如 [图2](激活S-GW_P-GW拓扑选择特性_63535133.md#ZH-CN_OPI_0163535133__fig1) 所示，根据 [实例说明](实现原理_70166786.md#ZH-CN_TOPIC_0170166786__zh-cn_concept_0163535132_p5) 中的场景进行配置。

脚本

配置4个S-GW网元和4个P-GW网元，并将4个SGW关联到TAI-FQDN：TAC-LB01.TAC-HB11.TAC.EPC.MNC001.MCC123.3GPPNETWORK.ORG；将4个P-GW关联到APN-FQDN: HUAWEI1.COM.APN.EPC.MNC001.MCC123.3GPPNETWORK.ORG。根据业务需要，指定匹配的IMSI前缀为1230010000的用户，采用标准方式选择P-GW，并且优先策略为P-GW优选。

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2SELGW03", SWITCH=ENABLE;
```

//打开S-GW/P-GW拓扑选择开关。

```
SET SMFUNC: TOPSELCFG=YES;
```

//对指定用户选择S-GW优选或者P-GW优选策略。

```
ADD GWSELPLCY: SUBRANGE=IMSI_PREFIX, IMSIPRE="1230010000", APNNI="huawei1.com", SLCTM=STANDARD_SELECTION_MODE, SELPLCY=PGWFIRST;
```

//在MME上增加Hostfile文件的A解析记录。

//增加SGW1网元。

```
ADD IPV4DNSH: HSINDEX=1, HOSTNAME="topon.s5.gw32.ch.example.com", ADDRSECTION=SECTION1, IPV4ADDR1="10.2.3.10", PRIORITY1=10,WEIGHT1=100;
```

//增加SGW2网元。

```
ADD IPV4DNSH: HSINDEX=2, HOSTNAME="topon.board3.sgw3.ch.example.com", ADDRSECTION=SECTION1, IPV4ADDR1="10.2.3.11", PRIORITY1=11, WEIGHT1=101;
```

//增加SGW3网元。

```
ADD IPV4DNSH: HSINDEX=3, HOSTNAME="topoff.board3.sgw3.ch.example.com", ADDRSECTION=SECTION1, IPV4ADDR1="10.2.3.12", PRIORITY1=12, WEIGHT1=102;
```

//增加SGW4网元。

```
ADD IPV4DNSH: HSINDEX=4, HOSTNAME="topon.s5.sgw4.ch.example.com", ADDRSECTION=SECTION1, IPV4ADDR1="10.2.3.13", PRIORITY1=13, WEIGHT1=103;
```

//增加PGW1网元。

```
ADD IPV4DNSH: HSINDEX=5, HOSTNAME="topoff.S5.gw32.ch.example.com", ADDRSECTION=SECTION1, IPV4ADDR1="10.11.1.10", PRIORITY1=20, WEIGHT1=110;
```

//增加PGW2网元，将PGW2的IP地址配置为一个无效IP。

```
ADD IPV4DNSH: HSINDEX=6, HOSTNAME="topon.vip.pgw2.ch.example.com", ADDRSECTION=SECTION1, IPV4ADDR1="10.11.1.11", PRIORITY1=21, WEIGHT1=111;
```

//增加PGW3网元。

```
ADD IPV4DNSH: HSINDEX=7, HOSTNAME="topoff.vip.pgw3.ch.example.com", ADDRSECTION=SECTION1, IPV4ADDR1="10.11.1.12", PRIORITY1=22, WEIGHT1=112;
```

//增加PGW4网元。

```
ADD IPV4DNSH: HSINDEX=8, HOSTNAME="topoff.vip.pgw4.ch.example.com", ADDRSECTION=SECTION1, IPV4ADDR1="10.11.1.13", PRIORITY1=23, WEIGHT1=113;
```

//增加Hostfile文件的NAPTR解析记录。

//增加DNS通配记录。

```
ADD AREADNS: DNTYPE=SGW_TAI, TAC="0x1101", TACRANGE="0x1120";
```

//将SGW1关联到通配区域（TAC1101 - TAC1120）。

```
ADD DNSN: FQDN="TAC-LB01.TAC-HB11.TAC.EPC.MNC001.MCC123.3GPPNETWORK.ORG", HSINDEX=1, ENTITY=SGW, INTYPE=S11, S5PROTOCOL=GTP, PRIORITY=10, WEIGHT=104;
```

//将SGW2关联到TAC1101。

```
ADD DNSN: FQDN="TAC-LB01.TAC-HB11.TAC.EPC.MNC001.MCC123.3GPPNETWORK.ORG", HSINDEX=2, ENTITY=SGW, INTYPE=S11, S5PROTOCOL=GTP, PRIORITY=12, WEIGHT=103; 
```

//将SGW3关联到TAC1101。

```
ADD DNSN: FQDN="TAC-LB01.TAC-HB11.TAC.EPC.MNC001.MCC123.3GPPNETWORK.ORG", HSINDEX=3, ENTITY=SGW, INTYPE=S11, S5PROTOCOL=GTP, PRIORITY=11, WEIGHT=102;
```

//将SGW4关联到TAC1101。

```
ADD DNSN: FQDN="TAC-LB01.TAC-HB11.TAC.EPC.MNC001.MCC123.3GPPNETWORK.ORG", HSINDEX=4, ENTITY=SGW, INTYPE=S11, S5PROTOCOL=PMIP, PRIORITY=13, WEIGHT=101;
```

//将PGW1关联到“HUAWEI1.COM”。

```
ADD DNSN: FQDN="HUAWEI1.COM.APN.EPC.MNC001.MCC123.3GPPNETWORK.ORG", HSINDEX=5, ENTITY=PGW, INTYPE=S5, S5PROTOCOL=GTP, PRIORITY=20, WEIGHT=114; 
```

//将PGW2关联到“HUAWEI1.COM”。

```
ADD DNSN: FQDN="HUAWEI1.COM.APN.EPC.MNC001.MCC123.3GPPNETWORK.ORG", HSINDEX=6, ENTITY=PGW, INTYPE=S5, S5PROTOCOL=GTP, PRIORITY=21, WEIGHT=113;
```

//将PGW3关联到“HUAWEI1.COM”。

```
ADD DNSN: FQDN="HUAWEI1.COM.APN.EPC.MNC001.MCC123.3GPPNETWORK.ORG", HSINDEX=7, ENTITY=PGW, INTYPE=S5, S5PROTOCOL=GTP, PRIORITY=23, WEIGHT=112;
```

```
ADD DNSN: FQDN="HUAWEI1.COM.APN.EPC.MNC001.MCC123.3GPPNETWORK.ORG", HSINDEX=8, ENTITY=PGW, INTYPE=S5, S5PROTOCOL=GTP, PRIORITY=24, WEIGHT=111;
```

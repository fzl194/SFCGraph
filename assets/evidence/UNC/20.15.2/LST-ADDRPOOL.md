## [命令功能](#ZH-CN_MMLREF_0209652305)

**适用NF：GGSN、SMF、PGW-C**

该命令用于查询所有地址池或指定地址池下地址段的信息。

## [注意事项](#ZH-CN_MMLREF_0209652305)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209652305)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652305)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| POOLNAME | 地址池名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定地址池的名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~79。不支持空格及特殊字符“#”、“$”和“&”等，由“-”、“_”、数字、字母和“.”组成，不能以“.”开头，不区分大小写。<br>默认值：无<br>配置原则：无 |
| SINGLEIPFLAG | 子池类型 | 可选必选说明：可选参数<br>参数含义：该参数用于指定地址池子池类型。<br>数据来源：本端规划<br>取值范围：<br>- Enable（使能）<br>- Disable（去使能）<br>默认值：无<br>配置原则：<br>- 配置小地址池对性能影响较大，正常情况下不建议配置。<br>- 当现网中经常出现的多个SPU同用一个地址池时，配置小容量地址池主要解决由于地址分配不均导致在用户激活过程中无可用的地址而激活失败的问题。<br>- 由于原有的地址池分配处理中每个地址段最多可以划分为1024个子段，对于小地址池来说，每个子段只有一个地址，则小地址池中每个地址段的地址个数不能超过1024个。对于小地址池中总的地址个数不做限制，理论上每个小地址池最多可以配置1024*64=65536个地址。 |
| POOLID | 地址池ID标识 | 可选必选说明：可选参数<br>参数含义：该参数用于指定地址池ID标识。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是1~80000。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209652305)

- 查询地址池名称为“lap”的信息： LST ADDRPOOL: PoolName="lap";
  ```
  %%LST ADDRPOOL: PoolName="lap";%%
  RETCODE = 0  操作成功。

  结果如下
  ------------------------
           地址池名称  =  lap
             子池类型  =  去使能
           IP地址类型  =  IPv4
           地址池类型  =  本地
       白名单检查开关  =  去使能
              绑定VPN  =  去使能
            VPN实例名  =  NULL
     地址租约使能标志  =  去使能
         地址租期(秒)  =  10
              IMS开关  =  使能
     是否允许地址重叠  =  去使能
       地址池锁定标志  =  解锁

  (结果个数= 1)

  ---    END
  ```
- 查询所有的地址池信息： LST ADDRPOOL:;
  ```
  %%LST ADDRPOOL:;%%
  RETCODE = 0  操作成功。

  结果如下
  ------------------------
  地址池名称  子池类型  IP地址类型  地址池类型  白名单检查开关  绑定VPN  VPN实例名  地址租约使能标志  地址租期(秒)  IMS开关  是否允许地址重叠   地址池锁定标志

  Pool1       去使能    IPv4         本地       去使能          去使能   NULL       去使能             10            使能     去使能             解锁
  Pool2       去使能    IPv6         本地       去使能          去使能   NULL       去使能             10            使能     去使能             解锁
  lap         去使能    IPv4         本地       去使能          去使能   NULL       去使能             10            使能     去使能             解锁
  lap2        去使能    IPv4         本地       去使能          去使能   NULL       使能               16            使能     使能               解锁
  (结果个数 = 4)

  ---    END
  ```

## [输出结果说明](#ZH-CN_MMLREF_0209652305)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 地址池名称 | 该参数用于指定地址池的名称。 |
| 子池类型 | 该参数用于指定地址池子池类型。 |
| IP地址类型 | 该参数用于指定IP地址类型。 |
| 地址池类型 | 该参数用于指定地址池类型。 |
| 白名单检查开关 | 该参数用于指定静态地址用户的白名单检查开关。 |
| 绑定VPN | 该参数用于指定是否绑定VPN。 |
| VPN实例名 | 该参数用于指定VPN实例名称。 |
| 地址租约使能标志 | 该参数指定是否使能IP地址租约功能。 |
| 地址租期(秒) | 该参数指定用户释放之后，对应用户IP地址的等待释放时间。取值为0时，表示用户去激活后，其使用的本地地址将被立即释放，且可重新分配给其他用户使用。取值非0时，表示用户去激活后，其使用的地址不会立即释放，而是等待配置的时间之后才能被使用。 |
| IMS开关 | 指定使能IP地址租约功能是否对IMS用户有效。 |
| 是否允许地址重叠 | 指定该地址池是全局地址池，还是局部地址池，局部地址池允许地址重叠。 |
| 地址池锁定标志 | 该参数用于指定地址池锁定标志。 |
| 是否分配特殊IP地址 | 该参数用于指定是否分配特殊IP地址。IP地址中主机号全0或全1的IP地址是特殊IP地址。 |
| 是否开启地址使用率的告警功能 | 该参数用于指定告警“ALM-100297 地址池组使用率超阈值”是否对本地址池内IP地址使用率生效。 |
| 地址池ID标识 | 该参数用于指定地址池ID标识。 |

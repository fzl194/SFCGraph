# 调测支持FWA接入

- [操作场景](#ZH-CN_OPI_0000002053227418__1.3.1)
- [必备事项](#ZH-CN_OPI_0000002053227418__1.3.2)

## [操作场景](#ZH-CN_OPI_0000002053227418)

本操作指导介绍在运行网络中调测支持FWA接入的操作过程。

## [必备事项](#ZH-CN_OPI_0000002053227418)

前提条件

- 请仔细阅读[WSFD-010112 支持FWA接入特性概述](WSFD-010112 支持FWA接入特性概述_16422889.md)。
- 完成[激活支持FWA接入（AMF）](激活支持FWA接入（AMF）_22148921.md)、[激活支持FWA接入（SMF）](激活支持FWA接入（SMF）_89264753.md)。

数据

该操作无需准备数据。

## [操作步骤](#ZH-CN_OPI_0000002053227418)

1. FWA测试终端接入网络，建立PDU会话。
2. 进入 “MML命令行-UNC” 窗口。
3. 查询当前在线的FWA用户的PDP上下文数。
  **[DSP SMPDPNUM](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/接入管理运维/查询PDP上下文数/查询会话管理的PDP上下文数（DSP SMPDPNUM）_09653799.md)** : QRY_SCOPE=SUMMARY, QRY_CLASS=FWA;
  预期结果：当前在线的FWA用户的PDP上下文数与实际规划一致。
  ```
  %%DSP SMPDPNUM: QRY_SCOPE=SUMMARY, QRY_CLASS=FWA;%%
  RETCODE = 0  操作成功

  结果如下
  --------
                                    查询分类  =  FWA
                                     ...
               N11接口SMF上激活的QFI上下文数  =  1
                              QFI上下文总数   =  1
                                     ...

  (结果个数 = 1)

  ---    END
  ```
4. 查询当前在线的FWA用户的会话上下文数。
  **[DSP SMSESSIONNUM](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/接入管理运维/查询会话上下文数/显示会话管理的会话上下文数（DSP SMSESSIONNUM）_09652085.md)** : QRY_SCOPE=SUMMARY, QRY_CLASS=FWA;
  预期结果：当前在线的FWA用户的会话上下文数与实际规划一致。
  ```
  %%DSP SMSESSIONNUM: QRY_SCOPE=SUMMARY, QRY_CLASS=FWA;%%
  RETCODE = 0  操作成功

  结果如下
  --------
                                   查询分类  =  FWA
                                     ...
           N11接口SMF上激活的SA会话上下文数  =  1
                          SA会话上下文总数   =  1
                                     ...

  (结果个数 = 1)

  ---    END
  ```

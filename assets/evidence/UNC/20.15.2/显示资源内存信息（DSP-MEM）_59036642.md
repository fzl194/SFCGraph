#### [命令功能](#ZH-CN_CONCEPT_0259036642)

该命令用于显示资源的内存使用率信息，包括总内存、已使用的内存、剩余的内存及内存使用率。

在日常的维护活动中，可使用本命令查看系统当前的内存信息，以确认内存使用是否处于合理的区间。

#### [注意事项](#ZH-CN_CONCEPT_0259036642)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0259036642)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0259036642)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RESOURCENAME | 资源名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定要查询CPU使用率的资源名称。通过<br>[**DSP RES**](../资源实例管理/显示资源信息（DSP RES）_59036939.md)<br>命令可以查询资源信息。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，区分大小写。<br>默认值：无<br>配置原则：当不输入时显示所有资源的信息。 |

#### [使用实例](#ZH-CN_CONCEPT_0259036642)

- 显示所有资源的内存使用率：
  ```
  DSP MEM:;
  ```
  ```
  RETCODE = 0  操作成功

  显示资源内存信息
  -------------------------
  资源名称    系统总内存（Kbytes）       系统已用内存（Kbytes）    系统剩余内存（Kbytes）    系统内存使用率（%）
  OMU1        18946096                   3423360                   15522736                  18                 
  OMU2        18946096                   2714824                   16231272                  14                 
  (结果个数 = 2)
  ---    END
  ```
- 显示名为OMU1的资源的内存使用率：
  ```
  DSP MEM: RESOURCENAME="OMU1";
  ```
  ```
  RETCODE = 0  操作成功

  显示资源内存信息
  -------------------------
                资源名称  =  OMU1
    系统总内存（Kbytes）  =  7866764
  系统已用内存（Kbytes）  =  1177836
  系统剩余内存（Kbytes）  =  6688928
     系统内存使用率（%）  =  14
  (结果个数 = 1)
  ---    END
  ```

#### [输出结果说明](#ZH-CN_CONCEPT_0259036642)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 资源名称 | 资源名称。 |
| 系统总内存（Kbytes） | 用于说明资源的总内存大小（单位：Kbytes）。 |
| 系统已用内存（Kbytes） | 用于说明资源已经被分配的内存大小，被分配的内存即为被使用内存（单位：Kbytes）。 |
| 系统剩余内存（Kbytes） | 用于说明资源未被分配的内存大小，未被分配的内存即为剩余内存（单位：Kbytes）。 |
| 系统内存使用率（%） | 用于说明当前资源内存使用率（单位：%）。 |

#### [命令功能](#ZH-CN_CONCEPT_0000001258631837)

**适用NF：MME、AMF**

该命令用于查询系统反压流控检测门限参数。

#### [注意事项](#ZH-CN_CONCEPT_0000001258631837)

无

#### [操作用户权限](#ZH-CN_CONCEPT_0000001258631837)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001258631837)

无

#### [使用实例](#ZH-CN_CONCEPT_0000001258631837)

查询PAE寻呼反压流控启动/恢复阈值：

```
%%LST REVERSEFCPARA:;%%
RETCODE = 0  操作成功

操作结果如下
------------------------ 
      控制策略  =  百分比
 过载阈值（%）  =  50
 恢复阈值（%）  =  5
过载阈值（个）  =  1
恢复阈值（个）  =  0
(结果个数 = 1)

---    END
```

## [命令功能](#ZH-CN_MMLREF_0000001232766175)

**适用NF：NCG**

该命令用于查询AGF基于TOPO异常响应处理动作。

## [注意事项](#ZH-CN_MMLREF_0000001232766175)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001232766175)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001232766175)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RCTYPE | RC类型 | 可选必选说明：可选参数<br>参数含义：该参数用于指定RC的类型。<br>数据来源：本端规划<br>取值范围：<br>- DEFAULT（针对未指定的异常返回码设置处理动作）<br>- VALUE（针对指定异常返回码设置处理动作）<br>- TIMEOUT（等待响应超时）<br>默认值：无<br>配置原则：无 |
| CODETYPE | 指定异常返回码类型 | 可选必选说明：该参数在"RCTYPE"配置为"VALUE"时为条件可选参数。<br>参数含义：该参数用于指定异常返回码的类型。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是300~999，0~10。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001232766175)

查询AGF基于TOPO异常响应处理动作:

```
LST AGFDISCRCACT:;
RETCODE = 0  操作成功

结果如下
--------
            RC类型  =  针对指定的异常返回码设置处理动作
指定异常返回码类型  =  900
  Failover模板标识  =  ccpfot1
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001232766175)

| 输出项名称 | 输出项解释 |
| --- | --- |
| RC类型 | 该参数用于指定RC的类型。 |
| 指定异常返回码类型 | 该参数用于指定异常返回码的类型。 |
| Failover模板标识 | 该参数用于指定融合计费Proxy Failover模板的名称。 |

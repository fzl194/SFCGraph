## [命令功能](#ZH-CN_MMLREF_0247815993)

**适用NF：SMF、AMF、NRF、NSSF、NCG**

该命令用于查询切片实例管理。

## [注意事项](#ZH-CN_MMLREF_0247815993)

无

#### [操作用户权限](#ZH-CN_MMLREF_0247815993)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0247815993)

无

## [使用实例](#ZH-CN_MMLREF_0247815993)

查询所有的切片实例管理：

```
%%LST SLICEINSTINFO:;%%
RETCODE = 0  操作成功

结果如下
--------
    网络分片实例信息  =  1
网络分片子网实例标识  =  1
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0247815993)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 网络分片实例信息 | 该参数用于指定网络分片实例信息。 |
| 网络分片子网实例标识 | 该参数用于指定网络分片子网实例标识。 |

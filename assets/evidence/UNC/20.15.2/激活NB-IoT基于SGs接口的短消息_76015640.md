## [操作场景](#ZH-CN_OPI_0276015640)

本操作指导介绍在运行网络中激活NB-IoT基于SGs接口的短消息特性的操作过程。

本特性支持NB-IoT终端发起EPS网络的短消息传输。NB-IoT终端可以不用发起联合的EPS/IMSI附着，只需附着到EPS网络通过MME与MSC的交互实现短消息数据传输业务，丰富了NB-IoT的数传方式。

> **说明**
> 适用于MME。

## [操作步骤](#ZH-CN_OPI_0276015640)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开SGs短信开关。
    a. 查询某指定用户范围的相关配置。
      [**LST M2MPLCY**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/M2M管理/M2M策略参数配置/查询M2M策略参数(LST M2MPLCY)_26145766.md)
    b. **可选：**如果 [步骤 2.a](#ZH-CN_OPI_0276015640__step_a) 未查询到该指定用户范围的相关配置，则执行此步骤，为该用户范围打开SGs短信开关。
      [**ADD M2MPLCY**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/M2M管理/M2M策略参数配置/增加M2M策略参数(ADD M2MPLCY)_72225443.md)
    c. **可选：**如果 [步骤 2.a](#ZH-CN_OPI_0276015640__step_a) 查询到系统已存在该指定用户范围的相关配置，且SGs短信开关未打开时，则执行此步骤。
      [**MOD M2MPLCY**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/M2M管理/M2M策略参数配置/修改M2M策略参数(MOD M2MPLCY)_72345365.md)
3. 打开本功能的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)

## [任务示例](#ZH-CN_OPI_0276015640)

任务描述

启用NB-IoT基于SGs接口的短消息。

脚本

//打开SGs短信开关。

```
ADD M2MPLCY: SUBRANGE=IMSI_PREFIX, IMSIPRE="123037301", APNNI="HUAWEI1.com", PSMSW=OFF, EDRXSW=OFF, CPOPSW=NOT_SUPPORT, SGSSMSSW=ON;
```

//打开本特性的License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2GSDT01", SWITCH=ENABLE;
```

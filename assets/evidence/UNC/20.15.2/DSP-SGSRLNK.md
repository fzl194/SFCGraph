#### [命令功能](#ZH-CN_MMLREF_0000001414563449)

**适用NF：SMSF**

此命令用于查看SGS服务端链路状态。

#### [注意事项](#ZH-CN_MMLREF_0000001414563449)

无

#### [本地用户权限](#ZH-CN_MMLREF_0000001414563449)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001414563449)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001414563449)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SRT | 查询方式 | 可选必选说明：可选参数<br>参数含义：该参数用于指定查询SGS服务端链路状态的方式。<br>取值范围：<br>- “ALL(所有)”：查询所有SGS服务端链路的状态。<br>- “LNK(链路)”：通过指定链路索引查询SGS服务端链路的状态。<br>- “LSX(链路集)”：通过指定链路集查询SGS服务端链路的状态。<br>默认值：无 |
| LNK | 链路索引 | 可选必选说明：条件可选参数<br>参数含义：待查询链路的索引。<br>前提条件：该参数在<br>“查询方式”<br>设置为<br>“LNK(链路)”<br>时生效。<br>取值范围：0~6400<br>默认值：无 |
| LSX | 链路集索引 | 可选必选说明：条件可选参数<br>参数含义：待查询链路所属链路集的索引。<br>前提条件：该参数在<br>“查询方式”<br>设置为<br>“LSX(链路集)”<br>时生效。<br>取值范围：0~2000<br>默认值：无 |

#### [使用实例](#ZH-CN_MMLREF_0000001414563449)

1. 查询系统内所有的SGS服务端链路状态信息：
  DSP SGSRLNK: SRT=ALL;
  ```
  %%DSP SGSRLNK: SRT=ALL;%%
  RETCODE = 0  操作成功

  操作结果如下
  ------------
  RU名称           进程号  链路索引  链路集索引  VPN名称  VLR号  链路状态  

  LINK_SP_RU_0064  0       2         1           NULL     1      故障      
  LINK_SP_RU_0064  1       1         1           NULL     1      故障      
  (结果个数 = 2)

  ---    END
  ```
2. 查询指定链路索引的SGS服务端链路状态信息：
  DSP SGSRLNK: SRT=LNK, LNK=1;
  ```
  %%DSP SGSRLNK: SRT=LNK, LNK=1;%%
  RETCODE = 0  操作成功

  操作结果如下
  ------------
      RU名称  =  LINK_SP_RU_0064
      进程号  =  1
    链路索引  =  1
  链路集索引  =  1
     VPN名称  =  NULL
       VLR号  =  1
    链路状态  =  故障
  (结果个数 = 1)

  ---    END
  ```

#### [输出结果说明](#ZH-CN_MMLREF_0000001414563449)

| 输出项名称 | 输出项解释 |
| --- | --- |
| RU名称 | SGS服务端链路的资源单位名称。<br>取值说明：1～63位字符串 |
| 进程号 | SGS服务端链路的进程号。<br>取值说明：0~11 |
| 链路索引 | SGS服务端链路的索引。<br>取值说明：0~6400 |
| 链路集索引 | SGS服务端链路所属链路集的索引。<br>取值说明：0~2000 |
| VPN名称 | SGS服务端链路的VPN名称。<br>取值说明：1～31位字符串 |
| VLR号 | SGS服务端链路的VLR号。<br>取值说明：1~16位十进制数字 |
| 链路状态 | SGS服务端链路所处状态。<br>取值说明：<br>- 正常：表示SMSF/VLR与MME之间的通信在该链路上正常。<br>- 故障：表示SMSF/VLR与MME之间的通信在该链路上中断，需要检查网络状况，分析故障原因。详细操作参见[ALM-80709 SGs Server链路故障](../../../../../../../../../网络运维/故障处理/UNC告警处理/业务告警/SMSF/ALM-80709 SGs Server链路故障_13960689.md)告警处理。<br>- 去激活：表示该SGS服务端链路处于去激活状态。 |

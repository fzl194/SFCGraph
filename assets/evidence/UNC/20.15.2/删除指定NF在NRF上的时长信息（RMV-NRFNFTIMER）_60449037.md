# 删除指定NF在NRF上的时长信息（RMV NRFNFTIMER）

- [注意事项](#ZH-CN_MMLREF_0000001360449037__1.3.2)

## [命令功能](#ZH-CN_MMLREF_0000001360449037)

**适用NF：NRF**

该命令用于删除指定NF在NRF上的时长信息。

## [注意事项](#ZH-CN_MMLREF_0000001360449037)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0000001360449037)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001360449037)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| NFINSTANCEID | NF实例标识 | 可选必选说明：必选参数<br>参数含义：该参数用于表示NF实例标识。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~36。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001360449037)

运营商不需要在NRF上单独设置实例标识为"88888888-6666-1234-5678-123456789ABC"的时长信息时，执行如下命令。

```
RMV NRFNFTIMER: NFINSTANCEID="88888888-4444-1234-5678-123456789ABC";
```

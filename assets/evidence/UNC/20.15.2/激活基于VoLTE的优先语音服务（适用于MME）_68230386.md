## [操作场景](#ZH-CN_OPI_0168230386)

本操作指导介绍在运行网络中激活基于VoLTE的优先语音服务业务特性的操作过程，包括两种方案，配置MPS业务功能和配置语音寻呼优先业务功能。

*表1 优先级方案对比*

| 方案 | 是否需要用户签约 | 应用业务场景 | 部署建议 |
| --- | --- | --- | --- |
| 语音寻呼优先业务（推荐） | 否 | 仅针对VoLTE/CSFB被叫业务 | VoLTE组网场景下推荐部署本方案，以提升被叫寻呼成功率，提升用户语音业务体验。 |
| MPS业务 | 是 | 针对所有业务 | MPS业务的优先级高于语音寻呼优先业务，该方案可提升特定签约用户享用网络资源的优先级，但部署后特定签约用户使用LTE精准寻呼和智能寻呼功能时会失效，VoLTE组网场景下不推荐部署。 |

> **说明**
> 适用于MME。

## [必备事项](#ZH-CN_OPI_0168230386)

前提条件

- 请仔细阅读[WSFD-102004 基于VoLTE的优先语音服务特性概述](特性概述_70010134.md)。

- 本特性需要在MME、eNodeB、SGW-C/PGW-C、PCRF、SPR、P-CSCF等网元上完成基本业务的数据配置。
- 用户需要在HSS签约相应数据。

## [操作步骤](#ZH-CN_OPI_0168230386)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. **推荐：** 部署语音寻呼优先业务。
  [**SET MMFUNC**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/MM扩展功能管理/设置移动性管理扩展功能(SET MMFUNC)_26145512.md)
4. **可选：** 部署MPS业务。
  [**SET MPSARP**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/MPS配置/MPS ARP信息配置/设置MPS ARP配置(SET MPSARP)_26305312.md)
  > **说明**
  > 步骤3和步骤4部署时二选一，推荐部署语音寻呼优先业务，部署MPS业务后签约用户使用LTE精准寻呼和智能寻呼功能会失效。

## [任务示例](#ZH-CN_OPI_0168230386)

任务描述

任务一（推荐方案）：启用基于VoLTE的优先语音服务业务，选择语音寻呼优先业务方案。

任务二：启用基于VoLTE的优先语音服务业务，选择MPS业务方案，设置MPS的ARP优先级临界值为7。

脚本

// 进入 “MML命令行-UNC” 窗口。

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2EMPS01", SWITCH=ENABLE;
```

//配置优先级功能。

//方案一（推荐方案）：打开语音寻呼优先功能。

```
SET MMFUNC: VOICEPAGINGPRIORITY=YES;
```

//方案二：开启MPS功能开关，并设置优先级临界值。

```
SET MPSARP: MPSFUNC=ON, PRILVL=7;
```

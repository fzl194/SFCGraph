## [操作场景](#ZH-CN_OPI_0168358257)

本操作指导介绍在运行网络中激活LCS特性的操作过程。

位置定位服务LCS（Location Services）是一种获取手机在移动通信系统中的位置，并对位置信息进行相应的转换、计算，提供出地理坐标值的业务。

## [必备事项](#ZH-CN_OPI_0168358257)

前提条件

- 请仔细阅读[WSFD-106401 位置定位服务（LCS）（适用于SGSN）特性概述](特性概述_68358254.md)。
- 请先完成Lg接口数据配置，请参见[配置到GMLC的数据（Over IP）](../../../../../初始配置/UNC初始配置与调测/组网对接配置/配置AMF&MME&SGSN/配置SGSN/配置到GMLC的数据/配置到GMLC的数据（Over IP）_82467136.md)或[配置到GMLC的数据（SG Over IP）](../../../../../初始配置/UNC初始配置与调测/组网对接配置/配置AMF&MME&SGSN/配置SGSN/配置到GMLC的数据/配置到GMLC的数据（SG Over IP）_29306561.md)。
- 完成加载License。

## [操作步骤](#ZH-CN_OPI_0168358257)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. 打开LCS开关。
    - 打开用户归属PLMN的LCS功能允许开关。
      [**ADD HPLMN**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/归属网络运营商管理/MNO管理/MNO网络配置表/增加本地PLMN(ADD HPLMN)_26146074.md)
      > **说明**
      > 如果该本地PLMN已存在，请使用 [**MOD HPLMN**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/归属网络运营商管理/MNO管理/MNO网络配置表/修改本地PLMN(MOD HPLMN)_26305884.md) 命令。
    - (可选)打开漫游场景下的LCS开关。
      [**ADD CONNECTPLMN**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/互联PLMN管理/增加互联PLMN(ADD CONNECTPLMN)_26305852.md)
      > **说明**
      > 如果该互联PLMN已存在，请使用 [**MOD CONNECTPLMN**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/互联PLMN管理/修改互联PLMN(MOD CONNECTPLMN)_26146044.md) 命令。
    - (可选)2G场景下，打开LCS开关。
      [**MOD NSE**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/Gb接口管理/信令实体管理/修改信令实体(MOD NSE)_26305838.md)
      > **说明**
      > - 如果BSC和SGSN之间存在多个NSE，则需要针对所有NSE设置参数“是否支持LCS”为“YES(是)”。
      > - 如果后续BSC与SGSN之间需要新增NSE，同样需要设置参数“是否支持LCS”为“YES(是)”。
4. **可选：**配置LCS参数。
  [**SET LCSPARA**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/LCS/LCS软件参数表/设置LCS参数表记录(SET LCSPARA)_72225477.md)
  > **说明**
  > 在系统初始化的时候，已经为LCS参数设置了缺省值，一般不需用户修改。
  >
  > 如果GMLC限定属于HPLMN，则要求 [**ADD GMLC**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/LCS/GMLC配置/增加GMLC配置(ADD GMLC)_26145796.md) 命令中配置的 “GMLC所属MCC” 、 “GMLC所属MNC” 与 [**ADD HPLMN**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/归属网络运营商管理/MNO管理/MNO网络配置表/增加本地PLMN(ADD HPLMN)_26146074.md) 命令中配置的HPLMN相同，且发起请求的GMLC的GMLC number必须在 [**ADD GMLC**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/LCS/GMLC配置/增加GMLC配置(ADD GMLC)_26145796.md) 命令中已经配置。否则， UNC 会拒绝用户定位请求。
5. 配置LCS Client与所属的GMLC号码的对应关系。
  [**ADD GMLCCLIENT**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/LCS/GMLC和LCS Client对照表/增加GMLC和LCS Client对照关系(ADD GMLCCLIENT)_72225481.md)
  > **说明**
  > 在移动始发的位置业务中，SGSN根据手机发送信息中携带的LCS Client信息，和配置的LCS Client与所属GMLC号码的对应关系，获得对应的GMLC。
6. 配置 UNC 进行本地定位时，SAI与地理位置经纬度坐标的对照关系。
  [**ADD SAIGEO**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/LCS/SAI和地理坐标对照表/增加服务区标识与地理坐标对照关系(ADD SAIGEO)_72345391.md)
  > **说明**
  > SGSN利用SAI与地理位置经纬度坐标的对照关系，完成SAI与地理位置经纬度坐标之间的转换，提供给LCS MT/NI/MO流程使用。
7. 配置 UNC 进行本地定位时，小区地理信息表。
  [**ADD CELLGEO**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/LCS/小区地理信息/增加CELLID与地理坐标对应关系(ADD CELLGEO)_26305598.md)
  > **说明**
  > SSGSN利用小区地理信息完成CID与地理位置经纬度坐标之间的转换，提供给LCS MT/NI流程使用。
8. 配置SGSN控制面IP地址与SGSN号码对照关系。
  [**ADD SGSNIPNUM**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/LCS/SGSN地址和SGSN号码对照表/增加SGSN控制面IP地址与SGSN号码对应关系(ADD SGSNIPNUM)_26305614.md)
  > **说明**
  > 在延迟定位过程中，当用户在其它SGSN重现时，本SGSN需要根据用户所重现的SGSN的IP地址来获得该SGSN号码，并在SGSN给GMLC的上报消息中携带此SGSN号码传送给GMLC，使GMLC能根据此SGSN号码向用户重现的SGSN重新发起定位流程。

## [任务示例](#ZH-CN_OPI_0168358257)

任务描述

激活LCS功能。

脚本

//设置本特性License开关为“ENABLE”。

```
SET LICENSESWITCH: LICITEM="LKV2LCS02", SWITCH=ENABLE;
```

//打开用户归属PLMN的LCS功能允许开关。

```
ADD HPLMN：MCC="460", MNC="00",LCS=YES;
```

//可选：打开漫游场景下的LCS开关。

```
ADD CONNECTPLMN：MCC="461", MNC="03",LCS=YES;
```

//可选：在2G场景下，打开LCS开关。

```
MOD NSE: NSEI=2740, LCS=YES;
```

//可选：配置LCS参数。

```
SET LCSPARA: GLHP=NO_LIM, LCS2GPOLICY=LOCAL_POSITION;
```

//增加GMLC和LCS Client的对照关系，LCS Client的号码为861380123456789，GMLC的号码为861390123456789。

```
ADD GMLCCLIENT: CLIENTNUM="861380123456789", GMLCNUM="861390123456789";
```

//增加SAI为4600907551234与北纬5度15分25秒西经35度45分55秒，服务区半径为35000米的地理位置坐标的对应关系。

```
ADD SAIGEO: SAI="4600907551234", LAT=NLA, LAD=5, LAM=15, LAS=25, LOT=WLO, LOD=35, LOM=45, LOS=55, RAD=35000; 
```

//增加CID为460090755123400与北纬5度15分25秒西经35度45分55秒，小区半径为5500米的地理位置坐标的对应关系。

```
ADD CELLGEO: CID="460090755123400", LAT=NLA, LAD=5, LAM=15, LAS=25, LOT=WLO, LOD=35, LOM=45, LOS=55, RAD=5500;
```

//增加SGSN控制面IP地址与SGSN号码的对照关系，SGSN控制面IP地址为10.2.3.4，SGSN号码为861390123456789。

```
ADD SGSNIPNUM: IPV4="10.2.3.4", SGSNNUM="861390123456789";
```

## [命令功能](#ZH-CN_MMLREF_0000001870382329)

该命令用于查询NWDAF的实例信息。

## [注意事项](#ZH-CN_MMLREF_0000001870382329)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001870382329)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001870382329)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| INSTANCENAME | NWDAF实例名称 | 可选必选说明：可选参数<br>参数含义：该参数用于唯一指定某一个NWDAF实例。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是1~50。本参数的构成字符只能是字母A~Z或a~z、数字0~9、下划线“_”和中划线“-”，例如，NWDAF_Instance_0。<br>默认值：无<br>配置原则：<br>本参数来源于ADD NFUUID命令中的“NF实例名称”参数。 |

## [使用实例](#ZH-CN_MMLREF_0000001870382329)

查询NWDAF信息。

```
%%LST NWDAFINFO:;%%
RETCODE = 0 操作成功

结果如下
--------
    NWDAF实例名称  =  NWDAF_Instance_0
NWDAF数据分析事件  =  QOS分析
(结果个数 = 1)

----    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001870382329)

| 输出项名称 | 输出项解释 |
| --- | --- |
| NWDAF实例名称 | 该参数用于唯一指定某一个NWDAF实例。 |
| NWDAF数据分析事件 | 该参数用于指定NWDAF支持的分析事件类型。 |

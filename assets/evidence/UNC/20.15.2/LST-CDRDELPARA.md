#### [命令功能](#ZH-CN_CONCEPT_0251174270)

**适用NF：NCG**

该命令用于查询当前话单删除、话单备份和话单分发任务的话单删除参数信息。

#### [注意事项](#ZH-CN_CONCEPT_0251174270)

无。

#### [本地用户权限](#ZH-CN_CONCEPT_0251174270)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_CONCEPT_0251174270)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0251174270)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| CDRDELOBJ | 话单删除对象 | 可选必选说明：可选参数<br>参数含义：用于修改话单删除、话单备份和话单分发三种类型任务的话单删除参数。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- CDRDELTASK：话单删除。<br>- CDRBACKUP：话单备份。<br>- CDRDISTR：话单分发。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0251174270)

查询话单删除参数：

```
LST CDRDELPARA:;
```

```
RETCODE = 0  操作成功。

结果如下:
--------
话单删除对象    话单删除开始时间    话单删除时间间隔(毫秒) 

话单删除        04:00               20                     
话单备份        04:00               1000                   
话单分发        04:00               1000                   
(结果个数 = 3)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0251174270)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 话单删除对象 | 用于修改话单删除、话单备份和话单分发三种类型任务的话单删除参数。 |
| 话单删除开始时间 | 用于设置话单删除、话单备份或分发任务的话单删除开始时间，系统默认04:00开始话单过期删除任务。 |
| 话单删除时间间隔(毫秒) | 用于设置话单删除、话单备份或分发任务的话单删除休眠时间。话单删除任务每删除1个话单文件休眠此参数时间间隔，话单备份分发任务每删除1000个文件休眠此参数时间间隔。 |

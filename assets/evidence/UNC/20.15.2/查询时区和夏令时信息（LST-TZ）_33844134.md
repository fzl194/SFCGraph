## [命令功能](#ZH-CN_MMLREF_0233844134)

本命令用于查询时区和夏令时信息。

## [注意事项](#ZH-CN_MMLREF_0233844134)

当 “标准时类型” 配置为 “UTC(UTC)” 时，查询结果中 “时区” 显示为UTC，否则显示为GMT。

## [参数说明](#ZH-CN_MMLREF_0233844134)

无。

## [使用实例](#ZH-CN_MMLREF_0233844134)

星期输入方式时区信息查询结果如下：

```
%%LST TZ:;%% 
RETCODE = 0  操作成功  
操作结果如下
------------               
                时区  =  UTC+08:00     
      是否执行夏令时  =  是         
        日期输入方式  =  星期方式          
            开始月份  =  1月 
    开始的第几个星期  =  第一个  
            开始星期  =  星期日       
            开始时间  =  18:54           
            结束月份  =  2月
    结束的第几个星期  =  第二个   
            结束星期  =  星期日       
            结束时间  =  18:55
  时间偏移量（分钟）  =  10    
 (结果个数 = 1)  
---    END 
```

日期输入方式时区信息查询结果如下：

```
%%LST TZ:;%% 
RETCODE = 0  操作成功  
操作结果如下 
------------               
                 时区  =  GMT+08:00     
       是否执行夏令时  =  是         
         日期输入方式  =  日期方式          
             开始月份  =  1月             
               开始日  =  2           
             开始时间  =  18:49           
             结束月份  =  2月             
               结束日  =  1           
             结束时间  =  18:50 
   时间偏移量（分钟）  =  10 
  (结果个数 = 1)  
---    END 
```

## [输出结果说明](#ZH-CN_MMLREF_0233844134)

命令执行正常，会返回命令执行成功的提示信息。输出结果说明如 [表1 输出项说明](#ZH-CN_MMLREF_0233844134__table1384221010309) 所示。

*表1 输出项说明*

| 输出项名称 | 输出项解释 |
| --- | --- |
| 时区 | 用于指定设备所处时区，其值域类型为字符串，结果显示+表示东时区，-表示西时区。 |
| 是否执行夏令时 | 用于指定是否配置夏令时：<br>如果支持夏令时则取值为“是”；反之为“否”； |
| 日期输入方式 | 用于指定日期的输入方式：<br>如果为日期模式则为“日期方式”，如果为星期模式则为“星期方式”。 |
| 开始月份 | 在支持夏令时的情况下，开始月份取值可以为：1~12月。 |
| 开始日 | 在支持夏令时并且日期输入方式为"Date(日期方式)"的情况下，日期的取值为1~31。 |
| 开始时间 | 在支持夏令时的情况下，开始时间其格式为“HH:MM”，“HH”代表时，“MM”代表分。<br>取值范围：00:00～23:59。 |
| 结束月份 | 在支持夏令时的情况下，结束月份取值可以为：1~12月。 |
| 结束日 | 在支持夏令时并且日期输入方式为"Date(日期方式)"的情况下，结束日期的取值为1~31。 |
| 结束时间 | 在支持夏令时的情况下，结束时间其格式为“HH:MM”，“HH”代表时，“MM”代表分。<br>取值范围：00:00～23:59。 |
| 开始的第几个星期 | 在支持夏令时并且日期输入方式为"Week(星期方式)"的情况下，开始的第几个星期取值可以为：FIRST表示第一个星期、SECOND表示第二个星期、THIRD表示第三个星期、FOURTH表示第四个星期、LAST表示最后一个星期。 |
| 开始星期 | 在支持夏令时并且日期输入方式为Week(星期方式)"的情况下，开始星期的取值可以为：Mon(星期一)、Tue(星期二)、Wed(星期三)、Thu(星期四)、Fri(星期五)、Sat(星期六)、Sun(星期日)。 |
| 结束的第几个星期 | 在支持夏令时并且日期输入方式为"Week(星期方式)"的情况下，结束的第几个星期取值可以为：FIRST表示第一个星期、SECOND表示第二个星期、THIRD表示第三个星期、FOURTH表示第四个星期、LAST表示最后一个星期。 |
| 结束星期 | 在支持夏令时并且日期输入方式为"Week(星期方式)"的情况下，结束星期的取值可以为：Mon(星期一)、Tue(星期二)、Wed(星期三)、Thu(星期四)、Fri(星期五)、Sat(星期六)、Sun(星期日)。 |
| 时间偏移量（分钟） | 在支持夏令时的情况下，时间偏移量的取值范围可以是：1~120。 |

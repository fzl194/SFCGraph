# 删除流行为下级联流策略（RMV QOSACTRDRPOLICY）

#### [命令功能](#ZH-CN_CONCEPT_0000001600440825)

该命令用于在流行为视图下，删除修改级联流策略。

#### [注意事项](#ZH-CN_CONCEPT_0000001600440825)

该命令仅适用于非NP卡场景和NP卡非加速模式场景。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600440825)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600440825)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| BEHAVIORNAME | 行为名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定流行为的名称，不允许为系统预定义策略default。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～31。<br>默认值：无<br>配置原则：区分大小写，不支持空格。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600440825)

流行为下删除配置级联流策略：

```
RMV QOSACTRDRPOLICY:BEHAVIORNAME="1";
```

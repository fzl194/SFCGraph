## [命令功能](#ZH-CN_MMLREF_0000001977419808)

**适用NF：NCG**

该命令用于查询告警服务器相关配置信息。

## [注意事项](#ZH-CN_MMLREF_0000001977419808)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001977419808)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001977419808)

无

## [使用实例](#ZH-CN_MMLREF_0000001977419808)

查询告警服务器相关配置信息：

```
LST NCGALMSRV:;
```

## [输出结果说明](#ZH-CN_MMLREF_0000001977419808)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 告警服务器名称 | 该参数用于指定告警服务器名称。 |
| IP类型 | 该参数用于指定告警服务器的IP类型。 |
| 主用告警服务器IPv4地址 | 该参数用于指定主用告警服务器的IPv4地址。 |
| 主用告警服务器IPv6地址 | 该参数用于指定主用告警服务器的IPv6地址。 |
| 主用告警服务器端口号 | 该参数用于指定主用告警服务器端口号。 |
| 备用告警服务器IPv4地址 | 该参数用于指定备用告警服务器的IPv4地址。 |
| 备用告警服务器IPv6地址 | 该参数用于指定备用告警服务器的IPv6地址。 |
| 备用告警服务器端口号 | 该参数用于指定备用告警服务器端口号。 |
| 派单告警列表 | 本参数用于指定派单告警列表。 |
| 不派单告警列表 | 本参数用于指定不需要派单告警列表。 |
| 增量查询新增告警时间间隔（秒） | 该参数用于指定增量查询新增告警的时间间隔。 |
| 全量查询告警时间间隔（秒） | 该参数用于指定获取全量告警用来判断告警恢复情况的时间间隔。 |
| 定时重发送故障告警时间间隔（秒） | 该参数用于指定定时重发送故障告警的时间间隔。 |

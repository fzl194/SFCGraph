#### [命令功能](#ZH-CN_CONCEPT_0251174234)

![](修改接入控制（MOD ACTRL）_51174234.assets/notice_3.0-zh-cn_2.png)

此命令不能动态生效，需要执行“RST VNFC”重启服务，并且修改接入控制可能导致话单业务不可用。

**适用NF：NCG**

该命令用于修改接入控制模块采用的IP地址和端口号。

#### [注意事项](#ZH-CN_CONCEPT_0251174234)

- 该命令执行后，需在“MML命令行 - UNC”窗口执行“[**RST VNFC**](../../../../../平台服务管理/单体服务公共功能管理/系统管理/复位系统/重启系统（RST VNFC）_59103634.md)”命令重新启动系统才能生效。
- 执行此命令修改IP地址及端口号属于危险操作。可能导致链路连接异常，从而导致CG无法收到话单。修改生效后，需执行[**DSP ACTRL**](显示CGF与对端网元链路状态（DSP ACTRL）_51174236.md)命令检查链路状况，确保链路正常。

#### [本地用户权限](#ZH-CN_CONCEPT_0251174234)

manage-ug；system-ug

#### [网管用户权限](#ZH-CN_CONCEPT_0251174234)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0251174234)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| ACID | 接入控制标识 | 可选必选说明：必选参数<br>参数含义：用于在NCG中定义一条接入控制的数据记录。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～32。<br>默认值：无<br>配置原则：<br>- 标识一条接入控制配置数据记录的关键字段之一（其取值不能被修改），用于指定需要对哪一条接入控制配置数据记录的参数进行修改或调整，操作员可以使用[**LST ACTRL**](查询接入控制（LST ACTRL）_51174235.md)命令查询获得。<br>- 该参数不能包含“\”。<br>- 不能输入的特殊字符请参考“[**特殊字符表**](../话单稽核/增加话单稽核（ADD CDRAUDIT）_51174239.md#ZH-CN_CONCEPT_0251174239__table_0365FEF0)”。 |
| AGID | 接入网元分组标识 | 可选必选说明：可选参数<br>参数含义：用于区分不同域的接入网元。<br>数据来源：对端协商<br>取值范围：字符串类型，输入长度范围为1～32。<br>默认值：无<br>配置原则：<br>- 用于重新定义该接入控制对应的接入网元分组标识。<br>- 不能输入的特殊字符请参考“[**特殊字符表**](../话单稽核/增加话单稽核（ADD CDRAUDIT）_51174239.md#ZH-CN_CONCEPT_0251174239__table_0365FEF0)”。 |
| IPRID | IP资源标识 | 可选必选说明：可选参数<br>参数含义：用于表示IP资源。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～32。<br>默认值：无<br>配置原则：不能输入的特殊字符请参考“<br>[**特殊字符表**](../话单稽核/增加话单稽核（ADD CDRAUDIT）_51174239.md#ZH-CN_CONCEPT_0251174239__table_0365FEF0)<br>”。 |
| PORT | 端口 | 可选必选说明：可选参数<br>参数含义：用于表示接入控制模块采用的端口。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～65535。<br>默认值：无<br>配置原则：无 |
| RIPTYPE | 重定向IP地址类型 | 可选必选说明：可选参数<br>参数含义：用于表示重定向IP地址属于IPv4网络还是IPv6网络，需根据当前局点规划选择其中一种。<br>数据来源：对端协商<br>取值范围：枚举类型。<br>- IPV4：IPv4。<br>- IPV6：IPv6。<br>默认值：无<br>配置原则：用于重新定义该接入控制使用的重定向IP地址类型。 |
| RIPV4 | 重定向IPv4 | 可选必选说明：条件可选参数<br>前提条件：该参数在“RIPTYPE”配置为“IPV4”时为条件可选参数。<br>参数含义：用于表示接入控制模块采用的重定向IPv4地址。<br>数据来源：本端规划<br>取值范围：IPv4地址类型。<br>默认值：无<br>配置原则：当且仅当本命令中的“IP地址类型”参数为“IPV4(IPv4)”时该参数有效，用于重新定义该接入控制使用的重定向IP地址。 |
| RIPV6 | 重定向IPv6 | 可选必选说明：条件可选参数<br>前提条件：该参数在“RIPTYPE”配置为“IPV6”时为条件可选参数。<br>参数含义：用于表示接入控制模块采用的重定向IPv6地址。<br>数据来源：本端规划<br>取值范围：IPv6地址类型。<br>默认值：无<br>配置原则：<br>- 当且仅当本命令中的“IP地址类型”参数为“IPV6(IPv6)”时该参数有效，用于重新定义该接入控制使用的重定向IPv6地址。<br>- IPv6地址必须是全球单播地址，不能为::、FFFF:FFFF:FFFF:FFFF:FFFF:FFFF:FFFF:FFFF、2001:X:X:X:X:X::、环回地址(::1)、链路本地地址(FE80::/10)和组播地址(FF00::/8)。<br>- 该参数的掩码缺省为FFFF:FFFF:FFFF:FFFF:FFFF:FFFF:FFFF:FFFF，不可更改。 |
| RPORT | 重定向端口 | 可选必选说明：可选参数<br>参数含义：用于接入控制模块采用的重定向端口。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～65535。<br>默认值：无<br>配置原则：当“IP资源标识”指定的端口故障时，可以切换到“重定向端口”。 |

#### [使用实例](#ZH-CN_CONCEPT_0251174234)

修改一个接入控制采用的IP示例：

```
MOD ACTRL:ACID="ps_actrl",AGID="PS_GROUP_1",IPRID="IP_Ga_CDRReceive_1st",PORT=9991,RIPTYPE=IPV4,RIPV4="192.168.3.5",RPORT=9999;
```

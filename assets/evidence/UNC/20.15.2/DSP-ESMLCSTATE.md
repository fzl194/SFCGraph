#### [命令功能](#ZH-CN_MMLREF_0000001126145818)

**适用网元：MME**

此命令用于查询指定ESMLC设备状态，以及MME与ESMLC之间的所有链路状态。

#### [注意事项](#ZH-CN_MMLREF_0000001126145818)

- 此命令执行后立即生效。

#### [本地用户权限](#ZH-CN_MMLREF_0000001126145818)

manage-ug；system-ug；monitor-ug；visit-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126145818)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组；G_4，来宾级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126145818)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| ESMLCID | E-SMLC 标识 | 可选必选说明：必选参数<br>参数含义：该参数用于指定要查询的E-SMLC 标识，E-SMLC 标识可以通过<br>[**LST LCSAPLNK**](../LCSAP链路配置/查询LCSAP链路配置(LST LCSAPLNK)_26305618.md)<br>命令查询。<br>取值范围：0～255<br>默认值 ：无 |

#### [使用实例](#ZH-CN_MMLREF_0000001126145818)

查询 “E-SMLC 标识” 为 “3” 的ESMLC状态

DSP ESMLCSTATE: ESMLCID=3;

```
%%DSP ESMLCSTATE: ESMLCID=3;%%
RETCODE = 0  操作成功。

输出结果如下
--------------
      E-SMLC 标识  =  3
      E-SMLC 状态  =  稳态
    E-SMLC 优先级  =  3
E-SMLC 定位用户数  =  30
           链路数  =  3
仍有后续报告输出
---    END

%%DSP ESMLCSTATE: ESMLCID=3;%%
RETCODE = 0  操作成功。

输出结果如下
--------------
链路索引            RU名称            进程号      链路状态

5                   USN_SP_RU_0064    0           稳态
11                  USN_SP_RU_0065    1           无效
17                  USN_SP_RU_0064    0           稳态
(结果个数 = 4)
共有2个报告
---    END
```

#### [输出结果说明](#ZH-CN_MMLREF_0000001126145818)

| 输出项名称 | 输出项解释 |
| --- | --- |
| E-SMLC 标识 | 参数含义：E-SMLC标识，标识整网络中某一可用E-SMLC。<br>取值范围：0～255 |
| E-SMLC 状态 | 参数含义：E-SMLC的设备状态，只要存在一条可用链路，E-SMLC为NORMAL稳态或者OVERLOAD过载态。<br>取值范围：<br>“NORMAL(稳态)”<br>、<br>“OVERLOAD（过载）”<br>、<br>“UNAVAILABLE（无效）” |
| E-SMLC 优先级 | 参数含义：优先级，定位业务选择E-SMLC时，根据优先级选择高优先级ESMLC，0优先级最高，5优先级最低，不同E-SMLC可以配置相同优先级，相同优先级的E-SMLC，根据负荷分担原则选择E-SMLC。<br>取值范围：0～5 |
| E-SMLC 定位用户数 | 参数含义：该参数表示当前进程内用此E-SMLC设备进行用户定位计算的用户数。<br>取值范围：0～4294967295 |
| 链路数 | 参数含义：该参数表示链路集下配置链路数，一条链路集最大4条链路。<br>取值范围：0～4 |
| 链路索引 | 参数含义：该参数用于指定E-SMLC链路索引。该参数用来在系统范围内部唯一标识一条E-SMLC链路。<br>取值范围：0～23 |
| RU名称 | 参数含义：该参数用于指定<br>SPP<br>资源单元名。<br>参数范围：1~63位字符串 |
| 进程号 | 参数含义：该参数表示会话所在SPP进程的进程号。<br>取值范围：0～20 |
| 链路状态 | 参数含义：该参数表示与E-SMLC链路状态，偶联建立成功，则为NORMAL稳态，否则为UNAVAILABLE态。<br>取值范围：<br>“NORMAL（稳态）”<br>、<br>“UNAVAILABLE（无效）” |

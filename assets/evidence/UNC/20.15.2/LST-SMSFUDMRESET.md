## [命令功能](#ZH-CN_MMLREF_0000001099973534)

**适用NF：SMSF**

该命令用来查询SMSF的UDM重选参数信息。

## [注意事项](#ZH-CN_MMLREF_0000001099973534)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001099973534)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001099973534)

无

## [使用实例](#ZH-CN_MMLREF_0000001099973534)

运营商希望查询SMSF的UDM扫描任务重选开关是否开启，执行如下命令：

```
%%LST SMSFUDMRESET:;%%
RETCODE = 0  操作成功

结果如下
------------------------
      是否触发扫描任务重选UDM  =  是
       UDM重选扫描速率(个/秒)  =  5
UDM重选时是否重新获取签约数据  =  否
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001099973534)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 是否触发扫描任务重选UDM | 该参数表示是否触发SMSF扫描任务进行重选UDM。 |
| UDM重选扫描速率(个/秒) | 该参数表示SMSF重选UDM时的扫描速率。 |
| UDM重选时是否重新获取签约数据 | 该参数表示SMSF重选UDM时是否向其重新获取签约数据。 |

## [命令功能](#ZH-CN_MMLREF_0209654359)

**适用NF：SMF、GGSN、PGW-C**

该命令用于查看已设置的前缀分割符和后缀分割符。

## [注意事项](#ZH-CN_MMLREF_0209654359)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209654359)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209654359)

无

## [使用实例](#ZH-CN_MMLREF_0209654359)

查询已设置的前缀分隔符和后缀分隔符的配置信息：

```
%%LST DOMAINSEPARATOR:;%%
RETCODE = 0  操作成功

结果如下
--------
    前缀域名分隔符  =  @%
    后缀域名分隔符  =  #/
前缀域名分隔符开关  =  使能
后缀域名分隔符开关  =  使能

(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209654359)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 前缀域名分隔符 | 该参数用于指定作为前缀的分隔符。 |
| 后缀域名分隔符 | 该参数用于指定作为后缀的分隔符。 |
| 前缀域名分隔符开关 | 该参数用于打开或关闭前缀域名分隔符开关。 |
| 后缀域名分隔符开关 | 该参数用于打开或关闭后缀域名分隔符开关。 |

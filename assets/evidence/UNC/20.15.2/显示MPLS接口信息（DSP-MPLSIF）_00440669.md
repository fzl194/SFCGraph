#### [命令功能](#ZH-CN_CONCEPT_0000001600440669)

该命令用于显示MPLS接口信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600440669)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600440669)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600440669)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IFNAME | 接口名 | 可选必选说明：可选参数<br>参数含义：该参数用于指定接口名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。<br>默认值：无<br>配置原则：请使用LST INTERFACE命令查看可用接口。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600440669)

显示MPLS接口信息：

```
DSP MPLSIF:;
```

```

RETCODE = 0  操作成功。

结果如下
--------
            接口名  =  Ethernet67/0/6
接口配置的MPLS MTU  =  1500
         接口MTU值  =  1500
       生效的MTU值  =  1500
      接口协议状态  =  接口Up
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600440669)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 接口MTU值 | 用于表示接口的MTU值。 |
| 生效的MTU值 | 用于表示接口有效的MTU值。 |
| 接口协议状态 | 该参数用于表示接口的协议状态。<br>- 接口Up：接口协议状态处于正常的启动状态。<br>- 接口Down：接口协议状态出现故障。 |

其余输出项请参见ADD MPLSIF的参数说明。

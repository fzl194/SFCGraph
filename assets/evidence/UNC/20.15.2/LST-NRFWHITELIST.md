## [命令功能](#ZH-CN_MMLREF_0000001088537098)

**适用NF：NRF**

该命令用于查询NF白名单中的NF实例。

## [注意事项](#ZH-CN_MMLREF_0000001088537098)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001088537098)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001088537098)

无

## [使用实例](#ZH-CN_MMLREF_0000001088537098)

查询白名单中的网元实例列表，执行如下命令：

```
LST NRFWHITELIST:;
%%LST NRFWHITELIST:;%%
RETCODE = 0  操作成功

结果如下
------------------------
NF实例标识  =  instance_01
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001088537098)

| 输出项名称 | 输出项解释 |
| --- | --- |
| NF实例标识 | 该参数用于表示需要增加到白名单中的NF实例标识。 |

## [命令功能](#ZH-CN_MMLREF_0296242196)

**适用NF：PGW-C、SMF**

该命令用于查询M2M关联的UPF组。

## [注意事项](#ZH-CN_MMLREF_0296242196)

无

#### [操作用户权限](#ZH-CN_MMLREF_0296242196)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0296242196)

无

## [使用实例](#ZH-CN_MMLREF_0296242196)

查询M2M关联的UPF组：

```
LST M2MUPGROUP:;
RETCODE = 0  操作成功

结果如下
--------
UPF组名称 

upfgrp1
(结果个数 = 1)

---    END
```

#### [命令功能](#ZH-CN_MMLREF_0000001172225671)

**适用网元：SGSN**

该命令用于查询GBP进程PDP用户上下文资源使用状态。

#### [注意事项](#ZH-CN_MMLREF_0000001172225671)

无。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172225671)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172225671)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172225671)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RUNAME | RU名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定SPU资源单元名。该参数可以通过<br>[DSP RU](../../../../../平台服务管理/单体服务公共功能管理/系统管理/资源管理/RU管理/显示资源单元信息（DSP RU）_59103857.md)<br>命令查询。<br>数据来源：整网规划。<br>取值范围：0~63 位字符串<br>默认值：无 |

#### [使用实例](#ZH-CN_MMLREF_0000001172225671)

查询GBP资源使用状态表:

LST GBPDPRESTBL: RUNAME="USN_VSU1";

```
%%LST GBPDPRESTBL:RUNAME="USN_VSU1";%%
RETCODE = 0 操作成功。

输出结果如下
--------------
RU名称      进程号   PDP资源拥塞标志 
USN_VSU1    1        0 
USN_VSU1    0        0 
 
(结果个数 = 2) 

---END
```

#### [输出结果说明](#ZH-CN_MMLREF_0000001172225671)

| 输出项名称 | 输出项解释 |
| --- | --- |
| RU名称 | 该参数用于指定SPU资源单元名。 |
| 进程号 | GBP进程的进程号。 |
| PDP资源拥塞标志 | 查询板PDP资源是否拥塞标识，1表示PDP数目已经超过进程PDP规格，拥塞后请参考ALM 80636告警帮助进行处理，0表示未超过规格。 |

## [命令功能](#ZH-CN_MMLREF_0235102634)

**适用NF：PGW-C、SMF**

该命令用于查询用户去活时内部诊断字段取值到去活原因的映射关系。

## [注意事项](#ZH-CN_MMLREF_0235102634)

无

#### [操作用户权限](#ZH-CN_MMLREF_0235102634)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0235102634)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| DIAGNOSTICS | 诊断原因值 | 可选必选说明：可选参数<br>参数含义：该参数用于指定SMF内部的去活诊断字段原因值。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~1000。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0235102634)

查询内部诊断字段取值为259的映射关系：

```
%%LST N40DIAGTRIGGER: DIAGNOSTICS=259;%%
RETCODE = 0  操作成功

结果如下
--------
           诊断原因值  =  259
             去活原因  =  正常去活                
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0235102634)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 诊断原因值 | 该参数用于指定SMF内部的去活诊断字段原因值。 |
| 去活原因 | 该参数用于指定内部去活诊断字段对应的去活原因。 |

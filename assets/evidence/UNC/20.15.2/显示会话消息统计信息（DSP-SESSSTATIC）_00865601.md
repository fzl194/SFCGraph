#### [命令功能](#ZH-CN_CONCEPT_0000001600865601)

该命令用于显示Diameter会话消息统计信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600865601)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600865601)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600865601)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001600865601)

显示Diameter会话消息统计信息：

```
DSP SESSSTATIC:;
```

```

RETCODE = 0 操作成功

结果如下
-------------------------
            对端地址  =  10.1.1.100
        发送对端消息  =  33
        接收对端消息  =  34
        丢弃用户消息  =  1
        最大延迟时间  =  1
  发送失败的消息计数  =  0
接收到的错误报文计数  =  0
(结果个数 = 1)
--- END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600865601)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 对端地址 | 该参数用于表示对端IP地址。 |
| 发送对端消息 | 该参数用于表示发送到对端消息统计。 |
| 接收对端消息 | 该参数用于表示接收对端消息统计。 |
| 丢弃用户消息 | 该参数用于显示丢弃用户消息统计。 |
| 最大延迟时间 | 该参数用于显示最大延迟时间。 |
| 发送失败的消息计数 | 该参数用于显示发送对端失败的消息计数。 |
| 接收到的错误报文计数 | 该参数用于显示接收到对端发送的错误报文计数。 |

#### [命令功能](#ZH-CN_MMLREF_0000001126145770)

**适用网元：MME**

此命令用于查询当前配置的M2M控制参数。

#### [注意事项](#ZH-CN_MMLREF_0000001126145770)

无

#### [本地用户权限](#ZH-CN_MMLREF_0000001126145770)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126145770)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126145770)

无。

#### [使用实例](#ZH-CN_MMLREF_0000001126145770)

查询M2M控制参数配置记录：

LST M2MCTRL:;

```
%%LST M2MCTRL:;%%
RETCODE = 0  操作成功。

查询结果如下
-------------------------
        NB-S1模式寻呼窗口差值  =  相同
        WB-S1模式寻呼窗口差值  =  相同
                 eDRX寻呼优化  =  是
                   用户面偏好  =  使用UE偏好
               信令优先级指示  =  否
                     例外报告  =  否
             无线优化寻呼开关  =  关闭
           eNodeB属性检查开关  =  关闭
       eNodeB覆盖增强检查开关  =  关闭
               用户面建立优化  =  是
               NB-IoT DDN优化  =  否
            CE Mode B DDN优化  =  否
             缓存包数取值来源  =  使用本地配置
                     缓存包数  =  0
MME发起的CP到UP切换速率(次/秒) =  1
      NB-IoT用户支持多PDN连接  =  关闭
         下行数传延迟释放连接  =  0
(结果个数 = 1)

---    END
```

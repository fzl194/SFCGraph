## [命令功能](#ZH-CN_MMLREF_0000001870382345)

**适用NF：PGW-C、GGSN、SGW-C**

该命令用于查询获取UPF服务区绑定TAI/LAI的方式。

## [注意事项](#ZH-CN_MMLREF_0000001870382345)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001870382345)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001870382345)

无

## [使用实例](#ZH-CN_MMLREF_0000001870382345)

以下命令用于查询获取UPF服务区绑定TAI/LAI的方式： LST UPAREABINDFUNC:;

```
%%LST UPAREABINDFUNC:;%%
RETCODE = 0  操作成功

结果如下
--------
                  生效方式 = 采用前缀的方式
(结果个数 = 1)

---    END
```

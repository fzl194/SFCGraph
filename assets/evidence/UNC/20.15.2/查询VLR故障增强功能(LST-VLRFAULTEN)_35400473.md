#### [命令功能](#ZH-CN_TOPIC_0000001935400473)

**适用网元：MME**

该命令用于查询VLR全故障场景增强功能开关。

#### [注意事项](#ZH-CN_TOPIC_0000001935400473)

无。

#### [本地用户权限](#ZH-CN_TOPIC_0000001935400473)

manage-ug;system-ug;monitor-ug

#### [网管用户权限](#ZH-CN_TOPIC_0000001935400473)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_TOPIC_0000001935400473)

无

#### [使用实例](#ZH-CN_TOPIC_0000001935400473)

查询VLR全故障场景增强功能开关信息：

```
LST VLRFAULTEN:;
```

```
查询结果如下
-------------------------
   VLR故障增强功能开关  =  关闭
(结果个数 = 1)

---    END
```

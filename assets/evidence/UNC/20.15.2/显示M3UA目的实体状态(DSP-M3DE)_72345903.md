#### [命令功能](#ZH-CN_MMLREF_0000001172345903)

**适用网元：SGSN、MME、SMSF**

该命令用于查询M3UA目的实体状态，目的实体是从本地实体的角度出发，在本地实体所在的所有信令网络中可见的实体。

#### [注意事项](#ZH-CN_MMLREF_0000001172345903)

- 系统会自行选取正常可用的信令进程（SPP/SGP），因此，执行该操作的时候要确保系统中有可用的信令进程，否则，操作将不能正常执行。
- 支持目的实体索引、目的实体状态、目的实体名称的单独查询方式和任意组合查询方式。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172345903)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172345903)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172345903)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RUNAME | RU名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定SPU资源单元名称。该参数可以通过<br>[DSP RU](../../../../../../平台服务管理/单体服务公共功能管理/系统管理/资源管理/RU管理/显示资源单元信息（DSP RU）_59103857.md)<br>命令查询。<br>取值范围：0~63位字符串<br>默认值：无<br>配置原则：无 |
| PT | 进程类型 | 可选必选说明：可选参数<br>参数含义：该参数用于指定进程的进程类型。<br>取值范围：<br>- “SGP（SGP）”<br>默认值：无<br>配置原则：无 |
| PN | 进程号 | 可选必选说明：可选参数<br>参数含义：该参数用于指定进程号。<br>取值范围：0~11<br>默认值：无<br>配置原则：无 |
| DEX | 目的实体索引 | 可选必选说明：可选参数<br>参数含义：该参数用于指定查询的目的实体的索引。<br>取值范围：0~1279<br>默认值：无<br>配置原则：无 |
| DES | 目的实体状态 | 可选必选说明：可选参数<br>参数含义：该参数用于指定查询目的实体时需要匹配的状态，即只有符合该状态的目的实体才显示。<br>取值范围：<br>- “INACCESS（不可达）”<br>- “ACCESS（可达）”<br>- “NOCONG（未拥塞）”<br>- “CONG（拥塞）”<br>默认值：无<br>配置原则：无 |
| DEN | 目的实体名 | 可选必选说明：可选参数<br>参数含义：该参数用于指定目的实体名称，标识目的实体。<br>取值范围：长度不超过32的字符串<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_MMLREF_0000001172345903)

查询实体索引为1的M3UA目的实体状态：

```
DSP M3DE: DEX=1;
```

```
%%DSP M3DE: DEX=1;%%
RETCODE = 0  操作成功。

查询M3UA目的实体状态
--------------------
 目的实体索引  =  1
   网络指示语  =  国内备用网
 目的实体编码  =  0x491
 本地实体编码  =  0x1449
     是否可达  =  是
     是否拥塞  =  否
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_MMLREF_0000001172345903)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 目的实体索引 | 待查询的目的实体的索引。 |
| 目的实体编码 | 用于指定该目的实体在No.7信令网中的信令点编码。<br>说明：信令点编码结构与对应的本地实体的信令点编码结构一致； 目的实体编码为16进制数，可以输入0x为前缀。即输入“0xB23”和“B23”均表示实体编码为“0xB23”。 |
| 网络指示语 | M3UA目的实体的“网络指示语”取值，与该目的实体所对应的M3UA本地实体的“网络指示语”一致。<br>取值说明：<br>- “国际网”<br>- “国际备用网”<br>- “国内网”<br>- “国内备用网” |
| 本地实体编码 | 本地实体编码。 |
| 是否可达 | 目的实体是否可达，有是、否两种状态。正常情况下应该为是。<br>取值说明：<br>- “否”<br>- “是”<br>说明：不可达时，参考<br>**ALM-10337 M3UA目的实体不可达**<br>帮助进行处理。 |
| 是否拥塞 | 目的实体是否拥塞，有是、否两种状态。正常情况下应该为否。<br>取值说明：<br>- “否”<br>- “是”<br>说明：拥塞时，参考<br>**ALM-80634 M3UA链路拥塞**<br>帮助进行处理。 |

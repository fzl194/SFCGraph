# 删除TA级切片限制区域码（RMV RSTRNSAREACODE）

- [注意事项](#ZH-CN_MMLREF_0000001171516453__1.3.2)

## [命令功能](#ZH-CN_MMLREF_0000001171516453)

**适用NF：AMF**

该命令用于删除TA级切片限制区域码。

## [注意事项](#ZH-CN_MMLREF_0000001171516453)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0000001171516453)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001171516453)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| AREACODE | 区域编码 | 可选必选说明：必选参数<br>参数含义：该参数用于唯一标识AMF服务的某个区域，该区域由一个或者若干个跟踪区组成。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是1~128。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001171516453)

删除编码为“jq001.pd006.sh.mcc123.mnc45”的TA级限制切片所属区域编码，执行如下命令：

```
RMV RSTRNSAREACODE:AREACODE="jq001.pd006.sh.mcc123.mnc45";
```

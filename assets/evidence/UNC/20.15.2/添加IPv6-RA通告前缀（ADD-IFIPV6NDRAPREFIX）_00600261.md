#### [命令功能](#ZH-CN_CONCEPT_0000001600600261)

该命令用于设置接口IPv6 RA通告前缀。如果用户希望路由设备只发布指定的前缀，则使用该命令配置前缀信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600600261)

- 该命令执行后立即生效。
- 该命令最大记录数为65535。
- 指定的前缀不能为链路本地地址（fe80::）、组播地址（ff00::）以及其他接口已经使用的前缀（包括接口地址前缀和RA消息携带的前缀）。
- 该命令可以在VNRS_VNFC的Ethernet接口，Ethernet子接口，Eth-Trunk接口，Eth-Trunk子接口以及Tunnel口上配置。
- 接口名称可以通过LST INTERFACE命令获取。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600600261)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600600261)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IFNAME | 接口名 | 可选必选说明：必选参数<br>参数含义：该参数用于指定IPV6前缀接口名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。接口名称由接口类型+接口编号组成。<br>默认值：无<br>配置原则：请使用LST INTERFACE命令查看可用接口。 |
| IPV6ADDRPREFIX | IPv6地址 | 可选必选说明：必选参数<br>参数含义：该参数用于指定IPv6前缀地址。<br>数据来源：本端规划<br>取值范围：IPv6地址类型。<br>默认值：无 |
| PREFIXLEN | IPv6前缀长度 | 可选必选说明：必选参数<br>参数含义：该参数用于指定接口IPv6地址前缀长度。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～128。<br>默认值：无 |
| PREFERTIME | 优先生存期（s） | 可选必选说明：必选参数<br>参数含义：该参数用于指定地址前缀优先生存期。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～4294967295。优先生存期不能大于有效生存期。<br>默认值：无 |
| VALIDTIME | 有效生存期（s） | 可选必选说明：必选参数<br>参数含义：该参数用于指定地址前缀有效生存期。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为0～4294967295。<br>默认值：无 |
| NOAUTOCONFIG | 无状态地址配置标志 | 可选必选说明：可选参数<br>参数含义：若配置使能，则知会本链路内的主机RA报文中的前缀是用于无状态地址配置的。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- on：使能。<br>- off：不使能。<br>默认值：off |
| OFFLINK | 本地链路标志 | 可选必选说明：可选参数<br>参数含义：若配置使能，则知会本链路内的主机RA报文中的前缀不是分配给本地链路的。主机若向该前缀指定的地址发送报文时，需要经过默认路由器转发。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- on：使能。<br>- off：不使能。<br>默认值：off |

#### [使用实例](#ZH-CN_CONCEPT_0000001600600261)

添加接口IPv6 RA通告前缀：

```
ADD IFIPV6NDRAPREFIX:IFNAME="ethernet64/0/3",IPV6ADDRPREFIX="2001:db8::1",PREFIXLEN=64,PREFERTIME=10,VALIDTIME=10;
```

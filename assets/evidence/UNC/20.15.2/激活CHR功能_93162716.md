## [操作场景](#ZH-CN_OPI_0000001093162716)

本操作指导介绍配置 UNC 和CHR服务器的对接，以及激活CHR功能的操作过程。

- 在激活CHR功能之前，当已实例化的UNC不带UCF服务时，需要上线UCF，具体流程参见《UNC软件安装》手册>安装UCF服务。
- 完成UNC上IP路由数据的配置，具体流程参见《UNC初始配置与调测》手册>配置VNF侧IP路由数据。
- (可选)安装部署IPsec，建议部署IPSec保证传输通道安全、数据安全。
- 配置UCF与CHR服务器对接数据。
- 开启CHR功能，配置CHR服务器类型以及发送策略。

## [操作步骤](#ZH-CN_OPI_0000001093162716)

准备阶段：动态上线UCF，以及配置UNC和CHR服务器IP路由数据。

**非SDN组网** ：

1. 动态上线UCF，具体流程参见《UNC软件安装》手册>基于VNF LCM安装UNC（非SDN）>[安装UCF服务（非SDN）](../../../../../软件安装/基于VNF LCM安装UNC（非SDN）/安装可选服务/安装UCF服务_52091480.md)或《UNC软件安装》手册>基于NFVO安装UNC（非SDN）>[安装UCF服务（非SDN）](../../../../../软件安装/基于NFVO安装UNC（SDN）/安装可选服务/安装UCF服务_67742476.md)。
2. 完成UNC上IP路由数据的配置，具体流程参见[《UNC初始配置与调测》手册](../../../../../初始配置/UNC初始配置与调测/组网路由配置/配置VNF侧IP路由数据（非SDN）_75096860.md)>配置VNF侧IP路由数据（非SDN）。

**SDN组网** ：

1. 动态上线UCF，具体流程参见《UNC软件安装》手册>基于NFVO安装UNC(SDN组网)>[安装UCF服务（SDN）](../../../../../软件安装/基于NFVO安装UNC（SDN）/安装可选服务/安装UCF服务_67742476.md)。
2. 完成UNC上IP路由数据的配置，具体流程参见[《UNC初始配置与调测》手册](../../../../../初始配置/UNC初始配置与调测/组网路由配置/配置VNF侧IP路由数据(SDN)_45606309.md)>配置VNF侧IP路由数据(SDN)。

2. 配置IPsec，具体参见《UNC安全配置》手册>可选配置>网络安全配置>配置IPsec功能。

  [**ADD UCFIP**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/UCF管理/UCF本地IP/配置UCF报表本地IP资源池（ADD UCFIP）_51253792.md)

## [任务示例](#ZH-CN_OPI_0000001093162716)

任务描述

此处以 UNC 作为SMF网元为例。

步骤一：配置UCF和CHR服务器对接数据。

步骤二：激活CHR功能，配置整系统CHR上报策略、存储策略以及CHR服务器类型。

- 打开CHR功能License，打开全局粒度的整系统CHR上报开关，打开APN为“HUAWEI.COM”的整系统CHR上报开关。
  > **说明**
  > CHR License仅控制整系统的正常流程CHR单据。
- 设置PDU Session Release/Registration Update流程成功和失败时均上报CHR单据，流程控制模板索引设置为10，并且上报至高性能CHR服务器。
- 配置整系统正常CHR、整系统异常CHR保存至UCF。

步骤三：（可选）配置小范围CHR上报策略以及存储策略。

- 配置小范围CHR单据上报的业务流程和信令事件。设置PDU Session Release/Registration Update流程成功和失败时均上报CHR单据，流程控制模板索引设置为11。
- 打开小范围CHR单据上报开关，并配置小范围CHR保存到OMU和UCF。

脚本

步骤一：配置UCF和CHR服务器对接数据。

//配置UCF报表本地IP资源池。

```
ADD UCFIP: PROTOCOLTYPE=TCP, IPTYPE=IPv4, IPV4="10.181.23.253", PORTSTART=6000, PORTEND=6031, INDEX=1;
```

//配置CHR服务器接入点IP地址等信息。

```
ADD UCFSVRIP: PROTOCOLTYPE=TCP, ACCESSNAME="ACCESS1", UCFSVRNAME="UCFSVR1", IPTYPE=IPv4, IPV4="10.80.128.1", SVRPORT=10500;
```

步骤二：激活CHR功能，配置整系统CHR上报策略、存储策略以及CHR服务器类型。

//打开CHR功能License，打开全局粒度的整系统CHR上报开关，打开APN为“HUAWEI.COM”的整系统CHR上报开关。

```
SET LICENSESWITCH: LICITEM="LKV2CHR04", SWITCH=ENABLE;
```

```
SET GLBCHRFUNC: CHRREPORTSW=ENABLE;
```

```
ADD APNCHRFUNC: APN="HUAWEI.COM", CHRREPORTSW=ENABLE;
```

//设置PDU Session Release/Registration Update流程失败时上报CHR单据，流程控制模板索引设置为10，并且上报至高性能CHR服务器。

```
ADD SESSNCHRPRCTMPL: TMPLIDX=10, SESSNCHRFAILPRC=PDUSESSNREL-1&MOBILREGUPDATE-1;
```

```
SET SESSNCHRCFG: HISRVPRCTMPLIDX=10;
```

//配置整系统异常CHR保存至UCF。

```
SET SESSNCHRSTORECFG: CHRFAILSAVE=ENABLE, CHRALLSAVE=DISABLE;
```

步骤三：（可选）配置小范围CHR上报策略以及存储策略。

//打开小范围CHR单据上报开关，并配置小范围CHR保存到OMU。

```
SET SESSNCHRSTORECFG: CHRSPESUBSAVE=ENABLE, CHRSAVEPATH=SAVE_OMU;
```

（可选1）//配置小范围CHR的过滤条件为IMSI为“123030123456789”的用户。

```
ADD CHRRPTSUBID
: IMSI="460030123456789", TMPLIDX=11;
```

（可选2）//配置针对位置N2TAC为0x123123的 用户进行本地存储。

```
ADD CHRRPTLOCINFO: LOCTYPE=N2TAC,TAC="123123";
```

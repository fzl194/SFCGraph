#### [命令功能](#ZH-CN_MMLREF_0000001172345163)

**适用网元：SGSN、MME**

此命令用于删除区域码记录。区域码用于定义允许或者禁止用户漫游的位置区域，借助一个完全的区域码列表，运营商可以确定用户是否能在此区域漫游。

#### [注意事项](#ZH-CN_MMLREF_0000001172345163)

- 此命令执行后立即生效。
- 此命令执行后会导致签约了本ZC（区域码）的用户不能从本记录的位置接入系统。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172345163)

manage-ug；system-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172345163)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172345163)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| ZC | 区域码 | 可选必选说明：必选参数<br>参数含义：待删除的区域码信息。<br>取值范围：0x0000~0xFFFF<br>默认值：无 |
| IDTYPE | 标识类型 | 可选必选说明：必选参数<br>参数含义：待删除的区域标识类型。<br>取值范围：<br>- “LA(位置区)”：表示该区域标识类型为位置区。<br>- “RA(路由区)”：表示该区域标识类型为路由区。<br>- “SA(服务区)”：表示该区域标识类型为服务区。<br>- “TA(跟踪区)”：表示该区域标识类型为跟踪区。<br>默认值：无 |
| MCC | 移动国家码 | 可选必选说明：必选参数<br>参数含义：待删除的移动国家码的信息。<br>取值范围：位数为3的十进制数字字符串<br>默认值：无 |
| MNC | 移动网号 | 可选必选说明：必选参数<br>参数含义：待删除的移动网号的信息。<br>取值范围：位数为2或3的十进制数字字符串<br>默认值：无 |
| LAC | 位置区域码 | 可选必选说明：条件必选参数<br>参数含义：待删除的位置区域码的信息。<br>前提条件：该参数在<br>“IDTYPE(标识类型)”<br>设置为<br>“LA(位置区)”<br>、<br>“RA(路由区)”<br>或<br>“SA(服务区)”<br>时生效。<br>取值范围：0x0000~0xFFFF<br>默认值：无 |
| RAC | 路由区域码 | 可选必选说明：条件必选参数<br>参数含义：待删除的路由区域码的信息。<br>前提条件：该参数在<br>“IDTYPE(标识类型)”<br>设置为<br>“RA(路由区)”<br>时生效。<br>取值范围：0x00~0xFF<br>默认值：无 |
| SAC | 服务区域码 | 可选必选说明：条件必选参数<br>参数含义：待删除的服务区域码的信息。<br>前提条件：该参数在<br>“IDTYPE(标识类型)”<br>设置为<br>“SA(服务区)”<br>时生效。<br>取值范围：0x0000~0xFFFF<br>默认值：无 |
| TAC | 跟踪区域码 | 可选必选说明：条件必选参数<br>参数含义：待删除的跟踪区域码的信息。<br>前提条件：该参数在<br>“IDTYPE(标识类型)”<br>设置为<br>“TA(跟踪区)”<br>时生效。<br>取值范围：0x0000~0xFFFF<br>默认值：无 |

#### [使用实例](#ZH-CN_MMLREF_0000001172345163)

删除一条区域码为0x1234，标识类型为位置区，移动国家码为123，移动网号为03，位置区域码为1的记录：

RMV ZC: ZC="0x1234", IDTYPE=LA, MCC="123", MNC="03", LAC="1";

## [操作场景](#ZH-CN_OPI_0000001464014514)

本操作指导介绍在运行网络中激活eNodeB与en-gNodeB之间通过EN-DC SON信息传送功能特性的操作过程。

EN-DC 自组织网络SON（Self Organization Network）信息传送即eNodeB与相邻en-gNodeB之间通过MME消息交互，实现配置信息的传递。

> **说明**
> 适用于MME。

本场景应用于eNodeB与en-gNodeB之间通过EN-DC SON流程自动发现X2地址的组网。

如 [图1](#ZH-CN_OPI_0000001464014514__fig1) 所示，eNodeB2和en-gNodeB连接于同一个MME即MME2；eNodeB1连接于独立的MME1。eNodeB1与en-gNodeB，en-gNodeB与eNodeB2之间互为相邻，之间通过X2接口传递切换和eNodeB/en-gNodeB状态信息。

## [必备事项](#ZH-CN_OPI_0000001464014514)

前提条件

- 请仔细阅读[WSFD-104402 eNodeB SON信息传送功能特性概述](../特性概述_66292211.md)。
- 完成加载License。
- MME、eNodeB网元上首先完成基本业务的数据配置。S1–MME、S10接口的数据配置，按网络规划配置数据进行，具体请参见[配置到eNodeB的数据](../../../../../初始配置/UNC初始配置与调测/组网对接配置/配置AMF&MME&SGSN/配置MME/配置到eNodeB的数据_82626680.md)、[配置到MME的数据](../../../../../初始配置/UNC初始配置与调测/组网对接配置/配置AMF&MME&SGSN/配置MME/配置到MME的数据_82626684.md)，本文不再赘述。

数据

该操作无需准备数据。

## [任务示例](#ZH-CN_OPI_0000001464014514)

任务描述

激活eNodeB与en-gNodeB之间通过EN-DC SON信息传送功能特性。

脚本

//打开功能开关

```
SET LICENSESWITCH: LICITEM="LKV2SON02", SWITCH=ENABLE;
```

//设置EN-DC SON功能开关。

```
SET S1CMPT: ENDCSON=SUPPORT;
```

## [命令功能](#ZH-CN_MMLREF_0000001189266901)

**适用NF：PGW-C、SMF**

该命令用于查询离线业务零用量容器上报方式。

## [注意事项](#ZH-CN_MMLREF_0000001189266901)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001189266901)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001189266901)

无

## [使用实例](#ZH-CN_MMLREF_0000001189266901)

查询离线业务零用量容器上报方式：

```
%%LST OFLZEROUSGRPTSW:;%%
RETCODE = 0  操作成功
结果如下
--------
    上报方式  =  上报
上报业务终结  =  使能
上报费率切换  =  使能
     上报QHT  =  使能
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001189266901)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 上报方式 | 该参数用于配置SMF生成的离线业务零用量容器是否向CHF上报。 |
| 上报业务终结 | 该参数用于配置控制包含FINAL、ABNORMAL_RELEASE Trigger的离线业务零用量容器是否上报。 |
| 上报费率切换 | 该参数用于配置包含TARIFF_TIME_CHANGE Trigger的离线业务零用量容器是否上报。 |
| 上报QHT | 该参数用于配置包含QHT Trigger的离线业务零用量容器是否上报。 |

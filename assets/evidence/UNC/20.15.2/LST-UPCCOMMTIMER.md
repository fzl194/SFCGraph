## [命令功能](#ZH-CN_MMLREF_0000001102357460)

**适用NF：SMF、SGW-C、PGW-C、GGSN**

该命令用于查询用户面控制通用定时器的时长。

## [注意事项](#ZH-CN_MMLREF_0000001102357460)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001102357460)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001102357460)

无

## [使用实例](#ZH-CN_MMLREF_0000001102357460)

查询用户面控制通用定时器时长，执行如下命令：

```
%%LST UPCCOMMTIMER:;%%
RETCODE = 0  操作成功

结果如下
------------------------
转发隧道不活动定时器时长(秒)  =  4
        延迟释放定时器(毫秒)  =  1000
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001102357460)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 转发隧道不活动定时器(秒) | 该参数用于设置SMF源侧ULCL UPF与目标侧ULCL UPF间的N9转发隧道的不活动定时器的时长。该定时器小于SMCOMMTIMER设置的Service Request（TREDIRECTREL参数）、Xn切换（TXNREL参数）、N2切换（TINDFWDTNL参数）流程释放阶段定时器时生效。 |
| 延迟释放定时器(毫秒) | 该参数用于设置UPF通知SMF流量乱序需延迟释放N4会话时，延迟释放定时器的时长。 |

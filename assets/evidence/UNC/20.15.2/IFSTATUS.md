#### [命令功能](#ZH-CN_CONCEPT_0000001549960906)

在监控接口的状态或检查接口的故障原因时，可执行该命令获取接口的状态信息。用户可以根据这些信息进行流量统计和接口的故障诊断等。

若不指定IFNAME参数时，则显示所有接口的状态信息；若指定IFNAME参数时，则可以显示指定接口的状态信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001549960906)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549960906)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549960906)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IFNAME | 接口名 | 可选必选说明：可选参数<br>参数含义：该参数用于指定接口名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。接口名称由接口类型+接口编号组成。<br>默认值：无<br>配置原则：请使用LST INTERFACE命令查看可用接口。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549960906)

显示接口的状态信息：

```
DSP IFSTATUS:;
```

```
RETCODE = 0  操作成功

结果如下
-------------------------
接口名                  接口物理状态                接口协议状态                接口IPv6协议状态                接口带宽

NULL0                   接口up                      接口Up                      接口Up                          -
GigabitEthernet0/0/1    接口up                      接口Up                      接口Down                        1Gbps
Tunnel1                 接口down                    接口Down                    接口Down                        0Mbps
(结果个数 = 3)
---   END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001549960906)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 接口物理状态 | 该参数用于显示接口物理状态。<br>- 接口up：接口物理层处于正常的启动状态。<br>- 接口down：接口物理层出现故障。<br>- 接口offline：接口离线。 |
| 接口协议状态 | 该参数用于显示接口的IPv4协议状态。<br>- 接口Up：接口IPv4协议状态处于正常的启动状态。<br>- 接口Down：接口IPv4协议状态出现故障。 |
| 接口IPv6协议状态 | 该参数用于显示接口的IPv6协议状态。<br>- 接口Up：接口IPv6协议状态处于正常的启动状态。<br>- 接口Down：接口IPv6协议状态出现故障。 |
| 接口带宽 | 用于显示接口的带宽。 |

其余输出项请参见ADD IFCONTROLFLAP的参数说明。

## [命令功能](#ZH-CN_MMLREF_0000001189159153)

**适用NF：AMF**

该命令用于查询NG PEI组性能统计对象信息。

## [注意事项](#ZH-CN_MMLREF_0000001189159153)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001189159153)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001189159153)

无

## [使用实例](#ZH-CN_MMLREF_0000001189159153)

查询系统配置的所有NG PEI组信息：

```
%%LST PERFNGPEIGRP:;%%
RETCODE = 0  操作成功

结果如下
--------
NG PEI群组名称

123/13
464564454
(结果个数 = 2)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001189159153)

| 输出项名称 | 输出项解释 |
| --- | --- |
| NG PEI群组名称 | 该参数用于标识NG PEI组的性能统计对象名称。NG PEI组内的PEI成员通过ADD NGPEIGRPMEM命令进行增加。 |

## [命令功能](#ZH-CN_MMLREF_0000001241422663)

该命令用于查询是否进行CRL检查。

## [注意事项](#ZH-CN_MMLREF_0000001241422663)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001241422663)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001241422663)

无

## [使用实例](#ZH-CN_MMLREF_0000001241422663)

查询CRL检查：

```
LST PKICRLCHECK:;
        RETCODE = 0  操作成功

        结果如下
        -------------------------
          CRL检查  = FALSE
        (结果个数 = 1)
        ---   END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001241422663)

| 输出项名称 | 输出项解释 |
| --- | --- |
| CRL检查 | 该参数用于表示是否进行CRL检查。 |

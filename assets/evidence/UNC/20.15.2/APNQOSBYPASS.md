## [命令功能](#ZH-CN_MMLREF_0000001222438291)

**适用NF：SMF**

该命令用于查询APNQOSBYPASS的配置信息。

## [注意事项](#ZH-CN_MMLREF_0000001222438291)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001222438291)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001222438291)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| APN | APN名称 | 可选必选说明：可选参数<br>参数含义：该参数用来指定APN名称。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是1~63。<br>默认值：无<br>配置原则：<br>该参数使用ADD APN命令配置生成。 |

## [使用实例](#ZH-CN_MMLREF_0000001222438291)

查询APN为"huawei.com"的配置信息：

```
%%LST APNQOSBYPASS:APN="huawei.com";%%
            RETCODE = 0  操作成功

            操作结果如下
            ------------
            APN名称  =  huawei.com
            5G用户QoS索引  = 5

            (结果个数 = 1)

            ---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001222438291)

| 输出项名称 | 输出项解释 |
| --- | --- |
| APN名称 | 该参数用来指定APN名称。 |
| 5G用户QoS索引 | 该参数表示用户QoS索引。 |

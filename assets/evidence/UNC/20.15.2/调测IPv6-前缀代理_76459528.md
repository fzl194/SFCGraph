# 调测IPv6 前缀代理

- [操作场景](#ZH-CN_OPI_0276459528__1.3.1)
- [必备事项](#ZH-CN_OPI_0276459528__1.3.2)

## [操作场景](#ZH-CN_OPI_0276459528)

当已经部署IPv6 前缀代理特性时，需要对本功能进行调测，确保功能可以正常使用。

> **说明**
> 适用于 GGSN-C、 SGW-C、PGW-C、SMF。

## [必备事项](#ZH-CN_OPI_0276459528)

前提条件

- 请仔细阅读 [WSFD-104004 IPv6前缀代理特性概述](特性概述_76459525.md) 。
- 完成 [激活IPv6前缀代理](激活IPv6前缀代理_76459527.md) 。

数据

| 类别 | 参数名称 | 取值样例 | 获取方法 | 说明 |
| --- | --- | --- | --- | --- |
| 用户信息查询 | 用户IMSI号（IMSI） | 123000123456789 | 测试终端自带 | - |
| 测试终端使用的APN | APN名称（APN） | huawei.com | 从已完成的配置数据中获取 | 取自UE透明接入运营商自身IP网络中配置的APN实例名。 |
| 地址池名称 | 地址池名称（POOLNAME） | pool01 | 从已完成的配置数据中获取 | - |

工具

测试终端

## [操作步骤](#ZH-CN_OPI_0276459528)

1. 执行 [**LST LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/查询License配置项开关（LST LICENSESWITCH）_09651570.md) 命令，查询License中是否允许使用支持IPv6 前缀代理。
    - 如果“SWITCH”为“ENABLE”，请执行[2](#ZH-CN_OPI_0276459528__step10476194711114)。
    - 如果“SWITCH”为“DISABLE”，则执行**[**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)**命令打开本特性对应的License配置开关。
2. 在 OM Portal上建立用户跟踪，在 “参数配置” 栏输入用户IMSI 。
3. 测试终端 使用“huawei.com”的 “APN” 接入网络。
4. 查看OM Portal上用户跟踪任务结果。
    - 如果访问成功，请执行[5](#ZH-CN_OPI_0276459528__step4)。
    - 如果访问失败，请参考调测UNC的OM Portal浏览功能。
5. 检查Reply消息中是否携带IPv6代理前缀信息。
    - 如果是正确携带，调测结束。
    - 如果没有正确携带，请执行[6](#ZH-CN_OPI_0276459528__step20418274171)。
6. 执行 [**LST APNADDRESSATTR**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/APN管理/基于APN的地址管理控制参数/查询基于APN的地址分配属性（LST APNADDRESSATTR）_09651663.md) 命令，检查该APN的配置是否正确。
  ```
  LST APNADDRESSATTR: APN="huawei.com";
  RETCODE = 0  操作成功。
  APN的地址分配信息 
  -----------------
                             APN名称  =  huawei.com
                                    ...... 
                        IPV6分配方式  =  LOCAL
                                    ......  
  (结果个数 = 1) 
  ---    END
  ```
    - 如果APN的“IPV6分配方式”与规划值一致，请执行[7](../WSFD-205101 支持Routing Behind MS/调测支持Routing Behind MS_76358999.md#ZH-CN_OPI_0276358999__cmd571354733510)。
    - 如果APN的“IPV6分配方式”与规划值不一致，请参考[激活IPv6前缀代理](激活IPv6前缀代理_76459527.md)重新配置后再次执行[3](#ZH-CN_OPI_0276459528__step19124153015128)。
7. 执行 [**LST ADDRPOOL**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/UE地址管理/UE地址池管理/地址池管理/查询地址池（LST ADDRPOOL）_09652305.md) 命令，检查地址池信息的配置是否正确。
  ```
  LST ADDRPOOL
  : POOLNAME="pool01";
  RETCODE = 0  操作成功。
  地址池信息
  -----------------
                            地址池名  =  pool01
                           IP地址类型 =  IPV6
                          地址池类型  =  Local
                      检查地址合法性  =  Enable
                                    ...... 
  (结果个数 = 1) 
  ---    END
  ```
    - 如果地址池的“IP地址类型”和“地址池类型”属性值与数据规划不一致，请参考[激活IPv6前缀代理](激活IPv6前缀代理_76459527.md)重新配置后再次执行[3](#ZH-CN_OPI_0276459528__step19124153015128)。
8. 收集信息并寻求技术支持。
    a. 执行 **[EXP MML](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/配置管理/配置导出管理/导出MML文件（EXP MML）_47200033.md)** 命令将当前配置数据导出为MML脚本文件并保存。
    b. 收集并保存上述所有查询信息。
    c. 收集归纳所有信息并联系华为技术支持解决。

## [命令功能](#ZH-CN_MMLREF_0000001213800466)

**适用NF：SMF、SGW-C、PGW-C、GGSN**

该命令用于查询GTPU参数。

## [注意事项](#ZH-CN_MMLREF_0000001213800466)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001213800466)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001213800466)

无

## [使用实例](#ZH-CN_MMLREF_0000001213800466)

查询GTPU参数

```
LST CPGTPUPARA:;

%%LST CPGTPUPARA:;%%
RETCODE = 0  操作成功

结果如下
--------
		主动发送路径探测消息开关  =  开
			路径探测消息间隔(秒)  =  60
	路径探测消息重发时间间隔(秒)  =  3
		路径探测消息最大重试次数  =  3
	  路径故障后去激活上下文开关  =  开
			  故障后重试探测次数  =  30
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001213800466)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 主动发送路径探测消息开关 | 该参数用于控制SMF是否主动发送路径探测Echo消息。 |
| 路径探测消息间隔(秒) | 该参数用于设置SMF发送GTPU路径探测消息时间间隔。 |
| 故障后重试探测次数 | 该参数用于设置GTPU路径故障后，SMF为等待路径恢复而发送路径探测消息的次数。 |
| 路径探测消息重发时间间隔(秒) | 该参数用于设置SMF发送GTPU路径探测消息重发时间间隔。 |
| 路径探测消息最大重试次数 | 该参数用于设置SMF发送GTPU路径探测消息最大重试次数。 |

#### [命令功能](#ZH-CN_CONCEPT_0000001600440705)

该命令用来查询无线路由地址借用表信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600440705)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600440705)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600440705)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IFNAME | 接口名称 | 可选必选说明：可选参数<br>参数含义：该参数用来表示借用地址的接口。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。<br>默认值：无<br>配置原则：请使用LST INTERFACE命令查看可用接口。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600440705)

查询无线路由地址借用表信息：

```
DSP WLRSRCINTERFACE:;
```

```

RETCODE = 0  操作成功

结果如下
------------------------
     接口名称  =  Tunnel1
     Peer地址  =  10.1.1.1
   源接口名称  =  LoopBack1
         属性  =  0
是否发送给FES  =  TRUE
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600440705)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 接口名称 | 用来表示借用地址的接口。 |
| Peer地址 | 用来表示使能地址借用的Peer对端地址。 |
| 源接口名称 | 用来表示被借用地址的接口。 |
| 属性 | 用来表示扩展属性。 |
| 是否发送给FES | 用来表示地址借用表是否发送给FES。 |

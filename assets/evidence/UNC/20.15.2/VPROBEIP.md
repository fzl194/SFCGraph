## [命令功能](#ZH-CN_MMLREF_0000001388882740)

该命令用于查询vProbe的报表本地IP资源池。

## [注意事项](#ZH-CN_MMLREF_0000001388882740)

执行命令前请确认vProbe服务处于上线状态，可通过DSP FUNCTIONSETINFO命令查询确认。

#### [操作用户权限](#ZH-CN_MMLREF_0000001388882740)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001388882740)

无

## [使用实例](#ZH-CN_MMLREF_0000001388882740)

运营商A查询vProbe配置单据上报的本地IP资源池配置信息：

```
%%LST VPROBEIP:;%%
RETCODE = 0  操作成功

结果如下
--------
    索引号  =  1
  协议类型  =  TCP
    IP类型  =  IPv4
  IPv4地址  =  10.185.23.253
  IPv6地址  =  ::
起始端口号  =  6000
结束端口号  =  6002
   VPN名称  =  _public_
      描述  =  NULL
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001388882740)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 索引号 | 该参数用于指定记录索引号。 |
| 协议类型 | 该参数用于指定vProbe报表使用的协议类型。 |
| IP类型 | 该参数用于指定IP地址的类型。 |
| IPv4地址 | 该参数用于指定vProbe本端的IPv4地址。 |
| IPv6地址 | 该参数用于指定vProbe本端的IPv6地址。 |
| 起始端口号 | 该参数用于指定vProbe报表地址的起始端口号。 |
| 结束端口号 | 该参数用于指定vProbe报表地址的结束端口号。 |
| VPN名称 | 该参数用于指定报表传输使用的VPN名称。 |
| 描述 | 该参数用于指定报表IP资源池的描述信息。 |

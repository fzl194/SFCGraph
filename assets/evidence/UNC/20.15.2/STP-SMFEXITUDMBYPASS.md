# 停止会话退出UDM Bypass状态任务（STP SMFEXITUDMBYPASS）

- [注意事项](#ZH-CN_MMLREF_0000001258800313__1.3.2)

## [命令功能](#ZH-CN_MMLREF_0000001258800313)

**适用NF：SMF**

该命令用于停止退出UDM Bypass。

## [注意事项](#ZH-CN_MMLREF_0000001258800313)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0000001258800313)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001258800313)

无

## [使用实例](#ZH-CN_MMLREF_0000001258800313)

当需要停止会话退出UDM Bypass任务，执行如下命令：

```
STP SMFEXITUDMBYPASS:;
```

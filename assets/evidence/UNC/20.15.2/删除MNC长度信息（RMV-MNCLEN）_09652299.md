# 删除MNC长度信息（RMV MNCLEN）

- [命令功能](#ZH-CN_MMLREF_0209652299__1.3.1)
- [注意事项](#ZH-CN_MMLREF_0209652299__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0209652299__1.3.4)
- [使用实例](#ZH-CN_MMLREF_0209652299__1.3.5)

## [命令功能](#ZH-CN_MMLREF_0209652299)

**适用NF：SMF**

该命令用于删除指定MCC号对应的MNC长度。

## [注意事项](#ZH-CN_MMLREF_0209652299)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0209652299)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652299)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| MCC | 移动国家码 | 可选必选说明：必选参数<br>参数含义：该参数用于指定移动国家码。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度是3。取值范围000～999。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209652299)

删除当前UNC所属的MCC为460：

```
RMV MNCLEN:MCC="460";
```

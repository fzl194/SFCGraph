#### [命令功能](#ZH-CN_CONCEPT_0225485673)

该命令用于查询所有CPU对应的能力基线值。

#### [注意事项](#ZH-CN_CONCEPT_0225485673)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0225485673)

G_1，管理员级别命令组；G_2，操作员级别命令组； G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0225485673)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SERVICEINSTANCE | 服务实例 | 可选必选说明：必选参数<br>参数含义：该参数表示大颗粒服务实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，通过LST VNFC命令获取。该参数不能取值为VNFP、ACS、VNRS_VNFC的服务实例名称。<br>默认值：无<br>配置原则：只能填写通过LST VNFC命令查询到的管理代理标识。 |

#### [使用实例](#ZH-CN_CONCEPT_0225485673)

查询所有的CPU能力基线值，执行以下命令：

LST CPUCAPABILITY: SERVICEINSTANCE=" vnfc " ;

```
RETCODE = 0  操作成功。

结果如下
--------
  CPU类型  =  x86_64
  CPU代数  =  Gold 6161 CPU
  CPU主频  =  2100
CPU基线值  =  1000
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0225485673)

| 输出项名称 | 输出项解释 |
| --- | --- |
| CPU类型 | 该CPU对应的类型。 |
| CPU代数 | 该CPU所使用的代数。 |
| CPU主频 | 该CPU对应的主频，单位为MHz。 |
| CPU基线值 | 该CPU对应的能力基线值。 |

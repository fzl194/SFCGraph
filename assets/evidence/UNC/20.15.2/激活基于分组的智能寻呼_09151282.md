## [操作步骤](#ZH-CN_OPI_0000002109151282)

  **[SET LICENSESWITCH](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)**

  **[ADD NGPEIPSPLCY](../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/N2接口管理/NGAP接口寻呼管理/智能寻呼策略/增加基于分组的智能寻呼策略（ADD NGPEIPSPLCY）_35319045.md)**
3. 配置AMF Pool迁移流程中，AMF向目标AMF传递用户上下文时UeContext信元中是否携带PEIPS信元以及寻呼分组标识。
  **[SET AMFSBICMPT](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/移动性管理/MM协议参数管理/AMF服务化接口兼容性参数管理/设置AMF服务化接口兼容性参数（SET AMFSBICMPT）_98011756.md)**

  **[SET NGMMFUNC](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/移动性管理/MM协议参数管理/5G移动性管理/设置5G移动性管理功能（SET NGMMFUNC）_09653748.md)**

## [任务示例](#ZH-CN_OPI_0000002109151282)

任务描述

任务1：IMSI前缀为12305且寻呼概率为大于90%小于等于95%的用户使用配置的寻呼分组0。

任务2：指定DNN为huawei.com，SST为1，SD为010101的用户使用配置的寻呼分组1。

脚本

任务1的脚本为：

//开启License项配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2SGPPG01", SWITCH=ENABLE;
```

//配置基于分组的智能寻呼策略。

```
ADD NGPEIPSPLCY: SUBRANGE=IMSI_PREFIX, IMSIPRE="12305", FEATURECOND=PAGINGPRO, PAGINGPRO=P95, PAGINGGRPSRC=USE_CONFIG, PAGINGGRPID=0;
```

//配置AMF Pool迁移流程中，AMF向目标AMF传递用户上下文时UeContext信元中携带PEIPS信元以及寻呼分组标识。

```
SET AMFSBICMPT: SNDPEIPSINFO=YES;
```

//配置寻呼UE时，如果用户的新TAILIST和旧TAILIST都有效时，针对旧TAILIST做增强处理。

```
SET NGMMFUNC: PAGINGENSW=ON;
```

任务2的脚本为：

//开启License项配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2SGPPG01", SWITCH=ENABLE;
```

//配置基于分组的智能寻呼策略。

```
ADD NGPEIPSPLCY: SUBRANGE=ALL_USER, IMSIPRE="12305", FEATURECOND=DNN_NSSAI, DNN="huawei.com", SST=1, SD="010101", PAGINGGRPSRC=USE_CONFIG, PAGINGGRPID=1;
```

//配置AMF Pool迁移流程中，AMF向目标AMF传递用户上下文时UeContext信元中携带PEIPS信元以及寻呼分组标识。

```
SET AMFSBICMPT: SNDPEIPSINFO=YES;
```

//配置寻呼UE时，如果用户的新TAILIST和旧TAILIST都有效时，针对旧TAILIST做增强处理。

```
SET NGMMFUNC: PAGINGENSW=ON;
```

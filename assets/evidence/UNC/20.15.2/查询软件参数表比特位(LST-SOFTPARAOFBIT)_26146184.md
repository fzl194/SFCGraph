#### [命令功能](#ZH-CN_MMLREF_0000001126146184)

**适用网元：SGSN、MME**

该命令用于查询软件参数表，查询结果以二进制形式输出。同时，该命令也支持查询指定软件参数中某个比特位的值。

#### [注意事项](#ZH-CN_MMLREF_0000001126146184)

无。

#### [本地用户权限](#ZH-CN_MMLREF_0000001126146184)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126146184)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126146184)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| DT | 参数类型 | 可选必选说明：必选参数<br>参数含义：软件参数的数据类型。<br>取值范围：<br>- “BYTE（单字节类型）”<br>- “DWORD（无符号整数类型）”<br>- “BYTE_EX（扩展单字节类型）”<br>- “DWORD_EX（扩展无符号整数类型）”<br>- “BYTE_EX_B（扩展单字节类型B）”<br>- “DWORD_EX_B（扩展无符号整数类型B）”<br>默认值：无。 |
| PARANUM | 参数索引 | 可选必选说明：可选参数<br>参数含义：软件参数索引。<br>取值范围：1~512<br>配置原则：<br>- 当参数类型取值为“BYTE”时，参数索引的取值范围是“1~240”。<br>- 当参数类型取值为“DWORD”时，参数索引的取值范围是“1~60”。<br>- 当参数类型取值为“BYTE_EX”时，参数索引的取值范围是“1~224”。<br>- 当参数类型取值为“DWORD_EX”时，参数索引的取值范围是“1~56”。<br>- 当参数类型取值为“BYTE_EX_B”时，参数索引的取值范围是“1~464”。<br>- 当参数类型取值为“DWORD_EX_B”时，参数索引的取值范围是“1~116”。<br>默认值：无。 |
| POSITION | 比特位 | 可选必选说明：可选参数<br>参数含义：参数比特位的位置。<br>取值范围：1~32<br>配置原则：<br>- 对于软件参数，右边的第一位记为比特1。例如，参数值 = 0000 0001，表示比特1的参数值为1。<br>- 当参数类型取值为“BYTE”，“BYTE_EX”或“BYTE_EX_B”时，比特位的取值范围是“1~8”。<br>- 当参数类型取值为“DWORD”，“DWORD_EX”或“DWORD_EX_B”时，比特位的取值范围是“1~32”。<br>默认值：无。 |

#### [使用实例](#ZH-CN_MMLREF_0000001126146184)

- 查询当前参数索引为1，比特位为1的单字节类型软参的设置情况：

LST SOFTPARAOFBIT: DT=BYTE, PARANUM=1, POSITION=1;

```
%%LST SOFTPARAOFBIT: DT=BYTE, PARANUM=1, POSITION=1;%%

RETCODE = 0  操作成功。
查询结果如下
--------------
参数类型  =  单字节类型
参数索引  =  1
  比特位  =  1
  参数值  =  0
(结果个数 = 1)
---    END
```

LST SOFTPARAOFBIT: DT=DWORD;

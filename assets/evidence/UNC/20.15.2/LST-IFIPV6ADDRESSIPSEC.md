## [命令功能](#ZH-CN_MMLREF_0000001121521228)

该命令用于查询接口的IPv6地址配置信息。

## [注意事项](#ZH-CN_MMLREF_0000001121521228)

该命令在VNFP和VNFC上都可以执行。

#### [操作用户权限](#ZH-CN_MMLREF_0000001121521228)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001121521228)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IFNAME | 接口名 | 可选必选说明：可选参数<br>参数含义：该参数用于指定需要配置IP地址的接口名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~63。接口名称由接口类型+接口编号组成。<br>默认值：无<br>配置原则：<br>请使用<br>[**LST INTERFACEIPSEC**](../接口配置/查询接口（LST INTERFACEIPSEC）_80592504.md)<br>命令查看可用接口。 |

## [使用实例](#ZH-CN_MMLREF_0000001121521228)

查询LoopBack4的信息：

```
LST IFIPV6ADDRESSIPSEC: IFNAME=""LoopBack4"";
RETCODE = 0  操作成功

结果如下
-------------------------
              接口名  =  LoopBack4
            IPv6地址  =  fc00:0000:0000:0000:0000:0000:0000:0000
        IPv6地址前缀  =  128
        IPv6地址类型  =  NULL
(结果个数 = 1)

---   END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001121521228)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 接口名 | 该参数用于指定需要配置IP地址的接口名称。 |
| IPv6地址 | 该参数用于指定需要配置的接口IP地址。 |
| IPv6地址前缀 | 该参数用于指定需要配置的接口IPv6地址的网络掩码。 |
| IPv6地址类型 | 地址类型。 |

## [命令功能](#ZH-CN_MMLREF_0000001870382317)

**适用NF：SMF**

该命令用于查询层二规则。

## [注意事项](#ZH-CN_MMLREF_0000001870382317)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001870382317)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001870382317)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| L2RULENAME | 层二规则名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定层二规则名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~63。不支持空格，不区分大小写。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001870382317)

- 查询层二规则名称为“rule1”的层二规则：
  ```
  LST L2RULE: L2RULENAME="rule1";
  RETCODE = 0  操作成功

  结果如下
  --------
    层二规则名称  =  rule1
     QoS属性名称  =  qosprop1
   层二过滤器名称  =  filter1
  (结果个数 = 1)

  ---    END
  ```
- 查询所有的层二规则：
  ```
  LST L2RULE:;
  RETCODE = 0  操作成功

  结果如下
  --------
    层二规则名称  =  rule1
     QoS属性名称  =  qosprop1
   层二过滤器名称  =  filter1
  (结果个数 = 1)

  ---    END
  ```

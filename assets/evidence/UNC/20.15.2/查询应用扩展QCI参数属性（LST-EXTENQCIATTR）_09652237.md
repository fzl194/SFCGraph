## [命令功能](#ZH-CN_MMLREF_0209652237)

**适用NF：SGW-C、PGW-C、SMF**

该查询应用扩展QCI参数属性配置。

## [注意事项](#ZH-CN_MMLREF_0209652237)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209652237)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652237)

无

## [使用实例](#ZH-CN_MMLREF_0209652237)

显示应用扩展QCI参数属性配置。

```
%%LST EXTENQCIATTR:;%%
RETCODE = 0  操作成功

结果如下
--------
       用户接入的扩展QCI控制开关  =  不使能
     Ga接口话单的扩展QCI控制开关  =  不使能
  Gy接口CCR消息的扩展QCI控制开关  =  不使能
 Radius请求信令的扩展QCI控制开关  =  不使能
N40接口请求消息的扩展QCI控制开关  =  不使能
                 扩展QCI取值范围  =  扩展QCI的范围为128到254
    扩展QCI映射为标准QCI的缺省值  =  9
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209652237)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 用户接入的扩展QCI控制开关 | 该参数用于指定本地用户是否申请non-GBR资源以及漫游用户是否使用扩展QCI进行接入。该开关仅适用网元为S-GW、P-GW、SMF。 |
| Ga接口话单的扩展QCI控制开关 | 该参数用于指定在Ga接口话单消息里是用标准QCI还是用扩展QCI。 |
| Gy接口CCR消息的扩展QCI控制开关 | 该参数用于指定Gy接口CCR消息是用标准QCI还是用扩展QCI。该开关仅适用网元为P-GW、SMF。 |
| Radius请求信令的扩展QCI控制开关 | 该参数用于指定Radius鉴权和计费请求消息里是用标准QCI还是用扩展QCI。 |
| N40接口请求消息的扩展QCI控制开关 | 该参数用于指定N40接口请求消息是用标准QCI还是用扩展QCI。 |
| 扩展QCI取值范围 | 用来配置扩展QCI的取值范围。 |
| 扩展QCI映射为标准QCI的缺省值 | 该参数用于指定扩展QCI映射为标准QCI的缺省值。该参数缺省值为9。 |

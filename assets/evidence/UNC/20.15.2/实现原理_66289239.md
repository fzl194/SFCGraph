#### [鉴权事件的原理](#ZH-CN_CONCEPT_0166289239)

**图1** 鉴权原理图

<br>

![](实现原理_66289239.assets/zh-cn_image_0229736722_2.png)

1. UE向UNC发送请求消息：
    - 在Gb模式下，UE发送IMSI/PTMSI Attach Request、Inter/Intra/Period RAU Request、Deactive PDP Context Request、Activate PDP Context Request、Detach Request等消息。
    - 在Iu模式下，UE发送IMSI/PTMSI Attach Request、Inter/Intra/Period RAU Request、Intra/Inter RAU after Relocation Request、Paging Response Request、Service Request、Detach Request等消息。
    - 在S1模式下，UE发送Attach Request、Inter/Intra TAU Request、Service Request、Detach Request等消息。
2. UNC收到消息后，会判断该消息发起的流程是否会触发必选鉴权事件。
    - 是 => 执行步骤[3](#ZH-CN_CONCEPT_0166289239__li58611674349)。
    - 否 => 执行步骤[4](#ZH-CN_CONCEPT_0166289239__li118611472347)。
3. 如需要对UE进行鉴权，UNC则对UE发送Authentication Request启动鉴权流程。
4. 如果不需要对UE鉴权，UNC直接处理该事件，不对UE进行鉴权。

#### [灵活鉴权的流程](#ZH-CN_CONCEPT_0166289239)

鉴权判断如 [图2](#ZH-CN_CONCEPT_0166289239__fig625193376) 所示。

**图2** 鉴权判断

<br>

![](实现原理_66289239.assets/zh-cn_image_0229736721_2.png)

1. UNC判断该事件是否为必选鉴权事件，如果为必选鉴权事件则需要鉴权；否则该事件为可选鉴权事件，继续进行下述步骤。
2. UNC 判断该用户是否属于特定用户群，不属于的话就使用系统默认的鉴权配置；否则，继续进行下述步骤（判断是否到达设定的鉴权周期）。
  > **说明**
  > 系统默认的鉴权配置是指采用参数 “鉴权周期” 、 “鉴权事件” 、 “鉴权事件上限” （ [**ADD GBAUTHCIPH**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/用户安全管理/Gb模式用户安全参数/增加Gb模式用户安全参数（ADD GBAUTHCIPH）_26145642.md) 、 [**ADD IUAUTHCIPH**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/用户安全管理/Iu模式用户安全参数/增加Iu模式用户安全参数(ADD IUAUTHCIPH)_72225327.md) 或 **[ADD S1USRSECPARA](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/用户安全管理/S1模式用户安全参数/增加S1模式用户安全配置(ADD S1USRSECPARA)_26305460.md)** ）的默认值。
3. UNC判断该用户的鉴权间隔时长是否达到了通过[**ADD GBAUTHCIPH**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/用户安全管理/Gb模式用户安全参数/增加Gb模式用户安全参数（ADD GBAUTHCIPH）_26145642.md) 、 [**ADD IUAUTHCIPH**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/用户安全管理/Iu模式用户安全参数/增加Iu模式用户安全参数(ADD IUAUTHCIPH)_72225327.md) 或**[ADD S1USRSECPARA](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/用户安全管理/S1模式用户安全参数/增加S1模式用户安全配置(ADD S1USRSECPARA)_26305460.md)**命令设定的鉴权周期，达到则需要鉴权；否则，继续进行下述步骤（判断触发事件的NAS消息是否在可进行灵活鉴权的事件配置列表中）。
4. UNC判断触发事件的NAS消息是否在可进行灵活鉴权的事件配置列表中，未在配置列表中则不需要鉴权；否则，继续进行下述步骤。
  > **说明**
  > 可进行灵活鉴权的事件配置列表通过 [**ADD GBAUTHCIPH**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/用户安全管理/Gb模式用户安全参数/增加Gb模式用户安全参数（ADD GBAUTHCIPH）_26145642.md) 、 [**ADD IUAUTHCIPH**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/用户安全管理/Iu模式用户安全参数/增加Iu模式用户安全参数(ADD IUAUTHCIPH)_72225327.md) 或 **[ADD S1USRSECPARA](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/用户安全管理/S1模式用户安全参数/增加S1模式用户安全配置(ADD S1USRSECPARA)_26305460.md)** 命令中的 “鉴权事件” 参数设定。
5. UNC判断该用户的灵活鉴权事件发生次数是否达到了通过[**ADD GBAUTHCIPH**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/用户安全管理/Gb模式用户安全参数/增加Gb模式用户安全参数（ADD GBAUTHCIPH）_26145642.md) 、 [**ADD IUAUTHCIPH**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/用户安全管理/Iu模式用户安全参数/增加Iu模式用户安全参数(ADD IUAUTHCIPH)_72225327.md) 或**[ADD S1USRSECPARA](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/用户安全管理/S1模式用户安全参数/增加S1模式用户安全配置(ADD S1USRSECPARA)_26305460.md)**命令设定的鉴权频率（鉴权事件上限），达到则需要鉴权；否则，不需要鉴权。

> **说明**
> - 如果发生了鉴权流程，则鉴权周期和鉴权次数都会回到初始值。
> - 如果鉴权事件上限为0时，不进行鉴权。

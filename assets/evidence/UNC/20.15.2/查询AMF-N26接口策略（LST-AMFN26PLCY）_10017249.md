## [命令功能](#ZH-CN_MMLREF_0000001310017249)

**适用NF：AMF**

该命令用于查询AMF的N26接口部署策略相关参数。

## [注意事项](#ZH-CN_MMLREF_0000001310017249)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001310017249)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001310017249)

无

## [使用实例](#ZH-CN_MMLREF_0000001310017249)

查询AMF的N26接口部署策略，执行如下命令：

```
%%LST AMFN26PLCY:;%%
RETCODE = 0  操作成功

结果如下
--------
        AMF N26接口部署方式  =  N26接口独立部署
      是否支持UAM内部互操作  =  否
          UAM内切换识别模式  =  通过DNS查询结果识别
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001310017249)

| 输出项名称 | 输出项解释 |
| --- | --- |
| AMF N26接口部署方式 | 该参数用于指示AMF的N26接口部署方式。 |
| 是否支持UAM内部互操作 | 该参数用于指示AMF N26接口部署方式为独立部署时，UAM内部的4/5G互操作场景是否直接在UAM内部处理，不再对外发送GTPC消息。 |
| UAM内切换识别模式 | 该参数用于控制UAM识别4/5G切换流程是否为UAM内部切换流程时的识别模式。 |

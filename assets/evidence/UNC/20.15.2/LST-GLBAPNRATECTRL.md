## [命令功能](#ZH-CN_MMLREF_0264343885)

**适用NF：PGW-C**

该命令用于查询全局APN速率控制配置信息。

## [注意事项](#ZH-CN_MMLREF_0264343885)

无

#### [操作用户权限](#ZH-CN_MMLREF_0264343885)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0264343885)

无

## [使用实例](#ZH-CN_MMLREF_0264343885)

查询全局APN速率控制配置信息：

```
%%LST GLBAPNRATECTRL:;%%
RETCODE = 0  操作成功

结果如下
--------
APN速率控制开关  =  不使能
   上行时间单位  =  不限制
   最大上行速率  =  0
   下行时间单位  =  不限制
   最大下行速率  =  0
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0264343885)

| 输出项名称 | 输出项解释 |
| --- | --- |
| APN速率控制开关 | 该参数用于控制开启和关闭APN速率控制功能。 |
| 上行时间单位 | 该参数用于表示APN速率控制的上行时间单位。 |
| 最大上行速率 | 该参数用于表示单位时间内可发送的最大上行数据包数。 |
| 下行时间单位 | 该参数用于表示APN速率控制的下行时间单位。 |
| 最大下行速率 | 该参数用于表示单位时间内可发送的最大下行数据包数。 |

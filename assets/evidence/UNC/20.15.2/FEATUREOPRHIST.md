## [命令功能](#ZH-CN_MMLREF_0214567237)

查询特性上下线历史。

## [注意事项](#ZH-CN_MMLREF_0214567237)

无

#### [操作用户权限](#ZH-CN_MMLREF_0214567237)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0214567237)

无

## [使用实例](#ZH-CN_MMLREF_0214567237)

DSP FEATUREOPRHIST:;

```
%%DSP FEATUREOPRHIST:;%%
RETCODE = 0  操作成功

结果如下
------------------------
                          操作类型  =  特性上线
记录操作对象信息(Feature和Pod信息)  =  usnom-pod:1,usn-pod:1
                      处理开始时间  =  20191216-14:17:35
                      处理结束时间  =  20191216-14:17:35
                          结果说明  =  NULL
                          处理结果  =  处理成功
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0214567237)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 操作类型 | 操作类型。<br>取值说明：<br>- Enable（特性上线）<br>- Disable（特性下线） |
| 记录操作对象信息 | 记录操作对象信息。 |
| 处理开始时间 | 处理开始时间。 |
| 处理结束时间 | 处理结束时间。 |
| 结果说明 | 结果说明。 |
| 处理结果 | 处理结果。<br>取值说明：<br>- OK（处理成功）<br>- Nok（处理失败）<br>- PartlyOK（部分处理成功）<br>- Processing（处理中） |

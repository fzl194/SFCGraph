## [操作场景](#ZH-CN_OPI_0000002490437921)

本操作指导介绍在运行网络中激活VoLTE承载延时保活特性的操作过程。

用户在VoLTE通话过程中，进入无信号区域，短时间内MME保留VoLTE承载（包括QCI=1（语音）和QCI=2（视频）），当用户移动到有LTE信号区域后，MME协助用户重建无线承载，即用户在无信号区域与有LTE信号区域移动时不掉话，提升用户业务体验。

> **说明**
> 适用于MME。

## [必备事项](#ZH-CN_OPI_0000002490437921)

前提条件

- 请仔细阅读[WSFD-201208 VoLTE承载延时保活特性概述](特性概述_90477973.md#ZH-CN_CONCEPT_0000002490477973)。
- 操作人员已经登录U2020。
- 完成加载License。

数据

| 类别 | 参数名称 | 取值样例 | 获取方法 | 说明 |
| --- | --- | --- | --- | --- |
| **SET ESM** | SAE承载保留时长(s) (TSAEBRHOLD) | 60 | 整网规划 | 语音增强特性相关参数 |

## [操作步骤](#ZH-CN_OPI_0000002490437921)

1. 进入 “MML命令行-UNC” 窗口。
2. **可选：**根据 [部署建议](实现原理_57358344.md#ZH-CN_CONCEPT_0000002457358344__zh-cn_concept_0298532351_section37531832152113) ，设置VoLTE承载保留时长，缺省值为60秒。
  **SET ESM**
3. 打开本特性的License配置开关。
  **SET LICENSESWITCH**

## [任务示例](#ZH-CN_OPI_0000002490437921)

任务描述

开启VoLTE承载延时保活特性，并设置保活时长为60秒。

脚本

//设置VoLTE承载保留时长。

```
SET ESM: TSAEBRHOLD=60;
```

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2BHVT01", SWITCH=ENABLE;
```

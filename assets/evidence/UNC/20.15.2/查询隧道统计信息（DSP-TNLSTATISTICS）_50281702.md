#### [命令功能](#ZH-CN_CONCEPT_0000001550281702)

该命令用于查询隧道统计信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001550281702)

只有配置隧道才能使用该命令。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550281702)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550281702)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001550281702)

查询隧道统计信息：

```
DSP TNLSTATISTICS:;
```

```

RETCODE = 0  操作成功

结果如下
-------------------------
  隧道类型  =  Gre
  隧道数目  =  1
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001550281702)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 隧道类型 | 用于表示隧道类型。 |
| 隧道数目 | 用于表示隧道数目。 |

## [命令功能](#ZH-CN_MMLREF_0000001189078987)

**适用NF：AMF**

该命令用于为NG PEI组性能统计对象增加PEI成员。

## [注意事项](#ZH-CN_MMLREF_0000001189078987)

- 该命令执行后立即生效。

- 同一个设备型号核准号码(PEITAC)不能添加到不同的PEI群组中。

- 最多可输入5000条记录。

#### [操作用户权限](#ZH-CN_MMLREF_0000001189078987)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001189078987)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| NGPEIGPN | NG PEI群组名称 | 可选必选说明：必选参数<br>参数含义：该参数用于标识NG PEI组的性能统计对象名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~128。“NG PEI群组名称”已经通过ADD PERFNGPEIGRP配置。<br>默认值：无<br>配置原则：无 |
| PEITAC | PEI设备型号核准号码 | 可选必选说明：必选参数<br>参数含义：该参数用于标识用户的设备型号核准号码（TAC）。TAC由PEI的前8位数字组成，由欧洲型号认证中心分配，用来标识某一型号的手机。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度是8。PEITAC编码为10进制数，按照字符串格式输入，字符串长度为8，只能由数字（0-9）组成。<br>默认值：无<br>配置原则：无 |
| UEDESC | 设备详细信息 | 可选必选说明：可选参数<br>参数含义：该参数用于标识设备详细信息。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~255。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001189078987)

增加一个NG PEI组性能统计对象成员，NG PEI群组名称为“phone_xx”，群组成员设备型号核准号码为“12345678”，设备信息为“phone_xx_1”：

```
ADD NGPEIGRPMEM: NGPEIGPN="phone_xx", PEITAC="12345678",UEDESC="phone_xx_1";
```

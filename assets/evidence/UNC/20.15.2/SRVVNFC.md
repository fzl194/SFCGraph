#### [命令功能](#ZH-CN_CONCEPT_0129627049)

该命令用于查询消费者信息。

#### [注意事项](#ZH-CN_CONCEPT_0129627049)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0129627049)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0129627049)

无。

#### [使用实例](#ZH-CN_CONCEPT_0129627049)

查询服务VNFC。

DSP SRVVNFC:;

```
%%DSP SRVVNFC:;%%
RETCODE = 0  操作成功。

操作结果如下：
--------------
   服务VNFC ID  =  5
  服务VNFC类型  =  CSLB
  申请服务类型  =  LB服务 & TN服务
服务IP地址规格  =  小规格
    IP业务模式  =  业务模式1
IP接入网络个数  =  1
    TS业务模式  =  NULL
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0129627049)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 服务VNFC ID | 参数含义：该参数用于表示消费者的唯一标识。 |
| 服务VNFC类型 | 参数含义：该参数用于表示消费者类型。 |
| 申请服务类型 | 参数含义：该参数用于表示申请的CSLB服务类型。<br>取值范围：<br>- “LB服务”<br>- “LB服务 & TN服务” |
| 服务IP地址规格 | 参数含义：该参数用于表示申请的服务地址规模，按照范围分为不同规格。<br>取值范围：<br>- “大规格”<br>- “中规格”<br>- “小规格” |
| IP业务模式 | 参数含义：该参数用于表示IP业务模式。<br>取值范围：<br>- “NULL”<br>- “业务模式1”<br>- “业务模式2”<br>- “业务模式3”<br>- “业务模式4”<br>- “业务模式5”<br>- “业务模式6” |
| IP接入网络个数 | 参数含义：该参数用于表示IP接入网络个数。 |
| TS业务模式 | 参数含义：该参数用于表示TS业务模式。<br>取值范围：<br>- “NULL”<br>- “业务模式1”<br>- “业务模式2”<br>- “业务模式3”<br>- “业务模式4”<br>- “业务模式5”<br>- “业务模式6” |

## [操作场景](#ZH-CN_OPI_0172659574)

调测基于区域的语音策略控制特性。

## [必备事项](#ZH-CN_OPI_0172659574)

前提条件

- 请仔细阅读[WSFD-201005 基于区域的语音策略控制特性概述](特性概述_70014727.md)。
- 完成[激活基于区域的语音策略控制](激活基于区域的语音策略控制_64009949.md)。

数据

该操作无需准备数据。

## [操作步骤](#ZH-CN_OPI_0172659574)

- 运营商规划在特定跟踪区内可以使用VoLTE语音业务，例如TA_1内允许用户使用VoLTE语音业务，TA_1之外区域不允许用户使用VoLTE语音业务。
  在相关网元均完成本特性的配置后，可以采用以下步骤检查特性工作是否正常：
    1. 创建用户跟踪。
    2. 用户在TA_1内开机附着。
    3. 观察MME向用户发送的Attach Accept消息中EPS Network Feature Support信元的bit1的取值。
      **预期结果** ：bit1取值为1。
    4. 用户移动到TA_1外的跟踪区。
    5. 观察MME向用户发送的TAU Accept消息中EPS Network Feature Support信元的bit1的取值。
      **预期结果** ：bit1取值为0。

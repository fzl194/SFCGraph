#### [命令功能](#ZH-CN_MMLREF_0000001172345975)

**适用网元：SGSN、MME**

该命令用于查询系统内所有SGP进程上的SBc链路状态。

#### [注意事项](#ZH-CN_MMLREF_0000001172345975)

无。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172345975)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172345975)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172345975)

无。

#### [使用实例](#ZH-CN_MMLREF_0000001172345975)

DSP SBCAPLNK:;

```
%%DSP SBCAPLNK:;%%
RETCODE = 0  操作成功。

输出结果如下：
-------------------------
     CBC索引  =  0
      进程号  =  6
      RU名称  =  USN_SP_RU_0064
   IP地址类型  =  IPv4
    本地地址1  =  10.2.62.20
    本地地址2  =  255.255.255.255
   本地端口号  =  29168
    对端地址1  =  172.27.168.1
    对端地址2  =  255.255.255.255
   对端端口号  =  29168
      VPN名称  =  _public_
     链路状态  =  正常
(结果个数 = 1)
```

#### [输出结果说明](#ZH-CN_MMLREF_0000001172345975)

| 输出项名称 | 输出项解释 |
| --- | --- |
| CBC索引 | CBC的索引，此CBC索引在系统范围内唯一。 |
| 进程号 | SBc链路所在SGP进程的进程号。 |
| RU名称 | 查询所在的资源单元名称。 |
| IP地址类型 | SBc链路IP地址类型。 |
| 本地地址1 | SBc链路第一个本地IP地址。 |
| 本地地址2 | SBc链路第二个本地IP地址。 |
| 本地端口号 | SBc链路本地端口号。 |
| 对端地址1 | SBc链路第一个对端IP地址。 |
| 对端地址2 | SBc链路第二个对端IP地址。 |
| 对端端口号 | SBc链路对端端口号。 |
| VPN名称 | SBc链路的VPN名称。 |
| 链路状态 | 查询SBc链路状态的查询结果。<br>取值说明：<br>- “正常”：表示MME网元与CBC网元间之间的通信在该链路上正常。<br>- “故障”：表示MME网元与CBC网元间之间的通信在该链路上中断，需要检查网络状况，分析故障原因。详细操作参见SBc链路故障告警处理。 |

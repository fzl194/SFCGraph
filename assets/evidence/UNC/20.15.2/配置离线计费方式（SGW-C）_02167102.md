## [操作场景](#ZH-CN_OPI_0302167102)

SGW-C的离线计费方式通过本地配置决定，本地配置有两种粒度：APN和计费属性CC。这些方式的优先级依次降低。如 [图1](#ZH-CN_OPI_0302167102__fig4886191182311) 所示为SGW-C的离线计费方式选择图。

## [操作步骤](#ZH-CN_OPI_0302167102)

1. 基于用户归属属性（本地用户、漫游用户、拜访用户）使能SGW-C的离线计费方式。
  **SET SGWCHARGECFG**
  > **说明**
  > 针对同一个用户，不同属性下规划的计费方式要一致，否则用户计费会出错。例如，针对CC取值为NORMAL的本地用户，如果设置NORMAL用户进行离线计费，又设置本地用户进行在线计费，则最终用户计费将出错。
2. **可选：**配置SGW-C的Home PLMN。
  **ADD NGSGWPLMN**
  > **说明**
  > 当PGW-C与SGW-C部署在同一台设备上，期望SGW-C和PGW-C对同一用户按照不同的漫游属性进行计费时，可以通过该命令单独配置SGW-C的Home PLMN用于判断用户的漫游属性（仅对IMSI与 **ADD NGPLMN** 配置匹配判断为本地用户的用户生效）。期望SGW-C和PGW-C对同一用户按照相同的漫游属性进行计费时，不需要配置该命令。
3. 配置使能SGW-C的离线计费方式。
    a. 基于用户计费属性使能SGW-C的离线计费方式。
      **SET SGWCHGMETH**
    b. 基于APN使能SGW-C的离线计费方式。
      **SET SGWAPNCHGMETH**
4. 设置SGW-C基于号段组的计费方式。
    a. 配置IMSI及MSISDN号码段。
      **ADD IMSIMSISDNSEG**
    b. 配置IMSI/MSISDN/IMEISV号码段组。
      **ADD SUBSCRIBERIDSEGGRP**
    c. 设置SGW-C基于号段组的计费方式。
      **ADD SGWSEGGCHGMETH**
5. **可选** ：配置SP合一用户产生SGW话单。
  **SET OFCCDRPARA**

## [任务示例](#ZH-CN_OPI_0302167102)

任务描述

- 任务一：配置SGW-C基于APN使能SGW-C的离线计费方式。
- 任务二：配置SGW-C基于用户计费属性使能SGW-C的离线计费方式。
- 任务三：配置SGW-C基于号段组的计费方式。
- 任务四：配置SGW-C对本地用户不使能离线计费，漫游用户和拜访用户使能离线计费，Home PLMN为12368。

脚本

//任务一：配置SGW-C基于APN实例使能离线计费功能。

```
SET SGWCHARGECFG:HOMEOFFLINE=DISABLE,ROAMOFFLINE=ENABLE,VISITOFFLINE=ENABLE,SGWCGFLAG=USE_LOCAL;
```

```
ADD APN: APN="apn-test";
```

```
SET SGWAPNCHGMETH: APN="apn-test",OFFLINE=ENABLE;
```

//任务二：配置SGW-C基于用户计费属性使能SGW-C的离线计费方式。

```
SET SGWCHARGECFG:HOMEOFFLINE=DISABLE,ROAMOFFLINE=ENABLE,VISITOFFLINE=ENABLE,SGWCGFLAG=USE_LOCAL;
```

```
SET SGWCHGMETH: CHARGECHAR=DEFAULT, OFFLINEFLAG=ENABLE;
```

//任务三：配置SGW-C基于号段组的计费方式。

```
SET SGWCHARGECFG:HOMEOFFLINE=DISABLE,ROAMOFFLINE=ENABLE,VISITOFFLINE=ENABLE,SGWCGFLAG=USE_LOCAL;
```

```
ADD IMSIMSISDNSEG:SEGMENTNAME="testmsisdn1",SEGMENTTYPE=IMSI,SEGSTART="13900100100",SEGEND="13900100199";
```

```
ADD SUBSCRIBERIDSEGGRP: SEGGROUPNAME="testmsisdngr1", SEGMENTNAME="testmsisdn1", SEGMENTTYPE=IMSIMSISDN;
```

```
ADD SGWSEGGCHGMETH:SEGGROUPNAME="testmsisdngr1", PRIORITY=2, OFFLINEFLAG=ENABLE;
```

//任务四：配置SGW-C对本地用户不使能离线计费，漫游用户和拜访用户使能离线计费，Home PLMN为12368。

```
SET SGWCHARGECFG:HOMEOFFLINE=DISABLE,ROAMOFFLINE=ENABLE,VISITOFFLINE=ENABLE,SGWCGFLAG=USE_LOCAL;
```

```
ADD NGSGWPLMN: NOID=0, MCC="123", MNC="68", DESC="sgw homeplmn1";
```

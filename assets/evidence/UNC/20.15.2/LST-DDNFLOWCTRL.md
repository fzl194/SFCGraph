## [命令功能](#ZH-CN_MMLREF_0264343879)

**适用NF：SMF、SGW-C**

该命令用于查询DDN流控参数。

## [注意事项](#ZH-CN_MMLREF_0264343879)

无

#### [操作用户权限](#ZH-CN_MMLREF_0264343879)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0264343879)

无

## [使用实例](#ZH-CN_MMLREF_0264343879)

查询DDN流控参数信息，执行如下命令：

```
%%LST DDNFLOWCTRL:;%%
RETCODE = 0  操作成功

结果如下
--------
Wal值  =  0
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0264343879)

| 输出项名称 | 输出项解释 |
| --- | --- |
| Wal值(包每秒) | 该参数用来设置DDN消息流控阈值。 |

#### [命令功能](#ZH-CN_MMLREF_0000001126146118)

**适用网元：MME**

本命令用于查询支持容灾备份的用户特征。

#### [注意事项](#ZH-CN_MMLREF_0000001126146118)

无。

#### [本地用户权限](#ZH-CN_MMLREF_0000001126146118)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126146118)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126146118)

无。

#### [使用实例](#ZH-CN_MMLREF_0000001126146118)

查询支持容灾备份的用户特征记录：

LST RESTOUSR:;

```
%%LST RESTOUSR:;%%
RETCODE = 0  操作成功。

输出结果如下
--------------
IMSI前缀  =  12303
(结果个数 = 1)
---    END
```

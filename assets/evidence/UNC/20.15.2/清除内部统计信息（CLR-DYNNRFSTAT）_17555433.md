# 清除内部统计信息（CLR DYNNRFSTAT）

## [命令功能](#ZH-CN_MMLREF_0217555433)

**适用NF：NRF**

清除NRF内部统计信息。清除之前的NF访问信息，内存中记录从当前开始的NF访问信息。

## [注意事项](#ZH-CN_MMLREF_0217555433)

- 该命令执行后立即生效。

- 主备或双活组网的场景下，如果需要配置此命令，则两个NRF上均需执行此命令，且配置参数一致。

#### [操作用户权限](#ZH-CN_MMLREF_0217555433)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0217555433)

无

## [使用实例](#ZH-CN_MMLREF_0217555433)

将内部统计项清零：

```
CLR DYNNRFSTAT:;
```

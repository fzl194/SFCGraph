#### [命令功能](#ZH-CN_CONCEPT_0209897146)

**适用NF：PGW-C、SMF**

该命令用于查询用户位置信息。当运营商希望查询用户位置信息时，则执行该命令。

#### [注意事项](#ZH-CN_CONCEPT_0209897146)

如果不输入位置名称，表示查询系统中所有用户位置信息。

#### [操作用户权限](#ZH-CN_CONCEPT_0209897146)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0209897146)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| LOCATIONNAME | 位置名称 | 可选必选说明：可选参数<br>参数含义：指定位置名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。不支持空格，不区分大小写。<br>默认值：无<br>配置原则：无 |

#### [使用实例](#ZH-CN_CONCEPT_0209897146)

- 假如运营商需要查询位置名称为test_loc_1的位置信息：
  ```
  LST USRLOCATION: LOCATIONNAME="test_loc_1";
  ```
  ```

  RETCODE = 0  操作成功

  位置信息
  --------
         位置名称  =  test_loc_1
         位置类型  =  路由区域标识
       移动国家码  =  123
       移动网络码  =  12
         位置区码  =  0x1234
         路由区码  =  0x12
         小区标识  =  NULL
         服务区码  =  NULL

       NR小区标识  =  NULL
         跟踪区码  =  NULL
  (结果个数 = 1)

  ---    END
  ```
- 假如运营商需要查询所有的位置信息：
  ```
  LST USRLOCATION:;
  ```

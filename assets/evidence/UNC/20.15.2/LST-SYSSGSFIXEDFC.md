#### [命令功能](#ZH-CN_CONCEPT_0000001369082810)

**适用NF：SMSF**

此命令用于查询VLR整系统Sgs固定速率流控参数。

#### [注意事项](#ZH-CN_CONCEPT_0000001369082810)

无。

#### [本地用户权限](#ZH-CN_CONCEPT_0000001369082810)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_CONCEPT_0000001369082810)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001369082810)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001369082810)

查询VLR整系统Sgs固定速率流控参数，可以用如下命令：

```
%%LST SYSSGSFIXEDFC:;%%
RETCODE = 0  操作成功  
操作结果如下 ------------              
                                             流控功能开关  =   ON
                   Location Update Request速率门限(个/秒)  =  2000       
                         Detach Indication速率门限(个/秒)  =  2000                  
                                    Paging速率门限(个/秒)  =  2000
 
(结果个数 = 1)  
---    结束
```

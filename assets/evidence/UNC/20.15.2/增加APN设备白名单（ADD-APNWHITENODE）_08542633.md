## [命令功能](#ZH-CN_MMLREF_0000001608542633)

**适用NF：PGW-C、SMF**

该命令用来配置APN跨省漫游流量限制设备白名单。支持配置SGW设备的IP地址段，以及SMF设备的NF实例标识。

## [注意事项](#ZH-CN_MMLREF_0000001608542633)

- 该命令执行后立即生效。

- 该命令配置需要通过[**SET APNACCESSCTRL**](../../APN管理/基于APN的接入属性控制/设置APN访问控制参数（SET APNACCESSCTRL）_09654434.md)命令，设置PROVRORESTRICT参数使能后生效。

- 最多可输入100000条记录。

#### [操作用户权限](#ZH-CN_MMLREF_0000001608542633)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [使用实例](#ZH-CN_MMLREF_0000001608542633)

- 配置IP地址为10.0.0.2，mask为27的SGW设备IP地址段进入APN跨省漫游流量限制设备白名单，可执行如下命令：
  ```
  ADD APNWHITENODE: APN="huawei.com", NODETYPE=SGW, IPVERSION=IPV4, NODEIPV4="10.0.0.2", NODEIPV4MASK=27;
  ```
- 配置NF实例标识为“072215ac-1632-4834-ad8d-eb08a909f34f”的SMF设备进入APN跨省漫游流量限制设备白名单，可执行如下命令：
  ```
  ADD APNWHITENODE: APN="huawei.com", NODETYPE=SMF, NFINSTANCEID="072215ac-1632-4834-ad8d-eb08a909f34f";
  ```

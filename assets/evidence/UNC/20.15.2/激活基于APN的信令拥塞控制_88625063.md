## [任务示例](#ZH-CN_OPI_0188625063)

任务描述

对APN为“example.com”的终端启用APN拥塞控制功能，打开backoff timer开关，设置拒绝原因值为CONGESTION，当此APN的附着次数达到30 times/h时， UNC 拒绝该终端的附着请求。

脚本

//打开MM拥塞控制功能License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2ASCC01", SWITCH=ENABLE;
```

//配置拥塞控制规则。

```
ADD APNCTRL: SUBSCRIBEDAPN="example.com", APNTYPE=CONGESTION_CTRL, BACKOFFSW=ON, ATTACHREJCAUSE=CONGESTION, ATTACHTHRESH=30;
```

#### [命令功能](#ZH-CN_CONCEPT_0000001186708204)

**适用NF：MME、AMF**

该命令用于查询系统中SCTP接入流控参数。

#### [注意事项](#ZH-CN_CONCEPT_0000001186708204)

无。

#### [本地用户权限](#ZH-CN_CONCEPT_0000001186708204)

manage-ug;system-ug;monitor-ug

#### [网管用户权限](#ZH-CN_CONCEPT_0000001186708204)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001186708204)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001186708204)

查询SCTP接入流控参数信息：

```
LST SCTPINITFLOW:;
```

```
操作结果如下
------------
      流控开关  =  开启
S1固定流控速率  =  2000
N2固定流控速率  =  2000
(结果个数 = 1)

---    END
```

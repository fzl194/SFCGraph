## [操作步骤](#ZH-CN_OPI_0249797716)

- SGW-C网元配置
    1. 进入 “MML命令行-UNC” 窗口。
    2. 设置基于整机的承载具备支持SGW-C RESTORATION的能力。
      [**SET FASTRECOVERY**](../../../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/网络管理/业务快速恢复/全局参数/设置全局业务快速恢复配置（SET FASTRECOVERY）_31453525.md)
    3. 配置MME POOL实例。
      [**ADD MMEPOOL**](../../../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/网络管理/业务快速恢复/MME Pool/增加MME POOL（ADD MMEPOOL）_31453510.md)
    4. 配置与SGW-C对接的MME POOL。
      [**ADD MMEIPTOMMEPOOL**](../../../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/网络管理/业务快速恢复/MME IP/增加MME IP（ADD MMEIPTOMMEPOOL）_31453389.md)
      > **说明**
      > 如果是IPv6组网，需要配置IPv6地址。
- PGW-C网元配置
    1. 进入 “MML命令行-UNC” 窗口。
    2. 设置基于APN的承载具备支持SGW RESTORATION的能力。
      [**ADD APN**](../../../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/APN管理/APN/增加APN配置（ADD APN）_09653747.md)
    3. 配置基于整机的防闪断定时器时长和留承载时长。
      [**SET FASTRECOVERY**](../../../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/网络管理/业务快速恢复/全局参数/设置全局业务快速恢复配置（SET FASTRECOVERY）_31453525.md)
    4. 配置SGW-C POOL实例。
      [**ADD SGWPOOL**](../../../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/网络管理/业务快速恢复/SGW Pool/增加SGW POOL（ADD SGWPOOL）_31453512.md)
    5. 配置与PGW-C对接的SGW-C POOL。
      [**ADD SGWIPTOSGWPOOL**](../../../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/网络管理/业务快速恢复/SGW IP/增加SGW IP（ADD SGWIPTOSGWPOOL）_31453511.md)
    6. 配置MME POOL实例，并绑定MME对应的SGW-C POOL。
      [**ADD MMEPOOL**](../../../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/网络管理/业务快速恢复/MME Pool/增加MME POOL（ADD MMEPOOL）_31453510.md)
    7. 配置与SGW-C对接的MME POOL。
      [**ADD MMEIPTOMMEPOOL**](../../../../../../../OM参考/命令/UNC MML命令/业务服务管理/接口管理/网络管理/业务快速恢复/MME IP/增加MME IP（ADD MMEIPTOMMEPOOL）_31453389.md)

## [任务示例](#ZH-CN_OPI_0249797716)

任务描述

使能SGW-C故障后的VoLTE承载故障快速恢复功能。

脚本

- SGW-C网元配置
  // 进入 “MML命令行-UNC” 窗口。
  //设置承载具备支持SGW RESTORATION的能力。
  ```
  SET FASTRECOVERY:RESTORSGWSWITCH=ENABLE;
  ```
  //配置MME POOL实例。
  ```
  ADD MMEPOOL:MMEPOOLNAME="mmepool";
  ```
  //配置与SGW-C对接的MME POOL。
  ```
  ADD MMEIPTOMMEPOOL:MMEPOOLNAME="mmepool",IPVERSION=IPV4,IPV4ADDRESS="10.10.10.10";
  ```
  ```
  ADD MMEIPTOMMEPOOL:MMEPOOLNAME="mmepool",IPVERSION=IPV4,IPV4ADDRESS="10.10.10.11";
  ```
- PGW-C网元配置
  //设置基于APN的承载具备支持SGW RESTORATION的能力。
  ```
  ADD APN:APN="ims",RESTORPGWSWITCH=ENABLE,PDTNSWITCH=ENABLE;
  ```
  //基于整机的防闪断定时器时长为120秒，保留承载时长为59分钟。
  ```
  SET FASTRECOVERY:PRTFRQBRKTIME=120,HOLDONTIME=59;
  ```
  //配置SGW POOL实例。
  ```
  ADD SGWPOOL:SGWPOOLNAME="sgwpool";
  ```
  //配置与PGW-C对接的SGW-C POOL。
  ```
  ADD SGWIPTOSGWPOOL:SGWPOOLNAME="sgwpool",IPVERSION=IPV4,IPV4ADDRESS="10.10.11.10";
  ```
  ```
  ADD SGWIPTOSGWPOOL:SGWPOOLNAME="sgwpool",IPVERSION=IPV4,IPV4ADDRESS="10.10.11.11";
  ```
  //配置MME POOL实例，并绑定MME对应的SGW POOL。
  ```
  ADD MMEPOOL:MMEPOOLNAME="mmepool",SGWPOOLNAME="sgwpool",BACKUPSWITCH=ENABLE;
  ```
  //配置与SGW对接的MME POOL。
  ```
  ADD MMEIPTOMMEPOOL:MMEPOOLNAME="mmepool",IPVERSION=IPV4,IPV4ADDRESS="10.10.12.10",BACKUP=TRUE;
  ```
  ```
  ADD MMEIPTOMMEPOOL:MMEPOOLNAME="mmepool",IPVERSION=IPV4,IPV4ADDRESS="10.10.12.11",BACKUP=TRUE;
  ```

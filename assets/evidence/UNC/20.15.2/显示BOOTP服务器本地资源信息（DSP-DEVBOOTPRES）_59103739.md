#### [命令功能](#ZH-CN_CONCEPT_0259103739)

该命令用于显示BOOTP服务器本地资源信息。用户可使用该命令查询BOOTP服务器本地资源的MAC地址、端口号等信息。

#### [注意事项](#ZH-CN_CONCEPT_0259103739)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0259103739)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0259103739)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SERVICEINSTANCE | 服务实例 | 可选必选说明：必选参数<br>参数含义：该参数表示大颗粒服务实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，通过<br>**LST VNFC**<br>命令获取。<br>默认值：无<br>配置原则：只能填写通过<br>**LST VNFC**<br>命令查询到的管理代理标识。 |

#### [使用实例](#ZH-CN_CONCEPT_0259103739)

显示BOOTP服务器本地资源信息：

```
DSP DEVBOOTPRES:
SERVICEINSTANCE="vnfc"
;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0259103739)

| 输出项名称 | 输出项解释 |
| --- | --- |
| RU编号 | 用于表示由网元分配的资源单元编号。 |
| 设备ID | 用于表示设备ID。 |
| 硬件类型 | 用于表示资源的硬件类型。 |
| 网络IP | 用于表示BOOTP服务器给资源分配的网络IP地址，该IP地址以运行环境为准。 |
| 网络MAC1 | 用于表示BOOTP通信使用的第一个MAC地址。 |
| 网络MAC2 | 用于表示BOOTP通信使用的第二个MAC地址。 |
| BoardInf配置下发时间 | 用于表示BoardInf配置下发时间。 |
| BOOTP服务器端口号 | 用于表示BOOTP服务器端口号。 |
| BOOTP客户端端口号 | 用于表示BOOTP客户端端口号。 |

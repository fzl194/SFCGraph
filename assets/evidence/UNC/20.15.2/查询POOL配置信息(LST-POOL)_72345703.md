#### [命令功能](#ZH-CN_MMLREF_0000001172345703)

**适用网元：SGSN**

该命令用于查询POOL配置信息。

#### [注意事项](#ZH-CN_MMLREF_0000001172345703)

无。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172345703)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172345703)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172345703)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| POOLID | POOL标识 | 可选必选说明：可选参数<br>参数含义：该参数用于表示POOL标识。<br>取值范围：0~4095<br>默认值：无 |

#### [使用实例](#ZH-CN_MMLREF_0000001172345703)

查询POOL配置信息：

LST POOL:;

```
%%LST POOL:;%%
RETCODE = 0  执行成功。

查询POOL配置信息
-------------------------
 POOL标识  =  0
  NRI长度  =  2
(结果个数 = 1)
---    END
```

#### [命令功能](#ZH-CN_CONCEPT_0000001600841025)

该命令用于显示DNS代理上的域名，即主机名和IP地址的对应关系表。

#### [注意事项](#ZH-CN_CONCEPT_0000001600841025)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600841025)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600841025)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| HOST | DNS域名 | 可选必选说明：可选参数<br>参数含义：该参数用于显示DNS代理上的域名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～255。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600841025)

显示DNS代理上的域名：

```
DSP DNSAGENTIPHOST:;
```

```

        RRETCODE = 0  操作成功。

        结果如下
        --------
        DNS域名  =  huawei.com
       IPv4地址  =  192.168.1.1
           状态  =  有效
       VNFC索引  =  1
      (结果个数 = 1)
      ---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600841025)

| 输出项名称 | 输出项解释 |
| --- | --- |
| DNS域名 | 用于显示DNS代理上的域名。 |
| IPv4地址 | 用于显示DNS代理上域名对应的IPv4地址。 |
| 状态 | 用于显示域名状态。 |
| VNFC索引 | 用于显示VNFC索引。 |

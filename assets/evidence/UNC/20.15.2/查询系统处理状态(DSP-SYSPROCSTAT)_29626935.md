#### [命令功能](#ZH-CN_CONCEPT_0129626935)

该命令用来查看VNFC是否处于稳态，是否允许下发扩缩容或者动态上下线等命令。

#### [注意事项](#ZH-CN_CONCEPT_0129626935)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0129626935)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0129626935)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| SERVICEINSTANCE | 服务实例 | 可选必选说明：必选参数<br>参数含义：该参数表示大颗粒服务实例名称。<br>数据来源：本端规划<br>取值范围：字符串类型，通过<br>**[LST VNFC](../../../../单体服务平台功能管理/操作维护/配置管理/VNFC信息/查询VNFC（LST VNFC）_59036046.md)**<br>命令获取。<br>默认值：无<br>配置原则：只能填写通过<br>**[LST VNFC](../../../../单体服务平台功能管理/操作维护/配置管理/VNFC信息/查询VNFC（LST VNFC）_59036046.md)**<br>命令查询到的管理代理标识，不支持对"VNFC类型名称"为VNFP、VNRS_VNFC、ACS、IPSEC_VNFC的RU状态查询。 |

#### [使用实例](#ZH-CN_CONCEPT_0129626935)

查看VNFC下的系统处理状态的信息。

```
DSP SYSPROCSTAT:
SERVICEINSTANCE="CSLB_VNFC_999"
; 
```

```
RETCODE = 0  操作成功。

结果如下
--------
  状态级别  ScaleGroup的名字   状态
  系统      NA                 稳态
  本地      SG0_CSLB_IPFWD     稳态
(结果个数 = 2)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0129626935)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 状态级别 | VNFC下的状态级别。<br>取值范围：<br>- “系统”：指网元包含的VNFC。<br>- “本地”：指下发该命令的VNFC。 |
| ScaleGroup的名字 | VNFC下的ScaleGroup名字。<br>取值范围：字符串类型，长度为：1～63。<br>说明：“系统”级别状态对应的“ScaleGroup的名字”固定为“NA”。 |
| 状态 | 不同状态级别的状态。<br>取值范围：<br>- “稳态”：表示系统或者本地为稳态，允许下发扩缩容或者动态上下线等命令。说明：- 系统状态为稳态时，表示可以进行整机的操作，如整机扩容、动态上线和动态下线。<br>- 本地状态为稳态时，表示可以进行该VNFC级别的扩缩容。<br>- “非稳态”：表示系统或者本地为非稳态，不允许下发扩缩容或者动态上下线等命令。<br>- “状态获取超时”：表示可能有VNFC处于复位或者断链中，建议稍后重新执行该命令查询。 |

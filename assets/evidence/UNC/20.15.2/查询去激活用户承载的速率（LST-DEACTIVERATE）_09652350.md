## [命令功能](#ZH-CN_MMLREF_0209652350)

**适用NF：SGW-C、PGW-C、SMF、GGSN**

该命令用来查询去激活用户承载的速率。

## [注意事项](#ZH-CN_MMLREF_0209652350)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209652350)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652350)

无

## [使用实例](#ZH-CN_MMLREF_0209652350)

运营商配置了去活用户承载的速率，可以使用该命令来查询已配置的去活用户承载速率：

```
%%LST DEACTIVERATE:;%%
RETCODE = 0  操作成功

结果如下
--------
去活速率(个/秒)  =  1000
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209652350)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 去活速率(个/秒) | 该参数用于指定整系统每秒钟去活的承载数，对于2、3G接入表示承载数，对于4、5G接入表示会话数。 |

## [命令功能](#ZH-CN_MMLREF_0000001601560902)

**适用NF：PGW-C**

该命令用于查询PGW重定向功能设置。

## [注意事项](#ZH-CN_MMLREF_0000001601560902)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001601560902)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001601560902)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| APN | APN名称 | 可选必选说明：可选参数<br>参数含义：该参数用于指定APN实例名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~63。只能由“-”、数字、大小写字母和“.”组成，不能以“.”开头且不能出现连续两个“.”。不支持空格及“_”、“#”、“$”、“&”、“%”、“^”、“（”、“）”、“，”、“/”、“;”、“:”、“””、“`”等特殊字符，不区分大小写。<br>默认值：无<br>配置原则：<br>输入的APN名称需要符合APN命名规则，仅支持配置APN NI（Network Identifier），例如“huawei.com”。一般情况下用真实APN进行配置，但若开启虚拟APN映射功能，则使用请求APN进行配置。 |

## [使用实例](#ZH-CN_MMLREF_0000001601560902)

查询APN为“HUAWEI.COM”的PGW重定向功能开关配置：

```
%%LST APNPGWRNS: APN="HUAWEI.COM";%%
RETCODE = 0  操作成功

结果如下
------------------------
              APN  =  huawei.com
PGW重定向功能开关  =  不使能
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001601560902)

| 输出项名称 | 输出项解释 |
| --- | --- |
| APN名称 | 该参数用于指定APN实例名。 |
| PGW重定向功能开关 | 该参数用于指示是否支持PGW重定向功能。 |

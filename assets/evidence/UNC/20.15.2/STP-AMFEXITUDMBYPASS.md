# 停止用户退出UDM Bypass状态任务（STP AMFEXITUDMBYPASS）

- [命令功能](#ZH-CN_MMLREF_0000001258840357__1.3.1)
- [注意事项](#ZH-CN_MMLREF_0000001258840357__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0000001258840357__1.3.4)

## [命令功能](#ZH-CN_MMLREF_0000001258840357)

**适用NF：AMF**

该命令用于停止退出UDM Bypass。

## [注意事项](#ZH-CN_MMLREF_0000001258840357)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0000001258840357)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001258840357)

无

## [使用实例](#ZH-CN_MMLREF_0000001258840357)

当需要停止用户退出UDM Bypass任务，执行如下命令：

```
STP AMFEXITUDMBYPASS:;
```

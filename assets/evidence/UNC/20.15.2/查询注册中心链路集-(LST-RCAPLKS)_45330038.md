#### [命令功能](#ZH-CN_CONCEPT_0000001245330038)

**适用NF：SMSF**

此命令用于查询注册中心链路集配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001245330038)

- 无

#### [本地用户权限](#ZH-CN_CONCEPT_0000001245330038)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_CONCEPT_0000001245330038)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001245330038)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| LSX | 链路集索引 | 可选必选说明：必选参数<br>参数含义：该参数用于指定链路集的索引。<br>数据来源：本端规划<br>取值范围：0~1<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001245330038)

1. 查询链路集索引为0的注册中心链路集配置，可以用如下命令：
  LST RCAPLKS: LSX=0;
  ```
  %%LST RCAPLKS: LSX=0;%%
  RETCODE = 0  操作成功。

  操作结果如下
  ------------
  链路集索引  =  0
  注册中心号  =  0
    链路集名  =  noname
  (结果个数 = 1)

  ---    END
  ```

## [命令功能](#ZH-CN_MMLREF_0000001136529189)

**适用NF：SGW-C**

该命令用于查询SGW-C的全球唯一标识。

## [注意事项](#ZH-CN_MMLREF_0000001136529189)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001136529189)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001136529189)

无

## [使用实例](#ZH-CN_MMLREF_0000001136529189)

查询SGW-C的全球唯一标识，执行命令如下：

```
%%LST SGWCID:;%%
            RETCODE = 0  操作成功

            结果如下
            ------------------------
            移动国家码  =  460
            移动网号  =  01
            SGW-C标识  =  0
            (结果个数 = 1)

            ---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001136529189)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 移动国家码 | 该参数用于指定组成SGW-C的全球唯一标识的移动国家码信息。 |
| 移动网号 | 该参数用于指定组成SGW-C的全球唯一标识的移动网号信息。 |
| SGW-C标识 | 该参数用于指定组成SGW-C的全球唯一标识的SGW-C标识信息。 |

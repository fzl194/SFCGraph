## [操作步骤](#ZH-CN_OPI_0248458476)

1. 进入 “MML命令行-UNC” 窗口。
2. 配置 UNC 是否可以向UE发送网络信息。
  [**SET NGMMFUNC**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/移动性管理/MM协议参数管理/5G移动性管理/设置5G移动性管理功能（SET NGMMFUNC）_09653748.md)
  > **说明**
  > 如果此处配置 UNC 不可以向UE发送网络信息，则在AMF无法通过Configuration Update Command消息向UE下发网络标识、时区和夏令时等信息。
3. **可选：**配置NITZ下发时间时的本地时间根据UTC时间获取，避免向UE发送的时区不符合协议。
  **[SET AMFSOFTPARAOFBIT](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/软件参数管理/设置AMF软件参数比特位（SET AMFSOFTPARAOFBIT）_09651487.md)**
4. 基于区域范围和用户群组组合的粒度来灵活设置不同的网络名称。
    - 配置区域编码
      [**ADD AREACODE**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/通用配置管理/通用区域编码管理/增加区域编码（ADD AREACODE）_44006351.md)
    - 为指定的区域增加跟踪区成员
      [**ADD AREAMEM**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/通用配置管理/通用区域成员管理/增加跟踪区成员（ADD AREAMEM）_44006352.md)
    - 配置5G用户群标识
      [**ADD NGUSRGRP**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/通用配置管理/用户群组标识管理/增加5G用户群（ADD NGUSRGRP）_44006475.md)
    - 配置用户群成员
      [**ADD NGUSRGRPMEM**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/通用配置管理/用户群组成员管理/增加5G用户群成员（ADD NGUSRGRPMEM）_44006476.md)
5. 为指定区域和指定用户群配置网络名称。
  [**ADD NITZPLCY**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/移动性管理/NITZ管理/NITZ策略管理/增加NITZ策略（ADD NITZPLCY）_09652255.md)
  > **说明**
  > - UNC整系统的网络名称（运营商名称）已在[**ADD NGMNO**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/运营商管理/5G 移动网络运营商管理/增加5G模式移动网络运营商信息（ADD NGMNO）_09652141.md)命令中配置。当需要匹配的区域范围为所有区域，用户群组为所有用户时，则优先使用[**ADD NGMNO**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/运营商管理/5G 移动网络运营商管理/增加5G模式移动网络运营商信息（ADD NGMNO）_09652141.md)配置的移动网络运营商名称，其次使用[**ADD NITZPLCY**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/移动性管理/NITZ管理/NITZ策略管理/增加NITZ策略（ADD NITZPLCY）_09652255.md)配置的网络名称。
  > - 对于区域范围，网络名称与区域范围的匹配优先级从高到低依次为：“AREA_CODE(指定区域编码)”，“ALL_AREA(所有区域)”。
  > - 对于指定的用户（群），网络名称的优先级从高到低依次为：“USER_GROUP(用户群)/IMSI_PREFIX(指定IMSI前缀)”，“FOREIGN_USER(外网用户)”或“HOME_USER(本网用户)”，“ALL_USER(所有用户)”。
  > - USER_GROUP(用户群)与IMSI_PREFIX(指定IMSI前缀)为互斥关系。
  > - 同一个区域编码中，可以配置不同的USRGRPID，但是不同的USRGRPID在[**ADD NGUSRGRPMEM**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/通用配置管理/用户群组成员管理/增加5G用户群成员（ADD NGUSRGRPMEM）_44006476.md)中不能有相同的IMSIPRE成员。
  > - 当配多个指定区域编码时，不同指定区域编码对应的区域成员在[**ADD AREAMEM**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/通用配置管理/通用区域成员管理/增加跟踪区成员（ADD AREAMEM）_44006352.md)中不允许交叠。
6. 配置时区和夏令时信息。
  [**ADD NGTZLST**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/5G接入业务管理/移动性管理/多时区管理/5G多时区参数/增加5G多时区参数（ADD NGTZLST）_71994441.md)

## [任务示例](#ZH-CN_OPI_0248458476)

任务描述

在5G网络中启用多时区功能。用户群组标识为20，为该用户群组在特定区域配置网络名称，全称为“Ch Mobile”，简称为“CM”。特定区域的区域编码为“MyTown”，包含跟踪区TAC1（010101）和TAC2（010102）。

脚本

//设置 UNC 可以向UE发送网络信息和时区信息。

```
SET NGMMFUNC: SNDNETINFO=YES;
```

//配置NITZ下发时间时的本地时间根据UTC时间获取，避免向UE发送的时区不符合协议。

```
SET AMFSOFTPARAOFBIT: DT=Dw, DWORDNUM=5, VALUE=VALUE_1, POSITION=15;
```

//配置区域范围和区域成员。

```
ADD AREACODE：AREACODE="MyTown";
ADD AREAMEM：AREACODE="MyTown", MCC=123, MNC=456, BGNTAC=010101, ENDTAC=010102;
```

//配置用户群组和群成员。

```
ADD NGUSRGRP：USRGRPID=20;
ADD NGUSRGRPMEM：USRGRPID=20, IMSIPRE=123456;
```

//为指定区域和指定用户群配置网络名称。

```
ADD NITZPLCY：AREARANGE=AREA_CODE, AREACODE="MyTown", SUBRANGE= USER_GROUP, USRGRPID=20, FULLNAME="Ch Mobile", SHORTNAME="CM";
```

//配置时区和夏令时信息。

```
ADD NGTZLST: TZID=1, TZONE=E0000, DST=YES, SM=DATE, SMONTH=JAN, SDAY=15, ST=18&10&02, EM=DATE, EMONTH=MAR, EDAY=1, ET=23&10&56, TO=1;
```

//为指定区域的用户配置时区标识。

```
ADD NGAREATZ: MCC="123", MNC="456", BGNTAC="010101", ENDTAC="010102", TZID=1;
```

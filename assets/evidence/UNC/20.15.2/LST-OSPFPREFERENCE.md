#### [命令功能](#ZH-CN_CONCEPT_0000001600866409)

该命令用于查询OSPF协议路由的优先级配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001600866409)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600866409)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600866409)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | 进程号 | 可选必选说明：可选参数<br>参数含义：OSPF进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |
| ROUTETYPE | 路由类型 | 可选必选说明：可选参数<br>参数含义：路由类型。<br>数据来源：本端规划<br>取值范围：枚举类型。<br>- default：区域内和区域间路由。<br>- ase：AS-External路由。<br>- intra：区域内路由。<br>- inter：区域间路由。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600866409)

查询OSPF进程1下ASE路由的优先级配置：

```
LST OSPFPREFERENCE:PROCID=1,ROUTETYPE=ase;
```

```

RETCODE = 0  操作成功。

结果如下
----------
      进程号  =  1
    拓扑标识  =  0
    路由类型  =  AS-External路由
  路由优先级  =  150
路由策略名称  =  NULL
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600866409)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 进程号 | 该参数指定用户创建的OSPF进程号。 |
| 拓扑标识 | 该参数指定用户创建的OSPF拓扑标识。 |
| 路由类型 | 该参数指定OSPF路由类型。 |
| 路由优先级 | 该参数指定OSPF路由优先级，路由优先级的值越小，实际的优先程度越高。 |
| 路由策略名称 | 该参数指定路由策略名称，使用路由策略为特定的路由设定优先级。 |

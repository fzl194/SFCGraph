- [参数说明](#ZH-CN_MMLREF_0209653117__1.3.4)

## [命令功能](#ZH-CN_MMLREF_0209653117)

**适用NF：AMF、SMF、NSSF、SMSF、NCG**

该命令用于查询本地配置的对端NF实例支持的外部群组标识的信息。

## [注意事项](#ZH-CN_MMLREF_0209653117)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209653117)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209653117)

无

## [使用实例](#ZH-CN_MMLREF_0209653117)

查询对端NF的外部群组信息。

```
%%LST PNFEXGROUPID:;%%
RETCODE = 0  操作成功

结果如下
--------
  NF实例标识  =  udm_instance_0
外部群组标识  =  externalGroupId01
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209653117)

| 输出项名称 | 输出项解释 |
| --- | --- |
| NF实例标识 | 该参数用于指定NF实例标识。 |
| 外部群组标识 | 该参数用于指定外部组标识。 |

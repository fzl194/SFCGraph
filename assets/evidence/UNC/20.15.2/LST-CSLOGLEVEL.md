## [命令功能](#ZH-CN_MMLREF_0209587888)

此命令用于查询日志输出级别。

## [注意事项](#ZH-CN_MMLREF_0209587888)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0209587888)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209587888)

无

## [使用实例](#ZH-CN_MMLREF_0209587888)

查询日志输出级别：

```
%%LST CSLOGLEVEL:;%%
RETCODE = 0  操作成功

结果如下
--------
日志级别  =  调试级别
(结果个数 = 1)

---    END
```

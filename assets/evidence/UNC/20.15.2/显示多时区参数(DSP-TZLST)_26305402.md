#### [命令功能](#ZH-CN_MMLREF_0000001126305402)

**适用网元：SGSN**

此命令用于查询某个时区是否进入夏令时。

#### [注意事项](#ZH-CN_MMLREF_0000001126305402)

无。

#### [本地用户权限](#ZH-CN_MMLREF_0000001126305402)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126305402)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126305402)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| TZID | 时区标识 | 可选必选说明：可选参数<br>参数含义：该参数用于指定待查询时区信息的索引。<br>取值范围：1～24<br>默认值：无 |

#### [使用实例](#ZH-CN_MMLREF_0000001126305402)

查询 “时区标识” 为 “1” 的时区信息：

DSP TZLST: TZID=1;

```
%%DSP TZLST: TZID=1;%%
RETCODE = 0  操作成功。

输出结果如下
--------------
          时区标识  =  1
              时区  =  GMT+01:00
夏令时偏移量(小时)  =  1
(结果个数 = 1)

---    END
```

查询多条时区信息：

DSP TZLST:;

```
%%DSP TZLST:;%%
RETCODE = 0  操作成功。

输出结果如下
--------------
 时区标识  时区       夏令时偏移量(小时)

 1         GMT+03:30  0                 
 2         GMT+05:45  0                 
(结果个数 = 2)

---    END
```

#### [输出结果说明](#ZH-CN_MMLREF_0000001126305402)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 时区标识 | 显示时区信息的索引。<br>取值说明：1～24 |
| 时区 | 显示时区信息的时区。<br>取值说明：<br>- “E0000(GMT+00:00)”<br>- “E0100(GMT+01:00)”<br>- “E0200(GMT+02:00)”<br>- “E0300(GMT+03:00)”<br>- “E0330(GMT+03:30)”<br>- “E0400(GMT+04:00)”<br>- “E0430(GMT+04:30)”<br>- “E0500(GMT+05:00)”<br>- “E0530(GMT+05:30)”<br>- “E0545(GMT+05:45)”<br>- “E0600(GMT+06:00)”<br>- “E0630(GMT+06:30)”<br>- “E0700(GMT+07:00)”<br>- “E0800(GMT+08:00)”<br>- “E0845(GMT+08:45)”<br>- “E0900(GMT+09:00)”<br>- “E0930(GMT+09:30)”<br>- “E1000(GMT+10:00)”<br>- “E1030(GMT+10:30)”<br>- “E1100(GMT+11:00)”<br>- “E1130(GMT+11:30)”<br>- “E1200(GMT+12:00)”<br>- “E1245(GMT+12:45)”<br>- “E1300(GMT+13:00)”<br>- “E1400(GMT+14:00)”<br>- “W0100(GMT-01:00)”<br>- “W0200(GMT-02:00)”<br>- “W0300(GMT-03:00)”<br>- “W0330(GMT-03:30)”<br>- “W0400(GMT-04:00)”<br>- “W0430(GMT-04:30)”<br>- “W0500(GMT-05:00)”<br>- “W0600(GMT-06:00)”<br>- “W0700(GMT-07:00)”<br>- “W0800(GMT-08:00)”<br>- “W0900(GMT-09:00)”<br>- “W0930(GMT-09:30)”<br>- “W1000(GMT-10:00)”<br>- “W1100(GMT-11:00)”<br>- “W1200(GMT-12:00)”<br>说明：时区包括GMT+TIME和GMT-TIME两种形式，+代表进入东时区，-代表进入西时区。例如：GMT+14:00代表进入东时区后，时区与世界标准时区提前14小时；GMT-01:00代表进入西时区后，时区与世界标准时区延缓1小时。 |
| 夏令时偏移量(小时) | 显示夏令时相对于标准时的时间偏移量。<br>取值说明：0～2。<br>- 0 表示没有进入夏令时。<br>- 1 表示夏令时偏移量为1小时。<br>- 2 表示夏令时偏移量为2小时。 |

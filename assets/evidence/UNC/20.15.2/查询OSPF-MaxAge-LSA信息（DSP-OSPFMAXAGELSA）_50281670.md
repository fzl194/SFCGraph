# 查询OSPF MaxAge LSA信息（DSP OSPFMAXAGELSA）

- [命令功能](#ZH-CN_CONCEPT_0000001550281670__1.3.1.1)
- [注意事项](#ZH-CN_CONCEPT_0000001550281670__1.3.2.1)

- [参数说明](#ZH-CN_CONCEPT_0000001550281670__1.3.4.1)

- [输出结果说明](#ZH-CN_CONCEPT_0000001550281670__1.3.6.1)

#### [命令功能](#ZH-CN_CONCEPT_0000001550281670)

该命令用于查询当前设备中达到最大老化时间的Router LSA或收到远端flush的Router LSA的信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001550281670)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550281670)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550281670)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | 进程号 | 可选必选说明：可选参数<br>参数含义：OSPF进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001550281670)

查询OSPF进程1中达到最大老化时间或收到远端flush的Router LSA的信息：

```
DSP OSPFMAXAGELSA:PROCID=1;
```

```

RETCODE = 0  操作成功。

结果如下
------------------------
                   进程号  =  1
                   区域号  =  0.0.0.0
               路由器标识  =  10.10.10.1
    LSA报头中的链路状态ID  =  10.10.10.1
  LSA时限达到MaxAge的次数  =  0
  LSA上次达到MaxAge的时间  =  NULL
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001550281670)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 进程号 | 该参数用于表示OSPF进程号。 |
| 区域号 | 该参数用于表示OSPF区域号。 |
| 路由器标识 | 该参数用于表示OSPF路由器标识。 |
| LSA报头中的链路状态ID | 该参数用于表示LSA报头中的链路状态ID。 |
| LSA时限达到MaxAge的次数 | 该参数用于表示达到老化时间的LSA的次数。 |
| LSA上次达到MaxAge的时间 | 该参数用于表示最后一次达到老化时间的LSA的时间。 |

## [命令功能](#ZH-CN_MMLREF_0000001135599382)

**适用NF：SMF**

该命令用于查询QoS监测规则。

## [注意事项](#ZH-CN_MMLREF_0000001135599382)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001135599382)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [使用实例](#ZH-CN_MMLREF_0000001135599382)

查询“IMSI”为“123030123456789”的QoS监测功能配置：

```
%%LST QOSMONTRULE:MATCHMODE=IMSI,IMSI="123030123456789";%%
RETCODE = 0  操作成功

结果如下
--------
              IMSI  =  123030123456789
       QoS监测开关  =  不使能
      时延监测方向  =  往返监测
          上报频率  =  会话释放触发
下行时延阈值(毫秒)  =  NULL
上行时延阈值(毫秒)  =  NULL
往返时延阈值(毫秒)  =  NULL
等待最小值时间(秒)  =  NULL
      测量周期(秒)  =  NULL
(结果个数 = 1)

---    END
```

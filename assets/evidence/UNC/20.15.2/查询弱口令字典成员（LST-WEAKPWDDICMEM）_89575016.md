#### [命令功能](#ZH-CN_CONCEPT_0000001289575016)

**适用NF：NCG**

该命令用于查询弱口令字典中的成员。

#### [注意事项](#ZH-CN_CONCEPT_0000001289575016)

无。

#### [本地用户权限](#ZH-CN_CONCEPT_0000001289575016)

manage-ug；system-ug

#### [网管用户权限](#ZH-CN_CONCEPT_0000001289575016)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001289575016)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| WEAKPWD | 弱口令 | 可选必选说明：可选参数<br>参数含义：用于查询弱口令字典中的成员。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～32。<br>默认值：无<br>配置原则：若不输入，则表示所有查询弱口令字典中所有成员。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001289575016)

查询弱口令字典中成员信息 ：

```
LST WEAKPWDDICMEM:;
```

```
RETCODE = 0  操作成功
结果如下:

Cspdbg@2017

Huawei#12

(结果个数 = 6)
---    END
```

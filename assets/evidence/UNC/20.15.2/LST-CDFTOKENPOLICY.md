## [命令功能](#ZH-CN_MMLREF_0000001266773960)

**适用NF：NCG**

该命令用于查询cdf token权重上报开关。

## [注意事项](#ZH-CN_MMLREF_0000001266773960)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001266773960)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001266773960)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| WEIGHTSWITCH | cdf token权重开关 | 可选必选说明：可选参数<br>参数含义：设置CDF token权重开关。<br>数据来源：本端规划<br>取值范围：<br>- “ENABLE（使能）”：需要上报cdf token权重时，需要配置权重开关为使能<br>- “DISABLE（不使能）”：不需要上报cdf token权重时，需要配置权重开关为不使能<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001266773960)

查询cdf token权重开关配置：

```
+++    UNC/*MEID:0 MENAME:UNC_VNF_ncg001*/        2022-05-24 20:27:32+8:00
O&M    #3936
%%LST CDFTOKENPOLICY:;%%
RETCODE = 0  操作成功

结果如下
--------
cdf token权重开关  =  使能
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001266773960)

| 输出项名称 | 输出项解释 |
| --- | --- |
| cdf token权重开关 | 设置CDF token权重开关。 |

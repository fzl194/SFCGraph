# 查询NF的订阅门限（LST NRFSUBTHRESHOLD）

- [注意事项](#ZH-CN_MMLREF_0209653067__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0209653067__1.3.4)

## [命令功能](#ZH-CN_MMLREF_0209653067)

**适用NF：NRF**

该命令用于查询NF的订阅个数的上限。

## [注意事项](#ZH-CN_MMLREF_0209653067)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209653067)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209653067)

无

## [使用实例](#ZH-CN_MMLREF_0209653067)

查询NF的订阅门限：

```
LST NRFSUBTHRESHOLD:;
%%LST NRFSUBTHRESHOLD:;%%
RETCODE = 0  操作成功

结果如下
------------------------
   订阅门限  =  90
内部订阅门限 = 100
(结果个数 = 1)
```

## [输出结果说明](#ZH-CN_MMLREF_0209653067)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 订阅门限 | 该参数用于表示NF的订阅个数的上限。 |
| 内部订阅门限 | 该参数用于设置内部订阅个数的上限，仅在SCP和NRF联合部署场景下生效。 |

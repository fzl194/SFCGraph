#### [命令功能](#ZH-CN_CONCEPT_0129626994)

该命令用于显示CSDB插件的信息，包括插件的标识、连接状态、版本号等。

#### [注意事项](#ZH-CN_CONCEPT_0129626994)

通过命令 **[DSP DBINSTANCE](../实例管理/查询CSDB子实例信息（DSP DBINSTANCE）_29626987.md)** 查询CSDB子实例信息， 返回信息中初始容量和总容量为0时该子实例不支持查询插件信息。

#### [操作用户权限](#ZH-CN_CONCEPT_0129626994)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0129626994)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| INSTANCEID | 子实例标识 | 可选必选说明：必选参数。<br>参数含义：该参数用于指定特定子实例，可以通过<br>**[DSP DBINSTANCE](../实例管理/查询CSDB子实例信息（DSP DBINSTANCE）_29626987.md)**<br>命令查询获取。<br>数据来源：本端规划<br>取值范围：1～100。<br>默认值：无。 |

DSP DBPLUGIN: INSTANCEID=2;

#### [输出结果说明](#ZH-CN_CONCEPT_0129626994)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 子实例标识 | 唯一标识一个子实例。<br>取值范围：1～100。 |
| 虚拟化网络功能组件标识 | 客户端进程所在的VNFC标识。<br>取值范围：0～4294967295。 |
| 插件标识 | 客户端进程指定的插件标识。<br>取值范围：0～4294967295。 |
| 插件连接状态 | 插件的连接状态。<br>取值范围：<br>- “Uninitialized”：未初始化。<br>- “Initialized”：初始化。<br>- “Connected”：连接。<br>- “Steady”：稳态。<br>- “Remote Storage Unavailable”：远程存储不可用。<br>- “Other state”：其他状态。 |
| 插件版本号 | 插件的版本号。<br>取值范围：不超过63个英文字符。 |
| 资源单元名称 | 指定进程所在的资源单元名称。<br>取值范围：不超过63个英文字符。 |
| 插件进程标识 | 参数含义：插件在业务VNFC内的进程序号。<br>取值范围: 0-4294967295 |
| 物理机标识 | 插件所在的物理机标识。<br>取值范围：不超过127个英文字符。 |
| Pod名称 | 指定进程所在的Pod名称。<br>取值范围：不超过63个英文字符。 |
| 容器类型 | 指定进程所在的容器类型。<br>取值范围：不超过63个英文字符。 |
| 压缩模式 | 指定进程所在的压缩模式。<br>取值范围：<br>- “Closed”：关闭。<br>- “lz4”：lz4。<br>- “zstd”：zstd。<br>- “Other state”：其他状态。 |

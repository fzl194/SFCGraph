## [操作场景](#ZH-CN_OPI_0174013202)

当部署基于使用流量的内容计费时，实现基于不同业务的计费，需要激活该功能。

> **说明**
> 适用于 GGSN、 SGW-C、PGW-C、SMF。

## [必备事项](#ZH-CN_OPI_0174013202)

前提条件

- 请仔细阅读[WSFD-109004 基于业务流量的计费特性概述](特性概述_74013201.md)。
- 完成[激活离线计费](../WSFD-011201 支持离线计费/激活离线计费_25768943.md)、[激活Gy/Diameter在线计费](../WSFD-109001 Gy_Diameter在线计费/激活Gy_Diameter在线计费_27639894.md)或[激活支持融合计费](../WSFD-011206 支持融合计费/激活支持融合计费_73659885.md)。
- 完成[激活内容计费](../WSFD-109002 内容计费基本功能/激活内容计费_74013177.md)。
- 完成加载License。

## [操作步骤](#ZH-CN_OPI_0174013202)

1. 激活基于业务流量的计费的License。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
2. 修改在线计费模板/融合计费模板，配置基于业务流量的计费。
  [**MOD DCCTEMPLATE**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/计费管理/在线计费/信用控制/信用控制模板/修改DCC模板（MOD DCCTEMPLATE）_09896924.md)
  [**MOD CCT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/计费管理/融合计费/融合计费模板/修改融合计费模板（MOD CCT）_09652641.md)
  > **说明**
  > DCC在线计费模板和融合计费模板均已配置可参见 [激活Gy/Diameter在线计费](../WSFD-109001 Gy_Diameter在线计费/激活Gy_Diameter在线计费_27639894.md) 或 [激活支持融合计费](../WSFD-011206 支持融合计费/激活支持融合计费_73659885.md) ，此处通过MOD命令配置流量相关参数。

## [任务示例](#ZH-CN_OPI_0174013202)

任务描述

打开基于业务流量的计费的License，配置基于业务流量计费的流量参数。

脚本

//打开基于业务流量的计费的License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV3WPVBCS11", SWITCH=ENABLE;
```

//配置基于业务流量计费的流量参数（融合计费）。

```
MOD CCT: CCTMPLTNAME="cct-test", VQT=30;
```

//配置基于业务流量计费的流量参数（在线计费）。

```
MOD DCCTEMPLATE: DCCTMPLTNAME="dcc-test", VQT=30;
```

## [操作场景](#ZH-CN_OPI_0275538866)

本操作指导介绍在运行网络中激活网络语音业务信息上报功能的操作过程。

网络语音业务信息上报功能可以提供LTE网络中所有用户语音业务的CHR（Call History Record），CHR中包含语音业务流程的相关信息（如流程发生的时间、流程时长、位置、失败原因值等）。运营商可以通过这些信息快速定位故障原因，或对业务质量进行分析以提升服务质量。

> **说明**
> 适用于MME。

## [必备事项](#ZH-CN_OPI_0275538866)

前提条件

- 请仔细阅读[WSFD-201003 网络语音业务信息上报功能特性概述](特性概述_75543779.md)。
- 完成加载License。
- 完成激活[WSFD-202001 CHR功能](../../操作维护功能/WSFD-202001 CHR功能_50287959.md)。
  > **说明**
  > 可以在部署“WSFD-202001 CHR功能”特性的时候一并开启网络语音业务信息上报功能，避免相同命令再次执行，简化操作。
- 操作人员已经登录U2020/MAE。

数据

该操作无需准备数据。

## [操作步骤](#ZH-CN_OPI_0275538866)

1. 进入 “MML命令行-UNC” 窗口。
2. 设置需要上报CHR单据的信令流程。
  [**SET CHRCFG**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/CHR管理/CHR配置/设置CHR配置（SET CHRCFG）_72225295.md)
3. 设置CloudUDN类型。
  [**SET CHRLOWPERFSVRSUB**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/移动性管理/低性能CHR服务器的单据订阅条件/设置低性能CHR服务器的单据订阅条件（SET CHRLOWPERFSVRSUB）_72345083.md)
4. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)

## [任务示例](#ZH-CN_OPI_0275538866)

任务描述

开启网络语音业务信息上报功能。

脚本

//设置Gb/Iu/S1模式下ATTACH流程成功时上报CHR，Gb/Iu模式下ATTACH和RAU流程失败时上报CHR，S1模式下ATTACH和TAU流程失败时上报CHR，CHR单据设备号填充方式为SGSN_NUMBER。

```
SET CHRCFG: GERANSUCC=ATTACH-1, GERANFAIL=ATTACH-1&RAU-1,UTRANSUCC=ATTACH-1, UTRANFAIL=ATTACH-1&RAU-1, EPSSUCC=ATTACH-1, EPSFAIL=ATTACH-1&TAU-1, RPTDEVNO=SGSN_NUMBER;
```

//设置CloudUDN类型。

```
SET CHRLOWPERFSVRSUB: GERANSUCC=ATTACH-1, GERANFAIL=ATTACH-1&RAU-1,UTRANSUCC=ATTACH-1, UTRANFAIL=ATTACH-1&RAU-1, EPSSUCC=ATTACH-1, EPSFAIL=ATTACH-1&TAU-1;
```

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2NVIR01", SWITCH=ENABLE;
```

#### [命令功能](#ZH-CN_CONCEPT_0000001600866489)

该命令用于显示DNS IPv6动态域名。

#### [注意事项](#ZH-CN_CONCEPT_0000001600866489)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600866489)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600866489)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| HOST | DNS域名 | 可选必选说明：可选参数<br>参数含义：该参数用于显示DNS IPv6动态域名。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围为1～255。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600866489)

显示IPv6动态域名：

```
DSP DNSDYNAMICIPV6HOST:;
```

```

        RETCODE = 0  操作成功。

        结果如下
        --------
            DNS域名  =  huawei.com
           IPv6地址  =  2001:DB8::11
      老化时间（s）  =  544
        (结果个数 = 1)

        ---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600866489)

| 输出项名称 | 输出项解释 |
| --- | --- |
| DNS域名 | 用于显示DNS IPv6动态域名。 |
| IPv6地址 | 用于显示DNS IPv6动态域名对应的IPv6地址。 |
| 老化时间（s） | 用于显示动态DNS表项的老化时间。 |

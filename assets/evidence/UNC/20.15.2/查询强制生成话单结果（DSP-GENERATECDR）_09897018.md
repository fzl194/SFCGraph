#### [命令功能](#ZH-CN_CONCEPT_0209897018)

**适用NF：SGW-C、PGW-C、SMF**

该命令用来查询上次强制产生用户话单结果信息。

#### [注意事项](#ZH-CN_CONCEPT_0209897018)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0209897018)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0209897018)

无。

#### [使用实例](#ZH-CN_CONCEPT_0209897018)

查询上次强制产生所有用户话单信息：

```
DSP GenerateCDR:;
```

```

RETCODE = 0 操作成功

上次强制话单生成结果
-------------------------
                    POD名称  =  uncpod-012-30-0-243__1020__0
                       时间  =  2015-08-21 14:17:47
                 成功话单数  =  1
                 失败话单数  =  0
                     结果  =  成功
(结果个数 = 1)

---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0209897018)

| 输出项名称 | 输出项解释 |
| --- | --- |
| POD名称 | 用于指定POD名称。 |
| 时间 | 上次强制产生话单的时间。 |
| 成功话单数 | 成功话单数。 |
| 失败话单数 | 失败话单数。 |
| 结果 | 上次强制生成话单的结果。 |

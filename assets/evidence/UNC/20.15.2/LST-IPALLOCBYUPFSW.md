# 查询基于UPF的地址分配开关（LST IPALLOCBYUPFSW）

- [注意事项](#ZH-CN_MMLREF_0249644922__1.3.2)
- [参数说明](#ZH-CN_MMLREF_0249644922__1.3.4)

## [命令功能](#ZH-CN_MMLREF_0249644922)

**适用NF：PGW-C、SMF、GGSN**

该命令用于查询基于UPF的IP地址分配开关。

## [注意事项](#ZH-CN_MMLREF_0249644922)

无

#### [操作用户权限](#ZH-CN_MMLREF_0249644922)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0249644922)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| NFINSTANCEID | UPF实例标识 | 可选必选说明：可选参数<br>参数含义：该参数用于指定UPF实例标识。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是0~255。<br>默认值：无<br>配置原则：<br>该参数取值应与UPNODE中NFINSTANCENAME保持一致，使用该前需通过ADD UPNODE添加一组记录。 |

## [使用实例](#ZH-CN_MMLREF_0249644922)

查询基于UPF的IP地址分配开关：

```
%%LST IPALLOCBYUPFSW:;%%
RETCODE = 0  操作成功

结果如下
------------------------
              UPF实例标识  =  upf_instance_1
基于UPF的IPv4地址分配开关  =  继承
基于UPF的IPv6地址分配开关  =  继承
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0249644922)

| 输出项名称 | 输出项解释 |
| --- | --- |
| UPF实例标识 | 该参数用于指定UPF实例标识。 |
| 基于UPF的IPv4地址分配开关 | 该参数用于指定基于UPF的IPv4地址分配功能开关。 |
| 基于UPF的IPv6地址分配开关 | 该参数用于指定基于UPF的IPV6地址分配功能开关。 |

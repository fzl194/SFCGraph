#### [命令功能](#ZH-CN_MMLREF_0000001126145828)

**适用网元：MME**

此命令用于查询DCN。

#### [注意事项](#ZH-CN_MMLREF_0000001126145828)

无。

#### [本地用户权限](#ZH-CN_MMLREF_0000001126145828)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126145828)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126145828)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| DCNID | DCN ID | 可选必选说明：可选参数<br>参数含义：该参数用于指定DCN标识。<br>数据来源：全网规划<br>取值范围：0~255<br>默认值：无 |
| MCC | 移动国家代码 | 可选必选说明：可选参数<br>参数含义：该参数用于指定DCN所属PLMN的移动国家号码。<br>数据来源：全网规划<br>取值范围：3位十进制数<br>默认值：无 |
| MNC | 移动网号 | 可选必选说明：可选参数<br>参数含义：该参数用于指定DCN所属PLMN的移动网号码。<br>数据来源：全网规划<br>取值范围：位数为2或3的十进制数<br>默认值：无 |

#### [使用实例](#ZH-CN_MMLREF_0000001126145828)

1. 查询 “DCN ID” 为 “0” 的DCN记录：
  LST DCN: DCNID=0;
  ```
  %%LST DCN: DCNID=0;%%
  RETCODE = 0  操作成功

  操作结果如下：
  --------------
        DCN ID  =  0
  移动国家代码  =  123
      移动网号  =  01
       DCN类型  =  默认DCN
          描述  =  eMtc
  (结果个数 = 1)

  ---    END
  ```

# 设置智能双N7会话特性是否开启（SET IPEN7SESSFUNC）

- [注意事项](#ZH-CN_MMLREF_0000002080872792__1.3.2)

## [命令功能](#ZH-CN_MMLREF_0000002080872792)

**适用NF：SMF、PGW-C**

该命令用于设置智能双N7会话特性是否开启。

## [注意事项](#ZH-CN_MMLREF_0000002080872792)

- 该命令执行后立即生效。

- 系统部署完成后，已经存在初始记录，参数的初始记录值如下表：

| IPEN7SESSFUNCSW |
| --- |
| DISABLE |

#### [操作用户权限](#ZH-CN_MMLREF_0000002080872792)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000002080872792)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| IPEN7SESSFUNCSW | 智能双N7会话特性开关 | 可选必选说明：必选参数<br>参数含义：智能双N7会话特性开关。<br>数据来源：全网规划<br>取值范围：<br>- “DISABLE（不使能）”：智能N7会话不使能<br>- “ENABLE（使能）”：智能N7会话使能<br>默认值：无。<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000002080872792)

如果要创建智能双N7会话，则打开智能双N7会话特性开关：

```
SET IPEN7SESSFUNC: IPEN7SESSFUNCSW=ENABLE;
```

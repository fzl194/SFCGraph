# 删除智能分流ULCL部署模式（RMV MULDNNULCLMODE）

- [注意事项](#ZH-CN_MMLREF_0000001647540305__1.3.2)

## [命令功能](#ZH-CN_MMLREF_0000001647540305)

**适用NF：SMF**

该命令用于删除指定智能分流专网DNN的ULCL部署模式。

## [注意事项](#ZH-CN_MMLREF_0000001647540305)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0000001647540305)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001647540305)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| DEDDNN | 智能分流专网DNN | 可选必选说明：必选参数<br>参数含义：该参数指定智能分流专网DNN名称。<br>数据来源：全网规划<br>取值范围：字符串类型，输入长度范围是1~63。只能由“-”、数字、大小写字母和“.”组成，不能以“.”开头且不能出现连续两个“.”。不支持空格及“_”、“#”、“$”、“&”、“%”、“^”、“（”、“）”、“，”、“/”、“;”、“:”、“””、“`”等特殊字符，不区分大小写。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001647540305)

删除指定智能分流专网DNN的ULCL部署模式，智能分流专网DNN为“special.dnn”。

```
RMV MULDNNULCLMODE: DEDDNN="special.dnn";
```

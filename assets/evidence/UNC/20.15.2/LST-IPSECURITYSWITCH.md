#### [命令功能](#ZH-CN_CONCEPT_0000001238232949)

该命令用于查询CSLB中IP选项安全开关配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001238232949)

无

#### [操作用户权限](#ZH-CN_CONCEPT_0000001238232949)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组；

#### [参数说明](#ZH-CN_CONCEPT_0000001238232949)

无

#### [使用实例](#ZH-CN_CONCEPT_0000001238232949)

查询CSLB中IP选项安全开关的配置。

```
%%LST IPSECURITYSWITCH:;%%
RETCODE = 0  操作成功

操作结果如下：
--------------
      IPv4选项安全配置开关  =  IP安全开关关闭
IPv6扩展头选项安全配置开关  =  IP安全开关关闭
(结果个数 = 1)

---    END
```

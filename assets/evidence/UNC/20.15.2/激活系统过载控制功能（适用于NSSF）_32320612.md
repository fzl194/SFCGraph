## [操作步骤](#ZH-CN_OPI_0232320612)

1. 进入 “MML命令行-UNC” 窗口。
2. 设置基于CPU过载流控开关。
  **[SET FCSWITCH](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/服务通信管理/流控管理/设置流控开关（SET FCSWITCH）_09587940.md)**
3. 设置服务化接口上指定消息类型的固定速率。
  **[SET HTTPFIXEDFCMSG](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/HTTP功能管理/HTTP流控管理/HTTP指定消息类型固定速率流控管理/设置HTTP指定消息类型固定速率流控信息（SET HTTPFIXEDFCMSG）_84132110.md)**

## [任务示例](#ZH-CN_OPI_0232320612)

任务描述

系统中度或者重度过载时，启动自保流控。

脚本

//设置基于CPU过载流控开关。

```
SET FCSWITCH: SWITCH=ENABLE;
```

//设置服务化接口上指定消息类型的固定速率流控参数。

```
SET HTTPFIXEDFCMSG:MSGTYPE=NNSSF_NSSAIAVAILABILITY_NSSAIAVAILABILITYPOST,FCSWITCH=ON,THRESHOLD=4000;
```

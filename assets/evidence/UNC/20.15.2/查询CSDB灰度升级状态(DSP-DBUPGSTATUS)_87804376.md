#### [命令功能](#ZH-CN_CONCEPT_0000001187804376)

该命令用于查询灰度升级阶段状态。

#### [注意事项](#ZH-CN_CONCEPT_0000001187804376)

该命令回显结果中所显示的灰度升级阶段与通过 **[LST DBUPGSTAGE](查询CSDB灰度升级阶段(LST DBUPGSTAGE)_33083965.md)** 命令查询到的阶段一致。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001187804376)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001187804376)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001187804376)

查询到当前的 **“灰度升级阶段”** 为 **“CSDB灰度升级结束” ，** **“灰度升级阶段状态”** 为 **“已成功”** ：

```
%%DSP DBUPGSTATUS:;%%
RETCODE = 0  操作成功

操作结果如下：
--------------
    灰度升级阶段  =  CSDB灰度升级结束
灰度升级阶段状态  =  已成功
(结果个数 = 1)

---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001187804376)

| **输出项名称** | **输出项解释** |
| --- | --- |
| 灰度升级阶段 | 当前所处的灰度升级阶段。 |
| 灰度升级阶段状态 | 当前灰度升级阶段的处理状态。<br>取值范围：<br>- “执行中”：阶段正在处理<br>- “已失败”：阶段处理已失败<br>- “已成功”：阶段处理已完成 |

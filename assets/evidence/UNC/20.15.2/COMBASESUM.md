## [命令功能](#ZH-CN_MMLREF_0000001097800864)

该命令用于显示Base平面亚健康链路概况。

## [注意事项](#ZH-CN_MMLREF_0000001097800864)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001097800864)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001097800864)

无

## [输出结果说明](#ZH-CN_MMLREF_0000001097800864)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 源端节点ID | 该参数用于标识源端节点ID。 |
| 源端Pod ID | 该参数用于标识源端Pod ID。 |
| 总链路数 | 该参数用于标识与该POD有通信的Base平面总链路数。 |
| 亚健康链路数 | 该参数用于标识与该POD有通信关系的Base平面亚健康链路数。亚健康链路指的是该链路的亚健康度超过了阈值。亚健康度为一个统计周期内的（丢包率（千分比）+ 5 * 错包率（千分比））/ 2，统计周期、阈值可通过<br>[**LST COMBASEHEALTH**](查询Base平面的亚健康检测参数（LST COMBASEHEALTH）_32217475.md)<br>查询。 |
| 最小亚健康度 | 该参数用于标识亚健康链路中的最小亚健康度的值，。 |
| 最大亚健康度 | 该参数用于标识亚健康链路中的最大亚健康度。 |
| 平均亚健康度 | 该参数用于标识亚健康链路中的平均亚健康度。计算方式为统计源端POD为当前POD的所有亚健康链路的亚健康度的平均值。 |

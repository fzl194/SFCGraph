#### [命令功能](#ZH-CN_CONCEPT_0192520045)

该命令用于显示PAE转发实例信息。

#### [注意事项](#ZH-CN_CONCEPT_0192520045)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_CONCEPT_0192520045)

G_1，管理员级别命令组；G_2，操作员级别命令组; G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0192520045)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| ENDPOINT | 服务地址 | 可选必选说明：可选参数<br>参数含义：用于指定微服务的实例标识。如果不输入该参数，则显示所有实例信息。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为0～127。数字“0～9”、大写字母“A～Z”、小写字母“a～z”、特殊字符“-”、“_”、“.”、“+”、空格符以及中文字符，其他均为非法字符。<br>默认值：无<br>配置原则：使用<br>**[DSP PAENODE](显示PAE节点信息（DSP PAENODE）_92520008.md)**<br>查看服务地址。 |

#### [使用实例](#ZH-CN_CONCEPT_0192520045)

显示PAE转发实例信息：

```
DSP PAEFWDINSTANCE:;
```

```
RETCODE = 0  操作成功。

结果如下

vup-pod-1                             正常          在线      1113    有状态    是            是            
(结果个数 = 2)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0192520045)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 服务地址 | 用于指定微服务的实例标识。 |
| 健康检测状态 | 当前节点状态，分为初始化、正常和脱管。 |
| 服务状态 | 当前节点的服务状态，分为在线和离线两种状态。 |
| 资源ID | 资源ID。 |
| 部署类型 | 用于表示部署类型 |
| 发布网络标识 | 用于表示发布网络标识 |
| 健康检查标识 | 用于表示健康检查标识 |

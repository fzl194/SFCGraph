## [操作场景](#ZH-CN_OPI_0166313830)

本操作指导介绍在运行网络中激活Category 6网关选择的操作过程。

Category 6网关选择功能，是指在 UNC 上为特定类型的终端配置IMEI-TAC库，系统通过匹配终端IMEI和IMEI-TAC库，识别终端类型。在使用APN（Access Point Name）进行DNS (Domain Name Service)解析寻址网关前，将为特定类型终端定制的LABEL信息加入到APN-NI和APN-OI之间，然后将扩展的APN发送到DNS服务器解析，为这些终端用户选择高速网关。

> **说明**
> 适用于 SGSN、 MME。

## [任务示例](#ZH-CN_OPI_0166313830)

任务描述

完成基于IMEI选择网关配置，同时添加对应的定制NAPTR/A记录，在用户附着/PDN连接建立/PDP激活流程中需要选择网关时为特定终端选择指定的GGSN/P-GW，即高速网关，以匹配高速用户的业务。

脚本

//开启获取IMEI功能。

```
MOD GBIMEICFG: SUBRANGE=ALL_USER, GETIMEIPLC=IMEI;
MOD IUIMEICFG: SUBRANGE=ALL_USER, GETIMEIPLC=IMEI;
MOD S1IMEICFG: SUBRANGE=ALL_USER, GETIMEIPLC=IMEI;
```

//增加IMEI群组/IMEI群组成员

```
ADD IMEIGP: IMEIGPID=1, IMEIGPN="GwSelection";
ADD IMEIGPMEM: IMEIGPID=1, IMEITAC="35437906";
```

//启用IMEI选择网关策略，配置定制LABEL。

```
ADD GWSELBYIMEI: IMEIGPID=1, LABEL="cat6";
```

//指定匹配的IMSI前缀为1230010000的用户，采用P-GW优选策略，启用CC_IMEI选择网关策略，配置定制LABEL。

```
ADD GWSELPLCY: SUBRANGE=IMSI_PREFIX, IMSIPRE="1230010000", APNNI="HUAWEI1.com", SELPLCY=PGWFIRST, SLCTM=CUSTOMIZE_APN-OI_REPLACEMENT, LABEL=CC_IMEI, IMEIGPID=1, CUSLABEL="cat18";
```

//可选：为具备GUL能力的UE设置统一使用.ORG后缀的APN DNS域名。

```
ADD APNDNS: APNNI="HUAWEI1.COM", UEACCCAP=GERAN/UTRAN/EUTRAN_UE, DNSPOLICY=ORG;
```

//可选：设置DNS查询方式，ORG后缀使用NAPTR查询。

```
ADD DNSQ: SUBRANGE=All, DNSUF="ORG", DNSQUERYMODE=NAPTR;
```

//可选：增加 UNC （MME）用于选择高速的GGSN/P-GW的NAPTR解析记录。

```
ADD DNSN: FQDN="HUAWEI1.COM.cat6.APN.EPC.MNC003.MCC123.3GPPNETWORK.ORG", HSINDEX=1, ENTITY=PGW, INTYPE=S5, PRIORITY=10, WEIGHT=101;
```

//可选：增加 UNC （MME）用于选择高速的GGSN/P-GW的A解析记录。

```
ADD IPV4DNSH: HSINDEX=1, HOSTNAME="TOPON.S5.GGSN-PGW1.Cityb.NODE.EPC.MNC003.MCC123.3GPPNETWORK.ORG", ADDRSECTION=SECTION1, IPV4ADDR1="10.2.3.10", PRIORITY1=10, WEIGHT1=100;
```

//可选：增加 UNC （GnGp SGSN）用于选择高速的GGSN/P-GW的NAPTR解析记录。

```
ADD DNSN: FQDN="HUAWEI1.COM.cat6.APN.EPC.MNC003.MCC123.3GPPNETWORK.ORG", HSINDEX=2, ENTITY=GGSN, INTYPE=Gn, PRIORITY=10, WEIGHT=101;
```

//可选：增加 UNC （GnGp SGSN）用于选择高速的GGSN/P-GW的A解析记录。

```
ADD IPV4DNSH: HSINDEX=2, HOSTNAME="TOPON.GN.GGSN-PGW1.Cityb.NODE.EPC.MNC003.MCC123.3GPPNETWORK.ORG", ADDRSECTION=SECTION1, IPV4ADDR1="10.2.3.10";
```

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2C6GS01", SWITCH=ENABLE;
```

//打开基于计费属性选择网关License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2CCGSN02", SWITCH=ENABLE;
```

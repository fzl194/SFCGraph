## [命令功能](#ZH-CN_MMLREF_0000001168320997)

当前版本配置此命令不生效。

该命令用于查询HAF服务内部功能的参数。

## [注意事项](#ZH-CN_MMLREF_0000001168320997)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001168320997)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001168320997)

无

## [使用实例](#ZH-CN_MMLREF_0000001168320997)

查询域故障功能和多连接增强功能开关。

```
LST HAFFUNC:;
```

## [输出结果说明](#ZH-CN_MMLREF_0000001168320997)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 域故障开关 | 该参数用于指定域故障功能的开关。 |
| 多连接增强开关 | 该参数用于指定是否开启多连接增强功能。 |

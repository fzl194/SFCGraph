## [操作步骤](#ZH-CN_OPI_0193971028)

1. 进入 “MML命令行-UNC” 窗口。
2. 配置用户允许选择高速GGSN。
    a. 设置会话管理扩展功能
      [**SET SMFUNC**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/会话管理/SM扩展功能管理/设置会话管理扩展功能(SET SMFUNC)_26145684.md)
    b. 增加NSA控制策略。
      [**ADD NSACTRLPLCY**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/网络管理/NSA组网管理/NSA控制策略/增加NSA控制策略(ADD NSACTRLPLCY)_72225809.md)
    c. 设置MAP兼容性参数
      [**ADD MAPCMPTBYIMSI**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/MAP应用协议/MAP功能配置/增加MAP协议接口兼容性IMSI号段配置(ADD MAPCMPTBYIMSI)_26305278.md)
    d. 增加GTP-C V0/V1协议兼容性配置
      [**ADD GTPCCMPT**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/GTP-C协议管理/GTP-C V0 V1协议兼容性/增加GTP-C V0_V1协议兼容性配置(ADD GTPCCMPT)_72225601.md)
3. **可选：**配置支持DCNR的高速GGSN的本地Hostfile记录。如果运营商在DNS Server上已配置该对应关系，则不需要配置本步骤。
    a. 增加IPV4 DNS Hostfile的A解析记录。
      [**ADD IPV4DNSH**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/DNS/DNS Hostfile管理/增加IPV4 DNS Hostfile记录(ADD IPV4DNSH)_26145884.md)
    b. 增加DNS DNS Hostfile的NAPTR解析记录。
      [**ADD DNSN**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/GTP-C接口管理/DNS/DNS NAPTR管理/增加DNS NAPTR记录(ADD DNSN)_72225569.md)
4. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)

## [任务示例](#ZH-CN_OPI_0193971028)

任务描述

完成支持DCNR能力的终端用户网关选择配置，同时添加对应的定制NAPTR/A记录，在用户PDP激活流程中需要选择网关时为特定终端选择指定的GGSN，以保证NSA用户从2/3G切换至4G时，始终锚定在高速网关上。

脚本

//设置会话管理扩展功能。

```
SET SMFUNC: GUNSAUPSELPLCY=MSCAP_ARD;
```

//增加NSA控制策略。

```
ADD NSACTRLPLCY: SUBRANGE=IMSI_PREFIX, IMSIPRE="46000", ISDCNR=YES, ISSECRATRPT=YES, ISGUUPSEL=YES;
```

//设置MAP兼容性参数。

```
ADD MAPCMPTBYIMSI: SUBRANGE=IMSI_PREFIX, IMSIPRE="46000", SF=NR_AS_SECONDARY_RAT-1, NODCNR=YES;
```

//增加GTP-C V0/V1协议兼容性配置。

```
ADD GTPCCMPT: IDX=1, MSGCLS=TM, TMMSGTYPE=CREATE_PDP_CONTEXT_REQUEST, CRTPDPREQ=UPSELIND, UPSELFLAG=DCNR-1;
```

//增加 UNC 用于选择高速GGSN的A解析记录。

```
ADD IPV4DNSH: HSINDEX=1, HOSTNAME="topon.gn.gw1.cluster1.net27.example.com", ADDRSECTION=SECTION1, IPV4ADDR1="10.2.3.10", PRIORITY1=10, WEIGHT1=100;
```

//增加 UNC 用于选择高速GGSN的NAPTR解析记录。

```
ADD DNSN: FQDN="HUAWEI1.COM.MNC003.MCC460.GPRS", HSINDEX=1, ENTITY=GGSN, INTYPE=Gn, DCNR=YES, PRIORITY=10, WEIGHT=101;
```

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2UNCUP02", SWITCH=ENABLE;
```

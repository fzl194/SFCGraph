#### [命令功能](#ZH-CN_MMLREF_0000001126306186)

**适用网元：MME**

该命令用于查看MME作为客户端的SBc链路配置数据。

#### [注意事项](#ZH-CN_MMLREF_0000001126306186)

- 该命令执行后立即生效。

#### [本地用户权限](#ZH-CN_MMLREF_0000001126306186)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126306186)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126306186)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| LNKINDEX | 链路索引 | 可选必选说明：可选参数<br>参数含义：该参数用于指定待查询的SBc链路的索引。<br>数据来源：本端规划<br>取值范围：0~127<br>默认值：无<br>配置原则：此链路索引在系统范围内唯一。 |

#### [使用实例](#ZH-CN_MMLREF_0000001126306186)

1. 不输入SBc链路索引，查询已经配置的所有SBc链路数据：
  LST SBCAPLNK:;
  ```
  %%LST SBCAPLNK:;%%
  RETCODE = 0  操作成功。

  查询结果如下

          对端地址1  =  10.10.10.16
          对端地址2  =  255.255.255.255
         对端端口号  =  3868
   SCTP协议参数索引  =  0

           链路名称  =  To-HSS0
            vpn名称  =  
  UNC

  ---  END
  ```

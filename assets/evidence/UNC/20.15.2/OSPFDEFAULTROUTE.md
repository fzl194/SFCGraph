#### [命令功能](#ZH-CN_CONCEPT_0000001549961550)

该命令用于查询OSPF默认宣告路由配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001549961550)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001549961550)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001549961550)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | 进程号 | 可选必选说明：可选参数<br>参数含义：OSPF进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |

#### [使用实例](#ZH-CN_CONCEPT_0000001549961550)

查询OSPF默认宣告路由配置：

```
LST OSPFDEFAULTROUTE:;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001549961550)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 进程号 | 该参数指定用户创建的OSPF进程号。 |
| 拓扑标识 | 该参数指定用户创建的OSPF拓扑标识。 |
| 外部LSA开销值 | 该参数指定自治系统外部LSA的开销值，仅当使能度量为true时生效。 |
| 引入外部路由类型 | 引入外部路由类型，仅当缺省路由使能类型为true时生效。 |
| 使能度量 | 该参数指定是否使能缺省路由度量值。 |
| 缺省路由使能类型 | 该参数指定是否使能缺省路由类型。 |
| 延迟发布聚合路由的时间（s） | 该参数指定延迟发布默认路由的时间。 |
| 默认路由发布标志 | 该参数指定默认路由发布模式。 |
| 计算默认路由 | 该参数指定是否计算默认路由。 |
| 路由策略 | 该参数指定路由策略名称。 |

## [命令功能](#ZH-CN_MMLREF_0234667401)

**适用NF：PGW-C、SMF**

该命令用于查询用户激活和在线恢复场景CHF的选择模式。

## [注意事项](#ZH-CN_MMLREF_0234667401)

无

#### [操作用户权限](#ZH-CN_MMLREF_0234667401)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0234667401)

无

## [使用实例](#ZH-CN_MMLREF_0234667401)

查询当前CHF选择模式：

```
%%LST CHFSELECTMODE:;%%
RETCODE = 0  操作成功
结果如下
--------
首选模式  =  NRF
备选模式  =  Local
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0234667401)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 首选模式 | 该参数用于指定CHF的首选选择模式。 |
| 备选模式 | 该参数用于指定CHF的备选选择模式。 |

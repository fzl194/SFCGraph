## [操作步骤](#ZH-CN_OPI_0000001297000418)

操作场景：EMS下发（推荐场景）

管理面下发跟踪任务，入口界面：监控 > 全网端到端信令跟踪 > 跟踪管理，添加>管理面创建（具体请参考 [图1 登录界面](#ZH-CN_OPI_0000001297000418__fig7212128114813) ）

**图1** 登录界面

<br>

![](管理面下发跟踪任务（5G）_97000418.assets/zh-cn_image_0000001350045361_2.png)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本功能的License配置开关。
  [**SET LICENSESWITCH**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)
3. （可选）配置跟踪消息上报速率控制（建议使用默认值。如果修改，请联系华为技术支持）。
  [**SET FWSOFTPARABITS**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/软参配置管理/设置软件参数比特位（SET FWSOFTPARABITS）_37580503.md)

## [任务示例](#ZH-CN_OPI_0000001297000418)

任务描述

开启端到端用户跟踪特性，在网管上创建端到端跟踪任务。

脚本

//打开端到端用户跟踪的License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2ESTAM01", SWITCH=ENABLE;
SET LICENSESWITCH: LICITEM="LKV3ESTSM02", SWITCH=ENABLE;
SET LICENSESWITCH: LICITEM="LKV2ESTAM03", SWITCH=ENABLE;
```

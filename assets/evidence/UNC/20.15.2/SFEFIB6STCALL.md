#### [命令功能](#ZH-CN_CONCEPT_0000001600865569)

该命令用于查询IPv6所有的路由表项的统计。FIB6用于保存报文转发信息。

#### [注意事项](#ZH-CN_CONCEPT_0000001600865569)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600865569)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600865569)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RUNAME | RU名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定资源单元名称，可使用命令DSP RU查询得到。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。区分大小写。<br>默认值：无<br>配置原则：使用DSP RU查看RU名称。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600865569)

查询IPv6所有的路由表项的统计：

```
DSP SFEFIB6STCALL : RUNAME="VNODE_VNRS_VNFC_IPU_0066";
```

```

RETCODE = 0  操作成功。

结果如下
-------------------------
        路由总数   =  8
    公网路由数量   =  7
  FIB表规格容量   =  1048576
(结果个数 = 1)
---    END
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600865569)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 路由总数 | 用于表示公网和私网路由的FIB表总数。 |
| 公网路由数量 | 用于表示公网路由数量。 |
| FIB表规格容量 | 用于表示FIB6表容量。 |

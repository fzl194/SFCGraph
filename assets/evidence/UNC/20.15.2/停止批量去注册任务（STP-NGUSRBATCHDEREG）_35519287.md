# 停止批量去注册任务（STP NGUSRBATCHDEREG）

- [注意事项](#ZH-CN_MMLREF_0000001135519287__1.3.2)

## [命令功能](#ZH-CN_MMLREF_0000001135519287)

**适用NF：AMF**

该命令用于停止批量去注册任务，命令执行后，对剩余待去注册的用户不再发起网络侧去注册处理。

## [注意事项](#ZH-CN_MMLREF_0000001135519287)

该命令执行后立即生效。

#### [操作用户权限](#ZH-CN_MMLREF_0000001135519287)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001135519287)

无

## [使用实例](#ZH-CN_MMLREF_0000001135519287)

当需要停止批量去注册任务，执行如下命令：

```
STP NGUSRBATCHDEREG:;
```

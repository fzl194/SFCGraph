# 删除层二规则（RMV L2RULE）

## [命令功能](#ZH-CN_MMLREF_0000001870462597)

**适用NF：SMF**

该命令用于删除层二规则。

## [注意事项](#ZH-CN_MMLREF_0000001870462597)

- 该命令执行后只对新接入的会话生效。

- 如果存在引用了层二规则名称的层二规则绑定记录，则不允许删除层二规则。层二规则绑定记录通过LST L2RULEBINDGRP查询。

#### [操作用户权限](#ZH-CN_MMLREF_0000001870462597)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001870462597)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| L2RULENAME | 层二规则名称 | 可选必选说明：必选参数<br>参数含义：该参数用于指定层二规则名称。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围是1~63。不支持空格，不区分大小写。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0000001870462597)

删除层二规则名称为“rule1”的层二规则：

```
RMV L2RULE: L2RULENAME="rule1";
```

## [命令功能](#ZH-CN_MMLREF_0209652433)

**适用NF：AMF**

此命令用于查询系统中全部或者指定条件的5G寻呼规则。

## [注意事项](#ZH-CN_MMLREF_0209652433)

如果命令中的寻呼策略指示没有输入，查询结果中将以无效值255显示。

#### [操作用户权限](#ZH-CN_MMLREF_0209652433)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209652433)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| RULEIDX | 规则索引 | 可选必选说明：可选参数<br>参数含义：该参数用于指定5G寻呼规则的索引。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是1~1000。<br>默认值：无<br>配置原则：无 |

## [使用实例](#ZH-CN_MMLREF_0209652433)

- 查询所有5G寻呼规则，执行如下命令：
  ```
  %%LST NGPAGINGRULE:;%%
  RETCODE = 0  操作成功

  结果如下
  --------
          规则索引  =  6
        用户群类型  =  所有用户
          IMEI TAC  =  NULL
        MSISDN前缀  =  NULL
          IMSI前缀  =  NULL
          业务类型  =  N1N2TRANS消息
           DNN指示  =  所有DNN
      数据网络名称  =  NULL
        5G QoS标识  =  0
       ARP优先级别  =  0
      寻呼策略指示  =  255
        匹配优先级  =  50
          规则描述  =  NULL
      寻呼动作组合  =  最近访问GNB
  (结果个数 = 1)

  ---    END
  ```
- 查询一条“规则索引”为“1”的5G寻呼规则，执行如下命令：
  ```
  %%LST NGPAGINGRULE: RULEIDX=1;%%
  RETCODE = 0  操作成功

  结果如下
  --------
          规则索引  =  1
        用户群类型  =  所有用户
          IMEI TAC  =  NULL
        MSISDN前缀  =  NULL
          IMSI前缀  =  NULL
          业务类型  =  N1N2TRANS消息
           DNN指示  =  所有DNN
      数据网络名称  =  NULL
        5G QoS标识  =  0
       ARP优先级别  =  0
      寻呼策略指示  =  0
        匹配优先级  =  50
          规则描述  =  NULL
      寻呼动作组合  =  最近访问GNB
  (结果个数 = 1)

  ---    END
  ```

## [输出结果说明](#ZH-CN_MMLREF_0209652433)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 规则索引 | 该参数用于指定5G寻呼规则的索引。 |
| 用户群类型 | 该参数用于指定应用5G精准寻呼的用户的标识类型。 |
| IMEI TAC | 该参数用于指定应用5G精准寻呼的用户的IMEI TAC。TAC（Type Allocation Code）是组成IMEI的前8个数字。 |
| MSISDN前缀 | 该参数用于指定应用5G精准寻呼的用户的MSISDN前缀。 |
| IMSI前缀 | 该参数用于指定应用5G精准寻呼的用户的IMSI前缀。 |
| 业务类型 | 该参数用于指定应用5G精准寻呼的业务类型。 |
| DNN指示 | 该参数用于指定应用5G精准寻呼的DNN范围，即所有DNN，或者指定的单个DNN。 |
| 数据网络名称 | 该参数用于指定应用5G精准寻呼的数据网络名称（DNN）。 |
| 5G QoS标识 | 该参数用于指定SMF通过N1N2MsgTransfer（非PCF释放会话）触发寻呼场景下，使用5G精准寻呼功能的5QI匹配值。 |
| ARP优先级别 | 该参数用于指定SMF通过N1N2MsgTransfer（非PCF释放会话）触发寻呼场景下，使用5G精准寻呼功能的ARP优先级匹配值。 |
| 寻呼策略指示 | 该参数用于指定SMF通过N1N2MsgTransfer（非PCF释放会话）触发寻呼场景下，使用5G精准寻呼功能的寻呼策略指示。 |
| 匹配优先级 | 该参数用于指定本寻呼规则的匹配优先级。 |
| 规则描述 | 该参数用于指定寻呼规则的描述信息。 |
| 寻呼动作组合 | 该参数用于指定本寻呼规则使用的寻呼动作组合。 |

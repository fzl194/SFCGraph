#### [命令功能](#ZH-CN_TOPIC_0172291814)

本命令用于查询节点故障自愈开关状态。

> **说明**
> 该命令仅在Full-stack虚机场景下支持。

#### [注意事项](#ZH-CN_TOPIC_0172291814)

无。

#### [参数说明](#ZH-CN_TOPIC_0172291814)

无。

#### [使用实例](#ZH-CN_TOPIC_0172291814)

1. 查询系统节点故障自愈开关：
  ```
  %%LST NODEREPSWITCH:;%%
  RETCODE = 0  操作成功  

         节点故障自愈开关  =  开启
    Error节点故障自愈开关  =  关闭
  Unknown节点故障自愈开关  =  关闭
  (结果个数 = 1)  
  ---    END
  ```

#### [输出结果说明](#ZH-CN_TOPIC_0172291814)

命令执行正常，会返回命令执行成功的提示信息。输出结果说明如 [表1](#ZH-CN_TOPIC_0172291814__table819976173217) 所示。

命令执行异常，会返回提示没有对应的查询结果。错误码列表如 [表2](#ZH-CN_TOPIC_0172291814__table698716490471) 所示。

*表1 输出项说明*

| 输出项名称 | 输出项解释 |
| --- | --- |
| 节点故障自愈开关 | 用于返回当前系统的节点故障自愈开关状态。<br>取值范围：<br>- “开启”：开启节点故障自愈<br>- “关闭”：关闭节点故障自愈 |
| Error节点故障自愈开关 | 用于返回当前系统的Error节点故障自愈开关状态。<br>取值范围：<br>- “开启”：开启Error节点故障自愈<br>- “关闭”：关闭Error节点故障自愈 |
| Unknown节点故障自愈开关 | 用于返回当前系统的Unknown节点故障自愈开关状态。<br>取值范围：<br>- “开启”：开启Unknown节点故障自愈<br>- “关闭”：关闭Unknown节点故障自愈 |

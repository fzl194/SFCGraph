#### [命令功能](#ZH-CN_MMLREF_0000001126146240)

**适用网元：MME**

此命令用于查询当前S1接口均衡操作的结果。

#### [注意事项](#ZH-CN_MMLREF_0000001126146240)

- 此命令执行后立即生效。
- 在迁移过程中执行此命令，可以查询当前S1接口均衡操作的进度。
- 在迁移结束后执行此命令，可以查询执行此命令前最后一次S1接口均衡操作的执行结果。

#### [本地用户权限](#ZH-CN_MMLREF_0000001126146240)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126146240)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126146240)

无。

#### [使用实例](#ZH-CN_MMLREF_0000001126146240)

1. 查询S1接口均衡操作结果：
  DSP S1BALANCE:;
  ```
  %%DSP S1BALANCE:;%%
  RETCODE = 0  操作成功。

  操作结果如下
  -------------------------
              迁移范围  =  系统自动均衡迁移
              迁移状态  =  迁移中 
  需要迁移的eNodeB总数  =  500  
    迁移成功的eNodeB数  =  377
    迁移失败的eNodeB数  =  77
  (结果个数 = 1)

   ---    END
  ```

#### [输出结果说明](#ZH-CN_MMLREF_0000001126146240)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 迁移范围 | eNodeB的迁移范围。<br>取值说明：<br>- “系统自动均衡迁移” |
| 迁移状态 | 当前eNodeB迁移任务状态。<br>取值说明：<br>- “迁移中”<br>- “迁移结束” |
| 需要迁移的eNodeB总数 | 需要迁移的eNodeB总数。 |
| 迁移成功的eNodeB数 | 当前已经完成迁移的eNodeB数。 |
| 迁移失败的eNodeB数 | 迁移失败的eNodeB数。 |

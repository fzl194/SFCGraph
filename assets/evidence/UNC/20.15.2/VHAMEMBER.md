## [命令功能](#ZH-CN_MMLREF_0251852642)

该命令用于查询vHA ETCD集群节点的详细信息。

## [注意事项](#ZH-CN_MMLREF_0251852642)

无

#### [操作用户权限](#ZH-CN_MMLREF_0251852642)

G_1，管理员级别命令组；G_2，操作员级别命令组

## [参数说明](#ZH-CN_MMLREF_0251852642)

无

## [使用实例](#ZH-CN_MMLREF_0251852642)

## [输出结果说明](#ZH-CN_MMLREF_0251852642)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 节点名称 | 该参数用于表示集群节点的名称。 |
| 与Client通讯地址 | 该参数用于表示集群节点告知给客户端的通讯地址。 |
| 与Peer通讯地址 | 该参数用于表示集群节点告知给对端的通讯地址。 |
| 节点成员标识 | 该参数用于表示集群节点的唯一标识。 |
| 角色 | 该参数用于表示集群节点的角色。<br>取值说明：<br>- Leader（主节点）<br>- Follower（从节点）<br>- Unknown（无效） |
| 运行状态 | 该参数用于表示集群节点当前的运行状态。<br>取值说明：<br>- Healthy（健康）<br>- UnHealthy（非健康）<br>- Suspend（挂起） |
| Pod名称 | 该参数用于表示本节点所在的pod名称信息。 |
| Node名称 | 该参数用于表示本节点所在的node信息。 |
| 版本号 | 该参数用于表示本节点的ETCD版本信息。 |
| 存储数据大小 | 该参数用于表示本节点的db数据大小。 |
| 集群节点个数 | 该参数用于表示集群节点个数。 |

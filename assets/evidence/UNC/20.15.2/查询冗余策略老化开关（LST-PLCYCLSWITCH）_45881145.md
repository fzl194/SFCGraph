## [命令功能](#ZH-CN_MMLREF_0000001745881145)

该命令用于查询冗余策略老化开关。

## [注意事项](#ZH-CN_MMLREF_0000001745881145)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001745881145)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001745881145)

无

## [使用实例](#ZH-CN_MMLREF_0000001745881145)

查询冗余策略老化开关，ON表示开启，OFF表示关闭。

```
%%LST PLCYCLSWITCH:;%%
RETCODE = 0  操作成功

结果如下
------------------------             
冗余策略老化开关 = 打开
    老化阈值 = 10
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001745881145)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 冗余策略老化开关 | 该参数用于指示是否开启冗余策略老化。 |
| 老化阈值 | 该参数用于设置老化阈值，单位：次数。 |

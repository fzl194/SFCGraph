## [命令功能](#ZH-CN_MMLREF_0000001264061200)

**适用NF：AMF**

该命令用于查询5G紧急服务功能。

## [注意事项](#ZH-CN_MMLREF_0000001264061200)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001264061200)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001264061200)

无

## [使用实例](#ZH-CN_MMLREF_0000001264061200)

查询5G紧急服务功能，执行如下命令：

```
%%LST NGEMGSRVFUNC:;%%
RETCODE = 0  操作成功

结果如下
--------
本网用户是否允许紧急呼叫业务  =  NR接入的紧急呼叫回落
本网用户紧急号码列表下发开关  =  是
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001264061200)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 本网用户是否允许紧急呼叫业务 | 该参数用于指定是否允许本网用户使用紧急呼叫业务。 |
| 本网用户紧急号码列表下发开关 | 该参数用于控制AMF在给本网用户发送Registration Accept消息时，是否将配置的MCC的紧急呼叫号码列表携带在消息中发送给UE。 |

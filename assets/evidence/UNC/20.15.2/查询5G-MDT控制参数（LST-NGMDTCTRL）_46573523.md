## [命令功能](#ZH-CN_MMLREF_0000001146573523)

**适用NF：AMF**

该命令用于查询5G MDT控制参数。

## [注意事项](#ZH-CN_MMLREF_0000001146573523)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001146573523)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001146573523)

无

## [使用实例](#ZH-CN_MMLREF_0000001146573523)

查询5G MDT控制参数：

```
%%LST NGMDTCTRL:;%%
RETCODE = 0  操作成功

结果如下
--------
基于管理的MDT开关  =  开启
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001146573523)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 基于管理的MDT开关 | 该参数用于指定最小化路测（MDT）功能开关。 |

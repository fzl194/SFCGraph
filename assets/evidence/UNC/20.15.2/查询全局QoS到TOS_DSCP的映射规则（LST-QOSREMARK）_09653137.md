## [命令功能](#ZH-CN_MMLREF_0209653137)

**适用NF：SGW-C、PGW-C、SMF、GGSN**

该命令用于查询系统上下行QoS映射功能配置。

## [注意事项](#ZH-CN_MMLREF_0209653137)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209653137)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209653137)

无

## [使用实例](#ZH-CN_MMLREF_0209653137)

查询系统上下行QoS映射功能配置：

```
%%LST QOSREMARK:;%%
RETCODE = 0  操作成功

操作结果如下
------------
下行标记  =  使能
上行标记  =  使能
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209653137)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 下行标记 | 该参数用于设置下行标记的开关功能。 |
| 上行标记 | 该参数用于设置上行标记的开关功能。 |

## [任务示例](#ZH-CN_OPI_0000001193337664)

任务描述

某行业用户签约了5G LAN业务，该用户涉及多个分支机构，地理范围跨越了两个UPF（UPF_Instance0和UPF_Instance1）的范围，两个分支机构的终端基于两个UPF的N6接口实现数据转发互通。

脚本

//配置开启5G LAN功能开关。

```
SET SMFFUNC: NGLANSWITCH=Support, VNGROUPDATASW=Support;
```

//配置对5G LAN UPF的选择机制。

```
SET UPSELECTFLAG: NGLANUPFSELSW=LOCALABILITY;
```

//配置对端UPF支持5G LAN。

```
MOD UPNODE: NFINSTANCENAME="UPF_Instance0", NGLANSW=ENABLE;
MOD UPNODE: NFINSTANCENAME="UPF_Instance1", NGLANSW=ENABLE;
```

//配置5G LAN关键信元。

```
SET PFCPPVTEXT: FEATURE=NGLAN,SWITCH=ENABLE;
```

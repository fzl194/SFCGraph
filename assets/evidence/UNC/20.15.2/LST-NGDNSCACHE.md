## [命令功能](#ZH-CN_MMLREF_0000001255725191)

**适用NF：SGW-C、AMF**

该命令用于查询DNS缓存参数。

## [注意事项](#ZH-CN_MMLREF_0000001255725191)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001255725191)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001255725191)

无

## [使用实例](#ZH-CN_MMLREF_0000001255725191)

使用以下命令查询DNS缓存参数：

```
%%LST NGDNSCACHE:;%%
RETCODE = 0  操作成功

结果如下
--------
              缓存使能  =  TRUE
          最大FQDN数量  =  4
最大主机名控制节点数量  =  8
        最大主机名数量  =  10
 写缓存最大速率(次/秒)  =  6
   FQDN扫描速率(个/秒)  =  5
          老化时长(秒)  =  3
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001255725191)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 缓存使能 | 该参数用于控制是否开启缓存功能。 |
| 最大FQDN数量 | 该参数用于表示单进程存储的最大FQDN数量。 |
| 最大主机名控制节点数量 | 该参数用于表示单进程存储的最大主机名控制节点数量。同一个主机名如果支持N个接口，则会生成N个控制节点。 |
| 最大主机名数量 | 该参数用于表示单进程存储的最大主机名数量。 |
| 写缓存最大速率(次/秒) | 该参数用于表示单进程每秒写缓存的最大次数。 |
| FQDN扫描速率(个/秒) | 该参数用于表示每秒扫描FQDN的数量，扫描的目的是进行缓存记录的老化判断与执行。 |
| 老化时长(秒) | 该参数用于控制缓存的提前老化功能。提前老化功能生效后，系统以本参数的配置值提前老化缓存记录，否则以DNS服务器返回的TTL值进行老化。 |

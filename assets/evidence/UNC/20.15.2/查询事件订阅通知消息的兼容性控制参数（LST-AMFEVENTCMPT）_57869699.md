## [命令功能](#ZH-CN_MMLREF_0000001157869699)

**适用NF：AMF**

该命令用于查询AMF事件订阅的兼容性控制参数。

## [注意事项](#ZH-CN_MMLREF_0000001157869699)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001157869699)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001157869699)

无

## [使用实例](#ZH-CN_MMLREF_0000001157869699)

查询AMF事件订阅的兼容性参数，执行如下命令。

```
%%LST AMFEVENTCMPT:;%%
RETCODE = 0  操作成功

结果如下
------------------------
UE当前驻留的基站全局标识  =  No
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001157869699)

| 输出项名称 | 输出项解释 |
| --- | --- |
| UE当前驻留的基站全局标识 | 该参数用于控制上报事件订阅类型为LOCATION_REPORT、PRESENCE_IN_AOI_REPORT的通知消息时，是否携带UE当前驻留的基站全局标识。 |

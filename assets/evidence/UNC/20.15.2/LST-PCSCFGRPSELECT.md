## [命令功能](#ZH-CN_MMLREF_0209653049)

**适用NF：PGW-C、SMF、GGSN**

该命令用于查询配置的P-CSCF-GROUP的选择模式。

## [注意事项](#ZH-CN_MMLREF_0209653049)

无

#### [操作用户权限](#ZH-CN_MMLREF_0209653049)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209653049)

无

## [使用实例](#ZH-CN_MMLREF_0209653049)

查询P-CSCF-GROUP的选择模式：

```
%%LST PCSCFGRPSELECT:;%%
RETCODE = 0  操作成功

结果如下
----------------
选择模式  =  主备
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209653049)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 选择模式 | 该参数用于配置apn或全局绑定的两个p-cscf-group的选择模式。 |

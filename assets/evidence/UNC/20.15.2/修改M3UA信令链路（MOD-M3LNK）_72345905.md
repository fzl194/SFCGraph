#### [命令功能](#ZH-CN_MMLREF_0000001172345905)

**适用网元：SGSN、MME、SMSF**

该命令用于修改M3UA信令链路配置数据。

#### [注意事项](#ZH-CN_MMLREF_0000001172345905)

- 该命令执行后立即生效。
- 拥塞开始门限必须大于拥塞结束门限。
- 该命令在版本升级过程中禁止执行。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172345905)

manage-ug；system-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172345905)

G_1，管理员级别命令组；G_2，操作员级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172345905)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| LNK | 链路号 | 可选必选说明：必选参数<br>参数含义：该参数用于指定M3UA链路号。该参数用来在系统范围内部唯一标识一条M3UA链路。<br>数据来源：本端规划<br>取值范围：0~1279<br>默认值：无 |
| PRI | 优先级 | 可选必选说明：可选参数<br>参数含义：该参数用于指定M3UA链路的优先级。<br>数据来源：整网规划<br>取值范围：0~254<br>默认值：无<br>说明：0为最高优先级，254为最低优先级。 |
| CONSTHD | 拥塞开始门限 | 可选必选说明：可选参数<br>参数含义：保留参数，不建议自行配置。 |
| CONETHD | 拥塞结束门限 | 可选必选说明：可选参数<br>参数含义：保留参数，不建议自行配置。 |
| SCTPINDX | SCTP 参数索引 | 可选必选说明：可选参数<br>参数含义：该参数用于指定SCTPPARA表的索引号，指定本M3链路使用的SCTP协议栈参数。<br>前提条件：SCTPPARA表中要有对应的SCTP参数索引号，否则输入的索引号无效，无法正确修改记录。<br>数据来源：整网规划<br>取值范围：0~65534<br>默认值：无<br>说明：需要在<br>[**LST SCTPPARA**](../../SCTP管理/查询SCTP协议参数(LST SCTPPARA)_72226019.md)<br>时可以查询到相应的链路索引。 |
| EXCHGMODE | 交换模式 | 可选必选说明：可选参数<br>参数含义：该参数用于设置在链路集工作模式为“IPSP”时，SGSN网元和对端网元的交换模式。<br>数据来源：对端协商<br>取值范围：<br>- “SINGLE（单交换模式）”<br>- “DOUBLE（双交换模式）”<br>默认值：无<br>说明：- 当链路集的工作模式为“ASP”或者“SGP”，本参数将不生效。<br>- 本参数设置值需要和对端网元的交换模式保持一致。 |
| LKN | 链路名 | 可选必选说明：可选参数<br>参数含义：该参数用于指定链路名称，标识M3UA链路。<br>数据来源：本端规划<br>取值范围:1~32位字符串<br>默认值：无 |

#### [使用实例](#ZH-CN_MMLREF_0000001172345905)

将链路号为0的M3UA链路的优先级改为1，链路名改为“R6_M3LNK1”：

MOD M3LNK: LNK=0, PRI=1, LKN="R6_M3LNK1";

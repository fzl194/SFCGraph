#### [命令功能](#ZH-CN_CONCEPT_0209896805)

**适用NF：PGW-C、SMF**

LST CHGINITPRI命令用来显示用户激活过程中RADIUS计费消息和在线计费消息交互优先级。

#### [注意事项](#ZH-CN_CONCEPT_0209896805)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0209896805)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0209896805)

无。

#### [使用实例](#ZH-CN_CONCEPT_0209896805)

查询RADIUS计费和在线计费消息优先级：

```
LST CHGINITPRI:;
```

```

RETCODE = 0  操作成功。

激活阶段计费模式优先
--------------------
RADIUS计费消息交互的优先级  =  1
  在线计费消息交互的优先级  =  1
(结果个数 = 1)
---    END
```

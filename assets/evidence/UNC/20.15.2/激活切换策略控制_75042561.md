## [操作场景](#ZH-CN_OPI_0275042561)

本操作指导介绍在运行网络中激活切换策略控制特性的操作过程。

切换策略控制就是根据用户属性确定用户在GSM和UMTS网络中的切换策略，来引导GSM、UMTS和LTE的网络业务承载和网络负荷。

> **说明**
> 适用于 SGSN 。

## [操作步骤](#ZH-CN_OPI_0275042561)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  **[SET LICENSESWITCH](../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md)**
3. 启用业务切换策略功能。
  [**SET COMPATIBILITY**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/QoS管理/QoS兼容性管理/QoS兼容性配置/设置QoS兼容性配置(SET COMPATIBILITY)_72345835.md)
  > **说明**
  > 此命令用于启用2G、3G业务切换策略功能。 “SRVHANDOVER2G” 取值为 “FIRST_PDP_ONLY” 表示2G只支持第一个PDP业务切换， “SRVHANDOVER3G” 取值为 “MULT_PDP” 表示3G支持多PDP切换， “GTL” 取值为 “YES” 表示支持GSM网络到LTE网络的切换策略下发， “WTL” 取值为 “YES” 表示UMTS网络到LTE网络的切换策略下发。
4. 打开E-UTRAN Service Handover信元和Service UTRAN CCO信元中的Service E-UTRAN CCO Value的下发控制开关。
  [**SET SOFTPARA**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/操作维护/软件参数管理/软件参数/设置软件参数表(SET SOFTPARA)_26146182.md)
  > **说明**
  > “PARANUM”为60的BIT1-BIT4是E-UTRAN Service Handover信元和Service UTRAN CCO信元中的Service E-UTRAN CCO Value的下发控制开关。BYTE_EX60 BIT1-BIT4配置为0表示 UNC 始终不下发；配置为其他非0值表示 UNC 可以下发。
5. 设置RNC的协议版本为R8及以上版本。
  [**MOD RNC**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/Iu接口管理/Iu接口RNC信息/修改Iu接口RNC信息(MOD RNC)_26305850.md)
  > **说明**
  > 为了避免RNC的协议版本较低时下发E-UTRAN Service Handover信元可能导致的RELOCATION流程失败，还需要设置RNC的协议版本“RNCVER”为R8及以上版本， UNC 才会下发E-UTRAN Service Handover信元。
6. 配置业务切换策略。
  [**ADD SRVHANDOVER**](../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/会话管理/业务切换策略/增加业务切换策略(ADD SRVHANDOVER)_26305476.md)

## [任务示例](#ZH-CN_OPI_0275042561)

任务描述

启动3G、2G业务切换并且多PDP支持业务切换。

设置 UNC 支持GSM/UMTS网络到LTE网络的切换策略下发。

增加业务级别为“Background”，用户级别为“高端用户”，业务切换策略为建议GPRS切换到 WCDMA 。

增加业务级别为“Background”，用户级别为“低端用户”，业务切换策略为建议WCDMA切换到GPRS。

脚本

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2SRVHD02", SWITCH=ENABLE;
```

//启动3G、2G业务切换并且多PDP支持业务切换，设置 UNC 支持GSM/UMTS网络到LTE网络的切换策略下发。

```
SET COMPATIBILITY: SRVHANDOVER2G=MULT_PDP, SRVHANDOVER3G=MULT_PDP, GTL=YES, WTL=YES;
```

//打开Service UTRAN CCO信元中的Service E-UTRAN CCO Value和E-UTRAN Service Handover信元的下发控制开关。

```
SET SOFTPARA: DT=BYTE_EX, PARANUM=60, VALUE="5";
```

//设置RNC的协议版本。

```
MOD RNC: RNCX=1, RNCVER=R8;
```

//配置业务级别为“Background”，用户级别为“高端用户”，WCDMA业务切换策略为“GPRS(建议切换到GPRS)”，GPRS业务切换策略为建议切换到WCDMA。

```
ADD SRVHANDOVER:SRVLVL=BACKGROUND, USRPRI=HIGHLEVELUSER, WSHS=GPRS, GSHS=WCDMA; 
```

//配置业务级别为“Background”，用户级别为“低端用户”，WCDMA业务切换策略为建议切换到GPRS，GPRS业务切换策略为“GPRS(不建议切换到WCDMA)”。

```
ADD SRVHANDOVER: SRVLVL=BACKGROUND,USRPRI=LOWLEVELUSER,WSHS=GPRS, GSHS=GPRS;
```

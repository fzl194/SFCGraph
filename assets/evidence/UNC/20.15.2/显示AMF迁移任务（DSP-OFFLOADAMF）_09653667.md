## [命令功能](#ZH-CN_MMLREF_0209653667)

**适用NF：AMF**

该命令用于查询AMF用户迁移进度信息。

## [注意事项](#ZH-CN_MMLREF_0209653667)

- 如果当前系统正在执行迁移任务，则显示当前迁移进度信息，否则将显示最近一次的迁移记录信息。
- 此命令对类型为“IMSI(IMSI)”的迁移任务不生效。针对类型为“IMSI(IMSI)”的迁移任务，可通过用户跟踪等手段查看迁移进度。

#### [操作用户权限](#ZH-CN_MMLREF_0209653667)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0209653667)

无

## [使用实例](#ZH-CN_MMLREF_0209653667)

查询AMF用户迁移任务进度信息，执行如下命令：

```
%%DSP OFFLOADAMF:;%%
RETCODE = 0  操作成功

结果如下
------------------------
               迁移类型  =  全部用户
               迁移状态  =  迁移中
                目标AMF  =  NULL
         目标AMF IP信息  =  NULL
           迁移启动时间  =  2019-12-23 12:09:54 +0800
           待迁移用户数  =  1
           已迁移用户数  =  0
     迁移任务时长(分钟)  =  0
         剩余时间(分钟)  =  1
设定的迁移速率(千/分钟)  =  1
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0209653667)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 迁移类型 | 该参数用来显示所启动任务的迁移类型。<br>取值说明：<br>- “ALL（全部用户）”：全部用户<br>- “PART（部分用户）”：部分用户<br>- “RATE（百分比用户）”：百分比用户<br>- “IMSI（指定IMSI）”：指定IMSI |
| 迁移状态 | 该参数用来显示当前任务的状态。<br>取值说明：<br>- “OFFLOADING（迁移中）”：迁移任务正在运行。<br>- “OFFLOADFINISHED（已结束）”：迁移任务已经结束。<br>- “OFFLOADSTOPED（已停止）”：迁移任务已经通过STP OFFLOADAMF命令停止。 |
| 目标AMF | 该参数用于显示当前任务设定的目标AMF名称。 |
| 目标AMF IP信息 | 该参数用于指定用户需要迁移到的目标AMF的ApiRoot的IP信息。 |
| 迁移启动时间 | 该参数用于显示任务启动时间。 |
| 待迁移用户数 | 该参数用于显示当前任务的计划迁移用户数。 |
| 已迁移用户数 | 该参数用于显示已经迁移的用户数。<br>1）如果“迁移状态”为“迁移中”，表示系统当前已经迁移的用户数。<br>2）如果“迁移状态”为“已结束”，表示上一次迁移任务中实际迁移的用户数。待迁移AMF上的Registered用户数变化，除了已迁移用户的影响外，还有可能因为UE正常的位置移动、注册或去注册等业务流程导致的用户数变化，因此待迁移用户数和已迁移用户数不一定完全相同。<br>3）如果“迁移状态”为“已停止”，表示上一次迁移任务中实际迁移的用户数。由于迁移任务是通过用户手工执行STP OFFLOADAMF命令强制停止，因此待迁移用户数和已迁移用户数会有所不同。 |
| 迁移任务时长(分钟) | 该参数用于显示迁移任务的执行时长。当前迁移任务状态为“迁移中”时，则显示当前迁移任务已经执行时长。 |
| 剩余时间(分钟) | 该参数用于显示当前迁移任务剩余时间的估计值。当前迁移任务状态为非“迁移中”时，则显示为0。 |
| 设定的迁移速率(千/分钟) | 该参数用于显示当前迁移任务设定的迁移速率。 |

## [命令功能](#ZH-CN_MMLREF_0296805382)

**适用NF：AMF**

该命令用于查询AMF的SMSF故障处理策略。

## [注意事项](#ZH-CN_MMLREF_0296805382)

无

#### [操作用户权限](#ZH-CN_MMLREF_0296805382)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0296805382)

无

## [使用实例](#ZH-CN_MMLREF_0296805382)

查询AMF的SMSF故障处理策略，执行如下命令：

```
%%LST AMFSMSFRESET:;%%
RETCODE = 0  操作成功

结果如下
--------
SMSF故障重选开关  =  打开
 扫描速率(个/秒)  =  5
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0296805382)

| 输出项名称 | 输出项解释 |
| --- | --- |
| SMSF故障重选开关 | 该参数用于指定AMF和SMSF的接口故障或者SMSF的状态更新为去注册时，是否删除用户上下文中SMSF的信息，以便后续有SMS相关业务时重选SMSF。 |
| 扫描速率(个/秒) | 该参数用于指定每个DS每秒扫描多少个用户，扫描到后对符合重选SMSF的用户向正常的SMSF进行注册。 |

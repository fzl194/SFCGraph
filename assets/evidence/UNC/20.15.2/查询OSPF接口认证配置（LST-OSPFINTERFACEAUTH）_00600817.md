#### [命令功能](#ZH-CN_CONCEPT_0000001600600817)

该命令用于查询OSPF接口认证配置。

#### [注意事项](#ZH-CN_CONCEPT_0000001600600817)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600600817)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600600817)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| PROCID | OSPF进程号 | 可选必选说明：可选参数<br>参数含义：OSPF进程号。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围为1～4294967295。<br>默认值：无 |
| AREAID | 区域号 | 可选必选说明：可选参数<br>参数含义：区域号。<br>数据来源：全网规划<br>取值范围：IPv4地址类型。点分十进制格式。<br>默认值：无 |
| IFNAME | 接口名 | 可选必选说明：可选参数<br>参数含义：接口名。<br>数据来源：本端规划<br>取值范围：字符串类型，输入长度范围为1～63。<br>默认值：无<br>配置原则：请使用LST INTERFACE命令查看可用接口。 |

#### [使用实例](#ZH-CN_CONCEPT_0000001600600817)

查询OSPF接口认证配置：

```
LST OSPFINTERFACEAUTH:;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001600600817)

| 输出项名称 | 输出项解释 |
| --- | --- |
| OSPF进程号 | 该参数用于表示OSPF进程号。 |
| 区域号 | 该参数用于表示OSPF区域号。 |
| 接口名 | 该参数用于表示OSPF接口名称。 |
| 认证模式 | 该参数用于表示OSPF接口的认证模式。OSPF支持报文验证功能，只有通过验证的报文才能接收，否则将不能正常建立邻居关系。 |
| 密文口令类型 | 该参数用于表示密文口令类型，当认证模式配置为HMAC-SHA256、Simple、MD5或HMAC-MD5时生效。 |
| 简单认证密码 | 该参数用于表示简单认证密码，当认证模式配置为Simple时生效。 |
| MD5/HMAC-MD5/HMAC-SHA256 密码 | 该参数用于表示MD5/HMAC-MD5/HMAC-SHA256密码，当认证模式配置为HMAC-SHA256、MD5或HMAC-MD5时生效。 |
| 密文验证字标识符 | 该参数用于表示密文验证字标识符，当认证模式配置为HMAC-SHA256、MD5或HMAC-MD5时生效。 |
| KeyChain名称 | 该参数用于表示KeyChain名称，当认证模式配置为KeyChain时生效。 |

#### [命令功能](#ZH-CN_CONCEPT_0000001150612759)

**适用NF：SGSN、MME、AMF**

该命令用于查询系统中SCTP缓冲区的模式。

#### [注意事项](#ZH-CN_CONCEPT_0000001150612759)

无。

#### [本地用户权限](#ZH-CN_CONCEPT_0000001150612759)

manage-ug;system-ug;monitor-ug

#### [网管用户权限](#ZH-CN_CONCEPT_0000001150612759)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001150612759)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001150612759)

查询当前系统中SCTP缓冲区模式：

```
LST SCTPBUFFERMODE:;
```

```
查询结果如下
-------------------------
 接收端缓冲区模式  =  私有模式
 发送端缓冲区模式  =  私有模式
(结果个数 = 1)

---    END
```

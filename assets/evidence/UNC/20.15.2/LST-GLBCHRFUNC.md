## [命令功能](#ZH-CN_MMLREF_0000001089350516)

**适用NF：SGW-C、PGW-C、GGSN、SMF**

该命令用于查询全局CHR功能配置。

## [注意事项](#ZH-CN_MMLREF_0000001089350516)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001089350516)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001089350516)

无

## [使用实例](#ZH-CN_MMLREF_0000001089350516)

查询全局CHR上报功能配置：

```
%%LST GLBCHRFUNC:;%%
RETCODE = 0  操作成功

结果如下
--------
                 CHR上报开关  =  使能
             支持上报CHR的会话数 = 0
        支持上报小范围CHR的会话数 = 3
支持上报小范围CHR的用户会话老化时长 = 24
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001089350516)

| 输出项名称 | 输出项解释 |
| --- | --- |
| CHR上报开关 | 该参数用于全局控制开启和关闭CHR上报功能。 |
| 支持上报CHR的会话数 | 该参数用于指定支持上报CHR单据的会话数量。 |
| 支持上报小范围CHR的会话数 | 该参数用于指定支持上报小范围CHR单据的会话数量。 |
| 支持上报小范围CHR的用户会话老化时长 | 该参数用于“指定支持上报小范围CHR的会话数”大于0时，选中的上报小范围CHR的用户会话的老化时长。<br>超时后会重新选择上报小范围CHR的用户。 |

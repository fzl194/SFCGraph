# 调测基于位置信息的分流策略控制（SMF）

- [必备事项](#ZH-CN_OPI_0000001107996028__1.3.2)

## [操作场景](#ZH-CN_OPI_0000001107996028)

本操作指导介绍在运行网络中调测本地分流策略控制特性的操作过程。

## [必备事项](#ZH-CN_OPI_0000001107996028)

前提条件

- 请仔细阅读[WSFD-223001 基于位置信息的分流策略控制](WSFD-223001 基于位置信息的分流策略控制特性概述_87840550.md)特性原理概述。
- 完成[部署基于位置信息的分流策略控制（SMF）](部署基于位置信息的分流策略控制（SMF）_53995959.md)。

数据

| 类别 | 参数名称 | 取值样例 | 说明 |
| --- | --- | --- | --- |
| 用户信息 | 国际移动用户识别码（IMSI） | 123000123456789 | - |
| 用户DNN信息 | DNN名称（DNN） | Huawei.com | MEC区域内的DNN |
| 用户位置信息 | 用户位置信息TAI（用户位置信息TAI） | 123010006 | MEC覆盖区域对应的TA |
| ULCL UPF IP地址 | IP地址 | 192.168.126.11 | 分流UPF的IP地址 |

## [操作步骤](#ZH-CN_OPI_0000001107996028)

1. 用户进入MEC覆盖区域，发起PDU会话建立流程，创建PDU会话标识为5的PDU会话。
2. 进入 “MML命令行-UNC” 窗口。
3. 查询用户的PDU会话信息。
  **[DSP PDUSESSION](../../../../../OM参考/命令/UNC MML命令/业务服务管理/会话管理/接入管理/接入管理运维/查询PDP上下文信息/显示PDU会话（DSP PDUSESSION）_09653215.md)**
  预期结果：UE的会话创建流程中，选择到IP地址为192.168.126.11的ULCL UPF插入了用户会话。

  ```
  DSP PDUSESSION:QUERYTYPE=IMSI, IMSI="123000123456789", DSPINFOTYPE=DETAILED, WLNETWKTYPE=NW5G, PDUSESSIONID=5, QFI=1;
  RETCODE = 0  操作成功  

  结果如下 
  --------------  
              国际移动用户标识 = 123000123456789
                              ...
               用户位置信息TAI = 123010006
                              ...
             UP的PFCP IPv4地址 = 192.168.126.11
                              ...
                      ULCL标识 = True
                              ...
  (结果个数 = 1) 
  ---    END
  ```

#### [命令功能](#ZH-CN_MMLREF_0000001126145426)

**适用网元：MME**

用户通过 [**STR VLROFFLOADBYLAI**](启动IMSI分离4G用户任务(STR VLROFFLOADBYLAI)_26305240.md) 启动对4G用户的IMSI分离任务后，执行本命令可以查询该任务的运行状态等信息。

#### [注意事项](#ZH-CN_MMLREF_0000001126145426)

此命令执行后立即生效。

#### [本地用户权限](#ZH-CN_MMLREF_0000001126145426)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001126145426)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001126145426)

无。

#### [使用实例](#ZH-CN_MMLREF_0000001126145426)

查询 [**STR VLROFFLOADBYLAI**](启动IMSI分离4G用户任务(STR VLROFFLOADBYLAI)_26305240.md) 启动的IMSI分离4G用户扫描任务的运行状态和进度：

DSP VLROFFLOADBYLAI:;

```
%%DSP VLROFFLOADBYLAI:;%%
RETCODE = 0  操作成功。

操作结果如下：
--------------
               任务的运行状态  =  正在运行
            任务的处理进度(%)  =  60
                 已处理用户数  =  16000
已发起IMSI Detach请求的用户数  =  8000
(结果个数 = 1)

---    END
```

#### [输出结果说明](#ZH-CN_MMLREF_0000001126145426)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 任务的运行状态 | 显示任务的运行状态。<br>取值说明：<br>- 停止：任务未启动或者任务正常结束。<br>- 正在运行：任务正在运行。 |
| 任务的处理进度（%） | 显示任务已处理的用户数占所有需要处理的用户数的百分比。<br>说明：- 本命令连续两次查询所显示的任务处理进度，可能会因两次查询期间系统内大量用户数突增，导致第二次查询的进度比第一次查询的进度小。<br>- “所有需要处理的用户数”与使用[**DSP USRPDPNUM**](../../../系统管理/用户数据库管理/显示用户和承载上下文数(DSP USRPDPNUM)_72345955.md)命令查询指定主用SPP进程得到的输出结果中“静态用户数”的值保持一致。 |
| 已处理用户数 | 显示任务已经处理过的用户数。<br>说明：- 该值是执行[**STR VLROFFLOADBYLAI**](启动IMSI分离4G用户任务(STR VLROFFLOADBYLAI)_26305240.md)命令后，系统内当前所有主用SPP进程已完成扫描的用户数之和。 |
| 已发起IMSI Detach请求的用户数 | 显示任务已经处理且MME已发起IMSI Detach请求的用户数。 |

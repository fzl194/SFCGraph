## [操作步骤](#ZH-CN_OPI_0163510668)

1. 进入 “MML命令行-UNC” 窗口。
2. 打开本特性的License配置开关。
  [**SET LICENSESWITCH**](../../../../../../OM参考/命令/UNC MML命令/平台服务管理/操作维护/License管理/设置License项的开关（SET LICENSESWITCH）_09654190.md) :
3. 设置4G网络设备标识获取和检查策略。
  [**MOD S1IMEICFG**](../../../../../../OM参考/命令/UNC MML命令/业务服务管理/Pre 5G接入业务管理/控制面管理/业务安全管理/设备检查管理/S1模式IMEI配置/修改S1模式IMEI配置(MOD S1IMEICFG)_26145638.md)

## [任务示例](#ZH-CN_OPI_0163510668)

任务描述

在4G网络中启用设备标识检查功能，配置IMEI获取策略为优先获取IMEISV，实现对所有用户使用S13接口进行IMEI检查。IMEI获取失败允许接入，IMEI检查超时允许接入，允许灰名单用户接入，不允许黑名单用户接入，允许EIR返回用户原因失败时接入，允许EIR返回设备原因失败时接入，EIR响应超时时长为5。

脚本

//打开License配置开关。

```
SET LICENSESWITCH: LICITEM="LKV2EIR02", SWITCH=ENABLE;
```

//设置4G网络获取和检查IMEI的策略。

```
MOD S1IMEICFG: SUBRANGE=ALL_USER, GETIMEIPLC=IMEISV, CHKIMEIPLC=IMEISV, ITTYPE=S13, IMEIACCESS=YES, CTFLAG=YES, GRALLOW=YES, BLALLOW=NO, USERERR=YES, PROVIDERR=YES, EIROUTTIMER=5;
```

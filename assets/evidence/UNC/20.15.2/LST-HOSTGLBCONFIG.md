#### [命令功能](#ZH-CN_CONCEPT_0000001600441209)

该命令用于查询主机收发忽略管理平面接口上送的TCP或UDP协议报文的校验和。

#### [注意事项](#ZH-CN_CONCEPT_0000001600441209)

- 该命令执行后立即生效。
- 当前版本不支持此命令。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001600441209)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001600441209)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001600441209)

查询主机收发忽略校验和：

```
LST HOSTGLBCONFIG:;
```

```

        RETCODE = 0  操作成功

        结果如下
        -------------------------
          忽略校验和  = TRUE
        (结果个数 = 1)
        ---   END
```

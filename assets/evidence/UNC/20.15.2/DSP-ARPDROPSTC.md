#### [命令功能](#ZH-CN_CONCEPT_0000001550281274)

该命令用于查询ARP丢弃报文统计。

#### [注意事项](#ZH-CN_CONCEPT_0000001550281274)

无。

#### [操作用户权限](#ZH-CN_CONCEPT_0000001550281274)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_CONCEPT_0000001550281274)

无。

#### [使用实例](#ZH-CN_CONCEPT_0000001550281274)

查看ARP丢弃报文统计：

```
DSP ARPDROPSTC:;
```

#### [输出结果说明](#ZH-CN_CONCEPT_0000001550281274)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 源IP地址 | 用于显示报文的源IP地址。 |
| 目的IP地址 | 用于显示报文的目的IP地址。 |
| 源MAC地址 | 用于显示报文的源MAC地址。 |
| 目的MAC地址 | 用于显示报文的目的MAC地址。 |
| 接口名称 | 用于显示接口名称。 |
| 丢弃类型 | 用于显示丢弃类型。 |
| 报文丢弃计数 | 用于显示报文丢弃计数。 |
| 最后丢弃报文的时间 | 用于显示最后丢弃报文的时间。 |

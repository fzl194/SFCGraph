## [命令功能](#ZH-CN_MMLREF_0000001823622958)

**适用NF：NRF**

该命令用于查询漫入场景SUPI白名单。

## [注意事项](#ZH-CN_MMLREF_0000001823622958)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001823622958)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001823622958)

无

## [使用实例](#ZH-CN_MMLREF_0000001823622958)

查询所有漫入场景SUPI白名单信息：

```
%%LST ROAMINSUPIWL:;%%
RETCODE = 0  操作成功

结果如下
--------
号段起始字符串  =  123456789876501
号段结束字符串  =  123456789876505
(结果个数 = 1)

---    END
```

## [输出结果说明](#ZH-CN_MMLREF_0000001823622958)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 号段起始字符串 | 该参数用于表示号段的起始号码。 |
| 号段结束字符串 | 该参数用于表示号段的结束号码。 |

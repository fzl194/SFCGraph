## [命令功能](#ZH-CN_MMLREF_0000001652280610)

该命令用于显示特定局向的HTR信息，显示信息包含HTR流控阈值和流控状态等HTR流控信息。

## [注意事项](#ZH-CN_MMLREF_0000001652280610)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001652280610)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001652280610)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| OFCIDX | 局向索引 | 可选必选说明：必选参数<br>参数含义：该参数用于显示局向索引。<br>数据来源：本端规划<br>取值范围：整数类型，取值范围是0~4095。<br>默认值：无<br>配置原则：无 |

## [输出结果说明](#ZH-CN_MMLREF_0000001652280610)

| 输出项名称 | 输出项解释 |
| --- | --- |
| POD名称 | 该参数用于显示进程所在的POD名称。 |
| 进程标识 | 该参数用于显示进程ID。 |
| 状态 | 该参数用于显示HTR状态。<br>取值说明：<br>- “Initial（初始）”：初始状态<br>- “Normal（正常）”：正常状态<br>- “PreNormal（预正常）”：预正常状态<br>- “PreOverload（预过载）”：预过载状态<br>- “Overload（过载）”：过载状态 |
| 初始放通阈值 | 该参数用于显示该局向进入HTR状态第一个周期的放通阈值。单位：个/秒，Normal状态显示0。 |
| 最小安全边界 | 该参数用于显示该局向最小安全边界，单位：个/秒，Normal状态显示0。 |
| 最大安全边界 | 该参数用于显示该局向最大安全边界，单位：个/秒，Normal状态显示0。 |
| 发送请求总数 | 该参数用于显示该局向累计发送请求总数。 |
| 接收对端过载响应总数 | 该参数用于显示该局向累计接收对端过载响应总数。 |
| 等待对端响应超时总数 | 该参数用于显示该局向累计等待对端响应超时总数。 |
| 请求流控总数 | 该参数用于显示该局向累计被HTR流控的请求消息总数。 |
| 周期内发送请求数 | 该参数用于显示该局向前N个周期发送请求数，格式为"XXX/XXX/XXX/ ..."。(其中N最大值为5，取决于局向当前所处的HTR流控状态，初始状态显示为空"NULL")。 |
| 周期内失败率 | 该参数用于显示该局向前N个周期失败率，格式为"XXX%/XXX%/XXX%/XXX%/..."。(其中N最大值为5，取决于局向当前所处的HTR流控状态，初始状态显示为空"NULL")。 |
| 周期内接收对端过载响应数 | 该参数用于显示该局向前N个周期接收到对端回复过载码失败响应数，格式为"XXX/XXX/XXX/XXX/ ..."。(其中N最大值为5，取决于局向当前所处的HTR流控状态，初始状态显示为空"NULL")。 |
| 周期内等待对端响应超时数 | 该参数用于显示该局向前N个周期等待对端回复响应超时数，格式为"XXX/XXX/XXX/XXX/..."。(其中N最大值为5，取决于局向当前所处的HTR流控状态，初始状态显示为空"NULL")。 |
| 周期内流控请求数 | 该参数用于显示该局向前N个周期流控请求数，格式为"XXX/XXX/XXX/ ..."(其中N最大值为5，取决于局向当前所处的HTR流控状态，初始状态显示为空"NULL")。 |

## [命令功能](#ZH-CN_MMLREF_0000001915082974)

**适用NF：SMF**

该命令用于查询H-SMF通用控制开关。

## [注意事项](#ZH-CN_MMLREF_0000001915082974)

无

#### [操作用户权限](#ZH-CN_MMLREF_0000001915082974)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

## [参数说明](#ZH-CN_MMLREF_0000001915082974)

无

## [使用实例](#ZH-CN_MMLREF_0000001915082974)

查询H-SMF通用控制开关：

```
%%LST HSMFCOMMONSW:;%%
RETCODE = 0  操作成功

结果如下
------------------------
服务节点IP地址开关 =  不使能
```

## [输出结果说明](#ZH-CN_MMLREF_0000001915082974)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 服务节点IP地址开关 | 该参数用于控制SEPP组网场景下，N40接口消息是否不携带服务节点IP地址。 |

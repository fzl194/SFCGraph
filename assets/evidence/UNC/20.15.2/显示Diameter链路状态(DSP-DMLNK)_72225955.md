#### [命令功能](#ZH-CN_MMLREF_0000001172225955)

**适用网元：SGSN、MME**

该命令用于查看Diameter链路状态。

#### [注意事项](#ZH-CN_MMLREF_0000001172225955)

- 该命令执行后立即生效。
- 如果查询结果中某条链路的状态信息显示为NULL，可以确定该链路所在SGP进程故障。告警台会存在告警ID为1003的“模块故障”告警。

#### [本地用户权限](#ZH-CN_MMLREF_0000001172225955)

manage-ug；system-ug；monitor-ug

#### [网管用户权限](#ZH-CN_MMLREF_0000001172225955)

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

#### [参数说明](#ZH-CN_MMLREF_0000001172225955)

| 参数标识 | 参数名称 | 参数说明 |
| --- | --- | --- |
| LINKIDX | 链路索引 | 可选必选说明：可选参数<br>参数含义：该参数用于指定待查询的Diameter链路的索引。<br>取值范围：0~1279<br>默认值：无<br>说明：如果不输入，表示查询系统内所有Diameter链路的状态。 |

#### [使用实例](#ZH-CN_MMLREF_0000001172225955)

1. 不输入查询参数，查询所有Diameter链路状态信息：
  DSP DMLNK:;
  ```
  %%DSP DMLNK:;%%
  RETCODE = 0  操作成功。

  输出结果如下
  --------------
  链路索引    链路名称     RU名称            进程号        链路状态

  2           noname       USN_SP_RU_0066    1              故障        
  1           noname       USN_SP_RU_0066    3              故障        
  0           noname       USN_SP_RU_0066    4              故障        
  6           noname       USN_SP_RU_0066    2              故障        
  (结果个数 = 4)
  ---    END
  ```
2. 输入Diameter链路索引，查询指定的Diameter链路状态信息：
  DSP DMLNK: LINKIDX=0;
  ```
  %%DSP DMLNK: LINKIDX=0;%%
  RETCODE = 0  操作成功。

  输出结果如下
  --------------

     链路索引  =  0
     链路名称  =  noname
       RU名称  =  USN_SP_RU_0066
     进程号   =  4
     链路状态  =  故障
  (结果个数 = 1)
  ---    END
  ```

#### [输出结果说明](#ZH-CN_MMLREF_0000001172225955)

| 输出项名称 | 输出项解释 |
| --- | --- |
| 链路索引 | Diameter链路的索引，该参数用来在系统范围内部唯一标识一条Diameter链路。 |
| 链路名称 | Diameter链路名称。 |
| RU名称 | 该参数用于指定<br>SPU<br>资源单元名。 |
| 进程号 | 查询出的Diameter链路中的进程号。 |
| 链路状态 | 查询Diameter链路状态的查询结果。<br>取值说明：<br>- “正常”：表示MME与HSS，EIR或GMLC之间的通信在该链路上正常。<br>- “故障”：表示MME与HSS，EIR或GMLC之间的通信在该链路上中断，需要检查网络状况，分析故障原因。<br>- “拥塞”：表示MME与HSS，EIR或GMLC之间的通信在该链路上的数据量达到链路负荷门限。需要调整组网或扩容。 |

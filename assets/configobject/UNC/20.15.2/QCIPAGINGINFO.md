# QCIPAGINGINFO（QCI寻呼策略参数配置）

## 说明

**适用网元：MME**

该命令用于增加QCI（QoS Class Identifier）寻呼策略。系统可以通过对特性QCI的寻呼策略参数配置来决定寻呼的等待间隔和次数。

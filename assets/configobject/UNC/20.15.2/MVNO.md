# MVNO（MVNO配置信息）

## 说明

**适用网元：SGSN、MME**

此命令用于增加MVNO的配置。MVNO（Mobile Virtual Network Operator，移动虚拟网络运营商）利用MNO（Mobile Network Operator，移动网络运营商）授权的网络资源，提供业务（包括MNO提供的业务以及MVNO自己定制的业务），并能对授权的资源进行维护。

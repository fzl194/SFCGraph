# EMTCBESM（S1模式eMTC SM协议参数）

## 说明

**适用网元：MME**

该命令用于设置S1模式CE Mode B（一种eMTC终端）用户的SM协议参数。

当网络中有CE Mode B终端处于无线信号覆盖较差的区域导致业务流程成功率较低时，可通过本命令调大定时器时长以改善业务成功率。

# DCNMAP（DCN映射关系）

## 说明

**适用网元：MME**

当部署DCN时，此命令用于根据用户的签约数据UE Usage Type配置指定DCN所服务的用户范围。UE Usage Type表示UE的使用特征，根据UE Usage Type可以选择特定的专用核心网。执行此命令前需要通过 [**ADD DCN**](../DCN配置/增加DCN(ADD DCN)_72225505.md) 增加专用核心网。

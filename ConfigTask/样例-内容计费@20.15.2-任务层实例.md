### `@TaskRule@00001` URRGROUP 必须同时配在线+离线 URR

| 字段 | 值 |
|---|---|
| `rule_id` | `UDG@20.15.2@TaskRule@00001` |
| `task_ref` | `UDG@20.15.2@ConfigTask@00001` |
| `rule_name` | URRGROUP 必须同时配置在线 URR 和离线 URR |
| `rule_type` | `consistency_rule` |
| `rule_logic` | URRGROUP 下必须同时配置在线 URR 和离线 URR（UPURRNAME1/DOWNURRNAME1 指离线、UPURRNAME2/DOWNURRNAME2 指在线）；只配一种且与用户计费模式不匹配时无法计费。编号无优先级语义。 |
| `violation_effect` | 无法计费 |
| `severity` | warning |
| `status` | active |
| `source_evidence_ids` | [`output/.../部署UPF_28493406.md`] |

## ② 命令真相（mml_commands.jsonl）
**功能**：当接口上使能简单流分类，并且报文在接口下行转发检查PHB时，可以使用该命令配置当前域的下行IP报文的服务等级和颜色映射成对应的DSCP值，或者下行VLAN报文在路由器内部的服务等级和报文颜色对应路由器外部的802.1p的值。
**notes（规格/上限→应投影 atom rule）**：
- 该命令执行后立即生效。

| 参数 | 描述 | 数据来源 | 必选 | 默认 | 取值范围 |
|---|---|---|---|---|---|
| DSNAME | DS域名 | local_planned | required | 无 | 字符串类型，输入长度范围为1～31。 |
| PHBTYPE | PHB类型 | local_planned | required | 无 | 枚举类型。 |
| PHBVALUE | PHB值 | local_planned | required | 无 | 整数类型，取值范围为0～63。802.1p的取值范围是0到7;DSCP值的范围是0到63。取值越大优 |

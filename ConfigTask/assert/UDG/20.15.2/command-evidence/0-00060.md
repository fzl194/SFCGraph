## ② 命令真相（mml_commands.jsonl）
**功能**：该命令用于修改接口的IPv6使能情况，用户可通过该命令修改接口的IPv6 MTU和Autolinklocal标志。
**notes（规格/上限→应投影 atom rule）**：
- - 该命令执行后立即生效。
- 该命令在VNRS_VNFC上的Ethernet接口，Ethernet子接口，Eth-Trunk接口，Eth-Trunk子接口以及Loopback口，Tunnel口上配置执行。
- 缺省情况下，接口上不使能IPv6。
- 若IPv4 MTU与IPv6 MTU都配置的时候，实际取两者之间的较大值配置至vNic。

**参数真相表（代码解析）**：

| 参数 | 描述 | 数据来源 | 必选 | 默认 | 取值范围 |
|---|---|---|---|---|---|
| IFNAME | 接口名 | local_planned | required | 无 | 字符串类型，输入长度范围为1～63。接口名称由接口类型+接口编号组成。 |
| ENABLEFLAG | IPv6使能标志 | local_planned | required | 无 | 枚举类型。 |
| MTU | IPv6接口最大传输单元（byte） | local_planned | conditional | 无 | 整数类型，取值范围为1280～9600。 |
| AUTOLINKLOCAL | 自动生成链路层地址标志 | local_planned | conditional | 无 | 布尔类型，输入格式为“TRUE”或者“FALSE”。TRUE：接口自动生成链路地址，FALSE：接口 |

## ② 命令真相（mml_commands.jsonl）
**功能**：**适用NF：SGW-U、PGW-U、UPF**

该命令用来增加对端网元所属的GTP数据面IP地址段，即配置GTP路径管理黑白名单中的地址段。
**notes（规格/上限→应投影 atom rule）**：
- - 该命令执行后立即生效。
- 该命令最大记录数为200。
- ADD ECHOIPLIST是配合SET UPGTPPATH使用的。
- 使用LST UPGTPPATH查看EchoList类型为黑名单还是白名单。
- 如果对端网元的IP地址落在ADD ECHOIPLIST配置的地址段内：
    - 配置GTP路径管理名单属性为白名单时，系统向对端网元发送GTP信令路径Echo探测或是GTP数据路

**参数真相表（代码解析）**：

| 参数 | 描述 | 数据来源 | 必选 | 默认 | 取值范围 |
|---|---|---|---|---|---|
| IPVERSION | IP地址版本类型 | global_planned | required | IPV4 | 枚举类型。 |
| IPV4ADDR | IPv4地址 | local_planned | conditional | 无 | IPv4地址类型。 |
| IPV4MASK | IP掩码 | local_planned | conditional | 无 | 整数类型，取值范围为1～32。 |
| IPV6ADDR | IPv6地址 | local_planned | conditional | 无 | IPv6地址类型。 |
| IPV6MASK | IP掩码 | local_planned | conditional | 无 | 整数类型，取值范围为1～128。 |

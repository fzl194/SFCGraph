## ② 命令真相（mml_commands.jsonl）
**功能**：该命令用于配置接口的IP地址，包括逻辑接口及物理接口。物理接口是真实存在的接口。逻辑接口是指能够实现数据交换功能但物理上不存在、需要通过配置建立的接口。
**notes（规格/上限→应投影 atom rule）**：
- - 该命令执行后立即生效。
- 该命令最大记录数为65535。
- 该接口需要支持配置IP地址。
- 该命令在版本升级过程中禁止执行。
- 每个主用接口最大允许配置256个接口IP地址：1个主IP地址和255个从IP地址。
- 该命令的必选参数IFNAME可通过LST INTERFACE命令查询到。

**参数真相表（代码解析）**：

| 参数 | 描述 | 数据来源 | 必选 | 默认 | 取值范围 |
|---|---|---|---|---|---|
| IFNAME | 接口名 | local_planned | required | 无 | 字符串类型，输入长度范围为1～63。接口名称由接口类型+接口编号组成。 |
| IFIPADDR | IPv4地址 | local_planned | required | 无 | IPv4地址类型。0.0.0.0～255.255.255.255。 |
| SUBNETMASK | IPv4地址掩码 | local_planned | required | 无 | IPv4地址类型。0.0.0.0～255.255.255.255。 |
| ADDRTYPE | IPv4地址类型 | local_planned | optional | main | 枚举类型。 |

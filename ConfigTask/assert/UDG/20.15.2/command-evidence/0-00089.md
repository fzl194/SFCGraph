## ② 命令真相（mml_commands.jsonl）
**功能**：**适用NF：PGW-U、UPF**

此命令用于配置错误码信息。运营商可以根据报文中错误码进行不同的重定向动作处理。
**notes（规格/上限→应投影 atom rule）**：
- - 该命令执行后立即生效。
- 该命令最大记录数为100。
- 单个ErrorCodeName下最多配置500组错误码范围。
- 整机最多可以配置2000组错误码范围。

**参数真相表（代码解析）**：

| 参数 | 描述 | 数据来源 | 必选 | 默认 | 取值范围 |
|---|---|---|---|---|---|
| ERRORCODENAME | 错误码名称 | global_planned | required | 无 | 字符串类型，输入长度范围为1～31。不区分大小写，不支持空格。 |
| ERRORCODEOP | 错误码范围操作码 | local_planned | required | 无 | 枚举类型。 |
| ERRORCODESTART | 错误码范围起始值 | local_planned | conditional | 无 | 整数类型，取值范围为0～1000。 |
| ERRORCODEEND | 错误码范围终止值 | local_planned | conditional | 无 | 整数类型，取值范围为0～1000。 |

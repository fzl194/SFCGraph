## ② 命令真相（mml_commands.jsonl）
**功能**：**适用NF：PGW-U、UPF**

该命令用于配置Gy接口一次重定向的URL白名单列表，并配置是否包含重定向携带信息。匹配上该白名单的报文将被重定向到OCS下发的URL。
**notes（规格/上限→应投影 atom rule）**：
- - 该命令执行后立即生效。
- 该命令最大记录数为1。
- 若命令配置为重定向URL中可包含重定向携带信息。而如果重定向携带信息包含原始请求URL，名称为reqUrlName，携带用户的MSISDN，名称为msisdnName，携带用户的IMSI，名称为imsiName，携带用户的IMEI，名称为imeiName，携带用户的IP信息，名称为msipName，则URL重定向中报文内容为： Locat

| 参数 | 描述 | 数据来源 | 必选 | 默认 | 取值范围 |

| EXTFLTNAME | 扩展过滤器名字 | local_planned | optional | 无 | 字符串类型，输入长度范围为1～31。不支持空格，不区分大小写。 |
| REDIRAGETIME | 重定向完成时间（秒） | local_planned | optional | 无 | 整数类型，取值范围为5～600。 |

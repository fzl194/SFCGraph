**功能**：**适用NF：PGW-U、UPF**

该命令用于配置系统进行PPP协商时所使用的参数，包括系统侧与用户鉴权时使用的主机名、系统的最大接收单元、PPP协商请求的超时时间。
**notes（规格/上限→应投影 atom rule）**：
- - 该命令执行后立即生效。
- 该命令最大记录数为1。
- 该命令存在系统初始记录，参数的初始设置值如下表：

| 参数标识 | HOSTNAME | MRU | TIMEOUT |
| --- | --- | --- | --- |
| 初始值 | UPF | 1500 | 3 |

| 参数 | 描述 | 数据来源 | 必选 | 默认 | 取值范围 |
|---|---|---|---|---|---|
| HOSTNAME | 本端主机名称 | local_planned | optional | 无 | 字符串类型，输入长度范围为1～31。区分大小写。 |
| MRU | 最大接收单元 | local_planned | optional | 无 | 整数类型，取值范围为64～1500。 |
| TIMEOUT | 超时时间（秒） | local_planned | optional | 无 | 整数类型，取值范围为1～10，单位是秒。 |

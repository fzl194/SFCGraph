## ② 命令真相（mml_commands.jsonl）
**功能**：**适用网元：SGSN**

该命令用于增加缺省计费属性参数。缺省计费属性是指当外网用户签约的HLR没有为外网用户指定计费属性时，SGSN将会按照该外网用户的MCC和MNC对应的缺省计费方式对该外网用户实施计费。
**notes（规格/上限→应投影 atom rule）**：
- - 该命令执行后立即生效。
- “用户类型（ST）”相同的记录数最大为10，本表记录总数不超过20。
- 对于具有相同ST的任两条记录，如果MCC相同，有效长度为2和3的MNC的前两位不允许相同。
- S-CDR中填写的计费属性按照如下的优先等级进行选择：
    1. 通过 [**LST CHGGNCCCFG**](../Gn接口计费属性选择策略配置/查询Gn接口计费属性选择策略(LST CHG

| 参数 | 描述 | 数据来源 | 必选 | 默认 | 取值范围 |

| ST | 用户类型 | 整网规划 | required | 无 | <br>- “ROAMING(漫游用户)”：表示使用归属PLMN的GGSN的外网用户。 |
| MCC | 移动国家码 | 整网规划 | required | 无 | 位数为3的十进制数字 |
| MNC | 移动网号 | 整网规划 | required | 无 | 位数为2或3的十进制数字 |
